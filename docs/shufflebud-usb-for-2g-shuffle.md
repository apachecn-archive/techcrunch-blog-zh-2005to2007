# 用于 2G Shuffle 的 Shuffle | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/04/shufflebud-usb-for-2g-shuffle/>

如果你自豪地拥有 2G iPod Shuffle，但似乎不习惯需要基座来同步，那么看看 Incipio 的 ShuffleBud 吧。它是一个小适配器，连接到 Shuffle 上的 3.5 毫米端口，以提供 USB 访问。

如果你选择使用它，你可以放弃携带 Shuffle 的基座和电缆，因为 ShuffleBud 即使缩小了尺寸，也可以完成所有相同的任务。哦，最棒的是它只需要 6 美元。
