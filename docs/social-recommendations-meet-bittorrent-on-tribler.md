# 社交推荐在 Tribler 上遇到 BitTorrent

由代尔夫特理工大学和阿姆斯特丹自由大学开发的 BitTorrent 客户端为 BitTorrent 世界带来了动态的社交推荐。

[Tribler](https://web.archive.org/web/20221130211239/https://www.tribler.org/)4.0 版本采用了流行的概念，并赋予其坚实的 Web 2.0 扭曲。和 Last.fm 风格一样，Tribler 根据用户的历史记录推荐下载。

客户端还支持 YouTube 浏览。Tribler 背后的引擎使用从观看 YouTube 视频收集的数据，结合下载历史，来推荐新的视频观看。

将各种 Web 2.0 思想融合到 BitTorrent 客户端中可能是发展的逻辑进步，但瑞士军队的方法会对广大受众产生巨大的吸引力吗？

Triber 是作为一个研究项目的一部分而设计的，该项目有 20 名科学家参与，€为此投入了 600 万英镑。作为一个研究项目，它很有趣，但如果这个概念很强，它只能通过 BitTorrent 使用社区未来对 Tribler 的下载和使用来证明。
