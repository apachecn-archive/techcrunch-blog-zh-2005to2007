# PlayStation 3 玩 PS2 游戏有问题

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/14/playstation-3-has-problems-playing-ps2-games/>

![](img/1e703a63ec99d18b2f9fd8da5dbb3d53.png)

索尼已经承认 PlayStation 3 在 PlayStation 2 游戏上存在问题。问题出在新旧 PS2 游戏上，包括《最终幻想》系列。到目前为止，PS3 在显示菜单和图形时遇到了问题。计划在 PS3 上玩新发布的 PS2 游戏如*最终幻想 XII* 的用户将会有一个软件更新。我知道有不少人卖掉了 PS2，以便有足够的钱购买 PlayStation 3。如果索尼不尽快发布新游戏机的补丁，将会有很多人感到愤怒。不过这不全是索尼的问题，微软去年的 Xbox 360 也面临类似的问题，不过问题很快就得到解决。让我们希望下周一切都会好起来。
