# 泛泰推出紧凑型 WinMo 智能手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/27/pantech-launching-compact-winmo-smartphone/>

。泛泰不会在舞会之夜哭泣，所以他们计划开发一款运行 Windows Mobile 5 的智能手机，但像摩托罗拉 L7 SLVR 一样小。将会有两个滑出式键盘，一个是像 Cingular 8125 或威瑞森 XV8700 那样水平滑出的全 QWERTY 键盘，另一个是垂直滑出的小键盘，这样你就可以拨号，像三星的滑盖手机一样。

没有发布日期的消息，但是天才少年很快会有一些照片。如果有一样东西我们几乎和女人、婴儿、小狗、小猫、汽车、船、云杉鹅和的照片一样喜爱，那就是手机的照片。
