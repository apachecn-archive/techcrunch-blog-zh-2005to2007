# Prosper 推出–社会贷款

 [](https://web.archive.org/web/20120520032744/http://www.prosper.com/) 总部位于三藩市的 [Prosper](https://web.archive.org/web/20120520032744/http://www.prosper.com/) 今天推出(Prosper 在其隐身阶段被[称为 Circleone](https://web.archive.org/web/20120520032744/http://www.techcrunch.com/2005/10/27/circleone-rumored-to-launch-soon/) )。

像总部设在伦敦的(并由基准银行资助的)Zopa 一样，Prosper 是借款人和贷款人的市场。

个人借款人说他们需要借多少(最多 25000 美元)和他们将支付的最高利率。感兴趣的贷方说他们想借多少，以什么样的利率借。Prosper 将价格最低的贷款人组合成一笔贷款。

![](img/b119b3a5e7e533498120bed20e764208.png)

根据 Prosper 的说法，完成一笔贷款通常需要 2-4 天。然而，借款人可以要求加急贷款，这将采取符合最低要求的金额和利率的第一贷款人。

Prosper [收取多项费用，包括向借款人收取 1%的成交费，向贷款人收取 0.5%的年度贷款维护费。](https://web.archive.org/web/20120520032744/http://www.prosper.com/public/help/topics/other-fees.aspx)

Benchmark 资助了 Zopa 和 Prosper。其他繁荣投资者包括 Accel、Fidelity 和 Omidyar。
