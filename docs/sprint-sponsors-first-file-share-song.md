# Sprint 赞助第一首文件共享歌曲

![biz031.jpg](img/568b19d49a5c6527b8216d3eddc0811d.png)

在过去十年的大部分时间里，唱片公司一直在为 P2P 网络担忧，这是有充分理由的。2000 年销量最高的专辑卖出了近 1000 万张，而 2006 年销量第一的专辑卖出了不到 400 万张。自 2002 年以来，共享文件的个人数量几乎翻了一番，2006 年有 940 万用户在网上共享各种东西。Sprint 希望他们能够扭转局面，或者至少从炒作中获利。

Sprint 与大西洋唱片公司和 ArtistDirect 的 Media Defender 合作，发布了第一首有主要支持者的文件共享歌曲。嘻哈艺术家 Plies 将成为为期三个月的试验品，他的 1600 万首歌曲将被上传到文件共享网站。当你下载任何 Plies 歌曲时，你会看到一个嵌入的 Sprint 标志，旁边是艺术家的名字和曲目名称。作为回报，大西洋唱片公司、Media Defender、Plies 和他的出版公司将分享一笔“可观的六位数”费用。Plies 是谁？
