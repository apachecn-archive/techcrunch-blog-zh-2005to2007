# 松下 PT-AE1000:全 1080p 高清家庭影院

松下试图在这个假期带着它新的 AE1000 投影仪来到你的树下。这个光滑的黑盒子能以 1080 便士的最高分辨率播放你的视频。

更重要的是，它有 11，000:1 的对比度，这是总热度。松下将这款投影仪瞄准了高端家庭影院爱好者，尽管你应该可以找到低于 5000 美元的投影仪，这确实将它带入了 1080p 的中档(对于那些用 100 美元作为卫生纸的人来说)。

我们喜欢 2 倍光学变焦和球面透镜系统。理论上，它应该使图像更清晰，对比度更高，是的，这是一个词——查一下(编辑:实际上请不要)。当然，我们还没有亲眼见过，所以这可能只是一堆松下的公关噪音。
