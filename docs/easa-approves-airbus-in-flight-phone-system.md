# EASA 批准空客飞机上的电话系统

![airbus.png](img/2455a2e2a9337235301ee505503f2f16.png)
住在欧洲？比如安静的飞行？当心；欧洲飞机安全局(EASA)最近批准了空客机载移动电话系统的试运行。幸运的是，该系统可以设置为某种“安静模式”，在这种模式下，只能发送和接收数据传输。尽管如此，充满闲聊的旅行还是有可能的。你只想小睡一会儿。
