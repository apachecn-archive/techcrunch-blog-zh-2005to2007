# 新手机功能:空气传播病原体和生物威胁预警系统

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/02/new-phones-feature-airborne-pathogen-and-bio-threat-warning-system/>

如果你像布莱克一样，那么你会担心细菌、生物恐怖主义、标准恐怖主义、禽流感、非典、嬉皮士、尘螨以及任何以“痘”结尾的东西如果 Gentag 的这款新手机能够投入生产，他将会爱上它的技术。

作为一个概念，它没有真正的规格清单，但它似乎是一个相当标准的蛤壳。让这款手机脱颖而出的是一个传感器，它可以提醒用户空气中潜在的有害物质，生物或化学物质。你可以选择你想被警告的内容，剩下的就交给它了。对于我们这些患有哮喘和/或忧郁症的人来说，这是一项伟大的技术。
