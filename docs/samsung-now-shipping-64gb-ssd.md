# 三星现在推出 64GB 固态硬盘

![](img/c4bac05549b9c00517f3583633b68c6f.png)

今天早上，三星宣布将开始发运其新的 1.8 英寸 64GB 固态硬盘。那是 64GB 的色情数据，可以快速访问。使这款固态硬盘流行的一些功能包括更快的启动时间，防跳过数据保护，因为没有移动部件，以及更长的产品寿命。它还使用更少的电力，这意味着没有必要使用你上周看到的 600 瓦电源。

由于驱动器是针对 UMPC 和平板电脑制造商，没有给出价格。但由于固态硬盘还没有那么普遍，我想它不会那么便宜。也许三年后这些会变得一毛钱一打，但不是今天。
