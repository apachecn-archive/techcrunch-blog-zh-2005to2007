# RIAA 对苹果:更多的数字版权管理

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/07/riaa-to-apple-more-drm-please/>

。RIAA 认为苹果只需要向竞争对手开放它的 DRM，而不是用清醒的思维承认 DRM 是一个问题而不是一个解决方案。

从表面上看，RIAA 的抱怨是有道理的，其想法是应该有一个全行业的、单一的 DRM 标准。这将使任何便携式设备与任何音乐商店兼容。问题是，这给了 RIAA 直接控制许可证清算所，这意味着他们将最终控制你的歌曲，而不是苹果，微软，敦促或任何其他人。

所以实际上，RIAA 的意思是，乔布斯的无数字版权管理音乐的想法并不能让人们购买更多，但通用的数字版权管理是关键。RIAA 又一次只见树木不见森林。
