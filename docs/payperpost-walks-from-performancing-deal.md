# PayPerPost 的业余时间

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/01/04/payperpost-walks-from-performancing-deal/>

一家公司[宣布](https://web.archive.org/web/20220818073904/http://blog.payperpost.com/2006/12/payperpost-acquires-performancingcom.html)收购另一家公司，然后又退出交易，这种情况并不常见，但 PayPerPost 不是一家典型的公司。

今天，在 PayPerPost 博客上的一篇[帖子](https://web.archive.org/web/20220818073904/http://blog.payperpost.com/2007/01/performancing-deal-is-off.html)中，该公司表示*“我们……挖掘了度量平台，遗憾地发现这不是我们现在正在寻找的。”这发生在正式宣布收购的一周之后。*

一般来说，负责任的公司在宣布交易之前会“深入研究”收购目标。

不管发生了什么，这都不太好。交易宣布后，Performancing 将他们未收购的资产转移到一个新的域名中，并重新启动了这项服务。鉴于交易已经正式宣布，他们当然停止了与其他潜在收购者的谈判。在并购领域，这就是所谓的“被留在祭坛上”，因为你与之交谈的每个人都想知道为什么之前的交易会失败。

Performancing 应该有一个更明确的收购协议，所以他们不是完全没有责任。但是 PayPerPost 正在成为一家越来越荒谬的初创公司，并且成为投资者 Draper Fisher Jurvetson 的眼中钉。
