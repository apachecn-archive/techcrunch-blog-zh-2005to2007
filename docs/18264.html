因此 Plaxo的新社交网络，名为Pulse ( 我们上个月写过)，仍然漏洞百出。罗伯特·斯考伯和马特·马歇尔说它会在周一正式发布，所以希望到时候他们能解决所有的问题。目前，它是活的，但不是真的工作。

Plaxo有效地将自己变成了一个社交网络。就用户而言，最大的重叠将是专业人群——LinkedIn。但是新的功能和界面都是脸书的。特别是，用户可以添加自己的数据流，如博客帖子、flickr照片和亚马逊愿望清单。然后你可以在网站上看到你所有朋友的信息流。它与脸书去年推出的新闻订阅几乎相同。

Nick Gonzalez喜欢Pulse是一个开放平台的事实，并将其比作脸书。我不反对他写的东西。但是我注意到一些读者对那篇文章相当恶毒的评论，他们说，在Plaxo公司早期的所有垃圾邮件投诉之后，他们再也不能信任它了。

用户会涌向Plaxo并像今天使用脸书一样使用Pulse吗？我认为他们不会。脸书发明了社交网络的新闻提要。追赶总是很难。当你还在与声誉问题作斗争时，事情会变得更加困难。

即便如此，在很久以前放弃Plaxo服务后，我还是在Plaxo上创建了一个新账户。我喜欢新闻提要不局限于Plaxo应用程序，我打算给它一个公平的机会。也许这会转移我对脸书的部分注意力。我们走着瞧。
