# VP180 Skype 鼠标

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/15/vp180-skype-mouse/>

如果有人有任何理由认为鼠标不应该与 Skype 电话结合在一起，现在就说出来，否则就永远保持沉默。没人吗？很好。就像 [USB Hub +蓝牙加密狗](https://web.archive.org/web/20210228041938/http://crunchgear.com/2006/08/15/four-port-usb-hub-with-bluetooth-dongle/)和[安防摄像头+加热器](https://web.archive.org/web/20210228041938/http://crunchgear.com/2006/08/15/520tvl-security-camera-with-built-in-heater/)一样，这个东西把我们喜欢的两样东西，合二为一成了我们喜欢的一样东西。

这款鼠标的工作原理和普通的双键滚轮鼠标一样，但是当你把它翻转过来时，会有一个小键盘和通话/结束按钮。遗憾的是，没有屏幕，所以你只能在显示器上切换 Skype 联系人。

我们可以看到的唯一缺点是，键盘和鼠标可能是你桌子上最脏的部分，所以把它放到嘴边可能不是个好主意。随便说说。
