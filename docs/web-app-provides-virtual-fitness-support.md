# Web 应用提供虚拟健身支持| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/12/08/web-app-provides-virtual-fitness-support/>

[![traneo_logo.gif](img/cb4bea69e1acd0ae757ed27081638322.png)](https://web.archive.org/web/20160414164801/http://www.traineo.com/) 我在研究生院教瑜伽和动感单车，所以健康和健身是我略知一二的话题。据我所见，这个行业是采用新技术较慢的行业之一，这就是为什么我很高兴找到 [Traineo](https://web.archive.org/web/20160414164801/http://www.traineo.com/) 。

现在，我知道 TechCrunch 的读者可能不会对个人健身软件感到兴奋，但在你尖叫之前，“为什么这在 TechCrunch 上？”在评论部分，考虑到 Traineo 最近与一些公司达成了令人印象深刻的赞助协议，如[碧欧泉](https://web.archive.org/web/20160414164801/http://biotherm.groups.traineo.com/)和欧莱雅。在线社会支持团体是可以赚钱的，所以请听我说完。

Traineo 是一个健康和健身支持网站，用户每天向一群朋友报告他们的体重、饮食和锻炼情况，这些朋友互相帮助监督和支持。我不愿称 Traineo 为社交网络，因为与 MySpace 不同，它的目标不是拥有尽可能多的朋友。目标是有一个你信任的精选小组来帮助你度过保持身材的苦差事。

我以前和我的朋友一起尝试过很多这样的在线程序。《自我》杂志今年早些时候有一个旨在让我们所有人“正好赶上泳装季节”的软件，但软件很可怜，社交网络方面甚至更糟。我还尝试了基于订阅的 MyFoodDiary 和 PEERTrainer，我认为这需要用户做太多的工作。Traineo 是免费的，非常简单。

Traineo 不需要输入你在玉米煎饼里吃了多少盎司奶酪，在咖啡里放了多少奶油，它只允许你估计一下你每天的饮食情况。根据你的年龄、体重和目标，它会告诉你应该达到多少卡路里的目标，在一天结束时，你会告诉程序你吃得是差、一般、一般还是好。

“我们真的把事情简单化了，”Traineo 的创始人之一 Alasdair McLean-Foreman 通过 IM 说道。“人们发现坚持节食/锻炼很难，所以软件应该是最容易的部分，否则人们会放弃并失去兴趣。”

锻炼日志也很简单。你从可用的锻炼类型中选择，告诉它你做了多长时间，以及你感觉强度如何。我告诉 Traineo 我想减掉 4 磅，称重让我可以定期更新来跟踪我的进展。

对自己的进步不那么害羞的用户可以加入更大的支持团体，要么是赞助团体，要么是围绕不同人口统计数据的团体，如母亲或技术爱好者。事实上，我没有找到一个技术爱好者团体，但我创建了一个名为 TechCrunch 的团体。请随意加入。

Traineo 是由一些非常健康的软件开发人员编写的。许多运动员为该公司工作，包括麦克林-福尔曼，他曾代表英国参加田径比赛，是 2004 年哈佛田径队队长。

其他人也喜欢 Traineo。在[对新型健身网络服务的综述中，CenterNetworks 称其为“一个优秀的全方位应用”](https://web.archive.org/web/20160414164801/http://www.centernetworks.com/web-2-0-fat-off-challenge)
