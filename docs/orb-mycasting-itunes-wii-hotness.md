# orb my casting:iTunes+Wii = Hotness

Orb MyCasting 是一个可以让你将音乐和照片从电脑传输到 Wii 的程序。使用 Wii 的内置 Opera 浏览器，Orb 在你的子网内创建一个迷你服务器，并在 Wii 的屏幕上显示你的内容。

这并不是什么令人惊讶的消息，但这显然是一个很好的方式来避开目前围绕流媒体(MPSCSSM (c))的大量和普遍的愚蠢行为。如果在你的磁盘上，它就是你的，对吗？对吗？
