# 裸奔搜索让社交网络变得无所不包

[![streakrlogo.png](img/34053f31818ba5e67b7fd9855de1dc05.png) ](https://web.archive.org/web/20220811003529/http://www.streakr.com/) Vivek，over at Startup Squad，最近[发现了](https://web.archive.org/web/20220811003529/http://startupsquad.com/2007/03/15/streakr-search-across-social-networks/)一个新的社交网络和社交网络元搜索引擎[裸奔](https://web.archive.org/web/20220811003529/http://www.streakr.com/)。主 URL 仍然显示[网站即将推出](https://web.archive.org/web/20220811003529/http://streakr.com/)。新的引擎可以让你在主流网络(MySpace、Hi5、Bebo 和脸书)以及它自己的社交网络上搜索个人资料。这似乎是一个吸引人们进入他们主要服务的诱饵，就像 Wink 推出自己的个人资料搜索和 Rapleaf 推出 UpScoop 一样。档案管理工具[profile link](https://web.archive.org/web/20220811003529/http://www.beta.techcrunch.com/2006/12/28/profilelinker-takes-meebo-approach-to-social-networking/)也有一个搜索引擎。

裸奔者的社交网络对酷孩子来说就像美味，对 Trig 来说不那么花哨。它包括一个个人资料，一个工具栏，和一个偶然发现的功能，可以让你浏览给定类别中的链接。这里是视频的[一个。](https://web.archive.org/web/20220811003529/http://www.streakr.com/streakr/?cat=4&x=34&y=46)

这些简介看起来很像 MySpace，由常见的细节、关于我、照片和诱发癫痫的布局组成。[森雅](https://web.archive.org/web/20220811003529/http://www.streakr.com/Xania)是裸奔者的汤姆。然而，在 MySpace 放博客和评论的地方，裸奔会放上你最喜欢的链接和你对每个链接的“赞”评级。您可以手动将链接输入到您的个人资料中，或者使用裸奔工具栏将链接添加到您的个人资料中并对其进行投票。工具栏还提供了一个到主站点上所有其他功能的接口，目前仅用于 IE，需要。NET framework，下载和安装需要一天的时间。

还有几个其他网站提供社交网络元搜索。以下是一些内幕:

**眨眼**
[眨眼](https://web.archive.org/web/20220811003529/http://wink.com/)快速简单。它搜索 Friendster、MySpace、Bebo、LinkedIn 和 Live Spaces。它还具有高级搜索功能，如位置、性别、身份、年龄和兴趣。它还允许您在第一次搜索后通过这些字段缩小搜索范围。

**profile link**
[profile link](https://web.archive.org/web/20220811003529/http://profilelinker.com/)是最全面的搜索引擎，拥有 84 个社交网站，包括综合、博客、文化、交友、专业、学生和特殊兴趣网络。不幸的是，你必须登录才能使用它。

**up scoop**
[up scoop](https://web.archive.org/web/20220811003529/http://upscoop.com/)领先于易用性。不像其他的，UpScoop 根据你地址簿中的所有联系人通过电子邮件进行搜索。它搜索 Bebo、同学会、Ecademy、Flickr、Friendster、Hi5、Livejournal、Multiply、MySpace、Ringo、Tickle、Tribe、Yelp、Mog 和 LinkedIn。虽然它可以一下子找到你的大多数朋友，但它的一些缺点是，搜索最后几个朋友可能需要几个小时，并且需要交出你的电子邮件凭据。
