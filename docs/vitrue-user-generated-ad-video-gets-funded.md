# ViTrue:用户生成的广告视频获得新投资者

[![](img/b2d39d01ac18dfb07caa6cdd26ffd5c3.png)](https://web.archive.org/web/20201205113908/http://www.vitrue.com/) 用户生成视频分享网站 [ViTrue，Inc.](https://web.archive.org/web/20201205113908/http://www.vitrue.com/) 来了一些声誉良好的新投资者，其中包括谷歌的原始投资者之一、常被称为“硅谷教父”的罗恩·康韦上个月，该公司宣布了 Comcast Interactive Capital 和 Turner Broadcasting 的重大系列“A”轮融资——尽管他们只能说是个位数的数百万美元。其他最近的投资者包括考克斯战略执行副总裁达拉斯·克莱门特(Dallas Clement)、Landmark Communications 的创始高管巴恩斯·斯坦利(Bahns Stanley)和其他一些人。

ViTrue 托管用户生成(UG)视频，类似于 YouTube，以广告为重点。这种相对较新的广告类型被称为品牌视频社区，旨在促进“品牌参与”所以如果你爱旧金山巨人队，(谁不爱呢？)，可以创建一个推广团队的视频，通过 ViTrue 发布。

目前，ViTrue 只有 Moe's Southwest Grill、Lance Foods、辛辛那提孟加拉虎队和一个叫做[的书呆子联盟](https://web.archive.org/web/20201205113908/http://nerdleague.sharkle.com/)的自制视频。不完全是我想“接触”的品牌，但我不是说这种广告不会有效。现在下结论还为时过早，但泰德·特纳对此深信不疑。

也许这就是在线视频世界最终解决如何将 UG 视频货币化并避免版权诉讼的方法？天知道 YouTube 值多少钱，但谷歌一直没怎么谈论他们将如何从收购中获利。此外，他们还不得不过滤非法上传的版权材料。因此，如果赞助 UG 内容是新的计划，ViTrue 只能希望和祈祷有人会为 Moe 的西南烧烤提供一些搞笑的东西。(投稿人注意:竞技吃永远是赢家！)

特纳广播公司将在今年晚些时候测试这一模式，届时它将发起一项名为[“有趣还是不有趣？”](https://web.archive.org/web/20201205113908/http://tbs.sharkle.com/index.html)

ViTrue 对 UG 视频并不陌生。该公司拥有[Sharkle.com，](https://web.archive.org/web/20201205113908/http://www.sharkle.com/)在线视频分享社区。
