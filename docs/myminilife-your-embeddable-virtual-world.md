# MyMiniLife:你的嵌入式虚拟世界

 [](https://web.archive.org/web/20220929152204/http://www.myminilife.com/) [MyMiniLife](https://web.archive.org/web/20220929152204/http://www.myminilife.com/) 是一个基于 Flash 的虚拟世界/社交网络。用户创建并定制一个角色，然后建造一个虚拟空间，添加墙壁、地板、门、窗户等。然后，用户可以在空间中添加从灯到大炮的定制商品，并在商品中嵌入视频或照片元素，或链接到网页(例如，在嵌入中单击我的轻便摩托车)。

有些用户创造幻想世界；其他人则创造了几乎与他们真实的家或公寓一模一样的复制品。其他用户可以参观你的家，四处走动，查看图片和视频，然后留下笔记或与主人实时聊天。所有的物品都需要“钱”来添加到世界中(并且可以在以后出售以获得现金)——并且一些高级物品相对于你开始游戏所获得的现金来说是非常昂贵的。然而，该网站是免费使用的，并且支持广告。更多的钱是通过每日登录和其他行动积累起来的。

图形相当简单，但是很容易定制我的头像和建立一个简单的空间。与 Habbo Hotel 和 Cyworld 等更成熟的竞争对手不同，MyMiniLife 允许用户通过 Flash 小工具在任何其他网站上嵌入他们的世界，例如他们的 MySpace 页面。
