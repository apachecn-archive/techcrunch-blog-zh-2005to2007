# Proporta 发布 iPod Nano 防溅运动套件

![](img/f7ee1827ad38a39d1e0766e4eaef39fe.png)

如果你真的像你一样笨手笨脚，那么你应该知道一个好的外壳对你的个人电子产品的重要性。Proporta 刚刚为 iPod Nano 2G 发布了新的防溅运动套件。准备好接受打击，它配备了双层硅胶外壳、保护性透明屏幕、可调节臂带、挂绳和皮带夹，以防你需要看起来像 43 岁。

这种防溅美容产品现在已经上市，售价约为 40 美元。水手们欢呼吧！
