# Napster 和摩托罗拉达成协议

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/24/napster-and-motorola-enter-agreement/>

我们是否看到摩托罗拉在努力应对即将到来的 iPhone？据 PhoneContent.com 称，Napster 点播音乐订阅服务将很快转向手机。该网站报道称，Napster 和摩托罗拉已经宣布了一项营销协议，将通过摩托罗拉的便携式订阅设备系列提供对 Napster 音乐库和其他内容的轻松访问。这两家公司正在北美、英国和德国开展推广活动，为一系列摩托罗拉音乐手机提供 Napster 音乐体验，包括 ROKR Z6 和 Z6m 手机。

据 PhoneContent.com 报道，该营销协议的第一阶段是为摩托罗拉手机用户提供一个月的 Napster to Go 免费订阅服务。这将通过零售和美国和欧洲的运营商以包装盒内和售后促销的形式提供。Napster to Go 与具有 Windows Media DRM 功能的摩托罗拉手机兼容，这一推广活动预计将于今年下半年启动。
