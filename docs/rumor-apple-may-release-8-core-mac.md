# 传言:苹果可能发布 8 核 Mac 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/26/rumor-apple-may-release-8-core-mac/>

![](img/f21fbbecd5ce1a07f27bd48077a507e6.png)

距离苹果发布 Mac Pro 只有几个月的时间，它配备了两个 2.66GHz 的双核英特尔至强“Woodcrest”处理器，但现在有传言称，苹果正准备发布一款总共有八个内核的新电脑。如果新的八核 Mac 真的发布，它可能会采用两个英特尔的四核至强“Clovertown”处理器，这将为要求苛刻的用户提供他们一直渴望的功能。

就我个人而言，即使我使用强大的定制 Mac Pro，我也经常发现 Mac 在运行 Logic Pro 7 或 Pro-Tools 和一堆插件时会发出轧轧声。我希望 8 核 Mac 能够消除这种滞后，打开运行更多应用程序(和插件)的大门，而不是让我同时卖掉我的汽车。谣言还说，我们应该期待在 11 月中旬的某个时候听到关于新 Mac 的一些消息，如果是这样的话，让我们希望它在 12 月前的某个时候发布，这样我们就可以利用圣诞节作为一个借口来挥霍几千美元。
