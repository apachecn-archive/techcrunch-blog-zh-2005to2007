# 罗约变得相关——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/03/16/rojo-gets-relevant/>

[![](img/45d7c488ffa47b117b964bf8078d19dd.png)](https://web.archive.org/web/20211018043057/http://www.rojo.com/)今晚添加了一个伟大的新[相关性功能](https://web.archive.org/web/20211018043057/http://blog.rojo.com/rojoblog/2006/03/new_at_rojo_moj.html)，结果是一种**个性化挖掘**。

你现在可以按日期或相关性查看你的提要，相关性是根据其他用户的投票(比如 diggs)、总阅读量和总用户标签来决定的。结果——这是一个很好的方法来整理你的每日订阅，找到最受欢迎的帖子。

罗约还增加了一些[类别](https://web.archive.org/web/20211018043057/http://www.rojo.com/today/)——突发新闻、娱乐、科技等。，其中包括编辑挑选的提要，并根据相关性和新鲜度进行排名。这是一个从你的订阅源列表之外的来源找到热门帖子的好方法。

罗约在功能方面再次领先，相关性现在是一个热门话题。问题是，这是否足以在极其拥挤的 feed 阅读器领域推动用户采用。理查德问了一个不同的问题:“我想知道这会不会把博客从沉睡中唤醒？”
