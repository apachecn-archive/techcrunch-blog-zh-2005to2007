# 简单命令行的用处

[](https://web.archive.org/web/20220811121045/http://www.sugarcodes.com/)[【Sugarcodes】](https://web.archive.org/web/20220811121045/http://www.sugarcodes.com/)是一个新网站，它有非常基本的功能，但像老网站 [YubNub](https://web.archive.org/web/20220811121045/http://www.yubnub.org/) 一样，对许多用户来说是一个令人敬畏的生产力工具。它基本上只不过是一个命令行，在这里可以使用一个简单的专用命令对任何 web 服务进行搜索。“g books”在谷歌上搜索书籍。“yt U2”在 YouTube 上搜索 U2。方便之处在于，一旦您知道了命令，您就可以从这个命令行执行任何操作，省去了访问站点的步骤。

我们以前写过 [YubNub，包括它在 MobileCrunch 上的](https://web.archive.org/web/20220811121045/http://www.beta.techcrunch.com/2006/01/15/revisiting-yubnub/)[移动功能](https://web.archive.org/web/20220811121045/http://mobilecrunch.com/2006/01/14/yubnub-mobile-20-application/)。YubNub 有额外的功能让它变得非常有用——包括一个浏览器插件[为用户节省了访问 YubNub 网站的步骤。用户也可以创建自己的命令。Sugarcodes 似乎还没有插件。](https://web.archive.org/web/20220811121045/http://mobilecrunch.com/2006/01/14/yubnub-mobile-20-application/)

这些不是生意，但是 YubNub 和 Sugarcodes 对于经常上网的人来说是非常方便的工具。而且 YubNub 插件的浏览器不动产成本是最小的。
