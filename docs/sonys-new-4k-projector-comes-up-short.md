# 索尼新的 4k 投影仪不尽人意

我们都在谈论 4k，这是近超高清未来的乌托邦式的解决方案。在 96DPI 的 292 英寸左右的显示器上，你真的不需要更大的东西(是的，没错)。

热爱这种下一代技术的索尼公司刚刚为 4k 人群推出了一款新的投影仪，使用了新的 SXRD(硅 X-tal 反射显示器，或对我们来说是巫术)，分辨率为 4,096 x 2,160，即 880 万像素。“真正的”4k 分辨率通常被认为是 1000 万像素或更高，但这仍然足够好，大多数人不会注意到。

这不是你的普通家庭影院设置，这是真正的数字影院舞台，你可以期待价格标签匹配。虽然这是一个令人印象深刻的分辨率，但仍然不足以显示重新制作的的[全分辨率，但这可能会很快改变。](https://web.archive.org/web/20201125030948/http://crunchgear.com/2006/10/08/red-one-prototype-lens-found-remainder-remains-awol/)
