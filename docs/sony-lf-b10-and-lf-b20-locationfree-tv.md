# 索尼 LF-B10 和 LF-B20 定位免费电视

这两个索尼定位免费单元可以让你在任何你喜欢的地方放置你的视频，很像 Sling Media 的 Slingbox。LocationFree 可让您从任何连接的 PC 或笔记本电脑上控制和观看电视。LF-B10(200 美元)和 LF-B20(250 美元)这两个基站允许您控制两台与复合视频或 S-Video 连接的设备。这两个单元都使用 H.264 而不是 MPEG-2 对视频流进行编码，从而以更低的带宽成本提供更高的视频质量。

您可以将它与 LocationFree 电视机顶盒配合使用，这样您就可以在自己家里的另一台电视上观看 LocationFree。举例来说，这可以让你在你办公室楼上客厅的主电视上收看你的频道。LF-B10 和 LF-B20 都使用以太网，但 LF-B20 具有 802.11b/g，因此您可以无线连接。最重要的是，您可以使用您的索尼 PSP 观看内容，只需更新到 2.50 固件或更高版本。Slingbox 支持更多的设备，但索尼的似乎是后起之秀，也有一些不错的功能。
