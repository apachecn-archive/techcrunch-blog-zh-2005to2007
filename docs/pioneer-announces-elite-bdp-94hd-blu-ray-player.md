# 先锋宣布推出精英版 BDP-94HD 蓝光播放器

![pioneer-elite-bdp-94hd-blu-ray-disc-player.jpg](img/8f4626f4203ebea8ed33c7c632d4aa18.png)

与今天宣布的新的第八代等离子体一起的是先锋公司的蓝光播放器，精英 BDP-94HD。没有太多值得流口水的东西，但是 BDP-94HD 可以通过 HDMI 消费电子控制技术无缝集成到您的家庭影院系统中。一个遥控器就能控制你所有的家庭影院设备…太棒了！不幸的是，你必须处理 1.2 HDMI 连接，但它确实拥有优质的杜比 TrueHD 无损高级音频。这款新播放器本月上市，售价 1000 美元。
