# Prosper.com 将于周二宣布里程碑

周二[](https://web.archive.org/web/20220811121727/http://www.prosper.com/)[Prosper.com](https://web.archive.org/web/20220811121727/http://www.prosper.com/)，一个[个人对个人的贷款网站](https://web.archive.org/web/20220811121727/http://www.beta.techcrunch.com/2006/02/05/prosper-launches-social-lending/)于二月推出，将宣布几个相当重要的里程碑:10 万会员和 2000 万美元的资助贷款。

他们比总部位于英国的竞争对手 Zopa 更快地达到了这两个里程碑，Zopa 最近被命名为商业 2.0“颠覆者”

Prosper 允许成员请求高达 25，000 美元的贷款(平均资助贷款为 5，000 美元)，然后其他成员以各种利率提供资助贷款。Prosper 将贷款分成多个部分来分散风险，然后从提供最有吸引力利率的贷款人那里获得资金。自该网站于 2006 年 2 月启动以来，已经资助了 4，000 多笔贷款。Prosper 通过预先从借款人那里收取 1%的贷款金额，并向贷款人收取 0.5%的年度贷款维护费来赚取收入。

有趣的事实:Benchmark 投资了 Zopa 和 Prosper，随着 Zopa 向美国市场扩张，这两家公司将很快展开直接竞争。
