# 赌博上瘾者的社交网络

 [](https://web.archive.org/web/20210802191052/http://www.gottabet.com/) 总部位于伦敦的 [Gottabet](https://web.archive.org/web/20210802191052/http://www.gottabet.com/) 是一个处理小额友好赌注的社交网络。用户设置赌注，要么是实际的钱，要么是 Gottabet 货币，称为“花生”，没有现金价值，但给了吹牛的权利。

一个例子[赌注](https://web.archive.org/web/20210802191052/http://www.gottabet.com/event/detail/374)是“杰克·鲍尔在第六季会杀多少人？”参与的费用是 30 个花生，获胜者将得到全部奖金。对于基于现金的下注，Gottabet 收取底池的 5%的费用。赔率可以设定，收益的一部分可以捐给慈善机构。举个例子——两个女人[和任何 Gottabet 成员打赌，她们可以在某个日期前减肥。赌注也可以保持私人或邀请。现金赌注的结算是通过信用卡和借记卡完成的。](https://web.archive.org/web/20210802191052/http://www.gottabet.com/event/detail/329)

Gottabet 也是一个社交网络——所有用户都有一个[个人资料页面](https://web.archive.org/web/20210802191052/http://www.gottabet.com/gottabe/Niharika),他们可以在那里添加朋友。对用户的评价不仅是根据他们收集了多少花生，还有他们的声誉。

Gottabet 表示，他们正在开发该产品的手机版。我想他们也会为人们在其他网站上推广他们的赌注创建小工具。目前，他们有一个工具来添加一个链接到来自脸书的赌注。

由于赌博在许多国家是非法的，Gottabet 将试图通过要求现金赌注只有“有限的价值”来保持低调。
