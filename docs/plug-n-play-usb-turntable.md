# 即插即用 USB 转盘| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/06/plug-n-play-usb-turntable/>

![](img/36945ed272a701412803cd09d093991c.png)

即插即用 USB 转盘允许你通过 USB 在你的计算机上听你的黑胶唱片。如果你只是想再听一遍那些流畅的 LP，转盘也可以连接到你的立体声系统。这是一个非常好的产品，对于 DJ 来说，他们有数百张黑胶唱片供他们采样。即插即用 USB 转盘还配有软件，一旦你的文件传输到你的电脑上，就可以编辑和清理它们。当然，我们不知道编辑软件工作得有多好，也不知道它最终如何以数字方式制作出美妙的黑胶声音，所以最好尽可能保持你的黑胶原始。该转盘零售价为 139 美元，配有一个毛毡防滑垫，45 转适配器，线路水平 RCA 电缆，USB 电缆和 CD-ROM 上的记录软件。
