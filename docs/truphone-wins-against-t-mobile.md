# Truphone 战胜 T-Mobile 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/07/16/truphone-wins-against-t-mobile/>

通过 WiFi 网络提供免费网络电话服务的 Truphone 在发展 VOIP 业务的过程中遇到了不少障碍。

一些运营商[故意禁用 Truphone 在其诺基亚 N95s 上使用的 VOIP 功能](https://web.archive.org/web/20150924192537/http://www.truphone.com/blog/blog.tru?entry=video_showing_internet_telephony_removed)。VOIP 已经被贴上了标签，被认为是恐怖分子不被追踪的一种可能的交流方式。最后，T-Mobile [拒绝将](https://web.archive.org/web/20150924192537/http://truphone.blogspot.com/2007/06/t-mobile-tries-new-tactic-to-kill-off.html)与 Truphone 的网络和 VOIP 运营商互联，使得无法呼叫 T-Mobile 的号码。这项规定并不适用于 T-Mobile 投资的另一家 VOIP 初创公司 Jajah，因为 Jajah 使用现有的电话网络而不是 VOIP 与被叫用户进行最后的连接。

今天终于带来了一些好消息，因为该公司已经获得了针对英国 T-Mobile 的禁令，迫使当地运营商根据竞争法在 7 月 23 日星期一之前与这项服务互联。
