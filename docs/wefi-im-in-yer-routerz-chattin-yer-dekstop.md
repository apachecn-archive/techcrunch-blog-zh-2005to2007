# WeFi:我在你的路上，在你的车站聊天

让我们一起创建一个 Web 2.0 小工具公司，好吗？请从 A 列和 b 列各选一项。见鬼。挑几个。

| A | B |
| 网络社交 | 手持设备 |
| 支持 Ajax | 猴子笼 |
| 携带好友列表 | 垃圾桶 |
| XML 源使用 | 山羊农场 |
| 超简单 | 路由器 |
| 互动的、感应的 | 气象站 |
| （同 VoiceoverInternetProtocol）网络电话 | 带振动的 MP3 播放器 |

 雇佣一个斯坦福大学的毕业生来帮助 biznes 然后启动。你的公司必须提供任何人真正想要或需要的东西吗？绝对不行。以 WeFi 为例。你设置了一个无线接入点。你分享一下。你可以和分享你的 Wi-Fi 的人聊天，前提是他们不只是嘲笑你的性感，也不忽视他们必须安装/访问一些网站才能使用你的 Wi-Fi 的事实。

是的，朋友。这是一个社交网络无线网络。想跟踪陌生人？想变得像“嗨！我看你喜欢无线网络。我们能见面吗？没有？”在我上大学的匹兹堡，曾经有一个电话亭。你走过电话亭，一个住在俯瞰电话亭的公寓里的家伙会打电话给电话亭，如果他喜欢你的样子。他会深呼吸然后邀请你上去。这听起来有点像 Web 2.0 版本。
