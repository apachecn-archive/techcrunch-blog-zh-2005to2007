# 雅虎为搜索添加歌词快捷方式

上周，雅虎开始在雅虎音乐上发布《T2》的歌词，这是它们第一次合法地出现在互联网上。今天他们[宣布](https://web.archive.org/web/20221006200500/http://www.ysearchblog.com/archives/000447.html)一个雅虎搜索快捷方式，可以很快找到歌词。

只需键入艺术家姓名或歌曲名称和“+歌词”即可在其他搜索结果上方看到该艺术家或歌曲的歌词结果。例如，参见甲壳虫乐队[、U2 乐队](https://web.archive.org/web/20221006200500/http://search.yahoo.com/search?p=the+beatles+lyrics)[和史密斯乐队](https://web.archive.org/web/20221006200500/http://search.yahoo.com/search?p=u2+beautiful+day+lyrics)[的歌词。](https://web.archive.org/web/20221006200500/http://search.yahoo.com/search;_ylt=A0oGkjKtmj9Gik8BaCVXNyoA?p=the%2Bsmiths%2Blyrics&ei=UTF-8&fr=yfp-t-501&x=wrt)

雅虎歌词并不完美——版权限制迫使他们将歌词显示为图像，而不是免费文本，以避免刮擦。然而，这也阻止了搜索引擎索引。
