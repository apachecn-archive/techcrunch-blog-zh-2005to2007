# 东芝推出具有昂贵功能的廉价笔记本

![](img/13eb48eedcae31d89b90236da15114ac.png)

东芝在笔记本电脑方面有起有落，但卫星 A135-S4527 总体上看起来是一笔相当稳定的交易。A135 预计将于本周以低于 800 美元的总价格在零售店上市，它配备了一些令人印象深刻的功能，如双核 1.73GHz 英特尔酷睿双核 T2060 CPU，1GB 内存(bravo Toshiba！)、DVD 刻录机、120GB 硬盘和 Vista Home Premium 来启动。

是的，所有这些都不到 800 美元。东芝是如何成功的？我想这可能与非法军火交易有关，但谁知道呢？无论哪种方式，那些寻找功能丰富，价格低廉的笔记本电脑的人可能应该在它发布时检查一下。看起来很有希望。
