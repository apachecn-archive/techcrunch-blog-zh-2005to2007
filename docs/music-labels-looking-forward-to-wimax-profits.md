# 音乐公司期待 WiMax，利润| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/09/music-labels-looking-forward-to-wimax-profits/>

我们知道 WiMax 是 4G 标准，最终将取代当前的 3G 无线技术。它便宜，速度快，现在正在部署。虽然更快的下载速度对你这个用户来说是好事，但让你垂涎三尺的是音乐公司和内容提供商。

十年前让互联网如此流行的概念是内容点播的想法。如果你想读点什么，你可以马上读。随着家庭宽带普及率的提高，这种风气已经蔓延到多媒体领域。错过了上周那集《迷失》？没问题，去美国广播公司，你想什么时候看就什么时候看。问题是，我们变得越来越移动(因此这个网站)，把高带宽内容放在用户手中(付费)的想法到处都是赚钱的。

不过，不要相信我们的话。查看路透社整理的不错的摘要，包括为一些不太懂技术的读者提供的一些关于 WiMax 的一般背景。文章中提出了许多好的观点，但要点是:用更少的钱获得更多的带宽意味着更多的移动下载。
