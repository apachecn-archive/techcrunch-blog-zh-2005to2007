# 我的喷气背包在哪里？

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/24/wheres-my-jetpack/>

我们知道你们中的大多数人除了《嘎吱嘎吱齿轮》之外什么都不读，但是有一本书正在流传，这本书对齿轮爱好者来说很有兴趣。 *[我的喷气背包呢？](https://web.archive.org/web/20150906171244/http://www.amazon.com/Wheres-My-Jetpack-Amazing-Science/dp/1596911360/ref=pd_bbs_sr_1/002-9861797-6671237?ie=UTF8&s=books&qid=1180026272&sr=8-1)* 展示了多年来许诺给我们但从未实现的许多惊人的发明。

作者丹尼尔·克拉尔是一名机器人专家博士，所以他可能知道自己在说什么(至少和我们一样)。我一读完菲利普·罗斯的《普通人》就去拿一本。

我的喷气背包在哪里？ 出自丹尼尔·克拉尔【无冕之王】
