# 宣布 TechCrunch 日本

 [](https://web.archive.org/web/20220124191200/http://jp.beta.techcrunch.com/) 我非常高兴地宣布，TechCrunch 现已在[jp.beta.techcrunch.com](https://web.archive.org/web/20220124191200/http://jp.beta.techcrunch.com/)推出日语版。

TechCrunch 的所有内容都将被翻译成日文，并在英文版发布后一天左右发布。此外，日本的创业公司将会出现在网站上，其中许多帖子会被翻译成英文，放在 TechCrunch 的主网站上。RSS 源是[jp.beta.techcrunch.com/feed/](https://web.archive.org/web/20220124191200/http://jp.beta.techcrunch.com/feed/)。

我对 TechCrunch Japan 寄予厚望—[TechCrunch France](https://web.archive.org/web/20220124191200/http://fr.beta.techcrunch.com/)现在是法国最大的博客之一([如果不是最大的](https://web.archive.org/web/20220124191200/http://ouriel.typepad.com/myblog/2006/05/is_techcrunch_j.html))。
