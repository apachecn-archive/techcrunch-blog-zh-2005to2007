# Web 2.0 的公司，第 2 部分——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/10/05/the-companies-of-web-20-part-2/>

这是出席 [Web 2.0 大会](https://web.archive.org/web/20220112091111/http://www.web2con.com/)发射台研讨会的第二批公司。在这里见[第一部分。](https://web.archive.org/web/20220112091111/http://www.beta.techcrunch.com/2005/10/05/the-companies-of-web-20-part-1/)

我的朋友 Ethan Stock 展示了昨晚发布的 Zvents。我们已经在这里写了 [zvents](https://web.archive.org/web/20220112091111/http://www.beta.techcrunch.com/2005/09/26/zvents-launches-next-week/) ，在这里写了。简而言之，Zvents 帮助你创建和定位成千上万的每月本地事件，并且有大量令人敬畏的 ajax、标签和其他 web2.0 的东西。

Ron Rasmussen 谈到了 KnowNow，一个有趣的基于 RSS 的提醒系统(他们称之为“elerts”)。我想更好地理解这一点，并希望本周能与罗恩坐下来谈谈。

 Ian McCarthy 给我们做了一个关于 [Orb](https://web.archive.org/web/20220112091111/http://www.orb.com/) 的诱人演示，它可以让你在不需要任何硬件的情况下将内容从你的家庭电脑传输到任何 wifi 设备。它对视频、照片等非常适用。他甚至在他的客厅里拉起了一个视频摄像头，并用 Orb 打开了灯。酷毙了。虽然现在只有个人电脑。

迈克尔·坦纳今天解除了 Wink 的密码保护，所以我们终于可以看到了。Wink 是“以人为本的搜索”,我认为他们正在做一些强大的事情。它们获取基本的搜索结果，并允许人们对它们进行标记和排名，以创建一个更好的结果集。他们称他们的技术为“tagrank”。

该死，迈克尔，回复我的邮件，明天给我一个面试机会。:-)

 Matthew Gertner 在 allpeers 上演讲，这是一个开发 firefox 应用程序的开放平台。Allpeers 目前处于私人测试阶段。

Bart Decrem 给了一群人演示。[什么](https://web.archive.org/web/20220112091111/http://www.beta.techcrunch.com/2005/08/26/flock-social-browsing-is-cool/) [更](https://web.archive.org/web/20220112091111/http://www.beta.techcrunch.com/2005/09/17/flock-evolves-announces-public-launch/)能不能[我说](https://web.archive.org/web/20220112091111/http://www.beta.techcrunch.com/2005/10/05/flock-expanding-beta-today/)关于羊群？我喜欢这种不自然的方式。如果他们能找到一种方法将 Pandora 直接集成到 Flock 浏览器中，我就再也不会离开我的电脑了。

但说真的，我已经拿到了新版本，并将在本周做一个完整的简介。

创始人 Bob Wyman 谈到了 PubSub、[结构化博客](https://web.archive.org/web/20220112091111/http://structuredblogging.org/)和他们的新 [LinkRanks](https://web.archive.org/web/20220112091111/http://www.pubsub.com/stats.php) 产品，我们[在这里写了](https://web.archive.org/web/20220112091111/http://www.beta.techcrunch.com/2005/09/26/pubsubs-linkrank-list-of-most-influential-blogs/)。更多关于 PubSub 的信息，我们最喜欢的潜在搜索引擎，[这里](https://web.archive.org/web/20220112091111/http://www.beta.techcrunch.com/2005/06/14/profile-pubsub/)。
