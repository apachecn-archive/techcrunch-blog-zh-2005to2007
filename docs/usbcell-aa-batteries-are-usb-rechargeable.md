# USBCELL AA 电池可通过 USB 充电

你有没有想过用电脑给 AA 电池充电？我们也不是在谈论 USB 充电器，我们只是在谈论把你的 AA 插入 USB 端口。你也是吗？我们知道我们做到了，有了这一对 24 美元的 USBCELL AA 电池，我们一生的梦想已经实现了。

电池适合任何 USB 端口，无论是你的笔记本电脑，个人电脑，还是供电的 USB 集线器，也可以与几种常规充电器配合使用。然而，这些电池不能与 60mA 充电器一起使用，充电器充电时间不到 2 小时，而是使用 250mA 充电器，充电器充电时间为 7 小时。否则你会制造一个时间漩涡，将你所爱的人吸进齐塔人区域，在那里你只能在感恩节和总统日去拜访。你知道他们每年都改变总统日。
