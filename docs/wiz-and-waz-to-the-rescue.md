# Wiz 和 Waz 来拯救

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/15/wiz-and-waz-to-the-rescue/>

如果你整天玩电子游戏，你可能会变成一个斑点……至少我们是这么听说的。但是如果为了拯救宇宙，你需要变成一滴呢？嗯，有一个棘手的问题我们无法回答。然而，我们可以告诉你，到了九月，这将成为一个更大的问题。在手机游戏里。

Eidos Interactive 宣布 9 月推出 SolaRola，在那里你可以从邪恶的冷酷无情的 Ping 手里拯救宇宙……嗯，你猜它是两个小斑点。进入 Wiz 和 Waz，两个兄弟将在 25 关的难题挑战中与邪恶战斗。

我们仍在等待有关支持的运营商、价格和手机的消息。但是斑点在上面。
