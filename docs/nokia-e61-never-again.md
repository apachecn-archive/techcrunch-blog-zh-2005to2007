# 诺基亚 E61，下不为例——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/11/nokia-e61-never-again/>

看来诺基亚的 E61 智能手机的性能达不到黑莓的标准，我们在 [ideagroup](https://web.archive.org/web/20210305012306/http://pure.typepad.com/ideagroup/ "Notes From A Chronic Entrepenuer") 的朋友确实感受到了这种痛苦。E61 显示出从联系人列表中缓慢拨号，降低了他的整体工作效率，并且显然 WiFi 连接不够壮观。这是如此糟糕，他只是想把这个家伙扔出去，现在它显然是挂在易贝，等待下一个吸盘。

此外，E61 可能会给人一种错觉，认为它支持轻松处理 VOIP 和 GSM 电话，但实际上，它甚至做得很差。最糟糕的是？嗯，E61 对于嘈杂的环境来说声音不够大。诺基亚希望你在会议上如何使用它？来吧诺基亚:我们期待更好的。
