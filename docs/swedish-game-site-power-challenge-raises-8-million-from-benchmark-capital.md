# 瑞典游戏网站 Power Challenge 从 Benchmark Capital 融资 800 万美元

基准资本投资 800 万美元与瑞典游戏网站 [Power Challenge](https://web.archive.org/web/20221205074321/http://www.powerchallenge.com/) 合作，该公司将于今天宣布。Power Challenge 是一个丰富的互联网应用程序——一个用 java 运行的游戏，玩起来非常像视频游戏。用户建立和定制他们的团队，并与他人进行比赛以获得分数和地位。游戏质量很高，旨在推动传统 PC 游戏和在线游戏之间的界限。

Power Challenge 及其姐妹网站 Manager Zone(今年早些时候收购)表示，它每天增加 4000 名新玩家。他们将利用融资所得的现金资助新游戏的开发，目前仅限于足球和曲棍球。这些游戏目前只能在 Windows 电脑上玩。

该公司成立于 2004 年。
