# 坦格勒的嵌入式讨论

[![](img/4505d4b116da8da5057496f5b3ebfcd8.png)](https://web.archive.org/web/20220517182728/http://www.crunchbase.com/company/tangler) 澳大利亚创业公司 [Tangler](https://web.archive.org/web/20220517182728/http://www.crunchbase.com/company/tangler) 创造了下一代论坛产品，无需刷新页面即可进行实时讨论。他们的论坛产品既有同步的也有异步的——这意味着它与 Meebo(基于网络的聊天)的竞争不亚于与现有论坛应用的竞争。用户还可以轻松地将富媒体嵌入到讨论中。

我们第一次写 Tangler 是在 2006 年 2 月[日](https://web.archive.org/web/20220517182728/http://www.beta.techcrunch.com/2006/02/21/tangler-untangling-communication-on-the-web/)，当时它正处于深度开发阶段。在过去的六个月里，他们一直在悄悄地与 beta 合作伙伴合作，最近他们敞开大门，允许任何人创建一个新的论坛。上周，Tangler 上创建了第 1000 个论坛。

Tangler 论坛也是分散的。任何讨论/论坛都可以轻松嵌入第三方网站。讨论在论坛的所有实例上同时发生。[看这里](https://web.archive.org/web/20220517182728/http://wheel.blogs.com/adwentures/2007/07/go-for-goalz.html)一个嵌入式论坛的例子。

使用 Tangler 作为他们论坛的创业公司的例子包括 [Weewar](https://web.archive.org/web/20220517182728/http://www.tangler.com/group/7466/discuss/) 、[particles](https://web.archive.org/web/20220517182728/http://www.tangler.com/group/5556/discuss)和 [Omnidrive](https://web.archive.org/web/20220517182728/http://www.tangler.com/group/5402/discuss/) 。

同样参见我们最近对 [Meebo 房间](https://web.archive.org/web/20220517182728/http://www.beta.techcrunch.com/2007/05/14/meebo-launches-meebo-rooms-oh-and-meebo-now-has-ads/)的报道。有趣的是看到网络聊天和论坛朝着相同的最终产品碰撞。
