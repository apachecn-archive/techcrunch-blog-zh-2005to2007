# OLPC 人喜欢玩毁灭战士

VIDEO

在过去的一年里，我们确信 OLPC 团队一直在努力为非洲的贫困儿童设计 100 美元的笔记本电脑。我们为他们对工作如此投入而自豪，但每个人都需要偶尔休息一下。来了经典的 FPS *毁灭战士*来娱乐团队。似乎克里斯多佛·暴雪(总色情名称)和他在 OLPC 办公室的伙伴们在 OLPC 上扔了厄运并且玩得很开心。

Doom 在 OLPC 上看起来、听起来和玩起来都很棒，而且由于笔记本电脑侧面的 d-pad，控制也很好。偷看上面的视频，看到一群呆子拍摄小鬼，而像女学生咯咯地笑。
