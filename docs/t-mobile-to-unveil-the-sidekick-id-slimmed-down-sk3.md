# T-Mobile 将推出 Sidekick ID:瘦身版 SK3

![](img/bf6511a90e6b26a40af8ebddcefe311a.png)

在过去的几个月里，关于“Sidekick 2.5”的谣言满天飞。我注意到了这一点，几周后，我向一些消息来源核实了一下。果不其然，一款“Sidekick 2.5”作为 Sidekick 3 的精简版替代品正在开发中。今天，有人向我泄露了这款设备的一张照片，你可以在上面看到一个被称为“丑陋”的彩色图案。虽然它不是最漂亮的设备，但 Sidekick ID(是的，就是它的名字)拥有你在 SK3 上喜欢的所有功能，只是没有所有的垃圾。
 那么名字里有什么呢？Sidekick ID 被削减，以展示通信功能，而不是通常包含的额外废话。你*不会*在 Sidekick ID 上找到以下一些功能:蓝牙、MP3 播放器、数码相机或 microSD 插槽。

不过，你会在 Sidekick ID 上找到的内容可能会受到大多数人的欢迎。一个带有 AIM 的 IM 客户端，Yahoo！、MSN、电子邮件客户端、网络浏览器，当然还有电话，都是 Sidekick ID 核心的一部分。一个更新的危险操作系统和新的菜单风格当然会出现，但不要期待一个激进的新设计变化或类似的东西。当然，T-Mobile 的 myFaves 将是 Sidekick ID 不可或缺的一部分。

导航将包括一个类似 Sidekick 3/Tempest arcade machine 的轨迹球，但目前还不知道 d-pad 是否会成为 Sidekick ID 的最终设计。因为我们只有一张 T-Mobile 员工的照片，所以很难判断左边是扬声器还是 d-pad。或许两者都有？有这种可能，但从外表来看，我觉得是个扬声器。

粉丝最喜欢的键盘设计将与 Sidekick ID 保持一致，允许您随心所欲地发送文本和即时消息。显示屏仍然会翻转，并发出可笑的大声音，这样每个人都知道你在使用 Sidekick。别担心。你对我们来说还是很特别的。

因此，在介绍了大部分规格和特性之后，还有两个问题:“它的价格是多少，什么时候能上市？”

振作起来！

Sidekick ID 将于今年在 Q2 上市，最有可能在夏季上市，价格将低于 Sidekick 3。我从两个渠道听说，正在考虑 149.99 美元的价格点，以保持其可负担性，并使其对大众可用。我是说面对现实吧。T-Mobile 发布 Sidekick 3 并让人们支付 300 美元的 2 年合约，彻底赚了一笔。当然，他们现在可以放弃合同，但是 T-Mobile 仍然赚了不少钱。

然而，随着更多的核心功能和更少的像 MP3 播放器这样的噱头，Sidekick ID 有机会成为今年夏天最热门的通信设备之一。你可以把它想象成一部黑莓手机，供那些在旅途中仍想与朋友、家人和同事保持联系的非专业人士使用。

不管怎样，就是这样。这是跟班身份证上的毒品。它不被称为“Sidekick 2.5”，它缺乏某些功能，这在这种情况下不一定是一件坏事，它将于今年夏天推出。享受吧。
