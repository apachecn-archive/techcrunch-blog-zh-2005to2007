# T-Mobile 用 Wing 替代 MDA(可能)

很可能是因为没有人关心这个设备，T-Mobile 已经开始放弃 MDA 了。作为替代，T-Mobile Wing(又名 HTC P4350)将接管全部 QWERTY/PDA 功能，搭载全新的 Windows Mobile 6.0。MyMobile911 报道称，该设备将有一个弹簧加载的滑动键盘，与 Cingular 8525 相同的屏幕，并且整个设备都有一层蓝色橡胶涂层，以保护它免受你将把它扔向的墙壁的影响。有传言称，翼将于 8 月由 T-Mo 出道。
