# PatentMonkey:同行专利审查开始，进入 SecondLife

![SecondLife](img/61f53a747d59b72a982639a88b8339e0.png)在过去的两周，对等专利开放了一个 USPTO 批准的项目，允许对未决专利申请进行[公开审查。注册会员被鼓励通读可用的专利，并提供关于现有技术和显而易见性的意见，然后可以成为 PTO 审查员审查非专利参考文献的一部分。](https://web.archive.org/web/20201124130956/http://www.peertopatent.org/patent/list)

今天，Peer-to-Patent 宣布他们已经在第二人生的[民主岛上占据了一个空间，允许成员与视频互动并查看项目信息。](https://web.archive.org/web/20201124130956/http://cairns.typepad.com/peertopatent/2007/06/peer-to-patent-.html)
