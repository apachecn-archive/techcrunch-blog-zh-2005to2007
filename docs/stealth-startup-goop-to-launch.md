# 秘密创业公司 Goop 即将推出——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/04/01/stealth-startup-goop-to-launch>

关于总部位于帕洛阿尔托的秘密创业公司 [Goop](https://web.archive.org/web/20220119201051/http://64.34.165.73/) 的谣言已经流传了几周。一个接受电子邮件提交进行测试的标准登录页面现已推出，一些关于该公司的信息已经泄露。

2006 年 4 月 1 日早些时候，首席执行官 Peter Gibbons(以前在 TPS 自动化公司 Initech 工作)向我介绍了该产品。Goop 当然拥有下一代 web 2.0 特性，可以与更成熟的公司竞争。著名天使投资人 Josh Kopelman 提供的 3000 万美元巨额融资应该会给它足够的持久力来吸引大约 20，000 个电子邮件地址，它需要被一个大公司收购。

核心功能集包括一个完全交互式的 3D 社交标签环境，第一个剪切和粘贴书签，以及一个 Ajax/Flash 丰富的界面，带有圆角和易于观看的彩色粉笔。此外，所有常见问题都有 RSS 提要。

然而，该服务最有趣的方面是许多 del.icio.us/flickr-like 功能，允许任何人创建自己的基于主题的主服务版本，创建与任何 XUL 或力士接口的混搭，并在基于 Soap 的平台上通过完全封闭的 API 向用户突出显示结果。目前，该服务仅可在 Mac 平台上使用 Internet Explorer 浏览器下载。报名参加封闭测试[这里](https://web.archive.org/web/20220119201051/http://64.34.165.73/)。Goop 是赢家。
