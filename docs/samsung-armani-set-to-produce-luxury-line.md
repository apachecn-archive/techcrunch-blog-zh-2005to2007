# 三星、阿玛尼将生产奢侈品系列

三星和 LG 之间的竞争再次升温。LG 通过与普拉达合作推出了高端豪华手机，现在三星据说正在与阿玛尼谈判生产一系列奢侈品。

市场消息来源证实，韩国消费电子产品制造商和意大利设计师已经在今年年初的队列，他们目前在谈判的最后阶段。据三星发言人称，手机并不是此次合作的唯一成果。我们可能还会看到阿玛尼系列的电视机。
