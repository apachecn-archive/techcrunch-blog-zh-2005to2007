# 孢子即将成为蒸汽制品| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/20/spore-about-to-become-vaporware/>

![](img/d2d9360cafadb0c8f7cffd5785ddbc2b.png)

这很遗憾，但在软件行业这并不是什么新鲜事。威尔·莱特备受期待的*孢子*现在再次被推迟，看起来它永远不会发布到等待的世界。这款游戏的发行商艺电公司表示，这款游戏现在已经从 08 财年进入 09 财年，这基本上意味着这款游戏还看不到发布日期。

由于没有发布日期和持续的猜测，游戏将继续被推迟，它开始看起来像*孢子*永远不会看到商店货架。这让我觉得很奇怪，考虑到游戏的工作版本是存在的(不久前在 E3 展示过)。除非团队也在开发控制台版本，否则我真的看不到我们有机会像飞船里的刺蜥蜴一样摇滚。
