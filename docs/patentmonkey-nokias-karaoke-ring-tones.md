# PatentMonkey:诺基亚的卡拉 ok 铃声

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/28/patentmonkey-nokias-karaoke-ring-tones/>

[![Gnarls Barkley by PhotoKeith](img/c069a335a3af7be42f1f94d403ba02d6.png)](https://web.archive.org/web/20160301042502/http://www.photokeith.com/welcome.html "Gnarls Barkley by PhotoKeith")

本周，诺基亚获得了一项不仅仅是铃声的专利，还有可以用第二种音调修改的铃声，或者他们所谓的“[复合铃声](https://web.archive.org/web/20160301042502/http://www.patentmonkey.com/PM/patentid/7248900.aspx)”。

哦，对了，让你所有的朋友录下你最喜欢的歌曲，这样就能知道是谁在打电话。跳跃后的细节…

诺基亚的新专利包括使用铃声变化来快速帮助你现在可能收到的不同类型的通信，例如电子邮件、文本或电话。该专利的细节描述了修改部分标准铃声的选项，以使用户能够知道到来的消息是电子邮件还是呼叫。首先，对该技术的描述似乎涵盖了[复音铃声](https://web.archive.org/web/20160301042502/http://en.wikipedia.org/wiki/Ringtones#Types_of_Ringtones)的变化，如放大或删除曲调中的乐器。好吧，不那么令人兴奋，但潜在的实用性。

至于细节…

> 1.一种用于在可操作来经由通信网络接收通信的通信设备中提供铃音告警的方法，所述方法包括:检测呼入通信；确定呼入通信的至少一个通信特征；将第一铃音增强与所述至少一个通信特征相关联；以及通过将第一铃声增强附加到基础铃声来生成复合铃声警报，其中所述至少一个通信特征包括多个通信特征，并且将预定铃声增强与多个通信特征中的多个通信特征中的每一个相关联，其中第一铃声增强包括将辅助铃声附加到基础铃声。

现在，诺基亚，如果移动设备可以确定关于呼叫源的信息，除了它与呼叫者或电子邮件源的通信类型之外，通过向基础铃声添加二次增强来修改铃声是非常可能的。

诺基亚设想，这也可以超越移动设备，带到个人电脑平台，提供类似的听觉通知。
