# 本周来自 Cingular/美国电话电报公司/随便什么地方的新黑莓手机

在& T

下个月，期待一款名为 8800 的性感模特的发布。8800 看起来非常像[珍珠](https://web.archive.org/web/20130628172239/http://crunchgear.com/?s=pearl)的更老、更胖的兄弟，是一个全 QWERTY 功能的 BB，具有令人惊叹的功能。或多或少是对现有的 8700 系列 Cingular 目前提供的升级，8800 增加了流行/麻烦的 Pearl 轨迹球，这意味着这可能是 RIM 即将推出的所有 BBs 的标准功能。

然而，据《我们的天才》报道，今年晚些时候，一款全新的掌上电脑将会亮相。9000 系列(9900？9009?9201?)将采用完全重新设计的黑莓 GUI，运行它的新硬件，可能还有 3G 连接。然而，这款手机不会只面向 Cingular/美国电话电报公司。寻找全面的版本，因为它实际上是一个新的黑莓类，每个运营商都适合一个。
