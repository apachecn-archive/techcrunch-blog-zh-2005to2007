# Sun 用 JavaFX 回应 AJAX、Silverlight

![sun.png](img/cc6bfc8414b6641867ebbd1de5f74b03.png)在 [Infoworld](https://web.archive.org/web/20221002135742/http://www.infoworld.com/article/07/05/07/javafx-javaone_1.html) 和 [CNet News](https://web.archive.org/web/20221002135742/http://news.com.com/2100-1007_3-6181922.html) 报道，Sun 将在今天的 [Java One 大会](https://web.archive.org/web/20221002135742/http://java.sun.com/javaone/sf/index.jsp)上推出一个基于 Java 的产品系列，名为 JavaFX。

该声明紧随最近广受欢迎的微软 Silverlight 发布，作为一个产品，它也将直接与 AJAX 竞争。

JavaFX 据说是 Java 的一个新的脚本变体，专注于消费者通信市场的开发，包括桌面、移动客户端和电视。第一个发布的产品是 JavaFX Mobile，一个用于移动设备的软件系统。

在演示结束之前，我们不会对新产品有更多的了解，但我的第一想法是，我们在这里看到的是 Sun 的追赶游戏。面向对象的应用程序编程语言 Java 曾经是最酷的产品，新一代应用程序将在此基础上推出，接管我们的桌面。然而这从未发生过。这款专注于移动技术的新产品，如果今天受到开发者的欢迎，有可能让 Sun 重新回到应用游戏中来。
