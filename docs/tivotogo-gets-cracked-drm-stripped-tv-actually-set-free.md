# TiVoToGo 被破解，数字版权被剥夺。电视居然免费？

我不确定 TiVo 的借口是什么，不让包括我自己在内的非 Windows 用户使用 TiVoToGo 的功能。可能和我从其他不提供 Mac/Linux 支持的公司那里得到的一样:没有足够的用户。但正是这种态度促使人们去做像打破 DRM 这样的事情。

PVRWire 有一个项目的报告，该项目开发了一个命令行程序，将从 TiVoToGo 文件中剥离 DRM，这将允许您在任何设备上播放 Windows 专用文件。如果你想了解更多，有一个[wiki 详细介绍了项目的进展](https://web.archive.org/web/20130627214122/http://www.alt.org/wiki/index.php/TiVoToGo)，但基本上，该程序摆脱了 DRM，将文件从。录制到。mpeg，可以在许多设备上播放。更重要的是，可以上传到很多网站。
