# NEC TYPE-N01:史上最伟大的笔记本电脑？

就规格而言，NEC 和 Takara 的 TYPE-N01 可能不是操场上最快的孩子，但它在速度上的不足，在风格上得到弥补。正如你所看到的，TYPE-N01 被打扮成各种各样看起来吓人的装束，包括一个红色按钮和一个看起来可能是指纹扫描仪的东西。

真的，这只是一台精心设计的配有 15.4 英寸 WXGA 显示屏的 L 型笔记本电脑。在引擎盖下，它有一个 1.6GHz AMD mobile Sempron 3200+处理器，1GB DDR2 RAM，80–100 GB 硬盘，双层 DVD 刻录机，802.11 a/b/g 和一个 FeliCa 端口。

TYPE-N01 的售价将从 1231 美元到 1476 美元不等，产量仅限于 300 台。
