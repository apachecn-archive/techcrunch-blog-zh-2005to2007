# 官方 PSP 开发包从易贝撤回

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/06/official-psp-devkit-gets-pulled-from-ebay/>

![](img/e821a746037b7d9e4f4566b8ec7cf6ab.png)

似乎昨天，有人在易贝发现了一个官方的索尼 PSP 开发者工具包。在拍卖还剩不到两天的时候，最高出价只有 329 美元——低于我的预期。根据拍卖，待价而沽的是一个 PSP，电缆，和一个开发者的电脑用于编程和编译游戏。终极自制者的幻想。

来自论坛:

> “从我们收集到的信息来看，这个工作站是用来测试和编译 PSP 游戏的。目前还不清楚正在开发的游戏是否带有索尼官方的数字签名算法(允许它在 PSP 上工作，而不需要自制黑客)，但这场拍卖的获胜者可能会收集一些很酷的信息，这些信息可以用于自制社区的利益。”

不幸的是，拍卖最终被易贝取消。看起来索尼发现他们的一个“不太出名”的开发者试图赚快钱。然而，在不久的将来，另一家上市可能只是时间问题。有点像 OJ 被取消的书。
