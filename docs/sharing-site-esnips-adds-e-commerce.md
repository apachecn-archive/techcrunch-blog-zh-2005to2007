# 共享网站 eSnips 增加了电子商务

 [](https://web.archive.org/web/20221001073918/http://www.esnips.com/) 以色列 [eSnips](https://web.archive.org/web/20221001073918/http://www.esnips.com/) 是一个非常简单的在线社交网络，用于分享图像、视频、文件和几乎任何其他东西。我们曾在这里写道[当它最近重新推出](https://web.archive.org/web/20221001073918/http://www.beta.techcrunch.com/2006/03/21/checking-out-esnips/)时，现在该网站增加了 PayPal 功能，允许用户直接在网站上购买和销售。该公司已经看到人们试图通过他们的网页进行买卖，并表示增加电子商务才有意义。目前 eSnips 不收取任何交易费用——他们只是希望这个功能能让网站更吸引用户。我很好奇这是如何实现的。

该公司报告称，绘画和照片是网站上最受欢迎的商品，其次是珠宝、“其他商品”和工艺品。网站上还出售音乐、视频和游戏。用户通过其帐户获得 1 GB 的免费存储空间。收入是通过销售额外存储和 Adsense 产生的。任何项目也可以用 HTML 小部件在场外显示。项目是使用仅适用于 Windows 的工具上传的。

ESnips 是一个朴实无华的网站，似乎只是为人们工作。该公司目前正在举办“开放工作室周”，突出那些在网站上分享他们作品的精选艺术家。我猜这只是进一步证明，用户生成内容、数据可移植性和社交网络的 Web 2.0 世界并不一定要华而不实才能吸引人们的注意力。

这看起来像是十年前会失败的事情，但现在在线分享内容的想法已经变得更加主流，这可能真的有用。它是否已经成为主流，足以支持像 eSnips 那样的商业模式？我们会看到的。
