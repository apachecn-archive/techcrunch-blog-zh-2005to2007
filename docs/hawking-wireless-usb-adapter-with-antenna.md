# 带天线的霍金无线 USB 适配器

![hawkingadapter.jpg](img/11b024dbd80c8e01df4e0889b1896fc9.png)这款 USB 无线适配器采用可拆卸天线，超越了普通 USB 无线适配器。通常，人们购买这些产品是为了在没有内置 802.11b/g 无线功能的旧笔记本电脑上使用，通常这些产品的接收效果很差。不再有了！
