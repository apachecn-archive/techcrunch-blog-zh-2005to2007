# 我们都是海盗

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/26/were-all-pirates/>

一项针对 2600 名美国人的研究发现，其中整整 18%的人都是盗版电影，我怀疑这不是一个有代表性的样本。这是什么意思？这意味着，通过推断，有 2500 万肮脏的，可怕的下载者。在这个群体中，只有极少数人使用 CinemaNow 或亚马逊 Unbox 等下载服务，而 80%的人使用 BitTorrent 等点对点盗版工具。

> #典型的电影下载者是 29 岁；63%的下载者是男性，37%是女性。一个下载者通常会在他的电脑上存储 16 部完整长度的电影。
> #在接受调查的人中，56%的人在某个时候在电脑上看过 DVD，而 29%的人在上个月在电脑上看过 DVD。大约 25%的人在电脑上看过流媒体电视节目。

这个世界要做什么？哭？尖叫？加入美国电影协会的自杀协议？以上都不是，朋友们。这项调查基本上证明了可下载内容是下一个大事件，如果我是一个打赌的人，我会开始投资流媒体视频服务。有多少 MP3 在媒体公司想出怎么卖之前就转手了？几百万人。几百万。

不管蓝光和 HD-DVD 的人怎么说，未来是 IP。物理磁盘是一个死胡同。
