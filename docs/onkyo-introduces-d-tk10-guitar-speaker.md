# 安桥推出 D-TK10 吉他扬声器

![](img/4f37975e6795d331e72dc3d44e865292.png)

D-TK10 吉他扬声器由安桥和著名吉他制造商 Takamine 开发，是一款模仿吉他的革命性音箱。它采用基于高端吉他结构的自由振动薄壁外壳，创造出更饱满、更丰富的声音。这款扬声器具有精心布置的支柱、纵梁和厚度变化(像吉他一样)，因此它可以驾驭和控制通常会在另一款扬声器系统中丢失的声音。低音音箱由桃花心木制成，配有一个用于低音的 4 英寸锥形低音扬声器和一个 1 英寸的环形驱动高音扬声器，可以再现高达 100 kHz 的频率。

> “安桥 D-TK10 还具有镀金接线柱，可与香蕉插头、铲形接线片或裸线兼容。扬声器经过磁屏蔽，允许它们在视频设备附近工作，而不会降低画面质量。”

这些是一些制作非常好的扬声器，制作如此之好，以至于安桥一个月只发布 50 对，因为每对都投入了大量的工作。每副大约需要 1999.00 美元，但是如果你有一个高端工作室，并且正在寻找一些很棒的新显示器，D-tk10 绝对值得一听。
