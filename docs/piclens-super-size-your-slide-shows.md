# PicLens:超大尺寸的幻灯片

 [](https://web.archive.org/web/20220701013416/http://piclens.com/) [PicLens](https://web.archive.org/web/20220701013416/http://piclens.com/) 是来自 [CoolIris](https://web.archive.org/web/20220701013416/http://cooliris.com/) 的新照片查看插件。这个工具可以让用户观看来自 Flickr、Photobucket、脸书和图片搜索结果 Google、Yahoo！，又问。到目前为止，该软件在功能和用户控制方面很少，但它在视觉上如此引人注目，无论如何你都可能会喜欢使用它。第一个版本仅适用于 Safari。

CoolIris 是一家我们在之前[已经在这里写过的公司。他们迄今为止的旗舰产品是一个浏览器插件，提供搜索结果和其他网页的弹出预览。有些人喜欢它，但其他人(包括我自己)觉得它与其说有用，不如说刺耳。这种新产品使用起来更有趣。CoolIris 还提供免费的铃声制作服务，一些公司已经开始这样做了。](https://web.archive.org/web/20220701013416/https://beta.techcrunch.com/tag/cooliris)

下载插件后，当鼠标悬停在任何受支持网站的图片上时，这些图片会显示一个两框图标。单击该图标会将您带到全屏视图，以启动幻灯片放映。只有在你启动 PicLens 的页面上显示的图片才可以在幻灯片中看到(谷歌图片是 18 张，Flickr 是 24 张)，而且似乎没有任何方法可以改变图片的显示速度。作为概念的第一次验证，这很棒。对于任何 Safari 用户来说，下载并使用它来增强图像浏览都很容易。希望有一天会推出 Firefox 插件。该公司表示，Windows 版本将很快推出。

可能没有办法直接将其货币化，在缩略图周围包装广告风险很大——在其他人的全尺寸幻灯片放映后放置后卷广告是不可能的。
