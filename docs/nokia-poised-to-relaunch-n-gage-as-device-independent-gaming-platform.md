# 诺基亚准备重新推出 N-Gage 作为独立于设备的游戏平台

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/01/31/nokia-poised-to-relaunch-n-gage-as-device-independent-gaming-platform/>

诺基亚，保佑你的心。我们喜欢你制造相当时尚和复杂的手机，它们容易使用并且有一些能量。我们也喜欢你在 Symbian 上做的事情，尤其是 S60。但是，拜托，N-Gage？还是？

我们肯定你放弃了游戏优化设备的想法。你让 N-Gage 成为 N 系列手机上的游戏平台的想法很好。这样，无论你有哪一款 N 系列手机，你都可以有自己选择的游戏。

但是你会选择高端的图形和音乐吗？你认为在任天堂 DS 和 PSP 的世界里，人们想要的不仅仅是手机上的纸牌游戏吗？显然是这样，因为你已经计划在下周马德里的 GDC 会议上宣布你的平台。反正街上是这么说的。如果你做了，我们不知道该怎么想。但是祝你好运，诺基亚。你的愿景是宏伟的，我们希望你能实现它，没有什么比在我们的 S60 手机上再次看到托尼·霍克更让我们自豪的了。
