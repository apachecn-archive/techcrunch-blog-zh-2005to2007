# SMS M500 投入生产

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/08/sms-m500-goes-into-production/>

![m500_pic5.jpg](img/f9f523bb235272562160dc3c2a2be91c.png)

SMS M500 是一款手表。但不是一般的手表。这是一款四频 GSM/GPRS、触摸屏、支持 Java 和 WAP 的手表。如果军情六处真的很便宜而不是欧米茄海霸王，这就是 Q 会给邦德的。它基本上是你手腕上的智能手机。不要老是查看手机信息，你可以看看你的手腕，这在你寻找下一个[短信炮友](https://web.archive.org/web/20151005124707/http://crunchgear.com/2007/05/31/sex-messaging-the-silent-booty-call/)的时候会更加显眼。M500 支持上述短信、彩信、MP3 和 MP4 播放，但 128MB 的内置内存根本无法容纳。在光明的一面，它有数据传输和充电的 USB 端口以及蓝牙 2.0。你真的想要一个，对吗？嗯，它终于投入生产，并于 8 月 31 日开始从香港工厂发货，9 月 10 日上市。欢乐！
