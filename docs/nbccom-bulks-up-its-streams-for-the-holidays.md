# NBC.com 为迎接假期而扩建了河流

那些想寻找更多方法避免假期不和家人说话的人应该去 NBC.com。该网络在其网站上增加了“周五之夜”的前九集和情景喜剧“30 摇滚”的前七集。并且它们大部分是无商业广告的；每集之前只有一个广告。

NBC 在其他节目播出后一直在播放，但在接下来的一周就被删除了。《周五之夜之光》和《30 摇滚》是第一批有权在网站上存档的作品。不过，一旦假期过后新的剧集开始，这些也会消失。
