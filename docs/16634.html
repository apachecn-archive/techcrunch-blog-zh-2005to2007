雅虎上的Ojas Rege！手机——TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2007/07/05/qa-ojas-rege-on-yahoo-mobile/

Mobile Crunch与雅虎全球移动产品副总裁Ojas Rege聊了聊移动内容、广告和iPhone的重要性(雅虎为iPhone提供雅虎邮箱、推送电子邮件服务、OneSearch以及金融和天气应用)。

移动紧缩:iPhone会对移动市场产生什么影响，如果有的话？

瑞吉:我们很兴奋。我们确信这将会产生影响。归根结底，它将推动消费者对精彩、易用、酷的服务的需求。它拓展了消费者心中的可能性艺术。

它让业内的所有玩家对自己说，“嘿，创建引人注目和直观的应用程序是完全可能的。”

Mobile Crunch: 它的影响会因为它的高价位而减弱吗，也就是说，它会在较低的SRP下产生更大的影响吗？

Rege: iPhone现在是一款高端设备。随着时间的推移，价格点会发生变化。我们在其他技术中也看到了这种情况。公平地说，价格点将会演变。在雅虎，我们希望看到这些类型的平台走向尽可能多的人。

移动紧缩:iPhone将如何影响移动广告？iPhone独特的网络浏览方式将如何影响雅虎在其空间上销售的广告或广告类型？

Rege: 这个有三个组成部分。

1.任何驱使人们使用移动数据服务的东西都是好的。这就创造了更多的临界质量来销售广告。

2.在雅虎，我们不会为所有手机打造相同的体验。我们在不同的电话平台上优化我们的服务。任何让我们做更多事情的东西也为更多的广告商打开了大门。

3.最重要的一点是:你必须有能力让广告与用户相关。相关性高的简单广告是高效的。

这是雅虎智能广告的自然结果。

SmartAds的高级概念是:使用你所知道的关于用户的东西来定制广告。我们使用个性化来更容易地定位广告。我们在SmartAds中为PC平台所做的一切都适用于移动设备——我们不受媒体类型的限制。

在我们所有的服务中，我们都在向个性化发展——也就是说，确保我们能够为用户和平台提供量身定制的服务。确保这些服务对他们来说是真正重要的。

Mobile Crunch: 我们能否展望未来，问一问两年后甚至一年后我们会在广告或移动领域看到什么样的趋势？还是科技发展太快，无法做出这样的预测？

Rege: 像iPhone这样的新技术为很多人打开了大门，他们会说，“哇，我可以在我的手机上做很多很酷的事情；不仅仅是语音和文字。”随着用户意识到他们拥有的生命线——手机——也与他们想在这个世界上做的许多事情相关，我们将会看到一个转变。正如我之前提到的，它拓展了消费者心中的可能性艺术。这吸引了很多人，创造了更多的广告需求。通过扩展用户可以利用技术做的事情来增加需求的整个概念，推动了我们如何提供服务。

Mobile Crunch: 衡量移动广告指标与标准的互联网广告报告和衡量有何不同？

当你是一个广告客户，正在制作一个移动广告时，你想让它成为你在电脑上所做的品牌延伸吗？人们正在试验。这是非常早期的市场。标准指标:展示次数、转化率——这些都适用于手机。核心指标将保持不变，但扩展指标将有所不同。手机的实验时机已经成熟。

Mobile Crunch: 随着越来越多的人可以使用移动技术，你是否认为用户指标会发生变化？

Rege: 随着手机使用变得更加大众化，人们所做事情的性质将会改变。我们的移动产品遍布全球。每个国家都处于不同的发展阶段。在美国，我们有许多不同的连接方式:高宽带普及率，高个人电脑普及率。东南亚的许多国家没有这些选择。在许多地区，人们没有其他选择去做。我们必须拥有一款不依赖用户访问电脑的产品。我们必须在移动优先的背景下提供服务。

Mobile Crunch: 移动设备浏览网页的方式是否正在走向标准化？还是技术的速度让公司朝着不同的方向发展？

也许有一天我们会有真正的标准化，但我不确定我会有多大年纪。我们不会屏住呼吸。我们认为，在操作系统和浏览器方面，世界将继续分裂。

Mobile Crunch: 雅虎认为移动市场最大的挑战是什么？帮助用户理解什么是可能的。有时候，这个行业过于依赖技术。关键是找到简单、有吸引力的应用程序。

如果我有一个关于推动移动平台使用的愿望，那就是想出让用户了解他们可以在手机上做什么的好方法，并以一种不复杂的方式向他们提供服务。
