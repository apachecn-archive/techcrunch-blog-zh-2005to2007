# PS3 的 SACD 能力不如 SACD

就像当前蓝光和 HD DVD 之间的战争一样，高保真音频格式领域也曾有过一场争夺霸权的斗争:超级音频 CD 对 DVD-Audio。事实上，“奋斗”有点言过其实，因为两者都没有真正起飞到任何程度。而且，根据[CNET 的大卫·卡诺伊在《渴望](https://web.archive.org/web/20130628200337/http://crave.cnet.com/8301-1_105-9680711-1.html?tag=head)上的一篇帖子，看起来索尼可能无意中有意决定了它创造的 SACD 的命运。

似乎即使 PlayStation 3 有播放光盘的能力，它在输出方面也有所欠缺，没有多通道模拟音频连接来处理多通道轨道。Carnoy 继续解释说，虽然索尼可能有一天会启用 HDMI 来数字处理 SACD 音频，但你仍然需要一个配备 HDMI 的 A/V 接收器来接收它。
