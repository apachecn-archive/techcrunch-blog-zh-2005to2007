# 专利战:Local 获得本地搜索专利

在被美国专利商标局授予第 7，200，413 号[专利后，Local.com](https://web.archive.org/web/20220521053737/http://patft.uspto.gov/netacgi/nph-Parser?Sect1=PTO1&Sect2=HITOFF&d=PALL&p=1&u=%2Fnetahtml%2FPTO%2Fsrchnum.htm&r=1&f=G&l=50&s1=7,200,413.PN.&OS=PN/7,200,413&RS=PN/7,200,413)[的股价今天上涨了 52%](https://web.archive.org/web/20220521053737/http://www.reuters.com/article/technology-media-telco-SP/idUSBNG31155620070702) 。根据该专利的文本，它涵盖了他们所谓的“增强型目录辅助”，该服务将根据用户通过运营商、短信、WAP 或语音输入的关键词提供特定的广告推荐。

这是对他们最近授予的专利号为 [7，231，405](https://web.archive.org/web/20220521053737/http://patft.uspto.gov/netacgi/nph-Parser?Sect1=PTO1&Sect2=HITOFF&d=PALL&p=1&u=%2Fnetahtml%2FPTO%2Fsrchnum.htm&r=1&f=G&l=50&s1=7,231,405.PN.&OS=PN/7,231,405&RS=PN/7,231,405) 的专利的补充，该专利涵盖了网络爬行和地理编码基于位置的内容。这似乎类似于谷歌已经拥有的正在申请的专利。为了获得更多信息，Bill Slawski 列出了一份详尽的谷歌本地搜索专利申请清单，包括术语定义。他有一个类似的页面，关注微软的本地搜索专利。

许多其他专利已经出现在本地搜索空间。Free 411 的叮当网络公司最近获得了一项专利，在提供查号服务之前滚动上下文广告。谷歌最近申请了一项雄心勃勃的本地搜索专利。

本土搜索公司已经开始了专利战。总部设在伦敦的 Geomas 公司拥有 5，930，474 项专利，该专利使用户能够从地图上选择信息，该公司正利用这项专利在 T2 对威瑞森提起诉讼。

对于 Local.com 将如何使用这些专利，T4 有些怀疑。然而，首席执行官 Heath Clarke 在一份相当明确的竞争参考资料中指出，“我们鼓励目录帮助，并帮助有兴趣使用我们知识产权的 411 公司与 Local.com 签订许可协议。”
