# 第三屏幕媒体加入 Hovr 游戏

第三屏媒体已经将 Hovr 添加到其移动广告网络报告 ClickZ News 中。这个免费的手机游戏发行商在二月份作为一个支持免费广告的手机游戏网络推出，应该会加强广告网络吸引年轻人市场的努力。直到最近，Hovr 一直在销售自己的广告库存，但根据合作伙伴关系，第三屏幕媒体将销售和管理广告。Hovr 是第一家通过第三屏幕媒体的网络销售广告的游戏提供商，该公司正在被 AOL 的 Advertising.com 子公司收购。
