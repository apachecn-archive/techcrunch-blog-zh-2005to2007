# 纽约为未来的出租车做好准备

[![futuretaxi.jpg](img/a91ba9bbcddcbcc4e7367e273a73e442.png)](https://web.archive.org/web/20210303010033/https://beta.techcrunch.com/wp-content/uploads/2007/04/futuretaxi.jpg "futuretaxi.jpg")

目前，纽约正在举办某种形式的[汽车贸易展](https://web.archive.org/web/20210303010033/http://www.autoshowny.com/)，其中一个亮点是展示未来出租车的展览。它被称为出租车 07，包括各种概念和原型，你可能会在纽约街头看到一天。许多概念侧重于改善残疾人的无障碍环境，而另一些则试图尽可能体现环保意识。标准出租车(图片)结合了这两种方法:内部有足够的空间来容纳轮椅，出租车专用的动力系统改善了油耗和排放。替代燃料驱动的出租车也在展出，但这永远不会在现实世界中流行起来。
