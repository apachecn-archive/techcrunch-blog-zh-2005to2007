# Scrybe 可能会在办公应用程序中设立一个新标准

 [](https://web.archive.org/web/20211018065657/http://www.iscrybe.com/) [杰森·克拉克](https://web.archive.org/web/20211018065657/http://www.downloadsquad.com/2006/10/20/scrybe-the-online-productivity-suite-im-dying-to-try/)今天发现了一家伟大的新创公司，叫做 [Scrybe](https://web.archive.org/web/20211018065657/http://www.iscrybe.com/) 。它还没有推出，但他们正在使用 YouTube 来展示该产品(见下面的视频)。更多的公司应该这样做——这让公司能够向每个人展示他们的产品到底是什么，并在关键的预发布阶段控制信息。

Scrybe 看起来是一个独特的在线日历应用程序，当你离线时也能很好地工作。只需在浏览器中打开该网站，就像你在线一样——该应用程序将在你下次在线时进行同步。日历视图会根据您正在查看的内容自动展开和最小化。它还允许无缝的剪切和粘贴以及与 Office 文档的集成，以及从网站上添加书签和获取内容的方式。他们甚至考虑过如何正确打印日历、待办事项和其他内容(这个功能让我想起了我们报道过的可能是技术最不先进的[初创公司)。](https://web.archive.org/web/20211018065657/http://www.beta.techcrunch.com/2005/09/04/pocketmod-type-stuff-and-print-it/)

在线/离线功能是我最喜欢这个产品的地方。这是我们几周前围绕[电子邮件应用](https://web.archive.org/web/20211018065657/http://www.beta.techcrunch.com/2006/09/29/why-the-new-mac-webmail-is-important)讨论的事情，也是 Adobe 即将推出的 [Apollo 平台](https://web.archive.org/web/20211018065657/http://labs.adobe.com/wiki/index.php/Apollo)的承诺的一部分。

这绝对是一个值得关注的创业公司。期待本月发布。
