# 石斑鱼功夫动作(幽默)

**今天向所有员工发出了一封内部 Grouper 电子邮件**，上面写着“ *Grouper Street Fighter。描述:它来找你了……看石斑鱼毁了比赛*”。这封邮件还有一个视频链接，展示了 Grouper(字面意思)击败竞争对手的过程。

我最喜欢的部分是当石斑鱼飞踢 YouTube 的时候。
