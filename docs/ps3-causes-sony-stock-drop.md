# PS3 导致索尼股票下跌

![](img/9f2df0d4c4ad7d0e644484ce97c622a3.png)

与普遍的看法相反，我并不讨厌 Playstation 3。事实上，我迫不及待地想得到一台——如果它能用的话。Macquaire Equities 的分析师 David Gibson 发布了一份报告，称由于今年 TGS 的 Playstation 3 设备出现技术问题，索尼的股票最近下跌了 2.75%。

> 我们担心，这样一个负面问题在如此接近全面生产的时候出现，对公司来说显然是一个负面消息。

吉布森的报告指出，Playstation 3 单元过热，行为不稳定，不得不反复关闭。索尼再次声明 Playstion 3 本身没有任何问题。他们认为我们有多蠢？甚至 1Up 在一个 EGM(电子游戏月刊)编辑玩*赛车风暴*的时候也遇到了一个有缺陷的 Playstation 3！索尼公司的奈奈子·加藤表示，问题是由于通风不良和 200 个 PS3 单元过于靠近而产生的热量。她还表示，这是商展上所有开发商的共同问题。从现在开始，我会忽略她要说的话，希望索尼在 PS3 上市前能正常运行。
