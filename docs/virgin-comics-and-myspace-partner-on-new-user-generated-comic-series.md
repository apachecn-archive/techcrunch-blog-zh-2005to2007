# 维珍漫画和 MySpace 合作开发新的用户生成漫画系列

Myspace 有一个有趣的新项目:他们与维珍漫画公司合作创建了[联合漫画](https://web.archive.org/web/20210126093825/http://www.myspace.com/coalitioncomix)。资深漫画作家迈克·凯里将在网站上指导创作全新漫画的过程。他们承诺将采纳用户的想法和投票来引导故事。

主持这个节目的人是迈克·凯里，他是一名前会计师，为一系列流行漫画撰稿，包括《蝙蝠侠》、《x 战警》、《终极幻想 4》等。

这听起来更像是一个噱头——好的创意作品很少是群体的产物，因为真正激进的东西会被大多数人淘汰。如果你想看这一切的展开，请在 8 月 2 日网站发布时回来查看。
