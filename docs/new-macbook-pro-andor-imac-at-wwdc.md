# WWDC  发布新款 Macbook Pro 和/或 iMac

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/23/new-macbook-pro-andor-imac-at-wwdc/>

![](img/a64f50b896d92a6aa6d1dcfbe1b667eb.png)

又是一周，又是一个苹果大谣言。这个源于下个月即将开始的 WWDC。分析师表示，史蒂夫·乔布斯将在大会期间推出新版 iMac 和新款 Macbook Pro。虽然 Macbook Pro 不会再更新了，但 iMac 肯定需要大修。有传言称，新的设计将更薄，并完全重新设计。

我现在不太相信苹果公司的谣言，但这一条似乎是真的。随着 iPhone 发布日期越来越近，苹果公司完全有理由更加兴奋。也许圣罗莎会带着 Macbook Pro 初次登场…
