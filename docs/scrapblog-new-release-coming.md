# Scrapblog 新版本即将发布

[![](img/2e87a8e42b712165433aa0d72656c1c4.png)](https://web.archive.org/web/20230203024456/http://scrapblog.com/) 明天， [Scrapblog](https://web.archive.org/web/20230203024456/http://techcrunch.com/2006/09/25/scrapblog-to-offer-powerful-web-photo-layout-tool/) ，一个混合媒体共享和在线杂志网站，将在迈阿密的 We Media Conference 上展示他们基于 Flash 的在线剪贴簿应用程序的新版本。新产品的公开版本将于三月问世。不过，联合创始人卡洛斯·加西亚让我快速浏览了一下重新设计的网站，我很喜欢我所看到的。剪贴簿已经是一个伟大的产品。新版本运行更流畅，具有合适的桌面应用程序的外观和感觉，并整合了更多类型的媒体和编辑工具。

新版本具有与原版相同的拖放功能，但通过遵循相同的菜单栏隐喻和全屏选项，更接近地模仿了桌面应用程序。他们还稍微增强了编辑功能，但没有让它对网络来说过于密集。用户将能够使用“编辑”和“属性”工具箱来控制照片和图层的转换，效果，以及照片的裁剪。还添加了帧之间的过渡(各种类型的渐变和擦除)。

该版本将允许用户从更多网站导入照片，如 Webshots、Photobucket 和 Yahoo！。他们以前只支持 Flickr。音频和视频也将被添加到产品中。用户可以选择让歌曲和他们的剪贴簿幻灯片一起播放，并将 YouTube 视频嵌入他们的页面。你将能够旋转和缩放视频，就像照片一样。Scrapblog 还希望将幻灯片导入 YouTube。剪贴簿页面已经可以导出到 Flickr 帐户。

Scrapblog 发展得很好，看起来正在从剪贴簿的利基中扩展出来，因为它更像是 RockYou、 [BubbleShare](https://web.archive.org/web/20230203024456/http://techcrunch.com/2007/01/04/bubbleshare-finally-gets-its-payday/) 、 [Slide](https://web.archive.org/web/20230203024456/http://techcrunch.com/2007/01/06/rumor-slides-venture-round-was-20-million/) 、 [Photobucket](https://web.archive.org/web/20230203024456/http://techcrunch.com/2006/07/04/photobucket-distributing-custom-flock-browser/) 和 [Filmloop](https://web.archive.org/web/20230203024456/http://techcrunch.com/2007/01/06/filmloop-dips-toes-into-the-deadpool/) 等幻灯片放映网络应用。

Scrapblog 目前通过卡洛斯·加西亚的前公司 Nobox 进行私人融资。
