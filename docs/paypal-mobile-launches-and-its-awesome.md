# PayPal 手机发布——太棒了 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2006/03/22/paypal-mobile-launches-and-its-awesome/>

PayPal 已经推出了它的移动支付平台，当然，叫做 PayPal Mobile。一小时前，MobileCrunch 的 Oliver Starr 在[获得了](https://web.archive.org/web/20221209133528/http://mobilecrunch.com/2006/03/22/paypal-goes-mobile/)的独家新闻，也在[遭到了](https://web.archive.org/web/20221209133528/http://hardware.slashdot.org/hardware/06/03/22/2353239.shtml)的攻击。

这给现有的移动支付提供商带来了麻烦。PayPal 的解决方案是完全成熟的，易于使用，并且有一些很棒的功能，超越了给朋友发短信的方式。

要使用该服务，[点击此处](https://web.archive.org/web/20221209133528/https://www.paypal.com/us/cgi-bin/webscr?cmd=xpt/mobile/MobileOverview)并登录(您必须拥有或创建一个 PayPal 帐户才能查看)。您需要注册一个电话并创建一个 PIN。PayPal 使用一个自动系统给你打电话并验证 PIN，然后电话就可以使用了。

要进行个人对个人付款，您可以将付款金额和收款人的电话号码发送给 PayPal，或者拨打免费电话并按照指示进行操作。

贝宝手机也可以用来支付东西。这叫做文字购买:

> 无论何时，当你在海报上、杂志上、活动中看到你想买的东西旁边的“购买”文字时，你都可以通过短信安全地订购该商品。
> 用手机购买 CD、DVD、书籍、电子产品、配件等等。

要使用它，你发短信给贝宝的项目编号，他们会打电话给你确认购买。完成了。

我已经确认使用 PayPal Mobile 不收取额外费用。如果你有一个商业或高级帐户，正常的收费收取金钱适用。当然，大多数移动运营商对短信收费。
