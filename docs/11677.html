荒谬:一个Digg克隆人对另一个Digg克隆人采取法律行动

原文：https://web.archive.org/web/http://www.techcrunch.com/2007/03/19/ridiculous-one-digg-clone-takes-legal-action-against-another/all-comments/

把这个归档在“荒谬”下Digg clone SuperGu 已经向(开源)Digg clone Pligg 发送了一封停止&终止的信，指控其大量侵犯版权。

投诉包括:SuperGu声称拥有ajax效果、侧边栏、登录框、“阅读更多”链接、导航标签、表达“由...提供动力”、标签、RSS图标和其他常见的网页设计元素和布局。

您的yget模板的整体布局和功能在外观和感觉上与SuperGu的软件相似。首先，投票箱的位置和形状基本相似，只有在投票选项和总投票制表器之间反转人字形的微小变化，其次，yget模板顶部的导航选项卡“已发布”、“未发布”和“提交新故事”是相同的。第三，SuperGu上的左侧垂直菜单只被移到了右侧。第四，“搜索”字段与SuperGu在同一个位置。第五，位于搜索字段下方的“排序”功能与SuperGu位于相同的位置，尽管使用了不同的词来实现对用户的相同意义。第六，“排序”旁边、“搜索”下面的“RSS”按钮(右上)类似SuperGu的RSS图标。第七，“标签”功能位于与SuperGu相同的位置，位于storybox中“发帖者”下方和“故事描述”上方第八，“Storybox”与SuperGu位于相同的位置，并包括相同的“评论”、“将链接添加到...”(保存)和“告诉朋友”(电子邮件)功能。第九，屏幕下方的页码与SuperGu的设计相似，功能与SuperGu相同。第十。措辞“由Pligg提供动力”与“由SuperGu提供动力”相同最后，“侧边栏”菜单在外观和操作方法上与SuperGu相同。在侧边栏菜单中，注册/登录框的外观和操作与SuperGu相同。此外，应用于侧边栏菜单中所有框的Ajax效果在它们打开和关闭时是相同的。对话框通过一个按钮打开/关闭，该按钮的操作和外观与SuperGu相似，只是将向下箭头放置在圆形而不是方形中有所不同。“阅读更多”链接与SuperGu的“更多”按钮操作相同，并且位于相同的位置，因此我们认为您的产品侵犯了我们客户的版权。而且。基于你的yget模板和SuperGu之间的大量相似性和复制，这不太可能是偶然的结果，而是有预谋的复制SuperGu的软件设计。

更荒谬的是:SuperGu创始人詹姆斯·菲尔普斯被抓到雇佣一名开发人员来构建他声称Pligg窃取的模板，他说“我只对克隆Pligg中的‘Yget’模板感兴趣:pligg/templates/yget/”这是一个EFF参与的完美案例，我期待看到GuperGu创始人詹姆斯·菲尔普斯的强制性视频道歉。
