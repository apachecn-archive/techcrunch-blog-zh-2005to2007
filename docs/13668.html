Windows Live Hotmail首次亮相

原文：https://web.archive.org/web/http://www.techcrunch.com/2007/05/07/windows-live-hotmail-debuts/comment-page-2/

Hotmail可能是Web 1.0的第一个重大成功，它已经发展了。

微软宣布在全球范围内推出36种语言的Windows Live Hotmail的版本,并辅以AJAX技术。

这项新服务是对之前Hotmail服务的巨大改进，吸收了超过2000万测试人员的意见。然而，今天的产品仅仅是从测试版到正式推出的一步。

我们之前报道过Windows Live Hotmail:早在二月迈克尔·阿灵顿查看了当时的测试服务，结果并不乐观，新的Hotmail排名第三，仅次于Gmail和Yahoo！邮件。

Windows Live Hotmail最初拥有3000万用户，在未来几个月内将增加超过2.5亿用户。

新的Hotmail基于全新的代码，标志着Windows Live品牌的持续整合。

未来几周，微软将发布一款新的免费邮件客户端，旨在取代Outlook Express、Windows Mail和Windows Live desktop。新的客户端被称为Windows Live Mail，将是一个桌面客户端，在用户桌面上本地提供Hotmail的功能集。它还将支持非Hotmail帐户的管理。

登录新的Hotmail还有一些额外的好处；支持是内置的，允许即时通讯的接触，或VOIP呼叫接触，与一次点击，这也将配合与微软的桌面即时通讯平台。
