# 糟糕:生产前的 HTC 设备可能会发送给零售商

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/12/oops-pre-prod-htc-units-possibly-sent-to-retailers/>

![HTC Advantage](img/662e9f622ef8072ff63260320a3d3883.png)

啊哦。似乎新的 HTC Advantage 已经被运到 Amazon.com 的 CompUSA，可能还有其他地方作为预生产单位。当设备第一次启动时，用户会收到“仅供测试，不出售”的信息。

通过电脑更新软件可以很容易地修复这个故障，但是如果从硬件的角度来看这些设备还没有准备好投入生产，这可能是一个大问题。
