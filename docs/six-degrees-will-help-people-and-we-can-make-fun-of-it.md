# 六度会对人有帮助(我们也可以取笑)

SixDegrees 是一个新网站，它敦促用户向名人慈善机构捐款，然后在他们的网站上放置一个小工具来宣传这些慈善机构。它没有得到很好的解释，看起来只不过是一个包装[网络为好](https://web.archive.org/web/20220119082317/http://www.networkforgood.org/)，一个真正优秀的慈善社交网络。当然，这也是为了重振凯文·贝肯在 T4 停滞不前的职业生涯。

用户被敦促在他们的网站上添加一个名人小部件，该部件与一个或多个特定慈善机构的捐赠页面相关联。通过小部件进行的捐赠会被跟踪，总数会显示在小部件中。用户也可以创建他们自己的定制部件，没有名人照片。

培根在下面的视频中详细讲述了《六度分离》电影以及随后的[游戏](https://web.archive.org/web/20220119082317/http://en.wikipedia.org/wiki/Six_Degrees_of_Kevin_Bacon)取笑他在这么多电影中出现的事实。然而，所有这一切与这项服务的联系并没有得到真正的解释，尽管可以假设，因为我们在全球社区中关系如此密切，向慈善机构捐款是件好事。

我们创建了一个小部件来测试它。捐款都捐给了红十字会。
