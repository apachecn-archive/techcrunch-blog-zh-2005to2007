# 华盛顿现在看维基百科。这应该不错。TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/04/22/washingtonwatch-now-a-wiki-this-should-be-good/>

 [](https://web.archive.org/web/20131110061955/http://www.washingtonwatch.com/) [【华盛顿观察】](https://web.archive.org/web/20131110061955/http://www.washingtonwatch.com/)，一个我们去年[写的关于](https://web.archive.org/web/20131110061955/http://www.techcrunch.com/2006/08/06/keep-an-eye-on-government-washingtonwatch/)的网站，分解了提议的美国联邦立法和监管背后的成本，然后以评论和简单的是/否投票的形式要求你的直接反馈。看到一些拟议中的立法给每个家庭带来的平均成本，我觉得很有趣。

该网站此前通过民意测验和评论征求用户反馈。他们现在也把它变成了一个维基，所以用户可以编辑账单摘要。我预计游说者、律师和国会工作人员将根据他们自己的议程大量参与旋转立法，我们将很快看到华盛顿特区版本的维基百科战争的开始。这应该很有趣。
