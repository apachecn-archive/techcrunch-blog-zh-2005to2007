# Wii 的夜晚是真实的，可用于冬季

![](img/799cf5dfedf64d6d8ddf9ccf3563b34a.png)

任天堂 Wii 的*夜*是真实的，将于今年冬天问世，名为*夜:梦之旅*。在过去的一周里，我们已经看到了一堆指向续集的[传言](https://web.archive.org/web/20230322164148/http://crunchgear.com/2007/03/21/nights-sequel-planned-for-nintendo-wii/)，但现在游戏的制作人已经正式宣布了这款游戏的存在，称“*之夜*飞入一个新的梦幻世界，有一个新的故事，在一个新的平台上，名为 Wii。”我年轻时没有土星，所以我真的不知道该期待什么，但我只听说过它的好消息。

你认为现在找到一个 Wii 很难，就等着这个宝石出来吧。
