# 一些我们从未听说过的公司就 iPhone 专利起诉苹果

[![dnfiphone.jpg](img/28670951a2c82f5604e1cf0a3b6ac6be.png)](https://web.archive.org/web/20201129134852/https://beta.techcrunch.com/wp-content/uploads/2007/08/dnfiphone.jpg "dnfiphone.jpg")

也许你想知道——也许你不想知道？——某个不知名的公司正在起诉我们的苹果公司，声称其 iPhone 中的触摸屏侵犯了其一项专利。Conglom-O 的继任者 SP Technologies 公司表示，苹果公司“已经侵犯了，并且现在正在侵犯”(哦，听起来多么官方)其“用户无法终止的计算机可读键盘显示方法和介质”专利。

[*永远的核弹公爵*](https://web.archive.org/web/20201129134852/http://en.wikipedia.org/wiki/Duke_nukem_forever) 会在这套衣服冲击苹果之前问世。
