# 更名和新功能:Google IG 将更名为 iGoogle

谷歌的个性化主页 Google IG 已经更名为 [iGoogle](https://web.archive.org/web/20230301084050/http://www.igoogle.com/) ，并将于明天在 igoogle.com 重新发布。今天，一些博客作者被带到谷歌总部，正在报道细节。

iGoogle 是谷歌进入拥挤的可定制 Ajax 主页空间的进入者，在这里 [My Yahoo](https://web.archive.org/web/20230301084050/https://techcrunch.com/2007/03/08/all-new-my-yahoo/) 是[的明确领导者](https://web.archive.org/web/20230301084050/https://techcrunch.com/2007/03/13/a-little-perspective-on-ajax-home-pages/)，每月有超过 5000 万的独立访问者。iGoogle 目前的月独立访客刚刚超过 700 万。

但谷歌从未推广过 iGoogle，而且它有许多优于 MyYahoo 的功能。最重要的是谷歌的个性化主页支持 widgets(他们称之为 gadgets)。雅虎顽固地拒绝将 My Yahoo 属性与他们的 Konfabulator widgets 平台整合。通过 Google widgets，用户可以将 Gmail、Gtalk 和其他服务直接带到他们的主页。最后，谷歌允许用户自定义 iGoogle 的模板——他们说 30%的用户选择这样做。

谷歌现在在其主页上提供了一个“个性化主页”链接，这将为该网站带来大量新用户。当心，雅虎。
