# 新的亚马逊网络服务即将推出:“SDS”

***更新:**对 SDS 的引用已从以下链接的亚马逊网站中删除。原文保存在本帖底部。*

 [](https://web.archive.org/web/20221130203007/http://aws.amazon.com/s3) 一项名为“SDS”的全新亚马逊网络服务在[这个讨论客户 YouOS 的亚马逊网页](https://web.archive.org/web/20221130203007/http://www.amazon.com/b/ref=sc_fe_c_1_182241011_2/104-1990029-1486358?ie=UTF8&node=242471011&no=182241011&me=A36L942TSJ2AJA)上被提及(参见倒数第二段——截屏也在下面),并且正在与一些精选的亚马逊合作伙伴进行测试。经过一点挖掘，我们听说它可能代表“简单数据服务”，并将于今年某个时候推出，尽管另一个消息来源说这个名字是不正确的。亚马逊的一名代表不愿评论这项服务是否存在。

亚马逊的网络服务部门(参见我们对 [Mechanical Turk](https://web.archive.org/web/20221130203007/http://www.beta.techcrunch.com/2005/11/04/amazon-finally-shows-itself-as-the-matrix/) 、[简单存储服务](https://web.archive.org/web/20221130203007/http://www.beta.techcrunch.com/2006/03/14/amazon-grid-storage-web-service-launches/) (S3)和[弹性计算云](https://web.archive.org/web/20221130203007/http://www.beta.techcrunch.com/2006/08/24/exclusive-amazon-readies-utility-computing-service/) (EC2)的讨论)今年一直在掀起波澜，许多备受瞩目的初创公司正在使用一种或多种服务来加快部署时间和[省钱](https://web.archive.org/web/20221130203007/http://blogs.smugmug.com/onethumb/2006/11/10/amazon-s3-show-me-the-money/)。在最近的一次采访中，[亚马逊首席执行官杰夫·贝索斯](https://web.archive.org/web/20221130203007/http://www.talkcrunch.com/2006/11/14/interview-with-jeff-bezos/)讨论了亚马逊围绕其网络服务部门的战略。
