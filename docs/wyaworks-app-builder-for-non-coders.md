# wya works–面向非编码人员的应用构建器

新创公司 Wyaworks “旨在为网络发展做博客为出版业所做的事情”。换句话说，Wyaworks 希望帮助非编码人员构建有用的 web 应用程序。顺便说一下,“Wya”代表“编写应用程序”。

他们的第一个应用程序叫做 [WyaCracker](https://web.archive.org/web/20221006213155/http://www.wyacracker.com/) ，一个表单生成器。该网站是在私人测试，但测试访问代码可以通过电子邮件从左边栏的“尝试测试”链接请求。

Wyacracker 允许用户构建简单的 web 表单，并在网站上或通过电子邮件发布。结果数据可以在网站上查看或导出到 excel。

它的设计非常粗糙，但核心功能是可行的。不像竞争对手[的表单组装](https://web.archive.org/web/20221006213155/http://www.formassembly.com/)，它也是一个很棒的表单生成器，它是免费的(公平地说，Form Assembly 在这一点上有更健壮的功能)。

我真的很喜欢这样的应用程序，它执行的任务对编码人员来说很容易，但对非编码人员来说却非常复杂。我希望他们对网站进行重新设计，让底层应用程序闪闪发光，我期待着未来的产品发布。要了解更多关于 Wyaworks 的信息，请查看菲尔·西姆的博客。
