# Moto ROKR e6 基于 Linux 的智能手机获得屏幕时间

我们之前已经告诉过你关于[即将发布的基于 ROkR e6 Linux 的 Moto 智能手机](https://web.archive.org/web/20210306062438/http://crunchgear.com/2006/10/19/motos-touchscreen-linux-based-rokr-e6-clears-fcc-excites-fanboys/)，虽然它已经通过了 FCC 的审查，但我们仍然没有看到它。不过，今天，我们有一个小视频的野兽在行动，似乎我们听到的规格(200 万像素自动对焦摄像头，蓝牙，QVGA 触摸屏等)是正确的。

令人兴奋的是，Moto 的智能手机没有使用 Windows Mobile，而是开发了自己的基于 Linux 的操作系统。据传三星也在做同样的事情，但没有 Moto 享受的速度快。虽然我们希望看到有人能替代 WinMo，但 Linux 是否是答案还没有定论。

也就是说，如果做得好，Linux 可以做任何事情。如果你喜欢智能手机，你喜欢 Linux，你会喜欢跳转后的视频。
