# 索尼开发视频随身听

根据最新统计，即使 iPod 保持着 243%的市场份额，索尼也不会放弃他们的 MP3 播放器 Walkman 系列。哦，不，他们实际上正在努力制造一个可以播放视频的随身听。

他们的高级副总裁说，他们正在制作一款处理图像的产品，从任何逻辑评估来看，这意味着玩家既可以处理图像，也可以处理视频。我们猜测“图像”可能意味着“移动的图像”，从日语翻译过来就是不可靠的。

关于视频随身听的其他细节尚未公布，但如果它们要对市场产生任何重大影响，视频播放器要么必须得到视频商店的支持，要么处理大量编解码器——例如 DivX 和 XviD 以使它们对电影下载者更友好。
