# Talkster 为企业推出在线服务

[![](img/078b382475d765122f788bf0f7a0ae81.png)](https://web.archive.org/web/20220120014647/http://talkster.com/) 今晚， [Talkster](https://web.archive.org/web/20220120014647/http://talkster.com/) 在 VoIP 市场推出了一个新的基于在线状态的通信平台的测试版，重点是企业，旨在通过别名连接呼叫者，无论他们使用什么设备。

Talkster 与类似 [Jajah](https://web.archive.org/web/20220120014647/http://www.beta.techcrunch.com/2006/09/26/jajah-just-launched-killer-voip-product/) 、 [Rebtel](https://web.archive.org/web/20220120014647/http://www.beta.techcrunch.com/2006/07/26/rebtel-makes-international-calling-easy-and-cheap/) 、 [Hullo](https://web.archive.org/web/20220120014647/http://www.beta.techcrunch.com/2006/08/22/hullo-to-enter-voip-war-with-free-product/) 、 [ConnectMeAnywhere](https://web.archive.org/web/20220120014647/http://uk.beta.techcrunch.com/2006/09/14/is-moip-the-next-buzzword/) 和 [Fring](https://web.archive.org/web/20220120014647/http://fring.com/) 的服务相似，它使用 VOIP 来拨打廉价(目前免费)的长途电话。但是这些服务是高度针对消费者市场的，而 Talkster 不是。首席执行官詹姆斯·万莱斯称消费者网络电话市场是一场“竞相压低费率的竞赛”Talkster 不想玩那个游戏。Talkster 将目光投向了企业。

Talkster 通过必要的软件桥梁实现电话和个人电脑之间的通话。然而，与 Jajah 和 Hullo 不同的是，Talkster 不需要下载，可以通过手机和桌面浏览器在网络上工作。这使得它比 Jajah 的基于 Symbian 的客户端应用更广泛。Talkster 将在 2007 年推出 Java 客户端。目前，该系统连接移动电话、VOIP(即时消息语音)、VOIP 电话和 PSTN 电话，并有可能连接到任何使用 SIP 的服务。你还可以使用 Talkster 向 MSN、[、Google Talk](https://web.archive.org/web/20220120014647/http://www.beta.techcrunch.com/2006/07/28/google-talk-swings-back/) 和 [Gizmo](https://web.archive.org/web/20220120014647/http://www.beta.techcrunch.com/2006/07/20/gizmo-nukes-standard-voip-business-model/) 上的几乎任何手机或好友拨打免费手机电话。

在线注册这项服务后，你需要输入联系人的别名、电子邮件地址(对于 VoIM)和电话号码，这是一项繁琐而又必要的任务。VoIM 联系人必须接受您的邀请才能加入 Talkster。

![](img/0d192a6d7ef64fe9be3e1b899567a505.png)当您用手机登录 m.talkster.com 时，会看到一列联系人的 VoIM 在线/离线状态，您可以通过 VoIM 连接到联系人，也可以自动转接到他们选择的电话号码。在长途电话的情况下，Talkster 充当两个呼叫者之间的 VOIP 桥梁。打本地电话时，Talkster 不碍事，直接连接。

如果你不想在你工作的电脑上打电话，Talkster 有呼入或回叫选项，让你通过电脑打电话，但使用电话连接。该选项具有明确的成本优势，这取决于您的移动计划。

Talkster 的设备不可知模型让人想起基于文本的 [JumpChat](https://web.archive.org/web/20220120014647/http://www.beta.techcrunch.com/2006/11/09/the-y-combinator-companies/) 和 [Swarmteams](https://web.archive.org/web/20220120014647/http://www.beta.techcrunch.com/2006/09/18/swarmteams-communicate-like-animals-by-sms/) ，但加入电话呼叫使该模型更加强大。网络界面是一种牺牲，但它至少会让 Talkster 出现在尽可能多的人面前。一个易于使用的客户端将真正使服务活跃起来，并使它比 Jajah 或 Rebtel 更有吸引力。尽管如此，我们还没有看到这是否是我们一直在等待的杀手级网络电话服务。

Talkster 非常依赖其企业目标作为主要的差异化因素。我们无法让该公司说出为什么它更适合企业而不是消费者，但我们得到承诺，2007 年将增加企业功能。与此同时，测试版将测试核心服务。
