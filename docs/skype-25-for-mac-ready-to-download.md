# Skype 2.5 For Mac 可供下载

今天，这家互联网通讯公司在邮件中为 Mac OS X 用户发布了一款新版本的免费应用:Skype 2.5。

测试版从 11 月中旬就开始了，但是最终版本今天才正式发布。除了稍微更新的外观，还有一个新的生日提醒功能，可以向世界上任何一部手机发送短信，并与最多九个人举行电话会议。
