# 美国军队将离开 MySpace，YouTube，乐趣

为了“节省带宽”,国防部封锁了官方电脑对 YouTube 和 MySpace 等管道的访问。许多雇主都会安装这样的过滤器，但这次不同:

> 住在美国基地的军人和家属仍可以通过私人互联网浏览这些网站，但此举让在伊拉克和阿富汗使用流行图片和视频分享网站的军人很少或根本无法访问这些网站。

 当然，抛开所有的“带宽”问题，国防部有一种道德监管的声誉，可能不喜欢他们最优秀和最聪明的人浪费他们的脑细胞在 MySpace 上与他们的高中恋人聊天。

不要以为这不会被忽视:我们在海外的大部分男孩几乎没有高中毕业(MySpace key demo)。我们的观点:男人被击中胯部的视频对士气来说是一个巨大的福音，国防部在这里真的是大错特错了(没有双关语)。

如果你问我们，看可爱的小猫家庭电影和大学生出丑的照片的自由不正是我们的士兵为之奋斗的吗？而且，严肃地说，你会认为国防部希望我们的人感觉到他们与家乡的家人和朋友有联系。可悲的是，这只会切断他们。
