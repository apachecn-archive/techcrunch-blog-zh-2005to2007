# 使用 Farecast 查找航班信息(或者只飞往西南方向)

 [](https://web.archive.org/web/20221225065350/http://www.farecast.com/) 总部位于西雅图的 [Farecast](https://web.archive.org/web/20221225065350/http://www.farecast.com/) ，现在处于私人测试阶段，是一个机票价格比较工具，它还使用预测算法在购买机票时推荐*。因此，这个想法不仅仅是向用户展示谁的机票最便宜，还展示了等待是否有意义。期待总部位于明尼阿波利斯的 Flyspy 在不久的将来推出类似的服务。*

该网站设计良好，界面清晰地传达了价格和预测信息。在测试的这一点上，你必须选择西雅图或波士顿作为出发城市，所以我查看了西雅图-洛杉矶航班。最便宜的选择是 329 美元(美国航空)，Farecast 以 80%的把握告诉我，最低票价将在未来 7 天内上涨 50 美元。

好的，太好了。但西南航空的往返机票价格为 308 美元，不包括在价格比较中。这才是真正的问题所在。Farecast 是一个很好的解决方案，可以从航空公司完全混乱的定价中提取有用的信息。但是西南航空公司不玩这些游戏，也不向 Farecast 这样的比较引擎开放他们的服务。所以最低和最容易理解的价格被排除在服务之外。

虽然我喜欢 Farecast，并会用它来看看我是否能以低于西南航空公司的价格获得航班，但我真正希望看到的是我们的航空公司从集体的屁股中拿出集体的头脑，采用一种有意义的定价模式。例如，其中一个不需要像 Farecast 那样的专有算法就能理解。
