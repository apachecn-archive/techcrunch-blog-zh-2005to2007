# 天蓝色工作室预览 Trillian Astra

[![](img/158727107876cb12bbfc80d09288cb3a.png) ](https://web.archive.org/web/20221129045330/http://www.ceruleanstudios.com/ap/) [天蓝色工作室](https://web.archive.org/web/20221129045330/http://www.ceruleanstudios.com/ap/)已经开始提供一些关于其流行的跨网络即时通讯工具 Trillian 的下一版本的信息。新版本名为 Trillian Astra，有一个[预览网站和视频](https://web.archive.org/web/20221129045330/http://www.ceruleanstudios.com/ap/)现在可用，是 [Jordan Running](https://web.archive.org/web/20221129045330/http://www.downloadsquad.com/2006/11/21/trillian-astra-sneak-preview/) 首先为我们指出的。Trillian 允许用户使用一个应用程序与许多其他 IM 系统上的人进行 IM。

这个新版本将会在一个拥有大量忠实用户的程序中看到一些大的变化。以下只是该公司表示该软件最新版本中的一些新东西。Trillian 仅适用于 Windows。阿斯特拉目前在私人阿尔法，我们只是很兴奋地报道这第一次预览。

现在 IM 中有很多创新，我们计划在接下来的几天里在这个领域发布一个概述，但是这个 Astra 预览版太热门了，我们认为它应该有自己的帖子。

我们在这篇文章的结尾嵌入了该公司的宣传视频。

**特性**

该应用程序允许用户在 Yahoo！、ICQ、AIM、Bonjour、Windows Live、Google Talk 以及现在的 MySpace IM。该公司表示，MySpace IM 集成已经成为一段时间以来的主要功能需求。

新版本允许用户选择多达 5 个文件同时拖放文件传输；Trillian 会自动将它们压缩成 zip 文件发送出去。图像可以立即放入窗口进行文件传输。

Astra 将比以往任何时候都更容易从多台计算机上访问。每当你在任何一台电脑上登录 Astra 时，该程序都会下载你的联系人列表。

**网络存在**

每个用户都有一个个人 Trillian Astra 网页，其中集成了支持离线消息传递的网络聊天小工具。您也可以选择关闭 web 发起的聊天会话的通知。你的 Astra 页面可以充满你的信息，这些信息来自你在社交网站上提供的 URL，以及显示你的 Flickr 流、你正在听的歌曲、当地天气等的小部件。这些网页同步将与您的桌面客户端同步，并将支持逐个访问者的个性化-换句话说，我可以选择是否要查看您的音乐、天气、博客订阅等。该部件还将支持一次点击大小最大化。

桌面客户端将包括一个名为“知识栏”的 RSS 新闻收报机，它根据客户端算法确定的上下文相关的新闻条目进行循环。Astra 将支持 TabletPC 输入，因此用户可以通过即时消息发送手写信息。

新版本将允许用户搜索和过滤他们的联系人列表，朋友的头像将能够以磁贴格式显示。最后，这个新版本承诺运行更快，并消除内存泄漏。

总之，一个非常令人印象深刻的预览。如果伊鹂莎能做到这一切，而且有充分的理由相信他们能做到，那么这将是一个吸烟产品。
