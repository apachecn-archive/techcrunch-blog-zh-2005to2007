读者们，你们好。在过去的几天里，最好的评论者是 MIA，但现在它将在美国东部时间晚上 9 点左右出现。感谢您的精彩评论，它们的质量一直在提高，我们真的很感激我们有如此活跃的读者群。

总之，**职业杀手**在告诉我们他对苹果粉丝的看法后赢得了第一名。我们同意。

> 为此干杯，尼古拉斯。苹果公司制造非常有灵感的产品，吸引了非常忠实的粉丝群。。。但是防御性已经变得难以置信的失控了。如今批评苹果就像骂某人的母亲是妓女一样严重。我很惊讶人们会这么关心。去享受你的系统，让那些讨厌的人去恨吧，对吗？就我个人而言，我发现任何其他方法都反映了用户的集体心态。(和往常一样，可能是少数敢于直言的人破坏了其他人的想法。)
> 
> 不过，有人可能会说，苹果公司应该对崇拜他们的粉丝们长期以来的垃圾态度负责。他们最近的广告活动(我是苹果电脑，我是个人电脑)就像是垃圾话政治。我更希望看到一家公司让自己的产品为自己说话，而不是在竞争中低打出手。

**兰斯**因分享他对[格式战](https://web.archive.org/web/20151009032354/http://crunchgear.com/2007/06/27/interview-with-ken-lowe-vp-of-business-development-and-strategic-marketing-for-sigma-designs-talks-about-blu-ray-and-why-hd-dvd-is-dead/)的看法而获得第二名。

> 我把这场格式之战视为几十年前 VHS 与 Beta 之战的翻版。Lowe 先生的观点似乎是(因为没有直接引用，所以我只能做出假设)蓝光是唯一可能的赢家，因为光是因为更大的数据容量，它就是更好的格式，削减成本的播放器不会有任何影响。
> 
> 回过头来看，这听起来像是索尼反对 VHS 的老理由。贝塔是更好的产品，更好的质量，更好的一切。但是 Beta 失败了，因为 VHS 的价格比它低，磁带也更便宜。我还认为这就是 Windows 比 Mac 更流行的原因——硬件和软件在 Mac 上可能更好，但在 Windows 机器上更便宜。
> 
> 归根结底，在一场技术大战中，给定两个大体上平等的选择，消费者选择了更便宜的选择。我不认为价格差异能如此容易地被忽略。

里奇·卡登在 iPhone [数据计划](https://web.archive.org/web/20151009032354/http://crunchgear.com/2007/06/27/edge-network-could-cost-apple-tens-of-customers/)上的输入使他排在第三位。

> 我认为这是一个明智的决定。让我们面对现实吧，就像上面评论的那样，苹果是基于美元来选择美国电话电报公司的。(坦率地说，在我看来，这真的只是威瑞森和美国电话电报公司之间的事)。这就剩下了 3G (HSDPA/UMTS)或 EDGE 的选择。EDGE 是更明智的选择。为什么？
> 
> 因为美国电话电报公司没有覆盖全国的 3G 网络。因此，要么他们将不得不在地理上限制可用性(不好)，要么他们将让生活在边缘地区的人购买 3G iPhone，并获得较差的体验。我们知道乔布斯对他的产品的劣质体验的感受。
> 
> 我个人认为边缘网络并没有那么慢。他们已经对它进行了翻新(我想它被称为 Operation Fine Edge ),我可以说，使用诺基亚 N95，我已经看到数据传输速度最近有所提高。
> 
> 另外，我不认为 iPhone 是数据密集型的。我目前平均每月在手机上使用 500MB，这包括下载音乐/播客、流媒体视频、痴迷地查看 Gmail 以及一般的网上冲浪。
