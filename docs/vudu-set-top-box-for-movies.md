# Vudu:电影机顶盒

![vuduboxmarkmark.jpg](img/fd9d88370e2162eca9c8e21ca4393a8d.png)

Vudu 是一家小公司，它正在解决一个老问题:如何在不口吃、不缓冲、不强迫你走到邮箱前的情况下把电影放到你的电视上。解决办法？P2P。

Vudu box 将存储您可能喜欢观看的电影的开头，然后有条不紊地-快速地-从 Vudu 网络上的同行下载电影片段。“但是等等，”你会说，“难道不需要有很多同辈人在……”嘘！别把这些告诉 Vudu！

是的，在网络上需要有很多的同行，这是我认为这个产品会动摇的地方。我不想成为一个扫兴的人，但是我可以看到 Vudu 成为一个专利农场，在我看到一个提供电影的独立业务之前，这项技术最终会出现在康卡斯特的盒子里。

事实上，Vudu 似乎是一个现代的，虚张声势的 TiVo。我认为这是错误的。TiVo 可以让你抓取你已经拥有的东西——直播电视——并播放它。这项工作传统上落在了 Hitachis 和 JVC 的肩上。它类似于录像机，人们得到了录像机并购买了数百万台 TiVos。有线电视公司不擅长视频存储。这对他们来说是陌生的，他们无法忍受他们的客户可能想要跳过频道。因此，他们一直置身于市场之外，或者发布功能相对较弱的设备。

Vudu 正在践踏供应商的地盘。苹果电视及其同类产品还不错——这些都是时移设备，如果你认为 100%的苹果电视用户都在他们的电视上观看 ITMS 的内容，那你就错了。第一次黑客攻击是为了让它播放 DivX，这是有原因的，而不是因为有人没有先见之明，在 MPEG4 记录他们孩子的 bris。

但是视频点播是一个管道问题。有线电视公司不想要任何东西，除了他们自己的服务占用大量的带宽，不管有没有 P2P，你都会在这个设备中有很多上下游。

Vudu 将出售几个盒子，用打破它们的网络效应，他们的技术将出现在 Comcast Pioneer SuperOnDemand 机顶盒中。我和其他人一样喜欢创业，但不是这个。
