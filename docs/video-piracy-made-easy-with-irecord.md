# 使用 iRecord 轻松实现视频盗版

有了流媒体网络公司的新 iRecord，你的电脑现在已经过时了。无论如何，这至少是一种从你的 iPod、PSP 或其他 MPEG-4 播放设备上获取音频或视频的方法。这是一个小型视频捕捉/编码设备，具有模拟电视和音频输入以及 USB 输出。

iRecord 使用 H.264/ACC 采集和编码视频，使用 MP3 采集和编码音频，大小约为 iPod 的两倍。酷的是，它没有自己的内部存储；插入播放设备后，它会直接录制到设备的硬盘(或闪存或其他什么)上，因此您可以立即播放。

朋友们，这很酷。它还可以记录你高兴地连接到 USB 端口的任何 USB 大容量存储设备，所以你甚至可以把它用作穷人的 DVR，尽管 200 美元的价格标签，穷人不会很快得到一个。
