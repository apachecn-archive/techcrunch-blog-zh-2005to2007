# T-Mo 滴两个新的性感，性感三星

T-Mobile 刚刚推出了两款不同配置的超薄三星手机:T619 翻盖手机和 T629 滑盖手机。中档手机具有相似的功能，提供 130 万像素的摄像头，蓝牙，T-Mobile 网络，并包括雅虎！、MSN 和 AIM 信使。

T629 基本上是 T-Mobile 版的 Cingular D807，确实是一款不错的手机，与 T619 不同的是，滑块包括一个可以播放 MP3 的 MicroSD 插槽，可以在国际上使用。

这两款手机现在都可以从 T-Mobile 获得，T619 的价格为 79.99 美元，T629 的价格为 149.99 美元。
