# Saitek 的高科技复活节篮子

![](img/5be9851fc40db0d316d84a1b0d56e5f2.png)

好吧，Saitek 实际上并没有发布高科技复活节篮子，但他们确实把这个放在了一起，作为春季/复活节宣传的一部分。他们的篮子里有四个 Saitek 的笔记本鼠标，有橙色、黄色、绿色和粉色可供选择，每个 19.99 美元，他们建议将这些鼠标作为礼物或添加到您的篮子中。离圣诞节还有 276 天，所以你不能责怪一家公司的尝试！
