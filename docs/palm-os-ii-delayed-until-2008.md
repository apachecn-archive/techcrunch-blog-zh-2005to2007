# Palm OS II 推迟到 2008 年

![](img/80cd5606d34010d32a6c25e6f57a681a.png)

Palm 新的基于 Linux 的移动操作系统刚刚被推迟到 2008 年。今年 5 月，当该系统发布时，Palm 公司声称，在“OS II”上的设备将在年底出现在消费者手中。然而，最近 Palm 的首席执行官 Ed Colligan 说“我们正在开发的基于新 Linux 平台的产品，要到明年的某个时候才会出现。”

他还确认了 Palm 对 Palm 平台和 Windows Mobile 的承诺。
