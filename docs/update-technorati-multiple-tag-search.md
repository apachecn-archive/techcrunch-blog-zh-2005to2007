# 更新–Technorati(多标签搜索)

 [Niall Kennedy](https://web.archive.org/web/20201129134444/http://www.technorati.com/about/staff.html?s=niall_kennedy) 刚刚宣布 [Technorati 现在允许多个标签搜索](https://web.archive.org/web/20201129134444/http://www.technorati.com/weblog/2005/08/42.html)。

有了这个新功能，你可以进行一次搜索，找到你输入的两个或更多标签中的任何一个。

尼尔说:

> 您现在可以在 Technorati 中搜索多个标签！只需用单词“或”分隔每个标签，就可以将带有指定标签的帖子添加到搜索结果中。
> 
> 多个标签是追踪你最喜欢的话题的好方法，同时也考虑到了人们给他们的帖子加标签的各种方法。[学院或大学标签搜索](https://web.archive.org/web/20201129134444/http://technorati.com/tag/college%20OR%20university)显示由 Technorati 索引的最新帖子，标签为“学院”或“大学”你也可以在感兴趣的领域混合标签，比如通过对 PSP 或 GameBoy 的[标签搜索来跟踪移动游戏行业。](tag%20search%20for%20PSP%20OR%20GameBoy)
> 
> 我相信您对多标签搜索如何帮助您发现新内容以及了解您所关注主题的最新动态有更多的了解。尽情享受吧！

好吧，酷。这为某些搜索节省了一点时间。

想真的哇我们？创建“和”功能，并允许我们通过多个关键字过滤。那会让我们很快得到超级相关的结果。TechCrunch 的
