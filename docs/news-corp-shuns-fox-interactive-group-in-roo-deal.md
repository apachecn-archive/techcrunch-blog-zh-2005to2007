# 新闻集团在鲁迈拉运营公司交易中避开福克斯互动

华尔街日报[今早报道](https://web.archive.org/web/20221203093002/http://online.wsj.com/article/SB117003197066590694-search.html?KEYWORDS=roo&COLLECTION=wsjie/6month)(在付费墙后面)新闻集团将宣布对在线视频初创公司 [ROO](https://web.archive.org/web/20221203093002/http://www.rootv.com/?bt=NS&bp=MAC&bst=FF&biec=false) 投资 1200 万美元。然而，与互联网初创公司的其他投资和收购不同，新闻集团的子公司福克斯互动没有参与这笔交易。福克斯互动拥有 Myspace，并投资了一些初创公司，如[simple hired](https://web.archive.org/web/20221203093002/http://www.beta.techcrunch.com/2006/04/18/simply-hired-takes-135m-from-fox-and-foundation-capital/)。他们是新闻集团的互联网部门。

据我们所知，福克斯互动不仅没有参与交易，他们甚至不知道这件事。有传言称，福克斯互动的高管们是在今天早上阅读《华尔街日报》的文章时才听说这笔投资的。他们不开心。一位内部人士告诉我们，他们不相信新闻集团会在没有福克斯互动参与(或至少知情)的情况下进行互联网交易，并表示他无法相信新闻集团会投资那场“该死的灾难”。

更尴尬的是，福克斯互动一直在与鲁迈拉运营公司的竞争对手 Brightcove 进行单独对话，bright cove 最近在一轮融资中筹集了近 6000 万美元，传言其估值为 2.2 亿美元。据报道，在融资结束前，福克斯互动正在谈判以 2.5 亿至 3 亿美元收购 Brightcove。

这显然是一个右手不知道左手在做什么的例子。

自从前任负责人罗斯·莱文索恩离开，取而代之的是一位几乎没有互联网经验的高管，福克斯互动就被削弱了。新闻集团在没有咨询福克斯互动的情况下对一家互联网初创公司进行了重大投资，这一事实表明，新闻集团不再指望他们的互动子公司来指导他们的在线战略。
 ***更新:**我收到另一位内部人士的电子邮件，称《华尔街日报》最初的报道是不正确的，新闻集团获得了基于特定收入/业绩阈值收购鲁迈拉运营公司股票的认股权证。尽管来自福克斯互动公关公司的报道称，他们确实知道这笔交易，但我们坚持我们最初的评估，即这对他们来说是一个完全的惊喜。*
