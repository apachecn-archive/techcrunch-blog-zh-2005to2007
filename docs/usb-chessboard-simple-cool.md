# USB 棋盘:简单=酷

![ulife008500_03_l.jpg](img/06c5193d2ef30aed37fb203bd987cf5b.png)

我的 Treo 上有两个独立的国际象棋程序(已付费，谢谢)，但不管我的骑士移动技能在虚拟世界中变得多么邪恶，我总是在真实的国际象棋比赛中被打败。我认为这与我们相对于数字感知物理板的方式有关。或者可能是我瞎编的。

这个漂亮的 USB 棋盘可能是我的答案。它记录了物理棋子的移动，就像它是一个虚拟棋盘一样，所以你可以在保持真实感的同时与计算机对战。我们喜欢它。

我们很高兴它能够通过互联网扮演朋友，但这是通过老式的电子邮件。很烂，但还是很酷。谁给你带来了这个闪亮的书呆子灯塔？白兰度，就是他。
