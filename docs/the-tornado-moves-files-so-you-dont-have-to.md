# Tornado 会移动文件，因此您不必| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/01/03/the-tornado-moves-files-so-you-dont-have-to/>

。

以 60 美元的一次性费用，Tornado 可以让你通过 USB 连接在基于 Windows 的电脑之间传输文件。一端放在一个系统中，另一端放在另一个系统中，然后您只需在两者之间随心所欲地拖放。也没有要安装的软件。一旦你连接上了，你会看到一个分屏界面，上面显示了电脑的驱动器和文件夹设置(见跳转后的截图)。

但这就是它的全部功能。如果您经常需要在计算机之间移动大文件，这是很方便的，但在其他情况下，这项任务可以通过闪存驱动器或 CD-R 来完成(尽管直接 USB 连接会更快)。
