# 松下为 2007 等离子生产线增加了 42 英寸 1080p 设备

![](img/dfe63f3e636bcbe8fd45716d9171d1ce.png)

正如我[提到的](https://web.archive.org/web/20210225052948/http://crunchgear.com/2007/03/30/pioneer-to-show-off-8th-gen-plasmas-this-may/)，松下昨天参观了哥谭市，展示了今年的等离子电视阵容，包括将于 6 月上市的首款 42 英寸 1080p 型号，售价 2500 美元。(哇，连载句预警。)松下在今年早些时候的消费电子展上展示了许多这样的等离子电视，但如果能和它们单独相处，而没有一大群人挤过来反对我，也无妨。

今年有九款新机型，其中两款是 2006 年的延续，包括 103 英寸的 1080p giant。所有型号都至少有两个 HDMI 端口，一个 SD 卡插槽和一个新的防眩光涂层。最高端的型号可以解码 h.264，所以你可以加载一个充满类似编码电影的 SD 卡，并在大屏幕上观看它们。在昨天所有的演讲者中，松下的工程师是最有活力的(高层管理人员[通常](https://web.archive.org/web/20210225052948/http://crunchgear.com/2007/03/29/video-of-the-iphone-at-ctia-hilarious-exec-blathers-and-blathers/)不太会说话)，他解释了为什么新的等离子体比液晶显示器更适合处理快速移动的图像(想想体育)，因为它们不会像液晶显示器那样模糊图像。同样值得注意的是，松下与 GalleryPlayer 合作，让你在电视上观看高分辨率的艺术作品。对于像我这样一个自命不凡的人来说，这是一笔丰厚的额外津贴。

Crave 的男孩和女孩们实际上测试了一种新的等离子体，并普遍喜欢它，但出于某种原因，绿色看起来不太对。
