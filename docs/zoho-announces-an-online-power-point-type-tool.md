# Zoho 宣布推出一款在线幻灯片工具

在线办公套件 [Zoho](https://web.archive.org/web/20230118023949/http://zoho.com/) 本周推出了一款新的演示工具，名为 [Zoho Show](https://web.archive.org/web/20230118023949/http://zohoshow.com/) ，用于在网上创建和展示远程演示。Zoho 母公司 [Adventnet](https://web.archive.org/web/20230118023949/http://www.adventnet.com/) 是 TechCrunch 的赞助商。

[Thumbstacks](https://web.archive.org/web/20230118023949/http://techcrunch.com/2006/03/09/thumbstacks-ajaxflash-web-powerpoint/) 似乎是 [Zoho Show](https://web.archive.org/web/20230118023949/http://zohoshow.com/) 面对的首要对手，两者都有一定的优缺点。

这一最新产品是第 12 款 Zoho 网络应用，包括文字处理器、电子表格、聊天、CRM、应用创建器等等。

Zoho Show 最重要的功能是可以导入 Power Point(以及打开 office)文件。这是它相对于拇指堆栈的主要优势。不幸的是，这些文件有 1 MB 的上传限制，所以大量的 Power Point 文件无法工作。Zoho 文件也可以导出为 power point 文件，或者只需点击一下就可以上传到网上，而 Thumbstacks 可以导出为 HTML。

Zoho Show 和 Thumbstacks 都提供远程控制的幻灯片在线显示，并且都可以很容易地在公开和私人之间切换。选择 private 时，如果您尝试加载公共 URL，两个系统都会显示一条错误消息。

Zoho Show 似乎不支持箭头的创建和操作，这对于演示来说似乎是个大问题。拇指堆有。根据 Thumbstacks 开发者的博客，这是一个真正的技术壮举，浏览器对箭头的支持。相反，Zoho 提供了一些箭头图像，你可以四处移动和调整大小。我不希望不得不那样做。

这两项服务都支持从 Flickr 轻松导入照片，但只有 Zoho 会提示你登录以访问你标记为私人的照片。我可以想象基于私人网络的演示文稿与私人网络存储的照片放在一起，如果这是你正在寻找的，你可能不想使用拇指堆栈。

Thumbstacks 似乎从一开始就在 MAC 电脑上苦苦挣扎，并且幻灯片放映一次只能有 5 个观众。总的来说，我在带有拇指堆栈的 Mac 上收到了更多的错误信息和用户界面问题，但能够在幻灯片中添加箭头对我来说非常重要。虽然 Zoho Show 是一个商业供应商提供的更好的产品，但在他们添加对箭头的支持之前，至少我可以想象自己使用 Thumbstacks 来创建大多数幻灯片，只是为了在电话上向人们展示。如果我与使用 Power Point 的人合作做某件事，我会使用 Zoho Show 我会请他们帮我把箭头放进去。

公平地说，现在 12 个服务强大的 Zoho 套件令人印象深刻。当 Zoho 发布的时候，我们在这里回顾了 Zoho 的作者，很明显，该公司很认真地想随着时间的推移改进他们的产品。这里希望 Zoho Show 增加它的能量点上传限制，并增加对箭头的支持。
