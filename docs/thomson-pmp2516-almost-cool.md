# 汤姆森 PMP2516:几乎很酷

汤姆森中国为我们带来了这款轻薄小巧、类似 iPod 的手机，PMP2516。它几乎兼容你想要的任何格式，MP3，Mpeg 1-4，Jpeg 图片，WMA，ASF，WMV 和 WAV。2.5 英寸的液晶屏幕大小正好适合观看，这个小家伙甚至还支持电视输出连接，这确实很棒。USB 2.0 和 1GB 存储完善了这些功能。
