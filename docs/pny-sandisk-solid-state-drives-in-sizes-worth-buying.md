# PNY，闪迪固态硬盘的尺寸值得购买

![ssd-5000-family-64gb.jpg](img/3878fa796d376caa57971b6bec2df7b2.png)在 2007 年台北国际电脑展上，闪迪展示了其固态硬盘(SSD)系列的两款新产品:[闪迪1.8 英寸 64GB UATA 5000 和 2.5 英寸 64GB SATA 5000 固态硬盘产品。这被 PNY 发布的 2.5 英寸 128GB 固态硬盘所超越。闪迪和 PNY 分别宣称读取速度为 67MBps 和 66MBps。这对于快速启动和文件访问来说怎么样，嗯？](https://web.archive.org/web/20201026013921/https://crunchbase.com/organization/sandisk)

两家公司都没有列出驱动器的价格，因为它们现在似乎只给原始设备制造商。对于大量订单，PNY 以 350 美元左右的价格出售其 32GB 硬盘，所以你可以算一下。此外，不要指望 PNY 驱动器直到 2007 年第三季度(其 128GB 的驱动器，我的意思是，32GB 的型号目前正在发货)。然而，SanDisk 的新 64GB 型号现在已经上市。
