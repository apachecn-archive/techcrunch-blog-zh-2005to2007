# 它有木头:木制电脑模块

![](img/a2b94b2e8e7bed1ff372d606b28571c7.png)

电脑，因为没有更好的词，是美丽的。这些定制的电脑是由枫木、杨木、胡桃木、冈卡洛·阿尔维斯木、枫木、豹木、帕多阿克木等制成的。正如你所看到的，电脑并没有完全被木头覆盖，木头只是电脑的外壳。内部机构是明确的，让你看到所有那些高端组件，你付出了血汗钱。在你挑选出你喜欢的机箱模型后，你就可以挑选组件了。这些电脑并不便宜——即使是最便宜的电脑也要 6000 美元。所有的电脑都使用 AMD 处理器(对不起，英特尔粉丝)，华硕豪华主板，GeForce 7900 GTX，2GB 内存和 960GB 硬盘。当你分解每个部件的价格时，你并没有多付多少钱。

如果你不想定制你自己的系统，Suissa 也为特定的工作提供个人电脑。您可以在家庭/办公室、游戏和娱乐之间进行选择。你总是可以从其中一个开始，然后不断升级。瑞士电脑公司也接受定制订单，所以如果你的钱包能承受的话。您也许可以偷偷将英特尔酷睿 2 双核至尊处理器安装到您的电脑中。
