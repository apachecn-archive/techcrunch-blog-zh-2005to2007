# 准备稳定投票。

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/09/ready-steady-vote/>

2006 年第三届年度“[网站](https://web.archive.org/web/20160420162800/http://www.websiteoftheyear.co.uk/)提名名单今天刚刚揭晓。从今天开始到 12 月 1 日，英国互联网用户可以通过对被提名网站的内容、导航和设计进行排名，为他们最喜欢的网站投票。

年度网站是由在线调查机构 [Metrixlab](https://web.archive.org/web/20160420162800/http://www.metrixlab.com/site/uk/home.asp) 运营的泛欧洲人民选择奖，并得到了 [Nielsen//Netratings](https://web.archive.org/web/20160420162800/http://www.nielsen-netratings.com/) 的支持。

去年，年度网站选举在八周内吸引了超过 150 万的独立访客。去年的赢家包括 [Zylom](https://web.archive.org/web/20160420162800/http://www.uk.zylom.com/index.jsp) 、[never statist](https://web.archive.org/web/20160420162800/http://www.neverstandstill.com/)、 [Puredance](https://web.archive.org/web/20160420162800/http://www.puredance.co.uk/) 、Google、192 和 [Profileheaven](https://web.archive.org/web/20160420162800/http://www.profileheaven.com/) 。

> “今年，我们扩大了类别，以反映不断变化的 Web 2.0 格局:拍卖、博客、汽车、社区、娱乐、金融、娱乐和游戏、健康、ISP、就业、导航、新闻、门户、搜索引擎、购物、体育、电信、旅游、电视和广播、视频共享。”MetrixLab 的 Frans-Anton van Gils 说

每个类别都有两个奖项:年度最佳网站(在内容、导航和设计方面平均得分最高的网站)，以及年度最受欢迎网站(票数最高的网站)。
