# preview seek–新的网络搜索引擎

![Preview Seek](img/2f581d7ecb4d295e16136df291da39fa.png)**Company:** [PreviewSeek](https://web.archive.org/web/20201202012800/http://www.previewseek.com/) **Launched:**

2005 年 8 月

**Status:**

天使投资

**Location:**

英国伦敦

PreviewSeek 是洪成昌在伦敦创建的一个新的搜索引擎，它以许多创新的功能悄悄地给人们留下了深刻的印象。

基本的搜索结果很棒，但没什么好兴奋的。有用的功能包括消除查询歧义(你是在搜索水果“苹果”，还是计算机“苹果”？)，搜索页面的预览(想想 Browster)，更好的细化搜索。

## 理解您的疑问

PreviewSeek 在尝试确定查询的含义方面做得很好。输入“ [Java](https://web.archive.org/web/20201202012800/http://beta.previewseek.com/?where=web&vid=l242319538I1129054663&twin=n&query=java&x=0&y=0) ”，你会得到一个带有大量查询选项的结果集，包括岛屿、编程语言，甚至咖啡。

## 预览结果

这是我最喜欢的功能。除了没有下载和广告之外，PreviewSeek 允许你以与[浏览器](https://web.archive.org/web/20201202012800/http://www.browster.com/)(我们的[个人资料](https://web.archive.org/web/20201202012800/http://www.beta.techcrunch.com/2005/08/11/update-browster-firefox-surging-ahead/))非常相似的方式预览搜索结果。这是一个快速浏览结果的好方法，不需要点击预览。

## 搜索细化

对于任何给定的查询，PreviewSeek 会在左侧栏建议一些优化选项，这将极大地帮助您深入特定的搜索。
