# Powerset 会拉个谷歌吗？

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/10/05/will-powerset-pull-a-google>

**更新:**今天早上我收到了一封来自可靠消息来源的电子邮件，内容是*“…我已经看过了演示(并和那些家伙一起度过了相当长的时间)…这项技术非常好，肯定会对网络搜索查询的一个重要子集有所帮助。但他们肯定有自己的工作要做。索引比传统的网络搜索要昂贵得多，这意味着要做一些真正的扩展和性能工作……当然，我不能说得太多，否则会惹恼 NDA 的上帝……(哦，那些 VentureBeat 的数字在资金方面是错误的。)"*

* * *

[![](img/e5afd88822fb4bd7f263d5a2bd3dd0ba.png)](https://web.archive.org/web/20220810220032/http://www.powerset.com/) 我还没有看到即将推出的搜索引擎 [Powerset](https://web.archive.org/web/20220810220032/http://www.crunchbase.com/company/powerset) 的演示，但据报道，许多看过的人对此印象深刻，[说](https://web.archive.org/web/20220810220032/http://venturebeat.com/2006/10/04/powerset-that-secretive-little-search-engine-company/#more-2087)他们再也不会使用谷歌了。

Powerset 想让人们在搜索时使用自然语言，包括一些今天搜索引擎忽略的词(创始人 Barney Pell 所说的“[停用词](https://web.archive.org/web/20220810220032/http://www.barneypell.com/archives/2006/10/powerset_and_na.html)”)。像大多数人一样，我学会了在搜索引擎上使用不同的语言。我使用重要的关键词，其他的都省略了。其中一些搜索习惯甚至蔓延到了我的个人通信中，我发现即时消息和电子邮件的快速讨论经常看起来像是疯狂的、毫无意义的词汇来回传递。十年前，我不知道这些讨论意味着什么。今天，他们能有效地与我已经熟悉的人进行快节奏的商务对话。

因此，尽管我认为今天的搜索远非完美，但我并不认为解决方案是创造一种更好的方式让计算机理解我们的意思。坦率地说，我认为我们会继续以计算机能够理解的方式更好地交流，其他领域也会有进步(深度网络、更好的算法、富媒体搜索等)。).

但是 [Pell](https://web.archive.org/web/20220810220032/http://www.barneypell.com/archives/2006/10/powerset_and_na.html) 提出了一个令人信服的论点，即自然语言搜索对于传达意义和意图非常重要。他用示例搜索来说明他的观点——“给孩子的书”、“由孩子写的书”和“关于孩子的书”都相当于今天搜索引擎的“书的孩子”。他的核心论点是，如果不使用自然语言，我们可能没有办法恰当地表达“儿童书籍”这一查询。

Powerset 正在寻找大笔资金来推出他们的新引擎。风险投资家总是谣言的最佳来源(向他们询问信息的最佳时间是在他们放弃一笔交易，或者被另一家公司收购之后)。如果他们与公司没有经济利益关系，就会信口开河。关于 Powerset，[的传言](https://web.archive.org/web/20220810220032/http://venturebeat.com/2006/10/02/bold-start-up-powerset-about-to-raise-10m-to-take-on-google/)称，该公司正试图在 2000 万美元的投资前估值基础上筹集 1000 万美元。这是一大笔钱，但如果 Powerset 推出谷歌，没有人会在意。
