# 惠普 LED 笔记本即将上市

超越苹果，也超越你，戴尔。在你们这些笨蛋之前，惠普将推出一系列带 led 的笔记本电脑。AUP 和 CMO 将从第三季度开始向惠普发运笔记本电脑的 12.1 英寸面板。有消息称，戴尔计划在惠普之后的某个时间开始生产和销售 LED 笔记本电脑。那么苹果什么时候会加入呢？
