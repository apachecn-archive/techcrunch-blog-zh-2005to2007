# Platial 将在 Where 2.0 推出多媒体混搭

[![](img/3c8028b5a6885608a951155f8a367d25.png)](https://web.archive.org/web/20220928185122/http://www.platial.com/) 多媒体和用户生成的在线地图服务 [Platial](https://web.archive.org/web/20220928185122/http://platial.com/) 将在明天开始的 [Where 2.0 发布会](https://web.archive.org/web/20220928185122/http://www.google.com/url?q=http://conferences.oreillynet.com/where2006/&revid=1218653146&sa=X&oi=revisions_inline&ct=result&cd=1)上于附近宣布一个名为[的新现场服务。这项新服务将把新闻、照片、视频、事件和地点的 RSS 源与谷歌地图和谷歌地球结合起来。内容将从路透社，Flickr，Eventful，YouTube 和其他用户的地图映射。您选择的位置的更新也将可供订阅。](https://web.archive.org/web/20220928185122/http://today.platial.com/)

Platial 是众多允许用户在谷歌 API 基础上轻松创建自己地图的服务之一，但它以其迅速的采用和资金支持而闻名。该公司表示，Platial 在推出后的前 5 个月里，在 300 个城市托管了 20 万张地图。例如，请看[公共生物燃料站](https://web.archive.org/web/20220928185122/http://www.platial.com/system/map/1711)(令人惊讶的是，全是美国东海岸)或者这个家伙的[无望的浪漫地图](https://web.archive.org/web/20220928185122/http://www.platial.com/paiges/map/1535)，记录了一个世界价值的个人心碎，显然是因为一个女人。

该公司的支持者包括凯鹏华盈、奥米迪亚网络、蒂姆·奥雷利和克莱·舍基。

Platial 的最新服务对各种来源的 RSS 提要进行了复杂而合理的使用。利用如此多的联合多媒体资源可以使 Platial 在一个包括竞争对手 [Wayfaring](https://web.archive.org/web/20220928185122/http://wayfaring.com/) 、 [CommunityWalk](https://web.archive.org/web/20220928185122/http://communitywalk.com/) 、、 [Frappr](https://web.archive.org/web/20220928185122/http://frappr.com/) 等的领域中处于有利地位。这些服务中的每一个都有独特的功能集，但来自异地的多媒体和新闻的显示尤其引人注目。

如果用户未来能够在自己创建的地图上叠加选定的提要，或者对这一新服务与基本的 Platial 服务进行一些其他创造性的集成，那就更是如此了。“今日邻近”服务似乎主要是作为地图创建之外的辅助服务；它将用于监控 Platial 用户制作的基于位置的多媒体、事件和地图，这些用户与您关注相同的位置。
