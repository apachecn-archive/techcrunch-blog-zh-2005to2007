# T-Mobile 推出新款三星手机

![](img/87863458f6c6d0a8ee2d2f5d5a37770f.png)

我们[最近报道了](https://web.archive.org/web/20210305104639/http://crunchgear.com/2006/09/20/t-mo-drops-two-new-sexy-sexy-samsungs/)T-Mobile 将发布两款新的三星手机。今天，它已经成为官方——T-Mobile 和三星推出了 t629 滑盖和 t619 翻盖。

两款手机都有 130 万像素的摄像头和 EDGE 连接。t629 有一个 microSD 插槽，两年合约的价格为 249.99 美元或 149.99 美元，而 t619 的合约价格为 199.99 美元或 49.99 美元。

今天上午还宣布了一个较小的模型，t509s，又名李子。这款水果味糖果棒厚度为 0.39 英寸，配有 VGA 摄像头和 EDGE。它的售价为 199.99 美元，两年合约的售价为 49.99 美元。
