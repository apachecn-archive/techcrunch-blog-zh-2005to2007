**有什么新消息？**

Simpy 是一个社交书签服务，可以比作 [Shadows](https://web.archive.org/web/20221130214139/http://www.beta.techcrunch.com/?p=92) 、 [del.icio.us](https://web.archive.org/web/20221130214139/http://www.beta.techcrunch.com/?cat=19) 、 [Furl](https://web.archive.org/web/20221130214139/http://www.beta.techcrunch.com/?p=30) 、 [Yahoo My Web 2.0](https://web.archive.org/web/20221130214139/http://www.beta.techcrunch.com/?p=56) [和](https://web.archive.org/web/20221130214139/http://www.consultantcommons.org/node/239) [others](https://web.archive.org/web/20221130214139/http://www.dlib.org/dlib/april05/hammond/04hammond.html) 。

Simpy 今天发布了新功能，包括

–书签网站的全文搜索
–书签的标签云
–移除、重命名、拆分和合并标签

![](img/7a647e9b35a100f1f59dd2031a46171f.png)

来自奥蒂斯的简单博客:

发布日志中有详细信息，但简而言之，我很高兴地宣布新的 Simpy 版本已经发布了！

最显著的特性是搜索和标记。

全文搜索又回来了！不仅可以全文搜索自己的书签，还可以全文搜索任何其他 Simpy 用户的公共书签。在我看来，这是大而有用的。通过标签浏览很好，但也有局限性。全文搜索可以做得更多，因为 Simpy 支持字段搜索，所以您总是可以使用类似 tags:foo 的查询来模拟标记搜索(更多信息在 FAQ 中)。

您现在可以看到您的标签群(只需使用 Simpy 页面顶部导航栏中的“标签”链接)。标签云是您的标签及其使用分布的更直观的表示。伴随标签云而来的是 4 个新的标签功能:移除、重命名、拆分和合并。这四个功能可以让你随心所欲地修改你的标签。如果他们还不够，我洗耳恭听。"

发行说明这里是[这里是](https://web.archive.org/web/20221130214139/http://www.simpy.com/simpy/ReleaseLog.do)。

这些是有用的附加功能。出于某种原因，我真的很喜欢看到标签云格式的数据——如果 RSS 聚合器以这种格式显示新的未读内容，我会很高兴——这将使我更容易快速找到我想要阅读的内容。

我还必须说，我被[阴影](https://web.archive.org/web/20221130214139/http://www.shadows.com/)“阴影页面”的想法震惊了——简单却非常非常有用，我希望看到 simpy 和其他人考虑这个想法。
