# 三星 i570 震撼塞班操作系统

![samsung-sgh-i570-live.jpg](img/1d26bdf4ff7ca17838e74ec066418af7.png)

最新出现在网络上的三星滑盖会让任何一个 WinMo 的反对者高兴的不可开交。我们真的知道蛤蜊快乐吗？无论如何，SGH-i570 是一个完整的 QWERTY 滑块运行 Symbian 9.1。它的一些奇妙的特征让我想知道它是否会在美国出现。

视频通话在这里并不存在，所以前面的 VGA 摄像头不一定会被使用，但其臀部的 200 万像素将会被使用。160MB 的板载存储空间只够存放几首歌曲，感谢 gawds 的 microSD 插槽。Wi-Fi、HSDPA、EDGE 和蓝牙也让这个滑块成为一个诱人的机器。让我们希望它能到达我们的海岸，因为三星设备上的塞班操作系统是坏人中的坏人(最好中的最好)。
