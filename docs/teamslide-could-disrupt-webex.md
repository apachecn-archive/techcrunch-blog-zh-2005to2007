# Teamslide 可能会扰乱 WebEx——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/04/24/teamslide-could-disrupt-webex/>

 [](https://web.archive.org/web/20210227072909/http://www.teamslide.com/) 总部位于德国慕尼黑的 [Teamslide](https://web.archive.org/web/20210227072909/http://www.teamslide.com/) 是 WebEx、Placeware 和 GotToMeeting 等服务的一个新的、成本非常低的替代品，允许人们在线共享应用程序(如 power point)。

WebEx、Placeware、GoToMeeting 等的典型应用。是销售电话。主持人或演示者使用 WebEx 软件建立通话。与会者在预定时间点击电子邮件中的链接，每个人都加入到虚拟会议中。主机能够共享应用程序(powerpoint、浏览器、pdf、Word 等)。)并进行推介。

所有现有服务的一个缺点是价格以及对主持人和与会者的浏览器插件的需求。WebEx 和 Placeware 的最低定价都是 375 美元/月，GoToMeeting 的最低定价是 50 美元/月。任何使用过这些服务的人都知道，除了最低价格之外，他们还要收取使用费。

刚刚推出的 Teamslide 拥有其他服务的许多功能，更易于使用(主持人或与会者无需插件或其他软件),而且价格低廉。缺点是，至少在其当前的实现中，它需要一个[服务器端软件安装](https://web.archive.org/web/20210227072909/http://www.teamslide.com/pages/check/check.php)。

这项服务对于主持人和与会者来说只需要一个浏览器，有一个 Ajax 界面，可以用来分享演示幻灯片(以一种专有的团队幻灯片格式)。他们已经创建了一个插件，用于从 [powerpoint](https://web.archive.org/web/20210227072909/http://www.teamslide.com/pages/take-a-tour/8-powerpointaddin.php) 轻松转换。在他们的[演示](https://web.archive.org/web/20210227072909/http://www.teamslide.com/demo/demo9/admin/login.php?todo=load&username=demo&password=demo)中查看服务的更多细节。

根据演示(上面的链接)，这是一个服务的现场实现，Teamslide 比 WebEx 更容易使用(我是许多失败的 WebEx 演示的受害者)。Teamslide 的一个明显的局限性是，它只共享演示幻灯片，而 WebEx 在共享浏览器和其他应用程序时确实大放异彩。但是到目前为止，除了面对面的会议之外，WebEx 唯一真正的替代方案是在演示之前通过电子邮件发送文件。像 Teamslide 这样的服务，以及像 [Thumbstacks](https://web.archive.org/web/20210227072909/http://www.beta.techcrunch.com/2006/03/09/thumbstacks-ajaxflash-web-powerpoint/) 这样的 powerpoint 在线版本，正在开始改变这种情况。
