# 呜哇！辛普森手机游戏上路了

距离《辛普森一家》电影《T2》上映还有一周多一点时间，剧本的一些细节尚未泄露。但最终得到证实的是，EA Mobile 的一款游戏正在路上。《辛普森一家:崩溃的几分钟》现在已经可以在英国的 Jamster 和 T-Mobile 上看到了，但是有传言说美国版本正在路上。

在游戏中，你扮演荷马，并可以与各种受欢迎的角色互动，包括巴特，伯恩斯先生，克鲁斯蒂，楼警官和阿普。情节包括荷马必须阻止核熔毁！哦！
