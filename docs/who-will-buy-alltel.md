# 谁会收购 Alltel？

威瑞森真的会吞并 Alltel 并成为全国最大的移动电话提供商吗？两年前，华尔街日报报道了一次可能的合并，但那显然从未发生过。我们知道 T-Mobile 和 Cingular 运行在 GSM 网络上，因此排除了它们，Sprint 正忙于整合 Nextel 的网络。这使得威瑞森成为唯一可行的候选人，但 Alltel 报道的 300 亿美元的价格可能会动摇威瑞森对整个 enchilada 的兴趣。美国技术研究公司的股票分析师林宇民说，这很有可能发生，因为他们都使用 CDMA 网络，这将使威瑞森成为该国最大的供应商。斯坦福集团股票研究分析师迈克尔·纳尔逊不这么认为。

> “威瑞森根本不是一个心甘情愿的买家，”他说。“Sprint 目前还没有能力进行如此大规模的变动，所以这真的取决于私募股权。”

随着 FCC 今年即将拍卖 700MHz 频谱，Nelson 说:

> "他们可以用少得多的 300 亿美元建立自己的网络。"

我不是一个喜欢打赌的人，但我不认为这会发生。两年前没有发生，那么现在有什么不同呢？
