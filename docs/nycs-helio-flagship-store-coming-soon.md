# 纽约 Helio 旗舰店即将开业

[Helio 的](https://web.archive.org/web/20210417131450/http://crunchgear.com/?s=Helio)最新旗舰店出现在纽约超级时尚的 Soho 区。在过去的几个月里，我一直在观察这个店面的进展，直到大约三周前，这个“棚子”才出现。在相当长的一段时间里，它只是一个普通的店面，只是从里面贴了一个 Helio 的标签。我从裂缝中探出头来，试图了解商店的进展情况，但我运气不太好。我看到很多准备工作正在进行，但没有什么能与圣莫尼卡的时尚旗舰店相提并论。尽管如此，我还是会让你们知道这家店的进展，因为它就在我家附近的街上，我每天都会路过这个怪物。如果你在这个地区，那就自己去看看，这家商店位于休斯顿北部的百老汇，在百思买和 Urban Outfitters 之间。
