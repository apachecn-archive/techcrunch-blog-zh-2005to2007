# 索尼将涉足手机游戏

索尼似乎正在寻找一个新的游戏领域。虽然该公司凭借 PlayStation 3 在客厅占据主导地位，但 PlayStation 便携手持系统一直存在问题。现在，索尼电脑娱乐欧洲公司正寻求在便携领域采取不同的举措，进军手机游戏。该公司已聘请前 THQ 无线高管巴里尔·布鲁根(Balir Blugan)负责移动业务开发。据 PocketGamer 报道，像《逃离》和《原始》这样的游戏可能会被移植到手机上，索尼可能会开发自己的游戏或者与其他开发商合作。索尼爱立信 PlayStation 手机的传言会是真的吗？敬请期待！
