# 简介:PodTech

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/06/21/profile-podtech/>

**公司:** [波德泰](https://web.archive.org/web/20201201083140/http://www.podtech.net/)

![](img/1133a975827c14364812bb92bc6587aa.png)

**成立:**2005 年 5 月 23 日([链接](https://web.archive.org/web/20201201083140/http://furrier.typepad.com/john_furrier/2005/05/podtechnet_prev.html))

**什么事？**

[PodTech](https://web.archive.org/web/20201201083140/https://crunchbase.com/organization/podtech) 是一组精彩的播客访谈，更新非常频繁，以顶级科技名人为特色(最近的播客包括对[罗伯特·斯考伯](https://web.archive.org/web/20201201083140/http://www.podtech.net/?p=55)、[戴夫·霍尼克](https://web.archive.org/web/20201201083140/http://www.podtech.net/?p=46)、[约翰·马可夫](https://web.archive.org/web/20201201083140/http://www.podtech.net/?p=41)、[克里斯·安德森](https://web.archive.org/web/20201201083140/http://www.podtech.net/?p=38)和其他人的访谈)。

引起我们注意的是约翰正在采访的人的绝对素质(更不用说素质了)，以及他的新企业起飞的速度。仅仅两周之后，PodTech 就有超过 66，000 名访客，超过 21，000 次音频播放和 9，000 次 MP3 下载。对于一个除了口碑之外没有营销的网站来说，这已经不错了。

用他们自己的话说，*“我正在建立我的播客网站，名为 PodTech.net——这是一个播客信息系列，致力于成为一个时移广播节目，在新的时移媒体辛迪加世界中介绍硅谷。有人说硅谷正在失去影响力。我认为硅谷正在发生变化。我的节目旨在通过播客和博客产生关于硅谷的信息，并将其推向世界，然后将一个社区循环回硅谷。是的，硅谷正在改变……但它的影响力正在增加，因为它现在是虚拟的和联合的！”*

这是我们的十大播客之一，我们从约翰那里了解到他列出的即将播出的节目会让我们大吃一惊。敬请期待…
