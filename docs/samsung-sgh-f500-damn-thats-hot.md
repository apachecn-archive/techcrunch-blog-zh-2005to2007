# 三星 SGH-F500:该死的是热的

昨天在香港举行的国际电信联盟 2006 年电信展上，三星展示了 SGH F500，即 Ultra Video，这是第一款原生支持 DivX 的手机。是的，这是一个奇怪的电话。它不太可能很快或永远出现在美国，我们通常不报道这种事情，但这是一个如此疯狂的设计，我想确保你们都好好看看它。T2 的韩国人在小玩意方面远远领先于美国，这让我有点羡慕。

这款手机实际上有两个面:一面是带键盘和小屏幕的手机，另一面是大屏幕和多媒体键。它在 2,100MHz 频段上支持 UMTS，在 900、1800 和 1,900MHz 频段上支持 GSM/EDGE，并支持大量音频和视频文件类型，包括前面提到的 DivX。
