# 维珍移动将失去移动电视产品

正如我们上周报道的那样，维珍移动正在终止 Movio，这是一个表现不佳的移动电视分支，是通过与英国电信合作推出的，采用 DAB 电视标准。现在，随着合作关系的解除，维珍移动似乎可能会停止销售和生产龙虾 700TV 手机，这是该运营商的旗舰移动电视设备。用户仍然可以收听 DAB 广播，但这似乎不足以吸引设计笨拙的手机。

维珍移动没有排除在未来推出使用 DVB-H 标准的新移动电视服务。希望他们下次能设计出更好的手机。
