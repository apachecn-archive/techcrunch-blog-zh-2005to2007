# Wordie 就像没有照片的 Flickr

[![wordie_logo.jpg](img/da44b579fad1cabcf6df569eb0a51286.png)](https://web.archive.org/web/20220927230827/http://wordie.org/) 在本周早些时候的一次晚宴上，Flickr 联合创始人斯图尔特·巴特菲尔德无意中提到了一个名为 [Wordie](https://web.archive.org/web/20220927230827/http://wordie.org/) 的网站，其口号是“像 Flickr，但没有照片”

巴特菲尔德被这个网站逗乐了。巧合的是，我们前一天刚刚和 Wordie 创始人约翰·麦格拉思谈过话，我们也是。沃尔蒂就像一包戏法。这是用户记录他们喜欢并想记住的单词的一种方式。

“这是一种组合玩具，字典，辞典和社交网站…但主要是一个玩具，”麦格拉思说。

如果你访问 Wordie，你会发现这个网站更有趣。你不必建立你的词典，因为那里没有定义，只有定义的链接。但是你可以记录下你听到或读到的单词，并且知道你会想再次使用它们，比如[穴居人](https://web.archive.org/web/20220927230827/http://wordie.org/words/word?id=troglodyte)或[水粉](https://web.archive.org/web/20220927230827/http://wordie.org/words/gouache)。当您查找新单词时，您会看到各种定义的链接，以及标记该单词以供记忆的其他用户的列表。这很傻，但不知何故像 Flickr 一样让人上瘾。

当麦格拉思决定为他的朋友们写 Wordie 的时候，他刚刚写完收藏者的社交网站 Squirl。它是用 Ruby on Rails 编写的。

“我收到很多人的电子邮件，说他们认为这是一个荒谬的想法，然后花了两个小时输入单词并检查其他人的话，”麦格拉思说。"它有一种简单化的魅力。"

巴特菲尔德在一次关于我们如何定义社交媒体和 Web 2.0 应用的谈话中提到了 Wordie。我不确定 Wordie 是不是也是，但是对于我们这些 Web 2.0 行业的人来说，它是一个很好很有趣的方式来发现/使用我们的幽默感。
