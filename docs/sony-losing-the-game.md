# 索尼输掉了游戏——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/14/sony-losing-the-game/>

《洛杉矶时报》将索尼的游戏部门比作纽约扬基队。“两人都有高薪人才和伟大的进攻，但都在努力赢得今年的比赛，”洛杉矶时报特约撰稿人亚历克斯·法姆和布鲁斯·华莱士今天的文章说。分析师们声称，虽然任天堂的 Wii 和微软的 Xbox 360 是为了立即影响市场而设计的，但索尼的 PlayStation 3 走在了市场的前面，因此要到 2008 年才会看到销售的起飞。

当然，到那时对 PlayStation 品牌来说可能就太晚了。随着包括《下一个侠盗猎车手》在内的游戏在 Xbox 360 上的出现，这可能会伤害索尼，索尼指望这些独家游戏来帮助增加对 PlayStation 2 的需求。《洛杉矶时报》的报道称，对索尼来说，这还不是最糟糕的，该报道称，“日本国际游戏开发商协会的 Kyoshi Shin 表示，那里的许多创作者正在将他们的注意力转移到 Wii 上。”
