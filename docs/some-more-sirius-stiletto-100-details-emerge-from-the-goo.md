# 更多的小天狼星高跟鞋 100 的细节出现了

你可能渴望得到一些可靠的发布日期和定价信息。好吧，我们只能给你后者，因为它在克兰奇菲尔德的预购价是 349.99 美元。

不过，我们可以给你一些最终的规格:2GB 的内部存储，调频发射机，MP3 和 WMA 播放，彩色屏幕，卫星和 WiFi Sirius 交付。它的尺寸为 2 英寸 x 4.5 英寸 x 0.68 英寸(采用超薄电池)，2 英寸 x 4.5 英寸 x 1 英寸(采用普通电池)。

它在 2GB 的存储器上存储了 100 小时的 Sirius 编程。使用超薄电池，它只能持续 2 小时的 Sirius 接收(哎哟)，4 小时的 WiFi Sirius 模式(体面)，10 小时的 MP3/WMA 播放模式。有了扩展电池，一切分别在 2 小时、4 小时和 20 小时翻倍。
