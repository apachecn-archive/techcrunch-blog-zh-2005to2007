# 有史以来最好的部件——Gapingvoid

 [](https://web.archive.org/web/20221205084549/http://www.gapingvoid.com/widget/) 如果你不知道休·麦克劳德和他在 [Gapingvoid](https://web.archive.org/web/20221205084549/http://www.gapingvoid.com/) 上的每日漫画，你就不在酷人群中。他的艺术是聪明的，有洞察力的，总是及时的。经常是[怪](https://web.archive.org/web/20221205084549/http://www.gapingvoid.com/Moveable_Type/archives/002943.html)。他也不介意偶尔惹毛别人。

休作为嘉宾加入了吉尔摩帮，参加了四月的播客。我们讨论了休如何让自己的作品获得更多关注，于是就有了博客小工具的想法。嗯，两个月后休的[推出了](https://web.archive.org/web/20221205084549/http://www.gapingvoid.com/Moveable_Type/archives/002978.html) it。想在你的博客或其他网站上展示他每天的漫画？[在此获取代码](https://web.archive.org/web/20221205084549/http://www.gapingvoid.com/widget/)。
