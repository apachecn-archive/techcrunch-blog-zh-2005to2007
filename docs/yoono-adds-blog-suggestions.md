# Yoono 添加博客建议

火狐和 IE 链接建议插件 Yoono ，最近在他们的产品中加入了博客和文章建议。新的建议已被添加到 Yoono 侧边栏的顶部，并显示与您当前正在查看的页面相关的故事。例如，访问 Techcrunch 会得到右图所示的结果。
![](img/17ca2d7097a9380144253a0a697c53cd.png)

Yoono 有点像 [Del.icio.us](https://web.archive.org/web/20220630100039/http://www.beta.techcrunch.com/tag/del.icio.us/) 和 [StumbleUpon](https://web.archive.org/web/20220630100039/http://www.beta.techcrunch.com/tag/StumbleUpon/) 的组合。Yoono 不会在互联网上东倒西歪，也不会在带标签的链接中搜索，而是根据你正在浏览的页面，在侧边栏或滚动条中给你一个建议列表。它不会保留你的冲浪历史。建议是基于对超过 140，000 个用户共享的书签和实时书签(RSS 提要)的分析而生成的。通过查看用户共享的书签以及它们是如何按文件夹组织的，Yoono 找到了相关链接。这些链接中的每一个都被赋予一个流行度分数，这个分数部分基于将该页面作为书签共享的用户数量。上个月，他们每天提供 600 多万条建议。

我发现 Yoono 的新功能是一个有用的补充，实际上，我已经被吸引到了关于比尔盖茨最近露面的博客文章的线索中。然而，这项服务仍然需要一些调整。每隔一段时间，我都会收到一些我看不懂的语言的建议，或者是来自博客的重复报道，其中包含多个重叠的内容。新的建议确实填补了产品中的一个空白，为每个网站提供了更频繁变化的建议。社区的书签变化不够频繁，无法为我经常访问的页面提供新的建议。Yoono 也一直在考虑另一个解决这个问题的想法，短期书签在给定的时间段后到期后会自动清除。

关于文章建议的更多信息，请查看 [SphereIt](https://web.archive.org/web/20220630100039/http://www.beta.techcrunch.com/2006/05/22/timecom-adds-sphere-it-links/) 。
