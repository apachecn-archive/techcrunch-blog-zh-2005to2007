# XMHD (Gmail 硬盘)–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/01/profile-xmhd-gmail-hard-drive/>

## 这是什么？

XMHD 是一项允许你将谷歌 Gmail 账户用作虚拟硬盘的服务。如果您没有 gmail 帐户(这需要邀请)，不要担心。他们会邀请你的。

![](img/3945744f2153884b03af9e155d8e5243.png)

XMHD 类似于我们昨天描述的 [Gmail Drive](https://web.archive.org/web/20201201062242/http://www.beta.techcrunch.com/?p=113) 。然而，Gmail Drive 是你的 windows 机器上的一个实际的虚拟驱动器(只有 Windows)，XMHD 在其网站上运行一切，因此是独立于平台的。你只需要一个网络浏览器和一个 gmail 账户就能使用它。

也没有账号注册或者下载。只需给它你的 gmail 证书，它就会为你登录 gmail 账户，并向你展示它自己的前端。

![](img/04f35a784bf9e60e6af90ec6016270fe.png)

用*自己的话说，“XmailHardDrive.com 是怎么工作的？XmailHardDrive.com 直接连接到你的 Gmail 帐户，然后把它变成一个便携式虚拟硬盘，你可以从世界任何地方访问。您只需输入您的 Gmail 用户名和密码，就像其他第三方软件一样，然后使用用户名和密码组合通过 Gmail 服务器的验证，并将您连接到您的个人帐户。因此允许用户能够将文件上传到他们的 Gmail 帐户。接下来，当用户上传文件时，该文件将作为电子邮件附件上传到他们的 Gmail 收件箱中，但是该文件经过加密和压缩，因此用户可以存储更多文件。然后，该文件被保存为。xmhd 文件，可以通过 XmailHardDrive.com 解压缩和重新下载。现在有了 FlingIt 功能，用户可以将刚刚上传的文件发送给有 Gmail 账户的朋友。目前最大文件上传大小为 20mb，但是随着我们获得更多的赞助商，我们将取消上传限制，您将能够上传更大的文件！”* [链接](https://web.archive.org/web/20201201062242/http://www.xmailharddrive.com/beta/about.xmhd)

您可以轻松地从硬盘(或其他连接的驱动器)添加文件。您一次最多可以上传 10 个文件。

文件被压缩并存储在一个专有的。xmhd 格式，但是可以在网站上双击并以其原始格式打开(并保存、编辑等)。).还有一个很酷的“发送”功能，可以通过电子邮件将文件发送给 gmail 账户中存储的任何 gmail 联系人。如果这个人没有 gmail 账户，他们就必须申请一个，然后用 xmhd 打开文件。所有这些对电子邮件收件人来说都有点麻烦——尽管 XMHD 表示他们的文件格式对于压缩目的是必要的(因此你可以在你的 gmail 帐户上存储更多数据)，但这也是一种很好的病毒式接触服务，因为你的文件收件人必须登录 XMHD 才能查看它们。当然，你也可以直接用电子邮件发送你的文件，这没什么大不了的，这是一个很好的功能。
