威瑞森即将推出KRZR

原文：https://web.archive.org/web/http://techcrunch.com/2006/09/22/verizons-krzr-drop-imminent/

你的RAZR就要(正式)烂了。过度生产，现在俗气的手机图标是如此无处不在，它现在有着与K-Fed演唱会门票一样多的“酷”。幸运的是，它即将被新的摩托罗拉KRZR(克拉泽)取代，这是RAZR(剃刀)的后续产品。看起来威瑞森无线将会是第一个将传闻已久的KRZR推向市场的阵营，而且它应该随时会退出。其他运营商(很可能是所有运营商)应该很快也会加入进来，但随着威瑞森的测试网站已经展示了基于Flash的交互式手册，他们看起来好像已经抢先了一步。

KRZR是非常RAZR-ish，有点薄的脸(我希望我是)。它的大部分功能与最新的RAZR相同，包括EVDO、130万像素的摄像头和带MicroSD存储的MP3播放器。KRZR在前面添加了一些非常像巧克力的触摸感应隐藏媒体按钮，并通过蓝牙将DUN带到大众面前。

如果你是当前超薄RAZR的粉丝，这款手机就是为你准备的。它在一个很小很小的手机里有一个不错的功能列表，应该会很受欢迎。一旦你得到了你的，你就可以公开嘲笑那些RAZR用户是2006年的 so 。
