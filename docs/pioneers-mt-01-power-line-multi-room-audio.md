# 先锋的 MT-01 电力线多房间音频

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/21/pioneers-mt-01-power-line-multi-room-audio/>

先锋的 MT-01 电力线音响系统通过您的电力线播放音乐，而不是通过 WiFi、蓝牙和神奇的无线仙女居住的 2.4GHz。这样，干扰更少，也更方便，因为你的设备无论如何都需要插上电源。

多房间音频设置在每个扬声器中也有运动检测器，因此当您走进房间时，声音会激活。我们不确定它是否会在一段时间后自动关闭，所以每隔几分钟就要挥动手臂来保持声音，这有点烦人。MT-01 还支持六个扬声器，并配有一个 2x25W 网络扬声器和一个小型 5W 网络扬声器。

输入是 USB A 和 B，以及三个音频输入。它还可以支持从一个房间到另一个房间的多个音频流，所以你可以在一个房间听你的 iPod，同时在另一个房间看电影。
