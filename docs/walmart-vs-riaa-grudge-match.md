# 沃尔玛与 RIAA:旗鼓相当的比赛

在任何事情上，我们并不经常站在沃尔玛一边，但不知何故，我们发现自己与他们联合起来对抗一个更大的威胁——RIAA，就像眼镜蛇和特种部队不得不联合起来对抗 Gamemaster 一样。

沃尔玛似乎希望 RIAA 再次降低 CD 的定价。此前，美国政府以零售抵押品敲诈为幌子，迫使 RIAA 就价格操纵达成和解(RIAA 的标签将拒绝向使用 CD 作为亏损领头羊的零售商提供营销材料，大多数大型便利店都是这样做的)。现在，沃尔玛告诉 RIAA 停止人为抬高 CDs 的成本，这样他们就不会在 CDs 上赔钱。

我们希望律师们进行一场笼斗，希望沃尔玛成功，因为这些价格会反映在其他零售商身上，包括那些你不会去地狱购物的零售商。
