# Viewscore 聚合小工具评论

[![](img/66e50216ef13004b5356153be887440d.png)](https://web.archive.org/web/20211028085120/http://www.viewscore.com/)[CrunchGear](https://web.archive.org/web/20211028085120/http://crunchgear.com/2006/08/30/viewscorecom-review-aggregator/)的约翰·比格斯写了一篇关于以色列一家名为 [Viewscore](https://web.archive.org/web/20211028085120/http://viewscore.com/) 的初创公司的文章，该公司收集了来自网络上的电子产品评论。他说这就像是小玩意的 metacritic.com。不过，这里最酷的技术是，该网站将使用不同等级的网站之间的数字评级标准化(十分制或五星制被转换为满分 100 分)，并使用语义分析来确定不使用数字评级的评论的数量。所有评论都是平均的，可以单独查看。你可以比较价格和评价评论的质量。还显示了每个产品的基本信息和许多小工具的产品比较页面。

CrunchGear 的编辑比格斯说，到目前为止，小工具的数据库相当不错。该公司告诉我，它打算将其技术扩展到小工具以外，但这是最好的起点，因为网上已经有这么多小工具的评论。

在相关的元评论网站消息中，多主题评论搜索引擎 [iNods](https://web.archive.org/web/20211028085120/http://inods.com/) (见 [Mike 的评论](https://web.archive.org/web/20211028085120/http://www.beta.techcrunch.com/2006/01/10/more-edge-reviews-inods/))今天发布了他们的 1.0 版本。

如今网上有如此多的可用内容，以至于对其进行高质量的聚合并提供一些有价值的附加价值是一项重要的服务。我认为这一领域还有很大的发展空间。
