# Playaway 一体化有声读物评论| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/03/playaway-all-in-one-audiobook-review/>

![](img/fb039fe79ce8a264750e1d1c5c4169e1.png)

无论数字音频播放器变得多么便宜，出售一次性使用的想法仍然会让大多数消费者矛盾地耸耸肩。毕竟，DAP 的伟大之处在于它的灵活性——你可以在任何时候装载你想要的东西。你不是专辑曲目订单的奴隶，加载新音乐非常容易。

那么，Playaway 销售一款预装了一本有声读物、没有其他可识别功能的锁定型 DAP，可能会想些什么呢？

![](img/37600084de040613d3bdc195b6bdf07a.png)
如今，针对技术恐惧者的电子产品市场似乎正在蓬勃发展。从商业角度来看，这是一个绝妙的主意——去掉关键功能，收取“简单性”的额外费用看看那些可怕的[吉特巴手机](https://web.archive.org/web/20160927144012/http://www.techcrunch.com/2007/05/29/jitterbug-because-cell-phones-scare-old-people/)。

这似乎是 Playaway 的目标市场——那些需要打电话或听有声读物，但害怕电源按钮和电线的人。

这个想法很简单:你打开盒子，你有一个相对苗条的 DAP，它的有用性只持续到包含有声书的时间(据我所知，没有黑客公开如何侵入这个东西并加载你自己的音乐，我也不明白为什么有人会特别想这样做。)你需要的一切都包括在内。我所说的一切是指一对手机，它们让达美航班上发放的免费手机看起来像是森海塞尔的手机。

![](img/a8436b0c4271585a789d5989fb7b10ff.png)

我们的测试单元装载了村上春树的《海边的卡夫卡》。音质相当不错。虽然我肯定有一些音响发烧友会嘲笑斯蒂芬·金的声音失去一些喉音的想法，但聪明人通常会以较低的音质加载有声读物，以节省空间(这些东西很大。)真正的问题是附送的耳机，很烂。起初，没有声音传来，我以为整个事情被终结了。后来，我扭了几下耳机插孔，意识到手机上的传感器需要放在正确的位置——这是你已经用了一年的廉价耳机的常见问题，但对于任何开箱即用的东西来说，这都是荒谬的。幸运的是，你可以把自己的插头插入插孔，我强烈建议你这样做。

令人沮丧的一点是:没有可辨别的方法让书离开玩家。我猜约翰·格里森姆现在不需要担心他的书会出现在 SoulSeek 上了。

![](img/c74681383d0b59795723ae3f17ed123a.png)

所包括的控制是绰绰有余的。你可以在章节之间跳跃，调节音量(感谢上帝！)，快进/快退，如果你不介意花栗鼠给你读你的书，它甚至有一个均衡器控制和提高播放速度的能力。谢天谢地，当你打开它时，它会自动从你停止的地方重新启动。我想，有点像数字书签。

然后还有浪费的问题。这些天来，人们都在谈论那些被填埋的小玩意，这种公然浪费的东西可能看起来很恶心。令人欣慰的是，它附有说明，说明如何邮寄回制造商进行回收，以换取下次购买时的可观折扣。当然，不包括这一点会受到道德上的谴责。

那么该不该买这个东西呢？我猜它可以成为一个很好的圣诞礼物，但是现在小型数字音频播放器非常便宜，所以它只能作为一个新奇的东西。特别有趣的是各种书籍的价格范围——有些书便宜到 30 美元，而有些书高达 90 美元。虽然这对一次性播放器来说似乎很多，但这实际上是基于 CD 的有声读物的价格，所以你不必为内容支付太多的溢价。所以，如果你在道德上反对从 P2P 上获取有声读物，或者你碰巧非常非常技术愚蠢(在这种情况下，我无法想象你现在正在阅读 CG)。)

它使用一节可更换的 AAA 电池。
