# 只有 244 份 Vista 在中国销售

[![chinavista.jpg](img/9bb8c97582977401d5096b769b12b216.png)](https://web.archive.org/web/20210228080759/https://beta.techcrunch.com/wp-content/uploads/2007/04/chinavista.jpg "chinavista.jpg")

微软在 Windows Vista 上市的头两周就在中国卖出了 244 份正版。(该操作系统在全球上市的第一个月就售出了约 2000 万份。)所以要么中国没人用微软的操作系统，要么他们在那里做了很多盗版。我猜是后者。
