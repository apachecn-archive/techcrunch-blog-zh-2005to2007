# Tail rank——长尾的工具

TailRank 刚刚为他们新的分析服务发布了一款 alpha 产品。

我的理解是针对博客内容长尾的个性化搜索/推荐/排名引擎。

凯文·伯顿([罗约](https://web.archive.org/web/20210619071207/http://www.beta.techcrunch.com/2005/06/17/profile-rojo/)的联合创始人)，是 TailRank 的创始人，并在他的[博客](https://web.archive.org/web/20210619071207/http://www.feedblog.org/2005/09/what_is_tailran.html)上写了关于这次发布的文章(如果你还没有订阅这个博客的话)。

> TailRank 是下一代博客排名系统，它考虑了从其他 Web 2.0 产品和服务中学到的许多经验。不用说我真的很兴奋！我热爱我的工作，现在这个领域有很多创新。
> 
> 我们仍处于早期 alpha 开发模式，仍在组建团队。一切都很顺利，我推荐的每个人看起来都很兴奋。
> 
> 我希望在几周内有一个公开版本，但请随意注册一个测试帐户，我们会尽快给你发送一些东西。很明显，在接下来的几周里，我会在我的博客上经常谈论 TailRank，所以请随意订阅。

凯文将在明天的 TechCrunch Meetup 上演示该产品。
