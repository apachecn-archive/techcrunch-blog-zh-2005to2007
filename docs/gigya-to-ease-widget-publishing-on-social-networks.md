# Gigya 简化社交网络上的小工具发布

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/04/09/gigya-to-ease-widget-publishing-on-social-networks/>

 [](https://web.archive.org/web/20160416114010/http://www.crunchbase.com/company/gigya) 把一个小工具放到一个网站上，不管是博客、MySpace 页面还是其他什么，都有点麻烦。用户通常需要复制一个嵌入代码，登录到他们的网站，然后粘贴到适当的位置。虽然这并没有被证明是一个不可逾越的障碍，但那些找到了让用户更容易在自己的网站上添加小工具的方法的小工具初创公司，与竞争对手相比，增长率明显更高。

Slide、RockYou 和 Photobucket 是这一领域的早期实验者。他们没有强迫用户进行剪切和粘贴，而是简单地要求用户输入他们的 MySpace(或其他社交网络)凭证，然后代表他们直接将小部件放到网站上。当他们在 2006 年开始这样做的时候，每个人都期望 MySpace 会因为安全原因切断他们的联系，但是这从来没有发生过。MySpace 允许这些公司以用户身份登录并发布小工具。Slide、RockYou 和 Photobucket 实现了爆炸式增长。选择不提供这一功能的竞争对手 FilmLoop 停滞不前，现在[在死池里](https://web.archive.org/web/20160416114010/http://www.techcrunch.com/2007/01/06/filmloop-dips-toes-into-the-deadpool/)。

但是提供这个功能有点麻烦。有许多大型社交网络需要处理，它们偶尔会更改自己的 API 或登录程序。发生这种情况时，该特征会中断，直到做出更改。因此，今天大多数小部件公司简单地坚持尝试和真正的“剪切和粘贴”方法来增加小部件。

以色列初创公司 Gigya T7 于 2006 年 11 月推出了一款工具，允许人们通过电子邮件向他人发送小工具，而不仅仅是将它们发布在网站上。他们说，他们最初的产品表现不错。现在，他们正在推出一款名为 [Wildfire](https://web.archive.org/web/20160416114010/http://www.gigya.com/wildfire) 的新产品，该产品将允许 widget 生产商将其 widget 直接嵌入到更大的社交网络中(目前支持 MySpace、脸书、Friendster、Hi5、Xanga、Blogger 和 Tagworld)。小部件创业公司有一个自己动手的选择，他们直接与一些较大的公司合作。

一旦 Wildfire 与合作伙伴 widget 站点一起实施，用户只需选择他们使用的社交网络，键入他们的凭据和他们希望它出现的站点区域，然后点击提交。这个小工具将被放在他们的 MySpace 或其他社交网络页面上。我在下面嵌入了一个非常高层次的服务概述视频。

Snapvine 和 T2 已经将 Wildfire 整合到他们的网站中。其他一些小工具初创公司将在下周推出 Wildfire。

Gigya 没有向合作伙伴收取服务费用，称他们计划向移动和其他可以开始创收的领域扩张。联合创始人和 CMO·鲁利·埃利泽罗夫表示，他们希望 Gigya 成为一个小工具平台，他们的前两个产品(Gigya 和 Wildfire)只是一个开始。

到目前为止，该公司没有花多少钱就做了很多事情。2006 年 11 月，他们从 Benchmark Capital 和第一轮融资中仅筹集了 65 万美元。尽管这轮融资规模很小，但这些都是一级投资者，他们必须看到一个良好的长期商业计划。
 ***更新:**据[广泛报道](https://web.archive.org/web/20160416114010/http://www.thealarmclock.com/mt/archives/2007/02/israels_gigya_r.html)该公司实际融资 400 万美元。这是不正确的。在去年 11 月的一轮融资中，他们迄今只筹集了 65 万美元。*
