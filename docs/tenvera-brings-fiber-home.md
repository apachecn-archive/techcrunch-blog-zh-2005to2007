# Tenvera 将光纤带回家

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/08/31/tenvera-brings-fiber-home/>

![](img/6bef14c3146da069104d6477745ae634.png)

总部位于田纳西州的 Tenvera 公司开发了一种相对廉价的方式将纤维放入家中(FITH)。这样做的好处是显而易见的，随着光纤入户越来越普遍，你的老式室内铜线的劣质性将比以往更加明显。Tenvera 将其新的 FITH 产品称为“系统 5 ”,据估计，它只占一个建筑总造价的 2%到 3%。他们将于九月在 CEDIA 展示第五系统。

[感谢田纳西州的 Tenvera](https://web.archive.org/web/20140606215613/http://www.engadget.com/2006/08/31/fiber-in-the-home-thanks-to-tennessees-tenvera/)
