# 科学家发现如何连接量子计算机

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/07/scientists-figure-out-how-to-wire-quantum-computers/>

[![quantum1.jpg](img/190e1154a77a5f21e485321fce03eb15.png)](https://web.archive.org/web/20180127065253/http://old.crunchgear.com/wp-content/uploads/quantum1.jpg "quantum1.jpg")

D-Wave 的[猎户座](https://web.archive.org/web/20180127065253/http://crunchgear.com/2007/04/09/scientists-call-out-d-wave-systems-quantum-computer/)可能是量子计算领域朝着正确方向迈出的一步，但它的许多批评者仍在等待更可信的东西出现。RIKEN 是一个日本研究小组，他们现在能够“可控地耦合量子位”。虽然不是第一个完成这项工作的人(加州大学伯克利分校的一个团队去年就完成了)，但日本的发现证实了量子位耦合是真实的。*连线*称量子位耦合“类似于电路板上晶体管的连线。当量子位耦合在一起时，它们会相互影响——就像经典逻辑门一样。”

所有这些只是说，科学家们已经找到了如何“连接”量子计算机的方法。这张图片显示了连接在一起的量子位。很显然。
