# 基于 Palm OS 的 Treos 终于获得了对 Exchange  的直接推送支持

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/25/palm-os-based-treos-finally-get-direct-push-support-for-exchange/>

如果你们中的任何一个商业用户因为 Palm 缺乏推送电子邮件支持而对基于 Palm OS 的智能手机(read: Treo)与黑莓手机持观望态度，那么他们最终会做出决定。Palm 今天宣布对流行的 Treo 的 700p 和 680 型号进行升级，使其能够与 Microsoft Exchange 进行电子邮件同步。

该功能的缺乏导致许多人远离 Palm OS 阵营，Palm 希望从周一开始在其网站上免费进行的软件升级将改变这一点。然而，不清楚的是基于 Palm OS 的 Treos 的未来会发生什么，因为即将到来的软件化身令人困惑地被命名为 Garnet OS(T2)，并且在不久的将来也没有可预见的 Garnet Treos。

该更新将作为 OTA 下载提供给 700p 和 680 用户，我们 650 死忠再次被冷落。
