# 简单的 Web 2.0 流量趋势跟踪器

[![](img/e9374f421746e9e28946571a86fd5da4.png)](https://web.archive.org/web/20220929142849/http://movers20.esnips.com/TableStatAction.ns?reportId=100)[eSnips](https://web.archive.org/web/20220929142849/http://www.beta.techcrunch.com/2006/10/31/esnips-secures-2m-for-media-sharing-site/)的人们已经创建了一个[简单的网站](https://web.archive.org/web/20220929142849/http://movers20.esnips.com/TableStatAction.ns?reportId=100)来追踪“Web 2.0”的流量趋势。该网站使用 Alexa 数据。我很乐意看到有人创建一个类似的网站，通过使用各种[其他衡量标准](https://web.archive.org/web/20220929142849/http://www.beta.techcrunch.com/2006/08/19/check-for-bookmarks-with-socialmeter/)，例如 Technorati 中的反向链接数和 Technorati 索引的博客中的提及数，来对这些 Web 2.0 网站(或者更确切地说，[这些网站](https://web.archive.org/web/20220929142849/http://www.beta.techcrunch.com/2007/02/05/web-20-directory-go2web20-upgraded))进行排名。
