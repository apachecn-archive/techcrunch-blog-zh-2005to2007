# 雅虎！首次亮相 oneSearch 移动门户| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/20/yahoo-debuts-onesearch-mobile-portal/>

雅虎！正在积极开发新的移动门户网站，名为 oneSearch，它显示了这一点。作为门户网站，雅虎！知道内容是王道，获取这些内容是移动 2.0 的全部意义。进入 oneSearch，一个新的页面(在你的手机上的 m.yahoo.com ),它聚集了大量的雅虎的功能放到一个易于使用的页面上，包括 Y！信使，Y！电影，甚至是便携版的 flickr。

更重要的是，这项服务真的很快，不一定是因为后端的东西很强大，而是因为你正在寻找的内容以一种相当直观的格式组合在一起。在 EDGE 连接上寻找 300 的门票不到 20 秒，非常容易使用。

这是我的雅虎的一部分！服务。虽然你可以不用登录就能使用它，但这样做对你自己没有好处，因为系统可以利用你的偏好来加速找到你要找的信息。而且是免费的。
