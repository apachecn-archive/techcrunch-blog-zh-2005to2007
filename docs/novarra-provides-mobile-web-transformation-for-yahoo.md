# Novarra 为雅虎提供移动网络转型！

![novarra1.jpg](img/14aae91a416d49ab28ce37695aada929.png)今天，下一代移动互联网服务提供商 Novarra 宣布，Novarra 的 Vision server 平台将为雅虎提供网络转换服务一站式搜索服务。来自 oneSearch 的链接将被转换为使用 Vision server 技术的手机链接，该技术基于开放标准。

这项服务已经在亚洲启动并运行，并将于八月中旬在美国推出。
