# 面部追踪手机即将问世

FotoNation 已经为手机创造了一个面部跟踪系统，它可以让你的杯子保持在相框里，而不会引起任何大惊小怪。它可以与 Symbian Series 60 和 Windows Mobile 5.0 一起工作，这意味着该软件适合大多数较小的平台。它有助于对焦、曝光和色彩平衡，还可以自动裁剪场景中的人脸。它还没有上市，但是现在已经可以申请许可了。
