# 网飞提供电脑视频流

另一个品牌站出来，开始提供人们真正想要的东西。从今天开始，网飞将开始向订户免费提供流媒体电影。

> 周二，黑斯廷斯先生将开始回答这个问题。网飞正在推出一项服务，将电影和电视节目直接传送到用户的电脑上，不是下载，而是流媒体视频，不保存在电脑内存中。这项服务对网飞的用户是免费的，旨在让该公司在互联网电影发行的雏形世界中获得一个立足点。

目前的目录包括大约 1000 部电影，并且只能在 Windows 下工作。订购 18 美元的用户可以一个月看 18 个小时的电影，同时观看标准的三张 DVD。有趣的一点是:

> “(网飞首席执行官)黑斯廷斯先生说，“市场是微观的。”。“在很长一段时间内，DVD 将是一个非常大的市场。”
