# Vista:极限科技回答，升级还是全新安装？

所以你已经做到了。你已经咨询了你的配偶、律师、牧师和治疗师，你决定放弃 XP，转而使用 Windows Vista。你甚至选择了你想要的口味。但是当把一个 Windows 操作系统升级到另一个时，一个令人困扰的问题总是:全新安装还是升级当前安装？

在过去，全新安装总会带来更稳定、更快的系统，减少遗留问题。然而，全新安装也意味着重新下载和配置所有你喜欢的应用程序，如果它们能在新的操作系统上运行的话。这不是一个容易做出的选择，但幸运的是，Extreme Tech 的人已经做了一些功课，并在 Vista 升级或 Vista 全新安装之间进行了激烈的竞争，结果令人惊讶。

事实证明，作为 XP 升级版运行的 Vista 与作为全新安装版运行的 Vista 在基准测试中得分几乎相同。此外，Windows 升级中常见的系统膨胀和孤立文件几乎不存在。这意味着从 XP 升级到 Vista 实际上比从旧操作系统升级到 XP 要容易得多。如果你真的要尝试 Vista，请放心，升级过程应该会很顺利，这意味着更少的麻烦。你也可以更快地试用你庞大的新操作系统。
