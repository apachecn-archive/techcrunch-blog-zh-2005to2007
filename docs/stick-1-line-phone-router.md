# Stick 单线电话路由器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/17/stick-1-line-phone-router/>

因此，你有一部电话、一部传真机、一台 TiVo 录像机和一台运行 AOL 拨号上网的旧 Windows 95 机，它们都连在墙上，共用一根电话线。是不是点了 4 行就掏鼻子了？当然不是，你得到的是电话路由器，它“每次都会筛选并自动将所有语音、传真和调制解调器呼叫路由到正确的设备。”

我们不知道这种魔力是如何工作的，但如果它很方便，可以为您省钱，144.99 美元的价格标签是非常值得的。
