看看这个运行 Linux 的 AMD 机器的屏幕截图。我们看到了什么？摇摆不定、透明的窗口、火爆的 UI 效果、令人惊叹的 3D 任务切换器，以及足以让 [Vista Ultimate](https://web.archive.org/web/20201205124956/http://crunchgear.com/2007/01/30/a-copy-of-windows-vista-ultimate-and-his-boy/) 看起来像[午夜指挥官](https://web.archive.org/web/20201205124956/http://www.ibiblio.org/mc/)的视觉盛宴。

我并不是说我们都出去买一份 Ubuntu 的拷贝，但也许这就是我要说的。这台机器在运行什么？一个较旧的 GeForce 卡，一个相当快的处理器，和 Ubuntu Edgy Eft 与 [Beryl 0.1.3 窗口管理器](https://web.archive.org/web/20201205124956/http://www.beryl-project.org/)。
