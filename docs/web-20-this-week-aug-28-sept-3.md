## 2.SEW 评论 RSS 搜索引擎

SEW 的克里斯·谢尔曼写了一篇关于 RSS 搜索引擎的综合评论:

> 许多人错误地将 RSS 搜索称为“博客搜索”虽然许多博客确实提供 RSS 提要(自动创建提要是大多数博客软件的一个功能)，但并不是所有的博客都有提要。此外，RSS 几乎可以用于任何基于网络的内容。RSS 基本上是一个相对简单的规范，它使用 XML 以标准的方式组织和格式化基于 web 的内容。

## 3.RSS 与电子邮件

弗雷德·威尔逊[写了](https://web.archive.org/web/20160202181901/http://avc.blogs.com/a_vc/2005/09/email_vs_rss_co.html)他的读者来自哪里，并揭示了一些关于电子邮件和 RSS 订阅相对受欢迎程度的非常有趣的数据。底线是只有大约三分之一的读者通过 RSS 访问他的内容。其他人直接登陆网站，或者阅读他的日常邮件。

> 8 月份的网页浏览量:~90，000
> 8 月份的网页访问量:~60，000
> 8 月份的网络独立访客:不知道，但我猜大约有 10，000
> 电子邮件订户:1055
> RSS 订户:~5500

后来:

> 事实是，在我的大约 17000 名听众中，只有三分之一的人使用 RSS。我的读者可能比普通读者更专业(例如，只有一半人使用 ie 浏览器，几乎 40%的人使用 Firefox)。

## 4.Ajax Office——不是“如果”,而是“何时”

Ajax 和其他在线办公类型的应用程序正在遍地开花。理查德·麦克马努斯不厌其烦地找到了许多，并在这里写下了它们。老实说，目前还没有好的全套解决方案。AjaxOffice 只是一个有明确意图的网页。 [Writely](https://web.archive.org/web/20160202181901/http://www.writely.com/) 有一个优秀的 ajax 文字处理器，但需要构建其余的。大家伙们还要多久才会进来？
