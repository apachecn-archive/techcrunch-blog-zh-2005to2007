# OPML 实验-2.0 版

我们的 [OPML 实验](https://web.archive.org/web/20230213033303/https://techcrunch.com/2005/09/29/opml-an-awesome-experiment/)和[戴夫·温纳](https://web.archive.org/web/20230213033303/http://archive.scripting.com/2005/09/29#When:7:36:29AM)(这里也听听戴夫的播客)已经[吸引了一群人的注意](https://web.archive.org/web/20230213033303/http://radio.weblogs.com/0001011/2005/09/29.html#a11295)，一些真正聪明的人正在开发利用博客目录的新应用。

这是我们目前掌握的情况:

1.  2005 年 9 月 29 日，我创建了 TechCrunch 帖子的 OPML 目录，你可以在 Scripting.com 的侧边栏看到。这很容易用 [OPML 编辑器](https://web.archive.org/web/20230213033303/http://www.opml.org/)完成，并且每天更新。
2.  那天晚上晚些时候，Niall Kennedy 创造了一个可移动的模板来自动完成这项工作。
3.  马特·莫楞威格为 WordPress 做了同样的事情。
4.  今天，我们从 [Kosso](https://web.archive.org/web/20230213033303/http://blogs.opml.org/kosso) 那里得知，他已经创建了一个 flash 小部件来显示目录(左边的图片),并将它放在了他的博客的[右边栏中。它有一个很棒的设计，避免了 Dave 网站上原始框的页面刷新。](https://web.archive.org/web/20230213033303/http://blogs.opml.org/kosso)

一旦我找到时间，我将实现 Matt 的 wordpress 插件来自动输出我们的帖子，并在 Kosso 的允许下，将他的小工具放在我们的网站上，以取代我们右边栏的基本分类系统。

OPML 太棒了。
