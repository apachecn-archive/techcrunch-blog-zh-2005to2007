Netvibes个人主页

由Florant Fremont在法国发起的项目Netvibes是基于个性化主页理念的最新网络应用，和其他一些应用一样(如 Start 和 Google的个人主页)，它是由一些精心开发的Ajax驱动的。他们的目标最好用他们自己的话来表达，所以引用他们关于9月8日发布的博客帖子，

它的目的是在互联网上提供一个地方，你可以设置你喜欢的内容，一个超级简单的方法来建立你的个性化主页，随时随地可用。不需要注册，但是如果你想从另一台计算机访问你的页面，你可以用你的电子邮件地址和密码注册。

这个项目本身看起来很不错:模块很直观，页面交互很容易在几秒钟内完成。很明显，还有一些其他的东西，比如Flickr的模块，feed搜索，书签等等(一个合适的列表可以在他们的博客上找到)。总而言之，虽然它是许多人正在处理的问题的另一种解决方案，但Netvibes看起来是一个非常好的基于Ajax的主页，所以你应该绝对去看看并得出你的结论。
