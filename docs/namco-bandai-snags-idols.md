# 纳木错万代捕捉偶像

Namco Bandai 与 FremantleMedia 达成协议，以手机游戏的形式将后者的偶像电视节目带到欧洲、中东和亚太地区。

Namco Bandai Networks Europe 的总经理大久保正治说:“偶像是世界上最大的电视节目之一，数百万粉丝很快就可以随时与他们最喜欢的节目互动。”音乐人才搜索计划是世界上最成功的电视特许经营之一。这是一笔多年的交易。
