# 智能手机现在:欢迎

![](img/3b6a6fbe41b43ca728c069edddd608cd.png)

欢迎来到[智能手机现在](https://web.archive.org/web/20130627211723/http://www.crunchgear.com/smartphones-now)，这是我们对智能手机领域的首次年度调查。这组文章将从今天开始一直持续到本周五，内容涉及主要的智能手机操作系统，以及今年我们最喜欢的一些精选。

今年我们注意到的一件事是智能手机市场的主流化。黑莓珍珠(Blackberry Pearl)和摩托罗拉 Q 等手机已使智能手机成为普通手机买家的可行选择，Cingular 99 美元的诺基亚 E62 has 是一款具有 QWERTY 键盘和黑莓兼容性的智能手机，几乎每个人都可以接触到这些奇迹。智能手机不再适合放在皮带皮套里，而是放在钱包、公文包和背包里。
