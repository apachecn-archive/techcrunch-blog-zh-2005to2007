# 参议院提出新的 ESRB 立法

![](img/d59af11ff228366412a0f951c8dbdebb.png)

参议员萨姆·布朗巴克提出的《视频游戏分级法案真相》要求 ESRB 实际播放完整的游戏。目前，ESRB 根据开发者给他们的视频演示给游戏打分。为了记录在案，我从来没有玩过暴力或非暴力游戏，并认为 ESRB 评级是不恰当的。布朗巴克还表示，“它还将禁止视频游戏生产商和分销商向评级机构隐瞒或隐藏可播放的内容”。隐藏或保留可播放的内容？如果一个游戏演示有大量的暴力，它会自动将游戏分级为 M 级。我们看不到的是 A 级(仅限成人)游戏。也许这就是布朗巴克想要的？

> 除了新的要求，布朗巴克的法案还要求美国联邦贸易委员会明确定义描述游戏内容的参数，以及什么将被视为游戏内容的错误描述。当前的视频游戏分级系统需要改进，因为评论者看不到游戏的完整内容，甚至不玩他们应该分级的游戏。为了使视频游戏的评级有意义并值得父母的信任，游戏评级必须更加客观和准确。
