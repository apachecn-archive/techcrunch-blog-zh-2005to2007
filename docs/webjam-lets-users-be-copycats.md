# Webjam 让用户成为模仿者

[![webjam_logo.jpg](img/28fc99c152652dce348798d58c42385c.png)](https://web.archive.org/web/20220816002232/http://about.webjam.com/) 一家名为 [Webjam](https://web.archive.org/web/20220816002232/http://about.webjam.com/) 的欧洲公司本周末在巴黎的 [LeWeb3](https://web.archive.org/web/20220816002232/http://www.leweb3.com/) 大会上发布。它是一个带有大量社交网络组件的个人网络聚合工具。

起初，我认为 Webjam 就像一个爱好者[发言人](https://web.archive.org/web/20220816002232/http://www.beta.techcrunch.com/tag/Spokeo/)，因为它允许你用 RSS 订阅源和来自 [Flickr](https://web.archive.org/web/20220816002232/http://www.beta.techcrunch.com/tag/flickr/) 等网站的个人内容定制你的个人资料页面。在这两个网站上，用户都可以在他们的网络中分享他们的网页，但 Webjam 将这一功能推进了一步。通过 Webjam，用户可以复制他们在 Webjam 网络中找到的其他页面。他们可以用其他人的内容创建一个新页面，然后随心所欲地修改它。

例如，假设您发现一个宣传即将上映的电影的 Webjam 页面。你决定要与你的朋友分享它，但认为其中的一些内容是令人不快的或蹩脚的，或者出于某种原因不值得分享。如果该页面是公开的，您可以复制该页面，现在该页面将成为您的，您可以随意更改。你可以改变颜色，布局，内容。您可以从其他网站添加电影评论。这是你的 Webjam 帐户下的一个新页面，有一个新的 URL。

如果你创建了一个你不希望任何人编辑或复制的网站，你可以将其设置为私有，但 Webjam 认为大多数人会将他们的页面公开。毕竟，模仿是奉承的最好形式。

对于私人使用，如果您想将主页设置为除了标准主页元素之外还提供各种社交组件，如您姐姐的 Flickr 相册或您朋友推荐的当日网页，Webjam 非常有用。您可以轻松地将其他人页面中的元素复制到您自己的主页中。一按下 Edit，页面的每个组件就变成了一个模块或小部件，可以通过拖放进行编辑。唯一不可移动或删除的组件当然是广告模块。

Webjam 中的页面是由标签组织的。创始人说，他们预计人们将开始在每个标签内建立社区。

“它非常强大，因为它将交流的力量带到了一个新的水平，”Webjam 的联合创始人兼董事总经理 Yann Motte 说。“分享没问题，但那又怎样？我们将通过允许人们在 Webjam 正在做的事情的基础上建立社区来使人们变得更好。如果你不知道如何运营一个网站，你可以去 Webjam，选择一个你喜欢的，然后复制它。”

我不认为 Webjam 是最容易学习的网站。有许多工具和功能，这并不是一件坏事，但它确实比大多数社交网站花了更长的时间来弄清楚。但是[教程](https://web.archive.org/web/20220816002232/http://help.webjam.com/wj/tutorial/)写得很好。

Webjam 一周前非正式推出，但他们没有告诉我到目前为止有多少用户注册。莫特说:“我们想把这些信息留给自己，但我们会说，我们从一开始就没想到会有这么多用户。”。该公司正在完成第一轮融资。
