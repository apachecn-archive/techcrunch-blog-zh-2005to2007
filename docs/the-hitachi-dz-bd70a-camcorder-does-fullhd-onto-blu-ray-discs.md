# 日立 DZ-BD70A 便携式摄像机可以全高清播放蓝光光盘

[![dzbd70a.jpg](img/5c2e3d8833cc2e2044556284e98b8ce5.png)](https://web.archive.org/web/20130628211354/http://old.crunchgear.com/wp-content/uploads/dzbd70a.jpg "dzbd70a.jpg")

日立的 DZ-BD70A 是第一台可以将 1080p 视频录制到蓝光光盘上的高清摄像机，出于某种愚蠢的原因，现在被称为“全高清”。一张碟片能让你看大约一小时的全高清视频。这款摄像机也可以用作普通的 jane 数码相机，它有一个 SD 卡插槽来存储这种照片，分辨率高达 432 万像素。如果你还在用 DVD(我想大多数人都是这样)，要知道这款摄像机可以很高兴地录制成标准清晰度的 DVD 光盘。

这真是奇怪的一周。不[比](https://web.archive.org/web/20130628211354/http://crunchgear.com/2007/08/01/panasonic-announces-worlds-smallest-vertical-hd-camcorder/) [少](https://web.archive.org/web/20130628211354/http://crunchgear.com/2007/07/30/panasonic-intros-two-hd-camcorders/) [三家](https://web.archive.org/web/20130628211354/http://crunchgear.com/2007/08/01/canon-ivis-hg10-high-def-hdd-camcorder/) CE 公司已经宣布或展示了高清摄像机。原因很明显，因为这正是杂志开始为他们的假日指南编辑小工具列表的时候。尽管这个夏天对你们来说似乎过得很慢(当然我们也注意到了)，但现在事情应该会有所进展。

别担心。很快就会有一个接一个的产品发布，让你垂涎欲滴。然后圣诞老人来了，我们都可以玩我们的新玩具。
