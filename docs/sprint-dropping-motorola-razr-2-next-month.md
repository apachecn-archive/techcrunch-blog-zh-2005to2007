# 下个月加入摩托罗拉 RAZR 2

![sprintv9m1.jpg](img/f165db68c3a3a25523d82aeda08ab34e.png)

据 BGR 的匿名消息人士称，Sprint 将在下个月的某个时候推出 RAZR2 V9m。规格保持不变，但突出的一点是数字模式下的 3 小时通话时间。嗯。时间不多了，摩托罗拉。支持的服务将包括 Sprint TV、音乐商店、图片邮件、视频邮件、NFL 移动应用程序、点播、无线备份和蓝牙 A2DP。尽管 RAZR 很烂，但这款还是相当酷的。外部触摸屏按钮非常好用。
