# TimeBridge:现在通过网络同步您的会议

[![](img/48d198d0bbce4ddd1ede65441d8fb018.png) ](https://web.archive.org/web/20220516092730/http://www.crunchbase.com/company/Timebridge) [TimeBridge](https://web.archive.org/web/20220516092730/http://www.crunchbase.com/company/Timebridge) 是一家总部位于旧金山的初创公司，它想做好一件事:帮助安排会议。

它们最初是去年年底推出的深度集成的 Outlook 插件[。虽然最初将它们与其他调度竞争对手区分开来，但我有一种感觉，插件需求增加了使用系统的不必要的摩擦。](https://web.archive.org/web/20220516092730/http://www.beta.techcrunch.com/2006/11/07/the-launchpad-13-at-web-20-summit/#more-3600)

现在，TimeBridge 重新推出了一个完整的网络界面，可以与您的 Google 和 Outlook 日历集成，并很快支持其他日历。新版本将他们推向总部位于蒙特利尔的竞争对手 Tungle 的方向，后者集成了更多的日历。Tungle 的不同之处还在于它使用了一个在你和你的联系人系统之间运行的 P2P 系统，而不是像 TimeBrigde 那样在一个中央服务器上运行。

[![timebridgescreensmall.png](img/bddbee7b7361b2038dea31c21e51c92e.png) ](https://web.archive.org/web/20220516092730/https://beta.techcrunch.com/wp-content/uploads/2007/08/timebridgescreenbig.png) TimeBridge 的原始 Outlook 插件将它们的全部功能带到了您的桌面上。所有这些功能现在都可以在 web 应用程序中使用，使用插件在您的计算机和 web 之间推送日历数据。一个日历上的更新会反映在其他日历上，并且在网络上的任何地方都可以访问主日历。

要召开会议，您可以登录到他们的网站，并填写一个类似电子邮件的表单，其中包含与会者的电子邮件地址、会议主题和可能的会议时间。使用 TimeBridge 时，您不必下载任何东西，但如果您安装了插件，它会有所帮助。如果你的与会者没有时间桥，你可以根据你的个人日程安排来建议时间。但是，如果他们的日历中集成了 TimeBridge，您可以在边栏中选择时间来查看他们计划的时间段。

在您发出会议请求后，每个参与者都会在他们的电子邮件中收到完整的会议请求表单。该表单列出了可能的会议时间，参与者可以选择“不好”、“好”或“最好”。接受会议请求会在日历上为使用 TimeBridge 的人放置时间占位符。

然后，系统根据与较早时间有关联的与会者的响应，通过点数选择最佳会议时间。如果即将召开的会议时间尚未确定，您可以选择会议时间或向没有响应的人发送提醒。使用 TimeBridge 的与会者将会在他们的日历上看到已确认的会议时间。
