# Zenum Organum 便携式媒体播放器:外观就是一切

![](img/e6329f43940c2d95fd29712dd099d857.png)

Zenum Organum 便携式媒体播放器以其整体美感弥补了其令人印象深刻的统计数据和功能。换句话说，它看起来真的很好。你会称之为受 iPhone 启发吗？。那些特征？试试 2GB 的存储，1.8 英寸的屏幕，支持 MP3，ASF，WMA 和 WAV。非常基本的东西，这就是为什么 Zenum 必须尽力让它变得闪亮漂亮。年底应该会到。

泽纳姆拉了其中一个“这是如此可怕的基本，但男孩看起来膨胀！”去年，当它开始展示它的 [Opteris](https://web.archive.org/web/20210227214749/http://crunchgear.com/2006/09/20/zenums-opus-opteris-pocket-pc-smartphone-looks-great/) 智能手机时，真是让人目瞪口呆。顺便说一句，在下周的 CeBIT 展会上，Opteris 可能会重见天日。不要让任何人告诉你，风格并不比内容更重要。
