# 任天堂召回原装 Wii 遥控表带

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/15/nintendo-recalls-original-wii-remote-strap/>

![](img/7b3b47e0ede169072a9da3bfe096172c.png)
在[提醒](https://web.archive.org/web/20131104145827/http://www.nintendo.com/consumer/wii_safety.jsp)顾客如何正确使用腕带后，任天堂宣布完全召回原来有问题的腕带。要申领你的新表带，请填写这张[在线](https://web.archive.org/web/20131104145827/http://www.nintendo.com/consumer/strapreplace.jsp)表格，任天堂应该会在不久的将来给你一个新的。

如果你在等待更换表带时无法停止玩 Wii，你可能需要考虑采取额外的安全措施来确保你的电视继续存在。
