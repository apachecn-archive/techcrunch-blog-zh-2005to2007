# Pixelotto 你必须在里面才能赢…

亚历克斯·图的第一个想法——“百万美元主页”在一夜之间引起了网络轰动。这是一个来自英国大学生的非常聪明而大胆的(销售和/或营销)想法，让他在很年轻的时候就成为了百万富翁。为什么我没有想到它，但像所有的好主意一样，事后看来它们是显而易见的。从那以后，我有幸与亚历克斯见过几次面，也听过他的演讲和真人版 beatbox。

上次在伦敦举行的[互联网爱好者会议上，亚历克斯暗示了他的新想法。“我要捐出一百万美元！”在他宣布之后，我们简短地谈论了一个事实，即像任何新的乐队一样，在他们的第一张专辑取得初步成功之后，最难的是“第二张专辑综合症”，并最终决定他们的生死——想想詹姆斯·布朗特、蒂朵、阿拉尼斯·莫里塞特或莫比。](https://web.archive.org/web/20230305032132/http://internetpeeps.com/blog/2006/11/15/ideas-qa/)

我对亚历克斯说，我想他可能也有同样的问题。人们对他肩负的传递新的、原创的、完全左领域的东西的期望肯定是巨大的。

等待结束了，Alex 现在公开宣布了他的新想法—[Pixelotto](https://web.archive.org/web/20230305032132/http://www.pixelotto.com/)。对不起亚历克斯，但我完全没有印象，因为它几乎是同样的事情了，~~所以我不会链接到它，不，它不是十亿美元的网站。如果你想了解更多，我相信你可以在谷歌上找到它。~~

我要说的是，主要的想法是[drumroll]这次你付给亚历克斯每像素 2 美元，然后他会宣布，有人在最后，一个像素购买者赢得 100 万美元，亚历克斯得到另一个！？我想这可能行得通，因为有足够多的人去买彩票，那么为什么这些人不把他们的钱给亚历克斯而不是玩 pixelotto 呢？
在某种程度上，我理解为什么个人和中小企业可能会觉得购买一两个像素以在网站上显示他们的徽标很有吸引力，但为什么要购买 Bebo 和 Last Minute 呢？到目前为止，已经产生了不到 20 万美元，所以谁会笑到最后，我怀疑可能是亚历克斯，但在人类 beatbox 唱歌之前，还有 40 万像素要出售！？

不出所料，英国最新的亚历克斯·图崇拜者推出了“tickettoamillion”

**更新 2** :我现在已经链接到 pixelotto，并按照评论中指出的那样修改了帖子。我希望亚历克斯一切顺利，并很高兴被证明是错误的，但对于一个聪明和精通网络的人来说，我希望它是更有创意的东西。
