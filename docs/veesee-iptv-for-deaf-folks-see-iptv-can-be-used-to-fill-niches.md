# 面向聋人的网络电视:看，网络电视可以用来填补空白

[![veesee.jpg](img/04055fa7277c1261b15ee73781a1fc04.png)](https://web.archive.org/web/20230109072551/https://techcrunch.com/wp-content/uploads/2007/05/veesee.jpg "veesee.jpg")

看那个！有人用技术来做好事，而不仅仅是为了输出更多的百万像素或每秒多少。这家名为 VeeSee 的英国 IPTV 服务(想想 [Joost](https://web.archive.org/web/20230109072551/http://crunchgear.com/2007/02/22/joost-for-mac-reviewed-huh-looks-cool/) )迎合了聋人的需求，节目以英国手语为通用语言。该公司认为，在传统电视节目中，缺乏适合聋人的节目来满足聋人的需求。IPTV 频道实际上是提供 900 个流媒体频道的大型电视门户网站 ViewTV 的一部分。这种内容需要一点时间来创建，所以在未来的几天或几周内，预计只有三到四个小时的时间。就目前的情况来看，比现在多得多。

另外，我们已经有一段时间没有在电视上看到手语女了。在这里，她古怪的德国对手。
