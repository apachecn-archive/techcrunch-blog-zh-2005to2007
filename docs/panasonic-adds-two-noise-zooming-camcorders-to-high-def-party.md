# 松下为高清派对增加了两台降噪摄像机

松下刚刚正式发布了两款新的消费者高清摄像头，加入他们已经拥有的 boss prosumer 系列。

HDC-SD1 使用 SDHC 卡，被吹捧为“世界上最小最轻的 3cd 高清摄像机”，而 HDC-DX1(这些家伙什么时候才能知道型号名称！)将您的视频放入 DVD。

它们都非常小，所以内置图像稳定功能是必须的。然而，最整洁的特征似乎是这个:

> HDC-SD1 和 HDC-DX1 采用世界上首款 5.1 声道环绕声系统，配有 5 个麦克风 2。当用这些型号制作的录音在 5.1 声道家庭影院系统上播放时，观众会被清晰、细致的声音所环绕，使他们感觉就像置身于行动之中。变焦麦克风功能将麦克风的动作与摄像机的动作联系起来。例如，当放大远处的主题时，麦克风也会放大并记录该主题的特定声音。

如果我没看错的话，这意味着你完全可以用这个东西在聚会上监视别人。只需放大它们，剩下的就交给麦克风了。整洁！

他们都是在三月袭击的。SD1 的价格是 1500 美元，DX1 的价格是 1400 美元。
