# ViddYou 为 Vloggers 推出 Blogger

[![viddyoulogo.png](img/ad20c80fe65e9701bb124ccb50cc9671.png) ](https://web.archive.org/web/20220128110809/http://viddyou.com/) [ViddYou](https://web.archive.org/web/20220128110809/http://viddyou.com/) 今天刚刚推出他们的视频博客网络。根据 Mefeedia 的说法，大多数的 vlogging 发生在人们的个人网站上，这些网站倾向于像已故的伟大的 [Ze Frank](https://web.archive.org/web/20220128110809/http://www.beta.techcrunch.com/2006/10/27/rocketboom-almost-10x-ze-franks-downloads/) 这样的网络节目。另一个更私人的，一半是由社交视频网站捕捉的。ViddYou 专注于后一类，加强和推广通常隐藏在 YouTube 等社交视频网站中的“个人和博客”部分。

该网站基于简单的社交网络设计，由 vlogs 和社区组成。Vlogs 跟踪用户最新视频、朋友、兴趣和个人简介视频。社区将兴趣相似的博客捆绑在一起。目前，社区只能由公司创建，并仅限于一般，旅游，音乐和忏悔。

整体用户体验简单明了。您可以将视频发布到您的 vlog，方法是将它们记录在您网络摄像头上的网站小部件中，或者从手持或手机摄像头上传(通过电子邮件彩信)。如果你安装了一个 [3GP](https://web.archive.org/web/20220128110809/http://en.wikipedia.org/wiki/3gp) 视频播放器，你甚至可以在手机上观看视频。他们目前将视频限制在五分钟内，以阻止非法发布内容。你上传的每一个视频都可以被评分、回复(视频或文本)、嵌入，并附有一组放在录像下方的 Flickr 照片。

我们之前已经见过各种各样的其他个人视频服务: [Vlip](https://web.archive.org/web/20220128110809/http://www.beta.techcrunch.com/2007/03/15/vlip-launches-vlogging-for-dummies/) 、 [Ustream](https://web.archive.org/web/20220128110809/http://www.beta.techcrunch.com/2007/03/26/build-your-own-justintv-with-ustream/) 、 [Stickam](https://web.archive.org/web/20220128110809/http://www.beta.techcrunch.com/2006/02/14/pimp-my-blog/) 、 [Blip.tv](https://web.archive.org/web/20220128110809/http://www.beta.techcrunch.com/2006/08/15/eyespot-partners-with-bliptv-veoh-to-provide-online-video-editing/) 以及[其他社交视频网站](https://web.archive.org/web/20220128110809/http://www.beta.techcrunch.com/2005/11/06/the-flickrs-of-video/)。ViddYou 只专注于视频日志而不是网络节目；这有助于将其与其他服务区分开来。Vlip 由视频评论线程(视频评论的公告牌)组成。Ustream 和 Stickam 支持实时节目流。Blip.tv 对网络节目进行分类，而其他社交视频网站倾向于将视频日志淹没在堆积如山的其他视频中。
