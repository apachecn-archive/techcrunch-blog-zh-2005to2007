# 汽水机器人流行了 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/12/pop-goes-the-soda-bot/>

![](img/e7092482eeaebf902a9df680d397296b.png)

这个转换玩具套件可以让你把你的易拉罐变成某种电动机器人虫子，可以振动、滑动和嗡嗡作响(没错，我来自中西部。叫“pop”)。虽然出售它的英国公司 [Great Gizmos](https://web.archive.org/web/20221025120902/http://www.greatgizmos.co.uk/index.htm) 还没有公布价格或发布日期，但我可以想象它在喝啤酒的兄弟会成员和环保主义者中非常受欢迎。这会让他们有更好的事情做，而不是把罐头塞进他们的脑袋里，或者花 5 美分买下它。
