# 变形金刚爱上鞋子

![](img/09f77834048d6328e7802494b2332918.png)

我们已经通过[一个 iPod 基座](https://web.archive.org/web/20230322164148/http://crunchgear.com/2007/01/23/transformers-ipod-docks/)看到了对即将上映的《变形金刚》电影的期待，但是把它提升一个档次怎么样？如果你真的觉得大胆，你可以买一双印有变形金刚的运动鞋。

它们基本上是缩小了一半的耐克(不，你不能穿它们)，每个都展开成一个危险的、挥舞着武器的机器人。如果你熟悉这个系列，你会得到运动鞋形式的护卫队和威震天——保证带出你内心的孩子。几乎可以说他们是伪装的机器人？

不过，它们看起来确实很棒。59 美元可以让你得到一对机器人，还不包括系鞋带的课程。
