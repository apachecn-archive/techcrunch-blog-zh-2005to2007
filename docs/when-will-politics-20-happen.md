# 政治 2.0 什么时候会发生？

鉴于 Le Web3 的 Loic Le Meur 提出的一个问题是网络将如何改变政治体系，我认为这个来自 Stuart Bruce 的链接是恰当的。西蒙·迪克森写了精彩的《theyworkforyou.com，他用谷歌地图创建了一个新的英国议员的[混搭。我喜欢时不时地关注我当地的议员特里萨·梅(Theresa May)在做什么，她如何代表我投票，以及她把纳税人的钱花在了哪里。](https://web.archive.org/web/20181120053518/http://maps.findless.co.uk/mpmap.php "Findless.co.uk: UK MPs")

这让我想知道为什么我们仍然有超额支付(费用，养老金，监督欢乐)(欧元)议员代表我们。过去，他们把我们的声音带到议会，但是考虑到现代的通讯手段，如 digg(投票)、电子邮件和博客(双向交流)、rss(感兴趣的话题)等，我们还需要保留同样的政治制度吗？
