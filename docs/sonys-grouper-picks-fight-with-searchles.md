# 索尼的石斑鱼选择与搜索者战斗

今天早上，石斑鱼给社交搜索引擎[searches](https://web.archive.org/web/20220929203837/http://www.beta.techcrunch.com/2007/02/08/searchles-nabs-another-angel-round/)发了一封信，当然，这封信被发布到了网上。以下是搜索者[的回复](https://web.archive.org/web/20220929203837/http://blog.searchles.com/2007/03/sonys-grouper-tells-searchles-to-cease.html)。Grouper 不喜欢 Grouper 视频在该网站的新搜索电视产品上播放。Grouper 的主要抱怨是在 Searchles 播放器中对视频进行了重新皮肤处理，删除了它们的品牌和反馈功能。

如果这样的抱怨有任何效力的话，那对媒体混搭者来说是个坏兆头，比如视频多媒体播放器 [Splashcast](https://web.archive.org/web/20220929203837/http://www.beta.techcrunch.com/2007/01/29/splashcast-launches-one-player-to-bind-them-all/) 和 [Cuts](https://web.archive.org/web/20220929203837/http://www.beta.techcrunch.com/2007/02/21/cuts-launches-amidst-online-video-editor-war/) 视频编辑器。这对于 AdBrite 的视频播放器来说尤其糟糕，它并不拥有自己的内容，而是发布自己的广告。
