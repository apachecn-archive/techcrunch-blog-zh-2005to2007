# 关于移动 Web 2.0 的新书| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/08/27/new-book-about-mobile-web-20/>

阿吉特·乔卡尔和[余炳贤](https://web.archive.org/web/20160202203240/http://www.tonyfish.com/)，两位著名的英国移动网络演讲家，刚刚写完他们的新书《[移动网络 2.0](https://web.archive.org/web/20160202203240/http://www.mobileweb20.futuretext.com/) 》。以下是这本书涉及的一些热门话题。![](img/d500da664a9ccee99935dad519c84426.png)

*   移动内容和不断变化的力量平衡(用户生成内容的力量)
*   我不是一个数字，我是一个标签(对电信业数字管理的影响)
*   多语言移动接入(任何人，任何地方，只要有手机运行..)
*   移动 web 2.0 和数字融合(移动 web 2.0 是数字融合的驱动力)
*   Ajax 和移动小部件的颠覆性力量
*   基于位置的服务和移动 web 2.0 (LBS)从未真正起飞。移动 web 2.0 会有帮助吗？)
*   移动搜索:比你手机上的谷歌多得多。

这些话题中的大部分(如果不是全部的话)肯定会在未来的某个时候在这个博客上讨论，因为我也计划回顾一下英国的移动 Web 2.0 初创企业。因此，我不时地邀请两位作者成为特约撰稿人，我很高兴地说他们已经同意了。；-)

此外，我还邀请了另一位著名的英国移动专家丹尼尔·阿普尔奎斯特成为我们的特约撰稿人。除了在沃达丰担任高级技术策略师之外，丹尼尔还是 W3C 移动网络倡议的主席和伦敦移动星期一活动的组织者。

如果你感兴趣，下一次 MoMoLondon 活动将于 9 月 4 日举行。
