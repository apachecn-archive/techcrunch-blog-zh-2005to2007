# Plazes 重整旗鼓，向 Twitter 学习

![plazesmap.png](img/dd159bec18f4bc7a2be83fe8c6fb03a7.png)
[Plazes](https://web.archive.org/web/20221228064535/http://plazes.com/) ，总部位于苏黎士和柏林，是一个将你和你的朋友连接到你消磨时光的地方的社交社区。他们已经存在有一段时间了(我们在 2005 年 6 月 TechCrunch [推出](https://web.archive.org/web/20221228064535/http://techcrunch.com/2005/06/11/plazes-gets-traction-at-reboot-70/)的那天对 Plazes 进行了简介)，他们最近[在一轮风险投资中为](https://web.archive.org/web/20221228064535/http://techcrunch.com/2007/02/05/european-startup-plazes-raises-e27-million/) €筹集了 270 万美元。

Plazes 让用户告诉其他人他们在哪里，他们在做什么。目前，用户必须下载软件，该软件根据 IP 地址、来自路由器的网络 id 和其他信息自动确定用户的位置，然后将你放在 Plazes 地图上。如果这是一个 Plazes 成员以前没有去过的地方，你可以命名和描述这个地方，并添加 Flickr 照片。尽管 Plazes 做得很好，但要求用户下载软件并在他们改变位置时使用软件的摩擦造成了摩擦并减缓了用户增长。借鉴 Twitter 的简单性，Plazes 正在改变其处理位置的方式，并为该产品增加了时间和活动维度。

新功能将于下周推出。不再需要客户端软件来设置位置。相反，用户可以通过 Google maps mashup 简单地添加一个地方(并通过建议机制得到帮助)，还可以说他们过去是否在那里，将来是否会在那里，或者现在是否在那里。用户也可以说他们在做什么，这是一个非常类似 Twitter 的活动。用户可以通过 Plazes 网站、客户端软件或通过移动设备将这些数据发送给 Plazes。一个即时通讯界面即将推出。
新版本还将保留用户的更新历史，并允许他们发布未来的位置，让你可以计划或偶遇。这个时间表也可以让你的朋友全面了解你何时何地在一起。

[![plazessmall.png](img/a6c99bbc15ead87d20a6ba4b6fc2dda2.png)](https://web.archive.org/web/20221228064535/https://techcrunch.com/wp-content/uploads/2007/05/plazesbig.png) 好友可以订阅人物或地点，并从那里看到数据流。还可以组成包括人员和地点的组，并且 feed 信息显示这些人员在所选位置时在做什么。

Plazes，000 名会员中的 250 名将在下周前获得更新。Plazes 吸引了 60%的美国人，其余大部分是欧洲人。

Plazes 显然试图降低参与障碍，以获得更多用户和现有用户的更多参与。该产品将明显更容易使用，添加活动信息意味着用户更新的频率可能比他们改变位置时更高。
