# 中国将下调手机资费

![china_flag.jpg](img/d7bf60c516e133886f84fcaad8a77005.png)亚洲电信报道称，中国信息产业部(MII)在其官方网站上表示，中国应全面实施单向收费，同时降低电信服务的一般费用。MII 还将通过降低最高限价来降低国内漫游费，具体价格还有待确定。该部进一步敦促电信运营商简化复杂的服务包。

这是中国政府首次正式确认单向收费将在全国范围内实施。
