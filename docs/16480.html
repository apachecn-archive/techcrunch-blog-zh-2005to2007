原文：https://web.archive.org/web/http://techcrunch.com:80/2007/07/02/new-laptops-from-toshiba-draft/

东芝似乎真的已经在市场两端的笔记本电脑上迈出了步伐:超便携和游戏。

Portege R500是东芝在移动领域的最新产品。令人惊讶的是，这款两磅重的笔记本配备了一个光盘驱动器，它号称是世界上最薄的，厚度为令人印象深刻的7毫米。

点击查看更多图片和规格…

R500包含一些很酷的功能和选项，如可用的64GB固态硬盘存储，当结合没有光驱时，笔记本电脑的重量下降到令人难以置信的1.72 磅。固态硬盘和LED背光显示屏的结合显然为R500提供了世界上最好的电池续航时间，估计为10小时。

除了上面提到的硬件，这款超便携设备还配备了蓝牙2.0和draft n wireless。1.2 GHz Core 2 Duo和标准1GB RAM确保它是一款出色的启动性能。

起价1999美元

东芝的怪物17英寸卫星X205配备了所有常见的桌面替换装饰品。虽然与外星人的同类产品相比略显逊色，但1999美元的起价也相当便宜。

基本配置包括:

–Nvidia GeForce Go 8700m GT，配备256 MB DDR 3图形内存

–2GB DDR2内存，配有1gb英特尔睿频内存

–17英寸WXGA显示屏

–蓝牙2.o EDR(增强型数据速率)

–802.11 a/g/n无线

–高清DVD光盘

–4个带低音炮的扬声器

–240 GB存储2×120(抱歉，没有raid！)

多花500美元，CPU就能提升200MHz，硬盘空间提升到320(仍然没有raid ),显存提升到512MB。
