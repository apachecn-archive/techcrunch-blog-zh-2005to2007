# Streamburst 提供创新的非 DRM

纪录片制作人史蒂夫·奥黑尔今天宣布，他的电影《寻找山谷》[现在可以通过一项名为](https://web.archive.org/web/20221207202759/http://www.insearchofthevalley.com/) [Streamburst](https://web.archive.org/web/20221207202759/http://www.streamburst.co.uk/) 的创新服务下载——在最初的 DVD 销售令人失望之后，他希望这一举措能够提高该项目的销售。Streamburst 提供了一种阻止和跟踪盗版的方法，而不限制购买文件的人使用这些文件。

去年 12 月发行 DVD 时，我们写了奥黑尔采访硅谷名人史蒂夫·沃兹尼亚克、约翰·沃诺克、盖伊·川崎、克雷格·纽马克、杰夫·拉斯金、蒂姆·奥莱利、丹·科特基和许多其他人的电影。这些采访探索了硅谷的历史，以及为什么它会成为这样一个科技创新中心。

如果不是因为盗版问题，http://web.splashcast.net/p/提供完整的电影下载可能是大多数独立电影制作人的首选发行方式。虽然有些人很容易嘲笑媒体巨头的担忧，但小型独立电影制作人的职业生涯显然直接取决于他们尽可能多地将作品货币化的能力。(而且他们更容易让人同情。)

输入 [Streamburst](https://web.archive.org/web/20221207202759/http://www.streamburst.co.uk/) ，O'Hear 选择在线分发。这家新成立的英国公司对版权采取了一种有趣的方法。Streamburst 采取了两个步骤来防止电影盗版，而不是像 DRM 阻止的那样给那些想在多个设备上观看他们购买的电影并合法使用内容的观众戴上手铐。

第一，每部电影开始时都有一个 5 秒钟的显示，显示购买拷贝的人的名字，就像他们的信用卡上显示的一样。第二步是 Streamburst 从下载文件的每个副本中剥离出一系列不可检测但独一无二的位。该公司声称，这种签名将生存大多数编辑和格式转移。这个想法是，被命名的心理障碍将阻止许多人非法分发文件，那些没有阻止的人可以通过从任何进入非法文件共享网络的副本中剥离的独特的一系列比特来识别。

无论如何，这都不是一个不可战胜的计划，但 Streamburst 可以帮助内容发行商更舒适地提供他们的作品供下载。《寻找山谷》的制作者就是这种情况，该片现在有高质量、便携和移动格式[，通过 Streamburst](https://web.archive.org/web/20221207202759/http://isotv.streamburst.tv/) 售价 8 美元。
