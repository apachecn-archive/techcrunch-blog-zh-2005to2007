# PlayStation 3 慈善发布会

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/18/playstation-3-launch-line-for-charity/>

![](img/2b064d319966d67b376287b895b52f7d.png)

所以 PS3 昨天在美国发售了 40 万台。各地的游戏玩家涌入电子商店，希望得到一台梦寐以求的游戏机。有各种各样关于暴力和恶作剧的报道。一台 PS3 甚至就在购买它的商店门前被砸碎，而那些排队的人则厌恶和恐惧地看着。

这个故事有点不同，也许有点感人。为了给[儿童剧](https://web.archive.org/web/20150912121219/http://www.childsplaycharity.org/about.php)筹集资金，由[便士街机](https://web.archive.org/web/20150912121219/http://www.penny-arcade.com/)、 [ScrewAttack](https://web.archive.org/web/20150912121219/http://www.screwattack.com/LaunchLine.html) 的家伙们建立的帮助生病儿童的慈善机构为了一台 PS3 排了五天队，然后将其拍卖给慈善机构。拍卖产生了可观的 3200 美元，全部捐给了儿童游戏公司。

啊，高尚的事业。有点让我觉得自己是个混蛋。不管怎样，去捐吧。说真的。
