# 太阳能无线扬声器——为无线而欢呼

![](img/6449398d946cbfaabcdf4a9509625827.png)

这款音箱使用太阳能给 7 节 AA 充电电池充电。电池依次驱动 5 瓦 RMS 扬声器。它的射程约为 150 英尺，装在一个防风雨的外壳里。你可以花大约 152 美元买到它们。是只有我觉得还是它看起来有点像热水瓶？
