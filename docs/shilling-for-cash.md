# 先令换现金——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/02/shilling-for-cash/>

![shill.JPG](img/bc91c9393e3388efa64041624a81e7d5.png)这不算是一篇技术文章，但这是我们在 [TechCrunch](https://web.archive.org/web/20210412213134/http://www.beta.techcrunch.com/2006/06/30/payperpostcom-offers-to-buy-your-soul/) 的年长但仍然很酷的叔叔们的一篇有趣的报道。PayPerPost 将博客作者与广告商联系起来，这意味着“骗子”可以与“男人”联系起来，出售“垃圾”。

真正的问题是，人们希望博客写手们写一些关于他们推销的产品的好东西，而且他们不会公开他们之间的关系。当我们在 CG shill 的时候，我们是公开的，所以每个人都知道。然而，大多数“长尾”上的博主没有我们所拥有的广泛的联系和渠道，幸运的是，因此他们不能拒绝这个机会。那会是什么呢，joesipodgaragegear.info？女士还是老虎？

毫无疑问，博主应该得到尊重和访问权，即使是世界上最差劲的黑客和当地新闻媒体也应该得到尊重和访问权。不幸的是，很难获得并维持 access 和 PayPerPost 假装提供的那种服务。但这是错误的访问，它被称为 shilling。博客应该谨慎对待这种做法，如果不是厌恶的话。
