# 未来学家:我们将在 CES 上看到什么

再过几个小时，整个科技行业(除了我自己——我正在度一个非常需要的假期)将涌入拉斯维加斯参加 CES，CrunchGear 的团队将准备好给你今年塑造年度事件的最新独家新闻。

很多人在每年的这个时候玩 CES 猜谜游戏，预测谁将发布什么。我们克朗彻奇公司想更进一步。在很大程度上，除非另有说明，接下来的预测不仅仅是猜测，而是我的消息来源给了我很好的理由相信我们将在 ces 上看到的东西。我不得不对此含糊其辞，以免与 CrunchGear 法律团队发生冲突。所以准备好被取笑吧。

点击跳跃看看有什么……


**摄像机:**泄密者认为日立是第一个使用混合 DVD/硬盘摄像机的公司(*免责声明:我知道这件事已经有一段时间了，但由于 NDA，我无法在博客上发表。日立公司已经批准我提及此事。这个承诺很酷:一键即时复制硬盘上的视频应该可以让你在长达一周的假期中不用跑来跑去。但是不要认为日立是唯一一个意识到混合摄像机潜力的公司。至少有另一家公司几乎可以肯定已经瞄准了这个市场，并且正在采取一种完全不同的方式…*

电视:LCD 和等离子质量的下降和提高，以及 SED 即将到来的必然性，让我相信 DLP 正在利基市场之外的最后挣扎——反批量生产的反弹似乎很大。惠普在科技上的放弃对事业没有帮助。然而，期待在 CES 上看到一些东西，这应该给 DLP 芯片制造商德州仪器带来希望，不需要再次依赖图形计算器游戏。这并不是说降价液晶和等离子不会很快锁定 DLP，但它们至少还有一个不错的市场份额。如果这涉及到 LED 种类的 DLP，我不会感到惊讶…

**手机:**在不知道苹果的“不要叫它 iPhone”将会发生什么的情况下，预计从明年 1 月开始，将会有大量白色手机从生产线上下线。2006 年是黑色的一年，2007 年将是白色的一年(当然，有彩虹的阴影。)这将可能包括新的模型，以及熟悉的朋友的白色版本。2007 年将是白色手机年，苹果进入市场只会进一步推动这一趋势(你真的能想象他们的第一部手机不是白色的吗？)

**DVRs:** 我喜欢 TiVo——这个品牌让我充满了温暖的模糊感觉——但我猜这些家伙不会有太多的新硬件，因为系列三刚刚下线。然而，他们已经成为内容交付的忠实粉丝，所以一些独家交易并不稀奇。不幸的是，我对他们将在展会上展示什么一无所知。我所知道的是，你应该留意其他提供增强版媒体中心 DVR 的公司。其中一些产品试图成为满足您所有客厅需求的一站式商店，其方式在消费者层面上从未真正尝试过，目前的设计趋势可能表明，至少有一个这样的模型可能看起来像 Roomba…

暂时就这样了。记得收听 CrunchGear 对本次极客马拉松的热血报道。
