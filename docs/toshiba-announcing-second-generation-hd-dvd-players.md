# 东芝发布第二代 HD DVD 播放器

东芝将于今年晚些时候推出的新播放器将使用 HDMI 版本，该版本将带宽翻倍，并支持 720p 和 1080p 播放。这两款机型 HD-A2 和 HD-XA2 将取代它们的 HD-A1 和 HD-XA1 播放器，这两款播放器存在缺陷和其他问题，导致销售速度低于预期。

HD-XA2 将具有 HDMI 1.3、1080p 分辨率和 297MHz/12 位视频 DAX，具有 4 倍过采样和图片设置功能。对你来说，这可能看起来像是技术术语——的确如此——但它允许用户定制对比度、亮度、噪声阻挡和其他参数，以优化图像输出。

HD-A2 在 10 月份将为 499 美元，HD-XA2 将为 999 美元。
