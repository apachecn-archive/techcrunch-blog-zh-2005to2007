# 网页过滤:电脑说没有？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/29/web-filtering-computer-says-no/>

![](img/7a18c596bea8103eb70d6717f6514ca5.png)

你有没有不去关注手头的工作，而是在工作的时候查看足球比赛的结果，进行网上购物，或者只是在即时消息上和朋友聊天？嗯，如果这与你的工作无关，那么一些员工可能很快就会试图通过使用网络过滤技术来阻止你访问他们认为不合适的网站或应用程序。

总部设在苏格兰的 Bloxx 就是这样一家互联网过滤公司，他们开发了一种技术，使组织能够更好地控制他们的员工、学生等。互联网的使用。Bloxx 目前正在庆祝赢得爱丁堡晚报商业卓越奖的“创新和技术”奖，但网络过滤在工作中是好消息还是坏消息？

员工**信任与控制**这个棘手的问题最近在微软技术大会的主题开幕式上被讨论。艾德，在悉尼。安妮·基拉(Anne Kirah)，微软高级设计人类学家(？)说:

> 世界各地的公司都在说，哦，你不能在上班的时候上网。她说:“你不能在工作时间使用即时通讯工具。”。这些是数字移民的想法。

基拉将“数字移民”定义为没有出生在数字生活方式中的人，他们认为这是一种干扰，而不是生活中不可或缺的一部分。年轻一代的工人从出生开始就使用电脑和手机，她称他们为“数字原生代”。

基拉认为，这种冲突之所以经常发生，是因为雇主的生产率基准是基于一些不再存在的东西。

> 在旧世界，我们通过朝九晚五坐着来衡量生产力。我们朝九晚五地来上班，除了工作你还会做什么？我自己仍然是这种心态，因为我是老一辈的人。当我被即时消息轰炸时，我发现这非常困难，我发现我只是骨折，但那只是我。但是经常发生的是，我们把自己的经历翻译成‘好吧，我做不到，所以别人也做不到。如果他们在做，那一定意味着他们没有集中注意力。"
> 
> “我们实际发现的是，这些孩子伴随着它成长；他们已经学会了如何在社交的同时工作，这就是他们在找到第一份工作之前做学校作业时所做的。”
> 
> “我认为最重要的是文化正在发生变化。这并不是一个完全的改变，因为如果你记得朝九晚五的工作，你仍然会打电话给医生，因为下午 5 点以后你不能打电话给[手术室]，因为它关门了。或者你想打电话给商店，看看他们是否有一些东西。或者你可能会接到朋友的电话或者打个电话。问题是，当它可见时，它变得更加明显。”
> 
> *“在英国，有人被解雇了，因为他们在网上看报纸，做了这些被认为是错误的行为。法官实际上是支持这位员工的，他说:“这个人在网上做的事情和我们一直做的事情有什么不同，那就是在休息时间看五分钟报纸。”。“我们打那个电话有什么不同？”所以他看不出区别。衡量生产率的标准是你生产了什么，而不是你如何利用时间。”*
> 
> ***来源** : [APC 启动](https://web.archive.org/web/20160327062410/http://www.apcstart.com/)*

一些公司，如 Virgin Money，决定以独特的方式正面解决这个问题，用新的“电子休息”取代传统的喝茶休息时间。除了午餐时间，该公司还推出了 15 分钟的员工“电子休息”时间。这意味着员工可以在每天上午 11 点去购物和处理其他在线个人事务。该计划是根据一项对 1223 名办公室职员的独立调查得出的，调查显示 56%的人在工作时间通过网络买卖商品。

在同一主题演讲中，微软澳大利亚技术社区组经理弗兰克·阿里戈说

> 人们对工作中有限的互联网接入感到非常沮丧，因此他们正在寻找自己的解决办法。人们越来越多地使用不容易被公司防火墙阻挡的匿名代理，将他们自己的无线宽带服务用于个人笔记本电脑或工作 PC，或者通过移动电话和 PDA 访问即时消息。

> 比尔·盖茨几年前就说过，如果你担心互联网的生产力，你担心的是有人从你的文具柜里偷钢笔……还有更重要的事情要担心

![](img/bf5cca873ca196690963ae7092d5ab1b.png)

最近我在一个客户网站上工作，这个网站几乎屏蔽了所有的互联网接入。沮丧的开发人员会带着他们自己的笔记本电脑，回到他们的家用电脑，以便获得完全免费的网络访问！

我知道需要在信任和控制之间取得平衡，但是您的公司会选择哪一种呢？它会阻止对合法网站的访问，从而影响工作效率吗？这需要长时间致电 it 部门来移除被阻止的访问。

希望你的电脑不会说不！
