# Wii 传感条黑客

![](img/5c45103c31a3c3bdfab9fe7333ff7c7a.png)

看起来 Wii 的感应条是靠 Wii 游戏机本身输出的 7.5 伏电压运行的。因此，一个勤奋的年轻人，如果他专心致志，可以从 Radio Shack 添加一个 DC 适配器。这正是 AVSForums 的人所做的，结果好坏参半。

问题是连接到传感条的电缆非常细，很难焊接。请注意，并非不可能，但很困难。看起来最好的办法是撬开传感条，铺设一条新的 DC 电缆。无论如何，讨论相当热烈，如果是初级的。

还有[这首小曲](https://web.archive.org/web/20130627212453/http://doctabu.livejournal.com/64758.html)，让你创建一个功能齐全的酒吧，可能有点不太吸引人——视频后跳转。

或者你可以[只用蜡烛](https://web.archive.org/web/20130627212453/http://crunchgear.com/2006/11/27/wii-works-with-candles/)。
