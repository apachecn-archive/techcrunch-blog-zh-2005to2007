# 索尼 NSC-GC1 上市

![nsc-gc1_3q_lcd_open_540×359.jpg](img/8d8222e341f6030cb190744a59081b72.png)

索尼不会错过最新的技术潮流，所以他们发布了 NSC-GC1，这款网络共享摄像头迎合了社交网络妓女和视频博主的需求。摄像头以 MPEG-4 格式拍摄视频，并以 QVGA/15fps 至 VGA/30fps 拍摄快照，同时拍摄 500 万像素的静态照片。“分享标记”本能地将你的视频和照片上传到网上。天啊。它超级光滑时尚，仅重 5 盎司，宽 1 3/16 英寸，高 4 1/8 英寸，深 2 英寸。如果您使用 2GB 卡的默认设置进行录制，CAM 可以在 Memory Stick PRO duo 卡上录制大约五个小时的不可下载内容。9 月份找，200 美元左右。
