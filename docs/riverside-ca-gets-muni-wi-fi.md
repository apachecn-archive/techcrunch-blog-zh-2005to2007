# 加州河滨市获得市政无线网络

![bfsep06p16b.jpg](img/9da3dfdc27035ea05bb64a3b77d0bab2.png)

我不确定加州河滨有多少 CrunchGear 读者，但如果你是，那么如果你在市区，靠近亨特科技园或亚当斯汽车中心，早上你会发现一个不错的惊喜。Riverside 将是美国电话电报公司首次涉足市政和公共使用的 Wi-Fi 无线宽带互联网服务。

广告支持的免费 Wi-Fi 目前覆盖 3 平方英里，最终将覆盖 55 平方英里的美丽河畔，提供免费和付费订阅服务。该市警察局将测试网络，在巡逻车之间运行视频，因为他们已经厌倦了在车站上传视频的繁琐任务。

美国电话电报公司和 MetroFi 将为低收入居民提供免费服务，但也将提供基于订阅的高速互联网接入服务，每天 7.99 美元，每周 15.99 美元。
