# 酒吧营和 OPML 路演 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/16/profile-bar-camp/>

在湾区度过一个多么美好的周末:

## 周五至周日的酒吧营:

对于我们这些今年没有被邀请参加[足球营](https://web.archive.org/web/20221129090059/http://wiki.oreillynet.com/foocamp05/index.cgi)的人来说，有一个看起来同样(或者更)令人兴奋的选择:[酒吧营](https://web.archive.org/web/20221129090059/http://barcamp.org/index.cgi?HomePage)。不需要邀请。

不是反 Foo 阵营，只是不一样。我期待着参加(并写博客)这个活动。正如罗斯·梅菲尔德[所说](https://web.archive.org/web/20221129090059/http://barcamp.org/index.cgi?FooCamp)，“重要的是当好人聚在一起时，伟大的事情就会发生。”

Socialtext 将捐赠其办公室和 wifi 的使用权。

罗斯·梅菲尔德:[酒吧营不反 FOO](https://web.archive.org/web/20221129090059/http://ross.typepad.com/blog/2005/08/band_camp.html)

## 周六 OPML 路演:

戴夫·维纳在《Geeks.opml.org》中的评论:

> 加州 OPML 路演，8 月 20 日，下午 7 点！
