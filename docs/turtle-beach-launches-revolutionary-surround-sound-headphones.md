# 海龟滩推出革命性的环绕声耳机

![surroundphones41.jpg](img/f995f3c724ee00fb972e75961beeeaa7.png)

你准备好革命了吗？你是吗？Turtle Beach 正在全力推出 Ear Force AK-R8，其目标是成为 PC 游戏耳机领域的佼佼者。这款耳机共有八个扬声器和双低音扬声器，都是专业级的，非常适合游戏玩家。USB 供电的主宰者提供真正的环绕声，这不是开玩笑的游戏。

一系列其他功能使 AK-R8s 成为严肃游戏玩家的必备耳机。它们不仅提供环境效果，而且当它们打开时，你还可以听到你周围发生的事情。可拆卸的吊杆麦克风可以根据您的具体需求进行调整，10 波段、6 声道均衡器也不差。总之，这似乎是完美的设置，还包括 5.1 或 7.1 模拟和数字输出。现在只需 149.95 美元就能买到一双。
