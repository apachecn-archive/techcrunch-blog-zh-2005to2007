# 有史以来最酷的光环 3 雕像

![](img/fb88afb5fdacd15bd9d999359c81bc7f.png)

随着《光晕 3》的发布越来越近，相关商品已经开始流出。以上是今年你会看到的四个最酷的*光环 3* 小雕像。由库布里克制作，这些令人敬畏的斯巴达小雕像有红色，蓝色，绿色和该死的活跃迷彩！每个雕像都有 2.5 英寸高，是从日本进口的，所以它们一定很贵，对吗？

纳达！18 美元可以让你从拥有 360 的邻居那里获得四件套和街头信誉。他们现在可以预订，所以拿出签证，开始收费。
