# 虚幻 3 引擎不来 Wii 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/06/unreal-3-engine-not-coming-to-wii/>

![](img/ced03888dad8ce90a160dc20e5f98a37.png)

随着 Wii 上的游戏看起来像 PSOne 的 Jetmoto，大多数消费者都希望假以时日，Wii 游戏会变得更好看一些。见鬼，也许一两年后，游戏会有类似战争机器的图像，对吗？不对！

Epic 声明《虚幻 3》引擎永远不会出现在任天堂 Wii 上。为什么？Epic 表示，该引擎不是为任天堂的硬件设计的。

因此，虽然我呼吁 Wii 粉丝为其劣质图形哭泣，但我建议，如果你想在主机游戏上看到一些疯狂的细节和极端的灯光，就拿起 Xbox 360 或 Playstation 3。因为，你知道，Epic 和那些游戏机一起完蛋了。
