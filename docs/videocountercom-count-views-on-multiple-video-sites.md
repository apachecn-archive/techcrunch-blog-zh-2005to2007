# VideoCounter.com:统计多个视频网站的浏览量

[![videocounter.png](img/5ce0cec8bbfad66622279a600d1cebb2.png)](https://web.archive.org/web/20210924025452/http://www.videocounter.com/MC/)[【VideoCounter.com】](https://web.archive.org/web/20210924025452/http://www.videocounter.com/MC/)允许用户追踪他们的视频每天在多个在线视频托管网站上被观看的次数。

使用 VideoCounter.com 很简单。用户只需注册一个免费账户，输入要追踪的视频，剩下的事情由 VideoCounter.com 来做。VideoCounter.com 提供了视频每天被观看的次数，以及视频在每个门户网站上被观看次数的统计数据。

支持的视频网站包括 YouTube、Revver、Daily Motion、MySpace、iFilm 等。

Chris Pirillo [最近](https://web.archive.org/web/20210924025452/http://chris.pirillo.com/2007/06/29/video-ads-in-web-video-success/)发帖称，在整个视频中出现赞助商标志的覆盖广告是视频广告的终极形式。这是一个公平的要求，但缺少的组件是一个简单的方法来跟踪多个网站上的视图。VideoCounter.com 并不跟踪下载量，但它确实将多点视频服务的统计数据整合在一起。这些统计数据对于追踪广告曝光率或者仅仅是为了好玩而知道是有用的。
