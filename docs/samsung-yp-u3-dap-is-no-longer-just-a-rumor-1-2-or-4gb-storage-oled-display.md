# 三星 YP-U3 DAP 不再只是谣言:1、2 或 4GB 存储，有机发光二极管显示器| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/18/samsung-yp-u3-dap-is-no-longer-just-a-rumor-1-2-or-4gb-storage-oled-display/>

[![samsungu3off1.jpg](img/5a629703db4dd6b4d64bfaf4a0434fee.png)](https://web.archive.org/web/20170608050958/http://old.crunchgear.com/wp-content/uploads/samsungu3off1.jpg "samsungu3off1.jpg")

信不信由你，三星确实为它制作了一些精致的 DAP，但在一个充满 iPod 和 iPod 配件的世界里，很难注意到它们。几天前第一台[与我们的](https://web.archive.org/web/20170608050958/http://crunchgear.com/2007/05/07/samsung-yp-u3-specs-leaked/)相遇的 U3，看起来将同时挑战存储容量为 1、2 和 4GB 的 [iPod shuffle](https://web.archive.org/web/20170608050958/http://crunchgear.com/2006/11/01/ipod-shuffle-2-hands-on/) 和 [iPod nano](https://web.archive.org/web/20170608050958/http://crunchgear.com/2007/06/05/major-in-mac-get-a-free-ipod-nano/) 。与 shuffle 不同，U3 实际上有一个显示器，即使它只是一个四线显示器。而且，像大多数低端 DAP 一样，不需要使用任何特殊的软件来加载音乐:只需拖放播放器能够容纳的深度即可。

最高容量机型的最高价格为 150 美元，这应该会让那些希望物有所值的人感到高兴。但我想这有点像校园法则:如果你没有[品牌 x]，那么你就不酷。不，过了 11 岁就说不通了。
