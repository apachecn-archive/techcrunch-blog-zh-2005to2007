Web 2.0的公司，第2部分

这是第二批参加“网络2.0大会”的公司。参见第1部分。

我的朋友伊桑·斯托克向我展示了昨晚发布的T4 Zvents。我们在这里写过事件，在这里也写过事件。简而言之，Zvents可以帮助你创建和定位成千上万个每月的本地事件，并且拥有大量令人敬畏的ajax、标记和其他web2.0东西。

罗恩·拉斯穆森谈到了KnowNow，一个有趣的基于RSS的提醒系统(他们称之为“elerts”)。我想更好地理解这个问题，希望本周能和罗恩坐下来谈谈。

伊恩·麦卡锡给我们做了一个关于 Orb 的演示，它可以让你在不需要任何硬件的情况下，将家里电脑上的内容流式传输到任何wifi设备。对于视频、照片等，它的非常好用。他甚至在自己的客厅拉起一个视频摄像头，用Orb打开电灯。很好。不过，这只是PC业务。

迈克尔·坦纳今天解除了对Wink的密码保护，我们终于可以看一看了。Wink是“以人为本的搜索”，我认为他们正在寻找更强大的工具。他们把基本的搜索结果，并允许人们标签和排名，他们创造一个更好的结果集。他们把自己的技术称为“tagrank”。

该死的，迈克尔，回复我的邮件，明天给我一个面试机会。:-)

马修·格特纳在 allpeers 上展示了一个在firefox上开发应用程序的开放平台。Allpeers目前处于私人测试阶段。

巴特·德洛姆做了一个Flock演示。关于Flock我还能说什么？我喜欢这种不自然的方式。如果他们能找到一种方法将Pandora直接集成到Flock浏览器中，我就再也不会离开我的电脑了。

但说真的，我已经拿到了新版本，并将在本周做一个完整的简介。

创始人鲍勃·怀曼谈到了PubSub、结构化博客和他们的新 LinkRanks 产品，我们在这里写了这个。更多关于PubSub的信息，我们最喜欢的潜在搜索引擎。
