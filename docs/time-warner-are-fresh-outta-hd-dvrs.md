# 时代华纳刚刚卖完高清数字录像机

![](img/94fe0920e315f9f127666d61644b4a6c.png)

正如 Biggs 在他的 [Wii rant/post](https://web.archive.org/web/20210307094306/http://crunchgear.com/2007/01/26/one-mans-wii-pinion/) 中提到的，TiVo 很烂，因为对直接来自有线电视提供商的 DVR 有很高的需求。事实上，需求是如此之高，以至于时代华纳有线电视公司跟不上需求，高清数字录像机盒子已经卖完了。

那些担心超级碗的人可以松一口气了。时代华纳说，它有很多没有 DVR 功能的普通高清盒子，所以你至少可以看到精彩的高清比赛——你只是不能 DVR，这样你就可以在涂着凡士林的时候以慢动作观看所有的马屁。
