# 幻影再次滑落

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/08/phantom-slips-yet-again/>

![](img/b70c60ee8eb3c765bc8e4c5c4f4c1663.png)

正当事情看起来可能会为蒸汽友好型[幻影、](https://web.archive.org/web/20201022071613/https://crunchbase.com/organization/phantom) 稍微振作起来时，它宣布其备受吹捧的笔记本电脑的发货已经推迟。你会记得 lapboard 是 Phantom 命运多舛的游戏系统中出现的一个组件。

实际上，比格斯[上个月晚些时候有机会检查了 lapboard](https://web.archive.org/web/20201022071613/http://crunchgear.com/2006/10/30/phantom-lapboard-and-alienware-entertainment-center-in-the-wild/) 。他似乎对此持谨慎乐观的态度，所以我猜测那里确实有产品。也许问题不在于产品，而在于其极其无能的管理。

然而最终，这是一个悲惨的故事。请记住，在其存在的六年里，幻影娱乐公司(前身为 Infinium Labs)还没有生产和销售过一个产品。我们知道 lapboard 是真实的，但谁知道我们是否真的能看到它。
