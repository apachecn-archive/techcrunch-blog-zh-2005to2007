# 诺基亚 Wisair 无线 USB 照相手机| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/08/wisair-nokia-wireless-usb-camera-phone/>

试图将其超宽带(UWB)和无线 USB 解决方案融入每一款消费产品的芯片组制造商之一 Wisair 在 3GSM[展示了无线 USB 在照相手机中的应用](https://web.archive.org/web/20151201053242/http://crunchgear.com/?s=3GSM)。

使用嵌入了 Wisair 542 芯片的诺基亚 N 系列手机，该公司将向 WiMedia Alliance area 的参观者展示(我们其他人只能相信他们的话)当你可以使用无线 USB 将图片和视频从手机传输到电脑上时，这将是多么棒的事情——以无线方式。假设运营商没有锁定这些功能来阻止你使用它。
