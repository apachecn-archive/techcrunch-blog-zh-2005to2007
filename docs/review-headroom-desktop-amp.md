# 回顾:桌面放大器扩展空间

![](img/4dd1c9270a1ab1a0d2a426249c8d0a3e.png)

如果你碰巧有一些昂贵的耳机，一个好的耳机放大器真的可以在仅仅令人印象深刻的声音和真正的音响发烧友荣耀之间产生差异。进入 [HeadRoom 桌面放大器](https://web.archive.org/web/20210412001211/http://www.headphone.com/products/headphone-amps/the-desktop-line/headroom-desktop-amp.php)(599 美元)，这是一个相对紧凑的耳机放大器，具有一系列输入和输出选项以及一些热门的内部组件。我最近有机会用一套森海塞尔 HD650s 和 Etymotic ER4S canalphones 进行了测试。这是真相。

哑光黑色金属机身在桌子上不占太多空间。顶部有点凸起，但是你仍然可以把东西放在上面而不会掉下来。输出和输入都在背面，只有一对耳机输出在正面(四分之一英寸和八分之一英寸)。前面的绿色 LED 告诉你放大器打开了。

由于我已经通过 M-Audio Transit USB(它也有一个录音输入)输出了我的 PC 音频，我只是将 Transit 的光学输出连接到 amp 的光学输入。我为我的 iPod 使用了一个模拟输入，为我的 M-Audio Studiophile DX4 监听音箱使用了模拟输出。

在背面，信号源选择器开关可让您选择模拟或数字输入，而数字输入选择器可在三种不同的数字输入类型(USB、光纤、同轴电缆)之间切换。模拟输入选择器允许您在两个模拟输出之间进行选择。起初，我发现自己希望所有的开关都在前面，但一旦我通过感觉记住了后面的开关，我就克服了它。

在前面，除了电源开关，两个耳机插孔和大黑色音量旋钮，还有一排四个开关。其中一个可以激活后部输出，这样你就可以轻松地在耳机和扬声器之间切换，这可能是桌面放大器最有用的功能。其他三个开关以一些非常漂亮的方式改变声音，这取决于源材料和听音设备。

亮度滤波器提供两个高音增强器和一个关闭位置；大多数纯粹主义者会关闭这一设置，尽管我发现滤波器对于打出一些声学爵士乐轨道来使骑钹和高帽真正歌唱很有用。增益选择器允许您根据输出源的阻抗来设置输入电平，使用高设置来驱动大型扬声器，使用中或低设置来驱动大多数耳机。

crossfeed 开关使用 HeadRoom 专有的 crossfeed 电路，该电路可以改变声场的形状，以适应耳机与耳朵的接近程度(与扬声器相比)。我的总体印象是交叉馈送通过将中音带到你的脸上来温暖声音。一些轨道真的受益——面向人声的曲调和任何具有严重声相效果的东西——而原声乐器轨道在没有它的情况下听起来更真实。

总的来说，桌面放大器在饱满度、平衡度和细节方面改善了我的森海塞尔 HD650 和 Etymotic ER4S 耳机。众所周知，用耳机很难实现 Soundstaging，但森海塞尔和桌面的结合真的让我忘记了我正在听耳机(只要我保持头不动)。Etymotic canalphones 听起来很棒，但我用它们定位声音有点困难。

如果你买了一辆法拉利，你不会给它装上光头胎，那么为什么要买一对没有像样耳机放大器供电的漂亮易拉罐呢？如果你对花 849 美元买台式放大器不感兴趣(带 DAC 选项，给你数字输入；基本版 599 美元)，你至少应该看看 HeadRoom 的低端放大器，比如非常便携的可选电源(399 美元)——总共大约 1800 美元。
