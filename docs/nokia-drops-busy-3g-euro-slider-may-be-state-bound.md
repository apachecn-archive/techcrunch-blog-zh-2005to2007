# 诺基亚放弃繁忙的 3G 欧元滑块，可能是国家限制

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/21/nokia-drops-busy-3g-euro-slider-may-be-state-bound/>

今天，诺基亚在欧洲发布了一款新手机，我们应该会看到它在美国的 Cingular 上市。6288 是一个小滑块，充满了主要的 3G 功能，并有黑色或白色(你好，苹果现在使用彩色，咄)。

你从 UMTS 和 QVGA 320 x 240 屏幕开始，然后在后面添加一个带闪光灯的 200 万像素摄像头，给它专用的变焦和照片控制，添加一些蓝牙 2.0 和 MiniSD 插槽，你就有了一个 spicey 肉丸，或者至少是一个热辣的相机手机。但诺基亚决定超越这一点，为这个强大的设备配备了第二个摄像头，前面有一个 VGA 用于视频会议，然后包括一个易于使用的视频编辑功能，用于完美的彩信。滑头。

6288 是由 Cingular 暂时提供的 6282 的欧洲升级版，在补贴和折扣之前，包括 512MB 存储卡在内的价格应该在 415 美元左右。因为 Cingular 从未推出过 6282，所以期待这家伙明年在 Q1 以 299 美元左右的价格给我们带来惊喜。
