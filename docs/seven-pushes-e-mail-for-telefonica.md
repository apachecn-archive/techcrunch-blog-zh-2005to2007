# SEVEN 为西班牙电信推送电子邮件

使劲，使劲推。这是推送电子邮件软件开发商 SEVEN 的话。今天，SEVEN 宣布推出适用于包括诺基亚 N95 在内的各种手机和智能手机的推送电子邮件软件，通过西班牙电信的移动部门提供邮件 Movistar Empresas 和邮件 Movistar Profesional 服务。这将为用户提供通过移动电话实时访问其公司和个人邮件账户的便利。

SEVEN 的首席执行官罗斯·博特(Ross Bott)表示:“西班牙电信的多样化手机和智能手机产品为其用户提供了移动享受。“将这些时尚的手机与 SEVEN 通过 Movistar 提供的移动电子邮件服务相结合，客户可以随时随地保持联系。作为世界上最大的推送电子邮件软件提供商，我们致力于提高用户的连接性并增加他们的移动电子邮件访问。”
