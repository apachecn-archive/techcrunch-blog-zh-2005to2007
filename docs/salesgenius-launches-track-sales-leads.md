# SalesGenius 启动:跟踪销售线索

 [](https://web.archive.org/web/20220521051720/http://www.genius.com/) [Genius，Inc.](https://web.archive.org/web/20220521051720/http://www.genius.com/) 今晚推出他们的第一款产品: [SalesGenius](https://web.archive.org/web/20220521051720/http://www.genius.com/products/salesGenius.html) 。它面向销售和营销团队，允许这些专业人员在其公司网站上跟踪销售线索的活动。WebEx 前高管、首席执行官大卫·汤普森上周给了我一个演示。

许多分析包已经提供了跟踪网站用户行为的能力。SalesGenius 的不同之处在于，它完全不需要公司方面的 it 集成，而是将行动与特定的销售线索联系起来。他们用来做这件事的方法实际上相当聪明。

如果您是销售专业人员，您可以使用 SalesGenius Outlook 插件或基于 web 的电子邮件产品向销售线索发送电子邮件。任何返回到您公司网站的 URL 都会自动转换为 Genius URLs，这些 URL 通过代理提供服务，并允许 Genius 跟踪他们在网站上的点击和后续操作。然后，销售专业人员可以使用 Genius 提供给他们的附加信息联系销售线索。

这不是一个面向消费者的应用程序，但是很多销售人员阅读这个网站，我认为这可能与他们非常相关。基本产品是每个用户每月 49 美元，包括 30 天的免费试用。

这项服务的某些方面会引发隐私问题。想了解更多，请看丹·法伯(Dan Farber)和雷夫·尼德曼(Rafe Needleman)的文章，丹·法伯说这“不完全是天才之举，但很有用”。
