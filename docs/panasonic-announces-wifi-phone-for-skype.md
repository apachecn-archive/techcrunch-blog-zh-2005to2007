# 松下宣布为 Skype 提供 WiFi 电话

![](img/73187c24d864157495ef082d4dc4b554.png)

松下通信公司正在开发一款用于 Skype 的 WiFi 手机。WiFi 电话将允许 Skype 用户从任何 WiFi 接入点向其他 Skype 用户发送和接收电话。到 2006 年第四季度，WiFi 电话核心还将具备 SkypeOut、SkypeIn、Skype Voicemail 和呼叫转移功能。这款手机不需要浏览器认证，它将通过 Skype 认证。
