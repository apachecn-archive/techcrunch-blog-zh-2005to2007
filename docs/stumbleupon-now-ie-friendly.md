# 现在是友好的

广受欢迎的火狐扩展软件[今天早上,](https://web.archive.org/web/20220702144242/http://stumbleupon.com/)发布了一个面向网络浏览器用户的工具栏。这项服务可以让你浏览朋友和其他与你兴趣相投的用户推荐的网站。用户也可以写网站评论。**最终结果是一个非常引人注目的用户体验，现在它与市场主流浏览器一起工作，很可能会爆炸式增长。**

该公司由加勒特·坎普、杰夫·史密斯和贾斯汀·拉弗朗斯领导，并获得了谷歌董事会成员拉姆·施拉姆、Lotus 创始人米切尔·卡普尔和其他几人的投资。针对特定位置和人口统计组的用户的页面最多向每个用户显示一次，每个目标访问者的费用为 5 美分。付费展示的页面都有明确的标记。

该公司告诉我，现在通过 StumbleUpon 每天平均有 230 万个页面被浏览。迄今为止，FireFox 扩展已有 230 万次下载，600 万个网站通过该系统进行了审查。支持 14 种语言。该公司出现在 TechCrunch 主办的[SuperNova](https://web.archive.org/web/20220702144242/http://www.beta.techcrunch.com/2006/07/03/the-supernova-12/)互联创新者展示中。

对于许多人来说，这正是那种可能让网络再次变得有趣的东西——包括对 IE 的支持是关键。我知道我通过 RSS 提要、 [TechMeme](https://web.archive.org/web/20220702144242/http://techmeme.com/) 、 [Digg](https://web.archive.org/web/20220702144242/http://digg.com/) 等找到了大量高质量的内容。但对世界上大多数人来说，StumbleUpon 将会满足他们对团体精选的高质量内容和在线社区的迫切需求。我相信我也会抽时间多陪陪它。
