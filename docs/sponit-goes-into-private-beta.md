# Sponit 进入私有测试版

最近，我和一位名为 [Sponit](https://web.archive.org/web/20210116141137/http://www.sponit.com/) 的创始人进行了交谈，这项新服务将在下个月左右推出。由三名加州大学伯克利分校的学生创建的 Sponit 是一个创新的群体“博客”平台，令人相当上瘾。

正如我以前说过的，我是一个封闭测试版的吸盘，给我们你的电子邮件页面。但我也有机会测试 sponit。它仍处于非常早期的阶段，但它是一个有趣的应用程序，并增加了整个 web 2.0 的讨论。
