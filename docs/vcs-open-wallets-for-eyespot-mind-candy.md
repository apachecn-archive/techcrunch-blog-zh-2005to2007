# 风投们打开钱包寻找眼中钉，心灵糖果

这是 Web 2.0 融资公告的重要一天。今天早上，我们发布了 GotVoice 以 300 万美元结束首轮融资的消息。现在，总部位于圣地亚哥的在线视频编辑服务公司 [Eyespot](https://web.archive.org/web/20220626074613/http://eyespot.com/) 宣布结束其第一轮融资。这笔资金由硅谷公司 Gabriel Venture Partners 牵头，总额为 370 万美元。 [VentureBeat](https://web.archive.org/web/20220626074613/http://venturebeat.com/2006/10/25/eyespot-gets-37m-for-video-mashing-service/) 似乎已经打破了这个故事。

[![eyespot logo](img/255050814bf8e49035b695f64ae0fff8.png) ](https://web.archive.org/web/20220626074613/http://eyespot.com/) Eyespot 直接向消费者提供拖放式 flash 视频编辑，并与 Blip.tv 和 Veoh 等公司合作[。竞争对手](https://web.archive.org/web/20220626074613/http://www.beta.techcrunch.com/2006/08/15/eyespot-partners-with-bliptv-veoh-to-provide-online-video-editing/) [Jumpcut](https://web.archive.org/web/20220626074613/http://jumpcut.com/) 被雅虎[收购！上个月。从手机铃声到视频短片，在线多媒体编辑显然被认为是 YouTube 一代的下一步。迈克尔·埃斯纳和 Overture 的上一任首席执行官泰德·梅塞尔是 Veoh 的董事会成员。Blip.tv 是 CNN 新公民视频计划的基础。像这样的合作关系让 Eyespot 看起来像是 VideoEgg，这家公司为 Bebo 和 Dogster 等社交网站提供基于浏览器的视频捕捉。据 Liz Gannes 报道，Eyespot 还与狮门娱乐、Current TV、Zomba/Jive、TVT、哥伦比亚、Epic、Island Records 和 Concord Records 有合作关系。多好的名单啊！](https://web.archive.org/web/20220626074613/http://www.beta.techcrunch.com/2006/09/27/yahoo-has-acquired-jumpcut/)

[![](img/4995e2edc4ff5c3d9d50c5e1a6909ce6.png) ](https://web.archive.org/web/20220626074613/http://www.mindcandydesign.com/people/index.html) [ TechCrunch 英国](https://web.archive.org/web/20220626074613/http://uk.beta.techcrunch.com/2006/10/25/mind-candy-announces-7m-in-funding/)今天也爆出了一个融资故事。在线游戏公司 [Mind Candy](https://web.archive.org/web/20220626074613/http://www.mindcandydesign.com/people/index.html) 宣布从 Accel Partners、Index Ventures 和 NewMedia Spark 获得了 700 万美元的投资。《心灵糖果》制作了一款非常受欢迎的游戏，名为《困惑之城》。TechCrunch 英国公司的 Sam Sethi 表示，这款游戏融合了真实世界的事件、网站、短信、电视、印刷品和各种其他媒体，创造了一种身临其境的游戏体验。有关该公司及其基于这笔资金的扩张计划的详细信息，请参见 Sethi 的报道。
