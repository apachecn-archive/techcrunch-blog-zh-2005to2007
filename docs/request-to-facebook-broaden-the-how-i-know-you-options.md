# 对脸书的请求:拓宽“我如何认识你”的选项

![](img/57ee51fa00dd05cee08820a75760436f.png)

当你在 facebook 上确认一个朋友时，你会被要求陈述你是如何认识这个人的。但是选择实在是太窄了。“我们勾搭上了”很少是描述你通过博客、网络等认识的人的恰当方式。事实上，大多数选择对于涌入这项服务的数百万非大学生成年人来说并没有真正的用处。是时候添加更多选项了。

有 14 个选项可供选择(见上图)，但我从未找到一个能恰当描述我的友谊的选项。[戴夫·维纳](https://web.archive.org/web/20220926102830/http://www.scripting.com/stories/2007/05/29/easyUpgradeToFacebook.html)提出了许多新的选择，包括“欣喜于”和“幻想于”。

我认为应该有一个可以添加自由文本的框。选择应该包括成人友好的选项，如“在会议上相遇”或“投资者”

你怎么想呢?你会选择什么选项？
