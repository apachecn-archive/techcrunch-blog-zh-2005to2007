# 闪迪金存储卡大赛

由于 Sandisk 不生产周围有辣妹照片的杀手级产品，他们选择了下一个最好的东西:黄金。在日本，Sandisk 正在为购买 Extreme IV 8GB CompactFlash 或 Ultra ll 4GB SDHC 卡的消费者举办一场比赛。如果你买了一个特别标明的包装，你就可以赢得一张纯金版的卡。我知道，非常令人兴奋。总比哪天见迈克尔·乔丹强。

CF 卡在美国价值约 5000 美元，而更小更薄的 Ultra II SDHC 卡售价约 2300 美元。由于它们不起作用，无论谁赢得它，都可能在一周内失去它，因此将它卖给购买黄金的人可能是他们的最佳选择。
