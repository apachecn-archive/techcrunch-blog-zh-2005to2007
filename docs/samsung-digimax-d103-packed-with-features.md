# 三星 Digimax D103 功能丰富

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/21/samsung-digimax-d103-packed-with-features/>

![](img/fec3adb7db262080ec6be26285739db2.png)

三星即将推出一款功能丰富的相机，那些想要升级的人可能会感兴趣。虽然没有确切的上市日期或设定价格，但 Digimax D103 值得坚持。这个 1010 万像素的庞然大物配备了 3 倍光学/10 倍数字变焦，2.5 英寸液晶显示器，PictBridge，4 厘米微距(非常适合那些特写)，漂亮的 MPEG-4 电影录制功能，ISO 1000，以及获得独特镜头的效果。

D103 前面还有一个把手，有助于在拍摄时保持相机稳定。它甚至有多种模式可供选择，适合各种可能的拍摄。这个设计并不是最酷或最时髦的，但有了 1010 万像素，谁在乎呢？
