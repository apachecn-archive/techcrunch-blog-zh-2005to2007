# YouTube 本地化了 9 个国家特定的版本

[![](img/c3fd7446f034fca4e919a12457cb3f5e.png) ](https://web.archive.org/web/20221002101328/http://www.youtube.com/) [YouTube](https://web.archive.org/web/20221002101328/http://www.beta.techcrunch.com/tag/youtube) 将于[今日推出](https://web.archive.org/web/20221002101328/http://googleblog.blogspot.com/2007/06/youtube-in-9-more-domains.html)YouTube 的九个国家特定版本。

YouTube 本地版本将适用于巴西、法国、爱尔兰、意大利、日本、荷兰、波兰、西班牙和英国，包括 youtube.fr 和 youtube.jp 等相关国家的特定域名。

新网站包括完全翻译的网站内容，并将最终提供特定国家的流行内容。

新网站将作为一个选项提供给所有 YouTube 用户，尽管 YouTube 将通过提供本地版本的直接链接来鼓励本地用户进行转换。

YouTube 在推出国家的广播合作伙伴包括 BBC、France 24、西班牙 Antena 3 和 Cuatro TV、葡萄牙 RTP、荷兰 VPRO 和 NPO，以及许多足球(足球俱乐部)和非营利组织。

提供 YouTube 本地化版本的唯一不同寻常之处在于，这是前所未有的。迄今为止，YouTube 取得了惊人的成功；向非英语观众开放该网站将推动新一轮增长，进一步巩固 YouTube 作为互联网视频点播领先目的地的地位。
