微软办公室圆桌会议:解决昂贵的电话会议？

Microsoft Office圆桌会议是一种桌面设备，提供同步语音和视频会议。最酷的是，从视频的角度来看，它产生了“一个360度的全景视频，并列显示了参加会议的每个人的图像。”然后，该系统可以识别并聚焦正在发言的个人，以便在会议中发表评论时获得关注。

圆桌会议与Office Communications Server 2007和Office Live Meeting协同工作，因此会议参与者可以在一个同构环境中集成虚拟演示、共享白板和传输文件。该系统还记录会议，以便用户可以回去回顾发生了什么。

圆桌系统目前正在微软校园内进行测试，并将很快进入微软技术采用计划的beta测试阶段。该系统预计将于2007年中期上市，价格不到3000美元。
