# 索尼爱立信 Z310 外观漂亮，功能强大

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/06/sony-ericsson-z310-comes-with-good-looks-good-features/>

![](img/3484d7c324fe61383383ff6535c17c67.png)

我会把它交给索尼爱立信:他们制造了一些很棒的手机，Z310 肯定是其中之一。这款翻盖手机正面采用镜面处理，有黑色、青铜色和粉色三种颜色。集成的图标显示出现在手机的前面，类似于索尼的 8GB MP3 随身听。

Z310 的技术规格相当不错。为 EDGE 做好准备，支持 HTML 的网页浏览，一次充电 7 小时通话时间/300 小时待机时间，蓝牙，MP3 播放，接听电话时的环境光效果，使用 VGA 摄像头的照片博客(他们什么时候才能学会？！)，红外，和 IM。

对于我所猜测的入门级手机来说，Z310 的设计相当不错。如果只有公司可以了解 VGA 相机是如此 2004 年，那么这款手机可能已经是完美的。哦，好吧，这仍然是一个拉德手机。Z310 计划于 2007 年在 Q1 上市，但价格尚未确定。如果你想升级，请在新的一年里睁大眼睛寻找这颗宝石。
