米波令人印象深刻的增长——技术危机

原文：https://web.archive.org/web/http://techcrunch.com/2005/12/12/meebos-impressive-growth/amp/

米波 ( 原创简介)显然将筹集风险资本和/或在不远的将来被收购。我昨天和创始人塞斯·斯特恩伯格谈到了网站的增长，他透露了一些令人印象深刻的数据。

Meebo 现在每天平均有250，000次登录。12月7日，即发布12周后，Meebo成功登录236，000次，发送了6，534，948条信息，传递的信息总数约为13，069，896条。

Meebo还询问用户在哪里使用该产品，以及为什么。结果是:

你在哪里使用meebo？

工作:59.7%
学校:36.8%
家庭:30.0%
网吧:7.3%

你为什么用米波？(自由形式答案编码)

否则IM难以/无法访问:33%
Web IM优于软件IM: 18%
meebo更容易:11%

我知道事情有些不对劲，因为我在最初的meebo帖子中收到了狂热的评论，大部分是学生们乞求找到绕过学校封锁Meebo网站的方法。这里只是一个的例子:

我不能用米波。
我想访问雅虎和hotmail邮件。你能通过我检查邮件找到另一个网站吗？

我急需它。
