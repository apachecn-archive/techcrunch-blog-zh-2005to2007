# Zune 发布日令人头痛

![](img/7415737c21018d3b6a97ca37bd6fa6f0.png)

只是在这里安装旧的黑色 Zune，伙计们，我在这里说，Zune 市场要么是运行在一个旧的 386 完成 Windows 98 的副本，要么有人要做一些解释。像 Engadget 的 Ryan 一样，我在一台相当新的 Gateway 笔记本电脑上完成了 Zune 安装 rigamorale，现在正在耐心等待市场的出现。我在美国东部时间今天早上 9:25 开始尝试，但我仍然没有下载我的由“怪异”艾尔·扬科维奇写的[白人和书呆子](https://web.archive.org/web/20210303055237/http://crunchgear.com/2006/09/28/white-and-nerdy/)。

我已经重启了三次，每次运行 Zune 应用程序时，它都会重新设置一遍，然后在市场上死机。让我们祈祷这些是开幕日打嗝。
