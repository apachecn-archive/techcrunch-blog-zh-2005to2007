# 索尼爱立信 Z610i，带隐藏式外部显示屏

![sez610.jpg](img/6dc1e6391554cbcb8e48dbad2108e0fb.png)索尼爱立信 Z610i 翻盖手机不仅拥有镜面光洁度，使其成为索尼爱立信近年来最闪亮的手机之一，其外部还有一个隐藏的“魔镜”显示屏，只有在使用时才能看到。我们不知道那是做什么用的，除了炫耀，但它看起来很酷。

这款手机有一个 200 万像素的摄像头，支持推送电子邮件和 RSS，2.0 英寸的内置屏幕，用于扩展内存的记忆棒微型插槽，从 2006 年第三季度开始有黑色、粉色和蓝色可供选择。
