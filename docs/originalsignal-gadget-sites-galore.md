# 原始信号:小工具网站丰富

![](img/feb938eb0ed33e634f947e5a064e87ad.png)

我采访了刚刚推出 T3 小工具的 T2 still point Media 的 Tako Steinz。OriginalSignal ，一个小工具网站的测试聚合器。现在，它搜索了 15 个主要网站，包括 Crunchgear 和 T6，并在你每次访问页面时报告新条目的出现。你可以把它想象成一个 NASA 的“大棋盘”式系统，而不是 NetVibes。

Tako 和他的 5 人团队想要一个非常基本的界面，并正在计划一些有趣的“信号”,包括政治和新闻。他们在几天前发布了主网站，现在我们说话的时候，他们正在给它添加“信号”。然后，他们将捕捉点击数据，创建一个中央仪表盘，显示当天最受欢迎的故事。它现在非常基础，但仍处于测试阶段，似乎是一个足够好的想法，值得再看一眼。
