# 东芝准备在高清格式大战中反击

东芝对索尼在高清格式大战中获胜不太高兴。东芝 HD DVD 公司的副总经理 Olivier Van Wynendaele 称蓝光支持者的说法是“宣传”。Wynendaele 对每一项主张进行了分析，并为每一点提出了合理的论据。

蓝光光盘协会上周表示，他们的格式在美国的销量是 HD 3:1。Wynendaele 认为 PS3 的拥有者应该对此负责，这不是因为他们购买光盘。他们只是用优惠券兑换索尼的免费蓝光光盘。他还表示，相比已经售出的 3 万台蓝光播放器，东芝已经在美国售出了 20 万台 HD DVD 播放器。Wynendaele 认为 [PS3](https://web.archive.org/web/20201129111519/http://crunchgear.com/2007/01/26/the-playstation-3-is-the-best-console-ever/) 的销量不应该被计算在内，因为不能保证他们会为自己购买蓝光光盘，或者他们甚至根本不会去看蓝光光盘。平心而论，这两种格式都不能声称胜利，因为高清播放器没有足够的市场份额，不到 1%，Wynendaele 也同意这一点。

索尼公司预计年底推出的[降价](https://web.archive.org/web/20201129111519/http://crunchgear.com/2007/03/05/more-sony-blu-ray-price-cuts-coming/)和更便宜的[蓝光播放器](https://web.archive.org/web/20201129111519/http://crunchgear.com/2007/02/26/sony-announces-600-blu-ray-player-raises-eyebrows/)怎么样了？东芝似乎对此也有答案。他们借鉴了沃尔玛的做法，计划不惜一切代价削弱索尼。也许这场战争还没有结束。
