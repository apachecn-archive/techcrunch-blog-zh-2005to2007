# 三星首次亮相折叠暴行，激怒了全知

这款来自三星的新设备在上帝面前令人厌恶。在 Duder-onimy 的某个地方，上帝愤怒地说，笔记本电脑不应该和手机躺在一起，就像手机和 PDA 躺在一起一样。这不是上帝的意图，读你的圣经。此外，你会发现没有提到鸭嘴兽，同样令人厌恶。

不过，这还算不错。

P9000 是一款超便携的 origami 风格的 Windows XP 笔记本电脑，配有 30GB HD、256MB RAM 和 1GHz Transmeta 处理器。听起来就像是另一个花哨的超便携设备，但想想它内置的语音 CDMA、数据 EV-DO 和移动 WiMax。加上蓝牙支持和 130 万像素的前置摄像头，你就有了一个一半是 PC/一半是手机混合/一半是怪物(这有太多一半了)，真的不能被称为智能手机(见:怪物)。

在明年年初的某个时候，在 SoKo(上帝说，韩国是非精英的地方)寻找这种被提的迹象，并为他们永恒的灵魂祈祷。
