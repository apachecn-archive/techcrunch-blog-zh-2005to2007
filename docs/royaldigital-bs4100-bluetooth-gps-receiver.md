# RoyalDigital BS4100 蓝牙 GPS 接收器

这款蓝牙 GPS 接收器通过蓝牙工作，这意味着只要你有一个支持蓝牙的设备——手机、PDA 或笔记本电脑——你就可以在任何地方获得 GPS 数据。通过在这些移动设备上安装合适的软件，您甚至可以将这种组合用作导航系统。

BS4100 本身的电池寿命为 16 小时，3 个 led 指示灯用于指示其状态，并可以通过 USB 端口供电。它甚至配有车载充电器和防滑垫，所以你可以把它安装在你的仪表板上。他们的网站上没有定价信息，所以你必须发电子邮件来报价。
