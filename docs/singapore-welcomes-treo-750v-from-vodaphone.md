# 新加坡欢迎沃达丰的 Treo 750v

等一下。什么？“V”不代表沃达丰？很可能不是。Palm 和微软上周末推出了 Treo 750v，这是其进入亚洲市场的第一款 Windows Mobile 手机。Treo 750v 曾是沃达丰的独家产品，也是本季度的必备智能手机。

沃达丰旗下的 M1 将于 10 月推出 UMTS Treo。我们仍在等待 Cingular 跟上它，说出那些神奇的“现在可用”的话，但在那之前，我们仍然是最后一个没有天线 Treo 的主要智能手机爱好者大陆。

尽管如此，我们仍然在手枪上领先世界！
