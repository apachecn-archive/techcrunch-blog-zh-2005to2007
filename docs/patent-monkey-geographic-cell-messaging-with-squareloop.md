# 专利猴:使用 SquareLoop 的地理细胞信息| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/02/patent-monkey-geographic-cell-messaging-with-squareloop/>

本周，初创公司 [SquareLoop](https://web.archive.org/web/20151001121845/http://www.squareloop.com/) 凭借其地理预测技术击败朗讯-阿尔卡特的 GMS，在 CTIA 获得[一等奖](https://web.archive.org/web/20151001121845/http://www.ctia.org/media/press/body.cfm/prid/1682)。该公司成立于 2005 年，当时它从技术转让组织 [MITRE](https://web.archive.org/web/20151001121845/http://www.patentmonkey.com/PM/FolderID/7f7b651c4d6747a8ab783176e4d1e92c.aspx) 获得了专利技术的权利，该专利技术允许通过现有的手机网络向某个地理区域的用户广播信息，同时还确保了最终用户的隐私。这项技术的主要用途是允许地方和联邦政府通过现有的蜂窝网络向目标地理区域快速广播紧急信息。Squareloop 于 2006 年开始对[应急响应](https://web.archive.org/web/20151001121845/http://www.squareloop.com/images/stories/SquareLoop-Manassas%20Beta%20Press%20Release%20-%20FINAL.pdf)应用程序进行现场测试。

这家初创公司还开始与移动运营商建立联系，以提供更广泛的定位广告和内容服务。

此外，像这样的地理信息似乎可以与快速发展的服务很好地融合，如[躲避球](https://web.archive.org/web/20151001121845/http://www.dodgeball.com/)、 [Twitter](https://web.archive.org/web/20151001121845/http://twitter.com/) 或 [Swarmteams](https://web.archive.org/web/20151001121845/http://home.swarmteams.com/) ，因为这项技术有可能与私人群组信息相结合，发出大量通知。

最后，将地理博客的理念与类似 [Outside.in](https://web.archive.org/web/20151001121845/http://outside.in/) 的服务结合起来，互联网变得更接近真实的工作，因为它同时是全球性和地方性的。
