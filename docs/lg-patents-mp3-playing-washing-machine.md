# LG 专利 MP3 播放洗衣机

![](img/8c4cf2fe4736f91d8e6db108d8f82fc6.png)

下次你去购买新洗衣机时，你可能会关心地询问 LG 即将推出的可以播放 MP3 的洗衣机。LG 刚刚申请了世界上第一台 MP3 播放洗衣机的专利，我敢肯定，任何洗衣服的人都会高兴得跳起来。

为什么这么重要？去洗衣服，听几分钟机器的声音。那种嗡嗡的、响亮的、令人讨厌的声音很快就会过时*。LG 的洗衣机会让你立刻陷入可怕的困境。可惜价格可能会超过 3000 美元大关。如果这对你来说太贵了，拿把吉他，开始在洗衣房里摇摆吧。*
