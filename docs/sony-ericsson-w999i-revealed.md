# 索尼爱立信 W999i 透露

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/19/sony-ericsson-w999i-revealed/>

![](img/826706c93abf4e4c94b81040aafddd67.png)

就音乐手机而言，我相信索尼爱立信过去是，现在仍然是唯一一家做得好的手机制造商。随身听系列手机总是设计精美，功能丰富，并有很强的音乐播放能力。但现在这一切都成为了历史。为什么？看上面那个手机。这应该是传闻中的索尼爱立信 W999i 随身听手机。它看起来绝对令人惊叹，我相信它的规格也同样令人惊叹。

有消息称，这款手机将配备 HSDPA，2.4 英寸 262k 彩色屏幕，标准的 200 万像素摄像头，调频收音机，据称该设备只有 12.5 毫米厚。这张照片似乎也显示了索尼爱立信的新 QWERTY 垫。还没有发布日期，但如果你不是 iPhone 类型的男人或女人，这绝对是一件令人眩晕的事情。
