# Wacom 推出兼容宽屏的图形平板电脑| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/31/wacom-ships-widescreen-compatible-graphics-tablets/>

有两个品牌严肃的图形艺术家可能永远不会放弃:苹果和 Wacom。任何真正从事数字设计的人都知道，Wacom 平板电脑对于他们来说就像画笔对于传统艺术家一样不可或缺。

他们中的许多人现在面临的问题是，苹果(和其他电脑制造商)正在倾向于宽屏(更接近，如果不是，16:9 的纵横比)显示器，而不是传统的显示器(4:3 的纵横比)。Wacom 的平板电脑反映了传统的 4:3 宽高比，因此它推出 16:9 平板电脑似乎是再自然不过的事情了，它的 Intruos3 系列也是如此。

凭借高达 12×19 英寸的可用表面，新平板电脑可以在整个屏幕上进行像素到像素的映射，甚至可以用于在双显示器上表达移动。这些平板电脑还具有 Wacom 的标准 ExpressKeys，即左手边的可编程快捷按钮。此外，还配有流行的 Grip Pen 手写笔和五键无线鼠标，它是您所需的一切。较小的版本，4×6 英寸，对于美国笔记本电脑用户来说甚至是便携的。滑头。
