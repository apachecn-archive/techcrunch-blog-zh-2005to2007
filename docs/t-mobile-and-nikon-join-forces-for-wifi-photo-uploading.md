# T-Mobile 和尼康联手进行 WiFi 照片上传| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/29/t-mobile-and-nikon-join-forces-for-wifi-photo-uploading/>

相机将照片直接从相机上传到 T-Mobile 的服务器。所有者可以上传 50MB 的图片到 T-Mobile，在那里它们被存储并被无聊的 T-Mobile IT 员工慢慢地盯着看两周，之后它们被删除并最终保存在他们的个人档案中。

购买尼康 Coolpix S7C 的用户可以免费使用一年，一年后你要么支付服务费用，要么购买一台新相机。
