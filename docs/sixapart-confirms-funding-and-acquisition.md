# SixApart 确认融资和收购

[![sixapart](img/07c889411032e131e2013ab3b01e6618.png)](https://web.archive.org/web/20221002122912/http://www.sixapart.com/)

正如之前广泛猜测的那样，TypePad 博客平台和 MovableType 博客工具的制造商 [Six Apart](https://web.archive.org/web/20221002122912/http://www.sixapart.com/) 今天宣布，他们已经完成了 1200 万美元的 C 轮融资。这轮融资来自 Focus Ventures、Intel Capital 和 August Capital，融资总额达到 2300 万美元。

Six Apart 似乎在 TypePad 的大量用户基础上做得很好，他们最近宣布了 TypePad business class 和一项使更多企业更容易进入博客的新计划。SixApart 还有很多事情要做，最新一轮的融资应该可以帮助他们度过下一个增长阶段。

SixApart 今天也宣布他们已经收购了 SplashBlog，具体金额没有透露。SplashBlog 是一个针对手机和 PDA 的博客解决方案，我们应该很快就会看到这项服务与 Six Apart 的现有服务相集成。
