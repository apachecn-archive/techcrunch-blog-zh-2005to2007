# 维斯塔·SP1:一切都会好起来的，更快

![VistaSP1](img/02f964337aa8aff75d279e6fc1c309e7.png)

微软的第一个 Vista 服务包显然会大大提高系统性能。哈利路亚。

据《APC Magazine》的 James Bannan 称，“性能有了非常显著的提高…硬盘抖动大大减少，总体而言，系统似乎更加流畅，响应速度也更快。”

该服务包主要包含错误修复和增强功能，以及一个新的“创建恢复光盘”应用程序，可用于创建系统恢复 DVD。

更新仍处于私人测试阶段，但以下链接中包含的大量信息应该是公众对 SP1 真正准备就绪时所能期待的相对接近的近似值。
