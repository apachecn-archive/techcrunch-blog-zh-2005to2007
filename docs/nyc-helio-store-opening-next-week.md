# 纽约 Helio 商店本周开业？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/17/nyc-helio-store-opening-next-week/>

![peter-005.jpg](img/b9d6a3654e9f7f97d9614c251fa35d36.png)
