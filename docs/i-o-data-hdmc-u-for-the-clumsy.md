# 输入输出数据 HDMC 大学，笨拙的

我会第一个承认我不够协调。没有一个星期我不撞到墙上，从什么东西上掉下来，差点弄瞎自己，等等。然而这不仅仅局限于身体伤害。就我的装备而言，我是一个火车残骸。我口袋里的日常用品寿命不到三个月。我看过的闪存盘多得数不清。

也就是说，我对 I-O 数据中的 HDMC-U 很感兴趣。这是一种便携式硬盘，既防水又防震，看起来相对坚固。它将于本月上市，8GB 和 12GB 容量的起价为 136 美元。
