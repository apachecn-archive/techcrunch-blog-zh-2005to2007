# 使用谷歌合作| TechCrunch 推出玩具搜索网站

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/07/toys-search-site-launches-using-google-co-op/>

[Mad For Toys](https://web.archive.org/web/20151007142221/http://www.madfortoys.com/) 是一个新的搜索引擎，它使用[谷歌的合作服务](https://web.archive.org/web/20151007142221/http://www.google.com/coop/)对英国 250 家在线玩具店进行利基搜索。玩具分为四类，传统、当代、收藏品和模型(如模型汽车等)，尽管许多商店都是手工挑选的。《疯狂的玩具》是由互联网创意咨询公司[未来逃脱](https://web.archive.org/web/20151007142221/http://www.futurescape.co.uk/)制作的。

它工作得很好，但对 Futurescape 来说并不太坏，它显然有点不靠谱(嘿，祝你好运)，他们在圣诞节前推出它也不足为奇。我认为，我们将会看到更多围绕谷歌合作的简单实现创建品牌的尝试，这是我们所知的英国首批尝试之一(尽管可能会有更多)。

TechCrunch 英国网站搜索由 Google Coop 提供支持，但与 Mad for Toys 不同，它没有商业野心。
