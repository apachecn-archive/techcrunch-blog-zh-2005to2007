简介:Feedburner(新功能)

公司: 喷灯

地点:芝加哥

以前简介:2005年6月14日

有什么新消息？

上周四，Feedburner举办了一个“黑客马拉松”日(见 Joe Kraus on hackathons )，工程师们放弃他们的项目一天，致力于可以在一天内设计、实现和测试的新功能。

他们想出了七个新功能(链接):

–Ping服务器–提醒FeedBurner您的博客已经更新，并希望FeedBurner立即更新您的FeedBurner feed，而不是等待正常的30分钟自动更新。
–给你的Feed添加地理标签–给你的FeedBurner feed添加地理位置。
–标题/描述刻录机–允许您在不更改源提要或博客工具的情况下更改提要的标题和/或描述。
–feed medic–通知您FeedBurner在定期处理您的源feed时遇到的任何错误。
–Javascript发行量报价器–启用了感知API，允许你将每日报价器放在你的博客上，显示你的发行量是上升还是下降。
–image cast–对SmartCast服务的增强，超越音频和视频外壳。
–stats tracker Dashboard Widget(仅适用于Mac)–使用适用于Mac OS X Dashboard的全新StatsTracker widget查看您的订阅源的传播情况。

我们最喜欢的是用于快速更新的新ping服务器(weblogs.com ping服务器将很快在这里进行介绍)，以及circulation ticker(放在您博客上的整洁工具)。当Keith去度假试用statstracker小工具时，我正试图找到一台mac电脑。
