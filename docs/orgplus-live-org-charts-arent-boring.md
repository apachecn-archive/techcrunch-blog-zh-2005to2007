# OrgPlus Live:组织结构图不无聊！

当一个听起来乏味的任务变成 Web 2.0 时，总是有点奇怪。典型的例子是创建组织结构图。软件供应商 Human Concepts 开发了 OrgPlus，这是此类活动的首选应用程序，而
[OrgPlus Live](https://web.archive.org/web/20221201170547/http://www.orgpluslive.com/default.aspx) 是该软件的新网络版本。不认为你需要组织结构图？好吧，也许只是你不知道更好。

当然，创建组织结构图的一些目标是获得公司结构的视觉概念，建立项目团队，并随着组织的变化跟踪每个人的角色和联系信息。但是组织结构图可以用来设计项目，建立~~一个博彩公司~~一个 NCAA 绅士的办公室游泳池，啤酒乒乓球锦标赛(谢谢丽莎！)甚至为体育联盟规划团队——基本上是任何需要结构和秩序的事情。想想 Visio online。

OrgPlus Live 带来了基于 Windows 的软件(起价 189.95 美元)的组织结构图优势，并将其应用于基于网络的版本，由于 Flash 的奇迹，该版本可以在任何平台上使用。拖放式创建、简单的文本输入和字段编辑、简单直观的界面、可靠的安全性、从 Excel 导入现有数据的能力等等，现在每月费用低至 9.99 美元。它还处于测试阶段(像其他互联网一样)，但是在我使用它的短暂时间里，事情看起来相当稳定。如果你仍然看不出它有多大用处，我敢说你可以自己尝试一下——有 14 天的免费试用期，不需要安装或下载任何东西。
