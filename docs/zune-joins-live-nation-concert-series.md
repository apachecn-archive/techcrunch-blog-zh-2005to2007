# Zune 加入现场国家音乐会系列

![bnt760×180livenation.jpg](img/097d10a0a2c1118431905d69f309b8de.png)

如果我有一个 Zune，并且觉得有必要参加今年夏天 28 场 Live Nation 音乐会中的一场，那么我也可以加入这个社交网络。Zune 将在演出中占据主要位置，让音乐会观众试用 Zune，并查看[热门、新的夏季颜色](https://web.archive.org/web/20210119095220/http://crunchgear.com/2007/06/04/hey-its-a-zune-and-its-red/)。Zune 点实际上是转换的货运集装箱，这可能有某种回收信息，但我仍然对此持观望态度。

Zune 也给了你，读者一个机会，通过在[Zune.net](https://web.archive.org/web/20210119095220/http://www.zune.net/en-us/social/events/2007/06/livenation/default.htm)注册，在它的 Zune Zoom Away 比赛或 30GB Zune 中观看你最喜欢的乐队的现场表演。还可以下载 Zune Zone 代金券。前 50 名携带优惠券前往 Zune 地点的人将在 Zune 专区获得 VIP 待遇，这是一个在精选音乐会上的特殊区域，有私人服务员和 Zune 草坪椅。这一切听起来像是祖恩-瑞菲克。
