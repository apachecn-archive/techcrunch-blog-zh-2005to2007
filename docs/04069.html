雅虎！去电视放更多雅虎！电视上的属性

雅虎今晚将发布一个新的测试版本，有更好的Flickr集成，画中画观看功能，以及来自雅虎的电影推荐！电影、在本地网络上访问音乐的能力以及对大量电视调谐卡的支持。基本的雅虎！Go for TV产品是一款适用于Windows机器的可下载数字视频录制服务。第一版4月推出。这是与微软的Windows Media Center的竞争，Windows Media Center是购买时预装在电脑上的，而不是从雅虎免费下载的。

雅虎！Go for TV以前允许用户浏览来自Flickr的各种通用流(例如“兴趣”)，但现在将提供对个人照片的访问，以便这些照片可以在电视上与一群人分享。Windows Media Center允许显示照片，但不包括联机组件。

雅虎！雅虎上的电影推荐引擎！电影让用户定义他们从印度到好莱坞的连续口味，然后给10部明显揭示心理的电影打分。然后，它会推荐你附近影院正在播放的电影、DVD上可以买到的电影以及电视上正在播放的电影。以前版本的雅虎！Go for TV没有提供任何个性化的推荐。Windows Media Center允许通过 MovieLink 购买电影。

微软的产品已经提供了访问本地存储音乐的功能，当雅虎！启动电视节目。我不会在这里尝试对这两种服务做一个广泛的比较，但是如果你对媒体中心类型的服务感兴趣，那么你可以看看雅虎！从今天开始，Go for TV将成为一个功能更加全面的选项。Flickr用户可能会特别感兴趣。
