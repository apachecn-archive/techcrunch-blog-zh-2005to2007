# 三星 M55 现在有 HD DVD 燃烧器

三星宣布对 M55 笔记本电脑进行修改，这可能会导致地球停止旋转。嗯，也许不会，但 HD DVD 球迷会很高兴听到这个消息。新的 M55 将配备 Core 2 Duo T7200、2GB RAM、120GB SATA 驱动器、GeForce 7600、蓝牙、Wi-Fi 和 17 英寸屏幕以及前面提到的 HD DVD 燃烧器。费用大约为 3200 美元。

三星在 BR 和 HD DVD 的问题上犹豫了一段时间，但看起来他们可能已经做出了决定。非常非常有趣。
