# 希捷凭借迈拓品牌成为权威

抱歉，我猜这是一个[硬盘](https://web.archive.org/web/20130729233056/http://crunchgear.com/2006/10/02/buffalo-drivestation-duo-offers-a-lot-of-hard-drive-to-love/)的早晨。希捷科技(Seagate Technology)今天上午明确说明了它将如何定位今年早些时候收购迈拓时获得的迈拓品牌内置硬盘。简而言之，迈拓将是主流品牌，希捷将是高端品牌。您将能够获得容量高达 320GB 的 DiamondMax 驱动器，它们将具有更高的普通消费者缓存大小和性能，并提供三年保修。Maxtor 的 MobileMax 笔记本电脑驱动器将具有高达 80GB 的容量和 5,400rpm 的旋转速度，以支持主流笔记本电脑。

相比之下，希捷硬盘具有更大的容量(目前高达 750GB)，更快的旋转速度和标准的五年保修，该公司已经与
合作了一段时间。
