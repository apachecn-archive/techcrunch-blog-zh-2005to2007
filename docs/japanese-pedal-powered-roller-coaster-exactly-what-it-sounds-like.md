# 日本踏板动力过山车:听起来像什么

我记得小时候在 DC 实地考察时，在史密森尼博物馆看到一些类似物理学的展览，当时我脑海中的想法是“我绝不可能坐上去。”我在这里的情感呼应了我年轻时的情感。这是一个你踩着车的过山车。不，真的。绿色游乐设施在轨道上有手推车，由乘坐它们的人推动。仅此而已。

我向日本人致敬，他们把安全抛到了九霄云外。如果这不能证明他们没有拉尔夫·纳德先生，我不知道什么是。
