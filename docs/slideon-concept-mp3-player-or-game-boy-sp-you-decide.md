# Slideon 概念 MP3 播放器还是 Game Boy SP？你决定| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/17/slideon-concept-mp3-player-or-game-boy-sp-you-decide/>

[![elslideon1.jpg](img/054132ac6562920bbc14caa5de618bb1.png)](https://web.archive.org/web/20131215025613/http://old.crunchgear.com/wp-content/uploads/elslideon1.jpg "elslideon1.jpg")

这个 Slideon 概念 MP3 播放器有一个简单的前提:滑盖手机越来越受欢迎，那么为什么不做一个滑盖 MP3 播放器呢？这款基于 flash 的播放器在不使用时会将仅有五个按钮的按钮藏在屏幕下方。屏幕看起来很大，一个可爱的幽灵般的角色漂浮在无限的蓝色中。里面还有一张热辣的照片。

[![elslideon2.jpg](img/14e9b49000a5f9d3509cd16fb3549134.png)](https://web.archive.org/web/20131215025613/http://old.crunchgear.com/wp-content/uploads/elslideon2.jpg "elslideon2.jpg")

我喜欢 Slideon 是因为它的极简设计，用一句老掉牙的空洞的话来说。这让我想起了一个游戏男孩前进 SP。不，事实上，它完全让我想起了 SP。
