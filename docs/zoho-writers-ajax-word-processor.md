# zoho Writer——另一个 Ajax 文字处理器

最近，我们一直在关注大量的 Ajax 办公应用程序(参见[彼得·瑞普](https://web.archive.org/web/20221129070205/http://earlystagevc.typepad.com/earlystagevc/2005/09/ajax_office_is_.html)和[理查德·麦克马努斯](https://web.archive.org/web/20221129070205/http://blogs.zdnet.com/web2explorer/?p=20))以及[其他](https://web.archive.org/web/20221129070205/http://www.beta.techcrunch.com/2005/10/02/writeboard-launches-but-needs-feature-upgrades/) [协作工具](https://web.archive.org/web/20221129070205/http://www.beta.techcrunch.com/2005/09/27/jotspot-live-the-perfect-wiki/)。

坦率地说，许多人(包括我们)认为 [Writely](https://web.archive.org/web/20221129070205/http://www.beta.techcrunch.com/2005/08/31/writely-process-words-with-your-browser/) 、 [Jotspot Live](https://web.archive.org/web/20221129070205/http://www.beta.techcrunch.com/2005/09/27/jotspot-live-the-perfect-wiki/) 、 [Writeboard](https://web.archive.org/web/20221129070205/http://www.beta.techcrunch.com/2005/10/02/writeboard-launches-but-needs-feature-upgrades/) 、 [Chalk](https://web.archive.org/web/20221129070205/http://www.beta.techcrunch.com/2005/08/05/profile-chalk/) 和其他人最终面向的是同一个客户群——那些想要在线创建、分享和集体编辑文档的人。

Zoho Writer 是最新加入的，和其他人一样好。想 Word +组编辑+ Ajax。这是一个丰富的 ajax 应用程序，允许共享和群组编辑，并且像 Writely 一样，有一个伟大的 wysiwyg 编辑界面和优秀的图像导入和操作功能。

事实上，它与 Writely 非常相似，只是它们还不支持 Word 格式的导入/导出或标记(即将推出)。

也看看 Zoho 的背包式产品 [Zoho Planner](https://web.archive.org/web/20221129070205/http://www.zohoplanner.com/) 。

迫不及待地想看到谷歌(据传)将于今年推出的完整 Ajax 套件。
