# 自然无线激光鼠标 6000，微软的疯狂寻找新鼠标

![](img/19cfc50c3970a36f8de642f890125011.png)
Everything USB 有来自微软的新款 Natural 无线激光鼠标 6000 的评测。除了使用一种奇特的新 LAAAAZER 光束引擎，鼠标还提供了一种新的人体工程学设计，据说可以减少手腕张力，增加舒适度。我认为这对于基本用户来说可能是一个有效的选择，但对于游戏玩家和超级用户来说就不那么好了。

我总是抽象地在市场上寻找更好的鼠标，但我想我可能有点迟钝。我已经使用罗技 G5 游戏鼠标一年多了，总体来说很满意。在此之前，我用了 MX 300，因为我不知道有多久(年龄)。希望有人会很快发布更好的东西，因为我觉得我需要有所行动。
