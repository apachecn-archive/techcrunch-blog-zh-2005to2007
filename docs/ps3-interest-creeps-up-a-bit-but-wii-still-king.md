# PS3 的兴趣有所上升，但 Wii 仍然是王者

[![warover.jpg](img/d70058dc33bece7c777e6eb228ee1853.png)](https://web.archive.org/web/20210411002958/https://beta.techcrunch.com/wp-content/uploads/2007/04/warover.jpg "warover.jpg")

Fanbois 生活在互联网上，因此通过跟踪相关搜索来衡量这三个视频游戏控制台的兴趣水平是有意义的。来自英国一家货架公司的重要结论表明，虽然从亚马逊等零售商的搜索量来看，PS3 的搜索量自该系统在欧洲推出以来增加了 127%，但 Wii 和 Xbox 360 仍然是最受欢迎的。Xbox 360 的搜索量增加了 250%，而 Wii 的搜索量增加了 579%。稍微算一下，这意味着 Wii 比 PS3 强 4.5 倍。来吧，指责我有偏见。

不过，说真的，谁会想到 Wii 会在战争的这个时候如此轻松地“获胜”呢？我想休闲游戏玩家比我们想象的要多。
