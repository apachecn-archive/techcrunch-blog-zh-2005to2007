# X06:漫威宇宙在线即将登陆 Xbox 360 和 Windows Vista

![](img/bea4dcf0e09a828052a6cbf61e13e673.png)

微软游戏工作室和漫威娱乐已经证实，Cryptic 工作室将成为备受期待的《漫威宇宙在线》的开发者。Cryptic Studios 以其大 MMO *英雄之城*而闻名，这其实是一款很棒的游戏。*漫威宇宙在线*将跟随*英雄之城*的脚步，实际上将是为 Xbox 360 发布的第二款 MMO，也是目前为 Windows Vista 发布的唯一一款 MMO。

除了能够玩漫威超级英雄，没有太多关于游戏将包括什么的信息，但希望它将远离当前可用的漫威游戏。我已经在考虑如何定制我的策略了。会有对策的，对吗？我希望如此。
