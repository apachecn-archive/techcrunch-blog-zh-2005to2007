# 内置支付卡的手表

![](img/34cb40c9bfc33d5bd026140bae099b5b.png)

在土耳其，一家当地银行推出了一项新的促销活动，将使信用卡成为历史。在与[万事达卡、](https://web.archive.org/web/20201129025851/https://crunchbase.com/organization/mastercard) 的一笔交易中，该银行现在提供一款内置 PayPass 信用卡的手表。你所要做的只是在接受 PayPass 的柜台刷一下你的手表，你就可以走了。只要你的总购买价格低于 15 欧元，就不需要签任何东西。虽然 PayPass 不是什么新东西，我们在美国也有，但我当然喜欢看着信用卡。手表本身看起来也不算太寒酸，尽管我认为它不适合在夜晚外出时佩戴。为此，米老鼠永远称王。
