# USB 可折叠键盘| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/29/usb-roll-up-keyboard/>

![](img/3a08bfccb8d328f6dd53d70e00ab326d.png)

这里有一些对忙碌的音乐家来说很方便的东西。USB 卷式键盘有 49 个键，128 种乐器，100 种预先录制的节奏。你知道，它还可以卷起来。卷起式键盘还没有被广泛使用。一些音乐家一直在使用 Logic Pro 7 或 Pro-Tools。这两个程序都是基于 MIDI 的，通常被电子音乐人所使用。

有了新的 USB 支持，你可以在任何地方拥有一个 MIDI 控制器。由于按键看起来没有按键感应，你会失去一些人性化的功能，比如力度，但是你可以在程序中编辑类似的东西。键盘运行 46.00 美元，所以板载的东西可能听起来像废话。但是像 Logic Pro 7 这样的程序拥有的所有功能，你无论如何都不会使用键盘的许多板载功能。
