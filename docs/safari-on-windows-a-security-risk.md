# Windows 上的 Safari 存在安全风险

苹果新发布的视窗版 Safari 浏览器今天因为各种错误的原因遭到抨击，有独立报告指出该浏览器给视窗用户带来了安全风险。

Thor Larholm [发布了](https://web.archive.org/web/20210923181327/http://larholm.com/2007/06/12/safari-for-windows-0day-exploit-in-2-hours/)一个 0day 漏洞，该漏洞导致 Safari 在访问带有恶意代码的网站时关闭。

勘误表安全的大卫·梅诺[声称已经发现了 6 个漏洞，4 个拒绝服务和 2 个远程代码执行漏洞。](https://web.archive.org/web/20210923181327/http://erratasec.blogspot.com/2007/06/niiiice.html)

这款浏览器得到了褒贬不一的评价；快速渲染网站的能力受到视觉上外来(对 Windows 用户而言)字体渲染的影响，并且不支持 Vista Aero 界面。
