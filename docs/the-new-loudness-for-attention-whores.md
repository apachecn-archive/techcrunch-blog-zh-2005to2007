# 关注妓女的新声音

![y_4.jpg](img/a2a9bd714270d2565d888a29e7320a5e.png)

女士们请注意:如果你天生需要在工作时吸引更多的注意力，有比这件内置扬声器的裙子更好的方法。是的，这是一个概念，但我们希望它不会流行。把你的衣服变成泰迪熊只会激怒你周围的人。任何穿这个的女人无疑对音乐的品味也是有问题的。这就是重点。

谢天谢地，这不仅仅是一个移动的 iPod 基座。这种衣服运行自己的软件，并有某种环境感知型“传感器”。这套服装中的大脑接受感官输入，谱写出一首“旋律”，它应该与“正在发生的事情”合拍。随便啦。

如果你需要更多的“看看我！”随着时间的推移，我们可以想出许多其他不会惹恼我们的方式，比如蓝牙比基尼。努力吧，设计师们。
