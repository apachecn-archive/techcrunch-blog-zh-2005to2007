# 东芝发布 HD DVD 硬盘

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/05/toshiba-releases-hd-dvd-drive/>

东芝为世界带来了首款 HD DVD 写驱动器，同时仍支持旧格式，即 DVD 和 CD。SD-H903A 的一些样品驱动器将于本月开始发货。这不是最快的，但它是第一个，所以我们要给东芝一些信任，它在单层和双层 HD DVD-R 上都以 1 倍的速度燃烧。它的缓冲容量为 8MB，据说由于只有一个物镜的光学拾取头，它支持所有 HD DVD。CES 就在眼前，所以我们应该能在几天内瞥见它的能力，展位 7914。
