# Escopetarras–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/18/the-escopetarras/>

![](img/6c45c5120e3d29f9e732698c5cbe3091.png)

这些乐器被称为“Escopetarras”，是枪和吉他的融合，由塞萨尔·洛佩兹创作。洛佩兹是在站在一家被炸毁的波哥大俱乐部外，注意到他的吉他和一把士兵步枪的相似之处时获得发明灵感的(whaaa？).

> 从那时起，他就开始与乐器工匠路易斯·阿尔贝托·帕雷德斯合作创作乐器。最初由温彻斯特步枪制成，现在主要由每个人都喜欢的冷战突击步枪 AK47 制成。

我真的看不出这些有什么用，而且由于吉他的身体和形状，音响效果也不会太好，但是，我不认为任何购买聚变枪和吉他的人会坐在一起唱 Goom-By-Ya。
