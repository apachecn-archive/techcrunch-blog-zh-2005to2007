# 诺基亚推出 3109 经典| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/18/nokia-introduces-3109-classic/>

![](img/5b772df1ed9efebd9f24cd649b70dc20.png)

那些在欧洲的人应该注意来自诺基亚的 3109 经典。该产品包括蓝牙和内置音乐播放器的 microSD，但缺少摄像头。如果你想把你的 iPod 留在家里，或者如果你不能在工作环境中使用照相手机，这是一个完美的选择。除了蓝牙和音乐播放之外，这款手机基本上是一款实用的设备。它有一个电子邮件客户端，不错的电池寿命，短信——没什么特别的。它将以 189 美元的价格在 Q2 首次亮相，并将只在欧洲发售。
