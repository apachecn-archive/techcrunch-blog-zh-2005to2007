# Smack Talk Talks Smack——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/25/smack-talk-talks-smack/>

![](img/2d8e31efa748ba577ff148539a5bbac4.png)

好的，我们知道在 Xbox 360 上玩在线游戏最棒的事情之一就是告诉别人你“wtf？pwned“他们。就像是瞬间的满足感。Digital Innovations 利用了这一想法，创造了 GameDR Smack Talk，这是一种按下按钮就能说话的设备。Smack Talk 插入你的 Xbox 360 控制器的耳机插孔(它有自己的耳机)，用三节简单的 AAA 电池和一个按钮，你就可以选择告诉你的对手“来吧，让我开心一天”(也可以用这种声音)。起初，这款设备看起来并不太吸引人。现在我所能想到的就是杀死一个按下“趴下”按钮的人(也是那个声音)，然后嘲笑他。你可以选择定制任何你想要的声音剪辑。这意味着你可以得到随机的声音，音频剪辑，更多的单句和任何你想包含的东西。如果你还没有开始使用静音按钮，请做好准备。Smack Talk 售价 29.99 美元，如果你像我一样有幽默感，你肯定会从中获得一点乐趣。
