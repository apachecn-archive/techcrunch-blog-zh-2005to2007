# 任天堂 Wii:组件电缆的图形差异

![](img/377ffa8b06a97f55fc9327ca78559b5f.png)

任天堂 Wii 在高清电视上显示图像时存在一些严重的问题，尤其是在显示文本时。由于《塞尔达传说》没有任何画外音，这个游戏需要大量的阅读。当使用捆绑的复合电缆将 Wii 连接到 HDTV 时，这变得很困难。纹理看起来不明确，文字模糊，世界本身看起来也不那么热。和许多其他人一样，我一直试图买到 Wii 的组件电缆，这些电缆碰巧到处都卖完了。你可以在易趣上买到这些电缆，但必须支付 100 多美元。人们一直在问的一个大问题是，分量电缆是否真的会对高清电视产生足够大的影响。简短的回答是:是的。虽然这些图片没有公正地处理组件电缆，但 wiinintendo 表示，你需要看到游戏在运动，才能看到组件电缆实际上产生了多大的差异。至少在标题屏幕(如上图)中，文字足够清晰，不用眯着眼睛就能阅读(分量电缆屏幕在左边)。更多电影如下。
