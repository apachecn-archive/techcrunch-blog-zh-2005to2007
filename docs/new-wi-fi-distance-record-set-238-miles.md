# 新的无线网络距离记录:238 英里

[![wifirecord.jpg](img/07210e3c76ac9962a042cc59540978cd.png)](https://web.archive.org/web/20201031161256/http://old.crunchgear.com/wp-content/uploads/wifirecord.jpg "wifirecord.jpg")

委内瑞拉是一个美丽的国家，一名研究人员创造了一项新的无线网络距离记录，从他的研究机构向安第斯山脉的另一个研究机构发送了 238 英里的信号。他们都位于安德烈斯山脉。这里最令人印象深刻的是，类似的努力并没有产生这样的结果。英特尔在美国所做的最大努力也只能到达 20 多英里以外的地方。除了英特尔提供给他的专用设备之外，这位委内瑞拉研究人员还使用了一些现成的部件。

有趣的是，我在纽约北部的房子从一层到另一层几乎收不到 Wi-Fi 信号。
