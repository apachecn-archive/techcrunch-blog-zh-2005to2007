# 您的 Minis 试图重新启动起始页

[![](img/bc1c2f46322a98c1bbcffd6ceadcb8b4.png)](https://web.archive.org/web/20221205082516/http://www.yourminis.com/) 已经竞争激烈的 startpage 空间今晚出现了一个有趣的新戏，YourMinis.com以一个改变的网站和新的战略重新启动。

YourMinis 是 RSS 源和网络功能的完美结合，与 Pageflakes、Netvibes 和一长串其他 startpage 服务竞争。YourMinis 报告拥有超过 80 万个用户账户，但仍远远落后于其主要竞争对手。它确实在 Wow 部分得分很高——例如，我从来没有通过两次点击将圣地亚哥动物园的活潘达卡姆添加到 Netvibes。

就在几分钟前，该网站重新推出了一个 HTML 主页，重点突出了用户、特色小部件和热门小部件集合。起始页已经转移到 yourminis.com/start[在接下来的几周内，该公司将推出一个公共 API，供开发者用于构建他们自己的 YourMinis 小工具。](https://web.archive.org/web/20221205082516/http://www.yourminis.com/start)

该公司还将在下周的 Adobe 会议上展示与 Apollo 的集成——这意味着你的 Minis 小工具将能够在网络上构建，并在桌面上使用。

在 startpage 应用程序中突出社交听起来很有趣，但我不确定对它有多少需求。我很想被证明是错误的——也许一百万个迷你会开花，被渴望良好编译的起始页标签的用户吞噬。看着出现在左边的芝加哥小熊队特色页面(如果你愿意，可以作为迷你版),我可以想象它的吸引力。热门博客、视频源和其他关于给定主题的动态源，由希望分享该主题知识的人编辑。不幸的是，这样的事情已经尝试过很多次了。例如，丑陋可能不是 Squidoo 的唯一问题。

一个好的 API 和 Apollo 集成可能会很有趣，但我认为你的 Minis 的优势将仍然是服务鲜明的视觉吸引力和对基础的扎实处理。如果这还不足以在公司面临如此多的竞争时为其建立牵引力，我不知道会是什么。
