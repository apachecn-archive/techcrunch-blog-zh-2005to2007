# Searchles 抓住另一个天使轮

[![searchleslogo.png](img/d1c7e2cadd06081c19eb37d40a50d641.png)](https://web.archive.org/web/20220929201256/http://www.searchles.com/) 总部位于华盛顿特区的社区搜索引擎[searches](https://web.archive.org/web/20220929201256/http://searchles.com/)今天获得了另一轮天使投资，在之前 60 万美元的基础上又获得了 30 万美元。超过一半的资金用于开发搜索引擎 Dumbfind。

Searchles 允许你通过 Delicious 这样的书签工具保存和标记，但是在搜索和共享链接时有更精细的控制。在 Searchles 中搜索会根据用户与你的关系来浏览用户的“圈子”。您可以搜索每个人、您自己、朋友、朋友的朋友、喜欢您帖子的人和群组发布的链接。这里有一个在 Web 2.0 组中搜索“搜索引擎”的[例子](https://web.archive.org/web/20220929201256/http://www.searchles.com/search/?search=search+engine&tag=&scope=group&group=web+2.0&x=0&y=0)。正如 Digg [使用](https://web.archive.org/web/20220929201256/http://www.beta.techcrunch.com/2007/02/01/digg-removes-list-of-top-users/)所做的那样，Searchles 现在也能识别排名页面中的顶级用户和群组。

新的资金紧随 searcles 最近发布的“推我/拉我”小工具之后，该小工具将 searcles 的书签、标记、共享和评论带到链接保存的页面。当 javascript 小部件安装在一个链接上时，它会显示主 searches 站点中该链接的所有 searches 数据。在这个小部件中，用户可以保存搜索链接，标记它，评论，共享链接，并通过标记快速显示其他相关的搜索链接。
