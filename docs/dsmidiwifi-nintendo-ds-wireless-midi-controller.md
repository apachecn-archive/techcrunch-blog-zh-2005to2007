这太令人兴奋了。一些人已经开发出一种方法，将任天堂 DS 作为 MIDI 控制器无线使用*。DSMIDIWiFi 使用任天堂 DS 的触摸屏作为控制器，并将信号无线发送到你电脑上的主机程序，如 Garageband、Ableton Live 或任何你用来创作音乐的程序。创作者为您的音乐乐趣编写了三种不同的程序，它们的工作方式如下:*

**   DS MIDI 键盘充当 MIDI 键盘，包括通过垂直/水平滑动唱针进行的音高轮/ MIDI 控制调整。*   Kaos DS 是一个类似于 Kaoss pad 的 XY 控制器。您可以使用手写笔或手指来调整 MIDI 控制。*   Pulse DS turns the DS into a MIDI synthesizer that uses the DS’s PSG (programmable sound generator) to produce Game-Boy-like sounds.

    想象一下，能够像使用 Korg Kaoss pad 一样使用任天堂 DS。环境噪音和流体控制都与您的 DS 和手写笔。神奇的东西。这是我见过的任天堂 DS 最好的自制应用之一。查看他们的页面，了解最新发展。
