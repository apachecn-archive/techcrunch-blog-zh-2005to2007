# Netvibes 推出脸书 Widget 它有点小毛病

 [](https://web.archive.org/web/20221130202003/https://beta.techcrunch.com/wp-content/uploads/2007/08/netvibesfbb.png) 人们显然对所有新的脸书应用程序有些厌倦了——所以很高兴看到通过将脸书数据导入另一个应用程序来实现相反的工作。 [Netvibes](https://web.archive.org/web/20221130202003/http://www.crunchbase.com/company/netvibes) 今天发布了一个很酷的[新工具](https://web.archive.org/web/20221130202003/http://blog.netvibes.com/?2007/08/02/140-facebook-meet-netvibes-netvibes-introduces-the-facebook-widget)，它可以将某些[脸书](https://web.archive.org/web/20221130202003/http://www.crunchbase.com/company/facebook)数据拉进你的 Netvibes 页面。一旦安装完毕，这个小工具就会显示消息、戳、好友请求和其他信息。这个小工具还显示朋友的信息，并有搜索功能。

我可以安装它，但它无法从脸书获取数据。我们的两个实习生把它修好了。让我们知道它对你来说是否顺利。

如果脸书(或使用脸书 API 的第三方)创建一个通用的小部件来在任何网站上发布这种类型的信息，那就太棒了。
