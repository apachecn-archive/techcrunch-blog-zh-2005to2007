# 爱情酒:不用再猜什么酒配什么菜了

[![foodwine.jpg](img/a67f4c2b44de08758fbb7c3adec118ff.png)](https://web.archive.org/web/20210225214712/https://beta.techcrunch.com/wp-content/uploads/2007/04/foodwine.jpg "foodwine.jpg")

这里的老板比格斯很了解他的酒，所以他知道什么酒配什么食物。而我，却对这种神奇的魔法一无所知。葡萄酒爱好者应该通过在酒的正面贴上标签来解决这个问题。该系列有五种不同的葡萄酒，每一种都搭配不同的食物:意大利面、烤鸡、pizaa(谁会吃配有葡萄酒的披萨？)，烤牛排，烤三文鱼。奥尔埃！

这些酒的价格不知道，更不用说味道如何了。我说他们需要做一种配写博客的酒。
