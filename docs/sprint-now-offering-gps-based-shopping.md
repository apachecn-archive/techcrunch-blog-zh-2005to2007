# Sprint 现在提供基于 GPS 的购物服务

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/07/sprint-now-offering-gps-based-shopping/>

![](img/dc71b158247a9ff0c289c9f688921323.png)

太专注于博柏利店，甚至没有注意到一个街区外的新 YSL 店？听起来你喜欢购物。Sprint 用户可以通过尝试其新的 Slifter 服务来获得最热门的交易。Slifter 是一种基于 GPS 的手机购物方式。它会跟踪你的位置，并向你推荐你当前位置附近的地方。你也可以通过名称和产品来搜索商店，这非常有用。

这项奇怪命名的服务应该对那些喜欢在美好的一天在户外购物的男女有用，而不是那些在购物中心闲逛寻找橙色 Julius 的青少年。此外，如果你想找到一个经销店的宝石，你可以保存以前的搜索。这项服务将适用于部分手机，每月额外收费 2 美元。在零售世界里，这是很小的代价。

[Sprint 将基于 GPS 的购物添加到手机中](https://web.archive.org/web/20150914161836/http://www.electronista.com/articles/07/06/07/sprint.slifter/)
