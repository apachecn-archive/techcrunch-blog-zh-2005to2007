# 手机上的环球秀时间到了

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/08/showtime-for-universal-on-mobile-phones/>

房子里最好的座位可能是在公共汽车或火车上，或者任何你可以拿着手机的地方。TG Daily 报道称，移动娱乐公司 I-Play 已经与环球影业签署了一项协议，将向手机提供几十部电影的流媒体视频剪辑。

虽然你仍然无法观看完整的电影，但这些片段将来自粉丝的最爱，包括《速度与激情》、《动物之家》和《金刚》。I-Play 声称，与环球公司的交易是第一次将电影内容带到手机上，这将可以在最先进的手机上播放。剪辑将按流派和字母顺序排列。

斯普林特，威瑞森，罗杰斯和 Amp'd 的客户将首先获得新的视频服务，但其他运营商应该能够在不久的将来获得“电影分钟”。
