# Windows Mobile 6(正式)发布

好吧，这对于经常阅读该网站的人来说应该不陌生，但是 Windows Mobile 6 就要来了。我们已经在 Crossbow 上报道了[吨](https://web.archive.org/web/20201130075334/http://crunchgear.com/?s=crossbow)的信息，但是今天还有一点。即以官方公告的方式:

正如我们[上周](https://web.archive.org/web/20201130075334/http://crunchgear.com/2007/01/29/microsoft-renames-windows-media-6-to-standard-professional-classic/)提到的，WinMo 6 将有三个版本:触摸屏专业版、智能手机标准版和掌上电脑经典版。

我们现在知道了新的移动操作系统将宣传的一些功能:

*   HTML 电子邮件支持*   Windows Live for WinMo*   使用 Live Messenger 传输文件*   新的移动办公程序*   Vista 集成*   Calendar ribbon for convenient scheduling

    除了这些功能上的增加，WinMo 6 还将具有。NET Compact Framework 和 SQL Server。然而，也许最有趣的是一项新功能，允许用户从丢失的设备上擦除数据和功能。

    Windows Mobile 6 将于下周在 3GSM 展出，所以请期待我们糊涂的编辑 John Biggs 带来的完整报道。
