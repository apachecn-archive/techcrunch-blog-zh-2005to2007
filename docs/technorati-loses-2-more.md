# Technorati 又输了两个三个

[](https://web.archive.org/web/20221205065243/http://www.technorati.com/)

刚刚传出消息，Technorati 又失去了两名员工。坦特克·切利克在 Technorati 担任技术主管 3 年后离职，亚当·赫兹[也在担任工程副总裁 3 年后离职。Tantek 和 Adam 都在 Technorati 的技术开发中发挥了重要作用，Tantek 还是微格式和其他开放标准的发言人和传播者。](https://web.archive.org/web/20221205065243/http://adam.typepad.com/)

从博客帖子来看，他们似乎都离开了公司去寻找其他机会。不知道他们是否在一起工作，因为他们同时离开，并在 Technorati 密切合作。他们两个最重要的技术人员的离开对 Technorati 来说是一个巨大的打击。Tantek 给 Technorati 带来了很多技术上的可信度，这在他们与停机和性能问题作斗争时尤其需要。Tantek 还负责围绕公司及其服务形成的很大一部分开发人员社区，并在 Technorati 对标签和微格式的支持中发挥了重要作用。今天，你可以发现几乎每个博客平台都内置了 Technorati，它们的 API 被许多不同的网站用来显示最新的结果、声誉等等。由于 Tantek 和其他人(如今年早些时候离开谷歌的 Kevin Marks)给公司带来的开发者福音和支持，Technorati 在第三方中获得了稳固的采用。

这将对 Technorati 产生什么影响还有待观察，因为许多第三方开发人员都忠于 Technorati 的关键开发人员，他们中的大多数人现在已经离开了公司。Technorati 也有很多项目，比如微格式，是由那些已经不在的关键人物领导的。还有待观察的是，这些人会发生什么，最重要的是，Technorati 是否能够保持他们在开发者中有些滞后的信誉。
