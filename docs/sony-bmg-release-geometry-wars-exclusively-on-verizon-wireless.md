# 索尼 BMG 公司在威瑞森无线电台独家发行《几何战争》

流行的街机游戏《几何战争》现在可以在手机上玩了，但是你需要成为威瑞森无线用户才能玩这个游戏。索尼 BMG 音乐娱乐公司宣布独家发行《几何战争:威瑞森复古进化手机》。这款游戏由奇异创意公司开发，忠实于最初的主机版本。

控件仍将使用熟悉的四向或八向按钮设置，并依赖键盘作为主要界面。这款游戏有八种游戏模式，还有一个排行榜可以追踪你的状态。几何战争:复古进化手机在 Get It Now 虚拟商店出售，每月 3.49 美元或 6.49 美元无限使用。
