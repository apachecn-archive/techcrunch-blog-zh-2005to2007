# PopCatcher 接收调频广播，因此您不必

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/01/24/popcatcher-catches-fm-radio-so-you-dont-have-to/>

当我住在西雅图，一直听着 [KEXP](https://web.archive.org/web/20150926002225/http://www.kexp.org/) 的时候，我很想拥有 PopCatcher MusicDock MD-601。

从表面上看，它只是一个外观不错的调频收音机，顶部有一个基座，用于连接内置的 MP3 播放器，大概是为了直接将音乐录制到便携式设备上。仅此一点并不特别有趣，因为有许多数字音频播放器可以录制调频广播。然而，PopCatcher 的不同之处在于它能够区分音乐和广告、电台 id 和 DJ 聊天。

该设备智能地剪辑掉除音乐之外的所有内容，让便携式播放器只加载以 192Kbps 速度录制的 MP3 文件。你也可以把 DAP 连接到你的电脑上，把你捕捉到的任何内容下载到你的电脑上。

该公司的网站上没有定价或发布信息，但你可以在这里注册联系。
