# Myspace 和 Cingular 上

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/18/myspace-and-cingular-get-it-on/>

![](img/263cd8f5b3912009deb45ec171cdae94.png)

和平 Helio 用户！回头见！看来你不再是唯一一家与 MySpace 有协议的无线提供商了。Cingular Wireless 和 MySpace 联手，让其无线用户在外出途中也能访问他们在社交网站上的个人资料。你和你 15 岁的粉红色朋克女友可以查看你的个人资料，发表评论，从手机上传图片，做所有让 MySpace 如此有趣的事情。

哦，但是等等。你以为这会是免费的，不是吗？Cingular 的一个很好的姿态帮助了年轻的人群，对吗？不这么认为。准备好每月花 3 美元为你的手机购买一个移动 MySpace 应用程序。别以为这就完了。你很清楚他们会向你推销“兼容 MySpace”的手机。
