# Noka:昂贵的巧克力曝光| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/28/noka-the-expensive-chocolate-expose/>

![](img/0742bc7454ffeda398f3c36010142f90.png)

Noka 是一种昂贵的巧克力品牌，每磅价格约为 2080 美元，对于你的身体最终将处理成固体废物的东西来说，这是一笔相当大的投资。当地的美食博客 DallasFood 决定仔细看看这个自吹自擂的品牌，并发现 a)他们不自己制作或采购可可，b)他们可能没有他们声称的那么好。

但是要记住，味道是一个非常主观的东西。如果你把一大块巧克力放在一个别致的钢盒子里，另一块同样的巧克力放在一个标有“鸟粪和附带巧克力”的盒子里，你的大脑会自动偏好这个别致的品牌。我知道这有点跑题了，但是我知道你们喜欢巧克力，所以请过来看看这个帖子。

顺便说一句，这种骗局发生在许多高端行业，包括手表、雪茄，当然还有科技。如果你知道索尼等离子是谁制造的，你可能会尿裤子。
