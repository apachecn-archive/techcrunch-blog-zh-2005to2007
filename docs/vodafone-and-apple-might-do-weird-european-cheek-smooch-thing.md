# 沃达丰，苹果可能会做奇怪的欧洲脸颊亲吻的事情

ZDNET 亚洲正在发表一篇关于沃达丰可能成为在欧洲提供 iPhone 服务的潜在运营商的文章。也许吧。

根据这篇文章“瑞士瑞信银行认为沃达丰对苹果来说是一个很好的赌注，因为它能够在大多数欧洲市场提供存在。”

关于这种可能的合作关系，有趣的是沃达丰运营着一个真正的 3G 网络，而苹果公司设计的 iPhone 可以在美国的 2G 网络上运行。iPhone 可能需要升级以支持更快的数据网络。从消费者的角度来看，这是一个好问题，但许多人会说，苹果应该首先让它兼容 3G。

苹果公司还没有宣布 iPhone 在欧洲的发布日期，所以这一切都是不切实际的猜测，直到它有可能发生。
