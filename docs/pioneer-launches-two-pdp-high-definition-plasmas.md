# 先锋推出两款 PDP 高清等离子

![](img/66ebc8716877b88bddcd8722685cd076.png)

先锋已经在日本发布了两个以上的 PDP 高清晰度等离子体。这两款新机型包括 PDP-607HX(如图)和 PDP-427HXD。

这种等离子体使用了一种新的技术，一位先驱者已经标记了“新的 P.U.R.E .黑色面板”和“新的 P.U.R.E .驱动 II”。黑色面板可以再现行业领先的色彩水平，Drive II 具有新的数字信号处理器，可以增强等离子体的性能。60 英寸的 PDP-607HX 将售价 6770 美元，42 英寸的 PDP-427HXD 还没有价格。

目前还不知道美国是否会购买这双鞋，但是有了这个价格标签，我们可能无论如何都不会购买。
