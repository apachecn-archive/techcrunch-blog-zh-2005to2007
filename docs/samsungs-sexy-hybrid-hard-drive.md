# 三星的性感混合硬盘

不，这种混合硬盘实际上并不性感(尽管它是袒胸露乳的)，而且一般来说很难对硬盘产生兴趣。但是，由于三星在这一领域仍然处于劣势，该公司正在尽最大努力创新这一领域，而不是仅仅推出更大的尺寸([尽管它也这么做了](https://web.archive.org/web/20201026012936/http://crunchgear.com/2007/06/12/samsung-shipping-1tb-hdd/))。 [2.5 英寸 MH80 笔记本硬盘](https://web.archive.org/web/20201026012936/http://samsung.com/Products/HardDiskDrive/HybridHDD_FlashON/index.asp)具有 256MB 的集成闪存，可以帮助你更快地启动(虽然只快了大约 10 秒)，更快地从休眠模式中醒来，并为你的笔记本电脑延长 25%的电池寿命(尽管这与三星自己的 M80 硬盘相比)。

 如果你渴望亲自看到这些改进，这些硬盘——有 80GB、120GB 和 160GB 三种尺寸——目前可从 [CMS](https://web.archive.org/web/20201026012936/http://www.cmsproducts.com/) 获得，7 月份可从 [SimpleTech](https://web.archive.org/web/20201026012936/http://www.simpletech.com/) 获得。
混合动力的乐趣当然不会就此停止。三星的路线图包括闪存容量更大的硬盘，以及闪存容量更大的硬盘。
