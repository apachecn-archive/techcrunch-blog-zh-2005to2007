# R2-D2 说哔哔哔

这是我见过的最酷的星球大战小工具了。这个小 R2-D2 是 1/2 的比例，由千年隼控制。它的圆顶可以旋转 270 度，他的“眼睛”可以投射到墙上或天花板上 80 英寸的屏幕上。一个 DVD 播放器，iPod 基座，立体声扬声器系统，存储卡，USB 输入，AV 输入，11 个经典的 R2-D2 音效和更多的东西都藏在这个甜屁股的日兴好东西里。我绝对要给自己弄一个。

编辑:预计在三月份，支付大约 2500 美元。
