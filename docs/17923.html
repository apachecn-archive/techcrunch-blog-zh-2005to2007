美国可能在欧洲之前获得免费移动服务

原文：https://web.archive.org/web/http://www.techcrunch.com:80/2007/07/29/the-us-may-get-a-free-mobile-service-before-europe/

我们对印第安纳州的 Mosh Mobile 还不太了解，但它声称在美国客户购买手机后，将向他们提供免费的基于广告的移动服务。这一消息传出时，美国一家备受瞩目的移动虚拟网络运营商(MVNO)在斥资3.6亿美元后停止运营。

Blyk ，一款在欧洲备受期待的MVNO，有类似的模式(免费服务，广告支持)，已经错过了几个发布日期。Mosh Mobile已经推出，虽然是在一个私人的“测试版”。只有其他Mosh Mobile用户可以邀请您加入该服务。他们还没有透露他们的服务提供商是谁。

商业模式:“这项服务将通过短信广告、无线网络门户、插入无线网页的广告、免费的无线应用程序来支持，并将针对不同的用户群。”要产生任何一种真实的ARPU(每用户平均收入)，就必须有大量的广告。目前还不清楚用户是否能够容忍这种情况，也不清楚广告商是否会得到他们所期待的回报。

该公司表示，他们得到了资助，但没有透露细节。他们有一个推特账户来更新用户信息。

更新:我一直在和移动高管交谈……没有人听说过Mosh Mobile，它似乎没有任何运营商协议，而且域名刚刚注册的事实表明，这更像是一种希望，而不是现实。我们会继续的。
