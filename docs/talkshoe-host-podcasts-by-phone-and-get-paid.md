# TalkShoe:通过电话主持播客并获得报酬

[![](img/1b4602d4169a77dbc1e5fbcca4023a1a.png) ](https://web.archive.org/web/20221002124929/http://www.talkshoe.com/) [TalkShoe](https://web.archive.org/web/20221002124929/http://talkshoe.com/) 是一个播客服务和目录，它将录制的电话会议与节目主持人的收入分成结合在一起。它看起来很容易使用，社区元素让我认为这项服务会成功。我无法想象许多节目主持人会因此赚很多钱，但在线视频世界已经表明，当分享很容易时，人们就会这样做。

如果世界真的在改变，把用户生成的内容放在媒体的中心，有人会找到音频的正确公式。这看起来是朝着正确的方向迈出的一步。

下面是 TalkShoe 的工作原理。Windows 桌面客户端允许多达 25 人在幕后参与对话和聊天。电话会被自动录音并输入到网站的播客目录中。节目主持人从网站的广告收入中获得金钱，根据他们节目的听众数量以及推荐其他人使用这项服务。

谷歌广告、feeds 中的广告和播客中插入的音频广告都将被用来赚钱。

对话可以是公开的，也可以是私人的，网站访问者被鼓励打电话参加正在进行的现场录音。与会者可以通过电话、Skype、Vonage 或 Yahoo 拨打电话。该公司表示，SIP 支持即将推出。

完成的播客被输入到 TalkShoe 播客目录中，用户可以在那里写评论和留下评论。该网站旨在鼓励订阅和安排未来剧集。网站上还没有太多的活动，但看起来它真的会流行起来。

到目前为止，特色播客包括巴里·邦兹本垒打比赛、在家酿酒、意大利旅行以及 Mac 和 PC 爱好者之间的辩论。

很多人说播客对于非技术用户来说还是太难了。TalkShoe 看起来很简单。如果真的有很多人对制作或收听自制音频内容感兴趣，这项服务可能是他们这样做的一个好方法。
