# 维珍移动美国公司 IPO 启动

维珍移动的美国业务正在好转，这是理查德·布兰森的维珍集团有限公司和 Sprint Nextel 的合资企业。Virgin Mobile USA Inc .将其首次公开募股金额提高至 5.06 亿美元，是最初目标的五倍。部分吸引力在于，年轻人市场已经真正接受了这个成立于 2002 年的 MVNO 品牌，并成功锁定了 35 岁以下的客户。

IPO 筹集的资金将用于削减债务和支付 Sprint 的自有资金。截至 3 月底，Virgin Mobile USA 拥有 488 万客户，第一季度净收入从去年同期的 338 万美元跃升至 1540 万美元。显然，年轻人谈到了处女。
