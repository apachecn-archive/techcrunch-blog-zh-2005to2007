# YouTube 将和威瑞森一起移动？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/07/youtube-to-go-mobile-with-verizon/>

![](img/53382f1199b207cfc29b1eb908e60932.png)
[上周](https://web.archive.org/web/20141021112604/http://crunchgear.com/2006/11/02/youtube-mobile-in-2007/)报道称，YouTube 创始人查德·赫利表达了他在 2007 年底将 YouTube 移动化的雄心。现在看来，这家互联网视频巨头可能会比预期更快地接近这一目标。

路透社今天早上刊登了一篇文章，推测 YouTube 可能很快就会与威瑞森达成协议，将其视频传输到威瑞森的手机上。然而，用户似乎不能浏览 YouTube 并自由观看任何他们想看的视频。相反，看起来 YouTube 将通过 V Cast 和点播向威瑞森手机提供特定内容。

这篇文章假设这将使威瑞森在竞争中获得某种优势，但我认为这种说法可能有点过头了。Cingular 已经有了 [YouTube Underground](https://web.archive.org/web/20141021112604/http://www.youtube.com/underground) ，它向用户提供特定的 YouTube 内容:
