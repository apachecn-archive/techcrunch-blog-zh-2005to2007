# RealNetworks 将推出 Firefox

 [](https://web.archive.org/web/20220117023341/http://www.mozilla.com/firefox) 未来两年 [RealNetworks](https://web.archive.org/web/20220117023341/http://www.real.com/) 和 [Mozilla](https://web.archive.org/web/20220117023341/http://www.mozilla.com/) 的合作将意味着下载 RealPlayer、Rhapsody 和其他真正软件的用户也将获得 [Firefox](https://web.archive.org/web/20220117023341/http://www.mozilla.com/firefox/) 浏览器。昨晚发布的公告称，发布谷歌工具栏的合作关系也已经正式确定。

这对火狐来说是个好消息。Real 表示，其软件每天的下载量超过 200 万次，但不清楚其中有多少是现在将包含 Firefox 的软件。

长期以来，Real 一直因允许更多令人讨厌的软件与其软件一起下载而受到批评，但这似乎是一个伟大的举动。尽管 Firefox 现在在美国有 16%的市场份额，在其他地方更高，但我确信仍然有很多人从未听说过它或者不知道如何获得它。

合作是扩大市场份额的一个很好的方式，只要他们小心翼翼。购买硬件后，软件变得容易获得，这似乎远不如某些形式的合作伙伴关系那么强制性，包括昨天在谷歌 IE 工具栏中发现的有争议的“bug”——它阻止用户改变谷歌的默认搜索。

Real/Firefox 的合作看起来很棒，我想这真的有助于扩展 Firefox 的使用。更新:一些读者说这是一个糟糕的举动，下载一个程序意味着他们只想下载一个程序——这也是有道理的。对于火狐来说，这可能会适得其反，因为火狐最有眼光的客户也是它的支持者。FF 是否放弃了通过忠诚的早期用户进行诚实的口碑营销的方式，转而采用一种更恐怖的方式来扩大市场份额？艰难的抉择。
