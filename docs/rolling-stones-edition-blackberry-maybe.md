# 滚石版黑莓:也许吧

RIM 对法庭并不陌生，它大部分时间都在捍卫自己的专利，有时赢，有时输。不过，针对黑莓手机制造商的最新诉讼来自一个不太可能的源头:滚石乐队。然而，这不是新闻。这就是为什么米克和他的公司让 RIM 的律师们感到恶心。RIM 似乎有一个滚石版黑莓的想法，支付了许可费，然后放弃了计划，抛弃了这个项目，从而放弃了滚石乐队本应获得的版税。

虽然我们认为这里的英国人言简意赅，但让我们看看更广泛的东西:黑莓是一款商务手机，仅此而已。没有 RIM 后端，它只是一部带有有趣键盘的手机。我们不明白的是滚石乐队为什么会说“我是一个受人尊敬的商人”也许是因为男孩们有亿万美元，但我们认为这是个坏主意。[iPod 上的披头士](https://web.archive.org/web/20210419031156/http://crunchgear.com/2007/05/11/beatles-potentially-might-just-want-to-hold-your-ipod/)，不是的。

所以如果石头在法庭上赢了，可能会有一个石头特别版 BB。太好了。你会买一个吗？我们不会。
