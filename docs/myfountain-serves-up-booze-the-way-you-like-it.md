# 我的店提供你喜欢的酒

![myfountain_485.jpg](img/a537abe0b166a8f637e910215ede39ab.png)

我通常在晚上喝得太醉了，以至于不能调制一杯合适的饮料，这通常会把我送到瓷器大王或最近的垃圾桶。如果 MyFountain 做到了它所说的，那么我就再也不用担心调饮料了。我的肝脏和室友会感谢我的，我敢肯定。台面分配器可容纳多达 12 种类型的混合器和酒精，包括特定的喷水口和碳酸线。

那么所有这些是如何工作的呢？嗯，一个个性化的主页允许你通过 Windows XP 机器上的触摸屏输入、查找和交换食谱。我想知道这种特殊的饮料是不是叫做 BSOD？明年春天再找吧，2600 美元左右。
