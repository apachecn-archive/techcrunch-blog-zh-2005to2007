# Xoxide PrimoChill 定制水冷盒

真正的硬核电脑改装者用水冷来制造他们自己的机箱，但是对于我们这些大学毕业并有工作的人来说，这个 Xoxide PrimoChill 可以为你做所有的水冷和机箱改装工作。

该案例是一个定制的 X-Qpack 与激光蚀刻窗口，紫外线 PrimoFlex 管，和紫外线防水板。它还配有一个紫外蓝色储器和一瓶紫外蓝色 PC 冷却剂，这意味着如果你把它储存在你睡觉的房间里，由于所有这些紫外光，你将会梦见超人的孤独堡垒。

该机箱是一个小型微型 ATX，适合大多数英特尔和 AMD 微 ATX 主板(查看网站上的列表)。现在你可以成为一名 OG-moder，而不用花那么多时间。只需支付 349.99 美元，你就可以了。
