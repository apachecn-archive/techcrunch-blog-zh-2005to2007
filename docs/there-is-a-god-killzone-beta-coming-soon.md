# 有一个神:Killzone 测试版即将推出

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/16/there-is-a-god-killzone-beta-coming-soon/>

![](img/2c70a186ea621831485d2faf5d932b10.png)

虽然最近很多 FPS 炒作都集中在即将到来的*光晕 3* 多人测试版上，但另一则消息刚刚宣布，它可能与《士官长》和 Xbox 360 相媲美。Guerilla Games 的 Seb Downie，负责 *Killzone* 系列的团队，已经在 Killzone.com 官方论坛上发帖，称公测肯定会发生。

> “当然会有公开测试。我一直说，任何有在线模式的游戏都需要经过公开测试，以确保它能按预期运行。”

考虑到第一个杀戮地带的多人游戏太烂了，这是个好主意。测试版可能会在 2007 年底推出，因为游击队团队正在狂热地完成“杀手级应用程序”,这可能会重振人们对奄奄一息的 PS3 的兴趣。只有时间能证明一切。
