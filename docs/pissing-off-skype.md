# 惹恼 Skype——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/01/08/pissing-off-skype/>

Skype 并不擅长与他人合作。这是一个我每天都在使用的很棒的服务，但是它是一个有围墙的地狱花园。它避开了开放的 SIP 协议，而这个协议是 T2 和其他公司已经接受的。它的即时通讯平台与 Meebo 和其他聚合器不兼容。这是一个孤岛，除了(非常重要的)警告，它与 POTS 网络配合得非常好，这意味着您可以使用 skype 拨打和接听手机和普通电话网络的电话。

我们仍在等待一个真正的 Skype 黑客的到来，它将使我们能够访问 Skype 客户端软件之外的内容。但是今天， [Charlie Paglee](https://web.archive.org/web/20220629190427/http://www.voipwiki.com/blog/?p=40) 宣布他们已经破解了 Skype 界面，并推出了自己的“Skype In”和“Skype Out”功能。黑客在 Skype 中添加了一个按钮，增加了 VOIP 功能。突出显示一个 Skype 联系人，然后单击按钮通过 TalkQer Voip 服务与其通话。

TalQer 有几个 Skype 没有的好功能，特别是免费的呼入电话号码和语音邮件。但还不清楚这是否足以说服大量 Skype 用户添加这种额外的软件。我的猜测是不会，但这可能会惹恼 Skype，足以阻止他们，并可能提起一两起诉讼。
