# 潜水相机包

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/23/submersible-camera-bags/>

水下摄影一直是一项相当昂贵的爱好，其外壳高达数百或数千美元。有了 Ewa-marine 包，你可以带上你的相机，花 100 美元就能射到水里，而你的单反相机只需要 350 美元。

显然，PVC 袋是防水的，耐用而灵活，所以你可以没有太多麻烦地按下相机的控制按钮。你也许不能调整每一个设置，但是只要你在下水前配置好了一切，你应该没问题。
