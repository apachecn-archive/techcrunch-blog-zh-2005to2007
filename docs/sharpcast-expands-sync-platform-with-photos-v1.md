# Sharpcast 通过照片 1.0 版扩展同步平台

[![sharpcast](img/03f32f3671751939c7a07770047d7d2c.png)](https://web.archive.org/web/20220929210704/http://sharpcast.com/) 如果你在多台计算设备上工作或娱乐——笔记本电脑、台式机和/或手机，Sharpcast 可以成为一个杀手级的生产力工具，也是保持各种文件同步的绝佳方式。在一个非常基本的意义上，它就像是你所有文件的 IMAP。

[Sharpcast](https://web.archive.org/web/20220929210704/http://sharpcast.com/) 走出 [beta](https://web.archive.org/web/20220929210704/http://www.beta.techcrunch.com/2006/05/30/sharpcast-opens-to-the-public/) 今天发布了他们的照片同步应用 1.0 版本。Sharpcast Photos 实时同步您在桌面、网络和手机上添加和编辑的照片。

新版本现在允许你实时地协作编辑你的照片和图库。协作包括在公共或私人文件夹中共享文件和读/写权限的能力，以及客户端内置的实时聊天。

我看到的 Sharpcast 实时同步的演示令人印象深刻，在不到一秒的时间内将文件更新推送到所有平台。在他们的应用程序、网络或你的桌面上编辑一张照片，所做的更改会立即被推送到每个地方。但是 Sharpcast 并不想成为下一个 Flickr，相反，照片同步应用程序实际上是他们的文件同步 API 的演示。在照片的特定情况下，整个照片被存储在每个位置，对原始照片的编辑被表示为关于如何渲染最终版本的元数据方向。

他们目前有一个更广泛的文件同步应用程序，蜂鸟，正在工作中。蜂鸟将处理所有文件类型的同步。它本质上与照片分享应用程序是同一种动物，但少了一些花哨的功能。Hummingbird 会注意到你的更新，并将其推送到你的其他平台，但不会对照片产品进行同样的实时更新。为了避免两个人同时编辑一个文件的问题，他们必须被保存，然后在其他地方打开进行修改。虽然 Microsoft Word 支持并发编辑，但 Hummingbird 目前还不支持。
