# TeleNav 抵达意大利| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/12/telenav-arrives-in-italy/>

TeleNav 刚刚宣布与 Softec S.p.A .达成协议，向意大利交付 TeleNav GPS 导航仪。这项服务将立即在各种黑莓智能手机上推出。![](img/269a8a4709abbfe8c24f7097381126a6.png)

你会记得，几周前，我在黑莓珍珠手机上查看了 T2 的电话导航服务。TeleNav GPS 接收器和服务可以将几乎任何支持蓝牙的手机变成一个方向敏锐的伴侣。

除了规划旅行方向，TeleNav 还可以定位自动取款机、WiFi 热点等。它可以在步行模式下使用，在街上闲逛时为自己导航。
