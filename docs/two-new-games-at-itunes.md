# iTunes 上的两款新游戏

苹果公司仍在经营游戏业务，今天悄悄向 iTunes 发布了两款新的 iPod 游戏。许多粉丝得知他们现在可以在 iPods 上玩 Soduku 会很兴奋。我们不知道这是如何工作的，但我们对史蒂夫和他的公司有信心，让它变得可玩。

现在也出了 Solitaire(不好意思，皇家纸牌)。多年来，iPod 一直附带一款纸牌游戏，但除了克朗代克备用游戏之外，皇家待遇还包括几个附加版本。

它们每首 5 美元，现在可以在你的 iTunes 上下载。虽然我们仍然认为 iPod 游戏是值得的，但还是有人喜欢它。当然，有些人喜欢梅格·瑞恩，但人无完人。
