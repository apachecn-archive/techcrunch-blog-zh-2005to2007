# 万维网联盟发布了 GRDDL 规范的第一个版本

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/25/world-wide-web-consortium-releases-first-version-of-grddl-specification/>

[万维网联盟](https://web.archive.org/web/20141102210742/http://www.w3.org/2006/10/grddl-pressrelease)今天宣布他们已经在[语义网](https://web.archive.org/web/20141102210742/http://www.w3.org/2001/sw/)和[微格式](https://web.archive.org/web/20141102210742/http://www.microformats.org/)社区之间建立了一个重要的联系，任何阅读这个博客的人都会知道我对它们有很大的兴趣。

通过“**从语言方言中收集资源描述**，或者 [GRDDL](https://web.archive.org/web/20141102210742/http://www.w3.org/TR/2006/WD-grddl-20061024/) (读作“griddle”)，软件可以自动从结构化网页中提取信息，使其成为语义网的一部分。

那些习惯于用 XHTML 微格式表示结构化数据的人可以通过移植到语义 Web 来增加现有数据的价值，使用 GRDDL 的成本非常低。
