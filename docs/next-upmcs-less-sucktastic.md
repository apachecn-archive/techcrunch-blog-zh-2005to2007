# 下一个 UPMCs:没那么难吃(？？)

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/29/next-upmcs-less-sucktastic/>

![](img/157aa69fa6bd8d8ddac8d2a3034efacc.png)

到目前为止，超便携电脑还相当愚蠢——它们小巧、可爱、无用——但下一代应该会好得多。他们保证。

例如，英特尔正在展示一款雅虎品牌的 UMPC，它包括一个用于延长电池寿命的单核处理器。他们还期待嵌入式 GPS 和无线以及通过 Sierra Wireless 和诺基亚的蜂窝网络。

我见过几个超级计算机，印象不深。再说一次，比起功能齐全的 mili PC，我更喜欢基本的互联网平板电脑，但也有一些人需要一台小巧的 XP 电脑。我只是不认为当前的作物——或者英特尔描述的这种假作物——是他们想要的。
