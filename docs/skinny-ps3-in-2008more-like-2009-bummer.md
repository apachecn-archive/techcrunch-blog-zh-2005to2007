# 2008 年的瘦 PS3 更像是 2009 年，Bummer 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/16/skinny-ps3-in-2008more-like-2009-bummer/>

PS3 有点沙哑大家都知道，尤其是索尼。下一代游戏机尚未在欧洲和澳大利亚推出，但索尼已经在努力开发下一代 PS3。

日本总部发布的一份声明称，索尼预计在 2008 年或 2009 年初从 90 纳米芯片过渡到 45 纳米芯片。我们都知道索尼在 PS3 上的记录，所以我打赌在 2009 年之前我们不会看到任何变化。芯片尺寸的减小相当于一个不那么矮胖的瘦身模型。对索尼来说，这是一个正确的方向，我们知道他们能做到——看看 PS2 的瘦身版就知道了。让我们祈祷，希望他们尽快完成过渡。
