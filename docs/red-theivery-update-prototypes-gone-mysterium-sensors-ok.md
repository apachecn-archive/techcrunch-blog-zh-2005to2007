# 红色 Theivery 更新:原型不见了，神秘传感器正常

，Red One 超高清数字电影摄像机的制造商。该公司的创始人詹姆斯·詹纳德通过 DVX 用户的帖子为我们带来了这一更新:

> 虽然被盗物品包括许多计算机、驱动器、显示器、原型(包括我们“闪亮”的铝 IBC 原型)、ID 文件和我们的 Scratch 系统……但窃贼似乎没有获得任何传感器、传感器信息或我们的主要“弗兰肯斯坦”拍摄相机。所有的红军回来后才把所有的碎片拼在一起。我们明天将恢复拍摄和测试。这将是最容易被认定为被盗的作品。

引用中的原型就是上面照片中的那个。这是一个好消息，因为红色的心脏是超级精确的神秘传感器，仍然在红色的拥有。瑞德正与当地警方合作寻找罪犯，我们将在此为您提供最新消息。
