# T-Mobile Wing 评论| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/22/t-mobile-wing-review/>

T-Mobile 新款 Wing 智能手机的制造商 HTC 就像是伟大的会话玩家。他们融入背景，放下切分音，几个喇叭，一些牛铃，而载体——这个小类比的奥蒂斯·雷丁——成为焦点。不过，不要为 HTC 哭泣。他们通过让别人发光来赚大钱。

以机翼为例。作为对 [Cingular 8525](https://web.archive.org/web/20230322164148/http://crunchgear.com/2006/11/06/cingular-8525-hands-on/) 和 T-Mobile MDA 的更新，Wing 有一个滑出式键盘，运行 Windows Mobile 6。与前代产品相比，Wing 要纤薄得多，与同类智能手机相比，它给人的感觉是更具大众市场，并且直接面向普通电子邮件用户。这不是一个公路战士的手机，虽然说战士可能会在周末使用它。
 ![](img/1c5341de7e8bfef59382740a13e9902a.png)

包裹着防滑橡胶——称为“柔软触感”——机翼的设计灵感来自另一款休闲 QWERTY 设备 [Dash](https://web.archive.org/web/20230322164148/http://crunchgear.com/2006/10/11/t-mobile-dash-hands-on/) 。因为它运行 Windows Mobile 6，所以用户界面非常干净，用户友好。T-Mobile 和 HTC 联手增加了一些非常好的功能，包括为许多运营商提供一步式电子邮件设置(Gmail POP 明显不存在，但可以通过高级配置菜单访问),以及一个任务管理器，它可以准确显示任何时候都在运行什么，确保你不会有两个日历应用程序占用宝贵的资源。T-Mobile 还包括一个 IM 客户端，支持 AIM、Yahoo 和 MSN。最后，它拥有使 Windows Mobile 6 成为更受欢迎的操作系统的所有应用程序，包括改进的 HTML 电子邮件渲染、改进的桌面同步和更好的浏览器。它有蓝牙、200 万像素的摄像头和 myFaves 支持。为了迎合其目标市场——休闲电子邮件和短信用户——这款手机的主屏幕上有一个 MP3 播放器。它还可以录制视频。

机翼没多少 3G。这是一款兼容 GPRS/EDGE 的四频手机，但它确实有 Wi-Fi 来实现更快的连接。Wi-Fi 系统在启用时会自动扫描热点，并在需要时跳转。

一个有趣的附加功能是手机左侧的一个“滑块”,取代了 Dash 奇怪的触摸滑块。这个只是上下跳动，允许你轻松地在列表中移动。

在过去的几个星期里，我爱上了机翼，又从机翼上掉了下来。作为一名前 MDA 用户，我理解折叠式 QWERTY 键盘的吸引力——这也是吸引我前往 Helio Ocean 的原因之一。这款手机重 6 盎司，感觉比看起来要重。尽管 IM 客户端不允许我用回车键发送消息，但输入起来还是很舒服的，这个小毛病可能是由某个地方的错误配置引起的。像 HTC 的大多数设备一样，Wing 有一个 USB 端口用于同步和充电。Mac 用户请注意:[WinMo](https://web.archive.org/web/20230322164148/http://www.markspace.com/)的丢失同步无法在 Windows Mobile 6 上运行。

使用 Wing 就像使用任何其他智能手机一样。Windows Live Search 是 WinMo 6 的一个补充，几乎没有用，但 Exchange 和 Outlook 连接肯定会受到 Windows 用户的欢迎。WinMo 6 是足够开放的，它可以做任何事情，幸运的是，T-Mobile 足够善良，在开箱即用的软件包中添加了一个很好的连接软件，这是许多 WinMo 实现所缺乏的。

我喜欢机翼。例如，我喜欢它胜过大海，仅仅是因为它的大小。海洋提供了媒体和乐趣，而机翼提供了媒体和乐趣的潜力，这当然是一个困难的概念，但请想一想。

总的来说，Wing 是已经强大的“休闲”智能手机系列的一个很好的补充。两年合约 299 美元，足够便宜来吸引那些可能被黑莓吓跑的人，而且尺寸肯定比 Treo 更便于携带。它击中了一个甜蜜点——一个蓝色的音符，如果你愿意 HTC 似乎总是能够到达，即使当它被卡在星星后面。
