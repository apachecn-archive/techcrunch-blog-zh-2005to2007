# TROLLTECH 发布绿色电话

Qt 和 Qtopia 的制造商 TROLLTECH 周一晚上在旧金山发布了绿色电话。Greenphone 提供了所有电话基本操作的源代码 API 调用，如拨号器、日历、地址簿等。Trolltech 表示，他们认为，通过开放源代码，而不是添加新的应用程序，可以大幅增强典型手机中的许多简单服务。“Greenphone 是第一款面向无限软件创新的 Linux 移动开发设备，”一位发言人表示。

手机本身包含 Qtopia Phone Edition，也可以作为正常工作的手机使用。Greenphone 测试型号具有触摸屏和键盘用户界面、英特尔 XSale 312 MHz 双核、64MB Ram，并配有蓝牙。TROLLTECH 表示，这款手机面向开发者、应用专家和制造商，非正式价格约为 700 美元。
