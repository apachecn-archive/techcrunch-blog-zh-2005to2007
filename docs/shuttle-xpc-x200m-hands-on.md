# XPC X200m 航天飞机动手

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/17/shuttle-xpc-x200m-hands-on/>

虽然我肯定有人认为苹果是“迷你”电脑的先驱，但 Shuttle Computers 长期以来一直是业界领先的小型电脑设计者和制造商。XPC X200 航天飞机是这种领导能力的一个典型例子。

尺寸为 2.1×8.3×11.8 英寸(HWD)，这个黑白相间的小盒子是一台功能强大的媒体中心 PC。X200m 运行在 Microsoft Vista Ultimate 上，采用 2.16GHz 英特尔酷睿双核 2 T7400 处理器和 2GB 667 MHz DDR2 内存，有足够的 pop 来处理多媒体任务以及您可能想要做的任何工作。(价格从 1149 美元起，但我的审查单元的配置目前是 1833 美元。)它还具有内置的标清电视调谐器、数字音频输出和用于媒体中心遥控器的集成红外接收器。

如果你只是在寻找一台小巧时尚的电脑，我推荐你选择 [X100](https://web.archive.org/web/20140308102634/http://sys.us.shuttle.com/X100.aspx) ，它失去了更多面向媒体的功能。但是，无论是为您的家庭娱乐系统添加 Windows Media Center 体验，还是拥有一台兼具办公和娱乐双重功能的 PC，X200m 都是一个可靠的选择。点击跳转查看更多图片和完整评论。

前面的事情都很简单；只有一个吸入式 DVD RW 驱动器、电源按钮和一个 USB 端口。然而，仔细观察会发现更多一点。

电源按钮下方的黑点是与捆绑的 Media-Center 遥控器配合使用的红外接收器的眼睛(跳至底部查看图片)。接收器是内置的，这很好，而不是选择简单的方法来摆脱一个丑陋的外部盒子，它会吃掉你的 USB 端口。另一个不错的触摸:虽然它看起来像是光驱开口的延伸，但红外眼下方的插槽是一个多格式存储卡读卡器。在它的正下方是一个活动灯，使用时会发出蓝色的光。不幸的是，当你让电脑进入睡眠模式时，它也会不停地闪烁。

后面也挺直白的。该天线用于内置 802.11g 无线电台。不过，如果你想增加一个更强大的天线或一个更容易隐藏的天线，它很容易拧开。

从左上顺时针方向依次是无线天线连接、SDTV 调谐器和 FM 收音机的射频连接、DVI 输出和电源输入。电源是一个外部电源模块，就像你在笔记本电脑上看到的那样。它很大，但很容易隐藏。电视调谐器也很好，但我们更喜欢高清调谐器。

在背面，你还会发现四个 USB 2.0 端口、模拟音频 I/O 插孔、一个用于数字音频的同轴 S/PDIF 插孔、千兆以太网以及出于某种奇怪的原因，用于鼠标和键盘的 PS/2 端口。我不知道你为什么要在这样一个现代、紧凑的媒体中心增加这些东西。我宁愿有一个 FireWire 或 eSATA 端口，但无论如何。

远程媒体中心票价相当标准。它确实与航天飞机的外观相匹配，如果你要充分利用媒体中心扩大的 10 英尺接口，你真的应该有一个遥控器。中间的绿色大按钮会让您立即进入媒体中心界面。

Vista 中对 GUI 的升级非常棒，X200m 及其集成的英特尔 GMA 950 图形适配器也很好地处理了操作系统的新 Aero 图形功能。整体表现很好，从音乐切换到观看或录制电视到浏览照片都很快。我的审查单位有 250GB 的存储空间，但你可以得到多达 750GB 的系统。这是一个很大的空间来存储越来越多的音乐、照片和视频收藏，尤其是与 160GB 的 Mac mini 相比。

我注意到的一件事是，在录制电视节目时，这个装置真的变热了，所以你要确保这个装置有足够的空气流通。编码视频时的性能比系统大小的预期要好。它无法与配有独立显卡的全尺寸台式机相媲美，但也没有理由指望它能做到这一点。

几乎没有人不喜欢身材娇小、性感的穿梭者 XPC X200m。是的，它是昂贵的。但这是获得如此小的东西的成本，当你要求它做最轻微的要求时，它不会冻结。如果你喜欢玩游戏或摆弄东西，你显然会想远离这台电脑。但是，如果您需要一台功能强大的台式机，它不会占用比一本大书更多的空间，并且具有管理您的媒体收藏的能力和功能，X200m 是一个可以考虑的选择。
