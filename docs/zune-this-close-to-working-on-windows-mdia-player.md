# Zune *差一点*就能在 Windows Media Player 11 上工作了

[![zunewmp.jpg](img/457d45ebe6b0dd36be4e6bd900439e58.png)](https://web.archive.org/web/20210423083955/https://beta.techcrunch.com/wp-content/uploads/2007/05/zunewmp.jpg "zunewmp.jpg")

微软[Zune](https://web.archive.org/web/20210423083955/http://crunchgear.com/2007/05/17/unboxing-the-halo-3-zune-almost-as-exciting-as-reboxing-it/)使用专有软件来管理音乐，无线电领域的一些人不一定喜欢使用这种软件。最终目标是能够在 Windows Media Player 11 上充分使用 Zune，独立程序员正在缓慢但稳步地接近这一目标。Zunerama 论坛上的一位用户 ronaldmonster 非常非常接近让 WMP 认出他的 Zune。剩下的就是说服 WMP 与 Zune 同步，这被证明比 Zune 粉丝想听到的更难。保持信心，先生们，因为在 WMP 使用你选择的便携式媒体播放器只是时间问题。

想想看，如果 iPod 用 iTunes 以外的东西来传输歌曲，人们会有多生气。不过，我相信微软有它的理由。
