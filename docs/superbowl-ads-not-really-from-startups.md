# 创业公司的超级碗广告(不是真的)

今天是——XLI 超级碗。全世界数以亿计的人将吃垃圾食品，喝啤酒，看一整年最好的电视广告，这些广告不时被足球比赛打断。

六家初创公司(Meebo、Meez、Multiply、Plaxo、RockYou 和 Technorati)无法支付比赛期间 30 秒广告的 250 万美元以上，他们一起制作了低预算的“超级碗”广告，并将其放在 YouTube 上。其中一些非常有趣。其他的，就没那么多了。我们已经将它们全部嵌入到下面，并进行投票，看看你最喜欢哪一个。

顺便说一下，去年真正的超级碗广告仍然在 video.google.com/superbowl.html T4 的谷歌视频上。我想谷歌明天会用超级碗 XLI 的广告取代那些广告。一旦比赛在 T5 开始，雅虎也将会托管真正的超级碗广告。

我最喜欢的“创业”广告是 Technorati，尽管他们使用了我最喜欢的电影中的片段进行欺骗。Plaxo 和 Meebo 并列第二。
