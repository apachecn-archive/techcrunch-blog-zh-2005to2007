# 夏末推出新款 iMacs

![](img/a1633748d5b6edf8f7c6f8437f2f27fd.png)

Appleinsider 独家报道了一些彻底重新设计的 iMacs，它们将于 7 月下半月至 8 月中旬发布。根据这篇文章，看起来在不久的将来还会有一些其他的惊喜给 Mac 粉丝。正如早期的传言所表明的那样，低端的 17 英寸 iMac 将被排除在大修之外，最终将走上 Powerbook G4 的道路。
