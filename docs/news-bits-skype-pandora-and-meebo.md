值得一提的是，今晚或明天早上会有一些涉及 Meebo、Pandora 和 Skype 的产品更新。

## Skype 添加了 Skypecasts

Skype 将于周三发布其客户端软件的 2.5 测试版。有一些很酷的新功能:向全球任何手机发送短信的能力，改进的电话会议(你可以看到谁在说话)，以及其他小的改进。

Skype 还推出了一款免费新产品 Skypecasts 的预览版，该产品允许多达 100 人进行实时、适度的 Skype 对话。主机可以控制静音、弹出和“麦克风通过”。Skypecasts 可以通过 html 片段预先安排和推广。

Skype 仍然缺乏录音功能，虽然有一些好的第三方产品来弥补这一差距，但似乎没有一个产品能在新的英特尔 MAC 上工作。

## 潘多拉:180 万会员，新的社区反馈功能

 [![](img/c9c0e8b1cec36e072b6db9008f09bc0f.png)](https://web.archive.org/web/20221006173226/http://www.crunchbase.com/company/pandora) 音乐探索网站 [Pandora](https://web.archive.org/web/20221006173226/http://www.pandora.com/) 将于今晚或明天宣布其拥有 180 万用户。

在过去的几个月里，这些客户给了潘多拉多达 7000 万首歌曲的反馈。Pandora 还将宣布，他们将开始使用这一社区反馈来改进用户的播放列表，并提供更多相关的音乐。

## Meebo 聊天记录

 [![](img/d8b4874c19c39ce3b2ca60ab6caee03f.png)](https://web.archive.org/web/20221006173226/http://www.meebo.com/) 网络聊天初创公司 [Meebo](https://web.archive.org/web/20221006173226/http://www.meebo.com/) 一直在定期推出新功能。二月份，Meebo 给用户提供了创建账户的能力。本周的某个时候，他们将宣布用户将能够访问聊天记录。他们还将宣布增强的用户偏好，更好的图标等。
