# 道路天使移动帮助你避免没有乳沟的票

具有 GPS 功能的手机变得越来越普遍。GPS 的作用不仅仅是告诉你你在哪里，而是帮助你自动地与你周围的环境互动。英国的 Road Angel Mobile 正在使用手机 GPS 来帮助你避免超速罚单。

随着软件应用程序加载到您的诺基亚手机上，该手机与兼容的 GPS 单元配对，Road Angel 会在您驾驶时跟踪您，并提供您当前的速度。更重要的是，它提醒你接近雷达照片速度陷阱。我们还不确定这是否合法，但这确实很酷。

不知道他们是否计划在美国推广这项服务(不太可能)，但在一家国家运营商的帮助下，这种服务在这里发展不会太难。Cingular 或威瑞森会同意这种合法的恶作剧吗？希望如此，因为我的那辆起亚跑得比你想象的要快。
