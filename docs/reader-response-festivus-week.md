# 读者反应:节日周

![](img/c50bcfd975698716549246ba6a335cc8.png)
正如我之前提到的，读者对未来一周的反应将会很大。我们将在下周的每一天颁奖，而不是在周五颁发一个大奖。好家伙，他们是好奖品。

因为 Festivus 在下周，我们决定全力以赴，送出一些我们有史以来最好的奖品。事实上是如此之好，以至于你不得不跳着去阅读它们:

这就是它的工作方式。下周我们有五个奖项，每天一个。一如既往，周五的奖品将是最大的，但周一至周四的奖品也不逊色。

![](img/9bc889cae25f01f153fed7bfae2bd5b8.png)
我们将赠送四个闪迪闪存驱动器。两个 1GB Cruzer 微型硬盘和两个 1GB Cruzer Titaniums 硬盘。我们将首先赠送微型电脑，然后赠送钛型电脑。他们将每天获得奖励。

现在，我说我们下周有一个大奖，我不是在开玩笑。下周的大奖是价值 600 美元的全新[尼康 D40 数码单反相机套装](https://web.archive.org/web/20201202192418/http://www.nikonusa.com/template.php?cat=1&grp=2&productNr=25420)。而且大家都知道，[D40 是一款非常坚固的相机](https://web.archive.org/web/20201202192418/http://crunchgear.com/2006/11/29/holiday-buyers-guide-2006-dslr-cameras/)。该套件包括 600 万像素的 D40 和 3 个 18-55 毫米镜头。我真的无法告诉你们我们有多激动，能把这些送给你们，我们亲爱的，亲爱的读者。

![](img/725ea81aae5b1067170095dd6bcc6af8.png)
