# 最方便的银行提供手机版

自称为美国最便捷银行的商业银行推出了一个虚拟私人银行报告移动博客，使其成为第一个基于网络的私人银行平台。商业客户将能够通过黑莓或 Treo 等 PDA 智能手机或即将推出的 iPhone 访问他们的账户。

根据该公司的新闻稿，“这项技术为客户提供了他们所有资产的最新财务状况，并组织其他重要信息——如股票跟踪、医疗和保险信息、账号、世界新闻、天气、体育甚至他们最喜欢的葡萄酒。”这就是方便！
