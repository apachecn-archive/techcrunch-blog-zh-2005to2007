# 斯巴达 DX:为 DNA 测试而来，为 iPod Nano | TechCrunch 而留

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/13/spartan-dx-come-for-the-dna-testing-stay-for-the-ipod-nano/>

DNA 测试器在历史上一直是笨重的设备，背负着沉重的说明书，甚至更重的价格标签。我有一个女朋友，她曾经做过 DNA 测试，看起来她正在用一个鹰嘴咖啡机制作浓缩咖啡。那么，对于一个看起来像 VHS 磁带倒带机、售价 15000 美元的独立设备，你会怎么说？如果你是一名生物化学家，你可能会说“帮我报名吧！”

否则，你很难找到它的用处。我要弄清楚这是否能改善法医学甚至是系谱学，这将是未来一个巨大的潜在社交网络迷因(“GenrRater.com 寻找和评价具有相同等位基因的朋友！”)但是最精彩的部分呢？最精彩的部分？每次购买都可以获得一个免费的 iPod Nano。你能说超级棒吗？
