# SNES/NES 控制器 Wii 模块

![](img/c163fd98e263504a3614e23e258cfefc.png)

我不做任天堂 Wii 这种事情，但是我知道如果我在虚拟主机上下载一些游戏，我会想使用一个实际的 SNES 或 NES 控制器来真正增加复古游戏的体验。看起来 modder Raphael 也有同样的想法，因为他将一个 SNES 控制器接入 USB 电路，以连接到他的 Wii。如果你认为这就像插上一些适配器，摇一摇就行了，那就再想想吧。这个项目完全可行，但有点棘手。

你当然可以只买一个 [SNES 转 USB 适配器](https://web.archive.org/web/20201129112135/http://www.amazon.com/Nintendo-Smartjoy-Controller-Adapter-Converter/dp/B0009VIZSY)，下载 [SNES9x](https://web.archive.org/web/20201129112135/http://www.snes9x.com/) 和一些 rom，然后免费尽情玩。但这是有道理的。你的电话，老虎。
