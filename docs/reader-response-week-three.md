# 读者反应周三

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/11/reader-response-week-three/>

![](img/a50f1761c7f425fc158112ae88032fb2.png)
问候大家，希望你们有一个美好无聊的星期一。我知道我都快撑不住了。不过你很幸运，因为我拿到了我们读者反应系列的最新一期。

到目前为止，我们已经送出了[一个 2G iPod Shuffle](https://web.archive.org/web/20160315160344/http://crunchgear.com/2006/11/29/commenter-appreciation-begins/) 和[一个三星 21 点](https://web.archive.org/web/20160315160344/http://crunchgear.com/2006/12/08/reader-response-week-two/)，此外还有 50 份 [M2 转换](https://web.archive.org/web/20160315160344/http://www.google.com/url?sa=t&ct=res&cd=1&url=http%3A%2F%2Fwww.m2solutionsinc.com%2F&ei=b6d9ReKMKouGwQKQ6fCVCA&usg=__MRviTK-J3L4RsH7on14JOX4BV3Y=&sig2=ba4EO-aKJetYOA7L0YGqBg)。我们将在本周和周五再送出 50 份 M2 Convert，正好赶上那些新年决心，一位幸运的赢家将带走一个 [Nike + iPod](https://web.archive.org/web/20160315160344/http://www.apple.com/ipod/nike/) 适配器。

为了让你对它的运动效率有个概念，它单枪匹马地把比格斯从一团看不见的凝胶状物质变成了一团稍微不那么凝胶状的东西。你怎么能对这些结果提出异议呢？

规则一如既往。要赢，你只需在本周发表评论，并在输入你的名字时输入一个有效的电子邮件地址。我们将在周五使用我们的超级秘密公式选出一个获胜者(好的，实际上我们只是看看本周前几个评论者，然后看看他们中谁的评论最有见地)。
