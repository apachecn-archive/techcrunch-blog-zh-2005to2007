# 希捷免费代理 Go 160GB 动手

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/03/seagate-freeagent-go-160gb-hands-on/>

![freeagent.gif](img/59c1868f44c182f705d8c70588167791.png)

看着希捷的 FreeAgent Go 160 GB 这样的设备，一般用户都会打哈欠。存储？160GB？Feh。无聊。

但是希捷有一些锦囊妙计。实际上，免费代理 Go 是你的电脑环境和个性在你的手掌中。这是一款存储产品，专注于经常旅行、移动位置、携带 iPod、数码相机拍摄、录像的人。一般来说，这是一款专为我们这些极客打造的设备。
 ![free2.jpg](img/33e6ad5777a631597e61efab9499e9e8.png)
但真的只是一个便携式存储单元，对吗？是的，坏了，但这是一个带有主要道具的存储设备。这里的想法是把你的电脑带在身边。就是这样，整个计算环境包括网络收藏夹、在线密码、设置、软件程序、您的即时消息、电子邮件联系人、数字音乐和图片文件，以及您 PC 上的任何类型的数据。想想这个；我们去的每个地方通常都有一台电脑，为什么要随身携带呢？将它带到半个世界之外或街对面您朋友的家里，将完全同步的 FreeAgent Go 插入任何 Windows USB 2.0，瞧，您就拥有了自己的 PC。你听到我说的了，你电脑里的所有内容，都在硬盘里。。

你说，明白了。我这么做是因为……？这不是一个通过网络访问你的电脑的应用程序，这是你的电脑。因此，你不用随身携带笔记本电脑，而是随身携带免费的 Go。这实际上变成了你的笔记本电脑(或台式机),然后你可以安全地使用任何计算机和互联网连接。它很小，只有 6 盎司重。

那么你在盒子里得到了什么？你得到了好看的 FreeAgent GO mover 驱动器，并预装了软件和电子文档。它甚至配有专有的 USB 电缆和快速入门指南(实际上只有图片)。系统要求是 Windows XP 家庭版或专业版，或 Windows 2000 专业版，你需要两个开放的 USB 端口。它支持 OS X，但没有一款免费软件能在 MAC 电脑上运行。

该产品有五年的有限保修，这是杰出的在线技术支持。

所以现在，让我们试驾这只小狗。我打开了 FreeAgent Go 数据移动器(主机)和 USB 插头。请注意，USB 是 Y 型插头，有两个 USB 插头用于计算机侧，一个用于 FreeAgent Go 数据移动器。好了，我建立了连接，data mover 指示灯亮起，告诉我握手已完成。接下来发生的事情是在电脑屏幕上弹出自由球员的页面。非常直观，一会儿我就安装了 FreeAgent 工具，然后我就可以开始工作了，就像使用 iPod 一样简单。现在，我可以用你的电脑吗？

几分钟后，我成了一名携带数据的数据移动自由代理人。书桌不再装订；我可以自由地漫游世界，不断地要求使用别人的电脑来打扰他们。对我来说，好消息是这是一个很大的世界，留给我很多人去烦，呃…问。顺便问一下，有人想买我的笔记本电脑吗？
