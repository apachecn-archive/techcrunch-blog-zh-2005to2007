# 我所谓的第二人生事件

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/25/my-so-called-second-life-event/>

我有幸被 [NMK](https://web.archive.org/web/20150918230641/http://www.nmk.co.uk/) 邀请到[参加一个为期半天的研讨会](https://web.archive.org/web/20150918230641/http://www.nmk.co.uk/event/2006/10/24/mmorpg-media-space)，讨论大型多人在线角色扮演游戏(MMORPGs)和虚拟世界(当然主要是[第二人生](https://web.archive.org/web/20150918230641/http://www.secondlife.com/))的新 3D 世界现象。

如果不是我主持会议，我本可以做更好的笔记，我会尽快就此写点东西。然而，像往常一样，观众中也有一些伟大的博客作者，其中艾伦·帕特利克做了非常全面的报道。我们还在那里见到了内容丰富、近乎传奇的亚当·路透。

对我自己来说，我从讨论中挑选出的一个突出的发展——有很多——是像 MySpace 这样的社交软件网络与 Second Life 的最终融合。

这方面最显著的例子是(也可能是第一个)——Playahead[中的用户能够将他们的个人资料翻转到第二人生的虚拟角色中，而他们所有的偏好和网络都保持不变。对我来说，这可能是非常革命性的，我不会轻易使用这个词。想象一下，如果人们把他们的 MyPSace 个人资料甚至 LinkedIn 或博客翻到“第二人生”会怎样？由于林登实验室将使整个项目成为一个开源项目——同时保留对关键的名称空间设施的控制——我们很可能在不太遥远的将来都能够做到这一点。](https://web.archive.org/web/20150918230641/http://www.playahead.co.uk/)
