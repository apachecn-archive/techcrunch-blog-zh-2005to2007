# 三星 Z610 iPod 风格的电池出现在 FCC 上

三星 Z610 从 iPod 身上汲取了一些造型元素，比如白色机身、圆形边框和光滑的塑料外壳。它还可以播放 MP3/WMA/AAC 文件，有一个 200 万像素的摄像头，一个微型 SD 扩展槽，甚至在前面还有一个用于视频通话的辅助摄像头。那么这是否意味着它支持 3G？你打赌你的屁股它会！

不知道哪个服务将得到这款手机，但当我们知道更多的时候，我们会让你更新。
