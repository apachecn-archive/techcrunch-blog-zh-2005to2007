# Skype PocketPC 2.1 测试版发布

这里有一些严重的问题。Skype 推出了 PocketPC 的 2.1 测试版，包括:

*   一键 Skype 访问:设备主屏幕上的 Skype 图标允许用户轻松查看和呼叫联系人。*   多人聊天:用户可以使用动画表情符号与同事、朋友和家人进行多人聊天，当移动用户离线时，聊天消息将在用户下次登录时自动更新。*   增强的联系人列表:用户现在可以看到他们的联系人头像和心情信息。*   个人资料个性化:用户可以用移动设备上的相机拍照，并立即更新他们的个人资料。

没有什么惊天动地的，但你已经得到了一些用户界面的调整和联系人列表的一些明确的改进。
