# Wink 拿出半个 Odeo，部分清算

 [](https://web.archive.org/web/20221201174821/http://wink.com/) 搜索引擎初创公司 [Wink](https://web.archive.org/web/20221201174821/http://www.wink.com/) 已提出用剩余现金从投资者手中回购股票，价格为面值的 50%，据该公司相关人士透露。

迄今为止，该公司已经筹集了两轮总额为 700 万美元的融资。我们的理解是，一些投资者选择根据这些条款将他们的股票卖回给公司。Wink 的主要投资者 Greylock 减持了该公司的股份，但仍是其最大的外部股东。

Wink 的计划将是整合其剩余现金，专注于去年秋天推出的[人物搜索引擎。](https://web.archive.org/web/20221201174821/http://www.beta.techcrunch.com/2006/11/10/wink-now-searches-myspace-linkedin-and-beebo/)

我们听到了关于为什么会发生这种情况的两种说法。来自心怀不满的股东的第一种说法是，该公司未能执行，是时候将剩余资本返还给投资者了。公司推动的另一个故事是，他们只是做出了改变产品方向的战略决定，并为投资者提供了一条出路，因为这不是他们最初购买的故事。这两种说法可能都有一部分是对的，尽管很明显，除非格雷洛克是幕后黑手，否则不可能强行清算。而格雷洛克，尽管他们已经卖掉了一些股票，似乎仍然支持该公司。

Odeo 最近也从投资者手中买回。在那种情况下，公司被完全私有化，外部股东的初始投资得到了 100%的补偿。这看起来是创始人伊万·威廉姆斯和比兹·斯通的明智决定。虽然 Odeo 已经被挂牌出售，但 Twitter 已经实现了爆炸式增长，人气也非常高。

Wink 也会成功吗？那是用户自己决定的。但是组织包含在大型社交网络中的大量人类元数据可能是一个明智的选择。
