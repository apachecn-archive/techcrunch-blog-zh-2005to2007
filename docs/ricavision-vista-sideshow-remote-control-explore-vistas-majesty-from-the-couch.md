# Ricavision Vista SideShow 遥控器:从沙发上探索 Vista 的威严

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/22/ricavision-vista-sideshow-remote-control-explore-vistas-majesty-from-the-couch/>

![](img/7ebeafff4c30840993812e9c278b581c.png)

Windows Vista 的众多亮点之一是 SideShow，这是一种将电子邮件等信息输出到电脑显示器以外的屏幕上的技术。使用 SideShow，这款 Ricavision 蓝牙遥控器可以让您在 Vista 电脑的媒体库中导航、安排电视录制和阅读电子邮件，而无需离开沙发(只要您在电脑的 328 英尺范围内)。遥控器的 2.5 英寸 QVGA 液晶显示器足够大，可以显示所有的信息，而不会使你的视力疲劳；谁不想不用笨重的鼠标和键盘来控制音乐呢？

当然，这样一个昂贵的遥控器要 200 美元，但是想想邻居们会多么嫉妒，当他们看到你的遥控器不仅有一个屏幕，而且还能与你的 Vista 电脑互动。我想他们只会说“哇”
