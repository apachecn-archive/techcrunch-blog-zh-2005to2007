# 我的酷朋友 iHolic

韩国的 Dacos 公司推出了一款支持 DMB 的新 mp3 播放器。我知道我知道，我们不能在这里得到它，所以为什么还要麻烦告诉你。因为我喜欢在工作中激怒你们所有人！这就是为什么，有问题吗？:)

iHolic 配备了一个 2.4 英寸的 TFT LCD，这似乎是屏幕尺寸的新标准。它踢了一些严重的屁股，因为它自带内置外置扬声器和遥控器的扩展坞。正如我之前所说，它支持 T-DMB，视频播放，图像浏览器，SD 卡插槽，它记录你的咆哮和狂欢。它是一个 3.15x2x0.7 英寸的小家伙，重 76 克。

512MB、1GB 和 2GB 三种型号可供选择。2GB 型号的售价约为 200 美元。
