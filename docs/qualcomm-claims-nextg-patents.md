# 高通声称 NextG 专利| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/11/qualcomm-claims-nextg-patents/>

据 mocoNews 报道，高通正在申请 1000 多项基本专利。这些包括 OFDM、OFDMA (OFDM 接入)和 MIMO(多输入多输出)技术的专利，这意味着下一代移动技术可能由高通控制，该公司要求对使用这些技术的任何一项收取专利费。高通负责投资者关系和国际营销的高级副总裁比尔·戴维森说，WiMax 是一个例外，戴维森认为它不如其他移动技术好。
