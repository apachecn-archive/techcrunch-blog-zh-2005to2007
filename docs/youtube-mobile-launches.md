# YouTube 移动发布

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/16/youtube-mobile-launches/>

![](img/04f2472fc171824230540b7e05193c72.png)
YouTube 刚刚推出了针对移动设备优化的新服务。几天前我们提到了第五代爱可视播放器是唯一可以流畅播放 [flash 视频的移动设备。也许这能给阿奇一个机会。](https://web.archive.org/web/20141019050659/http://crunchgear.com/2007/06/14/archos-announces-405-605-and-tv-plus/)

新的 YouTube 移动网站主要是 xHTML，需要一个非常强大的数据计划，它还支持通过 MMS 上传视频给三个 vBloggers。现在你可以在旅途中一遍又一遍地观看[这个](https://web.archive.org/web/20141019050659/http://youtube.com/watch?v=F80nn5sXN1o)发生。
