# 苹果专利展示简单的类似纳米的 iPhone

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/06/apple-patent-shows-simple-nano-like-iphone/>

一个令人恶心和侮辱的现实是，有些人不愿意把互联网揣在口袋里到处走。然而，这些人确实喜欢听音乐和打电话。有什么比播放音乐的手机更好的一举两得的方法呢？

这根本不是什么新想法。我记得在 2000 年末，我是第一个也是最后一个花 400 美元买下三星骚动手机的人，听人们大声问手机怎么可能变得更先进。

手机变得更先进了。但有一件事永远不会完美，那就是一个易于使用的音乐播放器和一部易于使用的手机。如果有一家公司可以做到这一点，它可能是苹果公司。

苹果提交的一份专利申请展示了这样一款手机。它基于类似 iPod Nano 的外形，将由一个简单的电话/音乐播放器组成，并带有一个触摸板，可以根据手机的当前模式显示不同的符号。

![iPhone nano](img/3d0db9abc023311031f04251d89bddaf.png)

虽然我链接的文章提到这款手机对你的祖父母来说是完美的，但我倾向于认为，对于那些甚至不应该被允许驾驶机动车辆的人来说，这款手机可能仍然有点先进。

我真的认为这对那些厌倦了随身携带手机和 iPod 的人来说很棒。这对年轻的全职妈妈和像我父母这样不需要数据访问但仍想看看便携式音乐有什么大惊小怪的人来说可能会很畅销。

发送短信的易用性也是需要关注的。如果苹果在这方面做对了，那么青少年市场的闸门也会打开。
