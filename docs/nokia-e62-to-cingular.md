# 诺基亚 E62 到 Cingular 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/12/nokia-e62-to-cingular/>

![](img/9fb90b67ae95882d20400aa8d499eee6.png)

诺基亚 E62 的细节已经流传了一段时间，但更多的细节是在今天早上宣布的。它被吹捧为非常受欢迎的摩托罗拉 q 的主要竞争对手。E62 具有完整的 QWERTY 键盘和 1600 万色，320 x 240 分辨率的屏幕。它兼容许多移动电子邮件选项以及 POP3、IMAP 和 SMTP。它还预装了几个即时通讯应用程序。

用户还可以查看、编辑和创建文档、电子表格和演示文稿。它采用了诺基亚的 S60 浏览器。不幸的是，它不支持 HSDPA 或 UMTS，但无论如何，它很热。它将于 9 月 29 日以“低至 149 美元”的价格上市。
