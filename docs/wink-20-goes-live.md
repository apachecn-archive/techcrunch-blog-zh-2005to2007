# Wink 2.0 上线 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2006/09/06/wink-20-goes-live/>

 [](https://web.archive.org/web/20221205091416/http://www.wink.com/) 社交搜索公司 [Wink](https://web.archive.org/web/20221205091416/http://wink.com/) 刚刚上线他们重新推出的网站，更好地利用合作建立的收藏，提供 Firefox 工具栏和其他变化，我们在[的早期预览](https://web.archive.org/web/20221205091416/http://www.beta.techcrunch.com/2006/08/25/wink-20-to-launch-next-week/)中详细介绍了这些变化。Wink 是一家聪明、资金充足的公司，在一个潜力巨大的领域提供有用的服务。这次重新推出并不是非常令人兴奋，但它确实使社交搜索的主要参与者之一比以前更加有用。这家公司值得长期关注。
