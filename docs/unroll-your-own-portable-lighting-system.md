# 展开你自己的便携式照明系统

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/07/unroll-your-own-portable-lighting-system/>

![metrolights](img/3dcda443b9116d80d73b371cd9915047.png)

如果你是需要便携式照明解决方案的数十人中的一员，你一定会喜欢 Lampada Metro 卷帘照明系统。

这些灯开始看起来有点像一卷胶带，然后被分解成一个像品客薯片一样的半透明管，发出微弱的光。

如果你想使用这个，请让我们知道如何或为什么。我没有取笑他们，我真的很好奇这个产品的目标受众是谁。
