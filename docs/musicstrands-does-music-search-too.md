# MusicStrands 也做音乐搜索。

如果[谷歌音乐搜索](https://web.archive.org/web/20210422175447/http://www.beta.techcrunch.com/2005/12/15/google-music-search/)不适合你，那就去看看 [MusicStrands](https://web.archive.org/web/20210422175447/http://www.musicstrands.com/) ，这是一家年轻的公司，昨天[重新推出了网站](https://web.archive.org/web/20210422175447/http://blog.musicstrands.com/?p=32)。该公司总部设在科瓦利斯，或(那是哪里？)和西班牙巴塞罗那。

MusicStrands 对我投入的每个乐队都有结果。大量的统计数据，以及从亚马逊和 iTunes 购买音乐的链接。它还拥有出色的 web 2.0 功能，包括社交网络、音乐推荐和用户标签。他们还为小型和独立乐队提供了很好的支持选项，允许他们将自己的音乐融入混音中。基本上，MusicStrands 是寻找新音乐的绝佳资源。

**更新** : [克利姆](https://web.archive.org/web/20210422175447/http://25thdimension.blogsome.com/)在[评论](https://web.archive.org/web/20210422175447/http://www.beta.techcrunch.com/2005/12/15/musicstrands-does-music-search-too/#comment-5418)中指出，谷歌与 MusicStrands 不同，也做歌词搜索。说得好。
