# 雅虎的 Webzari 将链接搜索可视化

 [](https://web.archive.org/web/20220701023603/http://yahoo.com/) 雅虎！韩国团队推出了一项名为 [Webzari](https://web.archive.org/web/20220701023603/http://kr.webzari.search.yahoo.com/search/webzari?p=http://www.beta.techcrunch.com/2005/11/06/the-flickrs-of-video/&b=30) 的服务，用于可视化任何网页的入站链接权重，并在雅虎中保存可视化查询历史的社会化书签服务。界面是朝鲜语的，还不完全清楚它的效果如何，但是基本的功能集是可以辨别的，这是对未来的一个有趣的想法。雅虎！搜索团队[在其博客](https://web.archive.org/web/20220701023603/http://www.ysearchblog.com/archives/000333.html)上寻求对项目的反馈。

一半有用，一半有趣， [Webzari](https://web.archive.org/web/20220701023603/http://kr.webzari.search.yahoo.com/search/webzari?p=http://www.beta.techcrunch.com/2005/11/06/the-flickrs-of-video/&b=30) 加入了大供应商最近的可视化计划的行列。可视化不再只是小家伙的专利了。上周发布之前，我们介绍了新的 Digg Labs 可视化工具。

这里的想法是，雅虎！Site Explorer 会给你一个链接到一个 URL 的站点列表，但是 Webzari 可以让你很容易地看到每个站点的链接或页面有多大，以及它们的地理位置。它可以像 Technorati 链接搜索一样工作，在 flash 中显示“权威”和地理信息。相当不错。

我向“视觉思维公司”的概念开发人员达纳·史密斯询问了 Webzari，她告诉我这是一个很好的可视化的例子，帮助用户比单纯的文本更快地“看到森林和树木”。她说，如果节点包括网站标志或其他一些独特的图像，服务可以得到改善，因为“每个实体的独特性使可视化真正强大。”我认为很多可视化项目都没有关注这一点。

我一直使用 Technorati 搜索链接，我通常想知道我的搜索结果中每个网站的入站链接数量。Webzari 的搜索结果的可视化表示是非常有用的，我很想看到英文版的 Yahoo！
