# 老牌英国媒体集团登上混搭 cluetrain | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/24/old-uk-media-group-gets-on-the-mashup-cluetrain/>

据[媒体卫报](https://web.archive.org/web/20170713061452/http://media.guardian.co.uk/advertising/story/0,,1927833,00.html)报道，区域媒体集团 [Archant](https://web.archive.org/web/20170713061452/http://www.archant.co.uk/) 将通过一个名为【24.co.uk】[家园](https://web.archive.org/web/20170713061452/http://www.homes24.co.uk/)的房地产网站来增加其分类广告，以对抗广告收入向在线竞争对手的流失。

现在，通常情况下，TechCrunch 英国公司对这一点并不感兴趣，主要是因为大多数时候，报业集团只是简单地发布 Web 1.0 铲状软件，并在网上转载纸质广告。然而，这一次，Archant 似乎让 brain 参与了战略，并在阿姆斯特丹雇佣了自称为分类广告和社区软件开发商的 Rosetta。

其结果是一个谷歌地图混搭，允许购房者识别和放大可用的房产。为了资助开发，Archant 为房地产经纪人提供免费广告，只有当消费者通过网站组织看房时，他们才付费。

这项服务首先在诺福克推出，毫无疑问也会在伦敦推出，因为阿坎特还在伦敦北部经营火腿连锁店。事实上，阿坎特的旗舰报纸[东方日报](https://web.archive.org/web/20170713061452/http://new.edp24.co.uk/)甚至正在播放[视频公告](https://web.archive.org/web/20170713061452/http://new.edp24.co.uk/content/videoconsole/console/tvconsole.htm)，这些公告似乎都在 Quicktime 上。

也许应该有人告诉他们，他们应该让视频可以共享，这样就可以在全球范围内传播关于诺福克的突发视频新闻了？
