# Olympus Stylus 780:基本的 710 万像素傻瓜相机

奥林巴斯最新的傻瓜数码相机“全天候”Stylus 780 保留了去年秋季 Stylus 750 的许多基本功能(710 万像素图像传感器、2.5 英寸 LCD 和 5X 光学变焦)，同时增加了一些旨在从相机中挤出最后一丝美好的功能。它采用了 TruePic III 图像传感器(就像它更先进的兄弟一样， [E-410](https://web.archive.org/web/20210303000833/http://crunchgear.com/2007/03/05/olympus-e-410-digital-slr-for-pros-and-joes/) DSLR)来提高图像质量，并采用了双图像稳定器来确保你严重颤抖的手不会破坏完美的拍摄。同样值得注意的是奥林巴斯所谓的完美拍摄预览，它可以让你在观看 LCD 上的照片时摆弄各种相机设置:太暗，调亮；白平衡有偏差，调整到您满意的程度，等等。这种模式也是为了帮助新手用户习惯数码相机的行话，而不会在这个过程中永久性地破坏他们的任何镜头。

期待今年 5 月 349 美元的价格。只是不要找光学取景器，因为对于相机新手来说，它们太热了。
