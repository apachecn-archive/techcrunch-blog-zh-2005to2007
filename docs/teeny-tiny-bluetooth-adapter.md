# 极小的蓝牙适配器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/16/teeny-tiny-bluetooth-adapter/>

所以你还没有准备好使用一台新的蓝牙电脑，这没关系。你可以买一个 USB 加密狗，这样你就准备好了，但是你看起来很傻，因为蓝牙加密狗对于笔记本电脑就像蓝牙耳机对于你的脑袋一样:choad-making material。

Newton Peripherals 的出现让你的生活变得更轻松。它的 Mogo 是你保持时尚所需的蓝牙适配器。大约指甲盖大小，这是你能得到的最小的 USB 蓝牙适配器。留点空间留点面子(我应该去做营销)。
