# 雅虎！答案启动 API

 [](https://web.archive.org/web/20220119231706/http://answers.yahoo.com/) 大获成功的[雅虎答案](https://web.archive.org/web/20220119231706/http://answers.yahoo.com/)正在通过[今天发布](https://web.archive.org/web/20220119231706/http://blog.360.yahoo.com/blog-d8pH0dcoRKeB12yOcnUQp.9VCFos?p=8971)[的](https://web.archive.org/web/20220119231706/http://developer.yahoo.com/answers/)将它的全球用户共享知识社区推向一个新的高度，这是一个为对访问庞大的雅虎感兴趣的外部开发者准备的 API！答案网站。该公司表示，现在系统中有 3000 多万个答案，从有趣的到有用的都有。Answers 还拥有一个非常忠实的用户群。

开发者将能够通过用户、搜索关键字或类别访问 YA 数据。该公司表示，这种 API 与内部用来创建最新雅虎的界面几乎完全相同。信使/雅虎！回答组合和闪光徽章，雅虎！优惠。
