# Navio 902T2 媒体中心

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/15/navio-902t2-media-center/>

![navio902.jpg](img/79174ab37d09bf1c6a537cc9000be016.png)如果您正在寻找一个基于 PC 的高端媒体中心，Navio 902T2 配有 7 英寸触摸屏 LCD、1TB 存储空间、2GB RAM、奔腾 D950 双核处理器和用于存档所有节目的 DVD 刻录机。再加上 Sigmacom X1 高清电视调谐器和 GeForce 7900GTX，它将为玩电脑游戏和电影提供足够的动力。

即使有所有这些高端部件，4995 美元的价格似乎也有点高。7 英寸的液晶显示器是一种奢侈品，可能不会有太大的用处，因为您将使用实际的电视来观看 Media Center 2005 界面，并使用遥控器来控制它-使触摸屏成为一种浪费的功能。
