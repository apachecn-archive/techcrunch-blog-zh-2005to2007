# 微软 Zune 已经有了中国山寨版

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/15/the-microsoft-zune-has-a-chinese-knockoff-already/>

Oriphe Industrial Limited 显然在几个月前看到了泄露的微软 Zune 照片，并迅速投入工作。我们所说的工作是指窃取微软的设计。

他们的 MP4 音乐播放器看起来很像 Zune，可以播放 AMW/MTV/MPEG/AVI 视频、JPEG 图片、电子书文本文件和 MP3 音乐。他们的 MP3 音乐播放器可以播放 MP3/WMA/WAV，还有调频广播录音，这其实是我们不常看到的。

播放器有 128MB/256MB/512MB/1GB/2GB 三种尺寸，兼容 Windows 和 Mac。没有价格信息，但老实说，你打算买一个吗？
