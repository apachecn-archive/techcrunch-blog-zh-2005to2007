# Writeboard 推出，但需要功能升级

37 个信号[背包](https://web.archive.org/web/20221003124221/http://www.beta.techcrunch.com/2005/06/12/backpack-showcases-rails-ajax/)、[大本营](https://web.archive.org/web/20221003124221/http://www.basecamphq.com/)等的创造者。)[今天推出了](https://web.archive.org/web/20221003124221/http://www.micropersuasion.com/2005/10/37signals_to_re.html) [Writeboard](https://web.archive.org/web/20221003124221/http://www.writeboard.com/) ，一款在线构建可共享文本文档的应用(Jason Fried 的帖子是[这里是](https://web.archive.org/web/20221003124221/http://37signals.com/svn/archives2/writeboard_is_live.php))。

这是一个很好的协作工具，但最近发布的产品，如 [writely](https://web.archive.org/web/20221003124221/http://www.beta.techcrunch.com/2005/08/31/writely-process-words-with-your-browser/) 和 [jotspot live](https://web.archive.org/web/20221003124221/http://www.beta.techcrunch.com/2005/09/27/jotspot-live-the-perfect-wiki/) 有更丰富的功能集。

Writeboard 是免费的，它允许用户非常快速地创建一个新文档，对其进行密码保护，并添加可以编辑该文档的用户。

与竞争对手的解决方案不同，您必须使用特殊的标记语言来格式化文本(没有所见即所得)，您不能上传图像，并且没有 ajax 或其他功能来在页面上移动内容。这也是一个小错误——在我们的测试页面上根本看不到评论。坦率地说，如果不是 37signals，他们通常会创建很棒的应用程序，我们现在还不会对它进行分析。

然而，我们猜测他们会很快增加功能。我们也很期待[粉笔](https://web.archive.org/web/20221003124221/http://www.beta.techcrunch.com/2005/08/05/profile-chalk/)的推出。

[SolutionWatch](https://web.archive.org/web/20221003124221/http://www.solutionwatch.com/242/37signals-writeboard-for-collaborative-writing/) 已经发布了对 Writeboard 的全面审查。
