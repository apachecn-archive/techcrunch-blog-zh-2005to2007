# 三星 USB 显示器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/16/samsung-ubisync-usb-monitor/>

前段时间我们告诉你的三星 USB 显示器在 CeBIT 展出，它比我们想象的要好。据报道，只有三到五个外设可以通过一个 USB 端口工作，但已经证实，六个实际上是这个数字。一名三星代表表示，一个 2GB 内存和一个 2GHz 双核处理器的设置可以轻松支持六台显示器。没有关于价格的消息，但是他们会在五月下降。

[三星的 UbiSync 显示器:所有时间都是 USB 接口](https://web.archive.org/web/20151004142534/http://www.engadget.com/2007/03/15/samsungs-ubisync-monitor-rig-all-usb-all-the-time/)
