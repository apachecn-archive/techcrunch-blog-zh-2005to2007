# OS X 版 Skype(测试版)

Skype 很棒。用的人大多喜欢。但如果你是 Mac 用户，你在游戏中总是有点落后。幸运的是，Skype 的优秀员工正在解决这个问题。今天他们为 OS X 发布了带视频的 Skype 2 测试版。

现在你不用启动讨厌的 Windows XP 就能搞定电话公司，而且这个软件和以前一样容易使用。视频功能允许您在 Skype 应用程序中或全屏与朋友面对面交谈，就像我们在 80 年代想象的 GI Joe 可视电话一样。

其他 Skype 用户仍可免费拨打 Skype 电话，但拨打座机或手机则需付费，尽管这仍比大多数长途电话便宜，因此仍不失为一种好方法。
