# 索尼全力支持 UMD。不，真的，有关系

[![sonysupportsumd.jpg](img/f6f0b1b5c1839dee39c603d382c8c87e.png)](https://web.archive.org/web/20201028064815/https://beta.techcrunch.com/wp-content/uploads/2007/07/sonysupportsumd.jpg "sonysupportsumd.jpg")

尽管事实上，从科学角度来说，没人在乎 UMD，索尼公司也不打算放弃它。不是没门，不是没办法。一位高级营销经理称这种格式的未来是“光明的”，这是我听过的最有信心的一票。word wizard 将其描述为“独一无二”，然后责备开发人员不知道如何正确使用这种格式:

> UMD 是一个创造游戏的独特媒介，开发者在处理系统时需要明白这一点

哦，猫打架！索尼，这就是你获胜的方式，你称你的开发者为白痴，因为他们不知道如何解决可怕的加载时间和相对较小的存储空间。最后一点在 PSP 上比[任天堂 DS](https://web.archive.org/web/20201028064815/http://crunchgear.com/category/nintendo-ds/) 更成问题，因为开发者更倾向于将 PSP 视为一个家庭系统，只是[碰巧是便携的](https://web.archive.org/web/20201028064815/http://crunchgear.com/2007/06/25/psp-has-untapped-power-thats-why-it-stinks-wipeout-dev/)，而 DS 显然是一个便携的(简单的)系统。

你买了多少部 UMD 电影？我个人在零分附近徘徊。但话说回来，我是个廉价的愤世嫉俗者。
