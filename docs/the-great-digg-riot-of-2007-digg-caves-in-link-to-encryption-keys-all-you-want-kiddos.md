# 2007 年的 Digg 大暴动:Digg 陷入困境，链接到你想要的加密密钥，孩子们

[![digglogo.jpg](img/de8aa177140710857f8b57be1a6f1bf1.png)](https://web.archive.org/web/20181114071208/http://old.crunchgear.com/wp-content/uploads/digglogo.jpg "digglogo.jpg")

书呆子们大叫犯规，[Digg默许了。Digg 的创始人——也是很多人心目中的神——凯文·罗斯刚刚发布了一个解释，解释了昨天发生的关于 HD DVD 加密密钥故事被撤的所有恶作剧。简而言之，Digg 有一个政策，从首页删除它认为可能会惹上法律麻烦的故事——色情、盗版，还有这里的加密密钥。但由于如此多的用户虚拟骚乱，罗斯屈服了。](https://web.archive.org/web/20181114071208/https://crunchbase.com/organization/digg)

现在[挖掘故事](https://web.archive.org/web/20181114071208/http://digg.com/tech_news/Digg_This_09_f9_11_02_9d_74_e3_5b_d8_41_56_c5_63_56_88_c0_4)。

出于多种原因，这是相当重要的。第一，我想看到美国电影协会或地球警察或任何人试图关闭现在链接到代码的许多网站。第二，它展示了用户创造内容的力量:作为 Web 2.0 的所有者，你对用户创造的东西到底有多少权力？剥夺了这种权力，用户就会威胁要离开，这就是 Rose 的 Digg 的遭遇。从法律的角度来看，太多的权力会威胁到你自己的生存。

就我个人而言，我发现许多 Digg 用户自以为是的态度有点愚蠢，好像他们天生就有权在网站上发布他们想发布的任何内容。记住，如果罗斯被起诉，他会上法庭，而不是你们。

但是没关系，去传播你认为合适的令人讨厌的咒语吧。
