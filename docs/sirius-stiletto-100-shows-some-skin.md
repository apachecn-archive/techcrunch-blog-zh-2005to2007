# 天狼星高跟鞋 100 显示一些皮肤

![](img/f36f01f0e835cb0989af60f6223d8dee.png)

看起来新的 Sirius [Stiletto 100 接收器](https://web.archive.org/web/20130627214358/http://crunchgear.com/2006/08/15/stiletto-not-just-a-woman%e2%80%99s-shoe/)已经准备好拍摄特写镜头了。如果你还记得，它能够保存长达 100 小时的音乐(MP3/WMAs)，可以通过 WiFi 连接传输音频，当然，它也是一个 Sirius 接收器。

现在，我们可以看到它与来自 Directed Electronics 的新 [Soloist 扬声器系统](https://web.archive.org/web/20130627214358/http://www.orbitcast.com/archives/details-on-the-soloist-media-dock-for-sirius.html)一起颠簸研磨。所以准备好你的钱包，因为她要 399 美元。
