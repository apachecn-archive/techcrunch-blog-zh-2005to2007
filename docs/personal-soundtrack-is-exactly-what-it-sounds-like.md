# 个人配乐就像它听起来一样

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/19/personal-soundtrack-is-exactly-what-it-sounds-like/>

我住在一个可以被归类为“大城市”的地方，每天上班的旅程都不一样。有时候，我很早就下了地铁，漫不经心地走在密歇根大街上。有时候，我会非常晚，所以我必须快步走(因为跑步太逊了)才能准时到达。无论我如何去工作，我都把 MP3 播放器牢牢地放在耳朵里。当我迟到并试图尽快赶到我要去的地方，而安东尼奥·卡洛斯·乔宾正在演奏时，这有时会变得非常滑稽。

好吧，如果发明家格雷格埃利奥特有他的方式，这将不再是一个问题。他发明了一种 MP3 播放器，可以测量你移动的速度，并相应地播放音乐。如果你走得快，它会开始播放你的迷幻之王。慢下来，你所有的惠特尼·休斯顿的专辑都会得到发挥。这是一个伟大的想法，我只是希望他能找到一种方法来减肥的装置，因为原型是一个巨大的笨重的混乱。
