# 沃达丰发布三款迈凯轮梅赛德斯手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/09/vodafone-unleashes-three-mclaren-mercedes-phones/>

![](img/0925b158b8c1bd2ef6b91b2848f55840.png)

沃达丰现在似乎得到了所有很酷的手机，这三款新的迈凯轮梅赛德斯手机也不例外。新的 770SH、GX29 和 SGH-Z720M 是由夏普公司制造的，具有非常好的规格。除了由一家制造你永远买不起的汽车的公司设计之外，这些新手机还配有独家内容、银色镀铬外壳和红色亮点，以反映迈凯轮梅赛德斯车队的颜色。

将包括大量与赛车相关的壁纸、铃声、视频和其他内容，比如让你在红灯时娱乐的赛车游戏。它们现在以我们不知道的各种价格出售。
