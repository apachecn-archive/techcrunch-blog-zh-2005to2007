# Tivo 会把亚马逊拆箱吗？

 [](https://web.archive.org/web/20211017133504/http://www.unbox.com/) 据报道，亚马逊正在与 Tivo 谈判，以使从亚马逊 Unbox 服务下载的电影能够通过 Tivo 发送到电视机。奇怪的是，这两家公司的股票价格在一个小时内迅速下跌之前，今天早上都出现了快速上涨。或许在最初的热情过后，开箱后的现实被广泛视为一场技术灾难。

Amazon Unbox 确实为每次下载提供了 DVD 质量的文件，可以刻录到 DVD 上作为备份，但不能用 DVD 播放器观看。更多关于 Unbox 的细节，请看[我们对其发布的报道](https://web.archive.org/web/20211017133504/http://www.beta.techcrunch.com/2006/09/07/amazon-unbox-goes-live/)。

苹果在“ [Showtime](https://web.archive.org/web/20211017133504/http://www.beta.techcrunch.com/2006/09/12/ok-heres-what-apple-announced-today/) ”活动上发布了自己的机顶盒 iTV。有传言称，该公司正在与谷歌谈判，其中包括允许谷歌视频的视频在独立电视上播放。

最终，这可能是 DRM 的问题，因为这些文件应该可以在任何上市的机顶盒上看到。正如电影制片厂对下载服务提出 DRM 要求一样，下载服务也会小心翼翼地与设备制造商分享他们的 DRM。你只能想象如果没有这些事情的纠缠，这个行业能走多远。也许每个人都会破产，电影会免费，不过，这很难说。
