# Unfuddle 为开发人员提供安全的托管项目管理

 [](https://web.archive.org/web/20201124161521/http://unfuddle.com/home) [Unfuddle](https://web.archive.org/web/20201124161521/http://unfuddle.com/home) 是一款外观清爽的新型软件开发项目管理服务。任何使用过 Basecamp 进行项目管理的人都会对它的外观和感觉很熟悉，但它是专门为软件开发人员定制的。这是一个托管服务，具有基本功能，包括消息传递、里程碑、票证、权限和通过 Subversion 的源存储。虽然许多开发人员目前使用基于 wiki 的项目管理服务，但 Unfuddle 是基于博客模型的。

该网站背后的公司名为 Subventurate LLC，由哈维的两名开发人员 Joshua Frappier 和 David Croswell 组成。

拥有 15MB 存储空间和有限功能的单个用户的免费帐户，或高达 5GB 的 24 美元至 99 美元的付费帐户，以及拥有时间跟踪和 SSL 连接等高级功能的 50 个用户。
