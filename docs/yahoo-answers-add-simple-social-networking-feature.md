# 雅虎问答增加了简单的社交网络功能

 [](https://web.archive.org/web/20211022135307/http://answers.yahoo.com/) [雅虎问答在其类别](https://web.archive.org/web/20211022135307/http://www.beta.techcrunch.com/2006/11/30/yahoos-big-win/)中占据主导地位，但也有大量关于垃圾邮件和其他无用内容的投诉。雅虎今天对产品做了一个简单的改变，可以帮助对抗垃圾邮件。用户现在可以[追踪那些倾向于提供优秀答案的人](https://web.archive.org/web/20211022135307/http://blog.360.yahoo.com/blog-d8pH0dcoRKeB12yOcnUQp.9VCFos?p=19332)。

这不会减少网站上的垃圾邮件数量，但它将有助于突出更好的内容，并允许用户在答案中创建社交网络。要使用它，用户只需将鼠标放在他们喜欢的用户的头像上，然后点击“添加到我的联系人”来自联系人的最新内容显示在用户的主页和个人资料页面中。

这不会直接帮助正在搜索内容或提问的用户，但肯定会给只是浏览网站的用户带来更好的内容。
