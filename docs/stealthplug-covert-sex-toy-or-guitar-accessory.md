# StealthPlug:隐蔽性玩具还是吉他配件？

![](img/cf28c0df51f1aefb638261cf353519dc.png)

除了有一个幽默的产品名称，IK 多媒体的 StealthPlug 实际上是一个看起来不错的配件。在线缆的一端有一个标准的 1/4 英寸插孔，可以插入吉他。在另一端，你有一个 USB 电缆，可以很容易地插入你的电脑。该设备还具有 1/8 英寸立体声输出，因此您可以轻松地将自己的音乐录制到外部设备上。这是所有旅行音乐人完全想要的:一个简单、省事的解决方案，允许在旅途中用笔记本电脑录音。

看起来你也可以用它得到一些不错的免费软件。标题包括 AmpliTube 2、Tracktion 2.1、SampleTank 2 SE 和循环库中的 500MB。还不错！StealthPlug 现已上市，建议零售价为 129 美元。
