# SoloSub 向修复 RSS 迈出了一步

SoloSub 允许拥有 RSS 的网站指向一个 URL，并允许读者决定他们想要使用哪个 RSS 阅读器。这里的[就是](https://web.archive.org/web/20220930175322/http://solosub.com/sub/http://www.petefreitag.com/rss/)的例子。

今天的 RSS 提要是一件痛苦的事情——你要么需要包含大量的订阅按钮，以便让那些不完全理解“XML”按钮含义的人更容易理解，要么依赖于提要的自动发现，这似乎经常会中断。我花了很多时间与聚合器打交道，请求他们重新缓存 TechCrunch 上的提要，这仍然是一个正在进行的项目。

SoloSub 是朝着正确方向迈出的一步。它在订阅 feed 的过程中增加了一个步骤(用户点击 solosub 图标，然后选择他们的阅读器)，但是这个额外的步骤是值得的。

SoloSub 目前支持 Bloglines、My Yahoo、NewsGator 和罗约。需要添加很多好的阅读器。
