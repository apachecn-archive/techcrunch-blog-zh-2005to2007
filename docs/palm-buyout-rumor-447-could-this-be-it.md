# Palm 买断传闻# 447；会是这个吗？

。现在有谣言在 Palm 流传，一个追求者一直在打电话试图买下他们。

这些谣言困扰了 Palm 多年，作为回应，高层通常会在全公司范围内发送一封电子邮件，大意是“那是废话，我们是非卖品。”问题是，这一次，没有这样的电子邮件，谣言指向摩托罗拉作为神秘的追求者。

此外，拥有 Palm 8%股份的投资者马克·纳尔逊(Mark Nelson)最近呼吁该公司积极寻找潜在买家。此外，ThinkEquity 的分析师乔纳森·胡普斯(Jonathan Hoopes)认为，Moto 在 Symbol 上的举动是 Palm 向外部集团出售的一个激励因素，因为这使它们成为支持任何智能手机产品的更具吸引力的方式。Symbol 有助于 Moto 追赶 RIM，但还远远不够。胡普斯认为棕榈是这个难题的一大块。

就个人而言，我们仍然认为苹果与 Palm 是完美的搭配，因为他们都喜欢简单、干净的硬件，都是市场创新者，并且都有在逆境中生存的诀窍。随着苹果的超级双密 iPhone 即将在 T2 发布，拥有一个与所有主要移动运营商都有关系的经验丰富的硬件合作伙伴可能会有所帮助。
