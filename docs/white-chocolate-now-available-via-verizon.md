# 威瑞森| TechCrunch 现已推出白巧克力

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/05/white-chocolate-now-available-via-verizon/>

广受欢迎的 LG 巧克力的白色版本。这款音乐手机是对苹果 iPod 和 Cingular 的索尼-爱立信随身听手机的直接挑战，以其发光的触摸感应按钮而闻名。现在，有很多连字符，所以我希望你注意了，因为这个手机很可爱。

这是一个纳米外观(编辑:Hickey 是连字符疯狂)的 MP3 播放器，可以滑入手机，这是涂料。巧克力有一个像样的摄像头，它支持 microSD 和蓝牙。确保得到[扬声器更新](https://web.archive.org/web/20150425051448/http://crunchgear.com/2006/09/27/verizon-enables-speakerphone-on-chocolate/)，好吗？没错。
