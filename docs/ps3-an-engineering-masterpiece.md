# PS3 一款工程大作？

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/16/ps3-an-engineering-masterpiece/>

市场情报公司 iSuppli 称索尼 PlayStation 3 是以入门级电脑价格提供超级计算机性能的工程杰作。这是在该公司对该系统进行拆解分析后得出的，该分析估计 20GB 版本的综合材料和制造成本为 805.85 美元，60GB 型号为 840.35 美元；这些设备的售价分别为 499 美元和 599 美元。跳转后有一个图表，显示了价格的细分，一部分一部分。

iSuppli 还表示，在 Nvidia(双图形处理器解决方案)、IBM (Cell Broadband Engine CPU)和三星(advanced XDR DRAM)的帮助下，索尼已经提供了惊人的性价比。
