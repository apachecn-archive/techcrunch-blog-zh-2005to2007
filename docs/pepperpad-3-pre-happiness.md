# 胡椒垫 3 幸福前

上周我们得到了一个 PepperPad 3，我们刚刚得到了写它的许可，所以我们决定做一个快速的回顾前讨论，并在本周晚些时候进行全面的回顾。PepperPad 3 是一款带有蓝牙、WiFi、全 QWERTY 键盘和 Linux 操作系统的互联网平板电脑。它是一台很棒的厨房/客厅电脑，非常适合网上冲浪和收发电子邮件。
![](img/860af29d6591def129f6c102a746eebe.png)
我推迟全面审查的原因是我还没有完全发现 PepperPad 的所有秘密。你在这里看到的屏幕是主窗口，有一组应用程序。如果你不想再深究下去，你会发现你需要的一切就在那里。然而，这东西显然有一些锦囊妙计。配有内置硬盘和明亮的触摸屏，PepperPad 是一款非常有趣的设备，我迫不及待地想进一步探索它的曲线美。
