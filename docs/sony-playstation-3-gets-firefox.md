# 索尼 Playstation 3 获得 FireFox

![](img/7f7012309792afe0675470c8a4e3b12b.png)

Terra Soft 最近宣布，他们已经开发了黄狗 Linux v.50，这是下一代 Linux 操作系统，将支持 Playstation 3。Terra Soft Solutions 的首席软件工程师 Owen Stampflee 说:

> 从 Commodore 64 时代到现代超频和水冷家用电脑，专用游戏盒和家用电脑之间爆发了一场争夺最高质量游戏的战争。PLAYSTATION 3 提供了业界第一个卓越的 Linux 操作系统体验 Terra Soft 提供了第一个 Linux 操作系统。我很自豪在这一发展中发挥了作用。

这一切意味着什么？你可以为你的 Playstation 3 下载以下内容:

*   内核 2.6.16*   gcc 4.1.1 和 glibc 2.4*   Cell SDK 1.1*   OpenOffice.org 2 . 0 . 2*   火狐 1.5.0 和雷鸟 1.5.0*   鹦鹉螺 2.14*   一套个人配件、开发工具；声音和视频、互联网和网络应用

火狐是否会成为 PS3 的主要浏览器还没有定论，但即使不会，很多用户还是会下载它。
