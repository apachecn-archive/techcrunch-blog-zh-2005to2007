# 索尼宣布 600 美元的蓝光播放器，令人惊讶

当然，你可以买一个上变频 DVD 播放器，在你的高清电视上播放任何高清分辨率的标准 DVD。毕竟，它们只有 100 美元左右，有时甚至更少，而且升级的东西看起来真的不可思议。但是索尼最新的蓝光播放器仅售 599 美元，真正高清化的想法看起来不错。更重要的是，除了比索尼之前 1000 美元的蓝光播放器便宜，新的 BDP-S300 还将具有 CD 播放功能。不，真的！

当然，只有蓝光光盘才是高清的，蓝光光盘比标准 DVD 更贵，兼容的产品也很有限，但成为第一家以如此低的价格为播放器定价的公司是对 HD-DVD 的挑战。通过推出一款普通消费者实际上买得起的播放器(如果他们想买的话)，索尼终于在其最新的格式消耗战中做了一些正确的事情。

现在，如果它能压低光盘价格，并让一些其他工作室参与进来，索尼可能会赢得这场比赛，这是自里根政府以来它还没有在格式方面取得过的成绩。
