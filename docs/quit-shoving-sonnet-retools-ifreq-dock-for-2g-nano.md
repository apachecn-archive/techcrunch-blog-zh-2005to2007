# 放弃推挤:十四行诗为 2G 纳米重组 iFreq 码头

你可能知道也可能不知道，当苹果将 iPod Nano 更新到 2G 版本时，它稍微移动了一下基座连接器，使得大量第一代 Nano 配件与新型号不兼容。当然，你可以推啊推啊，拉啊，让一些东西有点合适，但不是所有的东西。好吧，Sonnet Technologies 今天宣布，其 iFreq 车载 FM 发射器/充电器所含的底座已被重新设计，可适用于 1G 和 2G iPod Nanos。万岁。

不熟悉 iFreq？一部分是前面提到的安装支架，它可以在你贴在仪表盘上的背胶磁盘上脱扣。这可以让你关掉 iPod，改变设置、曲目、艺术家等等，然后再把它放回原处。一根电缆从坞站延伸出来，连接到 iFreq 的机身(另一部分)，进入你的点烟器。车身上有一个液晶显示屏和控制装置，可以让你轻松锁定调频广播电台，通过汽车音响播放 iPod 的内容。

因此，如果你想为你的新 Nano 购买一个调频发射机，iFreq 的新底座肯定会适合你。但即使没有底座，它也可以与任何有基座接口的 iPod 兼容，所以很有趣。售价约 80 美元。
