# Vista:切换回来很容易

TC 的撰稿人 Nik Cubrilovic 最近与他的 Mac 发生了冲突，并开始全职使用 Windows Vista。可怕的部分？他真的真的很喜欢！

令人惊讶的是，我倾向于同意他的观点。我对 Vista 在我家里的一台旧机器上的表现非常满意，我现在期待着启动 Windows，而不是害怕它。就算我不可能回到 IE 和 office 的世界——除非苹果公司犯下一些重大失误——但知道其他人和我有相同的观点也很好。

Vista 不是微软的蠢事。这是对一个已经很受欢迎的产品的一次有价值和明智的升级。撇开安全问题不谈，Windows 将永远伴随着我们，我很高兴至少当我想打开我的电脑时，我会看到一些有用的东西，而不是另一个无用的东西。
