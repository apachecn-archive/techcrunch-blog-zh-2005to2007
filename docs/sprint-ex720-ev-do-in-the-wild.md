# 野外冲刺 EX720 EV-DO

Slashgear 的 Vince 拥有 Sprint EX720，我非常嫉妒。最新的 Rev. A 宽带卡让我的 s620 堪比鼻涕虫。苹果已经发布了支持这种卡的软件驱动程序，所以它肯定会受到苹果用户的欢迎。Sprint 移动宽带卡的唯一缺点是缺乏 Mac 支持。在 Mac 上运行之前，你必须在 Windows 机器上激活该卡。文斯平均下载速度高达 826 KB/s，上传速度为 103 KB/s。我真的想要一个！
