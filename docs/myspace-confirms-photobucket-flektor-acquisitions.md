# 福克斯互动确认收购 Photobucket，Flektor

福克斯互动今天在一份新闻稿中确认了对[![](img/07c4920ed19066a472c3f58b346afada.png)](https://web.archive.org/web/20220815104950/http://www.myspace.com/)和 [Flektor](https://web.archive.org/web/20220815104950/http://db.beta.techcrunch.com/c/flektor/) 的收购。没有关于价格的细节，所以我们假设早期的报道是正确的:Photobucket 的 3 亿美元(包括 5000 万美元的收入)和 Flektor 的 1500-2000 万美元。

我们之前报道过 MySpace 进行了这两项收购；事实上，母公司福克斯互动是买家。

参见 Techcrunch 数据库中的 [Photobucket](https://web.archive.org/web/20220815104950/http://db.beta.techcrunch.com/c/photobucket/) 和 [Flektor](https://web.archive.org/web/20220815104950/http://db.beta.techcrunch.com/c/flektor/) 。
