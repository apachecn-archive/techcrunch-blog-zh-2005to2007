# YouTube 视频广告:没有前置广告，没有背景

NewTeeVee 的 Liz Gannes 简介 YouTube 进军视频广告。我们报道了 YouTube 向其顶级内容提供商提供收入分成的举措[5 月 4 日](https://web.archive.org/web/20220627075553/http://www.beta.techcrunch.com/2007/05/04/youtube-launches-revenue-sharing-partners-program-but-no-pre-rolls/)，我当时注意到没有任何前置广告的迹象。仍然没有。

新的视频谷歌/ YouTube 广告立即可以与 Adbrite 的[视频服务](https://web.archive.org/web/20220627075553/http://www.beta.techcrunch.com/2007/01/04/adbrite-makes-brilliant-video-product)相媲美。两者的视频上都有文字广告。相似之处仅此而已。Adbrite 的服务提供了一个链接到外部网站的文本链接，而 YouTube 的新服务本质上是视频中的视频。点击广告文本导致视频广告出现在播放器的顶部。

这是谷歌寻求更好地利用 YouTube 赚钱的一个显著进步，但我仍然不相信。在新蒂威拍摄的样片说明了一切。在 Cingular/ AT&T 的《我的化学罗曼史》视频的基础上提供了一则广告。Cingular 可能是新的美国电话电报公司，但它与我的化学罗曼史有什么关系呢？完全没有。没有语境，没有语境匹配和传递。观看该视频或其他音乐视频的用户会对打断他们观看的美国电话电报公司可选广告感兴趣吗？

在某些方面，这是典型的谷歌骑墙派。他们需要从 YouTube 上获得更好的回报，但他们似乎不愿意对托管视频进行预播放和后播放。相反，他们现在提供了一种折中的选择，他们可能会认为这种选择赋予了观众更多的选择，但实际上这种选择只会让观众避免观看广告。

预卷和后卷，如果做得好，就像洋葱新闻网的杜沃斯威士忌宣传活动一样，效果很好。更重要的是，它们是不可避免的。

没有人喜欢看商业广告，然而正是商业广告商提供资金来维持我们现有的媒体形式。YouTube 可能是 Web 2.0 和用户生成内容的典型代表，但经济是不可避免的。谷歌要收回 15 亿美元还有很长的路要走。新的广告服务可能会带来一些额外收入，但我认为它不会成为大赢家。
