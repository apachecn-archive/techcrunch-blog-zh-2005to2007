# 3GSM 的可滚动显示装置

意大利电信和 Polymer Vision 将在 3GSM 展示他们最近同意共同制造的一种可卷曲的手机。基于 Polymer Vision 的 Readius 概念，该设备主要用于阅读报纸、书籍和其他类似数据。

该设备的尺寸小于标准手机，具有可展开的 5 英寸显示屏。它在高对比度反光纸显示器上投射 16 级灰度。

没有价格或任何东西，但比格斯将在 3GSM 的地板上，一定会得到这个小古怪的纲要。
