# 更安静的 Xbox 360s 即将上市

![](img/2c074f3f3a39dedda5843d33e0fab629.png)
我爱我的 360。这是一个伟大的系统，我有信心它会赢得主机战争。但是有一个问题。这是我房间里最吵的设备。当我坐在这里玩*战争机器*的时候，机器发出如此多的摩擦声，有时候我怀疑它会不会爆炸。

认识到存在问题，微软正在采取措施纠正问题。新的控制台将采用更安静的 VAD6038，由 BenQ-LiteON-Phillips 制造。较老的东芝-三星和日立-LG 硬盘已经过时。

也许我得去把我的换了。
