Vista比Mac OS更安全:胡说？


地狱已经冰封了！往山上跑！天要塌了！安全研究员迪诺·戴·佐维(Dino Dai Zovi)说，Vista被宣称比Mac OS X更安全。我们为什么要在乎这个人要说什么？首先，他在一周前的CanSecWest安全会议上获得了一万美元的奖金，因为他获得了Macbook Pro的shell访问权限。

当被问及哪个平台更安全时，他是这样回答的:

我发现Vista的代码质量，至少在安全性方面，比Mac OS X 10.4要好得多。通过观察安全补丁中受影响的组件可以明显看出，微软的安全开发生命周期(SDL)已经减少了新编写代码中的漏洞。我希望更多的软件供应商跟随他们的脚步，开发主动的软件安全开发方法。

但不要害怕，因为迪诺给Mac用户提供了一些建议，帮助他们在你的钻机上保持更安全的状态:

我建议Mac用户将其主要用户设为非管理员帐户，为重要密码使用单独的钥匙串，并将敏感文档储存在单独的加密磁盘映像中。我认为这些是相当简单的步骤，许多用户可以采取这些步骤来更好地保护他们计算机上的敏感信息。
