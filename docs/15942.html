Vivendi Games Mobile宣布2007年下半年游戏名称

当谈到移动游戏时，电子娱乐行业的主要参与者都拿出了大枪。几乎每个人的口袋里都有一部手机，这是一个比当前一代游戏机加起来还要大的市场！

Vivendi Games Mobile宣布了他们的下半场阵容，这是一个令人印象深刻的阵容。这包括原创游戏以及基于Sierra Entertainment特许经营的游戏。Vivendi Games Mobile将在2007年下半年推出10款新游戏。

Vivendi Games Mobile总裁Paul Maglione表示:“今天宣布的游戏将提供非常令人愉快的游戏体验，是我们内部工作室和全球外部开发合作伙伴开创性设计的产物。“我们期待与我们的分销合作伙伴合作，将这些游戏带给移动游戏玩家的大众市场。所有即将推出的游戏都将继续采用令人耳目一新的不同营销方式，运营商已经开始将这种方式与Vivendi Games Mobile联系起来，包括独一无二的数字和病毒式举措，以及基于用户生成内容的营销策略。”

这些游戏包括:
【泰坦的崩溃】——来自《崩溃》Bandicoot系列的最新动作平台游戏。

三角洲部队——这款游戏基于广受欢迎的Novalogic三角洲部队系列，让你指挥一队三角洲部队游骑兵，并包括一个新的“狙击范围”功能，这是手机游戏中的第一款。

休闲套装拉里:对帆船的热爱——一款手机上的拉里游戏。大概说够了吧！

冲击方块——起源于韩国的益智游戏，以两种不同的模式提供新的挑战。

Slide N ' Loop——这是一款由任务驱动的益智游戏，由资深玩具发明家Zeki Orak合作设计。

《幸存的高中》08——一部原创IP专营权，以新的图形、人物和故事情节为重点，讲述另一个学年。

幸存的好莱坞——如果你能在这里成功……等等，那是纽约。不管怎样，这本书让你努力成为一名真正的明星。

不可思议的机器——这款流行的PC益智游戏在手机上推出了60个原始的智力游戏关卡，外加20个要解锁的奖励关卡。
