# Wii 多人游戏在 Q2 2007

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/04/wii-multiplayer-in-q2-2007/>

任天堂的美国掌门人雷吉·菲尔斯-艾梅在一次采访中表示，在线 Wii 游戏要到 2007 年 Q2 才会上市。他暗示说第一款多人游戏将会在三月份开始下降，在那之后应该会有一个冲击。

我不太确定我对此的感受。对于 Wii 用户来说，发布后的六个月似乎是一段特别长的时间，因为他们不得不等待下一代游戏机最标准的功能之一:在线多人动作。

让我有点恼火的是，公司推出产品只是为了让他们出门，而客户却要等几个月才能得到他们承诺的功能。
