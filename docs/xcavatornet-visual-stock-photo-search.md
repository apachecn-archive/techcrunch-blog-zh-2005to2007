# Xcavator.net:可视化股票照片搜索

[![xcavator.png](img/c7474dc3c68fdf21a696a83b8ddcdae1.png)](https://web.archive.org/web/20221001030001/http://xcavator.net/)[Xcavator.net](https://web.archive.org/web/20221001030001/http://xcavator.net/)是基于视觉搜索技术的股票图片搜索门户。

Xcavator.net 为股票摄影提供自然直观的交互式搜索，为买家提供基于视觉内容和关键词的浏览体验。视觉搜索功能的关键是门户的颜色和图像搜索引擎，由 CogniSign 智能图像识别技术提供支持。

用外行人的话来说，Xcavator.net 提供了三种相互关联的搜索选项。传统搜索基于标记的关键字提供照片，与股票摄影市场中的其他搜索方式非常相似。Xcavator.net 变得有趣的地方在于颜色和图像搜索。Xcavator.net 允许颜色搜索匹配，例如，如果需要一张与宣传册或网站颜色匹配的库存照片，用户可以通过使用颜色表或在方框中插入准确的十六进制颜色来优化照片搜索。图像搜索根据用户上传的图像或通过拖放在初始搜索中找到的图像来提供类似的照片。

Xcavator.net 与其他视觉搜索网站竞争，包括 [Riya](https://web.archive.org/web/20221001030001/http://www.beta.techcrunch.com/tag/riya) 、 [Pixsy](https://web.archive.org/web/20221001030001/http://www.beta.techcrunch.com/tag/pixsy) 和 PicSearch。Xcavator 不一定比他们的任何竞争对手更好，但与众不同。颜色和相关的搜索功能没有 Riya 的搜索功能给用户带来同样的享受，但 Xcavator.net 的功能感觉更实用，而且肯定更精细地定位于利基股票照片搜索。

Xcavator.net 最近与 iStockphoto 签署了一项协议，将来自 3.8 万名贡献者的 180 万张图片输入 Xcavator.net 搜索数据库。该网站将于 7 月 2 日退出测试。
