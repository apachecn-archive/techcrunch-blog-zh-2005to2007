**什么事？**

BlinkList 是一个基于 ajax 的社会化书签服务。它非常整洁，有一些很好的特点。它也有一个经验丰富的全明星团队，由他们的公关明星领导(见下面的团队):-)。

![](img/d9d2bbde23fe0434d7ca22d81a113558.png)

我们最近对一些竞争服务进行了剖析——参见 [Shadows](https://web.archive.org/web/20220811122539/http://www.beta.techcrunch.com/?p=92) 、 [del.icio.us](https://web.archive.org/web/20220811122539/http://www.beta.techcrunch.com/?cat=19) 、 [Simpy](https://web.archive.org/web/20220811122539/http://www.beta.techcrunch.com/?p=109) 、 [Furl](https://web.archive.org/web/20220811122539/http://www.beta.techcrunch.com/?p=30) 、 [Yahoo My Web 2.0](https://web.archive.org/web/20220811122539/http://www.beta.techcrunch.com/?p=56) [和](https://web.archive.org/web/20220811122539/http://www.consultantcommons.org/node/239) [其他](https://web.archive.org/web/20220811122539/http://www.dlib.org/dlib/april05/hammond/04hammond.html)。

Blinklist 包含了这些网站的大部分功能，并增加了一些其他功能。
 **关键特性:**

–简单的书签功能，包括一个[书签小程序](https://web.archive.org/web/20220811122539/http://www.blinklist.com/users/thankyou)，如果你想拥有一键书签
–在标签和书签上添加“星星”，以便将来轻松找到它们

![](img/0400a9c6bac13786e3a37ff8a1307137.png)

–blink list 将获取屏幕上任何高亮显示的文本，并用文本自动填充描述字段(这是我所知道的唯一一个这样做的服务，非常有用)

![](img/fbf7a15b6ae6b1fa4adb7f5984305687.png)

–书签可以是私有的，也可以是公共的。你可以在制作书签的时候用电子邮件发给朋友，这也是一个非常有用的功能。TechCrunch 书签在这里是。
—永久标签页面——例如参见[www.blinklist.com/tag/web2.0/](https://web.archive.org/web/20220811122539/http://www.blinklist.com/tag/web2.0/)(通用标签搜索上的嵌套/过滤搜索功能将是一个很棒的补充——[阴影](https://web.archive.org/web/20220811122539/http://www.shadows.com/)有一个非常有用的“缩小结果”选项)
—每个用户的标签云，以及整个 blinklist 社区

![](img/64ef8b5787792f00d0e6d308192d5263.png)

–“相关标签”用于当前查看的标签
–“自动完成”标记书签的建议
–所有页面的 RSS
–轻松注册

在这里看演示[看一些正在使用的功能。](https://web.archive.org/web/20220811122539/http://www.blinklist.com/Theme/Media/BlinkCNN.htm)

总的来说，BlinkList 是社会化书签服务的一个有价值的补充，也是我们的最爱之一。它没有 del.cio.us 的用户群，也没有 Shadows 或 Simpy 的所有功能，但它确实有其他公司没有的独特功能。它很容易获得牵引力。

**建议的新功能:**

–带有关键字搜索
的整页缓存–阴影页面类型功能(这将真正有助于搜索引擎排名，非常酷)。
–各级嵌套/标签过滤
–显示给定 URI 下赌注的人数
