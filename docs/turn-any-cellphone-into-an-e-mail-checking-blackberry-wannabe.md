# 把任何手机变成电子邮件检查，黑莓的崇拜者

[![mobileemail.jpg](img/1c1cb48fdf3a00933e57f1242c3f5968.png)](https://web.archive.org/web/20210118171825/http://old.crunchgear.com/wp-content/uploads/mobileemail.jpg "mobileemail.jpg")

《纽约时报》的*正确地认识到，不是每个人都想看起来像个零，每 27 秒就拿出一部[黑莓](https://web.archive.org/web/20210118171825/http://crunchgear.com/2007/05/15/blackberry-8120-to-replace-the-pearl/)查看他们的电子邮件。也就是说，谁不想定期阅读一些电子邮件，只要它不会变得引人注目？波格研究了一些能让普通手机以某种方式查看电子邮件的服务和应用。呜呜~！*

 *Gmail for Mobile 是一款小型 Java 应用程序(因此，威瑞森用户无需申请),它在小屏幕上忠实再现了流行的基于网络的电子邮件客户端。这里唯一的问题是，你需要手动检查你的电子邮件——当电子邮件到达你的收件箱时，没有疯狂的哔哔声或振动。不过，这可能并不坏，因为自动检查电子邮件所需的所有网络活动确实会耗尽电池。

谷歌的竞争对手雅虎拥有自己的移动应用程序，名为雅虎 Go 2.0。除了这个可怕的名字，这个名字可以被配置成自动检查你的雅虎电子邮件。你也可以登录雅虎旗下的各种服务，比如 Flickr 和 maps。因为它试图成为一个万事通，所以运行速度比 Gmail 稍慢，在更少的手机上运行。

最后，还有 Teleflip，一种可以将所有电子邮件转换成任何手机都能接收的短信的服务。你必须建立一个电子邮件地址的白名单，让 Teleflip 把它发送到你的手机上。你还应该认识到，短信只能包含有限数量的字符，因此较大的电子邮件需要拆分。

原来如此。普通手机用户在旅途中查看电子邮件的几种方法。当然，你也可以打开你手机的网络浏览器，但是这样就没那么性感了。(是的，就好像查看电子邮件是一件让人心烦意乱的事情。)
