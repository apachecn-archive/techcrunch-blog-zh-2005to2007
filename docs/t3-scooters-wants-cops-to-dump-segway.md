# T3 滑板车希望警察抛弃赛格威

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/19/t3-scooters-wants-cops-to-dump-segway/>

T3 系列踏板车比赛格威(Segway)踏板车稍少古怪，但同时也更古怪，它们希望赢得各地警察的心和脚。T3 是一辆三轮踏板车，配有 LED 系统、GPS、手套箱和充电电池。它的电池可以在现场更换，以防警官昨晚忘记插电，每个电池的价格约为 1200 美元至 1800 美元。

该装置本身的成本是未知的，但应该比赛格威便宜，因为它缺乏运动平衡和其他简单的机械。它看起来真的很像电动滑板车，而不是自动平衡的。不管怎样，预计几年内至少会有一些警察开着这种车追你。除非你坐进车里。
