# 索尼 NW-A800 随身听亮相

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/19/sony-nw-a800-walkman-unveiled/>

![](img/be8c8f79f7be0bf6b561e195dc1f4a93.png)

虽然这些索尼 A800 随身听的照片看起来很棒，我希望这是最终产品的样子，但我认为这是 3D 渲染的。被称为“纳米杀手”的 A800 是索尼的最新随身听，将于 3 月份上市。A800 拥有巨大的屏幕和易于使用的控件，有紫色、黑色、蓝色或粉色可供选择。

虽然确切的细节还没有出来，但我们知道它应该有 2GB，4GB 和 8GB 版本，可以播放 8 到 10 小时的视频，并支持多种编解码器。如果曾经有一款随身听可以挑战 iPod(甚至有些挑战)，那么这款就是了。
