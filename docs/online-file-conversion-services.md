# 在线文件转换服务–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/11/04/online-file-conversion-services/>

上周推出的免费在线文件转换工具[![](img/985d1ffb8ec59e6e9bd88f99f4f4d665.png)](https://web.archive.org/web/20220808224729/http://www.zamzar.com/)。和类似的工具 [Media Convert](https://web.archive.org/web/20220808224729/http://media-convert.com/) 一样，Zamzar 允许用户上传文件并将其转换成多种格式。

Zamzar 的常见用途包括将 PDF 文档转换为 Word，将 iTunes AAC 文件转换为 MP3，将 Flash (flv)文件转换为 MP4 以供 iPod 使用。最多可以同时转换五个文件。这个过程需要几分钟时间，完成后，Zamzar 会通过电子邮件发送一个链接，指向转换后的文件。它在我们的测试中运行良好。但是，请注意，Zamzar 不能转换 DRM 文件。最大文件大小为 100 MB，但转换的文件数量没有限制。有关 Zamzar 的更多信息，请参见 [Go2web2](https://web.archive.org/web/20220808224729/http://go2web2.blogspot.com/2006/11/free-online-file-conversion.html) 。

虽然我们在测试中发现 Media Convert 转换文件的速度更快，但 Zamzar 的界面和设计比现有的服务 Media Convert 更好。Media Convert 还允许输入文件存储在本地或来自 URL，如果文件在互联网上，则为用户节省了一个额外的步骤。此外，Media Convert 会在您等待时转换文件，并在过程完成时在屏幕上显示下载链接。因为不需要注册，所以用户不需要像使用 Zamzar 那样提供电子邮件地址。但是，只有 50 MB 或更小的文件可以使用媒体转换进行转换。

这两个服务都很有用，尤其是在转换 flash 电影(来自 YouTube 等)时。)到 MP4，以便在 iPod 上观看。
