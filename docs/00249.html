概述:

YouTube很像flickr ( profile )，但是对于视频来说。你可以上传多种不同格式的视频。AVI。MOV还有。MPG)。视频可以是任意长度，但总大小必须小于100 mb。编辑限制包括没有“裸体，你的视频必须适合所有观众。”(链接)。如需更多要求，请参见他们的帮助页面。

这项服务对用户(出版商和观众)完全免费。YouTube将视频转换为flash格式，因此上传和播放速度极快。在我们的测试中，视频上传速度明显快于其他服务。

YouTube也是一个分享网络。你可以添加朋友，他们也是这个网络的成员，并且可以通过电子邮件将任何视频发送给任何人。YouTube还提供易于使用的复制/粘贴代码，用于发送电子邮件和在网站上发布。你也可以标记你的视频，这与flickr非常相似。总的来说，服务很好。

这项服务最近刚刚推出，但似乎有相当多的用户发布了大量内容。我们绝对最喜欢的是马特环游世界。它展示了一个前程序员在世界各地旅行并在著名的地方跳舞——这太棒了，在网站上非常受欢迎。其他喜欢的还有这个和这个(那是穿红色衬衫的我。好的，不是)(大家是怎么做到的？).

我怀疑YouTube会很快被收购和/或复制。我们喜欢它。
