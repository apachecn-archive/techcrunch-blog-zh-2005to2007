# ren koo“Beeta”发布

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/04/28/renkoo-beeta-launches/>

 [](https://web.archive.org/web/20211019031007/http://www.renkoo.com/) 总部位于帕洛阿尔托的[人酷](https://web.archive.org/web/20211019031007/http://www.renkoo.com/)，刚刚筹集了[$ 300 万美元的风险融资](https://web.archive.org/web/20211019031007/http://www.beta.techcrunch.com/2006/04/13/renkoo-nails-3m-in-funding/)，今天早上向 beta 测试者开放。他们会在最初的几百个请求中立即加入，然后慢慢加入新用户(如果你没有看到一个申请 beta 账户的框，他们还没有切换到新网站)。一旦你成为会员，你就可以邀请其他人使用这项服务。

像竞争对手 Skobee 一样，Renkoo 帮助人们计划活动(evite 前阶段)。想召集几个朋友，讨论下周何时何地去喝酒？仁库是帮你做到这一点的绝佳选择。据传，Friendster 创始人乔纳森·艾布拉姆斯的新公司 Socializr 也将进入这一领域。

我在 Renkoo 的开发周期中断断续续地测试过它，它发展得很快。首席执行官亚当·莱福金让我今天再走一遍，看看最近增加的一些功能。

 [](https://web.archive.org/web/20211019031007/http://www.flickr.com/photo_zoom.gne?id=136316699&size=o) 要使用 Renkoo，您需要设置一个新事件并邀请联系人。如果他们还不是 Renkoo 的用户，他们将会收到一个邀请。如果是，他们将收到新的事件通知。一旦人们开始就某个事件进行互动，事情就会变得有趣起来。

Renkoo 是早期的“ [Comet](https://web.archive.org/web/20211019031007/http://www.irishdev.com/NewsArticle.aspx?id=2173) ”应用程序(允许在浏览器中进行实时双向互动——Google Talk 和 Meebo 基于底层 Comet 技术)。这意味着活动页面会随着您的朋友提供的信息实时更新。这种体验类似于即时消息。未登录的用户一旦登录就能看到评论流。

Renkoo 还允许即时消息，电子邮件和短信通知事件，评论等。(全部由用户控制)。事件完成后，可以导出为 icalendar 格式(适用于 Outlook 和 iCal 等桌面日历)或 Google 日历格式。对其他 web 日历和微格式的支持即将推出。

Renkoo 由 KnowNow 的创始人亚当·莱福金和 Joyce Park 创建，Joyce Park 曾是 Friendster 的高级网络开发人员，也是 PHP5 和 MySQL 圣经的作者。
