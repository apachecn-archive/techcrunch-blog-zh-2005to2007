# 无臭味的榴莲

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/09/stink-free-durian/>

![08durianxlarge1.jpg](img/1de049cf9449bda20dfade99b76c69fd.png)

一名泰国科学家发明了一种无气味的榴莲，这是一种亚洲水果，因其令人作呕的气味而闻名，因此被许多酒店、航空公司甚至新加坡地铁禁止食用。榴莲新手经常形容这种水果闻起来像脏脚、没洗的更衣室或死猫，但爱好者们对这种独特的味道深信不疑，甚至声称逐渐喜欢上了它。这可能类似于斯德哥尔摩综合症。

> 东南亚某个地方的报纸很少不报道榴莲死亡的消息。泰国卫生部的营养学家 Wilailak Srisura 说，这种水果富含碳水化合物、蛋白质、脂肪和含硫化合物(因此有这种味道)，据说“令人兴奋”，因此对高血压患者来说可能是致命的。

粉丝们对亚洲最臭水果的甜味版本感到酸楚【纽约时报】
