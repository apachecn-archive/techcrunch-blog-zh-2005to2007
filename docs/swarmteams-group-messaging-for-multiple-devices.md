# swarm teams–面向多种设备的群发消息| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/19/swarmteams-group-messaging-for-multiple-devices/>

swarm 团队是为那些无论使用什么设备都需要相互发送信息的团体或群体而设计的。使用短信和电子邮件，无论你在哪里，你都可以联系到你的小组。有四个不同的应用程序，Swarm-it！，Swarm-pro！，虫群网！还有 Swarm-api！，四个显然非常激动人心的应用程序将你与大众联系起来。

蜂拥而上。是为普通人提供的短信和电子邮件应用程序，因此当您向 group point 发送消息时，该消息就会传播给群组中的所有人。想要电子邮件的用户通过电子邮件得到通知，想要短信的用户通过手机得到通知。

虫群网！工作原理是在你的网站上嵌入应用程序，允许你向你的读者广播消息。你可以通过你的网站、他们的手机或电子邮件联系他们(后两种方式取决于用户)。这真的是 Swarm-pro 的插件！不能作为单独的独立产品使用。

Swarm-pro！允许你创建多个可以互相交流的群组，然后这些群组可以被打上烙印并完全独立于其他群组行动。你也可以链接群组，例如，允许你公司的多个部门互相发送信息。有同行管理或所有者管理的群组，这意味着更多的是在企业内部和外部建立社区。

Swarm-api！是应用程序编程接口(API ),用于构建 Swarm 插件和应用程序，以整合到您的 Swarm-pro 中！蜂拥而上。产品。

你可以注册一个 Swarm-it！现在试用，但 Swarm-pro！截至本文发布时，正在进行有限的试用。你也可以在试用 Swarm 软件之前，观看他们的在线屏幕演示，更好地感受它的样子。
