# SXSW 2007:威尔·莱特主题演讲直播

威尔·莱特即将登台为 SXSW 互动峰会做最后的主题演讲。希尔顿宴会厅比我见过的任何其他主题演讲都要拥挤。各种形状和大小的成群结队的 SXSWers。我看到了各种各样的人，从粉丝到不那么粉丝的人。然而，这看起来是一个好的。

大卫·鲍依目前正在用私人助理轰走汤姆少校。两个巨大的投影屏幕正在显示来自哈勃的图像。有人正在介绍:

2:10-主持人继续介绍。目前正在讲述一个关于威尔·莱特控制一个机器人向人们发射快速发射乒乓球的故事。

2:10-威尔·莱特上台。他的手臂看起来严重骨折。

2:13–Wright 正在讨论讲故事的过程。他解释说，不同媒介之间的故事呈现出不同的弧线。他说，电影比游戏更有趣，因为弧线比游戏更线性。

所有这些背后的原因是因为赖特认为讲故事的最佳元素是移情。人们能够把自己放在角色的位置上。

在线性讲故事中，导演知道未来，所以通向结局的故事可以被适当地改编，以与结局最相关的方式来展现故事。交互式讲故事没有这种奢侈，因为参与者有能力影响故事的进展。

2:25–Wright 以惊人的速度快速浏览他的 Powerpoint 幻灯片。他自己承认咖啡因摄入过多，所以我想这并不奇怪，但很难看清其中一些。

2:39–仍在谈论讲故事。

2:41–展示孢子播放器工具的演示。它展示了游戏的巨大。莱特的单细胞生物刚被什么东西吃了。然后他试着扮演一个更大的蛞蝓生物，当他试图吃掉一个长得像长颈鹿的小动物时，又被吃掉了。

当前的演示展示了角色的创建过程。看起来还挺百搭的。玩家在选择角色细节时将有大量的选项可供选择。然后，系统会分析这些属性，并适当地制作动画。很酷的东西。

他向前跳跃了数百万年，直到他的新生物有了感知能力和太空旅行能力。这种观点已经扩展到包括源行星的宏观观点，而不是直接周围环境的微观观点。宇宙飞船很酷。莱特设想了巨大的星际战争和互动。

莱特刚刚展示了毁灭一个星球的能力。地球上所有的生物都濒临灭绝，因为他迫使气候发生变化，海洋干涸。

他刚刚创造了一块巨石，并把它扔在了新的星球上。低等生物已经包围了它，并开始崇拜它。他降落在这个星球上并展示了孢子百科全书。这个想法是扫描你遇到的所有生物来完整记录孢子的存在。
