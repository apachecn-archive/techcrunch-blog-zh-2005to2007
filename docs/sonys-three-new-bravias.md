# 索尼的三大新创意——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/27/sonys-three-new-bravias/>

![](img/aa132827ccadda904b60acdf1842f505.png)

你是瘸子吗？你总是喜欢落后一步吗？后进生？那么我们已经为您设计了一些电视。索尼公司宣布了一些薄的背投高清电视，据说比标准背投电视小 22%。排队的是 50 英寸的 KDF-50E3000，46 英寸的 KDF-46E3000 和 37 英寸的 37H1000。

前两款在 7 月份降价，后者将在 5 月份上市，售价为 1300 美元。我不明白你为什么不付 42 英寸液晶显示器的钱，但人各有志。
