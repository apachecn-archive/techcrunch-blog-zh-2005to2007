# 威尼斯工程的星星排成一行

Kazaa 和 Skype 的创始人尼可拉斯·曾斯特罗姆和贾纳斯·弗里斯推出他们新的基于电视的创业公司[威尼斯项目](https://web.archive.org/web/20210928034923/http://www.theveniceproject.com/)的时机再好不过了。

关于这家新公司的传言最早出现在 7 月下旬的[商业周刊](https://web.archive.org/web/20210928034923/http://www.businessweek.com/bwdaily/dnflash/content/jul2006/db20060724_713810.htm?chan=top%20news_top%20news)、[商业 2.0](https://web.archive.org/web/20210928034923/http://business2.blogs.com/business2blog/2006/07/skype_founders_.html) 和 [GigaOm](https://web.archive.org/web/20210928034923/http://gigaom.com/2006/07/24/skype-founders-take-on-tv/) 上。本周，更多的细节出现在 Janus Friis 的博客上，他说:

> 这很简单，真的——我们正试图将最好的电视和最好的互联网结合在一起。我们认为电视是有史以来最强大、最吸引人的大众媒体之一。人们喜欢电视，但他们也讨厌电视。他们喜欢(有时…)令人惊叹的故事，丰富性，质量本身。但是他们讨厌线性，缺乏选择，缺乏基本的东西，比如能够搜索。完全缺失的是我们现在从互联网上习惯的一切:标签、推荐、选择等等…电视是 507 个频道，什么也没有，我们想帮助改变这一点！

尽管上述内容仅仅描述了 YouTube，但《商业周刊》之前的文章给出了更多的提示:P2P 分发技术[基于 Kazaa 和 Skype 背后的](https://web.archive.org/web/20210928034923/http://featured.gigaom.com/2006/10/05/five-questions-with-skype-co-founder-janus-friis/)相同技术，处理电视网络，互联网上的时移电视。

现在正是时候。消费者的需求是存在的，电视网似乎准备迈出这一步。苹果公司已经获得了 220 多部电视节目的版权，并在 iTunes 上销售。谷歌将 YouTube 的市场估值定为 16.5 亿美元，显示了网络上简单节目剪辑的价值。电视网已经表明，他们愿意通过互联网尝试新的发行方式。如果 Niklas 和 Janus 已经成功地获得了网络节目的权利，我们即将面临的事情很可能会导致我一劳永逸地取消有线电视订阅。
