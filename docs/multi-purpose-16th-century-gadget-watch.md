# 多功能 16 世纪小工具手表| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/25/multi-purpose-16th-century-gadget-watch/>

CrunchGear 的老读者会知道，我完全赞成[融合](https://web.archive.org/web/20151223143130/http://crunchgear.com/?s=convergence)，即一个设备可以做更多事情的想法。这是 90 年代末的一个主要主题，我们可以在电脑上看电视，反之亦然，这是我们刚刚开始探索的事情。目前，我们看到许多小玩意都有次级用途。但这不是新的。事实上，这只 1590 年的怀表不仅仅是一只钟表。

除了用老方法报时，它还用真正老的方法报时，用的是日晷。不知道为什么手表需要日晷，但你有它。它还可以作为一个粉末瓶(用于合法物质)使用，并且有一个内置的指南针。我们喜欢一心多用。

它由木头和骨头制成，直径约为一张 CD，这意味着它非常适合文艺复兴时期的黄曲霉。
