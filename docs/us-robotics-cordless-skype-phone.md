# 美国机器人无线 Skype 电话

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/30/us-robotics-cordless-skype-phone/>

![](img/3071002ef5653f3628e8009bed02bd3b.png)

USRobotics 今天宣布了用于 Skype 的 USR9630 无绳电话和 USR9631 配件手机，这两款设备不是很自动化，可能也不是美国制造的。USR9630 是一个基站，可以通过座机为 Skype 提供服务，也可以通过 USB 为 PC 提供服务。它可以连接多达四个 USR9631 手机，覆盖整个房间。

该系统与其他类似设备的区别在于，当连接到座机时，它还可以接收 PSTN 呼叫。不幸的是，基站不能存储语音邮件。
