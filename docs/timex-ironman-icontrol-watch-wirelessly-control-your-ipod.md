# Timex Ironman iControl 手表:无线控制您的 iPod

[![timexicontrol.jpg](img/0d35ee238267eecfe650baa0ca6901ad.png)](https://web.archive.org/web/20221130202654/https://beta.techcrunch.com/wp-content/uploads/2007/08/timexicontrol.jpg "timexicontrol.jpg")

我觉得写 Timex Ironman iControl 很恶心。这并不是因为它让我觉得“蹩脚”或“愚蠢”——手表无线控制你的 iPod 或 iPhone，让你选择、暂停、倒带等。音乐——但是因为我最喜欢的手表评论网站还没有看过它。

除了看起来像费雪牌玩具之外，iControl 还拥有其他 Ironman 手表的所有功能，也就是说，它是一款运动手表，而不是一款你可以在正式场合佩戴的手表。你可以潜到水下 328 英尺的深度(你是谁，潜水侠？)并存储跑步数据，如跑的圈数和平均跑步时间。

这是你的标准运动手表，但它将在跑步时控制你的音乐。从理论上讲，这很好，但是在你对它做出判断之前，你肯定还需要一段时间。
