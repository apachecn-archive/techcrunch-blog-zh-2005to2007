# 创意禅视觉:M 固件更新，FM 录音回归

![](img/7dd75af16fe1c52ec5fd0d79044e8227.png)

Creative 今天宣布了 Zen Vision:M firmware 1.60.01 的可用性。新版本恢复了 M 用户录制 FM 电台的功能。没有确切的消息说为什么 Creative 选择了取消播放器，但我知道不止一个 M 用户会对这个功能的回归感到高兴。

推测认为，由于 RIAA 的高射炮火，创意已经禁用了该功能，但这似乎值得怀疑，因为它现在恢复了调频录音功能。哦，谁知道呢？
