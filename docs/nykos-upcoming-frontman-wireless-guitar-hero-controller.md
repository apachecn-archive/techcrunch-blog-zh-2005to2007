# Nyko 即将推出的主唱无线吉他英雄控制器

。Nyko 为即将到来的吉他英雄 2 发布了一款无线吉他控制器，采用了他们的商标无线[Nyko技术和 Gibson Explorer 的黑色和红色双色。](https://web.archive.org/web/20201022074128/https://crunchbase.com/organization/nyko)

1up 有机会尝试控制器，并表示所有按钮都感觉到位，吉他应该会在下个月上架，售价约为 59.99 美元。我们不确定 Red Octane 是否会像起诉自由 V 无线吉他制造商蚂蚁突击队那样起诉他们。如果你想知道它是如何工作的，看看[我们的评论](https://web.archive.org/web/20201022074128/http://crunchgear.com/2006/10/20/freedom-v-wireless-guitar-controller-review/)。
