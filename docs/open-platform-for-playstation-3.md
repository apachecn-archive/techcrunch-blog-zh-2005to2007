# PlayStation 3 的开放式平台

![](img/a3d28e7a8df59058faeb6182b8603ac0.png)

我们在浏览 [PlayStation 3 系统软件用户指南](https://web.archive.org/web/20230120124627/http://manuals.playstation.net/document/en/ps3/current/index.html)时，偶然发现了[这一页](https://web.archive.org/web/20230120124627/http://www.playstation.com/ps3-openplatform/index.html)，上面写着:

> 除了索尼电脑娱乐公司提供的系统软件之外，PLAYSTATION 3 允许在其上安装任何所需的第三方系统软件。详细信息将很快发布在该网站上。

这是一个有趣的补充。我们都知道 PS3 是最强大的产品之一，无论是计算机还是其他产品，所以看到索尼开放 PS3 进行真正的定制是很酷的。运行任何第三方系统软件的能力确实模糊了控制台和计算机之间的界限。大家对此怎么看？
