# 虚幻竞技场 III 是 PS3 的热门游戏

![ut3_logo2.jpg](img/b22a8cdf145614feef91dd0ee8f3b94d.png)

索尼最近确实在尽力让我收回我说的话。在 E3 宣布的是索尼在 2008 年前对《虚幻竞技场 III》的独家权利。在那之前，微软不能对这款游戏做出任何声明，这是一大震惊。它变得更加诱人，因为你将能够在 PS3 上运行 PC 版本的用户生成内容，如地图、皮肤和其他模块。这可能只是运气，也可能是索尼有史以来一直在努力的一个总体规划。PS3 是一个“开源平台”，这意味着索尼希望游戏玩家付费的免费内容，但至少它再次激起了人们的兴趣。你要做什么，微软？
