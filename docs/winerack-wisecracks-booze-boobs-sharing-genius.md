# 酒架俏皮话:酒+胸部=分享天才

你们中的一些女孩希望自己有更大的胸部。我们知道这一点，我们约会过你们中的 84 个人。你们中的一些人也喜欢喝酒，甚至在你们不应该去的地方。我们也和你约会过。如果你有点 A 柱，有点 B 柱，我们有酒架。

这是一款黑色胸罩，缝有气囊，可以装葡萄酒、啤酒、伏特加、雅格酒，应有尽有，还有一个方便的饮用管。不，这不是玩笑。是的，我们完全期待我们熟悉的人使用它。不，我们不会觉得从他们的鼻子里吸烈酒很脏，我们会感到*感激*，就像我们应该的那样。
