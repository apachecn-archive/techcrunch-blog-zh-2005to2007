# 为什么 Sling 这么讨厌 Palm，美国？

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/01/why-does-sling-hate-palm-america-so-much/>

照片本身可能是诱人和令人兴奋的，但这也是乐趣所在。举个例子，我们在 zatznotfunny 上找到的这张照片。它显示了 SlingBox 调谐器的盒子的一面。我们的秘密是处理不同的移动平台和操作系统的徽章，这对于电子产品来说是相当典型的。

有趣的是，当你剥开一个圆形的黑色贴纸时，会露出手掌的标志。出于某种原因，斯林不想让我们看到这个。为什么手掌 logo 会被遮挡？是因为 Access PalmSource 大约十亿年没有真正更新过 Palm OS 吗？是因为 Palm OS 的客户端(已经擅离职守多年？)还没准备好？硅谷鸡尾酒会上的一对情侣变坏了吗？我们注意到 Windows Mobile 是特色，所以智能手机以字母“w”结尾的 Treo 用户情况良好。但是我们其他人呢？

我们不知道，但我们肯定会找到答案的。
