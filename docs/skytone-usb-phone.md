# SkyTone USB 手机| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/08/27/skytone-usb-phone/>

又名 RST102 由 Radian Technologies 制造，可通过 USB 连接到任何笔记本电脑。放入一张 CD，打开 Skype，就可以出门了。

在我们下面链接的 Marc“The Sharc”or chant 的迷你评论中，值得注意的是 Skype 手机“感觉很有趣”。想想看:我们要么使用带耳机的 Skype，要么使用带麦克风和扬声器的网络摄像头。我们很少把东西放在头上，除非是手机，或者是固定电话。因此，Marc 说，用鼠标拨号和用 Skype 不同水平的音频质量说话的认知失调有点多。但我们都会克服的，对吧？
