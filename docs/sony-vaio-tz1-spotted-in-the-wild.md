# 索尼 VAIO TZ1 发现在野外

![picture-001.jpg](img/c4f72db891ef49fc8bee48cbe0d5a351.png)

还记得[索尼 VAIO TZ1](https://web.archive.org/web/20130628143735/http://crunchgear.com/2007/05/17/sony-vaio-type-t-laptop-now-with-led-backlighting/) 吗？如果你没有，我会提醒你。这是迄今为止最热门的 VAIO 笔记本电脑，亲自看到它是一种真正的享受，因为一般公众不会在一段时间内看到它，如果有的话。 [Michael Tchong](https://web.archive.org/web/20130628143735/http://ubercool.com/) 告诉我，多亏了 32GB 的固态硬盘，Vista 才变得“可以忍受”。他还提到，他想选择运行 XP，而不是提高整体速度。Vista 占用了 9GB，而 XP 占用了 5GB。当您的总容量为 32GB 时，额外的 4GB 会带来很大的不同。我非常嫉妒，当他告诉我他以前的笔记本电脑是华硕兰博基尼的时候，我不得不走开，躲在角落里哭泣。点击跳转获取更多图片。
