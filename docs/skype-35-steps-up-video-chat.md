# Skype 3.5 升级视频聊天

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/07/skype-35-steps-up-video-chat/>

Skype 推出了一种有趣的新方式与您的家人和朋友进行视频聊天，包括来自 Daily Motion 和 Metacafe 等网站的共享视频内容。亲爱的读者们，Skype 3.5 for Windows 允许您将从上述视频分享网站下载的视频加入到您的 Skype“心情”中想象一下，如果你愿意，你正在和你在哈萨克的朋友聊天，他不太理解奥斯曼帝国的“驼背”的概念，或者他没看过杰克·布朗从 45 英尺高的地方摔下。现在您只需将他们引向您的 Skype 档案。

Skype 3.5 还有一些功能，如自动重拨，您可以将来电转移到 Skype 上的另一个人或另一个组，而专业版用户可以将来电转移到座机或手机上。
