# Newsgator 发布有用的工具栏——关注博客| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/08/09/newsgator-releases-useful-toolbar/>

 [](https://web.archive.org/web/20150921143620/http://www.newsgator.com/) [Newsgator](https://web.archive.org/web/20150921143620/http://www.newsgator.com/) 刚刚宣布发布[一款新的测试版浏览器工具栏](https://web.archive.org/web/20150921143620/http://www.newsgator.com/news/archive.aspx?post=103)，供 Newsgator Online、Newsgator Inbox 和 FeedDemon 的 Windows 用户使用。增加了一些关键功能，将有助于该公司与竞争对手 [Bloglines](https://web.archive.org/web/20150921143620/http://bloglines.com/) 竞争。

用户现在可以在 IE 和 Firefox 中点击订阅任何页面的 RSS 订阅，这一功能使得 Bloglines 成为最容易使用的在线订阅阅读器。工具栏还显示新闻源的 Newsgator 订阅者数量、指向您正在查看的 URL 的入站链接、新闻源预览和通过关键字发现新新闻源的搜索框。

这是对该公司产品的一个非常聪明的补充，我很惊讶花了这么长时间，我希望它能用于 Mac。网络新闻专线提供一键订阅，但其他功能会很好。如果这只是一个 FireFox 插件，那就太理想了。

虽然 Bloglines 长期以来一直是寻找简单而强大的工具的人最喜欢的提要阅读器，但新的 Newsgator 工具栏意味着寻求许多这些功能的用户现在有了选择。Newsgator 提供了 Bloglines 没有的大量新闻功能，处理 OPML 文件更加优雅。不过，搜索功能可能不会像 Bloglines 那样好——Bloglines 由 Ask.com 支持，要求 Bloglines 用户在搜索结果显示之前订阅一个 feed，以排除垃圾博客。

Newsgator 是市场上领先的 RSS 公司之一。该公司已经为《新闻周刊》、《今日美国》和许多其他公司提供了白标解决方案。15%的 TechCrunch 用户使用 Newsgator 服务，仅次于 27%的 FireFox Live 书签。我们上次报道该公司时，它发布了 RSS 未来的路线图。
