# 微软收购移动广告公司

微软对移动领域的兴趣刚刚增加。ClickZ News 报道称，微软已经收购了 ScreenTonic，并打算将其功能整合到 adCenter 中。ScreenTonic 的核心竞争力在于移动显示广告，该公司的产品包括固定，动画和视频横幅；文本链接，广告词，间隙和视频广告牌。

随着最近收购 Tellme Networks 和 MotionBridge，微软对移动业务的兴趣越来越大。
