# StupidVideos.com 移动内容

YouTube 最近开始了其移动网站的测试版。StupidVideos.com 现在向移动观众提供超过 2.5 万个用户制作的视频。StupidVideos.com，一个纯视频网络的部门，希望吸引该网站的现有用户，以及发现移动网站的新消费者。这个移动网站是由 Quattro Wireless 公司建立的，该公司创建了允许广告支持的自适应移动网站。

视频在移动平台上仍处于早期，但像 [mywaves](https://web.archive.org/web/20130628180705/http://mobilecrunch.com/2007/05/10/abmob-and-mywaves-to-offer-mobile-video/) 这样的公司为移动手机提供聚合视频，以及一些广告支持的选项。StupidVideos.com 与聚合网站的不同之处在于，它为无线用户提供视频服务。
