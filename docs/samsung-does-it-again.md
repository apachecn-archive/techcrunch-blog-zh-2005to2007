# 三星又来了——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/27/samsung-does-it-again/>

三星刚刚发布了有史以来最薄的 HSDPA 滑盖！HSDPA？！？没错，Z720 或 Ultra Edition 13.8 通过高速下行链路分组接入加速，允许以 1.8 Mbps 的速度进行一些非常快的数据传输，据说比 3G UMTS 快 6 倍。据三星称，Z720 与市场上的其他产品不同，因为它将为用户提供更高质量的流媒体视频内容，并能够访问比任何其他 PDA 更多的文件。

现在，您可以通过 microSD slot 在 2.1 英寸 QVGA 屏幕上查看所有喜爱的节目或图片，或者使用其 HTML 和 WAP 2.0 功能上网冲浪。支持 AAC/AAC+/e-AAC+/WMA 的 MP3 播放器。一个漂亮的 300 万像素摄像头可以让你在办公室周围拍摄所有这些照片，或者拍摄办公室经理在休息时拍摄的视频，甚至还有第二个摄像头用于电话会议，这是我们在这里永远不会看到的，蓝牙和 USB 都是标准配置。这东西很棒，我不介意自己也有一个，一旦我给你更多细节，你也不会介意的。这将是第一款预装谷歌移动搜索和 Gmail 的手机。它甚至在菜单屏幕上有自己的图标。多厉害啊。Ultra Edition 13.8 还配有三星的 uGo 和 uTrack。uGo 上的界面让我想弄湿自己，因为它使用各种黑魔法，通过显示你附近的城市地标来自动调整自己，调整图片来代表夜晚或白天，并自动调整时间，确保你不会提前两个时区迟到。我想 uTrack 有点酷。如果你想增加找到被盗手机的几率。它通过向预定号码发送短信来提醒你 SIM 卡和位置的变化。但是你必须对那种事情感兴趣。

虽然它目前只在欧洲可用，但我真的希望三星会把它的一个版本带到美国，因为它会踢很多屁股。
