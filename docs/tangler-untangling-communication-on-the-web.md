# 网上交流

![tangler](img/17b292e7e81ecd020ab89d856fbd1ae3.png)

上周末，我采访了来自澳大利亚悉尼的初创公司 T2 的创始人兼首席执行官马丁·威尔斯。我看了一下他们在过去 18 个月中开发的应用程序，这是一个即时消息和通信应用程序，基于他们所谓的“即时分组”概念。

即时分组意味着任何数量的人都可以围绕他们在网上或其他应用程序中找到的感兴趣的话题参与对话。该应用程序允许用户订阅一个群组，然后可以链接到任意数量的博客、网站、音乐文件(在 iTunes 或其他应用程序中)、地图上的一个点或其他任何东西。对话是持续的，所以当其他人在的时候你不需要在那里，桌面应用程序会在群组中发生新的事情时通知你。

这缓解了一些人目前看到的即时消息应用程序与博客捆绑在一起的问题，因为它使每个群组成为用户可以发表评论和进行对话的实际目的地。Tangler 很好地将人们团结在共同的兴趣周围，并让他们参与对话。这是即时通讯已经发展了一段时间的方向，但是 Tangler 看起来已经做对了。

这些人最近花了一些时间在硅谷，为公司和产品的公开发布做准备。到目前为止，Tangler 一直由私人投资，但马丁表示，他们目前正在考虑来自美国的投资兴趣。

Tangler 计划在两个月后开放私人测试版——在此期间，你可以在他们的网站上注册你对他们产品的兴趣。我急切地等待着它的推出，因为我已经看到了许多尝试，试图让围绕一个主题或网站的交流变得更容易，似乎这些人正在以正确的方式进行，并且做得很好。
