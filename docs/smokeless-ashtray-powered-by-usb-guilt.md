# 由 USB 供电的无烟烟灰缸

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/20/smokeless-ashtray-powered-by-usb-guilt/>

无烟烟灰缸已经伴随我们好几年了，它们并不新鲜。通过使用空气对流或离子吸引，它们吸收由你的香烟引起的多余烟雾，以防止它影响你周围的人(或你的电脑)。

新的是这个 USB 供电的无烟烟灰缸。很明显，这个东西是面向老派的 Unix 程序员的，因为他们倾向于花几个小时在键盘前，嘴里叼着烟。这个工具有助于确保他们的女朋友的母亲不会受到烟雾中致癌物质的影响。

我们不明白的是，为什么它的形状像一辆汽车，配有 LED 头灯。token 外设厂商为什么一直把东西做成蹩脚的小车？iPod 基座，扬声器，集线器，现在还有烟灰缸。我猜，至少不是粉色的。
