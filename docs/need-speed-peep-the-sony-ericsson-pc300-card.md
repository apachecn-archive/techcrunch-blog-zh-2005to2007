# 需要速度？窥视索尼爱立信 PC300 卡

![](img/fb6596a86ab3ed368cfc1a258445f765.png)

在 CES 期间，如果有某种 EV-DO 卡或高速无线网卡就太好了。因为我现在知道约翰将为自己部落他们，我盯着最近宣布的索尼爱立信 PC300。

这个坏男孩不仅有 HSDPA，UMTS，EDGE 和 GPRS，而且它有一个邪恶的粉红色霓虹灯标签！它支持高达 3.6Mbps 的下载速度，并且还集成了天线。糟糕的是，我们必须等到第三季度才能修复快速无线接入。
