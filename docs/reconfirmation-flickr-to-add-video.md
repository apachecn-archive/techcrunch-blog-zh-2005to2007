# 再次确认:Flickr 将添加视频

 [](https://web.archive.org/web/20220813223104/http://www.crunchbase.com/company/flickr) 五月 [Flickr](https://web.archive.org/web/20220813223104/http://www.crunchbase.com/company/flickr) 的联合创始人斯图尔特·巴特菲尔德漫不经心地向我提到，他们将“很快”允许用户上传视频(见本文最后一句)。昨天，雅虎视频总经理 Mike Folgner [再次证实了这一点:“雅虎的 Flickr 照片分享网站也将增加视频，”他说。](https://web.archive.org/web/20220813223104/http://www.bloomberg.com/apps/news?pid=20601204&sid=aulvw5OZgWww)

这是另一个扔掉的声明，是一篇更大的文章的一部分，关于雅虎改造其视频网站，以提高与 YouTube 的竞争力。但坦率地说，这是他说的唯一有趣的事情。玩追赶很糟糕，这就是雅虎视频现在的处境。他们将不得不付出比 YouTube 两倍的努力，而且不太可能得到他们所期望的回报。

但是 Flickr 需要视频。很多时候，我旅行归来，把照片上传到 Flickr，然后把视频存在我的硬盘上。我希望这些视频就在照片旁边，有相同的标签和相同的设置(相册)。他们两年前就应该加上这个…今年晚些时候我会很高兴看到它。
