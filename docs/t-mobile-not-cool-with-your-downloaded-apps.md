# T-Mobile 对你下载的应用不感冒

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/27/t-mobile-not-cool-with-your-downloaded-apps/>

看起来在 T-Mobile 的土地上并不是一切都很顺利。该公司已经改变了政策，并告诉拥有 Java 手机(几乎是所有手机)和数据套餐的用户，他们不允许安装第三方网络应用程序。为什么政策会有如此剧烈的变化？T-Mobile 声称存在安全问题，但我们都知道这是一派胡言，旨在阻止独立手机销售。有了这个禁令，这意味着没有 Opera Mini，没有 Google Local，没有 Gemdrop 的完整版本，什么都没有。

从现在开始，如果你想在手机上安装任何东西，你必须通过地狱般的 T-Zones 界面来支付高价的应用程序和游戏。如果 T-Mobile 想留在游戏中，不要输给 Cingular 更多，它最好开始改变它的政策，回到正常的 post haste。
