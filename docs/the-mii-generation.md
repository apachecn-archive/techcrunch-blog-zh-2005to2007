# 信息产业部的一代

当任天堂有数百万 Mii 角色为他们工作时，谁还需要演员？从下周开始，美国的任天堂将推出新一轮的 Wii 广告，以现在成为 Wii Sports 和 Wii Play 等游戏一部分的流行漫画为特色。新广告将以一个家庭在他们的客厅为背景，另一个发生在公司董事会会议室——这表明任何地方都是创造迷你视频游戏版本的你的好地方。这些广告也可以在 [Wii 的 MySpace 页面](https://web.archive.org/web/20130628171139/http://myspace.com/howwiiplay)上看到。哦，米莉，哦，我的天！
