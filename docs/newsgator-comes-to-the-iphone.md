# NewsGator 来到 iPhone 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/24/newsgator-comes-to-the-iphone/>

[![newsgatorlogo.png](img/e63860a87060beea46cce9555db91c18.png)](https://web.archive.org/web/20160422034850/http://newsgator.com/) 购买一部 iPhone 已经开始成为进入一个专门制作的应用程序私人网络的独家俱乐部的门票。iPhones 有了自己的聊天、国际象棋和约会应用。现在，iPhone 用户可以坐下来，啜饮他们的拿铁咖啡，享受 NewsGator 提供的另一款美味的 iPhone 应用程序。

RSS 阅读器 NewsGator 今天发布了他们产品的 iPhone 版本。现在，当 iPhone 用户登录 m.newsgator.com 时，他们将收到一个免费的为触摸屏定制的 iPhone 优化版网站。

NewsGator Mobile for iPhone 可以与所有现有 NewsGator 产品上的订阅源同步，包括 NetNewsWire、NewsGator Online 和 FeedDemon。

他们已经有了 Java 和 Windows Mobile 手机的付费客户端，我们之前在报道过。这些客户端包括剪报和离线阅读等附加功能。

读者也应该熟悉大卫·坎恩在 iPhone 发布后不久开发的[谷歌阅读器皮肤](https://web.archive.org/web/20160422034850/http://davidcann.com/iPhonify/GoogleReader/)。
