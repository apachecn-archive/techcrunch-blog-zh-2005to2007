# vpod.tv 融资 510 万美元:第一眼

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/05/19/vpodtv-raises-51-million-damn-good-video-site/>

 [](https://web.archive.org/web/20220117015006/http://www.vpod.tv/) 我在我的 [Innovate 2006 综述](https://web.archive.org/web/20220117015006/http://www.beta.techcrunch.com/2006/05/19/europe-shows-its-stuff-innovate-2006/)中提到了 [vpod.tv](https://web.archive.org/web/20220117015006/http://www.vpod.tv/) ，一家总部位于巴黎和马德里的初创公司。今天他们宣布了 510 万美元的 A 轮融资，由 Innovacom 的 Frédéric Humbert 牵头。

vpod 是一项“同类最佳”服务，在[迅速](https://web.archive.org/web/20220117015006/http://www.beta.techcrunch.com/2006/04/05/online-video-sites-breeding-like-rabbits/)发展[和](https://web.archive.org/web/20220117015006/http://www.beta.techcrunch.com/2006/04/02/motionbox-best-online-video-sharing-so-far/)[日益拥挤的](https://web.archive.org/web/20220117015006/http://www.beta.techcrunch.com/2006/05/16/aol-to-release-youtube-clone/)视频共享空间。 **vpod 在代码转换、编辑/显示和货币化方面做着无人能及的事情。**

坦率地说，在线视频领域变得如此疯狂，以至于我把创始人 Rodrigo Sepulveda Schulz 和 Ivan Communod 的产品演示推迟到了会议结束。真正的赢家似乎是 Adobe——其新的 Flash 开发平台如此强大，以至于每个人似乎都在开发自己的视频产品，而且大多数基本上都是同一事物的变体。

 vpod 可能不同。它的总部设在欧洲，这将使它在美国玩家互相攻击时获得优势——同时他们可以增加欧洲市场份额。此外，像 Revver 一样，vpod 也专注于视频货币化。

vpod 将允许广告商轻松制作广告，并在他们自己的网站上播放和/或插入用户的视频。广告将被插入在视频的开始，中间或结尾，长度为 10-60 秒。用户可以付费退出广告。通过选择加入，他们将从 vpod 获得收入分成——大约在 50%的范围内。这比竞争对手 Revver 迄今为止所宣布的更进一步。

但是 vpod 在其他领域也大放异彩。他们可以从几乎任何来源获取视频，更重要的是，可以为几乎任何设备进行转码。想要在 PSP 上使用 Mpeg-4 吗？vpod 将允许您下载该格式。iPod 的 Mpeg？你也可以拥有它。只要告诉 vpod 你的设备，它就会给你正确的格式。

最后，在 vPod 上创建你自己的“频道”的工具是惊人的。唯一的闪光就在玩家周围。其他的都是 javascript 和 dhtml。例如，它们允许通过拖动鼠标来调整框的大小，这是我以前只在 Flash 中见过的。

vpod 处于私人测试阶段。在[主页](https://web.archive.org/web/20220117015006/http://www.vpod.tv/)注册。他们允许人们慢慢进入，为秋季发布做准备。
