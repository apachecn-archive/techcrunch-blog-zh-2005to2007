# 我离不开的 Web 2.0 公司

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/12/30/web-20-%20%20companies-i-couldnt-live-without/>

在过去的几周里，随着 2005 年接近尾声，有许多关于 2005 年“最佳”和 2006 年“预测”的帖子。我不打算写这些。鉴于我已经写了整整六个月的博客，颁发“最佳”奖对我来说似乎有些冒昧。尽管预测很有趣，但它们最终并不那么有用。

当我回忆再过几天就要结束的一年时，我想写的是我每天都喜欢和使用的 Web 2.0 公司。

今年我已经测试了一千多种产品，并且写了几百种。虽然我所写的一些公司获得了非常积极的评价，但我发现对产品价值的唯一真正测试是它的持久性:随着时间的推移，我是否会继续使用该产品，甚至可能会为此付费？

对于那些好奇的人来说，这里有一个简短的公司列表，它们吸引了我的注意力，我不会选择离开网络而生活:

**[博客](https://web.archive.org/web/20100113011831/http://www.bloglines.com/)**

我对 Bloglines 又爱又恨，但是他们最近显著地提高了性能，我真的很喜欢看到每个订阅源的订阅人数。这是最难列入清单的一个，但在一天结束时，我不能离开他们。

**[Del.icio.us](https://web.archive.org/web/20100113011831/http://del.icio.us/)** 
![](img/353e052ff33b0d3c9e799c615238beb0.png)我每天都会多次使用 del . icio . us 来存储和检索书签。我坦率地承认，还有更好的解决方案，在不久的将来，我很可能会转向其中的一个，但你必须承认 Del.icio.us 发明了社交书签现象。

**[FeedBurner](https://web.archive.org/web/20100113011831/http://www.feedburner.com/)** 
![](img/bd3b84eef1d66132ee319cdb2e159432.png)我喜欢 FeedBurner 提供的关于 feed 读者群的统计数据，并且拥有许多对我很重要的[高级功能](https://web.archive.org/web/20100113011831/http://www.techcrunch.com/2005/12/13/feedburner-integrates-web-services-into-feeds/)。不管我过去写了什么，我了解并信任 FeedBurner 团队。我只希望他们能把我页面上的广告去掉。:-)
 **[Flickr](https://web.archive.org/web/20100113011831/http://www.flickr.com/)**

![](img/b1de2c69995c309f0acb9c1c1bfebfda.png)我每天都越来越喜欢 Flickr。我喜欢根据我朋友上传的照片来了解他们在做什么，也喜欢从别人那里获得对我照片的评论。我开始回去上传几年前的旧照片。Flickr 太完美了。

**[测量图](https://web.archive.org/web/20100113011831/http://www.measuremap.com/)**

由 Adaptive Path 创建的 Measure Map 博客分析工具让我对谁在 TechCrunch 上看什么有了难以置信的了解。不过，他们需要处理大型博客的速度问题(有时需要几分钟来获取统计数据，或者只是中断)。
 **[Memeorandum](https://web.archive.org/web/20100113011831/http://tech.memeorandum.com/)**

当我没有时间阅读我所有的订阅时，Memeorandum 是我保持博客世界的方式。它也改变了我写博客的内容和方式。模因论是一种文化现象。

是的，有很多 Ajax 桌面，但是 Netvibes 似乎在功能上保持领先。flickr 的东西很棒。另外，我怎么能不喜欢把 TechCrunch 作为默认订阅源的服务呢？:-)

**[Omnidrive](https://web.archive.org/web/20100113011831/http://www.omnidrive.com.au/)**

我一直在等待像这样的事情。我预见有一天，像 Omnidrive 这样的服务会与新的个人电脑打包在一起，或者与网络电子邮件解决方案一起提供。我才买了几天，但是我被迷住了。公平披露:也有一些令人敬畏的竞争对手，我才刚刚开始关注它们。
 **[潘多拉](https://web.archive.org/web/20100113011831/http://www.crunchbase.com/company/pandora)**

每当我写作的时候，我都会听潘多拉——有时一天听几个小时。我从中发现了无数新的艺术家。
 **[Skype](https://web.archive.org/web/20100113011831/http://www.skype.com/)**

我能说什么呢？和 Vonage 一样，Skype 让我的电话账单几乎为零，它是我日常工作和个人生活不可或缺的一部分。然而，我愿意用应用程序共享来换取新的视频功能。

我用它的次数比谷歌还多。目前还没有人推出更好的产品。下半年，他们在搜索速度上取得了巨大进步。
 **[按](https://web.archive.org/web/20100113011831/http://www.wordpress.org/)**

我喜欢 Wordpress。实际上，让我重新表述这句话:我爱 Wordpress 1.5。2.0 版本让我想把笔记本电脑扔出窗外。但这是一款令人惊叹的软件，我所有的博客都在上面运行。
 **[雅虎地图](https://web.archive.org/web/20100113011831/http://maps.yahoo.com/beta)**

我使用雅虎地图，因为它允许多点驾驶指示，这是其他任何地图都无法提供的。当我不得不在同一个晚上参加三四个节日聚会时，这非常有用。
