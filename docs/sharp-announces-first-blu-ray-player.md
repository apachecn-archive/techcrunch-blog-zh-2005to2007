# 夏普发布首款蓝光播放器

![sharpbd-hp20s-lg.jpg](img/db836e99c096b27a25639b9dec41c19d.png)

夏普宣布了他们的首款蓝光播放器 BD-HP20S，其规格至少可以说相当诱人。对消费者来说，最大的难题是光盘开始播放需要多长时间，对吗？夏普显然听到了所有的抱怨，并采取了一些措施。“快速启动”功能声称在几秒钟内启动光盘，但当我相信它时，我会看到它。

细节有限，但我们知道它将有 HDMI v1.3，5.1 模拟输出，支持 Dolby TrueHD，以及 24fps 的 1080p 输出。预计今年秋天它的价格会超过 500 美元。
