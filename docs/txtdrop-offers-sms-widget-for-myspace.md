# TxtDrop 为 MySpace 提供短信小工具

 [](https://web.archive.org/web/20211017005243/http://www.txtdrop.com/) 基于网络的文本信息公司 [txtDrop](https://web.archive.org/web/20211017005243/http://txtdrop.com/) 今天宣布了一个 MySpace 档案(当然还有其他网页)的新工具，它创建了一个按钮，可以向网页所有者的手机发送短信。这意味着短信发送者不必为每条短信付费。该服务仅在美国和加拿大可用。

我不打算在这里张贴一个测试按钮！(部分原因是这种丑陋只有 MySpace 用户才会喜欢。)不过，我可以想象，许多人会发现这非常有用。SixApart 的米娜·特罗特(Meena Trott)表示，在线社交媒体的未来在于非常小的群体——如果你有一个仅限朋友的个人资料页面或其他方式，可以合理地期待可控数量的消息，那么这可能就是适合你的服务。

MobileCrunch 的 Oliver 深入探讨了严重的安全问题，以及他对这种服务如何变得更加可行的想法。请继续关注更多关于小部件浪潮的报道。
