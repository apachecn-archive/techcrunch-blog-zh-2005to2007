# Palm 转向 Linux

埃德·克里根在今天的投资者日主题演讲中宣布，Palm 将在今年晚些时候发布一款新的操作系统。正如你可以从大家都在发布的漂亮图表(上图)中看到的， [Palm](https://web.archive.org/web/20200813074025/https://crunchbase.com/organization/palm) 并没有走向 windows，而是打开了它的闪光灯，开始与 Linux 合并。从技术上来说，它将“结合 Palm OS Garnet 和 Linux 内核的各个方面”,正如 Palm 信息中心所写的那样。这一转变被视为以更低的成本提供更好的用户体验的必要举措，这款手机可以支持即时启动、即时应用切换和更长的电池寿命。新的设置也将运行 Opera 作为浏览器的选择。尽管如此，Palm 将继续支持 Windows Mobile 手机。

山姆·詹姆斯有科里根的[完整演示](https://web.archive.org/web/20200813074025/http://palmaddict.typepad.com/palmaddicts/2007/04/new_palm_os_wil.html)。
