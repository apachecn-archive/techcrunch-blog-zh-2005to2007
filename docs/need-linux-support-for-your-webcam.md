# 你的网络摄像头需要 Linux 支持吗？

我不是 Linux 用户，也不需要购买任何网络摄像头，但如果我处于这种情况下，我会感谢米歇尔·查德。这位 60 岁的法国医生独自负责为廉价的中国网络摄像头编写 235 个 Linux 驱动程序，这样当你意识到它不能在你的 Linux 机器上工作时，你就不会把它扔出窗外。是什么迫使一个人去写那些驱动程序？他还是那个疯狂的法国网络摄像头狂吗？不，他似乎是一个温和的人，只是想把事情掌握在自己手中。正如他们所说，他疯了，再也受不了了。
 这一切都始于 2003 年，当时 Michel 为他的女儿们购买了网络摄像头，但他很快意识到没有 Linux 支持，于是 spca50x 项目开始了。父亲不愿为孩子做的事让我热泪盈眶。他在没有任何支持的情况下这样做的事实令人震惊，但是大多数业余爱好者不会为了荣誉而做这种事情。他们这样做是因为他们不想等着大部队来听他们的请求，这对他们来说很有趣。干得好，米歇尔。如果你是 Linux 用户，你的网络摄像头有问题，那么看看米歇尔的[页面](https://web.archive.org/web/20130628194345/http://mxhaard.free.fr/spca5xx.html)。
