# 三星推出 1TB 硬盘

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/12/samsung-shipping-1tb-hdd/>

我们都喜欢的三星公司正在大力推进，凭借比其他任何产品都更优越的 1TB 硬盘在硬盘市场上立足。你说，胡说八道？一点也不。SpinPoint F1 是第一个使用三个盘片的产品，而不是 Hitachi 和 Seagate 在其 1TB 驱动器中使用的四个盘片。

三个 334GB 的盘片由 7，200RPM 马达驱动，具有 16MB 或 32MB 高速缓存。三星的专有技术提高了性能，降低了我们讨厌的噪音。据说读取寻道时间为 8.9 毫秒，而平均延迟仅为 4.17 毫秒。SpinPoint F1 王朝还将包括 320GB、500GB 和 750GB 型号。零售价还不清楚，但他们现在发货。
