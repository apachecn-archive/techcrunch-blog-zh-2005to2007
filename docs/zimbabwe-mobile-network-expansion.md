# 津巴布韦移动网络扩展

大规模失业、几十年的零星内战和其他经济困难不足以阻止人们保持联系。因此，津巴布韦，这个曾经被称为罗得西亚的非洲内陆国家，正在寻求在未来六个月内增加移动用户容量。

津巴布韦的 Econet 正在扩展该国的蜂窝网络，并在明年 2 月之前将用户承载能力从目前的 80 万增加到 120 万。Econet 与爱立信和中兴通讯签署了合同，为扩建提供设备。
