# 斯坦顿的 T.90 USB 高扭矩转盘

[![t90_angle.jpg](img/34d5000dd20f7e76ec6a569a29c65d78.png)](https://web.archive.org/web/20210304195116/https://beta.techcrunch.com/wp-content/uploads/2007/04/t90_angle.jpg "t90_angle.jpg") 
斯坦顿[的新 T.90 USB 高扭矩转盘肯定不是我们看到的第一个通过 USB 将你的黑胶唱片转换成电脑上的数字文件的](https://web.archive.org/web/20210304195116/http://www.ion-audio.com/)，但它绝对比大多数专业或业余 DJ 更能满足要求。

除了与其同名的高扭矩直接驱动电机，T.90 还具有一个内置的“键锁”，用于在不影响音高的情况下调整唱片的速度，一个可选范围的音高控制滑块和 33rpm、45rpm 和 78rpm 的回放速度。S 形拾音臂在“刮擦”时为您提供卓越的跟踪能力，并且有两个启动/停止开关用于混音或战斗设置。

还包括一个 S/PDIF 输出，一个斯坦顿 500B 墨盒，防滑垫和布防尘盖以及软件转换成数字录音或光盘。

更新:T.90 的建议零售价为 435 美元。当它上架时，预计它会停留在 TTX 的范围内。
