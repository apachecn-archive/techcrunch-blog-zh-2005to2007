# 选项 USB 环球冲浪图标 HSUPA 调制解调器

这个世界正在摆脱束缚，我为此感到兴奋。Option 刚刚宣布了其 GlobeSurfer HSUPA 调制解调器的 USB 版本。该器件能够吸入 7.2Mbps 的下行流，并泵出 2Mbps 的上行流。

它有一个旋转的 USB 头，所以它应该能够很容易地适应任何 USB 端口，尽管有讨厌的外壳或其他东西。据说它提供即插即用功能，并向后兼容 EDGE、GPRS 和 GSM。该调制解调器还支持三频/四频，基本上可以在任何操作系统上工作。

还没有关于价格或可用性的消息。
