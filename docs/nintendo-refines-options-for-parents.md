# 任天堂为父母改进选择

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/30/nintendo-refines-options-for-parents/>

![](img/712586141cbf2fc90aed901a57517204.png)

任天堂今天发布了 Wii 的更新，让用户可以更好地控制主机上可用的家长选项。仍然没有关于频道何时到来(新闻、天气和更多)或我们何时将看到 Opera 浏览器的消息。如果你决定升级你的 Wii，要考虑到 Wii 已经有的浏览器漏洞可能不再有效了。少数用户在加载某些游戏时也遇到了问题。看来我得去更新我的游戏机了，这样除了我没人能使用它。
