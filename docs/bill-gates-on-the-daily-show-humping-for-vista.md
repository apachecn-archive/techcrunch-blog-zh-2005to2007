# 比尔·盖茨在每日秀上为 Vista 而战

你不需要任何愚蠢的博客来说服你，Vista 是迄今为止最好的 Windows 操作系统。不，你需要微软的董事长比尔·盖茨，他昨晚和《每日秀》的约翰·斯图尔特坐了一会儿，讨论了操作系统、密码保护和那个讨厌的 F12 键。

看起来《每日秀》的工作人员全力以赴迎接主席的到访，因为 Vista 的标志无处不在。如果迷人的微软老板没有让你想得到 Vista，我们不知道什么会。跳完后看第二部分。

我们最喜欢的部分是当他完成后离开的时候。变焦！
