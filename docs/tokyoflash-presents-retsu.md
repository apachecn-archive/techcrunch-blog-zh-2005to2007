# TokyoFlash 呈现 Retsu

![](img/dcdc369bec5bf35bca288582bf65e525.png)

Saishin 的手表 Retsu 是我们见过的最小的 LED 手表之一，它只能从 TokyoFlash 获得。Retsu 只有 1.4 盎司，比 TokyoFlash 上的大多数手表都要轻很多。看时间也不像那里的大多数手表那样令人困惑:

> 令人惊讶的简单阅读，灯比赛的数字，然后离开留下一个灯再次确认数字。每个数字都重复这一过程来显示时间。如此简单，令人惊讶的是从来没有人想到过它。

好的，这样更容易，但是看起来要花更长的时间来显示时间。TokyoFlash 似乎对 Retsu 非常有信心，但有了那性感的外表，我不能争辩太多。Retsu 还采用了黑色麂皮表带和非常简单的设计。130 美元的定价也不算太低。
