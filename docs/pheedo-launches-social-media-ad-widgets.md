# Pheedo 推出社交媒体广告插件

[![](img/f2a6d4be429bb08db99c2c58f18baaed.png) ](https://web.archive.org/web/20220926083302/http://www.pheedo.com/) RSS 广告网络 [Pheedo](https://web.archive.org/web/20220926083302/http://pheedo.com/) 将于今晚宣布一项新服务——由 RSS 驱动并整合了几个社交媒体工具的广告插件。熟悉 [Techmeme](https://web.archive.org/web/20220926083302/http://techmeme.com/) 的[广告项目](https://web.archive.org/web/20220926083302/http://www.beta.techcrunch.com/2006/09/25/techmeme-invents-new-kind-of-advertisment/)的读者会认识到这里的想法——当今的公司正在制作创意内容来吸引访问者访问他们的网站，这些内容可以明确地作为广告加以利用。Pheedo 希望使这种方法可扩展并广泛部署。这些小工具可以放在任何网页和许多其他广告网络上。

这是一场赌博，但如果广告业加快生产引人注目的社交媒体内容的步伐，这可能是一场非常明智的赌博。

![](img/e97943420d2aac6d1951c112c24b072d.png)这些小部件可以由任何 RSS 提要填充，在 [Pheedo 主页](https://web.archive.org/web/20220926083302/http://pheedo.com/)上可以看到这些小部件的实例。提要中的条目后面是按钮，用于在广告商选择的第三方社会化书签站点中标记这些条目。 [Opera](https://web.archive.org/web/20220926083302/http://opera.com/) 例如，Pheedo 中的广告项目可以在 Furl、Del.icio.us、Digg 或 Technorati 中标记——Expedia 广告可以在 Del.icio.us、Reddit 或 Google 书签中标记。任何项目都可以通过电子邮件发送。

视频提要也可以在小部件中提供，现在显示一个链接的缩略图。

*你是否经常在 Reddit 或 Digg 中看到你想标记的广告内容？Pheedo 将面临的第一个障碍是，它的广告客户中很少有人可能制作出足够引人注目的内容来使这些标签按钮(或许还有整个小工具)看起来合法。这是一个伟大的愿景——广告商以一种智能、创造性的方式加入对话。也许将来我们会看到更多这样的事情发生。*

Opera 是一个有趣的例子，Pheedo 说，经过一个月的测试，他们的广告插件已经有 1000 名订户。令人印象深刻。如果越来越多的公司可以开始在线制作这种令人满意的广告内容，那么 Pheedo 的广告插件就可以很好地为这种市场转变提供基础设施。
