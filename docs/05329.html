许多PDA和智能手机爱好者更喜欢Windows Mobile，而不是Foldger的Crystals，他们都对下一代操作系统感到兴奋，目前代号为“Crossbow”当它上市时，它可能会被称为Windows Mobile 6，根据MS Mobiles泄露的图片，它看起来非常光滑。

Web 2.0风格的外观看起来非常Vista-rific，关于热浴缸的消息是，它将具有直接从Vista提升的文件夹浏览器。微软之前曾表示，它很可能会每年左右发布一个新版本的WinMo，这将取决于运营商和设备制造商来保持手持设备的更新。

期待Windows Mobile 6在明年年初推出几个版本，因为微软现在正准备向设备制造商提供Crossbow。
