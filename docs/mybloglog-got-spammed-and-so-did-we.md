# 我的博客收到了垃圾邮件(我们也是)

 [迈克尔·詹森](https://web.archive.org/web/20220818232514/http://www.soloseo.com/blog/2007/01/10/free-advertising-on-techcrunch-with-mybloglog-flaw/)表明发送垃圾邮件 [MyBlogLog](https://web.archive.org/web/20220818232514/http://www.mybloglog.com/) (最近[被雅虎](https://web.archive.org/web/20220818232514/http://www.beta.techcrunch.com/2007/01/08/yahoo-buys-mybloglog-no-they-didnt-wait-yes/)收购)是相当容易的，他用 TechCrunch 作为他的实验品。像许多其他博客一样，我们显示 MyBlogLog 小部件(右下侧边栏)，它显示最近访问过该网站的人，以及他们的照片。

他所做的只是创建了一个新帐户，其中包含了他想要的广告图片。然后，他在 Opera 浏览器中打开 TechCrunch，并将其设置为每分钟自动刷新一次。结果是“用户”不断返回 TechCrunch，并弹出到小部件的顶部。一些点击进入 MyBlogLog 用户页面的流量又回到了目的地。

考虑到这很容易做到，这当然值得努力。詹森很快扩大了他的测试范围，包括了另外二十个博客。我没有注意到这一点，但是 MyBlogLog 的创始人 Scott Rafer 注意到了，并告诉了我这件事。他说他们将在不久的将来阻止这种行为。

MyBlogLog 上还有很多其他垃圾邮件，尽管大多数都是用户可以在他们的页面上给其他用户留下的信息。除了一个乱七八糟的 MyBlogLog 网站之外，没有太大的危害，我相信随着时间的推移，他们也会采取措施限制这种情况。

顺便说一句，Jensen 说他这么做只是为了指出 MyBlogLog 的缺陷，而不是真正的垃圾网站。
