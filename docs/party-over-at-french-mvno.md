# 法国 MVNO 的派对

![ten.jpg](img/cbcc53b728c8c659de7b44a200f87f05.png)你认为 ampd Mobile 的情况不妙！嗯，法国 MVNO Ten Mobile 没有现金了，因此根据传闻，最后一次被看到是在法国高速公路上举着一个牌子，上面写着“fournira le service Mobile pour la nourriture”(“将提供移动食品服务”)。

好吧，这还不算太糟，但据 mocoNews 报道，这家拥有约 15，000 名客户的公司在过去一年中已经消耗了约 1，000 万欧元(1，336 万美元)，目前正在寻求额外的资金。Ten Mobile 曾为其客户提供 IP 数据服务和包括即时通讯在内的互联网功能，但其他运营商开始支持这些服务，使得 Ten 的产品变得不那么独特。
