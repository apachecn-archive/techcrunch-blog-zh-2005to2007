# 汤姆·克兰西的彩虹六号:维加斯支持 Xbox Live 视觉

![](img/eb6d2f97c78b26548661d928d6868e07.png)

厌倦了为下一代 *Uno* 使用你酷炫的新 Xbox Live Vision？好吧，现在终于有一个大标题将 Xbox Live 的愿景融入到它的游戏中。汤姆·克兰西的《彩虹六号:维加斯》将允许用户截屏他们的脸，然后放到他们自己定制的播放器上。定制优惠只适用于在线游戏，但这是最重要的。你也可以定制你的角色的头部来匹配你的脸——就像你真的在游戏里一样！通常的附加组件也是可用的——你可以定制武器、伪装、衣服和防弹衣。

如果你不熟悉*彩虹六号:维加斯*，这是该系列的第五款战术射击游戏，粉丝们迫不及待地想要得到它，尤其是在看到*幽灵侦察:高级战士*的成功之后。看起来我们在 11 月会因为*战争机器*、*彩虹六号:维加斯*、*使命召唤 3* 和 *F.E.A.R.* 而损失很多钱，这只是下个月即将推出的几个伟大的游戏。如果你打算买一台任天堂 Wii 和/或 Playstation 3，你最好攒下一大笔钱，以便能够跟上。
