# MySpace 与 Rogers Wireless 合作

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/13/myspace-partners-with-rogers-wireless/>

MySpace 正在走向移动化。新公司的 MySpace 宣布与加拿大无线运营商 Rogers Wireless 合作。交易的财务条款没有披露，但这种合作关系将允许罗杰斯的客户使用精选手机访问 MySpace。

客户将能够访问社交网站来检查他们的页面，编辑个人资料(以防你需要在旅途中更改信息)，并使用其他功能。加拿大版的 MySpace 于今年 5 月推出。
