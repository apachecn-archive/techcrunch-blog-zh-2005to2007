# Thanko 发布 USB Wii-mote 充电器

感谢日本，我感谢你给我这个超级棒的 USB Wiimote 充电器，这样我就不用每隔几天就去买新电池了。Thanko 充电器配有 USB 底座、镍电池和新电池盖，因此您可以为这些新电池充电。新电池可提供长达 25 小时的电量，并在 3.5 小时内充满电。从任何 USB 端口充电，并有蓝色发光二极管，让您知道何时完成充电。

月底将在日本上市，你可以花 55 美元买一双，或者花 30 美元买一只。
