# MusicStrands 的新功能

 [](https://web.archive.org/web/20221202105423/http://www.musicstrands.com/) [Musicstrands](https://web.archive.org/web/20221202105423/http://www.musicstrands.com/) 是我去年 12 月写的关于[的一个音乐发现和搜索引擎。他们最近推出了新的功能(包括今天的一些)，使它成为你使用的任何音乐播放器的一个极好的辅助服务。](https://web.archive.org/web/20221202105423/http://www.beta.techcrunch.com/2005/12/15/musicstrands-does-music-search-too)

联合创始人 Gabriel Aldamiz-Echevarria 上周向我介绍了他们的一些新功能。MusicStrands 最适合作为 iTunes 的伴侣(他们在主页上有一个 [PC 插件](https://web.archive.org/web/20221202105423/http://www.musicstrands.com/mystrands/download.vm)，Mac 版本即将推出(我正在我的机器上测试一个未发布的测试版，效果很好)。

这项服务会观察你听的内容，并开始根据这些行为进行联想。如果您的播放列表中包含多首歌曲，它们就会关联起来。如果你听一组歌曲，它们会变得相关联，等等。“我们正在建立一个巨大的歌曲和艺术家之间的关联矩阵，”加布里埃尔向我解释说。用户可以使用这个矩阵来发现新的音乐并建立播放列表。

以下是一些新功能:

1.  播放列表生成器，它基本上是告诉你:其他人和你有相同的音乐，并且想听和你现在想听的相同类型的音乐，他们会用推荐的歌曲建立播放列表。对于那些拥有成千上万首歌曲的人来说，这是一个关键的功能，可以帮助他们重新发现他们的音乐库，并帮助他们填充他们的 iPods。**如果你使用 iTunes 插件，MusicStrands 会查看你的曲库中的当前音乐，并帮助你从这些音乐中创建播放列表。**这绝对是杀手锏——我有太多从未听过的音乐(翻录老友记 CD 等)。).
2.  [m-charts](https://web.archive.org/web/20221202105423/http://www.musicstrands.com/m-chart) ，最酷的是你的“最近播放的曲目”的 m-chart，m-chart 会随着你的听歌自动更新，这样你的朋友和读者就可以通过你的听歌行为轻松发现音乐。你可以将它定制成你喜欢的颜色、尺寸、格式等。
3.  社交倾听:它帮助你实时了解你的朋友在听什么。因为朋友是发现音乐的最佳来源，MyStrands 会告诉你你的联系人现在在听什么，它还会告诉你还有谁在听你现在正在听的同一首歌。
4.  [世界地图](https://web.archive.org/web/20221202105423/http://www.musicstrands.com/map.vm)——告诉你谁在哪里听什么歌。

看看 MusicStrands [Labs](https://web.archive.org/web/20221202105423/http://labs.musicstrands.com/) 看看他们正在开发的其他功能。在我看来，有很多有趣的功能可以帮助用户发现音乐；但是它的杀手锏是 iTunes 的插件，可以帮助你在自己的音乐库中建立你不知道的关系。
