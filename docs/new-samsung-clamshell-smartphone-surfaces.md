# 新款三星翻盖式智能手机表面

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/05/new-samsung-clamshell-smartphone-surfaces/>

一款未发布的三星智能手机，似乎是针对黑莓粉丝的。

这款翻盖式智能手机采用 Windows Mobile 5，专为 GSM/EDGE 网络(支持 UMTS)而设计，配有一个 200 万像素自动对焦外置摄像头和一个用于视频会议的内置摄像头。从功能角度来看，MicroSD 插槽、FM 接收器和立体声蓝牙使这款相机变得非常完美。

有趣的是，它还具有三星的黑莓式混合键盘。[首次在 SGH-I719 上看到](https://web.archive.org/web/20131104144526/http://www.phonescoop.com/articles/samsung_2006_03/index.php?p=7)，键盘抛弃了传统的 T9，成为半 QWERTY 解决方案。每个 chicklet 键都有两个字母，所以双击可以将 Q 转换为 w。这是一种将完整的数据输入放入小型设备的有趣方式，RIM 粉丝在适应它方面没有任何问题。实用吗？那完全是一个不同的问题。
