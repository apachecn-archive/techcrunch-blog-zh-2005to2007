# PalTalk 将为社交网络提供大规模多人视频聊天

视频聊天公司 [PalTalk](https://web.archive.org/web/20221001094614/http://paltalk.com/) 将于周一推出一项新的社交网络服务，旨在验证将视频聊天纳入社交网络的概念。PalTalk 已经在为大型视频聊天室提供动力方面开展了蓬勃发展的业务，但该公司认为，他们的技术是创造他们所谓的社交网络 3.0 的下一个合乎逻辑的步骤。他们说，如果第一次迭代是基本的联网，2.0 是在网络上分层更丰富的自我表达，3.0 是实时视频通信。对我来说有道理。

PalTalk 通过 Windows 桌面客户端运行，并且已经证明了自己的高度可扩展性。它利用 GAIM 创建了一个涵盖 AIM、雅虎信使和 ICQ 的综合好友列表。该公司表示，其 400 万用户经常在任何给定时间开放 5000 个聊天室。他们曾经开放的最大的房间容纳了 3000 人。电台明星 Opie 和 Anthony 使用 PalTalk 作为他们的聊天室，经常同时与 1 到 2 千人互动。我问 PalTalk 背后的人这在逻辑上如何可行，他们说有很多人举手赞成。

该公司希望，周一推出的名为 People 2.0 的社交网络系统将展示视频聊天在社交网络中的实用性和可行性。这项技术将被授权给任何一群人在线聚集的地方。PalTalk 的技术目前是免费下载的，可以付费升级，但没有价格或服务水平的比较容易被发现。

没有 Mac 版的 PalTalk 是一个真正的遗憾，该公司表示这是他们最常提出的要求之一。周一公布的特定社交网络实现在其他功能和细节方面可能并不重要，但该公司确实告诉我，他们对追求开放的身份标准感兴趣——不管这有什么价值。

如果社交网络真的不仅仅是个人之间的联系，如果人们确实想与感兴趣的社区实时联系，那么群组视频聊天对我来说就有意义了。社交网络世界中的文本聊天现在如此缺乏活力，我可以想象视频聊天会让这个空间着火。人们会下载桌面客户端来访问该功能吗？大概吧。他们会付钱吗？也许吧，但是被许可方也可以免费提供。
