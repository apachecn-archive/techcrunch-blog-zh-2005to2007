# 平儿现在承诺自由

[![](img/7de64e193c58e2ef26749cd70833fdf0.png)](https://web.archive.org/web/20211024203056/http://pinger.com/) 语音消息服务 [Pinger](https://web.archive.org/web/20211024203056/http://pinger.com/) 不再要求接收者必须是 Pinger 会员才能在他们的手机上收听消息。相反，在收到来自 Pinger 用户的消息时，收件人将收到一条短信，提示他们点击并拨打一个本地号码来收听消息。以前非成员通过电子邮件发送 Pinger 消息。但是，发送消息和管理联系人列表仍将保留给 Pinger 成员。

Pinger 的正常服务是作为文本信息的替代，让用户可以在移动中互相发送语音信息，而无需接听对方的电话、等待或打字困难。用户所要做的就是拨入他们的 Pinger 号码，说出接收者的名字(对于群组消息),并在他们的 Pinger 邮箱中留下语音消息。然后，Pinger 成员通过文本收到消息通知，提示他们点击并收听消息，他们可以立即通过另一条 Pinger 消息进行回复。

据传闻，Pinger 总共得到了 1100 万美元的支持，其中包括来自 Kleiner Perkins cau field Byers 和 DAG Ventures 的融资。
