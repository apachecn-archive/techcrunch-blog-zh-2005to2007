# nodus psy clone PSP Dock 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/15/nodus-psyclone-psp-dock/>

Nodus Psyclone dock 是为 PSP 设计的，实际上它看起来就像是你不会羞于在家里展示的东西。有一个集成的 T-Amp，当你看 UMD、听音乐或看记忆棒上的电影时，它可以改善音质。

因为 PSP 被放置在扬声器里面的方式，当它被连接的时候，你几乎没有办法玩游戏。当然，除非你有一双婴儿般的手，或者你是《X 档案》第一季里那个在做伸展运动的人。
