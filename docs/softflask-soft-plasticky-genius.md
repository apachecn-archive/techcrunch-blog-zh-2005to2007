# Softflask:柔软的塑料天才

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/07/softflask-soft-plasticky-genius/>

150 年后，我们这个时代将不会被人们认为是转向绿色和可再生资源的时代。人们不会记得互联网第一次变得无处不在、无孔不入的时候。人们不会记得我们让放荡、刁蛮的女继承人出狱的那一天，因为他们违反了让其他人入狱的法律。不，读者，我们的时代将被铭记为你可以把酒偷偷放进任何你想放的东西里的时代。

这是软瓶。一个塑料袋——本质上是一个大牙膏管——装上你最喜欢的烈酒，然后藏在任何你能藏的地方。因为它柔软而有弹性，你可以把它藏在普通瓶子会造成难看凸起的地方。就像在你的裤子里。

这种创新是一代人的特征，我想第一个说我是这一代人中值得骄傲的一员。
