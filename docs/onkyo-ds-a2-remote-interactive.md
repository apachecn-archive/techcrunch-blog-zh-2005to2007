# 安桥 DS-A2 远程互动

![](img/ec020f5ce4fc24d21cd271080571a293.png)

安桥今天发布了 DS-A2 iPod dock，这是 DS-A1 的升级版。新的 dock 可以在电视上显示曲目列表和视频。它包括一个专用遥控器，可以与任何 A/V 接收器配对。如果用户将该装置连接到安桥接收器，则可以通过接收器遥控器对其进行控制。当然，它可以给对接的 iPods 充电。
