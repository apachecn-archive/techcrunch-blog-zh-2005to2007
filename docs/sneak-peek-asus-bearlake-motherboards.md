# 先睹为快:华硕 Bearlake 主板

英特尔的最新芯片组，代号 Bearlake，距离实际发布还有几周时间，但我们已经有了一些规格，如果你在不久的将来考虑一款新的台式机，你可能会觉得很方便。P35 芯片组构建在 65 纳米工艺节点之一上，无疑将在需要 1，333MHz 前端总线的 Core 2 微处理器上运行。Bearlake 芯片组是理想的，因为它们运行的功率更低，不会产生大量的热量，而且生产成本更低。它还支持 DDR3 内存，同时仍然支持 DDR2。

新的 P35 主板将支持多个 PCI Express x16 显卡插槽。主插槽将具有真正的 x16 连接，而第二个 PEG 插槽将具有 x1 或 x4 连接，这都取决于您选择运行的卡的类型。此外，ICH9R Southbridge 还具有额外的 6 个 PCI Express 连接通道(用于 x1 扩展槽或外围设备连接)，支持 12 个 USB 端口和 6 个 SATA 端口(带 RAID)，以及 ICH8R 的所有当前功能。没有关于价格或正式上市日期的消息，但我们会随时通知您。
