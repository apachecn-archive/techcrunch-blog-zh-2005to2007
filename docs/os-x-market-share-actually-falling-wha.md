# OS X 市场份额实际下降。什么？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/19/os-x-market-share-actually-falling-wha/>

这很有趣。一项市场研究表明，从 2006 年 4 月到 2006 年 8 月，Mac OS 的市场份额实际上从 4.33%下降到了 3.71%。从 2005 年 9 月到 2006 年 8 月，其份额从 3.74%上升到 3.71%。这意味着它实际上在 12 月份达到了 4.35%的峰值，但随着时间的推移逐渐下降。

同样有趣的是，在同一年中，Windows XP 实际上从 76.36%上升到 84.18%。这可以解释为企业从 Windows 2000、Windows 98 和 Windows ME 迁移到较新的 Windows XP 机器，而不是其他竞争者市场份额的损失。

那么这意味着什么呢？这意味着 MAC 电脑并没有像大家之前认为的那样取得巨大进步，而且它们肯定没有达到人们在讨论苹果份额时引用的 5%的数字。
