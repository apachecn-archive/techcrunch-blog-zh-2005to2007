# 长尾巴越来越胖了

本周，三个与数字相关的独立新闻故事吸引了我的注意。iLike 宣布它现在已经拥有 600 万注册用户，并且每天增加 30 万新用户。苹果的 Windows 版 Safari 浏览器目前已有 100 万次下载。最后是时空，我们在 6 月 5 日评测的 3D 浏览器突破了 100，000 的下载量大关。

这三者可能看起来没有明显的联系，但是它们都有一个共同点:庞大的用户数量。

不久前，100，000 名用户对于一个 Web 2.0 相关的企业来说还是很大的。如今，像 SpaceTime 这样的小型初创公司可以在两周内获得这些数字。三年前，600 万用户似乎是一个不可能实现的梦想，然而，iLike 加入了拥有 100 万或更多用户的 Web 2.0 网站的长长的、不断增长的名单。Web 2.0 产品正在提高其对更广泛受众的吸引力，这反过来推动了整个市场的增长:长尾越来越肥。

尽管 Web 2.0 市场的这种增长使其更难脱颖而出，但边际成本和投资回报潜力现在已经有所改善。以时空浏览器为例。很多人会立即质疑是否需要替代浏览器，但这并不是一个全有或全无的命题。由于搜索交易，SpaceTime 的每个用户都为公司带来了投资回报。一个普通的时空用户可能会通过点击谷歌广告或浏览易贝网站每月向公司返还 5 美元；10 万用户每月 50 万美元。这个数字可能会更低或更高，但它仍然是一个回报。Safari 将在苹果的类似模式下运行。寻找吸引力的需求在整个市场中所占的比例实际上已经下降。相反，创建可持续业务的门槛并没有随着潜在用户数量的增加而提高，如今初创公司在整个市场中所占的比例较小。

从开发者或初创公司的角度来看，越来越长的长尾应该被看作是什么:一个为聪明的初创公司提供了更多机会的市场。一个更大的市场使得今天和明天比昨天更适合建立 Web 2.0 业务。一条更宽的长尾意味着，作为一个整体，将会有越来越多的成功故事和可持续发展的创业公司，这是一个双赢的局面。
