# 威瑞森统一无线、有线、DSL 计费

。

注册威瑞森本地和长途电话的威瑞森无线客户将可以从他们的手机或家庭电话上无限制地拨打他们的电话和其他网络上的客户的电话。

这对威瑞森用户来说是一种激励，让他们将自己的通信套餐统一到 VZN 所谓的“一元自由”账单中(他们的帽子，不是我们的)。除了座机和手机通话，套餐还包括 DSL。

总的来说，这是电话公司与有线电视公司正面竞争的唯一真正方式，由于这些套餐本身就能节省大量成本，因此在很大程度上，它们对消费者来说是双赢的。我们唯一的保留意见是，它确实剥夺了消费者的一些选择，这是我们非常反对的。
