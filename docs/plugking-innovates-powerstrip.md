# 插拔创新电源板

![](img/a6dc16ba163afa740fe1e8a17a3f9bff.png)

这里有一个好主意:一个带有可互换插座的电源板，允许用户以他或她想要的任何顺序或方式排列所需的插头。

不过，它不一定只是一个无聊的电源板。有 iPods 和门铃的选择，还有三个插脚和几乎任何你能想象到的东西。它还没有投入生产，但我相信很快就会有人将这种插件推向市场。
