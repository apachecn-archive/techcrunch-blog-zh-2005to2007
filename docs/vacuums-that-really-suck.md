# 真的很烂的吸尘器！

如果你看过戴森吸尘器的广告，你可能会怀疑它们是否真的像广告宣传的那样好用。事实上他们有。不像那些蹩脚的 Hoovers，看起来像我的祖父母会认为看起来过时，戴森是你能得到的最时髦的人。现在，洁癖者(像我一样)有了新的庆祝理由。该公司正在推出一种新的滤罐装置——不仅仅是一个滤罐，而是第一个具有恒定吸力的滤罐。DC21 将于本月晚些时候上市，价格约为 500 美元，采用了相同的 Root Cyclone 技术、终身 HEPA 过滤、可清洗过滤器和脚踏刷子控制！对于那些只在绝对必要的时候吸尘的人来说——比如当你在地板上旋转芯片的时候——你也很幸运。DC18(470 美元)是该公司的第一款超薄真空吸尘器，重 15.7 磅，但仍然提供相同的超级吸力！
