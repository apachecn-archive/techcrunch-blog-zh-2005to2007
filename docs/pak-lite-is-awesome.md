# Pak-Lite 太棒了

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/15/pak-lite-is-awesome/>

我喜欢极简主义的产品。它们有种令人耳目一新的美味。随着一切变得超级复杂，拥有一些保持功能简单的东西是很好的。

以 Pak-Lite 为例。它有两个超高亮度的白色 LED 灯泡，只需在 9 伏电池上按一下即可。可能也踢烂了大多数巨型闪光灯。不过，我打赌它在水里不太开心。
