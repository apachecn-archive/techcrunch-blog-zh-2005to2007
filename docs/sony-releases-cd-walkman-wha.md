# 索尼发布 CD 随身听-什么？

为发明时间机器的索尼鼓掌怎么样！据我们所知，他们回到了 1997 年，把一群索尼工程师带到了 2006 年，并给了他们所有的工作。这是索尼发布新 CD 随身听的唯一合理解释。

撇开愚蠢不谈，这款随身听有 G 保护系统，支持 MP3 CDs，而且是圆形的。如果 *olde* 的 CD 播放器有 40 秒的防震保护，这些东西应该有 40 **小时**的防震保护。

现在让我们希望他们把他们的新时间机器用在好的方面，而不是坏的方面，把我们带到一个蓝光不再糟糕的时代。
