# 面向苹果电脑的 Skype 视频今天发布

 [](https://web.archive.org/web/20220929215719/http://skype.com/) Skype 正在[发布](https://web.archive.org/web/20220929215719/http://about.skype.com/2006/09/skype_introduces_video_calling.html)用于 MAC 电脑的 Skype 视频(在[这个](https://web.archive.org/web/20220929215719/http://www.skype.com/download/skype/macosx/)下载页面寻找 2.0 版本)。自去年 12 月起，Windows 版 Skype 视频就已推出。

我们还没有拿到这个软件，但是它不需要像 7 月测试版那样下载两次，并且有希望有一定程度的稳定性。

现在，如果他们能给 Skype 添加一个非常简单的通话记录功能，我们的播客生活将会变得更加愉快…
