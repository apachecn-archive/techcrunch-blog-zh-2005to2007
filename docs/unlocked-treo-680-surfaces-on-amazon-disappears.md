# 亚马逊上解锁的 Treo 680 Surfaces 消失

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/15/unlocked-treo-680-surfaces-on-amazon-disappears/>

 Palm Treo 680 可在亚马逊上预购，价格为 413 美元，发货时间为 1-2 周。然而，在我们完成这篇文章之前，该商品在网站上变成了“不可用”，这很能说明问题。

值得注意的是，亚马逊为你准备的预购版本是解锁的 GSM 版本，而不是几乎官方的 Cingular 版本，这意味着每台 413 美元很可能是批量价格。这给了我们一些线索，让我们知道 Cingy 将能够利用补贴做什么，并使真正便宜(甚至免费)的 Treo 的想法变得更加现实。
