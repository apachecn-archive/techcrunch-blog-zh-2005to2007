# 探索生物圈水族馆| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/23/discovery-biosphere-terrarium/>

我要大胆地说，这个发现生物圈太棒了。除了看起来很酷之外，这对孩子们来说是一次有效的教育经历。

它允许孩子们控制球体内部的环境。该系统有控制温度和湿度的选项，甚至可以模拟下雨。有点像蚂蚁农场，但可能更整洁。

现在售价 40 美元。
