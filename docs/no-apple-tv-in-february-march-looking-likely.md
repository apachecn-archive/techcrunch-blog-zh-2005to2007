# 苹果电视可能在 2 月和 3 月上市

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/26/no-apple-tv-in-february-march-looking-likely/>

在等你的苹果电视？是的，我也是。Apple store 网站已经将发货日期从 2 月下旬改为 3 月中旬，有些令人失望。路透社(Reuters)证实了苹果发言人林恩·福克斯(Lynn Fox)的日期变更，他表示，“苹果电视的包装时间比我们预计的要多几周，我们现在预计 3 月中旬开始发货。”正好赶上你真正的生日。

所以你有它。不过，期待有一天在商店里看到显示器，因为苹果应该急于让尽可能多的人知道这款设备。
