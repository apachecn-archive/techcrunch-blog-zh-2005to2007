# 诺基亚 5200 和 5300 图像

Mobile Scraper 设法找到了几张即将上市的诺基亚 5200 和 5300 手机外壳的照片。正如你所看到的图像留下了很多需要，但他们确实说明了一点什么，我们可以期待从这些即将到来的模式。

从外壳来看，它们都应该配备 130 万像素的摄像头，并通过专用的外部按钮支持 MP3 播放。这些手机可能会有标准的 62k 彩色显示屏。没有关于价格或可用性的消息。
