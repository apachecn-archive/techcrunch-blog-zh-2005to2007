# Octopart:小配件的搜索引擎

 [](https://web.archive.org/web/20220929200725/http://www.octopart.com/) 今天， [Y Combinator](https://web.archive.org/web/20220929200725/http://www.beta.techcrunch.com/2007/02/21/y-combinator-taking-apps-have-idea-will-travel/) 支持 [Octopart](https://web.archive.org/web/20220929200725/http://octopart.com/) 推出了一个电子零件的垂直搜索引擎，非常适合电气工程师和业余爱好者。该引擎通过聚合和标准化来自 Allied Electronics、Digi-Key、Mouser 和 Newark InOne 的产品数据，帮助电子产品买家找到最佳交易。这个网站是由两个物理学研究生创办的，他们在为实验室购买用品时，不断面临艰难的价格比较，感到沮丧。他们最近退出了 Octopart 和其他垂直行业的开发。

搜索支持高级搜索功能，如通配符、短语匹配和布尔运算符。结果通过不同的数量和可用性进行价格比较，并附有关于零件规格的 flashpaper 和 pdf 报告。在摆弄了发动机之后，我对[插入式继电器开关](https://web.archive.org/web/20220929200725/http://octopart.com/search?q=relay+switch)的价格和规格的了解比我想象的要多。两位创始人 Sam 和 Andres 希望将该引擎集成到业余爱好者网站中，从而有可能在你最新的深夜车库项目中找到所有零件的最便宜交易。
