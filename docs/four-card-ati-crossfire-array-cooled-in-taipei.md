# 台北四卡 ATI 交叉火力阵列液冷

本周在台北国际电脑展上，许多[参考设计](https://web.archive.org/web/20201201212106/http://crunchgear.com/2007/06/05/via-nanobook-umd-big-pda-or-tiny-laptop/)亮相。除此之外，还有 [CoolIT Systems](https://web.archive.org/web/20201201212106/http://crunchgear.com/?s=CoolIT) 为 AMD 的 ATI Crossfire(不，不是那个 [Crossfire](https://web.archive.org/web/20201201212106/http://crunchgear.com/2007/06/05/video-dont-get-caught-up-in-the-crossfire/) )配置的工厂密封免维护液冷系统，用于新的高性能镭龙 HD 2900 XT 显卡。

该组件使用“闭环液冷设计中的微通道技术”，将 [ATI](https://web.archive.org/web/20201201212106/https://crunchbase.com/organization/ati) 镭龙 HD 2900 XT 显卡缩小到单个 PCIe 插槽大小。随着更新的主板设计出现了四个 x16 PCIe 插槽，您将能够运行四个卡，而不用担心热量或风扇噪音。

冷却单元本身滑入两个标准的 5.25 英寸驱动器托架，这意味着任何人只要有一把十字螺丝刀和两个开口托架就可以安装这个东西。如果你想构建功能极其强大的游戏电脑，你会想考虑一下 CoolIT 的解决方案。点击跳转观看该公司使用参考设计超频新 AMD GPUs 的视频。
