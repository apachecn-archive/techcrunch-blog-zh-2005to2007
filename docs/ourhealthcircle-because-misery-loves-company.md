# 我们的健康圈:因为痛苦喜欢陪伴

[![ourhealthcirclelogo.png](img/2e2469efb31e9f197dcfbe9e8b6bb788.png)](https://web.archive.org/web/20201031104502/http://ourhealthcircle.com/) 去年 11 月，我们报道了[每日力量](https://web.archive.org/web/20201031104502/http://www.beta.techcrunch.com/2006/11/07/daily-strength-online-support-groups-and-aggregate-treatment-info/)的推出，这是一个为你的病痛提供帮助的社交网络和支持团体。现在，另一个网站 [OurHealthCircle](https://web.archive.org/web/20201031104502/http://ourhealthcircle.com/) ，为受压迫的观众起了一个乐观的名字，开始让谈论棘手的问题变得稍微容易一些。

该网站与 DailyStrength 非常相似，只是背景故事不同。用户创建匿名或个性化的个人资料，加入超过 750 个关于健康问题的团体，并开始讨论。该网站的灵感来自创业与技术中心的三名加州大学伯克利分校的学生，并得到了阿伯实验室的支持。

该网站目前只是作为一个预览开放，邀请是根据具体情况发出的。测试版将在迈克尔·摩尔的新电影《精神病人》上映的那周推出，大概是为了借这部电影给医疗保健系统带来的关注浪潮。

该网站是特殊需求利基网站趋势的一部分，如[处方 4 爱](https://web.archive.org/web/20201031104502/http://www.beta.techcrunch.com/2006/07/17/prescription4love-fills-a-real-niche/)。
