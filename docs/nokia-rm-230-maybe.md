# 诺基亚 RM-230:也许吧

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/05/nokia-rm-230-maybe/>

3GSM 会议即将召开，诺基亚最近向美国联邦通信委员会提交了一款可旋转音乐手机。我们不太确定这款手机将属于哪个系列，但它似乎将是 3250 的继任者，但我们要到 3 月 16 日保密要求到期时才能确定，或者在 3GSM。

根据 Phone Arena 的说法，新的诺基亚将是支持 HSDPA 的四频 GSM 和三频 UMTS。提交的唯一一张照片显示底部 1/4 会旋转，手机背面有一个播放/暂停/拍照按钮。该报告还指出，它将有蓝牙，GRPS 和 EGRPS。我们很快就会看到，希望越快越好。
