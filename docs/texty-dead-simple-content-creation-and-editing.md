# 文本:非常简单的内容创建和编辑

[](https://web.archive.org/web/20211026142119/http://www.texty.com/)[【Texty】](https://web.archive.org/web/20211026142119/http://www.texty.com/)是一个非常简单却有用的新互联网服务，你可以用它来快速创建和编辑网页上的内容，而无需任何 HTML 或编程技能。

进入网站，在所见即所得编辑器中输入文本，格式化并添加图片。点击一个按钮，获得一个嵌入代码。您的文本将出现在您添加代码的任何网站上。如果你想做些更改，请返回到 Texty 进行编辑。这些改变将流向你嵌入的任何网站。您还可以为一段文本添加评论功能，并创建 RSS 提要。

在网上已经有很多很棒很容易使用的内容管理系统。博客软件只是一个例子。但是，如果有人在博客之外的网页上工作，并想添加一些文本和图形，这是一个很好的解决方案。参见我们对拥有[类似工具](https://web.archive.org/web/20211026142119/http://www.beta.techcrunch.com/2007/08/08/from-garage-to-cubicle-js-kit-closes-12-million-seed-round/)的 [JS-Kit](https://web.archive.org/web/20211026142119/http://crunchbase.com/company/jskit/) 的报道。我很惊讶有这么多的人在寻找这样的东西。
