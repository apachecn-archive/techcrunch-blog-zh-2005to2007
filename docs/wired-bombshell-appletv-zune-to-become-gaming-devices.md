# 有线重磅炸弹:AppleTV、Zune 将成为游戏设备

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/08/wired-bombshell-appletv-zune-to-become-gaming-devices/>

我们绝不会开始猖獗的谣言和猜测，但有时我们会注意到一些漏网之鱼，我们无法抗拒。Xbox Live 的前总经理格雷格·卡内萨(Greg Canessa)也是苹果公司专门从事游戏的前员工，他在接受《连线》采访时悄悄透露了一个关于他在 PopCap games 的新职位的消息。

当被问及他将为 PopCap 做些什么时，负责 Bejeweled 等游戏的游戏设计者和发行人说了一些他可能不应该说的话。

部分:

> [我将]从 PopCap 的工作室中取出稳定的特许经营和游戏，并进行调整，为不同的平台定制它——添加多人游戏、新的游戏模式、高清，为 Zune、iPod、AppleTV、任天堂 DS、PSP 定制用户界面和显示。

等一下。Zune 和 AppleTV？是的，那人是这么说的。我们不确定这是官方认可的游戏还是类似 Tecmo Bowl 的东西，但 PopCap 的东西已经可以从 Xbox Arcade 和 iTunes 上获得，所以跳转到 Zune 和 AppleTV 并不是很好。

所以你有它，不是一个而是两个谣言在一句话中供猜测和讨论。我们向你致敬，卡内萨先生，你打开了这个难题。亲爱的读者，你怎么看？这是废话还是可能？就个人而言，我们认为这是有可能的。
