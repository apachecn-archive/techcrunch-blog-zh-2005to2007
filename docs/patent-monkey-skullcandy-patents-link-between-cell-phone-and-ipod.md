# 专利猴:Skullcandy 专利链接手机和 iPod

![](img/8659261ff6da593e2b6a3dc07f898de6.png)

[Skullcandy](https://web.archive.org/web/20230130100805/http://www.skullcandy.com/index.php?urlLocation=home) 一直在创造一些有趣的产品，比如他们的耳机、MP3 手表和[背包](https://web.archive.org/web/20230130100805/http://crunchgear.com/2006/12/01/camelbak-and-skullcandy-team-up-for-some-backpack-convergence/)。

他们刚刚获得了第一项个人便携式音乐播放器和手机集成专利[(美国专利 7，187，948)](https://web.archive.org/web/20230130100805/http://www.patentmonkey.com/PM/PatentID/7187948.aspx) 。这是一款[智能设备](https://web.archive.org/web/20230130100805/http://www.skullcandy.com/linkDetail.php?urlLocation=link)，它可以将你的手机和音乐播放器连接起来，当你有电话时，它可以覆盖音乐——它被恰当地命名为“链接”

Skullcandy 最初在 2002 年提交了专利申请，他们的地址是犹他州帕克城，难怪他们的设备是针对雪地人群的。他们甚至有将连接装置集成到头盔中的头盔。

Skullcandy 并没有从他们的[专利申请](https://web.archive.org/web/20230130100805/http://appft1.uspto.gov/netacgi/nph-Parser?Sect1=PTO2&Sect2=HITOFF&p=1&u=%2Fnetahtml%2FPTO%2Fsearch-bool.html&r=1&f=G&l=50&co1=AND&d=PG01&s1=20040198436.PGNR.&OS=DN/20040198436&RS=DN/20040198436)中得到他们想要的一切。具体来说，他们目前的有线版本的链接不在此专利范围内，只有无线版本。考虑到他们有好的律师，他们很可能接受了他们现在可以得到的权利主张，并且仍然在寻求更广泛的权利主张。

以下是索赔的详细情况:

> 1.一种用于无线集成双向通信设备和音频传送设备的便携式无线装置，其中所述便携式无线装置的大小和形状被配置为在使用所述便携式无线装置的过程中可以被便携式地携带在用户的手中或一件衣服中，所述便携式无线装置包括:用于从基本上可任意选择的音频传送设备无线接收第一音频信号的装置；用于从基本上可任意选择的双向通信设备无线接收第二音频信号的装置；独立于音频和双向通信设备的耦合设备，其中第二音频信号的特征在于阈值，第二音频信号相对于第一音频信号具有优先权，并且第二音频信号在达到阈值时中断第一音频信号，该耦合设备包括用于接收所述第一和第二音频信号的所述装置， 所述耦合装置还包括:用于将包括第一或第二音频信号中的至少一个的第三音频信号传输到声学设备的装置，该声学设备适于将第三声波转换到对应于用户听觉范围的音频范围。

这一切意味着什么？对于那些买不起或不想要 iPhone 的人来说，这是一种迂回的方式([苹果](https://web.archive.org/web/20230130100805/http://www.patentmonkey.com/PM/IMTBlog/tabid/63/EntryID/36/Default.aspx)把你的手放在你的耳朵上)——该声明教导了第三种设备，它连接手机和便携式音乐播放器，并决定这两种设备的优先级。

因此，链接的当前版本和任何未来版本都是富人的 iPhone，或者是一个不错的补充，如果你已经有了 iPod/Zune 和手机，哦，等等，那是我们大多数人。
