# SuTree:用户聚合教学视频

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2007/05/03/sutree-user-aggregated-instructional-videos/http://www.techcrunch.com/2007/05/03/sutree-user-aggregated-instructional-videos/>

总部位于以色列的互联网公司 E-learning Knowledge Solutions 最近推出了[SuTree.com](https://web.archive.org/web/20081006235917/http://www.sutree.com/content.asp?i=1)，这是一项视频聚合服务，用户可以从网上添加教学视频并对其进行分类，提供一个视频目录，这些视频通常会被 YouTube 和 MetaCafe 等网站上的竞争内容淹没。

该服务与 [Scouta](https://web.archive.org/web/20081006235917/http://www.techcrunch.com/2007/03/23/scouta-media-recommendations-soon-in-itunes/) 类似，因为这两种服务都鼓励用户自己添加内容，对其进行标记或分类，评论并添加到一个更大的目录中供他人共享，而不是主要由机器聚合内容，尽管 SuTree 目前没有任何自动建议视频的能力。

该网站有超过 5000 个视频，涉及广泛的兴趣领域，从儿童到宠物，电子产品，食品和商业。我对来源材料的多样性印象特别深刻，它不仅仅是许多聚合网站上流行的 YouTube 视频的通常集合，而是来自更广泛的领域，包括来自每个感兴趣领域的专业内容提供商的视频。

从 it Web 2.0 证书的角度来看，很容易将网站打包，导航是非常老派的，不会跳出来用 Ajax 和 DHTML 的优点来打击偶然的访问者，但对于初学者、母亲、有爱好或寻求学习新事物的目标受众来说，它提供了一个很好的全面的包，利用用户生成的输入向许多人传递一些伟大的知识。
