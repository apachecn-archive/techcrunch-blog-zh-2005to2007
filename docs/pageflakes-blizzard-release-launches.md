# Pageflakes 暴雪发布 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/07/19/pageflakes-blizzard-release-launches/>

 [](https://web.archive.org/web/20221129033432/https://beta.techcrunch.com/wp-content/uploads/2007/07/pf2b1.png) 定制主页初创公司 [Pageflakes](https://web.archive.org/web/20221129033432/http://www.pageflakes.com/) 今天早上在它所谓的“暴雪”发布下推出了一系列新功能。

在所有的新功能中，有两个值得强调的是社交网络和页面上可定制的主题。

直到今天，Pageflakes 用户可以创建他们自己使用的页面，和/或制作称为 Pagecasts 的公共页面。内容过去和现在都完全取决于用户。然而现在，每个用户也可以获得一个个人资料页面，并可以将其他 Pageflakes 用户添加为好友。实际上，Pageflakes 现在是一个社交网络，用户可以基于共同的兴趣联系起来。查看上面我的个人资料页面截图(点击查看大图)。有共同兴趣的用户显示在右下方。

Pageflakes 还发布了“主题化”的公共页面，并与许多知名媒体公司合作创建自己的页面播放——今日美国、滚石、CNN、华盛顿邮报、新闻周刊互动、今夜娱乐、内幕、Slate、AOL、Die Welt、Bild 等。这与 Netvibes 的 Universe 产品相当，尽管 Pageflakes 是实时的，任何人都可以创建主题公共页面(Netvibes 仍然需要合作伙伴关系，你还不能只是创建一个)。迄今为止，用户已经创建了超过 120，000 个公共页面，现在这些页面可以拥有自定义主题。TechCrunch 公共 Pageflakes 页面[在这里](https://web.archive.org/web/20221129033432/http://www.pageflakes.com/techcrunch)。

Pageflakes 还在继续推出更多的小工具——到目前为止已经有 24 万个了。新用户会喜欢自动定制功能，这种功能可以让他们根据几个问题快速创建定制页面。
