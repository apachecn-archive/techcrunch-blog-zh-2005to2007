# 是的，又一个 Ajax 桌面——page flakes

 [](https://web.archive.org/web/20220924195558/http://www.pageflakes.com/) 我最近[写到](https://web.archive.org/web/20220924195558/http://www.beta.techcrunch.com/2005/12/07/ajax-desktops-wont-stop/)Ajax 桌面空间变得疯狂拥挤。那是在谷歌发布他们的[桌面小工具 API](https://web.archive.org/web/20220924195558/http://www.beta.techcrunch.com/2005/12/14/googles-ajax-desktop-is-now-extensible/) 和 [favoor](https://web.archive.org/web/20220924195558/http://www.favoor.com/) 发布之前。现在我们有了一个新的 ajax 桌面的开发者版本，名为 [Pageflakes](https://web.archive.org/web/20220924195558/http://www.pageflakes.com/) (仅在 IE 中可用)。

Pageflakes 有很多功能——RSS 阅读、电子邮件检查、笔记、搜索等。它还有一个开发者 API，允许第三方创建模块。鉴于微软和谷歌有这个功能，它是一个必须甚至必须进入游戏。

Pageflakes 表示，为他们的网站开发模块比 Microst Live 或谷歌更容易——开发人员可以使用 MS Visual Studio，他们支持 ASP.NET 和 Atlas(模块也可以使用 PJP 或 JSP 构建)。当然，克服 Netvibes(现在是 TechCrunch 每日流量的重要来源)的用户忠诚度以及微软和谷歌的网络力量将会非常困难。
