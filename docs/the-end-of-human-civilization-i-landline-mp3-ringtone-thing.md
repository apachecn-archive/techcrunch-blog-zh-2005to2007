# 人类文明的终结 I:座机 MP3 铃声的事情

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/08/the-end-of-human-civilization-i-landline-mp3-ringtone-thing/>

如果你有 12 美元可以挥霍，并且想保证我再也不和你说话，你可以考虑买一个神奇的 MP3 铃声。当然，铃声并不新鲜，但这个可怕的小设备通过为你的固定电话提供铃声而更进一步。

不，再读一遍。它输入正确。这个 USB 连接的盒子可以存储一首 MP3 铃声。然后连接到你的座机上。当有电话打进来时，它会播放“我的驼峰”，或者凯斯·厄本的任何歌曲，或者任何你糟糕的判断允许你载入的歌曲。如果你认为拥有一个这样的东西是个好主意，我的意思是，那你可能对音乐的品味也很糟糕。我在这里做一个假设，但我有一个很好的感觉，这是正确的目标。

不要给自己或任何人买这个，哪怕是作为插科打诨。即使批发价是 9.59 美元，因为这是对自由和个人正义的威胁。已经警告过你了。
