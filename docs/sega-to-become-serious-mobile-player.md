# SEGA 将成为重要的手机玩家

据 GameDaily 今天报道，美国世嘉公司任命前电子艺界资深人士琳达·卓别林领导该公司在北美的移动游戏扩张。世嘉希望在西方的移动领域“成为一个重要的玩家”，因为该公司已经在日本和韩国拥有庞大的移动游戏业务。去年，世嘉的游戏机软件业务已经转向更加关注西方市场。Chaplin 将担任 Sega Mobile 的副总裁，负责北美、中美和南美的业务，包括之前由日本 Sega 负责的业务管理。这些包括现有和未来移动内容的产品开发、销售和营销。
