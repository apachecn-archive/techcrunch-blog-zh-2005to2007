# 租赁电影来到苹果公司

苹果公司现在意识到音乐市场和电影市场非常不同，他们可能已经有了足够的销售电影的经验。很多人想拥有自己的音乐，多年来一遍又一遍地听，但很少有人想拥有自己的电影。看了一次就继续前进，对大多数人来说似乎已经足够好了。

因此，苹果以 10 美元或以上的价格出售电影的模式可能会留下相当大的一块市场。这也解释了为什么他们现在显然正在与主要电影公司谈判通过 iTunes 租借电影的权利。

这项新服务可能会在今年秋天推出，30 天的租金为 2.99 美元。

这对苹果电视用户来说也是一个好消息，他们将能够下载新电影，并在客厅的电视上观看。我们可以完全放弃有线电视订阅的那一天即将到来；我计划在 2007 年底做这件事。

顺便说一下，这不是 iTunes 上第一次出现关于出租电影的传言。[参见我们一年前关于同一主题的帖子](https://web.archive.org/web/20221203084943/http://www.beta.techcrunch.com/2006/07/18/itunes-video-rental-announcement-forseen/)。在史蒂夫·乔布斯上台宣布之前，这还不是一件确定的事情。

许多其他服务已经支持电影租赁。详情见我们 2006 年末的综述文章。
