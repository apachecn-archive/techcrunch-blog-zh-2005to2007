# Vertu 星座:经济实惠的 Vertu

我们将不得不相信诺基亚关于这些新 Vertu 星座手机的可负担性的话，因为他们还没有公布价格，发布日期也在 10 月份。

有三种款式可供选择，一种是金色和黑色皮革，一种是抛光钢和黑色皮革，第三种是缎面钢和深棕色皮革。好消息是这些手机不会完全没有特色，因为它们配备了 EDGE、GPRS、蓝牙、USB 连接和音乐播放器。他们也在使用诺基亚 S40UI，比老版本 Vertu 的 UI 好得多。

尽管如此，即使他们称之为“负担得起”，它仍然是一个 Vertu，这意味着你的孩子不会很快去私立学校。
