# Windows Media 正在经历痛苦

![](img/b8b1792bb6547ac6b29148904565bab7.png)

据消息人士称，Windows Media 服务器和 FairUse4DRM 之间的问题目前已经削弱了 Windows Media DRM 的功能。如果用户试图从引用直播流的文件中剥离 DRM，FairUse4DRM 实际上会破坏 Windows Media 服务器上的 DRM，并使提要对其他用户无用。据称，技术人员现在正在疯狂地解决这个问题，但有消息称，这个故障仍然是一个谜:

> 我的 DRM 管理控制台已经说了 3 天了:
> 系统警报
> 
> DRM 问题
> 太平洋标准时间上午 10 点已知问题:DRM 系统目前正在调查认证问题。我们希望这个问题能尽快得到解决。

很明显他们行动不够快。三天了，还是没有解决。有人注意到与这个错误相关的任何问题吗？
