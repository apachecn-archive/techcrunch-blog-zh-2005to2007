# 新的 Thinkpads 将于本月发布

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/10/new-thinkpads-due-out-this-month/>

今天，联想发布了他们最新的笔记本电脑，ThinkPad T61p。这款 15.4 英寸宽屏移动工作站被誉为联想性能最高、最环保的笔记本电脑，经过 Linux 认证，具有 802.11n 连接、NVIDIA 的 Quatro FX 芯片组和可选的超宽带无线功能。

> ThinkPad T61p 提高了性能标准，联想的 Ultra Connect II 无线设计有助于提高 WWAN 和 WLAN 802.11n 性能，新的 NVIDIA Quadro FX 57OM 显卡经过 open GL graphics GPU 标准认证，采用英特尔迅驰专业移动计算技术，电池续航时间长，可通过电池扩展进行更多控制。用户甚至可以选择超宽带的下一代无线功能。有了联想的新冷却系统，这款笔记本电脑比以前的型号运行起来安静了 3 分贝，即使在最密集的任务下也降低了 10%。顶级的安全性，笔记本电脑提供可选的端口禁用，32 字节密码保护，集成指纹读取器和全磁盘加密。
> 
> 该工作站经认证可运行 20 多家独立软件供应商的应用程序，还经认证可运行 Novell SUSE Linux Enterprise Desktop 10 操作系统，并由联想的硬件问题帮助中心提供支持。该笔记本电脑还经过认证，可以运行 Red Hat 和 Turbo Linux 操作系统。

电池寿命还没有定论，但预计联想可能会提供一些令人印象深刻的数字。他们声称“与具有电源管理功能的上一代笔记本电脑相比，效率提高了 70%，最大限度地为客户节省了 47%的能源。”因此，如果它使用类似于 T61 电池的标准 4 芯电池，如果 70%的数字是准确的，你应该会看到从 2.5 小时跳到 4.25 小时——而同时使用大约一半的能量来充电。

ThinkPad T61p 的起价为 1814 美元，将从本月开始通过联想的业务合作伙伴部门销售。
