# 专利猴:奥林巴斯现实生活游戏的可穿戴显示器

![](img/cbe8190dcd1b5cfe4a80e92b7c9eb2ab.png)

奥林巴斯一直推出非常热门的概念，并保持强劲的创新势头[，2007 年至今已获得 76 项专利](https://web.archive.org/web/20210226165639/http://www.patentmonkey.com/PM/FolderID/489848ce445441e3b4e8c46a58d88f81.aspx)。从他们在数码相机方面的经验来看，进入数码显示器眼罩会有那么大的难度吗？嗯，他们一直在考虑这件事。

事实上，奥林巴斯并不是唯一涉足这一新兴领域的公司。

奥林巴斯获得了一项非常有趣的眼镜专利，包括:

> 本发明的头戴式显示设备包括:角度检测器，用于检测观察者头部的倾斜角度；显示部分，以将预定图像叠加在外部环境的图像上的方式显示该图像，该显示部分用于对应于由角度检测器检测到的观察者头部的倾斜角度，切换显示该图像的显示状态和不显示该图像的非显示状态。

结合整合图像和头部运动的能力可以建立在 WiiMote 的影响上，但有一个全新的“现实世界中的游戏”旋转。

另一项与数字观看相关的专利是 IBM 的[两项](https://web.archive.org/web/20210226165639/http://www.patentmonkey.com/PM/PatentID/7191338.aspx) [发明](https://web.archive.org/web/20210226165639/http://www.patentmonkey.com/PM/PatentID/6959389.aspx)，涵盖了眼镜和公共监视器设备之间的安全观看，由此监视器和眼镜在解密序列上同步，使得观看的数据不可被其他人读取[(相关专利列表)](https://web.archive.org/web/20210226165639/http://www.patentmonkey.com/PM/FolderID/558a7eb8f51849f8a707c1982f233fc9.aspx)。

虽然这些概念不同于立体视觉，但这两个领域在光学知识上有很大的交叉，这使得它成为了数码相机玩家的主要开发领域，如奥林巴斯和宾得，他们愿意扩展以显示更多的 3D 世界。
