# XM 提供停机信用

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/23/xm-offering-credit-for-downtime/>

![](img/9eae4c9b4158e09858336c96cfa1237e.png)

你可能听说过 XM 关闭了几乎一整天，然后[它又重新上线](https://web.archive.org/web/20141014003936/http://crunchgear.com/2007/05/22/xm-up-and-running-again/)。当你每月付费购买的服务出现问题时，你显然会期待某种形式的信用，这样[用户就不会取消他们的订阅](https://web.archive.org/web/20141014003936/http://crunchgear.com/2007/05/17/rumor-and-innuendo-xm-losing-20k-subscriptions-per-day-in-clash-with-opie-anthony-fans/)。幸运的是，XM 足够聪明，知道这一点，并为订户提供两天的信用。

如果你想获得免费通话时间，你需要拨打 1-800-967-2346 与 XM 联系。尽管订阅费是每月 12.95 美元，但两天的信用额度可能足以给你买一小袋多力多滋。
