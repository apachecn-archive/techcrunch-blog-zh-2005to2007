# 舒尔的直喷耳机

复活节前夕，舒尔推出了业内首款直喷式耳机，给世界各地的耳机爱好者带来了一颗重磅炸弹。由于耳机末端伸出一根小管，实际上直接靠在你的耳膜上，它们有望击败普通老式电话。但是别担心，它只有听起来那么危险的 70%。

(细则规定，在跑步、躺下、功夫格斗或以任何方式移动时，你不应该使用新的[舒尔](https://web.archive.org/web/20201021044932/https://crunchbase.com/organization/shure) AF500，因为它们可能会在你的耳膜上打孔。)
