# Zvents 下周发布

![zvents](img/b3659c2737222673153aa808760e82f5.png)**Company:** [zvents](https://web.archive.org/web/20230216134035/http://www.zvents.com/) **Launched:**

下星期

**Location:**

加利福尼亚州门洛帕克

今天晚上，我有机会在位于门洛帕克沙丘路的[网络服务风险投资集团](https://web.archive.org/web/20230216134035/http://www.nsv.com/)办公室观看了即将推出的 Zvents 服务的演示。

将这个标记为 ajax 展示。Zvents 是一个完整的事件生态系统，拥有搜索、事件创建、日历、共享和博客/网站管理员工具(这里有很多 ajax)来推广事件和日历。

它在许多方面看起来与 [EVDB](https://web.archive.org/web/20230216134035/http://eventful.com/) (现在 [eventful](https://web.archive.org/web/20230216134035/http://eventful.com/ceoletter) )相似，尽管搜索功能显然是更进一步的一代，它肯定是一个非常好看的网站。

zvents 将于下周开放，并将在 Web 2.0 大会上发布。

Peter Caputa [在 PC4Media 上写了这个空间](https://web.archive.org/web/20230216134035/http://worcester.typepad.com/pc4media/2005/09/event_search_ca.html)的详细分析，包括 eventful 和 zvents。
