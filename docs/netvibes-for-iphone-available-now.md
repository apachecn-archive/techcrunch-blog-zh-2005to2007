# 苹果手机的 Netvibes 现在可以使用了

可定制的主页网站 [Netvibes](https://web.archive.org/web/20220703112553/http://www.crunchbase.com/company/netvibes) 今天早些时候在[m.nv1.netvibes.com](https://web.archive.org/web/20220703112553/http://m.nv1.netvibes.com/)为苹果手机发布了该网站的预测试版。这个版本的网站在普通浏览器上看起来不太像，但在苹果手机上肯定可以。

对于苹果手机来说，已经最小化的网络感应被进一步缩减为一列小部件。很少有图形可以加快加载时间。

我使用 Netvibes 作为我最重要的订阅源的快速 RSS 阅读器。它在苹果手机上做得更好，我计划用我每天多次阅读的 15 条信息为一个特别的网络感应页面添加书签。

今晚也要关注脸书的 T7 博客。脸书公关表示，他们将很快宣布该网站的苹果版本。**更新:** [发布了](https://web.archive.org/web/20220703112553/http://www.beta.techcrunch.com/2007/08/15/facebook-iphone-ultrahype/)。
