# VOIP 创业公司 Fring 融资 1200 万美元 B 轮

[![Fring](img/73006a6dabbbb5a418b11d69bf991bec.png)](https://web.archive.org/web/20211017010229/http://www.crunchbase.com/company/fring) 总部位于以色列的 VOIP 初创企业 [Fring](https://web.archive.org/web/20211017010229/http://www.crunchbase.com/company/fring) 已经完成了一笔[估计](https://web.archive.org/web/20211017010229/http://www.globes.co.il/serveen/globes/docview.asp?did=1000242078)1200 万美元的第二轮投资，由美国风投基金 North Bridge Venture Partners 领投。VenFin 和之前的投资者 Pitango、Veritas 和 Yossi Vardi 也参与其中。如果投资估算准确，这将是一家 VOIP 公司的较大一轮融资。

Fring 是一个用于 Windows 和 Symbian 手机的移动应用程序，使用 VOIP 进行廉价/免费的移动通话和即时消息。与 Jajah 不同，但与 Truphone 一样，Fring 通过 Wi-Fi 互联网接入或您的 3G 或 GPRS 互联网数据计划发送电话和聊天。像 Skype 一样，用户拨打标准电话只需支付象征性的费用。然而，拨打其他网络电话是免费的。Fring 连接到标准电话、其他基于 SIP 的 VOIP 客户端和聊天应用程序(Skype、MSN Messenger、ICQ、Google Talk，甚至 Twitter)。

Fring 受到了来自 Crunchgear 和 TechCrunch 的好评，主要是因为它的多聊天功能和免费通话。然而，通话质量是个问题，而且 Fring 使用的 3G 手机可能很贵。你也应该看到我们之前对其他 VOIP 运营商的报道。
