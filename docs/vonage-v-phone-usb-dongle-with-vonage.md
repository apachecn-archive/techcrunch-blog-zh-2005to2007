# Vonage V-Phone:带 Vonage 的 USB 加密狗

如果你是那种喜欢 Vonage 胜过 Skype 的人，而且你也是那种经常出差的人，那么你可能会喜欢 Vonage 的这款 USB 手机。你已经可以订购一段时间了，但现在是从 Vonage 的网站发货。

这不是一部传统的手机，但仍然很酷。V-Phone 是一个 USB 加密狗，很像一个拇指驱动器，包含 Vonage Talk 软件和预装的联系人列表。插上附带的耳机，使用 PC 拨打电话，一切就绪。PC 更多的是作为一个界面使用，这是一个相当不错的想法。

USB 电话有自己的 Vonage 号码，这有点奇怪，但想必你可以将其更改为你自己的号码。不到 40 美元，现在就开始抱怨也不错。
