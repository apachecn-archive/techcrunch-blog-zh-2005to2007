# 新网站“跃入”应用创建领域——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/06/19/new-site-jumps-into-the-application-creation-space/>

 [](https://web.archive.org/web/20210116144823/http://www.longjump.com/) 就在一年多前 [DabbleDB 推出了](https://web.archive.org/web/20210116144823/http://www.beta.techcrunch.com/2006/03/11/dabbledb-online-app-building-for-everyone/)，让人们只需很少的编程技能就能创建快速的数据库驱动应用。从那以后，我们已经看到 Zoho Creator 、 [WyaWorks](https://web.archive.org/web/20210116144823/http://www.beta.techcrunch.com/2006/04/30/wyaworks-app-builder-for-non-coders/) 和 [Coghead](https://web.archive.org/web/20210116144823/http://www.beta.techcrunch.com/2007/04/16/coghead-announces-17000-developers-building-applications-visually/) 推出了他们自己的竞争应用。 [Salesforce AppExchange](https://web.archive.org/web/20210116144823/http://www.beta.techcrunch.com/tag/appexchange/) 可以说也是一个竞争对手，其他一些自建应用也是如此，如[雅虎 Pipes、Teqlo](https://web.archive.org/web/20210116144823/http://www.beta.techcrunch.com/2007/03/02/5-ways-to-mix-rip-and-mash-your-data/) 和[微软 Popfly](https://web.archive.org/web/20210116144823/http://www.beta.techcrunch.com/2007/05/18/microsoft-launches-popfly-mashup-app-creator-built-on-silverlight/) 。

这是一个拥挤的空间，这是有原因的。这些工具让非开发人员可以为企业创建有用的或者昂贵的应用程序，这些应用程序可以直接访问或者通过内部网访问。它们也可以被无限程度地定制，给公司更多的灵活性来使用现成的东西。这些工具中的大多数也允许简单的 web 应用程序构建，从简单的 web 表单到更复杂的交互式应用程序。

今天发布的 [](https://web.archive.org/web/20210116144823/https://beta.techcrunch.com/wp-content/uploads/2007/06/longjumpb.png) [龙跃](https://web.archive.org/web/20210116144823/http://www.longjump.com/)是最新的参赛者。这项新服务是 2003 年成立的已经盈利的业务 [Relationals](https://web.archive.org/web/20210116144823/http://www.relationals.com/) 的副产品。37 岁的创始人 Pankaj Malviya 说，公司近期不需要额外的资金。

LongJump 致力于让人们在没有任何编程技能的情况下创建应用程序——这也是 Coghead 所追求的。在昨天的演示中，我看到 Malviya 非常快速地创建了几个应用程序，包括一个高度定制的企业共享联系人和日历资源。

LongJump 也在以一种真正聪明的方式接近他们的社区——像 [Ning](https://web.archive.org/web/20210116144823/http://www.beta.techcrunch.com/2007/02/26/ning-in-full/) 一样，应用程序可以与其他 LongJump 用户共享、克隆和定制。共享是可选的，所有的业务数据都被剥离出来并替换为示例数据。但是在不久的将来，用户将能够在其他人已经创建的基础上进行构建，从而为 LongJump 提供一个很好的示例应用程序库。

易用性与社区共享功能相结合，是该产品真正周到的补充。我们将在几个月后再次与他们联系，看看应用程序库进展如何。
