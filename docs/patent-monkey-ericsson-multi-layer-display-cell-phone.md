# 专利猴:爱立信多层显示屏手机

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/09/patent-monkey-ericsson-multi-layer-display-cell-phone/>

![](img/138612d761bedc67db380df4b0e9702d.png)

继摩托罗拉的[多层液晶太阳能显示器](https://web.archive.org/web/20160421183239/http://crunchgear.com/2007/04/22/patent-monkey-lcd-solar-cell-smart/)之后，爱立信最近发布的专利又在另一个方向增加了一个功能:[多层显示器](https://web.archive.org/web/20160421183239/http://www.patentmonkey.com/PM/patentid/7205959.aspx)。爱立信想出了许多应用程序，通过将信息分层，使其更适合小小的屏幕。

苹果可能是做这种正义的公司，跳完之后你会看到已经做了多少工作。

爱立信的技术建立在 Asulab 发明的多显示器上，其中一个显示器叠加在另一个显示器上，阻挡了第二个显示器的视觉。

爱立信的概念设想使用这种技术作为一种方式来确保内容或信息被显示给正确的用户，这是一个主要的好处。爱立信的[原始应用](https://web.archive.org/web/20160421183239/http://appft1.uspto.gov/netacgi/nph-Parser?Sect1=PTO2&Sect2=HITOFF&p=1&u=%2Fnetahtml%2FPTO%2Fsearch-bool.html&r=1&f=G&l=50&co1=AND&d=PG01&s1=659109.APN.&OS=APN/659109&RS=APN/659109)除了使用表层作为屏蔽第二个屏幕的手段之外，还寻求了许多更广泛的方法，进一步的设计和描述强调使用表层作为交互介质来输入文本，在地图/图像上查看文本等。正如你将看到的，爱立信的想法有历史。

本发明的一些背景变得更加清楚，尽管在外观上有点混乱。在使用小屏幕进行数据输入时，比如 iPhone 即将上市，需要一个叠加键盘在人体工程学上是有价值的。敲击正确的键似乎是一个问题，在爱立信的设计中讨论过，并被 IBM 的技术所覆盖。

追溯到 1994 年的一份申请，IBM 创造了一个[动态触摸屏](https://web.archive.org/web/20160421183239/http://www.patentmonkey.com/PM/patentid/5565894.aspx)，用户触摸屏可以针对[视差](https://web.archive.org/web/20160421183239/http://en.wikipedia.org/wiki/Parallax)进行调整。具体来说，IBM 的发明者描述道:

> 用户正在使用的应用的初始按钮或一系列按钮被配置成感测用户触摸的位置。这些初始按钮被称为调节按钮。其余按钮(非调节按钮)的按钮感测区域从这些初始触摸位置进行校准。

虽然执行这种概念的第二层或多层显示器可能不是必要的，但在小小的 PDA/手机屏幕上做更多事情的想法可能正在达到一个临界点。
