# 三星和美国陆军联手开发移动 WiMAX

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/26/samsung-and-us-army-team-up-for-mobile-wimax-tomfoolery/>

三星和美国陆军通信电子研发工程中心宣布成立合资企业，在未来几个月测试一些移动 WiMAX 技术。实验将在福特公司进行。指挥、控制、通信、计算机、情报、监视和侦察移动小组(说真的，这是一个很长的@ss 名称，我想我现在有心皮隧道)。

CERDEC 的工程师将测试移动 WiMAX 产品的性能，看看其成本和可访问性是否能在美国军队需要在短时间内传递关键信息时提供帮助。

电信系统部门的三星电子副总裁这样说道:

> “移动 WiMAX 为人们的通信方式带来了各种机遇。我们非常高兴能够向美国陆军提供和介绍移动 WiMAX 技术，他们可能会有各种独特的实施方案。移动 WiMAX 继续在全球扩张，三星将在前进的每一步打开新的机会之门。”
