Nimbuzz Talk推出，连接手机和电脑

mobile runch在宣布了 Nimbuzz Talk 的测试版，这是一款连接手机和PC用户的移动/PC应用。该应用程序支持移动设备、PC以及移动到PC或PC到移动设备之间的语音、在线状态、电话会议和SMS消息传递。它目前可以和MSN、Google Talk以及他们自己的Nimbuzz聊天客户端一起工作。这让我想起了另一个移动/PC版本， Talkster 。

Nimbuzz用户之间的移动到移动电话按本地费率计费，这使得一些廉价甚至免费的国际电话成为可能。Nimbuzz的在线支持允许用户通过从好友列表中选择一个联系人来拨打电话。然后，该服务通过800号码连接用户。MobileCrunch的Oliver Starr表示，手机对手机的质量“非常好”，但是手机对电脑的质量“还有待提高”。

这项新服务还有一个“蜂音”功能，让离线联系人知道你在试图联系他们。在他们的下一个版本中，这将导致Nimbuzz在联系人的电话上醒来。
