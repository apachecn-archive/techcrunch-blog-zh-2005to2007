# Vonage 的新网关显示来电显示、语音邮件| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/03/vonages-new-gateway-displays-caller-id-voicemail/>

我对 Vonage 又爱又恨。我喜欢我不再使用 Vonage，我讨厌成为 Vonage 用户。如果你不喜欢我，你可能会有兴趣了解 Vonage 正在准备一个带有 LCD 屏幕的 VOIP 网关，可以显示来电显示和语音邮件状态通知。

这是为了“激励消费者自豪地展示他们的 Vonage 路由器，而不是把它藏在桌子底下。”如果你不告诉保留代表你要搬到澳大利亚内陆的中部，那里没有互联网接入，它不会帮助你在一个小时内取消你的帐户。我不得不那样做，这是一个真实的故事。

如果你想测试这个设备，Vonage 会给你一个月的免费路由器服务，外加 30 美元的主线话费。您必须在路由器上使用第二个 Vonage 颁发的号码，并且必须在 30 天内归还。

抛开我对 Vonage 的感觉，这似乎是一个很酷的设备，可能会吸引那些对 VOIP 服务持怀疑态度的人考虑更换他们的座机。
