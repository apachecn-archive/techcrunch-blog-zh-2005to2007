# 维基百科从亚马逊获得了一笔数目不详的资金

维基百科的盈利姐妹网站 [](https://web.archive.org/web/20210124101802/http://www.wikia.com/wiki/Wikia) [维基百科](https://web.archive.org/web/20210124101802/http://www.wikia.com/wiki/Wikia)今天收到了第二轮资金，全部来自[亚马逊](https://web.archive.org/web/20210124101802/http://www.beta.techcrunch.com/tag/Amazon/)。目前还不知道亚马逊支付了多少。

就在本周，Wikia [宣布](https://web.archive.org/web/20210124101802/http://www.beta.techcrunch.com/2006/12/04/wikia-to-buy-armchairgm-for-2-million/)他们将以 200 万美元的现金和 Wikia 股票收购在线体育社区 [ArmchairGM](https://web.archive.org/web/20210124101802/http://armchairgm.com/) 。

三月份，我们[报道了](https://web.archive.org/web/20210124101802/http://www.beta.techcrunch.com/2006/03/27/wikia-raises-4-million/)该公司从 Bessemer Venture Partners 和 Omidyar Network 获得了 400 万美元，天使投资人丹·吉尔默、雷德·霍夫曼、伊藤穰一和米切尔·卡普尔也参与了投资。

Wikia 的 Gil Penchina 告诉 GigaOm,“坦白地说，融资和收购是“非常独立的事件”,“只是碰巧在同一时间发生”,但是如果没有亚马逊的资金，收购很难完成。
