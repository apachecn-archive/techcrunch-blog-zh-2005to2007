# Treemo 将为关心多媒体制作的人建造家园

[![](img/b9ee3f6373abc81db17be84b99ef150f.png)](https://web.archive.org/web/20211026164334/http://www.treemo.com/) 用户生成的多媒体的增长似乎不会很快放缓，新公司在这一领域的推出也不会。对于那些有兴趣在一个社区中结合照片、视频、音频和文本的人来说，这个空间的最新入口是今天华盛顿州西雅图市 [Treemo](https://web.archive.org/web/20211026164334/http://www.crunchbase.com/company/treemo) 的推出。收入共享、[知识共享](https://web.archive.org/web/20211026164334/http://creativecommons.org/)许可和硬件回收计划是迄今为止差异化的关键点。对企业在其他网站上的影响力不满的人以及重度移动用户很可能会喜欢 Treemo。

该公司成立前名为 HyperMob，由在移动技术领域拥有丰富经验的高管组成。如果网站积累了足够多的用户，它的大部分将会得到巩固，但现在有一些基本的品质使它脱颖而出。

Treemo 拥有所有标准的内容分享社区功能，但将通过收入分享和对社会负责的转变使自己与众不同。该网站支持桌面和移动上传，Oliver over 在[mobile runch 今天对移动角度](https://web.archive.org/web/20211026164334/http://mobilecrunch.com/2006/09/07/treemo-launching-today-new-mobile-content-network-opens-public-beta/)进行了回顾。移动在 Treemo 中尤为重要，因为该网站提供 xHTML 格式的移动浏览应用程序，并将视频转换为 3GP 移动格式。

用户可以选择是否允许在视频和音频页面上做广告，并从中获得部分收入，或者决定不让广告出现在他们的视频页面上。公司创始人布伦特·布鲁克勒(Brent Brookler)告诉我，一旦达到临界质量，收入分成就会开始，分成比例可能是 50/50。除了广告收入之外，还会有一些额外的服务级别(尚未公布)。


还有拖放文件管理，flash 嵌入和公共或朋友网络权限级别。

尽管 Flickr 的免费账户每月上传上限为 20MB，YouTube 的单个视频文件上限为 100MB，但 Tremeo 的每月上限为 200MB，视频文件上限为 20MB。适合摄影师，非常短的视频适合手机。最大总存储空间为 1 GB。

Treemo 最有可能吸引用户的方式是通过社会责任的角度。它位于太平洋西北部，是自由派极客的温床。人们完全支持知识共享许可。相机可以通过 Treemo 捐赠给各种 501c3 组织，供有抱负的学生摄影师使用，因此可以在纳税时注销。Treemo 提供三个月的优质服务，以换取捐款，并在捐款人的用户名旁边放置一个图标。手机可以通过 Ripmobile 通过 Treemo 捐赠，换取各种电子商店的优惠券。Treemo 还将很快发起一系列针对“有社会意识”的多媒体内容制作者的竞赛。

目前还没有从 Treemo 导出文件的方法，但我敦促该公司的创始人尽可能简化这一过程，他向我保证，至少很快会有办法做到这一点。

随着越来越多来自主流媒体共享社区的用户对公司在这些网站上的影响力越来越不满意，以及随着用户生成的多媒体的激增，我认为一些人可能会喜欢 Treemo 的方法，使他们能够关闭广告或分享收入，将 CC 许可证应用到他们的工作中，并参与到处理电子垃圾这一巨大问题的社区中。
