# 基于网络的媒体播放器可让您将 iTunes 流式传输到任何电脑

[![steampad_logo.gif](img/4339c99394bc56d7dc1c68dbdc1cf752.png) ](https://web.archive.org/web/20220929005302/http://www.streampad.com/) [Streampad](https://web.archive.org/web/20220929005302/http://www.streampad.com/) 是一款基于网络的媒体播放器，用户可以从任何电脑上访问他们的数字音乐库。类似于 [Songbird](https://web.archive.org/web/20220929005302/http://www.beta.techcrunch.com/tag/Songbird/) ，它从互联网上下载你的个人图书馆，但它完全可以在任何浏览器中运行。

Streampad 是一项免费服务。用户下载一个 Java 桌面应用程序，该应用程序存储它在你的电脑上找到的所有 MP3 和 AC3 文件(受保护和不受保护)的元数据，包括来自 iTunes 的元数据。然后，当同一个用户从另一台电脑登录到 Streampad 时，他们可以直接从他们的家用电脑上听音乐。

Streampad 的创始人丹尼尔·坎特(Daniel Kantor)说:“当你请求时，这首歌就会从你自己的电脑中流出。“当您运行 Java 程序时，它会继续运行并像 Web 服务器一样工作，因此请求会发送到您自己的系统。这就像一个点对点系统是如何工作的。当你选择音乐时，所有信息都来自我的服务器 Streampad，但当你播放音乐时，它来自你的电脑。”

因此，用户不得不开着家里的电脑来远程访问他们的图书馆，这并不是每个人都喜欢的。或者拥有 [MP3tunes](https://web.archive.org/web/20220929005302/http://www.mp3tunes.com/) 账户的用户可以将他们的音乐存储在 MP3tunes 上，然后通过 Streampad 登录 MP3tunes，从他们的“储物柜”中播放音乐

这项服务有社交推荐的功能，但不如 last . FM 好。基本上，用户可以看到谁在流媒体音乐，他们在听什么。如果文件是网络上的公共文件，比如说播客，那么点击 URL 将允许我收听我朋友正在收听的内容。如果它不是公开的，但我的图书馆里也有，我可以点击这首歌，Streampad 会为我找到我的副本。如果这两种情况都不是真的，但我仍然想听我朋友正在听的东西，那就由我自己去通过 iTunes 或其他方式找到它。

Streampad 是 Kantor 使用 PHP 和 MySQL 作为数据库创建和资助的。它已经运行了一年，但是坎特在五月份正式推出了他引以为豪的版本。迄今为止大约有 5000 名用户。
