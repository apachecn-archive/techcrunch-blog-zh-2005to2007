# Zune 呢？

RoughlyDrafted 的 Daniel Eran 对 ms 实际上像我们认为的那样是一个强大的玩家的想法进行了手术。他关注了他们业务中可以被视为失败的两个方面——Windows Mobile 和 WMA——某种程度上表明了 Zune 注定要失败，或者更糟，成为廉价品。

> 他们忽略的一个大秘密是，微软从未在消费者硬件业务中赚取过巨额利润，其高管也没有证明过在提供消费者想要的硬件或创意娱乐方面有任何商业头脑，当然，不包括微软令人印象深刻的高度复杂的键盘和鼠标部门。
