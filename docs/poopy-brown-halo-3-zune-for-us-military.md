# Poopy 布朗 Halo 3 Zune 为美国军方

![poopbrown.jpg](img/3eabf7fb95f4dbcd156229890dad8ffc.png)

我想我们都知道微软是怎么处理那些卖不出去的棕色 Zunes 的。他们正在为美国军方制作 Halo 3 军用版，并将于本月在军事零售店以折扣价出售。MS 向华盛顿大都会的 USO 捐赠了 300 台特别版 30GB Zunes，今天出发去执行任务的士兵可以在巴尔的摩华盛顿国际机场的 USO 休息室特别访问 Zune 收听站，以便在出发前定制他们的播放器。很好是的，女士
