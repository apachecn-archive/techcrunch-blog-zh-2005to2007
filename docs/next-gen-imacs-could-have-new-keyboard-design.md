# 下一代 iMacs 可能会有新的键盘设计

![](img/a20c6080a3df7527d35746e68cda2f67.png)

我们都知道新的 iMacs 很可能会在年底前上市，但是你知道新的键盘会和它们一起上市吗？根据“可靠消息来源”和“消息人士”的说法，AppleInsider 的人声称，新设计的键盘将借鉴 Macbook 的“低调”键盘，使键盘的尺寸变得格外薄。

由于新的 iMacs 据传有拉丝金属设计和更薄的外观，这一举动完全有道理。此外，苹果在过去十年里并没有在键盘上做什么。当然，我们看到了一个无线蓝牙模型出现，但除此之外，它一直相对安静。希望这款新键盘比现有版本更容易清洁。
