# 欧宝 Antara 有可伸缩的自行车架

![](img/a64d435310fa1e91c999d6179921d209.png)

这是我一直在用的东西。2007 年，欧宝将推出 Flex-Fix 系统作为其 Antara 和 Corsa 车型的选装配置。该系统是一个可伸缩的两个自行车架，可以滑入和滑出保险杠。

它是完全隐藏的，允许用户避免到处绑那些讨厌的带子。这需要立即成为美国汽车的一个选项。给你的国会议员写信什么的。
