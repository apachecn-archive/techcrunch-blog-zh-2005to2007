OS X市场份额实际下降。什么？TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2006/09/19/os-x-market-share-actually-falling-wha/

这很有趣。一项市场研究表明，从2006年4月到2006年8月，Mac OS的市场份额实际上从4.33%下降到了3.71%。从2005年9月到2006年8月，其份额从3.74%上升到3.71%。这意味着它实际上在12月份达到了4.35%的峰值，但随着时间的推移逐渐下降。

同样有趣的是，在同一年中，Windows XP实际上从76.36%上升到84.18%。这可以解释为企业从Windows 2000、Windows 98和Windows ME迁移到较新的Windows XP机器，而不是其他竞争者市场份额的损失。

那么这意味着什么呢？这意味着MAC电脑并没有像大家之前认为的那样取得巨大进步，而且它们肯定没有达到人们在讨论苹果份额时引用的5%的数字。
