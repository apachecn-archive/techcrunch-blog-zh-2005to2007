# 海绵细胞，一个 Ajax 日历

 [](https://web.archive.org/web/20220926194322/http://www.spongecell.com/) [海绵细胞](https://web.archive.org/web/20220926194322/http://spongecell.com/)是在 Ruby on Rails 平台上构建的新 Ajax 日历。除了现在无处不在的拖放界面，海绵细胞正在增加人工智能方面，将自然语言变成结构化的日历条目，并有一个很好的移动界面。更多信息在[这里](https://web.archive.org/web/20220926194322/http://spongecell.com/info/press_release)。海绵细胞将于 2006 年 2 月 1 日星期三在旧金山举行一个公开的发布会。

Ajax 日历现在已经非常普遍了(据我所知，它们是 Ajax 主页的近亲),而且似乎正在大量繁殖。我的清单现在包括:

[仁古](https://web.archive.org/web/20220926194322/http://www.beta.techcrunch.com/2005/11/25/so-what-exactly-is-renkoo/)、[斯科比](https://web.archive.org/web/20220926194322/http://www.beta.techcrunch.com/2006/01/28/nine-startups-at-e27-summit/)和[津布拉](https://web.archive.org/web/20220926194322/http://www.beta.techcrunch.com/2005/09/12/zimbra-webajax-based-outlook-application/)也触及这个空间。
