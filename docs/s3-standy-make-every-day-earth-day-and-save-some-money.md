# S3·斯坦迪:让每一天都成为地球日，节省一些钱

地球日来了又去，但这并不意味着我们应该回到我们杀害地球的习惯。许多台式机用户明显忽略了电脑的低功耗设置，这些设置在待机状态下仍然允许网络驱动器和/或远程应用程序的功能。也许，这是一种公然的漠视，但是通过一点小小的指导，你可以为地球做一些好事，而且可能还能省下一些钱。

eXoid 在一步一步的演练方面做得很好，但是您可能想知道为什么有人需要费心这样做，对吗？eXoid 的计算估计，根据你的电脑电源设置，你可能需要额外支付 34.56 美元。随着时间的推移，你的电脑越来越慢，越来越耗电。

> 嗯，今天的电脑有双显卡，RAID 0 配置，和多核心处理器可以吃掉很多果汁。高端电源设备现在的额定功率超过 1000 瓦。如果我们只是合理地估计一台计算机空闲时消耗 400 瓦，我们会发现一些惊人的数字。
> 我计算过(每天 24 小时)*一个月 30 天= 720 小时
> 电费通常以千瓦时或“千瓦/小时”为单位。电费可能高达每千瓦/小时 0.12 美元
> 在这种情况下，我们一个月 24/7 全天候使用计算机的总成本如下:
> . 4 千瓦(400 瓦)* 720 小时* $ 0.12 千瓦/小时= 34.56 美元

大多数新电脑已经配置为进入 S3 待机模式，但检查并确认一下也无妨。将计算机置于待机模式，并聆听风扇的声音。如果它们仍在旋转，那么您的电脑没有为 S3 正确配置。让我们开始吧，让那个超级大家伙进入 S3 待机模式…这太有趣了！

> 检查您的 BIOS 设置:确保您的 BIOS 电源管理设置为 S3 待机。不幸的是，如果在安装 windows 时没有设置为 S3，Windows 将不会安装正确的设置来允许计算机以 S3 模式睡眠。然而，仅仅改变 BIOS 中的选项将会覆盖某些计算机上的 windows 设置，并且您可能会进入 S3 睡眠状态。您使用的是 USB 键盘还是鼠标？这可能会拖你的后腿。由于担心由于 USB 设备不兼容而导致计算机进入无法恢复的睡眠状态，微软决定，如果 USB 输入设备存在，它将不允许 S3 待机模式。幸运的是，这很容易通过使用注册表项来绕过。只需打开 regedit 并输入以下注册表条目，如这篇微软知识库文章中所述:
> HKEY _ 本地 _ 机器系统当前控制设置服务 susb
> " usbBIOSx " = DWORD:00000000
> ![](img/d7667c1506ef031f5b4ceffce23da09f.png)
> 输入此注册表项，然后重新启动应该允许您进入 S3 睡眠状态，即使在 Windows XP 中使用 USB 外设。
> 如果您的电脑进入待机模式，但随后立即开始备份，您的 USB 设备可能会给您带来问题，请检查您电脑的 BIOS 设置并禁用(或启用)USB 开机选项，更改这些设置可以让您的电脑正常工作。

到目前为止一切顺利，对吧？很好！现在，您已经将计算机配置为正确进入 S3 待机模式，让我们来了解一下，如果您将计算机用作文件服务器、远程访问点或 VNC 主机，当您需要计算机始终处于开启状态时，该如何使用它。eXoid 指出，最好的方法是在你的本地网络上设置一个静态 IP，下面是如何做的。

> 1.导航到控制面板:双击网络连接
> 2。右键单击您的局域网:选择属性
> 3。选择
> ![](img/5ec525dca113f4bd18687060f81891c7.png)
> 4【TCP/IP】。单击属性。
> 5。手动设置您的 ip 地址:(IP 地址通常可以是您想要的任何地址。在一定的限制内，子网掩码、默认网关和 DNS 设置应该与您的特定路由器相对应。查看您的路由器手册了解更多信息)
> ![](img/f822ceae2660af0cd309520508560ef8.png)
> 6。完成更改后，单击“确定”,让网络适配器重新配置并重新连接到网络。
> 7。通过打开浏览器或检查网络可访问性来测试您的网络，看您是否仍有连接。

伙计们，我们现在到了最后关头。以下是当静态 IP 上有流量时唤醒 PC 的方法。

> 1.启动到您的 BIOS，确保所有 WOL 和网络唤醒选项已启用。请参考制造商的主板手册。
> 2。重新启动进入 Windows:导航到控制面板:双击网络连接
> 3。右键单击您的局域网:选择属性
> 4。在第一个框中，它应该显示您的网络适配器。点击‘配置’
> 5。导航到“高级”选项卡:选择“唤醒功能”
> ![](img/4afc08d6c9d4d9f884226ff767f38f9c.png)
> 6。每个网络设备稍有不同，您想要启用所有功能，请选择“两者”或“魔术包和唤醒模式”
> 7。单击“电源管理”选项卡:选中“允许此设备使计算机脱离待机状态”复选框
> ![](img/4006ace0e50b2dd9aa5e48a4d093cbba.png)
> 
> 一旦您选择了这些选项并重新启动，您现在应该已经为您的计算机设置了一个非常强大的配置。现在，您可以允许您的计算机进入待机模式，但是只要您的网络适配器检测到您的特定本地 ip(在我的例子中是:192.168.0.200)上有任何活动，它就会迅速从 S3 待机状态恢复并执行所请求的功能。一旦网络未被使用，您的计算机将在您在“电源选项”设置中设置的指定时间后返回待机状态。
