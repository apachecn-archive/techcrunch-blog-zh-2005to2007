# TomTom Go 720 拥有用户可编辑的地图

[![tomtomgo720.jpg](img/59fba20f299ebe96eac2390fcf73bf60.png)](https://web.archive.org/web/20130628182601/http://old.crunchgear.com/wp-content/uploads/tomtomgo720.jpg "tomtomgo720.jpg")

TomTom 在木厂还有一个便携式 GPS 设备。这是 TomTom Go 720，专为美国和加拿大市场制造。这是一款典型的 GPS 设备，具有日益变得更加标准的所有功能，包括 4.3 英寸触摸屏和基本的多媒体播放选项。但 Go 720 确实做了一件有点不同的事情:它允许用户合作改善所包含的地图。

如果有一个城市街道特征没有反映在内置地图中，比如说，一座建筑物刚刚被拆除，或者一条道路刚刚被封锁了一段时间，您会注意到单元上的变化。然后，当使用 TomTom Home 时，所有这样的改动都会反映在单元本身上。智能，TomTom 和一个整洁的方式来区分你的产品从一些[其他](https://web.archive.org/web/20130628182601/http://crunchgear.com/2007/05/21/mio-digiwalker-c520-the-gps-device-that-talks-to-you/) GPS 单元在那里。

TomTom 720 应该会在 7 月底上市，给你足够的时间凑够钱给自己买一个。我们还不知道你实际会支付多少。
