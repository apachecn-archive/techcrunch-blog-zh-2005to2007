# PS3 导致索尼利润下降 68%

[![ps3homewrecker.jpg](img/b275375a50594646265b4fd4a8d17046.png)](https://web.archive.org/web/20210225131923/https://beta.techcrunch.com/wp-content/uploads/2007/05/ps3homewrecker.jpg "ps3homewrecker.jpg")

我不会告诉一家公司如何经营自己或自己的业务，但索尼经历了我记忆中最糟糕的一年。它刚刚公布的利润与去年相比下降了 68 %,我们都知道罪魁祸首是什么。没错，PS3 的表现不佳和拙劣的发布确实让索尼今年陷入困境。制造商在硬件上遭受损失并不罕见(尽管任天堂的 Wii 已经以 2:1 的比例[超过](https://web.archive.org/web/20210225131923/http://crunchgear.com/2007/04/03/wii-outsells-ps3-and-xbox-360-in-february-must-mean-its-the-best/)PS3)，但索尼的损失尤其明显，这要归功于昂贵的蓝光驱动器。

索尼表示，它将在今年确定其软件阵容，坦率地说，看起来并不那么热门。当然，可能像[*【GTA 4】*](https://web.archive.org/web/20210225131923/http://crunchgear.com/2007/04/23/grand-theft-auto-6-on-its-way/)和*刺客信条*这样的伟大游戏即将出现，但那些曾经只属于 PS3 的游戏现在也在 Xbox 360 上安家了。
