# 性侵犯者抱怨被移除的电脑

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/16/sex-offenders-complain-about-removed-pcs/>

![](img/a6de9a6274fd4ebff6de85cdbb025659.png)

在明尼苏达州，一群被监禁的性犯罪者被激怒了，因为看守没收了他们在牢房/房间里使用的电脑。所以像任何一个美国人一样，他们向上诉法院提起诉讼，却被一个只有半个大脑的法官驳回了。我猜出庭是这样的:

> **法官**:我为什么要让一群性犯罪者保留他们的电脑？
> 
> **囚犯**:这样我们就可以……啊……跟上……时事。你知道的。我们想
> 发布鲍勃在淋浴时被捅的视频。
> 
> 法官:是的，这次我不得不说“不”。
> 
> **犯人**:该死。

我认为这个裁决至少可以说是法院的一个明智的决定。我的意思是，有人希望一群被定罪的性犯罪者和强奸犯玩 WoW 或在 YouTube 上发布视频吗？没想到会这样。
