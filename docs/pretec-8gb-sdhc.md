# pretec 8GB SDHC

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/08/31/pretec-8gb-sdhc/>

![](img/d38d56066d7f0743dcbf9d26adda2649.png)

看起来 8GB 的 SD 卡将很快上市。Pretec 声称已经开发出了第一种这样的卡。该卡支持 SD 2.0，不向后兼容。用户目前可以订购一张卡用于取样。该公司还创建了一个 16GB 的闪存驱动器，据称是同类产品中最大的，但这显然是错误的，因为驱动器远远超过 16GB。无论如何，Pretec 公司表示，这种驱动器将于明年年初投入大规模生产。

我肯定这东西会很贵。一个更好的选择可能是坚持使用一个更小的卡，并简单地使用一个基于便携式硬盘的照片浏览器。这样你就可以得到大约 80GB 的内存，价格也差不多。
