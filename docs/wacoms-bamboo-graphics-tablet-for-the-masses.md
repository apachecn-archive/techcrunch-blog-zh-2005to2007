# Wacom 的竹子:面向大众的图形平板电脑

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/16/wacoms-bamboo-graphics-tablet-for-the-masses/>

![wacom_bamboo.jpg](img/9d5ad77e121dfe9f128e169eba4ef9a4.png)

Wacom 在 NAB 上给我们留下了深刻的印象，它的新竹迷你图形垫预览，竹将在本月晚些时候以仅 140 美元的价格发货。对于那些可能不熟悉这个概念的人来说，Wacom 的 pad 可以插入你的 PC 或 Mac 的 USB 端口，充当一个图形输入设备:你用一个特殊的手写笔画画，你计算机上的 pad 感知软件(例如 Illustrator)就会做出响应。手写笔还有其他功能，如内置工具和压力传感器，所以你真的可以获得类似纸笔的体验。方便设计师使用。

竹子将这个概念带到了一个新的地方，顶部有一个类似 iPod 的滚轮。据说这使得在 pad 过去没有很好解决的工具、设置和菜单中滚动变得简单。任何让数字艺术对我们这些非艺术白痴来说更容易的东西都是受欢迎的。
