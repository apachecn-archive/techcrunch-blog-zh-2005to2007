# Web 2.0 工作组

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/10/10/web20-workgroup/>

我虔诚地阅读许多博客，在我阅读的 400 多篇文章中，有 15 篇左右我会一天多次查看新内容。

其中的两个博客是理查德·麦克马努斯的读写网(T1)和弗雷德·奥利维拉的博客(T3)和 T4。

理查德和弗雷德写的博客对 TechCrunch 大加赞赏。我们通常关注新公司，理查德分析新网络的趋势，弗雷德关注设计、可用性和开发。为了全面了解这个领域，我强烈推荐阅读这三个博客。

今天我们组成了一个松散的联盟。你会在这个页面的左边栏看到一个“web 2.0 工作组”的标志——它链接到一个登陆页面——[web20workgroup.com](https://web.archive.org/web/20180319074430/http://www.web20workgroup.com/)。在那个页面上，我们有所有三个博客的链接，以及最近的帖子和我们的订阅信息。

随着时间的推移，我们将添加更多的博客。这不是博客网络或任何正式的东西。我们只是喜欢更紧密地协调我们的想法。

理查德和弗雷德也成了好朋友。理查德来自新西兰，他上周第一次来美国参加 Web 2.0 会议，他在这里期间一直住在我家。弗雷德来自葡萄牙，他在我们的 edgeio 项目工作期间也住在这里。
