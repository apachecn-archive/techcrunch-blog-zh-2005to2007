# 可堆叠硬盘驱动器；乐高造型

我们喜欢便携式数字存储设备，所以我们当然喜欢 Lacie 的这些砖式桌面硬盘。重量从 160GB 到 500GB，彩色编码的砖块就像乐高积木一样堆叠起来。

有了几兆字节的存储空间，你可以在你的办公桌周围建立一个堡垒，保护你自己免受同事傲慢的成年人的可怕外表的伤害。唯一的缺点是，它们不包括其他巨大的砖块，你只能使用标准的 2×3 图标。不过，你知道，这是乐高。不是官方的，是乐高！
