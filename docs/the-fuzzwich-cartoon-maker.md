# Fuzzwich 卡通制作人

[![fuzzwichlogo.png](img/89b23597af4d1bb4467b6b1eec535db8.png) ](https://web.archive.org/web/20220119232049/http://www.crunchbase.com/company/fuzzwich) [Fuzzwich](https://web.archive.org/web/20220119232049/http://www.crunchbase.com/company/fuzzwich) 是一个死简单的应用程序，用于创建和发布动画短片。动画编辑器预先填充了随机选择的音乐、背景和卡通人物。然而，你可以很容易地将它们更改为其他 42 个字符、15 首歌曲或 14 个背景中的任何一个。内容库将随着时间的推移而变化，他们计划很快添加一个角色编辑器。

有了基本的布局后，你可以通过在场景中拖拽小精灵来实时调整它们的大小。播放器会在剪辑播放时记录你的动作。一些角色和场景带有动画(移动手臂或嘴巴)，也可以添加语音气泡。发布后，它可以很容易地嵌入或链接到另一个网站。我自己 5 分钟做了一个，完全上瘾了。我是这些催眠熊猫的超级粉丝。

这项服务的一个严重限制是，用户不能上传自己的图像或声音文件，也不能将它们包含在动画片中。然而，他们确实计划让人们把他们的脸上传到漫画上。他们也有一个功能更丰富的动画工作室，但最初专注于这个更简单的编辑器，因为它可以为更广泛的观众所用。

我们已经介绍了另外两家初创公司，Aniboom 和 MyToons，它们的目标是成为在线卡通的“YouTube”。Fuzzwich 与 Aniboom 的应用程序和 MyToons 有些竞争，但也有所不同。Fuzzwich 提供了一种在网站上创建卡通的简单方法。Aniboom 有一个叫做 Shapeshifter 的动画师，它就像是一个专业的动画制作工作室。MyToons 不提供漫画创作工具。

Fuzzwich 是一家 Y Combinator 公司。
