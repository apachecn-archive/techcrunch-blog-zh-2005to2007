# NBC 环球和 MobiTV 宣布移动电视点播服务

NBC 环球宣布，他们将与 MobiTV 一起为美国市场提供点播移动电视内容。NBC 电视台的黄金时段节目，比如《T2》、《办公室 T3》和《T4 英雄 T5》，很快就能在你的手机上看到。包括 USA、Bravo、SCI FI、Telemundo 和 mun2 在内的一系列其他 NBCU 频道将提供广告支持的节目。MobiTV 也将有几个广告支持的免费点播节目，包括*周五之夜*。24 小时观看期内，每场黄金时段节目的服务费为 1.99 美元。目前还不知道提供商是谁，但是这项服务有望在 Q2 的某个时候上线。我的祈祷得到了回应:只要我想，海顿·潘妮蒂尔就在我的手机上。
