# 隐藏您的 Nano，让它更大

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/13/hide-your-nano-make-it-bigger/>

是啊。它让你的超薄 Nano 对于你的弹力牛仔裤来说太大了。我会抓住机会的，谢谢你。
