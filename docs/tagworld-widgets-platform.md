# TagWorld Widgets 平台

由 Draper Fisher Jurvetson 资助的拥有近 200 万会员的社交网络[将于周五在](https://web.archive.org/web/20200807161324/http://www.beta.techcrunch.com/2006/03/12/tagworld-hits-1-million-members-announces-75-million-from-draper-fisher-jurvetson/)[TagWorldWidgets.com](https://web.archive.org/web/20200807161324/http://www.tagworldwidgets.com/)推出一个 Flash widget 平台。

TagWorld Widgets 是一个开放的平台，他们邀请开发者在现有的 30 个左右的 Widgets 的基础上进行开发。开发者站点[在这里](https://web.archive.org/web/20200807161324/http://tagworld.com/-/widgets/WidgetSubmit.aspx)。这些小部件可以在任何网站上使用，TagWorld 正在大力推广这些小部件在 Myspace 等竞争社交网络上的使用。

有几项创新使其成为比 [BunchBall](https://web.archive.org/web/20200807161324/http://www.bunchball.com/) 和其他人已经提供的解决方案更有趣的解决方案(BunchBall profile [此处为](https://web.archive.org/web/20200807161324/http://www.beta.techcrunch.com/2006/02/05/embed-flash-games-in-your-blog/))。微件具有可定制的属性，允许网站发布者对其进行个性化设置。比如颜色、字体、速度等等。可以在整合到网站上后进行定制，而无需返回到 TagWorld 站点。
