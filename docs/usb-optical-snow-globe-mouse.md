# USB、光学雪球鼠标

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/16/usb-optical-snow-globe-mouse/>

今年圣诞节，唯一在你桌上活跃的生物应该是一只老鼠，一只雪景鼠，真的(哈哈！).这是你标准的带滚轮的光学 USB 鼠标，除了 palm rise 中该死的雪景，它是由透明塑料制成的。把它摇起来，人造雪通过里面类似天体滑行的物质慢慢落到地上。它停在一个半英寸的塑料雪人的脚边，开心地对自己微笑，尽管事实上下周的这个时候他将被融化到下水道里。

为了完成“节日快乐”主题的效果，鼠标的正面有一个大大的白色笑脸(可能是马上就要变软的雪人的笑脸？)，以滚轮为机头。

现在在日本可以买到，而且，知道我们的运气，在圣诞节之前在这里。
