# 博主私人内测筛选？

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/11/private-beta-screening-for-bloggers/>

![](img/423b4f32bf5ceecb7074e256cfc41cca.png)

因《T2》而出名的休·麦克劳德正在谈论邀请博客写手参加一部名为《哈勒姆·福》的新片的私人放映。

此刻

the Hallam Foe film screening [has been tentatively pencilled in for 28th September followed by a Geek dinner?](https://web.archive.org/web/20160530141917/http://www.getyourpeople.com/archives/2006/08/i_posted_this_o.html) The aim is to get the invitees to blog about the film and spread the word. Hugh describes it as a “what do you think” event which reminds me more about how software is launched using public betas to garner feedback, thus enabling the buzz to spread virally, before the formal press announcement.

> 这不会是你典型的“这里有一些免费票，现在去写我们该死的电影”的想法。首先，这部电影只能以粗略的形式放映，因为最终的剪辑还需要几个月的时间。在我看来，这更像是一个“这就是我们正在做的，你觉得怎么样”的夜晚。戴夫、科林和制作团队的一些人会出现，会见一些博客作者，并希望离开时能更好地理解博客世界。

我猜如果你想要门票，你需要让休在他的评论区知道，现在我还没有看到这个活动的维基。
