# PSP 固件更新支持远程游戏| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/21/psp-firmware-update-enables-remote-play/>

![](img/e4bf538bc5b1ccc777bb42ee27155764.png)

如果你是少数几个拥有 PlayStation 3 的人之一，并且想用你的 PSP 进行远程游戏，你必须下载新的 3.0 固件更新。新的更新将允许你从你的 PSP 到 PS3 观看图像和流媒体音乐，以及启用远程播放功能，允许你将你的 PSP 用作 PS3 的控制器。

新的固件更新也有一个自动 UMD 选项，允许用户选择他们是否希望 PSP 在插入设备时自动启动 UMD。还有一个我们不太了解的相机功能，因为索尼还没有发布 PSP 的相机插件。
