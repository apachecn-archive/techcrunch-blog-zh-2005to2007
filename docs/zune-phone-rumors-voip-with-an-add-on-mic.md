# Zune 手机谣言:带附加麦克风的 VoIP

来自知情人士的最新 Zune 手机传言称，Zune 将是 VoIP，考虑到它已经内置了 WiFi，这是有道理的。他们还说，将会有一个附加附件，适合麦克风的基座。

我们猜测，微软将把这一点与他们的大型 Live Messenger 结合起来，试图在桌面 VoIP 方面超越 Skype。他们已经发布了带有 Windows Live Messenger 品牌的网络摄像头和手机，因此 Zune 肯定会非常适合这个领域。
