# 富士通笔记本电脑加入索尼电池召回

富士通刚刚宣布召回 287，000 块电池——全部由索尼制造。该公司加入了苹果、戴尔、东芝、联想和索尼公司召回电池的行列，使召回的电池总数达到 760 万个。

据报道，宏碁本周早些时候与索尼讨论了召回电池的问题，但尚未发布召回通知。如果您有一台富士通笔记本电脑，请查看此页面，以了解您的笔记本电脑是否受到影响。
