# TDK 为高清摄像机开发迷你蓝光光盘，为每个人开发 200GB 蓝光光盘

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/05/tdk-develops-mini-blu-ray-discs-for-hd-camcorders-200gb-blu-ray-discs-for-everyone/>

今天来自 TDK 的两项蓝光发展。第一种是 200GB 的六层蓝光光盘，其“电影”容量是目前双层蓝光光盘的四倍。另一种是 8 厘米的迷你蓝光光盘，容量为 16.5GB，将用于高清摄像机和其他传统上使用迷你 DVD 光盘的可录制媒体。

乔，你为什么要在乎呢？嗯，我们假设如果你已经买了一个蓝光播放器，它可以读取未来的蓝光光盘。假设这种新的六层光盘可以在这些光盘上播放，一切都很好。但是如果你不得不升级你的播放器来支持更大的蓝光容量，那就糟了。
