# Technovision 的 205 英寸液晶电视:多大才算太大？

![](img/746d23fff7491fb6b1baebda5c0e4dd6.png)

电视迷们注意了:Technovision 的精英们在 CeBIT 上展示了一款 205 英寸的 Luxio 液晶电视。(想想看，在今年早些时候的 CES 上，人们对 108 英寸的电视垂涎三尺。)你很可能需要搬进一个更大的房子，才能放得下所有 8.3 x 15 英尺的电视。意大利制造商没有具体说明这款电视的价格，但我会大胆猜测，说你需要一份工资比最低工资高一点的工作才能买得起它。那就别把我算进去。
