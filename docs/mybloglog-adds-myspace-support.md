# MyBlogLog 增加了 MySpace 支持 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/12/10/mybloglog-adds-myspace-support>

 [](https://web.archive.org/web/20221001154737/http://www.mybloglog.com/) 随着 2006 年的临近，我今年最喜欢的网络服务是 [MyBlogLog](https://web.archive.org/web/20221001154737/http://www.beta.techcrunch.com/2006/10/19/mybloglog-readers-network-around-their-favorite-blogs/) 。尽管雅虎！[收购传闻](https://web.archive.org/web/20221001154737/http://www.beta.techcrunch.com/2006/11/17/mybloglog-to-be-acquired-by-yahoo/)几周前，这家公司还是私有的。如果我从事网络并购业务，它们会和音乐社交网络服务一起成为我的首选。

MyBlogLog 已经建立了下一代社交网络服务。如果 Friendster/MySpace/etc 是社交网站的 1.0 版，这就是 2.0 版。该服务创建了一个[分布式社交网络](https://web.archive.org/web/20221001154737/http://www.vestedventures.com/blog/2006/06/distributed-social-networking-website.html)平台——允许网站和博客在其访问者社区中实现社交网络。

今天，MyBlogLog 增加了对 MySpace 个人资料页面的支持。这是他们渗透 MySpace 市场的一种方式。MySpace 几乎提供了 MyBlogLog 所拥有的一切，除了 MyBlogLog 有两样东西是 MySpace 所缺乏的:

1.  MyBlogLog 将显示谁最近访问了你的网页(只有其他 MyBlogLog 用户)——这对于好奇的猫来说很方便，他们想知道谁在查看他们的个人资料。
2.  MyBlogLog 是一个分布式社交网络系统，它不仅允许你与 MySpace 的用户交流，还允许你与访问网络上成千上万的其他博客/网站的用户交流。

![](img/7ee254b93d7e98a928b205ff313e0c25.png)

它的工作原理是，你用 MyBlogLog 创建一个帐户，抓取一段代码，把这段代码放在你的 MySpace 个人资料中，然后你[可以看到](https://web.archive.org/web/20221001154737/http://www.myspace.com/stevepoland)还有哪些 MyBlogLog 用户访问了你的网站。

我对 MyBlogLog 服务的唯一疑虑是公司名称——它把他们的业务和博客联系在一起。实际上，他们应该有像 NHL.com、纽约时报和 Slashdot 这样的合作伙伴——这些网站的用户社区愿意更多地了解彼此，互相交流。在我看来，每个网站都应该有 MyBlogLog——它可以让你的用户相互交流，建立社区。我在等着听第一次婚姻的故事，它是由于 MyBlogLog 使两个用户在一个网站上相遇而发生的，这两个用户多年来都经常访问这个网站，但从来没有办法了解对方(或那个网站的其他访问者)。

几个月前，[我](https://web.archive.org/web/20221001154737/http://www.mybloglog.com/buzz/members/vestedventures/)通过我的博客与首席执行官[斯科特·拉夫](https://web.archive.org/web/20221001154737/http://www.mybloglog.com/buzz/members/rafer/)交换了信息，他告诉我他们一直在寻找一个替代域名——你对斯科特和他的团队有什么建议吗？
