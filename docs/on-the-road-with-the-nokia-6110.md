# 驾驶诺基亚 6110 上路

![](img/9a093f394ff3fcdd30a5f270b2f3b43f.png)

趋同是一块难啃的骨头。我还没有遇到任何人把他们的手机当作 MP3 播放器使用——尽管他们就在那里——现在诺基亚希望我们把他们的手机当作 GPS 设备使用。

![](img/182d8ff7e7af9cdf91eb8847f0359361.png)
我很高兴地报告，他们的解决方案，嵌入在 [slim 6110](https://web.archive.org/web/20210227171619/http://crunchgear.com/2007/02/12/nokia-6110-3110-plain-jane-phones-with-some-interesting-features/) 上，相当不错。他们使用 Route 66 软件来提供一个令人信服的理由，让你把手机贴在仪表盘上。

我今天开着 6110 四处兜风，它带领我们穿过了巴塞罗那曲折的小街。这款设备大约和他们的 Symbian 滑盖 N80 一样大，背面有双扬声器，可以大声喊出方向。有多种视图，包括“夜间”视图，它创建了地图的负片，只照亮您所在的路线。指令非常准确，考虑到尺寸，GPS 加载时间比我预期的要快得多。

![](img/afbe26262800b996f8a1fa9014f5069e.png)

一个非常有趣的功能是一个“行人”设置，它允许你像一个小地图一样带着这个家伙在城里到处走。昨晚我在镇上闲逛找旅馆时，那会特别有用。

就趋同性而言，这可能是完全正确的。我会用这款手机取代一个独立的 GPS 设备，因为它在这么小的一个包里提供了这么多。大约 400 美元——在美国还看不到——我们可能要等一段时间，直到我们可以咨询手机，而不是福多在沙无然的 *15 性爱之旅*。
