# SunRocket 的客户不要担心，发誓要拯救他们

Vonage 已经来拯救那些因[sun rocket](https://web.archive.org/web/20130628131940/http://crunchgear.com/2007/07/16/aw-heeall-nawgh-sunrocket-burns-out/)关闭而受损的 VoIP 客户，他们可以通过转换到 Vonage 来保留他们的号码，而且是免费的。如果你选择转换，这可能是个好主意，因为把你的新号码告诉你所有的朋友是一件痛苦的事情，那么你可以得到两个月的免费服务。这是个陷阱，对吧？你打赌你的甜屁股有，但这是相当无痛的。你只需要注册美国住宅保费无限计划，一切都是正确的。拨打 1-800-833-4806 给 Vonage 打电话，把那个号码转到其他地方。
