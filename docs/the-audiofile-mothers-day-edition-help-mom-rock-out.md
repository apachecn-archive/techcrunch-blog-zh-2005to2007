# 音频文件(母亲节版):帮助妈妈摇滚起来

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/11/the-audiofile-mothers-day-edition-help-mom-rock-out/>

本周日是母亲节。(如果我让你喝了咖啡，我很抱歉。)像往常一样，我把它留到最后一分钟给我妈妈买点东西。自然地，现在她~~终于~~明白了我是做什么的，她慢慢地对便携式音频更感兴趣了。所以我想今年我应该挥霍一下，给她买些结实、简单、价格合理的音响设备，但我必须尽快。我需要剩下的面包来支付我在布鲁克林的高昂房租。以下是我的行动计划…

![](img/6de5480aa25ebb5b655d7d147d4d7e13.png)
一点背景:我的母亲在 70 年代是 IBM 的一名电脑程序员，尽管她是技术出身，也是一名长期的 Windows 用户，但她现在并不完全是一名消费电子产品专家。无论我给她买什么，都必须是烤面包机——简单的，这样我就不用做太多的技术支持，但质量要足够好，我会自己听，我是个音频势利小人。而且我也不想被迫~~典当什么东西~~向她借钱买。

大约一年前，我给了她一个非苹果的 MP3 播放器(最初的免费 Napster 播放器),希望她会喜欢上数字音乐。虽然她喜欢我为她放在那里的两张专辑(鲍比·达林最佳专辑和尼尔·岱尔蒙德专辑)，但她从来没有学会使用 Windows Media Player 来抓取和传输音乐。

考虑到这一点(我想你明白这是怎么回事)，我认为 iPod 将是一个万无一失的计划。事实证明，当她给她丈夫买了一个 4GB 的 iPod nano 时，我有机会测试这是真是假。当他们试图设置它时，我接到的电话非常滑稽，最终导致了从 iPod 的基座连接器上拔插头的问题。哦。

那么有什么比 iPod 更简单呢？嗯……对于音乐管理软件，你找不到比 iTunes 更简单的了，即使它不是你最喜欢的应用。(我认为像这种还能播放音乐的闪存盘应该是最简单的，但是翻录并不直观，组织起来也很麻烦。)对不起 WMP 11 和 Winamp 粉丝，我就是不买账。

如果我不得不挑选一个 iPod 的替代品，那将是第二代 iriver Clix T1，因为它有一个出色的界面，可以与 Windows Media Player 以及 Windows Explorer (USB 大容量存储类)一起工作。由于许多 Windows 用户习惯于文件/文件夹组织，这是对音乐管理软件不感兴趣的人的最佳选择。

作为一个音频设备迷，我总是很失望看到有人戴着普通的耳塞。也就是说，让妈妈把东西塞到耳朵深处是不可能的，我希望她在走动时能够听到周围发生的事情。

我对妈妈友好的耳机的手动选择是森海塞尔 MX-500。它们只是普通的耳塞，戴上耳塞就不会太容易掉下来。售价 13 美元的耳机听起来也出奇的好——比任何普通耳机都好，而且比 iPod whities 更不容易被妈妈抢走。不过，如果我最终给她买了 Clix，附赠的耳塞应该没问题。

如果她不喜欢把东西塞在耳朵里，我会选择森海塞尔 PX-100，这是一款听起来很棒的平板耳机，售价约 40 美元；它们也可以很好地折叠在钱包里。~~我打赌她会喜欢一套 Bose QuietComfort 3。~~

当然，不管我给她买什么，我只知道我至少要花几个小时为她布置好，并教她如何使用。我认为底线是，对于拒绝学习新技术的人来说，没有什么事情是足够简单的。

但至少今年我的礼物会比礼券更贴心。
