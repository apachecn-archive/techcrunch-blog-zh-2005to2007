# Ziki，本周的新社交网络

 [](https://web.archive.org/web/20220816001827/http://www.ziki.com/) 总部位于纽约的 [Ziki](https://web.archive.org/web/20220816001827/http://www.ziki.com/) 是一个新的社交网站，它允许人们标记自己，并通过这些标记与他人联系。它还允许用户从他们的博客、flickr 照片、YouTube 视频等获取 RSS 源。

社交网络的加入让这个时代变得更加美好。

我在注册时遇到了各种各样的问题(我很擅长注册新服务)。Ziki 需要太多个人信息，比如我的生日。此外，我在尝试提交注册时多次出错，并且从未收到允许我登录的确认电子邮件。我的建议是:放宽对个人信息的要求，修改网络表单和电子邮件确认，慢慢鼓励你的客户在网站上添加更多内容。更多关于 Ziki [这里](https://web.archive.org/web/20220816001827/http://go2web2.blogspot.com/2006/04/ziki-lovely-social-network.html)。
