# 现在的智能手机:Palm OS–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/03/smartphones-now-palm-os/>

虽然 Palm OS 在美国的市场份额落后于 RIM 和黑莓，但它仍然是一个强大的操作系统，受到许多用户的青睐，他们相信它的可靠性和易用性。早在 1996 年，随着最初的 Pilot 1000 首次推出，该操作系统在功能和健壮性方面都有所增长，同时也变得更加易于使用。
 Palm OS 一度是美国手持设备的绝对领先操作系统，但微软对 OEM 手机制造商的大幅折扣和 RIM 黑莓 OS 的低开销已经造成了影响。也就是说，Palm OS 提供简单的功能，同时保留执行复杂和高级任务的能力，这使得它成为第一次使用智能手机的用户的好选择。就美国智能手机而言，它目前只能在 Palm 非常受欢迎的 Treo 系列上使用。

Palm OS 用户几乎普遍喜欢他们的设备。它是唯一一个原生支持 Mac OS 的智能手机操作系统，所以 MacBooks 和 IMAC 的用户可能不想再看这个流行的操作系统。这并不意味着 PC 甚至 Linux 用户不应该认真看待 Treos，因为扩展你的桌面或替换你的笔记本电脑是智能手机吸引力的一部分，这是 Palm OS 十多年来一直做得很好的事情。

## 我们拼写爱 P-A-L-M

但是不要让这种 Mac 比较愚弄了你；Palm OS 用户并不觉得自己错过了什么。从 Exchange 同步到网页浏览，Windows Mobile 的所有功能都是匹配的，而且通常更易于使用。此外，与黑莓相比，Palm OS 可以与更多的推送电子邮件系统配合使用，甚至可以与 RIM 的 BlackBerry Connect 配合使用，进行企业设置。

不过，Windows Mobile 和 RIM 并不是唯一的竞争对手。塞班，一个在世界其他地方非常受欢迎的操作系统(事实上，它是全球市场的领导者)，正在美国取得进展。诺基亚现在正在 Cingular 网络上销售一款价格适中的智能手机，采用 Symbian 风格，尽管其受欢迎程度仍不确定。Symbian 操作系统在用户忠诚度、易用性和可扩展性方面非常相似。

然而，在共享软件、免费软件和商业领域，Palm OS 比其他移动操作系统具有更多的附加软件，其中大多数都分享了 OS 的易用性理念。

## 谁，什么，在哪里？

目前，威瑞森和 Sprint 提供 Treo 700p，这是基于 Palm OS 和 CDMA 的智能手机版本，EV-DO 提供惊人的快速移动互联网连接。Cingular 销售支持 EDGE 的 Treo 650，并准备提供 Palm 的最新 Treo 650。当运营商开始销售时，这种无天线手机将有四种颜色，表面上是在未来几天的某个时候。

T-Mobile 是唯一一家不提供 Palm OS 设备补贴的美国主要运营商，尽管这更多是因为 Palm 的硬件而不是软件。T-Mobile 正在寻求其智能手机供应商承诺集成 WiFi，这是 Palm OS 设备尚未提供的功能。但是，这又是一个硬件问题，因为操作系统已经支持 WiFi 好几年了。这并不意味着你不能在 T-Mobile 上运行 Palm OS Treo，但这确实意味着找到并购买一个没有补贴和解锁的设备，这可能有点棘手。

最终，智能手机的好坏取决于你能用它做什么，这也是基于 Palm OS 的设备大放异彩的地方。目前有超过 65，000 个应用程序可供下载和/或购买，用户可以将任何 Palm OS 设备定制为他们想要的样子。由于智能手机应用程序本身的用途往往有限，用户可以通过下载的应用程序来混合和匹配他们想要的功能，从而减少存储膨胀并最大限度地提高效率。例如，使用谷歌地图应用程序查找位置实际上比通过网络浏览器更容易。电子邮件很简单，因为联系人有他们自己的应用程序(他们无缝地一起工作)，任务和备忘录之类的事情也是如此。

## 哪一个？

根据您使用智能手机的目的(商务或个人)，互联网上有几十种指南可以帮助您在程序迷宫中导航，准确找到您需要的程序。安装它们很容易；下载后点击图标，通过随附的电缆插入 Treo，它就加载了。或者，你可以把应用程序拖到你桌面的蓝牙传输程序中，选择手机，点击“确定”来加载它。你甚至可以直接从手掌大小的网络浏览器下载应用程序，升级和定制 Palm OS 设备非常非常简单。

总而言之，这就是它的本质:这是一个易于导航、学习以及使用的操作系统。如果你以前有过非智能手机的 Palm OS 设备，你会很熟悉，你的绝大多数传统应用程序会像以前一样工作。如果你是手持计算的新手，这是对当前市场的完美介绍。如果你是个老手，试试看。你一定会喜欢的。
