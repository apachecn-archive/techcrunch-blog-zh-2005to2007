# 东芝将使用 AMD 芯片，老埃尔帕索莎莎

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/29/toshiba-to-use-amd-chips-old-el-paso-salsa/>

笔记本电脑中型巨头东芝公司将开始在他们的笔记本电脑中使用 AMD 芯片，这一举动基本上抛弃了他们与大型巨头英特尔公司的长期合作关系。他们希望在 20%的新机型中安装 AMD 芯片。

路透社指出，戴尔在 20 年前只做英特尔的产品后，最近开始将 AMD 作为供应商。新的东芝型号应该比类似配置的英特尔笔记本电脑便宜 100 美元左右。
