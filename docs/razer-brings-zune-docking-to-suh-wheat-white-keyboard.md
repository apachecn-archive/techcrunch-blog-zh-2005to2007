# Razer 为 Suh-Wheat 白色键盘带来 Zune 对接

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/14/razer-brings-zune-docking-to-suh-wheat-white-keyboard/>

虽然它看起来像一个台球玩具，但这确实是一个带有 Zune dock 的理论键盘。它的背面还有两个 USB 端口，为您节省了更多宝贵的 PC 端口。它甚至是白色的，以配合你的白色 Zune。没有消息说这是黑色还是棕色，但我们猜测不是。

除了 Zune 连接之外，这个媒体键盘还有专门用于媒体应用的按键，包括对 Zune 本身的控制。好主意。否则，我们对这个明亮的白色 QWERTY 打字机没什么可说的。
