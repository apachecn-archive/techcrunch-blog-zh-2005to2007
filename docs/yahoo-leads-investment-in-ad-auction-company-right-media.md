# 雅虎！牵头投资广告拍卖公司 Right Media

 [](https://web.archive.org/web/20230208151040/https://direct.rightmedia.com/index.php) 纽约另类广告元网络[右媒体](https://web.archive.org/web/20230208151040/http://rightmedia.com/)今天完成 4500 万美元的 B 轮融资。该服务实时拍卖多个广告网络之间的广告空间。雅虎！领一轮，会拿公司 20%的股份。2005 年 6 月，Right Media 从 Redpoint Ventures 获得了 725 万美元的投资。红点也是这一轮的参与者。

我们在 8 月份对 Right Media 进行了简介，当时该公司发布了一个名为 RMX 直销的轻量级广告市场。

Right Media 的系统允许网站出版商加入多个广告网络，并自动显示来自任何一个网络的广告，只要该网络在实时竞价中为其广告空间支付最高的每次印象价格。该公司表示，通过其服务，每天有超过 20 亿次的展示交易。该服务包括一个过滤系统，以防止恶意软件通过广告传播，并执行出版商的标准。出版商能够对他们网站上的广告进行非常精细的控制。

自动竞价包括诸如每次展示的地理位置以及网站和用户的转换率(cookies)等因素。竞价是相对于预先确定的活动实时进行的 Right Media 的企业版目前每秒进行 30，000 次拍卖。

8 月，Right Media 有 50 家广告网络加入了其企业版，8 家加入了其小型出版商版。批评者质疑，如果没有大型广告网络参与他们的服务，Right Media 是否会是一项有用的服务，尽管这种批评只能针对小型出版商服务。Right Media 的企业部分的参与者包括福克斯互动、论坛报和 Looksmart。雅虎！显然支持这种模式，并将通过 Right Media 拍卖自己的非优质广告空间。

对雅虎来说，时机是正确的。在广告领域有所作为。自 9 月中旬公开讨论来自大型汽车和金融服务广告买家的广告收入下滑以来，该公司的股票暴跌，至今仍未恢复。更多的坏消息来自雅虎！今天——[报道第三季度利润下降 38%。利用非优质广告空间的拍卖系统可以帮助弥补公司广告战略中出现的一些漏洞。除了利用低流量网页之外，入股 Right Media 还有助于打击点击欺诈。实时竞价可以通过自动放弃竞价来快速响应低转换率。](https://web.archive.org/web/20230208151040/http://www.bloomberg.com/apps/news?pid=20601103&sid=aHuYKUVII1dg&refer=news)
