# Slingbox 改版| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/14/slingbox-gets-a-facelift/>

![](img/5589595881558d3fea0efd554d41c50c.png)

Sling Media 已经向 FCC 提交了第二代 Slingbox 的一些信息。新设备将通过 A/V 电缆或 S-Video 直接连接，而不是他们迄今为止使用的三合一电缆。

没有说是否有其他东西被更新，但没有提到 WiFi，所以看起来你的“去任何地方”的媒体设备仍然只能去以太网电缆。

实际上，我有点喜欢第一个设计，有点喜欢它。这个让我想起了 80 年代的玩具。不知道什么时候可以买到，但是我想是在假期之前。
