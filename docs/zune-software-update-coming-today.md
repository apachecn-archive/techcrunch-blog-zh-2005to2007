# Zune 软件今天更新| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/19/zune-software-update-coming-today/>

[Zunerama](https://web.archive.org/web/20150909063340/http://www.zunerama.com/) 的哈维传来消息，Zune 软件将于今天晚些时候更新。它将启用对 Windows Vista 的支持，并修复 Biggs 和 Goldman 等用户在安装 Zunes 时遇到的许多问题。

更新将是 22MB。它将通过微软自动更新和 Zune.net 的[发售。](https://web.archive.org/web/20150909063340/http://www.zune.net/)
