# nVidia 8600 和 8500 GPUs 现在官方:仍然摇滚 PureVideo HD 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/17/nvidia-8600-and-8500-gpus-now-official-still-rock-purevideo-hd/>

[![nvidia8600.jpg](img/db01a6046d35194e0c3b44b097b61c38.png)](https://web.archive.org/web/20150922033746/http://old.crunchgear.com/wp-content/uploads/nvidia8600.jpg "nvidia8600.jpg")

显卡巨头 nVidia 终于正式推出了中端 8500 和 8600 显卡。它们本质上都与高端 8800 一样强大，但 nVidia 限制了着色器的数量。或者，在英语中，卡片没有顶级卡片那么强大。大惊喜。

然而，价格并不太糟糕，8600 GTS 的标价为 229 美元，8500 在适当调整后可降至 89 美元。这些卡的一个重要特点是 nVidia 的 PureVideo HD。如果你曾经试图在电脑上观看高清视频，无论是通过高清光盘还是 rips，你都知道你需要一个非常强大的处理器来流畅地运行。通过将视频处理转移到 GPU，您的 CPU 不会融化。
