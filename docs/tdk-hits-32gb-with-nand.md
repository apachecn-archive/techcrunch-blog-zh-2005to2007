# TDK 凭借 NAND | TechCrunch 达到 32GB

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/18/tdk-hits-32gb-with-nand/>

![](img/e15b4bea910f986f1b0e15f7573eb81d.png)

TDK 宣布，其新的 IDE 兼容 NAND 闪存容量达到了 32GB。该闪存盘的尺寸比标准的 2.4 英寸笔记本电脑驱动器小约 20%，功耗更低，数据传输速率高达 33.3MBps。目前仅提供有限的样品数量，但随着所有类似设备的发布，预计明年某个时候将开始在笔记本电脑中看到这些设备。
