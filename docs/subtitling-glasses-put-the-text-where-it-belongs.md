# 字幕眼镜把文字放在它该在的地方

![subtitle-glasses.jpg](img/e0e205b95820043052fb574969d2665b.png)

我更喜欢看带字幕的母语的进口电影。进行一场激情的表演，却让一些蹩脚的配音演员来配音，这太糟糕了，你不觉得吗？然而，屏幕上的字幕会让人分心。我喜欢字幕眼镜背后的想法。

使用一个[通用的士兵式](https://web.archive.org/web/20230322164156/https://techcrunch.com/wp-content/uploads/2007/04/n-universalsoldier.jpg)光学系统，来自电视或 DVD 播放器的字幕被投射到你眼前的玻璃上，类似于平视显示器的工作方式。这样，你的眼睛可以扫描整个大屏幕，对话总是在你需要的地方。聪明。

目前，原型是为剧院观众准备的，但由于最难的部分是耳机，所以今年晚些时候可以寻找更低的消费版本，很可能在 100 美元的价格范围内。
