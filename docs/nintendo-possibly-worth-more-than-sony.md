# 任天堂可能比索尼更值钱

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/25/nintendo-possibly-worth-more-than-sony/>

![](img/90f0b3a077366c27ed53f135d6f3546b.png)

上周末，许多报道冲击了像 Digg 这样的网站，称任天堂的市场价值已经超过了索尼。从数字上看，索尼似乎仍以 6.64 万亿日圆领先于任天堂的 6.3 万亿日圆。随着极其受欢迎的 Wii 和 DS 系统的热销，难怪任天堂的表现如此出色。尽管这很棒，但我仍然觉得有人没有告诉我事情的全貌。

例如，索尼不像任天堂那样只生产一堆游戏硬件。索尼是一个巨大的电子产品制造商，每天在全世界销售数千美元的产品。你是在告诉我，在所有那些 [SACD 播放器](https://web.archive.org/web/20140904051107/http://crunchgear.com/2007/03/19/the-case-for-high-end-audio-the-super-audio-cd/)，随身听，和[高清电视](https://web.archive.org/web/20140904051107/http://crunchgear.com/?s=hdtv)之后，索尼没有比任天堂做得更好？我知道我们在谈论市场价值，但是拜托。此外，当事实不真实时，不要编造虚假的故事。看看这些数字。6.64 > 6.3。努夫说。
