# Veotag 的深度标记获得 75 万美元| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/05/16/veotags-deep-tagging-gets-750k/>

[![veotaglogo.png](img/c838131ea19e2546faca948d5888b5ab.png) ](https://web.archive.org/web/20161120071014/http://veotag.com/) [ Veotag ](https://web.archive.org/web/20161120071014/http://veotag.com/) veo 刚刚获得 75 万美元的天使投资。正如我们之前所报道的，深度标签是更长的在线媒体的一个重要特征。谁不想跳过主题演讲中无聊的部分呢？许多初创公司正在将深度标签整合到他们的服务中，如 Pluggd、 [Viddler](https://web.archive.org/web/20161120071014/http://www.techcrunch.com/2006/08/09/viddler-to-make-moments-in-video-searchable/) 、Motionbox 和 JumpCut 等。

Veotag 允许您将此功能添加到嵌入式播放器中的任何视频或音频中。你可以从你的电脑或通过网址上传内容(WMV，FLV，mp3)。任何人都可以注册一个帐户，上传内容，发布，并开始标记编程中的点。Veotag 播放器环绕内容，在右边显示带有时间戳的评论，链接到出版商选择的感兴趣的片段。你可以在这里看到盖伊·川崎[演讲中的播放器示例。](https://web.archive.org/web/20161120071014/http://www.veotag.com/player/?u=fqiufigrjh)

Veotag 有一个标签搜索引擎，它也被主要的搜索引擎编入索引。该索引然后可以被传送到广告服务引擎，用于围绕音频或视频的定向广告。

有兴趣将深度标签整合到他们自己的媒体中的读者也应该看看 [Click.tv 的](https://web.archive.org/web/20161120071014/http://www.techcrunch.com/2006/04/16/clicktv-moves-video-ideas-forward/)视频深度标签小工具和评论。
