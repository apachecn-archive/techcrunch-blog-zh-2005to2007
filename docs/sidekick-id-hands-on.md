# 助手 ID 动手–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/13/sidekick-id-hands-on/>

![img_6508.jpg](img/e4d32def79e4a07c4eee2330e0b83b5c.png)
死党 ID:别叫它东山再起

Sidekick 线是一个奇怪的动物。它既不是智能手机，也不是时尚手机，但它似乎非常不稳定地跨越了两者之间的界限，成为了每个人的一切。自从 SK3 问世以来，我一直在使用它，发现它是我用过的最好的信息手机，无一例外。Symbian，WinMo 和 Linux 都没有 Danger 的东西简单的操作系统和服务以及外形。

SK3 又大又丑，而且非常实用，但它几乎受到所有人群的欢迎。这是年轻人的黑莓手机和老年人的笨电子邮件手机。那么，什么是助手 ID 呢？除了几个明显的例外，这款精简的消息手机几乎做了 SK3 能做的一切，并且直接针对那些买不起或者不想要 SK3 的人。我确信这些观众是存在的，但是他们想要 SK ID 吗？

 ![img_6528.jpg](img/d116de447a8a6d9d5ab9b7b42497c819.png)
T-Mobile 把 SK ID 发给我，我立刻破解开 SK3 进行快速对比。正如所料，SK ID 没有摄像头或蓝牙。很公平。这款设备与 SK3 差不多大，屏幕更小，重量轻得多，看起来——但感觉不到——像一个塑料玩具，特别是安装了保险杠。这些保险杠最终将有多种颜色和风格，使它们更像是一个配件，而不是一个结构元件。今天晚些时候寻找各种保险杠的视频。

![img_6507.jpg](img/8e13ff3942bdf39c52e363bb333afb37.png)
中的方框。

SK ID 的两年合约价格为 99 美元，配有灰色外壳和保险杠。它的可拆卸盘子每套售价约 20 美元，将于 4 月 25 日在网上和零售商处发售。我一打开盒子，就被吸引去用一套蓝色盘子替换灰色盘子。虽然我最初怀疑这个功能的价值，但很明显每个人都想做同样的事情——这是 SK ID 魅力的一部分。就像过去的传呼机和诺基亚一样，这个东西正在乞求被塞进一个别致的盒子里或者用透明的塑料包裹起来。

按钮很有趣。SK ID 在手机左侧仍然有一个方向垫/听筒和一个小松鼠轮。所有其他按钮都完全被柔性塑料覆盖。屏幕大约 2.5 英寸宽，比 SK3 的小得多，而且非常暗。默认的主题，一个蓝色的风格和无处不在的 SK 男孩和女孩，看起来几乎过时了。很明显，夏普、Danger 和 T-Mo 对组成这款手机的单个硬件进行了一些改动，以使价格低于 99 美元。这并不意味着它廉价或脆弱——恰恰相反。这款 SK ID 实际上感觉比 SK3 更坚固，铰链也更好。

![img_6547.jpg](img/084f41c6b19ef6622adf5694459c1e10.png)
SUP LOL STFU

键盘几乎是一样的，只是键与键之间的空间大了一点，字体也更清晰了。通话和挂断按钮彼此有点接近，但除此之外，整个手机都是实心的。

![img_6532.jpg](img/4ccfaaf6aeaf3b2abd52f45dffb3cfb1.png)

如果你有一个 SK3，你不会想要 SK ID。不是给你的。这款手机售价 99 美元，面向的是那些可能会买 RAZR 的人，但他们会选择一款热门的小型信息手机。用户界面和 SK3 的完全一样。唯一真正的区别是没有摄像头或 MP3 播放选项。相反，Danger 增加了一个相册功能，让你可以将通过电子邮件发送的照片移到你的相册中。它也与 MyFaves 兼容。也没有存储卡插槽，也没有 USB 端口。有什么大不了的，对吧？相信我。我没有错过。

![img_6543.jpg](img/fc64107ee276d69db76102bf04b0fc68.png)
宝宝，妈妈

注意:我整个早上都在试图让 SK 服务在手机上弹出，但没有用(显然需要 2-48 小时)，但它可以与标准的 Danger Web 界面配合使用。大多数功能都被下载到网络上，所以 SK ID 几乎不保存任何联系人或帐户信息，而是从 Danger 的服务器上下载。

![img_6536.jpg](img/491a79cd75359ac400f5a09d71c9674d.png)

总的来说，我对 SK ID 并不失望，但也不惊讶。它介于摩托罗拉 RIZR 等具有有限消息功能的“标准”手机和 T-Mobile 及其他运营商提供的高端设备之间。没有什么比 Danger 的操作系统更容易开箱即用，即使是 Dash 也不例外。如果你不希望使用蓝牙或相机——我怀疑这将使超过驾驶年龄的人退出比赛——那么 SK ID 是一个很好的设备。那么，理想的受众是初中学生，对于这样一个年轻而多变的受众，我不难看出这款手机的价值。老年人，请远离。很快就会有适合你的手机，或者，如果你想方便地收发信息，就买一部 SK3 吧。如果 SK ID 是你的风格，我可以毫无保留的推荐。
