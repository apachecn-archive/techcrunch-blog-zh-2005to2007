# 西瓜 Zune 发现| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/21/watermelon-zune-spotted/>

![](img/9e4864662f0fa9fbd86e383ffb67cd19.png)

在“最近的新闻发布会”上，一名摄影师设法抓拍了一些传闻中西瓜色 Zune 的照片。虽然一个新的彩色 Zune 是令人兴奋的，因为燕麦麸皮出来在一个令人兴奋的新的棕色，什么是在里面是重要的事情要记住。

不幸的是，这只是同样的旧 Zune，在一个无聊，丑陋的新颜色。微软显然认为他们将通过不断发布新的颜色来作为主要玩家打入 DAP 市场。虽然它对苹果公司有效，但这是因为苹果公司实际上制造了一个不错的音频播放器。如果 Zune 不那么糟糕，也许人们会对新的颜色感到兴奋。不管怎样，西瓜版 Zune 售价 249 美元，预计今年夏天会降价。同样的无光泽的功能，同样的设计。继续前进，伙计们。
