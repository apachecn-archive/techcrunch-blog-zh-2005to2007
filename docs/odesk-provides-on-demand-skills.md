# oDesk 提供按需技能

[![oDesk](img/97f4f4d18dd7908b04f1e0a2d47e8f9f.png)](https://web.archive.org/web/20220929201514/http://www.odesk.com/) 我的初创公司 Omnidrive 在我们需要寻找合同 C++开发技能来完成一个项目时，首先找到了 [oDesk](https://web.archive.org/web/20220929201514/http://www.odesk.com/) 。乍一看，它就像许多其他帮助项目经理寻找人才的门户和市场之一，但仔细观察后发现，oDesk 具有独特的优势，使其成为寻找员工需求的最佳和最简单的方法之一。

经过两年的测试后，oDesk 于今年 5 月推出了一个合同人才市场。我们发现 oDesk 拥有大量(3600 个提供商)高技能人才，他们来自从文档编写、Java 开发到 C++开发的众多领域。寻找承包商的过程非常简单，在我们发布招聘广告后，几天内我们就收到了几十个回复，Omnidrive 在周末之前就让印度的一个高技能团队填补了这个职位。

供应商的高质量是 oDesk 高于竞争对手的地方，但是一旦你选择了合作的供应商，它就会大放异彩。oDesk 在供应商端使用网络摄像头和屏幕捕捉技术来监控工作进度。在线承包业务长期以来一直存在不信任问题，oDesk 的项目管理和监控技术无疑缓解了这一问题。我们能够使用网络摄像头观看并与我们的提供商交流，这巩固了仅持续几天的关系。

供应商主要来自印度、俄罗斯、乌克兰、美国、菲律宾和加拿大，该系统目前有 255 个职位空缺。oDesk 的独特之处还在于它不固定项目成本，而是按小时收费，这使得项目经理可以根据需要使用提供商。因为这是过程的一部分，oDesk 收取小时费用的佣金，目前每个月有数万个小时被收费。

oDesk 总部设在门洛帕克，他们已经从适马合伙人公司和环球投资公司筹集了 600 万美元。oDesk 吃自己的狗粮，因为他们 40 多人的团队中有 50%是 oDesk 提供商，他们使用自己的项目管理系统来构建平台。

oDesk 为寻找和管理熟练合同工提供了一个简单、方便和安全的选择。
