# Thanko Vonia 让你看起来像工具

Thanko 的 Vonia 头带的意图非常好，但是，说真的，你一定是在开玩笑。iPod Shuffle 并不比一本火柴大，显然，这对一些人来说太大了。如果你是其中之一，那么 Vonia 就很适合你，当然，如果你喜欢看起来像一个冲洗。你的 Shuffle 藏在发带的后面，插入一些老式的骨传导技术。唯一真正可取之处是能够在水下使用。来吧，看起来像埃尔冲洗。
