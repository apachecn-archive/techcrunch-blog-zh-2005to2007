# Xploder 即将登陆美国| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/09/xploder-is-coming-to-america/>

Blaze Games 今天宣布将在美国发布其 Xploder 高清电视包。该系统允许 PS2 用户将 PS2 的游戏进行向上转换，以便在 HDTV 上使用。

> Xploder HDTV Player 兼容超过 95%的 PS2 游戏，是一款独特的软件实用程序，可以释放 PS2 游戏固有的高清功能，为游戏玩家提供 480p、576p、720p 和 1080i 高清模式。在加载游戏之前，只需加载光盘，从简单的屏幕界面中选择所需的设置，并定位屏幕图像以获得最佳播放效果。

它将于本月晚些时候上市，售价为 49.99 美元，并配有一根高质量的 RGB 分量电缆，可将 PS2 直接连接到高清电视。
