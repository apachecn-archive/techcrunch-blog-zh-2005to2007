# USB 闪存盘，适用于旧的 MAC 电脑| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/07/twelve-gigabyte-usb-flashdrive-for-old-old-macs/>

 Pocketec 发布了一款一英寸大小的便携式 12GB USB 驱动器。这本身有点诱惑，但真正奇怪的是 AVING 上列出的统计数据:通过 Vista 与 Windows 98 兼容，以及 Mac OS 8.6。

等等，什么？是的，我们也不明白，但我们确实喜欢 USB 插头的弹簧刀动作和 1.7 盎司的重量。它看起来也构造得相当好，所以它不会像我拥有的其他该死的袖珍驱动器一样在你的口袋里粉碎。

我们预计它会在今年晚些时候上市，很可能会以 200 美元左右的价格作为其他品牌的 OEM。
