# Neoware 瘦客户机笔记本电脑:什么硬盘？

硬盘是给笨蛋用的。好吧，也许不是，但如果你的笔记本电脑被盗或丢失，它们很容易受到损害。虽然对于我们大多数人来说，一些小偷发现我们的“ScarJo_Naked_mov.avi”文件没什么大不了的，但这是大公司赖以生存的东西。含有敏感数据的笔记本电脑丢失的新闻时有发生，像往常一样，你不会想成为新闻的主角。

瘦客户机公司 Neoware 开发了一系列基于 Citrix 基础设施的瘦客户机笔记本电脑。运行 Linux 或 Windows XPe 的笔记本电脑不仅仅是终端。这款笔记本电脑拥有 15 英寸的屏幕，最大尺寸为 1600×1200，配有立体声扬声器、USB、调制解调器、VGA 输出、LAN 和 WiFi，其他笔记本电脑都有。

此外，他们专有的安全软件使得获取你的数据几乎是不可能的。如果不让别人知道你的情报很重要，那就考虑瘦。
