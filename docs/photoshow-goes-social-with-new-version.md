# 新版图片秀走向社交

上周五，SimpleStar 公司发布了其 [PhotoShow](https://web.archive.org/web/20210302060552/http://photoshow.com/) 产品的新版本，并将这一广受欢迎的照片和视频分享服务与早期采用者开创的许多发展同步，你很可能已经在 TechCrunch 上读到过这些。该服务现在支持社交分享和在线分类，这是其他媒体分享服务的典型特征；这个变化堪比[什么雅虎！去年夏天，当它从一个很大程度上的私人服务转变为一个更受 Flickr 启发的现象时，Photos 做到了。Photoshow 将在线共享与管理媒体的桌面应用程序结合在一起。](https://web.archive.org/web/20210302060552/http://www.beta.techcrunch.com/2006/08/17/yahoo-photos-to-leave-beta-today-api-finally-coming/)

不过，SimpleStar 最值得注意的一点是该公司在业务发展方面令人难以置信的敏锐。该公司报告称，由于与康卡斯特(Comcast)、沃尔格林(Walgreen's)和沃尔夫相机(Wolf Camera)等公司的合作，其桌面软件每天的新安装量达到惊人的 20，000 次。一年半前，SimpleStar 从 Venrock 获得了 600 万美元的资金。

PhotoShow 有一种特殊的味道，包括产品的价格(40 美元)，Windows 专用的桌面应用程序，与大型有线电视公司和互联网服务提供商合作，以及网站上到处都是装在纸板盒里的软件照片，尽管这些照片主要是通过下载获得的。这种特别的味道让我想对这家公司嗤之以鼻——但事实是，他们知道如何完成工作。

PhotoShow 将最初的产品货币化，这是一个令人费解的升级，售价 20 美元，显然是由许多图片主题组成的(“孩子、爱和爱国主题包”等)。)、你的媒体的 DVD 以及任何它能想到的向用户收费的东西。

SimpleStar 表示，其照片展示服务吸引的不是早期用户，而是最后 2/3 的用户。到目前为止，它似乎成功地拓展了这个市场；虽然 Alexa 确实指出不起眼的 Photoblog.com 获得的流量是 PhotoShow.com 的 4 倍，但这可能不太相关，因为 PhotoShow 对桌面软件的依赖。

我个人觉得这种服务居高临下，令人讨厌，但也许这比照片秀更能说明我的情况。毫无疑问，该公司值得其大型合作伙伴的认可，并决定通过此次最新升级为该服务添加更多社交功能。这可能是许多人推荐的策略的一个主要例子——专注于赚钱，让早期采用者承担最大的风险，并在以后实施符合市场的策略。
