# Vaja 发布新的 iPod Nano 保护套

![](img/9ca810bcc950f7c56ee8f8bf8f6cfd69.png)

所以你得到了一个彩色的 2G iPod Nano，但需要一个外壳？不要害怕。Vaja 刚刚宣布了 Nano 的四个新案例，老实说，它们看起来相当粗糙。有 34 美元的经典款(如上图)，40 美元的经典款上衣，30 美元的 I-Volution 皮套，45 美元的带挂钩的 I-Volution。所有箱子都可以根据您的喜好定制，有 1000 多种不同组合可供选择。我最喜欢的是带钩子的 I-Volution，因为在我的钥匙链上有一个 iPod Nano 几乎是有史以来最甜蜜的事情。
