# 时代华纳有线电视将扩大视频点播服务

![](img/7f42764ad554a70844d5f9049bc5866c.png)

自从我们的新公寓安装了时代华纳有线电视，我就不再是这项服务的狂热粉丝了。我确实喜欢快速视频点播功能，这让我可以在任何时候观看“康乔德的飞行”，但是看起来 TWC 将改善点播服务。一项名为“重新开始”的新功能可能会在年底推出，它将允许用户重新开始他们错过的电视节目。然而，你将无法跳过广告，所以这项服务的风险适得其反，那些 DVR 跳过杂乱。
