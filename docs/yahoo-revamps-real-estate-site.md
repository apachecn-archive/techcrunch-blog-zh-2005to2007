# 雅虎改版房地产网站

[![](img/03047b8b7ee9a922d78ad430bf41838d.png)](https://web.archive.org/web/20220629012317/http://realestate.yahoo.com/) 雅虎今天公布了其新的[房地产](https://web.archive.org/web/20220629012317/http://realestate.yahoo.com/)网站。该网站现在整合了一个交互式地图，显示按位置、价格、床位和浴室数量过滤的房价数据。它从 MLS 列表、分类、止赎和出租中提取数据。雅虎还通过使用他们的技术为雅虎地图的家庭价值混搭提供动力，将[与](https://web.archive.org/web/20220629012317/http://www.zillowblog.com/zillow_blog/2006/08/expanding_our_p.html)高调和[大量资助的](https://web.archive.org/web/20220629012317/http://www.beta.techcrunch.com/2006/07/24/zillow-raises-another-25-million/)合作伙伴 Zillow 完全整合在一起。新网站将使无数竞争对手的房地产网站的日子变得更加艰难，包括新来者，如 [Redfin](https://web.archive.org/web/20220629012317/https://beta.techcrunch.com/tag/redfin) 、 [Trulia](https://web.archive.org/web/20220629012317/https://beta.techcrunch.com/tag/trulia) 、 [Movoto](https://web.archive.org/web/20220629012317/http://www.movoto.com/) 和[RealEstateABC.com](https://web.archive.org/web/20220629012317/http://www.beta.techcrunch.com/tag/Real-Estate-ABC/)。

通过重新设计，雅虎支持寻找和列出一所房子，研究该地区(学校和社区)，寻找房地产经纪人，获得抵押贷款(当地利率)，以及评估你的房子。这使得网站比 Trulia 提供的价格列表更全面，但没有像 Redfin 和 Movoto 那样作为经纪人介入(注意，后两种服务仅限于少数几个地理位置)。
