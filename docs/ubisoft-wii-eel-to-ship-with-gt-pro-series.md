# Ubisoft Wii-eel 将随 GT Pro 系列一起发货

![](img/7d1a6475e4fc757842ec90916d8972cc.png)

这是几天前的事了，但仍然是件大事。你面前看到的是育碧的 Wii 驾驶游戏配件。它是由 Thrustmaster 制造的，在概念上是一个奇特的设计。将 Wii-mote 插入 Wii-eel，踢回你的懒小子，尽情比赛。

但是我觉得这个很酷的想法没什么问题。首先，我是一个软弱的嬉皮士。尤其是我的和平主义手臂。当我不得不一直张开双臂的时候，我怎么能赢得比赛呢？第二，我该如何直行？如果没有底座来稳定轮子，我会不会移动得太多来保持控制呢？也许我错了，但这东西看起来像是那些只在理论上是好的想法之一。该车轮将与 GT Pro 系列一起发运，该系列预计将于今年假期发布。有人要吗？
