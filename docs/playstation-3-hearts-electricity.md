# PlayStation 3 心电

![](img/ec5a3afd8cbefb1ea7c31d7fb32de474.png)

PlayStation 3 将需要总共 380 瓦的电力才能运行，这是 PlayStation 2 所需电力(45 瓦)的八倍多，是微软 Xbox 360 (160 瓦)电力需求的两倍多。这是一台耗电机器。如果你一年内每天玩 PS3 一个小时，大约要花 20 美元(感谢 Gizmodo)。
