# CafePress 为什么不用闪光灯？

有许多网站出售定制的赌场筹码(只需搜索一下)，但我们没有发现任何一个网站拥有在线设计工具，允许你自己制作筹码。[the chi lab](https://web.archive.org/web/20220810103820/https://www.thechiplab.com/)采取了额外的步骤，创造了一个非常好的 Flash 工具，用于设计你自己的赌场级芯片。创建工具是非常灵活和基于层的。它没有新的在线图像编辑器 [Fauxto](https://web.archive.org/web/20220810103820/http://www.fauxto.com/) 那么强大，后者也是基于 Flash 的，但它完成了工作。

所以我的问题是，为什么 [CafePress](https://web.archive.org/web/20220810103820/http://www.cafepress.com/) 和 [Zazzle](https://web.archive.org/web/20220810103820/http://www.zazzle.com/) 不实现 Flash 工具来帮助人们创建定制产品呢？Zazzle 的 Ajax 工具相当不错，允许一些基本的图像大小调整和移动，以及添加文本。Imagecafe 停滞在 90 年代——他们的产品创建工具几乎没有任何灵活性。两者都没有提供该实验室的功能。

Adobe 已经开发出了令人敬畏的工具，像 TheChipLab 这样的小型电子商务公司可以用它们来为客户提供超酷的服务。是时候让 CafePress 和 Zazzle 的产品副总裁们清醒过来，重新发明他们的产品了。Adobe 已经完成了所有繁重的工作，他们所要做的就是实现它。
