# WebShell:面向管理员的 Ajax 应用程序

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/29/webshell-an-ajax-app-for-admins/>

这是一个有趣的小家伙。这是一个用 Ajax 编写的用于 iPhone 的 SSH 应用程序，可以下载到您喜欢的服务器上。当我还是一名系统管理员时，我总是为我的便携式设备寻找 SSH 实现。没有一个通常是一个交易破坏者。我不认为这是一个理想的解决方案——它可能充满了不安全感——但这是我最近见过的最酷的 [iPhone 应用](https://web.archive.org/web/20140602075645/http://crunchgear.com/2007/06/16/latest-iphone-app-helps-you-find-gas/)。
