# PocketMod 类型的东西并打印出来| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/09/04/pocketmod-type-stuff-and-print-it/>

我本来不打算写关于 [PocketMod](https://web.archive.org/web/20230307054801/http://www.pocketmod.com/) 的文章，我知道人们会对我说些废话，但我不断看到关于它的帖子，所以我最终尝试了一下。

除了正确折叠纸张(见下面的截图)比打领结更难之外，还有一两件我喜欢的事情。但是我想得太多了。

对于那些不知道的人来说，PocketMod 是一个 flash 程序，它允许你在一张纸上打印模块，这样你就可以参考它并在上面做笔记。有日历，待办事项，井字游戏等等。你可以在一页上放置多达八个模块，它可以折叠成一本漂亮的小书。

为了保持条理，我白天会写很多实际的东西。在任何时候，我的口袋里都会有 3-4 张纸，上面记录着产品创意、要介绍的新公司等。所以我想我可能会开始用这个。尤其是现在我投入了这么多时间学习如何正确折叠。

关于 PocketMod 没有太多其他的可说了。我把折叠说明放在下面，只是因为我觉得你必须剪切和折叠这该死的东西，这太搞笑了。这应该是一个关于 web 2.0 的博客。:-)
