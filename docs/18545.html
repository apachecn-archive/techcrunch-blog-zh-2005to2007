更新提高了Windows Vista的可靠性


形象从戴夫那里偷来的

…它叫做OS X. HAR！HAR！继续摇滚你的Vista自动更新或点击下面的链接，因为Vista今天会少吸一点。更新改善了内存泄漏和冻结，并加快了一些小事的速度——如复制文件时剩余的估计读取时间——以确保您的Vista体验更加充满“哇”而不是“现在到底出了什么问题？”一些修复后跳转。

*如果计算机上禁用了用户帐户控制，您将无法成功安装网络打印机。如果网络打印机由基于Windows XP或基于Windows Server 2003的计算机托管，就会出现此问题。
*计算机从休眠状态恢复后，需要很长时间才能显示登录屏幕。
*当您将脱机文件同步到服务器时，脱机文件会损坏。
