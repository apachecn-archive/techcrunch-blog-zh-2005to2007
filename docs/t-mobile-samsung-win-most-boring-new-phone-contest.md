# T-Mobile，三星赢得最无聊新手机竞赛

我们不理解 T-Mobile 的新条款。三星(Samsung)的 myFaves 兼容翻盖手机确实很弯曲，但与该制造商为 T-Mo 提供的其他产品相比，体积较大，而且价格更贵。功能方面，它也很差，有 GPRS(不支持 EDGE)，标准 VGA 摄像头(这里没有百万像素)，和单色外部显示器。我们喜欢三星和 T-Mobile，因为他们在良好的语音网络上制造了坚实、有吸引力的手机，并提供了出色的客户服务，这就是为什么这款不优雅的手机令人失望的原因。

两年计划售价 50 美元的相机手机[无法与三星 t619](https://web.archive.org/web/20201022065633/http://crunchgear.com/2006/09/20/t-mo-drops-two-new-sexy-sexy-samsungs/) 相比，后者的外形更纤薄，130 万像素的拍照器，而且同样的计划是免费的。我们知道条纹是新的，T-Mobile，但这不完全是一件好事。我们唯一能想到的让这款新翻盖手机与众不同的是它是否搭载了升级版的三星操作系统，但我们甚至不确定它是否搭载了。下次你们对我们 T-Mobile 和三星这样做的时候，把手机命名为“Meh”。它现已发货，可在您值得信赖的当地 T-Mobile 商店购买。
