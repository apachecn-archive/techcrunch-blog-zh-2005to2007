# 任天堂 Wii 只有白色

。

只有一种颜色的 Wii 可用，索尼是唯一一家将拥有黑色、银色和白色多色游戏机的公司。虽然任天堂只是确认了一种颜色，但我们怀疑各种不同颜色的 Wii 版本将在发布后的 18 个月内发布，巧合的是，这也将比第一个 Wii 版本小，但存在铰链开裂的问题。它将被称为 Wii 不那么大。
