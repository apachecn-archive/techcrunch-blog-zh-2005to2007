# 专利猴:阿尔卡特以 15 亿美元战胜微软的细节

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/24/patent-monkey-details-on-alcatels-15-b-win-against-microsoft/>

![](img/af35047d018a8528fa4c9cd55a1d2b51.png)

以下是历史上最大的专利诉讼和解的一些补充细节:微软因其 Windows Media Player 侵犯了法国电信的 MP3 技术而被判向阿尔卡特支付 15 亿美元。陪审团发现微软侵犯了 [RE39，080“感知编码器/解码器速率环路处理器”](https://web.archive.org/web/20160317215711/http://www.patentmonkey.com/PM/PatentID/re39080.aspx)专利和 [US 5，341，457“音频信号感知编码”](https://web.archive.org/web/20160317215711/http://www.patentmonkey.com/PM/PatentID/5341457.aspx)专利。陪审团为 Windows Media Players 版本的销售设定了侵权的两项专利各 0.5%的版税(总计 1.0%)。如果你问自己这个案子到底是关于什么的，请继续阅读…

阿尔卡特获得了有史以来最高的专利侵权奖项，该奖项是基于微软对阿尔卡特专利 MP3 技术的使用。15 亿美元的和解金额远高于上世纪 90 年代初柯达因侵犯即时相机相关专利而被判向宝丽来支付的 9 . 09 亿美元。

该案件最初提交给弗吉尼亚州的联邦法院，朗讯与 Gateway 和戴尔就颜色记忆、视频搜索功能和用手写笔控制电脑的相关专利展开了竞争。微软加入了该案件，因为它有义务赔偿和补偿网关和戴尔。随后，朗讯、戴尔和微软在多个司法管辖区提起了各种其他诉讼，最终导致这些案件被合并到一个司法管辖区——加利福尼亚州南部地区法院。布鲁斯特法官然后根据技术将所有合并的诉讼分成几个案件(都在他的管辖范围内)。关于 MP3 专利技术的案子是第一个判决的。阿尔卡特去年收购了朗讯，引发了阿尔卡特诉微软的纠纷。

虽然很难追查，这里有一个[文件夹，里面是阿尔卡特和微软专利](https://web.archive.org/web/20160317215711/http://www.patentmonkey.com/PM/FolderID/203437fa7d934f5fa0b730f68ff1d195.aspx)相互之间的各种纠纷。在这个具体的案例中，陪审团发现 Windows Media Player 侵犯了' 080 专利的权利要求 1、3 和 4 以及' 457 专利的权利要求 1、5 和 10。如果你想知道，我们也是，这里有几个典型的声明…
