# WeatherBill 推出，宣布所有明星投资者

[![](img/c0bf26c9d3cffe036aa57926cac3d4a9.png)](https://web.archive.org/web/20221220164031/http://www.weatherbill.com/) 总部位于三藩市的 [WeatherBill](https://web.archive.org/web/20221220164031/http://www.crunchbase.com/company/weatherbill) ，让你对天气下注的服务(它也可以用于保险)，拿下了他们的登陆页面，并于今天推出。

两周前，我们写了关于这家公司的文章，当时它还处于私下测试阶段。WeatherBill 将电子商务网站与复杂的天气预报算法相结合，向个人和企业出售天气保险。

该公司没有透露他们这轮融资的规模。然而，它包括风险投资公司 NEA 和 Index Ventures，以及一些知名人士:del.icio.us 创始人约书亚·沙赫特，Skype 创始人尼可拉斯·曾斯特罗姆(通过他的 [Atomico](https://web.archive.org/web/20221220164031/http://atomicoinvestments.com/) 基金)和霍华德·摩根(idealab 和首轮资本)，以及其他拒绝透露姓名的人。

WeatherBill 非常适合业务受天气影响的中小型企业。如果一个高尔夫球场希望在每个雨天获得 1000 美元的报酬，WeatherBill 将为他们创建一个动态策略。

该公司由前谷歌员工大卫·弗里德伯格创建。

另外，WeatherBill 正在用 Flash 做一些非常好的事情(参见[工具](https://web.archive.org/web/20221220164031/https://www.weatherbill.com/tools)部分)。即使天气和保险不是你喜欢的，你也应该去看看。特别是如果你自己正在创建 Flash 应用程序的话(我在三月份对 Jobby 使用 Ajax 说过同样的话)。
