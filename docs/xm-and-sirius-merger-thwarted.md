# XM 和 Sirius 合并受阻

我和其他人一样支持避免垄断，但这一点对许多在这个问题上比我了解得多的人来说是有意义的。 [XM 和 Sirius 一直在探索合并](https://web.archive.org/web/20130628143149/http://crunchgear.com/2006/11/28/xm-and-sirius-in-talks-for-merger/)的可能性，但是根据目前 FCC 的规则，复数公司不能成为单一公司。这使得情况处于停滞状态，直到，也许，当前的规则手册被修订，以弥补合并。

这两个提供商的结合将非常有意义，因为它将允许双方联合努力和资源，从而降低运营卫星的巨大成本，然后，当然，将好处传递给你，有良知的消费者。他们将能够摆脱多余的程序，提供更专业的材料，还可能更快地开发新的服务和技术，而不是把大量的精力放在竞争上。

[XM + Sirius =哦等等，这不会发生的](https://web.archive.org/web/20130628143149/http://gizmodo.com/gadgets/xm/xm--sirius--oh-wait-its-not-going-to-happen-229521.php)
