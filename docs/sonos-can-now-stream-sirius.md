# Sonos 现在可以播放天狼星了

![](img/8b895d44030750a2d9c35309cb604c5d.png)

Sonos 是一家人人喜爱的在家中播放音乐的公司，它宣布与 Sirius 合作。交易:从今天开始，如果你有 Sonos 的 Zoneplayer 硬件，你可以下载软件，让你播放 80 个频道的卫星节目——包括霍华德·斯特恩和 NFL。它将免费 30 天，之后 Sirius 的新用户每月收费 12.95 美元，如果你已经是订户，则收费 2.99 美元。

当然，Sonos 已经从 Rhapsody 和 Pandora 获得了流媒体内容，这意味着这只是他们不断增长的联盟网络中的又一步，这可能使他们成为流媒体服务之父。对于一家甚至没有接近 CE 巨头的知名度或规模的公司来说，这并不坏。目前还不知道 XM-Sirius 的合并会对这笔交易产生什么影响。

同样注意:Sirius 将在今天晚些时候宣布一系列新的接收器，请继续关注即将发布的内容。
