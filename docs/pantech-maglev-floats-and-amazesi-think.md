# 泛泰磁悬浮浮动和惊奇…我认为

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/15/pantech-maglev-floats-and-amazesi-think/>

在最近急需的手机减肥浪潮中。泛泰希望凭借其采用磁悬浮技术的最新手机脱颖而出。磁悬浮，就像它的名字一样，基本上允许一个物体悬停在另一个物体上方，没有任何物理接触，并通过磁场保持在适当的位置。大多数滑盖手机都设计有弹簧，这让它们变得又肥又丑……我猜。

总之，泛泰很瘦，平均只有 9.9 毫米，击败了三星的厌食症[超版本](https://web.archive.org/web/20161104033920/http://crunchgear.com/2007/02/26/samsung-continues-to-slim-down/)。它配有 130 万像素的摄像头、MP3 播放器和 T-DMB。泛泰计划下个月在韩国推出这款手机，但是还没有关于价格的官方消息。如果在韩国表现良好，泛泰计划将其推向海外。耶！
