[![zenzuilogo.png](img/63bf8d17de5e10ce87691a0b8d98baa1.png)](https://web.archive.org/web/20211027134602/http://zenzui.com/) 微软支持的 [ZenZui](https://web.archive.org/web/20211027134602/http://zenzui.com/) 今天上午推出。这是一款新的移动浏览器，旨在通过小工具让你的移动设备更容易上网。该应用程序允许您通过平移和缩放网格来浏览 6×6 网格的网站小部件。每个小工具都是一个网站的移动优化版本，你可以通过放大它来进行交互。

从宣传视频的外观来看(见帖子顶部)，你可以放大到一个小部件，也可以放大到一个 4×4 的小部件网格。围绕网格的缩放和平移由你的触摸屏(如果你很幸运的话)或数字键盘(对于我们其他人来说)控制。虽然它目前只能在 Windows Mobile 设备上使用，但他们的目标是在今年晚些时候推出 Java 2ME 和 BREW 版本。

ZenZui 的缩放用户界面最初是由微软的雷德蒙研究实验室开发的，后来被剥离出来，成立了自己的公司。该公司与微软 IP Ventures 合作，从 SeaPoint Ventures 和其他投资者那里获得了 1200 万美元的首轮融资，微软保留了该公司的股份。

整个服务是一个聪明的方式来增加甲板上移动电话的位置。Deck placement 是移动电话上的一组默认应用程序。这是你的手机上最令人垂涎的空间，通过甲板上的位置进行分发决定了大多数移动应用程序的成败。目前很难获得甲板位置，因为它是由最高层的运营商决定的，并且通常包括一些收入分成。ZenZui 应用程序实际上将一个点变成了 36 个(他们希望变成 1000 个)。

如果 ZenZui 本质上不做运营商现在正在做的事情，对位置收费，这一切都很好。你看，ZenZui 并不是一种新的手机上网方式，而是一种新的手机上网方式。这一切都很好地总结在[这张图](https://web.archive.org/web/20211027134602/http://www.zenzui.com/images/contentProductsDiagram.png)显示了营销人员和开发者将内容输送到你的手机。

ZenZui 目前包含的 36 个景点都由内容合作伙伴提供支持，如 Zillow、Eventful、Kayak、OTOlabs、Avenue A、Razorfish 和 Traffic.com。他们将利用“公认的广告原则，如 CPA 和 CPM(我们称之为 CPZ-每变焦成本)”来盈利。为了鼓励开发，ZenZui 将与 widget 开发者分享广告收入。小工具开发者也可以选择发布一个无广告版本，它将显示来自非营利组织的信息。

ZenZui 是朝着正确方向迈出的一步。它从运营商那里移除了更多对手机“不动产”的控制，但是它仍然在这个过程中插入了不必要的成本，广告。我们相信，移动电话平台应该朝着开放标准的方向发展，这已经使桌面计算平台受益匪浅。

[Blupulse](https://web.archive.org/web/20211027134602/http://www.beta.techcrunch.com/2006/12/04/bluepulse-20-does-not-disappoint/) 是另一个 widget 平台(很可能会支持广告)。更开放的移动窗口小部件平台是诺基亚的 [Widset](https://web.archive.org/web/20211027134602/http://mobilecrunch.com/2006/06/12/monday-morning-review-new-widset-app-from-nokia-what-tweens-want-in-mobile-mobiles-for-sight-impaired-more/) ，和 [Opera](https://web.archive.org/web/20211027134602/http://www.opera.com/pressreleases/en/2006/02/14/2/) 。这些平台面临着第一次跳到人们的手机上的同样障碍，但随着它们更容易与我们的桌面连接，这将变得越来越容易。
