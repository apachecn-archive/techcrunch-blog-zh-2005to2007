# 终极游戏椅| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/05/ultimate-game-chair/>

![](img/65bf388bf95ead58e40af831616e48d0.png)

这是南方公园的男孩们昨晚可以用来参加他们的七周哇马拉松的东西。终极游戏椅 12 的振动马达与游戏的隆隆声同步。它还集成了 3D 立体声扬声器和地面效果照明。它将于本月晚些时候上市，售价 399 美元。

对于那些不需要乙烯基皮革或踢脚线的人来说，没有这些功能的猛禽型号售价 299 美元。
