# Plantronics 发布夏末耳机

Plantronics 有两个，数一数，两个新耳机——一个消除噪音，另一个保持噪音。

Voyager 520 是一款降噪蓝牙耳机，“具有强大的降噪技术，可提供商业品质的性能”。

它将花费你 99.99 美元，将于 8 月上市。

对于行动不便(但仍相当灵活)的人来说，还有隔音功能。音频 480 虚拟电话亭耳机“笔记本电脑多媒体和互联网通话。”这是目前 Plantronics 推出的最小的电脑耳机。

模拟版售价 89.95 美元，USB 版售价 109.95 美元——也将于 8 月上市。

我有一个来自 Plantronics 的 Gamecom Pro1 耳机，它真的非常非常好。加州圣克鲁斯的好东西。
