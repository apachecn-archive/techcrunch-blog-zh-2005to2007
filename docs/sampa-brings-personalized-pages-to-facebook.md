# Sampa 将个性化页面带到脸书

[![sampalogo.png](img/55382f475f780dc362d2225138481d9b.png) ](https://web.archive.org/web/20220516082156/http://www.crunchbase.com/company/sampa) [Sampa](https://web.archive.org/web/20220516082156/http://www.crunchbase.com/company/sampa) 是一个个人网站创建工具，让你定制自己的免费托管网站。与 [Weebly](https://web.archive.org/web/20220516082156/http://www.crunchbase.com/company/Weebly) 和 [Synthasite](https://web.archive.org/web/20220516082156/http://www.crunchbase.com/company/Synthasite) 不同，Sampa 更注重内容，而不是布局。网站创建的重点是通过模块向网站添加内容。他们最近以每月 4000 美元的烧钱率，上个月在他们的网站上获得了 100 万的页面访问量，并在今天将他们的服务添加到了脸书。

![sampasmall.png](img/ad3bb2eaf647a9c8ebaf84f478be038e.png) Sampa 站点围绕模块展开。您可以为博客、照片库、静态页面、YouTube、Flickr、Twango、Amazon、Delicious、Blue Dot 和 Blogger 添加模块。当您添加一个模块时，它的链接会被添加到顶部的菜单栏中。您可以限制任何人、朋友或不同的 Sampa 成员访问您的任何内容。页面的布局是基本的，允许您在模块间移动和调整列数。可以通过应用图库中的任何模板来更改设计。

除了脸书，你可以通过脸书创建和管理你的网页。您可以通过脸书画布页查看您和您的脸书朋友的页面。Sampa 还在你的个人资料页面上添加了一个迷你版的网站。

其他网站个性化服务可能会进入脸书，但考虑到该网站严格的斯巴达式设计原则，这将是一场艰难的战斗。
