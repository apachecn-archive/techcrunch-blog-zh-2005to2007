# PS3 易趣被骗| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/20/ps3-ebayers-get-screwed/>

![](img/030f0446f9d6101b55676d52c5b5fbca.png)

为了在发布日买到 PlayStation 3，我决定从周三早上开始在百思买(Best Buy)前露营。一点都不好玩。

接下来的两天充满了争斗、争吵和警察让我们离开。在一次调查中，我决定询问前 28 个人(只有 28 台游戏机可用)是否打算出售或保留他们的 PS3。在这 28 人中，有 25 人计划通过 Ebay 或 Craigslist 出售他们的游戏机。我的一个朋友(他在那里大便和傻笑)实际上以 750 美元的价格把他的位置卖给了易趣网。起初，我以为我选了一家糟糕的商店，但事实证明，几乎所有人、所有地方的人都想从 PlayStation 3 上赚钱。

11 月 17 日，PS3 游戏机在 Ebay 上的售价为 2k-3k。从周六开始，游戏机的平均价格出现了毁灭性的下降，游戏机开始以略高于 1k 的价格出售。Kotaku 还获得了 Ebay 页面的屏幕截图，以显示 PS3s 的价格下降。新的易趣 PS3 平均价格对游戏玩家来说很棒，但对那些露营 3 天试图赚点外快的易趣玩家来说肯定很糟糕。笨蛋。
