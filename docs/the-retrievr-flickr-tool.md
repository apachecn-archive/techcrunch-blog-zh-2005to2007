# Retrievr Flickr 工具——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/01/03/the-retrievr-flickr-tool/>

我不确定 [Retrievr](https://web.archive.org/web/20210928155850/http://labs.systemone.at/retrievr/) 是否有任何商业应用，但它确实是一项漂亮的技术。

Retrievr 在网站中内置了一个 Flash 草图板。画点东西——任何东西——它都会得到相似的 Flickr 图片。我非常粗略地画了一条与红色博客相交的黑线，得到了一些不错的结果(见图片)。
