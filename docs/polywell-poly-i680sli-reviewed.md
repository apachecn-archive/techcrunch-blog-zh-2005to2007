# Polywell Poly i680SLI 评论| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/10/polywell-poly-i680sli-reviewed/>

它们的特点是多重显卡和处理器都是在地狱里锻造的。也就是说，遇见魔鬼的右手。

Polywell i680SLI 由一家更知名的 tweaker 个人电脑制造商制造，拥有四核 CPU 和 10，000 rpm 驱动器，运行最新的 nVidia GeForce 8800 tx T2 显卡。

虽然这台电脑肯定是针对游戏爱好者的，但它仍然需要一点超频和按摩来击败更老的高端电脑，这正是粉丝们喜欢听到的。
