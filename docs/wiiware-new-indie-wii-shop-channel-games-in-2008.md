# Wii ware:2008 年新的独立 Wii 商店频道游戏

![Wii](img/4fff614191f5aee75d7c346a971f1a60.png)
任天堂刚刚公布了其新的 WiiWare 系统，该系统将通过 Wii 商店渠道提供独立发行商的游戏下载。任天堂的美国总裁雷吉·菲尔斯-艾梅说:

> “拥有小预算和大创意的独立开发者将能够把他们的原创游戏推向市场，看看我们是否能找到下一个轰动的游戏。WiiWare 为越来越多的 Wii 用户带来了新的创造力和价值。”

这应该是一个很好的交易。开发者有机会提供没有库存风险的游戏，游戏玩家也有机会接触到新的廉价游戏。此外，它将完善目前在 Wii 商店频道上发现的有些不足的产品。

新内容将于 2008 年 1 月开始提供。
