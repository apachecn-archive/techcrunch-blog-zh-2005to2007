# 将推出 page flakes 2.0 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/10/03/pageflakes-20-to-launch/>

 [](https://web.archive.org/web/20221129050830/http://www.pageflakes.com/) [Pageflakes](https://web.archive.org/web/20221129050830/http://www.pageflakes.com/) ，一家总部位于德国的 Ajax 主页服务，准备在未来一两天内推出一项重大的用户界面变更。我就新品发布采访了联合创始人兼首席执行官 Christoph Janz 和首席设计师 Jeremy Baines 你可以在 [TalkCrunch](https://web.archive.org/web/20221129050830/http://www.talkcrunch.com/2006/10/03/episode-14-interview-with-pageflakes-founder/) 收听播客。

此次发布的很大一部分将是对 Pageflake 最近增加的“[发布](https://web.archive.org/web/20221129050830/http://www.pageflakes.com/sharinghelp.html)”功能的推广，用户可以在这里创建包含联系信息、待办事项、家庭照片等内容的页面。或者公开发布内容，或者与几个朋友分享。该网站也正在完全重新设计(见下面的公司提供的屏幕截图-抱歉的玩笑)。

与主要竞争对手 [Netvibes](https://web.archive.org/web/20221129050830/http://www.netvibes.com/) 相比，Pageflakes 在讨论数字时采取了低调的方式。虽然 Netvibes(几天前推出了他们自己的[重新设计](https://web.archive.org/web/20221129050830/http://www.beta.techcrunch.com/2006/09/30/new-look-for-netvibes/))公布了他们 1500 万美元的风险投资以及用户增长(先是[100 万](https://web.archive.org/web/20221129050830/http://www.beta.techcrunch.com/2006/03/22/netvibes-to-announce-seed-financing-today/)，然后最近是[500 万](https://web.archive.org/web/20221129050830/http://software.gigaom.com/2006/09/15/netvibes-five-million-users-counting/))，但 Pageflakes 没有公布用户数量或他们的基准融资规模。

不管这些公司的营销方法不同，它们都在建立一个庞大而有价值的用户群，并且都有足够的中短期资金。这很好，因为两者都还没有产生任何收入。鉴于这些和其他新的网络创业公司的极低烧钱率，他们还需要一段时间来找出从观众身上赚钱的最佳方式。
