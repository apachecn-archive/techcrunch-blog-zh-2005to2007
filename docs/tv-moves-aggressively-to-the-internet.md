# 电视大举进军互联网

新闻集团旗下的福克斯电视网与其附属公司签署了一份为期六年的协议，允许其在互联网上重播电视节目。

美国广播公司、哥伦比亚广播公司和全国广播公司都在做类似的实验，iTunes 也有许多[电视节目](https://web.archive.org/web/20220521070700/http://flickr.com/photos/michaelarrington/128088840/)可供下载。这个领域正在快速发展。例如，在一项新的研究发表后，人们在录制的节目中跳过广告(惊讶吧！).

这是我们正在关注的一个领域。期待即将到来的对互联网上电视节目和电影市场现状的评论。我们将从人们已经在使用的“免费”非法服务的角度来看待这些问题。
