Xbox 360更新

正如我们昨天提到的，一个备受期待的Xbox 360更新计划于今天发布。正如承诺的那样，用户可以在今天早上登录并下载秋季更新。新版本增加了:

新功能

  • 通过VGA和分量电缆支持高清1080p视频模式。
  • Xbox 360 HD DVD玩家支持。
  • 从运行Windows Media Player 11、Zune软件或Windows Media Connect的Windows PC上流式传输WMV视频。
  • 播放来自USB闪存驱动器、Xbox 360存储设备等存储设备的视频。
  • 播放CD或DVD数据光盘上的视频。
  • Xbox 360无线耳机支持，包括Xbox指南中的电池电量指示器。
  • 视频支持50赫兹高清电视模式(仅限DVD和HD DVD)。
  • Xbox 360无线赛车轮支持。
  • 设置新发布的Xbox Live街机试用游戏的自动下载。
  • 支持即将发布的XNA Game Studio Express(需要单独下载和订阅。)
  • 来自Zune设备的流媒体音乐、图片和视频。

    • Xbox Live街机

  • 所有Xbox Live街机游戏在主机上闪电般的列举和列表。
  • 使用两个新的排序选项快速查找游戏:最近玩过的游戏和按类别。
  • 自动下载所有新的Xbox Live街机游戏。当您进入Xbox Live Arcade时,“自动下载”可以选择将新发布的游戏试用版添加到活动下载中。
  • 在Xbox Live Arcade中查看完整的成就信息，包括图标、成就点数详情和描述。
  • 好友排行榜:与好友进行比较，并直接从Xbox仪表盘查看排行榜详情。
  • 告诉朋友:向朋友发送消息，介绍您最新喜爱的Xbox Live街机游戏。

    • DVD播放器

  • 观看DVD电影时，可以选择禁用Xbox Live的通知。
  • 改进了对不正确制作的DVD上的音频流的支持。
  • 更好地处理不可读的光盘。
  • 更新了倒计时定时器。
  • HD DVD玩家支持。
  • 增加了对非变形内容的缩放支持。
  • 添加了对家庭设置的支持，以阻止未分级的内容。

    • 视频播放器

  • 改进了对WMV内容中扩展的全黑场景的支持。
  • 增加了对Windows Media Video 9高级配置文件视频的支持。
  • 当视频暂停时，屏幕保护程序将被激活。
  • 现在与DVD播放器共享用户界面和播放器控制。
  • 改进了从Xbox Live下载时的视频播放。
  • 增加了跳过控制(向前大约。大约30秒后。7秒)。
  • 改进了无法播放的视频内容的消息传递。
  • 增加了经过时间和剩余时间的显示。
  • 改进了碎片硬盘上高比特率高清内容的播放。
  • 改善了视频播放的视觉质量。
  • 增加了可选的显示模式:默认和信箱缩放。

    • 本地化

  • 中国的新区域设置。
  • 改进了控制台设置中的德语翻译。
  • 改进了内存区域的德语翻译。
  • 改进了家庭环境中的日语翻译。
  • 在图片区域添加了所有支持语言的本地化文本。
  • 改进了相机效果中的西班牙语翻译。
  • 改进了特定于语言环境的十进制分母。
  • 增加了DVD播放器标题菜单和主菜单的本地化支持。
  • 增加了对“试图读取媒体”系统消息的本地化支持。

    • 记忆区

  • 增加了对Xbox 360 HD DVD播放器内容的支持。
  • 添加了按类别查看内容大小的新页面。

    • 其他系统改进

  • 更新了对2007年新夏令时变化的支持。
  • 改进了对游戏的支持，以更好地处理多人比赛中的主机迁移。
  • 改进的Xbox Live Vision摄像头支持在缩放时改变房间照明。
  • 后台下载的视频出现在视频区。
  • 使用VGA时改进Xbox Live Vision摄像头支持。
  • 从PC播放照片幻灯片现在保持图片文件夹的层次结构。
  • 改进了在游戏或媒体区域播放和删除视频的用户体验。
  • 改进了Xbox Live注册时对无效字符的处理。
  • 更好地处理玩家卡中的水印。
  • 改进了Windows Media Connect的文本。
  • 改进了Media Center扩展器的用户界面性能。
  • 改进了使用Xbox 360无线适配器时对Buffalo WZR-G108无线路由器的连接支持。
  • 重新下载已删除的街机游戏不再重新列举街机页面上的所有游戏。
  • 改进了主题中文本的视觉显示。
  • 改进了注册Xbox Live并失去网络连接后对CD播放的支持。
  • 改进了对长项目名称存储区的支持。
  • 改进了从Xbox Live下载玩家图片的处理。
  • 改进了从Xbox Live下载许多小物品时的后台下载通知。
  • 改进了从帐户管理区返回Xbox仪表板后对自定义背景的支持。
  • 使用Xbox 360无线适配器提高了Windows Vista上的Windows Media Center扩展器和Xbox 360控制台之间的网络可靠性。
  • 改进了Xbox Dashboard和游戏内聊天的语音聊天质量。
  • 改进了无法连接到Windows Media Center扩展器时的消息传递。
  • 改进了使用USB集线器时的USB键盘支持。
  • 提高了删除各种类型内容时用户体验的一致性。
  • 改进了后台下载的低存储空间处理。
  • 改进了通过MTP协议从连接的数码相机传输图片的处理
  • 改进了存储设备已满时可选iPod支持下载的错误消息。
  • 改进了USB键盘的多键重复处理。
  • 扩展了家庭设置，支持阻止未分级的DVD和视频。
  • 增加了从下载区域的存储设备中清除空间的能力。
  • 增加了对低速和全速USB键盘的支持。
  • 改进了USB键盘的大写锁定行为。
  • 改进了Xbox Live市场和Xbox Live街机区域的外观和感觉。
  • 在Xbox Live注册中添加了兑换代码选项。
  • 删除了新下载开始时的系统消息。将显示已完成项目的消息。
  • 在Xbox Live Marketplace购买区增加了对CCV信用卡验证字段的支持。
  • 更新了Xbox Dashboard中关于将Xbox 360主机连接到Windows PC以流式传输数字媒体内容的“更多信息”说明。
  • 在Xbox Dashboard UI中增加了长文本描述的自动滚动。
  • 改进了播放视频文件时对DVD插入的支持。
