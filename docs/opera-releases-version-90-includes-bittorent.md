# Opera 发布了 9.0 版本，包括 bittorent

[![](img/0ea001755c36e9e6f53b7fbb6096ce27.png)](https://web.archive.org/web/20220516085759/http://www.opera.com/) 网络和移动(以及[任天堂 Wii](https://web.archive.org/web/20220516085759/http://opera.com/products/devices/) )浏览器 [Opera](https://web.archive.org/web/20220516085759/http://opera.com/) 今天发布了 9.0 版本，提供了许多很酷的新功能，包括内置的 bittorrent 下载器、标签预览和小工具。

这个位于挪威奥斯陆的浏览器起源于 1994 年，拥有一个忠实的用户群，尽管规模不大——尤其是在国际移动用户中。

对于有经验的 torrent 用户来说，bittorent 系统可能非常慢，但是一次点击下载 bittorent 文件的简单性可能真的会吸引新手。(对类似功能感兴趣的 FireFox 用户可能想看看我们对 AllPeers 的评论。)

选项卡预览是一个很好的功能——悬停在浏览器的任何选项卡上，会出现一个小的下拉窗口，显示屏幕截图、标题、URL 和其他信息。我知道当我有超过 20 个标签的时候，我总是会丢失火狐标签。

Opera 的新版本还包括许多桌面小部件，许多人可能会觉得很有用。尽管市场份额很小，这款浏览器确实有很多吸引人的功能。
