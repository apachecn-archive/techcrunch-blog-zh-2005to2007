# 维亚康姆对 YouTube 的诉讼毫无胜算

![](img/56241b61b868faed8778b34f5686af6b.png)

当维亚康姆[宣布](https://web.archive.org/web/20210227140420/http://crunchgear.com/2007/03/13/viacom-to-sue-google-and-youtube-for-1-billion-for-copyright-infringement/)将起诉 YouTube 和谷歌的时候，网上很多人都这么做了，“你永远也不会打败 YouTube，永远也不会打败 YouTube，等等”。事实证明，这些愚蠢的鹅可能是有道理的。
 《商业周刊》有一篇坚实的文章分析了为什么 YouTube 和谷歌可能有坚实的法律基础，根本没有真正的理由害怕维亚康姆。主要原因是，YouTube 的商业模式并不围绕版权材料(就像 Napster 当年做的那样)，而是围绕真正[伟大的](https://web.archive.org/web/20210227140420/http://youtube.com/watch?v=G3TEG9KbHSU)、[自制的](https://web.archive.org/web/20210227140420/http://www.youtube.com/watch?v=k4d5QVQR4Qo)、[用户生成的](https://web.archive.org/web/20210227140420/http://youtube.com/watch?v=_IJaXV3ywEY)内容。换句话说，维亚康姆不能反过来说 YouTube 完全靠其版权材料盈利。当侵权内容被报告给 YouTube 时，很快就会被删除。

另一个原因是，与过去被 RIAA 起诉而被遗忘的 P2P 公司不同，YouTube 背后有大量资金，如谷歌，可以用来打官司。

《商业周刊》提出了理性的解释，而不仅仅是“维亚康姆烂透了，YouTube 说了算！”

维亚康姆的诉讼不会扼杀 YouTube
