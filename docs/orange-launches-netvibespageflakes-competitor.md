# Orange 推出 Netvibes/Pageflakes 竞争对手

 [](https://web.archive.org/web/20221130202918/http://www.bubbletop.com/) 别人都有，为什么不多一个呢？Orange 是一家大型欧洲移动电话运营商，现在是法国电信的子公司，正在测试一款名为 [Bubbletop](https://web.archive.org/web/20221130202918/http://www.bubbletop.com/) 的可定制 Ajax 主页产品。

这个团队实际上是建立在旧金山附近，并且创造了 [Pikeo 照片分享](https://web.archive.org/web/20221130202918/http://www.beta.techcrunch.com/2006/11/29/new-phot-sharing-site-se-habla-espanol-et-francais/)产品。出于某种原因，他们决定在一个法国博客[上发布他们的发布故事，同时发布一个法语的采访/演示视频。不过，该产品本身似乎只有英文版本。](https://web.archive.org/web/20221130202918/http://www.leblogdebezier.com/?p=147)

我无法从视频演示中了解该产品的更多信息，尽管它似乎是一个标准的基于模块的 Ajax 主页，还包括社交网络功能，如在左侧栏中包括一个朋友区(我是猜测，见截屏)。

这是一个[过度拥挤的空间](https://web.archive.org/web/20221130202918/http://www.beta.techcrunch.com/2007/03/13/a-little-perspective-on-ajax-home-pages/)(现在谷歌也在大力推广[他们的 iGoogle 产品](https://web.archive.org/web/20221130202918/http://www.beta.techcrunch.com/2007/04/30/rebrand-new-features-google-ig-to-relaunch-as-igoogle/))。但 Orange 有近 9000 万移动用户试图向他们推广新的在线产品。如果我能进入测试版，我会在这上面贴更多。注册进入 Bubbletop 主页。
