# nyko Wii Party Station–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/12/nyko-wii-party-station/>

![e3-2007-nyko-wii-party-station-20070711042135475.jpg](img/147b5f735880f18adec5eca354e03c48.png)

我认为 Nyko 非常清楚它的目标人群。让我想想:他们以小组为单位玩游戏，因此有 4 个 Wiimotes 充电站。然后他们有一个奇怪的小得分读数，这样一个喜欢数字的球员就可以得分了。然后它有四个白色的饮料杯和一个芯片盘。最后，它有一个干手器。简单地说，Nyko 描述了我小时候需要的所有东西。是的，我是(曾经是！哈！)一个胖胖的、汗流浃背的、沉迷于电子游戏的笨蛋，还有一个对数字有严重强迫症的朋友，而我在任何时候最多只有四个朋友。派对电台将会是 24.99 美元，定价刚好在“圣诞钱”和“奶奶拜托了”之间
