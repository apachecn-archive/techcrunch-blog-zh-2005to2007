# Sling Media 为假期投放了三个弹匣

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/27/slingmedia-drops-three-slingboxes-for-the-holidays/>

# 吊索媒体投下三个假期吊索箱

Sling Media 的 Slingbox 是一种你可能在使用它之前不会意识到它的必要性的产品，就像平板电脑、 [TiVo 或避孕套一样。对于门外汉来说，它的简称是 Slingbox，让你可以在笔记本电脑、台式机或选择的移动设备上观看和控制你的家庭电视。第一个版本有一些非常小的缺点，但总的来说，它是一个伟大的产品。](https://web.archive.org/web/20190203131350/https://crunchbase.com/organization/tivo)

明天，Sling Media 将推出三款新机型，包括一款带有可选高清输入适配器的专业版，这是原版所没有的。其他特性包括宽屏(16:9)支持、高质量可编程视频压缩和对多达四个 A/V 源的完全控制。专业型号大约要 250 美元，高清适配器要 50 美元。

超出你的价格范围？Sling Media 增加了两款低端机型:Slingbox AV 和 Slingbox Tuner。AV 让您可以控制数字有线电视、卫星电视或 DVR 机顶盒。调谐器适用于使用标准基本模拟电缆的人，他们不需要访问机顶盒、DVR 或其他 AV 设备。两者的大小都是原版的三分之二，售价为 180 美元。
