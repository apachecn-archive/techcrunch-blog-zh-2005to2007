# 雅虎！回答发起问地球运动

![](img/3eec542355889d6f737ae68f786dd0d2.png) [雅虎！答案](https://web.archive.org/web/20220626091015/http://answers.yahoo.com/)今天推出了一项新的推广活动，名为“[向地球](https://web.archive.org/web/20220626091015/http://advision.webevents.yahoo.com/asktheplanet/index.html)提问”，名人、知名人士和据称有趣的人将提出问题，用户回答这些问题就有机会赢得奖品。名人阵容引人注目，应该会吸引很多关注。

以下人员将向用户提出问题:

*   玛丽莲·沃斯·莎凡特，世界最高智商的保持者，今天以一个关于在学校帮助孩子的最好方法的问题开始了宣传。
*   从周五到周日，你将有机会回答唐纳德·特朗普的一个问题，换取一个 5000 美元的消费账户。
*   美国消费者新闻与商业频道金融大师苏茜·欧曼的奖品是 3 盎司纯金。
*   咔嚓咔嚓，或者 NPR 汽车之声的汤姆和雷·马格里奥兹将参加。他们问题的奖品是一年的免费汽油。唐纳德·特朗普的问题只能得到 5000 美元！
*   Arrianna Huffington 将提出关于新闻和事件的问题，获胜者将获得 DC 双人游。惊心动魄。
*   阿尔·戈尔的问题与一辆免费的 2006 款普锐斯有关。
*   斯蒂芬·霍金的社会科学问题让一个人可以去史密森尼博物馆参观。
*   博诺将关闭活动和雅虎！将捐赠 25，000 美元给他的竞选活动以纪念获胜者。

它被称为“问星球”，但该公司声称，法律禁止向美国以外的任何人颁奖。这可能会引起一些愤怒。事实上，因为有这样一个名字，这场运动看起来相当以美国为中心。同样，本着大量页面浏览量的精神——我指的是科学调查，获胜者似乎是随机挑选的，用户被鼓励提供更多的回复以增加获胜的机会。该公司计划在美国各地张贴广告宣传这场运动。尽管如此，这看起来是一场有趣的运动，可能会提升雅虎的形象。回答有意义。让你的观众把他们的答案扔进帽子里是解决世界上最令人烦恼的问题的最好方法吗？你可以说我愤世嫉俗，但我不确定这是真的。
