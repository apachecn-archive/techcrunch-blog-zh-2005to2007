# 疯狂错误的谣言杀死了 WWDC

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/11/wildy-wrong-rumors-killed-wwdc/>

[![wwdcbroke.jpg](img/4e1cc8bbe705446e28f2120bc8cd21d1.png)](https://web.archive.org/web/20150919182158/http://old.crunchgear.com/wp-content/uploads/wwdcbroke.jpg "wwdcbroke.jpg")

“事实证明，我们几乎都错了。”一位关心此事的公务员曾经谈到某个中东国家存在非法武器。同样的话现在也适用于苹果的 2007 年全球开发者大会，在那里，互联网上流传的几乎每一个谣言都是完全错误的。没有新的 iMacs 没有新的 iPods17 英寸的 iMac 和 Mac mini 仍在销售；没什么。

我认为，这些猖獗且显然毫无根据的谣言开始损害苹果的形象。

至少从我浏览过的一些留言板的回复和我们自己的专业意见来看，对今年 WWDC 的普遍共识是，WWDC 糟透了。什么都没发生。标题是“苹果发布 Windows 版 Safari。”这可能发生在一年中的任何一天，它会很好。但不，这是杀死 WWDC 的炒作。

似乎在过去的 200 年里，我们一直在阅读和传播关于 WWDC 发布会的一个又一个谣言。早期有传言说会有黑色的 iMac，但是快速浏览一下新的苹果商店就发现没有这种电脑。你也找不到新设计的 iMac。据我所知， [17 英寸 iMac](https://web.archive.org/web/20150919182158/http://crunchgear.com/2007/05/21/apple-prepped-to-kill-off-17-inch-imac/) 还在我们身边。

没有[【top secret】](https://web.archive.org/web/20150919182158/http://crunchgear.com/2007/06/08/os-x-leopard-to-be-animated-os/)Leopard 的公告，当然，除非你考虑到 iChat“top secret”新功能的改进。我们没有。

没有 Wi-Fi iPhone，没有 3G iPhone。没什么。

从本质上说，所有由源源不断的谣言造成的炒作对我们来说有点扼杀了今年的 WWDC。当然，Safari 很酷(嗯，就像网络浏览器一样“酷”)，但有趣的是“还有一件事”甚至没有所有这些传言中的*暗示的那么多。*

这些谣言到底从何而来？

为什么我们都被它们困扰和困扰？
