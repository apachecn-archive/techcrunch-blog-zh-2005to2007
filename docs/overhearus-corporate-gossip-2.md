# 美国:公司八卦 2.0

如果你和大多数人一样，你可能也有过糟糕的经理和错误的公司计划，导致办公室里怨声载道。很难告诉给你开支票的人他们是问题所在，也不是我们所有人都能像布拉德·加林豪斯的[花生酱备忘录](https://web.archive.org/web/20221001201851/http://www.beta.techcrunch.com/2006/11/18/yahoos-brad-garlinghouse-makes-his-power-move/)一样获得媒体的关注。

Y Combinator 初创公司 [Overhear.us](https://web.archive.org/web/20221001201851/http://overhear.us/) 希望通过为员工提供一个公司专用的匿名论坛来改变这种情况，让他们直接说出他们对那些 [TPS 报告](https://web.archive.org/web/20221001201851/http://www.scribd.com/doc/3649/TPS-Report-from-Office-Space)的真实想法。Overhear.us 根据他们的电子邮件域名创建公司特定的论坛，是的，这意味着可以有一个公司论坛，你可以在那里抱怨 Gmail。您只需输入您的电子邮件地址就可以注册，电子邮件地址会发送给您一个验证链接，以确保您是该地址的所有者。这也意味着你可以通过输入同事的地址来邀请他们。一旦通过验证，您可以通过 cookie 保持登录状态，或者注册一个帐户用户名和密码来管理您在多个公司域中的活动。

留言板非常简单。您可以张贴和阅读与您的域名对应的主题留言板。消息中不存储任何用户 id，即使是为了内部目的。他们希望匿名会让人们更自在地表达他们真正的担忧和沮丧。由于匿名也可能导致滥用，董事会也是由社区管理的。用户投票评论和线程上下列表，想法是奶油将上升到顶部。

我们看到了隐藏在这些丑闻中的商业模式。除了为人力资源部门和管理层提供有用的信息外，他们认为管理层也喜欢澄清误解或回应批评。通过购买一个帐户(价格待定)，你的公司老板将无法调节董事会，但为他们的帖子购买一个特权位置，将他们移动到顶部，并让他们特别突出，以便于阅读。

Overhear.us 目前得到了天使投资，是制作了一个名为 [SocialMoth](https://web.archive.org/web/20221001201851/http://socialmoth.com/) 的发布秘密的类似网站的同一团队的产品。在这个领域有一家公司叫做[匿名员工](https://web.archive.org/web/20221001201851/http://www.anonymousemployee.com/)。
