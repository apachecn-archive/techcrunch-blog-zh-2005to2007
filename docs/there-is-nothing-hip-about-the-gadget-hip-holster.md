# 小配件皮套:激发你内心的“书呆子哈利”

![hipholster.jpg](img/491fc93c94db8fc8f82a93b58b376a98.png)

虽然我同意口袋膨胀是一个需要解决的问题，但小工具皮套不是。简而言之，这是一条腰带，上面有一个袋子，里面有口袋，可以装你的 iPod、手机、记忆棒、自尊、童贞等等。听起来不是很博学，但是我们以前不是叫这些腰包吗？

说真的，这比把手机挂在腰带上还糟糕。是的，把手机挂在腰带上会让你看起来很傻。

在你在评论中指出我之前，我知道你要说什么:我又来了，提出一个问题却没有提供解决方案。这就是我做事的方式。你认为我们应该对这个不断升级的问题做些什么？
