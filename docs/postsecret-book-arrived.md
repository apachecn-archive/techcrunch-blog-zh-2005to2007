# 秘密之书到了

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/12/24/postsecret-book-arrived/>

我在十月份[从亚马逊](https://web.archive.org/web/20220626075235/http://www.beta.techcrunch.com/2005/10/23/the-postsecret-book-cometh/)订购的[机密](https://web.archive.org/web/20220626075235/http://www.postsecret.com/)书到了。

昨晚我花了一些时间看了一遍，我真的很喜欢它。PostSecret 是比较有意思的 Web 2.0 公司之一(如果可以称之为公司的话)。它当然创造了对话，唤起了情感，并将线上和线下世界完美地融合在一起。

如果你想买这本书，可以在亚马逊[买到。](https://web.archive.org/web/20220626075235/http://www.amazon.com/exec/obidos/ASIN/0060899190/techcrunch-20/002-1922935-6175214)

说到书，看看 Om 的免费“最佳”电子书。哇，阿诺·盖尔菲的伟大设计。
