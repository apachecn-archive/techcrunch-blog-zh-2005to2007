# TrekStor 3-1 高清设置

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/13/trekstor-3-1-hd-setup/>

TrekStor 可能因其昂贵的 MP3 播放器而闻名，但他们将试图在 CeBIT 上用 3-1 高清设置来改变这种外观。3.5 英寸硬盘有一个双 USB 2.0 集线器，并有一个存储卡读卡器，能够破译九种不同的格式。它是一个理想的媒体中心，因为有 250 GB(160 美元)和 400 GB(230 美元)两种选择。你可以轻松地交换所有存储卡中的数据，只需点击一下开关，数据就会备份。如果你参加了 CeBIT，那就看一眼吧。
