# 真正的谷歌手机？

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/06/the-real-google-phone/>

![](img/27375f2ec82adea0e984530b93c31fa0.png)
这张由 [Engadget](https://web.archive.org/web/20180726181852/http://www.engadget.com/2007/01/18/the-google-switch-an-iphone-killer/) 泄露的图片可能显示了谷歌手机的一些重现

风险投资家西蒙·西蒙诺夫在他的风险投资博客 [High Contrast](https://web.archive.org/web/20180726181852/http://simeons.wordpress.com/) 上发表了一篇文章，他声称这是关于备受期待但仍处于推测阶段的谷歌手机的内部消息。据西梅奥诺夫称，谷歌已经让安迪·鲁宾负责开发 G-Phone。Sidekick maker [Danger](https://web.archive.org/web/20180726181852/http://danger.com/) 的创始人鲁宾在谷歌收购他的无线软件初创公司 Android 时来到了谷歌。据报道，在鲁宾的领导下，有一个大约 100 人的团队在这个设备上辛勤工作。西梅奥诺夫认为，鲁宾正在直接开发这款设备的操作系统。

他继续引用谷歌对 Reqwireless 和 Skia 的收购作为明确的迹象，表明谷歌正致力于创造自己的手机。Reqwireless 创建移动应用程序，在被收购之前，Skia 开发了一个基于矢量的演示引擎，能够在移动电话上呈现“最先进的”2D 图形。

谢苗诺夫将这些收购视为谷歌实验室 G-Phone 运转的证据。他还声称，与[谷歌橙色 HTC](https://web.archive.org/web/20180726181852/http://crunchgear.com/2006/12/17/google-and-orange-readying-gphone/) 手机的传言相反，新设备正在与三星联合开发(引用此[新闻稿](https://web.archive.org/web/20180726181852/http://www.google.com/press/pressrel/samsungandgoogle.html))和[*开关*代号](https://web.archive.org/web/20180726181852/http://crunchgear.com/2007/01/18/google-switch/)再次被传得沸沸扬扬。

据说这款新设备在设计上类似于黑莓手机。它将编码在一个 C++内核上，带有一个操作系统引导程序。像 Sidekicks 一样，Java 将被优化以在设备上运行。它将包括多种服务，如 G-Talk 等。

他确实提出了一个相当令人信服的案例，并且确实似乎考虑了围绕 G-Phone 的许多错综复杂的细节。那么有效吗？

我要大胆地站出来，同意西梅奥诺夫先生的意见。不难相信谷歌真的会研发这样一款设备。无线传播内容，比如定向广告，比通过管道传播要便宜得多。此外，如今宣称无线是未来几乎没有争议。

在技术允许的情况下，世界正以最快的速度走向自由化。毫无疑问，互联网上最大、发展最快的公司会不断寻找扩张的途径。
