# 一键博客评论:点击评论

[Clickcomments](https://web.archive.org/web/20220517221209/http://postreach.com/static/clickcomments) 旨在通过提供一种一键式非回避评论方法，消除在博客上发表评论时留下个人信息的需要。

![clickcomments.jpg](img/e0c649839c455c92707fe3c8f7fbbb60.png)

作为 Postreach 的一项服务，Clickcomments 由博客上包含的许多按钮组成，这些按钮与各种用户响应相关，如有见地、有趣和伟大的发现。用户只需点击一个或多个与他们对帖子的看法相关的内容，而无需留下个人信息或离开网站本身。

所有收到点击评论的帖子都会被列在 [Postreach 网站](https://web.archive.org/web/20220517221209/http://postreach.com/postreach/showcase/All%20Topics?tier=2&cc=0&tf=1&lang=1)上，有可能为使用该服务的网站提供流量。

Clickcomments 是一个没有威胁的插件，很容易在寻找吸引观众的额外方式的博客作者中获得青睐。
