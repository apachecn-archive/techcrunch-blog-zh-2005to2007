# thanko MP4-玩手表:做了很多，但看起来很糟糕

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/20/thanko-mp4-playing-watch-does-a-lot-but-looks-awful/>

[![thankomp4watch.jpg](img/f380c3b7a032cacc200ed71c0728934e.png)](https://web.archive.org/web/20160401031101/http://old.crunchgear.com/wp-content/uploads/thankomp4watch.jpg "thankomp4watch.jpg")

Thanko 生产的手表可能是我见过的最丑的手表，但在这一点上我会听从专家的意见。我们在 CG 公司关心的是这款手表，MP4 手表，有 2GB 和 4GB 两种版本，可以播放最流行的数字媒体文件格式，如 MP3、JPG，可能还有 MP4 视频。手表还可以录音。它实际上做了相当多，把我对除了显示时间之外做任何事情的手表的不合理的憎恨放在一边，看起来它可能是一个整洁的小玩具。虽然它似乎与 Mac OS X 不兼容。现在这是愚蠢的，尤其是考虑到它使用的是 USB 1.1 连接来传输数据。

所以，如果你能忍受在地铁上为了看《辛普森一家》( Simpsons)而在手腕上戴上一块小砖头，看起来很傻，那就准备好花 208 美元买这块手表吧。
