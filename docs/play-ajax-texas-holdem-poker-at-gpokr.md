# 在 gpokr 玩 Ajax 德州扑克

如果你喜欢阿贾克斯的[战舰](https://web.archive.org/web/20221005212201/http://www.beta.techcrunch.com/2006/05/25/battleship-ajax-sinkmyship/)，你会喜欢这个。总部位于加拿大多伦多的 [gpokr](https://web.archive.org/web/20221005212201/http://gpokr.com/) 是一款使用 Ajax 技术开发的多人无限德州扑克游戏。该网站由 Ryan Dewsbury 创建，在游戏中所有玩家之间也有实时聊天。该网站不使用真钱，Ryan 说他不打算改变这一点。

gpokr 是用 java 编写的，它使用谷歌[的 Web 工具包](https://web.archive.org/web/20221005212201/http://code.google.com/webtoolkit/)作为其客户端 Ajax(该工具包通过将服务器端的 Java 解释成 javascript 减轻了构建客户端 javascript 的负担)

还可以看看 Mark Roth 的 Ajax [hangman](https://web.archive.org/web/20221005212201/http://octagonsoftware.com/home/mark/gwthangman/) 游戏，它也是使用 Google Web Toolkit 创建的。两者都在 Web Toolkit 博客上讨论过，这里是。
