# 图书时间 5000 自动图书阅读器

 [https://web.archive.org/web/20201123193914if_/https://www.youtube.com/embed/gHESCNS28Tc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://web.archive.org/web/20201123193914if_/https://www.youtube.com/embed/gHESCNS28Tc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

视频 如果你太便宜，买不起索尼电子书阅读器，又懒得自己翻翻一本真正的书，那就有日本的自动读书器。你所要做的就是把你的书放下，设置翻页间隔(自动的，或者定时 X 分钟)，瞧，一台机器就可以翻页了，这样你就不用翻页了。

当然，你最终因缺乏锻炼和糖尿病而失明，这将阻止你享受任何类型的书籍，无论是“e”还是其他，所以你可能想四处走动，而不是成为一个懒汉。3000 美元就能买到这个。所以我们认为电子书阅读器可能是更好的解决方案。
