# 时代华纳有线电视上市

时代华纳今天宣布，时代华纳有线电视将成为一家上市公司。该公司预计股票将作为 A 类普通股在纽约证券交易所交易，交易代码为“TWC”。股票最早可能于 3 月 1 日开始交易。

此举是时代华纳有线电视申请破产保护时从 Adelphia 购买资产的结果。作为对这些资产的交换，TWC 向阿德尔菲亚支付了 1.56 亿股。现在，破产是最后确定的，大部分的 TWC 股份将分配给阿德尔菲亚股东。
