# NextWave 将收购 IP Mobile

美国电信初创公司 NextWave Wireless Inc .是一家技术和设备提供商，该公司宣布将购买 IP Mobile 69.23%的股权。这将使其成为继去年英国沃达丰集团退出后日本第一家外国移动电话运营商。预计日本总务省将批准这项收购。

如果交易如预期顺利进行，尚不清楚 NextWave 旗下的 IP Mobile 能否成为日本第五大移动电话运营商。几个障碍仍然存在，特别是 IP Mobile 的许可证将到期，除非该公司能在 11 月前开始运营。该公司计划提供高速数据通信服务，并于 2005 年秋季与软银移动公司(Softbank Mobile Corp .)和 Emobile Ltd .一起获得了手机服务的带宽。两家公司都已开始运营，而 IP Mobile 仍然落后于计划。
