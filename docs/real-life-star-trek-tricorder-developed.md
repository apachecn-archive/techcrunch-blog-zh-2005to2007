# 真人版星际迷航三录仪开发出来了

![](img/74e4a800d3c54a3bb40a7dbd97d7f8e3.png)

《星际迷航》的粉丝们(抱歉，我不是他们中的一员)应该感谢普渡大学的研究人员开发了一个现实生活中的[三录仪](https://web.archive.org/web/20130628150450/http://en.wikipedia.org/wiki/Tricorder)，这个著名的小设备可以分析和识别未知物质。该设备使用一种叫做离子龙头的东西，根据物质的质量来识别物质。将物体放入后，该设备根据其质量将一个物体的离子一个接一个地释放出来。是的，我希望你在化学课上专心听讲。

这在这个美丽的星球上有什么用？它可以用来确定一件食物是否被沙门氏菌污染，或者一个可爱的小娃娃是否被可卡因塞满。机场应该注意，一旦大规模生产，这种奇特的质谱仪将只需要 2000 美元左右。它也有 8 磅重的便携式和 20 磅重的固定式两种型号。这些额外的重量可以归因于它的板载 Windows 计算机和高容量硬盘驱动器的外壳。

我有没有提过*星际迷航*很蹩脚？好的。
