# AnyDVD 向 DRM 说再见

AnyDVD 帮助了那些想要破解无用的数字版权管理的蓝光迷。最初人们认为 AnyDVD 会发布一个测试版，但版本 6.1.3.0 已经发布，你现在可以自由地从你的蓝光光盘中取出 DRM。免费升级适用于 XP-64 和 Vista-64，并删除了区域编码。快乐的海盗！
