# 原始 Zune 聊天 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/15/primitive-zune-chat/>

我和我的朋友们无话可说，这将是完美的。ChikaZune 是一个图形文件的集合，可以使用 Zune 的 phot 共享系统发送回来。每张图片都包含文本片段——“你一会儿忙吗？”“哥们儿！”“我怀孕了。”你可以可行地创建一个非常简单的对话，如果你不是在讨论时事或土豚解剖，你一定会找到一些适用的东西。文本被分类到文件夹中以便于检索。

这不是《旗语》中的简爱，但也差不多了。
