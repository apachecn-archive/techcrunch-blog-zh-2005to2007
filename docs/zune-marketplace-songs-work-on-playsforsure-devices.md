# Zune Marketplace 的歌曲在 PlaysForSure 设备上工作(？？)

![](img/2da4feea48981725a7d703a26046c5ed.png)
警告:CrunchGear 不对音乐品味负责。

Zunerama 有一点令人兴奋，因为一位读者发现 Zune Marketplace 的歌曲现在正致力于一种创造性的禅，这意味着 Zune DRM 毕竟不是那么特别。

显然某人 13 岁的女儿开始把 Zune Pass 歌曲从她的 Zune 拖到她的 Zen。禅宗毫无怨言地吸收了音乐，但论坛确信，一旦 Zune 的 DRM 服务器发现发生了什么，她的欢乐将是短暂的。就我个人而言，只要她继续付账，我很怀疑会发生什么事情。官方允许 Zune/PlaysForSure 兼容并不符合微软的最佳利益，但如果这是一个错误或有目的的“bug”，那么我怀疑它不会很快改变。Zune 受到了伤害，他们可以获得的任何善意的提升都是一件积极的事情。
