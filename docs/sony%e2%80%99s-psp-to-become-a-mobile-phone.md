# 索尼的 PSP 将成为手机

索尼 PlayStation Portable 是一款不错的游戏系统，但它可能很快就会成为一款手机。只是不打算在美国国内打电话。正如我们的姐妹网站 Crunchgear 所报道的，在英国，PSP 可能会成为一款手机。索尼电脑娱乐欧洲公司与英国电信公司合作，将 PSP 转变成一种移动通信设备。

根据这项协议，用户将能够使用该系统通过互联网或 BT 无线热点进行语音通话、视频通话和发送信息。这项服务将利用即将推出的 PSP 摄像头和麦克风。虽然这不会是一部成熟的手机，但它肯定会给苦苦挣扎的系统一个急需的推动…如果只是在英国。
