web jam Bags 200万美元用于个性化页面社区

总部位于伦敦的WebJam是一个可公开编辑的个人页面社区，刚刚从法国早期风投I-Source Gestion获得了200万美元。你可以在这里看到我们之前的报道。

WebJam允许您创建任意多的个人页面，方法是使用ajax编辑器将专门的模块放到您的页面上。如果没有开放模块标准，这就有点太难了。默认页面从博客、个人资料或个性化起始页的模块开始。WebJam使创建或修改自己的页面变得更加容易，只需点击一下鼠标，就可以将模块甚至整个布局从其他WebJam页面复制到自己的页面。

模块包括个人发布(博客)、社区(朋友列表、公告板……)、媒体(照片、音乐、rss源……)和生产力工具(记事本、搜索、Gmail……)。这些模块中的每一个都可以保持公开，对朋友和注册用户可见，或者保持私有。每个页面都有一个社区，您可以邀请其他用户加入。您可以使用此功能在一定程度上模仿宁，首先创建一个安装了社区模块的中心群页面，然后邀请拥有个人资料页面模块的朋友加入。
