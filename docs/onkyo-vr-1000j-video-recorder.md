# 安桥 VR-1000J 录像机

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/20/onkyo-vr-1000j-video-recorder/>

安桥刚刚发布了最新的媒体录像机 VR-1000J，猜猜看？您选择从电视或任何其他视频源录制的任何内容都与索尼的小奇迹盒子、PSP 和您的手机完全兼容。有了 3 种不同的视频模式，您一定会对超精细(320×240，30fps)、精细(320×240，15fps)和正常(176×144，15fps)模式感到满意。VR-1000J 支持视频的 MPEG4，音频的 AAC/G.726/AMR，并有空间容纳存储卡或 SD 卡。
170 美元不算太贵。
