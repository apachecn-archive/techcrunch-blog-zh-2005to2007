# YouTube 给竞争对手带来致命一击

[![](img/e47cf1c81db5df7cadf184b5bf5ebb80.png) ](https://web.archive.org/web/20230116063154/http://www.youtube.com/) YouTube 联合创始人查德·赫利暗示了未来几个月 YouTube 的收入分成计划。BBC[推测](https://web.archive.org/web/20230116063154/http://news.bbc.co.uk/1/hi/business/6305957.stm)广告可能会采用 3 秒钟的预播放形式，但是查德·赫利在[他的评论](https://web.archive.org/web/20230116063154/http://www.buzzmachine.com/index.php/2007/01/26/davos07-chad-hurley-on-youtube-on-youtube/)中没有提到这一点。

数百万个 YouTube 视频是一个巨大的库存，最终给了广告商一个真正的理由开始投资视频广告的创作，并开始使用谷歌 AdSense 视频。

我相信会有三方寻求赔偿:

*   视频内容所有者——原创内容的所有者。(音频“指纹”版权系统已经在工作一段时间了，它将视频与内容所有者匹配，以补偿音频/视频内容所有者。)
*   视频内容创作者——将内容混搭到定制作品中的用户。
*   发布者——在其网站上向访问者显示视频的用户/公司。

其中一个问题是，广告是否会出现在 YouTube.com 以外网站的 YouTube 视频上。如果我是一个发布者，我想选择从我发布的视频中获得收入——但是如果我是 MySpace 的发布者(用户), MySpace 是最终的发布者。MySpace 加载了 YouTube 视频，但 MySpace 不允许在他们的网站上发布外部广告。此外，MySpace 一直在随机地[屏蔽外部插件](https://web.archive.org/web/20230116063154/http://techcrunch.com/2007/01/18/myspace-is-getting-a-little-testy/)并且没有任何解释，所以我想 YouTube 不会冒险在 MySpace 页面上看到的视频中显示广告。谷歌已经与 MySpace 同床共枕，这可能会导致 YouTube 视频中广告的 rev-share 讨论，并确保 YouTube 永远不会成为 MySpace 的不明原因的被禁受害者。

YouTube 的竞争对手一直在努力通过补偿视频内容创作者来使自己与众不同，但一旦 YouTube 以其老大的地位进入这场游戏，我质疑是什么让用户去 Revver、Guba、Metacafe 等不太知名的视频网站。竞争对手最近一直在挣扎——Revver 失去了两位创始人，Guba [失去了首席执行官和两位高管。](https://web.archive.org/web/20230116063154/http://gigaom.com/2007/01/05/guba-shopping-itself-around/)
