# 推出新的 MySpace 浏览器 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/15/new-myspace-focused-browser-launched/>

 [](https://web.archive.org/web/20221006021459/http://www.buddywave.com/) [BuddyWave](https://web.archive.org/web/20221006021459/http://www.buddywave.com/) 是一款新的浏览器，它完全专注于 MySpace 的数千万用户，这些用户每天在该网站上产生大约 20 次页面浏览量。该浏览器可在 Windows 机器上下载。大多数自定义功能需要通过浏览器登录您的 MySpace 帐户。我们遇到了很多问题，无法让登录正常工作；然而，试了几次之后，我们进去了。

BuddyWave 是 Internet Explorer 7 的精简版，增加了一些 MySpace 特有的功能。点击下面的屏幕截图查看大图。

像 [Flock](https://web.archive.org/web/20221006021459/http://www.flock.com/) 一样，BuddyWave 专门针对社交网络人群(尽管将 Flock 与 BuddyWave 进行比较是不合适的——Flock 是一个成熟的基于 Mozilla 的浏览器，而 BuddyWave 在其发展的这一点上充其量只是一个玩具)。BuddyWave 内置了两个主要工具:一个类似即时通讯工具的好友列表管理器，以及一个用于编辑 MySpace 个人资料的 Ajax 工具。请查看演示(链接到他们的主页)以了解这些工具的运行情况。

 [](https://web.archive.org/web/20221006021459/https://beta.techcrunch.com/wp-content/uploads/2007/02/buddywavebig.png) 现在为坏消息。由于 MySpace 没有官方 API，浏览器代表用户将请求代理回 MySpace。这意味着 MySpace 可以(也可能)马上关闭它。此外，MySpace 网站上任何形式或变量的改变都会破坏 BuddyWave，尤其是认证，直到 BuddyWave 更新他们的软件。BuddyWave 在浏览器中安装了一个自动更新程序，可能是为了处理这些不可避免的情况，尽管这也为 BuddyWave 下载和执行他们希望发送到您的计算机上的任何其他软件打开了大门。

另外，BuddyWave 最好的特性是 Ajax profile builder/editor。然而，他们所做的只是将第三方编辑器 [KillerKiwi](https://web.archive.org/web/20221006021459/http://killerkiwi.net/) 加载到浏览器窗口中。MySpace 用户可以通过任何浏览器轻松访问该网站。

在某些方面，BuddyWave 让我们想起了另一款基于 IE 浏览器的浏览器 [Browzar](https://web.archive.org/web/20221006021459/http://www.beta.techcrunch.com/2006/09/01/is-browzar-just-an-adware-machine/) ，该浏览器于去年推出，并获得了 15 分钟的声誉。然而，与 Browzar 不同的是，我们还没有看到任何迹象表明 BuddyWave 主要是一种向用户投放广告的方式。

如果你花大部分时间在网上浏览 MySpace，并且不介意使用 IE 浏览器，BuddyWave 可能是你会使用的。如果你不属于这一类，我们建议至少等到将来的版本解决了一些问题。

该公司位于新墨西哥州，由一群“22 岁的大学辍学生”创办。BuddyWave 博客是[这里](https://web.archive.org/web/20221006021459/http://www.buddywave.com/blog/)。其中一位创始人之前创建了[我的空间](https://web.archive.org/web/20221006021459/http://www.beta.techcrunch.com/2006/07/15/mywhatspace-sends-group-messages-in-myspace/)，我们去年对其进行了简介。
