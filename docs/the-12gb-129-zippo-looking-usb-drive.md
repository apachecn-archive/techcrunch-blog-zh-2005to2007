# 12GB，129 美元的 Zippo 外观的 USB 驱动器

避开普通的 NAND 内存，支持老式的机械硬盘，来自美国模块化的怪物驱动器是一种不同的拇指驱动器。这个庞然大物的大小和外观与你的标准 Zippo 大致相同，它将 12GB 的存储空间装入一个光滑的不锈钢外壳中。再加上末端的蓝色 led 来显示数据的读取和写入，您就有了看起来完全“老板”的东西

它已经准备好与 USB 2.0 和抗震。如果壮志凌云里的马华力用了 u 盘，这就是他会用的。它将于 11 月上市，售价 129 美元，考虑到你的极客中包含了多少摇滚乐，这个价格还不错。
