# 日本音乐杯:杯中的 DAP 扬声器

遇见音乐杯！它来自亚洲，是一个马克杯！用扬声器！和一个短小精悍的千斤顶！它播放你的 iPod 的音乐！这是我们在美国可能看不到的事情之一，但是，在这种情况下，我们可以接受。

真的。它由骨瓷制成，有黑色或白色(就像你的 iPod 一样)，底部隐藏着一个扬声器。我们不知道你为什么想要这个，但我可以看到，当没有其他想法出现时，无知的母亲会在斯宾塞礼品公司为他们喜爱 iPod 的儿子购买这个东西(编辑:有点像英国版“办公室”中蒂姆的 HatFM 布莱克)。
