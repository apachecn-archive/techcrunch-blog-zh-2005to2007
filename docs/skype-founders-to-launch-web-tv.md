# Skype 创始人将推出网络电视

互联网电话巨头 Skype 和老牌的 KaZaA 点对点文件交易服务公司的创始人计划在未来几个月推出一个广告支持的互联网电视网络。

随着更高质量的 P2P YouTube 的出现，这个代号为“威尼斯”的项目将允许业余爱好者和专业人士把他们的电视节目放到网上供任何人观看。他们还没有说他们将如何阻止有版权的材料被放在这项服务上，这意味着人们最有可能从电视上分享真实的电视节目，就像他们现在在 BitTorrent 上被分享一样。

如果他们的过往记录有任何迹象——KaZaA 是在线音乐的重大颠覆，而[SkypeT3 是传统座机的重大颠覆——这个威尼斯将是传统广播网络的一记重击。它会是最终将电视交易带给大众的东西吗？我们希望如此。](https://web.archive.org/web/20201129133321/https://crunchbase.com/organization/skype)
