# Safari 3 For Windows First Look

![](img/6a34717e0871d776ab1bed15dd1c6ddb.png)

既然 WWDC 战争已经结束，是时候检查善后事宜了。

我们找到了 Safari，并不意外地发现了一些初始错误。考虑到 Safari for Windows 仍处于测试版，大多数问题最终都会得到解决。

尽管史蒂夫似乎对 Safari 的基准很自豪，但页面的加载时间几乎没有变化。然而，Flash 速度已经提高，在 YouTube 或 MetaCafe 上浏览视频需要的耐心更少了。底线？还没准备好黄金时间。这是一个很好的尝试，但就我所见，在 IE7 甚至 Firefox 上几乎没有什么改进。简而言之，等待最终发布。
