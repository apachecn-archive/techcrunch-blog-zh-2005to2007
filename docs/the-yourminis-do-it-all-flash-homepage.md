# YourMinis 全能 Flash 主页

 [](https://web.archive.org/web/20211026154010/http://www.yourminis.com/) [YourMinis](https://web.archive.org/web/20211026154010/http://www.yourminis.com/) 是一款基于 Flash 的可定制主页产品，将与大量使用 Ajax 的类似产品争夺用户——[Netvibes](https://web.archive.org/web/20211026154010/http://www.beta.techcrunch.com/tag/netvibes)、 [Pageflakes](https://web.archive.org/web/20211026154010/http://www.beta.techcrunch.com/tag/pageflakes) 、[谷歌](https://web.archive.org/web/20211026154010/http://www.beta.techcrunch.com/2005/12/14/googles-ajax-desktop-is-now-extensible/)、[Live.com](https://web.archive.org/web/20211026154010/http://www.beta.techcrunch.com/2005/11/01/windows-live-more-than-an-ajax-desktop/)和 [more](https://web.archive.org/web/20211026154010/http://www.beta.techcrunch.com/2005/12/07/ajax-desktops-wont-stop/) 。这是由一家名为 [Goowy](https://web.archive.org/web/20211026154010/http://www.goowy.com/) 的初创公司推出的，该公司创建了一个基于 flash 的生产力套件(电子邮件、日历、即时消息等)。)去年——[看这里](https://web.archive.org/web/20211026154010/http://www.beta.techcrunch.com/tag/goowy)看我们的 Goowy 报道。

Goowy 首席执行官 Alex Bard 在本周的 Web 2.0 峰会上向我展示了该产品。YourMinis 是一个可靠的产品，提供完全可定制的模块，如 RSS 提要、Flickr 照片、YouTube 视频、POP 电子邮件等。用户可以创建多个标签来更好地组织信息，任何标签都可以变成公共 URL 并与朋友分享。他们还发布了一个 API，并允许第三方创建可供所有用户使用的模块。

你的 Minis 也有一个浏览器扩展，可以更容易地向网站添加信息。用户只需点击浏览器上的一个按钮，就可以将视频、照片和 RSS 源添加到自己的迷你页面上。这对于订阅 RSS 提要特别有用——一个模块会根据自动发现的提要自动添加到您的 Minis 中。

该网站上还有其他有趣的功能值得探索，YourMinis 无疑是一个展示 Flash 功能的橱窗。
