# 大众和 Sirius 罢工交易

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/25/volkswagen-and-sirius-strike-deal/>

![](img/0fdea25646af1818ac4a5ca38f77fbeb.png)

看起来天狼星和大众已经成功地完成了一项交易，将使天狼星卫星广播标准在选择大众从 2008 年到 2012 年。途锐 2、新甲壳虫、新甲壳虫敞篷车、GTI 和 GLI 车型都将标配 Sirius，但你仍需每月支付订阅该服务的费用。值得庆幸的是，大众包括 3 个月的免费订阅。

大众汽车还通过在途锐 2 和三重白色甲壳虫车型中提供 Sirius 标准，将 Sirius 纳入其加拿大业务。大部分都是一样的。看起来 XM 的粉丝们如果打算买 V-Dub 的话，他们必须找一家新的汽车公司或者一台新的音响。
