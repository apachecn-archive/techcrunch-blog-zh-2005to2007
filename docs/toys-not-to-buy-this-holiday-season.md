# 这个假期不要买玩具

从某种意义上说，我成了那个讲述“当我还是个孩子”故事的人，比如“当我还是个孩子的时候，我有一个由锋利的金属片制成的玩具木琴，从来没有人说过它有危险。”谢天谢地，美国公共利益研究小组(美国 PIRG)在 21 年前就开始做它的“玩具王国的年度麻烦”报告，除了别的以外，帮助减少假期去医院的次数。这份年度榜单今天发布，虽然榜单上没有“电子产品”本身，但却有很多有害的玩具和小玩意。

例如，Magnetix 磁性建筑套件的塑料建筑件和杆内有微小的磁铁，它们可能会脱落。报告称“如果一个孩子吞下不止一块磁铁，磁铁会相互吸引，导致肠穿孔或阻塞。”该产品的制造商重新设计了电视机，并在 3 月份开始了更换计划，但 PIRG 警告说，一些旧库存可能仍然在商店的货架上。

这份名单没有提到对排队购买索尼 PS3 或任天堂 Wii 的人的潜在伤害。如果你没有看过视频，请点击跳转。
