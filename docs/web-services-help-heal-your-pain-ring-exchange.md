# 网络服务帮助治愈你的痛苦:戒指交换

你和她分手了吗？她是不是在浴室里“哼”了太多次了？她的头发是不是有点掉色，和你的新音箱不相配？她想再谈谈你的“欺骗”吗？那么，抛弃她，就像一个坏习惯，转售她的订婚戒指，在一个可观的利润。

没错，朋友们，互联网终于找到了一种将心痛和深厚的情感转化为金融交易的方法——这是世界上最古老的职业多年来一直在做的事情。这个网站由年轻的约书亚·奥特曼创建，他发现自己的戒指在转售时价值只有三分之一，于是他把被抛弃和即将被抛弃的戒指放在了一起。我们认为独创性在美国已死。
