# 通用遥控器的 MX-810 Pro 向导遥控器:不适用于 Pater Obsoletis 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/15/universal-remotes-mx-810-pro-wizard-remote-not-for-pater-obsoletis/>

[![mx810.jpg](img/d9661f1b0648846c0111f59f573a9576.png)](https://web.archive.org/web/20150505155556/http://tctechcrunch2011.files.wordpress.com/2007/06/mx810.jpg "mx810.jpg")

我记得在 CES 上和通用遥控器的人聊过，我对自己说，“是的，这些人确实做了很多遥控器。”最新的遥控器 MX-810 Pro Wizard Remote 可以控制多达 24 种不同的设备(每个设备有 24 种不同的功能)，所有这些都存储在 ~~32GB~~ 32MB 的内存中。顶部有一个液晶显示器，可以完全定制，你可以在上面存储喜欢的频道等，以便快速访问。MX-810 同时兼容射频和红外。

如果你能超过 400 美元的价格标签，如果你是这些类型的遥控器的粉丝，你可能会这样做，那么它看起来可能值得你花时间。不过，我想知道握起来是否舒服。
