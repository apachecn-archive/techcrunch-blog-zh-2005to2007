# 雷·奥茨将取代盖茨成为首席软件架构师

[![](img/1fde556c274cb6cf4111bebbd3982527.png)](https://web.archive.org/web/20220818080531/http://www.microsoft.com/) 在[的新闻发布会上](https://web.archive.org/web/20220818080531/http://www.microsoft.com/presspass/press/2006/jun06/06-15CorpNewsPR.mspx)今天微软首席执行官史蒂夫·鲍尔默和董事长比尔·盖茨宣布，两年后比尔·盖茨将不再全职为微软工作。他将在那里做兼职工作，而不是全职在盖茨基金会工作。今天宣布了一项为期两年的过渡计划。

雷·奥茨将立即接替盖茨担任首席软件架构师。Ozzie 是微软基于网络的软件的最杰出的倡导者。自从他的公司 Groove Networks 于 2005 年 3 月被收购以来，他一直担任微软的首席技术官。

首席技术官 Craig Mundie 将立即成为新的研究和战略主管。
