# 博客音乐插件

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/04/11/tunefeed-music-widget-for-blogs/>

Faces.com 即将推出的新服务 Tunefeed 将允许人们上传音乐到一个储物柜，然后在网站上放一个 Flash 小工具，让访问者播放音乐。MySpace 已经有了一种有限的方式来做到这一点，TagWorld 有一个非常强大的服务。但对于我们这些有普通博客的人来说，Tunefeed 可能是一种分享音乐的有趣方式。我很期待在 TechCrunch 上测试这一点。

在 Tunefeed [主页](https://web.archive.org/web/20150919065112/http://www.tunefeed.com/)上注册测试版。您还可以在那里看到一个小部件的示例。另一家来自澳大利亚的酷公司。
