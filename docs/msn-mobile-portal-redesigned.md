# MSN 移动门户重新设计

本周，微软正式公布了 MSN Mobile 的新面孔，它拥有完全重新设计的外观。该门户提供电子邮件、新闻、体育、娱乐、本地电影列表、地图和路线以及其他 Windows Live 服务，包括 Windows Live Hotmail、Windows Live Messenger、Live Search 和 Windows Live Spaces。

内容包括访问来自合作伙伴的最新内容，包括 MSN 的独家在线体育合作伙伴 FoxSports.com 和 MSNBC。新的 MSN Mobile 现已在美国上市，在 2007 年的剩余时间里还将在全球市场上市。
