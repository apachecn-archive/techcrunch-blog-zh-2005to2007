Safari 3 For Windows First Look

既然WWDC战争已经结束，是时候检查善后事宜了。

我们找到了Safari，并不意外地发现了一些初始错误。考虑到Safari for Windows仍处于测试版，大多数问题最终都会得到解决。

尽管史蒂夫似乎对Safari的基准很自豪，但页面的加载时间几乎没有变化。然而，Flash速度已经提高，在YouTube或MetaCafe上浏览视频需要的耐心更少了。底线？还没准备好黄金时间。这是一个很好的尝试，但就我所见，在IE7甚至Firefox上几乎没有什么改进。简而言之，等待最终发布。
