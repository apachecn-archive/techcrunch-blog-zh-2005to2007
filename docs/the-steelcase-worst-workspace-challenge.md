# Steelcase 最差工作空间挑战| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/19/the-steelcase-worst-workspace-challenge/>

明天是狂欢节，我正准备喝得烂醉如泥。然而，在我离开之前，我必须花一点时间告诉你我们激动人心的新比赛，Steelcase 最差工作场所挑战。

这是一场空前的竞赛。

先说正事吧。这件事的奖项是令人难以置信的，我真的不能再兴奋了。大奖是一把全新的[钢制案](https://web.archive.org/web/20150703160355/http://www.steelcase.com/)椅。获胜者可以选择任何可用的[思考椅](https://web.archive.org/web/20150703160355/http://store.steelcase.com/articles.asp?id=155)或任何[飞跃椅](https://web.archive.org/web/20150703160355/http://store.steelcase.com/articles.asp?id=153)，不包括皮椅(所以基本上你可以在[黑色](https://web.archive.org/web/20150703160355/http://store.steelcase.com/ProductDetails.asp?ProductCode=LEAPV2)或[白金](https://web.archive.org/web/20150703160355/http://store.steelcase.com/ProductDetails.asp?ProductCode=46216179)底布飞跃椅之间挑选)。这些椅子的价值约为 900 美元，你可以选择你的封面颜色。我最近[回顾了一个飞跃](https://web.archive.org/web/20150703160355/http://crunchgear.com/2006/12/28/workspace-roundup-ergonomic-chairs/)，它几乎是有史以来最伟大的事情。

我们也有四个亚军的位置。我们在 [iFrogz](https://web.archive.org/web/20150703160355/http://www.ifrogz.com/) 的朋友已经提供了四张礼品卡，可以让你选择你想要的 iPod 保护套。这些案子很热门。

现在说说规则。要进入，我们需要最差工作场所的照片。这包括你的桌子、椅子、一般的工作环境等。我们希望看到灾难。到处都是纸，口香糖粘在东西上，咖啡洒了，椅子坏了，桌子砸了，等等。它必须是糟糕的，大写的 b。把你的参赛作品发到 crunchgear dot com 上，以“最糟糕的工作空间挑战”为主题。

工作环境最差的人将会赢得一把全新的钢制椅子，帮助你稍微改善一下糟糕的环境。而且，虽然 iFrogz 案例不会让您的工作环境变得更好，但它至少会帮助您少关心一点灾难。

我们将运行这个测试整整七天。这意味着下周一就关门了，所以开始工作吧。
