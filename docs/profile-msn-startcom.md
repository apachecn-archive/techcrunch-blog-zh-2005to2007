# 个人资料–MSN Start.com–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/03/profile-msn-startcom/>

## 这是什么？

Start.com，一个 RSS 聚合器和搜索服务，是微软的“T0”沙箱“T1”项目之一(“T2”虚拟地球“T3”是另一个)。这是一个 [AJAX](https://web.archive.org/web/20220517182430/http://en.wikipedia.org/wiki/AJAX) 应用程序。

![](img/a40c46ef28d030c72495cf024278cac1.png)Start.com 可与[谷歌的 IG 或个性化家居产品](https://web.archive.org/web/20220517182430/http://www.beta.techcrunch.com/?p=104)相媲美。主要的特性包括一个拖放界面，简单的 RSS 订阅(虽然自动发现似乎不是很好，它在我身上挂了很多次)，以及与 MSN 搜索的集成。提要和其他内容列在左侧的下拉菜单中，内容可以“固定”在页面的任何位置。Start.com 的博客在这里是。

## 主要特点:

*   在页面的任何地方“锁定”内容(提要或搜索结果)
*   将锁定的内容拖到页面上的任意位置
*   移除任何内容
*   增加或减少列数
*   改变页面的基本外观(“花岗岩”、“冰”等)。)

我唯一想不通的是如何从下拉菜单中删除预填充的“流行”提要。

像谷歌的 RSS 聚合器一样，这对于 RSS 新手或只阅读有限数量的提要的人来说是一个很好的服务。
