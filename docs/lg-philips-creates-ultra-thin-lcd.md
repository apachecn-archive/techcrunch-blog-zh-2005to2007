# LG 飞利浦制造超薄液晶显示器

![](img/59fe6f2a84cbeda9d9a0a29b7857b7fe.png)

你准备好迎接新的时尚了吗？LG Philips 发布了一款新的 LCD-TFT 显示屏，重新定义了轻薄。它的厚度只有 1.3 毫米，图像质量没有下降。

目前还没有任何订单，但这并不意味着什么。预计在 2007 年，这些将会在手机中随处可见。
