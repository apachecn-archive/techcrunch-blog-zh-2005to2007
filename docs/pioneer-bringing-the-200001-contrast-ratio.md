# 率先推出 20，000:1 的对比度

![](img/df711b5a4e2c1c10773c373baf30d7fe.png)

在上周的 Ceatec 会议上，先锋公司宣布他们打算在 2008 年北京奥运会之前发布对比度为 20，000:1 的等离子体。他们展出了一个 60 英寸的样品，但他们没有提到这些东西会花多少钱，或者会生产什么尺寸的模型。我所知道的是，如果图片是可信的，新的看起来比右边的 3000:11080 p 显示器好得多。
