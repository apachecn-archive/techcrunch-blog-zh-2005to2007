# T-Mobile 推出 USB 调制解调器，让人想起大号平板电脑

T-Mobile 推出了 web'n'walk USB 调制解调器，与沃达丰的竞争，但它更便宜，T-Mobile 说。USB 调制解调器兼容 PC 和 Mac，提供高达 1.8Mbps 的宽带速度。如果你在英国，你可以以 29.99 英镑的价格购买一个，HSDPA 连接的月费为 29 英镑。
