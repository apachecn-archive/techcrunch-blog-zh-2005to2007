# Veeker:用于移动视频的嵌入式播放器

[](https://web.archive.org/web/20220701011405/http://veeker.com/)mobile runch 的 Oliver 在新推出的移动视频网站 [Veeker](https://web.archive.org/web/20220701011405/http://veeker.com/) 上发布了[独家首发](https://web.archive.org/web/20220701011405/http://mobilecrunch.com/2006/10/25/veeker-launches-today-mobile-video-will-be-changed-forever-a-mobilecrunch-exclusive/)。该服务捕捉彩信发送的移动视频剪辑，然后将它们发送到联系人、公共和个人页面。一系列的短片可以通过 RSS 订阅，并且可以在另一个网站的嵌入式播放器中自动播放。换句话说，我用手机录制视频，通过 Veeker 发送，然后它出现在我网站上的播放器中。YouTube 确实支持来自手机的视频，但要重现 Veeker 承诺的功能还需要很多步骤。

批评者认为，移动视频不太可能在大众市场广泛使用。显然，维克尔是一家下注的公司。我不会对广泛移动视频创作的长期可能性发表评论，但我会说我对这家公司的目标印象深刻。

Oliver 对服务进行了更详细的描述，高管团队尤为重要。至少有两个人(罗杰·拉德曼和亚历克斯·凯利)之前的公司被大笔收购。我忍不住将 Veeker 与 Twitter 或 Dodgeball 等网站进行比较，但 Oliver 的想法更宏大。他相信这项技术有商业用途。我认为只需要一点点想象力，这样的服务就可以让很多事情成为可能。很简单，Veeker 可以很容易地将近乎实时的视频整合为整个网站的一部分。在 mobile run 查看[。](https://web.archive.org/web/20220701011405/http://mobilecrunch.com/2006/10/25/veeker-launches-today-mobile-video-will-be-changed-forever-a-mobilecrunch-exclusive/)
