# 日立推出 1tb 硬盘

如果你和我们一样，你会下载成吨成吨的~~下流、色情~~完全合法的音乐和视频。那么，您可能会遇到和我们一样的问题:您将所有这些数据存储在哪里？

答案来自打着全球首款万亿字节硬盘幌子的[日立](https://web.archive.org/web/20201204181514/https://crunchbase.com/organization/hitachi) 。相反，制造硬盘驱动器，因为问题是放弃 7K1000 的两个版本，Deskstar，适用于游戏或高端 PC 或一般用途，以及 CinemaStar，更安静的版本，具有更好的家庭影院/媒体中心/DVR 应用程序纠错能力。
