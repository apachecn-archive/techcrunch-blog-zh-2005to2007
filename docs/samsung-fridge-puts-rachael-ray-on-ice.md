# 三星冰箱把瑞秋·雷放在冰上

![](img/46c7409e5ba4914551b9136c2585df87.png)

哦，我多么希望我有这个冰箱来满足我在瑞秋·雷和 T2 的烹饪需求。三星 RH269 无线冰箱配有一台 10.4 英寸彩色 TFT 触摸屏平板电脑，配有数字记事簿、无线电视/收音机、数字日程安排器和食物管理软件。根据产品页面，这款平板电脑可以从冰箱门上拆卸下来，使用 2400 毫安时的三芯锂离子电池。

[、3500 美元](https://web.archive.org/web/20210302071032/http://images.businessweek.com/ss/05/06/wireless/source/8.htm)的冰箱并不便宜，但对于像我这样不记得上周购买的蔬菜是否仍然新鲜的伪厨师来说，它是完美的。
