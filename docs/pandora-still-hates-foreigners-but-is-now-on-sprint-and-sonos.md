# 潘多拉还是讨厌外国人，但是现在上了 Sprint 和 Sonos

[![](img/3f23bc36ac68f2782a0364b7e1739bed.png)](https://web.archive.org/web/20210226103919/http://www.pandora.com/) 美国最受欢迎的流媒体音乐服务 Pandora 刚刚推出两款新产品，分别是 Sprint 的 Pandora 和 Sonos 的 Pandora。

TC 拥有大多数热门的 SCOOPZ，但 Sprint 的 Pandora 将在 30 天的试用后以 3 美元/月的价格将电台流传输到 Sprint 手机上。Pandora for Sonos 可以让你通过 [Sonos 音乐系统](https://web.archive.org/web/20210226103919/http://www.beta.techcrunch.com/2006/09/14/sonos/)获得好东西，这对这项令人惊叹的服务/技术的粉丝来说绝对是一个加分项。

TechCrunch 还报道说:

> 潘多拉还重新设计了它的网站，以更好地整合听力和社区功能。该公司表示，他们现在有 690 万注册听众，他们播放了 47 亿首歌曲，投票数上升或下降了 5 亿次。他们每天播放整个目录的 94%,所以他们正在深入音乐的长尾。
