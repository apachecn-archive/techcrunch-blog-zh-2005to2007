# WordPress 博客的随机页面重定向

今天早上，我们写了关于 StumbleUpon 网站的具体问题和 Rafe Needleman 在 T2 的想法，即在博客上有一个按钮，可以弹出一篇随机文章。在这篇文章之后，我联系了 WordPress 的创始人马特·莫楞威格(TechCrunch 在 WordPress 上运行),问他是否有人为 WordPress 创建了一个插件来做这件事。他们没有，所以马特写了一个并发表了。

此处有[的插件。](https://web.archive.org/web/20230207093623/http://wordpress.org/extend/plugins/random-redirect/)[点击此处](https://web.archive.org/web/20230207093623/https://techcrunch.com/?random)查看随机的 TechCrunch 往期帖子，或者使用上面导航栏最右侧区域的 [![stumblecrunch.png](img/1624858fd68daddb3f0c2c215a1b6779.png)图标](https://web.archive.org/web/20230207093623/https://techcrunch.com/?random)。

如果 StumbleUpon 决定以小部件的形式提供这个功能，我们可能会集成它。Matt 创建插件不会给帖子分配任何值或评级。有了 StumbleUpon，读者更有可能对随机发布的内容感兴趣。
