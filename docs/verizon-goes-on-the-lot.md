# 威瑞森上停车场

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/30/verizon-goes-on-the-lot/>

威瑞森无线将成为福克斯互动媒体制作的真人电影《在停车场》的赞助商之一。该运营商将赞助与该系列相关的电影制作挑战，同时还提供幕后镜头，如 bios 和剪辑。威瑞森无线的客户将进一步有能力在真人秀系列节目中参与短信投票，该节目将由处理美国偶像投票的望远镜公司运营。
