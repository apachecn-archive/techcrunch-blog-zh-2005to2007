# Novatel V740 Rev A ExpressCard 带来速度(价格)

![](img/75db7bb4a0bd3f544f8a55ad4fb1e100.png)

面向威瑞森的 Novatel V740 修订版 A ExpressCard 将于 3 月底推出。除了有一个非常长的产品名称之外，无论你使用 OS X 还是 Windows，这张卡都会给你带来惊人的速度。然而，这张卡并不像我们希望的那样便宜。你可以在网上以 110 美元的价格从 3Gstore.com 购买一份 2 年的合同，据传闻，威瑞森可能会提供一张到 PCMCIA 适配器的 ExpressCard，并提供 50 美元的购买折扣。

你已经知道移动 EVDO 宽带并不便宜，所以也要准备好支付天文数字的月费。
