# 传说中的变形金刚小工具机器人原型

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/02/purported-transformers-gadget-bot-prototypes/>

我们总是讨论那些适合放在你口袋里的小玩意；手机，数码相机，游戏系统。但是，像你们中的许多人一样，我们从 80 年代早期到中期的一系列可爱的玩具中得到了对所有小玩意的热爱:[变形金刚](https://web.archive.org/web/20161012034312/http://crunchgear.com/2006/12/20/robots-in-disguise/)。

这就是为什么我们认为 7 月 4 日上映的新《迈克尔·贝变形金刚》电影的剧本要求一些能变成小机器人的小玩意非常合适。真正有趣的是，这些小家伙的动作玩偶正在制作中，在这里，我们为您带来了这些动作玩偶(假定)原型的照片，主要是因为在下周的 CES 之前没有太多事情发生，因为我们可以。更多在跳跃之后！

剧本明确地说一个角色的 iPod 得到了变形金刚的治疗，但是我们猜测他的粗鲁是和威震天串通好的，因为里面没有 MP3 播放器。那个邪恶的机器人。

此外，告诉全班:谁是你最喜欢的变形金刚？声波？反光板？感知器？布拉斯特？很多很多其他人中的一个？
