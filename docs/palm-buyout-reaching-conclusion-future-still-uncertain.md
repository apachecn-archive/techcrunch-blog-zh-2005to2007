# Palm 买断达成结论；未来仍不确定| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/20/palm-buyout-reaching-conclusion-future-still-uncertain/>

本周的某个时候，商业的车轮可能会转向，即将推出的 Treos 上的标志会变成诺基亚甚至摩托罗拉的标志。这两家公司已经对 Treo、[的制造商 Palm 出价几周了](https://web.archive.org/web/20160420225847/http://crunchgear.com/2006/09/24/palm-buyout-rumor-447-could-this-be-it/)，一些私募股权公司也加入了竞争，有传言称事情将在本周结束。诺基亚想要 Treo 的知名度，而摩托罗拉想要在智能手机市场有一个像样的立足点。可悲的是，如果这些公司中的一家得到 Palm，我们会看到他们吸收了这家受欢迎的智能手机制造商，而不是支撑它。

这就是为什么我们和 Palm 一样，支持私募股权公司。Palm 作为一个独立的组织是保持 Treo(以及平台)平稳发展的唯一途径，我们已经听说了关于 Treo 即将到来的修订的奇妙事情，在其他任何人的手中，我们都没有看到他们发挥出全部潜力。

但是谁知道呢？如果 Moto 或诺基亚最终与 Palm 合作，Palm 可能会被允许继续作为自己的子公司，做自己的事情。那么，诺基亚？摩托罗拉？如果你在读这个，你知道，做那个。Palm 受欢迎的部分原因是它是 Palm，我们希望你能明白我们的意思。
