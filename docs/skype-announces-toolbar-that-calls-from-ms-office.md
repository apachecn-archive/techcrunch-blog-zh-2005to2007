# Skype 宣布推出可从 MS office 拨打电话的工具栏

 [](https://web.archive.org/web/20221130213714/http://skype.com/) Skype 今天宣布发布了一些非常酷的工具栏，包括与微软 Office 交互的[工具栏。安装后，您可以点击 Word、Excel 或 Powerpoint 文件中的任何电话号码，Skype Out 就会呼叫该号码！它也能识别传真号码，但不会拨打这些号码。非常酷。](https://web.archive.org/web/20221130213714/http://www.skype.com/download/skypeofficetoolbar/)

Outlook、Outlook Express 和 Thunderbird 的工具栏测试版于上周发布。这些功能对电子邮件中的数字也有同样的作用，使用 Skype IM 跟踪电子邮件变得很容易。两周前，该公司发布了 Firefox 和 IE 工具栏的公共版本，可以调用网页上的号码，并根据网页主机的 ISP 添加国家代码。

该公司表示，估计有 30%的 Skype 用户用它来做生意，这些工具栏就是针对这些用户的。

当然这些都是只针对 Windows！Mac 用户最喜欢的就是这个[周二的 Skype for Mac 1.5 版本](https://web.archive.org/web/20221130213714/http://www.beta.techcrunch.com/2006/07/25/skype-sends-mac-users-some-love/)。
