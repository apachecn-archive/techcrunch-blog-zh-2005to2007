# 那些中国人喜欢发短信

![](img/5b73da73ef14d110f1f8c689beea485d.png)

如果你认为你的高中女友发了很多短信，等着看中国的预测吧。新华社周一称，预计农历新年期间将有超过 140 亿条短信发送到 T2。请记住，这只是在一周内！中国联通和中国移动都调整了网络，以弥补流量的增加。

此外，mainland China 有超过 4.6 亿手机用户，如果你仔细想想，发送 140 亿条短信也不是什么大事！
