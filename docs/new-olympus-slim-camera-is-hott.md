# 新款奥林巴斯超薄相机热销

![](img/c4ee8d4b1ad6bb2b1d9bd50262ac48c3.png)

奥林巴斯韩国公司昨天展示了它的新 Mju-730 型号。这款新的超薄相机拥有 700 万像素的镜头，3 倍的光学变焦，“一触式完美修复”功能，以及适用于新手用户的简单模式。

奥林巴斯没有宣布的是价格或可用性。加上好看的大液晶显示屏，这款相机可能会成为想要一款外观酷、有效的超薄相机的人的一大热门。
