# SkyRider 的目标是将 P2P 转化为利润

 [](https://web.archive.org/web/20220810110345/http://www.skyrider.com/) 总部位于山景城的隐形 P2P 公司 [SkyRider](https://web.archive.org/web/20220810110345/http://www.skyrider.com/) 今天正式宣布成立该公司，但对于今年秋天上线后将做些什么却含糊其辞，令人抓狂。该公司由前思科首席技术官爱德华·科泽尔领导，并获得了红杉资本(Sequoia Capital)和查尔斯·里弗(Charles River)的资助。该公司表示，将通过搜索关键词营销和在 P2P 基础上开发软件，使 P2P 软件商业化。

这是什么意思？这家公司的网站和发布会都聚焦于 P2P 盈利到底有多理想。我发现的一些可能的线索是，SkyRider 和另一家公司 CRight Inc .拥有相同的投资者、公关公司和语音信箱。几个月前，这家公司被悄悄地认定为反盗版系统。(更新:我已经确认，从今天起，CRight 将名称更改为 SkyRider。更多可用信息。)这是否意味着 SkyRider 将提供允许 P2P 网络检测和阻止其系统上的非法文件共享的软件？这当然会让很多企业更放心地通过 P2P 开展业务。它也可能是一种在 P2P 网络中搜索合法内容的手段。Om Malik 表示，尽管公司对自己的技术只字未提，但它将帮助 P2P 内容摆脱目前的孤岛状态。

几周前，我们报道了 Guba 的一个名为“张诗钟”的项目，该项目旨在检测媒体共享网站上的版权视频，以及类似 bittorent 的大型文件分发服务“红狐”。我们很想知道，SkyRider 最终会不会推出类似这两家公司的产品。

最新消息:有人告诉我，我的猜测是错的，在 SkyRider 中使用的是 CRight 的遗留技术，但它本质上并不是要阻止任何东西。大概这只是意味着，这只是它的许多功能之一的方式。今天模棱两可的新闻稿确实引起了轰动，我们拭目以待，看看今年秋天 SkyRider 是否能做到这一点。
