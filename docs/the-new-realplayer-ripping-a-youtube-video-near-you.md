# 新的 RealPlayer:抓取你身边的 YouTube 视频

[![realplayer.png](img/170efefc5a38e26e9712c85514bae04c.png) ](https://web.archive.org/web/20220816072002/http://www.realnetworks.com/company/press/releases/2007/rp11b107.html) [RealNetworks](https://web.archive.org/web/20220816072002/http://www.realnetworks.com/) 发布了新版本的 RealPlayer，包括一键视频抓取。

免费下载的视频播放器将允许用户保存和组织所有主要格式的视频文件，包括 Flash、QuickTime、RealMedia 和 Window Media，并将支持从 YouTube、Google、Yahoo！、Brightcove、AOL 和纽约时报。

抓取在线视频的能力并不新鲜。已经有许多程序可以提供类似的服务，但这是第一次在高用户群产品中提供这种级别的功能。

新的 RealPlayer 将不会下载或录制受 DRM 感染的视频，但会下载其他所有内容，这对于一群内容创作者来说，是一件令人头疼和容易被起诉的事情。每个内容创作者现在都将面临这样的现实挑战:如果他们的产品是免费的 DRM，它很可能会从原始的源站点被窃取，甚至被刻录到 CD 上。

我可能夸大了这个问题，一个真正的反 DRM 倡导者会认为消费者应该自由地在他们认为合适的地方以他们认为合适的方式使用内容。然而，内容创作者可以在不使用 DRM 的情况下施加版权限制，并且应该能够控制视频回放的方式和时间；免费观看的提议并不自动延伸到免费和无限制使用的提议，以免费播放电视或广播为例。

新版 RealPlayer 将于 6 月发布。对不起，Mac 用户，今年晚些时候才会真正启用抓取功能，Windows 目前只支持 Internet Explorer 和 Firefox。
