# 索尼质疑虚拟现实 PS3 控制器

[![3dsonynope.jpg](img/0934b5086c4335e44428c03efed9c290.png)](https://web.archive.org/web/20210414170409/http://old.crunchgear.com/wp-content/uploads/3dsonynope.jpg "3dsonynope.jpg")

因此，几天前[让全球](https://web.archive.org/web/20210414170409/http://crunchgear.com/2007/04/13/sony-patents-3d-virtual-reality-controller-for-the-ps3/)议论纷纷的索尼虚拟现实手套专利可能永远不会问世。索尼与 Next Generation 进行了一次聊天，并表示专利申请只是索尼每次提出新想法时都要做的一部分。仅仅因为它申请了专利并不意味着它有任何实际推出该设备的意图。

这真的不应该让任何人感到惊讶，因为专利总是为那些永远不会出现的东西而申请。(去任何一个苹果谣言网站，看看他们如何挑出专利的细节。)所以，索尼，谢谢你把美味的牛排摆在我们面前。
