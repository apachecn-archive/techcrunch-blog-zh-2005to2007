# 到 2012 年，短信将成为 670 亿美元的产业

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/20/sms-to-be-67bn-industry-by-2012/>

来自 [Portio Research](https://web.archive.org/web/20160407215547/http://www.portioresearch.com/) 的[报告](https://web.archive.org/web/20160407215547/http://www.portioresearch.com/MMF07-12.html)预测短信收入将达到 670 亿美元。这一增长主要归因于移动用户群预计将从 26.1 亿增加到 48.1 亿，发送 3.7 万亿条消息。预计亚洲市场增长最快，到 2008 年将占用户群的 50%。

短信约占运营商非语音计划收入的 75-80%,语音计划占全球移动收入的 80%。运营商之间的竞争一直在侵蚀语音利润，手机铃声和壁纸的移动内容提供商也奄奄一息，用户更容易通过互联网访问内容。最大的内容公司之一 Moderati 最近被 bell rock Media[收购](https://web.archive.org/web/20160407215547/http://moderati.com/press_releases/02.12.07.html)，并将向移动应用程序过渡。因此，运营商应该将注意力转向数据服务。

SMS 已经成为增长最快的信息服务，但是价格较高的 MMS、电子邮件和移动 IM 也有望健康增长。该报告建议运营商必须谨慎管理这些替代品的费率，以避免蚕食短信市场。
