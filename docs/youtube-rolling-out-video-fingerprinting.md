# YouTube 推出视频指纹识别

视频

拉什可以选择删除这段视频或从中获利

为了遏制 YouTube 上不断发生的侵犯版权行为，该公司宣布计划测试一种新的视频识别形式。迪士尼和时代华纳是第一批试水这种新形式的数字指纹的公司，如果试验成功，其他公司将很快跟进。这些新的指纹识别工具已经测试了大约一个月，可以在视频中挑选出唯一的标识符，以确保它们的合法性。

但是，如果发现非法上传的剪辑或电影，谷歌/YouTube 将联系所有者，看看他们是否希望剪辑被删除，或者他们是否愿意参与收入分成。似乎是一个体面的交易，每个人都赢了。没有人被起诉，公司赚钱，观众仍然可以享受优质内容。要是世界其他地方像 YouTube 一样就好了。等等不要。
