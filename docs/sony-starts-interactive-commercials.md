# 索尼开始互动广告

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/25/sony-starts-interactive-commercials/>

![](img/06583f8afff1ed42fcb9e73c76752ef1.png)

随着所有跳过广告的 PVR 的出现，索尼决定开始为拥有 PVR 的观众制作互动广告。这个广告实际上是为索尼的 Bravia 电视制作的，让观众从两种不同的结局中选择一个——一个是男性，一个是女性。这里的问题是:互动广告真的会让人们观看索尼的广告吗？也许一次或两次，但新鲜感可能会很快消失。这也不是我们第一次看到商业互动。我们以前看过那些“去 http://www.plug-site.com 看结局”的广告。那么，你们中有多少人真的特意去看了结局？没有吗？真的吗？索尼或许应该坐下来，在观众开始变得非常非常恼火之前，重新考虑一下这部电影。
