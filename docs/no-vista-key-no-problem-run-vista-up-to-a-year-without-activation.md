# 没有 Vista 密钥？没问题！运行 Vista 长达一年没有激活

![](img/48b3545d56c62e61d721a17a8b281018.png)

上个月，当约翰在他家的楼上烤一个可爱的水果蛋糕时，我从他的地下室偷了一份 Windows Vista Ultimate。当我把它安装到我朋友的电脑上时，一切都很好。过了一段时间，WGA 开始尝试打破 Vista。然后我才恍然大悟，Johnnypants 已经用上了盒子自带的序列号。

原来这只是一个小小的不便。由于一次注册表编辑，Vista 可以长达一年不被激活。Windows 专家布莱恩·利文斯顿说，随着更多的研究，人们可以发现一种永远推迟激活的方法。这让我松了一口气。在我的一生中，自从 Windows 3.0(是的，3.0，不是 3.1)以来，我从来没有为微软 Windows 付费，老实说，我没有计划在短期内开始。WGA 就是个笑话！
