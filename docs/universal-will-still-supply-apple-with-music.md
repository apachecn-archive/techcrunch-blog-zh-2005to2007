# 环球仍将为苹果提供音乐

![](img/10ceb3471af60b23bf123815970a48fd.png)

前几天，人们开始变得奇怪，因为[环球音乐集团宣布不再与苹果](https://web.archive.org/web/20210119084713/http://crunchgear.com/2007/07/01/breaking-umg-will-not-renew-itunes-contract/)续签 iTunes 合同。包括我在内的所有人都认为这是一个信号，表明 UMG 将彻底退出 iTunes，数百万首可用歌曲将永远消失。幸运的是，事实并非如此。

在意识到公众对该声明感到愤怒后，环球公司决定淡化这种情况，称将继续在 iTunes 上提供音乐，只是不会按照之前与苹果公司达成的旧条款。新协议允许 UMG 对独家预发行专辑收取更多费用，这意味着苹果将不得不为这些 iTunes 独家服务支付更多现金。希望我们不会很快看到 iTunes 曲目涨价。
