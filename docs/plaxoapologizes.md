# plaxo…道歉

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/03/27/plaxoapologizes/>

 [](https://web.archive.org/web/20220819083222/http://www.plaxo.com/) [Plaxo](https://web.archive.org/web/20220819083222/http://www.plaxo.com/) 首席执行官本·戈卢布[在他们的公司博客上为他们过去鼓励“熟人垃圾邮件”的行为写了一份公开道歉](https://web.archive.org/web/20220819083222/http://blog.plaxo.com/archives/2006/03/an_apology.html)。我不得不说，我从来没有[真的想过](https://web.archive.org/web/20220819083222/http://www.beta.techcrunch.com/2006/03/22/plaxo-now-with-less-evil/)会发生这种事。看到这个我又惊又喜。

见鬼，我甚至可能加入这项服务来表示我的支持。我的意思是没有任何讽刺的痕迹(真的)。道歉的主要内容如下:

> 在过去的几天里，Plaxo 一直在博客圈饱受批评。坦率地说，其中大部分是我们应得的……对于那些讨厌收到 Plaxo 更新消息、觉得我们在给熟人发垃圾邮件或受到服务伤害的人，我个人代表 Plaxo 的所有人道歉。我知道要赢得你们的信任，我们还有很长的路要走，我只能请求你们通过我们未来的行动来评判我们。

谢谢你，普拉索。从现在开始把我当成支持者。
