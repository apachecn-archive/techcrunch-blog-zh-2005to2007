# 索尼推出 VAIO FZ 系列| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/09/sony-introduces-vaio-fz-series/>

![](img/5ba9bc32c670a65e93a333c96ef48c81.png)

为圣塔罗萨的行动而疯狂？索尼公司今天推出了新的 VAIO·FZ 系列笔记本电脑。这些新笔记本将在下周左右上市，零售价在 1400 美元以上。配备了蓝光驱动器和时尚的设计，新的 FZ 系列看起来是最近上市的更好看的 VAIOs 之一。

具体来说，你会发现一个采用 XBRITE 技术的 15.4 英寸显示器，一个蓝光驱动器，专用媒体控制，HDMI 输出，支持 a/b/g 的 802.11n WiFi，一个 ExpressCard 3/4 插槽，以及处理 SD 和记忆棒格式的内部读卡器。CPU、RAM 和存储都是可定制的，但 1400 美元的基本型号 FZ130 配备了 1.8GHz Core 2 Duo、1GB RAM、英特尔 X3100 video(哎哟)和 160GB 硬盘空间，以及 DVD-RW 刻录机。

如果你想要那个蓝光光驱，你就得付钱。所有系统都包括 802.11n WiFi 和 Windows Vista Home Premium。
