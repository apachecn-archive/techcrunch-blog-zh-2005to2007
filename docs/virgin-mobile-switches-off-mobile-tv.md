# 维珍移动关闭移动电视| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/26/virgin-mobile-switches-off-mobile-tv/>

在欧洲推出移动广播视频服务不到 10 个月之后，维珍已经停止了这项服务，并将于明年初关闭这项服务。带宽提供商 BT Movio 正在取消与频率所有者 GCap Media 的合同。许多挫折困扰着这项工作，包括只有一部手机——龙虾 700 TV——能够接收这项服务。尽管进行了大规模的促销活动，销售额还是很低。

但更大的原因是 BT Movio 的服务基于 DMB(数字多媒体广播)并利用 DAB 网络，而欧盟则选择支持 DVB-H 事实上的移动广播技术。维珍航空表示，这项服务将持续到 2008 年 1 月底。这么久以来，我们几乎不认识你。
