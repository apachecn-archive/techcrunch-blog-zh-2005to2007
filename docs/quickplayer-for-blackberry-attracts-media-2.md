# 黑莓 QuickPlayer 吸引媒体

![](img/0032e220226e71d52185c7a661fddf11.png)

深呼吸苹果发布会还没结束，我们就已经收到了来自 CTIA 的信件。多好的一天啊！

好了，接下来是这样的。QuickPlayer Media Inc .宣布了黑莓 QuickPlayer 应用程序的第一轮媒体合作伙伴。QuickPlayer 允许向 BlackBerry 设备传送流式音频广播。它由黑莓 8700 系列、7130 系列和 Pearl 支持。

该公司现在与美国广播公司新闻，AccuWeather.com，清晰频道，Gaiam，MarketWatch.com，华尔街日报和韦斯特伍德有安排。这些协议将允许 QuickPlayer 用户从这些媒体提供商那里获得流媒体内容。
