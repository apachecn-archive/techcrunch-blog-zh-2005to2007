# 起搏器袖珍 DJ 最好是真的，因为我需要混合

[![pacemaker.jpg](img/b47c231be10c739507b15dcf9bfa60ca.png)](https://web.archive.org/web/20210306015448/https://beta.techcrunch.com/wp-content/uploads/2007/05/pacemaker.jpg "pacemaker.jpg")

亲爱的太空耶稣，请告诉我们这个起搏器口袋 DJ 是真的。谁能抗拒一个 120GB 的硬盘和你能忍受的混合和交叉渐变所有 ATB 和 AVB 的能力？(对文斯和我来说，这是相当多的。事实上，下周我们将会对 ATB 大放厥词。来加入我们吧。)不过，电池续航能力一般，混合播放时只能持续五个小时，勉强够组装一台通宵电视机。它应该在秋天上市，但是没有标价和一个不能正常运行的网站，我对此表示怀疑。
