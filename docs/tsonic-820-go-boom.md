# 音速小子 820 Go Boom | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/19/tsonic-820-go-boom/>

你认为 MP3 市场不是已经充斥着太多的型号，以至于你的大脑开始因为所有的选择而出血吗？创见不这么认为。欢迎 T.sonic 820 系列，它配备了巨大的 1.5 英寸“超级彩色”有机发光二极管显示屏。T.sonic 820s 拥有 mp3、照片、视频、调频收音机和电子书等常见的功能。真正吸引我眼球的一个特点是过多的录音选项。它可以记录其内置麦克风，外部或线路输入！很酷吧，我也这么认为。它还有什么？很高兴你问了。820s 还显示歌词，所以你可以摇滚出卡拉 ok 风格。

其他一些 dope 功能包括 A-B 重复功能，让您播放歌曲的某一部分，显然通过将该功能与 820s 可调可变轨道播放速度控制相结合，您已经使自己成为一种语言学习工具。啊？

T.sonic 820 还具有 7 种均衡器效果、可定制的用户均衡器模式和 12 种语言的歌曲/艺术家标题显示。T.sonic 820s 现在有黑色和白色两种颜色，它们似乎借鉴了苹果的 iPod 系列，因为 2gig 只有白色，4gig 有黑色。
