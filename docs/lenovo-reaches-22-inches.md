# 联想达到 22 英寸

![](img/29827f2090131e81425ef58071025da2.png)

秋叶原报道说，联想现在提供 22 英寸宽屏液晶显示器。该显示器支持 1680×1050 WSXGA+分辨率。它还具有 700:1 的对比度、300cd/m2 的亮度和 5 毫秒的响应时间。

D221 将于 12 月初在欧洲上市，售价 350 欧元。预计它将在美国国内以相对较快的速度下降，价格约为 460 美元。
