# 威瑞森无线走向农村

![verizon.jpg](img/818a07589178d4eb5dab391df0734aed.png)今天，威瑞森无线宣布已达成协议，将收购 Rural Cellular Corporation，以进一步增强威瑞森无线在其现有服务区附近市场的网络覆盖，并将覆盖范围增加 470 万，客户群增加 70 多万。根据交易条款，威瑞森无线将以约 26.7 亿美元的现金和债务收购 Rural Cellular。

截至今年 3 月 31 日，Rural Cellular 的网络已经服务了大约 716，000 名客户。它分布在五个地区，网络位于缅因州、佛蒙特州、新罕布什尔州、纽约州、马萨诸塞州、阿拉巴马州、密西西比州、明尼苏达州、北达科他州、南达科他州、威斯康星州、堪萨斯州、爱达荷州、华盛顿州和俄勒冈州。Rural Cellular 在其五个区域市场中同时使用 CDMA 和 GSM 技术，威瑞森无线公司宣布计划在现有的 GSM 市场中部署 CDMA 服务，将 GSM 用户转换为 CDMA 用户。

威瑞森无线公司总裁兼首席执行官洛厄尔·麦克亚当说:“威瑞森无线公司一直在寻找机会来增强我们客户的无线体验 Rural Cellular 市场的加入将使我们能够将我们的服务扩展到以前我们很少或没有业务的地区，并将使 Rural Cellular 的 Unicel 客户能够访问全国最可靠的网络和更广泛的语音和数据服务。"
