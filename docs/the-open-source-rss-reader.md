# 开源 RSS 阅读器 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/10/31/the-open-source-rss-reader/>

我已经用了[博客桥](https://web.archive.org/web/20221006100830/http://www.blogbridge.com/)几个星期了。这是我第一次认真地使用桌面应用程序来阅读订阅——我真的很喜欢基于网络的阅读器，因为没有软件下载、兼容性问题等等。，而且因为很多桌面阅读器都不是免费的。但是如此多的博客桥粉丝给我发邮件建议我尝试一下，我也这么做了。基于我所看到的，我给它竖起大拇指。

Blogbridge 也是开源的。创始人 Pito Salas 告诉我，至少有六名非雇员开发人员参与了这个项目。有一种商业模式——blog bridge 最终希望在开源代码基础上推出优质功能。目前，一切都是免费的。

Blogbridge 有三种版本——Windows XP、Mac OS X 和 Linux。它完全是用 Java 编写的。每周都有新版本发布——当前版本是 2.7 版。

默认用户界面是三个窗格。文件夹在左边，提要在中间，帖子在右边。导入 OPML 的文件很容易，你可以离线阅读(这是基于网络的阅读器的一大优势)，他们积极支持戴夫·维纳的[阅读列表](https://web.archive.org/web/20221006100830/http://www.reallysimplesyndication.com/2005/10/13#a1032)的想法(尽管还没有整合)。

每个窗格都有滚动条，这意味着，就像 Bloglines 一样，你可以在窗格中滚动，而不会失去在其他窗格中的位置(这让我和其他读者一样疯狂)。像所有的读者一样，Blogbridge 在恰当地呈现 HTML 方面做得很差，所以帖子看起来有点差。

Blogbridge 也有专家指南来帮助用户找到他们感兴趣的内容。我认识并尊敬的许多人都在那份名单上，包括[理查德·麦克马努斯](https://web.archive.org/web/20221006100830/http://www.readwriteweb.com/)、[杰夫·克拉维尔](https://web.archive.org/web/20221006100830/http://softtechvc.blogs.com/about.html)、[丹·吉尔默](https://web.archive.org/web/20221006100830/http://bayosphere.com/blog/dangillmor)和[谢尔·霍尔茨](https://web.archive.org/web/20221006100830/http://blog.holtz.com/)。

我是开源项目的忠实粉丝，Blogbridge 是一个优秀的读者。布洛布里奇位于马萨诸塞州阿灵顿。
