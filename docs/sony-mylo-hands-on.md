# 索尼 Mylo 动手

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/15/sony-mylo-hands-on/>

索尼 Mylo (MyLifeOnline)在一个非常小而时髦的包中提供了一套很棒的工具。索尼最新的个人通讯设备具备了所有能引起轰动的功能，但它在一些方面有所欠缺。滑块允许 QWERTY 键盘在不使用时保持隐藏。2.4 英寸的屏幕充满活力，非常好地照顾了视觉效果。Mylo 的销售对象是我这个年龄段的人，18-24 岁，但我很快就对它失去了兴趣。让我们记住，大多数 Mylo 的观众都在大学校园里，所以我有足够的理由对这款设备不感兴趣。这并不是说我不喜欢媒体功能和 WiFi 功能。如果你想买一个，不要犹豫，因为它有很多可取的功能，但我只是一个挑剔的人，我<3 my Sidekick3 (most of the time). The Mylo is not a cell phone, but it sure does look like one.

让我们从积极的方面开始，这个清单很长。UI 相当直观，易于浏览，很像 Sidekick。最好的功能是 WiFi，802.11b，按左上角的滑动键就可以轻松激活。它也支持 WEP 和 WPA-PSK。屏幕左下角的 WiFi 强度指示器以白色亮起，否则指示器呈灰色。如果这还不够，右边会亮起蓝色，让你知道 WiFi 已激活。

这套即时通讯客户端包括雅虎、谷歌和 Skype。AIM 目前还不被支持，但索尼的一位代表告诉我，它可能很快就会推出。Yahoo messenger 被简化了，看起来很普通，但这并不是一件坏事。Google Talk 运行得非常完美，我用起来一点问题都没有。我对 Mylo 上的 Skype 持怀疑态度，但我感到惊喜。即使配备了耳机/麦克风组合，语音质量也比我的 Mac 好得多。事实上，当连接到我的家庭网络时，互联网服务在各方面都更好。这也有助于你获得一年的免费 T-Mobile 热点服务。当我在城市里四处走动时，这就派上了用场。不幸的是，我没有机会使用 Ad-Hoc 应用程序，因为我从来没有发现任何人有 Mylo，所以我不能分享我的播放列表或音频流。

Mylo 上的多媒体很容易浏览，通过迷你 USB 添加/删除内容相当简单。板载 1GB 闪存和记忆棒 Pro Duo 插槽可以让您娱乐相当长一段时间。该播放器支持 MP3，安全和不安全的 WMA 以及 ATRAC 音频格式。支持的视频格式包括 MPEG-4/AAC。

现在我们来谈谈 Mylo 的缺点。首先是 WiFi。什么？！很可能你和我一样，总是希望与他人保持联系。纽约有很多免费 WiFi 点和星巴克，但当你需要查看电子邮件或给别人发即时消息时，去搜索就很麻烦了。另一方面，Mylo 包括一项免费网络服务，告诉你你所在地区所有的免费 WiFi 点在哪里。

QWERTY 键盘坏掉有很多原因。按键触摸起来不是很有特色，因为它们太小了。左撇子会喜欢空格键的位置，但这是我第一次抱怨 Mylo。后面的扬声器让我把音量调得比我预想的还要大。就像我说的，我很挑剔。我不喜欢 Home 键的位置，这是因为我习惯了 Sidekick。

也没有电子邮件客户端，某些网页的格式是一个真正的麻烦。您确实能够放大和缩小页面，但是您不应该担心格式问题。对于一款通信设备来说，索尼为什么在最关键的功能上有所吝啬？

最后但绝对不是最不重要的是 350 美元的价格标签。对于一个很容易被遗忘并放在抽屉里的通信设备来说，这是一个很大的负担。我宁愿带着我的 iPod 和 SK3，而不是 Mylo。我认为如果 T2 关于 VoIP 进入 PSP 的传言是真的，这种情况不会持续太久。
