# 戴尔用新的 IdeaStorm 网站向 Digg 致敬

[![googleadscape.png](img/6430e047966c844a773de4c6cd1206c1.png)](https://web.archive.org/web/20220926085546/http://www.dellideastorm.com/) 
戴尔今天推出了一个名为 [IdeaStorm](https://web.archive.org/web/20220926085546/http://www.dellideastorm.com/) 的客户关系网站，这可能是整个行业未来发展的一个有趣迹象。用户可以提交产品和功能请求、政策变更或任何他们想与戴尔社区分享的内容。然后，这些提交的内容会以 Digg 的方式进行投票。戴尔的这一举动发生在 [Yahoo！揭开了一个类似的网站](https://web.archive.org/web/20220926085546/http://www.beta.techcrunch.com/2007/02/15/yahoo-launches-digg-like-suggestion-site/)。

该公司还推出了 [StudioDell](https://web.archive.org/web/20220926085546/http://www.dell.com/content/topics/topic.aspx/global/shared/corp/media/en/studio_dell?c=us&l=en&s=corp) ，这是一个专注于戴尔问题的视频共享网站，包含公司视频和用户提交的内容。允许在 IdeaStorm 上发表评论，但不允许在 StudioDell 上发表评论。

公司[完全承认它欠 Digg](https://web.archive.org/web/20220926085546/http://direct2dell.com/one2one/archive/2007/02/16/6311.aspx) 的智力债务，称 IdeaStorm 是“留言板和 Digg.com 的结合”

Jeremiah Owyang [称](https://web.archive.org/web/20220926085546/http://www.web-strategist.com/blog/2007/02/16/dell-to-go-community-driven-feature-design-in-latest-web-strategy-idea-storm/)这是对“内部网已经过时了”这一观点的认可，尽管在我写这篇文章时第二受欢迎的提交材料是呼吁为戴尔员工建立一个单独的内部 IdeaStorm 网站。这让我怀疑戴尔的员工会对 IdeaStorm 有多大热情。主要建议是让戴尔对生态更加负责。

我认为这更证明了由 Digg 和 YouTube 流行的范例的有用性。就像雅虎！昨天刚刚推出的建议网站——戴尔的 IdeaStorm 看起来很像 Digg。许多人对雅虎的举动持批评态度，但是昨天报道中的迈克尔·阿灵顿和本帖中的你都支持这些公司的做法。

这些类型的网站非常聪明。如果网络爱好者批评大公司试图为合乎逻辑和广泛传播的过程申请专利(比如社交网络或混搭)，那么回头批评他们谦逊地跟随开创性初创公司的领导岂不是不公平？

你看不到对梅内米或[胡格](https://web.archive.org/web/20220926085546/http://hugg.com/)在掘客之路上遭遇惨败的愤怒。无数公司现在允许用户上传视频，只有最缺乏想象力的人才说他们都在试图剽窃 YouTube。我认为戴尔的新网站是一个勇敢的举动，会有更多的公司效仿。希望他们能把这种在线双向交流的趋势推向极限，并倾听用户的意见。
