# Sprint 现已推出 Novatel Merlin EX720 ExpressCard

![](img/9a5644273a83c3ec7dbcf58f199c820a.png)
我爱我有的无线宽带。这是自无线网络以来最伟大的事情。我已经说过很多次了，但是我想象有一天人们会开始放弃有线网络，转而使用类似的无线网络。

那些日子越来越近了。WWAN 越来越快，兼容性扩展越来越全面。今天，Novatel Merlin EX720 EV-DO 版本 A 卡已登陆 Sprint。

这是一种速度为 1.4Mbps 和 500kbps 的 ExpressCard。两年合约和折扣后，你现在可以花 179.99 美元买一台。价格有点贵，但它可以与 MacBook Pro 和任何你可以扔给它的东西一起使用。
