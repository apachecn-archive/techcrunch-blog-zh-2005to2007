# 大型韩国社交网络 CyWorld 在美国推出

 [](https://web.archive.org/web/20221222145528/http://us.cyworld.com/main/index.php) 我们这些美国人终于可以在新推出的美版韩国社交网络 [CyWorld](https://web.archive.org/web/20221222145528/http://us.cyworld.com/main/index.php) 中拥有自己的*迷你朋友圈*。迷你屋现在在美国被称为迷你屋，它们是用户头像(迷你)和卡通魅力的奇怪小空间，人们显然花了真金白银。大多数护身符似乎要花 5 个橡子 CyWorld 币(大约 50 美分？)尽管用现金购买更多的橡子要到下个月才能在美国网站上实现。所以，在增强你的能力时，要小心使用你的橡子。

新的美国网站在第一天遇到了一些技术问题，帐户创建有点混乱，但你可以进去看看网站。

根据 Katie Fehrenbacher 的报道，CyWorld 的母公司 SK Communications 在旧金山设立了一个 30 人的办公室，在美国版本上花费了大约 1000 万美元，并保证不惜一切代价在新市场上取得成功。还将通过 CyWorld 推出手机游戏和音乐销售。该公司已经在日本、中国和台湾推出了本地化版本。世界其他地区的本地化工作正在进行中。

据报道，高达 90%的 20 岁以下的韩国人在 CyWorld 上注册，这一市场份额连 MySpace 都羡慕不已。CyWorld 能否将其在一个国家的成功转化为其他国家的成功，这是一个关于社交网络领域本地化挑战的巨大考验。

我在这里大胆地猜测，绝大多数美国用户会更喜欢 MySpace 的开放空间和笨拙的代码，而不是 CyWorld 的迷你游戏、橡子和愚蠢的卡通头像。我想，如果你能说“那不是我，那是我的迷你米”，MySpace 的就业筛选风险就可以避免了随便啦。

如果 MySpace 和脸书都在努力将自己定义为包含有钱的年轻人的地方，这对 CyWorld 来说会有多难？也许在世界的其他地方，非常小的孩子经常在网上进行微支付购物(参见芬兰的[Habbo Hotel](https://web.archive.org/web/20221222145528/http://www.habbo.com/)—[20 美分交易中的 3000 万美元](https://web.archive.org/web/20221222145528/http://techcrunch.com/2006/07/10/habbo-hotel-microtransacts-itself-to-30m-in-revenue/))，但我认为这在美国并不常见。也许他们的目标是喜欢 HelloKitty 的成年人，也许我错了，我会感到惊讶。

该系统的一个明显的局限性是，美国用户不能与其他国家的用户交朋友，数据不能从一个国家的 CyWorld 移植到另一个国家。不过，我想这一切都是因为橡子。
