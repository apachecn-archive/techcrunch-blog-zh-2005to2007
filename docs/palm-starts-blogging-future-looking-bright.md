# Palm 开始写博客，未来一片光明

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/12/palm-starts-blogging-future-looking-bright/>

在过去的几周里，Palm 在传递信息方面确实很积极。嗯，不确定“积极”这个词是否合适，也许“主动”更合适，因为我们已经看到了新的 CDMA 3G 智能手机的[泄露，关于新平台](https://web.archive.org/web/20150227044852/http://crunchgear.com/2007/04/03/meet-the-treo-755p/)的[公告，以及现在的](https://web.archive.org/web/20150227044852/http://crunchgear.com/2007/04/10/hot-palm-develops-palm-oslinux-mash-up-platform-in-house-new-devices-on-horizon/)[博客](https://web.archive.org/web/20150227044852/http://blog.palm.com/)。我们猜测这个博客是 Treo maker 的一个尝试，以保持自己在小工具世界的雷达上，这是一个很好的举动。它目前只有一个“hello world！”贴吧，不过是新的。它会长大的。接下来呢。对米特罗姆尼的评论？Palm 对 Imus 问题的看法？

虽然我们不认为博客中会有任何未经 Palm 强大的公关团队仔细挑选的揭示性内容，但我们认为这更多的是一种文化，就像 Zune Insider 的 Cesar 为用户提供了一种与他们最喜爱的小工具背后的产品团队进行互动的方式。这对微软来说效果很好，我们认为 Palm 博客对这些猫来说会很好。

有趣的是，第一篇文章指出，这个博客不仅仅是针对常驻 Palm 博客作者的，而且我们会看到从实习生到高层的每个人的条目。我们喜欢这个想法，如果它足够简洁以保持我们的兴趣(尚未看到)，那么我们甚至可能会看到一些“官方泄漏”和病毒式的东西。将它添加到您的 RSS 阅读器中，并保持关注。
