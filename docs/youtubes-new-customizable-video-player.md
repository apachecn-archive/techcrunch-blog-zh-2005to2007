# YouTube 新的可定制视频播放器

![youtube](img/479311885b0fbf3a81802ca1cbcad12b.png)我们的数码宠爱爸爸， [TechCrunch](https://web.archive.org/web/20201129170502/http://www.beta.techcrunch.com/2007/07/20/youtube-launches-custom-player-feature/) 对 YouTube 新的可定制视频播放器进行了报道。

对于严重依赖 [YouTube](https://web.archive.org/web/20201129170502/https://crunchbase.com/organization/youtube) 视频并希望将所有内容聚集在一个地方的网站来说，这是个好消息。以前，你可能需要编写自己的代码或者依赖外部提供者，比如 [Splashcast](https://web.archive.org/web/20201129170502/http://www.beta.techcrunch.com/2007/01/29/splashcast-launches-one-player-to-bind-them-all/) 来完成这项工作。

YouTube 的新功能允许你选择一种颜色、一种布局，并使用一种非常简单的形式[轻松地将你的视频、收藏夹或播放列表添加到你的网站上。](https://web.archive.org/web/20201129170502/http://www.youtube.com/custom_player)

期待在不久的将来看到这些玩家到处出现(手指交叉！).
