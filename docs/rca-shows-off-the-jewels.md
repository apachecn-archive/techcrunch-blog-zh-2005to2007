# RCA 展示珠宝

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/03/rca-shows-off-the-jewels/>

RCA 推出了一系列新的数字媒体播放器，这表明并不是每个人都想(甚至需要)花几百美元买一个苹果 iPod。这一经济实惠的系列带有 Gem Line 品牌，包括多功能的 RCA Opal 音乐和视频播放器(75 美元)，可以存储多达 680 首歌曲或 8 小时的视频。该播放器兼容音乐下载和订阅服务，包括 eMusic、Yahoo！音乐、狂想曲和 Napster。Opal 的内置可充电锂电池一次充电可运行长达 15 小时。

Gem 系列的其余产品包括紧凑型 RCA Pearl(49 美元)，它被吹捧为 USB 拇指驱动器和音乐播放器的组合，以及用于健身房的耐用 RCA Jet 播放器(69 美元)，以及 RCA Jet Stream(TBD 价格)，它将在今年晚些时候加入无线耳机功能。
