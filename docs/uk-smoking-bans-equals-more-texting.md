# 英国禁烟令意味着更多短信

看起来正常点烟的人可能会用这些手指来发短信。7 月 1 日禁烟令生效后，由于没有香烟可吸，短信似乎成了英国吸烟者更多使用的方式。1700 万手机用户通过 Orange 网络发送的短信数量激增了 750 万条！

该理论认为，吸烟者需要一些东西来让自己有事可做，也可能是一种分散注意力的方式。也有人提出，发短信被用来支持那些试图戒烟的人！
