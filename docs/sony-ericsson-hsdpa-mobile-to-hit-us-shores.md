# 索尼爱立信 HSDPA 手机登陆美国

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/27/sony-ericsson-hsdpa-mobile-to-hit-us-shores/>

索尼爱立信首款支持 HSDPA 的手机将于第三季度登陆我国。Z750 翻盖手机也将是 SE 第一款采用下一代 Java 伞形标准移动服务架构的手机。新的 Java 平台将支持即时通讯、移动银行的方方面面，这让我相信美国电话电报公司将是运营商的首选。谁会关心这些东西呢，其他功能呢？

光滑的翻盖有一个很好看的外部显示器，可以告诉你谁在打电话，现在是什么时间，以及大多数外部显示器的所有常规功能。其他一些功能包括 200 万像素摄像头、调频收音机、媒体、3D 游戏、RSS 阅读器，以及对 Microsoft Exchange ActiveSync 和电子邮件(POP 和 IMAP)的支持。没有关于运营商(再次，我们猜测这是新的美国电话电报公司)或价格的消息，但它将有幻影灰色和玫瑰粉色。
