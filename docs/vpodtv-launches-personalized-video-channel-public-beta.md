# Vpod.tv 推出个性化视频频道公测版

[![](img/643980aaf6a4eafe28dae7479dbe4248.png)](https://web.archive.org/web/20211128172639/http://vpod.tv/) 推出一年后， [Vpod.tv](https://web.archive.org/web/20211128172639/http://vpod.tv/) ，基于 flash 的在线视频发布工具的制造商，发布了一些以前的高级工具进行公测。逐渐渗透到公众账户的工具将在他们的 myTV 产品的保护伞下。

现在，任何帐户都可以上传、发布和货币化他们自己的托管(600 Mb) [myTV 视频频道](https://web.archive.org/web/20211128172639/http://corp.vpod.tv/createyourownchannel.php)，就像这一个在法国演讲的[系列企业家](https://web.archive.org/web/20211128172639/http://portal.vpod.tv/loiclemeur/198209)。你也可以在下面看到他们嵌入式播放器的新版本。

频道由一系列流式视频(无倒带或快进)组成，可通过播放器底部的显示菜单导航，并由多个重叠菜单管理。这些菜单可以让你抓取一个单独节目的链接，RSS 源，或者通过电子邮件分享。发行商通过上传单个节目(使用新的桌面上传程序)到他们的帐户并在播放列表中订购视频来创建频道。然后，他们可以通过将播放器链接到自己的域并添加自己的徽标来个性化播放器，并通过 Netvibes 小部件进行整合。

还有其他几家创业公司涉足个性化互联网视频。 [Cozmo](https://web.archive.org/web/20211128172639/http://www.beta.techcrunch.com/2007/04/26/cozmo-wants-to-kill-your-tv/) 、 [Kyte](https://web.archive.org/web/20211128172639/http://newteevee.com/2007/05/15/kyte-tv-vc-cash/) 、 [me.tv](https://web.archive.org/web/20211128172639/http://me.tv/) 和 [Splashcast](https://web.archive.org/web/20211128172639/http://www.beta.techcrunch.com/2007/04/30/splashcast-plays-itunes-and-beyond/) 都有基于 RSS 的视频播放器，可以从其他网站托管的视频创建频道。尚未推出，世界电视还将推出个性化的视频播放器。
