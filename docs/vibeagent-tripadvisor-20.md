# VibeAgent:猫途鹰 2.0

[![vibeagentlogo.png](img/0bae3e649d8bd2a9164fbff324c6c626.png)](https://web.archive.org/web/20201125083816/http://vibeagent.com/) 互联网上任何有利可图的领域都必然会吸引新创公司的目光。这对于市值 788 亿美元的在线旅游行业来说尤其如此。它刺激了十几个元搜索引擎、旅行跟踪器和旅行规划工具的开发。

最新的旅游网站 [VibeAgent](https://web.archive.org/web/20201125083816/http://vibeagent.com/) ，你应该很熟悉吧。这是社交网络扭曲的猫途鹰。像猫途鹰一样，你可以使用该网站查找、查看和预订酒店(猫途鹰也提供航班服务)。然而，在猫途鹰，每个人都得到同样的评价。很难猜测海报背后的动机，或者它们与你的品味和偏好有多大关联。他们最近从公开上市的 TRX 公司总裁兼首席执行官 Trip Davis 那里获得了一轮未披露的种子资金。

[VibeAgent](https://web.archive.org/web/20201125083816/https://crunchbase.com/product/vibeagent) 通过展示评论者的个人资料，并根据您与评论者的关系对搜索结果进行排名，帮助为评论和搜索结果添加上下文。评论由文本、评级和该地区的照片组成。每条评论都由发布评论的人贴上标签，并在搜索结果中排名，相似的评论者、朋友和你加入的团体成员的评论会被推到最前面。这个想法是，这些人有你可以信任的观点，并且最接近你的观点。当然，如果你不加入这个系统，他们不会有太多的细节来作为他们结果的基础。

到目前为止，VibeAgent 坚持住酒店，因为他们希望专注于主观方面的购买。航班相当商品化，几乎每家航空公司都让你坐在同样拥挤的椅子上。酒店很难从价格标签甚至照片上评估，这就是 VibeAgent 想要个性化评论的原因。他们的下一个目标很可能是度假屋。

评论服务的一个主要问题是，在它成为一个自我维持的社区之前，要建立一个评论的临界质量。Yelp 通过关注几个大都市和大量的派对做到了这一点。VibeAgent 计划通过“旅游相关”奖励来奖励顶级用户。

虽然该服务仍处于私人测试阶段，但他们已经在 [InviteShare](https://web.archive.org/web/20201125083816/http://www.inviteshare.com/site.php?id=45) 上向我们发出了 100 个邀请。
