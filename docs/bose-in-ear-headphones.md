# Bose 入耳式耳机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/24/bose-in-ear-headphones/>

Bose 在一副入耳式耳机上打上他们的标志只是时间问题。虽然我喜欢我的 QC3s，但在我购买之前，我真的必须尝试这些东西。定价为 99.95 美元，它们镶有柔软的深色硅胶，并在一个小包装中提供 Bose 声音。不惊人，但值得研究。
