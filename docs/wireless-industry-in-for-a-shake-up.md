# 无线行业将迎来一场变革

![](img/11883d53257b0f1fc93dd47a5a6f052d.png)

联邦通信委员会主席凯文·帕丁有一个梦想。他设想美国拍卖的 700MHz 频谱将很快进入大众，而不是充满规则和限制的公司。锁定手机和冗长合同的日子已经一去不复返了。相反，我们都生活在一个任何设备都可以接入移动宽带网络的地方。

撇开摇滚幻想不谈，如果电信公司不再耍我们，而是给我们选择的权利，那就太好了。您知道吗，一些运营商要求制造商移除设备上的 WiFi，这样消费者就被锁定在他们的服务上了。可笑！希望我们能在未来几年看到凯文·帕丁和国会之间的无线改革。
