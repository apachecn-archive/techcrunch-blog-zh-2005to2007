# 专利猴:液晶+太阳能电池=智能

![](img/96b7b121de31713a08c2d5b4218b5d94.png)

手机电池寿命自问世以来一直是最大的问题之一。摩托罗拉从一开始就在那里，并且至少在探索聪明的解决方案。

摩托罗拉最近获得了一项在无线设备上结合液晶显示屏和太阳能电池的专利。虽然这种概念的成本使 iPhone 看起来很便宜，但拥有一个显示屏的概念也可以有双重目的，即接收光线为电池充电，这很有意义。

Moto 的专利指出，这种技术在纯黑色液晶显示器上表现更好，光通过率高达 75%。在市场上，自从这项技术提出以来，薄膜太阳能电池已经有了长足的发展。在太阳能电池的薄膜加工中，[维基百科注释](https://web.archive.org/web/20211026212259/http://en.wikipedia.org/wiki/Solar_cell#Transparent_conductors):

> 许多新型太阳能电池使用的透明薄膜也是电荷导体。目前研究中使用的主要导电薄膜是透明导电氧化物(缩写为“TCO”)，包括掺氟氧化锡(SnO2:F 或“FTO”)、掺杂氧化锌(例如:ZnO:Al)和氧化铟锡(缩写为“ITO”)。这些导电膜也用于平板显示器的 LCD 工业。TCO 的双重功能允许光穿过衬底窗口到达下面的活性光吸收材料，并且还用作欧姆接触以将光生电荷载流子从该光吸收材料中传输出去。目前的 TCO 材料对于研究是有效的，但是可能还没有优化用于大规模光伏生产。它们在高真空下需要非常特殊的沉积条件，它们有时机械强度差，并且大多数在光谱的红外部分具有差的透射率(例如:ITO 薄膜也可以用作飞机窗户中的红外滤光器)。这些因素使得大规模制造成本更高。

这个概念已经引起了包括精工、美能达和夏普在内的一些不同公司的兴趣。在过去的三十年里，摩托罗拉在太阳能方面的经验已经有点分散。

方便又环保，现在我们只需要在市场上看到更多。
