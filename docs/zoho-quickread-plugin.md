# Zoho QuickRead 插件 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/09/19/zoho-quickread-plugin/>

 [](https://web.archive.org/web/20221205085304/http://www.zoho.com/) 简单提一下 Zoho 今天在他们的在线办公套件中增加的一个小而重要的功能——一个叫做 QuickRead 的插件。它可以作为 Firefox 和 IE 的插件在 Zoho Writer 上使用，也将很快出现在他们的电子表格和演示应用程序上。

该插件的用户可以直接在互联网上打开 office 文档(目前为 Word ),而无需先将其下载到硬盘上，然后用桌面应用程序打开。相反，你只需右击链接并选择“用 Zoho Writer 打开”。关于 QuickRead 更详细的描述，请看 Zoho 的博客文章。

为什么这很重要？因为它消除了人们必须在他们的机器上安装 Microsoft Office 的另一个原因。当然，在这个插件可用之前，人们可以简单地下载文档，然后上传到 Zoho(或 Writely)并打开它。该插件只是将所需的步骤减少到一个——右键单击。Zoho 关注细节，这表明他们会成功。自从大约一年前我们第一次写他们以来，他们已经走了很长很长的路。
