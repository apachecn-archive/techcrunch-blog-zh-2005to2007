# Squadra Piloti: Web 2.0 异国汽车交易

 [](https://web.archive.org/web/20220810110525/http://www.squadrapiloti.com/) 这里有一个给[百万富翁](https://web.archive.org/web/20220810110525/http://www.beta.techcrunch.com/2006/04/12/millionaires24-email-for-the-rich/)(富人的电子邮件)人群的: [Squadra Piloti](https://web.archive.org/web/20220810110525/http://www.squadrapiloti.com/) (意大利语“司机团队”的意思)。这是一家由哈佛商学院二年级学生杰里米·伯尔创办的新公司。他为外国汽车爱好者创建了一个在线市场，“允许你将任何现有高端汽车的里程转换为一种货币形式，可以用来购买你想驾驶的任何其他汽车的里程。”

拥有一辆不适合你的法拉利 F430？也许你想开着别人的兰博基尼 Murcielago 去尝试新事物。Squadra Piloti 专为您打造。只需要注册并告诉它你有什么样的车，你想开什么样的车，以及任何参数(对方开的时候你必须在车里等等。).Squadra Piloti 会为你找到兰博基尼并安排好一切。你所要做的就是出现并驾驶它。

我打算注册一辆 1996 年的福特探险者，试着用一辆阿斯顿·马丁来换取一些严肃的驾驶时间。但是，即使没有人想借我的探索者，我仍然可以使用 Squadra Piloti:他们也允许车主付费出租汽车。

没有关于这项服务如何赚钱的消息。

如果这一切对你来说还不够酷，看看网站上一位名叫斯科特·斯皮德的[司机看似随意的照片。](https://web.archive.org/web/20220810110525/http://www.squadrapiloti.com/images.html)
