# Yodel 轶事——雅虎公司博客

 [](https://web.archive.org/web/20220926184315/http://yodel.yahoo.com/) 雅虎于太平洋时间今晚 10:30 推出了他们的第一个企业级博客[。第一个帖子是 Niki Dugan 写的，这里是](https://web.archive.org/web/20220926184315/http://yodel.yahoo.com/)。像 TechCrunch 一样，雅虎选择了 [WordPress](https://web.archive.org/web/20220926184315/http://www.wordpress.org/) 作为他们的博客平台。

我的朋友哈维·霍夫曼(Havi Hoffman)是雅虎影响者营销团队的一员，他给我发了一封电子邮件，告诉我关于这个博客的事情，他写道:“这个博客是企业沟通高级主管尼基·杜根的创意。(《裸聊》从一开始就在她的书架上。)我们才华横溢的暑期实习生保罗·斯塔马蒂乌(Paul Stamatiou)是佐治亚理工学院的大三学生，他在 www.pstam.com 大学写关于科技的博客，他担当了首席开发员的角色，在雅虎上创建了一个定制的 WordPress 博客。虚拟主机专业服务器。

雅虎有许多产品级博客，有些在 WordPress 上，有些在 Yahoo 360 上。根据 Technorati 的说法，谷歌的[企业博客](https://web.archive.org/web/20220926184315/http://googleblog.blogspot.com/)是最受欢迎的[之一，当然是托管在谷歌拥有的 blogspot 上。](https://web.archive.org/web/20220926184315/http://www.technorati.com/pop/blogs/)
