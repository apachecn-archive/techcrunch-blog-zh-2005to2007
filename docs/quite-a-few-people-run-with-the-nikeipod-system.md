# 相当多的人使用 Nike+iPod 系统跑步

![](img/a590a3f14b15d33ac097a72306b64316.png)

总的来说，我可以肯定地说苹果和耐克的交易是成功的。世界上最大的鞋制造商与最流行的 MP3 播放器制造商合作，这已经是天作之合了。现在耐克宣布 Nike+系统的用户已经记录了超过 2200 万英里。这足够跑一圈，然后在 In-N-Out 汉堡店停下来。

目前，耐克正计划在今年夏天向 Nike+iPod 官方网站添加一系列新功能，包括虚拟马拉松和挑战，以保持事情的趣味性。随着这个国家变得越来越胖，对日常锻炼感到厌倦，这是一种保持健康和享受快乐的绝佳方式。实现个人目标很有趣，但踢别人的屁股更有趣。
