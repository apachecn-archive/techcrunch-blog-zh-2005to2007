# 拉链 MP3 播放器设计理念

![](img/32c0e13129abaa0f97c7793add7fedf3.png)

如果我们不只是摆弄外套上的拉链，而是用它来控制我们听的音频，会怎么样？这似乎是设计师埃里克刘的心态。他的拉链 MP3 播放器结合了物理交互来控制设备。当你拉上拉链，你会喜欢你的曲子。拉链拉下，设备自动暂停。你也可以通过前后摆动拉链来控制音乐的音量和顺序。

包括无线耳塞，大概是由蓝牙或其他技术驱动的。目前这只是一个概念，但如果刘灿设法以 100 美元的价格大规模生产 1GB 的产品，他将拥有自己的 iPod Shuffle 的体面替代品。
