# 基于 Mozilla 的威尼斯项目

[![veniceproject_logo.jpg](img/4f56c3b1fb49c4fd08539505256c44fb.png) ](https://web.archive.org/web/20210925020241/http://www.theveniceproject.com/) [NewTeeVee](https://web.archive.org/web/20210925020241/http://newteevee.com/2007/01/11/venice-project-mozilla/) 有一个帖子详细介绍了[威尼斯项目](https://web.archive.org/web/20210925020241/http://www.theveniceproject.com/)背后的技术，这是由 Kazaa 和 Skype 创始人尼可拉斯·曾斯特罗姆和贾纳斯·弗里斯创办的新在线电视创业公司。一个非常有趣的披露——就像另一家初创公司 [Songbird](https://web.archive.org/web/20210925020241/http://www.beta.techcrunch.com/tag/songbird) 一样，该产品的一部分是基于 Mozilla 应用框架构建的。

威尼斯项目的两个问题大大降低了我对它的热情。首先，这项服务还没有引人注目的内容(尽管许可协议可能正在等待宣布)。其次，他们避开了浏览器，需要单独下载软件才能使用该产品。除非它高度病毒化，否则它不会飞起来，因为 YouTube 是非常受欢迎的基于浏览器的替代品。
