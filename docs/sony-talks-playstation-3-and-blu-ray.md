# 索尼谈论 Playstation 3 和蓝光技术

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/18/sony-talks-playstation-3-and-blu-ray/>

![](img/8d10848e98c0332bb1553500affe7f93.png)

> “在我们的发布会上，我们已经接近了今年蓝光光盘的 25GB 限制，”他声称。“明年，我们将把这个数字提高到 50GB，我预计在不久的将来，我们也会接近这个数字。”

索尼全球工作室老板菲尔·哈里森终于站出来回应所有针对索尼利用 Playstation 3 推动蓝光市场的恶意批评。由于 PS3 的上市日期很近，哈里森站出来谈论蓝光在 Playstation 3 中的渗透是一个很好的决定。现在有太多的人憎恨索尼，他们不知道索尼做出某些决定背后的原因，尤其是 PS3。我不是在这里选择立场，我只是想给每个人一个平等的机会使用他们的新游戏机。关于蓝光作为 PS3 的一部分只是为了给非互动内容火上浇油的话题，哈里森说:

> 和电影没关系。DVD 的容量不足以支持这种数据消费，或满足手机和 RSX 的数据消费需求——仅仅作为一种游戏设备，我们需要蓝光来提供 PS3 游戏使用的数据。不仅仅是图形。它是关于 7.1 音频，它是关于语音，它是关于在游戏中内置高达 1080p 的电影；它是高分辨率的纹理，它是动画，它是构成丰富多样的下一代体验的一切。一部分是视觉，一部分是声音，还有一部分是游戏的好处——更多的关卡，更多的细节，更丰富的体验。

最后，我们从索尼那里得到了一些真正的答案。也是时候了，因为我真的不能再忍受那些“PS3 过热，因为它们紧挨着”的垃圾了。如果哈里森说的都是真的，那么索尼为我们提供不同的东西是件好事。也许 PS3 中有全新的游戏体验在等着我们。
