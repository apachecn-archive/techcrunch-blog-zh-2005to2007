# 索尼和任天堂对裁员数字做出反应

上周，我们报道了美国索尼电脑娱乐公司和美国任天堂公司都在向一些员工叫停游戏。就索尼而言，裁员是全球重组的一部分，而在任天堂，似乎有一部分员工对从西雅图的住所迁走不太感兴趣。

今天我们听到的消息是，这些数字有点“离谱”在 SCEA，不是 80 到 100 名员工被解雇，而是只有大约 50 人将不得不担心如何支付房租、购买食物以及拿出 600 美元购买 PS3。至于任天堂，总裁 Reggie Fils-Aime 表示，上周该公司 90%的销售和营销团队的离职人数也很高。那个部门只有 60%的员工要离开。Fils-Aime 没有评论这 60%是否真的包括关键高管乔治·哈里逊、佩兰·卡普兰和贝丝·卢埃林。

此外，在接受《圣何塞水星报》采访时，他还表示，其中约 55 个职位将转移到硅谷的新办公室，另外 25 个职位将转移到纽约市，这两项工作都将在 9 月份进行。
