# 电脑用户的天赐之物

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/26/replay-media-catcher-a-godsend-for-pc-users/>

![](img/d10552a3fe0dbd2b85461a98a3ad2c67.png)

我不知道你们中有多少人会看美国广播公司的节目，但所有受欢迎的节目都可以通过 ABC.com 在线免费观看，比如《迷失的 T1》、《T2》和《实习医生格蕾》、《T4》。然而，麻烦的是，你必须观看无聊的 30 秒广告，有时基于 flash 的播放器会出现问题。你唯一的其他选择是从 iTunes 上购买节目或盗版，对一些人来说这两种选择都不可行。

来自 Applian Technologies 的重播媒体捕捉器。这个软件知道发生了什么。它知道你想从 ABC、YouTube、MySpace 等网站上抓取所有 flash 视频。它知道你想快速有效地完成它。支付 39.95 美元，你就可以在舒适的 iPod 上免费观看所有你喜欢的电影和节目。如果你是一个想偷 YouTube 视频的 Mac 用户，我个人推荐使用 [TubeSock](https://web.archive.org/web/20150919121347/http://stinkbot.com/Tubesock/) 来完成这项工作。
