# MyGame:创建和发布简单的 Flash 游戏

在线游戏网站 King.com 将于周二正式推出社交游戏网站 MyGame，这是一个集用户创作内容、休闲游戏和收入分享于一体的网站。

MyGame 用户可以玩现有的游戏，也可以使用游戏生成器开发自己的游戏。用户选择一个游戏模板，用文字、声音和照片个性化它，然后开始玩。

用户制作的游戏可以在社交网络上分享，也可以添加到博客和网站上。

我们在 5 月报道了一个类似的服务，PlayMyGame [。两者都有一个共同的想法:用户生成游戏。MyGame 进一步发展了这个概念，用户可以上传自己的游戏。MyGame 向游戏开发者提供游戏产生的广告收入分成，如果他们将 MyGame API 集成到他们的游戏中，还有机会获得预付现金奖金。](https://web.archive.org/web/20221130202631/http://www.beta.techcrunch.com/2007/05/28/wanna-take-a-shot-at-me/)

虽然像 MyGame 和 PlayMyGame 这样的网站的新奇价值是显而易见的，但提供一个用户生成游戏的出口，不仅可以创建游戏，还可以上传游戏并从中获利，这无疑是这个领域的下一步。MyGame 可能只是从少量游戏开始，但网站有机会利用其用户社区来构建更大的东西。
