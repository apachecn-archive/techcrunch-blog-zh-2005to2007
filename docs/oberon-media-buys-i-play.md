# 奥伯龙传媒收购 I-Play

据 mocoNews 报道，总部位于纽约的 Oberon Media 已经收购了总部位于伦敦的手机游戏提供商 I-Play。交易细节尚未披露，但交易预计将于 6 月底完成。I-Play 的商业模式包括在不同的手机和运营商之间移植游戏，这可能会与 Oberon 自己的平台集成。这家纽约公司托管 MSN 游戏和其他游戏，据 mocoNews 称，未来将以 I-Play 品牌推出其所有的移动媒体和移动游戏。
