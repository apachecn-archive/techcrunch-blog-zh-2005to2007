# NoFoodHere:实时搜索结果

[![nfh.jpg](img/27c586ce0e104cb54853935c17b6af52.png)](https://web.archive.org/web/20220929212134/http://nofoodhere.com/)

NoFoodHere 并没有试图改进搜索结果(该网站目前正在从雅虎搜索结果中抽取结果，并正在考虑转向谷歌)，而是试图改变用户与搜索引擎交互的方式。

选项卡浏览允许用户在同一浏览器窗口中同时运行多个搜索查询；不是一个惊天动地的功能，但有些什么方便。NoFoodHere 有趣的地方在于结果本身。随着 NoFoodHere 的出现，点击一页又一页的搜索结果的概念已经不复存在了。该网站首先会自动加载一长串搜索结果，然后当用户向下滚动页面时，会实时加载更多的结果。实际上，你可以在一个页面上得到上千个，甚至更多的结果，而不需要点击页面。这是一个非常好用的功能；在测试中，当我向下移动页面时，我发现它比谷歌阅读器更可靠。

NoFoodHere 来自澳大利亚的弗兰克斯顿(Vic)，墨尔本大学工程系三年级学生乔丹·贝利斯-麦卡洛克。
