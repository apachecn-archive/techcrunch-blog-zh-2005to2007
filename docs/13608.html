7.微软高管对营销人员针对谷歌的“我们是第二号”广告战犹豫不决。

6.当微软的Hotmail服务把所有的婚礼请柬都作为垃圾邮件退回时，雅虎有了新的想法。

5.推荐这两个人配对的计算机运行Linux。

4.反对这场比赛的电脑运行的是Windows Server Longhorn Beta 3。

3.老派人物鲍尔默认为新娘应该支付400亿美元的嫁妆。

2.法院书记员运行Windows 98的电脑在生成结婚证时崩溃了。

1.Semel拒绝在雅虎电脑上运行Windows Vista，直到Service Pack 1发布。
