# 小游戏手柄，大功能

![](img/6039082683a7232202f840e5add4c405.png)

MaxFire Pandora Pro 是一款小巧、可折叠、振动的 USB 游戏手柄。它有四个操作按钮，四个侧按钮，一个涡轮按钮和一根可伸缩的 USB 电缆。重量只有一盎司，非常适合笔记本电脑和蓝精灵。

游戏手柄还配有一个可以容纳 miniSD 卡的小盒子。
