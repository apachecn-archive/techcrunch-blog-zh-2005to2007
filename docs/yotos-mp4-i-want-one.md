# Yoto 的 MP4:我要一个！TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/20/yotos-mp4-i-want-one/>

不要把 Yoto 和 Yoko 搞混了，Yoto 在中国推出了 1GB mp4 播放器 V200，它有一个巨大的 2.4 英寸 QVGA 显示屏，用金属包裹着，所有的按钮和端口都放在旁边，这样你就可以在上班的火车上享受所有的 Pr()n。

为什么我们美国人在遇到这种小玩意时会被耍呢？当然，我们可能会有更好的播放器，但只需 50 美元，我肯定会喜欢这个播放器来美国，这样我就可以在火车上看视频，而不必花几百美元买一个 Zune，iPod 等。

现在我的咆哮结束了，这是 V200 发布的一些统计数据。通过 DivX 和 H.264 播放，配备了 miniSD 卡插槽和 RK2608A 芯片，可以用作电子书、照片浏览器和玩游戏。

如前所述，它的售价约为 50 美元。
