# 观看你的网络游戏《太空入侵者》

【YouTube http://www.youtube.com/watch?v=dtC6ZM0_m8U]

NetQoS 为监控系统流量的单调任务带来了一点视频游戏风格的戏剧性。他们的程序， [Netcosm](https://web.archive.org/web/20220927232752/http://www.netqos.com/network%2Dmonitoring/network-monitoring-labs.html) 监控流经你的路由器的流量，并将这些数据重组为有趣的善恶数据包冲突，就像你上面看到的那样。唯一的问题是，你的系统管理员是否觉得看到你的系统被“划斜线”比什么都不做更有趣。

Netcosm 是 NetQoS Performance Labs 的产品，目前只在他们的服务器上运行，但是他们有一个漂亮的 [FAQ](https://web.archive.org/web/20220927232752/http://www.netperformance.com/content.aspx?id=2690&cb_name=1) ，在那里你可以问他们关于他们正在享受的所有乐趣。
