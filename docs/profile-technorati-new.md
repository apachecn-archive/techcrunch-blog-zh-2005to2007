# 简介:Technorati(新)

**公司:** [Technorati](https://web.archive.org/web/20220927192013/http://www.technorati.com/)

![](img/0db7e4f67e5a733d4d9d5642a089c27a.png)

Technorati 全面推出了他们的新测试版，外观与 10 天前宣布的测试版略有不同。

新功能的完整列表可以在[这里](https://web.archive.org/web/20220927192013/http://www.sifry.com/alerts/arclinkhives/000319.html)(戴夫·西弗里的博客)和新的 Technorati 公司博客([链接](https://web.archive.org/web/20220927192013/http://www.technorati.com/weblog/2005/06/20.html))上看到。我早些时候在 technorati 网站上看到了这个，并在博客先驱报上看到了这个帖子。

新网站很棒，增加了许多新功能，我们期待未来的改进。很难说我喜欢它多过还是少于“旧的”Technorati——你会慢慢喜欢上它，新东西需要时间来适应。但到目前为止，我喜欢它，新功能集很强。
