# PS3 价格看不到下降的迹象；固件更新至 1.70

[![ps3boxes.jpg](img/8498187b1c21c19fcfca27d3375b301a.png)](https://web.archive.org/web/20201205093129/http://old.crunchgear.com/wp-content/uploads/ps3boxes.jpg "ps3boxes.jpg")

尽管已经比预定时间晚了 200 多万台，索尼表示它还没有“降低 PlayStation 3 价格的具体计划”。“该公司显然已经将该系统的未来与蓝光的成功挂钩:如果蓝光未能站稳脚跟，PS3 将陷入困境。虽然这可能是索尼的公开说法，但金融分析师表示，索尼被迫降价只是时间问题，部分原因是需求疲软，但主要是为了赶上 Xbox 360 和 Wii 的销售。以免我们忘记，在商店里找到 Wii 几乎是不可能的。

但是如果你有 PS3，你会注意到索尼发布了另一个固件更新，升级到 1.70 版本。此次更新提高了向后兼容性，并允许用户将 PSP 上玩的 PSone 游戏之间的游戏保存转移到 PS3 本身。
