# 东芝 1.8 英寸硬盘突破 100GB 大关

好的，如果你想在你的移动设备上增加存储空间，请举手，比如你的 MP3 播放器、PMP、PDA(你还在用 PDA？)、UMPC(如果你是五个购买者之一)或者你的超便携笔记本电脑。谢天谢地，有像东芝这样的公司将 1.8 英寸硬盘的存储容量增加到 100GB。

基于[垂直磁记录](https://web.archive.org/web/20210305101548/http://en.wikipedia.org/wiki/Perpendicular_magnetic_recording) (PMR)，东芝的新 MK1011GAH 使用两个盘片来实现大小，但它比以前的 1.8 英寸驱动器占用的空间小 10%。完整的规格可以在跳转后看到，但基本上是“我为什么要关心？”在这一点上，像这样的突破意味着你的下一代 ~~iPod~~ 便携式媒体播放器可能会拥有超过 100GB 的存储空间，同时体积更小、重量更轻。
