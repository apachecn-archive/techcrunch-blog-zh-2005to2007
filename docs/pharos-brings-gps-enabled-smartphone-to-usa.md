# Pharos 将 GPS 智能手机带到美国

我们喜欢 E-Ten glfiish X500，这是一款 Windows Mobile 5 智能手机，内置真正的 GPS 接收器。Pharos 将是美国第一个携带这种手机的品牌，称之为 GPS 手机，因为它富有想象力。明天将宣布，它将作为一款解锁的 GSM 智能手机出售，在“主要零售店”售价约为 700 美元，这意味着你可能真的会在周围看到这款高端手机。“解锁”销售意味着你插入标准的 T-Mobile 或 Cingular SIM 卡，一切就绪，无需配置或拨打所谓的“客户服务”热线。为了让价格更加合理，你还可以获得 WiFi、蓝牙、200 万像素摄像头和 MicroSD 插槽。

这里的目标不仅仅是有地图和方向，而是一个完整的基于位置的软件套件，它将带你去你想去的地方，这使它成为我们编辑的理想选择。我们希望看到更多的 GPS 单元与智能手机结合，随着今天的先进功能继续成为明天的标准，这是一件很好的事情。
