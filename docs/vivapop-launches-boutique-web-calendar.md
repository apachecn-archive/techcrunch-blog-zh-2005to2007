# Vivapop 推出精品网络日历

 [](https://web.archive.org/web/20221129052402/http://www.vivapop.com/) 总部位于康涅狄格州格林威治的 [Vivapop](https://web.archive.org/web/20221129052402/http://www.vivapop.com/) 推出了一项基于网络的日历和活动管理服务，该服务提供了一个非常好的界面、跨平台双向同步和语音识别，可用于移动访问。面对谷歌对 GCal 和 GMail 的整合，这个或任何网络日历是否有机会一直是个问题，但 Vivapop 的利基受众和优质功能集使其成为一部有趣的作品。

Vivapop 由前企业金融家伊丽莎白·沙无然·塔贝尔创建，已经发展了两年。通过手工批准参与组织的出版日历，提供高级功能和突出“高端”零售商的广告，Vivapop 旨在及时服务于新英格兰上层阶级和各地的富人。

不过，这种功能普遍令人感兴趣。与 Google Calendar 非常相似，可以使用大量 Ajax 管理多个日历，并从发布者向订阅者推送自动更新。在推出时，该服务加载了 100 多个公共和私人活动日历，这些日历来自主要位于康涅狄格州格林威治的组织。到目前为止，艺术组织、儿童活动中心和私立学校在列表中占主导地位，但任何地方的任何人都可以提供他们组织的日历以供出版。

Vivapop 除了门禁小区还有什么不同？有许多小事情，如零售商店列表和参与大都市地区的时间，以及根据您的地理位置自动调整时区。这里最有趣的功能是自动同步和 IVR。

Vivapop 用户可以在桌面、黑莓和 Treo 上的 VIvapop 和 Outlook 之间双向同步。下一步将是同步所有手机日历，然后是 iCal。太令人兴奋了。该公司使用来自加州雷德伍德城的同步技术。

第二个独特的功能是能够通过语音命令写入您的日历。用户可以拨打 Vivapop 号码，输入他们的接入码，然后通过按键或语音响应提示来添加日历项目。项目日期、标题和类型可以被 IVR 系统识别，而事件描述被记录为. wav 文件并放在您的日历上。

Vivapop 账户的前 30 天起价为 1 美元，随后每月 9.99 美元用于 Outlook 同步的基本服务，14.99 美元用于语音接入。该服务有一个漂亮的外观和感觉，有一个强大的功能集和一个清晰、可行的业务模型。我认为它很有机会赢得目标观众。
