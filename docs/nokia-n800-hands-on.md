# 诺基亚 N800 实践

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/23/nokia-n800-hands-on/>

![](img/0ed2987373495695f1bed00c4413bb80.png)

昨天拿到了 [N800](https://web.archive.org/web/20210301021104/http://crunchgear.com/2007/01/08/nokia-n800-uber-brief-hands-on/) ，我不能说我被[吹走了](https://web.archive.org/web/20210301021104/http://crunchgear.com/2007/01/18/mobilecrunchs-take-on-nokia-n800/)。我喜欢 N770——许多人不喜欢——这是一个令人印象深刻的升级，但它目前不是必须拥有的。一件有趣的小事？有一个巧妙的茎摄像头，弹出的一面。Maemo 几乎没有什么改进，但总体体验似乎比速度较慢的 770 快得多，也干净得多。
