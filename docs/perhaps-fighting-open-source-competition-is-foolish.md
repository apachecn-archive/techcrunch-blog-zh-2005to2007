# 对抗开源竞争可能是愚蠢的

去年一月，我们写了关于 [Nuvvo、](https://web.archive.org/web/20210302044940/http://www.nuvvo.com/)一家勇敢的初创公司挑战开源竞争对手的报道。然而，Nuvvo 现在正在出售。它可能很快就会进入[死池](https://web.archive.org/web/20210302044940/http://www.beta.techcrunch.com/tag/deadpool)，沉淀创始人在 Nuvvo [博客](https://web.archive.org/web/20210302044940/http://blog.nuvvo.com/2006/10/03/for-sale/)上写的话:

> 这并不意味着 Nuvvo 正在关闭。我们在世界各地的成千上万的用户对我们来说是珍贵的，我们将在可预见的未来继续提供服务。
