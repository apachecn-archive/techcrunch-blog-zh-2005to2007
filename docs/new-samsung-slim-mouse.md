# 新款三星超薄鼠标

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/18/new-samsung-slim-mouse/>

![](img/0bde9a530e8c3a36010e7f448140038b.png)

我从来不喜欢笔记本电脑专用鼠标。他们总是觉得自己又小又脆弱。他们通常看起来也很蠢。谢天谢地，三星为笔记本电脑和超便携电脑推出的新鼠标支持了我——那个神圣的发光环实际上是一个水平滚轮。这款鼠标只有 8 毫米，三星设法在里面塞进了五个按钮。它将推出黑色和粉红色。你知道，一想到一个女孩用粉红色的连接到她的 UMPC 让我想到未来，出于某种原因。

[三星宣布推出超薄鼠标](https://web.archive.org/web/20210224210204/http://www.newlaunches.com/archives/samsung_announces_super_slim_mouse.php)【通过[科幻博客](https://web.archive.org/web/20210224210204/http://blog.scifi.com/tech/archives/2006/10/18/new_mouse_turns.html)发布新品】
