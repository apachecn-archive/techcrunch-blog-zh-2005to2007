# Xbox Live 春季更新现已推出

![](img/cac4901de85a3d74cdd12dbd2f71be6f.png)

听好了，你们这些*光环*火枪手。Xbox Live 的春季更新已经推出，你们中的许多人将会欢迎大部分更新。要接收更新，请完全退出 XBL，等待几分钟或最多一个小时，然后重新登录开始接收补丁。其中一个新功能是 Live Messenger 集成，这样您就可以在 MSN Live 或 Windows Messenger 上与人交谈，不管它叫什么。

还有新的成绩弹出窗口，带有分值，更有组织的市场，XBL 街机上的“朋友现在玩”功能，你可以设置你的主机在漫长的下载完成后自动关机，这样你的妈妈就不会因为增加电费而打你了。在这篇文章的正下方的新闻稿中还可以找到一些其他的特征。现在开始更新！
