# 瑞典老太太获得世界上最快的互联网连接

虽然比她实际需要的多了 39.99999 千兆字节，Sigbritt Lö thberg 现在已经有了 40 Gbps 的互联网连接。她从来没有拥有过电脑，直到现在，但她生了一些著名的瑞典互联网家伙。

连接速度有多快？她可以在两秒钟内下载一张全高清 DVD。在典型的宽带连接中，这需要花费很多、很多、很多秒(甚至几分钟或几小时)。听着——她的房子离连接的源头有 1200 英里远。

根据这项技术的创造者，距离理论上是无限的。数据通过使用“新调制技术”的光纤线路传输，显然部署起来并不昂贵。

如果它真的可以不受限制地远离源头而不损失任何传输速度，有一天我们会从瑞典接入互联网吗？我肯定会全力以赴。

先是美女，然后是宜家，现在又是这个？瑞典，你宠坏我们了！
