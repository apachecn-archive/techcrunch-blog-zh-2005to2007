# Truemors 的屏幕截图:盖伊·川崎的谣言服务

 [](https://web.archive.org/web/20220129022256/https://beta.techcrunch.com/wp-content/uploads/2007/05/truemorsb.png) 上周，我们提到了盖伊·川崎的最新企业 [Truemors](https://web.archive.org/web/20220129022256/http://www.beta.techcrunch.com/2007/05/03/guy-kawasakis-newest-venture-truemors/) ，以及我们从私人测试人员那里获得的一些早期评论(“具有类似 twitter 功能的谣言报告公告板”)。

我们现在已经通过一些“借来的”凭证进入了私有测试版，并且已经为我们自己四处看了看。该网站建立在 WordPress 平台上，是一个基于类别的谣言网站，任何人都可以打电话、发短信或发电子邮件来造谣。

该公司与 Spinvox 合作，允许人们通过语音留言。然后 Spinvox 将它们转换成文本发布在网站上。

如果你愿意，你可以拨打 1-650-329-2020，留下你自己的谣言。短信要按以下格式发:短信“2020”到 55022。谣言也可以传到 post@truemors.com。

一旦谣言出现在网站上，其他用户可以留下评论并投票支持或反对。像 Digg 一样，所有的新帖子都被列出来供人们评论和投票。如果有足够多的人认为一个谣言是有趣的，它就会成为一个顶级列表(相当于 Digg 主页)。
