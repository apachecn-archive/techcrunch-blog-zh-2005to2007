# RIAA 又搞砸了！

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/05/riaa-messes-up-again/>

![](img/d78f9aa3bd6117b1844f76637062e970.png)

就在你认为 RIAA 的情况不会变得更糟的时候，它确实变得更糟了。九寸钉乐队目前正在欧洲巡回演出(顺便说一句，他们的新专辑绝对棒极了)，并一直在场馆的洗手间里藏着特殊的 USB 钥匙。发现 u 盘的人将获得即将发行的专辑《零年》中泄露的歌曲。

如果一个粉丝在浴室隔间发现免费的 MP3，他们当然会把它泄露到网上。现在，RIAA 正在向托管“泄露”曲目的网站发送停止令。甚至特伦特·雷诺本人也认可了这一营销活动，并认可了 u 盘上的音乐。听起来 RIAA 需要冷静下来，并意识到新专辑中泄露的歌曲实际上可能会增加新专辑的宣传。

文章说整张专辑没有泄露，[这是一个半谎言](https://web.archive.org/web/20201026111053/http://junkthatrocks.com/v3/?p=49)，但新专辑中的几首歌泄露出来并不是什么大事。接受它，享受 CD 销量的增长。
