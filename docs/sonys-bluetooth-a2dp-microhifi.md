# 索尼的蓝牙 A2DP 微 HiFi

索尼的 CMT-U1BT 是索尼(我们见过的)第一款采用 A2DP 蓝牙 2.0 规范的播放器，这意味着你可以从房间的另一端将音频传输到该设备。您可以使用支持 A2DP 的手机或带有支持 A2DP 的加密狗的 PC，无论哪种方式，您都不需要直接插入接收器。

如果你没有 A2DP 设备，你可以通过 USB 连接或使用免费的 A2DP 蓝牙接收器加密狗，它可以插入任何耳机插孔。该设备本身具有 2×25 瓦 RMS 扬声器和一个垂直插槽加载 CD 播放器，可以播放 MP3 CDs 和普通 CD。

这款设备的主要亮点是 A2DP，这意味着您可以将扬声器放在客厅中，并在家中的任何地方随身携带手机，无论走到哪里都可以有效地控制音乐。当你挤出一个 dook 时，你再也不用被一个糟糕的轨道卡住了——只需在你的口袋里伸手，快速前进。
