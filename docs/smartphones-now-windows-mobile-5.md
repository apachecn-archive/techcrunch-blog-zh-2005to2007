# 现在的智能手机:Windows Mobile 5

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/02/smartphones-now-windows-mobile-5/>

![](img/4218141bc0cd15938e3a6b3a78458b8a.png)如果你在过去一两年中听说过基于 Windows 的智能手机，那么它很可能运行的是 Windows Mobile 5.0。实际上，Windows Mobile 5.0 (WM5)是微软称之为 Windows CE 的一个子集，它运行的是 Windows CE 5.0 全部功能的一部分。Windows CE 的各种子集用于操作系统，如 Windows Mobile Pocket PC Edition、Windows Mobile Smartphone Edition 和 Portable Media Center。困惑了吗？

嗯，你需要知道的是，目前有两种不同类型的 WM5 设备可以购买。其中一个版本是 WM5 Pocket PC Edition (WM5-PPC)，它可以在更大的手机上运行，这些手机通常有触摸屏(几乎全部)或键盘(大约一半)。另一个版本是 WM5 智能手机版(WM5-S)，通常采用糖果棒、翻盖手机或扁蛤壳形状(像摩托罗拉 Q)。哪一个适合你？请继续阅读，寻找答案。

## WM 还是小而美:你决定

![](img/c6b71adf25c3993acb3372fefbdea998.png)那么，你为什么会购买 WM5 手机，而不是摩托罗拉 RAZR 或更小更漂亮的手机呢？功能，功能，功能！能够拥有一个像笔记本电脑一样功能的轻量级设备，你可以在口袋里进出，这是值得的。网页浏览、电子邮件、即时消息、媒体播放、RSS 阅读、Skype VoIP 通话、模拟游戏、组织和日程安排只是您可以做的几件事情。

将所有这些功能与无限制的数据计划结合起来，你就可以像黑莓一样收发电子邮件，无论你在哪里都可以联系到你——这大大减少了你的空闲时间。如果你有一个带蓝牙的 GPS 接收器，甚至是一个内置 GPS 的 WM5 设备，你也可以将你的手机作为便携式导航设备。如果您的提供商允许，您可以将智能手机连接到笔记本电脑，并将其用作便携式调制解调器，在任何地方都可以访问互联网。
