# 又在新闻里发帖子了

 [](https://web.archive.org/web/20220816083317/http://www.payperpost.com/) [PayPerPost](https://web.archive.org/web/20220816083317/http://wwaw.payperpost.com/) ，这家向博客写手支付广告客户产品的有争议的初创公司，明天将再次出现在新闻中。他们将宣布收购博客工具和服务公司[performance](https://web.archive.org/web/20220816083317/http://performancing.com/)(见我们之前关于 performance 的[报道](https://web.archive.org/web/20220816083317/http://www.beta.techcrunch.com/2006/10/18/performancing-launches-ad-network-for-bloggers/))。

此次收购的主要目的似乎是让 PayPerPost 获得 28，000 名表演用户，其中大多数是博客作者和 PayPerPost 的潜在客户。广受欢迎的表演型 Firefox 博客编辑插件[不会被收购，将被分拆成一个新的品牌。](https://web.archive.org/web/20220816083317/http://performancing.com/firefox)
