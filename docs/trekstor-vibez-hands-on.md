# TrekStor Vibez 手动操作

一个 MP3 播放器得到我的认可是非常罕见的。iPod 得到它是因为它“能正常工作”，Zune 不情愿地得到它是因为“它可能很快就能工作”然而，TrekStor Vibez 的石头不符合懦弱的审美。

![](img/ae18a30958c2e9be3f6daa1fbdd387c3.png)

这是一个 [8GB 音频播放器和照片浏览器](https://web.archive.org/web/20130627213605/http://crunchgear.com/2006/11/28/trekstor-vibez/)。相当标准的东西。它通过 USB 充电，有一个可移动的电池——它*鼓励*你把一个纸夹插到这个东西里，从背面弹出来，这是一个相当令人耳目一新的——非常直观的界面。然而，设计是最好的部分。正面包有橡胶塑料，背面是镀铬金属。它是球状的，非常小，有一个明亮的，可读的屏幕。它还有一个滚轮。但这不是普通的滚轮，这是一个物理滚轮，你可以将它旋转 180 度。轮子上甚至有一个小点，当你点击轮子上的一个基点时，它就会发光发亮。然后你有一个中央按钮和四个基本按钮来控制东西，就像 iPod 一样。

![](img/f1e1c357e92aba510553c88973df6f61.png)

这就是关键所在，不是吗？“就像 iPod 一样。”嗯，这是一个有趣的设计，否则也可以运行 MP3 播放器。TrekStor 在欧洲很受欢迎，今年只是在这里碰碰运气，所以这家公司绝对值得关注。

这个播放器的一个不寻常的方面是它没有内置调频调谐器。你实际上必须单独购买调谐器，它在一个每边大约 1/4 英寸的小芯片上，你必须将它压入一个小槽中。这不是一个主要问题，但显然任何人谁买了这个没有打算把一个小芯片到插槽。他们必须在产品上市前进行补救。

音频质量，如果你感兴趣，是标准的，没有惊喜。这是一个很好的 MP3 播放器。

![](img/d2936f8ec0ebc4dd648adef3618229f5.png)

除此之外，这个聪明的小 MP3 有一个相当不错的用户界面和一个独特的 DJ 功能，几乎就像 iTunes 动态播放列表一样，只是它们是在设备本身上运行的。如果他们给 Vibez 加了一个视频播放器，我会嫁给它。然而，就目前情况而言，我可以推荐亚马逊和其他地方售价 219 美元的这款产品。
