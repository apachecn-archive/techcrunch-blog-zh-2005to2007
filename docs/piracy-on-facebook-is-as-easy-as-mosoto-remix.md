# 脸书的盗版就像 Mosoto 混音一样简单

二月[![mosotologo.png](img/872ad4a9c2e20aa0bca7fe46e89cbd72.png)](https://web.archive.org/web/20221205090459/http://mosoto.com/)[Mosoto](https://web.archive.org/web/20221205090459/http://mosoto.com/)使用脸书 API [发布了他们的点对点文件交换和聊天应用。这是 API 推出时执行得最好的程序之一。现在，他们回来了，并集成到脸书平台，使它很容易赶上你的朋友在新闻和抢夺他们的音乐。](https://web.archive.org/web/20221205090459/http://www.beta.techcrunch.com/2007/02/07/mosoto-share-files-and-chat-on-facebook/)

Mosoto Remix 在您的个人资料中添加了一个小插件，用于共享您和您的朋友正在收听的音乐。在这个小工具中，你可以上传完整的歌曲，创建播放列表，并查看朋友最近上传/播放的音乐。如果你听到了你喜欢的东西，你也可以把你朋友的歌曲添加到你自己的账户上，或者从 iTunes 上购买。它本质上是 Box.net 小部件的一个专门实现，为所有文件提供存储。他们的画布页还可以让你和其他 Mosoto 用户聊天。

Remix 走了一条灰色路线，让用户能够上传和播放完整的歌曲，尽管他们不允许下载。Last.fm 等初创公司的脸书音乐应用程序(7 万多用户)和非常受欢迎的 iLike(300 多万用户)服务限制播放长度或顺序。脸书上还有几个其他的音乐插件。Audio (50 万以上用户)和 boom box(17 万用户)也允许会员上传歌曲和创建播放列表。

Audio 庞大的用户群提高了它的知名度，并促使它对盗版指控做出回应。在给用户的一封信中，这位音频开发商否认自己是“数字无政府主义者”，他说“我是音乐产业的好朋友，有人试图帮助它发展。我真的相信，音频可以提供的社交整合音频体验将引领音乐行业的未来——在相关性和收入方面都是如此。”

然而，其他音乐服务不得不为播放完整版权歌曲的特权付费。拉拉希望在未来两年支付超过 1 . 43 亿美元的版税。由于许可问题，潘多拉不得不[关闭国际接入](https://web.archive.org/web/20221205090459/http://www.beta.techcrunch.com/2007/05/02/pandora-to-shut-out-non-us-users-thursday-evening/)。由于这两个系统有着不同的规则，付费游戏和自由运动似乎不可避免地会发生冲突，脸书处于中间。
