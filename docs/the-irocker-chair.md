# iRocker 椅子

![](img/4d921e640b9f27fee876b41f4b46d243.png)

就在你认为 iPod 配件不能再荒谬了的时候，家具制造商 Adonis 宣布了一系列 iRocker 椅子。这些“交互式声音椅”带有一个内置的 iPod 基座，以及扬声器组，根据您购买的椅子而有所不同。

iRocker 椅子的价格从 99 美元到 600 美元不等，取决于哪种适合你的价格范围。高端型号配备了一套五个带遥控器的 Alltek Vision 扬声器(这有点矛盾)，所以你可以在不移动的情况下尽情摇滚和放松。低端型号配有一对标准扬声器。

无论哪种方式，两把椅子都包括一个辅助输入，以防你没有 iPod(还有人没有 iPod？！)，但你必须注意设备的电池。这些椅子也有各种各样的颜色，从超细纤维、中性黑色、骨色和马鞍色到人造革款式。所以，如果你喜欢坐着听音乐，iRocker 可能适合你。
