# 希捷为手机准备无线存储

希捷已经宣布了数字音频视频体验(或称 DAVE ),这是一项旨在为手机和其他 WiFi 和蓝牙设备提供 10-20GB 无线存储的技术。信用卡大小的驱动器将与无线设备同步，以便提供补充存储。

它已经开发了可以在 J2ME、BREW、Windows Mobile、Symbian 和 XCCC 上运行的软件。希捷表示，10GB 的定价应该在 150 美元左右。它应该会在今年夏天上市，20GB 的型号会在之后的某个时候上市。
