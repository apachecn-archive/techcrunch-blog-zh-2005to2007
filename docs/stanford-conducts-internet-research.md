# 斯坦福开展互联网研究| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/12/stanford-conducts-internet-research/>

![](img/ecd17769961a82301c6c3252883a1670.png)

斯坦福大学最近进行了一项研究，看看人们在互联网上做什么。根据他们基于 1996 年布局的网站，电子邮件仍然是最受欢迎的活动，90%的用户发送电子邮件。听着:聊天室是为年轻的匿名的 T2 开设的，就好像你不知道有成千上万的变态整天呆在里面一样。斯坦福还将互联网描述为“一个具有明显商业倾向的巨大公共图书馆”，这听起来有点像 T4。如果所说的图书馆有成堆成堆的山羊色情，也许我们会同意。
