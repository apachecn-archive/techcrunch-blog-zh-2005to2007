# 你说光学操纵杆？

三星给我们带来了一些手机的好处，他们推出了第一款带光学操纵杆的手机。本月在韩国上市的 SCH-V960 配备了 200 万像素的摄像头、GPS、文档浏览器、microSD 插槽和蓝牙。这款手机的伟大之处在于光学操纵杆和它允许用户使用的界面的易用性。这很像在手机上使用鼠标！咄。使用操纵杆，您可以在 Samsungs 的“我的屏幕”中导航，类似于桌面上的屏幕，并快速访问您的信息、相册和音乐。
现在我希望我的 Sidekick 3 也有这个功能:“智能照明”，这是一个照明传感器，可以控制 LCD 屏幕和键盘的亮度。哦，我疲惫的眼睛必须眯着，努力看清我的侧踢屏幕上的内容。哦，请请三星把这些带回美国。
