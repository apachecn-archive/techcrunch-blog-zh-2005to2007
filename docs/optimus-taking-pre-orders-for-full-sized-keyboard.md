# 擎天柱接受全尺寸键盘的预购

![](img/c41eb09f18a9e2626c083139667e87eb.png)

你肯定还记得 [Optimus 键盘](https://web.archive.org/web/20201124020529/http://www.artlebedev.com/everything/optimus/)，这是一个有 103 个键的神奇小工具，每个键都是一个可以单独重新编程的微型液晶显示器。一些消息终于再次传出:键盘将于 12 月 12 日开始接受预购。价格和数量尚未公布，但有传言说它的价格可能在 200 美元左右。在挖掘 Optimus LiveJournal 时，我们发现了这张用于重新编程按键的 Mac OS X 界面的图片。看起来难以置信的强大和易于使用。事实上，这个键盘的一切看起来都很棒，非常有用——现在让我们只希望它不会像[其他一些键盘相关产品](https://web.archive.org/web/20201124020529/http://www.infiniumlabs.com/home.html)一样变成蒸汽。
