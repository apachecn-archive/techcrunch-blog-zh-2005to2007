Nero Linux 3获得蓝光和HD DVD支持

Nero今天宣布Nero Linux 3现已上市。该软件的两个有趣之处在于它可以刻录蓝光和HD DVD光盘。Nero声称Linux 3是最强大和最通用的Linux刻录应用程序——我倾向于同意。

除此之外，它仍然只是刻录软件。伟大的燃烧软件，但它是什么。
