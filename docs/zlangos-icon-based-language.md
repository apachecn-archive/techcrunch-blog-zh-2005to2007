# Zlango 基于图标的短信语言

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/06/28/zlangos-icon-based-language/>

 [](https://web.archive.org/web/20220816010910/http://www.zlango.com/) [Zlango](https://web.archive.org/web/20220816010910/http://www.zlango.com/) 是一家新的以色列创业公司(有许多[创新](https://web.archive.org/web/20220816010910/http://www.beta.techcrunch.com/2006/03/01/lunch-in-israel-six-startups-and-a-vc/) [正在](https://web.archive.org/web/20220816010910/http://www.beta.techcrunch.com/2006/05/01/a-new-look-at-personalized-news/) [上](https://web.archive.org/web/20220816010910/http://www.beta.techcrunch.com/2006/04/21/confess-your-secrets-i4giveu/))，它创造了一种非常有趣的新语言和相关应用程序，可能会改变短信领域(最终也会改变电子邮件和即时消息)。以色列设计公司螳螂参与了这个项目。

这种语言基于图标或图片。每个图标都有特定的含义——一个人指着自己代表“我”，或者一颗心代表“爱”。今天，Zlango 语言中包含了 200 多个图标。

这项服务的用户为他们的 GSM 或 CDMA 手机下载一个 64K 的 Java 或 Brew 应用程序。应用程序将图标转换成 SMS 消息，然后在接收方电话上重新翻译回图标。为了直观地了解它是如何工作的，在这里查看 flash tour [，然后在这里](https://web.archive.org/web/20220816010910/http://www.zlango.com/pop.aspx?page=guide&guideId=1)试用模拟器[。](https://web.archive.org/web/20220816010910/http://www.zlango.com/content.aspx?page=try)

乍看之下，这不过是一款漂亮的移动软件，也是一种通过廉价且无处不在的短信网络发送图标信息的方式。

但我认为 Zlango 的文化内涵可能要深刻得多。

首先，两个用户可以使用 Zlango 进行交流，即使他们说的不是同一种语言。跨文化影响是显而易见的。

第二，Zlango 不是静态语言。很少使用的图标将随着时间的推移而消失，新的图标将被添加到语言中。然而，网站上没有明确说明的一件事是，用户是否能够像使用口语一样直接发展语言。在我看来，允许用户创建、使用和分享他们自己的图标，其中一些将最终成为主流用法，对 Zlango 的成功至关重要。

Zlango 声称，用户测试表明“大多数尝试过的年轻人都可以放心地说，他们在不到一个小时的时间里就掌握了这门语言。”在测试了模拟器几分钟后，我相信这是真的。只需几个按键就可以创建复杂的信息，比普通短信快得多。

我不知道这是否会受到用户的欢迎——我们必须等待即将到来的发布。下载软件肯定是采用的一个障碍，我怀疑 Zlango 将专注于与运营商达成协议，将这一点直接内置到手机中。他们将需要大量采用，以确保模仿服务不会在它有时间建立自己之前破坏潜在的网络效应。但话虽如此，看到一种新的基于图标的语言的诞生是很有意义的，看着这种语言像病毒一样传播，或者不传播，将会很有趣。关注 [Zlango 博客](https://web.archive.org/web/20220816010910/http://zlango.typepad.com/)的更新。
