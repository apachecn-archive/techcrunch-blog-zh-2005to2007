# 超冷技术保持 CPU 冷却，所以你不必

如今，CPU 冷却都是热电技术。Ultra Products 以 149 美元的 Ultra Chill-TEC 加入了珀耳帖效应处理器冷却器的竞争。

对于不熟悉的人来说，这种冷却器使用固态主动热泵，将热量从设备的一侧转移到另一侧。一个风扇，热管和铜板，然后悄悄地摆脱热量。最好的部分是，它可以帮助冷却超频的 CPU，其功率类似于液体冷却，但没有安装这些冷却器带来的所有设置问题。
