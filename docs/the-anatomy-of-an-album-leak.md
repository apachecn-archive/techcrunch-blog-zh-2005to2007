# 专辑泄露剖析

![](img/a6509daa29f36348974b0a3f54185866.png)
一张很棒的专辑，却在发行前整整三个月泄露

你一定要看看这个小博客上的这篇文章。这是一篇很好的文章，讲述了 CD 是如何在发行前几个月或几周被泄露的，以及它对音乐产业和歌迷的影响。不过，有时候泄密也有积极的一面，比如九寸钉乐队把 u 盘留在场馆浴室里让粉丝们发现。驱动器上是没有人听过的新歌，这反过来又让找到驱动器的顾客感到高兴。

还列出了去年一些最臭名昭著的泄露歌曲。这是一本很棒的读物。如果你对音乐或盗版有一点点兴趣，就去看看吧。
