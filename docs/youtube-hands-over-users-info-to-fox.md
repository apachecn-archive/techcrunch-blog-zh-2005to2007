# YouTube 将用户信息交给福克斯

据报道，在收到美国北加州地区法院的传票三周后，YouTube 发现一名用户被 20 世纪福克斯电视台指控在该节目播出前一周上传了该节目。这位名叫 ECOTtotal 的用户据称还上传了 12 集《辛普森一家》，其中一些已经很老了。根据 InternetNews.com 的一份报告，显然谷歌和 YouTube 愿意并且能够确认用户名 ECOTtotal 的所有者。

一些观察人士希望，当传票曝光时，谷歌抵制美国政府传票的历史将导致在本案中的不合作。不过，在这种情况下，道德高地显然要小得多。在被收购之前，YouTube 曾应要求提供用户名。

如果《24 小时》剧集在登陆 YouTube 之前就已经在 P2P 网络上流传，正如一些报道所言，那么这只不过是一次象征性的放血。这并不像雅虎在中国的一些行为那样应该受到谴责，在中国，该公司[据称](https://web.archive.org/web/20220928182325/http://www.rsf.org/article.php3?id_article=16402)交出了一些被中国政府指控从事颠覆活动的用户的信息。其中一些用户已经被送进监狱多年。

不过，我想知道 YouTube 用户对该公司披露个人信息会有什么感受。虽然传票发出时引起了相当多的关注，但我猜想大多数 YouTube 用户会容忍甚至支持该公司的决定。或许其他人会停止将他们的个人信息与用于非法活动的账户联系起来。隐私专家报告称，正是这些简单的事情让大多数人受到伤害。

尽管如此，随着维亚康姆的整肃，NBC 的恶语相向，现在这一切都发生在二月的上半月，YouTube 在不久的将来可能会是一个非常不同的地方。
