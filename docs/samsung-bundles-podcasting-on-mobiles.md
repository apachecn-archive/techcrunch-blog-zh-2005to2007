# 三星将播客捆绑到手机上

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/03/samsung-bundles-podcasting-on-mobiles/>

[![voiceindigologo.png](img/86a2c0e39daa79b6eb4e13f5d17a8746.png)](https://web.archive.org/web/20160528004140/http://samsung.voiceindigo.com/ht/home.do) 三星今天宣布，他们将在手机中捆绑 [VoiceIndigo](https://web.archive.org/web/20160528004140/http://samsung.voiceindigo.com/ht/home.do) 的广告支持播客服务。这项服务的第一款机型将是三星的双面 mp3 播放器/手机，[抢了](https://web.archive.org/web/20160528004140/http://www.samsung.com/Products/MobilePhones/Sprint/SPH_M620ZKASPR.asp)的风头。

VoiceIndigo service 是一个移动应用程序，它可以将播客下载到您的手机上，这样您就可以在线或离线收听全部内容。该服务还可以自动将一个频道的相关内容同步到手机上。该应用程序目前在带有 WAP web 客户端的 Symbian 平台上得到支持。

当 VoiceIndigo Mobile 正在播放您的播客时有电话打进来，您会在手机屏幕上看到来电显示。如果您选择接听电话，音频播放会自动停止。根据您的手机，播客也可以在您挂断电话后继续播放。VoiceIndigo 也可以用作桌面服务，您可以通过 URL 或 iTunes 下载播客。

现在有大量的移动播客服务。通过手机播放播客似乎是一个有吸引力的想法，因为与 mp3 播放器相比，手机的数量更多，但分发和复杂的移动平台使分发成为一个挑战。

VoiceIndigo 与三星的分销协议使其处于比大多数公司更有利的地位。Mobilecast 是一个类似的服务，也与 Alltel 和 3 等运营商有一些关系。然而，他们每月收取 3.99 美元的服务费。VoiceIndigo 支持广告。

[另外两项服务](https://web.archive.org/web/20160528004140/http://www.techcrunch.com/2006/11/12/listen-to-podcasts-on-any-phone/)，Podlinez 和 Fonopods 通过直接在电话上发送播客来规避分发和兼容性问题。
