# Pleo 现在可供我们预订

![](img/33621611937e71f5d5fe1e9bff411c61.png)

每个人最喜欢的电子恐龙现在离入侵我们的海岸又近了一步。Pleo 的制造商[UGOBE正在接受恐龙的预购订单，价格并不便宜。此前有报道称，Pleo 的价格在 250 美元至 300 美元之间，UGOBE 拿出了一台任天堂游戏机，并决定对这款 lil' guy 收取 350 美元。谈谈公路抢劫。](https://web.archive.org/web/20201130103315/https://crunchbase.com/organization/ugobe)

但是不要太兴奋。是的，UGOBE 和合作伙伴正在接受预购，但 Pleo 要到 10 月份才发货。同时，你可以去易贝买一辆二手的索尼 AIBO。我曾经养过一只狗，虽然这是一次超现实的经历，但那只狗比任何毛茸茸的动物踢的屁股都多。希望 Pleo 将在某些层面上进行比较。
