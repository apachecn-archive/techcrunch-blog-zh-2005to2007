# Skype 创始人将推出网络电视| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/26/skype-founders-to-launch-web-tv-2/>

关于电视世界和网络世界的融合和分离，有一些非常分裂的东西。

路透社报道称，Skype 的创始人计划在短期内推出广告支持的网络电视。Skype 的联合创始人 Janus Friis 说，这个代号为威尼斯的项目将“为消费者免费带来高质量的电视节目”。据推测，这将是另一个 P2P 客户端，类似于 Skype 和 Kazaa。

听起来这不会是“用户生成的电视”，尽管弗里斯说“我们不想再有任何诉讼”。换句话说，我们不希望受到那些把自己的电视节目混搭在一起，或者只是像在 YouTube 上那样简单地上传到服务上的版权所有者的指责。我同意他的观点，但是——众所周知——正是人们对 YouTube 上的电视所做的事情让它变得如此有趣——而不仅仅是重复播放的电视。

我想知道的是，“威尼斯”是由爱沙尼亚、[的黑客开发的，就像 Skype 是](https://web.archive.org/web/20160220193048/http://www.nytimes.com/2005/12/13/technology/13skype.html?ei=5088&en=06f1efc1e11e7049&ex=1292130000) …？也许我们的一些爱沙尼亚读者可以告诉我们。

与此同时——来自光谱的另一端——电视广播公司正拼命追赶 UGC，尽管前 Skype 的家伙们正试图在网上复制它。

在威尼斯故事开始的同时，在英国，Mint Digital 正在利用它的 [Bloombox.tv](https://web.archive.org/web/20160220193048/http://www.bloombox.tv/) 平台和现在的[Islandoo.com](https://web.archive.org/web/20160220193048/http://www.islandoo.com/)，前者旨在允许广播公司和广告商利用 flash video 进入用户生成的内容运动，后者被《独立报》气喘吁吁地描述为“与社交网站 MySpace 竞争的互联网现象”。《金融时报》援引格雷伦敦广告公司董事长大卫·艾伯茨的话说，他们让那些想上电视真人秀《海难》的用户“制作他们自己的诺基亚广告或三叶草广告”

现在，这听起来要么像又一个一无所知的营销人员开的一个恶心的玩笑，要么是一个非常聪明的想法，我还没有完全决定。
