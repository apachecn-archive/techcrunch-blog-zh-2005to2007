# 性感的 Mvix 媒体中心拆箱

你的电视还不能支持 Mvix MX-760HD 多媒体播放器，但 skippy.net 的人认为他们的可以。他们不仅写了这个设备有多光滑(我们说光滑是指它能输出 1080p，就像我们的任何电视都能处理它一样)，而且他们还拍了一些很棒的拆箱照片，并将它们放在 flickr 相册中。和普通色情片一样，业余爱好者制作的齿轮色情片最火。

如果你不能等待独立电视(或 Mac 媒体中心或其他什么)，这款设备可能会吸引你。除了通常的媒体格式，这个盒子还可以本地播放 DivX 和 Xvid。真性感。而且它是通过 802.11g 实现的，所以你不会(希望)经历你从次等“b”标准得到的口吃。混蛋们。如果你愿意，你可以通过 USB 2 控制这一切，我们建议你带上自己的硬盘，它是可升级的，你知道。只需 330 美元左右，你也可以拥有一个。
