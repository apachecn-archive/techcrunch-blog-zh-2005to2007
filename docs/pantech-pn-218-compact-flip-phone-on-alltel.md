# 泛泰 PN-218 紧凑型翻盖手机

Alltel 通常不以其时尚的手机而闻名，但这款来自泛泰的小型手机将在美国 Alltel 出售。它有一个 1.8 英寸的 TFT 屏幕，外面有一个 1.0 英寸的外部屏幕。VGA 摄像头、Brew 2.1 功能、扬声器电话和语音备忘录完善了该包。虽然这款手机很小，但也相当厚。合上时 0.96 英寸让你的口袋鼓了一大块。而且不是女士们喜欢的那种。
