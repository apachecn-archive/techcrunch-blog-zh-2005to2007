# 索尼 Vaio LA 系列媒体中心电脑

我们不会说这台电脑看起来像或者模仿了 iMac，因为它也不是独一无二的。我们只能说，这款一体式电脑看起来非常漂亮，应该适合媒体中心解决方案。

屏幕为 19 英寸，位于机器内部:英特尔酷睿 2 双核处理器、1GB 内存、300GB 硬盘、WiFi 和网络摄像头。它还配有 WIndows XP Media Center 2005，并具有内置的 NTSC 电视调谐器。您可以使用内置的 DVD 刻录机将录制的节目刻录到 DVD 上。该套装将于 10 月份上市，售价为 2499.99 美元。
