ValueClick以3.52亿美元收购比较购物运营商mezi media——TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2007/07/16/valueclick-acquires-comparison-shopping-operator-mezimedia-for-up-to-352million/amp/

太平洋时间2007年7月16日晚上8点28分

value click以高达3.52亿美元的价格收购了 MeziMedia ，这笔交易包括1亿美元的预付现金，另外还将根据MeziMedia截至2009年的收入和盈利情况支付高达2.52亿美元的款项

总部位于洛杉矶的MeziMedia是一家比较购物公司，运营的网站包括Smarter.com和在线优惠券网站CouponMountain.com。

华尔街日报报道 ValueClick首席执行官Tom Vadnais表示，收购MeziMedia符合ValueClick的收购理念，并“赋予ValueClick在美国的比较购物规模、更强的搜索营销专业知识和在中国的影响力。”

据说MeziMedia是盈利的，年收入为4000万美元。

继谷歌收购 DoubleClick和微软收购 aQuantive之后，最近有传言称ValueClick本身也是收购目标。
