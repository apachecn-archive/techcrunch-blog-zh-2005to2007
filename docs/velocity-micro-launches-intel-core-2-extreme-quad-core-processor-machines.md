# Velocity Micro 推出英特尔酷睿 2 至尊四核处理器计算机

我相信今天会有更多这样的“四核”产品发布，但 Velocity Micro 是我最喜欢的游戏 PC 和工作站台式机供应商之一。精品系统构建器(尽管你现在可以在百思买商店找到它的便利作品)与英特尔新的非常非常快的处理器是一体的。例如，有 Velocity Raptor Signature Edition 游戏 PC，完全装载了 QX6700 处理器和两个 768MB eVGA nVidia GeForce 8800GTX 显卡，今天的价格相当高，为 7395 美元。
