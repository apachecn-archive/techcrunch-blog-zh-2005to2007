# 英伟达下周将推出的下一款顶级 GPU:只是一个超频的 8800 GTX？

![](img/89840a870fa7ae514fd70b991bb1cc0e.png)

显卡专家 nVidia 将在下个月推出一系列新的显卡，其中似乎是超频的 8800 GTX 领先。“消息人士”称，nVidia 将把它的新顶级显卡称为 8800 Ultra，并打算超越 AMD 尚未发布的 Direct X 10 显卡。(这张卡本应在上周发布，但 AMD 临阵退缩，在最后一刻取消了发布。)

这条消息还表明，nVidia 将在发布 8800 Ultra 时推出低端 GPU。显卡的激烈竞争从未停止过。
