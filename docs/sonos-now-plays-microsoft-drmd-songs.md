# Sonos 现在播放微软数字版权管理的歌曲

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/01/08/sonos-now-plays-microsoft-drmd-songs/>

 [](https://web.archive.org/web/20160416114021/http://www.sonos.com/) 今天晚些时候 [Sonos](https://web.archive.org/web/20160416114021/http://www.sonos.com/) 将宣布他们的家庭音乐设备现在将支持微软的 DRM 音乐。这意味着来自 AOL Music Now、MTV Networks 的 URGE、Napster、沃尔玛、雅虎的音乐。音乐和 Zune Marketplace 都可以在 Sonos 设备上播放。从 9 月份开始，用户可以播放来自 [Rhapsody 的音乐——这意味着除了 iTunes 之外的所有主要音乐服务现在都在某种程度上得到支持。](https://web.archive.org/web/20160416114021/http://www.techcrunch.com/2006/09/14/sonos/)

Sonos 用户必须升级到该软件的 2.1 版，可从[这里](https://web.archive.org/web/20160416114021/http://www.sonos.com/support/software_updates/)获得。

苹果仍然牢牢控制着数字音乐产业，尚不清楚这是否会推动更多的使用。虽然 Sonos 将播放音乐服务的下载内容，但他们只会播放 Rhapsody 订阅用户的流媒体音乐。因此，如果你是 Sonos 用户，你最好确保你下载的音乐是 MP3 格式，或者如果你喜欢的话，使用 Rhapsody 订阅服务。

在今天的 DRM 世界里，事情变得简单了一点，但是我们认为被迫购买受保护音乐的日子将很快结束。同时，我们不建议从任何服务购买 DRM 下载。要么像比尔·盖茨推荐的那样，购买 CD 并翻录，要么加入订阅服务。我们不会谈论 BitTorrent 和 AllofMP3，因为我们的律师让我们停下来。
