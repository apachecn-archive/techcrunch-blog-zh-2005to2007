# Xbox 360 HD-DVD 附加软件可与 MAC 兼容…差不多吧

![](img/dadf113091ed21a046347400927e39b2.png)

不要弄湿你自己，也不要打你那些喜欢窗户的朋友的脸。UneasySilence 的人拿到了 Xbox 360 即将推出的 HD-DVD 播放器附件之一，并将其拆开，检查里面的套件。经过一番改装后，他们能够将它连接到一个 Windows 盒子上，并播放一些高清 DVD。然而，当他们将播放器插入 Mac 电脑的 USB 端口时，OS X 也立即认出了它，并能像平常一样播放 DVD。然而，使用 HD-DVD 没有任何结果，因为还没有可以在 Mac 上播放的软件。但是，是的，基本上这意味着你现在可以用微软收取的 199 美元买到一台正常工作的 HD-DVD 播放器。Mac 用户只能再等一会儿了。
