# 未来学家:潘多拉的未来是移动的

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/17/the-futurist-the-future-of-pandora-is-mobile/>

多亏了版权使用费委员会，从七月中旬开始，所有的网络电台将会看到更高的使用费。费用如此之高，不难想象音乐互联网的大片区域一夜之间会变成死气沉沉的空气。

最可怕的是失去潘多拉的前景——这是整个互联网真正伟大的事情之一。大约一年前，我有机会采访了 Pandora 的创始人 Tim Westergren，用他的话说，这项服务纯粹是出于他对拓展自己音乐领域的热爱。

就现在。潘多拉被围攻了。为了生存，它可能必须进化。它必须移动化。

如果你不熟悉 Pandora，显然你在过去的一年里没有踏进过美国的办公室，在那里，它已经从戴着耳机的小隔间工作人员疯狂地蔓延到戴着耳机的小隔间工作人员。简而言之:音乐基因组计划已经绘制出了大量基于模糊音乐特征的歌曲(想想“基于音乐套路的创作”或“东海岸说唱影响”)。你输入一首你喜欢的歌曲或艺术家，播放器就会接管——根据你的选择创建一个特别的电台，里面充满了与你喜爱的曲调有一个或多个共同特征的歌曲。

其中的精彩在于意料之外。这项服务不仅让你接触到不计其数的前所未闻的艺术家(许多音乐来自只是邮寄到项目门口的 CD)，而且通过剥去层层伪装，它为陈旧的商业音乐注入了新的生命。事实是，许多音乐迷并没有给所有艺术家一个聆听的机会。这可能是因为他们不太出名，或者，通常情况下，因为他们太出名了。

让我们这么说吧。我从来没有故意听过碧昂斯的歌。如果你问我是否喜欢碧昂斯，我可能会说“不”，并认为这是一个安全的赌注，不会对我有太大影响。然而，有了潘多拉，当诺尔斯女士突然出现时，我可能正在摇着一个模糊的鞋钉轨道。我不熟悉她的任何一首歌，除非我的潘多拉窗口现在充满了我的显示器，否则我无法知道她是谁。令人惊讶的是，我发现自己真的很享受。

这就是为什么 Pandora 不仅仅是对那些寻找任何曝光机会的不知名艺术家的一种恩惠，它还帮助一流的商业音乐人接触到热爱音乐的公众中完全放弃商业音乐的所有派别。

尽管如此，他们还是威胁要将其关闭。

那么潘多拉该怎么做才能活下来呢？答案很简单:移动化。

在移动设备上安装手机收音机或 Wi-Fi 正变得越来越便宜和普遍。如果有一种移动方式来访问 Pandora 的所有功能，这将比手机提供商目前提供的音乐内容的围墙花园，甚至是 XM 的 Inno 或 Sirius 的 Stiletto 等便携式设备上的数百个卫星广播电台都要整洁得多。想象一下，能够在旅途中输入一首歌曲或一位艺术家，然后在现场弹出一个充满惊喜的个性化播放列表。

要实现这一点，有两个选择:1)为高速 3G 手机推出一个程序，让它们能够接入 Pandora，2)推出一个 MP3 播放器或专门为 Pandora 设计的具有无线功能的专用音乐设备。

第一个，虽然显然是一个简单得多的提议，但并不能保证服务的长期可行性。首先，它无助于解决高额版税的财务现实。事实上，从经济角度来说，这可能是个糟糕的主意。Pandora 的大部分收入来自横幅广告——这对移动听众来说是一个冒险的提议。虽然一些有事业心的移动用户肯定已经将 Pandora 流式传输到他们的智能手机上，但这显然不是该服务的意图(它实际上阻止了某些版本的 Windows Mobile 接收它。)

类似潘多拉品牌的音乐播放器会是一个好得多的主意。这种设备需要他们与资金雄厚的制造商合作，让他们获得目前没有的财政资源。如果他们与 SanDisk 或微软合作，生产内置 Pandora access 的下一代 Sansa 或 Zune，他们可以获得巨额许可费，并拥有他们目前做梦也想不到的更多企业支持。

潘多拉不应该仅仅是一个网络电台，它应该是一个品牌。塞斯·波吉斯为他的专栏《未来学家》撰写了关于未来技术及其在个人电子产品中的作用的文章。它每周四出版，过去专栏的档案可以在这里找到。
