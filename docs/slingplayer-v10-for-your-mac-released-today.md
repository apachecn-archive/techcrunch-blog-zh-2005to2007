# 今天发布了适用于你的 Mac 的 SlingPlayer v1.0 版

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/30/slingplayer-v10-for-your-mac-released-today/>

Sling Media 宣布 Mac 用户 1.0 版将于美国东部时间今天下午 3 点在[下载](https://web.archive.org/web/20151230235610/http://ca.slingmedia.com/page/downloads.html)。1.0 版本包括侧面的遥控器，看起来很逼真，很酷吧？SlingPlayer for“Mac v . 1.0 是一个 Aqua 应用程序，遵循苹果的人机界面指南。”Sling 感谢 Mac 社区帮助他们塑造了整体吸引力，使其更加符合 Mac 界面的简洁和优雅。还有不到两个小时，伙计们！
