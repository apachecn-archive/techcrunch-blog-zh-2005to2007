# YouTube 将登陆欧洲 LG 手机

我们许多 CruchGear 员工都有一个共同点，那就是我们热爱日光海洋。另一个是无法治愈的麻疹，但那是另一篇文章。我们喜欢 Helio Ocean 的地方，除了它无处不在之外，还有拍摄视频并直接上传到 YouTube 的新功能。对于我们博客作者来说，没有比这更简单的方式来提升我们的媒体了。

LG 知道这一点，并与谷歌合作，为欧洲市场带来了具有相同 YouTubey 功能的 SGH-L760。这款手机的功能与海洋非常相似，特别是 200 万像素的摄像头，带有 YouTube 原生的 320×240 视频拍摄模式。不确定它是否能到达美国，但是这款 3G GSM 手机将会很好的适应美国电话电报公司的视频攻击。
