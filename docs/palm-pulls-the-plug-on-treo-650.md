# Palm 拔掉了 Treo 650 的插头

陈旧(但仍是我们个人的最爱)的 Treo 650 现在在 Palm 的网站上被列为“不再供应”。如果你试图选择它，系统会提示你看看 Palm 的其他智能手机，非常感谢。

然而，Cingular 仍然将 Treo 650 列为可用产品，尽管在折扣后仍然比其更年轻、更强大的兄弟 680 更贵。因此，如果你一直在推迟购买，这可能是你最后的机会。

或者你可以试试 50 美元的 Palm Trio 650 ，尽管质量值得怀疑。
