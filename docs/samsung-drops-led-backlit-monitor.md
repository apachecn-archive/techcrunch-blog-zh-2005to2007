# 三星水滴 LED 背光显示器

三星在今天的 CeBIT 上推出了一款新的 LED 背光 30 英寸液晶显示器 SyncMaster XL30。它具有 123%的彩色生产和 DLSR 相机用户会很高兴地听到，它把翠绿，深红色和其他自然颜色的真实形式。包括三星的专有软件*自然色彩专家*，以确保精确的色彩匹配以及 *Eyeone 设计& Eyeone 照片*。显示器设置可以设置为 Adobe RGB 和 sRGB，具体取决于您选择使用的颜色范围。它甚至配有一个可拆卸的遮光罩，以帮助阻挡不必要的光线。输入通过 USB 和 DVI-D 连接。XL30 还具有高清观看 HDCP。价格或上市日期尚未公布。
