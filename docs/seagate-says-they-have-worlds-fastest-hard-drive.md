# 希捷称其拥有“世界上最快的硬盘”

![](img/d519a349f5648487a77b921d20eedb93.png)

希捷肯定是自豪地展示其新的 Savvio 硬盘。希捷声称，这种 15，000 RPM 的驱动器是地球上最快的，因为它使用了 2.5 英寸的框架。空闲时，它的功耗也比大多数硬盘驱动器低。您甚至可以在闪电般的 2.9 毫秒内访问数据！哇，这真是令人兴奋的东西！我的意思是，想想你在 2.9 毫秒内可以做什么！你可以假装想象约翰裸体的样子，或者不再关心希捷和它的竞争对手之间的鸡巴尺寸竞赛。
