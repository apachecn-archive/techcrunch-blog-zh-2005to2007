# Web 2.0 和 VOIP 威胁着电信公司。事实。

大多数电信专业人士不相信他们的行业能够应对来自新的互联网和基于 IP 的服务的威胁。

根据电信分析公司 [STL](https://web.archive.org/web/20201125091223/http://www.telco2.net/) 的研究，该公司调查了 550 多人，超过 75%的人现在认为，如果电信公司要在 Web 2.0 世界中取得成功，他们需要“显著地改变他们的商业模式”。

这些忧心忡忡的高管提出的建议包括，在结构上把该行业分成不同的业务——一个只专注于网络，另一个专注于零售和服务业务。

STL 首席执行官 Simon Torrance 评论道:“电信行业已经意识到互联网和 Web 2.0 商业模式带来的威胁。它现在需要解决这些问题。”

调查的其他重要发现包括:

75%的受访者对电信公司在基于 IP 的世界中创造长期可持续增长的能力缺乏信心

68%的受访者认为电信公司已经失去了对其核心产品的控制——语音和消息不再是运营商的专属领域
76%的受访者认为运营商的控制文化(表现为“围墙花园”式的服务方式)不再可行，互联网世界的开放性更加重要

然而，对所有这一切的健康检查是，STL 显然是在让电信高管们比他们已经感到更加害怕，并向他们出售咨询服务。所以请记住这一点。
