# 任天堂 Wii 界面视频

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/14/nintendo-wii-interface-videos/>

任天堂发布了一些展示其 Wii 界面的官方视频。看起来好像你在用 Wiimote 控制和选择屏幕上的项目，就像演讲者在 Powerpoint 演示中用激光笔一样。

其他一些视频展示了如何定制你的虚拟形象的外观——比如让自己看起来像一个日本女人——还可以阅读新闻或查看天气。这些都是非常漂亮的应用程序，但是它们看起来真的是日本的 T2，我不是在说语言。用户界面设计看起来绝对是受日本的启发。
