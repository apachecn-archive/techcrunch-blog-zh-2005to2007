# Skype 认为不能运行 Skype 的手机不好

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/06/skype-thinks-phones-that-cant-run-skype-are-bad/>

![blurb_skype.jpg](img/36c7a469582c17331934c385a4de6580.png)

《今日美国》刊登了一篇关于 Skype 是如何被一些运营商拒绝让手机用户在他们的手机上运行外部应用程序的小文章。他们非常生气，事实上，他们写了一封措辞严厉的信给联邦通信委员会，他们打算把这件事一直进行到…什么？哦，对不起。

本质上，Skype 对威瑞森不允许人们在手机上运行第三方应用感到愤怒。Skype 就是这些被禁应用中的一个，因为大多数运营商都疯了——威瑞森是最疯的——并且不想让任何人做任何事情，Skype 被许多支持 Wi-Fi 的手机拒之门外，总体而言，这是一个新的市场。

反正故事就是这样。大狗们。
