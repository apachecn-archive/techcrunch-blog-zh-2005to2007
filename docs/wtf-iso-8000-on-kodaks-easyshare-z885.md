# WTF？柯达 EasyShare Z885 上的 ISO 8000

当百万像素不再吸引消费者购买相机时，制造商们显然转而宣传另一种规格:ISO。

这些字母代表国际标准化组织，在电影时代，ISO 后面的数字代表胶片速度或胶片曝光所需的光量。这里涉及到数学问题，我们就不赘述了，简而言之:数值越高，胶片越敏感，拍摄图像所需的光线就越少。然而，在数字世界里，这基本上是不可能的。

就个人而言，我拥有的最后一台胶片单反相机去了 ISO 1600。另一方面，柯达 EasyShare Z885 去了 ISO 8000。那是什么意思？老实说，我还没有看到相机的图像(在 PMA 展出的是一个前期制作，甚至对开机都很挑剔)，但是，我想你可以在几乎没有光线的黑房间里拍出黑色的东西，但仍然可以拍出东西的照片。这也是大多数(如果不是全部)供应商用来帮助减少图像模糊的方法。

更新:当我们最初发布这个的时候，我们忘记提到 Z885 甚至没有我们在 PMA 消费相机上看到的最高 ISO。当以 300 万像素的分辨率拍摄时，[奥林巴斯 FE-250](https://web.archive.org/web/20210302021020/http://www.olympusamerica.com/cpg_section/product.asp?product=1285&fl=2) 的 ISO 高达 10000。我们只能想象在那个 ISO 拍摄的图像中会有什么样的噪声。

 Z885 在其他方面毫不起眼。这是一款坚固的紧凑型相机，配有 2.5 英寸液晶显示器、810 万像素传感器和 5X 光学变焦。它包括许多柯达的颜色和操作功能，如自动场景模式检测和完美触摸，这可以改善照片中的阴影。总而言之，200 美元，假设图像质量不错，Z885 看起来像一个称职的紧凑型相机，比这个价格更值得。

顶部安装的模式拨盘非常便于拇指使用。
