# Wikia 正在发展，有人关注吗？

上个月在巴黎举行的 [Le Web](https://web.archive.org/web/20210622192650/http://www.leweb3.com/) 会议上，我浏览了首席执行官吉尔·潘奇纳的 [Wikia](https://web.archive.org/web/20210622192650/http://www.wikia.com/) 演示幻灯片，注意到一些事情，让我意识到它们有一天会成为一个巨大的网站。据该公司称，维基百科每天产生 250 万页面浏览量，并在稳步增长，他们的新文章增长率与维基百科早期的增长率几乎相同。Penchina 演示文稿的关键幻灯片如下，比较了两个网站在不同时间段内独特文章的增长情况。

![](img/2ea2779763bbdf39ccab5e36a91e2a4a.png)

因此，维基百科的盈利姐妹网站 Wikia 正在密切跟踪维基百科在新文章生成方面的早期增长。显而易见的结论是，适用于维基百科的方法可能也适用于维基百科。当然，用户生成内容和社区的力量也在发挥作用。

很难比较页面浏览量，因为维基百科显然已经多年没有跟踪他们自己的页面浏览量了，但维基百科肯定保持着自己的 250 万/天。

Wikia 证实，他们目前与谷歌就其网站上的文字广告达成了一项“默认”协议。大型广告网络(谷歌、雅虎、微软)都试图锁定与大合作伙伴的长期交易，通常是通过提供某种收入保证(见谷歌与 MySpace 的近 10 亿美元[交易和微软与脸书的](https://web.archive.org/web/20210622192650/http://www.beta.techcrunch.com/2006/08/07/google-pegged-to-search-myspace/)[交易](https://web.archive.org/web/20210622192650/http://www.beta.techcrunch.com/2006/08/23/facebook-does-ad-deal-but-not-with-google/))。看到他们每个人也开始追求维基百科，我不会感到惊讶，只是作为一种对冲，以防网站有一天发展到维基百科的一小部分。无论哪家公司长期锁定 Wika，未来都可能拥有非常好的资产。
