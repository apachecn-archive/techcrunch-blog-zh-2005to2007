# Yelp API 发布 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/08/02/yelp-api-released/>

 [](https://web.archive.org/web/20230219193416/http://www.yelp.com/developers/examples) 本地评论网站 [Yelp](https://web.archive.org/web/20230219193416/http://www.crunchbase.com/company/yelp) 悄然[今天](https://web.archive.org/web/20230219193416/http://www.yelp.com/developers/documentation/search_api)发布了一个 API，允许第三方访问该服务的大部分内容和功能。

API 允许许多输入请求。应用程序可以检索特定地理区域/位置的商业评论和评级信息，显示一个或多个商业评论和图片，基于电话号码提取商业信息等。对于每天多达 10，000 个请求，API 上没有商业界限——超过这个数目，你必须与它们达成协议。

我们[最近在 Yelp](https://web.archive.org/web/20230219193416/https://techcrunch.com/2007/07/09/yelp-is-way-cooler-than-i-thought/) 上开了一个玩笑，当时弹出了一个“全面服务”的景观设计师的评论——但实际上该网站做得非常好。他们已经在两轮中从 Bessemer 和 Benchmark 那里筹集了总计 1600 万美元的资金。竞争对手包括 Intuit 的 Zipingo 和今年早些时候被 Citysearch 收购的 Insider Pages。

康姆斯克显示 yelp 每月有大约 140 万美国访客和 600 万页面浏览量。
