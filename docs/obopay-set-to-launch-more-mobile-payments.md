# Obopay 即将推出:更多移动支付

**更新:**[obo pay](https://web.archive.org/web/20220810113301/http://www.obopay.com/info/)网站现已上线，尽管在四月份之前你不能[注册和使用该服务。](https://web.archive.org/web/20220810113301/http://www.obopay.com/info/signup.php)

总部位于帕洛阿尔托的 [Obopay](https://web.archive.org/web/20220810113301/http://www.obopay.com/info/) 将于周四某个时候推出(该网站目前受密码保护)。这是非常活跃的移动支付领域的最新产品(参见 [PayPal Mobile](https://web.archive.org/web/20220810113301/http://www.beta.techcrunch.com/2006/03/22/paypal-mobile-launches-and-its-awesome/) 和 [TextPayMe](https://web.archive.org/web/20220810113301/http://www.beta.techcrunch.com/2006/02/15/everyone-send-me-5/) )。

自从两周前马特·马歇尔发布了一个预告之后，更多关于 Obopay 的细节浮出水面。Obopay 将依赖手机上的 java 客户端，而不是像 PayPal Mobile 和 TextPayMe 那样的短信或短信支付。虽然这提供了一个更丰富、更安全的界面，但 Obopay 只对那些拥有支持 java 的手机的人有用:

> 我们首批移植的 25 款手机将跨越两个主要运营商(即 Cingular 和 T-Mobile)和一个我们正在试用的主要 MVNO。这一组包括五家不同的手机制造商(即诺基亚、摩托罗拉、京瓷、三星和索尼爱立信)，其中包括三家最大的制造商。在我们的移植工作中，我们的目标是四大运营商目前针对我们的最终用户群提供的所有手机。

这项服务本身将允许手机之间的个人对个人支付。Obopay 说“用户可以向几乎任何手机汇款”，这可能意味着短信(或电子邮件？)对于不受支持的手机——我只是不知道这将如何工作。用户还将获得一张附属于账户的借记卡，用于现实世界的支付，包括 ATM 取款。

至于费用，我现在得到的信息是“交易费用只是几分钱”。当我得到更多的信息时，我会更新。
