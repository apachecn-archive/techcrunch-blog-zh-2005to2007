## 概观

![](img/0e5e3d9f579a38a5df95f112778c50dd.png) [RemoteControlMail](https://web.archive.org/web/20221129080510/http://www.remotecontrolmail.com/) 会给你提供一个永久的(蜗牛式)邮件地址，收集你所有收到的邮件，扫描(外部)，通知你它的存在(与外部扫描一起)，如果你喜欢的话扫描内部(收费)，然后根据你的命令要么转发给你要么粉碎它。

您现在可以注册 RCM。这项服务将在几周内开始运行。

谁会想要这个？经常搬家或拥有多处住所的人。需要在另一个城市设立实体邮寄点的企业。担心身份盗窃的人(每个人都应该担心)。或者只是想对现实世界中的邮件进行更多网络控制的人。

我今天与创始人兼首席执行官 Ron Wiener 谈论了几天前推出的 RemoteControlMail，并对这项服务有了更好的了解。

处理真实的邮件并不有趣。无数的垃圾邮件源源不断地进入我们的邮箱(顺便提一下，其中只有 17%最终被回收)。作为一个国家，我们每年收到超过 2000 亿封邮件。

[远程控制邮件](https://web.archive.org/web/20221129080510/http://www.remotecontrolmail.com/)是第一家试图通过虚拟化我们现实世界的邮箱来解决这个问题的公司。

注册后,“远程控制邮件”会为您分配一个永久的邮寄地址，然后您可以将所有或部分收到的邮件重定向到该地址。

收到邮件后，RCM 将扫描包裹或信封的外部，并通过电子邮件或 RCM 网站通知您邮件的存在。您可以选择扫描包裹内部，然后将该物品转发给您(单独或与其他邮件一起)，或者粉碎并回收。

您的 RCM 邮箱示例:

约翰·史密斯
RCM # 1234
14525 SW milli Kan Way
Beaverton，OR 97005
