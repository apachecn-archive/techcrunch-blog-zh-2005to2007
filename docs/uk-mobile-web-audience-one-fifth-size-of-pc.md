# 英国移动网络受众只有个人电脑的五分之一

Telephia 和 comScore 发布了他们对基于移动设备和基于 PC 的互联网使用的研究结果，结果显示，在过去的一月份，英国有 570 万人使用移动设备访问网络，相比之下，有 3000 万人使用 PC 访问网络。这使得移动网络受众的规模约为英国基于 PC 的互联网受众的五分之一，该报告补充说，这些早期采用者往往是 35 岁以下的男性。

英国的男性用户占移动网络用户的 64 %,而 PC 互联网用户的 54 %,移动网络用户的 60%是男性，而 PC 互联网用户的 49%是男性。
