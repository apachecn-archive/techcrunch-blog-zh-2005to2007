# 新领域关注新闻故事之间的联系 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/05/01/new-sphere-focuses-on-connections-between-news-stories/>

 [](https://web.archive.org/web/20220928185833/http://www.sphere.com/) 当博客搜索引擎 [Sphere 在 2006 年 5 月](https://web.archive.org/web/20220928185833/http://www.beta.techcrunch.com/2006/05/01/new-blog-search-engine-sphere-launches/)推出时，它包含了一个独特的功能，即动态发现与你正在阅读的内容相关的故事，不管这两个故事是否有超链接。

这个名为“Sphere It”的特性越来越受欢迎，并帮助 Sphere 嵌入到顶级博客和新闻网站中。我们在每篇文章的底部都包含了一个 Sphere It 链接，它会弹出一个窗口，显示其他讨论相关问题的博客故事。Time.com 和其他主要新闻网站也做了同样的事情。今天，Sphere 总流量的很大一部分来自这些合作伙伴网站，它们利用这个特性为读者生成更多的内容(和页面浏览量)。

今晚 [Sphere 重新发布了他们的主页](https://web.archive.org/web/20220928185833/http://www.sphere.com/)来反映这个特性的有用性。该网站的主要领域分为四列。左边是主要话题，如头条新闻、美国新闻、娱乐、体育、科技等。点击任何主题，第二列就会填充来自 Sphere partners(如我们的网站、Time.com 网站和其他包含 Sphere It 功能的网站)的最近新闻，这些新闻已经引起了很多关注，这是根据该项目的页面浏览量(相对于该网站的平均值)和该公司未披露的其他因素(但可能包括对其他网站撰写类似内容的程度的分析)计算的。

![](img/ffa24d892d1165d2f6f1cb71334e9cf4.png)

点击任何一条新闻，它就会出现在第三栏。在第四列中，显示了相关的新闻项目。

你可以无限点击一个话题。点击第四栏的故事——它向左移动，第四栏显示与那个故事相关的新闻条目。

像 TechMeme 一样，新的 Sphere 网站可以成为人们经常查看的地方，看看博客和主流媒体上有什么新闻，并查看关于该主题的其他内容。它与 TechMeme 的不同之处在于，它不需要文章之间的链接。

想要被包含在 Sphere It 中的新闻网站必须将该功能添加到他们的网站中。Sphere 主页上有一个链接可以做到这一点。

迄今为止，Sphere 仅从 True Ventures、Trident Capital 和一些天使投资者那里筹集了 400 万美元。这家公司有八名员工。
