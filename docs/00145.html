Web 2.0本周(7月3日至9日)

原文：https://web.archive.org/web/http://www.techcrunch.com/2005/07/08/web-20-this-week-july-3-9/trackback/

本周，我们看到了许多有趣的web 2.0发展。然而，远远超过所有其他事情的是7月7日对伦敦的恐怖袭击。博客和互联网在向人们传递信息方面发挥了重要作用，因为电视远远落后，手机网络也堵塞了。

每周总结:

1。伦敦恐怖袭击

我正飞往欧洲参加一个会议，在飞机上享受我的第一次wifi体验，这时来自伦敦朋友的即时消息开始传来。一位朋友说:“管子里有一处暴露，他们认为这是一个电源转换器。”。几分钟后，“天哪，到处都是爆炸的报道——可能是公交车炸弹。”很快就有空乘人员和其他飞行员来找我更新信息。飞行员派了一名空服员回去听发生了什么事，因为她无法通过无线电获得消息。我似乎是飞机上唯一的新闻来源。

BBC的网站流量下降，人们转向博客了解新闻。维基百科被认为是最新的信息来源。其他人写博客讲述他们的经历。

从一个大媒体与博客的角度来看，弗雷德·威尔逊完美地总结了这一点，他说“这就是为什么CNN的24×7新闻频道已经过时了。”(链接到杰夫贾维斯)。

我们的心继续与每一个受影响的人在一起。911那天我在曼哈顿。现实需要几天时间来适应。

Blogherald引用了Netcraft的报告，该报告显示了2005年6月域名有史以来最大的月环比增长。其中一个原因包括“博客的爆炸性增长，越来越多的博客出于品牌目的购买域名。”

4。Ian Davis—“Web 2.0是一种态度而不是技术”

Ian写了一篇精彩的文章，进一步阐述了web 2.0的定义:“这是我的看法: Web 2.0是一种态度，而不是一种技术。这是关于通过开放应用和服务来支持和鼓励参与。我说的开放是指技术上的开放，包括适当的API，但更重要的是，社会上的开放，包括在新的激动人心的环境中使用内容的权利。当然，网络一直都是关于参与的，没有参与就什么都不是。这是一个最大的成就，网络超链接，从一开始就鼓励参与。不知何故，在90年代后期，网络失去了与它的根源的联系，自私的利益占据了主导地位。这就是为什么我认为Web 2.0标签是狡猾的:语义上它将我们链接回最初的网络和它所倡导的理想，但同时它也意味着新版本的再生。技术在不断发展，重要的是网络的社交面要跟上步伐。”
