# 索尼 Cyber-shot DSC-T10 现为粉色

索尼 Cyber-shot T10 已经发布了粉红色，这意味着女性应该去当地的索尼专卖店买一台，对吗？T10 有一个 720 万像素的传感器，3 倍光学变焦，56MB 的内部存储，支持 Memory Stick Duo 和 Memory Stick PRO Duo 介质，以及 PictBridge，便于打印。

电池可以让你在一次充电中拍摄 250 张照片，如果你购买了索尼 CSS-TNA: Cyber-shot station，电池的充电速度将比使用随附的充电器快 30%。我们提到过这是粉色的吗？
