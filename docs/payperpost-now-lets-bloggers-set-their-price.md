# 你的灵魂值多少钱？PayPerPost 现在允许博客作者设定价格

让广告商购买博客评论的[](https://web.archive.org/web/20220930040652/http://www.payperpost.com/)[PayPerPost](https://web.archive.org/web/20220930040652/http://www.payperpost.com/)[有争议的博客服务](https://web.archive.org/web/20220930040652/http://www.beta.techcrunch.com/2006/06/30/payperpostcom-offers-to-buy-your-soul/)发布了一款名为 PayPerPost Direct 的新产品。新产品是一个小部件，博客作者可以将其添加到博客中，宣布他们愿意出售博客帖子。

以前，博主们会仔细阅读 PayPerPost 网站上的列表，寻找赚取额外收入的机会。现在，博客作者可以在创建一个 widget 时设定自己的最低价格，让广告商直接来找他们。

这一特点微妙地改变了公司接近市场的方式。目前的系统让广告商为每篇文章支付固定的费用，而不管博客大小或权威的差异。结果是 5 澳元左右的低平均费用，这并没有吸引更大的博客。既然博主可以自己定价，并与广告商一对一接触，我们可能会看到一些较大的博客开始使用这项服务。竞争对手 [ReviewMe](https://web.archive.org/web/20220930040652/http://www.beta.techcrunch.com/2006/11/09/reviewme-launches-a-better-payperpost/) 也有类似的方法，向大型博客支付更多费用，尽管价格是基于算法的。

PayPerPost 保留直接产品产生的费用的 10%。他们正常产品的标准费用是 35%的折扣。

我们将拭目以待，看看是否有任何大型博客会很快采用这种方式。PayPerPost 仍然有很大的缺陷——它允许广告商要求积极的评论，并且他们不要求博客作者在帖子本身中披露这是赞助文本。
