# 研究称英国手机被用于 WC 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/30/study-says-uk-mobile-being-used-in-wc/>

在英国，吸烟可能不是长时间上厕所的唯一原因。蜂窝新闻报道说，许多英国人在厕所使用手机，以绕过办公室关于互联网使用的规定。T-Mobile 在全国范围内开展的研究发现，超过四分之一的英国劳动力仍然无法上网，他们正在通过手机上网。该研究还发现，23%的员工被禁止访问社交网站，11%的员工被禁止使用基于网络的电子邮件。

作为回应，T-Mobile 已经开始了一项“让互联网免费”的活动，鼓励用户尝试网络漫游服务。虽然研究表明，25%的消费者知道他们的手机上有互联网服务，但不知道如何使用。一旦他们弄明白了，这就意味着厕所要排队了。
