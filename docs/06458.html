Web 2.0获得了杂志般的待遇

原文：https://web.archive.org/web/http://techcrunch.com:80/2006/11/06/web-20-gets-the-glossy-magazine-treatment/

《卫报》的周末杂志已经发现了Web 2.0，并报道了该领域的一些热门公司，其中大部分对该领域的爱好者来说都是老古董(维基百科，Flickr等)。不过，很高兴看到英国的排在最后。FM 以为特色，主要集中在美国。

有各种各样的采访值得一试，在那里人们可以回答:什么是Web 2.0？你的大主意是什么？网上的下一件大事是什么？主要的伴奏部分开始时非常热情:

第二次互联网淘金热正在如火如荼地进行，这一次都是关于真实的人，创造，编辑和展示他们自己的生活和观点。约翰·兰彻斯特抓住虚拟世界，卫报记者采访了最聪明和最幸运的企业家，他们摧毁了旧的互联网，建立了一个全新的互联网

稍后，它变得更加具体:

现在正在发生的事情的简写术语是“Web 2.0 ”,这个名称是由网络商务倡导者Tim O'Reilly在2004年的一次会议上创造的，描述的是“一种态度而不是一种技术”。这个短语是第二次互联网淘金热的简称，紧随其后的是2000年结束的那次，那次淘金热可以说是历史上投资者资本遭受的最大损失。从商业的角度来看，这种新淘金热的决定性特征是，老牌公司正在向新贵投入巨额资金，这些新贵有三个共同点:他们从无到有，以惊人的速度成长；他们没有收入来源可言；他们的大部分内容都是由他们的用户提供的。因此，我们看到了默多克在2005年7月收购MySpace，雅虎在2005年3月收购Flickr，并有传言称即将以约10亿美元收购脸书，以及谷歌在10月9日以16亿美元收购YouTube，这是迄今为止金额最大的收购。这是一大笔钱砸在一些快乐的书呆子身上。查德·赫利和陈士骏在2005年2月才创建了YouTube。他们的作品以每月超过1亿美元的速度增值——这肯定是一项世界纪录。对于一家没有收入的公司的创始人来说，这是一大笔钱。

显然，这篇文章现在承认，鉴于所有这些交易都发生在去年，淘金热实际上可能已经结束了，但是，嘿，谁在计算…

然后，作者开始听起来像是他第一次发现了互联网，并困惑地发现人们用社交网络做各种事情。哇哦。最后一段听起来确实像是他已经没有什么可说的了，因为这是印刷品，所以他竭尽全力把它填充到最后一栏:

坐在电脑屏幕前的每个人都是世界的中心。每个坐在电脑屏幕前的人，越来越希望一切都与他们有关。这是我们第一次看到伴随网络成长的人想要从网络中得到什么。MySpace最聪明的地方之一是它的名字。

有点讽刺的是，这篇文章是在《旧金山纪事报》的一篇题为:“社交网站变得太好了:许多年轻人热衷于在线分享”的文章发表后两天发表的

这里有一段引言:

阿瑞卡·卡罗厌倦了分享。也就是网上分享。她分享了她的猫的生活。她分享了一份她最喜欢的电视节目和电影的清单(《实习医生格蕾》，言情片)。她分享了自己对湾区景点的评价(圣何塞的老意大利面工厂是两星，摩根山的星巴克是五星)。她被邀请来分享更多。

我能说什么呢？过度批评一份主流杂志关于我们建设和运营在线服务和社区的方式正在发生结构性变化的文章可能是不公平的。但这里有一个永恒的问题——现在Web 2.0正得到主流的认可，这是否意味着所谓的淘金热已经结束了？当然，简单的答案是，无论如何，这不是Web 1.0的淘金热。而且Web 2.0的很多态度和技术无论如何都有可能成为主流。尽管如此，有趣的是，一份精美的报纸杂志决定将所有这些放在英国的主流地图上。
