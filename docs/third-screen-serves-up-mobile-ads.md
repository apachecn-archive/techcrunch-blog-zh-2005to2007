# 第三个屏幕提供移动广告| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/09/third-screen-serves-up-mobile-ads/>

![ thirdscreen.gif](img/9c82b815bb34fe4feaa67aa399ce834a.png) Third Screen Media 是一家移动广告网络，在其集团的 185 个网站上提供广告服务，包括 Accuweather、Boston.com、福克斯新闻频道、Gannet 报纸网络、Maxim、Wapipedia 和 WWE，据报道每月提供 2.25 亿次广告展示。TSM 网络产生的流量表明移动互联网的采用，随着更多的用户开始使用他们的手机查看 WAP 网站上的内容，移动互联网的采用只会增加。
