# 威瑞森的无限数据计划没有那么无限

![](img/7981e14db3d63007e5a7352f9249a88c.png)

出门在外时，将 EVDO 记录在笔记本上无疑是一种享受。但所有美好的事物都必须结束，在这种情况下，它只是碰巧是“无限”这个词。一年多来，威瑞森一直标榜其 EVDO 数据计划是“无限制的”，但实际上，如果你一个月的下载量超过 5GB，你的服务就会因为担心你是盗版而被取消。Cybernet News 深入挖掘了威瑞森的 ToS，发现了以下情况:

> “无限数据计划和功能(如国家接入、宽带接入、一键通和某些 VZEmail 服务)只能用于以下目的的无线设备:(I)互联网浏览；㈡电子邮件；以及(iii)内部网访问(包括访问公司内部网、电子邮件和个人生产力应用程序，如客户关系管理、销售人员和现场服务自动化)。无限数据计划和功能不得用于任何其他目的。”

从那以后会越来越好。这些条款基本上是说，如果你一个月下载超过 5GB，你显然是盗版或从事威瑞森数据计划的“禁止使用”。这对你意味着什么？如果你不想冒 EVDO 服务[被取消](https://web.archive.org/web/20221221061109/https://techcrunch.com/wp-content/uploads/2007/04/vzw.jpg)的风险，你最好停止 P2P、Bit Torrent 和流媒体。否则，您可能会发现自己在寻找新的数据提供商。
