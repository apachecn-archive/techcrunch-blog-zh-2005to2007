# OQO 尚未死亡，出现在第一个零售点

在最初的一阵兴奋之后，我几乎忘记了小 OQO 模型 01+。(你知道，眼不见，心不烦。)Anyhoo，世界上最小的全功能 Windows XP 电脑将在这个假期在纽约市的 [RCS 体验](https://web.archive.org/web/20130627211358/http://www.rcsnet.com/www_site/index.cfm)进行首次零售。根据新闻稿“RCS Experience 以其在创造展示“设备”如何相互作用的环境方面的营销专业知识而闻名。”我从来没有去过这家商店，但网站上的图片让它看起来很酷，很有意思，所以也许是时候去旅行了。此外，我确实喜欢去电器商店，让店员说些蠢话。
