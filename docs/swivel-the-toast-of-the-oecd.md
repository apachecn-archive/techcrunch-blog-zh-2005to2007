# Swivel:经合组织的宠儿——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/07/11/swivel-the-toast-of-the-oecd/>

上周，我在两年一度的经合组织(OECD)T4 会议上发言。会议的一大主题是使用什么政策和产品来充分利用流入组织的所有官方政府经济、社会和环境数据。会议结束时，签署了伊斯坦布尔宣言，呼吁成员国将其官方政府数据作为公共产品在网上公开。

> 在正在研究的各种想法中，经合组织正在考虑建立一个基于 Web 2.0“维基”技术的互联网网站，介绍和讨论旨在制定社会进步指标的国际、国家和地方倡议。通过动态图表和其他分析工具向世界各地的公民提供指标，这一举措旨在根据可靠和可比较的统计信息，激发关于进展实际含义的讨论。

硅谷数据可视化和建模初创公司 [Swivel](https://web.archive.org/web/20211127064153/http://www.crunchbase.com/company/swivel) [参加了](https://web.archive.org/web/20211127064153/http://flickr.com/photos/michaelarrington/767387236/)，其他解决这些需求的初创公司如 IBM 也参加了(参见他们的 [Many Eyes](https://web.archive.org/web/20211127064153/http://services.alphaworks.ibm.com/manyeyes/home) 数据可视化产品)。

作为“新互联网”的代表，我绝对是大会的害群之马。我可能是唯一一个没有穿西装打领带的与会者(外面气温高达 100 度)，我的 Macbook Pro 吸引了这些只看 Windows 的政府人群的目光。但是他们真正感兴趣的是网络的发展和大多数新创公司的开放数据方法，以及如何利用这一点尽可能有效地将政府数据送到经济学家和其他人手中。

Swivel 在成为经合组织官方数据事实上的存储库方面有着巨大的优势。通过今年春天完成的一项交易，OECD 已经是 swivel 上的“官方来源”,并且已经在其网站上托管了大量的 OECD 数据。

除了他们与经合组织的官方关系，Swivel 还提供建模，以及 IBM 和其他瞄准该业务的公司提供的可视化工具(见我们在 Swivelbeta 版上的[原始帖子)。](https://web.archive.org/web/20211127064153/http://www.beta.techcrunch.com/2006/12/05/swivel-to-launch-this-week-communitize-your-data/) [Freebase](https://web.archive.org/web/20211127064153/http://www.beta.techcrunch.com/2007/03/09/this-is-cool-unless-it-achieves-consciousness-and-kills-us-all/) 是存放这种枣的另一个自然选择。无论发生什么，更广泛地传播这种统计数据的目标是崇高的。我希望它很快发生。
