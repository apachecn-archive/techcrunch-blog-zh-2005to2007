# zvents 今日发布| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/10/05/zvents-launches-today/>

今天， [zvents](https://web.archive.org/web/20161024063347/http://www.zvents.com/) [团队](https://web.archive.org/web/20161024063347/http://www.zvents.com/welcome/about)揭开了他们社交活动经理/日历的包装。宣布的时机再好不过了，因为 Upcoming.org(其竞争对手)[昨天宣布被雅虎收购](https://web.archive.org/web/20161024063347/http://www.waxy.org/archive/2005/10/04/yahoo_an.shtml)。

Ethan Stock 在 zvents 博客上发布了这次发布。主要功能包括同类最佳的活动搜索、活动标记、简单的博客和博客/网站小部件，以推广您的活动。zvents 智能地使用 ajax，网站感觉非常稳定。

Zvents 还开放了许多 API 来获取他们的数据，鼓励 mashups。我们对此表示赞赏，并期待事态发展。
