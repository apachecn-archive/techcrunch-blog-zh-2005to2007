# Plaxo 新版本发布:更多同步更频繁

[![](img/c3da33b0c1b291c9fa2b1db1471983d7.png) ](https://web.archive.org/web/20211127055130/http://www.plaxo.com/) [Plaxo](https://web.archive.org/web/20211127055130/http://www.crunchbase.com/company/plaxo) 发布了Plaxo 3.0，这是一个基于多点同步产品的新版本，提供了改进的同步选项以及 feed 风格的人物跟踪器。

Plaxo 3.0 下的自动“多路同步”包括谷歌日历、微软(Outlook、Outlook Express、Vista Mail 和 Hotmail)、雅虎(邮件和日历)、Mac OSX(用于通讯录、iCal 和 iPhone)、AOL (AIM 和 Classic AOL)、Mozilla Thunderbird 和 LinkedIn。新的多组件“智能地址簿”网络应用程序包括联系人、日历、任务、笔记等标签。改进后的地址簿组件包括通过 Jajah 的点击呼叫电话号码和雅虎的地图。

从 Plaxo[收购 HipCal](https://web.archive.org/web/20211127055130/http://www.beta.techcrunch.com/2006/05/01/plaxo-acquires-calendar-startup-hipcal/#more-1398) 获得的技术提供了新的日历功能，包括“即将到来”的视图，其中包括天气、重大事件的倒计时以及来自 Flickr 的嵌入照片。

Plaxo 新的“Pulse”功能聚合了来自朋友的内容。

据说 Plaxo 拥有 1500 万注册用户。之前的 TechCrunch 报道[点击这里](https://web.archive.org/web/20211127055130/http://www.beta.techcrunch.com/tag/plaxo/)。
