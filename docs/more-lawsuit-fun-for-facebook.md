# 脸书获得更多诉讼乐趣

在接下来的几个月里，脸书将不得不面对至少又一场诉讼。他们不仅仅是在处理 [ConnectU 的诉讼](https://web.archive.org/web/20220121000650/http://www.beta.techcrunch.com/2007/07/25/facebook-in-court-with-connectu-press-conference-at-3-pm-pst/)——上个月在宾夕法尼亚州，另一项针对他们的诉讼被提起。

这起诉讼的关键在于美国专利 [6，519，629](https://web.archive.org/web/20220121000650/http://patft.uspto.gov/netacgi/nph-Parser?u=%2Fnetahtml%2Fsrchnum.htm&Sect1=PTO1&Sect2=HITOFF&p=1&r=1&l=50&f=G&d=PALL&s1=6519629.PN.&OS=PN/6519629&RS=PN/6519629) (“为有共同兴趣的用户创建一个互动社区的系统”)，该专利于 2003 年发布，现由原告[Cross Atlantic Capital Partners](https://web.archive.org/web/20220121000650/http://www.xacp.com/)所有。

这是一起彻头彻尾的专利侵权指控，原告要求脸书支付专利使用费。专利声明涉及围绕具有“共同兴趣”的用户社区建立的社交网络。脸书，以大学、商业等为基础的网络。，自然是专利攻势的最大目标。

今天早些时候，我给脸书发了邮件，征求意见。他们还没有回复。
