# 飞利浦 lum alive–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/25/philips-lumalive/>

![](img/a3b153548cd744a0c94d0d018681a9b3.png)

我们已经看到了我们的可穿戴技术，但是飞利浦的 Lumalive 还是第一次。这项新技术将 led 直接集成到材料中。新的工艺不会留下 led 的可见痕迹，但该设计的表现与传统 led 完全相同。

除了 Lumalive 的所有显而易见的奇特应用，我可以看到它实际上对许多实际应用很有用。它可以预见地集成到跑步者和骑自行车者的背心中，以警告驾驶员他们的存在，等等。生产就绪模型将在 2006 年 IFA 车展上展示。
