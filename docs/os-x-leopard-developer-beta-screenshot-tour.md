# OS X 豹开发者测试版截图之旅

![leopardinstall.jpg](img/a8dfecfe0f751c41a66e9b7795c75d0b.png)
参加 WWDC 的一名开发人员带我们进行了一次 Leopard 新特性的截图之旅。一些更有趣的功能是 Spaces，它可以让你定义多达 16 个虚拟桌面来放置你的应用程序，Mail.app 中的待办事项列表和 Safari 中的 Dashboard 服务功能。

还有各种各样的东西，如 iChat 获得 Mail 的外观，Spotlight 中的更多搜索标准，以及在预览中做笔记。漫步到哈德曼克获得完整的图片预览。
