# 马里奥之声不是意大利人？！

本着我每周马里奥帖子的精神，这里有一个谷歌视频，是那个给马里奥和其他任天堂角色配音的人。根据 Charles Martinet 在[维基百科](https://web.archive.org/web/20210306120542/http://en.wikipedia.org/wiki/Charles_Martinet)的简介，他还为 Luigi、Wario、瓦路易基和蟾蜍沃斯配音。

虽然他的标志性游戏是《超级马里奥 64》，但我会永远记得马里奥的“我们走吧！”超级马里奥赛车 64 中的一句话钻进了我的大脑。
