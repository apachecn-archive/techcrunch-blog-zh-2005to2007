# 贾斯珀·莫里森设计的三星 E590 很差劲

![l_20070604.jpg](img/6c35909f25cee8d90dbea9ccef2426f8.png)

三星与设计师贾斯珀·莫里森合作，推出了一款多媒体丰富的手机，看起来也不错。嗯……这很好，但 E590 与三星已经提供的大多数糖果棒手机有什么不同吗？依我看确实不是。

我认为 320 万像素摄像头提供的许多功能弥补了它缺乏闪光的缺点。自动对焦、半快门、全景拍摄功能和可爱的豆包三脚架确实让 E590 更具吸引力。MP3 播放器、蓝牙 2.0、视频录制和 GPRS/EDGE 服务也有帮助。

E590 将于本月在欧洲、中国和东南亚上市，有贵族黑和雪银两种颜色。
