# 松下选择在外置 DVD 刻录机中使用总线电源

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/28/panasonic-opts-for-bus-power-in-external-dvd-burner/>

一件事，我们都是关于总线电源，这就是为什么我们对这个松下便携式 DVD 播放器/刻录机感到兴奋。把它插到你的 USB 接口上，就大功告成了。没有电池，没有电线。更重要的是，这是一个 DVD 多刻录机，这意味着无论你的 DVD 是哪种格式，你都可以覆盖。伙计，它也很漂亮。

还没有定价，但我们猜测它的价格会在 200 美元左右。尽管我们只是一家大型电子产品零售商的营销总监，但我们知道些什么呢？
