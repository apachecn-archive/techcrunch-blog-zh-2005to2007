# ComBOTS 是为你的头像提供简单的 VOIP 和文件传输

 [](https://web.archive.org/web/20221209132343/http://www.combots.com/) [ComBOTS](https://web.archive.org/web/20221209132343/http://combots.com/) 是一家德国公司，刚刚将其产品推向测试阶段，似乎旨在从 Skype 的潜在客户群中分一杯羹。你现在必须排队申请一个账户，但还是值得先看看这个网站。ComBOTS 是一个基于 Windows 的桌面客户端，通过 VOIP、文件传输、即时消息和大量的虚拟角色来促进一对一的通信。该服务宣传自己是无垃圾邮件、拖放式通信。

很明显，一家初创公司在推出之前花了很多钱在网站上，对我来说，ComBOTS 有些东西太精致了。尽管如此，我认为对于那些经常在网上与少数人交流的人来说，这可能是一个非常可行的产品。界面看起来非常干净和实用。该公司已经筹备了 3 年，由德国门户网站 Web.de 的一个团队创建。

这项服务最初是免费使用的，在持续使用一段时间后，用户需要购买套餐。根据持续时间的不同，订阅费每月在 2 到 3 欧元之间。还鼓励用户以大约 1 欧元或大约 13 美元(10 欧元)的价格购买具有各种表情或手势的化身。这项服务是免费的。

该公司周五宣布，它已与新线电影公司就《指环王》的化身 Paws(加菲猫)和《花生》的所有者联合媒体达成许可协议。小心 [CyWorld](https://web.archive.org/web/20221209132343/http://www.beta.techcrunch.com/2006/07/27/this-is-nuts-cyworld-us-opens-for-use/) 。

尽管有巨大的经济潜力，我认为头像和表情符号超载是愚蠢的——所以让我们看看这里的其他功能。

ComBOTS 桌面界面是一圈图标，当您将鼠标悬停在桌面上朋友的头像上时，就会出现这些图标。其中一个按钮是用于文件传输的。当一个文件被放入图标时，系统会提示你添加评论，然后文件就会被发送出去。该公司表示，可以发送大文件，甚至数百张照片。没有提到转移限制，但我相信对于作为目标的临时用户来说，这可能就足够了。该系统还支持离线传送传输的文件。

另一个按钮是网络电话，显然是免费的。在销售虚拟形象和廉价订阅的策略中，让 VOIP 成为亏损的领头羊是很有意思的。

我和其他人一样喜欢微笑或皱眉表情，但我认为网络电话将是这里最大的特色。如果它和系统的其他部分一样容易使用，我可以想象许多 Skype 用户会选择这个简单得多的工具。Skype 很棒，但如果只有一两个人经常用它来交流，那就有点过头了。

雅虎！带语音的信使是另一个可以满足这些需求的选择，包括对化身的明显重要的需求。和雅虎一样多。被认为是一家网络和电子邮件公司，然而，ComBOTS 声称是一个封闭的，无垃圾邮件的系统可能对一些人更有吸引力。

我知道，许多人指望系统能在消费者领域进行封闭的小群体沟通，并让基于无处不在的宽带的工具对非技术用户变得简单——但我发现 ComBOTS 令人厌恶地以高人一等的姿态出现。至少德国博客圈的一些草根阶层似乎也不满意，Technorati 发现标题像“[组合机器人必须死](https://web.archive.org/web/20221209132343/http://blog.computerwoche.de/2006/09/04/combots-must-die/)”我知道我宁愿帮助我的家人学习使用 Skype，也不愿在他们心情好的时候收到他们发来的咕噜咕噜的加菲猫头像。尽管如此，我认为像 ComBOTS 这样的工具确实存在一个巨大的市场，而且对许多人来说这可能是有意义的。
