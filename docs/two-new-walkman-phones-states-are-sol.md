# 两款新的随身听手机在美国上市

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/15/two-new-walkman-phones-states-are-sol/>

![](img/42d614fa4484ea314001f486d949a7cf.png)

索尼爱立信刚刚发布了两款我们永远不会在美国看到的新手机。w960(右图)配备了强大的功能，如 WiFi、用于浏览媒体的 240×320 触摸屏、320 万像素摄像头和 8GB 存储空间。

红色 w910 有一个很酷的播放模式，可以根据你的心情播放音乐。这个功能对于 emo 人群来说显然是没有用的。然而，更无用的功能之一是能够摇动手机来访问音乐商店。如果你喜欢把手机从包里拿出来看看艾薇儿·拉维尼的新专辑，这是一个很酷的功能。
