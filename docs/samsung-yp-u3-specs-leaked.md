# 三星 YP U3 规格泄露

三星最新推出的 U 系列 mp3 播放器对我来说没什么作用，但它确实比 iPod Shuffle 强多了。U3 将有 1、2 和 4GB 型号，配有 1 英寸有机发光二极管显示屏和 15 小时的嘻哈乐。它还具有 RDS 调谐器，支持 MP3，WMA，ASF 和 Ogg 文件。顶部的蓝光边缘隐藏了 USB 连接器，并没有做太多其他事情。U3 还没有“正式”宣布，所以没有上市日期和价格。
