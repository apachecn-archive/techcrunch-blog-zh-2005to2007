# Vista 毕竟不是一个失败，实际上比 XP 做得更好|技术崩溃

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/27/vista-not-a-flop-after-all-actually-doing-better-than-xp-did/>

[![vistalogo.jpg](img/002db394cb2ec3b0b7897eb9461f08e3.png)](https://web.archive.org/web/20150515213612/http://old.crunchgear.com/wp-content/uploads/vistalogo.jpg "vistalogo.jpg")

互联网上的传统观点认为，到目前为止，Windows Vista 是一个失败。但是你知道他们怎么说传统智慧。在最近发布的季度收益报告中(我们在前面提到了 T4)，微软称 Vista 所在的公司部门的销售额比上一年增长了 67%。在购买 Vista 的人中，整整 71%的人选择了价格更高的操作系统版本。此外，微软表示，85%的 Windows 销售是 Vista 销售，超过了同期 XP 的销售数字。

换句话说，Vista 几乎没有失败。

虽然一些企业可能仍然害怕跳转到 Vista，但 Vista 看起来确实不错。
