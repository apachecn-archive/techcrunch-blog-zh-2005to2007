# 下一个 iPod 或 iPhone 可能会有动态歌词显示

![](img/20a362ebb36986303d143ed25c425f46.png)

苹果公司最近公布的一项专利显示，下一代 iPod 或 iPhone 可能会有一种新的显示歌词的方式。专利显示，显示的文本将以某种方式突出显示，无论是颜色、下划线还是文本周围的方框。这可以用于卡拉 ok 或跟随有声读物。到目前为止，对于总部位于库比蒂诺的公司来说，这听起来是个聪明的主意。

然而，还有一个问题。你为什么想用 iPod 或 iPhone 来唱卡拉 ok？这两款设备的屏幕都很小，观看歌词会很困难，上帝禁止你用一只手来做，因为你要用另一只手来摇麦克风。也许苹果对 iPod 和 iTunes 的未来有所准备。
