# 索尼 Vaio 笔记本电脑泄漏

我真的很喜欢互联网。它让我的一天过得更快。一些笔记本评论论坛成员在欧洲索尼网站上做了一些挖掘，发现了最新的 VAIO 笔记本电脑。五个不同的系列将在 7 月发布，听起来肯定很棒。

11.1 英寸的 TZ，13.3 英寸的 SZ5 和两个新的 15.4 英寸的 FZ 将首次亮相，随着谣言的 13.3 英寸 CZ 和 14 英寸 CR 系列笔记本电脑。有一些猜测认为，TZ 系列将彻底彻底改革内外。新的机箱设计将包括一个内置的网络摄像头和重新设计的键盘。

要是我能在市场上买到一台新的笔记本电脑，并且愿意回到 Window 的机器上就好了！全力以赴。
