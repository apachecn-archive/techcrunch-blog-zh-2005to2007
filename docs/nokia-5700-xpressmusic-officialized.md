# 诺基亚 5700 XpressMusic 正式上市

没过多久， [5700 XpressMusic 传闻](https://web.archive.org/web/20130628171816/http://crunchgear.com/2007/03/05/nokia-5700-xpressmusic-in-the-works/)就正式面世了。来自诺基亚的最新 3G 手机拥有 [5300 XpressMusic](https://web.archive.org/web/20130628171816/http://crunchgear.com/2007/03/03/nokia-5300-xpressmusic-hands-on/) 的许多特征，但它有一个不错的转变，真的。旋转底部也不只是为了展示:每个位置都是手机的不同模式。5700 最明显的功能是音乐播放器，然后是 200 万像素的摄像头，然后是视频通话，然后是智能手机。

5700 配备了一个可以升级到 2GB 的 MicroSD 插槽。如果您不想使用 3.5 毫米插头耳塞或内置扬声器尽情摇滚，请将音频流式传输到 A2DP 蓝牙立体声耳机。音乐播放器还支持专辑封面、播放列表和带有音频可视化的 5 波段均衡器。5700 XpressMusic 预计将于下个季度在欧洲上市，价格为€350 英镑，并将有红色和黑色两种版本。
