# Zimbra 新增 1200 万客户——与 Comcast 合作

两家公司将于今晚宣布，康卡斯特将放弃其过时的网络邮件软件，代之以 Zimbra 的 Ajax 办公套件。这对康卡斯特的 1200 万宽带客户来说是个好消息，对 Zimbra 来说更是个好消息——这笔交易将使 Zimbra 的用户数量从目前的[600 万左右](https://web.archive.org/web/20221231065044/http://techcrunch.com/2007/01/25/zimbra-6-million-paying-customers/)大幅增长。

该协议还包括新的功能，包括让康卡斯特三网融合客户(VOIP 电话、互联网、有线电视)能够在线收听语音邮件，并通过电子邮件转发。用户也将能够从 Zimbra 客户端管理即时消息，两家公司正在将 Plaxo 的地址簿功能添加到组合中。

Zimbra 最近发布了一款新软件，可以让客户离线访问他们的网络邮件。该公司通过开源下载免费提供基本服务。他们收取客户服务费用，并通过经销商分销他们的高级版本。他们已经筹集了超过 3000 万美元的风险资本。
