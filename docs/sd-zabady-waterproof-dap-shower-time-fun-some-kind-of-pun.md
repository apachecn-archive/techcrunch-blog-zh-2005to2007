# SD 扎巴迪防水 DAP:淋浴时间的乐趣，某种双关语

[![zabady.jpg](img/773322cba79bff6ae14ef00b6d188661.png)](https://web.archive.org/web/20201021081503/http://old.crunchgear.com/wp-content/uploads/zabady.jpg "zabady.jpg")

我猜北方的夏天带来了防水小玩意的涌入。日本的最新产品是 Twin Bird 的 SD Zabady，这是一款防水的数字音频播放器，它坚持最基本的设计，为顾客带来愉快的防水体验。它内置的 128MB 存储空间被包含的 SD 卡插槽所抵消，使其总容量达到更主流(也更易管理)的 2GB。它可以播放 MP3 和 WMA 文件，每组 AAA 电池可持续使用约 18 小时。机器翻译的文本没有透露扎巴迪能在水下走多远，但照片似乎表明，它在任何真正的深度都是防水的。

这位身价 130 美元的球员偏偏在利物浦与 AC 米兰在冠军联赛决赛中相遇的这一天“倒下”。非常感谢 ESPN2 没有用高清显示比赛。这又不是什么大赛，也不是 2007 年。
