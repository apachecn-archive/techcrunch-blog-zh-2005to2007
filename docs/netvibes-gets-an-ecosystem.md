# NetVibes 得到了一个生态系统

 [](https://web.archive.org/web/20220816033450/http://www.netvibes.com/) 总部位于巴黎/伦敦的 [NetVibes](https://web.archive.org/web/20220816033450/http://www.netvibes.com/) ，提供个性化的 Ajax 主页，昨天发布了重大升级，包括新的插件和社区共享功能。NetVibes 首席执行官 Tariq Krim 向我们介绍了这个新版本。

首先，他们发布了 2 个新插件，可以在默认的 feed 菜单中访问。一个显示最热门的新闻，另一个帮助你监控易趣的活动。他们还发布了键盘控制,方便了页面和标签的浏览和使用(编辑、删除、帮助)。

![](img/a40d22eb11a983a5b138cae5cd81a045.png)但**关键创新在别处**。NetVibes 推出了生态系统(可在 eco.netvibes.com 访问)，这是一个由用户管理的 NetVibes 内容和服务目录。它将使任何人都可以轻松地创建、分享和评价任何提要、插件甚至标签。你可以通过点击“+”按钮直接在你的 NetVibes 主页上添加任何你想要的东西。该目录可以通过项目、标签和搜索引擎进行浏览。例如，你可以添加一个[“web 2.0”标签](https://web.archive.org/web/20220816033450/http://eco.netvibes.com/tabs/522/web-2.0)或者一个[“世界杯”标签](https://web.archive.org/web/20220816033450/http://eco.netvibes.com/tabs/389/world-cup-2006)到你的主页上，其中包括一个很好的提要选择。您也可以通过点击“发布到生态系统”按钮直接从您的主页填充目录，该按钮位于“刷新”功能左侧的[或每个“订阅块”右上角的](https://web.archive.org/web/20220816033450/https://beta.techcrunch.com/wp-content/uploads/2006/05/share_feed.png)或每个选项卡的菜单内(查看侧面的截图)。然后，您将进入一个编辑页面，给出您希望共享的项目的一些详细信息(见下面的截图)。分享现在只是公开的，我认为和你的朋友私下分享也很棒。

换句话说，NetVibes 更加面向社区，使用户能够与开发者互动，反之亦然。

NetVibes 也发布了一个 [API](https://web.archive.org/web/20220816033450/http://eco.netvibes.com/developers/) ，可以开发新的插件(超过 74 个[已经出现](https://web.archive.org/web/20220816033450/http://eco.netvibes.com/modules/))。从昨天开始增加了 8 个新的，一些非常有趣的还在路上(视频，网飞，myspace)

这发生在 PageFlakes [宣布来自 Benchmark](https://web.archive.org/web/20220816033450/http://www.beta.techcrunch.com/2006/05/30/benchmark-invests-in-pageflakes-ajax-desktop-war-heats-up/) 的投资之后。看起来现场比赛越来越激烈了。
