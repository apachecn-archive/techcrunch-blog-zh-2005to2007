# Podtech 到底是怎么回事？

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/07/27/whats-really-going-on-with-podtech/>

**更新:** [看到这里](https://web.archive.org/web/20230218173743/https://techcrunch.com/2007/07/27/dear-podtech-im-not-your-vp-marketing/)。

 [](https://web.archive.org/web/20230218173743/http://www.crunchbase.com/company/podtech) 有大量报道称，总部位于硅谷的 pod tech正在分崩离析。我采访了 Podtech 的首席执行官约翰·弗瑞尔，问他有多少谣言是真的。虽然他确实给了我一些废话，但这家公司至少还有一些生机。

2006 年 3 月，Podtech 筹集了当时规模较大的首轮融资——550 万美元。那时，他们很大程度上只专注于制作音频内容——对企业家的采访和付费的“商业信息”类型的企业播客。他们很快进入视频领域，从微软聘请了罗伯特·斯考伯，并开始认真制作大量内容。

快进到 2007 年的春天。Furrier 说，他们开始在视频领域发起激烈的竞争。该公司没有创造更多的内容，而是开始解雇一些专注于内容的员工，并在新的创意开发副总裁 Steve Gillmor 的指导下重新专注于整合第三方内容。像 1938 Media 的 Loren Feldman 和 National Banana 这样的人才加入进来，开始使用 Podtech 的 Flash 播放器，以换取有保证的广告收入。还有很多其他节目——见他们网站[的右边栏](https://web.archive.org/web/20230218173743/http://www.podtech.net/home/)。

所有员工都被鼓励花些时间创作内容，就像罗伯特·斯考伯在《斯考伯秀》中所做的那样。Furrier 说，除此之外，该公司不再从事内容创作业务。

但是 Podtech 需要第二轮融资…他们无法说服新的投资者以目前的流量水平和收入进入。两周前，现有投资者 US Venture Partners 和 Venrock Associates 以与第一轮相同的估值向该公司再投资 200 万美元。皮货商说，在收入和新资本之间，它们还能维持几个月。

Furrier 也在为公司寻找新的首席执行官。一些传言称，这是 Podtech 的风险投资者对新一轮资本提出的要求之一。Furrier 说这是他的主意，公司正处于需要一个不同类型的领导者的阶段。他说，他将继续担任公司总裁。

他们的新模式是一个很好的模式，发现并签下有保证收入的人才，然后将网站聚合给广告商。他们需要继续裁员，让他们的烧钱率非常低——他们将有效地成为他们合作伙伴的制作公司和广告销售团队。如果他们能在现金耗尽之前完成转型，波德科技仍有光明的未来。
