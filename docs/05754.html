Technorati宣布支持Open ID

原文：https://web.archive.org/web/http://www.techcrunch.com/2006/10/19/technorati-announces-support-for-open-id/trackback/

博客搜索引擎Technorati 今晚宣布，博主们现在可以用Open ID凭证宣称他们博客的所有权，更多支持Open ID的工作正在进行中。

Open ID提供了一个URL，您可以使用它在网络上的所有参与站点上标识自己。您可以自己处理物流，也可以付钱给许多不同的供应商来注册您的ID并解决对您的信息的请求。

传播Open ID的运动旨在让我们的生活更轻松，我们的数据更安全，并帮助用户避免大供应商不公平地滥用权力的封闭孤岛。Technorati对该标准的持续和有意义的支持是一件大事。将它加入到Technorati正在进行的前瞻性工作中，比如微格式。

许多其他供应商，包括像微软、甲骨文和太阳这样的大公司，都在参与开放ID的讨论，但对于新的网络社区中的主要参与者之一来说，做出这样切实的举动将为往往看起来很边缘的草根努力提供支持。当Web 2.0场景中更面向社区的玩家首先行动时，他们就可以对未来的格局施加更大的影响。

正如Technorati博客文章所指出的，其他支持Open ID的供应商包括开拓者 LiveJournal 和Six Apart的 Vox 。

有关Open ID的更多信息和背景，请参见伯克曼中心的IdentityGang.org，我在7月份对Open ID倡导者卡莉娅·哈姆林的采访以及维基百科关于 Open ID 的条目。
