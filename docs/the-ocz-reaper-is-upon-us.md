# (OCZ)死神来了

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/22/the-ocz-reaper-is-upon-us/>

害怕死神吗？通常是的，但不是 OCZ 的死神。PC2-8500 Reaper HPC 已经发布，游戏玩家应该感到高兴。

hoity-toity 高性能游戏 RAM 的时钟频率为 1066MHz，运行频率为 CL 5-5-5，采用 2GB 双通道套件。收割者配备了一个新的散热器，通过两个铜管散热，这两个铜管馈入一个紧凑的铝鳍片阵列。收割者还拥有一个 EPP 编程的 SPD，它与新的英伟达 SLI 芯片组兼容，它有 2.4V 的 EVP 额定值，尽管它只需要 2.1V。很漂亮，是吧？价格还没有消息。
