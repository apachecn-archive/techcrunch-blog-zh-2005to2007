聊天中最大的六个新想法

原文：https://web.archive.org/web/http://www.techcrunch.com:80/2006/11/24/the-six-biggest-new-ideas-in-chat/comment-page-2/

即时消息已经成为网上日常生活的一部分。我使用几种不同的服务，这取决于我想做什么和我想和谁说话。 AIM 非常适合与老朋友保持联系， Meebo 或 eBuddy 适合在任何地方注册，而 Skype 适合商务互动。我使用这些不同的服务是因为它们在某些关键特性上的优势。像 Meebo 、 Skype 、 Wablet 和 AOL收购的 Userplane这样的公司正在通过下一代网络即时通讯来推进这些功能。

实时通讯是当今网络上最具创新性的领域之一。以下是当今网络即时通讯中出现的一些重大想法，以及体现这些想法的服务。

1。互操作性
在AIM和ICQ取得初步成功之后，又冒出了其他几个聊天服务。像 Trillian 、 Gaim 、 Adium 和 Miranda 这样的服务开发了跨越不同协议进行通信的黑客技术。在一段时间的“猫和老鼠”之后，AOL通过对AIM进行小的修改来对抗互操作性，跨平台互操作性已经成为大多数新聊天程序的标准功能。雅虎信使和Windows Live信使今年夏天宣布了互操作性。AOL现在有一个开放的开发API，Google Talk运行在开放的Jabber协议上。显然，开放标准已经存在。

2。浏览器内聊天
AIM Express是聊天程序的早期版本，它脱离了可下载的客户端，转而在你的网络浏览器中运行。像 Meebo 和 eBuddy 这样的服务通过使用AJAX和桥接聊天协议开发了更丰富的用户界面。红杉支持Meebo(据传300-400万美元)，而低地基金eBuddy(500万欧元)继续争夺这一领域。Meebo通过他们的 MeeboMe 小工具允许在任何网站上进行嵌入式聊天，从而进一步拓展了业务。自从我们去年12月报道了Meebo的数据后，它就一直在稳步增长。每日登录和消息量增长了5倍，达到每天120万次登录和7000万条消息。Meebo声称每月有450万个uniques和70万个Meebo用户帐户。随着社交网站的普及，基于浏览器的即时消息将聊天带到了用户在网络上的位置。

3。基于位置的聊天即时通讯程序通过互联网将人们联系在一起。像 RadiusIM 和 Meetro 这样的新程序通过人们在现实世界中的位置将他们联系起来。RadiusIM是一个AJAX应用程序，而Meetro是一个可下载的客户端。两者都允许你填写你的位置和个人资料，以此来结识你所在地区的新朋友，甚至是另一个国家的人。与聊天的其他发展不同，基于位置的即时消息还没有在其他平台上被大量采用，这些平台继续根据用户生成的好友列表来联系人们。

4。灵活的身份
随着网络个人资料在网络上的增长，人们也越来越需要将你的私人面孔和职业面孔分开。虽然用户可以通过管理各种即时消息句柄来解决这个问题，但基于Flash的 Wablet ( 我们的报道)使剖析成为其系统的核心功能。Wablet允许你用不同的个人资料创建多个角色。然后你可以在网上嵌入这些聊天窗口，并控制每个访问者看到哪个角色。在不久的将来，Wablet将整合OpenID。聊天服务 ScribbleHere 目前使用OpenID。

5。上下文聊天
一些新的创业公司已经出现，并改变了即时通讯的上下文，从好友列表到网站和兴趣。虽然类似于老式的IRC聊天室，根据话题进行对话，但像me . medium、 Geesee 、新推出的 InCircles 、other online和 3bubbles 这样的公司已经在不同程度上将你在网络上的位置融入到聊天中。

3Bubbles是一个嵌入程序，可以为每篇博文添加一个帖子专用的聊天窗口。GeeSee走得更远，基于标签将不同网站的用户联系起来，例如，所有的技术博客可以共享一个公共聊天室。 InCircles，在这里是为了测试而嵌入的，操作类似，但是针对博客侧边栏进行了优化。OthersOnline和Me.dium结合了冲浪习惯，将兴趣相似的用户联系在一起。两者都是浏览器插件。OthersOnline专注于将经常访问相同网站、拥有相似个人资料的人联系起来，在下拉菜单中列出相似的用户。Me.dium的方法有点匿名，仅根据用户在相关网站上的存在，通过句柄连接用户。你与朋友和其他冲浪者的关系显示在“雷达”地图上，它向你显示其他用户访问你的网站或其他喜欢它的网站。

6。富媒体聊天
网络摄像头和麦克风出现在网络上已经有一段时间了，但是宽带、VOIP标准的增长，以及通过像 Skype 、 Google Talk 和 Yahoo！Messenger 已经扩大了在聊天程序中的使用。Skype和雅虎！支持拨打座机和手机(Skype年底前免费)。PalTalk创建了语音聊天室，可以同时主持数千人之间的对话。虽然语音和视频不适合许多IM用户进行的同步对话，但富媒体集成带来了基于文本的IM所没有的微妙和深度。
