# 你一整天都会读到的最安全的东西

![](img/20b30b0e142c1118b25ab456da3a9f9e.png)

我不知道你的情况，但上次我在家里检查烟雾报警器的时候，电池早就没电了。通常我只是关掉它或者出去买另一个电池，但是如果在电池之间的宽限期内发生火灾怎么办？我们中的许多人都倾向于忽视消防安全，杜邦公司也知道这一点，因此他们发明了一种再也不需要电池的烟雾报警器——T2。

该设备的工作原理是将一个标准灯泡拧入其中，然后将该设备拧入您的灯具中。这样，当您点亮房间时，设备会充电，您会受到保护。它甚至可以在没有电源的情况下工作 30 天。当然，39.99 美元对于烟雾报警器来说有点贵，但是想想电池和您将拯救的生命。
