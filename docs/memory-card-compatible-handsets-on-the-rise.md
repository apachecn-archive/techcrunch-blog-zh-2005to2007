# 存储卡兼容手机正在崛起

总部位于以色列的 mPortico 公司进行的一项调查显示，在过去的 24 个月里，带移动存储卡的手机在全球的普及率大幅提高。这项研究基于目前在美国、欧洲和亚洲销售的 990 部手机。最初的数据进一步证实，带有 MicroSD 和 MiniSD 扩展槽的移动设备目前占市场上手机的最大份额。
