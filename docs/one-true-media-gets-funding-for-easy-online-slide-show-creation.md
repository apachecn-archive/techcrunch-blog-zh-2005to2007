# 一家真正的媒体获得资金支持轻松创建在线幻灯片 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/06/11/one-true-media-gets-funding-for-easy-online-slide-show-creation/>

[![](img/490acb970cc15fa74b4dab5d66d8b836.png) ](https://web.archive.org/web/20221002115110/http://onetruemedia.com/) [一家名副其实的媒体](https://web.archive.org/web/20221002115110/http://onetruemedia.com/)今天将宣布从[凯鹏华盈和拜尔斯](https://web.archive.org/web/20221002115110/http://www.kpcb.com/)获得 500 万美元的新资金。该服务提供在线视频和照片幻灯片编辑和共享。

One True Media 的服务可用性很高，定位于家庭和在线社交网络用户，但在处理在线多媒体的病毒性质与其商业利益之间的紧张关系时倾向于锁定。这种紧张局势是由该公司的商业模式造成的，该模式旨在将数据存储和像 DVD 和相册这样的硬拷贝产品的创作货币化。鉴于该公司的目标人口统计数据，在不创建简单帐户的情况下将文件导出到其他系统或共享文件的困难可能不会对该服务的增长构成严重威胁。

这正是那种可以让基于网络的多媒体编辑和共享成为主流活动的高质量系统。这可能不会让“数据想要免费”的人群高兴，但它可以表明 Web 2.0 不仅仅是为极客准备的。

**历史**

成立于 2005 年 2 月并于同年 12 月推出的 [One True Media](https://web.archive.org/web/20221002115110/http://onetruemedia.com/) 由首席执行官马克·摩尔和联合创始人约翰·洛夫领导。摩尔是一名计算机科学家，在甲骨文开始了他的职业生涯，并创办了三家初创公司。Love 有 MBA，有 IT 高管背景。马克·弗莱彻(Mark Fletcher)是一名天使投资人和顾问委员会成员，直到上周还是 Bloglines 的负责人。

本周宣布凯鹏华盈投资该公司意味着一个真正的媒体加入了包括 WebMD、女性在线社区 iVillage、Audible、Amazon.com、谷歌和 450 多家其他公司的投资组合。凯鹏华盈团队成立于 1972 年，成员包括比尔·乔伊、威廉·r·赫斯特三世和许多其他知名人士。凯鹏华盈(Kleiner Perkins)合伙人兰迪·科米萨尔(Randy Komisar)是卢卡斯艺术娱乐公司(LucasArts Entertainment)的前首席执行官，他已加入 One True Media 董事会。

**可用性**

一个真正的媒体最强大的部分是系统的可用性。flash uploader 和下拉菜单用于选择从剪辑持续时间到过渡的所有内容，可能会为该服务的目标客户提供恰到好处的创意控制。缩放至裁剪和简单的平移功能可能看起来很老套，但当它是婴儿照片或在 MySpace 上设置为 Coldplay 的高中记忆时，那就没问题了。一首 Coldplay 歌曲是可用的，默认情况下还有其他 50 首歌曲，用户可以上传任何 Mp3 文件作为配乐。混音功能允许用户逐个剪辑地将音乐或视频声音降低到相对的程度，以防有人在吉他即兴重复之间说了一些重要的话。

**商业模式**

一个真正的媒体专注于帮助用户创建他们想要购买的 DVD 幻灯片(1 张 25 美元，数量越多价格越低)，变成印刷书籍(前 20 页 30 美元)或长期在线存储(4 美元/月)。高级订阅允许字幕，无限上传，20 GB 存储，个性化视频播放器，每月高达 5000 次观看，并访问 MP4 格式的文件。未付费账户可以存储文件 60 天。该公司还提供视频和照片扫描和修复服务。

对用户文件的访问仅限于高级订阅或购买 DVD。因此，如果你希望免费试用一种真正的媒体，那就把你创造的东西带走，把系统留给它——你无法做到这一点。这可能很多家长和 MySpace 用户都能接受。

更难的是，在向非注册用户展示在另一个网站嵌入视频的代码之前，需要创建一个帐户。当然，一个页面的源代码可以被查看，所需的代码就在那里，但是像 YouTube 这样的网站却不具备这种便捷性。使用“通过电子邮件发送此幻灯片”链接也需要注册一个帐户。任何人都可以在没有帐户的情况下观看幻灯片。

尽管如此，该公司表示，它正在寻求利用在线多媒体的病毒式传播。如果发生这种情况，它将不会像迄今为止照片和视频共享中的 Web 2.0 宠儿那样。一个真正的媒体是一种不同的业务，显然正在采取许多措施让用户注册。在这项服务上线的头六个月，就有超过 33 万用户注册。

**合作伙伴**

在极端可用性的背后，One True Media 与其他在线服务的合作将是其未来的最大因素。迄今为止，最大的两个合作伙伴是为博客提供视频播放器的 [TypePad](https://web.archive.org/web/20221002115110/http://typepad.com/) 和强生的[BabyCenter.com](https://web.archive.org/web/20221002115110/http://babycenter.com/)。BabyCenter 的功能之一是一个公告板，上面有到 True Media 的联合品牌 BabyCenter 视频页面的突出链接。

该公司将投入大量资源开发和提高类似合作伙伴的能力，以此作为其发展战略的关键。

One True Media 的可用性和高调合作给我留下了深刻的印象。我喜欢订阅的商业模式。不过，据推测，该公司希望大量的人会购买 DVD 或相册。我想，今天大多数习惯于在网上处理照片和视频的人不会花钱把他们的幻灯片刻录成 DVD。这项服务非常容易使用，但是，它可能会被更多有兴趣购买此类服务的人所采用。

如果与大型、非技术性的在线社区建立更多的伙伴关系，那么发现这项服务的人将会发现一个强大的制作引人注目的幻灯片的系统。尽管目前的趋势是在线免费多媒体服务，但他们可能正是愿意为这种服务付费的那类人。这似乎是可能的，凯鹏华盈和 One True Media 的合作前景看起来很好。
