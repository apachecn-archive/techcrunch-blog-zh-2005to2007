# 索尼发布 007 Vaio TX 笔记本和 Cyber-shot 相机

像邦德一样有点难回答，除非你是索尼。日本动力公司向你，松弛的乔，提供限量版 VGNTX007C“间谍装备”包，配有索尼的 Vaio TX 笔记本电脑和 720 万像素的 Cyber-shot 相机。

Vaio TX 配有 11.1 英寸液晶显示屏，重量为 2.8 磅，电池续航时间为 11 小时。Cyber-shot DSC-T50 有一个 3.0 英寸的液晶显示器和一个卡尔·蔡司 Vario-Tessar 镜头，可以拍摄到 ISO 1000。一切都很好，直到你看到它的要价是 4299 加元，或 3774 美元。除非你身上有女王陛下的秘密支票簿，否则你可能会想远离这个。
