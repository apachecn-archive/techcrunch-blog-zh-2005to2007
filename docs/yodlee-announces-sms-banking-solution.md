# Yodlee 宣布短信银行解决方案

短信现在可以用于手机银行和支付。今天，Yodlee Inc .宣布为其 Yodlee 移动解决方案推出一项新的短信银行功能。Yodlee 移动短信提醒可以帮助客户避免透支费用，而用户可以使用该服务来检查余额，查看交易，查看支出报告，甚至进行紧急账单支付。

“很明显，移动银行战略至少需要包含两个关键要素:1 .2)通过多种渠道访问的普遍性。Yodlee 产品开发部的 SVP 彼得·黑兹勒赫斯特(Peter Hazlehurst)说:“这是一些相关的、个性化的内容，能够满足消费者在旅途中的真正需求。“Yodlee 正在通过广泛的分发渠道(用户可以通过移动网络浏览器 SMS 访问他们的内容)和利用我们的核心优势来满足这两种需求:在一个简单的移动界面中安全访问几乎所有的金融账户。”

Yodlee Mobile 提供 SMS/文本消息功能以及 WAP 浏览器功能。
