# LRG 和黛安·冯·芙丝汀宝的限量版跟班 3

我们几周前给你看的那张照片非常准确，直到最后那个小小的太空入侵者。嗯，除了绿色 LRG 设计——如果你是一个小妖精，这很酷——还有黛安·冯·芙丝汀宝的黑红唇设计。

两款机型都有相同的键盘和 130 万像素的摄像头，但有不同的颜色，嗯，就是这样。期待特殊的价格溢价。
