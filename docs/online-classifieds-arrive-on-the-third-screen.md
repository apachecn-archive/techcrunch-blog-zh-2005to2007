# 在线分类广告出现在第三个屏幕上

![edgeiogumiyo.jpg](img/64fb37833dd068255abe70893f3baca7.png)
扫描报纸上的分类广告是如此上千年，甚至扫描网上也是去年的新闻。很快，你就可以通过手机搜索二手网球拍、子球和其他难以找到的物品。Gumiyo 和 edgio 宣布了一项新的合作伙伴关系，将在线分类广告引入移动领域，这将从 edgio 的房地产部分开始，Gumiyo 将通过列表向感兴趣的人提供这些信息。潜在买家很快就可以在手机上查看房源，并附上照片。

Gumiyo 首席执行官兼联合创始人 Shuki Lehavi 表示:“我们与 edgeio 的合作非常互补，非常有意义。“edgeio 致力于在线分类广告领域的创新，并提供了丰富的开放 API 供我们使用。这是我们从第一天起就向他们推荐 Gumiyo 列表的一个重要原因。现在，edgeio 卖家可以不费吹灰之力进入移动领域。”

未来的合作可能会看到 Gumiyo 利用 MMS、SMS、WAP 和 widgets 构建应用程序来增强搜索界面。继续搜索！
