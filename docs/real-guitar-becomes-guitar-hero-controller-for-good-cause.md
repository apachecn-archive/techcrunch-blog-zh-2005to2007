# 真正的吉他成为吉他英雄控制器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/06/real-guitar-becomes-guitar-hero-controller-for-good-cause/>

![](img/6825aee904188245f8eae05bf9fd2dbe.png)

值得一提的是，如果你今晚住在纽约地区，你应该来参观我们的朋友 [Joel Johnson 的](https://web.archive.org/web/20160828033417/http://www.dethroner.com/)慈善活动，名为 Funde Razor。此次活动的收益将捐给 [Penny Arcade 的](https://web.archive.org/web/20160828033417/http://www.penny-arcade.com/)儿童游戏慈善机构，该机构通过视频游戏捐赠等方式帮助医院里的生病儿童。这将是一个伟大事业的伟大事件，但也将有一个定制的吉他英雄 II 控制器出售。

这把吉他是由一个来自工具商的和蔼的小伙子制作的。他取出了一个标准的吉他英雄控制器，并着手将其安装在 Squier Strat 中，这是一个天才，因为它的流行设计和低成本。最棒的是，完整的操作说明已经发布到网站上，如果你有资源、时间和耐心，你也可以做一个。最后的结果看起来很出色，我很想用一把真正的吉他来撕碎《吉他英雄 2》。如果你敢的话，着手这个项目，但是一定要在今晚的活动中寻找乐趣，粉碎文件，帮助一个好的事业。

*更新*:乔尔表示[另外两个吉他控制器](https://web.archive.org/web/20160828033417/http://www.flickr.com/photos/jmillerid/sets/72157594407203447/)也将被拍卖。甜度！
