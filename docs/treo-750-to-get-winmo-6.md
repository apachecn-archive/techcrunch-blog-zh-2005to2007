# Treo 750 获得 WinMo 6 | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/27/treo-750-to-get-winmo-6/>

Palm 今天将宣布我们已经知道的消息:Treo 750 将在今年晚些时候进行 Windows Mobile 6 的改造。该设备的新版本将预装移动操作系统，并将为现有用户发布更新。

Palm 公司无线解决方案总监 Joe Fabris 说:

> Palm 将继续与微软紧密合作，最大限度地发挥 Windows Mobile 的优势，并为 Treo 客户提供最佳的移动体验。我们标志性的 Palm 体验与 Treo 750 上 Windows Mobile 6 的优势相结合，将继续为美国和世界各地的企业提供出色的生产力工具。

有关 Treo 750 的更多信息，请参见今年早些时候的[我的评论](https://web.archive.org/web/20160421001302/http://crunchgear.com/2007/03/02/treo-750-hands-on/)。它有一个设备的完整纲要。
