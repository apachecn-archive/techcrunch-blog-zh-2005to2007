# 发现新的光环 3 盔甲类型

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/28/new-halo-3-armor-types-discovered/>

![](img/5fcac92564ccee7ff506429a70d5593e.png)

作为兴奋的粉丝，任何光环 3 的消息对大多数 360 所有者来说都是好消息。似乎一个调查光环 3 测试版玩家深入挖掘了游戏使用的文件，发现了一点埋藏的宝藏。看到这篇文章的图片了吗？以下字符串是在 Halo 3 测试版的一些代码中发现的。他们展示了名为眼镜蛇、忍者、入侵者和调节器的新盔甲，当你作为斯巴达玩家时，所有这些都可以多人使用。

这还不是全部。更多的挖掘发现了两个精英球员的新设置。你可以选择默认，猛禽，或捕食者作为你的装甲类型。不幸的是，没有截图可用，但预计一些很快就会浮出水面。Bungie 正式发布一些新的游戏中的新装甲类型的截图只是时间问题。
