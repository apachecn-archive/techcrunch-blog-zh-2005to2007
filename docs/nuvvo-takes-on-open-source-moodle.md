# Nuvvo 挑战开源 Moodle

随着越来越多的教育在网上进行，在线学习管理系统在未来几年将会大规模增长。几天前，Nuvvo 在这个市场推出了一项有趣的服务。

Nuvvo 是免费的，它允许教育工作者用一个很棒的 Ajax 界面创建课程(观看[之旅](https://web.archive.org/web/20221203080818/http://www.nuvvo.com/tour)让我意识到 Ajax 在这样的应用程序中是多么有用)。还有一些模块可以帮助学生对课程收费(如果教育工作者选择收费)，创建测试，评分等。艰苦的工作是创建内容，而不是弄清楚如何使用 Nuvvo。

此处列出了当前的课程[。Nuvvo 通过广告和收取教育者设定的任何课程费用的 8%来赚钱。](https://web.archive.org/web/20221203080818/http://market.nuvvo.com/learn/)

自 2000 年以来， [Moodle](https://web.archive.org/web/20221203080818/http://moodle.org/) 一直是这一领域流行的开源选择。然而，Moodle 需要安装服务器，而 Nuvvo 是一个 web 应用程序。Moodle 通常也需要更多的技术专长，而 Nuvvo 则需要更多的手工操作。Nuvvo 的一名高管准备了一份两种解决方案的[对比图](https://web.archive.org/web/20221203080818/http://www.nuvvo.com/moodle)以及其他信息。
