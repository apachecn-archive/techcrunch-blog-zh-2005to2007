# 使用谷歌接管自动取款机比听起来容易

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/21/using-google-to-take-over-atms-is-easier-than-it-sounds/>

我们从来不提倡出去从自动取款机里偷钱(编辑:但我对此没有意见。–Blake)，但我们要警告 ATM 制造商，有了谷歌和一点耐心和独创性，这是相当容易做到的，尤其是如果机器是相当常见的 Tanax 迷你银行 1500。

你可能还记得上周弗吉尼亚州弗吉尼亚海滩的男子的[故事，他设法重新编程当地一家加油站的自动提款机，将 5 美元换成 20 美元，在其他顾客注意到之前，他带着一大笔钱逃走了。你可能会认为这个人是 l337 HaXX0r，但你错了。](https://web.archive.org/web/20161107064832/http://www.boston.com/news/odd/articles/2006/09/13/atm_reprogrammed_to_deliver_more_cash/)

事实证明，找到几种自动取款机的使用手册并不十分困难，包括加油站的那种。手册通常包含主密码，用它你可以或多或少地让机器按照你的意愿工作。

现在，在你打开谷歌之前，请注意，篡改自动取款机是一项严重的重罪，而且坐牢并不愉快(编辑:马特*会不会*知道)。记住这一点，让你的搜索基于单词“kristanna”和“loken ”,我们都会没事的。
