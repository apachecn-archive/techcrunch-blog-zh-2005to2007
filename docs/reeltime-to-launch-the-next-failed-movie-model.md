# 再次发布下一个失败的电影模型——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2006/11/25/reeltime-to-launch-the-next-failed-movie-model/>

总部位于澳大利亚的[](https://web.archive.org/web/20210925034046/http://www.reeltime.tv/)[reel time . TV](https://web.archive.org/web/20210925034046/http://www.reeltime.tv/)(不要与总部位于西雅图的[ReelTime.com](https://web.archive.org/web/20210925034046/http://www.reeltime.com/))将于本周推出一项新的电影下载服务，名为“实时 DTO”(download-to-own)。这项服务只对澳大利亚人开放，被称为第一个允许 DVD 刻录的合法下载服务(这并不准确，见下文)。今天早些时候，詹姆斯·班南发表了对 ReelTime 高管约翰·卡兰茨的采访。这项服务目前只能在 Windows 电脑上运行，将提供至少一家主要电影公司的电影——环球影业。购买的电影将作为三个独立的文件交付。一个用于 PC 观看，一个用于便携式设备观看(使用 Plays4Sure DRM)，一个用于刻录到 DVD。最后一个文件可以在三十天内刻录到 DVD 三次。之后就没用了。新版本的定价为 33.99 澳元(约合 26.50 美元)。

在美国，MovieLink 和 CinemaNow 都在试验 DVD 刻录功能，并且都与环球影业达成了协议。查看我们的报道[此处](https://web.archive.org/web/20210925034046/http://www.beta.techcrunch.com/2006/10/15/itunes-movies-v-the-rest/)和[此处](https://web.archive.org/web/20210925034046/http://www.beta.techcrunch.com/2006/10/24/aol-video-adds-paramount-product-still-lags-market/)了解更多关于这些和其他电影下载服务的信息。

ReelTime 的定价高得惊人。然而，抛开这些不谈，这项服务仍然存在问题。用户希望能够通过 DVD 在多种设备上消费内容，包括移动设备和普通电视。ReelTime 通过提供三个不同的文件来解决这个问题，每个文件都有不同的 DRM 方案(但是这些文件不能在 iPod 上播放)。ReelTime 试图满足这一需求的事实值得称赞。但是交付三个具有不同限制的文件并不是正确的解决方案。下载量太大，管理这些文件对用户来说是一笔很大的开销。

很明显，工作室正试图通过与几乎每一家初创公司(见本文中的[图表)达成交易来避免唱片公司](https://web.archive.org/web/20210925034046/http://www.beta.techcrunch.com/2006/10/24/aol-video-adds-paramount-product-still-lags-market/)的[命运，他们梦想着从数十亿美元的 DVD 和电影租赁市场中分一杯羹——只要价格足够高，让主要的线下 DVD 零售商满意，并且文件完全被 DRM 锁定。](https://web.archive.org/web/20210925034046/http://www.beta.techcrunch.com/2006/11/19/replacing-drm-with-a-music-tax-is-incredibly-stupid/)

但是有太多的服务在追逐对价格不敏感的消费者的非常小的市场，这些消费者愿意忍受 DRM 和对内容可移植性的限制。利润微薄。苹果和一两家其他公司可能会获得足够的市场份额来生存。其余的都在追逐一个[白日梦](https://web.archive.org/web/20210925034046/http://en.wikipedia.org/wiki/Pipe_dream)，他们的投资者也是如此。只要 BitTorrent 被用户广泛使用，笨重的替代品就没有机会。
