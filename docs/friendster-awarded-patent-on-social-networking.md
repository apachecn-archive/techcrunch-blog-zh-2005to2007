# Friendster 获得社交网络专利

 [](https://web.archive.org/web/20221209071920/http://www.friendster.com/) 萎靡不振的社交网络 [Friendster](https://web.archive.org/web/20221209071920/http://www.friendster.com/) 上个月末被授予[一项关于在线社交网络一些非常基本的特性的专利](https://web.archive.org/web/20221209071920/http://patft.uspto.gov/netacgi/nph-Parser?Sect1=PTO2&Sect2=HITOFF&p=1&u=%2Fnetahtml%2FPTO%2Fsearch-bool.html&r=1&f=G&l=50&co1=AND&d=PTXT&s1=7,069,308.PN.&OS=PN/7,069,308&RS=PN/7,069,308)。该专利涵盖了确定和显示已将个人信息输入社交网络的个人之间的关系；具体来说，确定谁在你的朋友圈里，谁不在。这项专利申请是在三年前提交的——这是美国专利商标局落伍的一个很好的例子。当 USPTO 试图寻找能够了解现有实践、评估非显而易见性等的审查员和基础设施时，许多领域的新技术提出了一个真正的问题。

Friendster 有许多知名投资者，今年 2 月，凯鹏华盈(Kleiner Perkins)也追加了投资。CNN 的[浏览器](https://web.archive.org/web/20221209071920/http://money.cnn.com/blogs/browser/2006/07/friendster-wins-patent.html)博客今天写道，“起诉竞争对手侵犯专利不是交朋友的方法——但却是赚钱的方法。”当被问及是否会有许可证或诉讼时，Friendster 总裁肯特·林德斯特罗姆告诉 RedHerring.com，“现在说这个还为时过早……我们会尽全力保护我们的知识产权。”

如果这项专利存活下来，并被认为与现在网络世界的一个巨大部门相关，我预计要求专利改革的声音会越来越大。然而，很难想象，如果这项专利及时获得批准，会发生什么。支持强有力的专利制度的人通常认为，除非创新获得专利保护，否则没有人会投资创新。显然，这个论点在这种情况下站不住脚。
