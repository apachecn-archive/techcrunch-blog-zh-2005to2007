# 三洋新推出的 42 英寸液晶显示器可以抵御恶劣天气

三洋 CE42LM4WPR 是一款 42 英寸的液晶显示器，它与同类产品至少有一个主要区别:它是防风雨的。没错，CE42LM4WPR 拥有 IP56 耐候认证，这意味着它可以承受各种典型电子产品难以忍受的户外危险。它不仅能防潮，还能应对“强水流”的冲击，所以你可以用水管之类的东西把它喷掉。

800:1 对比度的 WXGA 屏幕具有无眩光丙烯酸前面板，分辨率为 1366 x 768，视角为 178 度。它包括 HDMI 输入、三组 AV 输入和一个 RS232C 接口。不知道它的价格，但预计会很贵。
