# 雅虎将最终升级 MyBlogLog

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/05/13/yahoo-to-finally-upgrade-mybloglog/>

[](https://web.archive.org/web/20160307101852/http://www.mybloglog.com/)[【MyBlogLog】](https://web.archive.org/web/20160307101852/http://www.techcrunch.com/2006/10/19/mybloglog-readers-network-around-their-favorite-blogs/)，一个无处不在的显示网站最近访客照片的博客小工具，是 2006 年“立竿见影”的成功故事之一——[雅虎在大多数人还没有机会听说之前就收购了这家公司](https://web.archive.org/web/20160307101852/http://www.techcrunch.com/2007/01/08/yahoo-buys-mybloglog-no-they-didnt-wait-yes/)。

像许多博客一样，我们在 TechCrunch 上安装 MyBlogLog 小工具已经有几个月了。我们最终删除了它，原因是性能问题(它在一些情况下降低了网站的速度)和开始出现的数量惊人的[垃圾邮件。](https://web.archive.org/web/20160307101852/http://www.techcrunch.com/2007/01/10/mybloglog-got-spammed-and-so-did-we/)

大卫·达尔卡说,其中一些可能正在改变。昨天在芝加哥的一次会议上，MyBlogLog 社区经理 Robyn Tippins 告诉观众，可以期待几个即将到来的服务变化。

用户应该期待该服务的品牌重塑和重新设计，以及一个新的具有悬停功能的博客小工具。他们还将寻找一种方法来轻松关闭在线状态，这样用户就可以隐藏他们访问过某个网站的事实。

重新设计会很好。更好的是承诺 100%的正常运行时间。作为一家 widget 公司，放慢或关闭合作伙伴网站是不可接受的。MyBlogLog 的第一要务一定是性能。
