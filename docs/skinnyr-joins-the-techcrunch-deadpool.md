# Skinnyr 加入了 TechCrunch 死亡池

[![skinnyr.jpg](img/a63a74ce0e8c073f0f38cb03c3c5a19d.png) ](https://web.archive.org/web/20210928083121/http://www.skinnyr.com/) [与](https://web.archive.org/web/20210928083121/http://www.skinnyr.com/) [Fatsecret](https://web.archive.org/web/20210928083121/http://www.beta.techcrunch.com/tag/fatsecret) 在同一个市场竞争的减肥追踪和建议网站 Skinnyr 上市了。

所有者詹姆斯·托马斯告诉 TechCrunch，他正在出售网站，这样他就可以专注于他的社交歌曲创作社区。

据托马斯称，Skinnyr 拥有 1700 名用户。收入为 0，因为该服务从未货币化。托马斯为这块地要价大约 65，000 美元。

Skinnyr 通过加入 [TechCrunch Deadpool](https://web.archive.org/web/20210928083121/http://www.beta.techcrunch.com/tag/deadpool) 进行终极减肥治疗。
