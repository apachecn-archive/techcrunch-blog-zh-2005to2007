# 美通社和翁布里亚联手追踪博客

![](img/3a08d85c31a48b3dff6a02d2bb63a808.png)

![prnewswire.jpg](img/4508bdda1785b8b00500f837e036736d.png) [美通社](https://web.archive.org/web/20220816014929/http://www.prnewswire.com/)宣布与 [Umbria](https://web.archive.org/web/20220816014929/http://www.umbrialistens.com/) 合作，这是一家市场情报公司，专门从事博客研究和消费者生成媒体的新产品“MediaSense 博客测量”，允许客户评估关于他们公司、产品和品牌的在线对话的影响。

这项新服务建立在美通社与 Technorati 现有关系的基础上，并提供对话语气和参与者人口统计的图形化评估。

翁布里亚公司总裁霍华德·考尚斯基在新闻发布会上说:“博客作者是思想领袖和舆论塑造者，因此跟踪谁以及他们以多快的速度关注各种新闻对品牌和公关专业人士来说至关重要。”“如果是阳性，你想知道。如果是消极的蘑菇云，你肯定需要知道。这种合作关系让美通社的客户了解了这些知识。

虽然跟踪消费者生成媒体的公司的持续增长是 Web 2.0 最重要的驱动因素之一的持续成熟和接受的积极迹象，但必须问一个问题:为什么？

为什么公关专业人士需要一项服务来通过第三方了解博客作者对其客户的评价？

媒体监督服务仍然在支持公共关系方面发挥着重要作用，但这种老派模式来自互联网出现之前的时代，在互联网时代，通过第三方进行全国性媒体监督是必不可少的，原因很简单，因为没有替代方案，而且在许多情况下，对于平面媒体、广播和电视来说，今天没有一个包罗万象的替代方案。然而，博客和消费者生成的媒体是新时代的产物，一个在世界任何地方只需轻触按钮就能在线获取信息的在线时代。

许多公关专业人士联系并阅读 TechCrunch，因此我们或许可以得到一些答案:是不是有些公关专业人士不能在 Technorati 或谷歌博客搜索中键入“在此插入客户姓名”？

从谷歌新闻(Google News)、雅虎新闻(Yahoo News)和 Topix 等提供基于企业品牌名称的搜索结果的服务建立订阅源有多难？

参与 Web 2.0 世界的全部意义不就是消除信息传播的中间层，进行一对一的交流吗？

话虽如此，如果你有一个完整的公司费用账户，并喜欢你的信息填鸭式，那么这些博客跟踪服务是理想的。
