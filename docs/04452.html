寡妇租用旋转电话42年

原文：https://web.archive.org/web/http://techcrunch.com/2006/09/18/widow-leased-rotary-phone-for-42-years/

20世纪60年代，俄亥俄州坎顿市的一位寡妇Ester Strogen首次租赁了两部黑色旋转式电话。虽然大多数主要运营商在1985年就通知他们的客户不必再租赁，但Strogen肯定没有注意到这一点，电话公司——从这位可怜无害的女士那里每月获得29.10美元——也不想向她提及此事。

Strogen的家人本月最终取消了她的租约，并计算出她多年来为过时的手机支付了1.4万美元。显然仍有75万人在租房。我们都去奶奶家看看，确保她不再租房子了，各位。这是我们最起码能做的。
