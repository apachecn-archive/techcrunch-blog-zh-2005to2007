# 诺基亚 6110 向下导航很长一段路

![longwaydown.jpg](img/3a330085b579c39129c5fe0b6d5e253f.png)当失败的共产主义革命者和 t 恤偶像[Ernesto“Che”Guevara](https://web.archive.org/web/20130628192623/http://en.wikipedia.org/wiki/Che_Guevara)骑着摩托车环游南美时，他和他最好的朋友都没有任何高科技手机帮忙带路。也许如果车有诺基亚 6110 导航仪，他就不会试图把“社会主义正义”带到南美各国。好吧，这个世界永远不会知道，但是我们知道这部手机在伊万·麦格雷戈(*《猜火车》*、*《星球大战》*)和查理·布尔曼从苏格兰到非洲最南端的漫长旅途中帮助了他们。

这两个人都骑着宝马摩托车旅行，他们用诺基亚手机来确定路线，GPS 导航，网络搜索(“在非洲中部我们在哪里可以找到好吃的印度食物”可能是两个英国人的常见搜索)，当然还有回家的重要电话。这次旅行是他们最初的 *Long Way Round* 系列的后续，这是一个受欢迎的基于现实的系列，记录了他们一路上的冒险和不幸。与车不同的是，这两个最好的朋友知道一点资本主义技术，就像一部带有 200 万像素摄像头的手机要走很长的路。更重要的是，麦格雷戈就像一个 t 恤图标一样大，他的手不会在某个博物馆里证明他已经死了！
