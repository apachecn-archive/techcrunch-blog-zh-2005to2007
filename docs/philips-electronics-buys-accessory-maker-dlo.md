# 飞利浦电子收购配件制造商 DLO

[![wordmark_hq.jpg](img/62457b0b5551ce17d95c56c25e3fbf87.png) ](https://web.archive.org/web/20201026020409/https://beta.techcrunch.com/wp-content/uploads/2007/04/wordmark_hq.jpg "wordmark_hq.jpg") [皇家飞利浦电子](https://web.archive.org/web/20201026020409/http://www.usa.philips.com/index.html)今天宣布将收购总部位于美国的[数码生活用品公司](https://web.archive.org/web/20201026020409/http://dlo.com/index.tpl?command=showpage&cart=1176483606992252) (DLO)，该公司为苹果公司的 iPods 设计、营销和分销大量配件。这包括扩展坞、调频发射机、电缆和外壳。

该交易预计将于 2007 年第二季度完成，届时 DLO 将成为飞利浦消费电子部门“外围设备和配件”业务部门的一部分。目前还不知道这笔交易给飞利浦带来了什么财务或其他方面的损失。

虽然 DLO 确实为 Zune 和其他设备制造一些配件，但它的大部分产品都是用于 iPods 的。因此，我认为这意味着飞利浦将让 DLO 为该公司的 GoGear MP3 播放器制作配件，这可能真的有助于飞利浦在美国向那些通常会选择 iPod 作为配件的人销售一些产品。或者，也许 DLO 会继续使用大量的 iPod。
