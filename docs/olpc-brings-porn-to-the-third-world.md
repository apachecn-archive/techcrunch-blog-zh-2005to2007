# OLPC 把色情带到了第三世界

“每个孩子一台笔记本电脑”计划通过向第三世界的儿童提供网络色情达到了新的高度。

据尼日利亚官方通讯社周四报道，阿布加一所小学的笔记本电脑“出现了问题，因为学生们随意浏览含有露骨性内容的成人网站。”

得知 OLPC 项目背后的好心人的努力正在给当地带来实实在在的结果，令人感到温暖；为十几岁的男孩提供同样的机会接触网络色情，不管他们有多穷，也不管他们住在哪里。

然而，这个聚会将是短暂的；据报道，“每个孩子一台笔记本电脑”组织的一名代表称，OLPC 的电脑现在将安装色情过滤器。
