# SanDisk 向 Wii 展示一些 SD 爱

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/17/sandisk-shows-wii-some-sd-love/>

如果你的新任天堂 [Wii](https://web.archive.org/web/20150915111953/http://crunchgear.com/2006/11/15/next-gen-console-review-the-good-bad-and-the-ug-wii/) 不能使用普通的旧 SD 卡，闪存制造商 SanDisk 将推出特别版卡以配合游戏机的发布。

那么有什么特别的呢？首先，卡片是白色的(是的！是真的！一张白色 SD 卡！)并将印有“SanDisk For Wii”字样。可悲的是，这就是特殊性的终结。它们将有 512GB、1GB 和 2GB 三种尺寸，价格从 34.99 美元到 89.99 美元不等。

顺便说一下，这些卡可以用于存储游戏保存，备份 Wii 商店频道中的虚拟主机游戏，可以存储 JPEG 图像和 MP3 音频，以便通过 Wii 播放和收听，并可以接受 QuickTime 中创建的最大尺寸为 848 x 480 像素的视频文件。这些卡片还没有发布到该公司的网站上，但据说它们将与 Wii 一起出现在游戏和主要的消费电子产品商店中。
