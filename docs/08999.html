索尼VAIO VGX-TP1DT Roomba电脑

当比格斯和我在CES的大厅里漫步时，我们看到了这款外形奇特的索尼个人电脑，它正在进行某种网络直播。我们前往索尼展台试图找到它，但没有成功。

我们对摊位猴子的询问只得到困惑的表情。心烦意乱，我们回到了新闻发布室，在那里我们撕扯了文斯大约一个小时，直到是时候离开。

然而，我们的好奇心仍然被激起。所以感谢Engadget挖出了这只小动物。
T3】

原来它实际上是VAIO的两个新产品，VGX-TP1和VGX-DT1，它们链接在一起形成了巧妙地命名为VGX-TP1DT(又名Roomba PC)。TP1(顶部)圆柱是一台Vista家庭高级电脑，配有1.66GHz Core 2 Duo CPU。它具有1GB的内存，可扩展至2GB，160GB的硬盘可升级至500GB。该设备还具有一个带有HDMI、模拟RGB和光学数字输出的地面调谐器。然后是所有的标准噪音，如USB火线，记忆棒和SD插槽，以及WiFi。

DT1(底部)圆柱体实际上是一个数字调谐器，能够通过以太网和WiFi传输高清内容。然后，您的TP1设备可以以1920 x 1080/24Mbps或1440 x 1080/10Mbps的速率在1080i中录制这些内容。

它的面积只有10.6英寸，这使得它非常适合家庭影院环境。如果你感兴趣，这一对应该很快就会在日本上市，售价约为1，660美元。
