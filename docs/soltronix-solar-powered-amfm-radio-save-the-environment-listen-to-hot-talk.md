# Soltronix 太阳能 AM/FM 收音机:拯救环境，听热门话题

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/08/soltronix-solar-powered-amfm-radio-save-the-environment-listen-to-hot-talk/>

[![solarpowerhed.jpg](img/3f5dfbb4282f8bd41ac2341f16f4fbbb.png)](https://web.archive.org/web/20150923000558/http://old.crunchgear.com/wp-content/uploads/solarpowerhed.jpg "solarpowerhed.jpg")

从外表来看，继现在风靡一时的微型狗时尚之后，绿色的将成为下一个 SoHo 时尚潮流。赶上潮流的一个方法是使用 Soltronix 耳机 AM/FM 收音机。这是一副太阳能耳机，它的太阳能电池足够灵敏，只需 30%的光照就能运行。它的工作原理有点像那些混合动力汽车:太阳能电池产生的多余电力没有用来给耳机供电，而是用来给电池充电。所以不管有多少云层，你都可以利用它们。这个速度似乎有点慢，一个小时的太阳能充电需要 1-2 个小时的使用。

我不知道，但从图片来看，这些耳机看起来不会听起来那么热，特别是因为我已经被其他人宠坏了，[更高质量的耳机](https://web.archive.org/web/20150923000558/http://crunchgear.com/2006/10/27/battle-test-ultimate-ears-triplefi-10-pro/)。不过，慢跑时听听汉尼提的音乐可能是值得的。
