# 三星 i620 滑盖智能手机

![samsung-i620-slider-smartphone.jpg](img/5cd96c7d14c4b451767c65a9280ee4fa.png)

坦率地说，我对不断公布的所有新手机设计感到厌倦。我应付不了。我想要所有的，但是我没有钱或者方法去得到那些永远不会在我们的海岸上出现的。三星最新的滑块可能会说服我，但它可能运行在 WinMo 上，我不会去那里。然而，QWERTY slider 附带了一系列诱人的功能，如 WiFi、200 万像素摄像头、VGA 摄像头、蓝牙和 microSD 插槽。其他大部分功能都很无聊，但这看起来肯定比我讨厌的 21 点要好。
