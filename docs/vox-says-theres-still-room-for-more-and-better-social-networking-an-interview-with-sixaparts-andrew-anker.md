# 更多更好的社交网络仍有发展空间:采访 SixApart 的 Andrew Anker-

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2006/10/31/vox-says-theres-still-room-for-more-and-better-social-networking-an-interview-with-sixaparts-andrew-anker/>

上周，社交软件公司 SixApart 发布了其最新的社交网络和博客服务 [Vox](https://web.archive.org/web/20220818220302/http://vox.com/) 。发布会很高调，网站很漂亮，很多人([我自己也包括](https://web.archive.org/web/20220818220302/http://www.beta.techcrunch.com/2006/10/26/vox-lifts-off-and-youll-love-it/))认为 Vox 给市场带来了一些重要的东西。不是每个人都同意。

在接下来的 QandA 中，我询问了 SixApart 公司发展执行副总裁 Andrew Anker，关于 Vox 发布时的一些最大的批评。在 SixApart 工作之前，Anker 是 Wired Digital 的联合创始人兼首席执行官，也是 August Capital 的普通合伙人。我认为这些问题和答案会引起初创公司和一般社交软件从业者的普遍兴趣。

Anker 告诉我，他认为社交网络市场远未饱和，即使人们不使用它，保护隐私也是有市场的。他说，软件可以功能丰富，非技术用户也可以使用；在线广告的世界才刚刚开始超越“打猴子”式的广告，还有更多发展空间。

以下是 Anker 的回复，我会让你自己判断是否有说服力。下面的图片来自该网站 150 多个布局模板中的几个。

**马歇尔:**
有些人想知道 Vox 是否只是另一个社交网络——社交网络市场有多接近饱和？

安德鲁:
我们不认为它接近饱和，我认为这主要是由当前所有服务都增长良好的事实证明的。我们已经远离了社交媒体的零和市场份额阶段，在这个阶段，一家公司的新客户是其他人的损失。打个比方，我们认为我们正处于与有线电视在 20 世纪 80 年代中期相似的阶段。出现了一些突破性的成功，如 MySpace 和脸书，但我们距离拥有涵盖所有不同细分市场和客户使用案例的多样化属性还有很长的路要走。

对 Vox 来说，更重要的是，我们认为很少有产品能够满足更老的非“挂钩”市场的需求。MySpace 很棒，但它不是与你最亲近的朋友和家人分享孩子照片的最佳地方。

**马歇尔:**
有什么证据表明对以隐私为中心的个人表达工具有强烈的需求？

最简单的例子就是我们自己的 Live Journal，它拥有 1100 万注册用户，但 Flickr 在这个已经拥挤不堪的照片分享空间的惊人增长也证明了人们对隐私工具的需求。具体来说，我们认为比隐私本身更重要的是控制…让用户能够决定谁可以看到什么。事实是，Vox 上的大多数帖子和照片(就像 Live Journal 上的一样)都不是私人的。但我们的客户需要知道，如果合适的话，他们可以制作一些私密的东西。先创建，后决定谁可以查看。

脸书几个月前遇到的隐私问题都与控制有关。正如很多文章所写的那样——例如参见 [danah boyd](https://web.archive.org/web/20220818220302/http://www.danah.org/papers/FacebookAndPrivacy.html) 。脸书在做出这些改变时没有透露任何新的数据。但它们让用户感觉不到他们控制着自己的数据，这是用户抗议的根本原因。

Web 2.0 理念的基本原则之一是用户控制他们自己的数据。就我们而言，这既适用于在服务之间移动数据的能力，也适用于决定谁可以查看数据的能力。

**马歇尔:**
为什么读者要登录 Vox 才能发表评论？

Andrew:
这是一个暂时的限制，随着时间的推移会得到解决。我们在开发网站时仍处于快速迭代模式，这只是我们在发布前没有实现的功能之一。为了向所有人开放评论，我们需要比目前更好的垃圾邮件审核工具。一旦我们有了这些，我们将开放该网站向公众征求意见。明确地说，就像 Voxers 可以在 Vox 中控制谁可以发表评论一样(任何人、朋友、家人)，这个
也是用户可以设置的。

**Marshall:**
Vox 团队是如何为非极客提供功能丰富且易于使用的产品的？关于这些问题，未来会是什么样子？

构建良好的可用性就是要做好成千上万的小事，平衡非极客也能使用的丰富功能是这个过程的一部分。在 Vox 开发的早期，我们创建了一个两周的快速迭代周期，确保每两周严格地推送一次代码。通过这样做，我们确保了我们正在构建一个设计周期，总是在两周之内解决任何问题。

我们有遍布整个网站的 Hitbox，一个内部可用性实验室，一个阅读所有网站反馈的强大支持团队和一群热情的开发者，他们也是 Vox 的忠实用户。我们一直在
听取我们在可用性方面看到的隐性和显性数据，并不断调整网站，使其更加友好。

**Marshall:**
Vox 上的广告明显减少了，这对用户来说很好。然而，当 LiveJournal 最近推出赞助社区和功能时，一些人感到受到了伤害。管理页面上的广告、适度的面向公众的广告和来自合作伙伴网站的代销商收入是否有潜力充分转化，或者 Vox 是否会在几个月后崩溃并添加闪烁的横幅广告？

安德鲁:
我们当然在努力平衡 Vox 用户和广告商的需求，我不相信我们所追求的受众会对闪烁的横幅广告和“打猴子”式的信息反应良好。显然，我们是一家企业，需要赚钱。但我们并不认为赚钱和做好广告是两个相互排斥的概念。

正如上周宣布的那样，我们正与广告合作伙伴密切合作，以确保我们做得正确。早在社交媒体出现之前，我们认为社交媒体上的广告甚至更早。我们非常有信心，我们可以帮助它发展成为最有效的媒体之一。
