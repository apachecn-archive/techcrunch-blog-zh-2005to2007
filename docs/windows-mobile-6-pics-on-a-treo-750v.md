# Treo 750v 上的 Windows Mobile 6 图片

随着 Windows Mobile 6 ( [又名“Crossbow”](https://web.archive.org/web/20201129142323/http://crunchgear.com/?s=crossbow))在移动市场上的发展，我的 Windows 智能手机用户的问题是“它能在我的手机上运行吗？”答案越来越像是“是”，正如这些在 Treo 750v 上运行的 WinMo 6/Crossbow 的新照片所展示的那样。该操作系统已经分发给手机制造商和服务提供商，但我们还没有在市场上看到任何官方授权的手机或升级版。

不过，好消息是，这是因为运营商正在对特定设备的更新进行收尾工作。我们已经得到官方确认，T-Mobile Dash 将是第一批升级的手机之一，升级将对用户免费。还不错，真的。如果其他航空公司效仿，那么 750 上的弩，如图所示，将会非常普遍。
