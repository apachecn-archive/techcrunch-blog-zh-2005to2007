# 更新–浏览器(适用于 Firefox)

## 有什么新鲜事？

[浏览器](https://web.archive.org/web/20201201113738/https://crunchbase.com/organization/browster) 今天宣布了火狐浏览器 1.0[。查看我们的](https://web.archive.org/web/20201201113738/http://www.browster.com/pr_080205.htm)[之前的简介](https://web.archive.org/web/20201201113738/http://www.beta.techcrunch.com/?p=85)了解更多关于 Browster 的信息。

![](img/234e901ade7a2f5e9e4cb858784a9392.png)来自他们的[新闻稿](https://web.archive.org/web/20201201113738/http://www.browster.com/pr_080205.htm) : *“加利福尼亚州旧金山，2005 年 8 月 2 日——Browster，Inc .今天宣布其旗舰产品 Browster 1.0
作为 Mozilla Firefox 浏览器的扩展上市。Browster 是 Firefox 的第一个即时页面查看产品，为用户提供了最快的方式来查看搜索结果，只需将鼠标放在一个链接上。借助 Browster 独特的下一页功能，Firefox 用户可以零延迟地从一个结果跳到另一个结果。Browster 用户体验了一种更快、更容易和更有趣的方式来浏览网页。浏览器 1.0 火狐浏览器现在可以从 www.browster.com 免费下载。*

Browster 预取搜索结果中列出的网页，并包括第一个即时页面呈现功能。然后，用户可以在浏览器窗口中查看该页面或整个网站。当用户浏览他们的搜索链接时，Browster 正在申请专利的 LookAhead 技术会在用户将鼠标滚动到链接上时显示预览窗口。浏览器是一个革命性的，安全的，快速预览网页的方式。"

我们很高兴看到这个版本。Browster 现在将被永久安装和使用。伟大的应用。
