# Vertu 星座评论| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/03/vertu-constellation-reviewed/>

Sybarites 的花里胡哨的家伙们得到了一份 Vertu 星座评论。如果你刚刚加入我们，Vertu 是一个奢侈的手机系列，由制造诺基亚手机的同一批人制造……诺基亚。那么这款手机如何堆叠呢？

嗯，诺基亚 S40 的用户界面与其他诺基亚塞班操作系统手机相似，但“在图形方面更胜一筹”这款手机有一个礼宾按钮，可以呼叫 Vertu 的礼宾服务，有点像你手机的 OnStar。它还拥有蓝牙、GPRS 和 EDGE，这使得它在功能方面相当新，而不仅仅是一部包装在漂亮外壳中的旧手机。

作为奖励，它还拥有 WorldMate 软件来“跟踪航班、不同时区和汇率”，这对任何买得起 Vertu 的飞机乘客都很有用。
