# Vivee 为聊天电子邮件增添了新的内容

正如我们本周早些时候报道的，开车时发短信几乎和酒后驾车一样糟糕。我们也听到了很多解决这个问题的方法。也许等到到达你要发送或阅读电子邮件的地方才是最好的选择，但是我们又有什么资格告诉人们要多一点耐心呢！再说，反正没人会听我们的。他们忙着打电话或看短信！事实上，一些数字令人担忧。全国保险公司最近的一项调查发现，五分之一的人开车时发短信，在 18-34 岁的人群中，这个数字上升到三分之一！

但是我们这周也听说了一个解决方案，叫 Vivee，或者语音交互语音增强电子邮件。这是 USTelematics，Inc .推出的一项新服务，它会给你读电子邮件或短信。一个动画角色出现在屏幕上，并向你朗读收到的信息。

USTelematics 公司的产品经理 Charlene Montalbano 说:“安全应该是我们开车时首要关注的问题，保险研究表明，分心的司机造成了 80%的交通事故，因此一些州已经制定了法律，禁止在开车时使用手机。现在，立法者正在变得更加精确和有针对性，通过考虑专门禁止驾驶时发短信和阅读电子邮件的法律来消除司机分心。Vivee 不仅会让你在路上安全地集中注意力，最终，“她”可能会帮你避免被开罚单。”

Vivee 通过威瑞森无线 EVDO 高速网络连接到互联网，当你收到电子邮件或短信时，Vivee 会提醒你，以便你可以收听你的信息。为什么它需要会说话的化身，我们不能说。这个想法不是让你停止看设备吗？不过，任何让人们避免开车时发短信的产品或解决方案都是一件好事……让我们只希望用户不会觉得有必要看他们的 Vivee 头像，然后觉得有必要做出回应。
