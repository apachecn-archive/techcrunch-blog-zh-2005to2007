# MySpace 准备推出新闻服务，包括移动更新

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/19/myspace-poised-to-launch-news-service-includes-mobile-updates/>

MySpace 推出了一项新闻服务，使用了一种非常类似 Digg 的投票方式。虽然它适用于像 Digg 这样的网站，但我们有一种感觉，news.myspace.com 的头版将只不过是报道“OMG Staci 是这样一个笨蛋！！！1!"

现在是测试版(像互联网的其他部分一样)，但是报道称它应该在今天的某个时候发布。真正的问题是人们会使用它吗？我们不这么认为。当然，它属于一家名为新闻集团的公司，但它也是由拥有第一套绒毛般柔软的阴毛的人管理和居住的。

Helio 或 Cingular 上的用户可以注册手机新闻提醒，这意味着下次布兰妮做傻事时，你将是第一个知道的*。你也可以在你的 MySpace 页面上添加订阅源或主题。我猜很酷。有趣的是这项倡议:这是否意味着 MySpace 试图打造自己的网络替代品？我们认为是的。*
