没有计划吗？通过活动结识新朋友——TechCrunch

原文：https://web.archive.org/web/http://www.techcrunch.com:80/2006/10/23/no-plans-meet-new-people-via-activities

这周没有计划吗？或者，你有没有发现自己多了一张音乐会或体育赛事的门票，却没有人陪你一起去？过去，你的解决方案是费力地浏览Craigslist上的帖子，或者打电话给你所有的朋友。现在有两种基于网络的服务可以让你找到一起做事的人——谁要去和 MatchActivity 。

谁去是我印象最深的一个，虽然它缺少用户和活动(他们最近刚软推出)。该网站是简单的，非常深思熟虑的。这种商业模式是这样运作的:发布活动是免费的，然后“想去”参加活动的人，如果被选中，就支付一定的费用(在撰写本文时为1美元)——如果你没有被选中，你就可以拿回你的钱。因此，如果我有一张多余的音乐会门票，我可以(免费)张贴我有一张门票，并询问谁想去——我可能会得到4个响应者(每个人拿出1美元给谁)。然后我浏览他们的个人资料，选择我想和谁一起去——那个人的1美元归谁，其他3个人拿回他们的1美元。注:您的朋友可以免费申请/参加。该网站具有社交网络功能，允许您浏览个人资料、添加朋友、给人发消息、将人添加为收藏夹(以了解他们发布的任何活动)、向您的个人资料发布照片(如果他们是朋友，人们可以对他们发表评论)，以及向实际活动发布照片/评论。他们还提供了一些小部件，你可以将它们嵌入到你的博客或MySpace个人资料中，以推广某个特定事件或你的所有事件。

关于这个网站没有太多要说的，我爱它。它只需要用户和活动。我鼓励你发布一个活动，并在标题中使用“TC ”,这样你就可以与你所在地区的一些TechCrunch读者联系起来。所以继续前进，找到一个TC活动或发布一个活动——一场扑克游戏，或欢乐时光聚会，或钓鱼。《谁要去》由约翰·威尔创建、编码和设计。他也是(现在停滞不前的)Tagalag 的开发者，我们去年对其进行了简介。

match activity有一个约会的旋转，专注于为单身者寻找活动伙伴(但任何人都可以响应任何活动，无论他们是寻求男性还是女性)。通过成为高级会员(每月7.99美元，但在撰写本文时是免费的——每个人都是高级会员)，你可以实现3件事:1)你可以在你选择的系统中发布只发给好友的私人活动；2)可以向其他会员发送特定的1对1邀请；3)你的可靠性评级增加了3分(没有找到其他信息，但这似乎是不言自明的——人们可能会在实际参加活动时对你的可靠性进行评级)。

我喜欢MatchActivity的一点是，你可以发布一个活动或可用性——这样我就可以说我周四晚上有空，列出一些兴趣，也许可以发布一些想法，看看有人是否想做点什么。我还喜欢它有活动创意——这列出了即将在各个城市举行的活动(在撰写本文时只有3个城市——洛杉矶、纽约、旧金山——每个城市大约有5个创意；但这里有一个自动化系统的潜力，餐馆/酒吧/体育赛事/音乐会可以发布他们自己的活动，并成为一个特色活动的想法)。为了在网站上激发更多的“活动”(没有双关语的意思)，我会整合更多的活动想法，可能来自 Eventful API (正如Who's Going计划做的)。我也喜欢他们的“临时邮政编码”功能，这允许用户指定他们未来将前往的不同城市，从而对该位置的活动开放——这有点类似于谷歌的dodgeball.com服务所具有的功能(你切换你的城市，以便该城市的朋友知道你的位置)。需要澄清的一件事是搜索——我不确定是输入邮政编码还是城市名，它只提供50英里内的列表(没有改变距离的选项)。我也没有浏览过活动帖子，但我希望他们实施了可靠性评级，允许用户对他们与他人的活动体验进行评级——这在“谁去”中是缺乏的，可能会成为这些网站的一个重要功能(很像易贝的评级系统)。“谁去”觉得社交交易比金融交易更难量化(“我认为他是个呆子”)，并希望用户互相评论对方的个人资料，以证明他们之间的关系——例如，“嘿，我玩得很开心！”

很难说这两个网站中的任何一个会成功——它们需要参与。Who's Going提供的小工具可以更快地传播消息。我相信这些网站的简单性和专注性可以让它们腾飞——我绝对觉得有必要这样做。应该注意的是，来自“社交活动网站”的间接竞争非常多。Friendster创始人Jonathan Abram期待已久的 Socializr 是一个潜在的竞争对手(但仍未公开发布)。事件空间中的其他人包括雅虎的Upcoming.org、 Zvents 、750万美元资助的 Eventful 、300万美元资助的 Renkoo 、 Skobee 、 Involver 、以色列的 ILCU 、波斯顿的 HeyLetsGo 、 Weekendr 、 MySpace事件

我们在7月份简要介绍了MatchActivity和其他12个在线约会网站。

约翰·威尔关于谁要去的更多报道:

“谁要去”是为了帮助用户结识新朋友而全新设计的。相比之下，MySpace和Evite活动的目标是你已经认识的朋友。Eventful和Upcoming.org很大程度上是活动列表——你可能找不到一起参加活动的人。

Who's Going的功能更像是Craigslist或者一个交友网站。此外，MatchActivity.com也有类似的模式，专门针对约会市场。

(Whosgoing.com)网站的设计和测试是为了迎合广泛的受众，并足够灵活地满足用户的需求。该项目是从长远角度启动和发展的。
