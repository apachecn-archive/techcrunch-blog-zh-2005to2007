# Socialtext 推出拔插功能

[![](img/a8254954c585a04e7adf944872ac14b1.png) ](https://web.archive.org/web/20211026082723/http://www.socialtext.com/) [Socialtext](https://web.archive.org/web/20211026082723/http://www.socialtext.com/) ，一个企业维基工具，今天在巴黎 [LeWeb3](https://web.archive.org/web/20211026082723/http://leweb3.com/) 大会上发布了 Socialtext 不插电。它是一个拔出图标，让用户即使没有连接到互联网也能在他们的维基上工作。

当仍然在线时，用户可以点击蓝色的拔出图标，这将下载维基页面的选择，以便这些页面可以离线使用。一旦用户重新上线，更改将自动上传。

![socialtextunplug.jpg](img/ebbba6bd8f7f4b5fc680a2e677e05e13.png)“蓝色不插电图标类似于 RSS 图标，它向用户发出信号，表明在浏览器之外有一种不同的方式来使用内容。Socialtext 的首席执行官罗斯·梅菲尔德在他的博客中写道。

Socialtext Unplugged 是一个 HTML 文件中的应用程序。当拔出图标被激活时，它会以 Zip 文件的形式下载页面，尽管重新同步是通过 Socialtext 的 Wiki Web 服务进行的。
