# Vox 升空，你会爱上它

 [](https://web.archive.org/web/20220926094012/http://vox.com/) [六合](https://web.archive.org/web/20220926094012/http://sixapart.com/)昨晚宣布推出其最新社交网站 [Vox](https://web.archive.org/web/20220926094012/http://vox.com/) (Vox 公告[此处](https://web.archive.org/web/20220926094012/http://team.vox.com/library/post/happy-vox-launch-day.html))。拥有 LiveJournal、mobile Type 和 Typepad 的公司在这个新网站上做了很多正确的事情。在推出社交网站之前等待消费者欲望成熟的好处在 Vox 中显而易见。

这项服务在测试期间在一些人中建立了声誉，成为旧金山艺术精英的社交网络——但每个人都需要一个测试小组，这是一个很好的小组。Vox 最初被称为 Comet，我们在这里第一次写到它[。](https://web.archive.org/web/20220926094012/http://www.beta.techcrunch.com/2006/05/31/sixapart-to-launch-comet-renamed-vox-on-june-1/)

除了社交网站的所有基本功能，Vox 还包括广泛的隐私控制、博客帖子的标签云以及漂亮的 WYSWIG 合成页面。隐私级别是该公司战略的一个重要部分，特别是 Meena Trott 一段时间以来一直在谈论博客的未来将如何在小型、封闭的在线交流群体中找到。

个人资料页面不能在代码级别直接编辑，但有许多布局选项和超过 165 个外观鲜明的主题。还有简单的移动浏览和发布。媒体元素可以轻松地放入 Vox 页面，该网站与 YouTube、Flickr、Photobucket、iFilm 和 iStockphoto 集成。其中有几家是相互竞争的公司，它们都可以提供给用户，这很好。如果我昨天没有见到这家公司，并且看到他们的工作实际上非常面向社区和有趣，那么 IStockPhoto 对我来说似乎是一个奇怪的选择。用户喜欢 Vox 的一点是它的广告非常低调。这里的商业模式看起来很聪明。只有在管理页面、搜索结果和其他一些页面上有大的侧边栏广告，这很好。您可以查看个人资料页面和探索网站，而不用看大广告！在公共用户页面上有一些非常小的广告，用户被鼓励张贴他们喜欢的书籍和电影。读者可以通过附属链接购买这些内容，Vox 也将从中获利。我和 Six Apart 的 Anil Dash 谈过，他说广告将基本保持原样。

我希望 Vox 有一些不同之处。事实上，点击 Flash 媒体播放器会将你带到一个不同的 Vox 页面，上面几乎没有其他内容，这是非常反直觉的。如果在 Vox 中提供了对微格式的支持，并且做得和其他东西一样好，那就太好了。我也非常希望看到 OpenID 支持和用户数据的轻松导入/导出。Dash 告诉我，他们今天支持 OpenID 作为服务器(例如，你可以像 yourname.vox.com 一样在 LiveJournal 上发表评论)，并将很快添加对 OpenID 登录的完全客户端支持。Dash 表示，该公司正在努力做一些事情来简化出口。

不过，总的来说，我认为 Vox 在发布时看起来很棒。很明显，它是由一个经验丰富的团队建立的，他们一直关注市场如何发展以及人们在社交网络中想要什么。它非常悦目，看到人们趋之若鹜，我不会感到惊讶。有很多人渴望离开 MySpace，而作为大众兴趣社交网络，Vox 看起来是一个非常有吸引力的选择。
