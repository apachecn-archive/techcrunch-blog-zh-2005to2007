# 索尼爱立信 W660:看起来不像玩具的音乐手机

![](img/80a1dc2f2e411a8a3bd9c4ff5693d083.png)

索尼爱立信 W660 代表了数字音频播放器和手机之间的最新融合，这似乎是当今 T2 的时尚。3G 糖果棒有一个 200 万像素的摄像头，调频收音机支持，Java 2.0(你好，第三方应用！)以及在听音乐时可持续 25 小时的电池续航时间。它还具有音乐识别功能，试图通过分析几秒钟的歌曲来识别歌曲的名称。嗯，[21 点](https://web.archive.org/web/20130628172219/http://crunchgear.com/2006/11/30/samsung-blackjack-q-who/)不能很好地做到这一点，所以我有疑问。

W660 今年将在 Q2 的某些市场上市，有黑色和红色两种颜色。这可能是我第一次真正喜欢索尼爱立信手机的设计。狂喜一定就在附近。
