# Yoono Buzz It:彻底改造笔记本电脑

[![yoono.png](img/449c4c921a7fbf603c1fc20240c7fe20.png)](https://web.archive.org/web/20221001082622/http://www.yoono.com/) 社交冲浪推荐网站 [Yoono](https://web.archive.org/web/20221001082622/http://www.yoono.com/) 发布了一个新插件“Buzz It”，将谷歌笔记本等服务提升到一个新的水平。

大多数人都熟悉笔记本和剪贴簿风格的网站和插件。您可以添加注释、提醒和链接。Buzz It 采用了这些基础知识，并融入了丰富的 Web 2.0 体验。

该插件可用于 Internet Explorer 和 Firefox，并安装了一个新的工具栏。这是一个可以修复的初始缺陷。工具栏上的相关按钮(在 Firefox 中)可以拖动到浏览器顶部现有的占位符上。

使用 Buzz 记录网站或笔记非常简单，只需突出显示文本或页面，然后按下按钮或使用右键菜单。将出现一个包含相关信息的 Ajax 覆盖图。该插件从任何被浏览的网站中提取图像和视频，然后在 Buzz It 侧边栏中显示这些结果，将拖放功能添加到条目中。亚马逊图书链接甚至谷歌地图都可以内联生成，并添加到任何帖子中。

Buzz It 还兼作博客发布工具。大多数主流平台都支持，这是我用过的最好的博客发布工具之一。它需要一些工作，生成的标记低于平均水平，张贴的图像是从原来的网站，而不是在本地重新张贴；然而，对于 MySpace 用户或普通的博客作者来说，这些都不是大问题。它简单，干净，易于使用。

该插件集成了 Yoono 的社交冲浪推荐平台，并带有这类服务中常见的网站推荐风格工具。如果你在做这类事情，那么它是最好的服务之一，目前号称拥有超过 70 万会员。

是的，用(OMG！)另一个工具栏，但看过去，我可以诚实地说，这是令人印象深刻的。它很聪明，看起来很酷，一定会有广泛的吸引力。
