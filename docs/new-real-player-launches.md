# 新 Real Player 发布——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/06/26/new-real-player-launches/>

[![realplayer.png](img/962bd02d002aee08247f3dbfa20e70ee.png) ](https://web.archive.org/web/20210122142214/http://www.realnetworks.com/company/press/releases/2007/rp11b107.html) Real Networks 宣布在他们的[网站](https://web.archive.org/web/20210122142214/http://realplayer.com/)上公开发布他们的新 Real Player。我们[在本月初报道了测试版](https://web.archive.org/web/20210122142214/http://www.beta.techcrunch.com/2007/05/31/the-new-realplayer-ripping-a-youtube-video-near-you/)以及 [20 个测试版账户，这些账户很快就完成了](https://web.archive.org/web/20210122142214/http://www.beta.techcrunch.com/2007/06/04/try-the-new-realplayer-we-have-twenty-beta-accounts/)。

这款新播放器通过让用户在网络的任何位置保存和分享视频，致力于弥合桌面和互联网之间的视频鸿沟。新的播放器在你的浏览器中的任何视频中添加了一个“保存”按钮。当你点击它的时候，Real 会在你冲浪的时候在后台下载这个文件，不管它是什么格式。当你回放视频时，你可以选择与朋友分享视频，朋友会通过电子邮件发送一个原始视频的链接。有了免费版的播放器，你可以将视频刻录到 CD 上，可以在大多数 DVD 播放器上播放。付费购买专业版可以直接刻录到 DVD 上。未来的版本将允许你将视频推送到媒体播放器。

该播放器最显著的用途将是在视频巨头 Youtube 上，它已经吸引了一系列第三方下载工具，包括我们自己的。Real Player 将通过不下载受限制的内容来识别和遵守 DRM 限制。

该播放器仍然只适用于 PC，但他们预计今年晚些时候会推出 Mac 版。
