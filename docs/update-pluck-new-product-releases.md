## Pluck 推出了三款产品更新:

pull[于 2005 年 8 月 9 日宣布](https://web.archive.org/web/20230216182909/http://www.pluck.com/press/PluckPR-050808-Version2.html)三项产品更新——为 Internet Explorer、Firefox 及其网络产品(左边的网络产品图片)推出了 Pluck 2.0。一个关键的新特性是，所有这些产品都可以同步订阅源(并有望在下一个版本中同步已读/未读的帖子)。

Pluck 还很好地解释了什么是 RSS，以及如何为新的 RSS/Atom 用户订阅提要。例如，参见此处的 puck tour[和此处](https://web.archive.org/web/20230216182909/http://www.pluck.com/tour/)的“什么是 RSS”[。](https://web.archive.org/web/20230216182909/http://www.pluck.com/products/whatisrss.html)

以下是[新闻稿](https://web.archive.org/web/20230216182909/http://www.pluck.com/press/PluckPR-050808-Version2.html)的摘录:

> Pluck 版本 2.0 为 Firefox、Internet Explorer 和 Web 提供了跨平台的 RSS 阅读器套件
> 
> 德克萨斯州奥斯汀(2005 年 8 月 9 日)Pluck Corp .发布了业界第一的基于浏览器的 RSS 阅读器 2.0 版本，用于 Internet Explorer，并扩展了用于 Firefox 和 web 的 RSS 阅读器产品线。该产品套件是 RSS 领域中第一个允许用户在不同计算机、web 浏览器和操作系统之间同步重要 web 信息的产品，如 RSS web feeds 和书签。www.pluck.com 免费提供拔毛服务。
> 
> 今天发布的产品套件包括对 Pluck Internet Explorer Edition 和 Pluck Web Edition 的重大升级，还推出了一款名为 Pluck Firefox Edition 的新产品，用于流行和快速增长的 Firefox web 浏览器。Pluck Firefox Edition 在 Firefox 浏览器中提供了高级的 RSS 聚合和管理功能，使 Firefox 用户能够在易于查看的文件夹中轻松组织、查看、存储和共享 RSS Web feeds、互联网搜索结果、网页和其他信息。像所有的 Pluck RSS 阅读器一样，Pluck Firefox Edition 能够搜索感兴趣的主题以获得相关的 web 提要，并导入现有的书签和 Web 提要订阅集合。
> 
> 该套件中所有三种产品的新 Pluck 2.0 升级版为活跃的互联网用户提供了丰富的特性和功能，增强了他们查找、访问和管理网络信息的方式。此版本提供的功能包括:
> 
> *同步:使用户能够在多台计算机(PC、MAC 等)之间自动访问和同步他们的书签和 RSS web feeds。)、不同的 web 浏览器(Firefox、IE 等)和支持互联网的设备，因此 RSS 内容在工作中或家中总是可用的。
> *显著的 UI 改进:在 RSS 阅读、书签组织和订阅管理的所有方面都改善了用户体验。这种新体验旨在简化 RSS 学习曲线，提高 RSS 的采用率。
> * FeedFinder:这款来自 Pluck 的新提要发现引擎极大地提高了提要搜索的相关性，并为用户发现 web 提要开辟了新的途径。使用 FeedFinder，用户可以通过浏览 feed 分类、查看特色 RSS web feeds 以及在从互联网上收集的大量 feed 缓存中提交关键字搜索来找到感兴趣的 feed。
> *强大的搜索工具:允许用户通过一次点击和一个视图向多个搜索引擎发送搜索查询，节省时间并提高结果质量。

所有拔毛产品均免费。
