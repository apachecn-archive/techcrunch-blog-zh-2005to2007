# 回顾:DLO 的兴奋剂 iPhone 案件

对于那些不熟悉伟大的 DLO 的人，看看你的 iPod。如果它有一个箱子或一个码头，DLO 很有可能成为 T2。该公司拥有比苹果公司更多不同类型的 iPod 配件，其中大多数都很棒。

虽然我不是一个把电子产品困在箱子里的人，但我知道我是少数。你的 iPod 会掉下来，又脏又湿，某种保护性的盔甲是合适的。如果你是一个 iPhone 用户，那么保护你脆弱的投资是很重要的，DLO 也为你提供了保护。

上周末，我带着两个 DLO 的 iPhone 手机套去烧烤和其他娱乐场所，它们非常好用。配有皮带夹的奢华皮套非常适合卡其布和蓝色衬衫套装，它的风格足以让你看起来不像一个游客，但能让你轻松访问你的 YouTube 播放、iTunes 同步好友。

![photo_070707_003.jpg](img/050cef47a38935f6810eefc15105643d.png)

不过，我真正喜欢的是带耳机管理的硅胶外壳。对于那些用口袋来装装备的人来说，这是一种完美的避孕套。它完美地保护了 iPhone 免受适度跌落的影响，使它能够轻轻地弹回到背面的静止位置，这才是真正的乐趣所在。

![photo_070707_006.jpg](img/f7d55e6ba4697f2bb616f5d12f36f344.png)

DLO 已经在其 iPod 外壳上使用这种类型的耳机系统几个月了，所以它在这里包括它是很自然的。由于硅胶的柔软性(是的，我刚刚输入了)，这个盒子有一个整洁的派尔来包裹你的耳机电缆，末端有耳塞袋。它既优雅又简单，而且还能正常工作。

管理平台对 iPhone 整体厚度的增加可以忽略不计，当你考虑增加的保护时，价值就会累积起来。如果你做了一件蠢事，在 iPhone 上市前给自己买了一部，至少买一个这样的保护套，让它在好东西下架前保持良好状态。
