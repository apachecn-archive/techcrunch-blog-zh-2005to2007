# 东芝 Satellite P105-S9722 笔记本动手| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/12/toshiba-satellite-p105-s9722-notebook-hands-on/>

![](img/64d837b1706cfd7c1203a89c217816e4.png)

东芝的 Satellite P105-S9722 游戏笔记本电脑(为了方便我的手指，从现在开始将被称为 P105)最近在世界上发布，反响不错。这东西是一个完全的怪物，可以处理任何扔给它的任务。无论您是玩游戏、编辑视频、录制音乐，还是需要高性能设备的任何事情，P105 都很有可能完成。

![](img/71a2ab4c60edade1b8ea88bba2080993.png)

在我们深入了解规格之前，让我先谈谈 P105 的设计。瞥一眼这个笔记本的顶部，你会看到它有一个特殊的缟玛瑙蓝金属漆工作。这确实给了它一个光滑的外观，但也作为一个突出的指纹和粘性显示。此外，盖子不像我希望的那样容易合上。通常需要几次推动才能锁定。

P105 功能丰富。你有一个全尺寸键盘，一个内置功能按钮的触控板，一个指纹读取器，两个 Harmon Kardon 扬声器，以及顶部的一系列媒体功能键。虽然大多数这些功能都是不错的补充，但键盘的设计很差。请允许我进一步阐述:

东芝设计 P105 时考虑到了游戏玩家。显然东芝忘记提到的是，他们认为所有的游戏玩家都是左撇子。在键盘的左侧，所有键的比例都是 1:1。然而，当我们走到右边时，你会注意到这一边的 shift 键和标准键一样大。因为我是右撇子，每当我打字需要大写字母时，我不得不祈祷我刚才用小手指按的键是 shift 键。此外，箭头键是额外的小，是一个总在我的屁股疼痛使用。

![](img/8d4f1d4b741e6f43fe6cfec6318ffae4.png)

更不用说东芝在键盘的右侧还包括了一个全尺寸的数字小键盘。很高兴看到东芝缩小了数字小键盘的尺寸，增加了快捷键和箭头键的尺寸。但是关于键盘就说这么多了。让我们进入正题:规格。

卫星 P105 有一个巨大的 17 英寸宽屏 XGA 显示器，采用 TruBrite 技术。这个屏幕绝对是现象级的。电影看起来清脆，色彩丰富，游戏流畅。如今，你很难在笔记本电脑上找到更好看的屏幕。在内部，P105 配备了 2GHZ 英特尔迅驰酷睿 2 双核 CPU、2GB PC 5300 DDR2 RAM、可录制 11 种格式的 DVD 刻录机、GeForce Go 7900 256MB 显卡、多个音频输入、集成无线、VGA/S-Video/DVI 输出、4 个 USB 端口、Firewire、200GB SATA 硬盘、5 合 1 多媒体读卡器和 Windows XP Professional。所有这些使得 P105 的价格为 1999 美元，重 7.1 磅，并且是一个动力强劲的产品。

当我收到 P105 时，它是全新的，我很兴奋地看到等待我的是什么。不幸的是，当我启动 Windows XP 时，我的兴奋开始变得索然无味。东芝采取了戴尔式的举措，为这款出色的笔记本电脑装载了大量试用软件和优惠。我的桌面上有三列图标，都是间谍软件的试用版。这让我真的很不开心，因为我必须经历和删除所有这些废话，然后才能开始让它感觉像是我自己的。甚至几个星期后，每隔一个小时左右就会出现一些愚蠢的弹出窗口，让我知道我的电脑可能有危险，我应该购买一些可怕产品的完整版本。所以东芝，在未来，试着减少免费软件。我知道你把它放在那里赚了一些钱，但是这些东西都不好！至少包括一些新发布游戏的试用版，因为这应该是一款游戏笔记本电脑。

![](img/08ede69c5e79fa0b099545082fa32242.png)

一款运行良好的软件是指纹识别器。设置 P105 的指纹扫描仪非常简单，您可以使用它登录 Windows 并访问文档。这是一个非常好的安全特性，许多企业和高级用户会发现它非常有用。拥有一种额外的安全形式，这样任何时候都不会有人偷看你的文档。

当我打开笔记本前面的无线网络开关时，我注意到弹出了一个小小的连接指南。东芝的无线连接软件还不错，但并没有什么特别之处。我选择关闭它，进入 Windows 的无线设置。这里是事情走下坡路的地方。我与邻居的无线网络和我自己的无线路由器的连接总是像疯了一样中断。我尝试重置我的路由器，使用不同的设置和其他几种解决方法。最后，无线似乎有点古怪。

然而，我不打算为此完全责怪东芝。我家的无线网络一直有点问题，我确实在星巴克停下来测试了 T-Mobile 热点接入点上的 P105。看来星巴克里漂浮的咖啡因真的起到了作用。当我在网上冲浪、下载音乐和玩魔兽世界时，P105 飙升。看起来我自己的网络可能出了问题，但 P105 不会那么容易就离开。你可以告诉它有时使用无线网络有问题，希望东芝在发布这款笔记本的未来版本时会注意到这一点。

![](img/039c04f708cae58343b92cdc53aabadd.png)

但这应该是一个游戏笔记本电脑，对不对？确实是。我经常玩《魔兽世界》,但从来没有机会展示完整的细节，因为我知道如果我这样做，我的 FPS 率会直线下降。P105 上没有。我把球打到墙上，把每个设置调到最高，然后去了镇上。没有延迟，没有问题，只是全力以赴，在艾泽拉斯的世界里清澈见底的屠杀。我甚至能够在 WoW 和 iTunes 之间“alt-tab”切换，而不会跳过我的音乐的一个节拍——这是在全屏模式的魔兽世界和 iTunes 之间切换时通常会出现的问题。

多任务处理是这款笔记本擅长的另一个方面。我可以一边在 YouTube 上看视频，一边在 Outlook 中查看电子邮件，一边打开 WoW，在 Microsoft Word 中键入一篇文章。我知道，我知道…我应该检查一下 ADD 和所有的东西，但是当我可以用 P105 一次做这么多事情的时候，为什么还要麻烦呢？我很高兴我可以同时做这么多事情而不需要妥协。干得好东芝。

![](img/44ac6851d7d9bc83e878bd296ee9c2d8.png)

因此，一个问题仍然存在:值得为这款笔记本支付 1999 美元，并随身携带它不那么微妙的 7.1 磅吗？如果你是游戏玩家，答案是肯定的。对于游戏来说，这东西是个野兽，你扔给它的任何头衔它都会玩。17 英寸的 TruBrite 显示屏可以播放电影和游戏，为您提供充足的工作空间，并且有大量 USB 端口用于外设。

然而，如果你不像我一样经常玩游戏(我几乎只玩 WoW ),并且是一名学生或者只是一名普通的电脑用户，那么你应该尽一切办法避免卫星 P105-S9722。它的键盘设计、无线问题和捆绑软件是你可能想要远离的三样东西。此外，沉重的重量和高昂的价格并不能证明那些不是游戏玩家或预算有限的人值得购买。取而代之的是更轻更便宜的产品，比如东芝的卫星 U200-ST2091。它的起价为 899 美元，比 P105 便携得多。
