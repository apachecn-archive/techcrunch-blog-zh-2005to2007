# 旋转搜索 TiVo 上线| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/15/swivel-search-for-tivo-goes-live/>

[![swivelsearch.jpg](img/a2458ea94d6e04d731774867104d9a77.png)](https://web.archive.org/web/20150324055724/http://old.crunchgear.com/wp-content/uploads/swivelsearch.jpg "swivelsearch.jpg")

TiVo 今天推出了新的旋转搜索功能，试图让搜索不同的视频内容变得更容易。Swivel Search 是为宽带连接的 TiVo(T4)开发的，它是一种元搜索，可以搜索 TiVo 内容的许多不同来源，如 TiVoCast 和 Amazon Unbox(T6)，以找到你感兴趣的内容。比如说你在看一集*宋飞*想找其他类似的节目。嘭！旋转搜索然后找到了《老克里斯汀的新冒险》(因为朱莉娅·路易斯·德利法斯主演了两部电影)。就这么简单。
