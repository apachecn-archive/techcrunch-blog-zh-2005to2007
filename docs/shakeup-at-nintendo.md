# 任天堂大换血

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/07/shakeup-at-nintendo/>

美国任天堂公司的事情变得越来越有趣。Game Informer 报道称，三名关键玩家跳槽了，而且他们不是唯一的一个。该公司的销售和营销团队准备从西雅图搬到旧金山或纽约，大约 90%的员工决定不去。但大新闻是，三名关键高管，包括公共关系高级总监 Beth Llewelyn，营销和企业事务副总裁 Perrin Kaplan 和乔治·哈里逊营销和企业传播高级副总裁(右)也将离开公司。

唯一留任的关键营销高管是 NoA 董事长兼首席执行官君岛达己和总裁兼首席运营官·雷吉·菲尔斯-艾梅。该公司没有就这些传言是否属实给出官方说法，但任天堂的一次改组已经酝酿了很久。
