# 更新:Technorati 比之前报道的多获得 300 万美元

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/08/15/update-technorati-got-3-million-more-than-previously-reported/>

 [](https://web.archive.org/web/20170422162859/http://technorati.com/) 我们上个月报道了博客搜索引擎 [Technorati](https://web.archive.org/web/20170422162859/http://technorati.com/) 已经在 6 月份从之前的投资者 Draper Fisher Jurvetson 和 Mobius Venture Capital 获得了 760 万美元的 C 轮融资。今天的 [PE Wire](https://web.archive.org/web/20170422162859/http://hosting.mansellgroup.net/enablemail/ThomsonNewLetter/HostedWires/NewsLetters/Aug15-06.htm) 现在说 C 轮融资已经达到 1052 万美元。之前的投资让许多人对其影响感到困惑，今天的新闻确实回避了 Technorati 将何去何从的问题。

现在这个网站已经成立三年了，刚刚发布了一个重大升级。我们上次说过，我们假设本轮估值持平或下跌。这可能是一个长期的赌注，也可能是一个试图将网站推向行动的尝试，这样它就可以筹集更多的收入。这很难说，但今天的消息称 C 轮融资比之前报道的多了近 50%,这无疑让事情变得更加有趣。
