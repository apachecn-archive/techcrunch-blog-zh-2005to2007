# 美国将获得索尼爱立信 W610

![](img/1027f9c8df8442de4c69bfba5006c24c.png)

又是一天，又一部随身听手机出现在新闻中。索尼爱立信今天宣布，如果你住在美国，你将得到另一款 GSM 随身听系列手机 W610。

它将配备带 [RDS](https://web.archive.org/web/20210116043225/http://en.wikipedia.org/wiki/Radio_Data_System) 的调频收音机，200 万像素摄像头的 LED 闪光灯，改进的电池(感谢上帝)，支持 A2DP 的蓝牙和 512MB 记忆棒(Duo？)包含在内。

在功能集和设计之间，我想说 W610 看起来有点像旧的 W810，这远远不是开创性的。

索尼-爱立信最好开始拿出一些新的想法和设计，否则它将在阴沟里结束。你可以在 Q2 以高昂的价格从任何一家运营商那里买到 W610。
