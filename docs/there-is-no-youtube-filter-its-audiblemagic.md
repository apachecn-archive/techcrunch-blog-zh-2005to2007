# 没有 YouTube 过滤器；这是听觉上的

经过几个月激烈而公开的辩论，与谷歌的收购密切相关，据报道，YouTube 从 T2 获得了版权过滤技术的许可。《圣荷西水星报》援引两名未透露姓名的消息人士的话称，谷歌将很快公布领先的第三方过滤提供商 Audible Magic 为 YouTube 提供的过滤技术。

这是什么意思？这意味着，几个月来保证 YouTube 正在开发版权过滤技术并即将实施，要么是一个争取时间的诡计，要么是一个失败的努力，今天在压力下崩溃了。

[十天前](https://web.archive.org/web/20220110012518/http://www.beta.techcrunch.com/2007/02/12/myspace-to-implement-copyrighted-video-filtering/)宣布 MySpace 已经获得 AudibleMagic 过滤技术的版权保护许可。每个人都问的一个大问题是——这对 YouTube 意味着什么？虽然人们的反应各不相同，从屏息等待一项神秘的技术，到指责 YouTube 的黑手党般行为，但事实可能要平凡得多。可以说，YouTube 从一开始就不是一家科技公司。

谷歌和 YouTube 的发言人一再声明内容过滤迫在眉睫，但没有对美国商业研究中心的报告做出回应。
