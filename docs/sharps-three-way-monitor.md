# 夏普的三路显示器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/27/sharps-three-way-monitor/>

当我们路过去买一件漂亮的亚美尼亚制造的 t 恤时，http://tctechcrunch2011.files.wordpress.com/2007/01/harlots 会盯着我们看。没有人希望这样。

看，屏幕的工作原理就像那些奇怪的全息图像一样，当你移动卡片时，女士会盯着你看并眨眼。使用视差系统，屏幕从三个不同的角度显示三张不同的图片，并在您经过显示器时无缝切换。这是一个有趣的想法，但第一次看到它时会令人毛骨悚然，各位，现在我们离《银翼杀手》更近了。
