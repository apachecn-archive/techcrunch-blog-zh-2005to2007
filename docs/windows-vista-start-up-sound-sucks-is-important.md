# Windows Vista 的启动声音很糟糕，是神圣的

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/08/27/windows-vista-start-up-sound-sucks-is-important/>

为大比尔辩护，通知我们:

> 微软的史蒂夫·鲍尔给了罗伯特·斯考伯一些启动声音的理由，截至目前的计划无法关闭:1)“品牌体验的精神层面。一个简短、正面的确认，表明你的机器现在有意识并准备好做出反应”；2)“启动声音旨在帮助您校准或修复启动机器时发出的声音。”

作为回报，埃德·博特回击说，启动时发出的声音就像神庙里的屁一样响亮和突兀——就像 Windows XP 的声音，可能还有 Vista 的声音——应该可以由用户修改，如果用户希望的话，应该可以完全删除。但是，这个 shizz 是有灵性的，宝贝，你不要乱动灵性。
