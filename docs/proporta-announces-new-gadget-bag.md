# Proporta 发布新的小工具包

![](img/a9d3b6879142313df9fdae8778b63bc7.png)

老实说，我一直在寻找一个像这样的包包。我不再有一台可以工作的笔记本电脑，但是我有很多随身携带的小工具和小玩意儿，它们通常被随意地放在一个 North Face 包里。Proporta 的小工具包包括保护设备的厚衬垫，大量的独立隔层，甚至还有一个可以将耳机连接到 iPod/MP3 播放器的地方。它会花掉你大约 50 美元，现在就可以买到。
