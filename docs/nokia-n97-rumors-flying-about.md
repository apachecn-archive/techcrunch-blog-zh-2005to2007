# 诺基亚 N97 谣言满天飞

握着电话的诺基亚(哈！最后我让那个笑话起作用了)，你真的要发布 N97 吗？说真的，谣言满天飞，说诺基亚的新 N97 手机是真的，并慢慢成为人们手中的原型。Mobile Whack 有一些性感手机的热门图片，但它们是真的吗？传闻中这款产品的规格非常疯狂:巨大的 3 英寸显示屏，20GB 内存，带 miniSD 卡插槽，500 万像素摄像头，20 倍变焦。不可能？当然不是。但在我们看到具体证据之前，我们会对此持保留态度。如果这些照片确实是真实的，这款手机似乎有一个类似于 N93 的键盘和专用的音乐播放按钮。继续做梦吧，伙计们，往前走，这里没什么可看的…还没。
