# 热情的用户激发创造力

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/21/passionate-users-jumpstart-creativity/>

我们以前都见过这种情况。热情的用户希望给定的产品有更多或不同的功能，他们自己构建而不是等待。有时这样做纯粹是出于激情，有时也有盈利的动机。但结果往往是给最初的创业公司当头一棒。该公司的反应很大程度上说明了他们对用户的承诺。

2006 年的一个很好的例子是[用户创建的 Pandora 和 Last.fm 的混搭](https://web.archive.org/web/20211208232216/http://www.beta.techcrunch.com/2006/03/29/pandora-and-lastfm-togethersort-of/)(实际上是一种黑客行为)[很快被两家公司接受并支持](https://web.archive.org/web/20211208232216/http://www.beta.techcrunch.com/2006/04/21/update-on-pandoralastfm-mashup/)。该产品仍然存在，并且运行良好。

今年我们看到的两个例子是 [LinkedInABox](https://web.archive.org/web/20211208232216/http://www.linkedinabox.com/) 和 [WootAgent](https://web.archive.org/web/20211208232216/http://www.wootagent.com/) 。

 **[WootAgent](https://web.archive.org/web/20211208232216/http://www.wootagent.com/) 还在私下测试中。这是一个小的下载版本(目前只适用于 Windows 机器),铁杆 Woot 迷们会想用的

如果你不知道 Woot 是什么:它是一个成立于 2004 年的电子商务网站，每天都有一件商品出售。出售的数量是有限的，一旦买了，第二天才会出售。运费总是 5 美元，偶尔他们会出售“随机垃圾”，可能是无用的物品或令人难以置信的东西，如 20 美元的新液晶显示器。随机垃圾销售通常在几分钟内销售一空。

Woot 的问题是，人们经常忘记检查网站，他们错过了好的交易。WootAgent 位于系统托盘中，当有新 Woot 可用、Woot 快用完、Woot 售完时会自动弹出(见截屏)。对于除了忠实的 Woot 用户之外的任何人来说，这将是令人讨厌的，但对于那些关心的人来说，这是令人敬畏的。
 **LinkedInABox**

 [LinkedInABox](https://web.archive.org/web/20211208232216/http://www.linkedinabox.com/) 是一个 javascript 小部件，用户可以将其放在网站上(见 TechCrunch 法国编辑 [Ouriel Ohayon 的个人博客](https://web.archive.org/web/20211208232216/http://ouriel.typepad.com/)查看小部件，位于右侧栏)。这个小部件显示了 LinkedIn 个人资料中包含的基本信息，是专业博客或其他网站的一个很好的补充。坦率地说，这是 LinkedIn 应该自己做的事情，我希望他们接受这个产品。

我将寻找用户生成的产品和功能的其他例子，这些产品和功能增强了一个备受喜爱的核心产品，并在这里描述好的例子。**
