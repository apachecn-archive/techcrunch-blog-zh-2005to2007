# Palm Gandolf 规格披露| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/18/palm-gandolf-specs-revealed/>

[手掌甘多夫](https://web.archive.org/web/20151002003950/http://crunchgear.com/2007/06/10/mystery-vodafone-branded-palm-device-appears/)上泄露的规格显示[的一些传言](https://web.archive.org/web/20151002003950/http://crunchgear.com/2007/06/11/rumor-touchscreens-gone-for-all-but-high-end-palms/)其实是真的。Gandolf 将有两种版本可供选择；一个运行 WinMo 6，而另一个将运行新的无触摸屏 Palm OS。代号 Gandolf 将由 Treo 5XX 取代，并提供彩虹般的颜色。2.4 英寸的 QVGA 屏幕将有助于在 3G 网络和 HSDPA 上浏览互联网。Gandolf 将包括 256MB 的板载存储空间和 64MB 的内存。它的重量仅为 135 克，框架为 117 x 65 x 13.8mm 毫米。200 万像素摄像头和支持 A2DP 的蓝牙 2.0 完善了已知功能。价格和可用性仍然未知。
