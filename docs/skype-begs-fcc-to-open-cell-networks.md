# Skype 恳求 FCC 开放手机网络

![](img/1f827e163217674c636f7c2765ef8296.png)

Skype 一定是想利用美国人沉迷于手机的事实来赚钱。昨天，他们给 FCC 发了一封信，要求移动运营商允许用户运行他们选择的任何通信应用程序。简而言之，Skype 基本上希望 T-Mobile、美国电话电报公司、威瑞森和 Sprint 等公司允许客户安装手机版 Skype，以便用手机打便宜或免费的电话。

它可以想怎么叫就怎么叫，但 Skype 这样是不会有任何进展的。我很确定威瑞森会奋力抗争到底。电信公司还能如何盈利？如果 Skype 真的赢了，无线运营商将被迫提高手机价格来帮助平衡损失。有些人可能喜欢用 Skype 打更便宜的电话，但我喜欢以合理的价格购买新手机。
