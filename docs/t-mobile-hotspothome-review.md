# t-Mobile Hotspot @ Home Review

我的后门廊:热点！

我住在布鲁克林湾脊一条安静的街道上。我肯定不在这个国家，周围也没有任何可能干扰我手机信号的高楼。然而，当 T-Mobile 的代表去年夏天来我家时，他发现我的前门厅没有服务——是的，我有一个门厅，那又怎么样？—用科学术语来说，就是客厅里的几个酒吧。连地下室都不要问。我用 [Wi-Ex 蜂窝扩展器](https://web.archive.org/web/20210127033518/http://crunchgear.com/2006/11/15/wi-ex-zboost-yx500510-cellphone-extender-first-look-oh-wow/)补救了这种情况，但直到今天，我才终于看到了隧道尽头的光明。

T-Mobile 的 Hotspot @ Home 对于在家或关机时手机信号不好的人来说几乎是梦想成真。这项服务的费用是单条线路每月 19.99 美元，家庭套餐每月 24.99 美元，当你在 T-Mobile 或其他你可以访问的 WiFi 热点上时，可以在美国无限制通话。这项服务配备了 Linksys 或 D-Link 路由器——两者大致相同——专门针对 Hotspot @ Home 的 [UMA 服务](https://web.archive.org/web/20210127033518/http://en.wikipedia.org/wiki/Generic_Access_Network)进行了调整。

这项服务目前支持两款手机，三星 t409 和诺基亚 6086。我收到了 6086，我可以诚实地说，这不是我梦想中的手机。它有一个 VGA 摄像头，EDGE，蓝牙和音乐播放器，以及 AIM 和互联网接入，它非常小。然而，一般来说，它是一款“Meh”手机，如果 T-Mobile 推出对其他支持 WiFi 功能的手机的支持，如 [Wing](https://web.archive.org/web/20210127033518/http://crunchgear.com/2007/05/22/t-mobile-wing-review/) 或 [Dash](https://web.archive.org/web/20210127033518/http://crunchgear.com/2006/11/01/smartphones-now-the-t-mobile-dash/) ，我会非常兴奋。哦好吧。一个女孩可以梦想。这是我对这项服务的唯一疑问。说真的。我只是不喜欢电话。

那么当你打开手机时会发生什么呢？它开始寻找打开的热点，如果找到已经连接的热点，它会自动连接，热点名称会出现在屏幕上。就是这样。现在你所有的通话，不管是不是 T-Mobile 路由器，都是通过 WiFi 路由的。通话质量非常好，尤其是在像我这样一个奇怪的死区。它听起来不像你期望的 VoIP 那样有金属质感。取而代之的是，声音稍微柔和一些，没有普通手机通话的刺耳声音。这几乎就像从座机转到 Skype 一样——试试看。你会对这种差异感到惊讶。

该路由器可以作为家庭中的常规 WiFi 路由器使用，对通话质量没有影响。手机电池一次充电可以持续两天左右，对于那些需要定期连接 WiFi 的设备来说，这是一个令人印象深刻的壮举。

在我的测试中，使用 T-Mo 路由器时，声音从未掉线。我在我的 Airport Extreme 上使用它没有任何问题，切换也很无缝——我可以走出家门，走到街上，手机会无声无息地切换到蜂窝网络。根据计划，T-Mobile 提供每月 9.99 美元/19.99 美元的服务，不幸的是，这项交易可能会在本周 iPhone 的喧嚣中失败。幸运的是，T-Mobile 的员工将收到一本特殊的手册，教他们如何发现 badcellularitis，这是一个可以在任何时候袭击任何人的问题。你家里信号不好吗？你对目前的手机运营商感到失望吗？你是否生活在一个由于浓密的树叶、灌木或古代文明的废墟阻挡了手机信号塔而导致信号覆盖率较低的地区？向你的医生咨询 Hotspot@Home。
