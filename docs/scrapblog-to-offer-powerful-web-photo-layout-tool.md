# Scrapblog 为网络带来强大的媒体布局

 [](https://web.archive.org/web/20220129135326/http://scrapblog.com/) [Scrapblog](https://web.archive.org/web/20220129135326/http://scrapblog.com/) 将在本周的 [DEMO](https://web.archive.org/web/20220129135326/http://demo.com/) 上展示，他们的新软件是一个很好的例子，展示了基于网络的应用程序能做多少事情。这是一个 Flash 应用程序，允许用户将照片、视频、文本和音频拖放到背景图像上，以建立剪贴簿，可以打印为相册，刻录到 DVD，导出为 Flickr 的照片集，很快就可以导出为 YouTube 和其他视频网站的带解说的幻灯片。

Scrapblog 允许的对布局的平滑控制的数量确实令人印象深刻。这是一个吸引人的工具，超越了通常的剪贴簿人口统计，并可能在制作艺术家作品集和在线图形演示等方面有很大的用处。然而，即使是剪贴簿，也是当今美国领先的民间艺术之一。如果你是一个艺术势利者，不要因此而气馁；剪贴簿是很多有趣的使用它的网络界面。

该公司由卡洛斯·加西亚创办，目前自筹资金。它将通过印刷相册，刻录 DVD 和销售 pro-accounts 来赚钱。一些广告也可能出现在网站上。今天在 Scrapblog 网站上有一个不太复杂的免费测试版，但我在这里预览的网站将在今年年底前推出。

Scrapblog 允许用户移动和改变一长串图形元素的方向，添加阴影、文本和音频旁白等效果。*这就像在网络上拥有 Photoshop Elements*，但是添加了拖放图像、多媒体支持以及与您在 Flickr、Webshots 和 Photobucket 上的帐户双向同步。

Tabblo 提供类似的服务，但是 onetrumedia 是一个更直接的竞争对手。OneTrueMedia 给人一种更加封闭的企业感觉，但该公司也有许多高调的发行合作伙伴。
