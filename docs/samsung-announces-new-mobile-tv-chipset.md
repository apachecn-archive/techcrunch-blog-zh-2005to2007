# 三星发布新的移动电视芯片组

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/27/samsung-announces-new-mobile-tv-chipset/>

![](img/473ad1b75ebb2e10e04e839e87e4e0be.png)

三星刚刚发布了高性能、多标准移动电视芯片的细节，这将允许手机制造商在他们的设备上启用电视，而不管国家或标准如何。通过这项新技术，三星旨在满足移动电视行业的需求。耶。

这种新芯片让消费者可以选择他们想要的广播服务，而不必改变他们的设备或应用程序。三星系统大规模集成电路部门副总裁 Yiwan Wong 表示，移动电视市场有“巨大的增长潜力”。(王先生，你这样做太冒险了。)
