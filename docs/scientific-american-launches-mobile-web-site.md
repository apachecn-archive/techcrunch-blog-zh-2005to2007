# 《科学美国人》推出移动网站

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/08/scientific-american-launches-mobile-web-site/>

![sciam.jpg](img/d75133f17c1e5e2f9ceb5c45e2078353.png)需要搜索科学、技术、空间方面的每日新闻或寻找有趣的科学相关琐事…并且需要在旅途中完成这些工作？现在这一切就像你的手机浏览器一样近了。《科学美国人》推出了一个新的移动互联网网站。设计时考虑到了移动搜索和浏览，该搜索提供了报刊杂志中当前文章的列表，以及相关文章的链接。

*《科学美国人》*在线总经理兼副总裁米娜·勒克斯表示:“我们的移动服务的推出是我们出版物更广泛的品牌战略的一部分，旨在为我们的忠实读者提供新技术。”。“我们新的移动服务将允许我们的读者在旅途中轻松访问我们的内容和最新的突发科学新闻，为科学爱好者提供更多选择来访问来自《科学美国人》的信息，这些信息已经包括网站、博客、播客、RSS 订阅源和印刷版。”

《科学美国人》的这个新的移动版本是由 MindMatics 开发的，该公司还为多个索尼影业 DVD 发行、QPower Motor Oil 和许多制药公司推出了移动目的地。
