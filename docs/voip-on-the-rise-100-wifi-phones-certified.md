# VoIP 正在崛起:100 部 WiFi 电话获得认证

![](img/b95c456118098cb15f10d942fe5252cf.png)

随着每个新月的临近，VoIP 似乎越来越受欢迎。WiFi 联盟已经认证了 82 个双模式和 10 个 WiFi 专用手机，非常适合 VoIP 使用。该组织检查手机的互操作性和性能，以及干扰，以确保它们符合标准。该联盟现在正在推广 WMM(无线多媒体)和延长电池寿命。

希望到明年，我们将看到认证 VoIP 电话的显著增长，这可能是消费者注册 Vonage 和 Skype 等服务所需的推动力。
