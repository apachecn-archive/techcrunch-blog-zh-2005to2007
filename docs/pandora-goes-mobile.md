# 潘多拉走向移动——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/22/pandora-goes-mobile/>

在三藩市现代艺术博物馆，潘多拉刚刚宣布了他们为 Sprint 手机开发的新移动客户端。TechCrunch [在公告上有更多细节](https://web.archive.org/web/20201101020221/http://www.beta.techcrunch.com/2007/05/22/pandora-goes-mobile-and-sonos-and-more/)。

该客户端将允许 Sprint 的客户将他们的 Pandora 电台流式传输到他们的手机上。在接下来的 30 天里，客户可以通过手机免费前往 Pandora.com。之后，你将需要一个潘多拉高级账户，每月 3 美元，可以免费进入潘多拉。

对手机音乐感兴趣的读者应该看看我们对 [PartyStrands](https://web.archive.org/web/20201101020221/http://mobilecrunch.com/2006/11/12/partystrands-goes-mobile/) 、 [Avvenu](https://web.archive.org/web/20201101020221/http://mobilecrunch.com/2006/06/01/avvenu-announces-premium-services-and-mp3-streaming-from-your-pc/) 和 [Oboe](https://web.archive.org/web/20201101020221/http://www.beta.techcrunch.com/2005/12/02/oboes-web-music-locker/) 的报道。
