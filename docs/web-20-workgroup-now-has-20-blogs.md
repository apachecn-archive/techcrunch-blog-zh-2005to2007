# Web 2.0 工作组现在有 20 个博客

大约一个半月前， [
理查德·麦克马努斯、弗雷德·奥利维拉和我](https://web.archive.org/web/20220121193840/http://www.beta.techcrunch.com/2005/10/10/web20-workgroup/)组成了 [Web 2.0 工作组](https://web.archive.org/web/20220121193840/http://www.web20workgroup.com/)。我们现在已经发展到 20 个博客:

我们的目标是提供一个高质量的博客列表，这些博客写的是网络上的最新趋势。我们会继续定期添加博客，并致力于通过多种方式使其内容更易于访问。今天，我们将博客分成粗略的类别，为每个博客提供 RSS 源，并为那些想订阅所有博客的人提供一个聚合的 [OPML 源](https://web.archive.org/web/20220121193840/http://www.web20workgroup.com/web20workgroup.xml)。我们将很快增加更多的功能。

请继续给我们推荐好的博客。我们一直在寻找扩展对话的方法。
