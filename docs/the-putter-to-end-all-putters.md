# 结束所有推杆的推杆

![](img/0b9aa35eb7555b2eb23dfc24d968956e.png)

我从未喜欢过高尔夫球。我想这是因为我更喜欢需要大量出汗的运动(编辑:而且事实上这很无聊。布雷克)。不过，如果我拿起它，我绝对会买一个这样的推杆，这样我就可以打败任何我玩的人。来自韩国 Infinics 公司的 DiXX Blu 数字讲师。

推杆传感器和液晶显示屏帮助您在挥杆前将球杆与球完美对准。当你把它放在正确的位置时，它实际上给你开了“绿灯”。除此之外，推杆还会收集数据，如每次挥杆时的挥杆路径、挥杆速度、撞击位置和推杆面的角度。我不知道它是如何测量你脸部的角度的，但是它看起来很漂亮。本月晚些时候将在日本发售，售价为 9.8 万英镑(832 美元)。
