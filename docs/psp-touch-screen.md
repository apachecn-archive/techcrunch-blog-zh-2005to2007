# PSP 触摸屏

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/04/psp-touch-screen/>

![](img/29a68b719ec400aa8a8b14188ea7aa9d.png)

PSP TouchMe 是一个触摸屏，通过 USB 连接到 PSP，位于 PSP 的原始屏幕之上。TouchMe 是基于自制软件的设备，只能与特定的自制软件一起工作，所以不要指望索尼会开始许可与触摸屏兼容的游戏。不管怎样都很酷，对吧？TouchMe 通过拧入 PSP 的顶部安装到 PSP 上，价格约为 56 美元。TouchMe 还没有出售，没有关于它的发布日期信息，TouchMe 可能永远不会问世。现在，TouchMe 仍在测试中，我们甚至不确定它是否有效！如果真正把这个东西组装起来的自制软件开发商让它工作，自制软件开发商将离欺骗自己以为他们在玩令人垂涎的任天堂 DS 更近了一步。
