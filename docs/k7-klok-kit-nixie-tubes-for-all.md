# K7 Klok 套件:谢妮管的所有

![](img/7322768c1cd0a846e45759576c534fe6.png)

我们喜欢谢妮管。它们是真空管，曾经被用来读出旧的电子设备，如核反应堆控制室和詹姆斯·邦德维利安炸弹。现在你可以用 K7·克洛克套装制作你自己的谢妮钟，一个 129 美元的套装，里面有你像金手指一样摇滚所需的一切。
