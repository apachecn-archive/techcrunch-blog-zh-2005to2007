# TechCrunch 产品指数

我已经发布了一个关于 TechCrunch 上的公司和产品的索引。这是一种比使用搜索功能更快地找到我所写的东西的方法。事实上，我很难用标准搜索找到我自己的帖子。如果你知道这家公司的名字，进入这个页面，点击它就可以看到所有提到它的帖子的列表。

 [](https://web.archive.org/web/20220703180452/http://www.beta.techcrunch.com/company-index/) 这是自我生成。Rachel Cunliffe 获取了终极标签战士[WordPress 插件(由 Christine Davis 编写)的基本代码，并在几个关键方面进行了修改，以满足我的需求。](https://web.archive.org/web/20220703180452/http://www.neato.co.nz/ultimate-tag-warrior/)

结果是一个页面，列出了我写过的每家公司。出版商的界面基本上与 UTW 相同。Rachel 所做的更改主要是针对输出页面。

我们将在今天晚些时候或者明天把它发布到 WordPress 插件数据库中，这样任何为 WordPress 写东西的人都可以使用它。

计划中的改进还包括创建 OPML 格式的索引。

这个[索引](https://web.archive.org/web/20220703180452/http://www.beta.techcrunch.com/company-index/)也从任何 TechCrunch 页面的左侧栏顶部永久链接。
