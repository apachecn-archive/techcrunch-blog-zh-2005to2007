即时消息互操作性:不再只是一个梦想

原文：https://web.archive.org/web/http://www.techcrunch.com/2006/07/12/windows-live-yahoo-im-interoperability-begins-public-tests-today/trackback/

马歇尔·柯克帕特里克

微软刚刚宣布，今天是其即时消息客户端和雅虎之间实现互操作性的第一天！有语音信使。微软表示，这是两个不同的全球消费品牌第一次让他们的即时通讯客户端具有互操作性。该公司表示，拥有近3.5亿账户的用户群是全球最大的。IM互操作性花费了太长时间，我认为这永远不会发生。

Windows Live Messenger 是MSN Messenger的最新版本。雅虎！语音信使是雅虎最新的即时消息客户端！

IM系统之间缺乏互操作性一直是用户的主要抱怨。像Windows的 Trillian 或Mac的 Adium 这样的服务有助于缩小服务之间的差距，但却失去了用户可能更喜欢的某个系统的独特界面和一些功能。

今天的新闻应该会让即时消息对这两个系统的专用用户更有价值。很高兴看到，尽管两家公司都在努力为自己的消息服务开发增值界面——他们并不害怕失去客户，而是假装其他即时消息客户不存在。
