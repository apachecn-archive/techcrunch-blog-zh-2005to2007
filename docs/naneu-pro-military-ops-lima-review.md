# Naneu Pro 军事行动 Sierra 评论

虽然“携带单反相机和笔记本电脑”通常不被认为是军事行动，至少根据我所认识的海军陆战队员(嗨，克里斯！)，Naneu Pro Sierra 包包相当有独创性。这款包的设计可以容纳一台相机和一台笔记本电脑，结构紧凑，比例匀称，可以让一台 12 英寸的 Powerbook 和我的 Rebel XT 同居，就像大学处女发现第一次尝试禁忌之爱一样。

我在几次飞行中尝试了这个，发现作为一个标准的单反和笔记本电脑外壳，Sierra 是完美的。插入物可以保证单反一侧的安全，并且有足够的空间滑入笔记本电脑而不用担心损坏。不幸的是，当我取出填充的插页，并试图将其用作书籍和杂志的随身携带时，我发现了一些局限性。
几本书也淹没了笔记本电脑的口袋。然而，如果你需要一个像它宣传的那样的包——安全地携带你的相机和笔记本电脑而无需交互——这是一个赢家。

这是给谁的？移动中的相机极客，简单明了。尽管 Lima 的一个带子上确实有一个指南针，但你最好的选择是避免震慑战术，而是专注于携带你的相机和较小的笔记本电脑安全通过坦帕机场。Naneu 主页 89.99 美元。
