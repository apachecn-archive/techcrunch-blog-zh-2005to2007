# Newsgator Go！对于明天出局的手机

 [](https://web.archive.org/web/20221002161649/http://www.newsgator.com/) RSS 供应商 [Newsgator](https://web.archive.org/web/20221002161649/http://newsgator.com/) 将于明天宣布在 Windows 手机上阅读 RSS 提要的新服务。名为 Newsgator Go！该程序的价格约为 30 美元，并将与 Newsgator 家族的其他成员同步数据:Mac 版的 NetNewsWire、PC 版的 FeedDemon、Newsgator Online 和 Outlook 版的 Newsgator Inbox。Newsgator 产品是我阅读提要的最爱，我很高兴听到该公司将在未来几个月推出 Java 阅读器。

该产品最棒的一点是，它允许通过 Newsgator Online 中的一个界面指定特定的订阅源组，以便在不同设备之间共享。当我对 Newsgator Online 无法显示 1000 多个订阅源而不被阻塞感到失望时，我开始在我的桌面上使用 NetNewsWire，尽管该公司告诉我，他们在最近几个月大大提高了在线版本的速度。不管怎样，我只想要和需要手机上的这么多提要。Newsgator Go！还会在您的手机上本地缓存订阅源项目，这样您就可以在不上网的情况下阅读它们。

在旅途中阅读你的订阅源的其他选项包括 [Bloglines Mobile](https://web.archive.org/web/20221002161649/http://bloglines.com/mobile) 、 [FeedBurner 移动订阅源阅读器](https://web.archive.org/web/20221002161649/http://www.feedburner.com/fb/a/mfr)、多功能移动 hydra [Winksite](https://web.archive.org/web/20221002161649/http://www.winksite.com/site/index.cfm) 、Dave Winer 的用于选择订阅源的[移动新闻河](https://web.archive.org/web/20221002161649/http://www.beta.techcrunch.com/2006/08/29/dave-winer-ponders-mobile/)以及[即将推出的移动工具套件 SynapseLife](https://web.archive.org/web/20221002161649/http://www.beta.techcrunch.com/2006/08/31/synapselife-will-offer-lightweight-life-management/) 。
