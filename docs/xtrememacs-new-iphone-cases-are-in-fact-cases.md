# XtremeMac 的新 iPhone 保护套实际上是保护套

![xmaciphonecases.jpg](img/13f49268bf30b45d19d653b22f4f81e7.png)

iPhone 的所有者不可避免地希望让他们珍贵的财产看起来像他们购买时一样新鲜，但我们都有不同的品味，所以 XtremeMac 发布了 Onyx，SportWrap 和 TuffWrap 来满足他们。

[缟玛瑙](https://web.archive.org/web/20210124035838/http://www.xtrememac.com/cases/iphone/onyx/index.php)专为专业人士打造，配有精美的黑色皮革和白色缝线，还有一个一体式皮带夹，向董事会中的每个人展示你是一个~~工具~~老板。 [TuffWrap](https://web.archive.org/web/20210124035838/http://www.xtrememac.com/cases/iphone/tuffwrap/index.php) 为你的 iPhone 提供双色硅胶避孕套，兼具耐用性和时尚感。健身房的老鼠会喜欢氯丁橡胶臂带 [SportWrap](https://web.archive.org/web/20210124035838/http://www.xtrememac.com/cases/iphone/sportwrap/index.php) ，这样他们可以在健身房为女士们展示他们的 bi 和 tri。

Onyx 和 SportWrap 的预购价格为 29.95 美元。TuffWrap 也可预购，价格仅为 19.95 美元。预计 8 月份将在商店里看到这三款产品。
