# 专利猴:索尼起诉蓝光技术

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/25/patent-monkey-sony-sued-on-blu-ray-technology/>

Target Technologies 对索尼提起专利诉讼，声称其蓝光技术侵犯了 Target Technologies 的一项专利，该专利涉及索尼在其蓝光光盘中使用腐蚀性更低、成本更低的银基合金层。

Target Technologies 拥有一系列相关专利,可追溯到 1998 年 6 月的申请，涵盖了金属合金在光存储光盘反射层中的应用。索尼知道 Target Technologies 的专利吗？

索尼在光盘技术上几乎拥有 400 项 T4 专利。虽然 CNet 报道称索尼尚未对此案发表评论，但索尼确实有一项 [Target Technologies 专利](https://web.archive.org/web/20151002050156/http://www.patentmonkey.com/PM/patentid/6007889.aspx)引用了其一项[专利](https://web.archive.org/web/20151002050156/http://www.patentmonkey.com/PM/patentid/6972076.aspx)，这可以被视为故意侵权的理由。

塔吉特百货的申请文件没有说明其专利是涵盖了迄今为止生产的所有蓝光光盘，还是仅仅涵盖了其中的一部分。
