# 哦，我的上帝，是的:手机的受欢迎程度首次下降

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/03/oh-my-god-yes-cellphones-drop-in-popularity-for-the-first-time/>

[![nocellphones.jpg](img/383bc77ad6f999facb37039d04b36b36.png)](https://web.archive.org/web/20150926204835/http://tctechcrunch2011.files.wordpress.com/2007/05/nocellphones.jpg "nocellphones.jpg")

听到了吗？这是寂静之声。(不，不是[那个](https://web.archive.org/web/20150926204835/http://en.wikipedia.org/wiki/The_Sound_of_Silence)。)有史以来第一次，人们打手机的次数比前一年少了。研究人员对使用预付费手机的人进行了标记，发现他们比去年少打了 28.5%的电话，这是他们开始追踪以来十年来首次出现这种情况。可能的原因包括人们转向移动即时消息和电子邮件(因为这些服务通常更便宜)，或者人们只是“忘记”他们可以在旅途中打电话的事实。新鲜感可能已经过去了。很好的解脱。

我真的希望这是即将到来的事情的征兆，而不是数据异常。有多少次你的电影被打断了(或者，对于已经订婚的人来说，你和你配偶的晚餐被打断了)，因为某个笨蛋决定他必须在手机上大声说话？这不仅明显不文明，而且令人愤怒不已。因此，我不去电影院，除非是 A)星期二上午 11 点或者 B)约翰强迫我去。
