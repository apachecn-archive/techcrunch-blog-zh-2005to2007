# Webmail.us 用 ajax 和更多技术重新启动企业网络邮件

总部位于弗吉尼亚州的 [Webmail.us](https://web.archive.org/web/20220811114410/http://www.webmail.us/) 今天早上重新推出了面向小企业的网络电子邮件服务，它的 ajax 功能非常强大，使用起来非常流畅。它并不适合所有人，而且有一些局限性，但是对廉价的企业 webmail 感兴趣的小企业可能会发现他们的用户真的很喜欢它。搜索、ajax 和轻量级 RSS 集成是该产品的优势。

Webmail.us 自 1999 年就已经存在，但在 2003 年决定专注于电子邮件外包。该公司去年从私人投资者那里获得了大约 50 万美元的资金。现在，在 2006 年，该公司有了真正的 Web 2.0 的感觉，除了新的 ajax UI 之外，还有一个活跃的公司博客和频繁的软件更新。

该产品很简单，易用性是主要目标，但它也有一些很酷的功能，使其有趣。例如，在你的收件箱中点击一封邮件的标题，它会立即在一个 ajax 框中打开，感觉就像一个桌面应用程序。如果你喜欢 GMail 的 ajax 风格，你可能会更喜欢 Webmail 平台，它有非常流畅的感觉。网站上设置了[一键演示账户，你可以自己去看。](https://web.archive.org/web/20220811114410/http://www.webmail.us/) 

搜索可以限制在单个文件夹或所有项目中。可以轻松搜索各种字段，并且只能搜索包含附件的项目。拼写检查有一个很好的感觉和一个方便的恢复按钮。

日历、联系人和任务列表都集成在一起。你也可以订阅 RSS 提要，尽管现在提要阅读器还非常轻量级。它可能适合小型企业使用 RSS，但我希望它支持 OPML 进出口(更新:该公司说这是受支持的)和新闻视图的河流。

在如何组织你的电子邮件方面，Webmail.us 也相当有限。没有标记或标签，我甚至不清楚如何创建新文件夹。也许搜索的目的是充分满足这些需求，同时保持东西整洁。

支持与其他电子邮件系统的集成，该公司表示，他们有可怕的垃圾邮件过滤。我还没能测试出来。

每个邮箱的标准存储计划从 100mb 到 5GB 不等。价格低廉，从网上的证词来看，该公司的产品在重新推出之前满足了许多小企业的需求。肯定值得一看。理查德·麦克马努斯昨天写了一篇关于 webmail 启动空间的精彩概述，其中也包括 Webmail.us。
