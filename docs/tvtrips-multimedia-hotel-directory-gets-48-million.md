# TVtrip 的多媒体酒店目录获得 480 万美元| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/07/11/tvtrips-multimedia-hotel-directory-gets-48-million/>

[![tvtriplogo.png](img/233851b4094ea3bd7320401d4cad1a1c.png)](https://web.archive.org/web/20170430222801/http://www.tvtrip.com/)[推出](https://web.archive.org/web/20170430222801/http://www.techcrunch.com/2007/06/07/tvtrip-the-second-video-guide-for-hotels/)一个月后，TVtrip 获得了 Benchmark 欧洲分公司 Balderton 和 Partech 的又一轮融资，将于明天宣布。除了融资，他们还扩大了酒店视频目录，并在董事会中增加了一些旅游高管。

他们的名录已经超越了伦敦和巴黎，现在包括德国、意大利和西班牙。Erik Blashford(terra pass 首席执行官，Expedia 前首席执行官)、Thierry Antinory(汉莎航空营销和销售副总裁)加入了顾问委员会。

Trivop 还在开发一个酒店多媒体指南。
