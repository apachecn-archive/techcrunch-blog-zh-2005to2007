# Photobucket 分发自定义 Flock 浏览器

 [Photobucket](https://web.archive.org/web/20220813214340/http://www.photobucket.com/) ，一个[非常受欢迎的](https://web.archive.org/web/20220813214340/http://www.beta.techcrunch.com/2006/06/22/photobucket-vs-flickr-in-alexa-and-technorati/)照片分享服务，现在正在发布一个[定制版的](https://web.archive.org/web/20220813214340/http://photobucket.com/flock) [Flock](https://web.archive.org/web/20220813214340/http://www.flock.com/) Mozilla 浏览器。此处的是对该浏览器照片功能的一次浏览。这距离 Flock 第一个公开测试版的发布还不到一个月。

Flock 浏览器的 Photobucket 版本与 Flock 的版本完全相同，只是 Flickr 的功能被去掉了，Photobucket 成为照片整合的唯一选择。交易的细节没有公布(事实上，交易本身没有公布)，但我假设 Flock 正在与 Photobucket 分享该浏览器用户产生的搜索和其他收入流。基于未经证实但流传甚广的关于火狐搜索收入的传言，两家公司预计每年每个用户可从合作中获得 2-3 美元。

鉴于 Flock 在其浏览器的主要版本(del.icio.us、Flickr 和雅虎搜索)中与雅虎服务的紧密结合，我不会对很快看到雅虎版本的 Flock 感到惊讶。
