# iProjector 连接 iPod

第五代 iPod 最棒的功能之一是它可以将视频输出到电视上。这对旅行者来说很方便，因为他们可以在旅途中看几部电影，甚至是一季他们最喜欢的电视节目，并在酒店房间里自娱自乐，而不必支付 12 美元，也不必向老板解释“Ex Trek: the Next Penetration”。

他们可以简单地将他们的~~色情~~科幻爱情 DVD 放入他们的电脑，使用压缩程序(像 [M2 转换](https://web.archive.org/web/20201125031503/http://crunchgear.com/2006/12/11/reader-response-week-three/)，我们目前正在分发大量的)将视频转换成兼容格式，带上一些 AV 电缆，他们就搞定了。但是如果你不是一个漫步者(对不起，漫步者*人*)，还有其他选择。Ion Audio 的 iProjector 是一款相当普通的视频投影仪，除了一个值得注意的资产:一个 iPod 基座。投影仪以 800×600 像素的分辨率播放 iPod 中的视频，比 DVD 质量要好。

虽然有更高分辨率的投影仪，而且大多数用户的数字图书馆是其他格式的，但 iProjector 是一种保持简单的敏锐方式。

没有关于价格或可用性的消息，但在 1 月份的 CES 上正式推出该产品后，一切都将变得明朗。
