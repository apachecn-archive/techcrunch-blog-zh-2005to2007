# 松下 D-Snap SD850N DAP:是的，它可以播放音乐

[![pana1.jpg](img/8969b51b00f3bebcd6032dfba999c30f.png)](https://web.archive.org/web/20130628190609/http://old.crunchgear.com/wp-content/uploads/pana1.jpg "pana1.jpg")

如果你仔细观察，你会发现 Pansonic 已经更新了数字音频播放器的 D-Snap 系列。新上市的产品是 SD850N，这是一款基于 SD 卡的 DAP，附带 1GB 卡。就像它的祖先 SD800 一样，850N 具有噪音消除功能，可以阻挡高达 83%的不想要的外部噪音。很好，因为当我听 MP3/WMA/AAC 的时候，我不想听到纽约的噪音。

电池似乎也做了一些工作。当噪音消除关闭时，它可以持续长达 80 小时(打开时为 60 小时)，并且有一种特殊的省电充电模式，据说可以延长电池的长期寿命。如果你还不知道，最新的 D-Snap 就是“我也是！”真的只是烦。我敢打赌，你的手机现在播放 MP3。就用那个代替吧。
