# Zvents 完成 700 万美元的事件搜索

 [](https://web.archive.org/web/20220522103158/http://www.zvents.com/) 当地活动管理和日历供应商 [Zvents](https://web.archive.org/web/20220522103158/http://www.zvents.com/) 今天宣布完成总额为 700 万美元的首轮融资。这一轮由 VantagePoint Venture Partners 领投，包括 Red Rock 和 NetService Ventures。

Zvents 将强大的产品与 API、非常强大的团队和令人印象深刻的客户列表结合在一起。他们正在将自己树立为特定网络服务领域的领导者——他们本质上是一个 B2B 日历和事件小部件提供商。我认为相关基金进行了明智的投资。

Zvents 与其他媒体公司合作，提供一个 Ajax 丰富的活动日历，嵌入到他们的网站中。明年 1 月，Zvents 将推出其新的本地广告网络的测试版，进一步将它正在建立的媒体合作关系货币化。该公司的展示客户包括圣何塞水星新闻，迈阿密先驱报和丹佛邮报。Zvents 表示，它将很快推出一款免费的日历产品，供场馆、活动发起人和组织使用。该公司目前提供一个商业 API，并把初创社交网站 Maya 的妈妈作为客户。主要竞争对手 Trumba 最近才整合了 Ajax，并宣布停止对其免费和小型发行商产品的支持。Trumba 的客户包括《纽约时报》在线汽车版。

Zvents 还宣布来自 VantagePoint Ventures 的大卫·卡里克将加入公司董事会。卡里克是 DoubleClick 的联合创始人，也是 Ask 和 MySpace/Intermix 的投资者。卡里克将与电子艺界的联合创始人里奇·梅尔蒙(Rich Melmon)和部落网络(Tribe Networks)的创始首席技术官保罗·马蒂诺(Paul Martino)一起加入 Zvents 董事会。雅虎前首席科学家戈登·里奥斯说。搜索，已被聘为 Zvents 首席技术官。

该公司将于下个月在 Boston.com 推出服务，并于明年在美国和加拿大的前 50 个城市推出服务。我们之前对 Zvents 的报道是这里的。
