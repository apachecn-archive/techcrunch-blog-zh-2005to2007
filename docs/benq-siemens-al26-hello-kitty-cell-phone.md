# 明基-西门子 AL26 凯蒂猫手机

这款手机唯一重要的特点是它是一款 Hello Kitty 手机。所有其他的东西，像 130×130 的分辨率，32 和弦铃声，嗯，就是这样。除了和猫咪眉来眼去之外，手机没有太多的用处，这对凯蒂猫爱好者来说很好。

这款手机将于本月开始发售，最有可能是在日本。但是如果你真的喜欢凯蒂猫，你已经习惯了从那里进口你所有的商品。
