# 威瑞森和又大又胖的 RAZR 一起去 MAXX

无处不在的 RAZR 手机，推出快两年了，是不是越来越老了？你会让它退役，希望人们会喜欢它的继任者 KRZR 吗？不，你拿着以苗条著称的纤薄手机，让它变得更胖。

威瑞森显然认为 RAZR 品牌还有足够的生命力，可以为我们带来另一个变种，RAZR MAXX。顾名思义，这是 RAZR 系列的高端产品。抛开公认的假屏幕不谈，从各个方面来看，这款手机都要更大:它配备了一个带闪光灯的 200 万像素摄像头，microSD，EV-DO，“带有优雅灯光的未来触摸音乐键”，以及蓝牙(威瑞森风格，当然)。此外，如果它像它的 GSM 表亲一样，寻找双向视频会议。

我们知道 GSM 版本已经有一段时间了，它认为随着他们新的以 Z 为中心的字母主题手机的出现，我们当地的 GSM 运营商对这种野兽的布局一直犹豫不决。但是威瑞森把谨慎抛到了九霄云外，所以期待这个大家伙在年底前落马。
