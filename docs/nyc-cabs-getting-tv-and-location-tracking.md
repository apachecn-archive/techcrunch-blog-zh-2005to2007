# 纽约出租车获得电视和位置跟踪

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/18/nyc-cabs-getting-tv-and-location-tracking/>

![](img/903b33b170e398758dd023bc1f3aeb4d.png)

如果你像我们中的一些人一样住在纽约，你会知道出租车并不令人兴奋。当你烂醉如泥的时候，它们只是从一个地方到另一个地方的快速交通工具(嗯，至少有时候是这样)。然而，这一切都将改变，多亏了该市出租车和豪华轿车委员会上周展示的新触摸屏。

这款触摸屏将在未来几周内首次亮相，提供新闻、天气预报、餐馆评论，并显示您的出租车到达目的地的当前进度。那些在出租车上丢失个人物品的人可以打电话给出租车公司，更快地找到他们的物品。多亏了新系统，你也可以开始用信用卡支付了(差不多是时候了)。如果设备发出的声音变得令人讨厌，您可以将它静音，以便平稳地穿越市区。纽约人应该感谢上帝，并在假期留意这些。
