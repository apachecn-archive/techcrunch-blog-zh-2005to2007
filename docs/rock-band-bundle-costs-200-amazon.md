# 摇滚乐队套装售价 200 美元:亚马逊

[![rockband200a.jpg](img/05ad13ca553f698bbd0c6d7ca5cd8d72.png)](https://web.archive.org/web/20210119155504/http://old.crunchgear.com/wp-content/uploads/rockband200a.jpg "rockband200a.jpg") Anyone who doesn’t like Def Leppard can bite me, no disrespect

我和下一个家伙一样喜欢*吉他英雄*(虽然我[听说](https://web.archive.org/web/20210119155504/http://www.1up.com/do/reviewPage?cId=3161422)那个 [*摇滚 80 年代*](https://web.archive.org/web/20210119155504/http://crunchgear.com/2007/05/10/guitar-hero-rock-the-80s-comes-out-this-summer-for-the-ps2/) 有点让人失望)，但不要认为我会仅凭名字就盲目购买 [*吉他英雄 3*](https://web.archive.org/web/20210119155504/http://crunchgear.com/category/guitar-hero-3/) 。不，我已经盯上了 [*摇滚乐队*](https://web.archive.org/web/20210119155504/http://crunchgear.com/category/rock-band/) ，尽管如果亚马逊列出的价格——游戏、鼓、麦克风和吉他的价格为 200 美元——是真的，我现在可能不得不开始攒钱了。

我看过一些游戏的视频，但我不知道文斯扮演李尚义·罗斯时假扮萨米·哈加尔是否真的值那么多钱。当然，这些价格只不过是占位符，但是当它在假日购物高峰到来时，你可以期待游戏的零售价格在 200 美元左右。谁知道呢，也许我们会把玩游戏时表现得像个废物的视频贴出来让大家欣赏。
