# SimpleTech 500GB 外置硬盘动手操作

![img_6493.jpg](img/fbf2d801d64a7502faca596b099b6b8b.png)

好的硬盘有各种形状和大小，从受 IT 人士欢迎的[实用的四四方方的型号](https://web.archive.org/web/20210301134755/http://crunchgear.com/2006/09/08/maxtor-fusion-hands-on/)到艺术家和诗人用来存储他们的华丽作品的[花哨的便携式装置](https://web.archive.org/web/20210301134755/http://crunchgear.com/2007/01/02/stackable-hard-drives-lego-styling/)。SimpleTech 500GB 是这两者的结合:小而不显眼，足以被忽略，但其造型令人尖叫“我参加临界质量骑行，明年可能会去火人节。”

![img_6496.jpg](img/a9ab099db8a91f7420dbe34ba1088c27.png)

该驱动器使用壁式电流，并通过 USB 2.0 连接。顶部有一个启动备份的大按钮，它还包括一个 2GB 的 MyFabrik 帐户，用于将一些存储卸载到 myFabrik 独特的共享系统。

这款硬盘长约 7 英寸，全球闪存和高清设备制造商 SimpleTech 委托宾尼法利纳设计这款硬盘，也就是法拉利的设计师。很别致，是吧？

![drivestuff.gif](img/1bdde4ac0a8a4082915778024f97fb97.png)
内容可能已经转移。

令人惊讶的是，该驱动器附带了 Mac 和 PC 备份软件。Mac 版本需要一个小助手应用程序来操作顶部的按钮，但据我所知，这两个应用程序都来自 ArcSoft，是一样的。它是 NTFS 格式的，对 Mac 用户来说没什么用，但对 Windows 用户来说绝对是个好东西。

![arcsoft1.gif](img/7da2f6c364e6bd88f38376b47e87c839.png)
热 FR33 WAR3Z！

在我的 Mac Pro 上安装很简单——我只需运行 TotalMedia backup 安装程序，然后运行按钮助手。我不得不重新开始，这很痛苦，但我没有哭就熬过来了。

![img_6499.jpg](img/a03d9aa6b37a1d2e1836dc872c4bc64e.png)

开车其实很安静。我身边有三个硬盘，SimpleTech 设备是所有设备中最安静的。500GB 的价格约为 200 美元，160 GB 和 250GB 的较小型号分别为 99 美元和 120 美元。对于这样一个强大的驱动器来说绝对不坏。

驱动器，正如他们所说的，都是一样的。然而，有些人——包括 SimpleTech 会更加努力地去做一个好看、坚固的产品。
