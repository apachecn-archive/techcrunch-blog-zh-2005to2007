# 东芝在桌面 Linux 派对上插了一脚

![toshlin.jpg](img/ec3a12a53aa1d5921db56396cd49e54b.png)当然，今天早些时候，当互联网上出现报道称[戴尔可能会在今年晚些时候在消费者台式机和笔记本电脑上预装 Ubuntu Linux](https://web.archive.org/web/20130628211059/http://crunchgear.com/2007/05/01/dell-linux-round-2/) 时，涟漪就触及到了 Linuxverse。桌面 Linux 的想法是大多数 Linux 爱好者的想法，这正是它需要占据的运动类型。除了戴尔，[摩托罗拉也在为将于今年晚些时候上市的 Linux 智能手机版本](https://web.archive.org/web/20130628211059/http://crunchgear.com/2007/05/02/moto-rokr-e6-linux-based-smartphone-gets-screen-time/)打基础。不要被排除在外，东芝的 Luigi Cattaneo 最近表示，他希望看到 Linux 预装在其笔记本电脑上，因为这样的开源设备可以很容易地达到 700 美元以下。这并不意味着东芝必然会朝这个方向发展，但是如果东芝意大利公司的负责人正在考虑这个问题，就桌面选择而言，Linux 比一年前领先了一步。

但真正的问题是:消费者需要(或者说，确实想要)第三种选择吗？或者说 Linux 是操作系统的拉尔夫·纳德？操作系统历史 101 将在 2007 年变得有趣。
