# Wii 组件电缆实践

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/02/wii-component-cables-hands-on/>

![](img/34b513ad5ba1b3730a79c2ac5e420fd6.png)

几天前，我们在游戏[截图](https://web.archive.org/web/20150414170300/http://crunchgear.com/2006/11/28/nintendo-wii-graphical-difference-with-component-cables/)中看到任天堂 Wii 使用了分量电缆而不是捆绑的复合电缆。今天，我的 Wii 组件电缆终于到了，所以我自己能够看到不同之处。正如我在之前的帖子中提到的，我在使用*《塞尔达传说:暮光之城公主》*时遇到的最大问题是模糊的文本，这在游戏中有很多。我决定去一个有一些文本的场景，并使用复合电缆和分量电缆进行截图，以显示两者之间的差异。真的很难在图片中显示出这些电缆的区别，当你玩游戏时，这一切都在运动中。不过，使用 Wii 组件电缆，你将能够看到更清晰的线条，更好的显示文本和整体更好的显示效果。我的建议是，坚持使用任天堂便宜的 29 美元，远离 EB Games/GameSpot 或 Best Buy，它们都试图以 50 美元或更高的价格出售 Wii 组件电缆。对比如下。
