# 闪迪 Sansa 连接 Wi-Fi

SanDisk 展示了另一款 MP3 播放器，配有 2.2 英寸屏幕、照片播放器、内置扬声器、通过互联网和 USB 2.0 的社区连接。wi-fi 连接基本上与 Zune 的 wi-fi 功能相同。你可能知道也可能不知道，但这最初来自今年早些时候的 [Zing](https://web.archive.org/web/20210225034822/http://reviews.cnet.com/4531-10921_7-6535687.html) 。
