# 天狼星和 XM 今天合并？

![](img/e5015c583cc5b780beb3e642b3cf8e2e.png)

《纽约邮报》正在传播一些谣言，说 XM 和 Sirius 今天将合并，创造一个几乎势不可挡的卫星广播巨头。这笔交易将把 XM 和 Sirius 合并成一个拥有 1200 万用户的网络，让 Sirius 走出一个大窟窿。这两家公司在提供优秀内容的同时，现在都在学习旧媒体几年前学到的东西:如果你建立了它，就不能保证他们会来。天狼星受到的打击最大，也是获益最多的。

我个人会赞成合并。我只听 XM，加入天狼星阵容会让我更开心。我们会看到的。
