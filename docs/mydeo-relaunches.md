# Mydeo(重新)推出，谢天谢地，这不是另一个 GooTube 的崇拜者。

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/24/mydeo-relaunches/>

Mydeo 是一项在线视频流媒体服务，主要面向消费者和/或中小型企业，这可能会被你忽略。今天，他们推出了一项新的重新设计的服务，承诺提供更好的用户体验，一个[新的定价](https://web.archive.org/web/20130920051559/http://www.mydeo.com/content.asp?CID=65)模型和更好的性能，但让我们清楚的是，Mydeo 是**而不是**另一个 gootube 的崇拜者。

与 YouTube 不同，Mydeo 可以让你自由上传你选择的任何时长和分辨率的视频，更好的是，它们允许你在上传过程中不进行压缩或转码。

与 YouTube 和大多数其他在线视频网站不同，Mydeo **没有对你的内容主张任何版权或发行权，所以你不会发现你公司圣诞节派对上的尴尬视频在未来被数千个博客网站重播。**

与 YouTube 不同，Mydeo 上没有广告，也不允许成人内容，因为 Mydeo 是互联网观察基金会的积极成员。

与 YouTube 不同，Mydeo 不是 T21 的社交网站，人们不会访问 Mydeo 来观看你的视频或分享他们的评论。相反，Mydeo 只是让你能够选择人们将如何消费你的视频内容，要么通过给他们发送一个电子邮件链接，要么通过一个嵌入的链接放在你的博客上。

在这两种情况下，无论用户想要什么样的分辨率，内容都可以通过 windows media player 快速播放。

![Mydeo Screenshot](img/33b68efe7003a97bd9ba2ab49d2dc13b.png "Mydeo Screenshot")

那么谁会用我的视频呢？Cary Marsh (Mydeo 的 MD)告诉我，“目前大约有 70%的消费者和 30%的小型企业。”他们的会员统计数据同样有趣。显然，社交网络迎合的是年轻受众，但 Mydeo 的目标市场反映了他们的会员最多的地方；在 30-55 岁之间，高峰期在 30-35 岁左右，也就是人们有了孩子并购买了他们的第一台昂贵的摄像机。凯里补充说，他们的会员遍布 180 多个国家，现在他们 90%的收入来自英国以外。

出于多种原因，Mydeo 是我观察名单上的公司之一。我已经追踪他们超过 10 个月了。我认为他们有潜力成为英国成功的典范，原因有很多。首先，Mydeo 非常聪明地通过获得 DTI 技术创新拨款来筹集种子资金。这使他们在谈判第一轮风险投资时处于非常有利的地位，以扩大他们的盈利商业模式。

其次，当一家公司开始开发自己的虚拟生态系统时，这是一个明确的迹象，表明该产品已经达到了一个关键的临界点。目前，Mydeo 是唯一一家与 T2 Windows movie maker 成功合作并整合视频流媒体服务的欧洲公司，最近，T4 Muvee technologies 整合了他们的第三方视频编辑软件。此外，Tiscali 和 Orange broadband 等 ISP 最近也与 Mydeo 建立了联合品牌网站。

最后，看来我不是唯一一个把我的视频放在监视名单上的人。看起来卡里有望获得今年的“It 女性”奖。颁奖典礼将于 2006 年 11 月 2 日星期四在伦敦旧比林斯盖特举行。让我感到有点奇怪的是，尤安·森普尔是全部由女性组成的评委团中唯一的男性。尤安，你是怎么得到那份工作的，为什么？无论如何，祝卡里获奖和重新发行好运。
