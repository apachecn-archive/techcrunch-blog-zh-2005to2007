# Ustream.tv 现在具有更多交互性

[![](img/507211f42b8451c7f8c5be4c363f935e.png)](https://web.archive.org/web/20230301091939/http://www.crunchbase.com/company/ustream) 本月早些时候，我们[攻克了](https://web.archive.org/web/20230301091939/https://techcrunch.com/2007/06/22/will-live-video-catch-on-like-youtube#comments)不断发展的在线视频直播领域。与录制的视频相比，直播的原始形式是一种非常不同的格式。没有后期制作编辑来删除你的鱼钩或捕捉最好的部分。将最有趣的内容实时推送到观众面前是一个挑战。但是有观众可以互动，这就很重要了。

Ustream 今天发布的第三版重新设计完全着眼于观众和主持人的关系。他们为该平台增加了三个新的互动功能:喊表、实时投票和视频存档小工具。他们都是这项服务的可靠补充。

**喊计**

![shoutmeter.jpg](img/2d3d459db0f36eda0aef8ddcdd37019a.png)新的呼米旨在解决为观众带来最佳内容的问题。挑选最好的内容是一个严重的问题。Ustream 表示，他们每天制作超过 3000 小时的节目，并提供 25000 小时的视频内容。

呼叫器允许用户通过在侧边栏上大力点击来为当前频道的内容投票。来自更多用户的更多点击推动了欢呼栏，提示 Ustream 在频道上发生了一些好事。右边的横条反映了你自己的呼喊水平(红色)和人群(黄色)。没有消息表明他们将如何使用计价器数据。

这是对 Justin.tv 的 Tips 功能的又一次尝试，该功能允许观众投票选出直播视频流的片段。在所有 publisher 页面上可用的置顶列表中组织了置顶提示。

**现场民调**

Ustream 已经将现场投票整合到他们的广播控制台中，允许发布者实时向观众提出一些问题。

投票是在流的透明层上，所以你仍然可以看到节目。观众可以点击他们想要的答案，或者关闭盒子选择退出。

这是一个更严格的版本，简单地向聊天室提出一个问题，或者向观众征集电子邮件。

**档案图库模块**

![archive.jpg](img/f67efdeae1e26f396ef02c2c02a0a7c7.png) Ustream 不仅制作直播视频，还维护着一个过去视频的存档库。他们的新存档工具将让观众在不影响当前视频的情况下浏览旧节目。归档内容以前仅存储在普通页面上。

我猜 Ustream 将会在这个模块中加入“呼喊”功能，以帮助根据日期和兴趣过滤内容。

直播视频网站之间的演变已经变成了一种军备竞赛，因为网站寻求确定正确的外观、感觉和功能，以吸引最好的出版商和观众群。直播视频仍然不是主流，即使是像 Stickam 这样在 YouTube 繁荣时期存在的网站。我想，随着 live 在在线视频领域找到自己的位置，未来几个月我们会看到更多的创新。
