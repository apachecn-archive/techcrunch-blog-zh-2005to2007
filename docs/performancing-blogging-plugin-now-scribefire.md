# 表演博客插件现在 ScribeFire

 [](https://web.archive.org/web/20220818202742/http://www.scribefire.com/) 本周早些时候，当 [PayPerPost 宣布收购](https://web.archive.org/web/20220818202742/http://www.beta.techcrunch.com/2006/12/27/payperpost-in-the-news-again/)大部分[表演](https://web.archive.org/web/20220818202742/http://www.performancing.com/)资产时，很多人都在问这个广受欢迎的表演博客插件的未来，这个插件并没有被收购。

它还活着，并且已经被重新命名为 [ScribeFire](https://web.archive.org/web/20220818202742/http://scribefire.com/) 。该网站有一个登陆页面，将很快推出。与此同时，旧插件可以从 Mozilla 网站下载。

ScribeFire 表示，已经有 40 万人下载了该软件。没有关于有多少活跃用户的消息，但是很多博主对此深信不疑。
