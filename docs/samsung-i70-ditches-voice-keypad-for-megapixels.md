# 三星 i70 放弃语音，百万像素键盘

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/22/samsung-i70-ditches-voice-keypad-for-megapixels/>

三星今天推出了性感的 VLUU i70。配备 700 万像素照相/摄像机和 3 倍光学变焦、3 英寸外置 LCD、超薄设计、滑动外形和 HSDPA 3G 连接，会让任何极客垂涎三尺。但这不是电话，朋友。这是一款直射式全功能数码相机。

凭借其集成的调制解调器，i70 拍摄的照片可以直接从存储卡无线上传到网络画廊(或博客)。此外，ISO at 1600 意味着许多通常需要闪光灯的情况不再需要。因为它使用 HSDPA，所以需要一张 SIM 卡，而且它有接收短信的功能。真的。

除了那些令人眼花缭乱的数据之外，它看起来真的非常非常糟糕。我们自己的布莱克有一部类似风格的三星 point n ' shoot，他曾说这是他用过的最好的相机。我觉得赌这个美女安全。
