Web 2.0峰会上的启动平台13

原文：https://web.archive.org/web/http://www.techcrunch.com:80/2006/11/07/the-launchpad-13-at-web-20-summit%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20%20/

一年一度的 Web 2.0峰会今天在三藩市的皇宫酒店拉开帷幕。几个月前已经售罄的会议峰会明显比去年更大，数百人在周围转来转去，参观和被参观。

对我来说，去年会议的亮点是LaunchPad，13家年轻的初创公司向观众展示了他们的东西。参见我们去年的报道这里和这里。这些公司中的许多都做得非常好。只有一个人 Pubsub 进入了TechCrunch DeadPool 。

今年的LaunchPad可能比去年更具竞争力。200多家公司申请出席会议。只有13人被接受，每个人有5分钟的时间向观众展示他们的产品。我们在下面总结了各自发布的内容。

3B

3B 基于用户的输入创建三维浏览环境，用户然后可以创建化身来探索空间、邀请朋友、与他们聊天等。这里的一个演示是。我在笔记上写了“漂亮，有趣，完全没用。”经过进一步思考，我可以看到人们出于各种推广目的创造了一个3B，但它无法与第二人生争夺虚拟现实的注意力。

Adify

Adify 允许任何人创建垂直广告网络——他们的软件处理所有的细节，并允许出版商决定价格、广告类型和监控广告指标。他们的目标是帮助大约50万广告客户在他们想要的网站上实现定向广告的愿望。《华盛顿邮报》在他们的Blogroll平台上使用Adify。

说明是一个迎合“自己动手的人”的网站，包括许多创建东西的逐步说明。该公司称，有3万人经常使用该网站。今天，他们宣布推出新的协作功能。在不久的将来会有更多的介绍。

在椅子上

in the chair最恰当的描述是把乐器练习变成了电子游戏。这是一个网络应用程序，让音乐家在专业音乐家的录音旁边练习他们的乐器。通过将麦克风连接到你的电脑上，该应用程序可以向你提供实时反馈，告诉你你对他们曲库中任何歌曲的音符和节奏的掌握程度。用户还可以记录会话，并与他们的老师、祖母等分享。

oDesk

看我们之前覆盖的 oDesk这里。oDesk为寻找有才华的编程承包商提供了一个在线市场。他们于2006年5月启动。

我们已经报道 Omnidrive 将近一年了……他们今天在大会上宣布了他们的发布。他们有一个在线、windows和mac客户端，允许你无缝地将文件从你的计算机存储和同步到你的在线驱动器。他们的开放API也允许更专业的实现。参见我们之前的报道。

球体

我们已经关注博客搜索引擎Sphere已经将近一年了。今天他们为博客发布了Sphere It。TechCrunch已经将Sphere It整合到我们的模板中，并且在每篇帖子的末尾都包含了一个链接。单击它可以查看上下文相关的博客帖子和与故事相关的其他信息。

StikkitT23】

Stikkit ，即“便利贴的数字等价物”，于今日发布——参见TechCrunch UK，了解更深入的报道。用它来快速记笔记，Stikkit会对你写的内容做出明智的决定，并据此采取行动。提到一个名字和联系信息，它会把它存储在一个地址簿中。事件信息存储在日历中。Stikkit还支持标记和其他分类结构。我们会跟进斯蒂基特的情况。

Venyo

Venyo 推出了一款声誉管理应用，其中有一些有趣的变化。还没有太多细节。

时间桥

time bridge与Outlook 2003集成，帮助企业捆绑轻松的会议协调和项目协作。通过发送电子邮件邀请来协调会议，建议会议时间与每个人的时间表相匹配。与会者选择他们的最佳时间，Timebridge为该组寻找最佳时间。他们从梅菲尔德基金和Norwest Venture Partners获得了600万美元的资金。

转

Turn是一个新的广告网络，基于CPA或按行为计费，这意味着广告商只在某些特定行为发生时付费，如销售或用户注册。Turn计算一个行为实际发生的频率，考虑每个行为的付费，并相应地在网络中分发广告。看起来这背后技术含量很高，执行团队也很强大。

Klostu 今日推出。它利用了参与Boardscape留言板的3亿人，允许他们在所有留言板上创建单一登录和身份。将每个留言板变成一个社交网络。我们将对此进行更深入的研究。伟大的设计。
