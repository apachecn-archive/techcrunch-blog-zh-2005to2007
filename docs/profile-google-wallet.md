# 简介:谷歌钱包 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/06/20/profile-google-wallet/>

**公司:**谷歌(钱包)

![](img/731292d5e166230ff141259dff36257a.png)

**什么事？**

这项服务还没有推出，但是在博客世界(和华尔街日报)已经有一些传言，谷歌计划推出一项与 PayPal 竞争的支付服务。

这是 Web 2.0 吗？我*认为*如此。如果是 p2p，我想应该是 P2P，那么它肯定是 P2P。

《华尔街日报》文章:*“对于总部位于加州山景城的谷歌来说，支付服务可能代表着在线广告之外的重大扩张，在线广告占其去年 32 亿美元收入的 99%。在线支付提供商通常会从每笔交易中收取佣金。*

谷歌还有一座小山要爬——贝宝拥有 7200 万用户(贝宝在第一季度为易贝贡献了 2.331 亿美元，占其收入的 23%)。然而，他们之前已经通过搜索(v. everyone)、广告(v. overture)和电子邮件做到了这一点，尽管不是通过社交网络(见[orkut](https://web.archive.org/web/20210804035730/https://www.orkut.com/Login.aspx)——你上次登录是什么时候？)

Gary Price [指出](https://web.archive.org/web/20210804035730/http://blog.searchenginewatch.com/blog/050617-204801)谷歌支付公司[是由谷歌在几个月前创建的:*“我读了这篇文章后做了一点研究(喜欢那个深度网络(-)，发现两个月前谷歌向加州州务卿提交了文件，成立了谷歌支付公司。有意思。”加里还注意到有人注册了“*](https://web.archive.org/web/20210804035730/http://kepler.ss.ca.gov/corpdata/ShowAllList?QueryCorpNumber=C2742364&printer=yes)*[googlewallet.com](https://web.archive.org/web/20210804035730/http://www.whois.sc/googlewallet.com)”的*。

随着事态的发展会有更多的报道。
