# 去网站，相信我

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/13/q-bean-go-to-the-website-trust-me/>

Q-Bean 是一种无线音频传输设备，由一个兼容 Windows/Linux/Mac 的加密狗和一个小豆状的东西组成。它是如何工作的？你插上加密狗，它开始向 bean 传输，你甚至可以把它当成麦克风。你可以通过一个连接在豆子上的耳机来听声音。根据该网站的说法，Q 豆很酷，你可以把它夹在腰带上。它使用:

> LTB 专有的 ARIO 无线技术，这是一种真正的数字未压缩技术，可以解决 2.4 GHz 无线频段经常遇到的干扰问题。这种先进的射频技术提供了一个强大的非压缩 CD 质量的音频信号，具有高达 150(视线)的感觉范围，并使用安全的配对同步来确保一对一的连接。ARIO 无线技术类似于蓝牙，但适应性也很强，不会干扰其他媒体设备。这种先进的无线技术将提供零延迟的语音通信。

所以这有点像迷你耳机。而且是无线的。而且它的网站很讨厌，可能会引起佝偻病。现在售价 129.95 美元。单声道版本售价 119.95 美元，但你为什么想要单声道版本呢？
