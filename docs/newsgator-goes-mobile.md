# NewsGator 走向移动| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/19/newsgator-goes-mobile/>

![](img/c7ac63d082537952ac6f27e7ca70852d.png)

NewsGator 明天将宣布增加一个新产品，可能会引起轰动。NewsGator Go！是一个 RSS 聚合器，设计用于 Windows Mobile 设备。它将使用户能够在旅途中方便地阅读内容和管理 RSS 订阅。

新版本能够将文章和订阅与现有的 NewsGator 程序同步，如 NewsGator Online、Inbox、NetNewsWire 和 FeedDemon。使用 NewsGator Online，用户可以向特定的设备发送特定的订阅源，例如，可以设置移动设备只接收最重要的订阅源。它还可以格式化提要，使其非常适合移动显示。

NewsGator Go！for Windows Mobile 将与任何运行 Windows Mobile 2003 或 Windows Mobile 5 的设备兼容。它将于明天在 NewGator 上市，售价 29.95 美元。
