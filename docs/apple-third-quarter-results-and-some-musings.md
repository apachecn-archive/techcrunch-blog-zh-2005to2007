# 苹果第三季度业绩和一些思考

![250px-newton-img_0320_cleanup.JPG](img/7ba5f2d0964fb4493e7d0a6af252227b.png)
众所周知，[苹果公司宣布本季度收益增长 73%](https://web.archive.org/web/20210117025208/http://www.apple.com/pr/library/2007/07/25results.html)，股价上涨至 150 美元左右。他们售出了 9，815，000 台 iPods 和 1，764，000 台电脑，这相当于两条产品线都增长了约 30%。不算太寒酸。

然而，这说明了苹果公司和计算机行业的现状吗？显然，苹果是 MP3 播放器的老大。那是必然的。但是，笔记本电脑和台式机，更重要的是，iPhone 和它的无数竞争对手呢？

总的来说，在一个季度内拥有 100 万台电脑并不算多。2007 年，个人电脑制造商在 Q2 的出货量为[5880 万台](https://web.archive.org/web/20210117025208/http://www.tgdaily.com/content/view/32980/113/)，我们只能假设这个数字在本季度有所上升，而不是下降。这是微软赚大钱的地方。毫无疑问，苹果的收入在增长，但我担心接下来的几个季度会停滞不前。为什么？因为随着苹果的创新和世界其他地方的模仿。

说一下你对后端处理能力的要求。是的，我肯定你运行 Vista 的四核英特尔确实很酷，但 OS X 和推而广之的 iPhone OS 才是发展的方向。随着公司在桌面上转向 Linux，Vista 看起来越来越像 Gnome/KDE，一种新的技术和 UI 美学正在取代一些旧的范例。你每天都可以在 Web 2.0 中看到它——瘦客户端的回归、流畅的用户界面和解决单一问题的多种方法——你也可以在 OS X 和 iPhone 中看到它。这是一种感觉，事情在幕后和舞台上都运行良好。这是一个精确的设计，到目前为止很难建造，如果不是不可能的话。

视频

苹果将售出一百万部 iPhones。但我担心诺基亚、三星等公司会继续用更多新的 iPhone-y 设备蚕食他们的销售。手机用户界面是出了名的糟糕，iPhone 为便携式设备设立了惊人的高标准，就像牛顿曾经做的那样。

还记得你删东西的时候看到牛顿上出现的那个小尘云吗？那不是很棒吗？几乎每个人都能立即理解。这就是 Aero 应该有的样子，也是为什么越来越多的产品(例如 Office 2007)看起来像空白的石板，而不是充满图标的镜子迷宫。随着处理器能力的提高，神奇因素也在增加。这里有一个例子。当我在布达佩斯的时候，我在房间里走来走去，向人们展示 iPhone。我想向他们展示屏幕旋转，所以我把它带到一张桌子上，在纵向模式下转动它，但它不想变成横向。所以我敲了一下，突然整个屏幕都变了。整桌人都倒吸一口冷气。就像我刚刚变了一个魔术。Windows Mobile 6 或者 Symbian 都没有什么魔术，但是很快就会有了。所以祝贺你，苹果，一个伟大的季度，但你最好继续跑。每个人都在接近你。
