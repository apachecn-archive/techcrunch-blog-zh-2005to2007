# 威瑞森无线 G'zOne 类型-V 喜欢它粗糙

现在，这看起来像是 Sprint Nextel/摩托罗拉的一些好竞争。威瑞森无线“G”zone Type-V 是与基于 ip 的端到端网络解决方案供应商 ut 斯达康合作的产品，它将威瑞森典型的消费产品(200 万像素摄像头、V-Cast 支持、VZ 导航器、移动网络和完整的消息选项)结合在一个加固的蛤壳手机中，该手机符合“多项军事标准，包括暴露在超过 140 华氏度的温度、极端振动、太阳辐射、模拟每小时两英寸降雨量的风暴条件、强风和浸没在一米深的水中。”但谢天谢地，它看起来不太笨重；尺寸为 4.1 x 2.0 x 1.1 英寸(HWD)。

这款手机似乎缺少的是一键通服务，这是一个遗憾，因为这款手机看起来比其他坚固耐用的产品略时尚，而且肯定具有将您从工作场所带到家中使用的功能。还有，这个东西到底怎么发音？Gizz One？g 区？天呐。

不管怎么说，这款手机将于明天在威瑞森无线的网站上以 299.99 美元的价格上市，并签署了一份新的两年客户协议。
