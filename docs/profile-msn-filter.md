# 个人资料–MSN 过滤器–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/09/profile-msn-filter/>

## 概述:

[MSN Filter](https://web.archive.org/web/20210612154934/http://filter.msn.com/) 是微软最新发布的 MSN 产品。Filter 是一个类似于 [Weblogs，Inc.](https://web.archive.org/web/20210612154934/http://weblogsinc.com/) ( [profile](https://web.archive.org/web/20210612154934/http://www.beta.techcrunch.com/?p=31) )和 [Gawker](https://web.archive.org/web/20210612154934/http://www.gawker.com/) 的博客网络——这意味着他们雇佣博客作者来撰写特定主题领域的博客，并让他们去做。

他们已经改进了产品(通过添加博主的名字，这是他们最初被批评没有的东西)，我们假设随着时间的推移会有进一步的改进。

从[发布会](https://web.archive.org/web/20210612154934/http://spaces.msn.com/members/adam/Blog/cns!1p6aifxJS4b8QGAcmWMQi6Gg!225.entry)由 [Adam Sheppard](https://web.archive.org/web/20210612154934/http://spaces.msn.com/members/adam/PersonalSpace.aspx) 发布，Filter 的首席项目经理:

> 今天我们正式推出了 MSN Filter。Filter 对我来说是一项充满爱的工作，是我们与在线社区建立更多对话并与消费者建立更密切关系的宠物项目。这种模式本质上是纳米出版，最初由 Gawker Media 和 Weblogs Inc .的 Nick Denton 倡导。这两个伟大的博客网络都有自己的受众，他们将继续取得成功。
> 
> 像这样的日子是我喜欢在微软工作的原因。没有什么比看着一个想法通过一个专注的团队的辛勤工作变成现实更有意义的了。MSN 的人关心的是做出改变。
> 
> 通过 MSN Filter，我希望提高博客的知名度，并为用户提供一个向我们雇佣的博客作者提交有趣内容的声音和论坛。这是我们想要学习的东西，并随着时间的推移而发展。所以如果你在外面，我们在听！
