# Palm Treo 755p:评论

Palm 的 Treo 755p 以 Sprint 的方式今天到达。该设备与 AT & T 上的 680 和 [750](https://web.archive.org/web/20200708174644/http://crunchgear.com/2007/03/02/treo-750-hands-on/) 出自同一模具。然而，与 700 不同的是，755p 使用的是 Palm OS 5.4.9。

680 和 750 都获得了普遍积极的评价，但它们都因 Windows Mobile 的实施而受到了一定程度的抨击。我通常对 WinMo 设备很矛盾，但它们确实占了我智能手机体验的大部分。事实上，755p 是我用过的第一部 Palm OS 智能手机。所以让我们来看看它是如何支撑的。

多年来，Palm 一直被认为是 PDA 用户中的主力。它的实用功能不知疲倦地运行，它的设备的使用已经被预示为一种透明的任务。我愉快地同意这些观点，Palm OS 确实比我尝试过的任何版本的 WinMo 提供了更好的用户体验。一切都按其应有的方式运行，而且速度很快。

755p 比 750 要活泼得多。其中一个原因是 755p 采用的 312MHz 英特尔处理器超过了它的 GSM 同类产品。应用程序的响应时间是即时的，或者至少接近即时的。此外，755p 上包含的所有应用程序都有一个清晰可辨的位置。我没有发现臃肿，只是一个非常精简的用户体验。

该设备的另一个明显的好处是 EV-DO 的实现。当然，750 有 UMTS，但一致性是这个游戏的名字，在这场决斗中，EV-DO 每次都领先。它更频繁地获取更强的信号，并以更可靠的速度推送数据。我不记得有哪一次我找不到 EV-DO。这并不是说在某些时候没有，只是从来没有在我需要的时候。仅这一点就大大有利于 Sprint 和 755p。

现在，Treo 700p 的用户可能很难找到升级的明确理由。据我所知，只有两点不同。正如我上面提到的，755p 使用了新的 Palm 机身样式。这意味着它稍微小一点，并且有一个内置天线——所以设备顶部没有巨大的突起。其次，它使用的是略有修改的 Palm OS 版本，修订版 5.4.9。

那值得升级吗？我真的不能说。对我来说，这似乎主要是一个化妆品升级。因此，如果 700p 的外置天线对你来说是一个主要的阻力，那么尽一切办法去争取它。否则，您可能希望等待更明显的升级。无论如何，这是一款性能稳定的好设备。不过，我将保留这个版本的最佳字节标记，因为这是一个非常小的升级。
