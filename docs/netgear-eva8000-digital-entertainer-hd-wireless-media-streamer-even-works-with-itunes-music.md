# Netgear EVA8000 数字娱乐高清无线媒体流甚至可以与 iTunes 音乐配合使用

![](img/2dc4f359dbee3a4c605f55811c83c80d.png)

苹果的憎恨者或者想要比[苹果电视](https://web.archive.org/web/20210301204659/http://crunchgear.com/2007/04/02/help-key-putting-video-onto-your-apple-tv/)提供的更多开箱即用文件类型兼容性的人应该看看 Netgear EVA8000 数字娱乐高清版。这是一个无线(802.11g)媒体流，就像苹果最新的努力，但它更通用。

像苹果电视一样，它可以将你电脑上的音乐、视频和照片无线传输到电视上。由于它不是 802.11n，所以不要指望能够太有效地传输带宽饥渴的 1080p。剩下的正如你所料:一个做广告的像样的产品。只是这里发生了更有趣的事情。

虽然[笔记本电脑杂志](https://web.archive.org/web/20210301204659/http://www.laptopmag.com/Review/Netgear-Digital-Entertainer-HD-EVA8000.htm)称该设备不能播放从 iTunes 购买的音乐，但 CNET 的 Crave，你知道，测试了它，发现它确实可以工作。(它只是慢了一点，只对 Windows 用户有效。)Good ol' [Engadget](https://web.archive.org/web/20210301204659/http://www.engadget.com/2007/01/07/netgear-launches-digital-entertainer-hd/) 早在一月份就表示支持 802.11n，但它并不支持。(“这东西摇滚选秀 802.11n WiFi”哎呀！)这个故事的寓意？人们应该格外小心地获取和报告准确的信息，而不是为了“抢先”竞争而争先恐后。但是我知道什么？
