# 威瑞森知道你周五晚上会在哪里

T2 威瑞森:来参加我们的聚会，否则我们就开枪打死这条狗。

你会排队购买新的 [LG VX 9400](https://web.archive.org/web/20130628211153/http://crunchgear.com/2007/01/08/lg-makes-vx-9400-official/) 多媒体和电视电话，外加某种带 2GB 存储卡的音乐套件，价格仅为 99 美元，对吗？这就是为什么所有纽约地区的威瑞森商店将在周五晚上营业到很晚，因为预计会有成群结队的人敲门寻找一部仅比 Conair 座机稍好的手机，坐在电视机旁播放体育中心的重播。

是的，朋友们，大男孩就是这样玩阴的。当美国电话电报公司的商店将赶走成群的粉丝时，威瑞森正在发起一场公关攻势，在多条战线上贬低 iPhone。有趣的是，我们没有听到 Sprint 或 T-Mo 关于这个话题的消息，可能是因为他们可能知道什么时候该闭嘴，开始买更酷的手机。

iPhone 抢走了威瑞森的聚光灯。解决办法，求 PR。
