# MOG 清理了一些混乱

[![](img/7e8df44c579fbd3952a18ed95352b702.png)](https://web.archive.org/web/20210922061401/http://mog.com/) 音乐社区网站 [MOG](https://web.archive.org/web/20210922061401/http://www.mog.com/) 今天推出了他们网站的 2.0 版本。我们之前在社交音乐服务的[综述](https://web.archive.org/web/20210922061401/http://www.beta.techcrunch.com/2007/02/05/social-music-overview/)中报道过 MOG，当时他们推出了一款[嵌入式音乐播放器](https://web.archive.org/web/20210922061401/http://www.beta.techcrunch.com/2007/01/19/mog-launches-embeddable-mp3-player/)。

核心的 MOG 服务是一个名为 Mog-o-matic 的客户端应用程序，它可以监控你在各种媒体播放器上听的音乐。它还配有个人音乐个人资料页面，你可以在上面记录你喜欢的音乐，并从像你一样的用户那里找到新歌推荐。推荐要么基于你手动添加到你所听的艺术家收藏中的乐队，要么基于从 Mog-o-matic 收集的乐队。

MOG 的核心功能仍然存在，但他们已经重组并添加了一些新功能，以使该网站吸引非会员。

新设计通过将帖子按类型和内容分类，以及一个名为“魔法按钮”的新个性化工具，使过去统治主页的混乱变得有序。为了让非 Moggers 更容易浏览网站，帖子现在被标记为音乐评论或新闻，并根据帖子是否包括音频、视频或纯文本进行分段。新的“魔法按钮”可以让你根据自己的 Mog-o-matic 音乐档案对网站上的任何内容进行个性化过滤。

视频部分还包括一个非常酷的 YouTube 黑客，名为 [MOG TV](https://web.archive.org/web/20210922061401/http://mog.com/watch/tv/) (竞争对手 [iLike 也有类似的功能](https://web.archive.org/web/20210922061401/http://www.beta.techcrunch.com/2006/11/09/ilike-adds-music-videos-from-youtube/))。MOG TV 由给定艺术家的所有 YouTube 音乐视频组成。这让我想起了 MTV，当时他们真的播放音乐视频。您可以根据艺术家、个人资料中列出的音乐偏好、朋友偏好或 Magic 按钮来过滤 400K 索引 YouTube 视频中的播放内容。

MOG 与 Last.fm 非常相似，它的日志功能让听众可以发表关于任何乐队的博客文章。然而，MOG 已经将用户生成的帖子作为网站的重点，音乐推荐则是次要的。Last.fm 以他们的音乐推荐软件为主，用户生成内容为辅。最近点击量超过 50 万。
