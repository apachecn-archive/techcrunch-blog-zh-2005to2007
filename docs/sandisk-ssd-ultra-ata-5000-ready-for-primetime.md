# SanDisk 固态硬盘 Ultra ATA 5000 已准备好迎接黄金时代

![](img/cdc4e079d33319de3f945fd17b6cd8af.png)

SanDisk 认为，它已经准备好推出首款固态硬盘，进军移动个人电脑市场。SSD Ultra ATA 5000 1.8 英寸硬盘的大小为 32GB，可以在 35 秒内启动 Windows Vista Enterprise。这种驱动器因其速度而受到吹捧，这要归功于固态存储器，而不是像传统硬盘驱动器那样的移动部件。

新闻稿中的怪胎:

> "
> 由于没有移动部件，SanDisk SSD 不需要像传统硬盘驱动器那样旋转或寻找文件，从而使 SanDisk SSD 的工作速度更快。对于 512 字节的传输，SanDisk SSD Ultra ATA 5000 实现了 62 兆字节(MB)*/秒的持续读取速率和每秒 7300 次输入/输出(IOPS)的随机读取速率，比任何硬盘驱动器都要好 100 多倍。
> 这些性能数据提升了系统性能。例如，闪迪固态硬盘 UATA 5000 可以在 35 秒内启动笔记本电脑上的 Microsoft Windows Vista Enterprise。SanDisk SSD 实现了 0.12 毫秒的平均文件访问速率。**在 Windows Experience Index for Microsoft Vista 中，SanDisk SSD 在总分 5.9 分中得了 5.4 分，而在同一台笔记本电脑中进行的相同测试中，硬盘驱动器得了 3.7 分。***
> ”

哦，如果你把这些固态硬盘中的一个放进去，你的笔记本电脑将获得更长的电池寿命。该驱动器在活动时仅使用 0.4W，而标准硬盘上使用 1.0W。消耗更少的能量意味着更多的能量。每个人都赢了。目前还不知道这要花多少钱，也不知道什么时候会有更大的尺寸。
