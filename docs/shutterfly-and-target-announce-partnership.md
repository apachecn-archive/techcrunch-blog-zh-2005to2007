# Shutterfly 和 Target 宣布合作

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/07/shutterfly-and-target-announce-partnership/>

![](img/e4c0f251896f319b232f5bf400da8d94.png)

就我个人而言，我用 Flickr 拍照片，甚至不再用硬拷贝了。但是对于我们这些仍然有心使用 Shutterfly 的人来说，你现在可以从你的本地目标上获取你的数码照片了。今天，塔吉特百货和 Shutterfly 宣布了一项合作，你可以在网上以每张照片 19 美分的价格订购照片。

据说取货只需要大约一个小时，这意味着如果需要的话，你可以从他们那里挤出一份紧急工作，前提是你愿意为此付费。Shutterfly 的总裁杰夫·豪森博尔德说:

> “这种合作关系为 Shutterfly 与美国领先的消费品牌之一提供了首个同类零售渠道，我们很高兴在 Target 找到了理想的业务合作伙伴，为消费者提供他们期望从 Shutterfly 获得的优质体验，”

太棒了。下一次我真的需要打印照片，我会记住你 Shutterfly。考虑到通常每个人附近都有一个目标，听起来很有用。
