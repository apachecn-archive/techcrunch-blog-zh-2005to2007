# U-G-L-Y: ago7 UMPC

一般来说，我不会抨击那些我没见过面也没用过的产品，但是像 ago7 这样的超便携电脑将会扼杀已经很小很脆弱的 UMPC 市场。你能看看这个东西吗？需要结束它的痛苦。我今天早上发起攻击的原因是因为我收到了 ago7 的新闻稿，称这是“为移动在线活动寻求更高计算能力但又担心笨重笔记本电脑的手机用户”的一个新选择真的有人*警惕*笔记本电脑吗？

只需 799 美元，你就可以获得 1GHz 的 CPU、512MB 的内存、40GB 的存储空间以及内置的 WiFi 和蓝牙，所有这些都运行在 Windows XP 平板电脑版上。我对这一切都没意见。实际上，公司建议你将这款 UMPC 与蓝牙手机和数据套餐配合使用，以便在任何地方都能上网，对此我并没有生气。让我恼火的是，不管是哪家公司负责创建 ago7，都没有为它的外观付出任何努力。你想给人们一个超便携笔记本的可替代物，这很好，但请不要把它放在一个可以被儿童玩具超越的包装中。
