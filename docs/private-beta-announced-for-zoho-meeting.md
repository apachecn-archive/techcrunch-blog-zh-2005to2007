# 一个很好的 WebEx 替代方案:Zoho 会议

 [](https://web.archive.org/web/20211027005651/http://meeting.zoho.com/) Zoho 刚刚为 [Zoho Meeting](https://web.archive.org/web/20211027005651/http://meeting.zoho.com/) 发布了一个私人测试版，这是他们新的在线会议应用程序，将与 WebEx(最近被思科收购的[)的重要功能重叠——桌面共享。他们会吸引大量新用户加入——在主页上注册请求访问，或者在这篇文章的最后看嵌入的视频看演示。](https://web.archive.org/web/20211027005651/http://www.beta.techcrunch.com/2007/03/15/cisco-buys-webex-for-32-billion/)

Zoho Meeting 允许每个成员查看(放大/缩小)和远程控制主持人的桌面、聊天、添加参与者、向主持人发送电子邮件以及查看会议详细信息。用户也可以录制和下载会话(AVI 格式，Flash 即将推出)。聊天由 Zoho chat 处理，Zoho Chat 也将很快具备 VOIP 功能。基本上，如果你想远程演示，Zoho Meeting 将是一个非常有吸引力的选择。

要创建会议，您需要在 Zoho 帐户中创建会议，设置日期、描述，并通过电子邮件邀请与会者。但是，要广播您的桌面，您还必须安装一个 ActiveX 控制器(732 Kb)，这使得只托管 Windows。

[![zohomeetingsmall.png](img/da265dff5823ee98ae6ab0bc921a030f.png)](https://web.archive.org/web/20211027005651/https://beta.techcrunch.com/wp-content/uploads/2007/03/zohomeetingbig.png) 他们已经竭尽全力通过制作三种不同的风格让浏览器在任何地方都能工作:ActiveX(仅限 Windows)、Java 和 Flash。这意味着 Zoho meeting 可以在 Mac、Windows 和 Linux 机器上运行。闪存客户端有一些延迟，在 1.5 到 1.8 秒之间波动，Java 和 ActiveX 客户端表现更好。

但是，Zoho 并没有为了好玩而加入 Flash 浏览器。闪存查看器将让用户将会议嵌入到任何可以使用闪存的地方，比如网站。这种嵌入也适用于 Zoho Show，这将允许主持人在一张包含嵌入代码的幻灯片中向观看他们节目的人群远程演示。通过与 Zoho Chat 的集成，会议集成也将延伸到其他 Zoho 应用中。Zoho 用户很快就可以通过点击聊天框中的链接与任何 Zoho 用户召开会议，这样不仅可以说，还可以表达你的意思。
