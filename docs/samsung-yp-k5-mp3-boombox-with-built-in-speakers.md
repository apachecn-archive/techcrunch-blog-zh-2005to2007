# 三星 YP K5 MP3 内置扬声器

当您将三星 MP3 播放器与一组内置扬声器结合在一起时，您会得到什么？实际上，相当不错的东西。三星 YP-K5 MP3 播放器从正面看起来像一个普通的播放器，但在它的背面安装了一个滑动扬声器外壳。那怎么样呢？嗯，对于一个基于 flash 的 MP3 播放器来说，它有点笨重。实际上是 iPod Nano 厚度的三倍。

UI 本身就很吸引人。在 1.7 英寸的有机发光二极管屏幕上，有均衡器、滚动按钮和回放控制。这个单元的一些额外的奖励是一个闹钟和调频调谐器，这些东西你不会一直使用，但拥有它们是很好的。播放器正常播放时间为 30 小时，使用扬声器时为 6 小时，使用附带的交流适配器充电 3 小时。我们可以不用内置扬声器，因为不是每个人都需要它们，但它们应该对便便者有用。
