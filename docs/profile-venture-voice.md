# 简介:风险之声 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/07/05/profile-venture-voice/>

**公司:** [创业之声](https://web.archive.org/web/20201205122630/http://www.venturevoice.com/)

![](img/1bc79b4ecaa9691ecf48be668b5a96eb.png)

**投放:**2005 年 6 月 8 日

**什么事？**

风险之声制作采访有趣人物的播客。

用他们自己的话说，*“创业成功需要什么？我们正在通过打电话给企业家、风险资本家以及他们的朋友和敌人来寻找答案。这个播客(或者，对于外行人来说，是网络广播节目点播)以我们的对话为特色。倾听那些过着企业家生活的人的声音。从他们语气中听到的兴奋、恐惧和雄心让我们感受到他们正在经历的事情。我们对各行各业的人都感兴趣——那些已经成功或即将成功的人。我们既想听到成功，也想听到失败。除了职业生活，创业对个人生活也有影响；我们探索两者。如果您想推荐某人接受采访，或者对该节目有任何反馈，请联系我们。”*链接[链接](https://web.archive.org/web/20201205122630/http://www.venturevoice.com/2005/06/welcome_to_the.html)

![](img/454b1d6403565ead6d198dbd996d6d5e.png)

像 Podtech.net 一样，他们正在采访一些在技术行业非常引人注目和有趣的人。在他们上线的短暂时间里，他们发布了对以下人员的采访:

FeedBurner 的迪克·科斯特罗([第一部](https://web.archive.org/web/20201205122630/http://www.venturevoice.com/2005/06/vv_show_1_-_dic.html) ) ( [第二部](https://web.archive.org/web/20201205122630/http://www.venturevoice.com/2005/06/vv_show_2_-_dic.html))
[ad brite 的菲利普·卡普兰](https://web.archive.org/web/20201205122630/http://www.venturevoice.com/2005/06/vv_show_3_-_phi.html)(fucked company 创始人)
JotSpot 的乔·克劳斯([第一部](https://web.archive.org/web/20201205122630/http://www.venturevoice.com/2005/06/vv_show_4_-_joe.html) ) ( [第二部](https://web.archive.org/web/20201205122630/http://www.venturevoice.com/2005/07/vv_show_5_-_joe.html))

对于一个不到一个月的网站来说，这已经不错了。见鬼，也许有一天 TechCrunch 会登上 A 榜，接受 Venture Voice 的采访。:-)

我们期待听到更多来自风险之声的消息。
