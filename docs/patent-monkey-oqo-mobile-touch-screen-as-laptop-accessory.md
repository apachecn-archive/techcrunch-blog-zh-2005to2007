在 CES 上被比尔·盖茨加冕为“[类别界定 UMPC](https://web.archive.org/web/20160206121248/http://www.oqo.com/news/press_releases/2007-01-07_02_launch.html) ”之后，我们仔细阅读了 OQO 持有的一项专利。在某种程度上，一些申请的专利就像梦一样…这是真的吗？我就是记不起来了。OQO 创造了一款符合市场发展方向的设备…

如果您可以将一个触摸屏、手持计算 PDA 设备(使用触摸屏技术)结合起来，然后连接到一个带有操作系统的全尺寸笔记本电脑上，使这两个设备在分离时能够自主运行，在结合时能够统一运行，您会拥有什么？你会有一个非常惊人的 OQO 产品线，一个小型的“移动”移动设备，它利用那令人敬畏的新动态 UI 将[与](https://web.archive.org/web/20160206121248/http://www.patentmonkey.com/PM/PatentID/7054965.aspx)集成到你办公桌上的全尺寸机器中。

将这一概念更新到最近的新闻中，想象一下一部 iPhone 插入 MacBook Pro，在笔记本电脑中集成触摸屏动态显示和控制，然后可以分离，用作您的“移动”手机/音乐/互联网/移动设备。虽然摩托罗拉 Q 是一款[“非常强大的设备”](https://web.archive.org/web/20160206121248/http://www.crunchgear.com/2007/01/18/balmer-laughs-at-the-iphone-lies-completely)，但 OQO 可能拥有一系列关键的技术，这些技术将拥有苹果爱好者渴望的惊艳因素，同时成为商业用户所需的笔记本电脑+ [、黑莓](www.blackberry.com) +无线移动卡的新集成系统。

苹果在开发 iPod 和 iPhone 的过程中意识到了一件事，OQO 的发明利用了这一点——大容量蜂窝数据传输速度[不快](https://web.archive.org/web/20160206121248/http://www.physorg.com/news73142122.html)，但越来越受欢迎的是。一个联合目的的收费和文件更新程序符合人们所做的(或者应该做的，否则你最终会有很多[这些](https://web.archive.org/web/20160206121248/http://www.energizer.com/energitogo/index_flash.html))。现在，如果 iPhone 有 2 节电池,那么也许日常连接就没有必要了。

对于 iPhone 或 OQO 如何与您的办公室或家庭网络交互，有很多可能性。谁知道呢，这可能只是停留在我们的梦想清单上。
