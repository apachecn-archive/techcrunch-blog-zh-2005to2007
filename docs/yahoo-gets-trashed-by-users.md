# 雅虎被用户抛弃

[![yahootvlogo.jpg](img/0e59905a2d7971c2696f415b51e62802.png)](https://web.archive.org/web/20221203093233/http://tv.yahoo.com/) 雅虎遭到了用户的痛打，他们对[新的雅虎电视产品](https://web.archive.org/web/20221203093233/http://www.beta.techcrunch.com/2006/11/30/yahoo-tv-gets-a-new-do/)在他们自己的[博客](https://web.archive.org/web/20221203093233/http://yodel.yahoo.com/2006/11/28/anything-good-on-tonight/)上发布的评论感到愤怒。甚至雅虎娱乐的前负责人埃里克·施瓦茨也加入了他自己的抨击，并暗示雅虎已经迷失了方向。

雅虎的 Sal Taylor Kydd 在评论中也很活跃，[回应](https://web.archive.org/web/20221203093233/http://yodel.yahoo.com/2006/11/28/anything-good-on-tonight/#comment-5830)投诉，随后给出[状态更新](https://web.archive.org/web/20221203093233/http://yodel.yahoo.com/2006/11/28/anything-good-on-tonight/#comment-6065)关于所做的改变。

戴夫·温纳称这很有趣，我也同意。公司很难对用户反馈敞开心扉——尤其是当反馈是匿名的、不受约束的，并且放在自己的网站上的时候。给予员工直接、永久和实时回应的能力需要勇气，也需要高管承诺拥抱用户，即使他们讨厌你。

因此，尽管我承认雅虎的新产品并不出色，但我喜欢雅虎处理事情的方式。如果他们把旧网站作为喜爱它的人的一个选择，他们会很快赢得更多的用户。我们走着瞧。

我还注意到谷歌不允许在他们的企业博客上发表评论。为什么？
