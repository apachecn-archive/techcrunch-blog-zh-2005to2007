# 伟大的户外:一个明亮、恐怖的地方

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/02/the-great-outdoors-a-bright-scary-place/>

今天已经过去三天了，我们都该晒晒太阳了。无论你是住在潮湿、超湿的地方，还是在全球变暖的热浪中舒适地安顿下来，我们都鼓励你涂上防晒霜，阅读一些我们的户外小工具功能，然后尝试自己进入伟大的户外。

本周我们将谈论水上娱乐、露营娱乐和灭虫器。我们的目标是给你一个不错的酷装备样本，以取代你的 iPhone，当它在你为期 30 天的阿巴拉契亚远足的第二天坏掉的时候。

你可以点击[这里](https://web.archive.org/web/20160221182805/http://www.crunchgear.com/catgegory/greatoutdoors)阅读我们所有的户外故事，或者你可以关掉你的笔记本电脑，去看看外面广阔的世界有什么值得看的。你是我的选择。
