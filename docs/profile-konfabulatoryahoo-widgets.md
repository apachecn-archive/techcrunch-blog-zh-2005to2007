**被收购:**被雅虎收购，2005 年 7 月 25 日宣布
 **是什么？**

Konfabulator 昨天被雅虎收购。

用他们自己的话来说，*“Konfabulator 是一个用于 Windows 和 Mac OS X 的 JavaScript 运行时引擎，可以让你运行叫做小部件的小文件，这些小文件可以做你想让它们做的任何事情。小工具可以是闹钟、计算器，可以告诉你你的 WiFi 信号强度，可以获取你喜欢的符号的最新股票报价，甚至可以提供你当前的当地天气。”* [链接](https://web.archive.org/web/20211015195745/http://www.konfabulator.com/info/)

![](img/f30dda1b327a4b7133712de228e9b30f.png)

![](img/178805ecfa086f18c35971eff9aa05af.png)

Konfabulator 最初于 2003 年 2 月 10 日推出，仅用于 Mac。2004 年 11 月推出的 Windows 版本。

Konfabulator 需要下载，可以在 Mac 和 Windows 平台上运行。Mac 下载 5.4 mb，Windows 下载 10 mb。这项服务以前需要付费，但现在是免费的。

下载完成后，设置变得非常快速和简单。基本上，它在你的桌面上运行小的应用程序(有数以千计的应用程序可供选择)。

我最喜欢计算器、便利贴和 RSS 阅读器(尽管我想不出如何配置 RSS 阅读器来停止显示《今日美国》，所以放弃了)。还有天气、电池电量和其他你能想到的所有东西的小部件。一些会随着安装自动下载，但是它们很容易被删除，其他的通过下载就很容易添加(它们会自动安装到正确的目录中)。

小部件可以很容易地在桌面上移动、定制和完全钻研。

有[工具](https://web.archive.org/web/20211015195745/http://www.konfabulator.com/workshop)可以让任何人轻松创建自己的小工具。

Konfabulator 的外观和感觉与 Mac 上 OSX 的 Dashboard 非常相似，尽管有些人声称在 Konfabulator 上创建小部件更容易。

总体而言，它很酷，但将在晚上结束时卸载。我注意到安装后系统速度严重下降。
