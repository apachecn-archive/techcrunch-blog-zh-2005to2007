SanDisk 4GB M2卡面向手机用户

原文：https://web.archive.org/web/http://techcrunch.com:80/2007/03/27/sandisk-4gb-m2-card-aimed-at-mobile-phone-users/

我不是那种为了媒体目的而使用手机的人，因为手机上的软件程序太糟糕了。我从未在我的SK3上使用过MP3播放器，我的诺基亚5300甚至还没有问世。内存不足是我不使用这些功能的主要原因，但SanDisk 4GB Memory Stick Micro line可能会改变我的想法。

虽然它面向索尼爱立信Cyber-shot和Walkman系列，但它也适用于任何带有M2插槽的手机。它还向后兼容所有带记忆棒插槽的数码和手持游戏播放器。4GB M2卡要到今年晚些时候才会上市，目前还不知道价格，但它将在5月份发货给原始设备制造商。
