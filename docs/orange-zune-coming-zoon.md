# 橙色 Zune 即将发布

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/02/orange-zune-coming-zoon/>

你知道，当这种事情成为头条新闻的时候，这是一个缓慢的新闻日。朋友们，这是一个橙色的 Zune。比其他 Zunes 好吗？不会，除非你觉得橙色让某些物品在本质上比其他物品更好。不幸的是，情况并非如此。然而，如果你喜欢 Zunes，我们相信你会喜欢这个橙色的。当我在 J-school 的时候，他们从来没有告诉我我会写关于 orange Zunes 的文章。
