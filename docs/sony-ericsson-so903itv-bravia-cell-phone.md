# 索尼爱立信 SO903iTV Bravia 手机

我不是一个真正喜欢鲜红色设备的人，但从技术角度来看，索尼爱立信 SO903iTV Bravia 手机确实非常热门。除了红色的表面，它还在 3 英寸的宽屏显示器上显示风景视频。它还有一个能够时移录像的电视调谐器。

所有其他典型的手机功能也在那里，包括一个 200 万像素的摄像头和 microSD。这款也只在日本有售，所以不要开始起泡太多。
