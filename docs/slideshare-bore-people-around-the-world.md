# SlideShare:让世界各地的人感到无聊

TechCrunch 先睹为快，这是一个 PowerPoint 共享系统，你可以上传演示文稿，然后在任何网络浏览器上展示。把它想象成演示的 YouTube，只是少了一些孩子把曼妥思放进健怡可乐里。

虽然它不是小玩意，但如果你使用正确的移动浏览器，它肯定是移动的。它还可以用来帮助睡眠，前提是你能找到一份演示文稿，这份演示文稿是我多年前的同事克劳斯准备的。那个男孩的演讲能让一个六岁的高糖小孩睡着。
