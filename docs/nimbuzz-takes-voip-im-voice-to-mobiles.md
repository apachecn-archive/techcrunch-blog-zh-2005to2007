# Nimbuzz 将 VoiP、即时消息语音带到手机上| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/01/18/nimbuzz-takes-voip-im-voice-to-mobiles/>

我认识一个人，他在 Treo 上运行着一种 Skype 风格的软件，与朋友和家人通话，而没有占用他计划中的时间。这是个好主意，如果你的手机支持的话。剩下的人，看看 Nimbuzz。

Nimbuzz 是一个基于 Java 的程序，你可以将其加载到手机上，绕过手机内置的即时消息客户端，通过 Windows Live 与他人进行即时消息对话。但最棒的是，它还内置了对网络语音功能的支持，这意味着您可以与其他 Windows Live 用户(移动或非移动)甚至其他手机进行语音通话，使用 Windows Live 为您提供的低价方式。

这意味着任何运行 Java 的数据连接单元都有相当大的机会运行这种专门版本的 VoiP，完全绕过蜂窝语音网络。这很棘手，我们还没有看到它的实际应用，但这绝对是一个绝妙的想法，值得一试。
