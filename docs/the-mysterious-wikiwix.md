# 神秘的维基百科:隶属于维基百科？

 [](https://web.archive.org/web/20220625055221/https://beta.techcrunch.com/wp-content/uploads/2007/05/wikiwixb.png) 我们今天得到一个关于一个叫做[维基百科](https://web.archive.org/web/20220625055221/http://www.wikiwix.com/)的新搜索引擎的提示。这个网站对我来说是新的，网上关于它的信息很少(大约三个小时前的一个帖子是[这里是](https://web.archive.org/web/20220625055221/http://www.techjuicer.com/index.php/2007/05/19/wikipedia-search-engine-wikiwix/))。它是维基百科主搜索页面的[链接](https://web.archive.org/web/20220625055221/http://en.wikipedia.org/wiki/Special:Search?search=&go=Go)，尽管我不确定它是什么时候添加的。

像与维基百科无关的搜索引擎维基百科一样，维基百科只搜索维基百科。“联系我们”链接指向维基百科上的[这个页面](https://web.archive.org/web/20220625055221/http://fr.wikipedia.org/wiki/Utilisateur:Pmartin/Wikiwix)，它是法语的。该页面的翻译版本[在这里](https://web.archive.org/web/20220625055221/http://translate.google.com/translate?u=http%3A%2F%2Ffr.wikipedia.org%2Fwiki%2FUtilisateur%3APmartin%2FWikiwix&langpair=fr%7Cen&hl=en&ie=UTF8)。

该域名为法国居民马丁·帕斯卡尔所有。上面的联系链接早在 2007 年 3 月 20 日就添加了信息。该页面上的一个问题是*“谁是 Wikiwix 的幕后黑手，使用哪种技术，这是与 wikiédia 联系的内容？”*，这又增加了进一步的困惑。此外，维基百科已经被添加到维基百科搜索中，而维基百科没有提供这一功能，这一事实表明这两个实体之间存在某种联系。

当有更多信息时，我会更新这篇文章。我已经给吉米·多纳尔·威尔士发了邮件，等待他的回复。

***更新:**在评论里补充好信息。听起来这不属于维基百科。*
