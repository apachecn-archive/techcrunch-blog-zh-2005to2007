# Yappd 推出自称为“带图片的推特”

 [](https://web.archive.org/web/20221221073843/https://techcrunch.com/wp-content/uploads/2007/08/yappdb.png) 这种跟风服务不会持续太久。 [Yappd](https://web.archive.org/web/20221221073843/http://yappd.com/) ，一个 [Twitter](https://web.archive.org/web/20221221073843/http://www.crunchbase.com/company/twitter) 克隆，今天发布。在给我们的电子邮件中，他们把自己描述为“带图片信息的 Twitter”，这就很好地概括了这一点。这是一项可以让你迅速告诉全世界你在做什么的服务。您可以通过他们的网站、电子邮件或短信添加内容。

因此，当我们争论凯文·罗斯的 [Pownce](https://web.archive.org/web/20221221073843/http://www.crunchbase.com/company/pownce) ，这个领域的另一个新进入者，是否[与 Twitter](https://web.archive.org/web/20221221073843/http://techcrunch.com/2007/07/01/kevin-v-evan/) 有足够的不同以获得成功时，另一个充满希望的年轻枪手进入了这个领域，除了在你的状态信息中添加一张照片之外，几乎没有什么与众不同之处。

除非 Yappd 有一套绝妙的营销策略，否则我不认为他们会受到太多的关注。不过，我很喜欢照片功能。希望 Twitter 能尽快添加。
