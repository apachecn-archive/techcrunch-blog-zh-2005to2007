# 史上最差的任天堂 DS

![](img/d0d356902c0506b2a97f9daa317cc03b.png)

我在二手游戏机和游戏世界中见过的最糟糕的情况是，一台游戏机缺少交流适配器，或者一款游戏缺少包装盒和说明书。然而，没有什么能与日本涩谷展出的任天堂 DS 相比。Tsutaya 正在出售一台二手任天堂 DS，丢失并损坏了以下物品:

*   没有盒子*   没有说明书*   没有手写笔*   没有带子*   没有交流适配器*   表壳上的划痕*   Scratches on screen

    报告称，这款 DS 不仅濒临死亡，而且上面还有污垢，两块屏幕上都有很深的划痕。但有趣的是这个东西的售价是多少:一万日元或 100 美元左右。没错。人们愿意花 100 美元买一台完全毁坏、几乎不能工作的 DS。下次你在这里抱怨 Wii 短缺的时候，看看这篇文章，然后深呼吸…放松。
