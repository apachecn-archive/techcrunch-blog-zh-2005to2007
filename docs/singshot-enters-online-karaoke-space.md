# SingShot 进入在线卡拉 ok 领域 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/30/singshot-enters-online-karaoke-space/>

五月被福克斯互动[收购的](https://web.archive.org/web/20230220084733/https://techcrunch.com/2006/04/30/fox-acquires-ksolocom-confirms-newroo/) [](https://web.archive.org/web/20230220084733/http://www.singshot.com/) kSolo 在在线卡拉 ok 领域面临一些新的竞争。SingShot 明天将发布他们自己的在线卡拉 ok 版本。这项新服务本质上是受成功的 YouTube 用户界面影响很大的 kSolo 的翻版。

kSolo 产品的所有基本功能都在那里。表演者可以从一个至少有两到三年历史的歌曲库中选择歌曲，听一段演示，并录制自己的版本，同时跟着屏幕上的歌词走。希望通过允许用户分享他们的歌曲，由公众投票和评论，来复制 YouTube 等其他社交内容网站的成功。人类过滤有望在威廉·亨格斯的森林中挑出凯利·克拉克森。

这两种服务的最大区别在于细节。首先，kSolo 抛弃了 Flash，这是一个现在无处不在的在线视频平台，需要一个插件，只能在 Internet Explorer 上运行。SingShot 完全在 Flash 上运行，可以在所有浏览器上运行，这使得设置和开始运行变得轻而易举。用户只需改变他们的闪光灯设置，以允许程序访问他们的麦克风。虽然这两个程序都允许分享表演，但 SingShot 遵循简单的 YouTube 模式，允许简单的剪切和粘贴链接或嵌入，而 kSolo 有一个笨重的共享功能，允许你通过电子邮件发送链接，或者在稍加搜索后，将整个播放列表嵌入到一个页面上。

不过，这两种服务的导航功能都还不太好。我发现很难偶然发现新歌或找到那些我只记得部分标题或歌词的歌曲。相反，我发现自己过滤了广泛的类别(摇滚、流行或 80 年代)，不得不反复点击“下一页”。简单地在底部添加页码将有助于浏览类别。然而，与 SingShot 不同的是，kSolo 的搜索引擎未能识别出“披头士”与“the Beatles，the”相同。不过，在寻找新音乐方面，kSolo 在 SingShot 上确实有一个。他们有一个功能，可以让你找到与你刚刚录制的歌曲相似的音乐。

在今天的发布会上，SingShot 推出了一个极具侵略性的定价模式，向福克斯互动媒体发起了挑战。SingShot 为 kSolo 的“一周”提供了两周的免费使用，年费为 4.95 美元/月，季度费为 7.95 美元/月，月费为 9.95 美元/月。kSolo 的年费为 7.95 美元/月，季度费为 8.95 美元/月，月费为 9.95 美元/月。

kSolo 资金雄厚，拥有更大的用户群和歌曲库，但未来几个月将会看到这两个公司谁会胜出。

读者们也应该看看我们对 Bix 的报道，这是一项竞赛服务，似乎是这两个歌曲网站混搭的理想选择。
