# 专利猴:Vonage 资产出售的时机？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/14/patent-monkey-timing-right-for-vonage-asset-sale/>

Vonage 正在对抗可能的破产，需要一个新的 CEO 和投资者的挫败感。虽然它逃脱威瑞森诉讼的可能性有一些希望，一个积极的买家更激进的步骤可能只是一个技巧。

Vonage 就其专利侵权向威瑞森提起了上诉。最高法院关于专利显而易见性的裁决已经开始改变专利景观，提供了许多潜在的帮助，但对于 Vonage 的请求来说可能太迟了。虽然变通措施是有道理的，但考虑到这些新情况，一些创新的想法可能会给一个熟悉品牌、敢于冒险的投资者带来一笔丰厚的横财。

Vonage 在市场营销上投入了大量资金来获取客户:在其最近的季度更新[中提到，仅本季度就有 16.6 万客户获得了 3.1 亿美元。Vonage management 指出，这相当于支付 275 美元来获得一个新客户。公司生存的不确定性无助于赢得新客户，无论广告宣传多么迅速。这种品牌投资使该公司成为 VoIP 的代名词——一种不那么容易丢掉的资产。此外，在考虑出售时，一家没有固定首席执行官的公司可能会被视为一项资产。](https://web.archive.org/web/20140311001547/http://www.bloomberg.com/apps/news?pid=20601103&sid=a5ZllIG1FTM0)

最高法院 KSR 裁决对法院评估专利明显性的方式做出了巨大改变，并对专利执法案件产生了直接影响。许多批评家指责威瑞森的专利是显而易见的，潜在起诉者的深入审查应该挖掘这一分析。

在适当解决潜在问题的同时，Vonage 的[主要投资者](https://web.archive.org/web/20140311001547/http://finance.yahoo.com/q/mh?s=VG)还应考虑向竞争对手或新成立的实体出售企业资产的战略替代方案(例如，Vonage 控股公司仍将存在，但将同意出售账户、知识产权等)。威瑞森会再次起诉吗？是的。结局会一样吗？也许不是。对威瑞森专利显而易见性相关问题的审查对这一举措至关重要，但如果[第一联邦巡回法院案件](https://web.archive.org/web/20140311001547/http://271patent.blogspot.com/2007/05/common-sense-affirms-obviousness-in.html)是通过无效更多索赔向被告倾斜的迹象，这可能是收购公司的意外投资，也是 Vonage 现任董事会结束沮丧的一种方式。

涉案专利有: [US 6104711](https://web.archive.org/web/20140311001547/http://www.patentmonkey.com/pm/patentid/6104711.aspx) 、 [US 6287574](https://web.archive.org/web/20140311001547/http://www.patentmonkey.com/pm/patentid/6287574.aspx) 和 [US 6298062](https://web.archive.org/web/20140311001547/http://www.patentmonkey.com/pm/patentid/6298062.aspx)
