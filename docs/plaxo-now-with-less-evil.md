# Plaxo:现在少了一些邪恶——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2006/03/22/plaxo-now-with-less-evil/>

 [](https://web.archive.org/web/20220108084931/http://www.plaxo.com/) [普拉克索](https://web.archive.org/web/20220108084931/http://www.crunchbase.com/company/plaxo)往往会给人带来强烈的负面情绪。挫败感。厌恶。沮丧。

这么多人[讨厌 Plaxo](https://web.archive.org/web/20220108084931/http://www.google.com/search?hs=yPp&hl=en&lr=&client=firefox-a&rls=org.mozilla%3Aen-US%3Aofficial&q=i+hate+plaxo&btnG=Search) 的原因？我们这些不是客户的人，也就是绝大多数人，每天都会从那些选择注册 Plaxo 垃圾邮件服务的少数误入歧途的人那里收到多封电子邮件(我的名字，不是他们的)。

当你注册 Plaxo 时，他们很容易误导你，让你的整个地址簿充斥着你的联系信息和 Plaxo 广告。他们所做的任何更新也可能被发送到他们的整个地址簿。由于数百万人不知不觉地这样做了，我们这些非 Plaxo 用户承受了巨大的总成本:处理所有这些垃圾邮件。Jeff Nolan 和许多其他人已经通过将 Plaxo 放入他们的垃圾邮件过滤器来处理它。我经历了他们的多个步骤(很难找到)[过程](https://web.archive.org/web/20220108084931/https://www.plaxo.com/opt_out)，从他们的系统中删除了我的五封电子邮件中的每一封(这个过程包括回复他们的另一封电子邮件，以证明你拥有这个账户——多好啊)。

一路走来，当我们被迫处理 Plaxo 垃圾邮件和各种避免过程时，他们告诉我们，避免垃圾邮件的最佳方式是简单地成为 Plaxo 的成员。

> 如果您加入了 Plaxo，您可以填写一次您的联系信息，并选择谁可以查看您的信息。从那时起，更新通过 Plaxo 处理，这样你就不会再收到任何电子邮件了。

这就像一个跟踪者告诉他的跟踪者，如果她愿意嫁给他，他就会停下来。

如果你[写关于 Plaxo 的博客](https://web.archive.org/web/20220108084931/http://www.crunchnotes.com/?p=109)，或者公开表达你的不满，Plaxo 会派一名代表在评论中耐心地说服你。

嗯，Plaxo 终于宣布，他们已经骚扰了足够多的人，让他们加入，并且不会让用户很容易地向他们的整个地址簿发送垃圾邮件。他们选择的词语无疑告诉我们，至少在内心深处，他们完全知道他们让我们所有人多么痛苦:

> …我们一直知道更新请求是达到目的的一种手段— **我们的目标一直是获得尽可能多的成员，这样这些电子邮件就没有必要了。**看起来我们终于要走到那一步了。
> 
> 截至上周，我们已经超过 1000 万会员。我们现在每天有超过 50，000 名用户。由于这种巨大的增长，我们网络的深度，加上**我们成为好网民的衷心愿望，**我们已经开始逐步取消更新请求。
> 
> 这项功能可能会以某种形式一直存在，但**我们不再积极地推动新用户发送电子邮件**，并增加了限制，以防止现有用户发送大量邮件。在接下来的六个月内(考虑到我们基础的发布和升级)，您应该会看到这些消息逐渐减少。

鉴于 Plaxo 长期以来的立场，即他们的电子邮件爆炸是完全合法的，我在上面用粗体显示了我觉得特别不可思议的文字，而且真的，即使我们不知道，对我们来说*很好*。

所以总的来说，我很高兴 Plaxo 这样做，我为他们鼓掌。但是官方道歉怎么样？

没关系，反正他们会用电子邮件发送。
