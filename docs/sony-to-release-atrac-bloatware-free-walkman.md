# 索尼将发布无 ATRAC、无臃肿软件的随身听

![472975347_c86f7a5261_o.png](img/f2322faf2f85cd3bdff071ec061db339.png)

更让人惊讶的是？索尼正在发布一款拖放式随身听 B100，它不使用音乐加载应用，也不支持 ATRAC，或者有一个专门的 ATRAC 新闻网站？我选择后者。

这款随身听有 1GB、2GB 和 4GB 三种版本，应该会在下个月上市，可能只会在日本上市。我们在乎吗？不是特别明确，但很奇怪看到强大的索尼公司这样折腾，可能会打开新的市场。
