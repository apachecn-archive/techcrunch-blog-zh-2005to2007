# 沃达丰否认威瑞森收购传闻

![](img/da6c7a5e905bd46e1dde00f7b122fcb6.png)

今天早上，沃达丰对 Verizon Communications 感兴趣的消息开始传播。现在沃达丰否认它有兴趣以 1600 亿美元收购该公司，这将使其市值接近 3000 亿美元。仅供参考，沃达丰是 Verizon 通信公司的合作伙伴。他们成立了这个小公司你可能听说过 mdash 沃达丰拥有威瑞森无线 45%的股份。

《金融时报》称，该交易目前只是一纸空文，但它也表示，该计划已在过去几周得到讨论。想象一下，沃达丰的 3G 网络拥有威瑞森“最可靠的网络”。即时手机幸福。
