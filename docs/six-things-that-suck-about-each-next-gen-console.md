# 每台下一代游戏机的六大缺点

![](img/be83802216fe2d7133e037588ea2f29a.png)

我觉得 Wii 很烂也不奇怪。这是一个可怕的控制台，但没关系。PS3 和 Xbox 360 都有很多缺陷，让它们和任何任天堂游戏机一样糟糕。Pixel Perfect Productions 编制了一份列表，列出了这些游戏机如此糟糕的原因。值得注意的是 360 包括可怕的“红色死亡之环”，微软的积分系统，以及缺乏内置无线网络。所有真正困扰 360 的关键问题。

Wii 的缺点是动力不足，Wiimote 缺乏充电电池，游戏缺乏在线体验。如果讨厌索尼是你的风格，那么至少要体面地讨厌 PS3 没有隆隆声的能力，并且有一个黯淡无光的游戏库，这是 100%正确的。也许如果游戏行业看一下这个列表，下次游戏机大战来临时，将会有一场激烈的竞争。
