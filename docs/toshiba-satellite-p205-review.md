# 东芝卫星 P205 评论

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/19/toshiba-satellite-p205-review/>

![](img/9944136dc2f4663b13b06a343cb81ca3.png)
被称为 P205 的指纹磁铁

我评测过的过去几款[东芝笔记本](https://web.archive.org/web/20140215172310/http://www.crunchgear.com/?s=toshiba+satellite)不如市场上的其他产品出色。在 Satellite P205 中，东芝包含了内置网络摄像头、英特尔酷睿 2 双核 CPU 和 DVD 刻录机等功能。东芝是否已经解决了过去用平庸的评价困扰他们的问题？请继续阅读，寻找答案。

第一眼看到 P205，你会发现它有很多端口、明亮清晰的显示屏和全尺寸键盘，很多人都会喜欢使用。毕竟，这台笔记本电脑里有相当多的好东西。运行频率为 1.73Ghz 的英特尔酷睿 2 双核 T5300 CPU、200GB SATA 硬盘、5 合 1 多媒体读卡器、17 英寸 WXGA TruBrite 显示屏、2GB DDR2 RAM 和带 Labelflash 的 DVD 刻录机。不算太寒酸。还有一系列 USB 端口、Firewire 端口、以太网端口、ExpressCard 插槽和 802.11b/g 无线。尽管这些功能看起来很棒，但使用这款笔记本电脑并不好玩。它慢得要命，从睡眠中唤醒它比正常情况下要花更长的时间。我会说这些问题是由于 Windows Vista 家庭高级版和缓慢的硬盘。200GB 固然很棒，但不是以每分钟 4200 转的速度。

![](img/f372fae966db19afdd8328f8d7b8e5f7.png)

在 P205 上有了一个糟糕的开始后，我认为从这里只能是上坡。东芝在过去有一个巨大的无线连接问题，我很高兴地告诉大家，Windows Vista 或东芝已经解决了这个问题。我没有连接到我的家庭网络或公共 WiFi 网络的问题。我很高兴东芝注意到这一点，并努力解决这个问题，因为它开始变得荒谬。这是 2007 年，WiFi 时代的中心。没有理由会那么难。

正当我开始对使用这台笔记本感觉良好时，我遇到了更多的麻烦。我认为内置的网络摄像头会很棒。毕竟，这是一台巨大的 17 英寸笔记本电脑。如果苹果的 13 寸 MacBook 内置了很棒的摄像头，为什么东芝的 P205 不能？不幸的是，对我来说，使用附带的软件渲染可怕的图像。达到了不高于 5 FPS 的帧率，使用十分钟后，我放弃了，把笔记本电脑扔在沙发上睡觉了。

顺便说一句，东芝仍然在他们的笔记本上安装大量的垃圾软件。我意识到这在大多数基于 Windows 的笔记本电脑中司空见惯，但对东芝来说，至少去掉 P205 中三分之一的垃圾内容就太棒了。我不应该花几个小时删除这些对我没有好处的应用程序。为什么不包括一张可以安装的软件光盘呢？问题解决了。

那些对多媒体感兴趣的人肯定会比我更欣赏 P205。它拥有出色的显示屏和体面的集成显卡，可以运行 Vista 的 Aero 功能或大多数基本游戏，您可以安装并玩一会儿。看电影、视频或任何流畅运动的东西都是美丽的，这要归功于这个怪兽的显示屏。由于电池续航时间相当长，它几乎值得我们花力气去推一台 8 磅重的笔记本电脑。

那么东芝有进步吗？过去的问题都过去了吗？嗯，是也不是。无线问题已经解决，但垃圾软件和缓慢的操作系统让这东西感觉像 1998 年的笔记本电脑。在某一点上，P205 变得完全无法使用，因为硬盘和 CPU 因为一个运行的网络摄像头应用程序而变得疯狂。现在，你可以花 1000 美元左右买到 P205，但我想说，如果你想运行 Vista，就省下你的钱，买一台 Macbook 或 Macbook Pro。你得到了网络摄像头、显卡和漂亮的屏幕，没有垃圾软件和设计缺陷。

对不起，东芝，但也许是时候回到绘图板很长很长一段时间。
