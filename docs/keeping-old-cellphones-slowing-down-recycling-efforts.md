# 保留旧手机会减缓回收工作

你们大多数人放在床底下的那台 Startac 的电池可以得到更好的利用。欧洲回收 45%售出的便携式设备电池的目标正在受阻，因为消费者习惯保留他们过时的技术。过去一年，欧洲回收的锂离子电池总重量下降了约 100 吨。

所以去回收 2006 年的手机吧。现在*那是*老派。
