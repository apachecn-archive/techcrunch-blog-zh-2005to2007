# Vonage，现在安全了，需要你的帮助

我们很少对大公司的努力表现出友好，但作为一个多年来快乐的 Vonage 用户，我会接受的。Vonage 刚刚被[批准延期](https://web.archive.org/web/20230322164155/http://www.sci-tech-today.com/news/Vonage-Gets-Stay--Has-No-Workaround/story.xhtml?story_id=12100DIDILGQ)就在事情看起来很惨淡的时候，可以继续营业了。不用说，公司很生气。他们通过一个新网站 [FreeToCompete](https://web.archive.org/web/20230322164155/http://www1.freetocompete.com/) 来表达这种愤怒，这是他们“长期专利改革运动”的第一步

这个网站是你的标准请愿软件，有一些证明和事实。我怀疑公众舆论的高涨会使这种情况向任何一方转变，但众所周知，更奇怪的事情已经发生了。这里真正的外卖是什么？“苏联各部”正在破坏竞争。大家伙们掌管着带宽、频谱和硬件，他们不想让任何人——甚至是小老沃纳奇——夺走国家和国际通信的摇钱树。老男孩们在基础设施和研究上投入了太多，不允许任何事情从他们已经令人印象深刻的底线上刮下几便士。

所以祝你好运，沃纳格。我们有时讨厌你，但这只是因为你，像 TiVo 一样，是美国创新中的两盏明灯，而在过去的几个月里，你们两个都让我们失望，因为技术发展缓慢，而且有很多关于老鼠跳上沉船的消息。显然，现在不是这样了，但是当偶像动摇时，每个人都会受到影响。
