# Windows Live Messenger 速度超出测试版

[![](img/ea6dcdfe486ca740279a2c924cfdb3f3.png)](https://web.archive.org/web/20221002130250/http://www.live.com/) 
第一款 [Windows Live](https://web.archive.org/web/20221002130250/http://live.com/) 产品全球脱离 beta 状态[将于今晚](https://web.archive.org/web/20221002130250/http://www.microsoft.com/presspass/press/2006/jun06/06-19LiveEraPR.mspx)由微软发布。Windows Live Messenger 公测版仅用了六周时间。当测试版发布时，我们回顾了现在可用的大多数功能。

最值得注意的是通过界面和专用硬件包括语音通话，离线信息和一个简单的文件共享系统。

在等待 Vista 发布的人们多年的嘲笑和微软痛苦缓慢的开发的普遍声誉之后，Live Messenger 测试版的发布可能是这个新的后盖茨时代运营严重转变的第一个迹象。当周四宣布[盖茨将放弃他作为首席技术架构师的职位](https://web.archive.org/web/20221002130250/http://www.beta.techcrunch.com/2006/06/15/ray-ozzie-to-replace-gates-as-chief-software-architect-2/)给前首席技术官雷·奥茨时，Live 被认为是公司未来的主要发展方向。

该公司表示，明年将在全球推出 20 多项新的 Windows Live 服务。
