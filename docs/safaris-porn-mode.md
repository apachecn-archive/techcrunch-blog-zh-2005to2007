# Safari 的色情模式

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/12/safaris-porn-mode/>

![safaripornmode.png](img/90b62b1afe7d296768dcea591665c077.png)

TUAW 刚刚在新的 Safari 3 beta 中挖掘出“隐私模式”，这是一个删除下载文件、不保留 cookies、不保留密码或个人信息的功能。作为一个偶尔喜欢某些类型的网站的人——好吧，每 15 分钟一次——很高兴知道有这样的网站。在离婚协议期间，这肯定会节省律师费。

若要找到它，请点按“Safari”>“私人浏览”。嘣。超级秘密。
