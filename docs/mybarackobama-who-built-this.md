# 这是谁建造的？

每个人都在[谈论](https://web.archive.org/web/20230213100308/http://www.techmeme.com/070211/p15#a070211p15)总统候选人巴拉克·欧巴马今天在他的[网站](https://web.archive.org/web/20230213100308/http://www.barackobama.com/)上发布的新社交网络。专家们对它的优点、遗漏之处等给出了很多意见。

我真正想知道的是，谁为他们建造了这个？它推出了基本上功能完整且无 bug 的产品，如果没有一个扩展的测试版，这是很难做到的。我敢打赌，其中一个白标供应商为他们建立了这一点，虽然没有迹象表明，我已经能够找到在服务条款或网站上的任何地方。

[人物聚合网站](https://web.archive.org/web/20230213100308/http://peopleaggregator.com/)的马克·坎特证实不是他们。可能是最近收购的 [Five Across](https://web.archive.org/web/20230213100308/http://www.fiveacross.com/) ，它运营着一个 NHL 社交网络。Digg 上的一位评论者说，它看起来像是在运行“Friendster 1.0 alpha 软件，使用一种说不出的咒语”，这很有趣，但我认为不准确。

如果有人知道谁的软件是幕后黑手，请告诉我们。他们刚刚举办了一场大型营销活动。
