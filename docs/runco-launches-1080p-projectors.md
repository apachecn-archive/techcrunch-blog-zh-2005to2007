# Runco 推出 1080p 投影仪

Runco 宣布扩大其已经令人生畏的家庭影院投影仪阵容。该公司将增加七个新系统，每个都具有原生 1080p (1920 x 1080)分辨率，以满足您的观看乐趣。这些设备的价格从单芯片 DLP 反射系列型号的 11，995 美元到旗舰 3 芯片 DLP Xtreme 型号的 115，000 美元不等。所有的放映机都可以配备 Runco 的专有技术 CineWide 或 CineWide with AutoScope。该公司表示，新系列将于 10 月上市。

我想我需要这些东西中的一个。想象一下，3 芯片 DLP Xtreme 型号将其 1080p 的画质投射到一个[平面 x 屏幕](https://web.archive.org/web/20210422213734/http://crunchgear.com/2006/08/15/plana-100-inch-screen-they-might-think-its-a-plasma-tv/)上。神魂颠倒。
