# Napster 花花公子在 WoW 社交网站工作

![](img/521a47dc0889ddfd5c1c748c706500d2.png)

当盗版音乐盛行的时候，一个叫肖恩·范宁的家伙做了一个叫 Napster 的小程序。它使得 P2P 文件共享和盗版在不到一年的时间里激增。所以在肖恩卖掉 Napster，得到一些现金，并被起诉后，他显然坐下来，真正进入了这个你可能听说过的游戏:*魔兽世界*。

范宁现在在一个名为[破裂](https://web.archive.org/web/20220130092445/http://www.rupture.com/)的社交网站上工作，该网站以《魔兽世界》中的一个[流氓天赋](https://web.archive.org/web/20220130092445/http://www.thottbot.com/?sp=1943)命名。这个概念是合理的，将 WoW 玩家聚集在游戏之外是一个很好的主意，但是 Fanning 到底想做什么？

 他对自己的想法说了如下的话:

> “使用附加组件或软件下载，破裂进入游戏，自动汇集角色名称、个人资料和资源，并将其发布在个性化网站上。破裂也将汇集统计数据，以创建个人和公会排名，并为公会提供一个地方来组织他们的游戏。随着破裂跟踪每个成员随着时间的推移，这些个性化的档案演变。玩家可以在小组中或与其他人聊天，下载其他插件和游戏演示。”

听起来像是混合了 [MySpace](https://web.archive.org/web/20220130092445/http://www.myspace.com/) 的 [Thottbot](https://web.archive.org/web/20220130092445/http://www.thottbot.com/) 的改进版。我会咬，现在。
