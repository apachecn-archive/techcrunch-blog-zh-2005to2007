# 将任何 YouTube 视频放入你的 iPod

> 原文：<https://web.archive.org/web/https://techcrunch.com/2006/11/14/put-any-youtube--video-on-your-ipod/>

YouTube 上有大量的好内容，但版权侵权指控和 DMCA 的撤下通知意味着任何给定的视频都可能随时消失。拥有视频的本地副本意味着如果 YouTube 从他们的网站上删除视频，你也不会失去访问权。如果你把视频转换成 MP4 格式，你也可以把它传输到你的 iPod 上。

这样做似乎符合 YouTube 的[使用条款](https://web.archive.org/web/20210421120304/http://youtube.com/t/terms)，只要你没有将视频分发给其他人，并且你保留了文件的任何版权声明(见 2A 和第 4 节)。

有两个非常好的免费服务，可以把 YouTube 视频下载到你的硬盘上，转换成 MP4 格式，然后添加到 iTunes 上。对于 Windows 用户来说，试试 iTube，这里有[这里有](https://web.archive.org/web/20210421120304/http://www.benjaminstrahs.com/itube.php)。Mac 用户应该使用 [PodTube](https://web.archive.org/web/20210421120304/http://djodjodesign.free.fr/rightEN.html) ，虽然它需要使用 Safari。

如果您只想访问文件并直接查看它(VLC 媒体播放器将播放 flash 格式的视频文件)和/或将其转换为另一种文件格式，您可以使用 [Oyoom](https://web.archive.org/web/20210421120304/http://www.oyoom.com/) 下载任何 YouTube 视频，或者尝试名为 [Video Downloader](https://web.archive.org/web/20210421120304/http://javimoya.com/blog/youtube_en.php) 的 Firefox 扩展。我们还编写了自己的工具来下载 YouTube 视频，[可以在这个页面](https://web.archive.org/web/20210421120304/http://www.beta.techcrunch.com/get-youtube-movie/)上找到，它也嵌入在下面。只需输入[任何](https://web.archive.org/web/20210421120304/http://www.youtube.com/watch?v=9AlH2oYedfk&eurl=) YouTube 视频网址，点击“获取视频”按钮。

对于真正的“自己动手”者，奥赖利的埃里卡·萨敦在这里有一步一步的介绍。

一旦你有了。硬盘上的 flv 视频，使用 VLC 观看，或将其转换为其他格式。对于 Windows 用户，试试[重播转换器](https://web.archive.org/web/20210421120304/http://versiontracker.com/dyn/moreinfo/win/66712)。对于 Mac 用户，我们推荐 [iSquint](https://web.archive.org/web/20210421120304/http://www.isquint.org/) 。也有一些[在线文件转换服务](https://web.archive.org/web/20210421120304/http://www.beta.techcrunch.com/2006/11/04/online-file-conversion-services/)，但是它们不像桌面软件那样可靠，你必须等待上传和下载。
