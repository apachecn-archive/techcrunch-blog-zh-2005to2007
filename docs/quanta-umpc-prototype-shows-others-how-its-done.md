# 广达 UMPC 原型向其他人展示它是如何做到的

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/20/quanta-umpc-prototype-shows-others-how-its-done/>

现在这才像话。台湾大型笔记本电脑制造商广达电脑(Quanta)开发了一款超便携电脑(UMPC ),我真的可以把它藏在身后。(不像我今天早上提到的[POS](https://web.archive.org/web/20170708013225/http://crunchgear.com/2006/10/20/u-g-l-y-ago7-umpc/)。)这是该公司首次推出基于英特尔欢跃平台的原型 UMPC，如果属实，这意味着它可以播放高清视频和 5.1 声道环绕声，支持千兆以太网和 RAID，并且必须在英特尔双核 CPU 上运行。与 Ubergizmo 报道的 Windows XP 平板电脑版本相反，它还需要在 Windows XP 媒体中心版本上运行。真的没关系。很高兴看到有人在做一些看起来确实值得取代笔记本电脑的东西。
