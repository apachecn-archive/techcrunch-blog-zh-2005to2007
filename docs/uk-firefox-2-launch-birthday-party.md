# 英国火狐 2 发布&生日派对！

几周来，我一直在和伊恩·海沃德——Glaxstar 的医学博士——谈论关于火狐 2 在伦敦的发布会。到目前为止，聚会的地点还没有确定，遗憾的是日期也没有确定，因为 Firefox 主题发言人的航班安排有一些变化。

但有一点我可以确定的是，场地将会在伦敦市中心的“某个凉爽的地方”(能够容纳大约 200 人)

所以在日期和地点都还没确定的情况下为什么还要提呢！？为了帮助这次伦敦活动的组织者，如果你能在 Firefox [派对策划网站](https://web.archive.org/web/20130627203701/http://www.firefoxparty.com/parties/view/13)上注册你的兴趣，那就太好了。当然，如果你不能去伦敦，那么目前在世界各地有 620 个派对正在组织中，超过 5000 人已经登记了他们参加一个或几个派对的兴趣。我本人也将于 11 月 14 日前往都柏林参加他们的聚会。当然，你可以成立自己的政党，如果有人想在中部或更北的地方办一个，请告诉我。

最后，谁是 Glaxstar？他们是一家位于伯明翰的开发公司，专门为蓝筹公司开发 Firefox pro-extensions。最近，他们开发了雅虎令人敬畏的新的美味书签替换工具，我现在在 Firefox 2 中使用它。在这个扩展出现之前，我已经忘记了美味和标签的价值。
