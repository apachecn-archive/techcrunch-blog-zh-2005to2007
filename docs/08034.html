Ultra带来了快速、大而便宜的SDHC卡| TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2006/12/15/ultra-brings-sdhc-cards-fast-large-and-cheap/

Ultra，一家可能以其电脑外壳和电源而闻名的公司，今年已经向市场推出了许多额外的设备。今天早上，它宣布了一种8GB的SDHC卡，可以在任何可以使用高容量卡的设备上使用，价格为149.99美元。

除了容量大小，还有每秒18MB的读取速度和每秒3MB的写入速度，所以如果它在你的相机中，它应该以合理的剪辑存储快照。
