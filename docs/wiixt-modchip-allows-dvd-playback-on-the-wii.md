# WiiXT Modchip 允许在 Wii 上播放 DVD

![](img/577d5cdcb0e48bbc4ca150e536826b9d.png)

由于看不到 PS3 modchip，黑客和修补者似乎已经开始彻底利用任天堂 Wii。WiiXT 是最新推出的 modchip，被其创造者吹捧为“终极 Wii Modchip”。为什么？也许是因为它允许任何 Wii 或 Gamecube 备份播放，保持 VC 支持，并允许 DVD 的无区域播放。

这看起来像是一个坚实的努力，但直到我看到其中一个在专业动作回放旁边出售，我才称之为 vaportrash。只需阅读这些直接发布到网站上的内容:

> 我想转售/购买/试用 Wii XT——目前我们没有资金开始生产，要知道我们已经用砖砌了无数的 Wii 工作在这个芯片上，并且由于缺乏供应，购买更多的 Wii 会非常困难。当我们接近一个设定的发布日期时，在这个网站上会有更多的经销商和样品商的信息。

说实话，我很确定这只是一个 14 岁刚学 HTML 的孩子的假话，但是继续，证明我错了。
