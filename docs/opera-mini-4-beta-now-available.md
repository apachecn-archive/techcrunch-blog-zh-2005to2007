# Opera Mini 4 测试版现已推出

![opera.jpg](img/470d44ba21ac2094678e14d6bb3d72b0.png) Opera 软件即将进入一个新的维度。该公司刚刚发布了 Opera Mini 4 移动网络浏览器的测试版，代号为 Dimension。这个新版本基于任天堂 Wii 手机浏览器上引入的原始 Opera 功能，具有缩放选项。

Opera 首席执行官 Jon von Tetzchner 表示:“我们很高兴与快速增长的 Opera Mini 用户社区分享测试版。“测试版用户的反馈将使 Opera Mini 变得更好，我们感谢社区的努力。Opera Mini 的成功完全归功于人们的支持，正是他们让它成为了当今最受欢迎的移动浏览器。”

新的 Opera Mini 被设计成几乎可以在任何手机上运行，允许用户浏览他们喜欢的网站。
