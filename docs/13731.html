Sun用JavaFX回应AJAX、Silverlight

在 Infoworld 和 CNet News 报道，Sun将在今天的 Java One大会上推出一个基于Java的产品系列，名为JavaFX。

该声明紧随最近广受欢迎的微软Silverlight发布，作为一个产品，它也将直接与AJAX竞争。

JavaFX据说是Java的一个新的脚本变体，专注于消费者通信市场的开发，包括桌面、移动客户端和电视。第一个发布的产品是JavaFX Mobile，一个用于移动设备的软件系统。

在演示结束之前，我们不会对新产品有更多的了解，但我的第一想法是，我们在这里看到的是Sun的追赶游戏。面向对象的应用程序编程语言Java曾经是最酷的产品，新一代应用程序将在此基础上推出，接管我们的桌面。然而这从未发生过。这款专注于移动技术的新产品，如果今天受到开发者的欢迎，有可能让Sun重新回到应用游戏中来。
