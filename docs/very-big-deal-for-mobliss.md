# 对 Mobliss 来说非常重要

去年，Mobliss 完成了这笔交易，他们发布了热门智力竞赛节目*的手机版《一锤子买卖*。Mobliss 宣布，Endemol show 的手机游戏版本在美国的下载量已经超过了一百万次。Mobliss 也与 FremantleMedia 合作推出了移动电视频道 Atomic Wedgie，但它也没有完成交易。

Mobliss 项目管理副总裁约翰·洛斯基(John Loschky)表示，他预计这款游戏在美国的受欢迎程度将在其他地方重现。“随着电视游戏的持续成功，我们预计随着时间的推移， *Deal or No Deal* 手机游戏的订阅和下载量只会增加，不仅在美国，而且在国际市场，因为有了新的西班牙语版本。”
