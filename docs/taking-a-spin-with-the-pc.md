# 与电脑一起旋转| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/01/11/taking-a-spin-with-the-pc/>

许多游戏玩家已经流了太多的汗，但这更多的是因为糟糕的卫生条件，而不是游戏过程中的体力消耗。虽然 Wii 会让你的胖屁股离开沙发，但要真正感受到这种燃烧，还有 PCGamerBike，这是一款通过 USB 连接的迷你健身车，可以用于各种 PC 游戏。它实际上是 3D 创新岛世界的一个组成部分，这是第一个将健身设备融入体验的游戏，但它可以让你在玩体育和动作游戏时旋转。
