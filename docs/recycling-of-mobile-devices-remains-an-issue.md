# 移动设备的回收仍然是一个问题

这是一个我们不想看到的数字……在澳大利亚，去年只有 3%的废弃手机被回收。随着 870 万部手机被运往地下，大量手机可能会被扔进垃圾填埋场，因为普通用户每 18 个月到两年就会升级一次手机。根据澳大利亚移动通信协会最近发布的数据，手机回收的比例实际上正在下降！该组织现在呼吁世界各国政府让手机行业对收集废弃手机更加负责。

然而，该协会的研究表明，许多消费者将旧手机作为备份或备用。正如那些有一个旧抽屉的手机的人所知道的，也有一些项目可以将手机捐赠给那些需要的人。
