# NTT DoCoMo 发布 3G 老大哥手机

小心日本，你的健康 T2 NTT DoCoMo 的 3G FOMA F903iBSC(wtf)“商务用”手机抑制了你对色情的不健康沉迷。该手机的推出是为了应对信息泄露和非工作相关的使用公司发放的设备。NTT 真的在这一个上施加压力，因为照相机，外部照相机，USB/IR 端口和电子钱包服务被禁止。唯一的工作功能是一个有限的电话簿，只能容纳 101 个联系人，电子邮件，日程安排，数据框(？)、通话记录和备忘录，如果手机丢失，这些都可以找回。进一步的安全增强包括生物识别刷卡解锁手机一旦关闭。3 月 19 日在日本下降。当心美国公司，我们可能是下一个。
