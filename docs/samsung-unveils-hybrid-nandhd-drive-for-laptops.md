# 三星为笔记本电脑推出混合 NAND/HD 驱动器

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/08/samsung-unveils-hybrid-nandhd-drive-for-laptops/>

三星在 MH80 系列笔记本电脑硬盘中有一些有趣的东西。随着行业[开始将目光转向基于 NAND 的便携式闪存驱动器](https://web.archive.org/web/20140701073716/http://crunchgear.com/2007/03/07/wu-apple-going-solid-state-for-all-ipods-portables/)，而不是标准的基于磁盘的驱动器，三星开发了一种混合驱动器，将 80-、120-或 160-GB 标准硬盘与 128-256-MB NAND RAM 耦合在一个物理驱动器中。闪存 RAM 部分充当读/写任务的超级缓冲器；当信息从物理磁盘中取出时，它被存储在 NAND 中，无需物理磁盘不断旋转即可访问。由于驱动器上的传输几乎是即时的，NAND 充当了一个低功耗、快速的地方，将所需的信息“袖手旁观”以供访问，这意味着物理磁盘可以休息，从而节省宝贵的电池电量。

这是驱动器发展的下一步，旨在与微软的 ReadyDrive 配合使用，ready drive 是 Vista 中一个很少使用的组件，可以快速启动和关闭。不过，这款硬盘应该能提高任何笔记本电脑的性能，不管它运行的是什么操作系统，而且应该会在初夏上市，价格不会比现在的硬盘售价高多少。
