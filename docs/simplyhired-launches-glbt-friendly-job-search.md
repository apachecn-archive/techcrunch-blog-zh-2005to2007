# Simplyhired 推出 GLBT 友好求职

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/06/14/simplyhired-launches-glbt-friendly-job-search/>

[![](img/03f162ab844bdaa89bc7353ca282cd8f.png)](https://web.archive.org/web/20150919063409/http://www.simplyhired.com/) 
为了庆祝全国同性恋月，在线求职服务[simple hired](https://web.archive.org/web/20150919063409/http://simplyhired.com/)推出了 [GLBT(同性恋、双性恋、跨性别者)友好求职功能](https://web.archive.org/web/20150919063409/http://simplyhired.com/glbt)。人权运动给公司内部的新过滤搜索打了满分。

SimplyHired 指出，美国联邦政策没有禁止基于性别认同或性取向的就业歧视，34 个州的工人可以随时基于这些因素被解雇。这是不对的，很高兴看到一家初创公司开始帮助解决这种情况。

SimplyHired 总部位于加利福尼亚州的山景城，它还为妈妈友好型工作、狗友好型工作和其他类型的利基工作要求提供过滤器。
