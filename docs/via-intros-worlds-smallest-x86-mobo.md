# VIA Intros 全球最小的 x86 Mobo

你准备好…迎接世界上最小的个人电脑主板了吗？！威盛宣布推出 VT6047 微微 ITX 全功能世界上最小的 x86 主板。

微型 ITX 只有 10 厘米 x 7.2cm 厘米，比它的前身 17 厘米 x 17cm 厘米小得多。由于在电源管理、热效率和功能融合方面的进步，威盛以其在芯片层面的专业知识和创新而闻名。一如既往，当与 C7 处理器或无风扇 Eden 处理器等其他 VIA 产品配合使用时，Pico-ITX 将发挥其全部潜力。没有价格或上市日期的消息，但我们会随时通知你。
