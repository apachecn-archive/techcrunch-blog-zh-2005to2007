# 威瑞森放弃低端翻盖式 Windows Mobile 智能手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/02/verizon-drops-low-end-clamshell-windows-mobile-smartphone/>

看起来普通的翻盖手机可以访问你的 Exchange 服务器，能够从你的公司网络读取 Office 文档，以及 WinMo: Smartphone Edition 的所有其他(难以使用的)优点。

除了摇动 MS OS，VZN 翻盖手机还配有一个带闪光灯的 130 万像素摄像头(漂亮！)、蓝牙和 EV-DO。此外，您还可以获得一个用于音乐的 microSD 卡插槽。由于 WinMo 配备了 Windows Media Player 的微型版本，这款手机符合[的传言，即 VZN 正在将其音乐战略从 V-Cast 服务转向微软的 PlaysForSure](https://web.archive.org/web/20151001233423/http://crunchgear.com/2007/01/29/verizon-ditching-v-cast-for-playsforsure-for-musicphones/) ，这意味着你从任何 PlaysForSure 商店下载的音乐都可以在这款 149 美元的 Pantech 上播放。
