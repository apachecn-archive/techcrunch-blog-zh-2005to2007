# Timbuk2 博客

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/21/timbuk2-blogger/>

Timbuk2 发布了一款名为 Blogger 的新电脑包。它采用防弹尼龙外壳，三面板设计，内置填充笔记本电脑隔间，两个外部口袋和橡胶底部。这个包可以装下一个 15 英寸的 Powerbook，所以如果你想携带一些笨重的便携式电脑，你应该去别处看看。

Blogger 看起来是一个不错的小袋子，但要携带这个名字，我认为它应该能够携带一台单反和一台笔记本电脑。没有图片的博客算什么，对吧？对吗？！
