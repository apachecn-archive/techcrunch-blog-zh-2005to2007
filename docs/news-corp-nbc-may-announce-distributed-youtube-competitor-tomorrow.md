# 新闻集团、NBC 今天可能宣布分布式 YouTube 的竞争对手；谷歌嘲笑

几个月来，一直有传言称，一家电视网络牵头的合资公司将与 YouTube 展开竞争——我们第一次听到这个消息是在谷歌收购该公司的第一次传言[之后几天。从那以后，我们听说由于一个或另一个合伙人退出了小组，讨论陷入了僵局。](https://web.archive.org/web/20230125125717/http://techcrunch.com/2006/10/06/completely-unsubstantiated-googleyoutube-rumor/)

谣言现在卷土重来，来势汹汹。[洛杉矶时报](https://web.archive.org/web/20230125125717/http://www.latimes.com/business/la-fi-youtube22mar22,0,326504.story?coll=la-home-headlines)现在报道说一笔交易可能很快就会宣布，**并且这也由一位接近交易的消息人士直接向我们证实**。最早可能在周四宣布。

很明显，新闻集团和 NBC 也参与其中，维亚康姆和索尼也有可能参与其中。这不会是 YouTube 的直接竞争对手，因为重点将是向第三方网站分发内容:与雅虎、微软、美国在线和新闻集团的 MySpace 的交易可能已经到位。对于这些内容，可能有也可能没有专门的网站。

因此，总而言之，这是我们目前所知道的:这正在发生，并计划最早在周四发布新闻。除了新闻集团和 NBC，我们不知道还有谁参与其中。这项服务的实际推出时间是 2007 年夏天或更晚。

我认为这将得到疯狂的关注，然后可能会很晚才推出。众所周知，合资企业很难管理，将第三方分销合作伙伴加入其中会增加复杂性。当然，技术需要工作，这些公司现在不是为了建立基于网络的应用程序。谷歌似乎也不是特别担心:据报道，高管们称该项目为“小丑公司”
