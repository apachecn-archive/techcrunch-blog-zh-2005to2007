# 松下 Lumix DMC-FZ8:350 美元的长变焦镜头

徕卡 DC 镜头是松下 Lumix DMC-FZ8 的明星，它有 8 组 11 个元素，包括 3 个非球面镜头。12 倍光学变焦镜头是去年 DMC-FZ7 的延续，但现在支持 720 万像素传感器和新增强的 Venus III 处理器。还增加了原始模式。

 它不会进入你的口袋(除非你有一些大口袋，坦白说，这只是令人不安)，但相机整体紧凑，拿在手中感觉很好，非常平衡。

2.5 英寸 LCD 右上角的小操纵杆 nubbin 用于手动调整对焦和曝光设置。这绝对是相机的一个亮点，在你的拇指可以完美触及的地方。由于相机重量很轻，你可以使用它而不会感到相机会掉下来。

模式转盘可让您轻松旋转选项，包括以 30fps 的宽屏幕 VGA (848×480)录制的宽屏视频模式。
