# SoundTech LightSnake iChat USB 耳机和麦克风:苹果知道这件事吗？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/22/soundtech-lightsnake-ichat-usb-headset-and-mic-does-apple-know-about-this/>

[![ichatmic.jpg](img/4b92678e5d34103f09c7ee412170126f.png)](https://web.archive.org/web/20150911191841/http://old.crunchgear.com/wp-content/uploads/ichatmic.jpg "ichatmic.jpg")

我相信你以前听说过 [SoundTech](https://web.archive.org/web/20150911191841/http://crunchgear.com/2006/10/31/soundtech-announces-lightsnake-usb-intelligent-microphone-cable/) 。它制造各种各样的音频相关电缆等，最新的是 LightSnake iChat，这是一款 USB 耳机和麦克风，专为日常任务设计，如游戏和在 Skype 上打扰朋友。这里一个很好的梳妆台功能是“点亮时带电”，这使得该单元在处理声音时疯狂闪烁，而在简单通电时发光。我想你可以用这个假装成某种科幻英雄，与熟人交谈，然后惊叹它壮丽的光芒。

总之，这是一款坚固的耳机，来自一家倾向于不制造垃圾的公司。由于它是 2007 年的 USB 产品，它可以与 Windows、save [Vista](https://web.archive.org/web/20150911191841/http://crunchgear.com/2007/05/21/criticize-how-much-it-costs-to-run-vista-decently-and-get-beaten-up-by-readers/#comment-229779) 和 Mac 兼容，无需安装繁琐的驱动程序。
