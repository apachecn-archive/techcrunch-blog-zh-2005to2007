# 先锋想杀你

![](img/2a1d6466264653ccc9287d9076f57359.png)

我一直认为在你的汽车仪表板上放一个 DVD 播放器就像你在走钢丝的时候有人和你说话一样。看起来你只应该在你开很远的距离时使用它——或者在小便时以极高的速度行驶。要么这样，要么把它放在后座给孩子们。

先锋似乎不同意。以非常非常大的方式。他们为你的汽车引进了一个新的娱乐中心。我说的是 7 英寸宽的 TFT-LCD (234 x 480)屏幕，GPS 和导航系统，30GB 硬盘，DVD/CD-RW(出于某种原因只能以 Atrac3 格式刻录)，以及 50W×4ch 放大器，现在有一个 FM 调谐器，但最终会有一个 iPod 连接。你知道吗，我希望这真的能成功。我希望这个月晚些时候这个东西上市的时候全日本都能买。我还希望每个人都使用仪表板上的按钮，而不是附带的遥控器。遥控器是给胆小鬼用的。
