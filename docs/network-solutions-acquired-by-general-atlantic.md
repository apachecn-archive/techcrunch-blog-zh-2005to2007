# 通用大西洋公司收购的网络解决方案

我们刚刚得到消息，域名注册商(原来的那个)[网络解决方案](https://web.archive.org/web/20221001074714/http://www.networksolutions.com/)被[私募股权公司通用大西洋收购](https://web.archive.org/web/20221001074714/http://home.businesswire.com/portal/site/google/index.jsp?ndmViewId=news_view&newsId=20070206005943&newsLang=en)。价格尚未透露，但很快就会泄露。

网络解决方案是最初的域名注册商，也是 Verisign 的一部分，直到 2003 年从 Verisign 剥离出来。从那时起，他们已经失去了大量的市场份额，像 GoDaddy，eNom 和其他折扣业务。他们现在管理着大约 660 万个域名。
 ***更新:**正如我所怀疑的，价格漏的到处都是。有传言说这是一笔 8 亿美元的交易。*
