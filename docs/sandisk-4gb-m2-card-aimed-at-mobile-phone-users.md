# SanDisk 4GB M2 卡面向手机用户

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/27/sandisk-4gb-m2-card-aimed-at-mobile-phone-users/>

我不是那种为了媒体目的而使用手机的人，因为手机上的软件程序太糟糕了。我从未在我的 SK3 上使用过 MP3 播放器，我的诺基亚 5300 甚至还没有问世。内存不足是我不使用这些功能的主要原因，但 SanDisk 4GB Memory Stick Micro line 可能会改变我的想法。

虽然它面向索尼爱立信 Cyber-shot 和 Walkman 系列，但它也适用于任何带有 M2 插槽的手机。它还向后兼容所有带记忆棒插槽的数码和手持游戏播放器。4GB M2 卡要到今年晚些时候才会上市，目前还不知道价格，但它将在 5 月份发货给原始设备制造商。
