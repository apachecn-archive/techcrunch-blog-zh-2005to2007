# track stick Pro:GPS Snoop/Tracker 

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/08/trackstick-pro-gps-snooptracker/>

披萨店经理有一个很好的方法来确保他们年轻的送货员在工作时吸烟。这种跟踪棒是一种可安装的 GPS 设备，可以记录位置和温度，间隔时间长达 60 分钟。最旧的历史被更新的数据所取代。然后，您可以将设备放在支架上，并创建该设备特定路线的 Google Earth 覆盖图。伟大的保持你的司机诚实，并成为一个小微观管理者与一块燧石为心。

这些设备既可以连接到点烟器，也可以单独连接到汽车的电气系统。还有一个电源故障报警器和用于粘贴防篡改胶带的垫圈。没有定价，但预计这些都是几百元，这取决于您的车队规模。
