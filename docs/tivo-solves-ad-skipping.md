# TiVo 解决广告跳过？–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/28/tivo-solves-ad-skipping/>

![](img/987e4f2fd1b59884fe8bbd64edf45baa.png)

自从 TiVo 服务在本世纪初出现以来，电影公司在这方面就一直存在问题。争论的焦点在于 TiVo 给用户提供了轻松跳过广告的能力。这降低了广告成本，因为当潜在消费者可以掩饰他们的广告时，广告商不太愿意花大钱。

TiVo 下周将开始提供出现在录制节目结尾的小链接广告。当用户看完某个内容后，他们可以选择删除、保存或观看广告。

这似乎是一石二鸟的心态。它试图通过突出显示广告来安抚工作室和广告商，同时试图让消费者对边缘化的广告感到满意。它将在宽带连接系列 2 接收器上工作。我想知道它是否会成功。
