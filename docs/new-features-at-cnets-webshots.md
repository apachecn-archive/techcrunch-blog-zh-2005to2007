# CNET 在 TechCrunch 推出“大学直播”

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/04/04/new-features-at-cnets-webshots/>

[](https://web.archive.org/web/20220516081613/http://www.webshots.com/)，一家照片发布和分享服务公司，于 2004 年 8 月被 CNET 收购，开始推出社交网络功能，与 Flickr 等更前沿的照片服务竞争。上个月，他们推出了社交网络功能。今天晚些时候，他们将推出一项名为“大学直播”的新服务，目标是美国约 3000 万 18-25 岁的年轻人。

College live 是一个专门针对大学的网络照片网站，主要关注该大学学生的照片和活动策划。它只面向学生，Webshots 使用了和脸书一样的方法，试图保持一种独特的感觉——要求使用 edu 的电子邮件地址。这个想法是围绕这些照片和事件创造一种社区感，让学生们互相认识。它的目标观众和脸书一样，但提供不同的功能。虽然上传到特定大学网站的照片会被分组，但个人照片也会出现在网络照片上。

Webshots 是一个非常大的资产，拥有大约 4000 万总用户和 3 亿张照片。顺便说一下，CNET 作为一个整体仍在以健康的速度增长——去年第四季度，他们平均每月有 1.16 亿独立访客，比 2004 年同期增长了 13%。平均每日页面浏览量增加到 1.03 亿，比 2004 年增长了 22 %。CNET 表示，他们希望成为网络空间的维亚康姆，拥有许多独立的酒店，面向不同的市场。除了 Webshots，最近对约会网站[消费网站](https://web.archive.org/web/20220516081613/http://www.consumating.com/)和美食网站 [Chowhound](https://web.archive.org/web/20220516081613/http://www.chowhound.com/main.html) 的收购显示了这一战略的实施。Webshots 是帮助他们实现这一目标的另一个(非常大的)资产。
