# 吐司 8:TiVo-Tastic 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/23/toast-8-tivo-tastic/>

![](img/d3a1712a87f9f074f01033a7d2a9e644.png)

刚拿到盒装版的[吐司钛 8](https://web.archive.org/web/20221006053120/http://crunchgear.com/2007/01/08/roxio-steps-in-to-make-tivotogo-for-mac-a-reality/) ，我很困惑，但又开心，又害怕。我一直很喜欢吐司——它简单、直接，而且可以轻而易举地刻录固态硬盘。但是这些数字录像机是什么？

出于某种奇怪的原因，Toast 8 附带了一个 OS X 的 TiVo 传输程序。你会问为什么？因为 TiVo 不能完成他们自己的 OS X 版本，所以寻求 Sonic 的帮助来将软件构建到他们的最新版本中。这对那些有 TiVo 却没钱买 Toast 8 的人来说意味着什么？你有[替代品](https://web.archive.org/web/20221006053120/http://crunchgear.com/2006/12/05/how-to-download-tivo-content-to-macs/)。但是如果你想要官方的编码和解码尽量不要大惊小怪，你就有点卡住了。Roxio 为那些不想使用命令行或 Applescript 窗口的人开发了一款强大的基本产品。

TiVo 传输程序相当简单。你只需键入你的媒体访问密钥，选择你想工作的 TiVo，然后挑选几个节目。您可以将节目发送到 Toast 进行刻录或导出到 PSP 或 iTunes。该界面大大升级了一些常用的任务和设置，而不是隐藏在标签窗口中。

那么该不该升级呢？如果你有录像机，是的。我怀疑，这只是猜测，有人会破解 Toast 8 的 TiVo 功能，或者 TiVo 本身会在 Roxio 收回投资后提供一次性下载。我赌前者胜于后者。

除此之外，Toast 8 进行了相当大的升级，蓝光刻录、一键式 DVD/CD 复制、文件恢复、音乐控制软件和照片光盘刻录是一些主要的改进。该软件目前在线售价为 79.99 美元，所以看看吧。
