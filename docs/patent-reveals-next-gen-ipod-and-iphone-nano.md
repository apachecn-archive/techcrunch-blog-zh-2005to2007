# 专利揭示下一代 iPod 和 iPhone Nano

![](img/247f7163f0a31b55a670c40a8f58ffac.png)

现在，iPhone 几乎已经到来，苹果公司正在着手开发新版本的 iPod，可能还有 iPhone。最近的一份专利申请披露了一种新的全屏 iPod，它使用位于设备背面的触摸控制。用户将沿着背部滑动他的拇指来浏览菜单，同时施加压力来充当“点击”或“选择”。感觉我到目前为止？
 这项技术非常有意义。如果你有一个 iPod Nano，抓住它，看看当前的屏幕。你的大拇指会盖住显示屏，所以这么大尺寸的设备上的触摸屏不会真的工作。看起来臭名昭著的 iPod clickwheel 也将以虚拟化版本回归。看看专利图的右上角，你会看到一个数字小键盘，这可能意味着 iPhone Nano 或 iPod Nano 具有数字输入功能。不管怎样，现在最好的办法就是坐下来，放松，等着看这一切会变成什么样子。
