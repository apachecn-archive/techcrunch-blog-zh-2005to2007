# 飞利浦触摸屏留言板

![](img/2106e3fc4c6b962f8adaf37edef06b42.png)

这是一个很好的想法，可能并不太新。尽管如此，飞利浦还是推出了一款概念产品，这款产品本质上是一款大型镜面触摸屏，旨在用作家庭和办公室的留言板。

该设备显示其各种用户的缩略图头像。你只需点击你想给其留言的用户，该板允许你用手写笔写一条信息，或者你可以用板上的摄像头和麦克风录制视频信息。它还可以接收彩信和短信，以防你需要在最后一刻发送一条消息，告诉你晚饭要迟到了。很酷的东西。
