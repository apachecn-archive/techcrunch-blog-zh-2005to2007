# Time.com 增加了“Sphere It”链接

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/05/22/timecom-adds-sphere-it-links/>

总部位于旧金山的博客搜索引擎 [Sphere](https://web.archive.org/web/20220818231537/http://www.sphere.com/) 成立还不到一个月，已经有了嵌入在许多 time.com 文章中的“Sphere It”功能的链接([例子](https://web.archive.org/web/20220818231537/http://www.time.com/time/nation/article/0,8599,1192151,00.html))。Sphere 的首席执行官 Tony Conrad 说，他们目前正在和 Time.com 一起测试这项功能。

这些链接放在文章标题下面的显著位置，并直接链接到与该主题相关的 Sphere 博客搜索结果。我在我最初的关于 Sphere 的简介中写了关于 Sphere It 的内容——[,在靠近结尾的](https://web.archive.org/web/20220818231537/http://www.beta.techcrunch.com/2006/05/01/new-blog-search-engine-sphere-launches/)的“更多工具”下面。它是一个 Sphere 特性，允许用户从任何 URL 找到相关的博客内容。使用它最简单的方法是将 Sphere It bookmarklet 安装到浏览器中。在任何网页上点击它，就会出现相关的 Sphere 博客搜索结果。

与 Technorati 的“Technorati This”功能不同，它显示链接到被搜索的 URL 的博客条目，而 Sphere It 不报告链接。相反，它对被搜索页面中的文本进行语义分析，并返回与文章相关的博客结果。更多关于 Sphere It 和 Technorati This 的区别，请看 Sphere CEO Tony Conrad 的博客文章[这里](https://web.archive.org/web/20220818231537/http://sphere.wordpress.com/2006/05/12/week-one-in-the-rearview-mirror/)。

看一个例子，上面这篇文章的 Sphere It 结果，关于 Michael Hayden 将军被提名领导中央情报局，返回[这些](https://web.archive.org/web/20220818231537/http://www.sphere.com/search?partner=time&q=sphereit:http%3A//www.time.com/time/nation/article/0%2C8599%2C1192151%2C00.html) Sphere 博客搜索结果。快速浏览显示了我认为非常相关的博客内容。实时搜索的速度也令人印象深刻。

虽然这种类型的曝光对新的创业领域来说是极好的，但对博客领域来说也是极好的。我可以想象，普通的 Time.com 读者并没有完全意识到博客在突发新闻和(通常)睿智评论方面能提供什么。随着时间的推移，这应该会驱使更多的主流互联网用户进入博客世界。
