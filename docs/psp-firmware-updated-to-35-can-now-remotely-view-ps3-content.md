# PSP 固件更新到 3.5，现在可以远程查看 PS3 内容

[![pspxmb.jpg](img/1251768a4daf7d587c8b8527250f45d3.png)](https://web.archive.org/web/20210417145341/https://beta.techcrunch.com/wp-content/uploads/2007/05/pspxmb.jpg "pspxmb.jpg")

索尼在日本发布了 [PSP](https://web.archive.org/web/20210417145341/http://crunchgear.com/2007/05/04/top-five-psp-mods-that-dont-involve-throwing-it-out/) 固件 3.5，用户可以远程访问他们的 [PS3](https://web.archive.org/web/20210417145341/http://crunchgear.com/2007/05/30/sony-needs-to-drop-ps3-price-by-200-or-its-finished-bank-of-america/) 内容。只要它连接到 Wi-Fi 热点，PSP 就可以查看 PS3 上的所有视频、音乐和图片。上周的 PS3 固件更新让这一切成为可能。

最后，一个体面的 PSP 硬件利用，直到现在真的没有发挥其潜力。获得 PS3 仍然需要你重新配置你的债务，但是如果索尼想让 PS3 成功，让 PSP 起死回生，它就应该推销这种类型的“协同效应”。

现在带来 [OE 版本](https://web.archive.org/web/20210417145341/http://dark-alex.org/)。
