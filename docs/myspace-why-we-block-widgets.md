# MySpace:我们为什么屏蔽小工具

[![](img/bc6cbcaa61ccbc8f739a370b5a47423a.png) ](https://web.archive.org/web/20220818084538/http://www.myspace.com/) MySpace PR 回复了我们对周末报道的 [Imeem 堵塞](https://web.archive.org/web/20220818084538/http://www.beta.techcrunch.com/2007/02/25/imeem-blocked-from-myspace/)事件的评论请求。福克斯互动公司(MySpace 的母公司)SVP 企业传播部的朱莉·亨德森说:

> 如果一个小工具违反了我们的服务条款，我们会阻止他们。违规行为包括任何侵犯版权、带来安全风险、传播色情内容或参与商业活动的个人、小工具或软件。商业活动包括通过他们的小工具或软件在 MySpace 页面上销售广告。
> 
> 特别是在 Revver 的例子中，我们告诉他们，如果他们继续在我们的页面上销售广告，我们将阻止他们。他们拒绝停止在我们的页面上销售广告，所以我们屏蔽了他们。一点都不神秘。
> 
> 此外，我们目前或未来都没有收取“过路费”的计划。第三方小工具提供商只需遵循我们的服务条款…

新闻集团的首席运营官彼得·切宁去年发表的声明[仍然有问题:“如果你看看几乎所有的 Web 2.0 应用程序，无论是 YouTube，Flickr，Photobucket 还是任何下一代的 Web 应用程序，几乎所有的应用程序都是从 MySpace 上脱离出来的。”目前，MySpace 表示，他们没有计划向微件提供商收取网站分销权费用。但很明显，如果他们能找到禁止它们的借口，他们会的。](https://web.archive.org/web/20220818084538/http://www.beta.techcrunch.com/2006/09/12/myspace-we-dont-need-web-20/)
