# 魔兽世界:燃烧的远征第一印象

![](img/d8f9791249250f080ddf7b0b950dee16.png)

昨天我逛了一趟 GameStop，拿了魔兽世界*，*燃烧远征*的资料片。当我询问*燃烧的远征*时，他们说如果我没有预订，我就不能得到副本。哇，我想 800 万订户是件大事。我很失望，但并不愚蠢，我穿过街道去塔吉特百货，那里有大约 50 万本书出售。我付了 40 美元，回家安装了它。*

 *在我升级了我的账户并下载了一些补丁后，我准备好了。问题是，我的最高角色是一个 53 级的人类盗贼。因此，像 61-70 级和外域之类的东西在那个时候对我来说是不可及的。因为这差不多是扩展的一半，我觉得我已经损失了 20 美元。但后来我想起了燃烧远征有两个新种族:德拉内和血精灵。

![](img/07d41220fdc1beea026b97a85d4de39f.png)

我在我的主联盟服务器(Anetheron)上创建了一个 Dranei 角色。我给他起名叫费里斯布勒，到了 3 级后，我想起了为什么我尽量不在游戏中创造新角色。有点厌倦了德拉内，我决定尝试我的第一个部落角色:血精灵。我的血精灵沙蛛是圣骑士，长得真的很牛逼。他们看起来像愤怒的人肉色暗夜精灵。我真的很喜欢他们给我的新手任务，因为血精灵是一个全新的种族(连同德拉内人)，很多人都在周围捣乱。做一个血精灵很有趣，我完全可以看到更多的人使用血精灵而不是 Dranei。
