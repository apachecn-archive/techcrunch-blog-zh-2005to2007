# 先锋计划李书行第一印象

![pioneer-pdp-5010fd-flat-panel-display.jpg](img/a6b9f5cbd65933a592d9bc622dce9aaf.png)

我刚看完新的先锋高清电视回来，哇，我印象深刻。不过，720p 和 1080p 画质之间的差异几乎看不出来。细节与我们之前的[帖子](https://web.archive.org/web/20210302090347/http://crunchgear.com/2007/05/08/pioneer-8th-gen-hdtv-kicks-out-200001-ratio/)相同，但现在我们有了定价和发布日期。

**更新:**最佳模式不仅可以根据周围的光线调整画面亮度，还可以根据您正在观看的节目类型进行调整，无论是新闻、体育还是卡通。先锋的 720p 精英系列将于 7 月份推出 42 英寸和 50 英寸系列，售价分别为 3200 美元(PRO-950HD)和 4500 美元(PRO-1150HD)。9 月份将推出 50 英寸和 60 英寸的 1080p Elite，售价分别为 6000 美元(PRO-110FD)和 7500 美元(PRO-150FD)。

先锋标签下是 42 英寸和 50 英寸的 720p 型号，将于 6 月份降价，价格分别为 2700 美元(PDP-4280HD)和 3500 美元(PDP-5080HD)。1080p 50 英寸和 60 英寸电视机将于 9 月推出，售价分别为 5000 美元(PDP-5010FD)和 6500 美元(PDP-6010FD)。
