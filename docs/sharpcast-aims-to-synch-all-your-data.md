# Sharpcast 的目标是同步你所有的数据

[](https://web.archive.org/web/20221208223615/http://www.sharpcast.com/)[mobile runch 的 Oliver 今天发表了一篇关于 Sharpcast](https://web.archive.org/web/20221208223615/http://mobilecrunch.com/2006/08/14/sharpcast-is-synchronization-20/) 的长文，Sharpcast 目前是一种照片工具，可以毫不费力地在桌面、网络和移动界面上同步数据。每次您登录任何设备时，您的数据都会自动在所有设备之间同步，移动用户界面看起来就像桌面界面一样。Sharpcast 公司表示，该公司的同步技术将改变的不仅仅是照片的处理方式。奥利弗称其为“通用推送同步引擎”，并称其类似于黑莓技术，但适用于各种数据。

有了德雷珀费希尔朱韦森(Draper Fisher Jurvetson)、塞尔比风险投资公司(Selby Venture Partners)和适马合伙公司(Bernstein Partners)等投资者提供的 1650 万美元资金，Sharpcast 在推进其愿景方面处于有利地位。

跨平台同步说起来容易做起来难，看看 RSS 空间就知道了。如果 Sharpcast 能够超越照片，它将成为一项非常重要的技术。
