# TellMe 推出黑仔移动产品

 [](https://web.archive.org/web/20220315052356/http://www.tellme.com/) 如果你有一部支持新的 [TellMe](https://web.archive.org/web/20220315052356/http://www.tellme.com/) 移动应用的手机，你就再也不用 411 找生意了。它将于太平洋标准时间今天早上 5 点发布。

TellMe mobile 是一款免费的 Java 应用程序，可以安装在手机上。然后，您可以通过语音激活菜单找到正常的 411 信息。只要按住通话键，说出你要搜索的城市和州。然后说出企业名称。电话和地址信息出现在屏幕上。然后，您可以给企业打电话，查看地图和/或获取驾驶路线，并通过短信将信息发送给朋友。过去两周，我一直在用一台装有 Sprint 的三星 SPH-A900 测试 Tell Me，我被它迷住了(MobileCrunch 的 Oliver 也是如此)。最棒的是这项服务完全免费。
 [](https://web.archive.org/web/20220315052356/https://beta.techcrunch.com/wp-content/uploads/2007/01/tellmebig.png) 和新的 [Gmail mobile](https://web.archive.org/web/20220315052356/http://www.beta.techcrunch.com/2006/11/02/gmail-for-mobile-launches-today/) 应用一样，TellMe 需要一部支持 Java 的手机。这里有当前支持的设备列表。

点击右边的图片，查看应用程序使用流程的大图。如果您的手机不支持 Java 应用程序，您可以使用某个版本的短信告诉我服务。将您的查询(“比萨圣何塞”或“星巴克 94027”)发送到 TELLM，结果将会发送给您。
 **背景介绍**

TellMe 由前网景公司创始人迈克·麦库埃于 1999 年 2 月创立，总部位于硅谷。它现在有数百名员工。他们的核心业务是为电信公司提供 411 服务，为美国航空公司等公司提供语音激活的客户服务线路，以及他们自己的 1-800-555-TELL 服务，通过电话提供新闻和其他信息。

该公司在四轮风险投资中筹集了高达 2.39 亿美元的资金，尽管最近一轮是在 2001 年。该公司盈利且现金流为正——两年前，据报道他们创造了超过 1 亿美元的收入。虽然他们没有发表评论，但显而易见的是，TellMe 将于今年或明年上市。我们将会看到他们是否设法击败 NetSuite 在纳斯达克上市。
