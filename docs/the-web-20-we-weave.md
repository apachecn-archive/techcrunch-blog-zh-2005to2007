# 我们编织的 Web 2.0 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/12/the-web-20-we-weave/>

[http://web.splashcast.net/p/](https://web.archive.org/web/20221003152032/http://web.splashcast.net/p/)

[堪萨斯州立大学文化人类学助理教授 Michael Wesch 的整洁视频](https://web.archive.org/web/20221003152032/http://www.youtube.com/watch?v=6gmP4nk0EOE)。

该视频概括介绍了 10 多年前的网络和今天的社交网络(“Web 2.0”)之间的差异，重点介绍了 HTML 如何用于定义结构和风格特征(“表单”)，而 XML 将表单和内容分开，便于各种混搭的数据交换。

我们今天看到，随着网络的“widgetizing ”,各种各样的内容都可以插入网页——允许更广泛的内容分发。

迈克尔的视频提出了谁将组织所有这些数据的问题——并告诉我们会的。他揭示了我们点击的每一个链接和访问的每一个网页——我们都在教机器。

这让我想起了电影“ [I，机器人](https://web.archive.org/web/20221003152032/http://www.imdb.com/title/tt0343818/)”或“[黑客帝国](https://web.archive.org/web/20221003152032/http://www.imdb.com/title/tt0133093/)”——人类不是为“人”工作，而是为“机器”工作在电影《我，机器人》中，机器人最终开始学习如何感受、反应，并最终接管我们。

我不知道人工智能(AI)的现状，但我知道每月有超过 [2mm 的博客被创建](https://web.archive.org/web/20221003152032/http://technorati.com/weblog/2006/04/96.html)——其中大多数是人们以完全开放的日志格式喷涌的生活。最终，我认为人工智能将能够筛选所有这些信息，并有条不紊地对其进行分类，理解它，从中学习，并拥有自己的思想和感情——甚至可能接管我们！

我确信有数据支持我的观点，但与 10 年前相比，今天人们对互联网更加适应，对隐私的担忧也更少了。或者至少伴随着互联网成长起来的年轻一代不太关心隐私——他们通过网络向整个世界倾诉他们的想法。

也许有人会足够聪明地学习/组织人们的思想/想法/感受，搜索将从我们自满(并期待)的糟糕的“搜索结果”体验中发生革命性变化——相反，我们每个人都将在任何给定的时刻获得我们正在寻找的准确答案/信息(使用网络上的集体信息)。
