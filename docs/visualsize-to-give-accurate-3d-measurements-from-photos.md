# VisualSize 从照片中获得精确的 3D 测量值

南加州公司 VisualSize 将很快推出一项服务，提供 2D 照片中物体的精确三维尺寸。

为了精确地测量事物，他们需要同一事物的两张照片，但是从不同的角度。VisualSize 算法自动检测两幅图片中的特征点并找到匹配对。然后，它使用匹配对来计算两个相机位置的坐标(x、y、z 轴和 3D 坐标框架中的原点)，并使用从图像到平面到 3D 坐标框架的三角测量来重建 3D 场景。然后，VisualSize 可以高度精确地测量长度、角度、面积和体积。这个屏幕截图展示了一个原始输出的例子，但是服务本身是 pre-alpha 的。

这并不像, [Fotowoosh 通过从一张 2D 的照片中创建 3D 图像所做的那样性感。但它确实有商业价值。家居装修、室内设计、总承包估价、犯罪现场调查、军事规划、保险索赔评估等。都需要精确的测量。VisualSize 表示，他们可以以 0.1%到 5%的测量误差创建它们。](https://web.archive.org/web/20210507023810/http://www.beta.techcrunch.com/2007/04/15/fotowoosh-will-turn-any-picture-into-3d-image/)

该公司表示，他们将在其网站上提供这项服务。更重要的是，将会有一个 API 允许其他人围绕该服务构建应用程序。

创始人是王元芳(UCSB 大学教授)和梁平(UCR 大学教授)。他们通过以前创业公司的流动性事件自筹资金。
