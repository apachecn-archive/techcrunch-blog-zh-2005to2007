# Vista 服务包将于秋季推出

推迟购买 Vista 电脑还是等待 Vista 在您的企业解决方案中推出？据英特尔公司首席执行官保罗·欧德宁称，Vista 服务包将于 10 月或 11 月推出，泄露了微软最严格保守的秘密之一。

看，人们不喜欢买新软件。只要问问那些耐心等待 Windows XP 真正变得坚如磐石，然后再更换 Windows 2000 服务器的人就知道了。通过指定 SP1 到来的时间和日期，他们本质上是给每个人一个机会*而不是*现在购买产品，而不是等待几个月直到 SP1 下降。聪明的人们，那些电脑用户。

欧德宁在一次财务电话会议上透露了这个消息，这可能会让他获得一次去雷德蒙特殊再教育营的机会，这个营地位于盖茨头骨形城堡的顶端。
