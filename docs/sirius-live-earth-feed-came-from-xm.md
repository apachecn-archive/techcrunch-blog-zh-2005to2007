# 天狼星地球直播来自 XM | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/09/sirius-live-earth-feed-came-from-xm/>

![](img/d8a49c86e6ce0ff32caa1797838636cb.png)

对于那些认为将 XM 和 Sirius 合并在一起会是一件痛苦的事情的人来说，这证明了这就像凯文·贝肯在一部蹩脚的电影中得到一个角色一样容易。似乎 XM 是在东卢瑟福，新泽西州和伦敦的巨人体育场为 Live Earth 成立的，有 60 多名工作人员。摄制组录下了所有的音乐，并进行了几次采访，然后在……Sirius？上播出。

没错。原来 XM 正在实时向天狼星卫星广播和首映广播网提供信息。这意味着当你收听卫星广播时，你可以在 XM 频道和 Sirius 频道之间无缝切换。听起来大家都赢了。我很好奇这笔交易是不是即将进行的合并的一部分，或者是不是 Sirius 来 XM 公司索要热点信息。
