雅虎博客搜索启动

今晚7点，雅虎发布了一款博客搜索产品。约翰·巴特利得到了独家新闻。

不像他们昨晚发布的播客产品，博客搜索产品只不过是雅虎新闻的一个快速附件。坦率地说，我非常失望。

雅虎没有推出独立的网站，而是将博客搜索整合到雅虎新闻中。博客搜索结果显示在正常结果的右侧，在侧栏类型的区域。例如，参见关于“Web 2.0”的搜索。

我插入了一个红色的大箭头，并将图像放大到我们通常使用的两倍以上，这样你就可以在页面的右边区域看到结果。

雅虎还整合了Flickr和My Web 2.0搜索结果。目前还不清楚这些结果的相关性如何，但乍一看，有很大的改进空间。我发现用户界面完全无法使用。
