Pluck推出了三款产品更新:

pull于2005年8月9日宣布三项产品更新——为Internet Explorer、Firefox及其网络产品(左边的网络产品图片)推出了Pluck 2.0。一个关键的新特性是，所有这些产品都可以同步订阅源(并有望在下一个版本中同步已读/未读的帖子)。

Pluck版本2.0为Firefox、Internet Explorer和Web提供了跨平台的RSS阅读器套件

德克萨斯州奥斯汀(2005年8月9日)Pluck Corp .发布了业界第一的基于浏览器的RSS阅读器2.0版本，用于Internet Explorer，并扩展了用于Firefox和web的RSS阅读器产品线。该产品套件是RSS领域中第一个允许用户在不同计算机、web浏览器和操作系统之间同步重要web信息的产品，如RSS web feeds和书签。www.pluck.com免费提供拔毛服务。

今天发布的产品套件包括对Pluck Internet Explorer Edition和Pluck Web Edition的重大升级，还推出了一款名为Pluck Firefox Edition的新产品，用于流行和快速增长的Firefox web浏览器。Pluck Firefox Edition在Firefox浏览器中提供了高级的RSS聚合和管理功能，使Firefox用户能够在易于查看的文件夹中轻松组织、查看、存储和共享RSS Web feeds、互联网搜索结果、网页和其他信息。像所有的Pluck RSS阅读器一样，Pluck Firefox Edition能够搜索感兴趣的主题以获得相关的web提要，并导入现有的书签和Web提要订阅集合。

该套件中所有三种产品的新Pluck 2.0升级版为活跃的互联网用户提供了丰富的特性和功能，增强了他们查找、访问和管理网络信息的方式。此版本提供的功能包括:

*同步:使用户能够在多台计算机(PC、MAC等)之间自动访问和同步他们的书签和RSS web feeds。)、不同的web浏览器(Firefox、IE等)和支持互联网的设备，因此RSS内容在工作中或家中总是可用的。
*显著的UI改进:在RSS阅读、书签组织和订阅管理的所有方面都改善了用户体验。这种新体验旨在简化RSS学习曲线，提高RSS的采用率。
* FeedFinder:这款来自Pluck的新提要发现引擎极大地提高了提要搜索的相关性，并为用户发现web提要开辟了新的途径。使用FeedFinder，用户可以通过浏览feed分类、查看特色RSS web feeds以及在从互联网上收集的大量feed缓存中提交关键字搜索来找到感兴趣的feed。
*强大的搜索工具:允许用户通过一次点击和一个视图向多个搜索引擎发送搜索查询，节省时间并提高结果质量。
