# 欢迎来到社交网站，现在就付钱吧

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/12/welcome-to-the-social-now-pay-up-sucka/>

![patent450.jpg](img/738453e656a3db9b8be68bec6006f756.png)

这是有史以来最糟糕的专利创意，它来自雷德蒙，这有什么好奇怪的吗？Zune 是一个过时的设备，我可以自信地说，因为如果它起飞并成为一个主要的玩家，那么它会发生在公平的哥谭市。我只见过少数人加入“社交”，我从未在一个地方见过两个 Zunes。那么人们如何从这项新专利中赚钱呢？

在你不小心激活 Wi-Fi 之前，Wi-Fi 一直处于休眠状态，它会正常传输歌曲，但在第三次播放后，你会被提示购买歌曲。如果你选择这样做，那么发送给你的社交名流将会得到几分钱，而 Zune 市场和版权持有者将会财源滚滚。它也适用于“盗版”音乐，交易将使其成为合法的 MP3。不错的尝试，微软。
