# 触摸感应纸–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/06/touch-sensitive-paper/>

![](img/07e64cb37ea4a30a6a318f74906c4788.png)

吹娃娃鉴赏家欢欣鼓舞！新一波可触摸技术即将到来。一组瑞典科学家创造了嵌入金属颗粒的纸张，当被触摸时，会形成一个电路，产生录音。声音是通过嵌入纸中的小电磁铁
发出的。现在的目标是将这项技术应用于包装。然而，更多“有趣”的想法肯定会在前面。再加上出汗机器人 T4，周六晚上呆在家里变得更有趣了。
