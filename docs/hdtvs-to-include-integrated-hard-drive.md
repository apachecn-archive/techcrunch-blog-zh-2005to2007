# 高清电视将包括集成硬盘

![](img/e3ee05a5edf1f2d59b716e2e6ff2b7e7.png)

LG 宣布在韩国推出 LCD 和等离子高清电视，包括双高清调谐器和集成硬盘。消费者既可以同时观看两个高清频道，也可以在观看一个高清频道的同时录制另一个频道。这些电视将配备一个内置的 160GB 硬盘和一个 USB 2.0 端口。

韩国的建议价格将在 2667 美元至 5000 美元之间——取决于屏幕尺寸。
