# WeSmirch 自动挖掘污垢

# 韦斯米尔奇自动挖掘泥土

[![weSmirch](img/2f0a7dad06aa08d77d3888e49d3bd8f0.png)](https://web.archive.org/web/20201026042305/http://www.wesmirch.com/)

[WeSmirch](https://web.archive.org/web/20201026042305/http://www.wesmirch.com/) 是一个新的名人八卦和新闻门户，由 [Memeorandum](https://web.archive.org/web/20201026042305/http://www.memeorandum.com/) 提供支持。Memeorandum 背后的技术将从博客帖子或新闻网站中找到最受欢迎的故事，并使用专有算法将它们聚集在一起。读者得到的是一个单页的报纸视图，显示某个特定类别的重大新闻，并据此对它们进行排序，将新闻显示为来源链接的集群。

韦斯米尔奇发现新闻来源，并汇总最新的热点故事。WeSmirch 收集新闻的速度相对较快，这使得它成为一个无需访问大量独立网站就能获取八卦信息的好网站。它还会从更小的来源中找到潜在受欢迎的故事，并将它们显示在网站上。它很好地解决了重复的问题，即使没有故事之间的链接，也可以挑选出那些相同或相关的故事。

Memeorandum 的定期读者将从今晚开始注意到，Memeorandum 现在支持的所有三个网站都在故事集群中包含了图像。这些图片取自源文章，使网站整体看起来更好。

自去年 11 月推出以来，Tech.memeorandum 在博客圈大受欢迎，之前在 Techcrunch 上，我们已经写过关于 [Memeorandum 和这一类别的其他网站](https://web.archive.org/web/20201026042305/http://www.beta.techcrunch.com/2006/02/04/a-look-at-the-memeorandum-killers/)。Memeorandum 之所以出类拔萃，是因为它是该领域的第一家公司之一，尽管其他公司竭尽全力向该系统发送垃圾邮件，但它仍设法保持了一定的质量水平。

Memeorandum 的创始人 Gabe Rivera 的政策是扩大他所涵盖的类别，直到某个特定类别有足够多的有趣内容，人们才会真正想要阅读。科技博客社区非常活跃，有大量的故事来源和持续的新闻流，但在其他类别中，通常很难找到什么是受欢迎的算法，将一些可靠和可信的来源聚集在一起，总体上保持有趣的故事流。

随着这一领域玩家的更多发布和更新，这一类别正在升温。目前跟踪娱乐新闻的其他网站有 [Megite](https://web.archive.org/web/20201026042305/http://megite.com/index.php?section=entertainment) 和[blogni scient](https://web.archive.org/web/20201026042305/http://entertainment.blogniscient.com/)——至于哪个网站在报道娱乐和八卦方面做得最好，我会留给读者一个练习，因为这主要取决于你自己的喜好。
