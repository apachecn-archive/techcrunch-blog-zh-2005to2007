# 哦，天哪，又一个索尼电池在笔记本电脑上着火了

[![burninglaptopfun.jpg](img/638e6fe87c70b68e60d424e1412649e4.png)](https://web.archive.org/web/20210228005239/https://beta.techcrunch.com/wp-content/uploads/2007/05/burninglaptopfun.jpg "burninglaptopfun.jpg")

谁准备好迎接另一轮笔记本电池爆炸了？我相信每个人都记得去年整个[丑闻](https://web.archive.org/web/20210228005239/http://crunchgear.com/2006/10/03/sony-knew-of-battery-explosions-eleven-months-ago-ignored-dangers/)，索尼笔记本电脑电池爆炸，导致召回和索尼的许多尴尬时刻。来自日本的报道表明，作为东芝召回计划的一部分，索尼电池在四月份就着火了。据称，这一事件与之前的电池起火无关，而是由其他一些制造异常引起的。不过，我敢打赌，索尼和东芝都不想再和这类故事扯上关系了。

该死，又一个[惨败后的](https://web.archive.org/web/20210228005239/http://crunchgear.com/2007/04/26/my-lap-my-lap-my-lap-is-on-fire/)电池混乱。拉尔夫·纳德在哪里让他们改正他们的行为？
