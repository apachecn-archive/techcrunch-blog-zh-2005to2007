# Typepad 正在推出博客小工具

SixApart 的 Typepad 即将为博客作者发布[小工具。这将简化数百万博主的生活，他们到目前为止或多或少都要处理在模板中放置 html 代码，以便用第三方功能定制和丰富他们的博客。](https://web.archive.org/web/20220112014239/http://www.sixapart.com/typepad/widgets/)

首先，这与雅虎/苹果桌面小工具无关。Typepad 将为用户提供一种更简单、更无缝的方式，在你的博客中添加附加功能(feedburner、technorati、Pandora favorites songs 等)，而不必去该服务的网站(比如 Pandora)复制/粘贴 html 代码来包含上述功能。这些特性通常放在边栏或类型列表中。这对不精通博客的人来说尤其不友好，我们相信这将使第三方博客功能更受欢迎。

![](img/6fd0c53ac8206f6e98f5090b2fbde901.png)

Typepad 提供了一个小工具库(上面的截图)，目前包括[的 32 个项目](https://web.archive.org/web/20220112014239/http://www.sixapart.com/typepad/widgets/)，可以在 6 个类别(游戏/工作搜索/音乐/照片视频/出版/搜索)中选择。您可以订阅这个 [XML feed](//web.archive.org/web/20220112014239/https://feeds.feedburner.com/TypePadWidgets) ,以便在新项目出现时得到提醒。如果你有一个 Typepad 账户，你所要做的就是点击一个小工具，编辑标题，选择位置，然后发布到你的博客上。没有代码，没有混乱，没有犯错的余地。由于某些原因，在我写这篇文章的时候，一些插件还没有被激活，但是应该很快就会被激活。

> “……在 TypePad 上安装一个小工具比在任何其他博客服务上安装都要容易得多。TypePad 博客作者只需从 TypePad [小部件目录](https://web.archive.org/web/20220112014239/http://www.sixapart.com/typepad/widgets)中选择一个小部件。博主在构建者的站点上配置小部件，然后点击一次，将其发布到他们的 TypePad 博客上。用户在他们的 TypePad 博客上预览小部件后，他们可以确认他们的选择，小部件将出现在他们的博客上”

我尝试了这个功能(你可以在[我的 Typepad 博客](https://web.archive.org/web/20220112014239/http://ouriel.typepad.com/)上看到一个整合 [Sphere](https://web.archive.org/web/20220112014239/http://www.sphere.com/) 博客搜索的例子)，它工作得非常好，你可以轻松地重新定位或删除小部件，我相信它将从现在起为我节省一些时间。然而，喜欢玩 html 片段来定制自己的小部件(例如调整大小或删除一些文本)的精明的博客可能会走老路，仍然会手动操作。

![](img/f03742afd97754db6bb10fbd0a18c746.png)

但真正新奇的是(终于！) **Typepad 决定向社区开放他们的 API**来构建新的小工具，并扩大可供 Typepad 博客用户使用的服务和项目的选择。此处解释了操作方法[。](https://web.archive.org/web/20220112014239/http://www.sixapart.com/typepad/widgets/developers)

好主意从来不会独自出现。几天前发布的 WordPress 也为开发者提供了 SiderBar 小工具和 API 来构建新的项目，这是已经讨论过的。在博客中集成的容易程度堪比 Typepad。我喜欢的一点是，在一个漂亮的 Ajax 界面中，可以编辑你自己的文本，并把它放在边栏中。widget 目录(最近的评论/ feed/ Flickr / Delicious 和其他)还不是很大，但肯定会很快。更多的[这里](https://web.archive.org/web/20220112014239/http://lorelle.wordpress.com/2006/03/10/playing-with-wordpresscom-new-sidebar-widgets/)那里[那里](https://web.archive.org/web/20220112014239/http://www.liewcf.com/blog/archives/2006/03/wordpresscom-introduce-sidebar-widgets/)。

博客平台让用户更容易为他们的读者提供好看有趣的博客，这是个好消息。这应该会给博客平台行业带来一个新的质量标准。
