# 俄勒冈科学行动头盔摄像机

![](img/23c4a00e30ec881fe95f914d1490fdc9.png)

ATC-2000 行动头盔摄像机是一种面向行动的个人的新型活动摄像机。它以 30fps 的速度拍摄 640 x 480 VGA 的全彩色数字视频。正如人们对动作摄像机的期望，ATC2k 既防水又防震。

遗憾的是，它只支持 SD 卡，所以没有通过手机持续上传图片的蓝牙同步功能，但我想这至少比许多其他选择要好。它使用两节 AA 电池，并通过 USB 2.0 连接到您的计算机。ATC2K 现已上市，售价 129.99 美元。
