# 你的杜鲁门秀:有组织的生活博客

[![yts.jpg](img/b808b7589e612933a00150af75330d03.png) ](https://web.archive.org/web/20210919180249/http://www.yourtrumanshow.com/) [你的楚门秀](https://web.archive.org/web/20210919180249/http://www.yourtrumanshow.com/)周三推出一款致力于发布、评级和评论个人视频故事的产品。

这项服务结合了个人博客、用户生成的视频和社交网络，为用户提供了一个直观的界面，使他们能够记录自己的生活。

《楚门的世界》中的“V-link 界面”直观地连接了相关的故事情节，使用户可以很容易地找到并跟踪故事的发展。通常的用户评级系统也提供在关键导航点获得偏好的较高评级视频。

我的第一个问题自然是关于这个名字。华纳兄弟没有更新《楚门的世界》的独家使用权，所以现在它占据了 Web 2.0 创业公司的三分之二。

这个平台感觉有点像 YouTube 遇到了博客。是的，它是另一个视频托管平台 ala YouTube，但布局不同。重点是写个人故事的博客；功能包括按时间顺序提交到故事书。

《你的楚门秀》背后的团队旨在将《生活在镜头前》扩展为一个“明日在线真人秀明星，将用户生成的内容从单集视频迁移到多集系列”的网络。我完全不知道他们是否真的会做到这一点，但这是一个精心制作的利基视频托管网站，肯定不会因为缺乏良好的演示和编码而失败。
