# 灯单:制造商需要停止到处摆放无用的灯

[![blinkinglights.jpg](img/c253b0549804b3bc30ac151de5e7756b.png)](https://web.archive.org/web/20210228014356/https://beta.techcrunch.com/wp-content/uploads/2007/05/blinkinglights.jpg "blinkinglights.jpg")

那是可怕的灯光在闪烁！《计算机世界》的迈克·埃尔根对那些在今天的小玩意上到处安装愚蠢无用的灯的小玩意制造商感到愤怒。电脑显示器有灯告诉你*他们在*；路由器和调制解调器一插上电源就不停地闪烁；黑莓和其他手机似乎随时都会闪烁。够了！作为对这种疯狂行为的回应，Elgan 想出了一个小玩意灯的清单，其中他列出了几个关于这个小玩意光污染应该如何结束的论文。总而言之，它们是…

1.不再有夜晚闪烁的灯光

2.发光二极管都应该能够变暗

3.灯光闪烁应该是有原因的，而不是因为它看起来很酷

4.灯应该有一个开关

5.灯光！=好的设计。不要把它们当作症结。

你明白了吗？别再开灯了！

我们人民需要一个小工具灯清单
