# Saitek Pro 玩家指挥台

![saitek.jpg](img/7b2a0c2b9b740c1971adf71bd0b2556c.png)

Saitek 的这款新控制器采用直观的 20 按钮设计，让用户在 FPS 和 RTS 游戏中占据优势。Pro Gamer Command Pad 具有背光人体工程学键盘，可调节拇指区域，以获得最大舒适度。它有 4 路帽子开关和多达 120 个命令可以在设备上编程。

以前我经常去参加许多 [CPL](https://web.archive.org/web/20201022181841/http://www.thecpl.com/) 赛事，我不记得曾经见过锦标赛选手使用这样的东西。话说回来，当我退出舞台时，本质上模仿键盘的设备才刚刚开始出现。有谁知道现在有没有重要的玩家使用鼠标和键盘以外的东西？
