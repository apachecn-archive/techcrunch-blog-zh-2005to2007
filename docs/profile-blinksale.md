# 个人资料–blink sale–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/06/profile-blinksale/>

## 概述:

想要一种更简单的方式来管理发票？Blinksale 是给你的。即使发票对你来说不是问题，花点时间去看看他们的网站。这是视觉上和功能上令人惊叹的。它渗透了 AJAX 和 Rails，以及深思熟虑的、创造性的设计。这并不奇怪，因为一家专门研究 AJAX 的设计公司 [Firewheel Design](https://web.archive.org/web/20220810002648/http://www.firewheeldesign.com/web/) 创建了这个网站。但它也非常易于使用——这些功能不仅仅是为了给人留下视觉印象。我喜欢它，而且我从不寄发票。

Blinksale 以直截了当的方式满足简单的需求。对于小企业来说，发票总是一件麻烦事。你要么需要破解一个 Word 或 Excel 文档并转发，要么使用昂贵的会计软件。有些人只是在电子邮件中发送文本发票。它不是很漂亮，除非你使用会计软件，否则很少有或没有工具来帮助你跟踪和管理发送后的发票。

Blinksale 解决了所有这些问题。首先，创建和发送非常专业的发票需要一些时间。我实际上注册了一个新账户，创建了一张假发票，并在大约 5 分钟内将其发送给了我在阿基米德风险投资公司的合伙人基思·特雷(我向他收取了 25000 美元的“服务费”——让我们看看赖账者是否会付钱)。

Blinksale 目前支持 160 种货币。如果你能与 paypal 做生意，你可以在发票中包含你的 paypal 信息[并通过这种方式付款(在我看来，世界上有两种人:使用 paypal 一年或不到一年的人，以及被 paypal 欺骗并随后被 paypal“客户服务”忽视和/或斥责的人)。顺便说一下，注册后，blinksale 会为您创建一个独特的主页——我们的主页是 techcrunch.blinksale.com。](https://web.archive.org/web/20220810002648/http://www.blinksale.com/blog/invoices/000028.html)

创建发票后，有许多管理工具供您使用。

首先，Mac 用户可以将发票与 iCal 同步。对于我们其他人来说，xml 提要是可用的，您可以通过 RSS 订阅发票。[提醒](https://web.archive.org/web/20220810002648/http://www.blinksale.com/blog/reminders_and_thank_yous/000031.html)和[感谢](https://web.archive.org/web/20220810002648/http://www.blinksale.com/blog/reminders_and_thank_yous/000030.html)可以发送给客户。已付发票可以标记为已付并已结清。

## 订阅计划

Blinksale 有许多账户选项，从免费到每月 29 美元不等。付费计划都有 30 天的免费试用期，没有最短期限(并且在月底前终止不会退款)。付费计划允许定制模板(使用 CSS)，更多的用户，更多的发票，更多的客户，等等。

## Blinksale 宣言:

> 简而言之，我们大部分人工作的原因就是因为提供服务或出售商品而获得报酬。虽然许多小企业主、承包商和自由职业者喜欢他们的工作，但我们大多数人更喜欢滑雪。或者在湖上。或者在 60 英寸等离子电视前。我们工作是为了得到报酬。如果我们得到足够的报酬，也许我们就不用工作那么多了。为你的服务获得报酬应该是容易的，不痛苦的，快速的。这就是我们创造 Blinksale 的原因。[链接](https://web.archive.org/web/20220810002648/http://www.firewheeldesign.com/sparkplug/2005/July/the_blinksale_manifesto.php)
