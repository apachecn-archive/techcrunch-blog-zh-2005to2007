# 音频文件:RIAA 在抽苹果吗？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/09/the-audiofile-is-the-riaa-smoking-apples/>

音乐产业现在应该处于恐慌状态，但不是因为盗版。围绕数字音乐和数字版权管理的未来的困惑本周因史蒂夫·乔布斯给唱片行业的公开信而变得更加复杂。如果挪威消费者委员会和 RIAA 的回答像 SAT 考试中的阅读理解题一样被评分，这两个群体都不会超过高中。

乔布斯明确请求四大唱片公司允许在线音乐商店销售未经 DRM 授权的内容，这是一个荒谬逻辑的杰作。既然 CD 销售仍占唱片公司收入的 90 %,那么为区区 10%争吵有什么意义，这种想法是非常短视的。他公然忽略了一个事实，即 CD 销量正在下降，而数字下载量正在增加。

媒体喜欢指出数字下载无法弥补 CD 销量的下降。指望数字下载和 CD 销售保持某种完美的平衡，下载量无缝地填补空白，是愚蠢的。下载真的会取代 CDs 除非像 DRM 这样的技术不再是购买决策的考虑因素。

正如挪威消费者委员会(Norwegian Consumer Council)的托吉尔·沃特豪斯(Torgeir Waterhouse)敏锐地指出的那样，乔布斯接着指出，iPod 用户没有被锁定到 iTunes 音乐商店——我们抱怨的问题当然是相反的，iTunes 音乐商店的客户被锁定到 iPod。"

Waterhouse 又一次扭转了这一局面，称苹果公司有责任砍掉 DRM，而不是标签公司。“苹果和 iTunes 音乐商店应该解决唱片公司和 DRM 本身的问题，如果这个问题需要解决的话。”来吧。如果史蒂夫 un-DRM 的音乐，他要么会被起诉遗忘违反他的合同与~~魔鬼~~专业，或者他们会拉他们的目录和成本史蒂夫一大块的变化。

同样古怪的是 RIAA 的回应(通过洛杉矶时报):“苹果公司提议将 FairPlay 授权给其他科技公司是一个受欢迎的突破，对于粉丝、艺术家和唱片公司来说，这将是一个真正的胜利。”显然 RIAA 那边的药草有足够的效力让他们产生幻觉。史蒂夫没有提出这样的提议——他说开放 FairPlay 将意味着苹果“不再能够保证保护它从四大音乐公司获得许可的音乐。”

随着实体媒体的消失，如何说服大公司放弃他们正在培养的未来 5 年内成为他们生计的 2B 年度业务？这些钱的一部分来自未授权的音乐，但那只是一小部分。此外，尽管史蒂夫在迎合消费者，但他确实指出，在当前的 DRM 形势下，“消费者正得到源源不断的创新产品和广泛多样的选择。”

在 DRM 争论中被忽视的一个主要问题是，RIAA 并不真的对 DRM 感兴趣来防止盗版*现在*。如果是的话，他们会像攻击数字音乐一样攻击 CD。他们感兴趣的是让数字版权管理系统在*的未来*准确地运行，这样他们就可以避免在 CD 上发生的事情。

我是不是在说 DRM 不仅会存在，而且会无处不在？没错。但是[正如我在](https://web.archive.org/web/20210228011849/http://crunchgear.com/2006/12/29/the-audiofile-cant-they-all-just-get-along/)之前所写的，前景将会改变，因为大公司最终意识到消费者想要的不仅仅是一个文件的权利，而是一首歌的权利，这正是 Navio 试图做的事情。

你将支付比目前更高的价格，但仍比购买 CD(略)便宜，而且你可以随时下载你需要的任何 DRM 格式的歌曲。你仍然会受到限制，比如你可以在多少台设备或电脑上播放这些文件，但这比根据你在哪里购买音乐而被锁定在一个播放器上要好得多。这也消除了苹果开放公平竞争的需要。

你认为这会把人们推向非法下载吗？也许吧，但根据 NPD 集团的说法，“在美国，诉讼是电脑用户从未经授权的 P2P 下载转向合法下载的最主要原因。”到目前为止，已经有 10，000 起针对文件共享者的诉讼发生在 18 个国家。现在我们知道我们为音乐付出的钱都去了哪里…
