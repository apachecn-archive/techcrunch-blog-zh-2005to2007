# 雅虎用 API 程序打开巴拿马大门

[![yahoosearchmarketing_logo.jpg](img/ac2c161b954e84f26d226b8f70e42e65.png)](https://web.archive.org/web/20221001014211/http://searchmarketing.yahoo.com/) 雅虎[今天在](https://web.archive.org/web/20221001014211/http://yhoo.client.shareholder.com/press/ReleaseDetail.cfm?ReleaseID=246880)宣布了[雅虎搜索营销商业 API 计划](https://web.archive.org/web/20221001014211/http://searchmarketing.yahoo.com/api)，该计划将为企业和开发者提供免费和开放的巴拿马搜索营销应用程序接口(API)。

该计划将使广告商、开发商、广告代理商和技术提供商能够建立在雅虎的核心搜索营销技术之上，以增强他们现有的业务产品或创建全新的搜索营销工具和应用程序。

雅虎负责人苏珊·德克尔美国广告与出版商集团表示，开放巴拿马完全是为了获得市场份额:“通过提供对我们技术的开放访问，我们正在对我们的广告合作伙伴进行明确的投资，为开发者创造新的机会，并朝着实现我们的愿景迈出关键一步，以建立**行业领先的广告和出版生态系统**。”

向巴拿马开放免费 API 访问可能是两种情况之一:灵感触发的想法或绝望。考虑到交付 Panama 首先需要大量的准备时间，这看起来像是前者的决定，毕竟雅虎没有什么能在绝望的情况下迅速采取行动。开发者可能会在未来几天推出新程序，严格来说，从这个角度来看，这一提议肯定会受到业界的欢迎，并可能在寻求推翻谷歌主导的 Adwords 平台的过程中带来更大的挑战。
