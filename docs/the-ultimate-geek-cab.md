# 终极极客出租车——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/16/the-ultimate-geek-cab/>

![](img/d456c37dd03da0268e061a97894ab136.png)

我们中的一些人很难把自己从电脑前抽离出来，有时这没什么。但是如果你需要打游戏和上网冲浪，并且碰巧住在德克萨斯州的奥斯汀，你可能会想打电话给里基·格雷叫辆出租车。格雷在他由小型货车改造的出租车上安装了两台液晶电视，顾客可以在上面玩 Playstation 2 游戏、看电视和上网。

格雷卓越的服务，再加上极客的出租车，让我们在奥斯汀地区巡游时度过了一段美好的时光。他说，他未来的计划包括将 PS2 升级到 PS3，购买更大的液晶电视，并增加一个卫星天线，这样就可以播放更好的节目。如果他增加了一个小吃机，我很肯定每次旅行都值得走这条风景优美的路线。

一辆视野开阔的出租车:两台电视、一台笔记本电脑和一台 PlayStation 2
