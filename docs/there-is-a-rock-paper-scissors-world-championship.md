# 有一个石头剪刀布世界锦标赛

[![rps2.png](img/5bae96219b24004eee0a8fc32101348c.png)](https://web.archive.org/web/20221102172546/http://www.worldrps.com/) 来自世界各地的最伟大的战略家们将于 10 月 13 日齐聚多伦多争夺成为冠军中的冠军的权利。2007 年世界石头剪刀布冠军！

原来，2007 年国际世界 RPS 锦标赛是由本周在多伦多参加 Mesh 的一些 Web 2.0 业务人员组织的。尽管最初 OMG！事实上，石头剪子布是一项快速发展的运动，拥有自己的专业组织[世界 RPS 协会](https://web.archive.org/web/20221102172546/http://www.worldrps.com/)，并且在世界范围内定期举行竞技比赛。

对于那些不熟悉这个游戏的人，维基百科有一个[详尽的解释](https://web.archive.org/web/20221102172546/http://en.wikipedia.org/wiki/Rock,_Paper,_Scissors)。据说这种游戏可以追溯到中国的汉朝，后来传到日本和欧洲。虽然对一些人来说，石头剪刀布是一个轻浮的孩子的游戏，但对其他人来说却是一个严肃的追求。石头剪子布锦标赛已经被电视转播，有游戏策略和玩家策略，甚至有一本官方书籍在亚马逊上出售。

目前的石头剪刀布世界冠军鲍勃·库珀来自伦敦，今年的世界锦标赛是第七年。预计将有 500 多名参赛者。

现在你什么都见过了。往前走。
