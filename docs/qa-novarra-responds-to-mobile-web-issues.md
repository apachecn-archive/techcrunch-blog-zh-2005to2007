# 问答:Novarra 回应移动网络问题——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/22/qa-novarra-responds-to-mobile-web-issues/>

![novarra.jpg](img/d18e31afd443756a567f4611d753bec7.png)在我们发布了关于 [Novarra 与沃达丰](https://web.archive.org/web/20210126033136/http://mobilecrunch.com/2007/06/14/novarra-and-vodafone-uk-partner-to-deliver-mobile-internet/)合作为其移动互联网服务提供动力的消息后，Novarra 转码应用的批评者——包括[mobile aware](https://web.archive.org/web/20210126033136/http://mobilecrunch.com/2007/06/18/kevin-mccloskey-of-mobileaware-who-put-vodafone-in-charge-of-mobile-content/)的 Kevin McCloskey 指责该技术是一种通用的解决方案,“经常破坏公司网站的外观和感觉，从而削弱他们的公司品牌和身份。”关于网站应该如何在移动设备上呈现的争论仍在继续。诺瓦拉的解决方案是否像一些批评者声称的那样是一种倒退？我们询问了诺瓦拉公司的管理团队的意见。

自从沃达丰英国公司和诺瓦拉公司宣布交易以来，就有人认为这是移动网络搜索的一个错误方向。你对那些诋毁者说什么，他们声称这个应用程序会使使用一些网站更加困难？

可能这些人使用过这种质量的产品。大多数人发现，在普通手机上，在商业、新闻、邮件、滑雪等领域，这是一种相当引人注目的网络体验。下一次，当你被困在某个地方，等着回到笔记本电脑前在网上查找资料时；你可能还记得这条线索。

Novarra 的运营商客户遍布美洲、欧洲和亚洲的 2.5G 和 3G 网络。在这些部署中，消费者定期使用服务，没有抱怨，运营商对消费者的接受和使用感到满意。

在过去 3 年多的时间里，大多数运营商已经部署了许多带有完整网络浏览器的高端手机和 PDA。自 2002 年中期以来，Novarra 在 Palm 平台上发布了完整的浏览器。这些产品试图在手机上提供完整的网络。

我们认为，无论是传统内容还是未来内容，都有“为移动设备而创作”的角色。然而，人们不能忽视显而易见的事实。网络的力量在于数以百万计的网站，每一个都是相关的。

**Mobile Crunch:**Novarra 的转码技术开发了多长时间，在多少部手机上进行了测试？

**Novarra:** Novarra 的 web 转型解决方案是 7 年产品研发和 5 年商业部署的成果。它已经在数百部手机上得到验证。代码转换是解决方案的一部分。网络优化、手机可用性、网络平台整合、内容优化等。有助于转变移动用户的桌面体验。还支持所有移动特定内容。

**Mobile Crunch:** 转码应用是否在不断改进，以解决用户在尝试浏览各种网站时可能遇到的问题？

**诺瓦拉:**没错，这个问题相当大。随着 Web 的变化和发展，解决方案也在不断升级。这个行业正在开始一个旅程的下一个阶段，这个旅程可以追溯到很久以前。Novarra 致力于实现这一愿景，这也是我们取得今天成就的原因。我们愿景的核心是将质量和用户体验放在我们构建的一切的核心位置；这将帮助我们应对不断发展的网络。

记得 20 世纪 90 年代中期网络 1.0 的推出，有一种观点认为有些事情你不能做，因为网络不是为商业、视频、互动等而设计的。那么，你认为移动网络也会出现同样的问题吗？也就是说，人们在说，“你不能在移动网络上做这些事情”，这取决于创新者再次让它发生？

绝对的。有许多神话、恐惧和理论。有了创新和可靠的产品，这些障碍就被消除了。

**移动危机:**当我们听到很多关于 PC 的 Web 2.0 的时候，我们真的到了移动的那个点了吗？最近有人说，我们真的处于移动 Web 0.5 时代，我们还有很长的路要走。但是突然，人们发现他们可以在手机上使用网络，他们想要所有的东西，他们今天就想要。那么，移动真的能交付一切吗？

消费者的习惯可以很快改变。Novarra 的产品很容易部署到现有的消费者和他们的手机上，因此很快就被广泛采用。

将网络带到移动设备的进展速度正在快速变化，并且正在赶上常规网络功能(例如，视频)的能力。可能阻止一些东西进入手机的是不同参与者做出的商业决定:内容提供商、运营商、手机制造商等等。在移动网络上，带宽是比固定网络更加有限的商品，这将影响定价和商业模式。

人们会想要他们所能拥有的一切——只要价格合适。这是互联网的本质，也是一般技术的本质。

移动可能不会在明天提供“一切”，但作为当今互联网的补充，它有可能提供更独特和不同的东西。当然，Web 2.0 是一个明显的可能性。

诺瓦拉
