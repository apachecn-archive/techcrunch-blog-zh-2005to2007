索尼Type T型笔记本电脑现在与LED

有什么比在黎明时分醒来发现有一台LED背光笔记本电脑更好的呢？实际上，很多事情，但是如果日本的科技巨头能给我们送来最新的VAIO就好了。我们都在谈论盛传的LED Macbook(T2)。索尼可能会大赚一笔，但他们多年前就已经把头埋得太深了，所以这永远不会发生。

总之，索尼VAIO Type T十周年笔记本电脑拥有我们想要/需要的一切，包括前面提到的LED屏幕、可选的32GB固态硬盘和8小时电池续航时间，所有这些都装在闪亮的黑色外壳中，重量为2.68磅。其他值得注意的规格包括80GB硬盘、1GB RAM、11.1英寸全景屏幕和双层录制DVD驱动器。如果你觉得辣，那么你可以选择32GB固态硬盘和160GB硬盘的硬盘和DVD-R。还有一个可选的电池，你可以升级到18小时的任何你选择沉迷的乐趣，但等等，还有更多。

Type T有两个USB 2.0插槽，一个Firewire，Expresscard/34，SD/MMC/Memory Stick Pro/Duo，英特尔945GMS高速芯片组，802.11a/b/g，网络摄像头，指纹扫描仪和蓝牙2.0+EDR。真的，谁想要这些？
