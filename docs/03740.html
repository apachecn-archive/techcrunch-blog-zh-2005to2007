Xantrex PowerSource Mobile 100便携式充电器| TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2006/08/29/xantrex-powersource-mobile-100-portable-charger/

Xantrex的这款PowerSource Mobile 100重量约为一个移动笔记本电脑电池，可与“不同电压和功率水平的多种电子产品”配合使用，售价129.99美元。你为什么要用这东西？因为当插上这个便携式充电器的交流电源插座时，笔记本电脑可以多获得2小时的电力，索尼PSP可以多获得12小时的电力，iPod Nanos可以多获得72小时的电力。这果汁真他妈的多。事实上，这些东西中的一个应该足够你和你那喜欢拥抱的毛骨悚然的叔叔艾尔一起度过整个露营之旅了。那家伙很奇怪。
