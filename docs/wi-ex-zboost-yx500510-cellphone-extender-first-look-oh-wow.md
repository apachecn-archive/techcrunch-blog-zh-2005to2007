# Wi-Ex zBoost YX500/510 手机扩展器第一眼:哦哇

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/15/wi-ex-zboost-yx500510-cellphone-extender-first-look-oh-wow/>

![](img/64b1f99cc1d78011009407bac59dfc33.png)

不是听起来像一个厌倦的老黑客，但我很少对我看到的任何东西感到兴奋。我知道。我是一个厌倦的老黑客。无论如何，Wi-Ex zBoost 手机扩展器是那些实际上让我心跳加速的东西之一，所以当它出现时，我决定打开盒子，试一试。

399 美元的双 zBoost 扩展器是 800/1900MHz 蜂窝信号的频率助推器。它支持 CDMA、GSM 和 TDMA(抱歉，没有 Nextel iDen ),应该可以将你的信号增强到 3000 平方英尺。总之，理论上是这样的。

![](img/e73d2f828388dd2a1c33829f314971a1.png)

它配有一根长天线、同轴电缆和设备本身。只有三个灯，当你有一个好的信号时，东西开始闪烁绿色。否则，事情会变得疯狂，你知道你需要把一些东西移到别的地方。

![](img/fcf94d79f26703be71ea8d03e0840149.png)我在地下室工作，家里任何地方都没有手机信号。我通常在没有信号的情况下运行 Sidekick 3，并且我总是会掉线。有一天，T-Mobile 公司派了一位好心的先生去我们家测试我们的信号。我们的邻居很好，在我家外面——前面和后面——我们的信号很好。他真的进了房子两步——就在门槛处——他的信号几乎为零。所以这就是我要面对的。

我安装了 zBoost，把天线放在窗户外面，在短短的几分钟内，我的 SK3 上就有了三到四个信号条。当我确定这不是侥幸时，它会拍一张“事后”照片，但它相当有说服力。这是我喜欢看到的——一个很酷的产品，正如它所宣传的那样。我会给它一些进一步的测试，但我适时地留下深刻印象。
