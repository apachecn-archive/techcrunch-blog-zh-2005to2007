# 雅虎！移动服务广告和 Goes Gamma 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/12/yahoo-mobile-serves-ads-and-go-goes-gamma/>

[![](img/e96c36f2fa9d828e46821f7662f26048.png)](https://web.archive.org/web/20160407215542/http://mobile.yahoo.com/) 雅虎！今天在巴塞罗纳 3GSM 发布了三个声明:雅虎！Go mobile 2.0 (Gamma)，LG 手机上的预装，以及其移动广告网络的推出。雅虎！Go 是雅虎的移动部分！的三屏计划，以优化他们的个人电脑，电视和移动服务。

雅虎！Go 是一个可下载的应用程序，让用户可以访问搜索、地图、邮件、体育、Flickr、新闻、金融、娱乐和天气。该应用程序可以在 100 多部手机上使用，但 LG 的新协议意味着它也将预装在他们的许多新手机上。

雅虎！的 m.yahoo.com 移动广告平台将在欧洲、亚洲和美洲超过 19 个国家发布。广告将会出现在浏览器页面的顶部，让用户可以点击查看更多信息。
