# Phozi:给 MySpace 带来一个照相亭体验

[![phozi.png](img/3c6218387196c2546d8928fb12e1846b.png) ](https://web.archive.org/web/20220808131700/http://www.phozi.com/) [Phozi](https://web.archive.org/web/20220808131700/http://phozi.com/) 是一个图片托管服务，为在线 MySpace 一代提供日式照片亭体验。

在线图像托管并不新鲜。这是一个拥挤的市场，各种服务之间差别不大。Phozi 托管的照片可以在 MySpace 和所有常见的社交网络和博客服务上显示。照片可以直接上传，也可以通过网络摄像头或手机拍摄。

Phozi 在编辑上传图像方面与 if 提供的有所不同。Phozi 提供了各种工具来定制每个图像。用户可以添加边界，邮票和自由形式的文字到一个图像，这从帐户在功能上类似于日本的照片亭。

也不是没有竞争对手。 [Cameroid](https://web.archive.org/web/20220808131700/http://www.cameroid.com/) 提供类似的在线功能，而 [WebCamGreetings](https://web.archive.org/web/20220808131700/http://webcamgreetings.com/) 通过带有网络摄像头的软件提供这种功能。我喜欢 Phozi 的是它的简单。围绕图像提供选择，并且没有学习曲线。不需要注册。

我能找到的形容 Phozi 最好的词就是可爱。它不会吸引每个人，但 OMG lol 16 岁的女孩会喜欢它。
