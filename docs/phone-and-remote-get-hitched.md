# 电话和遥控器挂钩

![](img/9276937cb3f45c2b8d95f2daee8de446.png)

丢失手机或电视遥控器的日子已经过去了。现在你可以同时失去他们。

手机杂志[的戴夫·怀特(Dave White)在](https://web.archive.org/web/20210228041718/http://www.mobilemag.com/content/100/340/C11655/)上发布了一款即将在韩国发布的支持无线和短信的手机——它还可以兼作电视、DVD 或 PVR 遥控器。还没有价格或可用性的消息。

怀特哀叹这毫无意义，因为它只在韩国可用，我同意这一点——但它看起来仍然很酷。
