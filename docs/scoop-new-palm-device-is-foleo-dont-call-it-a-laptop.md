# 独家新闻:新的掌上设备“Foleo:”不要称它为笔记本电脑

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/30/scoop-new-palm-device-is-foleo-dont-call-it-a-laptop/>

11 点 30 分，Palm 平台以及 Treo 系列智能手机的幕后主脑杰夫·霍金斯(Jeff Hawkins)将发布一款新产品，让我们震惊不已。问题是，你不想等到 11:30。这就是我们在这里的原因，朋友。

霍金斯的新产品被粉丝们命名为“鹰”，这是一个有趣的装置。根据一个仍然欠我们人情的可靠消息来源，Palm 将称它为“Foleo”。听起来挺酷的。

Foleo 不是手持设备，不像钨丝灯或 Lifedrive。也不是智能手机，像 Treo。这是一个“智能手机伴侣”，一种更广泛地与智能手机交互的方式。它配有全尺寸 QWERTY 键盘和大液晶显示屏，并通过蓝牙与你的 Treo(或其他智能手机)配对。Foleo 上的编辑在智能手机上匹配，反之亦然。虽然这听起来像是一种与手机交互的方式，但实际上它还有更多的功能。

它将拥有自己的操作系统，运行 Palm 的家酿 Linux 操作系统，这是第一款采用该操作系统的设备。它还能够独立于智能手机运行应用程序，这意味着它可以取代许多人的笔记本电脑。

它将于今年夏天以 499 美元(AR)的价格推出，随后不久将有来自选定开发者的应用程序推出。我们的消息来源称这是完全合法的，但我们可能会从今天晚些时候的公告中了解更多。词。
