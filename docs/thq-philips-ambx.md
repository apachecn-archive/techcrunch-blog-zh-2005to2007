# THQ +飞利浦= amBX

飞利浦昨天宣布了 adventure gaming 中环境智能的到来，这还不算太寒酸。他们与 THQ 公司签约，使用他们的 amBX 照明技术交付其备受期待的游戏《断剑:死亡天使》。

amBX 代表了环境智能发展的一个进步，并将试图通过将游戏世界扩展到现实世界来彻底改变游戏体验。amBX 将为游戏玩家提供完整的“感官环绕体验”,让开发者能够在游戏过程中使用真实世界中的光线、颜色、声音甚至气流。

> “混合了脚本语言、软件引擎和架构，
> amBX 平台旨在通过启用的设备(如 led 颜色控制灯、有源
> 家具、风扇、加热器、音频和视频)提供全新的玩家体验
> ，所有这些设备都战略性地放置在玩家房间的周围
> 。”

粉丝？加热器？活动家具？听起来像是我奶奶的旧房间。
