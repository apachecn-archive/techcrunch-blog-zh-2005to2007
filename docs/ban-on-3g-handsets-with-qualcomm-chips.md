# 禁止 3G 手机使用高通芯片

美国国际贸易委员会下令禁止含有高通芯片的新 3G 手机，因为这些手机侵犯了博通的一项专利。这有点令人困惑…看，已经在美国销售的手机被排除在外，这包括同一型号的未来销售。此外，目前的手机品牌也不需要改变。但是任何新的 3G 设备都必须放弃高通芯片。
