# UBiQUiO 503G Pocket PC 玩得很安全，不会偏离常规

![](img/2c933b3f8b24bd25a6dd447a4da4f972.png)

哦，看，另一个来自 UBiQUiO 的掌上电脑，503G。像它的前身、501 一样，它看起来仍然可疑地像摩托罗拉 Q，并重复使用它的许多功能。你会发现 GSM/GPRS/EDGE 和 UMTS/HSDPA 设备支持 802.11b/g Wi-Fi、立体声蓝牙并运行 [Windows Mobile 6](https://web.archive.org/web/20201130050207/http://crunchgear.com/2007/02/12/windows-mobile-6-the-straight-dope/) 。然而，我拒绝称它为“移动发电站”。

坦率地说，它似乎没有做任何不同于任何其他掌上电脑。当公司实际上开始创新他们的手机和智能手机时，比如即将到来的[三星抢尽风头](https://web.archive.org/web/20201130050207/http://crunchgear.com/2007/03/26/samsung-upstage/)和 [Helio Ocean](https://web.archive.org/web/20201130050207/http://crunchgear.com/2007/03/27/helio-ocean-hello-nurse/) ，为什么要满足于这样的事情呢？毕竟，熟悉不会滋生蔑视吗，尤其是在没有合同的情况下接近 800 美元？
