# Ajax 语言翻译器

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/01/01/the-ajax-language-translator/>

上周我在[阿贾克斯](https://web.archive.org/web/20150917142350/http://ajaxian.com/archives/2005/12/the_aptlynamed.html)看到了乔尔·帕里什的[阿贾克斯翻译](https://web.archive.org/web/20150917142350/http://ajax.parish.ath.cx/translator/)。这是一个即时的 Ajax 应用程序，可以创建英语、西班牙语、德语、意大利语、葡萄牙语和法语之间的实时翻译。非常圆滑。

Ajax Translator，如 [Babel Fish](https://web.archive.org/web/20150917142350/http://babelfish.altavista.com/) ，对于一次翻译一个单词或者在两种语言中具有相同语法结构的简单交流非常有用，但它不允许语法不一致。对于更复杂的通信，后端很快就会落后。

虽然前端很酷。:-)
