# 汤姆森 EH308 DAP:闪亮、小巧、轻盈

[![eh308.jpg](img/2daa870d172d34ffd3d89befd875548e.png)](https://web.archive.org/web/20130628181158/http://old.crunchgear.com/wp-content/uploads/eh308.jpg "eh308.jpg")

汤姆森在 DAP 世界中是一个相当小的玩家(谁能和[苹果](https://web.archive.org/web/20130628181158/http://crunchgear.com/2007/05/29/want-free-ipod-games/)和[微软](https://web.archive.org/web/20130628181158/http://crunchgear.com/2007/05/28/one-million-zunes-somehow-sold/)相比呢？)，但如果策略得当，其闪亮的新 EH308 可以吸引一些眼球。这是一个 1.8 英寸有机发光二极管显示屏的小家伙。尽管它的尺寸很小，而且由于有机发光二极管显示器，电池寿命可与一些较大的家伙媲美，约为 15 小时。8GB 硬盘使其与最大的 nano 直接竞争——也就是说，除非苹果决定给我们一个惊喜，不在下周的 WWDC 发布新的 iPods。

EH308 可以播放所有正常的文件类型:MP3、WMA、JPG、WMV 和 MP4。别的不说，至少看起来不像屁股。
