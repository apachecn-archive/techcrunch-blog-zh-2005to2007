# 威瑞森大胆使用移动电子邮件

威瑞森终于摆脱困境，增加了一个支持雅虎的新电子邮件客户端、AIM、AOL、Windows Live Hotmail、Verizon.net 和 POP3/IMAP。哦等等。我说过他们把头从屁股里伸出来了吗？我的意思是他们把价格推得更高，因为他们收取 5 美元的服务费，而其他运营商一分钱都不收。这项服务在 LG enV 和 MOTORAZR V3m 等 select Get Now 手机上提供，由 OZ 提供支持。当然，如果你是美国选择高级计划的订户，你不会被收取 5 美元，但你必须每月支付 79.99 美元。帮我报名！
