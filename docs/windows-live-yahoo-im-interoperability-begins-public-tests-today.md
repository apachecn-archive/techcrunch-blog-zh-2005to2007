# IM 互操作性:不再只是一个梦想

微软刚刚宣布，今天是其 IM 客户端和雅虎之间[互通的第一天！带语音的信使可用](https://web.archive.org/web/20221209125722/http://www.microsoft.com/presspass/press/2006/jul06/07-12IMInteropPR.mspx)。微软表示，这是两个截然不同的全球消费者品牌首次让他们的即时通讯客户端实现互操作。该公司表示，近 3.5 亿账户的用户群是世界上最大的。IM 互操作性花了这么长时间，我以为它永远不会发生。

Windows Live Messenger 是 MSN Messenger 的最新版本。[雅虎语音信使](https://web.archive.org/web/20221209125722/http://messenger.yahoo.com/)是雅虎最新的即时通讯客户端。

IM 系统之间缺乏互操作性一直是用户抱怨的主要问题。像用于 Windows 的 Trillian 或用于 Mac 的 Adium 这样的服务有助于弥合服务之间的差距，但失去了用户可能更喜欢的一个或另一个系统的独特界面和一些功能。

对于这两个系统的忠实用户来说，今天的新闻应该会让 IM 更有价值。很高兴看到，尽管两家公司都在努力为自己的即时通讯服务开发增值界面，但他们并不担心失去客户，以至于假装其他即时通讯客户端不存在。
