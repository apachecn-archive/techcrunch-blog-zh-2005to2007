# 索尼爱立信发布新手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/08/sony-ericsson-announces-new-phones/>

![](img/572d2bcad90cf6088d5c1acb97042a84.png)

今天，索尼爱立信宣布了两款不能保证刺激你感官的新手机。T250 是一款非常基本的手机，可能会在当地运营商的预付费级别上销售。功能包括双频 GSM(哎哟)，VGA 摄像头，调频收音机，以及银色或黑色油漆工作。不算太坏，但是没什么特别的。

另一个竞争者是 S500 slider，这是一款带 EDGE 的四频段 GSM 手机，具有 200 万像素的摄像头，蓝牙，MP3 播放器和博客软件。本质上，我们希望在 CG 总部周围有一些这样的东西。这两款手机都将于今年第三季度上市，应该会登陆美国市场。
