# Strayform 尝试新的独立音乐模式

 [](https://web.archive.org/web/20210123050944/http://www.strayform.com/) [Strayform](https://web.archive.org/web/20210123050944/http://www.strayform.com/) 是一家得克萨斯州的初创公司，像 [SellaBand](https://web.archive.org/web/20210123050944/http://www.crunchbase.com/company/sellaband) 和[最近](https://web.archive.org/web/20210123050944/http://www.beta.techcrunch.com/2007/07/03/amie-street-celebrates-first-birthday-free-music-for-techcrunch-readers/)资助的[李正吉街](https://web.archive.org/web/20210123050944/http://www.crunchbase.com/company/amiestreet)一样，为未签约的艺术家提供一种推广和销售他们音乐的方式。

像 SellaBand 一样，艺术家注册，上传他们的一些音乐，然后为他们想要创作的新音乐提出建议。粉丝可以收听和下载音乐(DRM 免费)，并直接捐赠给他们喜欢的提案。这些提议都是不同的。例如，一位艺术家说，他会在歌曲中提到承诺最多的人的名字。

这项服务更像是塞拉班德，而不是李正吉街。SellaBand 还允许艺术家上传音乐并接受捐赠。如果捐款达到 50，000 美元，就像他们已经为几位艺人捐款一样，这位艺人将与一家唱片公司签约。相比之下，李正吉街只是让艺术家在网站上出售他们的音乐。下载从免费开始，随着下载次数的增加，价格稳步上升。

Strayform 几乎没有什么新闻，但是，莫名其妙地被福克斯新闻频道在一个 3 分钟的片段中报道了。该视频可在他们的主页上获得。我喜欢这项服务，但网站确实需要重新设计。
