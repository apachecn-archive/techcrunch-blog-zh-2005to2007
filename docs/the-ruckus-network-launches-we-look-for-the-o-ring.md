# Ruckus 网络推出，我们寻找 O 型圈| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/22/the-ruckus-network-launches-we-look-for-the-o-ring/>

又一天，又一个 iTunes 的竞争对手。Ruckus 已经存在了几年，但它最终团结起来，成立了一个连贯的音乐商店。但“音乐商店”可能是一个错误的术语，因为所有从 Ruckus 下载的音乐，甚至是主要唱片公司的音乐，都是免费的。

这个想法是，通过在一个易于导航、组织良好的系统中提供音乐，以及免除订阅费，唱片公司可以在自己的游戏中击败 P2P 下载，同时赚一些钱。

当然，这听起来不错，直到你真正看到它是怎么回事。

骚动不是对所有人开放的；基于广告的下载服务仅限于大学生，就像啤酒棒一样。此外，它不能在你的 iPod 甚至 Zune 上运行，因为歌曲是微软的 Windows Media WMA 格式，附带了烦人的数字版权管理。如果你的廉价奶奶给你弄了一个 PlaysForSure POSes，这是个好消息，但我们有一种感觉，其他 98%的女同学会让他们的 LimeWire 继续运行。
