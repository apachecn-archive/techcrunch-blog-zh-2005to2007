# 费城人获得基于 RSS 的音频旅游

![](img/d798c942c130ce3303caeeecf7b8f526.png)

如果你是像我一样生活在费城或来自费城的幸运儿之一，那你就等着享受吧。GoPhila.com 发布了一项名为“费城之声”的服务。这项服务的目标是这个星球上最好的城市的居民和游客，并提供全面的旅游，有趣的事实，生动的音乐，有趣的故事，以及一堆附加在名词上的形容词。了解这座城市的历史，以及它是如何成为今天这样繁荣的大都市的。这项服务甚至提供了一个 RSS 源，这样你就可以经常用新的“有趣的故事”自动更新你的 iPod 或 MP3 播放器。如果你不是费城人，或者你近期不会去那里，那么你可能应该停止关心，回去听 ABBA。
