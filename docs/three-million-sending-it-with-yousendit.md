# 300 万用 YouSendIt  发送

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/12/07/three-million-sending-it-with-yousendit/>

[![](img/f7e305f01b45fb7a5cc74c21453e8313.png)](https://web.archive.org/web/20160305124227/http://www.yousendit.com/) 文件传输公司 [YouSendIt](https://web.archive.org/web/20160305124227/http://www.yousendit.com/) 昨天告诉 TechCrunch，它最近已经超过了第三百万用户。虽然我们最初对 YouSendIt 的体验并不顺利，但我们后来认为它是一项相当好的服务。我们今天采访了首席执行官伊万·孔，了解公司最近的动向。

“新闻稿背后的真正原因，除了想稍微吹嘘一下，是我们是证明 Web 2.0、Office 2.0、Enterprise 2.0 的一个很好的例子，不管你想叫它什么，”Koon 说。“我们一开始是免费的，任何人都可以访问我们的网站，使用我们来发送大文件，我们已经非常成功地从面向消费者的网络服务发展成为面向商业用户的网络服务。”

Koon 说，YouSendIt 的大多数客户是中小型企业的商业用户。该公司还报告说，自今年 9 月以来增长了 300%，同期付费用户增长了 200%。

“就总收入而言，我们的季度环比增长率约为 50%，”Koon 说。“消费者不一定为服务付费，但企业用户非常愿意为服务付费，只要这有助于他们开展工作，并且是他们信任的服务。”

使用 YouSendIt，发件人可以选择密码保护或通过登录进行身份验证来控制谁可以接收他们的文件。Koon 说，该公司在东西海岸有 200 多台高度安全的文件服务器。

我们发现与 YouSendIt 相当的其他文件共享网站有 [Pando](https://web.archive.org/web/20160305124227/http://www.techcrunch.com/tag/Pando/) 和 [Zapr](https://web.archive.org/web/20160305124227/http://www.techcrunch.com/tag/Zapr/) 。
