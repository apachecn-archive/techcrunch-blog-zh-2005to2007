# 威瑞森 V640 EVDO ExpressCard 已审核

![v640macbook.jpg](img/c0d60c70662ab480fef98b061ad6f32c.png)

如果你的笔记本电脑中有一个 ExpressCard 端口，而不是老式的 PC 卡，你可能会想，是否所有的硬件制造商都会醒来，为你早期采用的自己创建卡。现在，一些公司已经加入了竞争，包括 威瑞森无线公司，为其 EVDO 连接提供了 Novatel V-640 ExpressCard，GearLive 的工作人员为了我们的阅读乐趣测试了一款。

判决结果？评测人员在他们的 MacBook Pro 中挖掘了这种卡，称赞它的简单设置和 370kbps 的全球下载速度，当他们靠近西雅图市区时，速度增加到了 48kbps。虽然没有达到理论上的最高速度 500kbps，但考虑到接收并不完美(与威瑞森无线打交道时，这并不奇怪)，这还是不错的。上传速度稍慢，为 114kkbps，但仍可接受。两年期合约 99 美元看起来很便宜，但无限宽带接入 59.99 美元可能有点负担。即便如此，Gear Live 还是给威瑞森 V640 打了 9/10 的分。看起来不错。
