# 南非初创企业推出移动 VoIP 服务

据南非开发者称，Yeigo 的 2.0 版本刚刚发布。Yeigo 是一款新的手机应用程序，它使用手机的互联网连接来拨打电话和发送短信或其他即时消息。最棒的是，这些选项都是免费的，比如用 Yeigo 打电话给其他人。打给非 Yeigo 用户的电话也会减少 80%。Yeigo 还将允许用户通过 GoogleTalk、MSN、Yahoo！，AIM，ICQ 和 Jabber。这项服务可以拨打国内和国际电话。
