**什么事？**

CommonTimes 是一个新闻社会化书签网站。网络上的用户将故事加入书签，并加上标签。突出程度由标记新闻源的用户数量决定。

![](img/a9f9e58e64d00cfe492bdffa9dc96053.png)

用他们自己的话说，*“common times 是一个新闻读者的社会化书签网站。或者，用更简单的话来说，CommonTimes 是一个新闻网站，它根据你选择添加书签的频率来发布新闻。我们的读者收集某些故事的范围越广，它们在我们网站上出现的位置就越显著。*

如果你认为主流媒体存在于自上而下的内容控制的一个极端，即一小群编辑决定新闻中出现的内容，CommonTimes 则完全相反——这是一个自下而上的新闻网站，基层网络读者在浏览时通过书签确定头条新闻。

相比之下，CommonTimes 之于新闻，就像 Del.icio.us 和雅虎的 MyWeb 之于互联网书签一样。与谷歌新闻(一个局限于主流媒体报道的封闭、自动化系统)相反，CommonTimes 是一个开放的社区系统，接受来自任何新闻网站或博客的内容，完全由我们的读者驱动。例如，虽然 Slashdot 和 Grist Magazine 提供了一个严格控制的自上而下的技术和环境新闻过滤器，很少成为主流媒体，但我们的部分提供了一个自下而上的故事视图，我们的读者认为来自任何来源的重要故事都可以很好地整合后者的故事。

我们社区决定的就是新闻。
[链接](https://web.archive.org/web/20201021065034/http://blog.commonbits.org/2005/07/announcing_the_.html)

有 web 2.0 元素:社会化书签、出版商和用户标签、书签故事的评论，以及所有内容的 RSS 提要。这是一个有趣的实验，我们期待参与其中。也有一些简单易用的书签工具，包括如何直接从 bloglines 添加书签的技巧。

要深入了解，请见布莱恩·德尔·维奇奥，他写了一篇精彩的博客(并向我们透露了这项服务)。
