## 2.弗雷德·威尔逊思考 Web 2.0 公司

我有机会在 Web 2.0 大会上见到弗雷德·威尔逊——我只希望我能更长时间地听取他的意见。他[写了一篇关于“点”对“端到端”解决方案的博客文章](https://web.archive.org/web/20220119072253/http://avc.blogs.com/a_vc/2005/10/point_solutions.html)。单点解决方案是独立的网络服务——想想被雅虎收购前的 Flickr。端到端解决方案是同一屋檐下的一组服务——想想像 MSN、Yahoo 等门户网站。一般来说，大众用户喜欢门户类型的解决方案，但是点式解决方案几乎总是更好。当然，投资者和企业家在决定最终退出策略时，需要考虑这一点。他们的目标必须是被门户网站收购吗？

## 6.博客订阅源数据

博客在线[在他们的博客上发布了一些很棒的数据和见解](https://web.archive.org/web/20220119072253/http://blog.ask.com/2005/10/deep_thoughts_a.html)。

虽然有很多很多博客，但 Bloglines 表示，如果一个博客至少有一个订阅者，那么它就很重要——如果你考虑到 Bloglines 的市场地位(仍然占主导地位),如果你忽略几乎每个人都订阅自己的订阅源这一事实，这是一个公平的衡量标准。以此衡量，大约有 130 万个“重要”博客。四个月前，这样的博客有 110 万个。这就构成了“长尾”。

事情很快就结束了。只有大约 37，000 个博客拥有 20 名或更多的读者，只有 437 个博客拥有至少 1，000 名读者。他们用来称呼大家伙的科学术语？“非常甜蜜”。:-)
