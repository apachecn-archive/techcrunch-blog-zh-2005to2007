# 理光推出两款新的加固型无线摄像头

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/30/ricoh-drops-two-new-ruggedized-wireless-cameras/>

如果你有一台 813 万像素的相机，你想改进它，你会怎么做？你会让它防震。而且防水。和防尘。然后你会有一个坚固的“运动相机”，非常适合户外运动者。

但是户外运动者需要轻便，所以你不会想带着那些讨厌的电缆。所以你要确保这个相机支持蓝牙。这样，您可以立即将任何或所有照片发送到您的配对设备(手机、pda、笔记本电脑)，而不必担心 USB 或卡传输，这在逃离熊时可能很棘手。或者，如果您真的很聪明，您可以添加内置 WEP 和 WPA/WPA2 安全功能的 WiFi。这样隔壁房间的人就不会浏览昨晚帐篷里约会的照片了。

幸运的是，理光的好人已经为您做到了这一点。他们还包括 26MB 内置存储空间、SD 兼容性和 2.5 英寸 LCD，所有这些都在两台摄像机中:Caplio 500SE 型，蓝牙售价 990 美元；W 型，没错，WiFi 售价 1100 美元。每个摄像头的重量都超过了一磅，所以“轻包装”是一个比喻，但至少你知道你不会毁了它，除非真的，真的尝试。

不好意思，露营者们，这是日本-只是为了这个相机。
