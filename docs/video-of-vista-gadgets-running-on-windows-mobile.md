小心 iPhone！看起来你不是镇上唯一一个运行小工具的设备。在这个视频中，我们看到 Windows Vista 小工具运行在基于 Windows Mobile 的手机上，这要感谢 Windows Mobile 开发者 Mel Sampat。很可爱，虽然我真的不认为这将很快成为除了梅尔以外的任何人可用。还有，谁在乎？无论如何，窗口小部件做的大部分事情都适用于 Windows Mobile(时钟、日历、布莱克的妈妈等。)

只是提醒一下，看起来基于 flash 的视频播放器不再工作了。点击网站上的 WMV 链接观看视频。
