如果你认为 Mac 用户在电脑上使用 Wiimote 时会被冷落，那你就错了。一个名为 DarwiinRemote 的完美应用程序允许你在 Mac 上使用蓝牙 Wii 控制器作为外设。在视频中，这位使用遥控器的绅士向我们展示了如何通过使用这款为 OS X 开发的程序来拖动窗口并与 OS X 互动。甚至还有一个实时图表在后台运行，显示 Wiimote 所做的活动。
