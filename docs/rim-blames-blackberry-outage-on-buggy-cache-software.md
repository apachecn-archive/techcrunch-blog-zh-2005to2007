# RIM 将黑莓停机归咎于错误的缓存软件

[![bberyreason.jpg](img/c5e17fbe8564018004a6084be9a17cbc.png)](https://web.archive.org/web/20210228081110/https://beta.techcrunch.com/wp-content/uploads/2007/04/bberyreason.jpg "bberyreason.jpg")

本周早些时候，由于 2007 年黑莓大停电，RIM 搞砸并几乎摧毁了西方文明。我们现在知道了灾难的原因:错误的软件。RIM 安装了一个未经测试的缓存软件，它不太符合标准。缓存和公司的数据库进行了激烈的斗争，让中层管理人员陷入了某种恐慌。

这个故事的寓意是永远不要安装未经测试的软件，企业版或其他。哦，还有别再沉迷于电子邮件了。两者都是同样好的教训。
