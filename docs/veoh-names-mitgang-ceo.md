# Veoh 任命 Mitgang 首席执行官

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/07/26/veoh-names-mitgang-ceo/>

[![](img/c7f4d165f06ff8cc69fe4ac4144e7d67.png) ](https://web.archive.org/web/20220808133757/http://www.crunchbase.com/company/veoh) P2P 和网络视频分享服务 [Veoh](https://web.archive.org/web/20220808133757/http://www.crunchbase.com/company/veoh) 宣布，雅虎前高管史蒂夫·米特冈被聘为首席执行官。

Mitgang 以前是雅虎的高级副总裁，领导了雅虎下一代上下文广告平台 Project Panama。

Veoh 在 6 月份完成了一轮 2600 万美元的融资[并推出了分布式电视平台](https://web.archive.org/web/20220808133757/http://www.beta.techcrunch.com/2007/06/14/veoh-takes-26million-round-c/) [Veoh TV](https://web.archive.org/web/20220808133757/http://www.beta.techcrunch.com/2007/06/20/veoh-announces-veohtv-a-sort-of-distributed-joost/) 。据《洛杉矶时报》报道，Veoh 的创始 CEO Dmitry Shapiro 已经被任命为首席创新官。
