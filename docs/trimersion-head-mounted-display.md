# Trimersion 头盔显示器

![](img/75dcd710f1588cbd311f47381dbf6fab.png)

Trimersion 头戴式显示器是一种虚拟现实设置，适用于 PC、PlayStation 2 和 Xbox 上的 FPS 游戏。您所要做的就是将适配器插入游戏外围设备，一切就绪。PC 不需要驱动程序，PlayStation 2 和 Xbox 也支持适配器。基本上，传感器跟踪 HMD，所以当你移动你的头，你的角色移动它的头。开枪的时候也是一样。

由于枪在 FPS 中并不真的移动，把枪举到屏幕上会很烦人，所以 Trimersion 也给了你玩鼠标的选项。HMD 售价高达 395 美元，并且只支持少数几款游戏，包括*光环、光环 2、反恐精英*和*荣誉勋章*。由于大多数与 HMD 兼容的游戏都是旧的，所以最好远离这个极其昂贵的产品，直到它至少与下一代游戏机和更多游戏兼容。
