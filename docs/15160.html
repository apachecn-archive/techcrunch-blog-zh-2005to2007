UMD:大PDA还是小笔记本？

起初我以为这位女士只是有一双男人的大手，但结果是VIA NanoBook UMD只有1.2×6.7×9英寸。这款1.87磅重的笔记本是与第一国际计算机公司(FIC)合作制造的参考设计，已被Packard Bell采用，作为EasyNote XS出现在欧洲市场。它目前正在Computex 2007上展示。

这款小巧的笔记本电脑采用1.2GHz VIA C7-M ULV处理器，由高达1GB的DDR2内存和30GB或60GB硬盘支持，可以挤出长达5小时的Microsoft Windows XP或Vista Basic excitation电池续航时间。



它有一个全键盘，配有我见过的最小的触摸板，一个7英寸的WVGA屏幕，内置802.11g Wi-Fi，蓝牙和以太网支持，外加一个DVI和两个USB 2.0端口。显然，在屏幕旁边有一个USB插槽，用于各种附加模块，如世界时钟和日历、GPS接收器、VoIP和宽带无线。(我假设其中一个模块是彩色屏幕旁边的单色东西。)
