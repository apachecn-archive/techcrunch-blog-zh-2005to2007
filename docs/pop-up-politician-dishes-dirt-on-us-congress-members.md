# 弹出式政客对美国国会议员大放厥词

 [](https://web.archive.org/web/20221002134350/http://sunlightlabs.com/popuppoliticians/) 当用户悬停在一个链接上时，有许多小工具会从外部弹出信息，但这是我见过的最酷的一个:[弹出政客](https://web.archive.org/web/20221002134350/http://sunlightlabs.com/popuppoliticians/)。这个来自[阳光基金会](https://web.archive.org/web/20221002134350/http://www.sunlightfoundation.com/)的 ajax 小工具可以让任何人将他们博客上的美国国会议员的名字链接到一个关于那个政客的弹出窗口。这里是[工具运行中的一个例子](https://web.archive.org/web/20221002134350/http://www.sunlightfoundation.com/node/914)。报道美国政治事件的博主可以很容易地将 Sunlight 的小工具放入他们的帖子中，以增加博客报道的价值，并帮助一个很酷的组织传播其工作。

该窗口包括国会百科上政客页面的链接，这是一个覆盖美国国会成员的维基百科，OpenSecrets.org 的页面跟踪上述政客的竞选捐款，以及华盛顿邮报的国会投票数据库。

这是 ajax、wikis 和其他 Web 2.0 技术被利用的一种简单但真正有用的方式。
