# SanDisk 的 Wi-Fi 比 Zune 的好，嗯，SanDisk，也许| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/23/sandisks-wi-fi-better-than-zunes-says-um-sandisk-maybe/>

昨天，我们提到 [SanDisk 用它的 Sansa Express](https://web.archive.org/web/20150926121604/http://crunchgear.com/2007/01/22/sandisk-sansa-express-vs-apple-ipod-shuffle/) 追逐苹果的 Shuffle 购物者。今天，CoolTechZone.com 表示，SanDisk 正在用其支持 Wi-Fi 的连接播放器寻找 Zune 的潜在买家。

不幸的是，该网站并没有详细阐述这个话题，尽管显然已经和 SanDisk 谈论过 DAP。

故事说，“在我们与 SanDisk 的会面中，该公司说，”Sansa Connect 是 Zune 所没有的。" SanDisk 当然是指 Sansa Connect 的 WiFi 功能."

SanDisk 发现 Wi-Fi 除了音乐分享还有其他用途，这真的令人惊讶吗？Sansa Connect 的无线功能(例如网络电台和访问 PlaysForSure 内容)肯定比 Zune 的社交网络功能更有用。我甚至怀疑这是否值得讨论。但是仅仅因为一个制造商给一个设备增加了一个功能，这就自动意味着它在“追求”另一个具有类似功能的产品吗？没错。

这里我不想吹毛求疵(但我是一个有点 ~~dick~~ skunk 的人，所以我会)，该网站错误地声称微软拥有第一个支持 Wi-Fi 的音乐播放器。MusicGremlin 在 2006 年初出了一张专辑，但那并不是第一张。我的观点是 DAP 市场还有很大的增长和创新空间。并非所有东西都是 Zune/iPod 杀手。
