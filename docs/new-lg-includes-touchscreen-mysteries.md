# 新 LG 包括触摸屏，神秘

当苹果发布 iChat Mobile(或者，你知道，iPhone，或者其他什么东西)时，我们许多博客专家都希望它能配备某种触摸屏。鉴于这种技术在过去几年里变得越来越便宜，触摸屏手机并没有人们想象的那么多。

除了某些智能手机，我们在许多手机上都看不到它们，索尼爱立信 Z558 是个例外。这就是我们激动的原因——激动！–查看 LG lk 850 的这些照片。除了底部的几个硬件摇杆，这款手机显然是通过触摸屏控制的。

这意味着界面可以根据用户使用手机的情况动态切换，我们可能会猜测虚拟 QWERTY 用于发送短信，但由于我们除了照片和型号之外什么也没有，所以这只是猜测。
