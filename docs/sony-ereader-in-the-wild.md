# 野生索尼电子书阅读器

来自 Make 的菲尔·托龙(Phil Torrone)拿到了一台索尼电子阅读器，然后开始对它拍照，而不是用它来阅读。无论如何，它实际上看起来相当不错，而且我们敢说，很有用！

到目前为止，从我们所看到的来看，字体很好，很大，即使在阳光下也很亮。它播放 MP3 和 AAC 文件，并使用 SD 卡来扩展内存。PT 将很快对电子书阅读器进行全面审查，如果有类似的东西，我们可能不得不自己去买一本，在公共交通工具上阅读。也就是说如果我们真的出去了。
