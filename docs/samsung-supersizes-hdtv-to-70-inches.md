# 三星将高清电视尺寸扩大到 70 英寸

![samsung.jpg](img/c739a695fc5910f35e370b876d61ac2b.png)

真不知道 SK 的公司为什么企图和三星竞争。他们拥有世界上最纤薄的手机，他们似乎是“世界第一”之王，所以最新的产品以其存在为世界增光添彩，不应该成为一个真正的惊喜。三星在韩国推出了一款漂亮的高清电视，一款 70 英寸的液晶电视，里面装满了发光二极管，婊子们。它还拥有 500，000:1 的动态对比度。像哇哦！

这是怎么回事？一种独特的局部调光技术允许 LED 背光在黑暗区域关闭。绿色变得更绿，因为能耗降低了 50%。这个坏男孩被包裹在闪亮的“钢琴黑”边框中，带有钛铬合金装饰和双隐藏式扬声器以及后置低音炮。这是事实，你知道它，你想要它。你可以在下半年给自己买一个，所以开始存钱吧。
