# 吊索媒体开发网络设备

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/20/sling-media-developing-networking-equipment/>

![](img/8903dbdfa129c438777c6da1ef2093c3.png)

看来，吊索媒体正在寻求打入网络业务。[根据最近的论坛帖子](https://web.archive.org/web/20150919155206/http://www.slingcommunity.com/blog/forum/thread/18652/Where-Can-I-buy-the-Slinglink---How-much-does-it-cost/)，Sling 正在开发一款 85Mbps 的电力线网络适配器，名为 SlingLink Turbo。该设备将与 Sling 目前的 Sling box 系列一起使用，并将于 2 月 27 日亮相。Sling 也在开发一种桥和开关组合，预计价格为 150 美元。
