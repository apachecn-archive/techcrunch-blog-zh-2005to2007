To-Conne i-Lit扬声器:没有那些烦人的人的即时俱乐部氛围

这个周末，我可以留在家里，用这些(而不是那些 ) i-Lit扬声器装饰每个角落，来混合我自己的DJ设备，从而为自己节省50美元(花在观看保罗·欧肯弗德的“DJ”节目上，如果你能把他的所作所为称为DJ的话)。看，它们会随着音乐的节拍以三种颜色中的一种(猜猜是哪几种)发光。它们不是最强大的，每个频道的额定功率只有3瓦，但是你显然只是为了好看才买的。随着这些连接到一些高节奏的节拍，它就像你自己的帕查在你的房子里，少了俱乐部毒品。我想这是个人选择。

这些即时交友软件将于下月登陆日本，价格合理，为47美元，仍低于我观看奥肯福德走过场的费用。
