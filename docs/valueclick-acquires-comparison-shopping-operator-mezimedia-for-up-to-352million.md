# ValueClick 以 3.52 亿美元收购比较购物运营商 MeziMedia

[![valueclick.jpg](img/3d63b7494c4fe1e0ce37d94cd0c4543f.png)](https://web.archive.org/web/20220813204030/http://www.valueclick.com/)

总部位于洛杉矶的 MeziMedia 是一家比较购物公司，运营的网站包括 Smarter.com 和在线优惠券网站 CouponMountain.com。

华尔街日报[报道](https://web.archive.org/web/20220813204030/http://online.wsj.com/article/SB118459404255267516.html?mod=rss_whats_news_technology&apl=y) ValueClick 首席执行官 Tom Vadnais 表示，收购 MeziMedia 符合 ValueClick 的收购理念，并“赋予 ValueClick 在美国的比较购物规模、更强的搜索营销专业知识和在中国的影响力。”

据说 MeziMedia 是盈利的，年收入为 4000 万美元。

继谷歌收购 [DoubleClick 和微软](https://web.archive.org/web/20220813204030/http://www.beta.techcrunch.com/2007/04/13/google-spends-31-billion-for-doubleclick/)收购 [aQuantive 之后，最近有传言称 ValueClick 本身就是收购目标。](https://web.archive.org/web/20220813204030/http://www.beta.techcrunch.com/2007/05/18/microsoft-pays-6-billion-for-aquantive/)
