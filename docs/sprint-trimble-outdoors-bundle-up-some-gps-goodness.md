# Sprint，Trimble 户外捆绑一些 GPS 的好处

如果你在 Sprint/Nextel 上，拥有他们 12 部支持 GPS 的手机中的一部，并且你经常发现自己在户外漫步(而不仅仅是在公园的男厕所里闲逛，Nicholas)，Trimble GPS 应用捆绑包每月 6.99 美元是值得的。该软件包为您提供了该公司的三种移动 GPS 解决方案:AllSport GPS、Geocache Navigator 和 Trimble Outdoors。这个组合非常棒，可以让你监控你的健身目标，使用你的手机进行地理藏宝和导航，以及户外旅行规划、地图绘制和 GPS 导航。点击跳转了解更多应用详情或[点击此处](https://web.archive.org/web/20181126143314/http://www.trimbleoutdoors.com/GPSpack.aspx)。

 AllSport GPS
通过 AllSport GPS 应用程序，希望快速启动或保持健身计划的手机用户可以通过监控户外锻炼的时间、速度、消耗的卡路里和行进的距离来跟踪锻炼活动。此外，用户可以将地图图像和推荐路线下载到手机上，查看成绩统计数据，并存储锻炼数据以监控进度或与朋友分享。该应用程序支持各种户外活动，包括步行、跑步、骑自行车和山地自行车。

地理藏宝导航器
地理藏宝是一种高科技寻宝游戏，正迅速获得全球消费者的广泛采用。Geocache Navigator 是唯一一款为手机带来地理藏宝功能的应用。用户的手机无缝连接到世界上最大的地理藏宝数据库。使用他们的 GPS 电话，地理勘探者可以使用一种独特构造的、类似雷达的界面来寻找宝藏，指引他们到达目的地。此外，手机屏幕为最有经验和新手用户提供基于指南针的导航、地图、速度、方向、位置、距离、缓存描述日志、提示和其他有用信息。

Trimble Outdoors
Trimble Outdoors 是一个完整的户外旅行规划和导航解决方案，允许消费者在线查看和规划旅行，将路线、地图和路点无线发送到他们的 GPS 手机，然后使用手机来指导他们的冒险。该应用程序允许用户获得他们的位置地图，用 GPS 面包屑轨迹跟踪路线，并标记和编辑沿途的路点。该应用程序还具有一个数字指南针，显示用户的坐标，速度和方向。用户还可以利用手机的多媒体组件来创建旅行的照片、视频和音频记录，以便在线分享和在谷歌地球上查看。
