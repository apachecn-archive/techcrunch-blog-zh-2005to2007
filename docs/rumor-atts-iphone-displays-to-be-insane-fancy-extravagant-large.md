# 谣言:美国电话电报公司的 iPhone 看起来疯狂、花哨、奢侈、超大

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/03/rumor-atts-iphone-displays-to-be-insane-fancy-extravagant-large/>

![iphoneatt.jpg](img/469a8035f77737f89178f2263b8abd2f.png)

从[曲线](https://web.archive.org/web/20150913052837/http://crunchgear.com/2007/05/03/att-gets-first-crack-at-blackberry-curve-surprised/)到 [iPhone](https://web.archive.org/web/20150913052837/http://crunchgear.com/?s=iphone) ，美国电话电报公司在今年夏天得到了所有热门手机的喜爱。然而问题依然存在:这家航空公司到底忠于哪个品牌？我们自己也发现很难去关心。这条曲线背后有美国电话电报公司的巨额营销预算，但有报道称，与我们即将经历的不可避免的 iPhone 痛苦相比，这条曲线显得苍白无力。

天才少年说，美国电话电报公司的零售店(你知道，以前是美国电话电报公司，后来是 Cingular，但现在不是了)正在增加拐角和端盖展示，这是我们这些凡人从未见过的。这些支架宽 7×3 英尺，有自己的电源，并与互联网(或至少是网络)相连。这比 [Sync](https://web.archive.org/web/20150913052837/http://crunchgear.com/?s=samsung+sync) 得到的夹子和手机充电器多一点。

我们猜测，该单元周围将会有巨大的、疯狂的平板显示器，帮助你“滑动解锁”的精灵，以及比基尼女孩，以确保没有遗漏任何东西。当这些家伙来袭时，你知道我们会第一个向你展示他们是多么的耀眼。
