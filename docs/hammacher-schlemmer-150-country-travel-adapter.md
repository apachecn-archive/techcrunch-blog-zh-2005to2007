# Hammacher Schlemmer 150 国旅行适配器

![](img/c1b661c66e2389015caaa03f45c6e7ae.png)
我不确定这款 150 国旅行适配器有多老，但值得注意。它非常小巧，可以同时为 AC 和 USB 充电。此外，它还有一个内置的电涌保护器。

该适配器可配置为在整个欧洲、澳大利亚、非洲和美洲工作。它将花费你 29.95 美元，但它的效用似乎超过了它的价值。
