# 作品中的个人审查

让你们卑微的记者们懊恼不已的是，我们试图让这里的事情保持平稳(这是“安全工作”，n00bz)。如果你支持这类事情，那么你很可能也是那种认为地铁有点粗鲁的人。你会想在 PTC 上市的那天买一台。

你会问，什么是 PTC？这是一个个人电视审查程序。使用您的 DVR，PTC 会监控输入的隐藏式字幕流。它会扫描 CC 流中的脏话，从而延迟电视的吞吐量。当它们被检测到时，就会从音频中过滤掉，表面上是用哔哔声甚至沉默代替了冒犯性的措辞，这让《杰瑞·斯普林格》的一集变得更加无趣。

虽然我们认为这会让《迷失》变得非常无聊，而且对屏幕上的图像没有任何影响，但这个想法很新颖。目前它还处于计划和理论阶段，但是如果你开车经过爱达荷州，你就会知道这种清洁设备有市场。
