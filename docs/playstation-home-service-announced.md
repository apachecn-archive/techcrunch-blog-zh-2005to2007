# Playstation Home 服务宣布推出

![](img/d0a8fb87cd67df07dcafe78eb453fbd2.png)

索尼似乎正在寻求与任天堂的 Mii 频道和*第二人生*竞争。菲尔·哈里森今天发布了 Playstation Home，在这里你可以与其他 PS3 用户互动，购买虚拟家具和服装，炫耀游戏中的奖杯。听起来有点像《动物穿越》*与*《第二人生》*的混合，而不是任天堂 Mii 的抄袭版本。*

 *你也可以通过一项名为 LittleBigPlanet 的服务来创建自己的游戏，这项服务将与 Playstation Home 同时推出。这是索尼将 PS3 推向许多人手中并使其成为游戏世界中更受欢迎的主机所真正需要的。

索尼宣布为游戏玩家提供 Playstation 家庭服务*
