# Open C 对移动应用开发者的挑战

![forumNokia.jpg](img/8923d46e8cd46d763d9f729b74e2ef14.png)今天，诺基亚宣布了首届“Open C 挑战赛”，这是一项为 Symbian OS 智能手机 S60 开发移动应用程序的全球竞赛。这个比赛是为了鼓励开源和免费软件开发者使用诺基亚的 Open C 环境。比赛的现金价格总计超过 2 万美元。

诺基亚全球开发者计划诺基亚论坛副总裁 Lee Epting 表示:“诺基亚邀请全球数百万开发者参加 Open C 挑战赛，这充分利用了开源和 Symbian 开发者的指数增长，他们目前正在为智能手机设备开发令人兴奋的新应用。“识别和奖励使用 Open C 构建或移植的最新和最棒的应用程序，将展示 Open C 已经在全球范围内推动 S60 智能手机应用程序开发的开发人员生产力的令人兴奋的增长。”

该比赛由诺基亚与 Orange 和 Symbian Developer Network 联合赞助，旨在让开发者应用他们的桌面和开源技能，决赛选手将被邀请在 10 月 16 日和 17 日在伦敦举行的 Symbian 智能手机展上向包括行业评委在内的一组小组成员展示他们的应用。

提交所有申请的截止时间是美国东部时间 2007 年 9 月 14 日晚上 11:59。有关注册流程的更多信息，包括参赛作品描述、资格要求、完整规则和奖品详情，可在[公开赛 C 挑战赛](www.forum.nokia.com/main/resources/technologies/open_c/contest.html)上在线获取。
