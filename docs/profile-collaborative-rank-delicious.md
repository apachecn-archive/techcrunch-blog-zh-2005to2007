# 个人资料-合作排名(Del.icio.us)

## 这是什么？

[合作排名](https://web.archive.org/web/20210731003311/http://collabrank.web.cse.unsw.edu.au/del.icio.us/)是澳大利亚悉尼新南威尔士大学创造的一项实验。这是一个搜索引擎前端的 [Del.icio.us](https://web.archive.org/web/20210731003311/http://del.icio.us/) 书签。

![](img/20d25fa925942d20c9cf1db68eb532b2.png)这个搜索引擎深受他们所谓的[“500 强”del.icio.us 标签者](https://web.archive.org/web/20210731003311/http://collabrank.web.cse.unsw.edu.au/del.icio.us/?cmd=helpful_users)的影响。这不仅仅是看谁标记了最多的页面，而是看谁快速地标记了东西，并且以其他人随后复制的方式(确认你以其他人认为合适的方式标记)。

用他们自己的话来说，*“del . icio . us 上给有用/及时的 URL 添加有意义的标签的用户(其他人随后也这么做了)将获得更高的 CollaborativeRank，这意味着他们的标签将对这个搜索引擎的排名产生更大的影响。”*我很自豪地说，我进入了 500 强名单，确切地说是第 499 名(哦，太好了，另一个需要竞争的对象)。:-)

这是一个搜索 del.icio.us 内容的非常有用的工具。
