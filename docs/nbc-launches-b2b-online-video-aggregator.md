# NBC 推出 b2b 在线视频聚合器

在苹果公司发布其预期的视频公告时，NBC 决定不袖手旁观，选择今天推出他们新的视频聚合服务，名为国家宽带公司，或 NBBC.com 的 T3。这是一个简短视频的市场，该公司的 230 家附属电视台和其他媒体合作伙伴可以去那里找到视频并发布在他们自己的网站上。广告收入将在制作视频的人、在自己网站上发布视频的人和 NBBC 之间平分。出版商可以采取地方电视台以外的多种形式；例如，RSS 供应商 [Newsgator](https://web.archive.org/web/20211205074213/http://newsgator.com/) 已经[宣布将参与](https://web.archive.org/web/20211205074213/http://www.newsgator.com/news/archive.aspx?post=106)并提供 NBBC 视频。其他合作伙伴包括 Break.com、About.com 和 CNet。

分销合作伙伴将能够选择一个完整的 NBBC 视频播放器、一个特定的主题频道来显示，或者为他们的网站选择单独的视频文件。广告将首先出现在伴随横幅广告的前滚动广告中，随着更长的视频被包括在内，中间滚动和其他类型的广告将会出现。玩家可以被广泛定制——一个简单的版本出现在右边。

NBBC 说，虽然每个人都在涉足视频，但并不是每个人都知道如何将视频打入当地市场。他们相信，附属电视台及其附属网站的网络将帮助他们做到这一点。我预计这意味着许多地方新闻网站的视频疲软状态将会改变，但这类网站将面临越来越大的商业压力，要求它们提供广告驱动的联合视频内容，而不是本地制作的或公民记者类型的视频。

从 iTunes 到 [CNN 接受用户生成的视频内容](https://web.archive.org/web/20211205074213/http://www.beta.techcrunch.com/tag/cnn)到[视频分享网站的用户走向专业](https://web.archive.org/web/20211205074213/http://www.beta.techcrunch.com/2006/09/10/top-social-media-users-getting-paid-is-the-balance-shifting/)——感觉无处不在的宽带和负担得起的摄像机对视频的影响才刚刚开始。
