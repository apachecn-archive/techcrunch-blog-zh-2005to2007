# LG 用 VX-8600 去翻转巧克力

威瑞森即将推出的巧克力翻盖手机被称为 XV-8600，它采用了与最初的巧克力滑盖版本(VX-8500)相同的巧克力设计。

与 8500 相比，翻盖手机 XV-8600 的分辨率较低，但其他一切都完好无损。刷新，立体声蓝牙，EV-DO，100 万像素摄像头，microSD 插槽，音乐播放器和外部触摸感应控制。目前还没有宣布这款产品何时上市，但应该会在下个月左右。
