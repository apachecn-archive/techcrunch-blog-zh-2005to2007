# Windows Live Writer | TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/08/13/windows-live-writer/>

微软[今天推出了](https://web.archive.org/web/20230222221132/http://windowslivewriter.spaces.live.com/blog/cns!D85741BB5E0BE8AA!174.entry) Windows Live Writer。这是一个免费的、[可下载的](https://web.archive.org/web/20230222221132/http://download.microsoft.com/download/f/9/a/f9a19f2d-cec4-4a25-9b0b-eb9655ea7561/Writer.msi)应用程序，运行在你的 Windows PC 上，允许你写博客文章。如果你有 Mac，就不要点击这个链接了，这只是 Windows PC 软件。

由于大多数博客工具都是基于 web 的，并不是所有的工具都允许所见即所得的编辑(您可以在屏幕上看到博客帖子的确切外观)，因此 Windows Live Writer 对于一些寻找更快、更容易的方法来发布博客的作者来说将非常有用。

像这样的工具已经存在很长时间了。例如 [Qumana](https://web.archive.org/web/20230222221132/http://www.qumana.com/) ，我[在一年前写了关于](https://web.archive.org/web/20230222221132/https://techcrunch.com/2005/06/15/profile-qumana/)的文章。RocketPost 是这一领域的另一家公司，尽管他们的软件收费 12 美元。[performance](https://web.archive.org/web/20230222221132/http://performancing.com/firefox)(一个人们赞不绝口的火狐插件)和 [Ecto](https://web.archive.org/web/20230222221132/http://ecto.kung-foo.tv/) ( [Om Malik](https://web.archive.org/web/20230222221132/http://software.gigaom.com/2006/08/13/windows-live-writer-write-on/) 用的就是这个)是另外两个。对这些现有服务的最大抱怨是，编辑现有的帖子是困难的或不可能的。我已经一年多没看它们了，所以我不知道它们是否已经进化到可以解决这些问题了。

Windows Live Writer 有许多功能——以所见即所得或 HTML 模式预览，支持大多数博客平台(Windows Live Spaces、Blogger、LiveJournal、TypePad、WordPress 等)，轻松插入照片和 Live.com 地图等内容，以及添加标签。好东西，但没有一个区别于现有的旧产品。

Live Writer 的与众不同之处在于，他们发布了一个 API，允许开发人员扩展该软件的功能，以发布其他内容类型。

糟糕的是，像谷歌和 Picasa 一样，微软忽略了 Mac 平台。
