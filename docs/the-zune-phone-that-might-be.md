# Zune 手机可能是

![](img/9a99383178d33d08d3b077329b074e2e.png)上个月[mobile runch](https://web.archive.org/web/20210304072205/http://mobilecrunch.com/2007/01/11/weighing-in-on-apples-iphone-your-mc-analysis-one-day-later/)和 [TechCrunch](https://web.archive.org/web/20210304072205/http://www.beta.techcrunch.com/2007/01/09/apple-announces-iphone-stock-soars/) 报道了乔布斯非常公开地发布 iPhone。史蒂夫·巴尔默嘲笑这款新手机定价过高，但微软可能会推出自己的手机版本。

《嘎吱嘎吱》一直紧紧跟随[线索](https://web.archive.org/web/20210304072205/http://crunchgear.com/2007/02/01/source-microsoft-zune-phone-in-the-works/)，它已经变成了一个相当可靠的故事。Crunchgear 报道称，周一，微软向联邦通信委员会提交了一份申请，申请一种可用于互联网通话的无线设备。他们接着说，该设备被描述为用于“消费者宽带接入和联网”。该设备将使用 OFDM 作为其通信协议，而不是 WiFi 或蓝牙。

标准 OFDM(正交频分复用)是一种调制方案，在未来即将到来的 [4G](https://web.archive.org/web/20210304072205/http://en.wikipedia.org/wiki/4G) 标准中广泛使用。简而言之，OFDM 将使更多的同时数据传输跨越移动塔。4G 本身并不是一项特定的技术，而是一系列技术和协议的集合，这些技术和协议将提供更大的移动带宽(100 Mbps(在手机网络中)到 1 Gbps(在本地 Wi-Fi 网络中)。CrunchGear 随后将这些点联系起来，指出 Sprint/Nextel 宣布了其基于 IEEE 802.16e 移动 WiMax 标准建设 4G 网络的计划，该标准与微软在 FCC 文件中的调制协议相同。Zune 手机被怀疑运行在 Sprint/Nextel 网络上，最早可能在 5 月份推出，比 iPhone 的发布日期提前。
