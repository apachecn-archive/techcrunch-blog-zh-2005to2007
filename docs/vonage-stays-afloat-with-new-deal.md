# Vonage 通过新协议保持运转

![](img/dff5b892890cd8cc409382923b527269.png)

在 Vonage folding 的传言和威瑞森臭名昭著的专利侵权案之后，事情看起来对这家受欢迎的 VoIP 供应商并不乐观。但现在，由于与 VoIP Inc .达成了一项为期两年的新协议，Vonage 现在可以继续生存下去，并赢回一些客户。Vonage 和 VoIP Inc .之间的交易将允许 Vonage 基本上外包客户拨打的电话。通过这种方式，Vonage 可以避免处理威瑞森的专利，并可以自由使用 VoIP Inc .的 VOICEONE 网络。

尽管经历了这一切，我不知道我是否会冲出去和 Vonage 签约。那些关注 VoIP 的人可能会喜欢他们在 Vonage 上看到的东西，但在签署任何东西之前，一定要货比三家。
