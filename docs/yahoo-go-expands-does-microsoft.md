# 雅虎！Go 扩张了，微软呢

![](img/20152166089c5735e8d6436adffde570.png)

今天早上对雅虎来说是个大新闻！拥有基于 Windows Mobile 的手机的用户。雅虎！宣布其广受欢迎的 Go 软件，以前只能在 Symbian 设备上使用，今天将在 Windows Mobile 手机上推出。雅虎！Go 现在可以运行 Windows Mobile OS 版本 2003 SE 和 5.0。

该服务允许用户从他们的电话服务，如 Y！邮件，搜索，照片，通讯录和日历，还有 Y！新闻、体育、财经等。雅虎！已经与包括摩托罗拉、诺基亚和 RIM 在内的一系列制造商合作，以确保即将推出的设备将预装该软件。摩托罗拉已经表示，它将预装并突出显示雅虎从 2007 年上半年开始，在全球范围内优化手机的移动性。

我不使用雅虎！邮件什么的，但是我欢迎 Go for Mobile 的到来。我敢肯定，谷歌在发布他们对此的答案方面不会落后太多。
