# 网络上的 windows Mobile“Crossbow”截图| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/09/windows-mobile-crossbow-screenshot-on-the-web/>

许多 PDA 和智能手机爱好者更喜欢 Windows Mobile，而不是 Foldger 的 Crystals，他们都对下一代操作系统感到兴奋，目前代号为“Crossbow”当它上市时，它可能会被称为 Windows Mobile 6，根据 MS Mobiles 泄露的图片，它看起来非常光滑。

Web 2.0 风格的外观看起来非常 Vista-rific，关于热浴缸的消息是，它将具有直接从 Vista 提升的文件夹浏览器。微软之前曾表示，它很可能会每年左右发布一个新版本的 WinMo，这将取决于运营商和设备制造商来保持手持设备的更新。

期待 Windows Mobile 6 在明年年初推出几个版本，因为微软现在正准备向设备制造商提供 Crossbow。
