# 雅虎！将地图和 Flickr 应用于旅行| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/10/yahoo-to-put-maps-and-and-flickr-to-work-on-travel/>

 [](https://web.archive.org/web/20161111014405/http://travel.yahoo.com/trip) 雅虎！今天，它的[旅行计划器](https://web.archive.org/web/20161111014405/http://travel.yahoo.com/trip)结束了 9 个月的测试期，这是一个有趣的旨在普及的酷 Web 2.0 应用的例子。用户可以从 Flickr 集合的弹出窗口中拖放 Flickr 照片，以说明旅行目的地。除了展示搜索结果的地理位置，地图还有商业用途。通过雅虎写博客！360 将目的地叙事提升到了一个新的高度，超越了短评。这很可能是主流采用 Web 2.0 的一部分。

旅游是一个拥挤的垂直行业，但 Flickr 是雅虎的强大资产。带进游戏中。同样，雅虎！地图不一定是谷歌地图丑陋的同父异母兄弟。这是一个不错的系统，很可能会受到在线旅游网站用户的欢迎。

我想象更多这样的事情可能会出现在持有雅虎的新实践中。黑客日事件。显然，该公司有很多工作要做。多个应用程序的有趣集成似乎是最容易实现的。
