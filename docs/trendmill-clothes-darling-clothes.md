# 衣服，亲爱的，衣服！

[![respectance.png](img/6d26f6d13dbf404e38cc9170f5c3e99b.png)](https://web.archive.org/web/20220121114659/http://www.trendmill.com/) 与本周小众社交网站不断发展的主题保持一致的是，一个服装爱好者的社交网站。

Trendmill 允许用户上传他们自己的衣服或他们希望拥有的衣服的图片。会员们可以“告诉人们”他们认为什么是热门的，对其他人的衣服进行投票，并合作“给出下一个时尚大趋势的总体感觉”。
Trendmill 还提供了社交网站的常见功能，包括消息传递、发布文章(博客)和与朋友联系(“随从”)。

TrendMill 与 StyleDiary、ShareYourLook 和 ShoutFit 在同一个领域竞争。与一些竞争对手不同，TrendMill 严格来说是一个社交网站，它通过时尚的界面和用户网站的高级定制来展示。人靠衣装，但很多人不会看这篇文章，所以这个网站并不适合所有人。然而，如果你认真对待你的范思哲和 YSL，你可能会喜欢 Trendmill。
