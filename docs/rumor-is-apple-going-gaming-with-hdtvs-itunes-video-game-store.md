# 苹果会用高清电视和 iTunes 视频游戏商店来玩游戏吗？

没有什么比美味的苹果酱和啤酒更适合我们的午餐了，今天我们两样都有。

这一条来自苹果内部人士，其消息来源称，库比蒂诺正在涉足已经拥挤的游戏市场。这不仅仅是猜测，保诚的分析师杰西·托尔托拉有证据表明苹果一直在雇佣游戏设计师。

当然，他们可能是为了已经建立的 iPod 游戏服务，想出更多像“宝石迷阵”这样的东西。或者，反过来，这些工作人员可能会将 iPod 游戏从现在的手机类扩展到其他领域。

或者他们可以在那里开始全新的东西，就像 2001 年的 iPod 一样。真正有趣的是，我们知道苹果有意拥有你的客厅，不久前有[关于即将推出内置无线功能的高清电视](https://web.archive.org/web/20201020115641/http://crunchgear.com/2006/10/24/apple-rumors-round-5253-iphone-evidence-and-giant-ass-tvs/)与 iTunes 电影商店合作的传言。

但是如果这些电视里面也有游戏机呢？

想一想；你可以从 iTunes 游戏商店下载一个游戏到你的 Mac(或者 PeeCee，Hodgman-lovers)，通过 FrontRow 界面浏览到它，发送到 HDTV，然后像玩普通游戏机一样玩。苹果拥有多项图形专利，这使得它完全可行。

这种组合设置实际上使游戏比硬件更重要:游戏机制造商通常会在硬件上有所损失，并通过游戏授权来弥补。苹果似乎不是走这条路的类型，所以结合游戏和显示硬件可以节省成本，并改变市场动态。

这就是苹果的想法吗？我们知道高清电视和视频游戏即将到来，再加上独立电视，这似乎一点也不牵强。
