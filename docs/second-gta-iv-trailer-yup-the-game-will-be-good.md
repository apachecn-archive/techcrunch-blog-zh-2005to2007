# 第二 GTA IV 预告片:是的，游戏会很棒

上周，《侠盗猎车手 4》第二部预告片的发布几乎淹没在所有的骚动中。这部预告片最令人印象深刻的是，它比我见过的任何 CE 公司发布的任何东西都更“时髦”和“时尚”，除了苹果。我们得到了比第一部预告片多得多的游戏片段和背景音乐。干得好，摇滚明星。希望这一个不会被[封杀](https://web.archive.org/web/20210302054231/http://crunchgear.com/2007/06/22/manhunt-2-temporarily-suspended-in-response-to-ban-ao-rating/)。
