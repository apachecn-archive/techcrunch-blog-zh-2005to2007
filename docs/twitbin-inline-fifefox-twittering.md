# Twittering:内嵌 Firefox Twittering

Twitter 用户目前有两个主要的界面，通过这两个界面，他们可以与日益流行的服务进行互动:通过 Twitter 网站本身或通过手机发送短信。

许多公司已经推出了第三方工具来增强基于计算机的 Twitter 交互。[twit terrorial](https://web.archive.org/web/20220119152013/http://www.apple.com/downloads/macosx/email_chat/twitterrific.html)在 Mac 用户中拥有强大的粉丝基础， [Twitteroo](https://web.archive.org/web/20220119152013/http://rareedge.com/twitteroo/) 一个 PC 版的 twit terrorial，最近 [Tweetbar](https://web.archive.org/web/20220119152013/http://mikedemers.net/projects/tweetbar) 将 Twitter 主页购买到 Firefox 的侧边栏中。

Infinimedia 的一个新的 Firefox 插件， [TwitBin](https://web.archive.org/web/20220119152013/http://www.twitbin.com/) ，用一个定制的 Twitter 内嵌应用将这些以前的平台带到了一个新的水平。

有时候这是最简单的事情，TwitBin 直接切入正题:你在 Firefox 窗口的左侧获得你的 Twitter 消息，按颜色分类。提供了一个文本框，用于添加您自己的推特:没有帖子。这个按钮是提供的，Infinimedia 认为这一点，并知道大多数人都理解点击 ENTER 的概念。

广告纯粹主义者不会喜欢插件的某个部分:一个 180×50 像素的横幅广告出现在插件的底部。它没有闪烁，也没有给我提供免费的 iPods 这是微妙的，所以我没有任何问题，但有些可能。

在没有任何其他竞争对手的情况下，这个插件应该会在 Twitter 粉丝大军中很受欢迎，而且从现在开始我会一直使用它。

开发人员与我们分享了一个值得强调的额外花絮:开发时间不到 2 周，成本不到 1000 美元。

聪明的思维和优秀的代码不一定是巨富或风投资助的专属领域。
