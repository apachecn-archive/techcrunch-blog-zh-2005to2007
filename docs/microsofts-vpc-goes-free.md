# 微软的 VPC 免费

如果你想在自己的电脑上运行一台虚拟电脑，微软正在做一件与众不同的事情:免费为你牵线搭桥。截至今天，虚拟个人电脑是免费的，但只针对 Windows 用户。这与微软的虚拟化方案相一致，该方案将 Virtual PC 等软件定位为测试环境和在一台机器内运行虚拟安全服务器的方式，而不是作为两台个人计算机。

这不是大多数消费者使用虚拟 PC 的目的，但是，嘿，它是免费的。我们热爱自由。
