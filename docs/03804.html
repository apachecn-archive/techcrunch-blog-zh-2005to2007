HTC TyTN(XDA Trion)3G Windows Mobile PDA手机现已上市

HTC智能手机的两种型号HTC TyTN或重新注册的XDA Trion——现在都可以在Dynamism的网站上购买。两款手机的售价都是899美元，这意味着它不是一次性手机。XDA Trion有一个QWERTZ键盘，在欧洲很常见，而HTC TyTN有一个QWERTY键盘，更适合美国。

更新一下，这些手机是四频GSM，有64MB内存，128MB ROM，一个MicroSD插槽，蓝牙，GPRS，HSDPA，EDGE，802.11b/g，一个200万像素的摄像头，一个用于视频通话的VGA摄像头，以及5小时的通话时间。由于兼容GSM，您应该可以在Cingular和T-Mobile上使用进口手机。
