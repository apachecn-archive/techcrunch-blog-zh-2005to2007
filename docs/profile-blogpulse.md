**什么事？**

BlogPulse 是一个典型的实时搜索引擎，具有趋势、对话跟踪和博客简介等附加功能。它的母公司是 Intelliseek。

用他们自己的话说，*“blog pulse 是进入博客圈的一扇窗户——每天打开它，发现博主们正在讨论的人、问题、博客、帖子、评论、花絮和新闻。”* [链接](https://web.archive.org/web/20210126100721/http://www.blogpulse.com/)

![](img/2eef5828715ff4ee48c61cfc393f2ebe.png)

BlogPulse 网站提供四种主要的服务，我们在这里介绍:搜索、对话跟踪、趋势和个人资料。

**搜索:**

BlogPulse 搜索允许通过关键字或 URI 进行搜索(但不允许标签/类别，这是绝对必须添加的)。高级搜索功能是[这里的](https://web.archive.org/web/20210126100721/http://www.blogpulse.com/search.html#advanced)。与 Technorati、Feedster 和其他公司相比，样本搜索产生了出色的结果，尽管这是一个快速发展的领域。总的来说，索引看起来非常及时和完整，搜索速度非常快。

![](img/6d83aabfca1f5422460be76ab4576265.png)

**对话跟踪器:**

对话跟踪器服务真的很酷。随着博客圈的发展，“对话”以分散和分布的方式发生在数百万个博客和其他网站上(而在 web 1.0 中，对话通常集中在一个站点上，如新闻组)。

*“blog pulse 趋势搜索”可让您创建图表，直观地跟踪特定关键词、短语或链接的“热门话题”。单独比较搜索词/链接，或使用所有三个字段将搜索词/链接与其他搜索词/链接进行比较。*

在左边的框中键入您的搜索词。在右边的框中键入每个搜索的描述性标签。然后选择你的时间框架:1 个月、2 个月、3 个月或 6 个月。

Blogpulse 已经收集了博客帖子的全文，并分析了引用，以创建一个关于关键词或 URI 的可视对话跟踪器。这很粗糙，但是对于跟踪讨论非常有用。请看视频[这里](https://web.archive.org/web/20210126100721/http://intelliseek.com/media/blogpulse/aBP-GenConversationTracker.mov)和[这里](https://web.archive.org/web/20210126100721/http://intelliseek.com/media/blogpulse/aBP-TechConversationTracker.mov)进一步解释。

会话跟踪器的一个关键价值是跟踪与单个帖子相关的病毒传播。

![](img/ce4db4d80ad6d5336d97b24ddc27bb88.png)

**趋势:**

[趋势](https://web.archive.org/web/20210126100721/http://www.blogpulse.com/trend)服务给出了与关键词或 URIs 相关的帖子的可视化图表。您还可以在单个图表上比较多个项目。他们也有[特色趋势](https://web.archive.org/web/20210126100721/http://www.blogpulse.com/trends.html)的高调新闻和其他讨论。

**简介:**

[档案服务](https://web.archive.org/web/20210126100721/http://www.blogpulse.com/profile)有关于特定博客的相当深入的信息，包括概述、帖子、引用、趋势、来源、邻居。看看这个。

Blogpulse 还有很多其他很酷很有趣的链接，统计数据等等。我们没有在这里做侧写。
