# 洗衣机，不需要洗涤剂

如果你像我一样，你大概每个季度洗一次衣服，甚至还没打开洗衣机的盖子，你就变得语无伦次，暴跳如雷，因为有人(可能是你，三个月前)用光了所有的洗涤剂。

好吧，只要不到 1000 美元，[海尔](https://web.archive.org/web/20201128023820/https://crunchbase.com/organization/haier) 就会卖给你一台洗衣机，它不使用洗涤剂，却能以某种方式把衣服洗干净。

WasH20 将水分解成 OH-和 H+离子成分，一旦分解，OH-隔离污渍，而 H+擦洗衣物。

头脑更迟钝的人或许能更好地解释这一过程，但我倾向于相信，这台机器的工作原理是基于海尔是一家相当成熟的公司这一事实。

最初在法国上市，美国的计划还没有消息。
