# 规模调整使规模调整变得简单

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/21/sizeasy-makes-sizing-easy/>

有一个问题经常困扰着我。我们在这里整理所有的产品信息，尺寸可能是最随意的。说什么 90 毫米长的东西对我来说真的没有任何意义。

输入 [Sizeasy](https://web.archive.org/web/20150629164739/http://www.sizeasy.com/) 。该网站允许用户输入各种尺寸，然后根据其他物体(如扑克牌、iPods 等)的比例调整尺寸。因此，如果你对空间推理有问题，你可能想给 Sizeasy 一个 looksie。
