# 向世界发送视频笔记…

![](img/6e7ced7e82dcd62702be78e991eb133e.png)

……但是世界想要看到它们吗？MobileCrunch 的 Oliver 刚刚获得了 Veeker 的独家消息，这是一种视频信息服务，允许你上传 MMS 视频，然后将其发送到你的个人页面、你的联系人或全世界。像之前许多引以为豪的视频/图片分享网站一样，Veeker 承诺成为人们的垃圾的粒状图片的一站式商店。

该系统是这样工作的:你把你的视频通过电子邮件/彩信发送到你在 Veeker 上的账户。它进入队列，被处理，然后出现在你让它出现的任何地方。这样，您的朋友和邻居就可以将您的视频嵌入到网页中，或者通过 RSS 源观看。现在看起来还没有，但是我们会让你知道奥利弗发来了他公文包内部的第一个视频。
