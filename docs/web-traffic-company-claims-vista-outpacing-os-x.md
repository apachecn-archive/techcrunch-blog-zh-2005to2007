# 网络流量公司声称 Vista 超过了 OS X

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/25/web-traffic-company-claims-vista-outpacing-os-x/>

汤姆·克拉兹特早上喝完咖啡，抬头看着网络流量公司 Net Applications 提供的最新数据，喊道:“错误！错误的。两个都是错误的。”

显然，Net Applications 声称，基于其“Hitlink”软件跟踪的操作系统种类，Windows Vista 正在侵蚀 Mac OS X 的销售。

根据 Krazit 的说法，这些数据的问题在于，Net Applications 只监控 40 万个网站，其中大多数是中小型企业网站，而且绝大多数出售的电脑都预装了 Vista。Net Applications 声称，Vista 的使用在 2 月份激增，大约是在 Vista 首次开始在大多数出售的新电脑上销售的时候。

因此，Vista 从无到有的事实会让这些数字看起来更加令人印象深刻。Krazit 自己做了一点研究，发现与 Net Applications 报道的情况相反。

> “让我们看看自去年夏天以来的总体趋势。2006 年 7 月，访问 Hitslink 网站的 Windows PC(Win 2000、XP 和 Vista)占总流量的 90.39%。当然，那时没有远景。今年 6 月，同样的组合占了 90.46%的流量，这意味着如果 Vista 正在蚕食任何东西，那就是 Windows XP 和 Windows 2000 的使用。
> 
> 同样的对比也适用于麦克 OS X？去年，基于英特尔的 MAC 电脑和老式 MAC 电脑的组合占 Hitslink 网站流量的 4.29%。今年 6 月，它占流量的 6%，随着 PowerPC Macs 的下降，基于英特尔的 MAC 在增长。
> 
> 因此，与去年相比，Mac OS 的使用量增长速度快于 Windows。"

总之，有趣的是，事实和数字很容易被歪曲。
