# 说出 Chiptune 的名字:8 位音乐的成长利基

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/08/name-that-chiptune-2-the-growing-niche-of-8-bit-music/>

![](img/277321298b67b27c098f54998b5df9dc.png)
随着组件变得越来越便宜和先进，音频一直在稳步向更高的保真度和分辨率发展。如果不是 24 位和 192kHz，许多人根本不感兴趣。但本周，我将目光投向了相反的方向:8 位音乐。由于硬件改造和软件排序器，整个音乐人社区都使用改进的手持游戏设备，如任天堂游戏机和 NES 盒作为现场乐器。起初，我无法决定这是青少年沉迷电子游戏与摇滚明星梦想的邪恶结合，还是一种合法的艺术形式，但我在上周末的[Blip Festival](https://web.archive.org/web/20160305105220/http://www.blipfestival.org/)——一系列庆祝 8 位音乐和视觉艺术的表演、讨论小组和放映——的经历让我明白了一些事情。

周五晚上，在位于曼哈顿华尔街附近一家已倒闭银行的大型艺术空间 the Tank，现场就像一场狂欢派对，有一大群人，许多彩色灯光，一个视频屏幕，音乐通过巨大的扬声器播放。但舞台上的这个人并没有操纵一套唱盘(或 ipod)——他只是在 QWERTY 键盘上按下按钮，连接着几个游戏机。在他身后是一个由手头的视觉艺术家控制的 50 x 20 4 英寸面板灯的灯光景观，在他面前是一群崇拜他的人。他的名字是[比特移位器](https://web.archive.org/web/20160305105220/http://bit.shifter.net/)(真名:约书亚·戴维斯，33 岁)，他是 Blip Festival 的联合组织者，32 名低级音频和视觉艺术家聚集在曼哈顿下城，参加为期四天的 chiptune 盛会。我拍了几张会场的快照——本着低级艺术的精神，我试图用我那蹩脚的手机相机拍一些有趣的照片。

![](img/cdaf3ae48502e759eff964903bb2c6e4.png)(移位器摇滚出锐舞风格。)

这个节日的另一个联合组织者是 [Nullsleep](https://web.archive.org/web/20160305105220/http://nullsleep.com/) (又名 Jeremiah Johnson，26 岁)，我最近有机会采访他。他通常在连接到 QWERTY 键盘的几个 Game Boy 和 Game Boy Colors 上演奏，让他像传统键盘手一样摇滚。我问他，面对音频技术的不断进步，是什么吸引他对 8 位音乐感兴趣:“对我来说，这与 it 的挑战有很大关系。挑战旧技术的极限，并试图让它做一些它从未想过要做的事情，我发现这真的很有意义。”

我还与 Blip 表演者[马克·德·纳多](https://web.archive.org/web/20160305105220/http://www.markdenardo.com/)进行了交谈，他将他的 PSP 音乐——运行一个 Game Boy 音序器的模拟——与他自己的木吉他演奏和人声相结合。这种结合是惊人的，虽然我最初觉得这就像是在配合录音音乐，但德·纳多确实自己创作了背景音乐。舞台上的一些技术问题促使我问他如何应对这样的挑战。他回答说:“我喜欢在舞台上和我的机器人打架。我们战斗的时候会做出更好的音乐。”伟大的男高音萨克斯管大师桑尼·罗林斯的影子，他总是声称当他真的生气时，他的演奏更好。

我听说过像吉诺·罗拜尔这样的先锋派艺术家用他放在手提箱里的一堆玩具和垃圾创作出动人的音乐。我也见过像 Mixmaster Mike 这样极具天赋的 DJ 用一套唱盘哄出一些令人难以置信的表演。这两者似乎都遵循了 Nullsleep 概述的相同原则，而 chiptunes 显然是这一原则的逻辑延伸。不同的是，8 位艺术家更关注实际创作乐器的 DIY 方面。Nullsleep 指出，“我们不是在重新混合音轨，我们是在重新利用视频游戏硬件来制造前所未闻的声音——结果听起来往往一点也不像游戏音乐。”

当你在表演期间环顾场地时，对技术的关注可能是最明显的——尽管有沉重的电子音乐，但很少有人真正在跳舞。他们被音乐、艺术家控制的闪光灯和显示各种抽象和不那么抽象的场景的视频屏幕的组合惊呆了。听起来像是在看某人玩电子游戏，嗯？

就我个人而言，我一直讨厌我最初的 Game Boy 上的编程音频(它早就被归入我的死亡电子产品抽屉了)，尽管我依稀记得在拯救黛西公主后从超级马里奥之地的结尾挖掘了 Hirokazu“Hip”Tanaka 的音乐。所以我确信我会对音乐节上的声音有些反感，尤其是作为一个铁杆声学音乐迷。然而，8 位音乐固有的局限性(以及艺术家们有时令人惊讶的复杂变通方法)正是这种形式如此迷人的原因。Nullsleep 是对的——结果通常一点也不像游戏音乐。
