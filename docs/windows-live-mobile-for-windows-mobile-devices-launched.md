# 面向 Windows 移动设备的 Windows Live Mobile 发布

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/14/windows-live-mobile-for-windows-mobile-devices-launched/>

微软刚刚推出了他们 Live 网站的移动版本，允许 Pocket PC 和 WM5 用户直接从他们的 pocketIE 或 Opera 移动浏览器访问邮件、空间、本地、Live beta、MSN beta 和天气。

该网站针对移动设备进行了重新设计——缩小了宽度，减少了图片数量——因此更易于使用。事实证明，微软 Live 计划非常有趣。这表明，如果微软真的想做点什么，他们可以在不到一年的时间里推出一个功能齐全的互联网应用套件。[移动直播](https://web.archive.org/web/20150927050140/http://mobile.live.com/)【直播[掌上电脑的想法](https://web.archive.org/web/20150927050140/http://www.pocketpcthoughts.com/index.php?action=expand,51268&/windows_live_for_mobile_launched.htm)
