# 视频广告:每个创业公司都有不同的解决方案

这可能看起来很奇怪，但我一直急切地等待着在我的病毒视频中看到广告的那一天。eMarketer 预计在线视频广告将在 2008 年翻一番，达到 13 亿美元，但是没有人真正为视频确定一个可扩展的广告平台。然而，谷歌一直在[悄悄测试](https://web.archive.org/web/20220327051350/http://www.beta.techcrunch.com/2007/05/11/youtube-video-advertising-no-pre-roll-no-context/)他们自己的系统，也有一堆其他创业公司在解决这个问题。

当他们试图创造最大的广告收入时，有几个关键问题他们都在努力解决。在哪里投放广告还有些不确定性(前期/后期/插播广告？).甚至广告的类型或长度也有待商榷。[最近的一项研究](https://web.archive.org/web/20220327051350/http://www.clickz.com/showPage.html?page=3626076)发现更长的广告在品牌推广中更有效，而传统智慧对坐着看更长广告的用户产生了怀疑。

在决定格式之后，确定视频的内容以产生相关的广告是另一个棘手的问题。对于不想冒险与煽动性内容联系在一起的大品牌来说，这也是一件可怕的事情。最后，这些广告平台需要出版商、广告商和交易市场。

他们就像视频的 adsense，根据视频的内容捆绑上下文文本广告。它看起来类似于 YouTube 的目标。当视频播放时， [Adap.tv](https://web.archive.org/web/20220327051350/http://www.crunchbase.com/company/adaptv) 挖掘相关的亚马逊产品和 Looksmart 广告，在关键时刻填充视频底部的广告栏。他们使用标签和其他元数据，以及语音到文本的翻译来找出视频的内容。

[![adbritelogomini.png](img/5579a952531679466b478fb8a0645806.png) ](https://web.archive.org/web/20220327051350/http://www.crunchbase.com/company/adbrite) AdBrite 是第一批用他们的 [InVideo 平台](https://web.archive.org/web/20220327051350/http://www.beta.techcrunch.com/2007/01/04/adbrite-makes-brilliant-video-product)在视频上叠加广告的公司之一。Adbrite 已经创建了一个类似于 YouTube 的嵌入式视频播放器。如果我们选择在 TechCrunch 上展示视频，我们可以使用这个嵌入式播放器，根据我们的选择，它将包括 Adbrite 广告和我们的徽标作为水印。任何人获取内容并将其嵌入自己的网站，都会显示相同的视频，带有相同的广告和水印。视频上的所有点击都会回到原来的网站。

[![broadramplogomini.png](img/48257302b1aa2cc6b1cdb403b3a3f1f0.png)](https://web.archive.org/web/20220327051350/http://broadramp.com/) 最有趣的广告，BroadRamp 希望[让你在视频上看到的一切成为可能的卖点](https://web.archive.org/web/20220327051350/http://cincinnati.bizjournals.com/sanantonio/stories/2007/06/11/story3.html?b=1181534400%5E1474128)。看到喜欢的 t 恤了吗？现在就点击视频购买吧。从视频中标记或以编程方式生成产品链接可能无法扩展或证明过于困难。然而，他们的核心业务仍然是视频内容传输系统。

[![everyzinglogo.png](img/1dbc4a019c844a520964ddc0f4671039.png)](https://web.archive.org/web/20220327051350/http://everyzing.com/) 原名 [Podzinger](https://web.archive.org/web/20220327051350/http://www.beta.techcrunch.com/2006/01/14/podzinger-launches-moves-podcast-search-forward/) ，Everzing 搜索音频和视频。由于他们不拥有内容，他们不能在视频内容上插入广告，但他们的语音到文本转录意味着他们可以帮助解决找到视频主题的问题。

[![blinkxlogomini.png](img/4af0055a8e93c65fdd7136785cb0cc39.png)](https://web.archive.org/web/20220327051350/http://www.crunchbase.com/company/blinx) 像 Everyzing 这样的视频搜索引擎，Blinkx 通过分析视频语音和元数据梳理出视频的内容。他们还声称使用视觉识别。然而，Blinkx 也利用他们的技术推出了基于视频内容的特别广告。

[![casttvlogomini.png](img/a868cfb753d660f01ff936428967463f.png)](https://web.archive.org/web/20220327051350/http://www.crunchbase.com/company/casttv) 另一个视频搜索引擎目前正在私服内测，[剧组。TV](https://web.archive.org/web/20220327051350/http://www.crunchbase.com/company/casttv) 查看视频的元数据和周围的链接，以确定视频周围的更多上下文。我们对它的工作效率印象深刻。然而，他们还没有讨论整合广告的计划。

为社交视频提供一个出色的、可扩展的广告平台解决方案，满足出版商、广告商和观众的需求，这只是问题的一部分。虽然找到最有效的格式需要大量的测试，直到消费者发现最有效的方法，但这些平台也需要视频内容来赚钱。由于具有目标内容的明确定义的视频属性可以与现有视频广告网络的赞助商合作，这些平台的理想市场仍然是有效地从社交网络和 YouTube 上漂浮的业余病毒视频中赚钱。然而，目前拥有网上视频最大份额的 YouTube 似乎正在慢慢开发自己的解决方案。

这使得成为一个目的地、合伙或收购成为可能的出路。Blinkx 和 Everyzing 等视频搜索网站目前正在将其搜索页面货币化，但无法通过在链接内容中嵌入广告来充分利用其平台。虽然这些网站提供更深入的视频搜索，但作为目的地网站存在也是一条艰难的道路，与谷歌、雅虎和美国在线等成熟的网络公司竞争。AdBrite 以一种略有不同的方式单干，用他们的 InVideo 播放器直接去找出版商。Adap.tv 一直在测试合作关系，在 MetaCafe 上测试他们的平台。

和大多数广告平台一样，广告商和出版商将会尝试它们的有效性。最终，能给这两者带来最大回报的创业公司将胜出。
