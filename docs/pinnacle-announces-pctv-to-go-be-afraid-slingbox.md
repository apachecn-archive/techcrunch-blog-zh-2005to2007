# 巅峰宣布 PCTV 走:害怕弹弓

![](img/87d0fca43aa99a19e07c3e0966879e73.png)

看起来顶峰公司已经感染了一种正在流行的旧弹匣病毒。他们刚刚宣布，在 11 月下旬 249 美元，你就可以购买 PCTV 去。这种新产品保持了其受欢迎的 PCTV 产品线，允许消费者通过无线网络在他们的 PC 上观看电视，与 Windows XP Media Center Edition 集成，并使用户能够将他们的 PC 用作 DVR。像 Slingbox 一样，只要你有一个像样的宽带连接，你可以在世界任何地方设置它来观看你的当地电视频道。这款盒子将在全国范围内的商店和网上销售。最好再次开始头脑风暴，因为 PCTV 看起来是你们的强大竞争对手。
