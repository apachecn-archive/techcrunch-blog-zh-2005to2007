# 东芝发布六款新笔记本电脑——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/16/toshiba-announces-six-new-laptops/>

![](img/adf0a9a97cbe170a79a72c2b11dc0982.png)

当我今天早上醒来，揉了揉眼睛，我看到了东芝发布的一份令人震惊的新闻稿，称已经发布了六款新笔记本电脑。由于涉及的内容太多，我将跳过诙谐的幽默，直接切入正题:规格。点击跳转查看完整列表。
 首先升空的，还有 P205 系列卫星。配备了 17 英寸的显示屏，Windows Vista，P105 这样的金属漆工作，以及采用“Labelfish”技术的 DVD 刻录机，这只是一种不同(和合法)的说法，即它是一种可以将文本打印到 CD 上的 Lightscribe 驱动器。预计低于 1000 美元的价格点。

其次，我们有 S637，配有 1.7 GHz 奔腾双核处理器、双层超多硬盘、Vista Home Premium、1GB 内存、120GB 硬盘和 802.11b/g Wi-Fi。相当不错的规格，虽然 CPU 可能只是高一点点。

S6267 采用 1.8 GHz Core Duo 处理器、双层 SuperMulti 驱动器、Vista Home Premium、2GB 内存、120GB 硬盘和 802.11a/b/g Wi-Fi。你都明白了吗？我是认真的，喘口气。要处理的权力太大了！

现在我们来看看 S6287，它只是一个 1.7 GHz 的 Core 2 Duo 处理器，一个双层 SuperMulti drive，2GB 的 RAM，一个 200GB 的大容量硬盘，Vista Home Premium，内置 Harmon Kardon 扬声器和 802.11a/g/n Wi-Fi。听起来很像其他一半的东芝笔记本电脑，但至少东芝是慷慨的内存和硬盘空间。加上那个 802.11n 也不疼。

快到终点时，你会发现 A205 系列。除了一款内置网络摄像头和标签鱼技术之外，其他都一样。该系列还在街区周围配备了 15.4 英寸的屏幕和 802.11n，以保持下载~~色情~~商业文件的无线速度。

在生产线的尽头，我们有自己的 S4577。它不是同类产品中最强的，它配备了 1.3 GHz 英特尔酷睿 2 双核处理器、1GB 内存、Vista 家庭高级版、160GB 硬盘和双层超级硬盘。对你来说不够强大？看看 S4617，它有 1.6 Ghz 的 Core 2 Duo CPU，2GB 的 RAM，Vista Home Premium，250GB 的硬盘，指纹识别器，Harmon-Kardon 扬声器(可能很糟糕)，以及双层 SuperUltraMultiXtra 驱动器。

最后一款是 S4639，它拥有 1.7 GHz Core 2 Duo 处理器、2GB 内存、280GB 大容量硬盘、双层 SuperMulti 驱动器、Nvidia GeForce Go 7300 显卡、Harmon Kardon 扬声器和指纹读取器。

布拉沃东芝。希望过去困扰你的无线问题和垃圾软件现在已经被清除了。
