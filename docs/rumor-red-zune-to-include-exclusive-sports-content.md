# 谣言:红色 Zune 将包括独家体育内容

。引起我注意的是我们听到的关于 Zune 的某种“运动包”的传言。当我周五晚上在 Zune Insider 联系塞萨尔时，他说他不能谈，他已经下班了。没问题，如果我不能通过官方渠道获得信息，我会看看我能从我的 MS 间谍网得到什么。我找到了很酷的东西。

我们听说西瓜红 Zune 即将推出，但我们的秘密工厂结合了这些传言，表明新的 Zune 将不仅仅是一个油漆工作，它还将包括体育相关的内容。他/她没有详细说明这些内容是什么，但表示除了红宝石色的设备，Zune 还将在其商店中推出与体育相关的新部分。猜想是能够将你最喜欢的球队加入到你的订阅中，这意味着你可以下载每场比赛作为播客在旅途中观看。我们不确定这背后有多少有效性，但这种侦察通常是在金钱上，体育内容总是赚钱的，所以我们将减少这一粒盐的大小。

美国每个州的每个酒吧都有电视直播每场比赛，我们不知道为什么有人会想和他们进行上周的比赛。但是，也有像我哥哥一样的人会重新观看几年前的比赛，重新体验那种刺激。我想这对他们来说是好事。猩红色的 Zune 应该会在今年夏天上市。
