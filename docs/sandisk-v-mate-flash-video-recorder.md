# 闪迪 V-Mate 闪光录像机

SanDisk V-Mate 的工作很简单。插入闪存卡(SD、MMC、MMCplus、MMCmobile、SDHC、MiniSDHC、MicroSDHC、Memory Stick PRO、Memory Stick Duo 和 Memory Stick PRO Duo)，将设备连接到复合视频源，它将使用 MPEG4 压缩将视频记录到卡上。SanDisk 表示，一个 1GB 的闪存卡可以支持长达 3.5 小时的任何视频。MPEG4 格式可以很容易地将卡插入手机或便携式媒体播放器，并开始观看您的视频。

10 月份上市时，预计价格仅为 123 美元。对于那些还没有完全掌握在个人电脑上为 iPod/手机/PSP 格式重新编码的人来说，这很有用。但是如果你真的想省下 123 美元，给自己买一份 Videora iPod Converter，你就可以自己为 MPEG4 格式化你的东西了。
