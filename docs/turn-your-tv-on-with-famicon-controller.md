# 用 Famicon 控制器打开电视

![](img/b4917c2d6903147a1cabd328352e2fc6.png)

如果你像大多数复古游戏玩家一样，每次看到任天堂的东西都会手心冒汗，你可能想看看这款可以用于电视的 Famicon 控制器。retro 控制器的 D-pad 可让您浏览频道，start 按钮可让您打开/关闭电视，select 按钮可让您更改输入源。

这款复古遥控器不仅得到了任天堂的官方授权，而且售价仅为 20 美元。该控制器只兼容有限数量的电视(网站上列出)，除了你的电视，你不能用它来做任何事情，使通用控制器不那么通用。

是只有我，还是遥控器让你想起了我们不久前发布的[复古闹钟](https://web.archive.org/web/20130627212240/http://crunchgear.com/2006/09/18/nes-controller-alarm-clock/)？
