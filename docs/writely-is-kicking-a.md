# Writely 正在踢一个* *

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/12/10/writely-is-kicking-a/>

 [ Writely ](https://web.archive.org/web/20220714211006/http://www.writely.com/) ( [之前简介](https://web.archive.org/web/20220714211006/http://www.beta.techcrunch.com/2005/08/31/writely-process-words-with-your-browser/))继续[引领在线 ajax word 产品的潮流](https://web.archive.org/web/20220714211006/http://www.beta.techcrunch.com/2005/12/08/rallypoint-offers-an-ajax-word-solution/)。

Writely 从第一天起就有很好的功能:导入和导出到 Word 格式，嵌入图像，所见即所得编辑器，拖放功能，与他人共享，以及标记文档。

他们还定期推出新功能，并在 Writely 博客上发布。我最近注意到的几个特点:

*   保存为 PDF 格式(见屏幕截图)——我测试过了，非常棒
*   本地化(外语)常见问题解答–支持的语言包括斯洛文尼亚语、中文、土耳其语和越南语
*   查找和替换
*   支持 RTF(富文本格式)

从 writely 直接创建 PDF 的能力是一个非常有用的工具，这是 Word 无法做到的(除非您单独购买或以其他方式获得了 PDF 软件)。Writely 不再是玩具，它正迅速成为 Word 的合法替代品。

下面是我希望看到的下一步——从 Excel 和 Power Point 开始，快速推出其他办公应用程序。

有传言称，Writely 现在正在寻求风险投资，并且很受欢迎。Writely 也是一个显而易见的收购目标——无论是产品还是成熟的团队。
