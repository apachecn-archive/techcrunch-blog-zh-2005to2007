# 诺基亚:现在有了 YouTube！

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/12/nokia-now-with-youtube/>

![](img/4cfc63b10b9f1399eddf3e7e763a535d.png)
那么这都是什么？诺基亚的新视频中心提供 RSS 视频源以及直接来自 YouTube 的内容，不需要浏览器。它最初在诺基亚的 fattie media 手机 N95 上提供，将用于 S60 手机，包括 N93i。是如何整合的？嗯，视频中心看起来有点像苹果的前排，通过其中一个旋转选择器提供 YouTube 视频。他们在会议中掩饰了这一点，所以我们姑且认为它存在，就此打住。
