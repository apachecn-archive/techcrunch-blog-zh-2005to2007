# Veoh 获得 2600 万美元的 C 轮融资

[![](img/edf3572c57337840e4dc86122d014ea3.png) ](https://web.archive.org/web/20211024230855/http://www.crunchbase.com/company/veoh) P2P 和网络视频分享服务 [Veoh](https://web.archive.org/web/20211024230855/http://www.crunchbase.com/company/veoh) 已获得 2600 万美元的 C 轮融资。这笔新资金使对 Veoh 的总投资达到 4125 万美元。据传闻，融资前的估值在 6000 万美元至 7000 万美元之间。

该公司的董事会中有迈克尔·埃斯纳，内容与像 Us Magazine 和 United Talent Agency 这样的公司打交道，正如 Marshall Kirkpatrick 在 2 月写的[，一些非常好的技术。](https://web.archive.org/web/20211024230855/http://www.beta.techcrunch.com/2007/02/12/veoh-relaunches-powerful-video-sharing-service/)

据 PEHub 报道，高盛引领了这轮投资，早期投资者 Spark Capital 和 Shelter Capital Partners 也增加了投资。

Veoh 与其他社交视频网站的不同之处在于，它专注于联合内容和视频质量。他们有一个桌面点对点视频播放器，可以以更高的质量播放更长的内容。对于出版商来说，Veoh 允许上传任何大小或质量的内容，并自动以适当的格式整合到 YouTube、MySpace 和 Google Video。

像 Metacafe 和 Revver 一样，出版商也可以从他们的内容中赚钱。然而，与这些竞争对手不同的是，Veoh 允许出版商直接向观众收取内容费用，并与该公司对半分成。一月份，Veoh 的创始人 Dmitry Shapiro 告诉 Beet.tv 他们拥有 400 万独立用户。

这是一大笔钱，尽管根据 Alexa 的数据，从长远来看，Veoh 已经悄悄进入了前 200 家在线网站。也许永远不会有第二个 YouTube，但 Veoh 是一个很好的尝试。
