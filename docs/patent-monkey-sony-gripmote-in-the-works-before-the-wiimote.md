# 专利猴:Wiimote 之前的索尼 GripMote？

在详细描述控制器的手持计算机交互设备上。

索尼的控制器被设计成适合手掌，并与用户的手指配合工作。当用户挤压设备时，施加在控制器传感器上的压力被检测到，产生用于系统或游戏交互的命令。控制器设计用于检测用户的手部动作和挤压动作。根据专利的描述，控制器可以使用蓝牙技术以及振动或触觉响应。索尼还设计了双无线独立手柄。

从表面上看，这项专利似乎没有给任天堂带来像 [Interlink 专利](https://web.archive.org/web/20190221033410/http://www.patentmonkey.com/PM/PatentID/6850221.aspx)和相关诉讼那样多的[问题](https://web.archive.org/web/20190221033410/http://cache.kotaku.com/assets/resources/2006/12/InterlinkvNintendo.pdf)。

随着“离开你的沙发”Wiimote 的流行，也许索尼可以迅速将 GripMote 从 PS3 的货架上撤下来。
