# 对 Firefox 2 进行基本调整

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/27/whip-firefox-2-into-shape-with-basic-tweaks/>

[火狐](https://web.archive.org/web/20160305084230/http://crunchgear.com/?s=firefox) 2 发布到现在还不到一周，网上到处都是评论，而且大多是正面的。但是有几个变化是我们个人不喜欢的。以关闭标签按钮为例。在以前的 Firefox 中，它位于标签行的最右边。现在每个标签都有自己的关闭按钮，很像 Camino。我们不喜欢这样。

我们尝试了一些扩展来解决这个问题，但是它们还没有更新，使得 Firefox 像康复诊所的加里·布西一样不稳定。

幸运的是，Lifehacker 的女士们整理了这个方便的小指南，来调整你的 Firefox，让它做你想做的事情。我们用它把“关闭标签”框放在我们想要的地方。你可以用它做更多的事情。不要让“高级配置”这个名字吓到你，这些简单的改变会让你的浏览器更快更容易使用。
