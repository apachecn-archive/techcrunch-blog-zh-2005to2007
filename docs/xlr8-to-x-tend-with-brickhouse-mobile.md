# XLR8 到 X-Tend 与 Brickhouse Mobile | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/03/xlr8-to-x-tend-with-brickhouse-mobile/>

内容创作者和发行商 XLR8 Mobile 与 Brickhouse Mobile 签署了一项多年协议，以扩展 XLR8 的品牌。这笔交易将有助于把 XLR8 的原创和收购内容带给世界各地的手机用户，包括比基尼鸡尾酒视频等产品，这些视频以“顶级名人模特”为特色，提供调酒技巧，这些公司称之为“性感的飞溅”。其他产品将更加生动，包括魅力，袒胸，裸体和软核。这些内容将通过国内和国际市场的运营商手机提供。

作为交易的一部分，Brickhouse Mobile 将扮演看门人的角色，以虚拟方式检查 id，并实施各种方法来确保内容只能由年龄合适的观众观看。
