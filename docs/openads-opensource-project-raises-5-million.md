# Openads 开源项目筹集了 500 万美元

Openads ，一个流行的开源广告服务器的开发者，已经从伦敦的广告网络 Unanimis 中分离出来，并筹集了 500 万美元的资金来支持他们的业务。本轮由 Index Ventures 领投，First Round Capital、Mangrove Capital Partners 和 O'Reilly AlphaTech Ventures 参与。Index Ventures 的索尔·克莱恩加入了公司董事会。首轮资本的 Josh Kopelman 也在他的个人博客上提到了他们的投资。

Openads，以前叫做 [phpadsnew](https://web.archive.org/web/20220627082828/http://phpadsnew.com/two/) ，从使用它的出版商数量来看，是最受欢迎的广告服务器解决方案。他们有 25，000 个出版商在 140 个不同国家的 100，000 个网站上使用该软件，使用 20 种语言。大约三十个广告网络建立在它的基础上，包括约翰·巴特利的调频出版公司。

该软件是 GPL 许可下的开源软件，因此可以免费使用。Openads 将通过在核心软件之上分层提供服务来赚钱，MySQL、WordPress、Zimbra 和其他公司已经成功做到了这一点。更多关于[的信息请点击](https://web.archive.org/web/20220627082828/http://db.beta.techcrunch.com/c/openads)。
