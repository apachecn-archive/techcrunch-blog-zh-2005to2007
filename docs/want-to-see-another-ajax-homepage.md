# HomePortals 可扩展主页

将[门户网站](https://web.archive.org/web/20200929144648/http://www.cfempire.com/home/)添加到不断增长的可定制主页/桌面列表中。

HomePortals 非常独特，因为每个模块都是可定制的，并且可以与其他 web 服务接口。因此，您可以创建一个新的模块(并允许其他人复制它)，和/或修改现有模块的属性。

例如，有一个预先创建的模块显示给定用户最近的 del.icio.us 书签、给定用户最近的 flickr 图片等。我还没有在我评论过的[其他服务](https://web.archive.org/web/20200929144648/http://www.beta.techcrunch.com/2005/12/18/yep-one-more-ajax-desktop-pageflakes/)中看到这种类型的功能。

HomePortals 也有一个非常好的博客工具模块(在 [HomePortals 博客](https://web.archive.org/web/20200929144648/http://www.cfempire.com/home/home.cfm?currentHome=/accounts/homeportals/layouts/blog.xml)上可以看到它的运行)。现在这变得越来越有趣了:我可以使用博客工具，添加我的 flickr 图片和 del.icio.us 书签，让访问者真正深入地了解我是谁以及我的生活中发生了什么。这就像 [SuprGlu](https://web.archive.org/web/20200929144648/http://www.suprglu.com/) ，但是博客工具是内置的，而不是拉进来的。
