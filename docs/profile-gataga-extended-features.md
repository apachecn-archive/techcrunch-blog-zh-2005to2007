# 配置文件:Gataga(扩展功能)

**公司:** [Gataga](https://web.archive.org/web/20201202005904/http://www.gataga.com/)

[<img src = ' https://beta . TechCrunch . com/WP-content/uploads/2005/07/gatagalogo . gif ' alt = ' ' border = 1/](https://web.archive.org/web/20201202005904/http://www.gataga.com/)

先前简介:[2005 年 6 月 13 日](https://web.archive.org/web/20201202005904/http://www.beta.techcrunch.com/?p=11)

**有什么新消息？**

Gataga 刚刚向 TechCrunch 发布了新功能:

–[Gataga 照片搜索](https://web.archive.org/web/20201202005904/http://www.gataga.com/photosearch.php)(通过 Flickr)
–[Gataga 手机](https://web.archive.org/web/20201202005904/http://www.gataga.com/mobile.php)

**照片搜索:**

![](img/0547fb69bc0714ad543581359e1c61b9.png)

新功能允许对 flickr 图片进行标签搜索。它工作得很好，如果有点慢(给他们时间，他们的服务器已经被用户压垮了)。这是对整体标签元搜索的一个很好的补充，有望在不久的将来与普通搜索整合。

我们最喜欢的测试是搜索塞尔玛·哈耶克，我们最喜欢的女演员。下面是结果(记住，现在只是 flickr):

![](img/cad45ed4c19548b95210494360935738.png)

搜索的 RSS 源是可用的。我想看到的网址改变为搜索以及设置 URI，一个简单的修复(允许张贴到搜索，电子邮件等)。).

**Gataga 手机:**

![](img/fce6336da0f2bddeabfde38bf11b35cb.png)

如果你有支持 XHTML (WAP 2.0)的手机或移动设备，你可以在你的移动设备上获得 Gataga 结果(将你的设备指向[gataga.com/m](https://web.archive.org/web/20201202005904/http://gataga.com/m))。目前只有 Technorati 和 Flickr 标签，但这些家伙很快就推出了新功能——预计很快会有更多。此处的屏幕截图:
