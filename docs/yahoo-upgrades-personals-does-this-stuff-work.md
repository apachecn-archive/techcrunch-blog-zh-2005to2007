# 雅虎升级个人信息——这有用吗？

 [](https://web.archive.org/web/20221129044418/http://personals.yahoo.com/) [雅虎征友网](https://web.archive.org/web/20221129044418/http://personals.yahoo.com/)是互联网上最大的交友网站(Match.com 第二)，所以当他们有所行动时，业内其他人都会注意到。

今天，他们将宣布一个名为快速浏览的新功能，它旨在让用户更快地搜索潜在的配偶。会员可以简单地从 15 个不同的特征中选择(不吸烟，喜欢旅游，等等),而不是上下滚动来查看个人资料信息。)称他们为约会“制造者”或“破坏者”然后，他们可以查看按是否包含交易破坏者组织的简档。这是更快剔除那些不受欢迎的人的好方法。

当然，这种东西可以帮助一些人找到匹配。但基本的人性在吸引力方面似乎更视觉化。人们不一定想要填写一份冗长的个性简介来帮助他们挑选潜在的匹配对象，然后发现没有身体上的吸引力。

这就是为什么我认为像 HotOrNot 的 [MeetMe](https://web.archive.org/web/20221129044418/http://meetme.hotornot.com/) 服务和它的 web 2.0 双胞胎 [YesNoMayB](https://web.archive.org/web/20221129044418/http://www.yesnomayb.com/) 这样的网站做得很好，我们[在 10 月](https://web.archive.org/web/20221129044418/http://www.beta.techcrunch.com/2006/10/20/getting-down-to-business-yesnomayb/)介绍过它们。你可以在几分钟内浏览数百人，点击你想了解的人，了解他们更多。如果对方也注意到了你，你们可以聊聊天，回顾一下所有决定人们是否相容的非视觉因素。

当谈到约会网站时，纯粹的数量是很重要的，雅虎、Match.com 和其他公司将继续做得很好。但是这个领域正在进行大量的试验，其中一些古怪的商业模式也会取得成功。
