# 爬壁机器人速度很快

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/02/wall-climbing-robot-is-speedy/>

![](img/72bf32d8a27de47432290dde2050d27a.png)

卡纳基·梅隆大学的一个机器人正在转头，并给出理由说“哦！”和“啊！”机器人以每秒 6 厘米的速度爬墙，如果你点击链接并观看视频，这是相当快的速度。那个小家伙会飞！它能够通过使用壁虎一样的吸盘吸附在它攀爬的表面上。来自文章:

> 当机器人的马达转动时，它的尾巴压在表面上，它的三角形腿向前旋转(a)。当它的前脚与表面接触时，尾巴与表面接触产生的电机扭矩将前脚压在表面上，同时拉开后脚(b)。当作用在后脚上的力达到一个临界点时，它就会从表面剥离，机器人向前迈步(c)。

非常酷的东西。一种可以爬墙的快速小型机器人听起来像是军方马上会采用的东西。不幸的是，机器人确实有一些问题。如果灰尘或污垢堆积在粘垫上，那么机器人就不能正确地爬墙。据我所知，伊拉克有很多沙尘。回到绘图板的家伙。偷看完整的文章和下面的视频链接。
