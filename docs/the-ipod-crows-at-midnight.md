# iPod 在午夜响起

![](img/93e651df062525e539eba650fd2c73be.png)

我在本周的《华尔街日报》上看到了一篇头版新闻(需要付费订阅)关于鸟类观察者使用便携式技术寻找鸟类的报道。一些人使用黑莓等无线设备来接收鸟类接近警报，而另一些人则使用 iPods 和扬声器来模仿鸟类的叫声。

这篇文章有趣的部分是争议:

> 争论的焦点之一是复制鸟鸣的移动技术的使用。亚特兰大的 MightyJams LLC 出售一款 iPod，里面装有 BirdJam 软件和 650 只鸟的歌曲样本……美国观鸟协会的道德准则建议不要对稀有或濒危鸟类采取这种策略，因为这会分散它们保护或喂养幼鸟的注意力。“这是一件非常个人化、自私的事情，”奥杜邦的格林先生说。

此外，根据这篇文章，手机会响并打扰野外的鸟，使用激光笔会伤害鸟的眼睛。

技术的诅咒在于，有时它会变得弊大于利。还有哪些技术会伤害它希望帮助的事物的例子？

总之，在网上，[可能 Jams LLC](https://web.archive.org/web/20210227044518/http://www.birdjam.com/welcome) 出售预装鸟叫声的 iPod 视频和 iPod Nanos。一个 324 美元的套装包括一个 [MacAlly](https://web.archive.org/web/20210227044518/http://www.macally.com/spec/ipod/mytune.html) 便携式扬声器系统。
