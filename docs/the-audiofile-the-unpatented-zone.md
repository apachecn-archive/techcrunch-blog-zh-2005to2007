# 音频文件:无专利区

![](img/9de873946db74b40ff5923936c8b4b47.png)

如果你还没有注意到，起诉数字音频相关专利的[专利流氓](https://web.archive.org/web/20210227223218/http://crunchgear.com/2007/02/26/apple-samsung-sandisk-sued-over-mp3-patent-infringement/) 和 [偷偷摸摸的 me-first 公司](https://web.archive.org/web/20210227223218/http://crunchgear.com/2007/02/24/patent-monkey-details-on-alcatels-15-b-win-against-microsoft/)的数量正在接近无穷大。当它真正到达那里时会发生什么？我们会被吸进另一个空间，在这个空间里，物理定律仍然有效，但是专利法失效了。超越视觉和听觉的维度。我们已经进入了…非专利区域。(嘟-嘟-嘟-嘟…嘟-嘟-嘟。)

在布满仙尘的街道上(或者那是硅？)，无忧无虑的的消费者戴着耳塞到处走。虽然它们带有熟悉的白色，但它们不是普通的耳塞。尽管没有任何明显的 Bose 标志，但它们都有形状有趣的硅胶尖，以获得“稳定的配合和逼真的声音”。突然，一个孩子使劲摇着头，一个耳塞有弹出来的危险——只是被一个非常像耳朵的弹性耳挂固定住了。一丝凉意顺着你的脊柱流下，但你还是挤到了公共广场(或者是一个松鼠？)就在前面。在路上，你瞥见阴影中有东西——一个专利律师的模糊身影。当一名无辜的年轻女子从她的钱包中取出一个带有触摸屏和点拨轮的闪亮矩形物体时，它滑回了附近的下水道。令人欣慰的是，一个大的白色苹果轮廓点缀着背面——终于，来自你自己的维度的东西！一群小孩和相似的玩家一起疾走，通过一个没有名字的协议无线地来回共享未经授权的 Ogg Vorbis 文件。作为这个宇宙中的一名访客，你会感到恐惧，但这只是在…无专利区的又一天。

一个戴着眼镜，留着山羊胡子，穿着牛仔裤和黑色高领毛衣的男人走过来问一些普通的问题。但当他靠近时，你可以看到他的苹果品牌音乐播放器的屏幕(这位年轻女士也有)，屏幕上显示的菜单系统太熟悉了，与 Creative 的 Zen 系列播放器上的一模一样。在这个未申请专利的区域，苹果公司从未向 Creative 支付 1 亿美元购买该界面的许可权——他们似乎只是简单地批发了一下。没有得到你的回应，这个人低头看着他的播放器，在触摸屏上做手势，把界面换成看起来可疑的 Windows Mobile，当他走开时，引起了深深的不安。

在非专利区，事情变得越来越奇怪，但当你听到远处响起空袭警报时，最后一击来临了。仿佛是在集体冲动的驱使下，视线内的每个人都拿出他们的“苹果”播放器，翻转到背面，去掉白色苹果形状的贴花，换上全新的白色香蕉贴花。观众对你的震惊和沮丧的窃笑声越来越大，直到它成为一个完整的笑声…你开始感到头晕…

在床上醒来，你被来自苹果、微软、Bose、Ultimate Ears 和 Creative 的代表包围着。叙述你在非专利区的噩梦般的旅行，你突然意识到你一直被熟悉的面孔包围着。“你在那里…你，你…”你看着你的床头柜，你的 iPod 像一个忠实的朋友一样坐在它的底座上。快速查看一下菜单和背面的标志，你会确信所有的专利在这个宇宙中都是完整的。

用眼角的余光，你发现阴影中有个黑色的东西在地板上移动——你拉起被子，害怕又一个专利律师。咻，感谢~~乔布斯~~上帝，这只是你的狗。
