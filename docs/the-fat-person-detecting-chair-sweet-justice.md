# 胖子探测椅:甜蜜的正义

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/01/the-fat-person-detecting-chair-sweet-justice/>

[![fuwapica.jpg](img/a35f547ced0ca2f39d6dfe4a99ab415f.png)](https://web.archive.org/web/20160505221406/http://old.crunchgear.com/wp-content/uploads/fuwapica.jpg "fuwapica.jpg")

胖子完全讨厌，对吧？胖胖的，身材走样的，挡着我的路，在飞机上占了那么多空间。已经够了。这就是为什么这把椅子来自——还有哪里？—日本让我太开心了。正常的时候，好人坐在上面，椅子散发出淡蓝色。当好心的老胖子坐下时，椅子会发出红光。红色像苹果的颜色。也许如果你吃一个你就不会这么胖了。

谢谢福娃 pica，这么神奇又需要很久的椅子的制造者。
