# 索尼 VAIO VGX-TP1DT Roomba 电脑

当比格斯和我在 CES 的大厅里漫步时，我们看到了这款外形奇特的索尼个人电脑，它正在进行某种网络直播。我们前往索尼展台试图找到它，但没有成功。

我们对摊位猴子的询问只得到困惑的表情。心烦意乱，我们回到了新闻发布室，在那里我们撕扯了文斯大约一个小时，直到是时候离开。

然而，我们的好奇心仍然被激起。所以感谢 Engadget 挖出了这只小动物。


原来它实际上是 VAIO 的两个新产品，VGX-TP1 和 VGX-DT1，它们链接在一起形成了巧妙地命名为 VGX-TP1DT(又名 Roomba PC)。TP1(顶部)圆柱是一台 Vista 家庭高级电脑，配有 1.66GHz Core 2 Duo CPU。它具有 1GB 的内存，可扩展至 2GB，160GB 的硬盘可升级至 500GB。该设备还具有一个带有 HDMI、模拟 RGB 和光学数字输出的地面调谐器。然后是所有的标准噪音，如 USB 火线，记忆棒和 SD 插槽，以及 WiFi。

DT1(底部)圆柱体实际上是一个数字调谐器，能够通过以太网和 WiFi 传输高清内容。然后，您的 TP1 设备可以以 1920 x 1080/24Mbps 或 1440 x 1080/10Mbps 的速率在 1080i 中录制这些内容。

它的面积只有 10.6 英寸，这使得它非常适合家庭影院环境。如果你感兴趣，这一对应该很快就会在日本上市，售价约为 1，660 美元。
