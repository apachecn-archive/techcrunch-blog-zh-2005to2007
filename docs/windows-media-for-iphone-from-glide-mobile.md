# Glide Mobile 的 Windows Media

尽管大肆宣传，iPhone 还是少了一些东西…特别是对 Windows Media 格式的支持，但 Glide Mobile 本周宣布，它现在将为 iPhone 提供新的格式支持，包括 Windows Media 视频和音频。iPhone 支持的格式包括:wmv。mpg，。mpeg，。avi，。divx，。mov、. mp4 和. 3gp。iPhone 支持的 Glide 音频格式包括:。wma。wav，。au、. mp3、. m4a、.aiff，。aac 和. ac3。

“EDGE 网络对于尖端的 iPhone 来说并不理想，”TransMedia 董事长兼首席执行官 Donald Leka 说。“Glide 通过自动提供针对 EDGE 和 Wi-Fi 访问优化的较低和较高分辨率版本的视频和音频文件来进行补偿。”

所有这些支持的格式都包括 EDGE 和 Wi-Fi 网络版本。
