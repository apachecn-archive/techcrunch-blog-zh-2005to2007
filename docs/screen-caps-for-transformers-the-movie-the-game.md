# 变形金刚:电影:游戏的屏幕帽

我们对即将上映的《变形金刚 T2》电影非常感兴趣。你不应该问为什么，这是最基本的。对于我们许多人来说，我们对小工具的痴迷始于变形金刚。它们是有用途的小型手持设备，很复杂(至少有些是)。抱歉，Gears)，并且能够在一所小学里分配地位。

我们也喜欢电子游戏。所以，在电影结束后，我们对这个游戏感到兴奋，我们真的很期待给你这些游戏中的镜头。是的，这将是一个很好的机会。
