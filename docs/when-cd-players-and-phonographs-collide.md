# 当 CD 播放器和留声机相撞时

[![cdphono.jpg](img/8e8b8fe8252e6093188051b5082aba7c.png)](https://web.archive.org/web/20210228041227/https://beta.techcrunch.com/wp-content/uploads/2007/05/cdphono.jpg "cdphono.jpg")

这是一个奇特的留声机形状的 CD 播放器原型。(这是实际创建的，而不仅仅是渲染。)设计者，雍洁玉，拆解了一台 CD 播放器，并把必要的部件重新组装成留声机的形状。我知道这是完全的疯狂。激光拾音器是标准唱机唱针的位置。一般来说，这是可以的，但奇怪的描述——“设计试图将用户带回到 20 世纪初留声机的黄金时代，在那里声音广播有一种神奇的感觉”——叶斯有点让人倒胃口。
