# Skype 2.6 测试版现已推出| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/31/skype-26-beta-available/>

Skype 刚刚发布了其网络电话软件的 2.6 版本作为公开测试版。有什么新鲜事吗？

新版本对 Skypecasts 做了大量的改进，到目前为止我还没有真正用过。新的“直播”标签列出了即将开始的活动，可能是从语言课到约会服务的任何事情。希望我还能过滤 skypecasts 或者完全关闭它。

Internet Explorer 和 Firefox 有一个新的浏览器插件，这意味着网页上的电话号码将被识别，Skype 用户只需点击该号码就可以使用 SkypeOut 进行呼叫。这是一个非常好的功能，但是当我加载 Skype 工具栏时，我以为它已经包含在以前的版本中了。

还有一个新功能是添加心情信息的链接。您的联系人会在他们的 Skype 界面上看到好友图标旁边的链接。

最后，Skype 正在利用其全球用户数据库，增加了一项功能，当 Skype 用户的任何联系人过生日时，该功能会提醒他们，前提是他们已将该信息输入了个人资料。很快就是我的 40 岁生日了，所以我要看看它是否有用。

说到生日，Skype 本周就满三岁了！他们有 700 万并发在线用户。干得好 Skype

测试版可从 Skype 网站下载。
