# Pligg，开源的 Digg 克隆，准备出售

 [Pligg](https://web.archive.org/web/20220417131436/http://www.pligg.com/) ，一个[有争议](https://web.archive.org/web/20220417131436/http://www.beta.techcrunch.com/2007/03/19/ridiculous-one-digg-clone-takes-legal-action-against-another/)但流行的开源内容管理系统，允许开发者快速创建[像 Digg](https://web.archive.org/web/20220417131436/http://www.crunchbase.com/company/digg) 一样的克隆，已经把自己放在[出售](https://web.archive.org/web/20220417131436/http://www.sedo.com/search/details.php4?domain=pligg.com)。

出售的资产包括域名、网站和对 SourceForge 账户的控制权，人们可以在这个账户上下载当前代码。

在一封电子邮件中，Pligg 表示，他们在该网站上有 1 万名注册用户，该软件已被下载 6 万次。

> Pligg 社区的发展超出了预期，超过 10，000 名注册用户分享和交易技巧，超过 60，000 次 Sourceforge 下载和 2 个 24/7 活动的 IRC 房间。这已经成为一个非常大的项目和社区来管理。我们都有白天的工作，努力找到足够的时间来跟上节奏。我们最近看到许多在线企业开发产品来支持 Pligg。销售从定制模板和模块到支持和安装的一切产品。这是一个明显的例子，有了合适的团队，Pligg.com 可以轻而易举地主宰这一领域的其他市场。Pligg 甚至创建了一个商店，用户可以在那里购买模块、模板和服务。不过这项功能还没有推出。
