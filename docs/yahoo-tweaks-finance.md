# 雅虎调整财务 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/17/yahoo-tweaks-finance/>

雅虎财经明天将对其[财经](https://web.archive.org/web/20221130203429/http://finance.yahoo.com/)网站进行一系列调整。[的变化](https://web.archive.org/web/20221130203429/http://news.com.com/2100-1038_3-6094693.html)包括增加了交互式股票图表、改进的留言板以及来自 ABC.com、CNN.com、Forbes.com 和 SmartMoney 的商业相关视频剪辑。雅虎也将为网站发布一个金融小工具[,我们在五月中旬曾报道过。](https://web.archive.org/web/20221130203429/http://www.beta.techcrunch.com/2006/05/19/yahoo-to-test-finance-widget-in-june/)

这些变化显然是对 2006 年 3 月发布的谷歌金融的回应。

自推出以来，谷歌金融并未占据显著的市场份额——相比之下，雅虎是页面浏览量最大的金融相关网站。它号称每月有 1000 万独立访客和 4.67 亿页面浏览量。每天大约有 150 万人访问雅虎财经。(资料来源:comscore，2006 年 6 月)
