# TopTenSources 获取样式馈送器

[![](img/ca381f3c1f7b38b4e5bda8330535d308.png)](https://web.archive.org/web/20220816004522/http://www.toptensources.com/) 人类编辑博客亮点网站 [TopTenSources](https://web.archive.org/web/20220816004522/http://www.toptensources.com/) 将于明天宣布收购社交购物网站 [Style Feeder](https://web.archive.org/web/20220816004522/http://www.stylefeeder.com/) 。我真的很喜欢这两家公司，很高兴看到合作关系如何发展。

TopTenSources 由资深技术专家 Halley Suitt 领导。该网站是基于人类编辑选择他们最喜欢的 10 个博客，主题广泛，从食物到博客到第二人生。

[Style Feeder](https://web.archive.org/web/20220816004522/http://www.stylefeeder.com/) 是由 Philip Jacob 创建的，他是一名程序员，在之前的几个创业公司都有工作经验。该服务使用一些经典的 Web 2.0 工具来创建高度可用的购物体验。一个 javascript bookmarklet 允许用户标记任何网站上出售的商品，并通过首先要求产品图像被点击来填充图像字段——非常好。URL 和描述是自动填充的，用户可以应用标签。然后，该商品会被添加到你的风格提要中，这是一个愿望列表页面，其他对你的风格和购物倾向感兴趣的用户可以跟踪这个页面。用户可以加入主题小组，并从他们的愿望清单中添加一盒小图片到他们在其他地方的个人网站。

Style Feeder 目前还没有盈利，但开发者 Phillip Jacob 告诉我，从附属链接到大型在线社区的私有标签版本的想法都在考虑之中。Halley Suitt 告诉我，TopTenSources 网站本身和 Style Feeder 之间的整合程度还有待确定，但 TopTenSources 很高兴地认为自己是一家在其旗舰网站之外提供多样化产品的公司。

我认为 TopTenSources 被低估已经有一段时间了，我很期待看到他们下一步的走向。
