# 在你的手机上使用 Skype

 [](https://web.archive.org/web/20221203092157/http://www.soonr.com/) 总部位于硅谷的 [SoonR](https://web.archive.org/web/20221203092157/http://www.soonr.com/) 推出了一款测试产品，用户可以通过手机访问 Windows 电脑上的应用程序和文件(目前还不支持 Mac，但即将推出)。如果你的手机可以上网，你就可以用 SoonR 访问你电脑上的 Outlook、图片和其他文件及应用程序。早在三月份 MobileCrunch 第一次进入测试版时，它就进行了回顾。

本周四，SoonR 将发布 Soonr Talk，它将允许用标准手机拨打 Skype 电话。它将通过 SoonR 的基本产品工作——通过浏览器在您的家用电脑上访问 Skype，并与一个或多个联系人发起通话。然后，SoonR 将从该电脑拨打您的手机(使用 Skype Out 点数),并通过普通的 Skype 通话功能拨打联系人的电话。联合创始人兼营销副总裁黄松向我展示了实际使用的产品，并在他的手机上敲了几下就建立了通话。效果很好。

SoonR 是免费的，并将在未来推出优质服务(例如，即使您的电脑关机，SoonR 也可以工作——SoonR 将缓存您希望能够访问的数据)。

一旦他们有了 SoonR 的 Mac 版本，我就加入。更多截图[此处](https://web.archive.org/web/20221203092157/http://flickr.com/photos/michaelarrington/tags/soonr/)。
