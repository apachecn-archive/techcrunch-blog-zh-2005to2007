# 威瑞森将在年底获得 Treo 700wx

威瑞森无线公司将在其 Treo 700w 中增加一个“x”。700wx 目前仅由 Sprint 提供。他们差别非常非常小，对拨号网络的支持是关键因素，但每个人都想在同一个游戏上。

它将保留 EV-DO，配有 Windows Mobile 5、全球定位系统、130 万像素摄像头、240×240 屏幕和 128MB 内置内存，与 Sprint 版本相同，但没有提到任何软件。期待天线运动的 Windows Mobile 5 为基础的 Treo 在圣诞节前一触即发。
