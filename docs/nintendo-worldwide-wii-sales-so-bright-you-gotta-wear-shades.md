# 任天堂 Wii 全球销量如此之大，你不得不戴上墨镜

[![wiiwins.jpg](img/e4f360f07424e7bd4a02792d3c907642.png)](https://web.archive.org/web/20210227013254/https://beta.techcrunch.com/wp-content/uploads/2007/05/wiiwins.jpg "wiiwins.jpg")

任天堂 [Wii](https://web.archive.org/web/20210227013254/http://crunchgear.com/2007/02/02/conan-whips-serena-in-wii-tennis/) 和 [DS](https://web.archive.org/web/20210227013254/http://crunchgear.com/2007/05/08/the-worst-nintendo-ds-ever/) 卖得如此之好，以至于任天堂甚至懒得再做销售预测了。现在有 584 万台 Wii 和 4000 万台 DSs，这些数字是任天堂没有预料到的。销售如此强劲，以至于该公司抱怨不得不不断重新评估其销售目标。所以，如果你以前不相信，现在就相信吧:Wii 正在(暂时)获胜，它将需要另一方的[杀手](https://web.archive.org/web/20210227013254/http://crunchgear.com/2007/05/03/xbox-360-making-for-slow-going-of-gta-iv/) [应用](https://web.archive.org/web/20210227013254/http://crunchgear.com/2007/05/03/the-coolest-halo-3-figurines-ever/)才能在这场主机战争中产生任何影响。只要每个人都对他们选择的游戏系统满意，我就很高兴。
