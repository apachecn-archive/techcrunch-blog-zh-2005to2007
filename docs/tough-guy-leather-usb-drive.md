# 硬汉皮革 u 盘| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/01/04/tough-guy-leather-usb-drive/>

大多数平庸的东西可以通过用皮革包装起来而从“愚蠢”提升到“有趣”。看:摇滚明星，方向盘，舞会日期。Digiwork 最新的 USB 拇指驱动器也是如此。在 1gb、2gb 或 4GB 的尺寸中，其他不起眼的闪存驱动器被包裹在感官的、奢华的皮革中。有了钥匙圈，这将是我们唯一真正放在钥匙链上的闪存盘。它说，“当然，我是一个极客，但我是一个*享乐主义*极客。”

我当然希望这些能让它回到美国。
