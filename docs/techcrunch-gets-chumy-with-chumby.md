# TechCrunch 与 Chumby 关系密切

![](img/babda5f6c99f310c5c76281b9c97961b.png)

[TechCrunch](https://web.archive.org/web/20191111110152/http://www.beta.techcrunch.com/) 本周末有机会体验一下 [Chumby](https://web.archive.org/web/20191111110152/http://www.chumby.com/) ，这是一款肯定会在明年产生巨大影响的设备。它因其完全的可攻击性而引人注目。一个人用它做什么，本质上只受限于他的想象力。用户可以对这个小型手持设备进行编程，以执行几乎任何想要的功能。

它配有 266Mhz ARM 控制器、32MB SDRAM、320 x 240 3.5 英寸触摸屏、WiFi、两个扬声器、音频输入/输出和一个 USB 端口。它可以通过触摸屏或外壳上的挤压传感器进行控制。内部构件可以很容易地从外壳中取出，并安装到另一个外壳中。预计将于 2007 年 3 月上市，建议零售价为 150 美元。开发人员现在可以访问 [Chumby](https://web.archive.org/web/20191111110152/https://crunchbase.com/organization/chumby) 并索取样品单元。
