# plantronics Pulsar 260–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/27/plantronics-pulsar-260/>

![](img/a0dfb0abaf01c907577fd6502f6419a3.png)

我通常不会对耳机感到兴奋，但 Plantronics Pulsar 系列已经生产了一些强大的立体声耳机，260 是最新的一款。这款耳机配有隔音耳塞和一个带麦克风的小型蓝牙转换器。

这款耳机售价 109.95 美元，包括内置接听按钮和快进/快退按钮。它还有一个立体声和扬声器的音频输出适配器，所以你几乎可以将你的音频流传输到任何东西。
