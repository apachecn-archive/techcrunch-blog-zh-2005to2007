# 诺基亚 N76 评测

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/02/nokia-n76-review/>

过去，你可以买到任何颜色的诺基亚手机，只要是深蓝色的。然而，现在诺基亚决定用 N76 为他们古板的产品线增添一点色彩，N76 是一款奇怪的小翻盖手机，配有漂亮的 2.4 英寸屏幕和专用音乐控件。

N76 是一款智能手机。它运行 Symbian S60，有 WCDMA 和 GSM 版本。它似乎还没有在我们的海岸上市——然而，如果我的小手里有它，那就意味着它应该随时会出现在网上和诺基亚的商店里。我现在不能确认，但看起来没有任何运营商会接这个老女孩，所以你可能需要花大约 499 美元。

这款手机本身非常时尚，拥有 200 万像素的摄像头和明亮的镀铬配色方案。它厚 13 毫米，内置 26MB 内存，带热插拔 microSD 卡插槽。一次充电大约持续 160 分钟，尽管我看到待机时间约为 10 天，但很少使用。

前面有镀铬按钮来控制音乐播放，还有一个 1.3 英寸的 birght 外壳显示屏。它支持大多数音乐播放，包括 Windows Media DRM 歌曲。还有一个调频调谐器。

该设备内置了一个 RealPlayer，因此可以播放多种视频格式和视频门户网站上的视频流。奇怪的是，这只小狗没有无线网络，这可能会吓跑一些购买者。

N76 是一种“高级时尚”智能手机。因为它运行 S60，你可以得到你想要的一切——邮件、即时消息、Symbian 应用支持。但因为它看起来像一个类固醇 RAZR，它不会让你看起来像一个太多的极客。一个非常好的权衡。

显然，尤其是 499 美元，N76 并不适合所有人。它看起来很棒，很有活力，但还有其他更强大的 N 系列手机。如果你正在寻找一部有点档次的智能手机，这是一个很好的选择。如果你正在寻找一台老爷车——甚至可能是带键盘的——去别处看看。
