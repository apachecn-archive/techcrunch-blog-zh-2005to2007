# 双簧管的网络音乐锁 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/12/02/oboes-web-music-locker/>

上周，我在的一篇[帖子中恳求这样做，这篇帖子是关于我想介绍的还不存在的公司(第一名——“更好更便宜的在线存储”)。](https://web.archive.org/web/20221002121145/http://www.beta.techcrunch.com/2005/11/21/companies-id-like-to-profile-but-dont-exist/) [MP3Tunes](https://web.archive.org/web/20221002121145/http://www.mp3tunes.com/) 用一个叫 Oboe 的产品套件钉住了它。

双簧管以每年 40 美元的价格提供无限量的在线音乐存储。该产品还有一个免费版本，尽管它的功能非常有限(例如，没有在线存储)。

我已经注册并支付了高级产品的费用。音乐同步是通过下载的应用程序(windown、mac 和 linux)完成的，这将永远持续下去。我有大约 10，000 首歌曲，它们以每小时 200 首的速度上传。这意味着我将上传大约 50 个小时。

歌曲上传后，我会写更多关于双簧管如何运作的内容。该网站承诺用户将能够创建和编辑播放列表，以 128k 播放音乐，甚至直接通过 itunes 播放。支持的格式包括 MP3，MP4，M4A，M4P，AAC，WMA，OGG，AIF，AIFF 和 MIDI。

在[FAQ](https://web.archive.org/web/20221002121145/http://www.mp3tunes.com/locker/cb/support/)中没有明确说明的一个特性是，用户是否能够将音乐下载回他们的硬盘(例如，在他们丢失了本地数据的情况下)。然而，FAQ 确实声明，“只需点击几下鼠标就能确保你永远不会丢失你的音乐。您甚至可以毫不费力地将它加载到您的另一台计算机上！”。允许这样做有明显的盗版问题，因为任何人用借来或偷来的帐户凭证都可以下载音乐。

MP3Tunes 是一家迈克尔·罗伯逊公司，他是 MP3.com 的创始人。双簧管是他允许用户在线欣赏音乐的最新尝试。此前，MP3.com 提供了一项名为 Beam-It 的服务，该服务允许用户通过将 cd 插入电脑来验证自己是否拥有该 CD，并随后能够直接从网上访问该音乐。

MP3Tunes 希望避免 Beam-It 的命运，只充当服务提供商，并假设用户合法拥有他们上传的音乐。[数字千年版权法案](https://web.archive.org/web/20221002121145/http://www.copyright.gov/onlinesp/)对服务提供商的责任做出了限制，“根据用户的指示，信息驻留在服务提供商控制或运营的系统或网络上”。

我希望他们赢得这场比赛。
