# Skype 以 41 亿美元出售给易贝

易贝[今天宣布](https://web.archive.org/web/20221224072441/http://www.mercurynews.com/mld/mercurynews/news/breaking_news/12625660.htm)以 41 亿美元收购 [Skype](https://web.archive.org/web/20221224072441/http://www.skype.com/) ( [以前的](https://web.archive.org/web/20221224072441/http://techcrunch.com/?p=70) [简介](https://web.archive.org/web/20221224072441/http://techcrunch.com/?p=193))。

26 亿美元已经预付——一半现金，一半易贝股票。另外 15 亿美元将根据绩效目标在 2008 年或以后支付。

Skype 拥有 5000 万注册用户，去年的收入约为 700 万美元。今年的收入预计为 6000 万美元。
