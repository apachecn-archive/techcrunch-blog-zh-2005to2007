# 公平游戏:在 iMovie 上的柔道动作

Seidai Software 开发了一个名为 FairGame 的免费脚本，可以从你的 iTunes 文件中删除数字版权管理。它显然使用 iMovie 创建了一个干净的、没有 DRM 的文件副本，看起来非常简单。

正如他们所说，生活会找到出路。
