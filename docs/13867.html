夏普的EM-One性感到疼

如果你还没有看过夏普的EM-One，请在繁忙的一天中抽出一分钟来看看这个简短的视频。专为日本市场打造，这是我想要的一款 Windows Mobile 手机。除了4英寸的触摸屏，你还有一个带媒体控制的双滑动QWERTY键盘，HSDPA，一个带闪光灯的130万像素摄像头，半GB的板载内存，一个520 MHz的处理器，beast拥有自己的GPU。这东西比我上一台笔记本电脑的规格更好，不是开玩笑。

我们可能会在某个时候看到美国版本(不包括数字电视调谐器)，但我们不会屏住呼吸。
