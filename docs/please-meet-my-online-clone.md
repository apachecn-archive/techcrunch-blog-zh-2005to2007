# 请见见我邪恶的在线克隆人

 [](https://web.archive.org/web/20200908231909/http://www.mycybertwin.com/) 人们可以把大部分时间花在网上，但很难把所有时间都花在网上。有时人们不得不工作、上学或吃饭。或者睡觉。而你不在的时候谁会在那里和大家聊天呢？直到现在这个难题还没有答案。现在，有了我的网络双胞胎。

这家总部位于澳大利亚的公司根据 79 个问题的个性测试和数百个额外的培训问题，创建了一个在线“克隆”用户。一旦完成，就会为该用户创建一个聊天机器人，它有自己的网页，可以嵌入 MySpace 或其他网站，或者可以代表你登录 Microsoft Messenger 并伪装成你。大约 10%被训练的迈克尔·阿灵顿克隆体[在这里](https://web.archive.org/web/20200908231909/http://mycybertwin.com/chat/techcrunch)(也在下面嵌入)。我已经教它尽可能积极地反社会。

MyCyberTwin 的创始人 Liesl Capper 和 John Zakos 认为这项服务将会吸引那些寻找用户特征来创造更多内容和更具粘性网站的社交网络和约会网站。该公司还拥有自己的社交网络，提供兼容用户之间的联系。
