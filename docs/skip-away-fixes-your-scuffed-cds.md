# Skip-Away 可以修复磨损的 CD

无论出于什么原因，你还在使用光盘吗？那是你的问题。它们很大，会被刮伤，并且不能保存太多数据。

如果你确实有一些被滥用的遗留信息，或者真的觉得有必要撕掉威猛的*粗心的耳语*！，那么你需要 Skip-Away，一个彩色盒子，用于修复划伤或翘曲的 CD。

这是全自动的，也就是说你把光盘放进去，按下一个杰特森风格的按钮，几秒钟后你就可以试着让你的 CD 重新工作。我们不知道这是如何真正修复磨损的，但如果你是老式的，它可能是必须的。
