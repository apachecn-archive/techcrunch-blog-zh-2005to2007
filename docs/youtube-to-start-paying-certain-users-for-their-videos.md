# YouTube 开始为某些用户的视频付费

[![moneytube.jpg](img/ef3a8418c24b445cde124f9f2bad50a1.png)](https://web.archive.org/web/20200803155006/http://old.crunchgear.com/wp-content/uploads/moneytube.jpg "moneytube.jpg")

通过谷歌的严格测试，你可能会开始为你上传的所有 [YouTube](https://web.archive.org/web/20200803155006/https://crunchbase.com/organization/youtube) 视频获得报酬。日常用户，甚至可能包括互联网超级巨星“孤独女孩”将从一个项目中受益，谷歌希望这个项目将鼓励用户向网站上传更多更好的剪辑。这一次，用户生成内容的创造者将因为创造了所有这些有价值的 Web 2.0 特性——YouTube、FaceBook、Twitter 等——而得到补偿。—和它们一样值钱。

为了开始收到你的一份行动，请联系 YouTube，它将考虑你的申请。如果它说是，就让$unknown 滚进来。请记住，不仅是选择和批准的个案基础上，但你最好有足够的观众来证明获得报酬。[这个](https://web.archive.org/web/20200803155006/http://www.youtube.com/watch?v=k4d5QVQR4Qo)切不了；[这个](https://web.archive.org/web/20200803155006/http://www.youtube.com/watch?v=o32oHGTOzTE)可能。
