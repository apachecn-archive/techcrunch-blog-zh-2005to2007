# 诺基亚 N75 评测

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/23/nokia-n75-review/>

![nokian75near.jpg](img/3ef90943d80245e49c7d3783b340a777.png)

我从来不明白拥有一部可以播放和存储音乐的手机有什么吸引力。这听起来可能很奇怪，但我更愿意随身携带我的 MP3 播放器和手机，而不是随身携带一个能做所有事情的设备。当然，有时候我会诅咒自己没有带包，因为带着钥匙、钱包、超大号手机和 MP3 播放器会变得很麻烦，而且非常烦人。我裤子的每个口袋都鼓鼓囊囊的，这也让我大喊“混球”。我买诺基亚 N75 已经有几个星期了，它的一切功能都很好，但这个芬兰翻盖手机最终改变了我对多媒体手机的看法吗？
 在我体验过的~~极度痛苦~~摆弄音乐手机的乐趣的所有音乐手机中，诺基亚 N75 被证明是一款出众的手机，对于那些真正想要一款做一些其他有价值的事情的音乐手机的人来说，你不必再找了。其简单的设计、易于使用的 Symbian UI 和 AT & T 的 3G 网络使其成为明显的赢家，但它远非完美。

首先，虽然铃声可以通过美国电话电报公司的 MEdiaNet 服务下载，UMTS 也能及时完成这项工作，但我真的希望 N75 能像三星 Sync 一样支持 HSDPA，三星 Sync 也在 AT&T 上提供。我应该提到 HSDPA 也比 UMTS 便宜吗？也许不是。

让我们来谈谈 N75 上的热门功能，即音乐。有一个专门用于音乐的完整子菜单，以及三个外部音乐控制和主键盘上的一个专用音乐控制。最大的遗憾是你不能直接在手机上购买歌曲。WTF？！事实上，集成播放器支持 MP3，WMA，AAC，eAAC+，M4A，以及可怕的 DRM 的对手只是让我说，“feh。”上传你从手机上购买的歌曲，然后把它们传送到你的电脑上，这是一件非常痛苦的事情。这有什么意义吗？

不管怎样，如果你通过美国电话电报公司音乐，Napster to Go，Yahoo！无限制，那么你就可以走了。这是一件好事，他们包括一个 USB 电缆，否则我会非常生气。通过诺基亚 PC 套件上传曲目或只是拖放。N75 有一个接受 2GB 卡的 microSD 插槽，所以要好好利用它。我很喜欢你可以即时创建自己的播放列表。前面提到的外部音乐控制可以让你在几秒钟内播放所有热门曲目。N75 还具有 MobiRadio 和 MobiTV 功能，以便您观看流媒体视频或收听流媒体广播。多亏了 3G，它工作得相当好，但是我建议你带上你的充电器，如果你是一个经常感到无聊的人。

除了明显的吸引力是音乐功能是相机。带有内置闪光灯和 10 倍变焦的 200 万像素无蔡司相机提供了六种不同的场景模式，适合任何情况，包括自动、人像、风景、夜间、夜间人像和用户自定义。在这个问题上，让我们开诚布公吧。任何 200 万像素范围内的手机拍摄的照片都不会登上任何杂志的封面，所以请记住这一点，或者看看 N95 和它的 500 万像素 snapper。N75 在大多数情况下都能拍摄出不错的照片，15fps 的 MPEG-4 视频录制速度仅受板载和 microSD 插槽中的内存容量限制。

让我们不要忘记 N75 的首要目的是什么:打电话。键盘的布局非常友好，超大的按键确保您每次都能敲到这些数字。通话的两端都很清晰，但对我来说，诺基亚手机的通话质量不是问题。我觉得尴尬的是，我不知道把耳朵放在哪里才能听到扬声器的声音，这是大多数蛤壳式扬声器通常都会发生的情况。N75 还感觉如果你施加任何压力，它都会在铰链处折断，这是事实，所以尽量不要这样做。通话时间相当令人失望，甚至没有接近它声称的四个小时的通话时间。一次充电两小时多一点。这太可怕了，但我们会把它归咎于 3G 网络。

其他一些值得注意的功能是 QuickOffice，它可以让你查看 Word、Excel 和 Powerpoint 文档，但除非你升级到完整版，否则你无法编辑它们。即时消息用户不必担心，因为 N75 可以让您与 Yahoo！、Hotmail 和 AIM 电子邮件和信使服务。N75 还支持 IMAP 和 POP3。PM 工具与其他 N 系列手机保持一致。

该设备本身价值 199.99 美元和 2 年合约，但你必须处理美国电话电报公司将让你跳过去享受它的铁环。
