# Wi-Fi iPod 正在路上，可能会在夏天准备好，让生活变得简单

[![ipodwifi.jpg](img/f2251a672eb95e5295b3e38ac10979ab.png)](https://web.archive.org/web/20201205152948/https://beta.techcrunch.com/wp-content/uploads/2007/04/ipodwifi.jpg "ipodwifi.jpg")

苹果的主要制造商已经暗示将在今年下半年发布 Wi-Fi ipod 的生产。台湾 *DigiTimes* 报道称，富士康将生产 iPods 成品，最早可能在夏季出货。目前还不知道这款 Wi-Fi iPod 是否会像 iPhone 一样有触摸屏。

当然，内置 Wi-Fi 的 iPod 自诞生以来就一直在传言，但富士康和任何人一样清楚苹果在做什么。
