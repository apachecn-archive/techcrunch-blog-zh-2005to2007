Windows Home Server今天向OEM发货| TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2007/07/16/windows-home-server-shipping-to-oems-today/

根据WHS团队博客上的一篇文章，Windows Home Server 软件今天将走向它的OEM合作伙伴。

WHS基本上是一个易于使用的服务器，允许家庭在多台计算机之间存储和共享数码照片、视频和文档。每天晚上可以自动执行备份，存储可扩展，并且可以从任何地方远程访问文件。

微软目前与六家原始设备制造商达成协议:富士通-西门子、Gateway、惠普、Iomega、Lacie和Medion。硬件设备应该会在今年秋天开始发货，起价在500美元左右。
