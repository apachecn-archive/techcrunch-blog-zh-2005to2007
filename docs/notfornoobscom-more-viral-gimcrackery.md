# NotForNoobs.com:更多的病毒漏洞

![Photobucket - Video and Image Hosting](img/1bbeca0941f866986f0c7a4aec009147.png)
好傻好傻。Gizmodo 和其他人认为，微软最新的 hork-tastic 病毒式营销尝试——称为 NotForNoobs——基本上是他们销售 Zune 高端媒体硬件的一种方式。关键是那条蛇，它是 [Razer](https://web.archive.org/web/20210122161401/http://crunchgear.com/2006/08/14/razer-protype-ipodzune-dockable-keyboards/) 的标志，它正与微软合作，用他们的 Razer 键盘与 Zune dock 一起摇滚出一些缓慢的堵塞。演示结束倒计时将于美国东部时间下周四午夜结束。

病毒式营销是一种很好的营销方法，但它有点像喊狼来了的男孩——这种东西过一段时间后就开始变得没用了。但是话说回来，仅仅是写下这些文字，我就落入了他们的圈套。
