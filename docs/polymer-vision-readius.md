# 聚合物视觉 Readius 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/13/polymer-vision-readius/>

我终于找到了离新闻发布室很近的聚合物视觉 Readius。这是每个人都在谈论的可滚动显示器，它实际上非常酷。该设备大约像手电筒一样大，有一个 5 英寸的灰度，16“色”屏幕，大约一秒钟更新一次。这更像是索尼阅读器，而不是其他任何东西，它似乎不会出售。相反，他们有一个模型挂在展台上，不停地打开和关闭，只有两个人在周围谈论它。显然，世人没有看到他们的天才。

包括电子书阅读器应用程序、RSS 阅读器和音乐/播客播放。这东西不可能播放视频，但这是一个诱人的想法。
