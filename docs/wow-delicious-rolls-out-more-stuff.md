# 哇！Del.icio.us 推出更多内容

除了 Del.icio.us 本周早些时候发布的强大而开放的[搜索功能，至少还有两个新功能值得注意。我还对下面的搜索功能有一些想法。](https://web.archive.org/web/20220124205230/http://www.beta.techcrunch.com/2005/10/24/337/)

## 媒体播放机

Del.icio.us 已经在网站中直接内置了一个媒体播放器。如果您找到 MP3 文件的链接，书签的左侧会出现一个小图标，单击该图标可以播放该文件。[试用一下](https://web.archive.org/web/20220124205230/http://del.icio.us/popular/system:filetype:mp3)。

## 标签包

[标签包](https://web.archive.org/web/20220124205230/http://blog.del.icio.us/blog/2005/10/bundle_up.html)实际上是标签的标签。这个功能以前就有，但是现在只需点击标签，就可以更容易地创建和编辑标签包。

## Del.icio.us 搜索引擎

过去几天，我一直在使用 del.icio.us 搜索功能。对于某些搜索，del.icio.us 的相关搜索结果比其他任何实时搜索引擎都要多得多。数据不一定很深，但最重要的结果是非常相关的。稍后可能会有更多的报道，但是 del.icio.us 可能已经悄悄地创造了网络上最好的(实时)搜索引擎。

## Delicious.com

最后一件事。最近有人向我指出[delicious.com](https://web.archive.org/web/20220124205230/http://www.delicious.com/)现在被重定向到 del.icio.us，这个域名是以约书亚的名字注册的。我认为获得域名是一个伟大的举动，我不介意用新的网址重新命名。
