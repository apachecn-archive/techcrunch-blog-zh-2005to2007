# Multiply 结束第一轮融资

[![](img/2a79f9f75aee9441c8734c432584ee63.png)](https://web.archive.org/web/20220926193241/http://www.multiply.com/) 在线社交网络 [Multiply](https://web.archive.org/web/20220926193241/http://www.multiply.com/) 已完成首轮融资，Transcosmos 投资 500 万美元，公司创始人投资 100 万美元。Multiply 是一种过滤所有网络功能的服务，从突出显示的用户到可见的标签云，通过一个带滑块的邻近过滤器。换句话说，用户决定他们是否想要查看他们的联系人列表上的人的信息，谁是朋友的朋友，等等。作为新的社交网站，我想说这个网站有相当大的生存机会。

自 2004 年 3 月推出以来，该系统已有近 300 万注册用户，最近还针对日本推出了本地化版本。该公司表示，它将使用新的资金来扩大特点，功能，营销和国际存在。网络上有如此多的社交网站，营销将是必不可少的。

创始人兼总裁彼得·佩扎里斯曾是 CBS SportsLine.com 的运营和产品开发总裁，负责 nfl.com、pgatour.com 和 cbs.sportsline.com 的运营。Transcosmos 为包括 Ask/Bloglines、RealNetworks、Hipcast/Audioblog、Pheedo、Edgio 等在内的投资组合增加了 Multiply。

![](img/7ae838b0ce0489b7637c43970dc08f04.png) Multiply 旨在通过密切关注将互动限制在已经相互联系的人之间来实现与众不同。除此之外，该系统还有许多有趣的功能，比如 Ajax 留言板和一个由用户关注组织的论坛，以及线程更新货币。留言板以近乎实时的方式在对话各方的屏幕上显示新的评论，论坛将根据用户之前是否阅读过这些评论来排列主题的优先级——因此，某个主题不会因为只有两个关注它的人之一发表了新帖子而出现在你的论坛视图的顶部。网站上到处都是 RSS，感觉很不错。

如果新的资金得到充分利用，Multiply 可能会在社交网络这块蛋糕上分得更大一块。
