# Technorati 的关键开发人员被谷歌抢走

据他的博客透露，[关键的 Technorati 开发者凯文·马克斯](https://web.archive.org/web/20220119010212/http://en.wikipedia.org/wiki/Kevin_Marks)已经离开公司，现在在谷歌。就在一个半月前，谷歌[领先](https://web.archive.org/web/20220119010212/http://www.beta.techcrunch.com/2006/12/28/google-v-technorati-and-hitwise-v-comscore/)成为 Technorait 的顶级博客搜索引擎，失去分数对他们来说是进一步的打击。

Technorati 向公众发布的关于其自身的信息很少，让我们主要猜测他们的财务健康状况。这一举动可能是麻烦的征兆。如果 Technorati 是一家上市公司，这将是卖空其股票的好时机。
