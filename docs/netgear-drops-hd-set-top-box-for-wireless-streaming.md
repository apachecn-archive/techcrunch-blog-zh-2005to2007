Netgear 凭借其数字娱乐高清媒体接收器击败了苹果的独立电视。机顶盒的工作方式很像即将上市的 iTV，它可以在你的个人电脑或 Mac 电脑网络上无线查找媒体，并允许你在家庭影院系统上播放所有内容。

虽然与 iTunes 下载不兼容，但该系统可以播放大多数其他媒体格式，包括 DRMed Windows Media 文件，甚至 XviD，以及对您的 Bit Torrent 粉丝来说令人兴奋的提议。

Digital Entainer HD 的非凡之处在于，除了您可能拥有的标准高清内容之外，它还支持应于今年推出的高清内容，通过 HDMI 连接到您的高清电视，分辨率高达 1080p。现在这里有一些缩写，但它的简称是下载高清，抓取高清，回放高清。
