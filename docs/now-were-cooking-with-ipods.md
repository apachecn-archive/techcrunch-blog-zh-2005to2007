# 现在我们用 iPods 做饭

就在几天前，我向 iPod 的上帝祈祷，希望有一个橱柜下的多媒体播放器来容纳我在厨房里的 iPod。沙赞姆。飞利浦发布了适用于 iPods、DVD、(S)VCD、MP3-CD、CD-RW 和图片 CD 的 DCD778 扩展坞娱乐系统。娱乐中心配有 8.5 英寸宽屏液晶显示器、内置 ATSC/NTSC 电视调谐器、数字 FM/AM 调谐器、两个 2 英寸扬声器、无线遥控器、烹饪定时器(真棒)和时钟。DCD778 兼容 4G、5G、mini 和 nano，下周上市，售价约为 400 美元。你能说“哇，厨房用的时钟和 iPod 基座！”不，可能不会。
