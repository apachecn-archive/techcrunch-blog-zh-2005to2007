# 现在我们有了 GodTube

 [](https://web.archive.org/web/20220818231547/http://www.godtube.com/) 我们写了 YouTube 上的克隆色情作品，所以为了平衡我们的因果报应，我们需要提到 [GodTube](https://web.archive.org/web/20220818231547/http://www.godtube.com/) ，一个类似 YouTube 的 Flash 视频分享网站，专注于使用“技术连接基督徒，以鼓励和推动福音在全球传播”所以当他们说上帝时，他们特指耶稣基督和基督教，而不仅仅是某个人可能信仰的任何一个上帝。

不要试图上传非基督教视频到网站。GodTube 依靠成员的“警惕”来快速标记不适当的内容。这与他们的“专有技术”相结合，帮助他们执行他们的社区标准。

我认为小众社交网络和其他类似的服务很棒——它给了有共同信仰和生活方式的人一个闲逛的地方。我认为 JesusTube 会是一个更好的名字选择，虽然，鉴于这是仅限于基督教的上帝。参见 TechCrunch 论坛上关于 MyChurch 的[讨论，这是一个“基督徒的脸书”。](https://web.archive.org/web/20220818231547/http://forums.beta.techcrunch.com/forums/thread.jspa?messageID=10027)
