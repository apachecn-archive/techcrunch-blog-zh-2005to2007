# Cingular 的新 Walkmanified 索尼爱立信

Cingular 今天在其网站上添加了一款新的索尼-爱立信随身听手机为“当前可用”。W300i 是一款功能齐全、价格合理的手机。它具有蓝牙功能，640 x 480 的摄像头，视频播放，访问 MSN，雅虎！，还有 AIM messengers，外置扬声器，当然还有带记忆棒微扩展的随身听音乐播放。蛤壳还具有外部控制功能，这是我们最近经常看到的一个漂亮的功能。

如果你能接受 2 年的合同，这个小家伙只会在 75 美元的邮件回扣后花掉你 19 美元。目前还没有消息表明 W300i 是否能与索尼爱立信新的 M-Buzz 商店合作，但如果他们开始以低端价格推出更多的这种中档手机，这似乎是一个明智的举动。
