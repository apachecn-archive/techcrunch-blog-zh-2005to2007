# 贝尔&罗斯 BR 01 幻影

![mailing-br01-phantom.jpg](img/51c34e4838441dd8f39c555afbcf6c2c.png)

贝尔&罗斯制造看起来像喷气式飞机读数器的大喇叭飞行员手表。他们的最新产品，幻影，有一个低能见度的脸，在黑暗中会发光，确保你的黑暗行动位置会被一个像手电筒一样发光的表盘泄露。

没有价格或可用性，但预计它在 4000 美元至 6000 美元的范围内。
