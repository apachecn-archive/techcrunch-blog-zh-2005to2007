# 专利猴:用 PS3 控制器做手机分机？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/22/patent-monkey-using-ps3-controller-as-extension-for-cell-phone/>

![](img/ad9525da856b22fa201ede585a28e793.png)

打[大学篮球 2K7](https://web.archive.org/web/20150806212054/http://ps3.gamezone.com/gamesell/p29737.htm) 的时候打不到手机？这项新功能可以让你使用游戏控制器、PS3 和平板电视来查看是谁打来的电话并接听。

索尼爱立信最近获得了一项的[专利，涵盖了无线传输内容或手机通话并使用遥控器或 PS3 控制器控制它们的过程。](https://web.archive.org/web/20150806212054/http://www.patentmonkey.com/PM/PatentID/7194259.aspx) 

核心发明描述了遥控器的使用，它可以作为你手机的延伸。例如，“在用户决定应答呼叫之前，可以在电视上显示呼叫者 ID 数据。或者，在应答呼叫时，RCD 可以将音频路由到与电视或立体声系统耦合的扬声器播放。如果输入信号是由文本、音频和/或视频组成的数据信号，则 RCD 可以控制电视，以便“播放”消息。如果电视机碰巧配备了数字视频录制(DVR)设备，RCD 可以自动暂停输入电视机的源信号，这样一旦 RCD/移动通信完成，用户就可以恢复观看，而不会错过任何内容。”

此外，本发明旨在包括连接到多个外围设备(包括游戏系统)的遥控器，作为连接到监视器的手段。本发明增加了允许内容被传送到设备的概念，增加了蜂窝电话作为接入扩展的移动对等网络的另一种手段(像对等消息系统 [SwarmTeams"](https://web.archive.org/web/20150806212054/http://home.swarmteams.com/) )。

融合会不会导致[游戏系统手机](https://web.archive.org/web/20150806212054/http://gizmodo.com/gadgets/home-entertainment/nintendo-wii-phone-rumor-pure-unicorn-fantasy-but-what-a-photochop-235258.php)的出现，让你可以随身携带并在那里玩游戏？诺基亚已经在这方面做出了一些承诺。
