# 社交音乐概述

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/05/social-music-overview/>

为了与 Mike 的[在线照片编辑概述](https://web.archive.org/web/20220816040556/http://www.beta.techcrunch.com/2007/02/04/online-photo-editing-overview/)的主题保持一致，我想报道一些进入社交音乐的人。音乐可能是第一种真正走向“Web 2.0”的富媒体类型，它已经成为初创公司非常受欢迎的地方。因此，有一些伟大的丰富的互联网应用围绕社会音乐。任何将音乐作为日常生活一部分的人，在寻找新音乐并与朋友分享时，都不会缺少选择。

fine tune

[![Finetune](img/372f08aff03e75c8455a9b640958c308.png "Finetune") Finetune](https://web.archive.org/web/20220816040556/http://www.finetune.com/) 是一个相对较新的用 Flash 编写的应用。这是我最喜欢的，我在我的 ZDNet 博客上做了报道。Finetune 的突出之处在于，除了标准的“艺术家电台”，它还允许用户建立特定歌曲的播放列表。最小播放列表是 45 首歌曲，每个艺术家最多可以有三首歌曲。使用自定播放列表，您可以确保只听您想要的歌曲。Finetune 也获得了积分，因为除了网页版，它还可以在 Wii 上运行，并有一个基于 Apollo 的桌面客户端。

**潘多拉**

Pandora 是这一系列应用的鼻祖，也是 Mike [离不开](https://web.archive.org/web/20220816040556/http://www.beta.techcrunch.com/2007/01/02/2007-web-20-companies-i-couldnt-live-without/)的 Web 2.0 应用之一。它是使用 OpenLaszlo 构建的，提供了列表中所有应用程序中最干净的体验。Pandora 使用音乐基因组项目来生成一系列你会喜欢的歌曲，基于你对以前歌曲的评价。你可以围绕艺术家、歌曲或专辑创建站点，还可以对 Pandora 选择的歌曲提供反馈(赞或赞)。Tech Crunch 对潘多拉星球的报道在这里。

**最后。FM**

[![last.fm](img/bc7e880321d009108866ee09995617f5.png "last.fm") last.fm](https://web.archive.org/web/20220816040556/http://www.crunchbase.com/company/last.fm) 是另一位 Web 2.0 老手，比其他人更倾向于社交。标签是 last.fm 体验的重要组成部分，除了能够收听“用户标签广播”之外，你还可以为任何歌曲添加标签，这是基于用户用特定类别标记的歌曲。last.fm 有一个单独的桌面应用程序，可以“搜索”你听的歌曲，并生成一个音乐档案，你可以与朋友分享。*点击这里查看 Tech Crunch 对 [last.fm 的报道。](https://web.archive.org/web/20220816040556/http://www.beta.techcrunch.com/tag/Last.fm/)*

**MOG**

[![MOG](img/8f7d7568259fcdfdc09a3697e47ec7ee.png "MOG") MOG](https://web.archive.org/web/20220816040556/http://mog.com/) 就是一个音乐社区。它非常以博客为中心，并围绕用户页面或“Mogs”展开。你也可以围绕你正在听的歌曲和你喜欢的艺术家来建立你的博客。这就为你建立了一个用户可以浏览和评论的个人资料。它还使用这个档案来推荐你可能喜欢的其他人或音乐。 *Tech Crunch 对 [MOG 的报道在此](https://web.archive.org/web/20220816040556/http://www.beta.techcrunch.com/tag/Mog/)。*

**radio blog club**

[![radio.blog.club](img/e3b0a8e550b7deed2590e66552580560.png "radio.blog.club") radio.blog.club](https://web.archive.org/web/20220816040556/http://www.radioblogclub.com/) 是另一个音乐服务，它根据你指定的艺术家或歌曲建立播放列表。听说的最少，不过界面不错。当你浏览网站并输入一个艺术家或歌曲时，它会为你建立一个包含 10 首歌曲的播放列表。根据我的经验，radio.blog.club 的推荐系统不是最好的，但是他们确实允许你在你的博客上嵌入他们的播放器。这似乎是最不健壮的应用程序，但仍然值得一提。

**MyStrands**

MyStrands 最初是 MusicStrands，是一个下载的桌面应用程序，可以与你当前的音乐播放器配合使用，根据你正在听的音乐提供推荐。在许多方面，它类似于 last.fm 的“Scrobbling ”,但 MyStrands 与你的移动设备相结合，似乎提供了一个更具社会性的推荐系统。通过在手机上捆绑音乐，MyStrands 比其他产品领先一点，它可以帮助你将所有的音乐收藏捆绑在一起。 *Tech Crunch 对 [MyStrands 的报道在此](https://web.archive.org/web/20220816040556/http://www.beta.techcrunch.com/tag/MusicStrands/)。*

**iLike**

![iLike](img/7515fb13118720c23281dc5a03e86899.png "iLike") [iLike](https://web.archive.org/web/20220816040556/http://www.crunchbase.com/company/ilike/) 是一个 iTunes 插件，让你的音乐库更加社会化。它还能追踪你在听什么，并推荐歌曲和有相似品味的人。它与 iTunes 界面很好地挂钩，并在你播放歌曲时推荐音乐。我听了一些相当晦涩的东西，建议是好的。他们也有一个 MySpace 的小工具，它被很好地放置在个人资料的“音乐”部分。 *Tech Crunch 关于 [iLike 的报道在这里](https://web.archive.org/web/20220816040556/http://www.beta.techcrunch.com/tag/ilike)。*

**iJigg**

iJigg 是一个类似 digg 的音乐发现服务，我玩得很开心。用户对单首歌曲进行投票，最受欢迎的歌曲会出现在首页的顶部。你不能用 iJigg 做任何“相关艺术家”，但是你可以按流派浏览，这样你就可以有针对性的发现你的音乐。iJigg 播放器也可以嵌入到其他网站上，这样你就可以和朋友分享了。随着这项服务越来越受欢迎，我认为这将是一个发现乐队的好方法。如果艾吉格在这里，Tech Crunch 的报道。
