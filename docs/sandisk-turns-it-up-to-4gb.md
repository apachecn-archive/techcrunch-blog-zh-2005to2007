# SanDisk 将其调高至 4GB

![](img/f655becf45bcc973b1cf1a8cdb6dc1f6.png)

这里没有定价的消息，但我们知道 SanDisk 已经在 3 月份发布了 4GB 的 microSDHC 卡。与以前的容量相比，这是一个巨大的进步，有人说这张卡本身就对 4GB 版本的苹果 iPhone 构成了威胁。完全是垃圾，但仍然是一个很好的尝试。不过不要急着马上去买这张卡，因为大多数支持 microSD 的手机最多只能支持 2GB。

到目前为止，[摩托罗拉摩托 Z8](https://web.archive.org/web/20210225025925/http://crunchgear.com/2007/02/12/moto-rizr-z8-mulitmedia-rizr/) 是第一款官方支持 4GB microSDHC 卡的手机。该卡将配有适配器，可以使用常规 SD 和 miniSD 卡插槽，这样你就可以将所有你在迈克尔杰克逊旁边的模糊镜头传输到你的电脑上。
