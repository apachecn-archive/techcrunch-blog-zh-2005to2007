# 如果你向未成年人出售暴力视频游戏，纽约会把你关进监狱

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/26/new-york-will-put-you-in-prison-if-you-sell-violent-video-games-to-minors/>

[![ministryoftruth.jpg](img/30362054cd1a96e0c2737560cf7a7c2c.png)](https://web.archive.org/web/20150816220136/http://old.crunchgear.com/wp-content/uploads/ministryoftruth.jpg "ministryoftruth.jpg")

电子游戏玩家(这些天我只是名义上的玩家)会在什么时候告诉他们如此关心的当选官员停止摆弄他们的爱好？在纽约这里，我们尊敬的州议会已经有了一项法案，将向未成年人出售暴力视频游戏定为 E 级重罪。(我周日刚买了 Wii 版的《生化危机 4》,发现 GameStop 的员工要求看我的身份证时，我感到很困惑；当店员告诉我他禁止两个孩子进入商店时，我真的笑了，因为他们公然让街上的人给他们买游戏。)我们这里有一条法律说，你犯了三次重罪，你可以期待在监狱里度过你的余生。所以，只想赚钱的 GameStop 员工先生，你最好每次都要身份证。否则。

该法案有望在我们无所事事的立法机关下个月恢复工作时签署成为法律。

难道没有一场真正的战争在进行吗？我不知道，我们是不是应该把精力放在别的地方？
