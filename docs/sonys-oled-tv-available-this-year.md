# 索尼的 OLED 电视今年上市

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/12/sonys-oled-tv-available-this-year/>

[![sonyoled.jpg](img/6c807a4b654a10669b555db8522a15dc.png)](https://web.archive.org/web/20131108103353/http://old.crunchgear.com/wp-content/uploads/sonyoled.jpg "sonyoled.jpg")

索尼将于今年推出其花哨的 OLED 电视，每月生产多达 1000 台。首批型号将有 11 英寸宽，0.12 英寸厚，价格将是目前平板显示器的“几倍”。未来可能是宏伟的，但看起来肯定会很昂贵。如果索尼确实在今年发布 OLED 电视，它将成为第一家这样做的制造商。不想再玩追赶[的](https://web.archive.org/web/20131108103353/http://crunchgear.com/category/ps3/)索尼？

有机发光二极管电视的实际生产将由索尼和丰田之间的合资企业负责，有点像其液晶显示器是如何与三星联合生产的。我要说的是祝你好运，兄弟，我急切地等待所有这些昂贵的新电视格式。
