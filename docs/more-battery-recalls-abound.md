# 更多的电池召回比比皆是

![](img/e7098ba7de70c37c168553c322a042d5.png)

戴尔公司今天早上宣布将召回另外 10 万块有问题的笔记本电脑电池。这一最新公告将使召回的电池总数达到 420 万块。

东芝还宣布召回 83 万块电池，这是应索尼的要求做出的。索尼电池爆炸是由于电池内残留的金属碎片。迄今为止，已有近 700 万块索尼电池因该问题被召回。
