# Vertu Aerius 蓝牙耳机

Vertu 不满足于只对手机过度收费，它还在向蓝牙耳机过度收费领域扩张。由雅各布·延森设计，这款耳机有一个按钮式接听和呼叫系统，降噪，耳钩和皮革袋。

毫无疑问，这些功能在 Vertu 手机上运行良好，但在支持蓝牙耳机的普通手机上也没有问题。老实说，在街上戴着这个就像戴着一个“抢劫我”的标志，表明你的钱比理智多。
