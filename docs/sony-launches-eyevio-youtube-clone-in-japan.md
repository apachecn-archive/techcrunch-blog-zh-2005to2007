# 索尼在日本推出 eyeVio YouTube 克隆版

[![eyevio.jpg](img/de4d6535dac58588d1f017c6a7090b5f.png)](https://web.archive.org/web/20210302030304/https://beta.techcrunch.com/wp-content/uploads/2007/04/eyevio.jpg "eyevio.jpg")

索尼又一次迟到了，它刚刚在日本推出了一个类似 YouTube 的视频分享网站，名为 eyeVio。用户上传他们的视频，可以是 MPEG-4、Flash 和 Quicktime 等格式。索尼表示，将密切监控网站上侵犯版权的视频，但他们具体计划如何做还不得而知。更糟糕的是，eyeVio 服务器已经不堪重负。太棒了。

索尼的这一最新努力只是索尼发布产品的又一个例子，该产品最终可能太小、太晚。(记住，我不知道日本用户是如何喜欢上 YouTube 的。)随身听、PSP 以及某种程度上的 PS3。在我看来，他们需要一点创意。
