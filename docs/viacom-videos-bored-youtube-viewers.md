# 维亚康姆的视频让 YouTube 观众感到厌烦

涵盖 YouTube 等在线服务的视频统计信息中心 Vidmeter 发现，只有 9%的 YouTube 视频因侵犯版权而被删除，这些视频占总浏览量的 6%。维亚康姆的视频——包括喜剧中心的内容——占这些视频的 2%。大部分被删除的维亚康姆视频是 MTV 的音乐视频，而不是乔恩·斯图尔特的滑稽。

迪士尼最受欢迎的下架视频？"南加州大学啦啦队极限楔形鞋"


关于在线视频，这告诉了我们什么？它大部分是本土的，大部分是短期的，大部分是病毒式的。广播公司可以喊狼来了，但他们肯定注意到，随着人们转向小屏幕，他们的商业模式正在迅速减弱。谁想等 6 个月才能看到《迷失》的下一集(本周，杰克和本玩了一场激动人心的跳棋游戏……本告诉杰克一个关于 J.J .艾布拉姆迷幻剂狂热梦想的小细节)，那时他们就可以看《南加州大学啦啦队长极限楔形鞋》这也是为什么我从来不想为 CG 或我工作过的任何其他网站做播客或视频广播的部分原因:没有人有时间或耐心坐在那里喃喃自语一个小时来获取一点信息。
