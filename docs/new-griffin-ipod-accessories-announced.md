# 新的格里芬 iPod 配件宣布

![](img/4dfe2d018311377f974566e6264fb710.png)

我没有 iPod，也很少后悔买了一个有创意的 Zen Vision:M——今天也不例外。格里芬已经宣布了更多的 iPod 配件，这些东西只是简单的傻瓜。首先，伊卡罗克。没错，现在你可以用你美妙的声音翻出一首歌的原始歌词，并用附带的调频发射机向全世界广播。今天，我们还有意大利语。对于 16 位立体声或 8 位单声道音频录音来说，它只不过是一个地方有两个麦克风。因此，如果你有两个 iPods，你可以用一个来演唱，让你的乐队用另一个来录制音乐。最后，格里芬还为我们提供了另一个名为 TrueCenter 的遥控坞站。不过，这款手机很独特，因为它内置了网络电台。太棒了。前进吧，消费者，消费苹果建立的市场。
