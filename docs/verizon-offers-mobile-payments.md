# 威瑞森提供移动支付

如果你是那种有赖账朋友的人，他们似乎从来没有现金，如果他们是威瑞森无线用户，你可以要回那 5 美元。正如我们的姐妹网站 Crunchgear 报道的那样[威瑞森正在增加移动支付选项](https://web.archive.org/web/20201204183229/http://crunchgear.com/2007/06/20/verizon-to-offer-mobile-payments/)，这意味着客户可以使用 Obopay Inc .的服务用他们的手机互相汇款。

这项服务将在未来几周内推出，Obopay 服务最终甚至可以让客户在零售收银台或自动银行取款机上挥动手机进行交易。预付费和计费选项将可用，但我们仍将不得不看看零售商如何快速适应新服务。
