# StyleMob:适合喜欢衣服的女性

[![stylemob1.jpg](img/fea015f20f25786b937d56c70c967d1f.png)](https://web.archive.org/web/20220818054619/http://www.stylemob.com/) 【街头风格社区】 [StyleMob](https://web.archive.org/web/20220818054619/http://www.stylemob.com/) 将于周四上市，推出一款部分热门与否、部分时尚杂志、部分 MySpace 的产品。

不管你喜不喜欢，StyleMob 鼓励用户上传照片让其他用户打分。这个概念是用户与其他用户分享他们的时尚，无论是商店服装、创造的服装还是它们的组合。

StyleMob 还涉足 MySpace 等社交网站。用户有个人资料、评论墙和朋友。StyleMob 成员可以发布建议问题(提供的例子包括“我应该继续拔眉毛吗？”或者“我应该穿什么去面试？”)到他们自己的“StyleCouncil”，这是一个由他们在 StyleMob 上找到的朋友和专家组成的私人造型师团队。

联合创始人萨沙·卡根(Sasha Cagen)用一种我作为一个男人不可能做到的方式描述 StyleMob，“我们早期的成员告诉我们，StyleMob 不仅仅是一个网站，它是一种思维模式和一种生活方式。StyleMob 是为每天早上的穿衣打扮注入创意。而是从那些品味和预算与自己相当的人那里获得灵感。”
