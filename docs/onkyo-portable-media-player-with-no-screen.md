# 不带屏幕的 Onkyo 便携式媒体播放器

安桥将 PMP 打造成一种自己动手的 PMP 方式。这款播放器配有一个 2.5 英寸的硬盘，你需要自己装进去——更不用说自己买了。然后，更好的是，它没有屏幕，所以你必须通过你的电视来播放你所有的视频。等等，我还没讲到最精彩的部分，支持的编解码器。

是的，因为外壳很大，可以容纳 2.5 英寸的硬盘，他们可以设法编写支持所有这些编解码器的程序:“MPEG-1/2/4，DivX，XviD(avi/mpg/MPEG/m2v/dat/ifo/VOB)，MP3/WMA/Ogg Vorbis/WAV 和 JPEG。”非常令人印象深刻的播放器，你既可以插入你的电视，并采取与你一起去听 MP3。它甚至配有遥控器和车载充电器，更不用说大量的电缆了。
