# Suunto t3:它什么都有

我们喜欢 Suunto——他们制造的手表很漂亮，有很多功能，而且风格独特，让他们的手表看起来不像是腕式电脑，更像是带数字的餐盘。t3 包括心率监测器、计步器以及实时卡路里燃烧读数。

黑色面大约 169.99 美元，标准 LCD 面 149.99 美元。计步器和心率豆荚单独出售。
