# 启动电源！你的 iPod Shuffle 的焦点问题

![](img/9d0599954296f61dd4c5303a182aa73a.png)

偶尔，一些看似简单的 iPod 配件会出现，这是必须的。随着多种颜色的 2G iPod Shuffle 的发布，Focal Powerbug 是对这些彩色生物的完美补充。你只需插入 Powerbug，让它充电，然后将附带的 USB 线插入 Powerbug 和你的 Shuffle，你就可以开始了。它甚至有一个小绿灯，让你知道它在充电。多么体贴。

你可以在二月底花 28 美元左右买到这个为你的 iPod 准备的很棒的小配件。
