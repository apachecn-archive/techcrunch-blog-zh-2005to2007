# 戴尔推出经济实惠的 E207WFP 宽屏显示器

E207WFP 是一款 20 英寸宽的平板显示器，响应时间为 5 毫秒，非常适合玩游戏或看电影，但更适合节俭的人，因为它的标价为 289 美元。既然这么便宜，你可以说服老板买两台，连起来组成一个 40 寸的宽屏系统，一台看盗版电影，另一台做工作。我们几年前就这样做了，我们的生产力飞速提高！
