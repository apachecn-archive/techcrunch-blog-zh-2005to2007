# MySpace 寻求移动合作伙伴

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/22/myspace-seeks-mobile-partners/>

MySpace 正在寻找移动合作伙伴，在整个欧洲推出和推广 MySpace 移动服务。

MySpace 欧洲移动副总裁让-保罗桑切斯(Jean-Paul Sanchez)最近刚刚从沃达丰(Vodafone)加盟，他正在就分销协议进行谈判，这些协议可能会包括营销合作伙伴关系。据桑切斯称，MySpace 正专注于三个关键业务领域:移动、视频和国际扩张。

MySpace 已经在美国达成协议，通过 Cingular 和 Helio 网络提供移动服务。桑切斯表示，这项移动服务在欧洲推出是为了响应该网站 1.3 亿用户的需求。
