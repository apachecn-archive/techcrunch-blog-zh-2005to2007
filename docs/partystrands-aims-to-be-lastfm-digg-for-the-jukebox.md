# Partystrands 的目标是成为点唱机的 Last.fm + Digg

[](https://web.archive.org/web/20220819215042/http://partystrands.com/)[【party strands】](https://web.archive.org/web/20220819215042/http://partystrands.com/)是一项将于下月推出的音乐服务，它将通过手机将汇总的推荐、投票和照片同步到一起。PartyStrands 由位于俄勒冈州科瓦利斯和西班牙巴塞罗那的公司 [MyStrands](https://web.archive.org/web/20220819215042/http://mystrands.com/) 创建，目标是酒吧和俱乐部。该网站将于 9 月 14 日上线。

在家中下载了 MyStrands 桌面应用程序的用户可以在参与地点通过短信登录，随身携带他们的播放列表和推荐。MyStrands 监控您在 iTunes 或 Windows Media Player 中的播放列表，并根据其他用户的类似播放列表推荐歌曲。

当许多人登录 Partystrands 时，该服务将汇总他们的播放列表和推荐，在视频投影仪上显示哪首歌曲正在播放(带有封面艺术)，并让用户通过短信对列表上的歌曲进行上下投票。用户还可以将他们在酒吧拍摄的照片发送到他们的 MyStrands 个人资料，以记录晚上的活动，并发送到会场的投影屏幕上与大家分享。至少不是视频。

MyStrands 的收入模式是与酒吧老板分享短信费用。这项服务首先面向欧洲，但将于今年秋天在美国推出。公司创始人 Gabriel Aldamiz-echevarria 报告说，MyStrands 已经从两家专门从事移动技术的欧洲风险投资公司筹集了 600 万美元。

我不确定我对此有什么看法。在酒吧的点唱机上播放音乐的部分魅力在于发现、随机机遇、排队等候和将钞票投入机器。我不知道在试图进行对话的同时上下投票选择歌曲、看到实时出现的照片(可能是你的照片)以及让音乐推荐自动反映每个人在家已经听的音乐会有多好。也许将来我们会避开某些酒吧，不是因为它们点唱机里播放的令人讨厌的音乐，而是因为它们的常客身上带有令人讨厌的特征。我知道我在家里听了很多奇怪的 Al，我不需要那让我在当地的时髦场所不那么受欢迎。

对于已经在 YouTube 和 MySpace 上生活的年轻人来说，他们认为我一个月发 100 条短信实际上是对这种媒体的忽视，Partystrands 可能会大受欢迎。在欧洲的成功并不令人惊讶，但这项服务在美国的可行性很可能取决于我们的移动活动能否赶上世界其他地区。
