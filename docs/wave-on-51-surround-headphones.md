# 5.1 声道环绕立体声耳机| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/20/wave-on-51-surround-headphones/>

![](img/fb6e491d6632e5a4df377619f93b7e15.png)

Wave-on 的 AON MDH-501H/D 耳机提供了 5.1 立体声环绕，这可能是有史以来最大的耳机。该设备配有一个支持 SPDIF/光学的杜比解码器。

说真的，看看这些东西的大小。看起来这个装置是针对游戏玩家的，但是我怀疑你戴太久会导致颈部损伤。

它将于本月晚些时候上市，售价为 149，000 韩元，折合成约 160 美元。
