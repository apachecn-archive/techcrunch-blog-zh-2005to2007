# Sky-Click:您的 Skype 网络呼叫中心

一项服务的成功有时可以根据它所产生的商业活动来衡量(想想 Ipod 的配件或者最近的 [SkyLook](https://web.archive.org/web/20220813211241/http://www.beta.techcrunch.com/tag/Skylook/) )。Skype 的成功给富有想象力的企业家带来了许多想法(Skype 经济)。

总部位于瑞士的 [Sky-Click](https://web.archive.org/web/20220813211241/http://www.sky-click.com/) 是由 [Ads-Click](https://web.archive.org/web/20220813211241/http://www.ads-click.com/) 发布的一个有趣的想法，该公司已经推出了一个类似白色标签 adwords/adsense 的平台(由 Wanadoo ISP 在欧洲使用)以及一个按通话付费的广告格式。

**Sky-Click 是一个 100%基于网络的解决方案，用于建立您的呼叫中心**，基于 Skype API，非常易于实施和使用。该服务不需要任何特定的硬件投资，并且可以很容易地集成到网站中(例如电子商务)。您只需在呼叫中心合作者的电脑上下载 Skype 客户端。在最终用户方面，你所要做的就是按下“呼叫”按钮，向呼叫中心发起呼叫(或视频呼叫)。

![](img/4275a79cc1bdad2ef57726e323b7dac2.png)

这个呼叫中心拥有专业解决方案的所有重要功能:时间管理、合作者可用性、呼叫调度、等待背景音乐、反馈管理(下面是一些独家截图)…

它是为任何类型的公司设计的，不管是大公司还是小公司。虽然我不确定大公司是否会率先采用这种解决方案，但我相信 Sky-Click 非常适合那些寻求在不影响客户服务政策的情况下降低运营成本的中小型企业。

当然，对于终端用户和公司来说，通话是免费的，服务比市场上的任何解决方案都要便宜:大约 10 美元/月/合作者。

Sky-Click 还可以用于其他目的，如远程培训或销售队伍管理。

Pascal Rossini(首席执行官)和 Cyril Lamblard(首席运营官)告诉我们，Sky-Click 应该在几周内开放，在此期间，您可以订阅他们的临时主页[以获得发布通知，甚至可以写信给 sky-click.com 信息中心。](https://web.archive.org/web/20220813211241/http://www.sky-click.com/)
