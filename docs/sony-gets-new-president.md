# 索尼新任总裁

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/30/sony-gets-new-president/>

![](img/58d342e08ebe07511cf33bfa79992e79.png)

索尼电脑娱乐美国负责人 Kaz Hirai 将取代 Ken Kutaragi 担任索尼电脑娱乐总裁。Kutaragi 被提升为 SCE 主席，SCEA 执行副总裁 Jack Tretton 将接替 Hirai 的位置。

> Kutaragi 将继续作为首席执行官监管整个 SCE 集团，并将充分行使他的权力来加强和进一步加快 PlayStation 业务的发展

*–索尼*

虽然索尼没有说明为什么决定推广 Kutaragi，但这很可能是因为 PlayStation 3 的发布出现了问题。在 Kutaragi 的领导下，北美收到了数量非常有限的 PS3 游戏机，一旦游戏机发布，它就只有负面新闻。索尼需要做出一些改变，希望我们很快就能看到。
