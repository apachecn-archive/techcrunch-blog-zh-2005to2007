# Zillow 添加用户生成的家庭信息

[![](img/c866d0b814d61431c4a515e91bbfaafa.png)](https://web.archive.org/web/20221003214230/http://www.zillow.com/) 西雅图 [Zillow](https://web.archive.org/web/20221003214230/http://www.zillow.com/) 是一个网站，允许人们获得美国 6800 万套住房的信息(如估计价值、面积、卧室和浴室数量等。).周三上午，他们将发布新功能，允许用户向 Zillow 的国家数据库提供关于自己房屋的最新详细信息。

该业主信息将与 Zillow 收集的信息一起出现在一个关于物业的信息页面上(见下面的屏幕截图)。如果一些关于住宅的信息不正确或过时，用户将有机会改正。还可以包括估价的其他信息，Zillow 将使用这些信息来确定房屋价值的更新估计。在添加任何信息之前，用户必须完成一个验证过程来声明一个家。

我希望人们在出售房屋时积极更新 Zillow 信息。

Zillow 不断创新，执行良好。迄今为止，他们已经筹集了 5700 万美元的巨额风险资金，并深深融入了雅虎房地产。我们之前在 Zillow 上的帖子是这里的。
