# 赞助商休息

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/04/27/thank-you-sponsors-and-readers/>

我们想花点时间感谢帮助我们保持 TechCrunch 运转的赞助商和读者…

**[Voxbone](https://web.archive.org/web/20160310062436/http://www.voxbone.com/)**–一家 VoIP 运营商，提供本地电话号码和全球免费电话号码的集中接入。他们有自己的网络，完全用于由思科路由器和交换机支持的语音流量。通过他们的网络，您可以为客户分配电话号码块，连接 VoIP 和 PSTN 电话线路之间的呼叫，共享电话流量容量，并使用他们的 API 创建您自己的解决方案。只需一个固定的月费。

——这就是我所说的地理社交网络。Plazes 可以帮助您的朋友在电脑或手机上查看您的位置和活动。你可以用它来与朋友和家人保持联系，或者寻找与你有相同位置或兴趣的 Plazes 用户。Plazes 还可以将你连接到你去的地方，让你将图片和媒体保存到你去的地方。

**[导管](https://web.archive.org/web/20160310062436/http://techcrunch.conduit.com/)**–个性化工具栏。Conduit 允许发布者(或者任何人)创建专门的工具栏，提供您和您的社区需要的所有工具。TechCrunch 有自己的工具栏(下载[这里](https://web.archive.org/web/20160310062436/http://techcrunch.ourtoolbar.com/exe))，可以让你搜索、了解整个 CrunchNetwork 的最新信息，甚至可以收听最新的 TalkCrunch 播客。如果这对你来说还不够好，那么[构建你自己的](https://web.archive.org/web/20160310062436/http://www.conduit.com/?vararrivalid=techcrunch&ref_id=techcrunch)。

——定制个人页面。Pageflakes 让您创建自己的个性化门户网站。您可以通过选择您感兴趣的内容来轻松定制您的页面，或者通过从基于 API 的各种小部件中进行选择来对页面进行微调。最近，我一直在用他们的[五子棋薄片](https://web.archive.org/web/20160310062436/http://www.pageflakes.com/Community/Content/Flakes.aspx?moduleKey=5919)。通过将页面公开来与朋友共享页面非常简单。不需要登录。
 **[【Edgeio】](https://web.archive.org/web/20160310062436/http://www.edgeio.com/view/marketplaces/)**——分类列表和内容在印刷中总是放在一起，因为广告商希望他们的信息到达出版商的读者手中。Edgeio 让广告商在网络上获得同样的曝光率，通过出版商联合列表来迎合广告商的受众。TechCrunch 自己的 CrunchBoard 由 Edgeio 的最新产品 Edgeio Marketplaces 提供支持，该产品允许出版商通过在自己的网站和 Edgeio 的整个网络上刊登广告来赚钱。
