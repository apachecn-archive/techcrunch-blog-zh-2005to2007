# Trulia 和 Mashery 推出房地产搜索 API 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/20/trulia-and-mashery-launch-real-estate-search-api/>

[![](img/f81d6b069faf669ebf9fc34f11b0f905.png)](https://web.archive.org/web/20160526210108/http://trulia.com/) 房地产搜索公司 [Trulia](https://web.archive.org/web/20160526210108/http://trulia.com/) [今天早上宣布其新 API](https://web.archive.org/web/20160526210108/http://www.truliablog.com/?p=124) 的可用性，以及由该 API 实现的两个有趣的混搭示例。外部开发商现在可以访问该公司的房地产数据和聚合用户搜索数据。

为了展示 API 可能实现的事物类型，Trulia 团队提供了两个他们自己构建的有趣的 mashups。Plotornot (一个关于 HotorNot 的游戏)关联了各种人口统计信息，如美国任何一个州的性别、婚姻和收入数据。TruliaHolic (鉴于相似性，可能是 Alexaholic 上的一个游戏)提供了美国任何城市或县的平均标价和搜索流行度差异的可视化。新 web 服务在房地产领域的使用非常热门，所以我希望我们会在其他网站上看到新 API 的许多有趣的用途。

关于今天的公告，最值得注意的一件事是 Trulia 在 [Mashery](https://web.archive.org/web/20160526210108/http://mashery.com/) 的帮助下创建了它的 API，我们在 11 月曾在这里介绍过 API 管理服务[。上周，Mashery 帮助推出了流量分析服务的 API Compete，该公司正在与其他三家公司合作开发即将发布的 API。](https://web.archive.org/web/20160526210108/http://www.techcrunch.com/2006/11/06/mashery-api-management-service-is-open-for-business/)

一个潜在的缓解因素是 Trulia API 只能根据其使用条款用于非商业用途。虽然像这样的术语经常被认为是开放的解释，但我很失望看到它。尽管这可能只是这个项目的第一步；该公司可能会提高其通话限制(现在每天只有 1000 次)，并在未来向商业用户开放。

Trulia 在房地产搜索领域的主要竞争对手包括 Zillow，Zillow 在 10 月份发布了他们自己的 APIs)，crowd sourced(众包)公司(T2)的 My-Currency(我的货币)和许多其他公司。点击查看我们之前对 Trulia [的报道。](https://web.archive.org/web/20160526210108/http://www.techcrunch.com/tag/trulia)
