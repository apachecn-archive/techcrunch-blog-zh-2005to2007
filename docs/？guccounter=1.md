# Soundpedia:一个体面的潘多拉替代品

[![soundpedia.jpg](img/7a13efcacf1f4a0be8d7c519c3e93876.png) Soundpedia](https://web.archive.org/web/20210211003319/http://soundpedia.com/) 是一个合法的音乐共享社区，提供音乐流媒体服务，为美国以外不能再使用 Pandora 的用户提供一个像样的 Pandora 替代品。

Soundpedia 带有一系列常见的 Web 2.0 音乐共享功能。该网站帮助用户发现新的音乐并与朋友分享。Soundpedia 既支持交互式用户播放列表，也支持从音乐播放器上传现有播放列表设置。用户可以与朋友聊天，发布评价，对艺术家发表评论，或者创建自己的音频/视频博客。

在不同的时代，人们很容易将 Soundpedia 视为又一个进入拥挤市场的音乐共享网站。然而，Soundpedia 满足了一个未满足的需求；这是比上次更潘多拉。调频和美国以外的世界已被封锁从潘多拉几个月。Soundpedia 与 Last.fm 不同。当然，Last.fm 的音乐推荐可以令人愉快，但在其他时候，你只想听某个特定的艺术家或专辑。Soundpedia 提供了全面的服务，这可能不是最美观的服务，但它提供了音乐。音乐可以按流派、艺术家或专辑播放，Soundpedia 的库非常丰富。我确实找到了那张没有完全收录的奇怪的专辑，但是歌曲被列在了 30 秒的剪辑中。创始人之一、纽约人格雷戈里·古墨告诉我，Soundpedia 正在定期增加播放列表；大多数用户不会注意到任何缺陷。

作为一个非美国人，我错过了潘多拉的功能；Soundpedia 刚刚点击了我的谷歌书签列表。
