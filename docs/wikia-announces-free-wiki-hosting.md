# 维基宣布免费维基托管

[![](img/c6e7db53b7fbfa9be7439a9d7aca8504.png) ](https://web.archive.org/web/20230130063140/http://www.wikia.com/) [维基百科](https://web.archive.org/web/20230130063140/http://www.wikia.com/)创始人吉米·多纳尔·威尔士信奉“人人免费”这就是为什么该公司今天推出了 [OpenServing](https://web.archive.org/web/20230130063140/http://www.openserving.com/) ，这是一项免费向任何维基开发者提供完整的虚拟主机支持的服务。

OpenServing 允许任何人建立和维护他们自己的合作网站。威尔士今天在电话中告诉 TechCrunch，OpenServing 也打算超越维基托管，包括自由软件、自由应用等。“基本上，任何东西都可以进入自由文化的模式，”他说。

Wikia 将通过互联网提供免费软件、带宽、存储、计算能力和内容。更重要的是，Wikia 将把 100%的广告库存和收入赠送给任何想要注册的博客/网站所有者。唯一的规定是开发者链接到 Wikia 的主页以增加广告收入。

威尔士说:“我们主要寻找的是它们将流量带回维基，但我们不只是考虑任何返回主页的链接，而是一个内容相关的链接。”“我们不只是要求一个广告，而是一个与我们的维基相关的文章和东西的链接。”

如果使用 OpenService 的网站在他们的维基上销售广告，他们没有责任与维基分享收入。

“基本上他们可以在自己的网站上自由销售广告，”威尔士解释道。“我们可能会在某个时候提供代理这些广告的服务，但这不是我们现在的模式。我们的模式基本上是说‘你保留广告收入，我们真的不担心这个。’"

这份声明至少解决了维基百科本月早些时候购买 ArmchairGM 背后的一些谜团。ArmchairGM 是 OpenServing 将提供的第一个自由授权软件包。同样值得注意的是，就在几周前，Wikia [从亚马逊](https://web.archive.org/web/20230130063140/http://techcrunch.com/2006/12/06/wikia-gets-an-undisclosed-amount-of-funding-from-amazon/)[获得了](https://web.archive.org/web/20230130063140/http://techcrunch.com/tag/Amazon/)一笔数目不详的资金，这可能是 Wikia 有能力做到这一点的一个重要因素。
