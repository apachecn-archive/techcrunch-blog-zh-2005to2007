# 三星 i550 细节浮现，仍不可用

![big.jpg](img/46292456dba42a21f1d39bba65b44ff6.png)

我已经有几天没有提到任何新的三星手机了，但今天发生了变化。三星的最新产品正在网上流传，它的功能令人印象深刻。据传 i550 支持 GPS 并运行 Symbian S60，这是对 Windows Mobile 操作系统的一个很好的改变。它被吹捧为智能手机，我倾向于相信这个标签是正确的。据称，它将运行在 3G 网络上，拥有 HSDPA、320 万像素摄像头、2.6 英寸显示屏、调频收音机、128MB 内存、microSD 和 microUSB。键盘的布局暗示了专用的 GPS、MP3 播放器和互联网按钮。不幸的是，官方没有透露 i550 的上市时间和市场。
