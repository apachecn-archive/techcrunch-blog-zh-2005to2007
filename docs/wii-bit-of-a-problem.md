# 有点问题

更多的 Wii 问题来为 PlayStation 3 降温。当使用 WiiConnect24 下载游戏机的初始更新时，这个特定的错误开始生效。IGN 报告说，用户遇到了几个错误代码，但代码 110213 和 32002 似乎是常见的。任天堂已经承认了这个问题，如果你不幸遇到这个问题，你有几个选择。

如果你的主机上没有任何重要的保存数据，你很幸运。你所要做的就是联系任天堂，它会在 3-4 天内给你送来全新的任天堂 Wii。然而，如果你确实在 Wii 上保存了重要的数据，你将不得不等待任天堂的发货标签，并将你的系统送去维修。第二种选择会让你在大约 8-12 天内没有 Wii，而且你还必须等待游戏机被寄回给你。我的建议是:忘记你出于某种原因决定在*超级马里奥 64* 上收集 70 颗星星的事实，并获得一个替换主机，如果你这样做，问题不会再次出现的可能性更大。
