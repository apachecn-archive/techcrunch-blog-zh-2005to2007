# 擎天柱迷你三键盘存在

![optimus_mini-ondesk.jpg](img/721476f0cd14149b0a4e6a01ac326371.png)
Thinkgeek 拿到了 Optimus Mini 三键盘的最终量产样品；只有三个键的键盘。尽管如此，每个按键上都有一个有机发光二极管显示屏，你可以用它来显示你想要的任何东西。例如天气，你有多少空闲内存，你当前的 CPU 使用率是多少，或者一个 3FPS 的网络摄像头流。目前只有 Windows 兼容，但 Mac 和 Linux 支持正在开发中。

9 月中旬发货，售价 159 美元。
