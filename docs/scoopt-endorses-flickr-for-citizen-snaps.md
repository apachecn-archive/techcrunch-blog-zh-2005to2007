# Scoopt 支持 Flickr 拍摄公民照片

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/11/scoopt-endorses-flickr-for-citizen-snaps/>

自称“公民新闻图片社”的 scoopt 现在将允许 Flickr 成员用关键词“Scoopt”标记他们的图片，使 Scoopt 能够将标记的图片推送给他们的媒体买家。Scoopt 还将鼓励买家使用 Flickr 作为商业图片资源。

Scoopt 创始人凯尔·麦克雷(Kyle MacRae)表示，这项新功能很有意义，因为“没有简单的方法将 Flickr 成员与买家联系起来”，而且“Flickr 成员不允许为他们的内容做商业广告”。Scoopt 的举动是非官方的，与 Flickr 没有正式关系。

销售收入由 Scoopt 和摄影师五五分成。摄影师总是保留全部版权。加入 Scoopt 是免费的。

Scoopt 拥有 12000 名会员，总部设在苏格兰。
