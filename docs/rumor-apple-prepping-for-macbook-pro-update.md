# 谣言:苹果准备升级 Macbook Pro

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/01/rumor-apple-prepping-for-macbook-pro-update/>

![macbook-pro.png](img/1b09e049804730f4ac5ceaa30ff3658f.png)

所有这些关于苹果的谣言都需要停止，因为它让我头疼，或者可能是因为我宿醉未醒。我不太确定是哪一个引起了我的恶心感，但是有另一个谣言正在流传，你可能已经听说过一两次了。Macbook Pro 系列什么时候更新？Macbook 上个月刚刚升级，MBP 已经有很长一段时间没出什么问题了。但是这个谣言可能站得住脚，因为有一些确凿的事实支持它。

**更新**:请继续阅读这个谣言的更新。

一位业内人士透露，目前 MBP 的产量已经下降，苹果在 T4 的直销店的开业让人们产生了疯狂的想法。还有一封主题为“新 SKU 的设置信息”的电子邮件发送给了美国分销商，其中包含了一系列更改。显然，15 英寸和 17 英寸型号的装运重量变化很大。15 英寸的减轻了 0.4 磅，而 17 英寸的型号从之前的 11.6 磅减轻了 11 磅。当然，这封电子邮件还没有得到证实，据报道，第二封电子邮件立即被发出，声称这是一个“错误”。我们不想做任何导致苹果股票暴跌的事情，因为那真的很糟糕。

所以现在的问题是，我们会在 WWDC 看到采用英特尔最新芯片组、Santa Rosa 和 LED 显示屏的 MBP 升级版吗？也许他们最终会干掉火线 800，搬到 eSATA？现在谁知道呢，但不到两周我们就会知道了。

更新:更新可能会比 WWDC 来得快。返校计划将于 6 月 5 日重新启动，在促销活动开始一周后更新产品系列没有太大意义，对吗？
