# Wii-DVD 版确认在日本发售

显然，日本比美国或欧洲更需要一个集成了 Wii 的小型 DVD 播放器，但任天堂日本公司刚刚证实，他们将得到一个具有 DVD 功能的“增强型”Wii。

> 目前有计划在日本发布一个具有 DVD 播放功能的 Wii 版本。目前还没有将这种模式引入欧洲或美国的计划。

日本的空间真的如此紧张，以至于一个普通的 DVD 播放器太大了，他们需要一个放在他们的 Wii 里吗？话说回来，也许有了 DVD 播放功能，改装者可以将 Wii 游戏刻录到 DVD 上。
