# 使用 HopStop 规划您的公共交通路线

谷歌可能是[环保意识强的](https://web.archive.org/web/20211129155448/http://www.beta.techcrunch.com/2007/05/14/who-cares-the-most-about-the-environment/)，但他们肯定不会帮助我们其他人变得更环保。目前，谷歌的公共交通地图工具 [Google Transit](https://web.archive.org/web/20211129155448/http://www.google.com/transit) 只处理橘子郡和明尼苏达州德卢斯等公共交通的堡垒，让我们这些在旧金山、纽约和其他任何地方的人盯着公交地图。

然而，Transit planner [HopStop](https://web.archive.org/web/20211129155448/http://hopstop.com/) 负责这些较大的地铁，覆盖波士顿、芝加哥、旧金山、纽约、华盛顿特区和新泽西州的几个城市。对于每个城市，HopStop 将绘制路线，并估计两个或更多地址之间步行和公共交通(公交车/地铁)的旅行时间。它还附带了一些其他非常有用的功能。你可以选择哪种旅行方式你更喜欢，步行，公共汽车，地铁，或三者的结合。此外，由于你并不总是知道你的目的地的街道地址，你可以从地图上选择。

除了指南，HopStop 还有一些新兴的本地指南和商业指南。而且如果你不喜欢界面，还有一个[开发者 API](https://web.archive.org/web/20211129155448/http://hopstop.com/freeapi/) 。

纽约和伦敦的 TechCrunch 读者也应该看看 Greg Brail 的[交通地图](https://web.archive.org/web/20211129155448/http://www.brail.org/transit/)。
