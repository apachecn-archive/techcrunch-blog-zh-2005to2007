# Nirvanix 将挑战亚马逊 S3

 [](https://web.archive.org/web/20211026153308/http://www.crunchbase.com/company/nirvanix) 期待在未来几周内推出新的圣地亚哥 [Nirvanix](https://web.archive.org/web/20211026153308/http://www.crunchbase.com/company/nirvanix) 。它旨在与亚马逊广受欢迎的 S3 存储网络服务竞争，并为网络开发者提供另一种在线存储选择。

该公司隶属于在线存储初创公司 [MediaMax](https://web.archive.org/web/20211026153308/http://www.mediamax.com/) (又名 Streamload)，也应该会在不久的将来宣布一轮 1200 万美元的融资。目前还不清楚 Nirvanix 和 MediaMax 到底是什么关系，该公司目前对这种关系的具体细节保密。不过，显然已经有了技术交换，MediaMax 现在使用 Nirvanix 服务来提供其存储产品的后端。此外，MediaMax 前首席执行官帕特里克·哈尔(Patrick Harr)现在在 Nirvanix 主持工作(MediaMax 创始人史蒂夫·艾弗森(Steve Iverson)重新担任了 MediaMax 的首席执行官)。

虽然该公司对资金以及它与 MediaMax 的确切关系保持沉默，但他们表示，他们将填补 S3 的一些已知漏洞。希望他们会提供服务水平协议(亚马逊没有)。

与此同时，MediaMax 一直在努力完成今年夏天早些时候困扰他们的[地狱般的技术过渡](https://web.archive.org/web/20211026153308/http://www.beta.techcrunch.com/2007/07/26/streamload-upgrade-goes-very-wrong-some-users-revolt/)(我猜这与 Nirvanix 的转换有关)。他们已经把他们的公司博客搬到了一个新的网站上，并且公开地和用户谈论他们面临的一些问题。

**披露:**我是 Omnidrive 的投资人，Omnidrive 也在在线存储领域，可以认为是 MediaMax 和 Nirvanix 的竞争对手。
