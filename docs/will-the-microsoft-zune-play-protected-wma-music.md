# 微软 Zune 能播放受保护的 WMA 音乐吗？

EFF 认为 Zune 不会播放受保护的 WMA 和 WMV 内容，这让我们很困惑。根据昨天的新闻稿，微软没有提到任何受保护的 WMA 播放，但确实提到了不受保护的 WMA 播放。

这有道理吗？如果没有任何形式的 WMA 保护，微软怎么能拥有音乐商店和 Zune 市场呢？既然微软表示，商店将提供购买音乐和订阅选项，那么如果没有受保护的 WMA，他们将如何保护他们的订阅音乐呢？

我们有根据的猜测是，Zune 将支持 WMA 和 PlaysForSure，这是 Napster、Rhapsody 和 Yahoo！无限制。他们不这样做是没有道理的。
