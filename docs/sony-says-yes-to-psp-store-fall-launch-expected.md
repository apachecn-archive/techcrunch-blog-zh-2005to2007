# 索尼同意 PSP 商店，预计秋季上市

[![pspsstore.jpg](img/daa1f4952c4d34c4ba4c7d99cec6f1ce.png)](https://web.archive.org/web/20210227005522/https://beta.techcrunch.com/wp-content/uploads/2007/05/pspsstore.jpg "pspsstore.jpg")

因此，索尼宣布它将推出一个 [PSP](https://web.archive.org/web/20210227005522/http://crunchgear.com/2007/05/04/top-five-psp-mods-that-dont-involve-throwing-it-out/) 商店，打算在今年秋天的某个时候开业。话又说回来，自从两年前苹果推出掌上电脑以来，我们就基本上认为这种情况会发生。实际上，这算不上什么声明，只是 SCEA 总统的一次性评论，说它现在应该已经到了，而且正在路上。没有关于你能买什么(大概是游戏和游戏相关的东西)或者多少钱的细节。

微软在 Xbox Live Marketplace 方面做得相当不错，所以让我们看看索尼是否会像过去几个月那样把事情搞砸。
