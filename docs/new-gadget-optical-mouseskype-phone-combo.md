# 新玩意:光电鼠标/Skype 电话组合

我只是偶尔钻研一下小工具领域，但是这个太好了，不能错过。

索尼宣布推出一款新的 VAIO 个人电脑外设，它既可以作为光学鼠标，也可以作为网络电话。这款蛤壳式鼠标将获得 Skype 认证，打开后它将成为一款网络电话，可以通过 Skype 拨打和接听世界各地的电话。功能包括:鼠标滚轮控制静音，音量和数字选择控制，回声消除技术和免提模式。

我觉得这个很棒，我想给我的 Mac 也配一个。
