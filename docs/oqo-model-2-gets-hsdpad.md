# OQO 模型 2 获得 HSDPA d 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/07/oqo-model-2-gets-hsdpad/>

![](img/0cfce725bbb7a7aa02661807de5bcc93.png)

在 Flickr 上，用户 tnkgrl 受够了她的 OQO Model 2 附带的 EVDO 卡。那么，当一个女朋友想要她的生活中有一个小 HSDPA 时，她该怎么做呢？把她的电脑改装一下，就这样。Tnkgrl 一意孤行，把 EVDO 像一个松散的倒刺一样扯了下来，换上了一张全新的 HSDPA 卡。

看起来不像花了*太多的工作*，但对于那些不喜欢电线或使你的保修无效的人来说，你可能想远离这个可怕的照片集。
