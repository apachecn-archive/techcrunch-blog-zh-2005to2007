# Xbox Live 功能有 TGS 和 XO6

![](img/90cfc287790de4ef0a6c2aa69031e7b5.png)

从 9 月 20 日开始，Xbox Live 为我们这些负担不起飞往日本参加东京游戏展的游戏迷提供了一个答案。Xbox Live 社区将能够下载大量的视频游戏演示、视频和更多 TGS 06 展会上展示的东西。

这还不是全部:Xbox Live 还将让用户访问在西班牙巴塞罗那举行的 X06。我们之前和 *E3 有过这样的机会:把它带回家*，这真是一个伟大的想法。

没有什么比听你的朋友告诉你他是如何得到《合金装备 4》的，以及它是如何在数十亿年后才会问世更糟糕的了。可下载的内容将涵盖九天的时间，我们将能够从 100 多种不同的内容中选择下载。哎呀，这是相当多的 360 硬盘空间。你也可以通过传统方式在 [Gamespot](www.gamespot.com) 查看信息。
