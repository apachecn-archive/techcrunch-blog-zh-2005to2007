# 安全地通过和离开

对于移动设备来说，确保信息安全越来越受到关注，尤其是当事物变得越来越互联时。因此，安全计算公司(Secure Computing Corp .)发布了一种新的软件认证工具，可以为移动设备和笔记本电脑生成一次性密码，这是一个受欢迎的消息。

《信息周刊》报道称，这种名为 Safeword MobilePass 的新解决方案旨在允许用户通过移动设备上生成的一次性密码访问 VPN、Citrix 和 Microsoft Outlook 以及其他应用程序。它可以在几个移动平台上运行，包括黑莓、Palm OS 和支持 J2ME 的手机。

该应用程序工具提供双因素身份验证，并为访问可信企业和消费者应用程序的用户建立身份。
