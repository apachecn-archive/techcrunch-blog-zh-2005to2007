# 雅虎推出个性化搜索

 [](https://web.archive.org/web/20221207201439/http://builder.search.yahoo.com/) 雅虎今天发布了[雅虎搜索构建器](https://web.archive.org/web/20221207201439/http://builder.search.yahoo.com/)的[公告](https://web.archive.org/web/20221207201439/http://www.ysearchblog.com/archives/000339.html)，这是一个可定制的搜索引擎工具，与 [Eurekster](https://web.archive.org/web/20221207201439/http://www.eurekster.com/) 和 [Rollyo](https://web.archive.org/web/20221207201439/http://www.rollyo.com/) 等个人搜索初创公司展开了一场激烈的竞争。

可以创建一个自定义搜索引擎，只搜索几个定义的站点，和/或整个雅虎搜索索引和雅虎新闻。搜索引擎可以被进一步定制以包括特定的搜索词以及用户键入的任何内容，排除某些关键词等。一旦完成，搜索引擎可以通过代码片段直接集成到网站中。

这与 Rollyo(见我们的帖子[此处](https://web.archive.org/web/20221207201439/http://www.beta.techcrunch.com/tag/rollyo))和 Eurekster 的 Swicki 产品(见我们的帖子[此处](https://web.archive.org/web/20221207201439/http://www.beta.techcrunch.com/tag/eurekster))直接竞争，我们用它们在 TechCrunch 上进行搜索——见右边栏。

像 Eurekster 一样，雅虎让搜索引擎创建者能够个性化结果页面，查看搜索统计数据，并包括一个常用搜索术语的标签云(这个标签云大大增加了搜索引擎的使用)。雅虎对与创作者分享广告收入只字未提——Eurekster 现在正在这么做。
