# Nimbuzz Talk 推出，连接手机和电脑

[![nimbuzz_logo.jpg](img/2f286302ba0e74399ded1c838ce45415.png)](https://web.archive.org/web/20220818061439/http://nimbuzz.com/)mobile runch[在](https://web.archive.org/web/20220818061439/http://mobilecrunch.com/2007/01/16/nimbuzz-talk-goes-live/trackback/)宣布了 [Nimbuzz Talk](https://web.archive.org/web/20220818061439/http://nimbuzz.com/) 的测试版，这是一款连接手机和 PC 用户的移动/PC 应用。该应用程序支持移动设备、PC 以及移动到 PC 或 PC 到移动设备之间的语音、在线状态、电话会议和 SMS 消息传递。它目前可以和 MSN、Google Talk 以及他们自己的 Nimbuzz 聊天客户端一起工作。这让我想起了另一个移动/PC 版本， [Talkster](https://web.archive.org/web/20220818061439/http://www.beta.techcrunch.com/2006/12/10/talkster-launches-presence-based-service-for-the-enterprise/) 。

Nimbuzz 用户之间的移动到移动电话按本地费率计费，这使得一些廉价甚至免费的国际电话成为可能。Nimbuzz 的在线支持允许用户通过从好友列表中选择一个联系人来拨打电话。然后，该服务通过 800 号码连接用户。MobileCrunch 的 Oliver Starr 表示，手机对手机的质量“非常好”，但是手机对电脑的质量“还有待提高”。

这项新服务还有一个“蜂音”功能，让离线联系人知道你在试图联系他们。在他们的下一个版本中，这将导致 Nimbuzz 在联系人的电话上醒来。
