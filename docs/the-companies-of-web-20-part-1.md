罗斯·梅菲尔德谈到了第一个所见即所得的维基编辑器。然而，他说这不仅仅是维基的一个工具。它是“网络开源同步编辑器”,他的愿景是它将被用于维基之外的许多网络应用。想免费试用[社交文](https://web.archive.org/web/20221231064616/http://www.socialtext.com/)？在 socialtext 上提到 web2con，就可以获得一年的免费五用户 wiki。

戴夫·佩尔介绍了 Rollyo，这是一款自己开发的搜索引擎。

你可以创建一个小型搜索引擎，只搜索那些你信任或认为有相关内容的网站，然后根据个人搜索进行搜索。他使用了一个非常引人注目的旅游搜索示例——只搜索 fodors、travelpost 和 frommers。保存的搜索可以是私有的，也可以是公共的并与其他人共享。

大卫·杨谈到了 [Joyent](https://web.archive.org/web/20221231064616/http://www.joyent.com/) ，这是一个针对小团体和公司的引人注目的网络套件，包括邮件、日历、联系人、文件等。，并允许开发人员在他们的数据上混合系统。大量的标签和“智能过滤器”。开放 API 以允许第三方应用。在这里游览[。](https://web.archive.org/web/20221231064616/http://joyent.com/experience/)

Rajat Paharia 展示了他超酷的 flash 平台 [BunchBall](https://web.archive.org/web/20221231064616/http://www.bunchball.com/) 。Rajat 也很友好，在当天早些时候给了我一个个人介绍。Rajat 谈到了开发人员如何同时需要基础设施和分发来发布应用程序。BunchBall 提供了这两者——一个灵活的 flash 平台(某些应用程序需要 Flash 8)以及开放的 API，新的第三方应用程序会自动跨平台分发。

目前的应用包括许多游戏和照片分享。Rajar 还表示，Metaliq 正在开发一款多用户德州扑克游戏，即将发布。

看看这个。与传言相反，Rajat 在给我演示时并没有在井字游戏中打败我。他撒谎。:-)

肯·里德谈论了他的新公司“真实旅行”。它是集中式的:-(带有一些非常好的标签和搜索/查找功能的用户内容:-)。

这个想法是利用用户内容和社交网络为旅游购物者创造个性化的体验，并为旅游业营销人员提供更有效的场所。因此，有望打破在线旅游业目前所处的死亡螺旋:竞相降低价格。

Satish Dharmara 对 Zimbra 做了一个绝对出色的介绍，尽管老实说 Zimbra 太酷了，充满了 AJax 的魅力，他可以站在那里喋喋不休，观众仍然会欢呼。

Zimbra 是一款“开源企业级可扩展协作服务器，具有智能在线备份和单邮箱恢复功能。它有分级存储管理”。这是什么意思？你不能从 Zimbra 网站上运行它，但是你可以把它安装在你自己的服务器上。这是它应该有的前景。
