# 尼尔森无线推出移动矢量

尼尔森媒体研究公司可能很快会测量人们在手机上使用的内容类型。mocoNews 报道称，尼尔森无线部门已推出其首款产品，名为 Mobile Vector，该服务旨在跟踪移动内容，如移动互联网和移动视频，同时还可以量化这对消费者整体媒体行为的影响。

这是为了配合尼尔森的随时随地媒体测量计划，该计划旨在跟踪所有视频平台上的电视使用情况，包括第三个屏幕上的电视使用情况。此外，这项服务将与尼尔森的其他产品一起工作，以跟踪手机的其他方面，如铃声销售。
