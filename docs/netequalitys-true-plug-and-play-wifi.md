# Netequality 的真正即插即用 WiFi | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/19/netequalitys-true-plug-and-play-wifi/>

![](img/525dec5fd2a6992118c256303bf65154.png)

当一家非营利公司把一个无线路由器放在一个插在墙上的外壳里时，你会得到什么？有人即插即用 WiFi 吗？Netequality 的新盒子是为负担不起互联网的低收入家庭设计的。它的工作原理是将 Meraki 无线网状路由器集成到一个更简单的安装介质中，成本仅为 79 美元，并将 DSL 或有线互联网插入其中一个盒子。一旦一个盒子有了互联网连接，在需要另一个上行链路之前，多达四个其他盒子可以从这个单独的连接中脱离。

这听起来无疑是将互联网接入偏远地区和低收入家庭的最佳方式。想在你的社区开始一个类似的项目吗？前往 [Netequality 的网站](https://web.archive.org/web/20141017014916/http://www.netequality.com/)了解更多信息。
