### 迈克尔·阿灵顿

![](img/bbf0ab1aeccadcc76a3eb9c47204e924.png)

这是沃尔玛的全新视频网站在火狐浏览器中的样子。它在 Internet Explorer 中运行良好，但是 CSS 在 Firefox 中加载不正确。一位设计师的观点是:“有人真的搞砸了。我可以在 30 秒内修好它。他们在发布之前在 Firefox 中测试过吗？”
