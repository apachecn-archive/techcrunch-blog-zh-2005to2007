# Nyko 为 PSP  设计的新充电器手柄

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/27/nykos-new-charger-grips-for-psp/>

![](img/e4b6e4651ef80082f22203ef1dddd57e.png)

如果你和我一样，在看电影时拿着你的 PSP 有严重的问题，你可能想看看 Nyko 的新充电器手柄。除了在使用 PSP 时提供更舒适的体验外，充电器手柄还可以为您提供 1.5 倍的额外游戏时间，以及在观看 UMDs 时保持 PSP 稳定而不会碰到保险杠按钮的能力。Nyko 还表示，充电器手柄可以缓解紧张，所以如果你花几个小时玩*铁拳:黑暗复活*，Nyko 的充电器手柄可能会符合你的胃口。
