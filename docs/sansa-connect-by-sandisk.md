# SanDisk 的 sansa Connect–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/12/sansa-connect-by-sandisk/>

Sansa Connect 是出现在 SanDisk 花名册上的最新 MP3 播放器。该设备在今年的 CES 上引起了不小的轰动，因为它创新性地使用了 WiFi，而且价格实惠。距离这款设备首次发布已经整整三个月了，但我现在拥有了一台，并有机会对它进行充分的测试。那么等待值得吗？

简而言之，是的。每隔几年，科技行业都会遇到一种能引起某种范式转变的设备。Sansa Connect 是数字音频播放器的转变。connect 采用 4GB NAND 内存，装在一个与 iPod Nano 大小相当的纤薄小盒子中。然而，它在几乎所有其他方面都是无可比拟的。

Nano 保持了 iPods 多年来的陈旧功能，Connect 带来了一系列新的独特想法，其中最值得注意的是它对 WiFi 的使用。与微软 Zune 不同，Connect 利用其 WiFi 的方式实际上对任何人都有用。该设备与雅虎协同工作。音乐，无需线缆即可传输音频和下载曲目。雅虎！音乐无限功能类似于 Zune 市场。虽然它不需要 Zune 令人困惑的信用系统，歌曲可以以 0.99 美元购买，但用户显然希望走订阅服务的道路。订阅费每月只需 8.99 美元，或固定年费 71.88 美元。

当登录到 Yahoo！通过 Sansa Connect 的音乐服务，用户可以播放各种 Y！网络电台除了通过 WiFi 下载曲目。这样做的一个问题是，歌曲只能通过推荐和流行音乐来提供。这鼓励你适当地评价你最喜欢的歌曲，这样设备就能最好地提供可下载的歌曲，但是如果你脑子里有一首歌却找不到它，我认为这是一件令人失望的事情。如果你喜欢你已经下载的一首歌，你可以使用 Zing 菜单点击下载整首歌——我真的不能强调我是多么迷恋这个功能。此外，我对雅虎印象深刻！主动加入该设备。它在这方面的参与是与其竞争对手的一个显著区别——我相信这将在短期内产生回报。TechCrunch 最近[在雅虎上更具体地报道了](https://web.archive.org/web/20201130075257/http://www.beta.techcrunch.com/2007/04/08/yahoos-new-media-device/)！迈克尔·阿灵顿采访了负责此事的高管之一。

除了下载音乐，Connect 还可以与您的 Flickr 帐户同步，以便随时随地显示您的照片收藏。这实际上是一个相对很酷的功能，因为它的 2.2 英寸屏幕是一个相当出色的显示器。它的另一个与众不同之处是内置扬声器。当然，这不是你能找到的最优质的扬声器，但考虑到它卡在这么小的设备里，听起来还可以接受。

它并不完美。这个界面肯定需要一点时间来适应——尽管我现在有点喜欢它了。感觉也有点单薄。如果你紧紧抓住它，塑料有时会嘎吱作响，我有一种感觉，它不会经受太多的摔落。但话说回来，你可能不应该放弃它。

有效的，有效的，这些元素完全改变了民主行动党的面貌。携带 MP3 播放器不再是被动的聆听， [SanDisk](https://web.archive.org/web/20201130075257/https://crunchbase.com/organization/sandisk) 已经将这一过程变成了主动的、参与性的交易。如果你在路上走了一段时间，离开了你的系统，你就不会局限于有限的音乐量。现在，即使远离机器，您也可以继续拓展自己的音乐视野。我相信这是微软在 Zune 上的“喷射”功能的野心，尽管它悲惨地失败了。

当微软发布 Zune 时，我就在想，用户还需要多久才能通过 WiFi 使用订阅服务，从任何可以连接的地方下载音乐。大约六个月后，这一天终于到来了，这是一件伟大的事情。我不会说这是一个 iPod 杀手或任何东西，但它肯定是一个纳米竞争者。售价 249 美元，比售价 199 美元的 4GB Nano 贵一点，但我认为增加的功能应该足以吸引大量买家。
