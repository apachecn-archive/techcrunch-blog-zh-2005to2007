# 索尼 Gachapin(？？)SE MP3 播放器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/01/sony-gachapin-se-mp3-player/>

我完全忘了索尼还在 MP3 市场！随着所有的 iPod，Zune，Sansa 和 Zen 的营销闪电战在索尼公司周围进行，保持良好和安静。我不确定 Gachapin 是什么或谁，但我认为它是某种龅牙毛毛虫。也许吧？

特别版 lime green E 系列随身听有 512MB 和 1GB 两种口味。它还带有一个 Gachapin 形状的布包和携带袋。200 的限量版将在日本独家发售，售价分别为 135 美元(1GB)和 110 美元(512MB)。
