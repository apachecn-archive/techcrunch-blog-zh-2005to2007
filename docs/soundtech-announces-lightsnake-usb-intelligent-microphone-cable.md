# SoundTech 发布 LightSnake USB 智能麦克风线| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/31/soundtech-announces-lightsnake-usb-intelligent-microphone-cable/>

LightSnake USB 智能麦克风电缆只是一种一端有 XLR 连接器，另一端有 USB 连接器的电缆。这两个接口允许你一端插入麦克风，另一端插入电脑。LightSnake 麦克风电缆采用嵌入式 A/D 芯片技术，可以增强音频信号，因此您不需要麦克风前置放大器。

对于没有混音器、Mbox2 或只是没有输入的艺术家和工程师来说，这种电缆是一个很好的工具。然而，在这个过程中你可能会损失音质。当您使用 SM57 或 SM58 等标准人声话筒时，这不是一个大问题，但当您使用电容话筒、心形话筒、缎带话筒和其他高端话筒时，由于音质原因，您不会想直接在电脑上做任何事情。

LightSnake USB 智能麦克风线还具有“带电点亮”功能，简单来说就是当线缆正确插入时，线缆两端都会亮起。你也不需要任何电缆驱动程序。对于使用 Logic Pro Audio 7、车库乐队、Cubase 和/或 Acid 的家庭音乐家来说，LightSnake 可能是比昂贵的混音器或 2 输入 Mbox2 更可行的选择。
