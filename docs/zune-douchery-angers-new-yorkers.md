# Zune Douchery 激怒了纽约人

[http://video.google.com/googleplayer.swf?docId =-3938916002153088312&HL = en](https://web.archive.org/web/20130628145822/http://video.google.com/googleplayer.swf?docId=-3938916002153088312&hl=en)

我不知道 Zune 的宣传人员在想什么，但他们凌晨 3 点在 [Max Fish](https://web.archive.org/web/20130628145822/http://www.maxfish.com/) 门前放了一辆巨大的 Zune 汽车，在这个过程中激怒了不止一个无限耐心的纽约人。即使在一个经常可以看到人们在凌晨 4:45 在街角排便的城市——真实的故事！—令人惊讶的是，这样的东西没有给主低音炮带来燃烧瓶。
