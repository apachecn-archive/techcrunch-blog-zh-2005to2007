# Vixy 将 YouTube 轻松引入 iPod

Vixy 发布了一款名为 FLV 转换器的工具，允许用户快速转换视频格式。例如，它可以让你轻松地在 iPod 上播放 YouTube 视频。

这很简单，只需将地址插入视频并选择输出格式。FLV 转换器做剩下的。它目前支持 AVI，MOV，MP4，MP3 和 3GP。据说该应用程序可以比典型的代码转换器更快、更“无损”地将 FLV 转换成 MPEG4。
