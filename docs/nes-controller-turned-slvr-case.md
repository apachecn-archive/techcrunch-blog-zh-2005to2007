# NES 主计长转向 SLVR 案

![](img/5fb7cd1e4798e8daba73cdd51516eae3.png)
耶。小妞们会喜欢的。

老派的 NES 管制员可能会受到打击。我记得连续几个小时玩*魂斗罗*和*超级 C* 试图击败他们，却发现自己尽全力把一个 NES 控制器扔在地板上。似乎这位戴着傻傻的汉堡帽的绅士也有同样的启示，他决定把过时的控制器变成一个坚固的手机套。

看起来这家伙拿了个打火机或者 Dremel 剪了一部分给他的 SLVR 显示器和按钮。可惜 SLVR 已经让你看起来像个傻瓜了。这只是雪上加霜。汉堡帽、NES 手机壳和眼镜——女士们？
