# Powerset 发布增长电子表格模型

[![](img/7559840c544695daf52201b84734654e.png) ](https://web.archive.org/web/20201126092223/http://www.powerset.com/) [在上个月的一篇帖子中](https://web.archive.org/web/20201126092223/http://www.beta.techcrunch.com/2007/05/05/will-powerset-have-powergrowth/)我写了关于 [Powerset](https://web.archive.org/web/20201126092223/http://crunchbase.com/company/powerset) 首席运营官史蒂夫纽康使用预测建模来猜测早期增长率，以便他们有足够的硬件来扩展。我说，这对他们有好处，因为他们公开发布了一些数据。我建议他们更进一步，自己发布预测模型:

> Powerset 应该发布模型本身(当然没有具体的 [Powerset](https://web.archive.org/web/20201126092223/https://crunchbase.com/organization/powerset) 假设)，让其他创业公司调整它供自己使用。大多数新公司没有优秀的骑师，也没有时间做这种工作。任何竞争问题都会被他们从这样做中获得的可观的善意(和链接果汁)所掩盖。

这正是他们将要做的事情。Newcomb 表示，他们将在未来几周内发布模型，但不包括 Powerset 的具体数据。不过，不要指望在 Google Docs 上看到这些(这是一个倾倒它们的完美地方，除了 Google 是 Powerset 的目标)。他们将提供电子表格供下载，或者让人们通过 Flash 浏览器在网上查看。
