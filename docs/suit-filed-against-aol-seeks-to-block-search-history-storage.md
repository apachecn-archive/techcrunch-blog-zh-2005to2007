# 对美国在线提起的诉讼；试图阻止搜索历史存储

 [](https://web.archive.org/web/20220626155011/http://aol.com/) 一场集体诉讼[今天在加州](https://web.archive.org/web/20220626155011/http://biz.yahoo.com/iw/060925/0166422.html)提起，控告美国在线[上个月泄露数千名用户的搜索结果](https://web.archive.org/web/20220626155011/http://www.beta.techcrunch.com/2006/08/06/aol-proudly-releases-massive-amounts-of-user-search-data/)。该诉讼要求每个受到影响的用户赔偿不少于 1000 美元，加州每个用户赔偿 4000 美元。隐私廉价！

该诉讼最有趣的部分是，原告试图迫使 AOL 停止存储所有搜索数据。该诉讼还要求用户搜索数据不得出现在进一步的搜索结果中，不得用于商业或非商业目的。虽然这可能是一个不切实际的要求，但这也是对当今网络上许多最有趣的发展的广泛攻击。我们的数据是有价值的，未经我们的允许就将其免费公开显然是一个可怕的想法，但这一宝贵的资源应该根据我们的愿望为我们的利益服务，而不是被摧毁。

该诉讼是由 Berman DeValerio Pease Tabacco Burt & puci llo 律师事务所代表三名 AOL 用户向美国北加州地区法院提起的。
