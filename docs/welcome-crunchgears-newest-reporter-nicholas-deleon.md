# 欢迎 CrunchGear 的最新记者尼古拉斯·德里昂

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/06/welcome-crunchgears-newest-reporter-nicholas-deleon/>

。尼古拉斯是 NYU 大学的三年级学生，是一个全面发展的大好人。他说:

> 你们好，男孩女孩们！我的名字是尼古拉斯·德里昂，我是 CrunchGear 的一名新作家，或者说是 CG，我们这一行的人这么称呼它。你可能还记得某个小工具博客上的我(读作: [Gizmodo](https://web.archive.org/web/20160325175438/http://www.gizmodo.com/) )，但是我想是时候换个环境了。我会尽我所能比较当今最热门的小玩意(！)到过去的国王和老学校的电视节目。这里有一个有趣的事实:上周六晚上，我没有出去，而是呆在家里看了几集《你怕黑吗？一边吃披萨，一边喝 Snapple。不要让任何人告诉你，博客作者的生活不比阿基里斯的生活更辉煌。

让我们都给他一个大大的 CG 拥抱。
