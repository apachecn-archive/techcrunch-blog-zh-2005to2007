# T-Mobile 手机盛会

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/29/t-mobile-handset-extravaganza/>

T-Mobile 度过了忙碌的九月，推出了四款新的三星手机和 RIM 黑莓珍珠手机。作为各种月底庆祝活动，Infosync World 整理了所有手机的小综述，包括 Trace aka SGH-T519，我们有一个 via 姐妹网站 CrunchGear 的[小视频](https://web.archive.org/web/20160407215522/http://mobilecrunch.com/2006/09/20/samsung-trace-quick-look/)。可悲的是，看起来这些手机都不会达到任何传说中的摩托罗拉 RAZR 类型的地位，而且，他们似乎都有点 meh 方面的事情。这对 Pearl 来说尤其如此，因为尽管它是目前最小、最轻的黑莓手机，也是唯一一款带摄像头的手机，但当你点击子菜单时，它的业务根源就会显现出来。
