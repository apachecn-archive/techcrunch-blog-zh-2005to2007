# 泰坦齿轮 2G 纳米外壳

![](img/a5aa2790fd4a626892850858054f1e4c.png)

如果你刚刚买了一台 2G Nano，并且正在寻找一款多功能的外壳，GizMac 可能正是你想要的。泰坦装备包括一个登山扣夹、一个臂章、一个 360 度旋转的腰带夹和一根挂绳。售价 24.99 美元。

不错的设置，但我不得不质疑它缺乏保护。看起来好像系统只是简单地夹在 Nano 上，让脸暴露在外，很容易受伤。要是溅到你闪亮的新 2g Nano 就太可惜了。真是太可惜了。
