# 演示中的热点

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/09/26/whats-hot-at-demo/>

[](https://web.archive.org/web/20210924234656/http://www.demo.com/)[演示](https://web.archive.org/web/20210924234656/http://demo.com/)会议正在圣地亚哥举行，一些明显的趋势正在 67 家独家入选的公司中显现。许多产品刚刚推出，还没有开发到应有的程度，但它们仍然令人兴奋。以下是会议最突出的一些主题，以及我最喜欢的公司朝这些方向努力的例子。

**分解它**

 *许多公司都致力于使数据和在线对象更加精细，便于携带和用户操作。

Pluggd 正在展示一项名为 HearHere 的新技术，该技术使用语音识别和语义分析，让用户在音频文件中搜索关键词和相关术语，这些关键词和相关术语显示在热图上，以便跳转到播客的相关部分。我在 Pluggd 上的[这篇文章中发布了更多的细节和一个截图。](https://web.archive.org/web/20210924234656/http://www.beta.techcrunch.com/2006/09/24/pluggd-to-make-podcasts-chunkier-searchable/)

BuzzLogic 展示了一款历时两年开发的企业社交媒体跟踪器。它可以发现和排名任何感兴趣话题的有影响力的博客帖子和主流媒体报道，在一个漂亮的用户界面中显示影响圈，并跟踪一组用户对新出现的对话所采取的行动。定价非常低，该公司收集并展示了复杂的搜索和 RSS 可以获得的同类数据，但在一个地方以一种可用的格式提供。我认为，在让非技术用户也能使用新网络的竞赛中，这可能是赢家。

[MindTouch](https://web.archive.org/web/20210924234656/http://mindtouch.com/) 发布了他们的 DekiBox 企业设备，该设备可以从电子邮件和附件中提取数据，并将其放在一个安全的维基中。

SportsStatz 与全国各地的高中和大学合作，捕捉体育赛事的视频和统计数据。该公司向学校提供相机和装有软件的笔记本电脑，球队可以获得免费的比赛磁带和统计数据。实时播放后仅 4 秒钟，该服务的个人用户就可以搜索游戏内部，找到特定的精彩内容，并在网上或用移动设备观看。最喜欢的球员的集锦可以自动订阅和发送。剪辑可以在社交网络环境中保存、共享和评论。这项服务还处于早期阶段，但已经在几个州完成了对高中篮球赛季的跟踪，并计划在今年年底前与 1000 所学校建立合作关系。虽然该网站仍在开发中，但该公司的演示确实令人印象深刻，捕捉和使用这样的内容显然有市场。

[Pixsense](https://web.archive.org/web/20210924234656/http://pixsense.com/) 使用正在申请专利的压缩算法将多媒体文件压缩高达 85%，目标是移动内容创作。几个行业的成熟公司都表示对该公司感兴趣，并制作每个人都喜欢的更轻便的移动多媒体。

 [](https://web.archive.org/web/20210924234656/http://www.adaptiveblue.com/) **我的最爱:**
[自适应蓝色](https://web.archive.org/web/20210924234656/http://www.adaptiveblue.com/)在 DEMO 上推出了它的蓝色组织者 out beta。作为 Firefox 的社交书签扩展，Blue Organizer 将自己的本体与你的标签结合起来，让你对你保存的每个项目执行一个很长的功能列表，并做许多智能的小事，如从主题数据库中收集标签，并在你访问页面三次后自动为它们添加书签。社交书签是一个拥挤的空间，但对于那些寻求一个良好构建的工具来平衡直观的用户体验和功能以取悦高级用户的人来说，Blue Organizer 可能是一个非常好的选择。这张照片的美在于细节。
 **滚自己的**

个性化是 Web 2.0 的一个关键目标，许多公司正在让内容提供商比以往任何时候都更容易实现这一目标。

[PrefPass](https://web.archive.org/web/20210924234656/http://prefpass.com/) 将在演示中上线，Adam Marsh 的服务瞄准了一个关键的在线痛点。用户在匿名配置文件中识别他们喜欢的 URL。PrefPass 从那些用于在参与网站上生成个性化内容的页面中提取标签，否则这些参与网站要么会提供令人生畏的注册要求(这使得网站与粘性相反)，要么会提供非个性化的广告和补充内容。当用户进入一个参与页面时，他们可以点击 PrefPass 徽章来授权访问他们的匿名偏好标签列表，然后网站可以提供有针对性的内容或广告。在 CustomCrunch.com 的，你可以看到 PrefPass 在运行，根据你的特殊口味对 TechCrunch 的帖子进行分类。

[NanoLearning](https://web.archive.org/web/20210924234656/http://nanolearning.com/) 是一种在 Flash 中制作教育游戏或培训模块的简单方法。用户提供内容(文本、图形、音频、视频、问题和答案),纳米学习提供模板和表格来创建您的模块。这里有一个例子，但是我可以想象很多人用这个工具快速开发教程和交互式演示。

 **我的最爱:**

[![](img/1fd0188f032d4b909bcbdf3773af93fd.png)](https://web.archive.org/web/20210924234656/http://www.widgetbox.com/)Widget market place[Widget box](https://web.archive.org/web/20210924234656/http://widgetbox.com/)不断为出版商提供更多方法，让他们拥抱松散的小块伦理，并将实时数据拉进他们的网站。Hummer Winblad 公司现在允许用户在他们的网站上放置一段代码来创建一个小部件字段，然后可以通过拖放来管理 200 个小部件中的任何一个，将它们移入或移出发布的页面。今天，该公司向我展示了一个浏览器侧边栏弹出窗口，让网站访问者继续与你的小工具互动，即使他们离开了一个特定的网址。要了解更多细节，请查看我们对 Widgetbox 的初步评论。Widgets 可能有一个愚蠢的名字(而 [Cute Overload](https://web.archive.org/web/20210924234656/http://cuteoverload.com/) 是目前网站上订阅最多的 widgets 之一)，但就像博客通过让非技术用户轻松在线发布原创内容改变了世界一样，widgets 也预示着一个新时代的到来，即用户混合和混合来自网络各地的动态数据。
 **让我们无拘无束地谈谈**

 *一些最有趣的公司正在推出服务，使沟通超越以前的限制。

JaJah 在 DEMO 上发布了一款移动产品，用户可以通过手机进行极低成本的 VOIP 通话。参见 [Mike Arrington 对此次发布的长篇评论](https://web.archive.org/web/20210924234656/http://www.beta.techcrunch.com/2006/09/26/jajah-just-launched-killer-voip-product/)以及昨晚关于竞争对手 [SoonR 与 WebEX](https://web.archive.org/web/20210924234656/http://www.beta.techcrunch.com/2006/09/25/webex-to-go-mobile-with-soonr/) 合作实现移动网络会议的新闻。

Flurry 正在展示他们的服务，该服务让 200 多部支持 Java 的手机提供简单的推送电子邮件和阅读 RSS 订阅。

[TotalView](https://web.archive.org/web/20210924234656/http://www.behere.com/) 将推出一款名为 BeHere 的企业 VOIP 视频会议产品，该产品可捕捉会议室的 360 度视图，并在每位与会者的桌面上提供易于配置的视图。用户可以共享整个系统的应用视图。我希望该公司在该技术上做得更多，比如提供实际的文件传输、反向聊天频道和会议录音——但该产品目前处于一个很好的位置，可以通过经销商分发到任何地方的 VOIP 会议室。他们已经在首轮融资中筹集了 700 万美元，正接近 B 轮融资的尾声，并将以低于 2000 美元的价格出售这项服务。这绝对胜过 VOIP 扬声器电话。

**我的最爱:**
[](https://web.archive.org/web/20210924234656/http://grandcentral.com/)[Grand Central](https://web.archive.org/web/20210924234656/http://grandcentral.com/)使用 VOIP 和 web 界面提供一个电话号码，可以根据来电号码将来电路由到您选择的任何电话，管理 web 上的语音邮件，记录通话等等。来自雅虎的高管们。GrandCentral 收购了 DialPad，它有一系列令人印象深刻的功能，我在这里回顾了这些功能。

许多人在会议上谈论的其他公司包括 [SystemOne](https://web.archive.org/web/20210924234656/http://systemone.at/) ，这是一个企业维基 CMS，它可以在你写作时分析你的文档，并在网上搜索你的提要、文件和维基的其他部分中的相关关键词。几周前，我[回顾了 SystemOne](https://web.archive.org/web/20210924234656/https://beta.techcrunch.com/tag/systemone) 。 [Koral](https://web.archive.org/web/20210924234656/http://www.koral.com/) 是另一个热门话题；这是一个企业 CMS，如果用户访问任何共享文件的过期版本，它会提供拖放文件组织、推荐标签和即时消息通知。微软的衍生社交网络服务 Wallop 显然也是一个热门话题。

所有被选中参加演示的公司都值得一看—[这是列表和视频](https://web.archive.org/web/20210924234656/http://www.demo.com/videoarchives.php)。这里弥漫着兴奋的气氛，创新是实实在在的。**
