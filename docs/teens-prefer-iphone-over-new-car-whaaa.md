# 比起新车，青少年更喜欢 iPhone，Whaaa？

![](img/c633e866a5b115f427556e4787f2c6b9.png)

根据 CNW 市场研究公司的一份报告，32%接受调查的青少年表示，一部新手机会给他们的朋友留下深刻印象，相比之下，20%的青少年选择了汽车。当你问他们，如果 cellie 是一部 iPhone，他们会有什么感觉时，说手机会让他们“酷”的青少年比例上升到了 70% 。不出所料，汽车也被鞋类、视频游戏、iPods 和电脑打败了。说真的，这太荒谬了。如果你的朋友对 500 美元的手机比对 30000 美元的交通工具更感兴趣，你需要离开的短途巴士。
