# 大声点，桑尼！奶奶听不见！

![](img/90261bcadbf47370b4d9d4ad8ef8b3d6.png)

老年人要么决定全力以赴使用电子设备，学习录像机时钟的工作原理，要么干脆放弃它们。但现在，多亏了 not 多科莫，老年人不必为了不用学习而放弃手机等必需品。

樱庭落-樱庭落电话是为有听力和视力问题的老年人设计的。它带有一个“慢速语音”功能，可以减慢来电者的速度，以便他们可以很容易地被理解，一个“清晰语音”功能，可以根据周围环境自动调节音量，一个自动文本阅读器，最重要的是，一个计步器。

听起来像是给我祖母的完美礼物。如果我能在美国抢到一个，把她的 SIM 卡换出来，我的生活会轻松得多。
