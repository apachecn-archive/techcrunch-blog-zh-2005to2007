# Usphere:颠倒大学申请流程

T2 申请大学要花很多时间搜索、比较和填写定制的表格。所有这些开销的后果是，大多数学生申请的学校越来越少，主要集中在知名学校。 [Usphere](https://web.archive.org/web/20211026162808/http://usphere.com/) 是一家位于伊利诺伊州埃文斯顿的初创公司，希望改变这种状况。

Usphere 允许学生填写一份申请，然后被他们的大学网络考虑。当你完成申请后，它会被扔进他们的申请人搜索引擎，只有他们网络中的 33 所学校可以访问。如果一所学校喜欢你，他们会给你发一封电子邮件，告知你录取的最低价格。申请服务的费用是 65 美元，尽管他们有几个免费的大学搜索工具。

申请不同于常春藤盟校和州立学校申请人熟悉的多校“标准申请”格式。Usphere 要求您上传任何成绩单、推荐信或文件，以帮助您的申请脱颖而出。然而，Usphere 用个性简介取代了申请短文。有点像大学里的 Match.com。

Usphere 将这项服务宣传为一个机会，可以申请到学生以前可能没有想过的“冷门”学校并被录取。他们不公开公布目录中的学校，但人们可以猜测它包括需要额外申请人的利基学校，申请人可能不太了解这些学校。
