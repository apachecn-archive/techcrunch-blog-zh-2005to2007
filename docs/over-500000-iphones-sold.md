# 售出超过 50 万部 iPhones

![](img/ba792b01add9f5ed4428f5fd2678586e.png)
~~工具~~消费者在苹果 SoHo 店排队

上周末有个小玩意叫 iPhone，对吧？显然它在销售方面做得相当不错。各种报道纷至沓来，称已售出 50 万或 52.5 万辆。就像 Playstation 3 一样，大城市的大多数人都选择了高端的 8GB 机型，而不是更便宜的 4GB 版本。

至于我一直在漫谈的所有客户保留问题，看起来 iPhone 正是医生为 AT&T 订购的产品。购买 iPhone 的消费者中，约有一半是新客户，他们只是为了苹果的必备小工具而加入。我得说这对于三天的工作来说是不错的。
