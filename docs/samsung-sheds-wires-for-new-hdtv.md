# 三星为新高清电视拆电线

![samsung-wireless-pdp-3.jpg](img/032ab217d6ad57363c98762220868b11.png)

三星宣布将很快在韩国推出 SPD-50P7HDT。这款 50 英寸高清等离子屏幕与同类显示器的不同之处在于其完全无线的外设连接。该设置允许用户通过新开发的无线技术连接 DVD 播放器、vcr 和其他设备，从而避免电缆混乱。它的售价预计为 4000 欧元，但不幸的是，目前在美国还没有销售。
