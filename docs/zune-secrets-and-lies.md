# Zune:秘密与谎言

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/17/zune-secrets-and-lies/>

iLounge 有一些关于 Zune 的热门内容，包括对 UI 和一些奇特无线功能的描述。以下是一些精辟的信息:

> Wi-Fi:诶。显然，你可以使用 Wi-fi 功能“租借”其他 Zune 用户的音乐一小段时间(一天)，让他们有机会从 Zune 音乐商店购买自己的歌曲。虽然这是一个很酷的想法，并且可能适用于无处不在的 iPods，但它在这里得到了一个“呃”,因为你不太可能随机遇到其他拥有 Zunes 的人，除非微软实际上免费提供音乐，否则你更不可能找到愿意从其最新的音乐商店购买歌曲的人。你也可以将 Zune 音乐下载到支持 Wi-Fi 的 Xbox 360 上，这也是一个很酷的想法，但需要 100 美元的无线适配器和 300-400 美元的游戏机。(尚不清楚是否需要硬盘——价格更高的 400 美元套装的一部分。)

> …
> 那个轮子:没那么酷。原来这里没有触摸感应滚动功能——它的形状像一个轮子，但实际上只是按钮。Zune 的圆圈左侧有一个菜单按钮，圆圈内有滚动按钮，圆圈右侧有一个播放/暂停按钮。还有一个保持开关，就像苹果的一样，在耳机端口附近。你可以按下“滚轮”上的上下键来上下滚动菜单。
