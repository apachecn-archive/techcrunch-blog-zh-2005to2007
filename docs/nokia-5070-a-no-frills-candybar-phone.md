# 诺基亚 5070–一款简洁的糖果棒手机

![](img/17425974f321634167b5f0b28aea8949.png)

那些享受简单生活的人会很高兴地欢迎诺基亚 5070。这款糖果棒风格的手机没有激进的设计，充满了基本功能！您将获得短信和彩信功能，令人惊叹的音频消息，分辨率糟糕的内置摄像头，收听 NPR 的调频收音机，立体声耳机，以及红色或蓝色油漆之间的选择。没有比这个更令人兴奋的了，是吗？

如果这还不令人兴奋，我就不知道什么才是了。它的零售价约为 135 美元，将在 Q2 期间在欧洲上市。由于缺乏功能，这款手机有点贵，但它看起来很坚固，可能是不喜欢刺激的人的选择。
