# 专利猴:网飞利用专利起诉反垄断

网飞订户 Dennis Dilbeck 提起集体诉讼，声称网飞通过利用欺诈手段获得的[专利](https://web.archive.org/web/20230322164155/http://www.patentmonkey.com/PM/FolderID/b91d9af1850d47b38563eb7ed80919e9.aspx)控制了在线 DVD 租赁市场，这导致了今年早些时候提起的一桩不同寻常的基于专利的反垄断案件(最近公开)，这与百视达对网飞的[反诉](https://web.archive.org/web/20230322164155/http://online.wsj.com/article/SB115024167013979480.html)一致。

迪尔贝克提出了一个案例，网飞在审查期间向专利局隐瞒了关键的现有技术，这将使两项专利无效，此后它一直用“虚假诉讼”来恐吓竞争对手。

迪尔贝克声称，网飞知道会破坏其专利申请的现有技术。证据呢。网飞在收到来自 NCR 的 10 项专利的通知后，非常担心这些专利，以至于该公司向 NCR 提起了自己的诉讼，以避免被起诉。迪尔贝克认为，NCR 专利会让专利局发现网飞的发明显而易见。之前提到的两个 NCR 专利现有技术示例包括 [6397199](https://web.archive.org/web/20230322164155/http://www.patentmonkey.com/PM/patentid/6397199.aspx) 和 [6842115](https://web.archive.org/web/20230322164155/http://www.patentmonkey.com/PM/patentid/6842115.aspx) 。法庭文件没有详细说明 NCR 的十项专利。

重要的是，NCR 的行动发生在网飞的专利还在申请过程中的时候。USPTO 可以利用欺诈作为理由，使授予的专利无效，这通常是在专利被用来对抗侵权案件中的另一方时进行的。

随着网飞专利的发布，Dilbeck 声称它能够对竞争对手实施专利垄断，从而提高网飞消费者的价格。迪尔贝克指出:

> 网飞提醒相关市场的潜在或实际竞争对手注意' 450 或' 381 专利(包括网飞对百视达的专利诉讼)的任何行为都是为了阻止和破坏竞争…

Dilbeck 声称，网飞是否知道其专利无效，并指出:*“网飞在起诉百视达时确实知道‘450 和‘381 专利无效且不可执行，其首席执行官甚至更进一步，称‘450 专利是一个‘笑话’*。

针对专利所有公司垄断的客户集体诉讼可能会为专利法领域的持续诉讼铺平道路，或者可能只是百视达诉讼的无根据延续。像大多数情况一样，这将归结于细节。
