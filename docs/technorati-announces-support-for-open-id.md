# Technorati 宣布支持 Open ID

 [](https://web.archive.org/web/20221007070219/http://www.technorati.com/) 博客搜索引擎 Technorati [今晚](https://web.archive.org/web/20221007070219/http://technorati.com/weblog/2006/10/144.html)宣布，博主们现在可以用 Open ID 凭证宣称他们博客的所有权，更多支持 Open ID 的工作正在进行中。

Open ID 提供了一个 URL，您可以使用它在网络上的所有参与站点上标识自己。您可以自己处理物流，也可以付钱给许多不同的供应商来注册您的 ID 并解决对您的信息的请求。

传播 Open ID 的运动旨在让我们的生活更轻松，我们的数据更安全，并帮助用户避免大供应商不公平地滥用权力的封闭孤岛。Technorati 对该标准的持续和有意义的支持是一件大事。将它加入到 Technorati 正在进行的前瞻性工作中，比如微格式。

许多其他供应商，包括像微软、甲骨文和太阳这样的大公司，都在参与开放 ID 的讨论，但对于新的网络社区中的主要参与者之一来说，做出这样切实的举动将为往往看起来很边缘的草根努力提供支持。当 Web 2.0 场景中更面向社区的玩家首先行动时，他们就可以对未来的格局施加更大的影响。

正如 Technorati 博客文章所指出的，其他支持 Open ID 的供应商包括开拓者 [LiveJournal](https://web.archive.org/web/20221007070219/http://www.livejournal.com/openid/) 和 Six Apart 的 [Vox](https://web.archive.org/web/20221007070219/http://www.vox.com/) 。

有关 Open ID 的更多信息和背景，请参见伯克曼中心的[IdentityGang.org](https://web.archive.org/web/20221007070219/http://identitygang.org/)，[我在 7 月份对 Open ID 倡导者](https://web.archive.org/web/20221007070219/http://netsquared.org/kaliya)[卡莉娅·哈姆林](https://web.archive.org/web/20221007070219/http://www.kaliyasblogs.net/Iwoman/)的采访以及维基百科关于 [Open ID](https://web.archive.org/web/20221007070219/http://en.wikipedia.org/wiki/OpenID) 的条目。
