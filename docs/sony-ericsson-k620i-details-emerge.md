# 索尼-爱立信 K620i 细节浮出水面

![](img/9a101b8d9f6bd3652c6c8c09cabcc5f7.png)

索尼即将推出的索尼爱立信 K620i 的一些细节已经出现，我们喜欢我们所看到的。除了非常酷的蓝色油漆工作(这可能是也可能不是最终的颜色)，这个吸盘只有 17 毫米厚，配备了 1.9 英寸，262k 彩色屏幕，蓝牙，USB 2.0(感谢上帝)，一个生病的 320 万像素摄像头，100 兆内存用于视频和照片存储，支持最大 1GB 的 M2 记忆棒卡。如果相机质量像大多数索尼-爱立信手机一样保持很高，那么我们就真的很享受了。不幸的是，还没有关于定价，发布日期，或它将与哪个运营商的官方消息。
