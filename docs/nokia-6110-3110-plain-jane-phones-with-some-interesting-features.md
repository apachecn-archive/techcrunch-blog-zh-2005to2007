# 诺基亚 6110、3110:拥有一些有趣功能的简单手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/12/nokia-6110-3110-plain-jane-phones-with-some-interesting-features/>

500 美元的 6110 是我们第一款有趣的手机，包括 GPS 和地图软件。它有 40MB 的存储空间以及 HSDPA 和辅助 GPS，它依赖于地面信号，即使你不在卫星覆盖范围内，也能收到信号。它支持 miniSD 和 200 万像素的摄像头。

![](img/2af5f4fb7737fc2ac36dbcc62046b25c.png)
3110 大大少配了区区 130 万像素的摄像头和 MP3 播放器。它有 GPRS 和蓝牙功能，应该会在 2007 年 Q2 奥运会上面市。
