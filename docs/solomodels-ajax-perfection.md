# solo models–Ajax 完美

 [](https://web.archive.org/web/20230218163652/http://www.solomodels.com/) 今天我偶然发现了亚利桑那州斯科茨代尔的 [Solomodels](https://web.archive.org/web/20230218163652/http://www.solomodels.com/) 。我写这篇文章是因为~~网站上有非常迷人的女性照片~~这是 Ajax、搜索和社交网络的典型例子。它迎合了模特、摄影师和经纪人的需求。

创始人 Hoss Etemad 写信给我描述了这项服务，我必须说我立刻就发现它很有趣。参与者可以创建个人资料、上传照片、添加成员为好友、聊天等。[搜索](https://web.archive.org/web/20230218163652/http://www.solomodels.com/search/index.cfm)功能非常棒——输入几乎任何物理属性，从胸围大小到眼睛颜色，然后找到与你匹配的模特。

他们为有兴趣尝试这项服务的人提供七天的免费试用。不过，Solomodels 完全是生意。该网站警告说:

> 我们的会员不是来约会的。
> 除了合法的模特相关工作之外，严禁联系我们的会员，否则将导致账户立即被取消。如果你正在寻找约会，请注册一个约会网站。

所以想都别想。

这是一个展示 web 2.0 的伟大网站。:-)
