# Safari Beta 升级至 3.0.2 | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/23/safari-beta-upgrades-to-302/>

![](img/b6bcab2193126ccd79001c5adccbdea8.png)

苹果希望 Safari 成为下一个大事件，我们为他们鼓掌，但在 IE 和 Firefox 需要担心之前，他们肯定还有很长的路要走。在发布的前 24 小时内发现一些[漏洞](https://web.archive.org/web/20150920041815/http://crunchgear.com/2007/06/12/0day-exploit-for-safari-for-windows-spotted/)后，苹果迅速将 Safari 升级到 Windows 的 [3.0.1](https://web.archive.org/web/20150920041815/http://crunchgear.com/2007/06/14/safari-for-windows-updated-to-301-fixes-exploits/) 。处于测试阶段，我们希望升级来得更快更激烈。针对 Mac OS X 和 Windows 分别进行了以下更改:

> -最新的安全更新
> -改进的稳定性
> -改进的 WebKit 对邮件、iChat 和 Dashboard 的支持
> 
> –最新安全更新
> –提高稳定性
> –修复文本显示、非英语系统和启动时间
