# Skype 推出 Skype Prime:通话收费

今天晚上，Skype 推出了一项新服务，让人们为 Skype 语音和视频通话付费。用户根据每分钟或每次通话来设定他们想要收取的费用。Skype 通过其 Skype 信用点数向支付方收费，然后通过 paypal (Skype 的姐妹公司)支付，扣除 30%的费用。

像去年推出的[以太](https://web.archive.org/web/20221207190211/http://www.beta.techcrunch.com/2006/06/22/ether-pay-per-call-launches-for-public-use/)(只收取总费用的 15%)，这对于某些类型的顾问来说是非常有用的服务。与以太网不同，Skype Prime 从庞大的用户群开始，并且是跨国运作的。你也可以说这两家公司都剽窃了旧的按通话付费号码。

要使用这项新服务，通话双方必须使用最新版本的 Skype(3.1.0.134)Windows 版。Skype 表示，其他平台最终也会出现。
