# 逐字彩色 LightScribe 光盘| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/13/verbatim-color-lightscribe-discs/>

在惠普或其他一些公司找到让光驱刻录彩色图像的方法之前，这些彩色 LightScribe CD-r 就像是给你的激光蚀刻标签添加颜色一样。

如果你不熟悉 LightScribe 驱动器，这是一种内置于光驱的技术，允许你将标签刻录到特殊涂层的 CD 和 DVD 的顶部。不幸的是，它们只能刻录灰度图像。所以这些光盘至少在背景中加入了绿色、蓝色、橙色、红色和黄色。售出 25 包，逐字今天宣布出货量的光盘，售价为 19.99 美元。
