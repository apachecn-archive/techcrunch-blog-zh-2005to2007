# 吊索准备 OS X 客户

![](img/0a70bc55804c10c1b539324067056c76.png)

在《卫报》的采访中，Sling Media 首席执行官 Blake Krikorian 宣布 OS X 的 SlingPlayer 将在下周面世。他还承诺，到今年年底，至少还会有另外一款操作系统面市。Zatz Not Funny 写道，Sling Media 已经[发布了职位空缺](https://web.archive.org/web/20130627212943/http://www.zatznotfunny.com/2006-04/slingbox-on-symbian/)正在寻找 Symbian 和/或 Palm (J2Me)的程序员，这可能表明他们正在关注其他操作系统。

听起来在吊索营有各种各样的忙碌。随着新的 FCC 文件和新的操作系统的支持，看起来他们试图保持进步，因为挑战者开始反对他们。我希望它能成功。
