# 零售商向索尼 PSP 发出最后通牒

![](img/fc4a88d4f358c99f23cb062c8a1d5260.png)

据 [1UP](https://web.archive.org/web/20130628191647/http://www.1up.com/do/newsStory?cId=3158013) 报道，零售商已经告诉索尼，除非 PSP 价格下降，否则他们将把它下架。尽管 1UP 没有指明零售商的名字，并且对细节含糊其辞，但该网站暗示，如果价格没有下降，PSP 很快就会发生重大变化。文斯有一些[细节](https://web.archive.org/web/20130628191647/http://crunchgear.com/2007/03/13/smaller-lighter-psp-coming-your-way/)关于重新设计可能需要什么，马特有一个[帖子](https://web.archive.org/web/20130628191647/http://crunchgear.com/2006/12/20/sony-preps-psp-movie-store-does-anyone-care/)关于一个新的 PSP 音像店正在工作中。

同时，1UP 与 PSP 的高级营销经理约翰·柯勒进行了交谈，他确认了重新设计。

> 据柯勒称，索尼正在减少零部件，并可能调整尺寸——类似于 PS1 和 PS2 的做法。“我至少可以说这么多，屏幕的大小不会改变，”他说。
