# 三星 WiMax 手机(别叫它蜂窝)| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/09/samsung-wimax-mobile-phone-dont-call-it-a-cell/>

我们已经厌倦了韩国人使用所有的移动电话。举个例子，三星在 WiBro (WiMax to us yanks)峰会上发现的这款功能丰富的超薄手机，我们在美国永远也看不到。为什么我们看不到它？因为它不使用标准蜂窝网络使用的 CDMA 或 GSM 格式。不，它必须变得非常时髦，像混合 VOIP 电话一样使用 WiMax 进行语音和数据传输。所以，从技术上来说，它不是一部手机，但它在各方面都是一体的。

它还拥有韩国的数字电视服务，以及 Wnidows Mobile 5:智能手机版、PTT 和笔记本电脑数据服务。它有双摄像头用于视频会议。和集成的 RSS 阅读器。

我们希望在这里看到这种创新的东西。在那之前，我们会继续抱怨威瑞森巧克力的恶心。
