# 威瑞森抛弃了 V-Cast 来玩音乐手机？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/29/verizon-ditching-v-cast-for-playsforsure-for-musicphones/>

。根据手册，新的高端 RAZR 利用 Windows Media 11 和 PlaysForSure，意思是你的雅虎！音乐订阅可以在手机上使用。

运营商向来自其他服务的音乐开放其硬件的想法并不新鲜，Cingular 早在它的超级时尚、性感、纤细、令人向往的 [ROKR](https://web.archive.org/web/20131217222758/http://crunchgear.com/?s=ROKR) 时就开始了这一趋势，这是一款兼容 iTunes 的音乐手机。这是一个难题，但它确实证明了人们想要手机上的音乐，但不想被锁在第二家音乐商店里。

虽然就合法音乐下载而言，PlaysForSure 仍处于 iTunes 的次要地位，但如果更多的运营商和制造商开始在他们的手机上安装非苹果竞争对手的软件，这种情况可能会发生变化。PlaysForSure 的工作方式和 iTunes 一样，可用的音乐大致相同，质量也刚刚好。然而，关键是手机的数量远远超过 iPod，如果它们都可以与用户选择的 PlaysForSure 商店兼容，那么这可能是对 iTunes 和 iPod 的第一个真正威胁。
