在gpokr玩Ajax德州扑克

如果你喜欢阿贾克斯的战舰，你会喜欢这个。总部位于加拿大多伦多的 gpokr 是一款使用Ajax技术开发的多人无限德州扑克游戏。该网站由Ryan Dewsbury创建，在游戏中所有玩家之间也有实时聊天。该网站不使用真钱，Ryan说他不打算改变这一点。

gpokr是用java编写的，它使用谷歌的Web工具包作为其客户端Ajax(该工具包通过将服务器端的Java解释成javascript减轻了构建客户端javascript的负担)

还可以看看Mark Roth的Ajax hangman 游戏，它也是使用Google Web Toolkit创建的。两者都在Web Toolkit博客上讨论过，这里是。
