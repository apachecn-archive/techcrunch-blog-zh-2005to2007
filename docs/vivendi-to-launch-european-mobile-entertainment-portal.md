# Vivendi 将推出欧洲移动娱乐门户网站| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/25/vivendi-to-launch-european-mobile-entertainment-portal/>

据《回声报》报道，法国多媒体巨头 Vivendi 可能很快会为第三屏幕推出一个多媒体网站。该公司成立了一个新的部门，Vivendi Mobile Entertainment，目标是开发这个新的门户网站，该网站被比作 Myspace 或 YouTube，将提供 Vivendi 和其他合作伙伴的音乐、视频、电影和游戏内容。这项服务将是一个付费网站，据报道，它可能会在今年秋天在法国推出，并随后推广到其他欧洲国家。
