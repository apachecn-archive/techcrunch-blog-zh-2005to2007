# Technorati 重新发布——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/24/technorati-relaunch/>

 [](https://web.archive.org/web/20220818141721/http://www.technorati.com/) [Technorati](https://web.archive.org/web/20220818141721/http://www.technorati.com/) 更新了他们的设计并增加了新功能，庆祝他们的三岁生日。如需了解更多信息，请参见 Technorati 博客上首席执行官戴夫·西弗里的[帖子](https://web.archive.org/web/20220818141721/http://technorati.com/weblog/2006/07/110.html)。

在这一点上所有人都竖起大拇指。这个设计比一年前的[前身](https://web.archive.org/web/20220818141721/http://www.beta.techcrunch.com/2005/06/20/profile-technorati-new/)好得多，也更容易阅读。

然而，新功能才是最重要的。特别是，该网站的新“ [Discover](https://web.archive.org/web/20220818141721/http://technorati.com/discover/) ”区域是一个很好的方式来把握博客世界关于任何给定主题的脉搏，包括新闻、技术、体育、商业和生活。这个功能显然旨在试图重现 Techmeme 在成为“博客世界的新报纸”方面取得的一些成功，并且比 Technorati 以前的努力要好得多。产品经理 Liz Dunn 将于本周晚些时候在 Discover 上发布更多细节。

还有其他值得注意的变化，包括对[收藏夹](https://web.archive.org/web/20220818141721/http://technorati.com/faves)部分和[热门](https://web.archive.org/web/20220818141721/http://technorati.com/pop/)部分的更新(与 [Technorati Top 100](https://web.archive.org/web/20220818141721/http://technorati.com/pop/blogs/) 列表的纯链接计数相比，更加强调用户收藏夹)。

Technorati 生日快乐，并祝贺重新启动和您最近的新一轮融资。
