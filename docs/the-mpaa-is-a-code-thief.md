# 美国电影协会是一个密码窃贼！TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/19/the-mpaa-is-a-code-thief/>

![](img/30734c954506117676b4f97deb6101ba.png)

你可能会认为，随着美国电影协会起诉人们侵犯版权，随着《幽灵骑士》等电影攀升至第一名，你可能会认为花 50 美元就能让一个博客作者购买他的软件。

“再想想！”森林博客发布平台的设计者帕特里克·罗宾喊道。罗宾最近发现[美国电影协会的博客](https://web.archive.org/web/20170707171059/http://www.mpaa.org/blog_default.asp)(目前已关闭)在没有支付适当许可费的情况下使用他的软件——个人许可费 10 英镑，商业许可费 25 英镑。

罗宾拿到钱了吗？没有。美国电影协会还没有回复他写的一封关于他的代码被盗的信。
