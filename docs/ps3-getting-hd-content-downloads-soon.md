# PS3 即将获得高清内容下载？

![](img/4c73b2f895aba162f81288dde9efb127.png)

1UP.com 最近与索尼公司的菲尔·哈里森谈论了 PS3 的未来以及 2007 年将会有什么新的发布。似乎索尼正准备在 Playstation 网络上推出高清内容，包括电影、音乐和电视节目。Harrison 还暗示 PS3 会有更大的硬盘，并继续让我们知道**不会很快降价。**

在玩了一段时间 PS3 和 Playstation Network 之后，我可以严肃地说，PSN 需要改进。它有一个平淡无奇的用户界面，一小批内容，没有电影或电视节目！有时候我只想花钱买一集《实习医生格蕾》的 DRM 版本，因为我想尽快看完。在我们知道索尼的超级游戏机有什么锦囊妙计之前，我们必须拭目以待。
