# Yahoo.icio.us？–雅虎收购 Del.icio.us

我刚刚与 Del.icio.us 的创始人 Joshua Schachter 通过即时消息进行了交谈。我问他对雅虎收购传闻有何评论，他让我看了 del.icio.us 的博客。

[雅虎收购了拥有大约 30 万用户的 Del.icio.us](https://web.archive.org/web/20230213025700/http://blog.del.icio.us/blog/2005/12/yahoo.html) :

> **y.ah.oo！**
> 
> 我们自豪地宣布 del.icio.us 已经加入雅虎！家庭。我们将一起继续改善人们在互联网上发现、记忆和分享的方式，重点强调社区的力量。我们很高兴能与雅虎合作！搜索团队——他们肯定有社交系统和改变网络的潜力。(我们也很高兴加入我们的孪生兄弟 Flickr！)
> 
> 我们要感谢一路上帮助过我们的每一个人——我们的员工、我们伟大的投资者和顾问，尤其是我们的用户。我们仍然希望得到您的反馈，我们期待在未来为您带来新的功能和更多的服务器。
> 
> 我期待着继续我的社会和社区记忆的愿景，并与 del.icio.us 社区和雅虎一起将它提升到一个新的水平！

竞争是如此美妙的事情。雅虎除了在过去几个月里推出了一系列新产品(而且步伐似乎在加快)，现在还拥有网络上最重要的两个标签属性——Flickr 和 del.icio.us

恭喜你，德尔·奥西欧·尤斯和弗雷德·威尔逊。还有乔什·科佩尔曼。
 **更新** : Jeremy Zawodny 在[雅虎搜索博客](https://web.archive.org/web/20230213025700/http://www.ysearchblog.com/archives/000223.html)上也发布了收购消息。“从今天起，del.icio.us 成为雅虎的一部分！家人。”
