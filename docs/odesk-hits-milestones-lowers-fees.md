# oDesk 达到里程碑，降低费用

周二，总部位于硅谷的 oDesk 将宣布他们达到了一些里程碑，并将降低价格。我们第一次[在 9 月初对公司](https://web.archive.org/web/20211127060555/http://www.beta.techcrunch.com/2006/09/08/odesk-provides-on-demand-skills/)进行了简介，我们认为 oDesk 填补了初创公司领域的一个重要空白——帮助公司为项目寻找外包开发帮助。

oDesk 是一个寻找开发人员来构建软件的地方，他们允许项目所有者在工作时通过屏幕截图和摄像机来监控开发人员——参见这里的演示。这是我们自己用过的服务，并计划再次使用。我们遇到的唯一不利之处是——发布的项目可以被公众看到，这对你的竞争对手来说是一个不利因素(如果他们足够聪明来检查网站的话)。

oDesk 做得非常好，将宣布 500，000 总计费小时。6，000 名开发者在该网站注册，每月有 1，000 个新项目发布。

oDesk 目前收取 23%的费用，这是在开发者每小时收费的基础上增加的。周二，他们将宣布费用已降至 10%(这一变化实际上在几个小时前已经生效)。
