# 索尼 PSP 获得 Wi-Fi 地理定位软件

索尼 PSP 的新软件将允许用户使用 GPS 配件进行室外地理定位，然后在建筑物内切换到由 Wi-Fi 信号提供的室内地理定位。

> Zenrin Map 2 和 Edia Maplus 的新导航软件仍将使用 PSP 的 GPS 附件，但增加了通过 Wi-Fi 提供的位置数据。例如，如果 PSP 连接到 Wi-Fi 热点，导航应用程序可以继续显示 GPS 卫星无法到达的建筑物内的位置。

你可以在美国国内以 59.90 美元的价格从 [Play-Asia](https://web.archive.org/web/20210122232415/http://www.play-asia.com/SOap-23-83-102j-71-bh-49-en-15-gps-84-j-70-1mij-43-7o.html) 购买日本独有的 GPS 接收器，或者分别以 95 美元或 99.90 美元的价格订购一个装有 [Zenrin 软件](https://web.archive.org/web/20210122232415/http://www.play-asia.com/SOap-23-83-102j-71-9f-77-1-49-en-15-gps-84-j-70-1sto-43-7o.html)或 [Edia 软件](https://web.archive.org/web/20210122232415/http://www.play-asia.com/SOap-23-83-102j-71-9f-77-2-49-en-15-gps-84-j-70-1mii-43-7o.html)的 GPS 设备。
