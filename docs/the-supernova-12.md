# 超新星 12 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/03/the-supernova-12/>

上周，在 Supernova 大会上，超过 100 家初创公司申请在 TechCrunch 赞助的[互联创新者](https://web.archive.org/web/20221006021739/http://www.supernova2006.com/go/connected-innovators)项目上展示他们的公司。12 家公司被选中，有机会向数百名观众推出他们的新产品。

我起草了一些在[发布会上演示和发布的产品的实时笔记，下面是我更完整的笔记。](https://web.archive.org/web/20221006021739/http://www.crunchnotes.com/?p=227)

**夏普卡斯特**

总部位于帕洛阿尔托的 Sharpcast 已经开发了一个平台，可以在你的电脑和移动设备之间同步应用数据。他们的第一个 showcase 应用程序是 [Sharpcast Photos](https://web.archive.org/web/20221006021739/http://www.beta.techcrunch.com/2006/05/30/sharpcast-opens-to-the-public/) ，它不仅可以将照片从一台设备/电脑推送到其他设备/电脑，还可以保持照片同步。在一个副本上进行更改，它也会将更改推送到其他副本上。还有许多新的应用程序(文档、日历、联系人)。该公司已经筹集了 1650 万美元的资金，它将是应用程序不可知的，所以你不必改用新软件。今天只有 Windows，Mac 即将推出。

**Webaroo**
 [Webaroo](https://web.archive.org/web/20221006021739/http://www.webaroo.com/)总部位于圣塔克拉拉，是[在 4 月](https://web.archive.org/web/20221006021739/http://www.beta.techcrunch.com/2006/04/10/store-web-content-offline-with-webaroo/)推出的一项新服务，允许 PC 用户(尚不支持 Mac)在离线时访问缓存的网络内容。Webaroo 提供预先选择的内容，称为“网络包”，用户也可以缓存他们想访问的任何网站。更多信息，请点击查看 TechCrunch Webaroo 评论[。](https://web.archive.org/web/20221006021739/http://www.beta.techcrunch.com/2006/04/10/store-web-content-offline-with-webaroo/)

**PostApp**

PostApp 是一家新公司，它允许用户将 web 服务直接放入他们的博客或其他网站，而无需具备使用服务提供商提供的 API 的技术技能。随着小工具的激增，PostApp 可能是在正确的时间出现的正确的应用程序。他们还从 Hummer Winblad 获得了 150 万美元的资金。点击查看完整简介[。](https://web.archive.org/web/20221006021739/http://www.beta.techcrunch.com/2006/06/22/postapp-launches-widgetbox-a-marketplace-for-widgets/)

**乙醚**

[以太](https://web.archive.org/web/20221006021739/http://www.ether.com/)超新星正式发射。他们创建了一个“易趣服务”，允许那些想在电话上出售时间的人这样做。在你的网站上放置一个以太标志——当有人点击它时，他们可以根据你设定的条件(价格、通话时间等)安排一个时间与你通话。).当你的电话响起时，电话那头有一个人已经给出了他们的信用卡信息，并在寻求你的建议。以太在三月进入[测试，我们在这里](https://web.archive.org/web/20221006021739/http://www.beta.techcrunch.com/2006/03/01/super-stealth-ether-to-launch-tonight/)报道了正式发布[。](https://web.archive.org/web/20221006021739/http://www.beta.techcrunch.com/2006/06/22/ether-pay-per-call-launches-for-public-use/)

**很快**

[Soonr](https://web.archive.org/web/20221006021739/http://www.soonr.com/) 是一个新的移动平台，我们之前已经在 TechCrunch 上[介绍过](https://web.archive.org/web/20221006021739/http://www.beta.techcrunch.com/2006/05/03/soonr-use-skype-on-your-cell-phone/)。到目前为止，他们推出的最有用的应用之一是能够在普通手机上使用 Skype(你只需支付从 Skype 到你自己手机的 Skype-out 费用，然后你就可以用 Skype 给你 Skype 列表上的任何人打电话)。Soonr 的 Mac 版本在 Supernova 发布。

**Zixxo**

看 Zixxo 有几种方法。对于用户来说，他们会根据你选择与他们分享的任何个人和人口统计信息，向你发送极具针对性的本地和全国优惠券。对于企业来说，这是一种非常划算的方式，可以接触到真正想收到优惠券的消费者。对于第三方来说，存在将用户和/或企业带到网络的收入共享机会。Zixxo 还很年轻，但是核心思想很强。如果他们开始获得动力，寻找一个潜在的快速收购该公司。

[StumbleUpon](https://web.archive.org/web/20221006021739/http://www.stumbleupon.com/) 是一款社交浏览应用。用户下载一个浏览器工具栏，可以找到不同类别的热门网站，对网站进行投票等。Stumbleupon 在 139 个国家拥有近 100 万注册用户，每天“绊倒”220 万个网站。广告商可以在目标受众面前以 5 美分的印象获得他们的广告。我使用这项服务。**
