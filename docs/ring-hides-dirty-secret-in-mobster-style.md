# 戒指以黑帮风格隐藏肮脏秘密

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/04/ring-hides-dirty-secret-in-mobster-style/>

我们不知道这为什么被称为鲻鱼戒指，但它肯定是一个“绅士的秘密”。在这个看起来很酷的金色小手指戒指里，埋藏着一颗蓝色的伟哥药丸。所以，当你表现得像乔·佩西一样，对你的搭档的新座驾大加赞赏时，你用戴着小指的手指做手势，你就可以在他面前炫耀你的男子气概了。

然后，后来，当你和你的卡波的女性朋友在一起时，即使你丢了你的阿玛尼西装，一丝不挂地在那里，你仍然可以吃药(真正的歹徒从不摘下他们的小戒指)。

它有两种不同的风格，取决于你是哪种无能的歹徒，来自佛罗里达的彭萨科拉。我们想收集一只小猫给拉什·林堡买一只。
