# 标签、微格式和 RSS 打败了语义网？

《IT 周刊》专栏作家蒂姆·安德森今天声称“语义网的到来需要很长时间”。他的论文有什么根据？

> 尽管“真正的语义网将比今天的互联网强大得多，不仅在搜索方面，而且在整合和聚集数据方面……谷歌的教训是，改进简单文本搜索的智能算法比引入语义标记的尝试更有影响力。取得的有限进展是自下而上的运动，如标签、微格式和 RSS(RDF 的一种方言)，而不是自上而下的努力，以说服组织发布 RDF 数据。”

他指出这样一个事实，即 [RDF(资源描述框架)](https://web.archive.org/web/20210304003404/http://en.wikipedia.org/wiki/Resource_Description_Framework)并没有广泛传播，仍然只是“在企业内部以一种更专业的形式”被使用。
