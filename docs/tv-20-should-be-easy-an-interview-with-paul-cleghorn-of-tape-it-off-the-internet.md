# 电视 2.0 应该是容易的:采访《把它从网上录下来》的保罗·克莱格霍恩

 [](https://web.archive.org/web/20210307070733/http://www.tioti.com/) 英国 TechCrunch 的 Mike Butcher 基于对新电视聚合网站[的创始人 Paul Cleghorn 的独家采访，写了一篇关于](https://web.archive.org/web/20210307070733/http://www.tioti.com/)[互联网电视的未来](https://web.archive.org/web/20210307070733/http://uk.beta.techcrunch.com/2006/11/01/whats-on-tonight-just-tape-it-off-the-internet/)的长文。我们之前对 TIOTI 的报道在这里。

克莱格霍恩的主要观点是，将电视搬上网络将比音乐产业的经历要容易得多。他的项目 TIOTI searches 和*聪明地从 iTunes、AOL video、亚马逊的 Unbox 服务和 BitTorrent 中寻找可用电视节目的链接。最终结果是一个功能丰富的聚合网站，Cleghorn 打算在上面销售广告。他说这就像“互联网的 TiVo”*

这是一篇[长而详细的帖子](https://web.archive.org/web/20210307070733/http://uk.beta.techcrunch.com/2006/11/01/whats-on-tonight-just-tape-it-off-the-internet/)值得一读——当你在那里时，你应该看看英国团队 TechCrunch 正在做的其余工作。他们在那边做得很好。
