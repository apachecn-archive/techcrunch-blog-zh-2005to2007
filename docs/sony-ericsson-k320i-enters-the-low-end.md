# 索尼爱立信 K320i 进入低端

K320i 有石墨灰、浅棕色和灰银色三种颜色，是 SE 面向 GSM 市场的最新低端手机。它有 4 倍数码变焦，VGA 摄像头，Java 支持，三频 GSM，蓝牙 2.0 和 15MB 的内部存储器。如果你对这些规格不感兴趣，记得我们说过它是低端的。

尽管是一款入门级手机，但它仍然拥有我们所期待的索尼爱立信的 K 系列风格。唯一主要的外观差异是有三种颜色可供选择。
