# 新 HTC 智能手机走向冲刺

随着 PPC-6800 的加入，Sprint 对智能手机的态度越来越认真。作为老化的 6700 的替代品，基于 HTC Titan 的设备是一个 WinMo 5，QWERTY 滑动发电站。像 BoyGenius 一样，我们希望看到它与 WinMo 6 一起发布，但升级应该很快就会推出。

这款手机充满了乐趣，包括 320×240 触摸屏、WiFi、蓝牙、microSD 和具有令人垂涎的自动对焦功能的 200 万像素摄像头。

遗憾的是，据说它不支持 3G 连接。但是有了 WiFI 和标准数据，你可能就不需要它了。
