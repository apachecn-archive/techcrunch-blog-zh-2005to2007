# 通过电线传输电视信号

![](img/ab5dd8f66f6de11ab6a8ab14206e107c.png)

一家强大的德国公司想出了一种方法，通过处理电流的同一根电线来传输电视信号。Devolvo 展示了其创造的 CeBIT，尽管目前这项技术一次只能传输一个电视信号。这项技术仍处于起步阶段(尽管各公司已经有一段时间可以通过相同的电线共享网络连接)，并使用 IP 通过电线传输信号，据说最大范围为 200 米。迪沃没有告诉任何人他们的新装置何时(或是否)可以实际使用，也没有告诉他们价格。反正那都是没用的信息。
