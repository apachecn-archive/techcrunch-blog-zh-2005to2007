# Weebly 推出博客平台，完成 65 万美元投资

[![](img/f410171d62a71fa25c2651c0fc7ad711.png) ](https://web.archive.org/web/20220630090031/http://weebly.com/) AJAX 网站编辑 [Weebly](https://web.archive.org/web/20220630090031/http://weebly.com/) 刚刚获得了 65 万美元的投资，并于今天推出了一个新的博客平台。这笔投资来自罗恩·康韦的 Baseline Ventures、Steve Anderson、Aydin Senkut、Paul Buchheit 和 Mike Maples。Weebly 计划将资金投入到新的个人和产品设计上。

Weebly 的核心产品是一个 [AJAX 网站编辑器](https://web.archive.org/web/20220630090031/http://www.beta.techcrunch.com/2006/11/05/checking-out-weeblys-ajax-site-creator/)，它使用模板皮肤和拖放网站内容小部件来创建个人页面，类似于你在各种个人起始页上控制布局的方式。以前，用户只能创建由文本、图像、视频等内容小部件组成的静态页面，以及 Google maps 或 adsense 等小部件。新的博客平台和所见即所得编辑器允许用户向他们的页面添加动态内容。

博客可以像任何其他 Weebly 页面一样添加到 Weebly 站点的导航栏中，除了一些专门的小部件。博客支持基本的博客功能，如发帖、评论、分类和存档。Webjam 在三月份筹集了 200 万美元，为用户的个人页面提供了一个类似的 AJAX 博客编辑器。然而，与 Webjam 不同，Weebly 允许在一个真正的 WYSIWYG 界面中编辑页面并在实际页面上发布。此外，每个帖子可以包含任何 Weebly 小部件，就像普通页面一样。像其他平台一样，帖子可以被起草、发布和标记。

此次升级还推出了一些新的小部件，比如 Twitter 徽章，不过你可以通过将代码放入标准的 HTML Weebly 小部件来嵌入任何小部件。Weebly 的创造者计划为流行的网络部件推出更多的包装器，并最终向社区开放平台。不幸的是，由于它是建立在自己的平台上，Weebly 无法利用其他流行博客引擎(如 WordPress)的现有插件。

真正的所见即所得编辑器是博客的一个受欢迎的补充，它已经减少了从 Geocities 到 Blogger 的网络发布的摩擦。
