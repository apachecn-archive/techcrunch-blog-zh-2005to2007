# 苹果 MacBook 精装版:喜欢，不喜欢，什么？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/11/rhinoskin-apple-macbook-hardcase-likes-dislikes-the-wha/>

如果你需要两个以上，可以享受免费送货和批量折扣。

一个典型的软边笔记本电脑外壳并不适合在户外保护你的 13 英寸 MacBook 免受撞击和刮擦。然而，用于箱子主体的坚硬的 ABS 塑料鼻皮。此外，内部有柔软的氯丁橡胶衬垫，可防止刮伤，并防止笔记本在运输过程中移位。

顶部的那个额外的箱子(也衬有氯丁橡胶)有足够的空间来存放苹果的电源适配器和一个旅行鼠标，如果你不需要使用它，可以很容易地拆卸下来。不过，这是你唯一的储物空间，这使得这款包更适合在家里或办公室使用，或者在不需要携带大量其他配件的短途旅行中使用。

如果你倾向于将笔记本电脑扔进汽车后座，摔在公园长椅或野餐桌上，或者只是对 MacBook 不够温柔，那么 RhinoSkin 适合你。如果你能忽略一些小细节的话。

用来保持箱子紧闭的厚重塑料夹子结实牢固。然而，它们在箱子的底部，所以当箱子打开时，夹子只是在周围摆动。如果犀牛皮把它们放在上面，夹子就不会碍事了。当然不是一个大问题，只是一个小烦恼。更重要的是夹子要结实，这些也是。

机箱的每一侧都有一个门，可以向下翻转，以便使用右侧的光驱和左侧的端口。这是一个小而周到的触摸，让你不必从机箱中取出 MacBook 来访问这些东西。

不幸的是，门是一种比箱子其余部分更薄、更不坚固的塑料，由两个小塑料小块固定在机身上，这与箱子其余部分更坚固的铰链完全不同。正常使用的时候不小心掉了一个。这让人摸不着头脑，因为案子的其他部分都很可靠。

箱子上没有把手。相反，RhinoSkin 包括两条可以快速连接的带子，让你可以像背包一样佩戴保护套。虽然我偶尔会喜欢打扮成《星球大战》中的突击队员，但我通常会在家里保密。不过说真的，我想痛打自己一顿，拿走我的午餐钱，因为我穿了这个箱子。

尽管如此，正如我之前所说，当你今年春夏去公园在小隔间和树下完成一些工作时，它是对你的 MacBook 的坚实而强大的保护。
