# T-Mobile 保持运转| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/10/t-mobile-keeps-spinning/>

周四，T-Mobile 车队宣布其赞助商德国电信决定继续赞助该车队，尽管在今年的环法自行车赛中出现了一些问题——尤其是兴奋剂事件。作为交易的一部分，T-Mobile 团队肯定会有一些不同的面孔。T-Mobile 表示，它已经同意车手和管理层的意见，并将拿出一部分工资建立一个 100 万欧元的基金，以帮助打击兴奋剂。

此外，作为交易的一部分，德国电信股份公司保留在出现进一步兴奋剂案件时立即终止承诺的权利。

T-Mobile 是职业自行车运动的最大赞助商之一，这项运动吸引了大量观众来到移动平台，尤其是在欧洲。今年的不幸事件对 T-Mobile 来说是个坏消息。根据目前的 T-Mobile 或其前身 Telekom，这家德国电信公司已成为 16 年来主要的自行车赞助商。这支车队在 1996 年和 1997 年分别与比约恩·里斯和简·乌烈芝一起赢得了两次环法自行车赛的冠军。
