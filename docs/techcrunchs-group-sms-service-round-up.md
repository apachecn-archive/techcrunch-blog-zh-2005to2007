# TechCrunch 的群组短信服务综述

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/28/techcrunchs-group-sms-service-round-up/>

![](img/6d473ac8628bd84fef96b64bc039506d.png)
Marshall Kirkpatrick，我们在 TechCrunch 的古怪表亲，当我们在复活节看到他时，他揍了我们一顿[写了一篇不错的文章](https://web.archive.org/web/20150807010327/http://www.techcrunch.com/2006/09/27/a-look-at-eight-multi-person-sms-services/)，详细介绍了最近出现的不同的[群组短信服务](https://web.archive.org/web/20150807010327/http://crunchgear.com/?s=SMS)。当然，有躲避球，但现在你有不止一个选择。

虽然他没有涵盖所有提供类似服务的公司(这很可能是不可能的)，但他确实列出了八大服务，不仅提供了细节，还通过方便的功能矩阵帮助您找到最适合您需求的服务，这样您就可以。

就个人而言，我们希望看到一个“霸王”服务，允许你与你使用的所有服务进行通信，但由于当时这是一个有限的市场，这可能要求很多。
