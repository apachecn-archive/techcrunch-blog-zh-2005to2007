# MySpace 微妙地转向强调视频

 [](https://web.archive.org/web/20220813222525/http://www.myspace.com/) 我们之前写过新闻集团[对](https://web.archive.org/web/20220813222525/http://www.beta.techcrunch.com/2006/10/09/is-fox-a-factor-in-the-googleyoutube-deal/)收购谷歌-YouTube 的不安。在过去的几周里，新闻集团一直在与 YouTube 进行一边倒的公关战——从新闻集团首席运营官·彼得·切宁最近的[评论称 60-70%的 YouTube 流量来自 MySpace (YouTube 称这是不正确的，MySpace 仅占 YouTube 总视频流量的约 5%)到华尔街日报的文章称国家新闻集团在谷歌交易完成前为收购 YouTube 做了最后的努力。鉴于视频剪辑的重要性(包括用户制作的和受版权保护的作品)，新闻集团感到不安的是，他们最大的竞争对手也是他们在 MySpace 上最大的收入来源。](https://web.archive.org/web/20220813222525/http://www.multichannel.com/article/CA6371185.html?display=Breaking+News)

这个周末，MySpace 的主页和用户页面上发生了一个微妙的变化，使得 MySpace 的视频更加突出和易于访问。主页的左上角现在有 4 个视频，吸引用户开始在 MySpace 上观看视频。用户个人资料页面现在有一个名为视频空间的部分，如果用户上传了视频，它会自动显示视频。在视频空间的视频下，观众可以选择将视频添加到他们的个人资料中，就像用户目前可以将任何 MySpace 音乐艺术家的音乐添加到他们的个人资料中一样——进一步个性化他们自己。观众还可以选择用文本评论来评论视频(YouTube 允许文本和视频评论)。

这些变化应该会对 MySpace 视频的使用产生重大影响。到目前为止，MySpace 还没有真正致力于吸引用户使用他们的视频服务。不知道 MySpace 视频的用户现在肯定会看到了。此外，MySpace 视频现在非常简单——用户上传他们的视频，视频会自动出现在他们的 MySpace 个人资料页面上；而在 YouTube 上，你必须上传视频，抓取 HTML 嵌入代码，然后粘贴到你的 MySpace 个人资料中。

MySpace 可以随时阻止 YouTube 视频显示在 MySpace 的用户档案上(就像他们阻止 flash 小工具中的 javascript 和出站链接一样)，迫使用户使用 MySpace 的视频上传系统。当然，这不会没有风险——用户的强烈反对可能会发生，类似于[脸书最近经历的](https://web.archive.org/web/20220813222525/http://www.beta.techcrunch.com/2006/09/06/facebook-users-revolt-facebook-replies/)。鉴于 MySpace 用户在创建个人资料和好友列表方面投入了大量时间，大多数 MySpace 用户可能最终会使用 MySpace 的视频系统，而不是跳槽去另一个社交网站。

根据最近基于访问市场份额的 HitWise 指标, YouTube 以 45.46%的市场份额排名第一，MySpace.com 视频以 22.99%排名第二。而 comScore 报告称 MySpace 在视频流和独立用户方面排名第一。
