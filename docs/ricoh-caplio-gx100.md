# 理光 Caplio GX100 | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/29/ricoh-caplio-gx100/>

理光最新的数码相机在一个小小的包装中具有强大的冲击力。GX100 有一个 24-72 毫米当量的高倍广角变焦镜头，通过可选的广角转换镜头，可以将广角放大到 19 毫米。这还不是最精彩的部分！理光的可拆卸电子取景器堪称数码相机世界的第一。取景器可以倾斜，因此在明亮的室外环境中拍摄更容易，并且提供 100%的覆盖范围，没有视差和完整的设置信息，就像在 LCD 上一样。1000 万像素的摄像头会让你所有的朋友都认为你是下一个安塞尔·亚当斯。好吧，不是真的，但如果你知道你在做什么，它可能会拍出一些精彩的照片。

使用 GX100 拍摄的图像有多种分辨率和压缩组合，包括 RAW 和 JPEG，以获得最高质量。附带的充电电池可以让你曝光 380 次或 35(？)用 AAA 电池。理光在 4 月份的最新降价为 399.99 美元(785 美元)。
