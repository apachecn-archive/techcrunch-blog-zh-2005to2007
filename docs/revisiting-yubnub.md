# 重访 YubNub–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/01/15/revisiting-yubnub/>

我在 6 月 13 日写了乔恩·阿基诺的 [YubNub](https://web.archive.org/web/20220701010030/http://www.beta.techcrunch.com/2005/06/13/internet-command-line-yubnub/) 回[。就在我开始写博客的两天后，所以没有多少人看那篇文章。](https://web.archive.org/web/20220701010030/http://www.beta.techcrunch.com/index.php?s=yubnub)

但是很多人都知道 YubNub。它是一个“互联网命令行”，允许任何人从 YubNub 命令行创建一个新功能并访问其他 web 服务。一个命令至少由两条信息组成——一个应用程序标识符和一个特定的命令。

一些更流行的命令是[这里是](https://web.archive.org/web/20220701010030/http://yubnub.org/documentation/jeremys_picks#Whois)。没有看到您需要的命令？创造自己的[。想不用去现场就用 YubNub？](https://web.archive.org/web/20220701010030/http://yubnub.org/command/new?name=)[安装](https://web.archive.org/web/20220701010030/http://yubnub.org/documentation/describe_installation)插件，跳过一些步骤。

奥利弗·斯塔尔(Oliver Starr)今天在 mobile runch 上写了一篇关于 YubNub 的文章，包括它在手机浏览器上的使用情况。
