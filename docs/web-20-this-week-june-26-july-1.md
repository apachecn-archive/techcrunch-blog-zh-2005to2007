**1。[我们出发了…第一个专注于 RSS 的风险投资基金宣布成立——1 亿美元](https://web.archive.org/web/20221129080418/http://www.siliconvalleywatcher.com/mt/archives/2005/06/were_off_to_the.php)** **“吉姆·摩尔和约翰·帕尔弗里已经启动了 RSS 投资者，资本为 1 亿美元。这是第一个专注于真正简单的聚合(RSS)的风险投资基金，RSS 是博客和企业新闻传播等媒体技术的核心技术。**

 *更多关于基金的信息[点击这里](https://web.archive.org/web/20221129080418/http://www.siliconbeat.com/entries/2005/06/30/you_gotta_be_nuts_a_vc_fund_dedicated_to_rss.html)。

**2。[奥莱利:Web 2.0 将建立在黑客的背上](https://web.archive.org/web/20221129080418/http://blogs.zdnet.com/BTL/?p=1565)**

Tim O'Reilly 今天在 Where 2.0 大会上谈到了召开一次以地图和定位技术为重点的会议的好处，并将其置于一个新平台 Web 2.0 的背景下

“谷歌地图和 Craigslist 是第一个真正的 Web 2.0 应用，这两个网站都没有参与——是一个开发者把它放在一起的，”他说。“黑客正在教这个行业该做什么。”

**3。[Troy Angrignon——转变范式:从 web 1.0 到 web 2.0 的 17 步心智进化过程](https://web.archive.org/web/20221129080418/http://www.troyangrignon.com/blog/_archives/2005/6/19/955532.html)**

**5。[尼克·布拉德伯里:微软、RSS 和关注](https://web.archive.org/web/20221129080418/http://nick.typepad.com/blog/2005/06/microsoft_rss_a.html)**

*“RSS 最强大的一点是，它将信息分解成单个的项目——如果你喜欢，可以说是一口大小的块——理论上，这使得工具和服务能够发现你在关注什么。对你关注的内容了解得越多，这项服务就能自动为你提供越多的相关信息(也能自动丢弃越多不相关的信息)。”*

我们对微软公告的讨论在这里是。他们的声明是一件大事。attention.xml 也是(或者至少可以是)。
