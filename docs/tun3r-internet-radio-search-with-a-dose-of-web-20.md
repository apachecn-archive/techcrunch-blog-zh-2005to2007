# TUN3R:带有 Web 2.0 味道的网络电台搜索

[![yun3r.jpg](img/6aa568864cbbbd28fd4aa63818a66e24.png)](https://web.archive.org/web/20211027122233/http://tun3r.com/)[【TUN3R.com】](https://web.archive.org/web/20211027122233/http://tun3r.com/)是一个定位于“Web 2.0 一代”的网络电台搜索引擎

TUN3R 不同于传统的广播门户网站，它用“The Dial”来问候用户:一个包含顶级互联网广播电台的网格。只需在表盘上拖动指针，听众就可以听到电台播放的音频样本，还可以浏览电台最近播放列表的节略版。

用户可以在电台的播放列表中搜索他们感兴趣的艺术家和歌曲，结果显示在“表盘”上

该索引包括 1，400 个电台，涵盖 40 多种语言和 25 种不同的音乐流派。TUN3R 通过向表盘上显示的电台提供增值服务来获得收入，这些增值服务包括用于提高可见性的定制彩色表盘图形、更频繁的播放列表采样和实时音频采样。

从界面上看，这有点像一个 1 美元像素的广告页面搭配一个 AM 收音机，但这种布局和想法确实很酷。该数据库是广泛的，并且能够查看播放列表以及样本音频改善了互联网广播搜索体验。如果你正在搜索一个在线电台，这绝对是一个值得访问的网站。
