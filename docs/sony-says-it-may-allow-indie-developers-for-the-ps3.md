# 索尼表示，它可能会允许 PS3 的独立开发者

[![ps3homebrew.jpg](img/34836f203bc3e1be6d82fd5899fd8f8d.png)](https://web.archive.org/web/20210301222341/https://beta.techcrunch.com/wp-content/uploads/2007/04/ps3homebrew.jpg "ps3homebrew.jpg")

受微软 [XNA 游戏工作室](https://web.archive.org/web/20210301222341/http://crunchgear.com/2006/12/14/xna-game-studio-express-from-microsoft/)的启发，索尼的菲尔·哈里森最近在一次采访中表示，索尼愿意向独立开发者开放 PS3。(是的，我们知道，Net Yaroze 几年前就这么做了，但当时没人在意。另外，今天的家酿社区能够生产比许多商业实体开发的软件更好的软件。)

哈里森说，从现在开始，这种发展将是视频游戏行业的一个“关键”方面，索尼希望成为其中的一部分。对于索尼来说，如果它真的完成了，这只能是一个好的举措。谁知道有多少年轻的、有抱负的程序员手上有下一个 *Katamari* ？
