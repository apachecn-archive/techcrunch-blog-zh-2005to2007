# Screencast-o-Matic 为网络浏览器带来了截屏创作

来自 Big Nerd Software 的[![screencast.png](img/7f38c5b70877e8cd21dc98fa30c14cc0.png)](https://web.archive.org/web/20220818075617/http://www.screencast-o-matic.com/)[screen cast-o-Matic](https://web.archive.org/web/20220818075617/http://www.screencast-o-matic.com/)为网络浏览器带来了截屏的创作。

通过使用一个 Java 小程序来记录音频和屏幕活动。然后，结果可以显示在网站上或导出到 Quicktime。

有各种免费和付费的 screencast 软件应用程序提供类似的功能，但 Screencast-o-Matic 是我见过的第一个严格基于浏览器的应用程序，本质上可以跨各种平台工作。

作为一项服务，它很容易使用。将显示一个框架，可以将其移动到适当的窗口上进行捕获。鼠标的移动可以通过一个红点来跟踪，记录就像按一个按钮一样简单。

感觉有点笨重，视觉上不会赢得任何美容奖项，显示选项有限。没有提供明显的嵌入代码，也缺乏支持将截屏视频直接上传到 YouTube 或类似视频托管网站的集成。

撇开问题不谈，对于临时用户来说，Screencast-o-Matic 是一个方便的工具，将来肯定会被其他人复制。
