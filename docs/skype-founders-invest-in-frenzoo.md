# Skype 创始人投资 Frenzoo

由 Skype 的四位联合创始人工程师成立的公司 [![frenzoo.jpg](img/d80fdb160967fadc58b09a913c90a1e6.png)](https://web.archive.org/web/20220522020734/http://www.frenzoo.com/) [环境声音投资](https://web.archive.org/web/20220522020734/http://www.asi.ee/)投资了总部位于香港的 3D 社交网络公司 [Frenzoo](https://web.archive.org/web/20220522020734/http://frenzoo.com/) 作为旨在将服务扩展到整个亚洲的 A 轮投资的一部分。

ASI 的所有者包括 Skype 的高级开发人员 Priit Kasesalu 和 Jaan Tallinn、Skype 的首席架构师 Ahti Heinla 和前 Skype 工程主管 Toivo Annu。

Frenzoo 使用户能够通过成为“初露头角的时尚达人”来与朋友联系。用户可以通过个性化 3D 角色来表达自己，并可以混搭各种品牌的衣服、鞋类、配饰、家具和生活用品。

Frenzoo 利用“RealFashion 3D 引擎”，利用图形硬件的最新进展，在真实的 3D 中模拟用户的角色、服装和生活环境。Frenzoo 应用程序是一个可下载的程序，提供本地社交网络浏览和即时消息。

交易条款没有披露。Frenzoo 目前处于封闭测试阶段，计划于 2007 年下半年推出。
