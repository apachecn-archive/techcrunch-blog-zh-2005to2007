# SlickrFrame 数码相框自动显示 Flickr 照片

![](img/4138702ac3ccfd802782cbfed27898e5.png)

这个 SlickrFrame 数码相框使用最近发布的开源软件来显示任何标有“slickrframe”字样的 [Flickr](https://web.archive.org/web/20210227142124/http://flickr.com/) 照片框架本身是由一台旧笔记本电脑制成的，并通过 Wi-Fi 连接到互联网。然后， [Slickr](https://web.archive.org/web/20210227142124/http://cellardoorsw.com/?page_id=4) 软件在 Flickr 上搜索所有带有这个神奇单词的图片。当它找到一张照片时，相框就会显示这张照片。简单，轻松，有趣。我们可以利用更多。

它的创造者看起来是一个谦逊、友好的小伙子，所以在工作的时候给他发一些难忘的照片来打发时间吧。拜托，不要穿超短裙，不要穿紧身衣，不要开柠檬派对。
