# 告诉我:机器人做搜索，你说脏话

mobile runch[的 Oliver 和 TC](https://web.archive.org/web/20201205024322/http://www.beta.techcrunch.com/2007/01/23/tellme-mobile-launches/) 的 Mike 测试了 TellMe 的最新版本，这是一种语音搜索服务，可以让你告诉机器人你在找什么。如果有必要的话，机器人会给你发一张地图，让你给朋友打电话或发短信告诉他们方位。

TellMe 需要一个很小的独立应用程序来绘制地图，而且现在似乎是免费的，所以这比从你的运营商那里支付 5 美元 411 有了相当大的改进。让我们祈祷它比标准的“嗯……我不确定我理解了你的意思。你能重复一下吗？”我们日常接触的机器人。
