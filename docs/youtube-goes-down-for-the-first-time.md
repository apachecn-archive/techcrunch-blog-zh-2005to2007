# YouTube 首次关闭

 [](https://web.archive.org/web/20220813114224/http://youtube.com/) 就在新的康姆斯克数据显示[YouTube](https://web.archive.org/web/20220813114224/http://www.beta.techcrunch.com/2006/08/15/comscore-myspace-video-traffic-doubled-in-july/)的流量创新高的同一天，该网站在今天早上宕机了六个小时，这是自 2005 年 2 月上线以来的首次意外宕机。尽管用户被告知新功能正在开发中，但媒体调查已经证实，导致网站瘫痪的实际上是数据库故障。美国时间晚上的定期计划停机很常见，但今天标志着该网站首次出现重大服务故障。

摆在桌面上的最大问题一直是关于 YouTube 的商业模式；今天的停电让你怀疑该公司是否能够扩大规模。YouTube 宣布，截至上个月，它每天提供超过 1 亿个视频。

**更新:**如今，YouTube 的问题已经遍布网络，得到了主流媒体的大量报道。康姆斯克今天也受到了巨大的压力，因为交通报道是在失败的当天发布的。对两家公司来说，这是一笔奇怪的公关横财，除非 YouTube 继续存在服务问题。
