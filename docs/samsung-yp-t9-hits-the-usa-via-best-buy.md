# 三星 YP-T9 通过百思买| TechCrunch 登陆美国

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/09/samsung-yp-t9-hits-the-usa-via-best-buy/>

百思买现在在其网站上以 199.99 美元的价格出售新款三星 YP-T9 DAP 播放器。当你考虑到它可以存储大约 1，000 首歌曲并播放 mp4 视频，而价格与一个只有音乐的 4GB iPod Nano 相同时，这是一笔相当不错的交易。它还配备了一个比苹果提供的更大的屏幕，一个调频调谐器，并与 PlaysForSure 配合使用。

遗憾的是，这不是我们听说过的 Bluetooh 版本，但在美国本土看到它真好，因为我们之前认为这将是亚洲独有的产品。如果你正在寻找 Nano 的替代品，这是一个很好的开始。
