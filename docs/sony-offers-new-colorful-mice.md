# 索尼推出新的彩色鼠标

![](img/4dbc6502df4ad0d3b4e91736e7197a21.png)

索尼宣布了两款新的旅行鼠标，颜色鲜艳，看起来有趣又幼稚。SMU-C3 是一款双色光电鼠标，而 SMU-M10 则更加小巧简洁。两者都支持 Windows 和 Mac OS X，并具有不同的颜色模式。它们都使用 800dpi 的传感器。M10 专为旅行设计，配有 USB 电缆包装，因此在旅途中不会缠绕在一起。C3 有一个可调节的绳索用于防缠绕措施。

它们还没有在美国上市，但目前已经在英国上市。SMU-M10 型大约要 30 美元，而 SMU-C3 型大约要 35 美元。
