# 歌曲依旧:齐柏林飞艇来到 iTMS

![](img/c1690b937a51d6266b0ae600a1e4571d.png)
漫谈下去！

最近使用过其他在线音乐服务吗？iTunes 音乐商店只是不漂浮你的船？没关系。苹果公司偶尔会表现糟糕，但现在有理由回到乔布斯公司钟爱的经济作物上来。齐柏林飞船将发行“母舰”，这是每个人都喜欢的吸毒乐队的最终专辑。

点击率会在那里。《全洛塔爱》、《年少轻狂》，甚至《阿喀琉斯最后的抵抗》都会奇妙地装点你的耳朵。这也是苹果的一大打法。随着 Zeppelin 在 iTunes 上首次亮相，现在只剩下披头士的目录了。目前还不知道“母舰”的价格，但我想如果所有的曲子都在的话，大概在 100 美元到 150 美元之间。
