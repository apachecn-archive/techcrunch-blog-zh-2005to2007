苹果电视被黑支持USB驱动器

我梦想成为世界级的黑客，只是因为我虚荣，想看到自己的名字在谷歌上第一个出现。黑客做了伟大的事情，这让我喜笑颜开。我更喜欢苹果产品被黑。如果你一直对苹果沉默寡言的继子苹果电视犹豫不决，那么这可能会影响投票，因为它已经被黑客攻击为使用USB驱动器作为主要驱动器，一切都可以完全同步。伙计们，别开玩笑了。

说明

该补丁是为Apple TV软件版本1.0编写的，因此经过了测试。如果你有1.1版本，补丁可能无法工作。请告诉我们你是否能让补丁在1.1上运行。

您需要什么:

*支持ssh的苹果电视。如果你还没有启用它，你可以参考这篇文章来启用它。有关如何在不打开机箱的情况下启用ssh的说明，请参考这个wiki页面。
*英特尔Mac或基于英特尔的*nix。这需要运行脚本来远程修补Apple TV上的内核。也许可以使用cygwin在Windows下运行安装脚本。但是，我们没有尝试过这个。
*已安装的Mac OS X 10.4英特尔版本。或“/系统/资源库/扩展”文件夹中内容的完整副本。
*来自Apple TV的“mach_kernel.prelink”文件的原始未修改副本。如果你的Apple TV上的内核还没有被修改，你可以告诉脚本从那里得到它。否则，您可以从Apple TV Software 1.1 update获得该文件，此处提供。
*使用“日志式HFS+”格式化的外部USB驱动器。这是苹果电视期待的格式。

程序

*完成上述所有步骤后，解压缩zip文件并阅读自述文件。在里面，你会找到如何运行脚本和如何让USB驱动器工作的说明。
*重要提示:在运行补丁脚本之前，请先备份您的Apple TV的内容。如果说打补丁有一个重要的步骤，那就是备份你的Apple TV。
*备份好Apple TV后，运行脚本“install-atvrhd.sh”并按照说明进行操作。这个过程应该不到5分钟。一旦苹果电视上的内核被修补，设备将重新启动。

使用它

*在未插入USB驱动器的情况下打开Apple TV。等待介绍序列(飞行电视屏幕等)。)然后插入USB驱动器。
*一旦插入，内置硬盘的内容将被复制到外置硬盘。这个过程可能需要很长时间。要解决这一问题，我们建议您首先擦除内部硬盘的内容，这样就不需要复制内容。【The Apple TV将在内容复制后自动重启。
*此时，您的Apple TV将使用外置硬盘作为其主存储器。
