# 活力季风二降低 CPU 温度，没有所有的液体冷却管

虽然英特尔和 AMD 可能会采取更多措施来降低处理器的工作温度，但超频仍然会导致一些高温，尤其是在机箱中塞满其他高性能组件的情况下。这通常会导致建筑商和买家转向高效但昂贵的液冷系统。小型游戏电脑供应商 Vigor Gaming 一直在投资另一种解决方案:Monsoon Active TEC CPU 冷却系统。以下是 Vigor 对其工作原理的解释:

> 热电冷却器(TEC)或珀耳帖冷却器是一种固态热泵，利用珀耳帖效应转移热量。现代商用 TEC 由许多电串联和热并联的 p 型和 n 型半导体对组成。这些耦合被夹在两个导热且电绝缘的衬底之间。热量的泵送方向可以通过改变充电 DC 电流的极性来改变。

你都明白了吗？嗯，只要知道这是一种冷却超频处理器的有效方式，而没有在 PC 内部设置液体冷却系统的麻烦或费用。至少 ExtremeTech 上的一篇评论是这么说的。该网站称，虽然第一版没有给人留下深刻印象，但第二代实现了降低温度的承诺。尽管代价是 149 美元和一些高于平均水平的风扇噪音。
