# 从手机发送照片——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/11/sendphotos-from-mobile/>

SendPhotos 5 可以让用户在多台电脑上同步他们的相册，用户很快就可以分享他们的手机照片。Avanquest 宣布即将发布 SendPhotos Mobile，允许用户自动将拍照手机拍摄的照片上传到“在线储物柜”，让用户直接访问图像。这些可以在手机或电脑上查看。

SendPhotos 5 软件和 2GB 在线存储现在售价为 29.95 美元，并将于今年晚些时候与 SendPhotos Mobile 兼容(价格待公布)。
