# 音频文件:苹果能承担得起偷懒吗？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/10/the-audiofile-can-apple-afford-to-slacker/>

收音机给了你两样重要的东西:它让你不陷入音乐的窠臼，你不必决定接下来听什么。既然 Last.fm 和 Slacker 等可定制的网络电台服务风靡一时，苹果让 iPod 用户分一杯羹的时机再好不过了。我嗅到了 iPod +懒鬼的伙伴关系，现在苹果让 GooTube 跳上了 iPhone 列车…

我花了几年时间才意识到，也许对没有音乐的人来说，听音乐并不是一件坏事，是我对收音机(包括调频和互联网)重新燃起的兴趣做到了这一点。现在，我不断地在这些网站上查阅资料，这无疑拓宽了我的视野。(快讯:爵士书呆子现在检查印第摇滚乐队！)

记得收音机吗？你并不拥有音乐，但是你可以免费听到它，如果它让你想起，你就去买它。网络电台很活跃，它开始与便携式播放器配合使用:看看 SanDisk 和 Rhapsody、iRiver 和 Rhapsody、SanDisk 和 Yahoo、Sonos 和 Rhapsody 的幸福婚姻，等等。

然而，对于 iPod 来说，它在巅峰时期是孤独的。苹果公司*仍然*坚持整个订阅服务的想法(像 Napster、Rhapsody 和 Yahoo 一样)，阻止 80%的音乐播放器拥有者在没有做学期论文价值的研究的情况下定期获得新音乐。当然，你可以得到一个 iPod Shuffle，并不断地用你自己的音乐库中的随机曲目自动更新上面的音乐。

除了 iTunes，还有其他兼容 iPod 的数字音乐的在线来源。但是除了 eMusic 之外，由于其无数字版权管理的 MP3，它在 iPod 上的表现一直很好，很难说有多少 iPod 用户真正使用它们。

**懒鬼**
网络电台世界中出现的一个更好的东西是一个叫懒鬼的网络电台服务，该公司即将放弃专用便携式音乐设备的圣杯。原型到处都是，它们看起来棒极了:WiFi，4 英寸的屏幕，我完全支持竞争，但如果苹果这么早就加入进来，那真的太棒了，因为我开始厌倦了滚动我的 80GB iPod 来寻找我有一段时间没听的东西。

这家在线 CD 店让你付费从其他用户那里购买光盘。但是你可以把你自己的 iTunes 库上传到 Lala 的服务器上，不管你的音乐是翻录的还是下载的(只要没有 DRM)。这意味着你可以从任何联网的电脑上访问你的音乐，甚至可以直接拷贝到你的 iPod 上。

在大约一个月的时间里，Lala 的用户可以从一个主要的唱片公司(华纳音乐集团)免费点播完整长度的歌曲——这是一个巨大的进步。但是那个服务[黑了](https://web.archive.org/web/20150909125012/http://arstechnica.com/news.ars/post/20070703-the-day-the-music-died-lalas-free-streaming-goes-dark-will-return.html)；该公司引用了服务器过载，我肯定这是真的，但我怀疑它的立即成功(尽管界面不稳定)可能吓得一些标签退出。

**其他赠品**
iTunes 上总有一些免费的赠品，还有一些免费的音乐下载网站，尽管通常它们都没有你听说过的东西。此外还有 SpiralFrog，这是一个大胆的新广告支持的免费下载服务，拥有来自环球唱片公司的 70 万首歌曲(这本身就是头条新闻)。当然，这些曲目都是 WMA 格式的，所以 iPod 将再次被从免费音乐盛宴中剔除。

虽然如果 iTunes 上有订阅服务就太好了，但你仍然可以用录音程序“时移”网络电台，比如窃听，它只是抓取你电脑播放的任何声音。如果你真的下定决心，你甚至可以使用外部声卡，如 M-Audio Transit USB，并将输出连接到输入，并录制模拟信号。(人们仍然用磁带录音机录下收音机里的东西，许多 MP3 播放器都有 FM 录音功能。)

如果下一代意义重大的 iPod(不仅仅是容量增加或内部变化)内置 WiFi(这仍是个问题)，Slacker 将是苹果接管数字广播并让 iPod 用户走出盒子的理想合作伙伴。想象一下，你可以无线更新你的 iPod，播放适合你口味的音乐——尽管我确信“免费”不符合苹果的愿景。
