# Zune 打折，导致微软亏损 2.89 亿美元

这是有充分理由的。当国内最大的科技公司(世界？)决定与 MP3 播放器的实际变体进行正面交锋，我们必须对此进行报道。当该公司利用内置 WiFi 等创新时(不管有多没用)，我们也会报道。我们喜欢 Zune——尤其是 Blake，但你们都知道他是个 PC 粉丝。

遗憾的是，我们不得不报告说，微软上个季度花费了 2.89 亿美元，这几乎不是该公司当时希望公布的数字。虽然我们从内部消息来源得到保证，Zune 的下一个版本(它是*而不是*Zune II)将解决当前模式的许多缺点，并且该集团有一个在工作中实现盈利的路线图。

这是个好消息，因为周末我们去了西雅图当地的积压商品和折扣店，我们震惊了——震惊了！——找到一堆售价 168 美元的近一打全新未开封的 Zunes。这比我们在 ze Web 上看到的 229 美元的现行价格低了 25%,对于想买 Zune 的人来说，这是一笔不错的交易。

然而，问题是:真的有人在寻找 Zune 吗？还是每个想要的人都得到了？或者这些只是几个从缝隙中掉进贫民窟的小商店？上面的数字意味着这些都有可能。

你们都是怎么想的？
