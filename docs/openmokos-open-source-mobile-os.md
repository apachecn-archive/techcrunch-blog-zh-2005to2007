# OpenMoko 的开源移动平台

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/15/openmokos-open-source-mobile-os/>

肖恩·莫斯-普尔茨(Sean Moss-Pultz)今年试图改变手机/设备行业。他希望消费者像关心台式机或笔记本电脑上的操作系统一样关心手机上的操作系统。 [OpenMoko](https://web.archive.org/web/20131104155652/http://www.openmoko.com/) 和 [FIC](https://web.archive.org/web/20131104155652/http://www.fic.com.tw/) 的 Neo1973 智能手机是这种变化的开始。

基于 Linux 的移动平台是完全开源的。与 Symbian、微软和大量其他可用的手机操作系统不同，OpenMoko 为开发人员提供了一种创建应用程序和提供服务的简单方法，而无需支付每部手机的许可费用，也不会有移植到多个平台的麻烦。

Gizmodo 很好地总结了今年平台和手机的首次发布。多点触摸屏手机将在未来几个月内发放给开发商，Moss-Pultz 表示，2007 年 9 月是 Neo1973 以 350 美元的解锁价格进入消费者手中的目标时间。对于一部 GSM 手机来说，这并不坏，因为它可以根据你的需求而增长，而不是为了获得新功能而不得不花一大笔钱升级到新手机。

上面的手机图片只是开发者的模型的样子。OpenMoko 的产品经理 Moss-Pultz 告诉 CrunchGear，他正在考虑装一个滑出式 QWERTY 键盘，因为他打算让手机和操作系统成为移动计算平台，为了尽可能快地输入文本，键盘是合理的选择。如果你想要更多的角度，Gizmodo 有几个设备的图片，但是，在跳跃之后，你会看到操作系统本身的镜头，所以你可以知道接下来会发生什么。
