# 沃达丰推出 YouTube 手机

英国拥有所有值得流口水的移动商品。欧洲主要市场的沃达丰客户将直接从[沃达丰直播获得 YouTube 内容！](https://web.archive.org/web/20200806014514/http://online.vodafone.co.uk/dispatch/Portal/appmanager/vodafone/wrp?_nfpb=true&_pageLabel=template05&pageID=IM_0001&redirectedByRedirectsImplServletFlag=true)在他们的手机上。:感叹:

该协议规定，YouTube 将每天特别为沃达丰的客户提供精选的新视频。但这还不是最精彩的部分。用户可以从手机上传自己的视频到 YouTube，也可以将自己喜欢的视频转发给家人和朋友。
