# 不升级到 Vista 的六大理由

我用的是运行 Mac OS X 和 Windows XP 的电脑。我完全打算在 Leopard 发布时升级到 Leopard，但是，我暂时会远离 Vista。我正计划对为什么这样做进行全面的抨击，但是《计算机世界/个人电脑世界》成功地指出了[我不打算购买和升级到新的 Windows 操作系统](https://web.archive.org/web/20200915032428/http://www.pcworld.com/article/id,128669-page,1/article.html)的所有理由。

作家迈克·埃尔根(Mike Elgan)抓住了这次发布会让我恼火的一切:高昂的定价(Home Premium 为 239 美元，升级版为 159 美元；终极版售价 399 美元，升级版售价 259 美元)，XP 还没有死(尽管微软不会介意你是否认为它已经死了)，升级和安装 Vista 肯定是一个非常耗时的项目， [Vista 还没有完全成熟，不能用于一般用途](https://web.archive.org/web/20200915032428/http://crunchgear.com/2007/01/23/vista-service-pack-1-a-band-aid-for-a-baby/)或[游戏](https://web.archive.org/web/20200915032428/http://crunchgear.com/2007/01/23/vista-gaming-10-reasons-it-will-be-hot-5-reasons-were-worried/)，你将需要新的硬件来获得不错的性能(那台 18 个月的旧电脑就是不行)。

最后，如果你无论如何都想买一台新电脑，Elgan 建议 Vista 是换成 Mac 的一个好理由。事实是，OS X 已经做了 Vista 做的大部分事情，而且已经做了一段时间了。除非你依赖于没有 Mac 版的纯 Windows 硬件或软件，或者你是一个铁杆 3D 游戏玩家，否则值得研究一下转向 OS X 的问题。 [Leopard 承诺将非常出色](https://web.archive.org/web/20200915032428/http://www.apple.com/macosx/leopard/index.html)如果苹果公司遵循其过去的定价，当它推出时，升级只需 129 美元。
