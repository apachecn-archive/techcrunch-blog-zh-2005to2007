# 初创公司聚合的统称是什么

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/07/what-is-the-collective-term-for-a-mashup-of-startups/>

几个月前，我和金融城律师事务所的合伙人克莱夫·格林加斯(T2)谈论英国创业市场。Clive 过去是一名软件开发人员，现在是一名专注于电子商务、互联网和 IT 的商业律师，他有着独特的见解。我们开始讨论今天的软件开发有多么不同，这主要是因为开发工具的简单性和通过整合现有技术快速创建新公司的无限机会。

在讨论的公司中，我们强调了 [OnOneMap](https://web.archive.org/web/20130627211302/http://www.ononemap.com/) 、 [Zoomf](https://web.archive.org/web/20130627211302/http://www.zoomf.com/) 、 [nestoria](https://web.archive.org/web/20130627211302/http://www.nestoria.co.uk/) 、 [TrustedPlaces](https://web.archive.org/web/20130627211302/http://www.trustedplaces.com/) 、 [Reevoo](https://web.archive.org/web/20130627211302/http://www.reevoo.com/) 和 [CrowdStorm](https://web.archive.org/web/20130627211302/http://www.crowdstorm.co.uk/) ，并同意很高兴看到所有这些初创公司都推出并做得如此之好，但 Clive 也警告说，其中一些公司过度使用和潜在依赖第三方技术，因为它们可能有退出战略，会被更大的公司收购。

> “有什么要紧的；一些投资者担心，混合网站过于依赖第三方 API。投资者的尽职调查可能会调查网站的架构和代码，以确定移除一个混杂的元素或 API，然后用另一个替换的难度有多大。你可能会问，如果整合后的网站运行良好并带来收入，那么删除和替换混杂的元素为什么如此重要？一个原因是减少特定 API 或混合元素所有者停业的影响。当然，当你聚合亚马逊、易贝、谷歌、微软和雅虎这样的巨头时，这种风险微乎其微。但是，如果你只与巨头合作，甚至会有风险。如果一个 Web 2.0 网站与某个特定巨头的 API 集成得太紧密，那么代码分离的风险和成本可能会吓跑潜在的购买者，从而关闭投资者的潜在出口。

![Nestoria - UK property search engine](img/487b6913e8e10b4afd7ec29256ca1749.png "Nestoria - UK property search engine")

如今有一家公司确实在使用谷歌地图，那就是 nestoria，但他们似乎已经注意到了 Clive 的警告，并正在与一些真正有趣的众包项目建立开源合作伙伴关系。第一个合作伙伴是[geograph.org.uk](https://web.archive.org/web/20130627211302/http://www.geograph.org.uk/)，他自称的任务是“制作**免费**可访问的不列颠群岛的教育有用的地理位置照片档案”。另一个合作伙伴是 openstreetmap“一个旨在为任何想要的人创建和提供免费地理数据(如街道地图)的项目”。这里有一个来自怀特岛的例子。

当然，现在这两个项目都还处于胚胎阶段，他们拥有的数据非常稀少，这意味着 nestoria 仍然依赖商业许可的地图和数据源。看看风险投资者开始对他们进行尽职调查时会发生什么，以及这是否会以某种方式影响他们的估值，这将是一件有趣的事情。

另一家房地产搜索引擎 Zoomf 采取了一个稍微不同的角度，决定与另一家名为 Trusted Places 的初创公司合作。中联重科知道，购买一处房产的最终决定不仅仅取决于房产的价格，还取决于周围的地理位置和设施。然而，仅仅知道你可能购买的房产附近有学校、酒吧和/或餐馆并不能告诉你它们是好是坏。因此，Zoomf 没有试图自己有机地获取这种社会知识，而是明智地寻求与可信的地方合作。这是一个伟大的双赢交易，因为 TrustedPlaces 扩大了其品牌的覆盖范围，并可能通过 Zoomf 网站找到新的评论者。另一方面，Zoomf 通过增加另一层知识来丰富用户的资产搜索。

我相信还会有更多类似的合作，我们也很乐意听到这些合作。昨晚我读到一篇关于 T2 的文章，他们与 CrowdStorm 的合作更多的是社交而不是商业方面，在 Last.fm 上组建了一个收听小组，这肯定会比听所有蹩脚的百货商店音乐要好——只要它不是 Wham 和去年圣诞节。
