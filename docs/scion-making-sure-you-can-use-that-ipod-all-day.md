# Scion 确保你可以整天使用 iPod

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/09/scion-making-sure-you-can-use-that-ipod-all-day/>

![](img/77d30d902f84c35bdb6cf66cb0316516.png)

虽然 Scion 目前只生产三种不同的型号，但你可以放心，你的 iPod 将会无缝集成到其中的任何一种。接穗宣布，其整个 2008 年阵容将具有完整的 iPod 连接。汽车配备的标准 160 瓦先锋系统现在将具有 iPod 控制功能，并将显示艺术家、专辑和曲目名称等信息。

一个普通的 iPod 适配器将被包括在内，以便在你驾驶并随着一张旧的 *Rush* 专辑摇滚的时候连接并给 iPod 供电。不要忘记安全也很重要。接穗将包括安装在车轮上的音频控制，这样你就不必把你的眼睛离开道路。
