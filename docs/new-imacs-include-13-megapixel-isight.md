# 新款 iMacs 包括 130 万像素 iSight | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/14/new-imacs-include-13-megapixel-isight/>

![](img/24f5fa1484069af3a6ccec440a042718.png)

当然，新的 iMacs 有一个可爱的新键盘和更新的外观。是的，在价格、零件、速度等方面有一些改进。你知道苹果也悄悄升级了 iSight 吗？新款 iMacs 和 Macbook Pros 内置 130 万像素 iSight 摄像头，可以拍摄 1280×1024 分辨率的视频。为什么像 iChat 和 Photo Booth 这样的应用没有利用起来？处理相机和软件占用太多 CPU。
