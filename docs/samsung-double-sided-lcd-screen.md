# 三星双面液晶屏

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/01/04/samsung-double-sided-lcd-screen/>

到目前为止，三星今年在世界上第一个双面 LCD 屏幕上表现良好，该屏幕能够在屏幕的两面显示两个独立的图像。这怎么可能？当然，随着三星新的双栅极 TFT(薄膜晶体管)架构的使用。新的双面屏幕不是一次只能操作一个像素的一个门，而是有两个门，允许它在正面显示一个图像，在背面显示第二个图像。该显示器还利用了萨米公司专有的 ASG(非晶硅门)技术。ASG 可以适应 TFT 栅极的增加而不增加驱动器集成电路的尺寸。每个显示器都是 2.22 英寸宽，QVGA 分辨率，支持 265k 颜色，透射主屏幕上的亮度为 250 尼特，辅助反射屏幕上的亮度为 100 尼特。在 CES 展会上找吧…哈哈哈一个显示器上的显示器上的显示器。我笑得前仰后合。
