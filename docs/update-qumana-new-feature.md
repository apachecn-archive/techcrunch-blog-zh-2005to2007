# 更新–Qumana(新功能)

## Qumana 推出更新功能

 [Qumana](https://web.archive.org/web/20201201234529/http://www.qumana.com/)(TechCrunch[简介](https://web.archive.org/web/20201201234529/http://www.beta.techcrunch.com/?p=18))是一个[所见即所得](https://web.archive.org/web/20201201234529/http://en.wikipedia.org/wiki/Wysiwyg)的博客编辑器，几乎可以与所有的博客软件兼容。查看我们以前的个人资料了解更多信息，但你可以拖放图像，让文本看起来完全像你想要的那样，没有 html 或格式标签，甚至在离线时也可以编写和编辑帖子。

在我们看来，它是最好的软件(注意:[其他人不同意](https://web.archive.org/web/20201201234529/http://blog.softtechvc.com/2005/07/weblog_posting_.html))，而且不像他们的大多数竞争对手，它是免费的。

昨天 Qumana [宣布了](https://web.archive.org/web/20201201234529/http://blog.qumana.com/blog/_archives/2005/8/10/1126874.html)一个新功能:能够改变以前发表的帖子。请注意，这项新功能适用于 Qumana 和非 Qumana 帖子，因此您现在可以开始使用他们的软件，并修改您使用现有博客软件撰写的旧帖子。
