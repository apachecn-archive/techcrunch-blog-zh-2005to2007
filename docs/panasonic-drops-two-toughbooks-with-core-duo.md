# 松下扔掉了两部配备酷睿双核处理器的笔记本电脑

松下的阿基里斯式(除非你击中正确的地方，否则坚不可摧)系列笔记本电脑最近更新了两个型号，CF-30 和 CF-19。这两款笔记本通过了从 36 英寸跌落 26 次、地面车辆振动、键盘溅水、极端温度、灰尘和海拔阻力等测试。也被称为“婴儿”系列测试。

CF-19 是左边的平板电脑，CF-30 是右边的标准笔记本。两者都有英特尔酷睿双核处理器和 80GB 硬盘，加上高达 4GB 的内存、GPS、蓝牙、WLAN 和一些 3G 功能，正如我们在这里看到的。3G 是 EV-DO 和 HSDPA，这意味着你可以使用 Sprint、Cingular 和威瑞森。
