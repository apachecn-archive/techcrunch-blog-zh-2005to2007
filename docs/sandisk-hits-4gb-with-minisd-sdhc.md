# 闪迪的 miniSD SDHC 达到 4GB

SanDisk 发布了 4GB miniSD 高容量(SDHC)闪存卡。在你兴奋之前，我应该指出这些东西还没有被任何设备支持。

![](img/46c860c3502500775dab2f85c7b604e5.png)
这些 SDHC 卡符合新的 SDA 2.0 标准，最终将允许 miniSD 包含高达 32GB 的容量。真的很神奇。这么小的东西却有这么大的容量。明年它们应该会被广泛使用，但是现在你可以只是坐着流口水。
