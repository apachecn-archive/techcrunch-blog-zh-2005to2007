# PlayStation 3 在纽约的活动| TechCrunch

我爱加利福尼亚和其他地方，但是所有这些发生在纽约的游戏事件让我希望我能出城一段时间。索尼将举办另一场面向公众的 PlayStation 3 派对(就像他们在 11 月 1 日举办的派对一样)。你可以玩 PlayStation 3 的发布游戏，还会有免费食物。如果你在纽约，有空的话，在下午 4:00 登记入住之前，去曼哈顿伍斯特街 160 号。
