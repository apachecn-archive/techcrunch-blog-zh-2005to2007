# 诺基亚 N95 预览版| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/06/nokia-n95-preview/>

诺基亚了解智能设计。与上帝不同，诺基亚密切关注产品的生命周期，他显然把香蕉做成优雅地适合人手的形状。虽然许多 N 系列手机是相似的，但它们通常是对以前产品的改进，在更小、更优雅的包装中提供惊人的功能。

诺基亚 N95 是诺基亚最新的媒体手机。这款超轻手机有一个 500 万像素的主摄像头，150MB 的内置内存，前面有一个迷你视频会议摄像头和 QVGA 屏幕。添加一个带有 2GB 卡的 MicroSD 插槽，不确定这是否是标准配置，但应该是，以及一个可与大多数其他产品媲美的音乐播放器，您就可以开始工作了。GPS 和 WiFi？那只是肉汤。

我买这款手机才两天，所以我没有时间真正深入了解网络功能，但我很高兴地告诉大家，从设计角度来说，这是我迄今为止见过的最好的 N 系列机型。在手机的边缘，我们发现了一个 3.5 毫米的插孔，一个小舱口下的热插拔 MicroSD，一个迷你 USB 插孔，相机按钮和音量控制。当你向上滑动手机时，它会显示一个键盘。当你向下滑动它会显示轨道控制按钮。最棒的是，这款手机由柔软的橡胶塑料包裹，重约 4 盎司。这就像他们拿走了过去让他们的手机可怕的一切——专有的 USB 端口、奇怪的闪存隐藏孔和沉重的电池——并做出了真正出众的东西。

60 系列手机并不适合所有人。作为一个智能手机平台，S60 是一个难对付的野兽。这并不是一个真正的智能手机操作系统，但它隐藏了许多强大的功能。我一直对 S60 手机中的电子邮件客户端感到失望，这款手机也不例外。幸运的是，N95 有一些惊人的用户界面改进，包括非常酷的旋转菜单系统和“3D 铃声”，只需快速点击按钮，即可将常规铃声变成奇怪的立体声墙。

然而，如果你喜欢 S60，我对 N95 的初步印象是，它比早期的 N 系列手机，甚至是 N80，都有了很大的改进。凭借强大的 CPU 和出色的图形处理器，N95 显示了小巧、轻便、智能设计的智能手机的价值。更多即将到来。
