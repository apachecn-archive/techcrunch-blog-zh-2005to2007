# 野外的天狼星高跟鞋 100

视频 如果你想知道高跟鞋在正常照明下的样子，而不是产品拍摄照明下的样子，那就来吧。还有一个 UI 如何工作的视频，向你展示了浏览和听霍华德的样子。

UI 放大和缩小看起来相当不错，但是从背面伸出的“扩展”电池确实使播放器的设计变得难看。当然，我们更希望得到扩展的电池，而不是耗尽电量，但对于时尚人士来说，你可以选择嵌入高跟鞋的超薄电池。总而言之，它相当薄，应该足够便携，可以像手机一样放在同一个口袋里。
