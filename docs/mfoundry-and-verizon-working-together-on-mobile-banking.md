# mFoundry 和威瑞森合作开发手机银行

移动应用平台提供商 mFoundry 宣布，它正与威瑞森无线公司合作开发银行和金融服务解决方案，威瑞森有望支持 mFoundry Spotlight 金融平台。Spotlight 平台允许容纳多种类型的金融服务内容，包括支持银行、经纪、信贷和支付解决方案，目前可以在市场上最流行的移动设备上运行。
