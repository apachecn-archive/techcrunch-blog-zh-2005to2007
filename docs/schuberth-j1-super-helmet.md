# 舒伯斯 J1 超级头盔

![](img/7f9148ccab4713413f3c1e032bda0cd2.png)舒尔伯斯 J1 头盔看起来很酷，有点让我想买辆自行车，这样我就可以用了。在这种与众不同的未来风格之下，是一系列旨在保护你大脑的高级功能。

头盔有一个可拆卸的下巴护板，一个光学 1 级无变形遮阳板，集成的遮阳板和一个通风系统，可以防止你的大脑沸腾——我见过这种事发生，太可怕了。这可能要花 660 美元，但与把头撞在人行道上相比，这笔投资似乎并不算太糟。只是一个想法。
