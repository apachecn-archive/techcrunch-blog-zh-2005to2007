# 雅虎！选择 WinMo–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/07/yahoo-go-for-winmo/>

![](img/4bc55316d9d2775ac11f968b4387d180.png)
[雅虎！Go for Mobile](https://web.archive.org/web/20200920052544/http://mobilecrunch.com/2007/02/12/yahoo-mobile-serves-ads-and-go-goes-gamma/) 现已推出 gamma 版本，适用于 Windows Mobile 设备。这个版本包含了 Symbian 人已经吹嘘过的所有东西，包括电子邮件、oneSearch(雅虎的移动搜索产品)、照片分享和 RSS/news ticker 类型的订阅。查看雅虎的移动网站，了解更多关于兼容性的信息，但是如果 T4 运行 WinMo T5，它可能会运行雅虎。
