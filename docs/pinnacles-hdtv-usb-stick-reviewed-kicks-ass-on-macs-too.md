# 品尼高的高清电视 u 盘评论，踢屁股(在苹果电脑上也是)

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/01/pinnacles-hdtv-usb-stick-reviewed-kicks-ass-on-macs-too/>

我们之前已经[提到过 130 美元的 PCTV HD Pro Stick](https://web.archive.org/web/20150914190915/http://crunchgear.com/2006/09/20/pinnacle-pctv-hd-pro-stick/) ，认为它非常漂亮，但实际上还没有机会玩一玩。不过没关系，因为 EverythingUSB 上的魔法精灵已经为我们做了这项工作，他们对这款设备做了我们所见过的最全面的评论，如果不是世界历史上最全面的话。

这值得一读，但我们会为你分解它:它或多或少的岩石，虽然软件的“古怪”，像梅格·瑞恩。附带的天线是经过深思熟虑的，它是 Mac 兼容的，你知道，规则。“为 iPod 录音”的功能很诱人。

跳跃后的坏消息。

我们希望你的笔记本电脑有一些电力，因为卡本身没有高清电视解码器，所以它依赖于你的笔记本电脑的 CPU。虽然这不会给你的 MacBook Pro 用户带来太多麻烦，但那些拥有普通 MacBook 或旧笔记本电脑的用户可能会有一些麻烦。为了让它发挥最佳效果，把数独放到一边去看 *24* ，因为你需要肌肉。
