# 体育+ TVGuide + Digg = RUWT？

[![ruwtlogo.jpg](img/00e13da6c28e4e53253d730a1c2d396d.png) ](https://web.archive.org/web/20220925143049/http://areyouwatchingthis.com/) [“你在看这个吗”](https://web.archive.org/web/20220925143049/http://areyouwatchingthis.com/) (RUWT？)是一个新的体育迷网站，您可以使用它根据社区投票和比赛的实时比分来关注电视上最受欢迎的体育比赛。这意味着在发展过程中，人们不会错过真正精彩的体育比赛。主页包含一个电视时间表，列出了不同电视网络上的所有体育赛事，您可以用它来为即将到来的比赛投票。默认订阅源基于东海岸时间，但您可以在美国和加拿大按邮政编码和有线电视提供商进行定制。电视节目表中的每场比赛都会告诉你它将在何时何地播出，这是一场比赛的书面预览，让你投票，并让你在评论帖中“大声说出”你的观点。该网站还跟踪 15 项运动的新闻、排名和团队。

![ruwtmini.jpg](img/b95e17a274d504757372b5c2d006d050.png)在网站上关注比赛的另一个主要方式是通过他们的体育特定记分牌页面，在那里他们按体育或球队实时列出每场比赛的投票。随着比赛临近或进入加时赛，记分牌会自动增加分数，以反映更具娱乐性的比赛。 [NCAA 篮球委员会](https://web.archive.org/web/20220925143049/http://areyouwatchingthis.com/ncaab)今天相当活跃，NHL 委员会[包括来自 YouTube 的比赛集锦。](https://web.archive.org/web/20220925143049/http://areyouwatchingthis.com/nhl)

这是社区投票模型的一个很好的应用，它极大地受益于分数的自动分析来确定兴趣度。他们还通过 RSS 和 iCal feeds 让你跟踪你的观察列表，检查你的朋友是否得到了你没有得到的游戏。
