# 为音响发烧友准备的 Tivoli 台式音响

Tivoli Audio 的这套桌面音响系统被宣传为音响发烧友的桌面音响。在这个手工涂漆的盒子里，有两个高品质的 3 英寸全频扬声器和后端口，一个内置的 5 英寸磁屏蔽超重低音扬声器和三个独立的放大器。

一个大的 LCD 显示 CD 曲目信息和无线电数据系统信息。它有一个 1/4”耳机插孔和一个内置的 AM/FM 天线。此外，还有两个辅助端口，允许您连接外部音频源。

现在可以从 Hammacher Schlemmer 买到，售价 600 美元。小心脚下，很陡。
