# 轨道真空:方格旗，干净的地板

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/17/track-vac-checkered-flags-clean-floors/>

我的朋友本讲述了一个很棒的故事，当电影首次在电影院上映时，他去看了《塔拉迪加之夜》。在他前面几排是一个五口之家，你可以看出他们非常*喜欢[纳斯卡](https://web.archive.org/web/20131216171030/http://crunchgear.com/?s=nascar)。不幸的是，他们似乎不明白这部电影是一部喜剧，一个笑话和对纳斯卡文化的讽刺。电影开始大约 20 分钟后，这家人走了出来，大概是要求售票亭里糊涂的少年退款。*

有些人确实把汽车绕圈看得很重，对他们来说，这就是 Track Vac 机器人吸尘器。如果 Roombas 是为书呆子准备的，那么 Track Vac 是为那些痛打书呆子的人准备的。有了无线遥控器，想成为恩哈特的人可以整理他的(或她！)穴，去破纪录。

更重要的是，当这个小家伙的电量耗尽时，它可以在自己的充电站“停车”。如果地板不需要吸尘呢？清洁工作结束后，汽车仍然可以运行很长时间，所以它可以在被喷上香槟和亲吻不是妻子的丰满模特之前胜利地跑几圈。所有这一切只为了不到 300 美元。这就是为什么我们是世界上最伟大的国家，朋友。
