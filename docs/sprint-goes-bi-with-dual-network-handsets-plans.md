# Sprint 推出双网手机，计划| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/16/sprint-goes-bi-with-dual-network-handsets-plans/>

虽然它在 QT 上已经有一段时间了，但 Sprint 今天已经正式推出了它的 CDMA + iDEN 硬件，将无线提供商自己的网络与多年前购买的 Nextel 合并。这意味着 Sprint 手机可以拥有所有的 CDMA 功能(例如，便宜的通话计划)，并获得 iDEN 的推动，与其他 Sprint 和 Nextel 手机通话。

除了将这些技术结合到摩托罗拉手机中，Sprint 还将具有双网功能的手机重新命名为“power source”Blend 和 Buzz。似乎型号正在快速消失，这只会让人们搞不清他们得到的是哪个版本(你知道有多少不同的 RAZRs 吗？很多)。
