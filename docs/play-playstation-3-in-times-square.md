# 在时代广场玩 PS3

![](img/c66211af2b25f730caf79e3efe6dbc52.png)

纽约时代广场的玩具反斗城已经设立了有四个 [PlayStation](https://web.archive.org/web/20201026104421/https://crunchbase.com/product/playstation) 3 单元的售货亭。到目前为止，四个 PS3 中只有一个可以玩，所以如果你在这个地区，你必须和其他人保持一致。如果你在这个地区，就去看看吧！到目前为止，似乎只有一个可玩的游戏(*赛车风暴*)，但我相信在不久的将来我们会看到更多可玩的游戏被演示。Destructoid 拍摄了一些信息亭和运行中的 PlayStation 3 的照片，为性感而跳跃。
