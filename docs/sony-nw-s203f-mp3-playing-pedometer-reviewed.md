# 索尼 NW-S203F MP3 播放计步器评论

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/10/sony-nw-s203f-mp3-playing-pedometer-reviewed/>

索尼 NW-S203F 的小瓶之所以引人注目，不仅因为它看起来像一个婚姻助手，它还配有一个计步器。没错，你可以用它的 1GB 在跑步时听音乐，但你也可以用它来跟踪你跑得有多好。如果你和我们一样是博客写手，那就不太好了。

控制器的工作原理是左右旋转旋钮，上下滑动到三个位置。然后，您可以跳过文件夹、暂停或浏览菜单项，这取决于它的位置。还有一个功能可以根据你的跑步速度改变播放列表——这是不能关闭的——所以它可以是一个很酷的功能，或者让你真的想把这个东西塞到别人的屁股里。

不幸的是，你不能在跑步结束后将数据上传到电脑上，也不能使用蓝牙耳机，但有一个调频调谐器和 16 小时的电池寿命。因此，它更像是一个业余爱好者的跑步小工具，而不是约翰评论的[耐克+iPod](https://web.archive.org/web/20161012034146/http://crunchgear.com/2006/10/03/nikeipod-hands-on/) 。
