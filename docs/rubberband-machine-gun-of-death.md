# 橡皮带死亡机枪

当我还是个孩子的时候，我妈妈经常拉我去高中礼堂参加艺术和手工艺展览会。太可怕了。一排排的百花香和愚蠢的废话。然而，有一个希望的灯塔。


这个老家伙，他的名字我不记得了，总是被藏在礼堂的后面角落里。他有给妈妈们的常规装备，但他也有一个橡皮筋枪的军火库，就像一些非法军火商一样安装在他的桌子后面。

他有所有无害的枪。单一发射器。双桶。甚至有一种霰弹枪可以一次发射五发子弹。然后是其中的一个，一个 12 桶，144 橡皮筋的怪物，它能够在几下猛烈的旋转曲柄中毁灭整个工艺展。

我想要它。我必须拥有它。但它不卖。“为了我自己的收藏，”他会说。每个违背自己意愿被拖到那里的小男孩，都会呆呆地站在那里看着那个美丽的东西，想象自己向站在那里盯着看的所有其他小男孩开火。

好吧，现在魔法在某种程度上消失了，但我仍然深情地记得它。我不再需要我曾经用过的橡皮筋机枪了，但是我内心深处还是渴望花 400 美元买一把。幸运的是，另一个更成熟的我意识到我会把自己的眼睛射出来。
