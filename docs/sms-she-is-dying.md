# 短信，她快死了| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/27/sms-she-is-dying/>

![skullphone.jpg](img/45a47d43e2e7ca2f6a874d2fa8aecd0c.png)

研究公司 Gartner 预测，到 2010 年，SMS 将变得过时或至少变得无足轻重。咄。这份报告可能只适用于澳大利亚，但在美国不难发现这种相关性。

随着 Joe Shmoe 拿起黑莓或其他一些具有电子邮件功能的智能手机，移动电子邮件每天都变得越来越普遍。威瑞森报告称，6 月份收发了 100 亿条短信，你可能会有不同的看法，但那只是因为他们向你收取了使用电子邮件的阴阳费用。

无论如何，更便宜的短信计划可能减少了通话时间，对我来说是这样，但电子邮件是最简单的方式，而且更便宜。不是每个人都能在工作时拿出手机发短信而不被经理们发现，但是你可以随时随心所欲地发电子邮件。
