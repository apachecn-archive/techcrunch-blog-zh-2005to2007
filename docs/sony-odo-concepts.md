# 索尼 Odo 概念| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/06/sony-odo-concepts/>

![](img/2d31370d74f1b355bdd60574d8266da4.png)

最近，索尼展示了一系列不需要电池或交流电源的概念产品。例如，旋转快照相机是用手驱动的。你把手指伸进去，旋转它，它就会产生足够的能量来给相机供电。对于喜欢露营、热爱户外或讨厌电池的人来说，这似乎非常方便。

亦作推动力弹装置。把这个小玩意拖过桌子给它供电，然后坐下来欣赏电影。您无需外部电源即可观看视频和图像。太糟糕了，这些目前只是概念产品，因为我想减少使用电池。也许有一天我们会看到一辆完全由脚驱动的 VAIO 或者其他同样荒谬的东西。但是如果脚动发电机爆炸了，我们都会有麻烦。
