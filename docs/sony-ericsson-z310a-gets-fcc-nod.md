# 索尼-爱立信 Z310a 获得 FCC 批准

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/01/sony-ericsson-z310a-gets-fcc-nod/>

![](img/db3d42372aed095c42a3beb962242c5e.png)

不久前，我给了你索尼爱立信即将推出的 Z310a 的内幕。嗯，它离成为现实又近了一步，因为它刚刚获得了 FCC 的批准。如果你忘了，这款翻盖设备配有 EDGE、全 HTML 网络浏览器、电子邮件、RSS 阅读器(显然是为了让你在旅途中也能享受美味)和录音机等功能。

目前还没有定价或具体运营商的消息，但我敢猜测，某个名字与“尿海”押韵的人将会购买它。
