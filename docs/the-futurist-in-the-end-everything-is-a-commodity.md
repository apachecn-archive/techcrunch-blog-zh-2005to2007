# 未来学家:最终，一切都是商品

![](img/b0bf196b32bc70ce2355cdd58592f22a.png)

不久前，我在《纽约客》杂志上读到一篇关于奢侈番茄酱的报道。如果“奢侈番茄酱”的想法让你大笑，你并不孤单。虽然顾客多年来一直在吃高端芥末产品，但即使是最高档的餐馆也仍然供应亨氏。事实是，虽然一些产品越来越多地被其高端优质产品所定义，但在组合中仍有一些产品只能被称为商品。

当然，这些队伍正在减少。Grey Poupon 对芥末的影响，Dyson 对吸尘器的影响，以及现在的商业干手器。当一个白色的通用冰箱足以让你的蔬菜保持凉爽时，一个没有零下的房子现在看起来过时了。

但是，产品周期是周期性的，曾经是商品的行业再次成为商品只是时间问题。

戴森和 Sub-Zero 可能让平凡的家用电器变得性感，但一旦你忘记了他们的产品激发的激情，他们的评论肯定是混杂的。*消费者报道*频频报道亚零的返修率很高，戴森最后也只是一般般的成绩。那么如果不是可靠性和质量让这些产品成为必备，那是什么呢？

答案当然是形象。就戴森而言，这很有趣，因为你不会像拿手机一样在咖啡桌上拿出吸尘器。理想的形象很大程度上取决于两个关键因素:设计和价格。设计是足够直接的:如果有人可以把一件通常看起来应该藏在壁橱里的产品加上一些颜色和形状，人们就会想要它。但是价格才是更有意思的:即使人们声称自己是捡便宜货的人，仅仅是让一件产品变得更贵通常就足以让一些人想要它。

你可能声称你能分辨出可乐和好市多品牌可乐之间的区别，但是当涉及到这个问题时，有相当多的人可能会在味觉测试中失败。尽管如此，使用花了你一大笔钱的产品还是有一定的满足感。它会成为你引以为豪并想要展示的东西。事实上，你是一个可以如此轻率地花费额外金钱的人——无论是在鹅肝上还是在手机上——增加了你的自尊和自我价值(不管这有多可悲)。)推销奢侈品的行为是人类历史上最伟大的社会工程之一。那就是:让人以浪费金钱为荣。

今天，仍然缺乏超级奢侈品选择的产品几乎为零。虽然，随着更便宜的物品越来越接近它们的奢侈品，曾经多样化的产品可能会再次成为商品。

这就产生了一个有趣的情况。曾经商品化的行业——比如吸尘器——现在提供了超昂贵的奢侈品选择。与此同时，曾经在低端和高端之间存在巨大差距的行业正在看到这种差距正在缩小，因为他们实际上已经商品化——Dirt Devil 现在以半价出售戴森式的无袋真空吸尘器，Vizio 电视与索尼没有太大区别。这创造了一个有趣的、相当可预测的产品周期。

**首先:**一个行业是从商品开始的。低端和高端产品在价格和受欢迎程度上几乎没有区别。
 **第二:**令人向往的奢侈品涌现。这些产品因设计和高昂的价格而激起热情。

第三:低端产品模仿高端产品的性能和设计。低端产品的价格会随着功能的增加而小幅上涨，而高端产品的价格会因为竞争而被迫下降。

第四:一个行业再次变得商品化，尽管产品看起来更好。
