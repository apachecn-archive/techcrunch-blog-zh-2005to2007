# Skype 在美国开始收费的时间到了

[![skype_logo.jpg](img/ce26042d696e0a87a691a85780a1d259.png)](https://web.archive.org/web/20221001071122/http://www.skype.com/) 从 2007 年开始，SkypeOut 通话在美国和加拿大境内将不再免费。该公司周三宣布，他们将开始对从 Skype 拨打任何手机或座机收取年费。从 PC 到 PC 的通话将保持免费。

无限年通话费用为 29.95 美元。如果您在 2007 年 1 月 31 日之前购买该计划，仅需 14.95 美元。如果没有无限制计划，用户可以为美国和加拿大境内的通话支付每分钟 2.1 美分，与国际通话费用相同。

“我们看到消费者愿意拨打价格合理的 SkypeOut 电话，”Skype 北美区总经理 Don Albert 告诉《纽约时报》。

作为一项促销活动，SkypeOut 通话从去年春天开始在美国和加拿大境内免费，尽管它在国际上一直是一项付费服务。考虑到大多数电话套餐平均每月 30 美元，每年 30 美元还是相当便宜的。我以 14.95 美元购买了该计划，并注意到您只能以折扣价购买一年。
