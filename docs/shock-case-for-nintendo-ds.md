# 震惊！任天堂 DS 的外壳

![](img/97ace76214b203d2aba9c7703eea80be.png)震惊！case 本质上是对任天堂 DS Lite 外壳的彻底改造。它有多种颜色，包括透明色(虽然我猜想透明色在技术上不是一种颜色，但你明白这一点)，所以，像 Raj 一样，你可以根据你的心的内容来调整颜色。

更重要的是，震惊！案件提供了一个更新的脆性铰链上发现的 DS Lite。扬声器也有更新，允许驾驶员发出更饱满的声音。还没有关于价格或可用性的消息，但我们会在有消息时通知您。
