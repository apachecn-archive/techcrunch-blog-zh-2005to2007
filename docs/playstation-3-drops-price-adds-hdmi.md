# Playstation 3 降价，增加 HDMI | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/22/playstation-3-drops-price-adds-hdmi/>

![](img/58735b92e6588cb1115c859d3d5092d7.png)

最后，来自索尼的一些好消息。在日本东京，肯·库塔吉(Ken Kutaragi)透露，低端 Playstation 3 型号将支持 HDMI。至少现在人们会考虑购买更便宜的 Playstation 3。Kutaragi 实际上表示，增加 HDMI 是因为索尼受到了负面的报道！

> 在 E3 时代，没有(关于)HDMI 的共识，索尼“担心它的杀伤力过大。”但媒体和制造商的负面反应导致该公司重新考虑。

我不知道让更便宜的 Playstation 3 拥有 HDMI 会有多“过火”，但索尼最近很奇怪。Kutaragi 还透露了一些更好的消息:Playstation 3 将会降价。兴奋吗？现在，坏消息是:价格下跌将只发生在日本和 T2。为什么我们不明白呢？因为美国的零售商说 PS3 的价格很划算。价格下降不多，从 499 美元降到 429 美元，但这是一个开始。也许下周，索尼会告诉我们，他们终于知道如何做运动检测**和**振动，交叉手指。
