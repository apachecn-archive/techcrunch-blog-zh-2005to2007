# Podango 推出公测版

[](https://web.archive.org/web/20220928190417/http://www.podango.com/)[【Podango】](https://web.archive.org/web/20220928190417/http://www.podango.com/)，我们[上个月写的](https://web.archive.org/web/20220928190417/http://www.beta.techcrunch.com/2006/08/29/podango-to-create-podcast-channels/)，今天进入公开测试，所以任何人都可以访问这个网站，如果他们愿意，可以申请成为一个站主任。

Podango 的目标是将可能每周发布的多个播客合并成一个每天 3-5 个节目的频道，并提供一个 RSS 源。Podango 将招募电台总监，他们将为他们的频道招募播客，使用 Ajax 拖放管理界面管理频道，并帮助招募广告商。希望听众能从让一个人选择给定类别的最佳播客中受益，并避免因节目将一个接一个排队而造成的广播时间停滞。想了解更多关于成为电视台台长的信息，请看本页。

Podango 还将为电台创造广告销售，保留 30%，与电台总监和播客分享 70%。

Podango 现在有六个直播站，首席执行官 Lee Gibbons 告诉我，随着电台主任的招聘，更多的电台将很快推出。点击这里查看 Lee 关于测试版发布[的博客文章。](https://web.archive.org/web/20220928190417/http://www.podango.com/blog/?p=30)
