# 谷歌音频广告的软发布

我们收到来自广告商的报告，称[谷歌音频广告](https://web.archive.org/web/20221007124333/http://www.beta.techcrunch.com/2006/12/11/startup-cries-foul-at-googles-new-radio-ads-product/)已经作为一个选项添加到他们的 Adwords 账户中，尽管我们还不能确定每个人现在都有这个功能(如果你没有看到，请告诉我们)。

谷歌向 Adwords 客户提供 400 美元的免费广告。

这项服务从去年年底的[开始在](https://web.archive.org/web/20221007124333/http://adwords.blogspot.com/2006/12/bringing-radio-advertising-to-google.html)进行[测试。广告商可以在广播电台购买广告位，按每次点击付费。谷歌还会使用专业的配音演员来制作你的音频广告(大多数电台也会免费这么做)。](https://web.archive.org/web/20221007124333/http://www.marketingpilgrim.com/2006/12/google-audio-ads-opens-up-radio-testing.html)
