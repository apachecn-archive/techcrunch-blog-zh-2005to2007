# 经济实惠的阿格系列

GE A730 和 A830 是该系列中预算友好的入门级数码相机，但他们声称“相机的火力几乎是其价格的两倍”。A730 有 700 万像素，而 A830 有 800 万像素。两者都有 3 倍的光学变焦，但 A730 有 4.8 倍的数码变焦，A830 有 5.1 倍的数码变焦。两者都有 2.5 英寸的液晶显示屏，与其余的有相同的功能和颜色选择。A 系列的标价将在 150 美元左右。
