# TechCrunch 西雅图 Web 2.0 派对将于下周举行

上周一，我们在伦敦举办了一场非常成功的 TechCrunch 派对。**接下来——与 [Redfin](https://web.archive.org/web/20230213030751/http://www.redfin.com/) 共同制作的西雅图派对。还有几天，下周三，2006 年 5 月 31 日。派对将在 ConWorks 举行，这是一个位于联合湖南岸的艺术仓库风格的空间，可容纳 800 人(当然，在这里举行派对看起来很有趣)。**

这项活动是免费的，对所有人开放。

**事件信息:**

**日期:**2006 年 5 月 31 日星期三
**地点:**华盛顿州西雅图市博伦大道北 500 号 Conworks([地图](https://web.archive.org/web/20230213030751/http://local.live.com/default.aspx?v=2&cp=47.623273~-122.335846&style=r&lvl=14&scene=3695119&sp=aN.ry3gqq4t4tbd_500%2520Boren%2520Ave%2520N%252c%2520Seattle%252c%2520WA%252098109%252c%2520United%2520States___) ) ( [查看](https://web.archive.org/web/20230213030751/http://local.live.com/default.aspx?v=2&cp=ry3ggy4t4tk4&style=o&lvl=2&scene=3695119&sp=aN.ry3gqq4t4tbd_Consolidated%2520Works_500%2520Boren%2520Ave%2520N%252c%2520Seattle%252c%2520WA%252098109__) )
