# OS X 豹仍未准备好，程序员仍在计算链表

![](img/d1c42eafd022ac2dea06a3520fcd84ec.png)可怜的豹子。尽管每个人都对新的 OS X 版本将于今年 4 月推出感到兴奋，但谣言树正在萌芽，抱怨它将在很长一段时间内不可用，程序员们自己也厌倦了被苹果邪恶的波斯剑臂团队领导推着走。

对于那些说 Leopard 已经准备好了，人们只是在撒谎的人，苹果程序员说“这只是测试版，不是最终版或金主版。”所以请闭嘴，让他们工作。
