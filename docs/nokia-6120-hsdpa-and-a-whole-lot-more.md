# 诺基亚 6120: HSDPA 和更多

![](img/d12074e09adda31201f8c0b9dca8159d.png)

我的，哦，我的是诺基亚 6120 看起来很可爱。那幅金属漆画太精美了，我差点误以为是 Vertu！关于外面的甜蜜已经说得够多了，让我们来深入了解一下里面丰富、粘稠的东西。首先，6120 采用带功能包 1 的 S60 第三版操作系统、四频 GSM 和 WCDMA/HSDPA。转到更面向用户的功能，你会发现蓝牙 2.0+EDR，一个 200 万像素的摄像头，带有非常受欢迎的闪光灯，microSD 和一个 2 英寸 QVGA 屏幕。

如果糖果棒手机是你喜欢的东西，那么诺基亚 6120 看起来是最有前途的手机之一。它的价格约为 350 美元，将于 6 月上市。不过要小心。如果你想要一个，你可能不得不最终进口它…
