# 景计划:截屏和截屏的三条腿狗

Jing 项目是 TechSmith 的一个相当新的屏幕捕获和截屏工具。

Jing 的目标是通过为两者提供一个程序来使截屏和截屏更容易。用法很简单。截屏工具和我以前用过的一样好；与 [Screencast-o-matic](https://web.archive.org/web/20221028072400/http://www.beta.techcrunch.com/2007/05/29/screencast-o-matic-brings-screencasts-to-a-web-browser/) 不同，它是一个基于软件的工具，质量上的差异显而易见。

Jing 项目是免费的(至少目前是免费的)，所以我不应该对它太苛刻。这是我喜欢的软件，并准备写一个完全有利的评论，直到我试图上传一个截屏。Jing 只在 Flash 中保存截屏(。swf)。这似乎不是什么大事情，直到你试图找到一个像 YouTube 这样的视频托管网站。swf 文件。

Jing 内置的共享功能允许用户直接将文件上传到，这是一个由 TechSmith 拥有和运营的截屏托管网站；然而，如果 YouTube 是美国，Screencast.com 将是一个失败的非洲国家；太恐怖了。我无法访问我通过 Screencast.com 界面上传的测试截屏，尽管我可以直接观看。在直接视图中，没有允许你在你的站点上嵌入截屏的代码；这是否只是付费主机计划的一个选项还不清楚，因为在那个页面上没有链接到帮助或常见问题页面。

截屏工具很方便，如果 Flash 文件不打扰你，那么它是一个很好的截屏工具。总的来说，景项目就像一只三条腿的狗；你可以很容易地爱上它，它会叫，但它不能做一只四条腿的狗能做和应该做的一切。
