# 三星 SpinPoint S166 硬盘没有噪音

虽然硬盘的大小和速度似乎总是在上升，但很少看到一家公司对快速旋转的硬盘发出的声音采取措施。然而，三星的 SpinPoint S166 硬盘驱动器系列确实通过使用其专有的 SilentSeek 和 NoiseGuard 技术来解决噪音问题，“以加快操作速度，同时降低噪音。”

本月推出的 7,200rpm 3.5 英寸硬盘有 80GB 和 160GB 两种规格，与标准硬盘相比，噪音水平降低了 15%。虽然这种减少使它们非常适合粘在媒体中心 PC 上，但对于该应用来说，存储容量小得令人尴尬。这里希望三星能够在不久的将来推出一款价格适中的 320GB 版本。
