# wet paint——迄今为止最好的维基？

总部设在西雅图的 [Wetpaint](https://web.archive.org/web/20211026130917/http://www.wetpaint.com/) 今天开门迎客，展示了几个使用其超级光滑的新维基平台的样本网站。

你可以通过冗长的[之旅](https://web.archive.org/web/20211026130917/http://www.wetpaint.com/demo_win.html)很好地了解用户是如何与平台上的维基互动的。基本上，这是一个非常用户友好的所见即所得类型的维基平台。要查看示例 wiki，请查看他们创建的 [xbox 360 站点](https://web.archive.org/web/20211026130917/http://www.wikixbox360.com/)。

这是一个托管的维基解决方案，Wetpaint 的商业模式是在维基的每个页面上添加上下文广告。

该公司成立于去年，由三一风险投资公司和弗雷泽科技风险投资公司支持。详见[硅拍](https://web.archive.org/web/20211026130917/http://www.siliconbeat.com/entries/2006/03/06/wetpaint_wikis_for_regular_people.html)、[约翰库克](https://web.archive.org/web/20211026130917/http://blog.seattlepi.nwsource.com/venture/archives/101794.asp?source=rss)和[佐利鄂尔多斯](https://web.archive.org/web/20211026130917/http://www.zoliblog.com/blog/_archives/2006/3/6/1799969.html)。
