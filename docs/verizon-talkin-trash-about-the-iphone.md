# 威瑞森在说关于 iPhone 的废话

事实证明，iPhone 真的“无所不能”，这让 AT & T 的竞争对手有点紧张。威瑞森无线公司把我算作一个用户，他们列出了一个清单，这个清单会让你希望你从来没有听说过 iPhone，更不用说——喘气！—取消你的 VZW 联系人，转到 AT&t。VZW 讨厌触摸屏，他说“如果你不看着一个你甚至感觉不到的键盘，你怎么打字？”说得好，你还有什么？“Wi-Fi 不是移动技术”怎么样。你必须在一个固定的地方使用它——前提是你知道在哪里可以找到它。一些无线热点让你付费使用。还有那些关于无线网络安全的烦人的担忧……”该死，VZW 又得了一分。

最后，VZW 快照部门的杀手锏是“靠近插头”我猜这意味着不可拆卸的电池将很快耗尽，但事实并非如此。但是不管怎样。是 BAAAD。

我要说的是，普通的威瑞森销售代表永远不会使用这些策略。任何打电话将电话号码转移到美国电话电报公司的人可能已经购买并支付了他们的 iPhone，那些可能认为 iPhone 很酷的普通人可能不会购买任何比 50 美元更贵的 RAZR。交易结束了，威瑞森。现在就看你的了，让制造商更容易地将酷手机推向市场，而不是用蹩脚的软件和低劣的无线标准来膨胀它们。哎呀。抱歉。你现在就是这么操作的，对吧？
