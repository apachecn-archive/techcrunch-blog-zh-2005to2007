# Skype 现在有呼叫转移功能

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/31/skype-now-has-call-forwarding/>

[【Skype】](https://web.archive.org/web/20210422222134/http://www.skype.com/)(之前的[简介](https://web.archive.org/web/20210422222134/http://www.beta.techcrunch.com/?p=70) ) [今天发布了他们客户端的新测试版](https://web.archive.org/web/20210422222134/http://www.skype.com/products/skype/windows/)——1.4.0.45。

有一大堆新功能(都列在下面)，但有一个真正有趣的功能，在我看来，它改变了整个 VOIP 游戏。Skype 现在有了真正的来电转移功能，允许您将 Skype 来电转移到任何普通电话上，即使您的 Skype 处于离线状态。

直到几分钟前我在马来西亚读了一篇名为 [Jan 的 Skype 博客，我才意识到这一点。来电转发本身对 Skype 来说并不陌生—](https://web.archive.org/web/20210422222134/http://zog.typepad.com/malaysia/2005/09/skype_newest_be.html)[Jyve](https://web.archive.org/web/20210422222134/http://www.beta.techcrunch.com/?p=119)在 Skype 附加软件中加入来电转发功能已经有一段时间了。

然而，使用 Jyve，您必须登录 Skype 才能转接来电。这是一个必要的限制(Jyve 无法控制)，但是当我登录 Skype 时，我不需要将来电转移。当我离开电脑注销时，我才想要这个。

所以当我看到这个的时候，我非常兴奋。但是从 Skype [下载页面](https://web.archive.org/web/20210422222134/http://www.skype.com/products/skype/windows/)或他们的[常见问题解答](https://web.archive.org/web/20210422222134/http://support.skype.com/index.php?_a=knowledgebase&_j=questiondetails&_i=304&nav=+%26gt%3B+%3Ca+href%3D%27index.php%3F_a%3Dknowledgebase%26_j%3Dsubcat%26_i%3D9%27%3ESkype+for+Windows%3C%2Fa%3E)看不出你是否必须登录才能使用呼叫转移功能。

所以我测试了一下。简和我在注销时互相打电话，Skype 尽职尽责地把电话转接到我们的手机上。完美！

在“高级设置”下，您实际上可以设置三个单独的转移号码，每个号码都会按照您设置的顺序响铃。当你离线时，一个特殊的图标出现，让人们知道他们可以打电话给你，它会转移到一个正常的电话。
