# 索尼用 GPS 设备找到自己

索尼今天宣布了其 NAV-U 系列 GPSes 的第二个新成员。NV-U71T 有一个 3.5 英寸的防眩光屏幕，该设备通过吸盘连接到您的挡风玻璃上。

新的 GPS 利用手势命令功能，允许用户通过在触摸敏感显示器上签署特定的动作来访问特定的命令。它还包括预装在 2GB 闪存中的大约 700 万个兴趣点。它还可以通过内置扬声器提供语音指示。

它的售价为 500 美元，在最初的 90 天后需要 4.95 美元/月的订阅服务。
