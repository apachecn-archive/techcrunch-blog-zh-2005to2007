我们的一些老读者可能还记得[一个愚蠢而不负责任的谣言](https://web.archive.org/web/20210225220949/http://crunchgear.com/2007/05/31/a-stupid-insane-mac-rumor-were-starting-right-here-new-macs-to-ship-with-vista/)我们就在这个博客上开始，这个想法是基于苹果公关人员的一个失误，未来的 MAC 电脑可能很快会预装 Vista 作为 Leopard 的双引导选项。这个想法不仅仅是运行 Windows，而是在 Mac OS 中运行 Windows 应用程序。你们中的许多人对这个想法嗤之以鼻，我们自己甚至认为它是假的。

然后，一位同事向我展示了上面的 VMware Fusion 视频中的 Unity，您可以看到这种情况正在发生。我们并不是说 Leopard 将会发布这种类型的场景，但它确实会火上浇油，增加思考的食粮，以及其他隐喻。
