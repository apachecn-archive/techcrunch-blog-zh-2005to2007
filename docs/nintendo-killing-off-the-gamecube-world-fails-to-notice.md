# 任天堂扼杀了 GameCube，世界没有注意到

![](img/ccba03fddb05d88123a48ac61bd9d63e.png)

Ohhhh *啪嗒*！根据 *GameDaily* 的采访，任天堂的 Perrin Kaplan 已经证实 GameCube 已经不复存在。卡普兰说，任天堂将不再生产 GameCube 游戏或系统，但如果人们仍然想浪费现金购买它们，“库存就在那里。”

大不了？那鸿到 GameCube 完成的时候，所有的游戏都可以在 Wii 上玩 Wii 已经开发了第二版。此外，你能诚实地说 GameCube 是一个“杀手”系统吗？没想到会这样。
