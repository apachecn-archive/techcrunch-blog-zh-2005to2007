# 转向推出混合 CPA 广告网络

 [](https://web.archive.org/web/20221224072250/http://turn.com/) 圣马特奥初创公司 [Turn](https://web.archive.org/web/20221224072250/http://turn.com/) 将在明天的 Web 2.0 大会上推出一个有趣的新广告平台。它将每次行动成本竞价与广泛的用户、网站和广告绩效分析相结合。该公司称自己是世界上第一个自动定向、竞价的 CPA 广告网络。

我经常写道，新的广告网络让我厌烦，因为这是描述一个有趣的广告网络的前缀，但最近我已经看到了足够多有趣的在线广告新方法，所以我要承认我对这个领域有了新的兴趣。说广告有创新空间是一种保守的说法。

Turn network 正在采取一种方法，将每次行动成本(CPA)投标和对 60 个上下文、行为、人口统计和性能因素的分析结合起来，以确定在何时提供哪些广告。该公司打算在后台完成所有的广告投放工作，这样购买广告就变得极其简单，但却异常有效。反正他们就是这么打算的。这让我想起了 [RightMedia 的](https://web.archive.org/web/20221224072250/https://techcrunch.com/tag/rightmedia)动态服务，即在给定时刻支付最高价格的任何子网络文本广告，以及 [BrightRoll 的](https://web.archive.org/web/20221224072250/https://techcrunch.com/tag/brightroll)基于持续转换性能选择视频广告格式。
 Turn 由 Alta Vista 前 CEO 兼 CTO 吉姆·巴尼特(Jim Barnett)和约翰·埃利斯(John Ellis)共同创立。随着该公司的推出，它的库存中有来自 1000 家广告商的 500 万条广告，由 30 家出版商运营，每月有 1100 万独特的访问者。该公司从 Norwest Venture Partners、Trident Capital 和 Shasta Ventures 获得了 1800 万美元的风险投资。

*在转转网购买广告不需要关键词选择和管理。*相反，购买者确定他们希望受众采取的行动，以及每次采取这些行动时他们愿意支付的金额。该行动可能是一个网站访问，电子邮件注册或完成交易。

当访问者来到一个网站时，在确定每个广告的成功概率时，关于这些用户的数据、网站的上下文分析、广告以及每个广告排列在该网站和相关网站中的成功都被考虑在内。然后，根据 CPA，相对于支付的价格考虑转换的可能性。*CPA 出价除以采取行动的概率等于广告每千次展示的有效收入。*这样广告就做好了！

广告网络在推出时必须看起来闪亮、新颖、有前途。只有时间才能证明这个网络的执行效果如何。是时候有人提出一种基于 CPA 和分析的混合方法了。
