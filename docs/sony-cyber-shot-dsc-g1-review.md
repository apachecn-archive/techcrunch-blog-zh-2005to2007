# 索尼 Cyber-shot DSC-G1 评论

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/31/sony-cyber-shot-dsc-g1-review/>

索尼 Cyber-shot DSC-G1 是一个傻瓜相机的庞然大物，部分原因是它漂亮的 3.5 英寸液晶显示器。凭借 2GB 的内部存储和卡尔·蔡司 Vario-Tessar f/3.5-4.3 38 毫米-114 毫米 3 倍变焦镜头和高质量的 600 万像素传感器，你很难不在拥挤的市场上给 G1 一个战斗的机会。然而，它 600 美元的高昂价格和糟糕的内置 Wi-Fi 功能几乎否定了 G1 所有的积极属性。


在深入研究最先吸引我的 G1 的 Wi-Fi 功能之前，让我们先来看一下细节。关于 G1，你首先会注意到的是它的大小。这当然不是一个你可以放在口袋里随身携带的精致小东西，除非你穿着 MC 锤子裤或者你想让生命之颚靠近你的垃圾。我一点也不夸张，伙计们；G1 是索尼硬件的 3.7×2.8×1 英寸(打开时约为 4.5 英寸)。

![imgp0153.JPG](img/cddbeea68aa39863d053c58acc76ef9a.png)

我最初喜欢 G1 的滑动外形；它增加了其坚固的设计。我相信你们大多数人都习惯用一个简单的动作来启动你的数码相机，那就是按下开/关按钮。我想让我的相机快速启动，这样我就可以捕捉那些只有在大量饮酒和/或吸毒时才会发生的特殊时刻。G1 让这个过程变得有点难，因为现在你需要先松开扣子，然后拉开相机。这是一个痛苦的屁股和四到五秒的总启动之前，你发射了照片没有帮助。

![imgp0152.JPG](img/fc70c663e613573fe185c4e8ebda7b24.png)

一旦 G1 启动并运行，并且你已经适应了按钮的布局，我发现在开始时很麻烦，在 UI 中导航就变得轻而易举了。这基本上和你在 PSP 或 PS3 上看到的一样。我在任何傻瓜相机上见过的最好的用户界面。恭喜你，索尼。相机部分允许你拍摄照片或拍摄带音频的视频(640×480 或 320×240 @ 30fps)，但你不需要我来弄清楚这一点。我真的很喜欢 Viewer 菜单，因为子菜单允许您查看您的整个相册、存储在外部存储器(支持高达 8GB 卡的 Memory Stick Duo)上的内容、您拍摄的最后一张图像、您查看的最后一张图像和幻灯片。你也可以通过音频播放器听一些音乐。稍后我会介绍一个沟通子菜单，请稍等。工具箱可以让你直接通过 PictBridge 打印图片，媒体工具可以记录你还有多少内存，还有一个标准的设置子菜单，可以显示所有常用的内容。

![imgp0122.JPG](img/66b35fad7c6162d77e499fe092964e9a.png)

至于[画质](https://web.archive.org/web/20130628175043/http://crunchgear.com/2007/06/29/an-iday-retrospective-with-snarky-captions/)你不会失望的。有 10 种不同的场景模式，本质上，应该涵盖你遇到的任何情况。超级防抖图像稳定补偿你的抽搐和/或妥瑞症的爆发。您还可以调整曝光、白平衡和一些拍摄模式，从连续连拍到每次 3 张照片，每张照片有 3 个曝光级别，间隔 1.0 EV。关于 G1，我唯一注意到并且真正不喜欢的是 ISO 噪音的数量，它在 200 的时候开始抬起它丑陋的头。你爬得越高，情况就越糟，但先进的光电技术应该不会有太大的问题来缓解这种情况。

让我们来看看超级酷、超级棒、超级棒的功能，那就是 Wi-Fi。这是我上个月拿到的第二台支持 Wi-Fi 的数码相机。它符合我的期望吗？不，一点也不。与[尼康 S50c](https://web.archive.org/web/20130628175043/http://crunchgear.com/2007/07/24/nikon-coolpix-s50c-review/) 不同，你只能将照片发送到另一个支持 DLNA(数字生活网络联盟)的设备，这真的很糟糕。如果你想把派对上的裸照立即上传到你的 MySpace 上，那就去别处看看吧。当然，除非你的朋友也有一辆 G1，但考虑到它的价格，我认为这不太可能。

您可以同步多达三个 G1，这样每个人都知道您在快门发挥作用时拍摄的是什么。或者你可以发送特定的照片，因为你可能不想让你的朋友知道你做的每一件事。

这是一个相当令人失望的功能，我期待着尝试，但可悲的是，这不是唯一的缺点。我不喜欢你的外部内存选择，这只是一个，但 2GB 的内部内存弥补了这一点。我想你可以使用外部插槽来放你所有的音乐，所以在这种情况下杯子是半满的。这东西太大了，你得把它插上才能充电。这意味着你需要随身携带许多垃圾来拍照，但可充电锂离子电池可以持续约 260 多张照片。

![imgp0123.JPG](img/2d5142d3b205c86380e8dc6d7bbc1cc6.png)

它很大，结实而且相当昂贵，但考虑到它能做什么，它是一个很好的会聚设备。幸运的是，这种网络拍摄的利大于弊…勉强。
