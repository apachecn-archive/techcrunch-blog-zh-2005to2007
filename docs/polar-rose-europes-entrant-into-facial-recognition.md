# 极地玫瑰:欧洲进入面部识别领域

 [](https://web.archive.org/web/20211128174854/http://www.polarrose.com/) 瑞典马尔默[极地玫瑰](https://web.archive.org/web/20211128174854/http://www.polarrose.com/)今天举行发布会，并提前宣布将于明年初发布一款产品。首席执行官尼古拉·尼霍姆上周末给我演示了即将推出的产品。

Polar Rose 正试图解决从照片中获取意义这一日益重要的问题。如果没有语义数据，一张照片对于试图将关键字查询转换成图像结果的计算机来说没有多大用处。谷歌开发了一款游戏[让用户为照片添加标签。Flickr 和它的一些竞争对手在让用户标记照片方面做得很好。当然新人](https://web.archive.org/web/20211128174854/http://www.beta.techcrunch.com/2006/09/01/google-image-labeler/) [Ookles](https://web.archive.org/web/20211128174854/http://www.beta.techcrunch.com/2006/12/18/ookles-to-launch-in-early-2007/) 也将很快发布他们自己的产品。但是网络上的绝大多数图像很少或没有与之相关的语义数据，因此实际上它们是不可见的。

极地玫瑰想要改变这一点。它们不会成为照片的目的地。相反，他们计划采取双管齐下的方法来赢得市场份额——为用户分发浏览器插件，为图片网站提供免费的 API。该插件的用户将能够添加互联网上任何照片的数据，并在以后搜索这些照片。集成该 API 的公司将免费获得用户创建的数据副本。

Polar Rose 背后的技术创建了一个人的虚拟 3D 图像，排除了光线和其他可能影响识别的条件。这个 3D 图像然后被用来猜测一个人何时出现在照片中。例如，该插件将提供一个链接，试图在互联网上找到被标记的人的其他图像。一开始这个数据会很单薄。但是，如果用户采用 Polar Rose 插件，搜索功能可能会成为一个有用的工具，可以找到更多的人的照片。

Polar Rose 上个月宣布获得北欧风险投资伙伴 510 万美元的首轮融资。
