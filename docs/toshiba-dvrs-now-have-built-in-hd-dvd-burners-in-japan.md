# 日本东芝 DVR 现已内置 HD DVD 刻录机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/12/toshiba-dvrs-now-have-built-in-hd-dvd-burners-in-japan/>

[![tosha600.jpg](img/224ce3de8fd9a376661f396025ec9599.png)](https://web.archive.org/web/20140818191812/http://old.crunchgear.com/wp-content/uploads/tosha600.jpg "tosha600.jpg")

我们的日本读者将会兴奋地听到东芝正在更新其 DVR，以包括内置的 HD DVD 刻录机。DVR 有 300GB (RD-A300)和 600GB (RD-A600)两种容量，有两个调谐器。正如去年双调谐器 [TiVo](https://web.archive.org/web/20140818191812/http://crunchgear.com/category/tivo/) madness 期间周围的任何人都知道的那样，拥有两个调谐器可以让你同时录制两个节目。此外，DVR 兼容 TrueHD 和 DTS-HD，这两种新的杜比音频格式让发烧友和粉丝(例如我)眼花缭乱。

DVR 支持高达 1080p 的分辨率，你可以在 600GB 的型号上存储大约 78 小时的高清节目。不过，我不太明白为什么你既需要硬盘又需要光盘。
