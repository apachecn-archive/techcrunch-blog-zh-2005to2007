# 沃达丰将提供加密短信

![](img/5d81cd281e17d87f82f81ddb70b2fbcd.png)

我们偏执的英国读者谁摇滚每个文件 128 位加密将高兴地知道，沃达丰英国将带来加密短信。该公司最近宣布计划为客户提供加密短信服务。通过与 Broca 合作，沃达丰将使用 Broca 的 SAMS 技术来确保所有信息都被加密。目前还不知道将使用何种加密技术。

这项服务通常面向需要绝对隐私的有商业头脑的个人。嘿，如果你在 Xbox 团队工作，不小心泄露了 360 的新型号，你会有什么感觉？每条短信都需要额外付费，并且是可以添加到现有计划中的额外服务。
