# 蓝牙天堂:Aliph Jawbone

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/07/bluetooth-heaven-aliph-jawbone/>

请允许我向您介绍目前市面上最奇特的蓝牙耳机，来自 Aliph 的 Jawbone。几个月来我一直听说这款设备，但直到 CES 上我才真正见到它。这一切都像人们所说的那样吗？

![](img/5ddcdc4da662528f838f1d2618af5ddd.png)
总之，是的。Jawbone 是市场上最好的蓝牙耳机。Jawbone 结合了先进的噪音消除技术、吸引人的美学、各种适合的选项和实惠的价格，是众多蓝牙耳机中最通用的竞争者

将耳机戴在耳朵上是一项简单的任务。套装包括四个耳机——左右各一个大耳机和一个小耳机——以及各种耳塞。我只花了几分钟就找到了一个能让我的大耳朵舒服的组合。Aliph 因包含一套套装而获得奖励积分。

![](img/0f3db9b90fca26e51d2a376697ced86e.png)
利用噪音屏蔽技术，颚骨能够抵消或显著降低环境噪音。它使你能够在通常不适合交谈的环境中交谈。该技术结合了创新的声学和音频处理技术，让您能够听到对方的声音，也让对方能够听到您的声音。

其专有的语音活动传感器可以准确识别您何时说话，从而根据背景声音调节您的声音。我在各种环境中尝试过，取得了很大的成功。该设备通信能力的最佳例子是在人群中。当我在拥挤的地方时，我通常很难辨别电话。凭借其动态音量和频率调节，Jawbone 几乎为我消除了这个问题，让我可以在任何地方自由交谈。该系统通过监控和调节干扰背景噪声来实现这一点。

它还能让另一端的人更好地听到你的声音。该设备使用多向麦克风处理您的语音，并从您发出的语音信号中消除背景干扰。

然而，也存在一些问题。Jawbone 拥有网格状的表面和三种时尚的颜色，是一款极具美感的设备。问题是它有点笨重。

我不经常使用蓝牙耳机，因为我觉得把东西挂在头上很尴尬，也很受限制。考虑到这款耳机的质量有点大，我觉得它比大多数耳机都要笨拙一些。这完全是一个个人问题，但我不能把它对设备太多。我知道这对很多人来说都不是问题，尤其是那些习惯使用蓝牙耳机的人。

总而言之，我认为 Jawbone 是一款值得称道的设备。它的售价为 119 美元，与竞争对手的价格相差无几。考虑到该单元的先进性及其令人印象深刻的噪音消除功能，我相信它的扩展周长可以忽略不计，有利于其明确的效用。如果你想买蓝牙耳机，我建议你看看 Jawbone。
