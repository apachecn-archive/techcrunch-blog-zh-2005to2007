# Sprint 发布详细的覆盖地图

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/11/sprint-releases-detailed-coverage-maps/>

Sprint 复制了 T-Mobile 一年半前的举措，为消费者发布了详细的覆盖地图，这样他们就可以看到 Sprint 的服务到底在哪里以及有多好。只要输入你的邮政编码，地图就会显示你的语音和 EV-DO 覆盖范围。对 Nextel 客户来说也是好消息，它也包括你的服务范围。这样，如果你打算转到 Sprint，你就可以知道你是否有足够的覆盖率。
