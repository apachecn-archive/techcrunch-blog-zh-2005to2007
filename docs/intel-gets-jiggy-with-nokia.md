# 英特尔与诺基亚关系紧张

![](img/594d1823dd71d611dc7e01d5a12a6069.png)

今天宣布，英特尔已与诺基亚达成协议，在其迅驰双核处理器中嵌入 HSDPA 连接模块。英特尔将处理从硬件实施到软件开发的所有生产要素。
