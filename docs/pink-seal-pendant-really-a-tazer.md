# 粉色印章吊坠真的是一款 Tazer 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/15/pink-seal-pendant-really-a-tazer/>

![](img/307b599e68bdb6ac0516137500797bca.png)

下次你的女孩不得不独自(混蛋)从你的公寓走回家时，确保她配备了这个看似无辜的粉红色印章吊坠。虽然这是一件非常漂亮的珠宝，但真正的秘密却藏在海豹的内脏里。看这个粉红色的吊坠实际上有一个内置的类似泰瑟枪的装置来阻止攻击者和抢劫犯。195 伏的电力将会输送到任何一个接受这东西的人身上。足以吓跑攻击者，但不足以满足 LAPD 的菜鸟。力量和风格的完美结合，如果你愿意的话。

想要一个吗？我们不知道为什么，但是你必须飞到日本去买一个。
