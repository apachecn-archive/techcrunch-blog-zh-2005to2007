# 微软 AdSense 竞争对手的试点邀请

微软期待已久的上下文广告平台 ContentAds 今天向其试点项目的潜在参与者发出了第一份邀请。微软表示，从“主要”(他们的说法)MSN 拥有的网站开始，ContentAds 将不仅使用关键字，还使用人口统计目标、地理目标和增量竞价工具来投放广告。听起来像 AdSense 加上一些对各种 MSN 网站读者群的人口统计的考虑——我们将看到当内容广告被释放到野外时会发生什么。我们可能很快就会看到。

更大的时间竞争谷歌的 AdSense，雅虎！出版商网络和该领域的其他参与者应该意味着出版商的收入削减更多，广告服务方式更多创新。不管怎么说，这只是理论上的，尽管微软姗姗来迟且安全地进入这个游戏，却留下了一个问题，即这里是否会有很多创新。来吧微软——给我们一个惊喜！

收到邀请的在线广告专家[詹妮弗·斯莱格](https://web.archive.org/web/20221225065348/http://www.jensense.com/archives/2006/08/msn_adcenter_in.html)透露了这个消息(我通过找到了[，并预测微软的加入对于那些每月没有数百万次展示的小出版商来说将是一个特别好的消息。](https://web.archive.org/web/20221225065348/http://mrwavetheory.blogspot.com/2006/08/microsoft-getting-contextual-with.html)

就在上周，微软宣布了可能是与脸书的巨额广告交易。

更新:微软的唐·道奇相信这种新的广告产品将比谷歌或雅虎有更好的点击欺诈保护。那会很酷。
