# 雅虎！选择[windows] mobile。TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/30/yahoo-go-for-windows-mobile/>

雅虎！Go for Mobile 于今年早些时候首次推出，适用于运行 Symbian 操作系统的移动设备。从今天起，任何运行微软移动操作系统版本 2003 SE 或 Mobile 5.0 的 Windows Mobile 手机都可以使用。

雅虎！Go for Mobile 集成了 Y！邮件，搜索，照片，通讯录和日历，还有 Y！新闻、体育、金融，让消费者可以在手机上快速访问他们的个性化内容，并且对以下国家的消费者免费:美国、**英国**、法国、德国、IT、es、AU &新西兰、in、ID、MA、PH、SG、其他(国际工程)。

此外雅虎！今天宣布他们已经和领先的设备制造商达成了全球分销协议，包括摩托罗拉、诺基亚和 RIM，通过嵌入 Yahoo！在选定的设备上使用移动设备。例如，摩托罗拉将预装并突出显示雅虎！从 2007 年上半年开始，在全球范围内优化手机的移动性。

在 MSN、Yahoo 和 AOL 之间的 Web 1.0 之战中，目标是赢得消费者的心，让他们成为桌面上默认的浏览器门户。随着宽带的出现，把所有东西都放在一个地方的需求已经消失了。

如今，移动设备是谷歌、雅虎和 MSN(健身俱乐部)之间的新战场。再一次缓慢的[移动] ![](img/43e3a0f180cf4a1d8c7488aab8b4ae9d.png)访问速度意味着消费者再次乐于见到健身俱乐部提供一个聚合的移动门户作为他们的默认主页，以节省时间和金钱。

过去，与 Windows mobile 操作系统制造商的分销协议确保了 MSN 在 Windows Mobile 平台上一直拥有相当大的客户群。今天的这一宣布将对 MSN 的移动优势构成重大挑战。

目前，网上还没有明确解释微软的战略回应是什么，或者我们可以对 Windows Mobile Live 有什么期待。如今，MSN 移动服务和一些测试版应用——空间、地图、搜索——有了一个松散的联盟。

至于谷歌，伦敦是他们主要的移动开发中心。谷歌的 Web n Walk 搜索策略在英国与 T-Mobile 合作得很好，但消费者仍然对使用手机上网持谨慎态度，因为与浏览相关的数据成本很高。他们也仍然对这种体验感到沮丧，因为即使使用 3G 手持设备，访问速度也很慢。赢得移动用户的芳心的战斗才刚刚开始。

在我看来，第一轮是谷歌的网络行走活动，第二轮是雅虎！随着今天的宣布，谁或什么将赢得下一轮呢？
