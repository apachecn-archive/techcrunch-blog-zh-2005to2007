# 明天就要登陆了:漂亮的 P2P 网络电话

![ooma.jpg](img/298ba1e3cbbfdbdd620837d72a50dfcb.png)当我们花时间推动我们的[摧毁你的固定电话竞赛](https://web.archive.org/web/20210227103221/http://crunchgear.com/2007/07/18/crunchgears-destroy-your-landline-contest/)时，你们这些读者被恳求摧毁你的传统固定电话，一些事情发生了，让我们想或许保留我们的[。Ooma](https://web.archive.org/web/20210227103221/http://www.beta.techcrunch.com/2007/07/18/ooma-launches-free-consumer-phone-service/) 是一种新的类似 Vonage 的网络电话服务，它调整了公式，足以在 SunRocket 等公司失败的地方生存。

VoIP 的理想已经存在，VoIP 到 VoIP 的通话几乎总是免费的。然而，拨打传统座机会产生一些相关费用(按次或按月)。正是这些成本的增加给其他网络电话提供商带来了问题。Ooma 想出了一个相当创新的方法来规避这些成本，这可能会有所不同。

Ooma 免费使用。你需要购买它的专有硬件，399 美元是一笔不小的投资。但是当你考虑到你从家里打电话(在美国境内)将永远免费时，就把它当作你未来的首付吧。

大多数传统的固定电话到 VoIP 的硬件由一个包含实际魔力的中央交换机组成。Ooma 使用一个基站和“卫星”交换机(称为 Scouts)来分散您的家庭电话网络(我正在注册商标，不要尝试)。

大多数传统的固定电话转网络电话提供商还会把你拨到最近的邻居交换机上，在城市地区每隔半英里左右，在农村地区每隔几英里，不管你打电话的人有多近。

然而，Ooma 喜欢点对点的交流。您的 Ooma 基站不仅允许您拨打电话，还能帮助其他人拨打电话。如果有另一个 Ooma 基站比交换机更靠近您要呼叫的人，则使用它。太棒了。

如果有足够多的人开始在互联网上使用 Ooma，最终公共电话网络将会越来越少被使用。点对点绝对是互联网的未来，Ooma 可能会证明它是所有通信的未来。

Ooma 将于明天早上发布，其病毒式营销团队由一位名不见经传的演员阿什顿·库彻领导(这位阿林顿老兄正在为我们的 TalkCrunch 姐妹网站采访他)。你很快就会听到很多关于它的事。

寻找 MTV 人群推送(MTV 人群还有意义吗？我已经 30 多岁了，我不知道)到处都是，即将到来的付费下载功能被添加到一些手机中(例如，铃声和不可能玩的游戏)。

此外，该公司还组建了令人印象深刻的教练阵容，这些人已经成为值得关注的对象。明天早上去看看。
