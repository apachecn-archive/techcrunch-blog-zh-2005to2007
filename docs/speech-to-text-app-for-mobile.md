# 手机语音转文本应用

作为一个在小学就学会打字的人，“发短信”这个概念对我来说是陌生的。我更喜欢用我所有的手指来写一条信息，而带有共享字母按键的设备可能会让人彻底沮丧。但如果我能说出一条信息，然后以文本形式发送出去，我会非常高兴。好吧，声音信号听得很清楚。该公司刚刚推出了 VoiceMode 2.0，这是第一款针对手机的连续听写解决方案。

“VoiceMode 2.0 是 VoiceSignal 在开发使用语音来简化手机操作的应用程序方面的又一个领先例子，”VoiceSignal 首席执行官 Rich Geruson 说。“VoiceMode 2.0 让人们能够自然地创建文本信息、文档或向日历等应用程序添加文本，这比他们只使用键盘要简单和方便得多。这是 VoiceSignal 的另一个非常令人兴奋的突破。”

这款应用让用户可以通过说出他们想要的内容来创建短信，而不必使用键盘。此外，VoiceMode 2.0 允许用户在手机上的任何基于文本的应用程序中输入文本，包括短信、彩信、电子邮件、笔记、日历甚至办公文档。VoiceMode 字典拥有数万个单词，并且可以添加其他单词以允许更大的定制。这些必须用键盘手动添加，但这就是生活。

VoiceMode 2.0 将在部分诺基亚设备上推出，并将于今年晚些时候登陆英国和美国。
