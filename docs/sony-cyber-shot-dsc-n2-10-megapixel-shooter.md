# 索尼 Cyber-Shot DSC-N2 1000 万像素相机

图片？好消息，1000 万像素的傻瓜相机刚刚正式发布。

这款相机有 3 倍的光学变焦，3 英寸的触摸屏 LCD 和 1/1.8 英寸的 CCD 传感器。它很好地利用了它的大屏幕和内置的相册，可以容纳 500 张照片。

索尼的相机有 56MB 的内部存储空间，支持 4GB 的 Memory Stick Duo 或 Memory Stick PRO Duo 卡，可以在幻灯片模式下显示 1100 张图像。它将于 10 月上市，售价 450 美元。
