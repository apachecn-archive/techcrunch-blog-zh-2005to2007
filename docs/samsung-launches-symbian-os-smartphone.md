# 三星推出 Symbian OS 智能手机| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/25/samsung-launches-symbian-os-smartphone/>

三星今天在西班牙马德里举行的 S60 峰会上发布了一款新的塞班操作系统智能手机 SGH-i400。哇，等一下。既纤薄又时尚的智能手机？不会吧！i400 正是如此，因为它配备了 S60，该系统提供了全面的语言支持系统，增强的可下载应用程序，以及多任务处理功能。

三星的 i400 是一款面向年轻职场人士的智能手机。它具有标准功能，如完整的网络浏览、蓝牙、200 万像素摄像头和 MP3 播放特定按钮。它还包括一个可扩展到 4GB 的 miniSD 插槽。我想 2.3 英寸宽的屏幕是可以接受的，但是谁想在非 QWERTY 键盘上啄来啄去呢？i400 将于 7 月在俄罗斯上市。俄罗斯？WTF？！
