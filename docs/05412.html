泄露的三星SGH-i760看起来不错

在过去的几个月里，我经常使用T-Mobile的三星T809。我真的很喜欢滑块的设计、功能和显示，尽管我从它那里获得了糟糕的电池寿命。这款泄露的三星SGH i760 Windows Mobile 5 Pocket PC智能手机配有滑出式QWERTY键盘，看起来像是T809的缩影。这些规格看起来非常美味，有蓝牙，802.11g无线，microSD插槽，2.8英寸QVGA触摸屏，200万像素摄像头，3G/HSDPA三频支持，以及向前视频通话的能力。那个键盘看起来也很宽敞。
