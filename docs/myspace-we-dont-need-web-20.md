# MySpace:我们不需要 Web 2.0

新闻集团首席运营官彼得·切宁今天告诉公司投资者，“如果你看看几乎所有的 Web 2.0 应用程序，无论是 YouTube，Flickr，Photobucket 还是任何下一代的 Web 应用程序，几乎所有的应用程序都被逐出了 MySpace。”[多频道新闻](https://web.archive.org/web/20221026092302/http://www.multichannel.com/article/CA6371185.html?display=Breaking+News)报道称，切尔宁表示，新闻集团没有理由不能建立平行业务，特别是针对 YouTube。“考虑到他们的大部分流量来自我们，”他说，“如果我们有足够的、甚至更强的竞争对手，我认为我们应该能够赶上他们，甚至超过他们。”

Chernin 今天在美林媒体和娱乐会议上的讲话没有被讨论的是该公司已经采取的措施，这些措施使外部公司更难在 MySpace 内传播他们的存在，如[在 Flash widgets 中阻止外部链接](https://web.archive.org/web/20221026092302/http://www.beta.techcrunch.com/2006/07/20/myspace-security-measure-disables-viral-spread-of-widgets/)。还会有更多类似的阻碍吗？[澄清一下，这是 Chernin 发表评论的背景，他没有讨论阻止其他公司的小工具。]

虽然竞争对手脸书[因向外部开发者开放 API 而赢得赞誉](https://web.archive.org/web/20221026092302/http://www.beta.techcrunch.com/2006/08/15/facebook-requests-developer-friends-with-new-api/)，但据了解，MySpace 这样开放的可能性可能为零。

目前还不清楚 MySpace 还能通过哪些功能来与 YouTube 竞争。MySpace 视频播放器有嵌入、相关视频、热门视频和观众评论。Chernin 说，MySpace 的视频努力到目前为止还很小，估计 YouTube 的 60%到 70%的流量来自 MySpace。随着 YouTube 社区发展出自己的明星，这种情况可能会变得不那么严重，这些明星最多将 MySpace 页面作为静态参考点。

切尔宁还表示，该公司正寻求在 MySpace 上放更多自己的商业视频。“你会看到我们开始在该网站的娱乐方面变得更加积极，”他说。YouTube 上的商业视频是一场豪赌，其中一些广受欢迎，一些则引发了用户的强烈不满。

总结一下:新闻集团的首席运营官说，Web 2.0 正在从 MySpace 上分流流量，他们可以建立自己的服务来与之竞争，网站上将会有越来越咄咄逼人的商业攻势。这听起来既傲慢又危险，就像是对 MySpace 依赖对于外部开发者来说风险太大的担忧的真实验证。

奥姆·马利克昨天在《商业 2.0》上发表了一篇题为[突然一切都来了 Widgets](https://web.archive.org/web/20221026092302/http://money.cnn.com/magazines/business2/business2_archive/2006/09/01/8384338/index.htm?postversion=2006091105) 的文章，他说“这里每个人都是赢家:MySpace，因为它变得更粘了；YouTube 和 Slide，因为他们获得了流量；和用户，因为他或她在一个页面上就能得到所有这些。”听起来 MySpace 的所有者可能不想玩一个所有人都赢的游戏。

我希望我们能听到新闻集团的其他人有不同的看法，或者彻宁的说法是有根据的。来自许多不同组织的创新交换数据，并允许用户将信息从一个平台转移到另一个平台，这是所谓的 Web 2.0 的关键特征。Chernin 的声明可能有商业意义，但对创新来说不是好兆头。如果你想知道跨站点小工具有多酷，看看 [Typepad 的方法](https://web.archive.org/web/20221026092302/http://everything.typepad.com/blog/2006/09/new_widget_upda.html)或者新的小工具市场 [Widgetbox](https://web.archive.org/web/20221026092302/http://widgetbox.com/) 。

我想没人会感到惊讶。也许所有讨厌 MySpace 糟糕代码的外部开发者会感受到一种新的自由感。
