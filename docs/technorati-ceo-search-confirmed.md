# Technorati 首席执行官搜索确认

 [](https://web.archive.org/web/20220815072053/http://www.technorati.com/) [Technorati](https://web.archive.org/web/20220815072053/http://www.technorati.com/) 聘请总部位于纽约的 [James 和公司](https://web.archive.org/web/20220815072053/http://www.jamescosearch.com/)为公司物色首席执行官。过去几周，该公司一直在接触硅谷和其他地方的潜在候选人，以评估他们对领导这家成立近四年的公司的兴趣。目前还不知道创始人兼现任首席执行官戴夫·西弗里是否会在公司担任新职位，或者会跳槽到其他公司。当然，也有可能找不到合适的人选来替代赛弗里。

Technorati 的统计数据最近大幅度上升，尽管他们目前的一些分销协议可能处于危险之中。据报道，该公司的员工上周被告知首席执行官的搜索。

Technorati 是最大的博客搜索引擎，blogsearch.google.com 位居第二。当然，如今谷歌对博客的索引比一年前要频繁得多，这表明谷歌在 Google.com 的主要搜索引擎很可能是 Technorati 最大的直接竞争对手。

Technorati 尚未回复要求置评的电子邮件。Technorati 寻找首席执行官的传言最早是由 Valleywag 报道的。
 ***更新:**我对戴夫·西弗里的询问促使他就此事写了一篇[博文](https://web.archive.org/web/20220815072053/http://www.sifry.com/alerts/archives/000494.html)。他说他将继续留在公司担任产品职务。我认为这是明智之举。*
