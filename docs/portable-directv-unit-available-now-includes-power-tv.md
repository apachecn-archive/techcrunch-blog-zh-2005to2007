# 便携式直播电视单元现已上市；包括电源、电视| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/04/portable-directv-unit-available-now-includes-power-tv/>

我住在公寓里。这是一个不错的公寓，可以看到西雅图的大部分风景，从市中心一直到 U 区，甚至是弗里蒙特桥。遗憾的是，和许多公寓居民一样，我只能看有线电视。我所在的大楼对直播风格的迷你菜有严格的规定。经理说这是因为装修，我们认为这更多的是因为康卡斯特给他的电视机提供了很大的折扣。

Sat-Go 可能是一种选择。当然，它的价格是 1499 美元，而且我仍然需要支付服务费，但是我可以带着这个便携式天线去我喜欢的任何地方。该装置不只是一个碟形天线，还包括一个接收器、调谐器和 17 英寸液晶显示器，整个装置依靠内部锂离子电池运行。太棒了。

不知道充电持续多长时间，但由于它没有移动部件，是一个被动接收器，我们猜测比笔记本电脑长。由于它配有一个香烟插头适配器，你真的可以把它带到任何你想看电视的地方，只要你在视线范围内。我想，我在这附近的许多酒吧里会很受欢迎。
