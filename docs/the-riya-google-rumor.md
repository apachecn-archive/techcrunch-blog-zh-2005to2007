# Riya-Google 谣言

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/11/17/the-riya-google-rumor/>

今早有传言称 [Riya](https://web.archive.org/web/20220811122043/http://www.riya.com/) (一家[杀手级照片面部识别初创公司](https://web.archive.org/web/20220811122043/http://www.beta.techcrunch.com/2005/10/26/riya-prepares-to-launch-alpha/))已经或正在被谷歌以 4000 万美元的价格收购。参见[奥姆马利克](https://web.archive.org/web/20220811122043/http://gigaom.com/2005/11/16/googles-riya-designs/)、[尼尔肯尼迪](https://web.archive.org/web/20220811122043/http://www.niallkennedy.com/blog/archives/2005/11/google_purchasi.html)和[保罗凯德罗斯基](https://web.archive.org/web/20220811122043/http://paul.kedrosky.com/archives/002058.html)。

即使每个人都在写博客，这也只是一个谣言。不过，这是一个谣言，一些公司的员工向我证实了这一点，这些公司显然也在寻找 Riya，但在价格变得过于昂贵后退出了。

这些谣言肯定会让明天晚上在我家举行的 Riya 发布会更加有趣。

我想强调的是，尽管我认识 Riya 的人，尽管我主持他们的发布会，但我没有从他们那里得到交易的直接确认(是的，我试过)。他们对此三缄其口，无论谣言准确与否都是可以理解的。

上个月我测试了 Riya，给我留下了深刻的印象。在这里看我的[简介](https://web.archive.org/web/20220811122043/http://www.beta.techcrunch.com/2005/10/26/riya-prepares-to-launch-alpha)。Riya 拥有坚实的技术和令人印象深刻的团队。
