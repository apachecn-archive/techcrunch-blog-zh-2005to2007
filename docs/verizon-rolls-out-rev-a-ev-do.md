# 威瑞森推出 EV-DO 牧师

为了提供各种服务，威瑞森今天宣布启动其 EV-DO 修订版 A 网络。首先，Rev. A 网络将在波士顿、里士满、芝加哥、盐湖城以及整个佛罗里达州内运行。此次升级将允许威瑞森无线数据计划的用户以 400-800 kbps 的速度下载。相当不错。

为了迎接发布会，威瑞森还宣布了 USB720 无线调制解调器。据说，它能够通过一个行业标准的 USB 接口连接到“所有最新的笔记本电脑、台式机和手持设备”它将于明天上市，但价格尚未确定。
