# Nvidia G80 详情

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/06/nvidia-g80-details/>

![](img/005bd7731c3b024d1c6ff69cc6402a37.png)

Nvida 已经发布了一系列关于下个月即将发布的零售 G80 主板的细节。该板将被命名为 GeForce 8800，并将有两种型号:GTS 和 GTX。

GTX 将拥有 575MHz 时钟和 768MB GDDR3 内存，时钟频率为 900Mhz。它的内存是 384 位，吞吐量为 86GB/s。

对于那些不要求绝对最高性能的人来说，GTS 适合更看重价值的消费者。它宣称拥有 500MHz 时钟和 640MB DDR3。内存只有 320 位，吞吐量为 64GB/s。

这两种主板都将符合 HDCP 标准，并具有双链接 DVI、VIVO 和 HDTV 功能。目前还没有价格方面的消息，但预计价格将在 11 月的第二周下降。
