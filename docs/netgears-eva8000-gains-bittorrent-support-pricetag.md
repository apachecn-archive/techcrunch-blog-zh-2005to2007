# Netgear 的 EVA8000 获得 BitTorrent 支持，价格标签

有传言称，苹果的 Apple TV 将于明天，即 3 月 20 日上市，虽然我们很兴奋地想看看苹果能给美国人的客厅带来什么，但我们中的一些人对这款设备缺少的一些功能有些担心。DivX？几乎没有。
 作为替代，人们可能会考虑 Netgear 的 EVA8000，[我们在](https://web.archive.org/web/20201128080802/http://crunchgear.com/2007/01/08/netgear-drops-hd-set-top-box-for-wireless-streaming/)之前已经看过的盒子。我们当时报道说它不能在 iTunes 上运行，但现在看起来可以了，这是一个不错的补充。但真正让我们再次思考这款设备的是它与 BitTorrent 配合良好的消息，这意味着你可以从流行的 BT 网络下载过多的~~非法~~免费内容，并在你的客厅以高清播放，高达 1080p，优于 Apple TV 的 720p。350 美元，价格相当有竞争力。如果你真的喜欢娱乐，EVA8000 也可以满足你，因为它可以全屏播放 YouTube 内容。很好。

我们向 Netgear 致敬，它努力让我们已经有的媒体可以在任何地方的电视上看到，无视电影商店的想法，并勇敢地面对诉讼。够了，网件，够了。
