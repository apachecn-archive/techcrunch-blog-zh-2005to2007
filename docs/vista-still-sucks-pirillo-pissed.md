# Vista 还是很烂，皮里洛很生气

![](img/218ae499f4ad845b37da47e8b12bd787.png)

Chris Pirillo 安装 Windows Vista 已经六个月了。地狱般的六个月。Chris 就像我们中的许多人一样，最初对 Vista 感到兴奋，却发现我们一半的软件和硬件都无法使用它。在皮里略的案例中，他日常工作使用的是一台传真/打印机一体机。经过几个月的挣扎，甚至一度使用过 VMWare，他觉得自己已经受够了。他又换回了 Windows XP。

考虑到 Vista 的缺陷和漏洞，这一举措是明智的。许多 Windows 用户已经切换回 XP，包括许多 iPod 用户，当他们发现 iTunes 在 Vista 上无法运行几个月时，他们被吓呆了。现在这个操作系统已经发布半年多了，是时候让微软开始运作了。让我们看看一些真正的补丁和服务包，它们会让 Vista 看起来更像 XP。
