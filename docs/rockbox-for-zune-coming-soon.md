# Zune 的 Rockbox 即将推出

如果你喜欢 Zune 的硬件(大屏、简单的控制和无线网络),但对软件或界面(或两者都有)感到遗憾，那么你可能想看看即将推出的 Rockbox 的 Zune 安装。

对于那些不知道的人来说，Rockbox 是一个开源固件替代项目，旨在让你根据自己的喜好定制音频播放硬件(如 Zune 或 iPod)，从界面到功能。这个想法是，你可以酿造自己的 Rockbox，然后在你选择的硬件上运行它，将用户体验的权力和控制权交给你。

虽然它还没有准备好，但开发人员正在开发 Rockbox 的 Zune 端口(尽管是在自愿时间内)，这与他们已经开始工作的东芝 Gigabeat 端口没有太大区别。这意味着在未来的某个时候，我们应该会看到一个可供下载的可用版本。但在你兴奋之前，值得注意的是，如果你在 Zune 上运行 Rockbox，你可能不仅会失去与 Zune Marketplace 的兼容性，而且很可能会使你的保修失效。这还不适合没有经验的人或胆小的人，但这是值得考虑的事情。
