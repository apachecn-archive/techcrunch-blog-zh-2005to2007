# 记住待办事项清单

![RememberTheMilk](img/8dd98c2ceff59c49a17b5f4d3c3fe323.png)**Company:** [Rememberthemilk](https://web.archive.org/web/20220702000257/http://www.rememberthemilk.com/) **Launched:**

2005 年十月

**Location:**

澳大利亚悉尼

 [RememberTheMilk](https://web.archive.org/web/20220702000257/http://rememberthemilk.com/) 是一个新的富含 ajax 的待办事项列表，类似于 37 Signal 的 [Ta-Da 列表](https://web.archive.org/web/20220702000257/http://www.tadalist.com/)。

记住，Milk 列表是按选项卡组织的。项目很容易输入(尽管有一个额外的点击让我很烦)。在列表之间点击非常简单。项目可以很容易地重新排序。您还可以共享列表和/或选择公开列表。

RememberTheMilk 做得很好的一件事是允许大量元数据与单个任务相关联。优先级可以用一个漂亮的颜色编码系统来设置，在设置完成日期方面也有灵活性。您也可以给任务添加注释。

一个非常好的功能是通过电子邮件添加任务的能力。

提醒可以通过电子邮件、即时消息或短信发送。你也可以通过 RSS 订阅列表。

总的来说，使用 RememberTheMilk 比使用 Ta-Da 列表要丰富得多。设置日期提醒特别有用。然而，在易用性方面有一个明确的折衷。使用 Ta-Da 列表不需要培训，而我似乎经常参考 RememberTheMilk FAQs 来理解功能。

RememberTheMilk 是由 Emily Boyd，Omar Kilani 和一个名叫 Bob T. Monkey 的填充动物创造的(我自己更喜欢活的吉祥物)。:-)
