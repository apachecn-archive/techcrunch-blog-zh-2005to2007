# Zillow 房地产数据来到你附近的网站

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/10/26/zillow-opens-real-estate-data-api/>

[![](img/b6d92f57a887bfb7b5ddf77aa1eeba12.png)](https://web.archive.org/web/20170410122822/http://www.zillow.com/) 房地产搜索公司 [Zillow](https://web.archive.org/web/20170410122822/http://zillow.com/) 已经开放了两个 API，允许其他网站利用其关于美国各地房屋的数据。该公司上个月制造了新闻。成为其房地产网站上预发布 API 的最早用户之一。总共有 30 多个 API 的预发布用户，还包括北加州和内华达州保诚、ZipRealty 和 Redfin。

除了自己的数据，Zillow 现在还包括房屋所有者上传的房屋信息[。该网站还提供了基于微软“鸟瞰图”的房屋 3d 视图如果你以前没有尝试过在 Zillow 中输入一些地址，我推荐你这样做只是为了好玩。](https://web.archive.org/web/20170410122822/http://www.techcrunch.com/2006/09/19/zillow-adds-user-generated-home-information/)

现在，第三方将能够在他们的网站上访问和整合两套数据中的任何一套，一套是关于房屋估价的，另一套是关于财产细节的。API 的使用是免费的，默认情况下，用户每天可以制作 1000 个。对 Zillow 来说，这是一个非常聪明的营销举措。对于小型网站来说，增加功能和复杂性的最好方法之一是将来自外部的丰富数据集成到自己的产品中。从视频搜索到联合照片，一切都是如此。对小型网站出版商来说，向高度可移植数据的转变是一个非常激动人心的时刻，他们比以往任何时候都更容易围绕自己的核心价值主张包装一个功能丰富的环境。

总部位于西雅图的 Zillow 已经筹集了 5700 万美元的风险投资，并于今年 2 月推出了测试网站。在 8 个月的时间里，从初创到成为你所在领域顶级网站的主要数据提供商之一，令人印象深刻。这在很大程度上是因为 Zillow 的高管团队，由 Expedia 创始人兼前首席执行官 Rich Barton 和 Expedia 前高级副总裁 Lloyd Frink 领导。

其他从事类似工作的初创公司包括 [Trulia](https://web.archive.org/web/20170410122822/http://www.techcrunch.com/2005/10/30/real-estate-vertical-search-with-trulia/) 、[realestatabc](https://web.archive.org/web/20170410122822/http://www.techcrunch.com/2006/03/29/zillow-has-competition/)和 [Redfin](https://web.archive.org/web/20170410122822/http://www.techcrunch.com/2006/05/31/redfin-can-rewrite-real-estate-rules/) 。
