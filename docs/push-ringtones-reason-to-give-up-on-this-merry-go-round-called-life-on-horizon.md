# 推铃声，理由放弃这个旋转木马所谓的“生活”在地平线上

![ringing.gif](img/16f1ad94dd9a2e78746c23326ce335de.png)

我可以想到一些我可以花 770 万美元购买的东西，这些东西不会像推环者那样令人讨厌，也不会那么愤世嫉俗。这个想法并不新鲜——有些人称之为回铃。当我给你打电话时，它会播放我选的歌。这是有史以来最糟糕的事情。

当然，像我这样的人会用它做好事。我希望你喜欢旅途中的“不要停止相信”。但我知道我认识的人会选那首“我的驼峰”的歌，强迫我把铃声调到静音。朋友们，这种行为没有任何借口。

Emotive Communications 获得了一大笔投资来完善这项服务，如果你想这么称呼它的话。如果适用的提供商选择使用这项服务，它应该可以在所有 3G、4G 和 VoiP 手机上工作，我们希望他们不要这样做。这意味着每个人，不仅仅是早期收养的青少年，都可以参与进来。

美国手机和网络电话运营商:听我说。我是专家。不要让它落入坏人之手，因为这是对美国一切神圣事物的爱。仅此而已。
