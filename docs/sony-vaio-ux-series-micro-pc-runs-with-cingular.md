# 索尼 VAIO UX 系列微型电脑采用 Cingular | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/08/10/sony-vaio-ux-series-micro-pc-runs-with-cingular/>

![sony_vaio_ux_micro_pc2.jpg](img/1981a3a78db3b757994e647b4b0d10b3.png)

折纸炒作来了又去，但这并不意味着超便携电脑不会符合我们这些电脑迷的要求。VGN UX180P 是一台 1.2 磅的迷你电脑，配有英特尔酷睿单核芯片和 30GB 硬盘空间。对于无线网络，你有 Cingular 的 EDGE 网络、蓝牙、Wi-Fi 或有线以太网，它看起来只需 1749.99 美元就能在亚马逊销售。它已经存在了一段时间，但这是我们第一次看到一个重型，超级性感的日本电脑在很长一段时间内登陆这些海岸。
