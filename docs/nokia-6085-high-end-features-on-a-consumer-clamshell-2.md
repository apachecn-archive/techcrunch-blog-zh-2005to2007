# 诺基亚 6085:消费者翻盖手机的高端功能

诺基亚今天向我们介绍一款新的 GSM 翻盖手机——6085。作为一款相当不错的大众市场手机，6085 拥有这一类别手机的所有标准功能，包括 VGA 摄像头、单色外部显示器、蓝牙和额外的 FM 调谐器。

这款手机的有趣之处在于，诺基亚决定添加通常为高端手机保留的功能，如支持 Flash 动画、微型 SD 卡插槽和带 MP3 播放的立体声蓝牙。

这款手机也支持 EDGE，所以很快就可以从 Cingular 或 T-Mobile 找到它，可能定位于 VZN 的巧克力系列。
