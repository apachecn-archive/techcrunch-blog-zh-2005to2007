# PepperPad 3 实践

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/28/pepperpad-3-hands-on/>

![](img/778db34f7512f72e7a08e1cc0f54f4b5.png)
![](img/391468f07570948624d96e3e201107da.png)pepper pad 3 是一台厨房电脑。不像以前的版本，也不像充斥在平板电脑战场上的多种平板电脑，它只是想成为一个方便的设备，用来查找一两个食谱，听一些互联网广播，也许还会播放一些预先录制的日间电视节目或在浴室里浏览网页。

它大约和一本平装书一样大，运行 AMD Geode 533-Mhz 处理器。我们收到的设备有 20GB 的磁盘空间和 256MB 的内存。这个垫子重约两磅，由橡胶塑料包裹，握起来很舒服。触摸屏相当敏感，分离式键盘虽然一开始很烦人，但你会越来越喜欢。还有一个网络摄像头，方向垫和小滚轮。总的来说，它非常适合握在手中，感觉足够结实，可以从厨房的柜台上摔下来。

目前，这些应用程序还相当基础。有一个视频播放器和一个音频播放器，还有一个电子邮件客户端和一个 AIM 客户端。所有应用程序都可以从屏幕底部、标签页或主图标屏幕访问。大多数媒体存储在磁盘上——这是一件相当讨厌的事情——但你也可以使用 PepperPad 的桌面应用程序与 Windows 桌面同步。没有添加网络共享驱动器的可见方法，但您可以连接外部 USB 驱动器，将内容拖放到 pad 上。

作为一名快乐的诺基亚 770 用户，我喜欢将 PepperPad 这样的设备与诺基亚的小型平板电脑进行比较。pepper pad 主要是“互联网平板电脑”，而不是像我们一直看到的 UMPC 垃圾那样的“没有键盘的笔记本电脑”。PepperPad 非常受欢迎，因为它是专门为不会使用笔记本电脑的人设计的。它坚固耐用，易于使用，并且足够小，不用时可以隐藏起来。也就是说，它也是黑客的梦想，在亚马逊上售价仅为 699 美元，和 770 差不多。由于其 Linux 基础，它可以运行几乎任何 Linux 应用程序。

为什么你应该买一个 Pad 而不是笔记本电脑？如果妈妈需要一台电脑来处理她的食谱、电子邮件和一些目标，而妈妈是最伟大的一代，不想要那些电脑病毒，这就是答案。如果格兰普斯想在地下室听当地的古典音乐，或者下载木工图案，好吧，口香糖，给你。如果你想在沙发上浏览网页，然后在打瞌睡时把这个东西放在你旁边的地板上，这就是你的设备。

如果你想完成任何工作，去买一台笔记本电脑。如果你想要的东西是即时的，有一个可接受的外形，并会及时融入你的生活，那么看看 PepperPad 3。
