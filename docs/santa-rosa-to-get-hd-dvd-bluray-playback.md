# 圣罗莎得到 HD DVD，蓝光播放

![](img/019e6ef8b9826f9036a763dfe8334d65.png)

根据英特尔移动图形发言人 Mike Choi 的说法，英特尔将开始在 Santa Rosa 的某些版本中使用第三方硬件 HD DVD 和蓝光解码器。使用单独的解码器将使具有更便宜的集成显卡的机器能够播放高清媒体，同时允许 Vista 的 Aero 界面流畅运行并降低成本。这将有望让更多的消费者采用这种格式，同时也可能让这场战争变得更加恼人。太好了。

考虑到微软永远不会支持另一种光盘格式，这个消息多少有点令人惊讶。
