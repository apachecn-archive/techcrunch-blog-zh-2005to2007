# 信号软件允许从 iPhone 控制 iTunes

![](img/d3ea29c827c4dafa900d733c740c9fcb.png)

Alloysoft 的一款新软件现在可以让你通过 iPhone 控制 iTunes、Winamp 或 Windows Media Player。它叫做 Signal，它使用 WiFi 通过网络向你的电脑发送信号来控制音乐和电影的播放。包括实时更新，这样你就可以知道如果你不在电脑旁边，哪首歌曲或视频正在播放。另外，iPhone 的界面看起来真的很光滑。

听起来很适合在家庭聚会中控制音乐。它的价格是 30 美元，对于这样的基础软件来说是相当昂贵的。对一些人来说，PC 版可能值得，尽管我不太确定为什么有人会买 Mac OS X 版。附带的苹果遥控器不是已经有效果了吗？
