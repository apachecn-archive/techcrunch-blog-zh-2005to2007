# 用罗杰斯？你的种子可能没用

![](img/7268a67c7af8456b2ca1930331c24e7f.png)

因为我不是加拿大人，我对罗杰斯的所作所为有点不知情。据 TorrentFreak 报道，由于过度消耗带宽，ISP 禁止了 BitTorrent 流量。输入加密，在接下来的一年里，每个人都是安全的。现在，为了遏制加密比特洪流流量，罗杰斯已经开始扼制所有加密传输。这意味着你的安全电子邮件、登录以及这两者之间的一切都被搞砸了，除了直接向罗杰斯投诉，你什么也做不了。
 TorrentFreak 还指出，罗杰斯的口号仍然是“共享大文件和更多”现在我在内心深处开心地笑着，真的希望你也一样。今天就给罗杰斯打电话，告诉他们你不希望你的加密流量被转移和阻塞！
