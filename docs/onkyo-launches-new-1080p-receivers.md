# 安桥推出新的 1080p 接收器| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/01/onkyo-launches-new-1080p-receivers/>

![](img/ccbd3cf472a9bf2361f40173924b708d.png)

安桥做了一些很棒的东西。如果我不是一直处于经济破产的边缘，我可能会给自己买一个。无论如何，TX-SR674 和 TX-SR604 接收机具有一系列高端功能，面向中级家庭消费者。

两款接收器都有两个 HDMI 输入和 7.1 环绕声或多声道声音。它们都有 Audyssey 2EQ 用于自动声学校准，并且支持 XM。TX-SR674 可以以 7×95 的速度推进，零售价为 799 美元，而 TX-SR604 只能在 7×90 的阈值下工作，零售价为 599 美元。
