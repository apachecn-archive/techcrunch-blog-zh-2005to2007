# 重新推出 CrunchBoard 工作板 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/03/09/relaunching-the-crunchboard-job-board/>

 [](https://web.archive.org/web/20221209142400/http://www.crunchboard.com/) 有了新版本的 [CrunchBoard 工作板](https://web.archive.org/web/20221209142400/http://www.crunchboard.com/)——我们已经去掉了我们的定制代码，并用新的 [Edgeio 市场产品](https://web.archive.org/web/20221209142400/http://www.edgeio.com/view/marketplaces/)取而代之。

Edgeio 是我共同创办的一家公司，我花了很多时间和他们一起开发产品功能，以确保它对我们和其他希望拥有自己的分类广告网站的网站来说是完美的。他们将定制页面，使其具有与您的主网站相同的外观和感觉，并处理所有的物流、客户服务和支付问题。如果你选择对列表收费，他们会收取总费用的 20%，否则产品就是免费的。

edgeio 有一个我们已经实现的功能——可以在你自己的求职板上交叉推广。因此，在列出工作的过程中，我可以问列表者他们是否愿意在另一个基于 edgeio 的板上列出他们的工作，可选地收取额外的费用。如果名单上的人选择这样做，费用将在我和其他求职者之间按我们希望的任何方式分摊。

另一个尚未推出的功能是联盟计划。当这一功能启动后，其他网站可以成为 edgeio 支持的网站的附属网站，并获得销售列表的佣金。将有一个小部件和简单的链接选项的子公司。一旦有了，我会尽快添加，并把三分之一左右的费用分给代销商。

我们现在也将把 Crunchboard 扩展到其他类型的相关列表。稍后会详细介绍。

对我们来说，重要的是为新的创业公司提供相关的工作列表，并尽可能容易地将雇主和雇员联系在一起。edgeio 让我们做到了这一点，没有管理所有 bug 的麻烦，也没有维护自己代码的麻烦。

**注意:**有很多其他优秀的求职板提供商可供选择([我在这里写了很多](https://web.archive.org/web/20221209142400/http://www.beta.techcrunch.com/2007/01/24/the-job-board-bubble/))，我很明显在选择 edgeio 的问题上有矛盾。尽管如此，我认为它的价格和灵活性使它超越了工作列表，成为一个定制的 craigslist 类型的网站，适合任何类型的列表，这是一个不错的选择。
