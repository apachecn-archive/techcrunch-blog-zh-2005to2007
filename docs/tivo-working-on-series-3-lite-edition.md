# TiVo 正在制作系列 3“精简版”

![](img/d781c7e31712b08580e737993b61b9c1.png)

有人说 TiVo 是在垃圾堆里。恕我不能苟同。如果现在给我一个 TiVo，我会毫不犹豫地接受。但是我会把它还回去，因为我知道康卡斯特提供了一个高清 DVR 有线电视调谐器，完全免费。一个盒子，简单多了。TiVo 知道这一点，它需要一个计划。这个计划就是 3 Lite 系列。

在 TiVo 最近的 Q1 收益报告中，讨论了一种比当前的 Series 3 更便宜的新盒子。这个“Lite”盒子可能会在第四季度推出，出于成本考虑，系列 3 的一些功能将被删除。一次录制两个节目的功能或 SATA 可能会消失。谁知道呢。我们将在接下来的几个月里看到一个新的 TiVo 盒子是否即将实现。
