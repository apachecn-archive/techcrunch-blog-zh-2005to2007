# 东芝 Gigabeat S60 评测

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/27/toshiba-gigabeat-s60-reviewed/>

![](img/f9a545d01e938a8993ff9290572cc54d.png)

Krunker 发布了对东芝 Gigabeat S60 的评论，除了一些缺点，他们似乎很喜欢它。这款 Zune 之前的 Zune 媒体播放器使用的是嵌入式版本的 windows，实际上它的用户界面看起来相当不错。视频播放很好，虽然没有基于 WiFi 的歌曲共享，但这款播放器仍然是一个不错的竞争对手。

他们似乎多次指出的一个主要问题是电池寿命。显然，Gigabeat S60 的电池续航时间并不太长，而且你必须使用 Windows Media Player 来管理你的音乐和视频。这很糟糕，但除此之外，球员体面地站了起来。尤其是价格方面。查看所有细节的完整评论。
