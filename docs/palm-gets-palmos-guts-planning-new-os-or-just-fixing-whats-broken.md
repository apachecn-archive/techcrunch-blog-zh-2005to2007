# Palm 得到 PalmOS 的内脏，规划新的 OS，还是仅仅修复坏掉的？

今天，Palm 公司宣布了一项价值 4400 万美元的许可协议，获得 PalmOS 5 (Garnet)源代码的许可。有人猜测，Palm 最终会授权休眠的 OS 6 (Cobalt)利用其多任务特性和内置的单手导航功能，或者成为 Access 下一代移动操作系统 ALP (Access Linux 平台)的早期授权厂商。

相反，Palm 现在不仅有权在其时髦的 Treo 智能手机上安装 OS 5，还可以篡改 OS 5 的内容。以任何标准来看，这个操作系统都是陈旧的，但这项独特的交易让 Palm 能够使用 Garnet 的任何或所有部分，这意味着它可能正在努力酝酿自己的操作系统，这确实是一个不同寻常的举动。

作为类比，想象一下如果戴尔购买了 Windows XP 的源代码权利。然后，它可以在销售的服务器上安装自己的操作系统，使它们与 windows 软件 100%兼容，而不必安装 Windows(或支付许可费)。这当然不太可能，但凸显了这一举措的价值。
