# Juicy pinppl 正在刷新移动社交网络

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/05/pinppl-mobile-social/>

在蓬勃发展的移动社交社区中，你必须有一个与众不同的地方。Pinppl (有些人发音为“菠萝”，尽管其创始人保罗·安东尼明确发音为“PIN 人”)通过建立一个专属于黑莓用户的社区来提供产品，所有这些都基于无处不在的黑莓 PIN。

PIN 是黑莓自己专有的用户寻址系统。每个黑莓手机都有一个(尽管许多用户可能不知道他们自己的)，黑莓用户可以像发送电子邮件或短信一样轻松地相互发送 PIN 码信息。PIN 信息是“点对点”的，只能从一个黑莓手机传到另一个。

PIN 也是现有的每一部黑莓手机的通用标识符，pinppl 正是利用了这一特性。这项服务本身是一个非常简单的社交网络，以标签、个人资料、联系和其他内容为特色。但事实上，这项服务不仅可以从黑莓自己的浏览器上使用，因为它的极简主义(但美学上令人愉悦)用户界面意味着你可以在旅途中比一些专注于商业的社交网络更容易地建立你的网络。(有没有试过在你的黑莓手机上关注 Linkedin 的链接？)我说“以业务为中心”是因为:让我们看看黑莓的用户群体是什么样的。

另外，当你在黑莓手机上浏览 pinppl 时，你可以从任何用户的个人资料中直接给他们发送 PIN 消息——非常聪明。我心中的问题是:这个社区最容易接受这种社交网络吗？时间会证明一切。更多地集成到黑莓平台本身(例如，无缝登录)也会很有用。

总而言之，这是一个令人兴奋的社交网络领域的新成员，有着有趣的倾向。
