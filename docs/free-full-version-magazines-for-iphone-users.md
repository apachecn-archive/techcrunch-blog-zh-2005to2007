# 面向 iPhone 用户的免费完整版杂志

![texterity](img/9a471a6f296e4cd30bf0955c2c3cc00b.png)

Texterity.com 向 iPhone 用户提供 20 多种免费杂志，全部包装在专为该设备设计的界面中。每本杂志都是完整展示的，这项服务不需要任何特殊的软件或插件来观看。

当数字杂志界面处于测试阶段时，所有这些优点都是免费的。测试期结束后还没有定价的消息。
