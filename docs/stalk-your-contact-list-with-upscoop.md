# 使用 UpScoop 跟踪您的联系人列表

[![](img/1de15390032ad265c2e7b97075b89a7b.png)](https://web.archive.org/web/20211202031351/http://upscoop.com/) 今天，声誉网络 [Rapleaf](https://web.archive.org/web/20211202031351/http://www.beta.techcrunch.com/2006/05/17/ebay-bans-rapleaf-links/) 发布了一项名为 [Upscoop](https://web.archive.org/web/20211202031351/http://upscoop.com/) 的新服务，该服务联合了许多初创公司(例如，参见[profile link](https://web.archive.org/web/20211202031351/http://www.beta.techcrunch.com/2006/12/28/profilelinker-takes-meebo-approach-to-social-networking/)和 [Wink](https://web.archive.org/web/20211202031351/http://www.beta.techcrunch.com/2006/11/10/wink-now-searches-myspace-linkedin-and-beebo/) )试图在社交网络之上添加一个元层。有很多这样的网络，很多人属于不止一个。跟踪你自己和你朋友的关系网是很复杂的。

Upscoop 旨在根据您的朋友的电子邮件地址，帮助您找出他们属于哪个网络。您放弃您的 AOL、Gmail、Hotmail 或 Yahoo 电子邮件帐户的电子邮件凭据(包括密码)。Upscoop 获取你的联系人列表，然后在多个社交网络中搜索，并试图在他们在主要社交网络中索引的 1000 万个个人资料中找到你认识的人的个人资料页面。搜索过程不是瞬间完成的，实际上需要几个小时。

点击任何结果都会(有时)将您带到该人的个人资料页面。然后，您可以将他们添加为好友，或者与他们进行互动。

要求人们提供完整的电子邮件凭证来完成搜索将是一个艰难的推销。但这比在 Wink 上逐个搜索好友要容易得多。对于认真对待与朋友联系的人来说，Upscoop 可能适合他们。
