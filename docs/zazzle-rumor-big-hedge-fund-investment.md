# Zazzle 谣言:大型对冲基金投资

 [](https://web.archive.org/web/20220121081827/http://www.crunchbase.com/company/zazzle) 这里有一个未经证实但来源可靠的传言:总部位于硅谷的 [Zazzle](https://web.archive.org/web/20220121081827/http://www.crunchbase.com/company/zazzle) 将完成第二轮融资，金额在 3000 万美元左右，投资前估值超过 3 亿美元。预计这轮融资将由一家尚未命名的对冲基金牵头。凯鹏华盈(Kleiner Perkins，曾在 2005 年领导该公司 1600 万美元的首轮融资)据说也将参与这轮融资。在接下来的几周内，期待公司的确认。

该公司与 CafePress 和其他公司竞争。
