# Nike+iPod 与 AnyShoe 一起失控| TechCrunch

对于那些想使用 [Nike+iPod 运动套装](https://web.archive.org/web/20140310232813/http://crunchgear.com/2006/10/03/nikeipod-hands-on/)但不想穿耐克鞋的人来说，现在有一种产品可以对你的困境表示一些同情。SwitchEasy 的 running any shoe 是 Nike+iPod 发射器的小型外壳。只需将发射器放在外壳内，并将其夹在您的鞋子上，即可进行 Nike-free Nike+iPod 操作。该产品售价 7.99 美元，价格适中，请尽情享用吧。
