# 机器人在一起工作

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/17/the-robots-are-working-together/>

任何看过《蛋壳里的幽灵》的人都应该知道这个故事的含义。来自比利时布鲁塞尔自由大学、意大利认知科学与技术研究所、自主系统实验室和 Dalle Molle 人工智能研究所的一组科学家开发了一个系统，通过该系统，一组机器人可以合作实现一个共同的目标。现在，这些目标包括[逐渐将一个用红色发光二极管照亮的物体运送到一个大的白色目标](https://web.archive.org/web/20170608213708/http://www.swarm-bots.org/dllink.php?id=587&type=movies)和[一起工作，从一个红色物体和一个蓝色目标](https://web.archive.org/web/20170608213708/http://media.newscientist.com/data/images/ns/av/dn10315V1.mp4)绘制出一条路径。

在未来，他们希望让机器人互相帮助完成一个机器人需要完成的任务。例如，如果一个机器人来到一堵他需要通过的墙，他会回去找一些伙伴，他们会想出他们是否能一起爬上去。不过，目前的一批可以适应。他们可以自己改变方向，以确保他们朝着正确的方向前进。伙计，这太酷了。Tachikoma 很快就要实现了。
