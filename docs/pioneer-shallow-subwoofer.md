# 先锋浅低音炮

先锋的 TS-SW2541D 正在向以前很少有超重低音扬声器涉足的领域进军:浅空间。低音扬声器只需要 3 英寸的深度就能产生声音饱满的低音。

这意味着它不需要一个巨大的外壳就能发出正确的声音。这也意味着它可以位于潜艇以前无法到达的空间。

![](img/a596db329889dd448c187bfc4d40b82c.png)
所以，考虑到它 180 美元的价格标签，期待在不久的将来看到汽车完全装满这些东西。会发生的。
