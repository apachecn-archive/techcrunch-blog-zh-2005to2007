# 你把电子邮件和照片联系起来了吗

 [](https://web.archive.org/web/20210916104955/http://www.weekendr.com/whoareyou/) 德国初创公司 [Who-R-You](https://web.archive.org/web/20210916104955/http://www.weekendr.com/whoareyou/) 是一个死气沉沉的简单社交网络。它也可能是最直接的在线约会网站之一(尽管它并没有声明它的用途)。

去那个网站，告诉它你的电子邮件地址，并添加一张你自己的照片。确认你的电子邮件。完成了。

该网站的访问者可以浏览图片，或键入电子邮件地址，看看是否有图片与之相关联。选择允许联系人联系他们的用户可以接收来自其他注册用户的消息。

这不是一个技术展示，也没有复杂的商业想法。但是就像我们描述的某些其他网站一样，仅仅因为一项新的网络服务不一定在流动性事件轨迹上，并不意味着它不值得一看。

正如我在第一段中提到的，你是谁可能是最简单形式的网上约会，因为人们会仔细阅读图片并联系他们觉得有吸引力的用户。

请留意同一家公司即将推出的另一个网站，名为 [Weekendr](https://web.archive.org/web/20210916104955/http://weekendr.com/) 。
