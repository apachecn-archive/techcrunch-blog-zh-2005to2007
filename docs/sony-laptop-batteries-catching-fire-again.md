# 索尼笔记本电池再次着火

![dellexplode_02.jpg](img/4adfc1f3a72383c106297feb5d6fc9a1.png)

“我有祸了。”这一定是索尼最近的座右铭，因为他们一直试图扑灭一场又一场大火。这一次，烧起来的不是戴尔，而是东芝。五月下旬，一台东芝笔记本电脑着火，罪魁祸首还是索尼电池。经过调查后，发现电池短路导致裤裆着火，我说的也不是红发女郎。东芝的[型号列表](https://web.archive.org/web/20210421071750/http://www.csd.toshiba.com/cgi-bin/tais/su/su_sc_dtlView.jsp?soid=1501060&ignore_sn=true)很广泛，所以我会看一看并采取适当的措施。
