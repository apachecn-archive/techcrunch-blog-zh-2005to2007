# 新一代成人网站综述

[![](img/5b627b79a2aff9c32743ea6c834e2c07.png)](https://web.archive.org/web/20230218162051/http://www.allposters.com/-sp/Sex-Posters_i1248676_.htm) 网络色情通常是一个前沿产业，而且总是[为了钱](https://web.archive.org/web/20230218162051/https://techcrunch.com/2007/05/12/internet-pornography-stats/)。该行业通常是互联网上的第一批新功能，这些新功能最终会渗透到主流消费者网站。

然而，最新的网络创新浪潮完全集中在消费者网站上。色情业的人已经采用了许多这些功能来创建大规模成功的第二代成人内容网站。2006 年中期，我们第一次接触到色情管道。其他人，如 [EroShare](https://web.archive.org/web/20230218162051/https://techcrunch.com/2007/01/01/eroshare-user-generated-porn/) ，也紧随其后。在最近的一篇综述文章中，我们还提到了虚拟世界红灯区。

[Go2Web2](https://web.archive.org/web/20230218162051/http://go2web2.blogspot.com/2007/08/watch-and-share-porn-web-20-style.html) 刚刚发布了一些额外的综述。其中一些有着很棒的名字——比如 MegaRotic、Fantasti.cc、NippleByte 和 StumblePorn。

这些网站中的许多只不过是更传统的色情网站的潜在客户生成工具(如果你要访问它们，[首先要采取一些保护措施](https://web.archive.org/web/20230218162051/https://techcrunch.com/2007/07/10/site-advisor-20-haute-secure-launches-to-detect-and-block-malware/))。但是看到这个行业尝试任何新的东西来获得新的资金是很有趣的。大量的竞争和大量的金钱推动了这一切。
