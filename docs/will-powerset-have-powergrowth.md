# Powerset 会有 PowerGrowth 吗？

![](img/45c92294adbc9e7b6e0242e61d76377a.png)

任何严肃的互联网创业公司都试图猜测早期的增长率，以便他们有足够的硬件来扩展(希望软件也在那里)。隐形搜索引擎 [Powerset](https://web.archive.org/web/20210411131719/http://www.crunchbase.com/company/powerset) (见我们的[早期报道](https://web.archive.org/web/20210411131719/http://www.beta.techcrunch.com/tag/powerset))对他们的建模非常透明，并在他们的公司博客上发布了他们的一些想法。

史蒂夫·纽康(Steve Newcomb)，联合创始人和首席运营官，写了[帖子](https://web.archive.org/web/20210411131719/http://www.blognewcomb.com/blog/2007/04/how_is_powerset_predicting_gro.html)，并详细介绍了他们如何建立预测模型。上图显示了在新的 [Snap Preview Anywhere](https://web.archive.org/web/20210411131719/http://www.beta.techcrunch.com/2006/11/18/add-snap-previews-to-any-site-for-free/) 发布并重新推出其产品后，Newcomb 应用于竞争对手[Snap.com](https://web.archive.org/web/20210411131719/http://www.snap.com/)的模型。

阴影区域显示预测的增长，而实际结果由蓝线显示。不错的工作，尽管我认为他们对模型做了一些调整以使其适合。

Powerset 应该发布模型本身(当然没有具体的 Powerset 假设)，并让其他创业公司调整它以供自己使用。大多数新公司没有优秀的骑师，也没有时间做这种工作。任何竞争问题都会被他们从这样做中获得的可观的善意(和链接果汁)所掩盖。
