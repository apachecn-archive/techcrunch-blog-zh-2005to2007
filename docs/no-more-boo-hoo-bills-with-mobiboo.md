# 再也不用为 Mobiboo | TechCrunch 支付账单了

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/04/no-more-boo-hoo-bills-with-mobiboo/>

总部设在伦敦的 mobi boo(T2)今天宣布正式推出其移动 VoIP 服务，这是英国第一个被分配了英国移动号码范围 07911 的商业 VoIP 电话网络。

> “Skype 等公司已经证明了 VoIP 的成本节约。Mobiboo 更进一步，复制了传统移动、DECT 和固定电话服务的真正组合，为住宅和商业客户提供了可靠的替代方案，从而实现了真正的物有所值。一个坚果壳，一个手机号码，DECT 功能和座机价格，以及 WiFi 的额外覆盖范围。”Mobiboo 的商务总监理查德·纽森解释道。

这项服务将使用户能够通过 ut 斯达康的 F3000 手机打电话，只要他们在一个**开放的** WiFi 热点或无线路由器的范围内。Mobiboo 还与欧洲领先的热点提供商[云](https://web.archive.org/web/20140814120649/http://www.thecloud.net/)合作，在英国提供良好的热点覆盖。

无论是在国内还是在国外旅行，Mobiboo VoIP 都具有显著的成本节约意义，因为国际漫游费变得不存在，例如从热点地区拨打英国座机每分钟仅需 2 便士，而拨打其他 Mobiboo 电话仍然免费。

最近去了一趟法国，我发现 T-Mobile 要向我收取每兆 7.50 英镑的费用，让我使用 GPRS/UMTS 在 MDA 手机上浏览网站。没有漫游数据交易！？不要安静一天一次的“网上漫步”活动。

因此，为了不被敲竹杠，我决定用房间里的 wifi 连接查看电子邮件、上网冲浪，并使用我的 Skype 移动客户端打电话。结果没有通话费用或敲诈性的数据费用，也没有撕毁帐单；-)
