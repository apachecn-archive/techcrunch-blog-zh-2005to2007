# MyFabrik:文件共享和更多

![](img/cf749d9d6bc79d22c81611dacd495667.png)

MyFabrik，与 Maxtor 关系不太密切，是一个新的在线数据存储/共享/组织应用程序，让您只需一个网络浏览器就可以远程管理文件。您可以私下或公开上传和下载图片、音乐、视频和文档，具体取决于您设置的权限。这款网络应用的 Web 2.0 特性来自于标记、与他人分享，以及将你选择的文件快速发布到易贝和 MySpace 上的能力。

它的目标是成为媒体分享的一站式商店，攻击。Mac 和其他一些 2GB 的播放器每年 30 美元，可以与 Maxtor 的 Fusion 系列硬盘无缝交互。
