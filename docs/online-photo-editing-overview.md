# 在线照片编辑概述 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/04/online-photo-editing-overview/>

前几天推出的 [Picnik](https://web.archive.org/web/20230202235453/http://picnik.com/) a [给我们带来了又一个在线照片编辑工具。像](https://web.archive.org/web/20230202235453/http://go2web2.blogspot.com/2007/02/one-click-photo-fixing.html) [Fauxto](https://web.archive.org/web/20230202235453/http://www.fauxto.com/) 一样，Picnik 使用 Flash，而大多数早期的编辑工具都使用 Ajax 进行浏览器内编辑。

由于所有电脑都配有旋转、调整大小和裁剪照片的基本软件，所以使用在线服务需要有令人信服的理由。上传一张照片到这样的服务，编辑它，然后下载回你的硬盘，成本太高。为了弥补这一点，大多数服务都允许你将编辑过的照片直接传输到 Flickr、Webshots 或其他在线照片服务，省去了用户来回上传和下载的麻烦。

大多数这些在线服务还提供编辑工具，超越简单的旋转、调整大小和裁剪，并开始进入 Photoshop 的领域。以下是几个比较好的例子，以及我们最近的测试笔记:

**Fauxto**

 [](https://web.archive.org/web/20230202235453/http://www.fauxto.com/) [Fauxto](https://web.archive.org/web/20230202235453/http://www.fauxto.com/) 是一款基于 Flash 的 Photoshop look-like。这是我们所知的唯一基于图层的在线工具，也是目前最好的。但如果你只是在寻找照片编辑，而你又不熟悉 Photoshop，Fauxto 的复杂性会让你感到沮丧。如果你已经熟悉 Photoshop，你可能已经有一份拷贝了。Fauxto 看起来很可爱，这是一个很好的闪存应用示例，但我不确定他们的目标市场是谁。

**Picnik**

皮克尼克是这个街区的新成员，他们是目前为止最好的。它也是基于闪存的，速度最快，用户界面最直观。完成编辑后，您可以将照片直接传输到 Flickr。Picnik 已经取代了基于 Ajax 的 PXN8，成为我们最喜欢的在线照片编辑工具。

**图片 2 生活**

[](https://web.archive.org/web/20230202235453/http://www.picture2life.com/)[picture 2 life](https://web.archive.org/web/20230202235453/http://www.picture2life.com/)是一个基于 Ajax 的图片编辑器。它专注于抓取和编辑已经在线的图像。工具选择一般，用户界面差。网站上有一些 bug。照片编辑后可以传输到 Flickr、23 和 Imageshack。

**预加载器**

 [](https://web.archive.org/web/20230202235453/http://www.preloadr.com/) [Preloadr](https://web.archive.org/web/20230202235453/http://techcrunch.com/2006/07/18/new-stuff-for-flickr-freaks/) 是一个使用 Flickr API 的 Flickr 专用工具，甚至用于帐户登录。该服务包括基本的裁剪、锐化、色彩校正和其他工具来增强图像。Preloadr 是专门为 Flickr 设计的，这一事实可能并不是一个优势——其他一些服务一样好，甚至更好，也提供了 Flickr 集成。

**PXN8**

 [](https://web.archive.org/web/20230202235453/http://www.pxn8.com/) [PXN8](https://web.archive.org/web/20230202235453/http://www.pxn8.com/) 是最好的基于 Ajax 的编辑器(在 Picnik 发布之前也是最好的),它有一个很棒的用户界面，主要特性都在大图标上突出显示。基本的“增强”功能很好地解决了图片的明显问题。编辑过的照片可以传输到 Flickr 或 [Webshots 的 AllYouCanUpload](https://web.archive.org/web/20230202235453/http://techcrunch.com/2006/05/29/cnets-allyoucanupload-is-disruptive/) 服务。

**狙击**

[](https://web.archive.org/web/20230202235453/http://www.snipshot.com/)[Snipshot](https://web.archive.org/web/20230202235453/http://www.snipshot.com/)，以前叫做 Pixoh，是另一个非常好的基于 Ajax 的编辑工具，它脱颖而出是因为它的设计高于平均水平，而且他们的服务中有一个 API。我们更喜欢 PXN8 的功能和 UI，但只是勉强。
