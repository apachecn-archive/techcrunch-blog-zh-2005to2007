视频

这看起来像 6G iPod 的用户界面吗？它看起来足够真实不是吗？苹果确实在 MacRumors 上放出了一群狂热的律师，他们删除了原始视频，所以这让人觉得这是真的。视频非常清楚地表明，用户界面是基于点击轮的，这将是一个巨大的失望，苹果粉丝肯定会反抗。有人真的关心“世界时钟”现在向你展示一个以世界地图为背景的真正的时钟吗？我不这么认为。

CrunchGear 从[可靠消息来源](https://web.archive.org/web/20160422033120/http://crunchgear.com/2007/07/17/crunchrumor-6g-ipod-to-join-iphone-apple-tv-in-featuring-youtube-content/)获悉，下一代 iPod 将拥有宽屏触摸屏。普遍的共识似乎是，6G iPod 将在 iPhone 减去手机之后问世。也许，仅仅是也许，苹果在他们的集体头脑中有一个宽屏触摸屏 iPod 将会消灭 iPhone 的销售，但那只是疯狂的谈话。这个视频看起来像是一个假的和/或旧版本的用户界面。
