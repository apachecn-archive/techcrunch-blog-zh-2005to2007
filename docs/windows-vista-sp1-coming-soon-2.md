# Windows Vista SP1 即将推出(er)

![msdownloadsvistasp1.gif](img/834a91294537da1ff2c7f205c5ec5505.png)

显然 Windows 无法让它的大嘴巴闭上。经过一番挖掘，一些网站已经发现在微软下载中提到了 [Windows Vista SP1](https://web.archive.org/web/20201127000619/http://crunchgear.com/2007/05/21/windows-vista-sp1-coming-soon/) ，这是微软为更新和其他大文件下载提供的服务。

这是什么意思？本质上看起来 Service Pack 1 存在于微软自己的封闭工作空间中，一些技术作者没有意识到它还没有普及。我怀疑这将很快消失，但它确实意味着有某种 SP1 候选人在野外的某个地方，这意味着它可能在今年夏天准备好。好侦探，关心这东西的人！
