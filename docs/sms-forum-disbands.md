# 短信论坛解散

本周，致力于开发和推广 SMS 以造福无线行业的非营利组织 SMS 论坛宣布将解散。该组织将在今年年底关闭其网站。最初成立时是一个短消息对等论坛，后来这个组织改成了 SMS 论坛，并从 2001 年开始发展壮大。自成立以来，SMS 论坛为运营商、开发商和用户提供了一个分享和交流想法以及为整个行业提供解决方案的平台。SMS 论坛基本上解决了许多问题，小组认为没有必要进一步推广该标准。
