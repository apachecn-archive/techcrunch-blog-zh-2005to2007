# 任天堂的新 Zapper: Hotness 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/11/nintendos-new-zapper-hotness/>

任天堂今天在 E3 为广受欢迎的 Wii 推出了几个新的硬件配件，包括马里奥手推车的方向盘和一个新的 Zapper。对于那些可能太年轻而记不起的人来说，最初的 NES 行动系统与最初的 Zapper 一起发货，你用它来打鸭子，仅此而已。但是猎鸭非常有趣，Zapper 成为了标志性的东西。

我们希望现在你的虚拟枪可以有动作感应，过去十年中许多流行的街机射击游戏可以击中 Wii。带动作感应的死亡之屋？给我们报名。我们真正希望的是一个新的，2007 年版本的《猎鸭记》,在其中我们终于可以射杀那只该死的狗，它因为我们的失踪而窃笑。我讨厌那只狗。
