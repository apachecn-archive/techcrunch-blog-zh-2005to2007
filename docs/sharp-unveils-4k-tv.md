# 夏普发布 4k 电视

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/06/sharp-unveils-4k-tv/>

2010 年，你将会看到 4k 视频。即以每秒 60 帧的速度拍摄 4096×2160 像素的视频。或者类似的东西。这将 HD 提升了 4 倍。未来是你的电视，带给你的公司会是夏普。

没错，就是那款以贫血的“组织者”而闻名的夏普，1999 年圣诞节，当你要一台 Palm Pilot 时，你的祖母送了你一台。夏普最近向世界展示了其新的 64 英寸 ASV“超级高清电视”。以今天的标准来看，4:2 的长宽比有点奇怪，但几年前的 16:9 也是如此。但是看看这些数字:那是 884 万像素。那可能比你的数码相机还多，能人。

短期内不要寻找这种超高 rez 屏幕，因为它是证明概念的原型。但事实是，夏普(夏普！？)加入 4k 游戏意味着 4k 可能有未来，而且它可能会很快到来。你将能够在你的客厅里观看用神秘的[红色一号摄像机](https://web.archive.org/web/20201123231559/http://crunchgear.com/2006/09/25/red-theivery-update-prototypes-gone-mysterium-sensors-ok/)拍摄的镜头，你的孩子会认为它一直都是这样的。我们讨厌你的孩子。
