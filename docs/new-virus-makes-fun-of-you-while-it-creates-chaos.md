# 新病毒在制造混乱的同时取笑你

![](img/8fa2ef6ecf854f6546159ae083d67bf7.png)

特洛伊病毒已经存在很多年了，但是 Outlook 的傻瓜仍然认为他们可以通过下载附件看到杰西卡·辛普森的裸体。最新的特洛伊木马之一 BotVoice-A 现在将病毒的概念又向前推进了一步。它不仅会删除你的文件，还会通过你的扬声器嘲笑你，同时破坏你的电脑。

当您的文件消失时，病毒使用 Windows 文本阅读器对您说出以下句子:

> 你被感染了，我重复一遍，你被感染了，你的系统文件被删除了。抱歉。祝你愉快，再见。

就在你认为事情不会变得更糟的时候，它变得更糟了。僵尸之声-A 禁用 Windows 注册表编辑，所以你不能修复它造成的任何损害。据说这种病毒是以假冒 MP3 或视频文件的形式通过 P2P 传播的，所以在盗取这些 CSNY MP3 时要小心。
