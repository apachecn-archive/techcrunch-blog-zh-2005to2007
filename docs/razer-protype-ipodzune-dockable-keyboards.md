# Razer Pro |型 iPod/Zune 可对接键盘

这款 Razer 键盘获得了第一个宣布的 Zune 兼容键盘的荣誉。这款 Pro |不仅支持 Apple iPod，还配有媒体中心键、可编程键、32KB 板载内存、两个 USB 端口和一个音频外设线路。它甚至有镀金的 USB 连接器，怪物电缆公司多年来一直用它来证明其抬高的价格。

键盘应该会在今年 11 月左右发布，大约是微软 Zune 发布的时间。如果键盘有一个 iPod dock 和一个 Zune dock，而不是一个键盘一个，这可能是一个很好的购买。
