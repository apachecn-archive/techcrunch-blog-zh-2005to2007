# 看见红色:超高清

它被指责为蒸汽产品，但红色的现在已经正式亮相，并且可以预订。

对于那些不知道的人来说，红色的是红色的第一台相机，红色是由 Oakley 的创始人詹姆斯·詹纳德创立的数字视频公司。让红色如此神奇的是，作为一名光学爱好者，Jannard 设计了一款质量可与 35MM 胶片媲美的数字电影摄像机，它完全模块化，因此可以随心所欲地定制，并且成本远低于质量较低的摄像机。他的目标是让数字电影产业焕然一新，他可能已经做到了。

这是第一款低于 10 万美元的相机，可以每秒 60 帧的速度拍摄 4k 视频(4520 x 2540 像素)。也就是 1100 万像素。有消息说詹姆斯·卡梅隆已经订购了三辆。你现在可以在 red.com 花 1000 美元预订你的。我喜欢这个东西，它有很好的规格，看起来就像新变形金刚电影中的威震天。
