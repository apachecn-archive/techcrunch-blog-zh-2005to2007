# O2 抛弃 I-mode

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/18/o2-to-ditch-i-mode/>

![O2.jpg](img/243066dbfb867e82c18ce5f047d861b8.png)据《福布斯》报道，英国运营商 O2 将放弃其 i-mode 服务，因为在过去两年中，该服务仅注册了 260，000 名互联网电话用户。此外，移动运营商将从 7 月底开始不再销售任何新的 i-mode 手机，但尽管如此，在未来两年内仍将继续支持这 25 万用户。福布斯补充说，这项技术在日本非常流行，但在欧洲却没有引起同等程度的兴趣。
