简介——星球网2.0——TechCrunch

原文：https://web.archive.org/web/http://www.techcrunch.com:80/2005/07/11/profile-planet-web-20/

服务: 星球网2.0

推出:大约两周前

什么事？

Planet Web 2.0是一个网站(更重要的是一个RSS feed ),它从Web 2.0发行商那里收集内容。对于任何跟踪web 2.0的人来说，这是一个必须添加的提要。内容的来源链接在侧边栏上，包括TechCrunch。:-)

上周，我们在阅读理查德·麦克马努斯关于机器人和内容创建/聚合的精彩且备受讨论的帖子时，首次发现了该网站。

我们对Richard的帖子有很多想法(总的来说，完全同意他的观点)。我们的感觉是Planet Web 2.0，它不遗余力地赋予内容属性，正在以正确的方式进行这一不可避免的演变，我们支持他们。

关于重新发布内容的争论可以追溯到网络本身。当然，RSS让这变得更加容易。事实上，RSS的全部目的就是重新发布内容。争论的焦点是它是否应该只在RSS阅读器中重新发布，或者它是否可以在任何地方重新发布。然而在现实中，在线RSS阅读器和网站有什么区别呢？不多，特别是当你想到博客和其他读者是如何分享信息的…

编者按:要追踪关于RSS和版权的“老”争论，请看杰森·卡尔卡尼斯、罗伯特·斯考伯、 Makeyougohmm 开始。

所以TechCrunch的立场是:我们喜欢Planet Web 2.0，认为它所做的是一种负责任的方式来推广内容，同时给那些写它的人适当的归属。

在形成我们的立场时，我们注意到Ian Davis(Planet Web 2.0的创建者)对Richard帖子的评论:

“我认为你发现了内容再混合的一些本质特征，这些特征以前可能没有被意识到。聚合和重新混合内容需要尊重。我们应该尊重原作者的意图，以及他们在创作内容上投入的时间和精力。改变用途、假冒、不归属——这些都是对待内容和人的不尊重方式。希望我在Planet Web 2.0(以及我主持的其他类似网站)上尊重作者。也许会有更多的流量流向他们，或者会帮助人们发现他们已经认识的相关作家。不管怎样，我欣赏写作中的工作，我只希望我能写得有我所读的一半好。”
链接

只要这是他的立场，他的行动反映了他的立场，我们就始终支持他。

与此相关的是， Nivi (另一个必读博客)最近创建了两个聚合feeds风险投资频道和企业家频道T10。他做的归因比星球网2.0少，但很明显他的动机是纯粹的。我们两者都推荐。

更新:我通过电子邮件与Ian Davis进行了交谈，他在自己的网站上提供了以下见解:

“我创办PW的动机是收集这个领域中我尊敬的人的作品。我想了解Web 2.0
的前景，并跟上新的发展和应用。我已经有了一个小的私人订阅列表，但我决定发布并从中汇总信息，而不是把它藏在心里。我推断，如果我对此感兴趣，那么肯定会有其他人同样感兴趣。希望人们会通过他们已经读过的人发现它，并发现其他伟大的作家。这个网站只活跃了几周——我想我是在7月1日注册的。

我对开放数据访问、协作和语义网有着长期的兴趣。早在2000年，我就参与了RSS 1.0
规范的编写，当时我认为这是开放数据网络的第一步
，RSS暗示允许重用。
现在，有了Web 2.0，我想我有了一个令人兴奋的机会将
同样的原则应用到更广泛的内容和服务中。事实上，
我刚刚在一家名为Talis的英国公司开始了一个新的角色，领导蚕项目的开发，这是一个支持Web 2.0
应用的平台。

我玩得很开心。感觉像是第一次发现了
的马赛克:)”
