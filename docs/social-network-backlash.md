# 社交网络反弹(幽默)

我想很多人最近肯定收到了太多的链接请求，因为今晚我遇到了两个“反社交网络”(尽管其中一个只是个笑话):Isolatr 和 Snubster。

## 隔离器

 [](https://web.archive.org/web/20221209124241/http://www.isolatr.com/) 孤立者不是真正的服务，但应该是。Metroblogging 的 Sean Bonner 创建了这个网站，承诺“帮助你找到别人没有的地方”。加分内容，查看[常见问题](https://web.archive.org/web/20221209124241/http://isolatr.com/faq.html)(每个答案都是“否”)。我喜欢对细节的关注，比如称它为 beta 服务，去掉名字中的“e ”,并添加来自 Doc Searls 和 Xeni Jardin 的虚假引用。

克里斯·皮里洛(Chris Pirillo)多年来一直在寻求这样的东西(但却是真实的)。[山谷谷](https://web.archive.org/web/20221209124241/http://www.valleywag.com/tech/social-network/isolatr-the-antisocial-network-160190.php)有更多。

## Snubster

Bryant Choung 的 Snubster 是一个真实的直播服务。这有助于你通知人们和全世界，你对某人很生气。“通知”功能告诉他们，他们已经犯了“违规”，并处于暂停期。如果他们真的搞砸了，你可以声明他们“对我来说已经死了”。每个用户都有一个个人资料([示例](https://web.archive.org/web/20221209124241/http://www.snubster.com/viewList.php?passListID=102))，他们可以在其中获得有趣的细节。我唯一的问题是，为什么 snubster 名字里的“r”前面有个“e”？
