# TiVo 桌面 2.4 公开测试版下降

TiVo 已经发布了其桌面软件的 2.4 beta 版本。新版本允许用户将 PC 上的非 DRMed 视频进行转码，以便在 TiVo 上播放。

除了代码转换功能，这项新服务还可以将高清照片从 PC 播放到 TiVO。TiVo Desktop Plus 升级服务的费用是 25 美元。抱歉，还没有 Mac 版本。
