# 简介–ice rocket

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/07/29/profile-icerocket/>

**公司:** [冰箭](https://web.archive.org/web/20210920195732/http://www.icerocket.com/)(即将更名为[布洛格斯库尔](https://web.archive.org/web/20210920195732/http://www.blogscour.com/))

![](img/e5dd58d88eaeebcf86a5db1109278e9b.png)

**地点:**德克萨斯州达拉斯

**什么事？**

IceRocket 是一个博客搜索引擎，非常好的一个，归马克·库班所有。在我们看来，他们还没有准备好迎接博客搜索之王的挑战，但是他们功能丰富，在某些方面做得非常非常好。

![](img/d30efdff3cb18efc33063d6f490994d8.png)

现在人们比较博客搜索引擎有两种方式:给定关键词或标签的总链接数，以及给定博客显示的总链接数。在两次测试中，IceRocket 似乎都表现不错(但[并不总是](https://web.archive.org/web/20210920195732/http://radio.weblogs.com/0001011/2005/07/17.html#a10670))。在 TechCrunch 的链接上，IceRocket 似乎是最新的，本周(实时网络中的事情变化很快)。

 IceRocket 有很多很棒的搜索和其他工具，包括通过[关键词](https://web.archive.org/web/20210920195732/http://blogs.icerocket.com/search?q=techcrunch)、[标签](https://web.archive.org/web/20210920195732/http://blogs.icerocket.com/search?q=tag%3Atechcrunch)(见他们的标签云[这里](https://web.archive.org/web/20210920195732/http://blogs.icerocket.com/tag/))和[网址](https://web.archive.org/web/20210920195732/http://blogs.icerocket.com/search?q=http%3A%2F%2Fwww.beta.techcrunch.com)进行搜索。

他们也有一个[趋势](https://web.archive.org/web/20210920195732/http://trend.icerocket.com/)服务(见这里的预定义趋势竞赛[，见鬼，他们甚至有](https://web.archive.org/web/20210920195732/http://www.icerocket.com/c?p=morebuzz)[电子邮件](https://web.archive.org/web/20210920195732/http://mail.icerocket.com/)(当我还是伦敦 GNR 的首席运营官时，我把我们的一家公司，NamePlanet，卖给了 [NetIdentity](https://web.archive.org/web/20210920195732/http://www.netidentity.com/) ，另一家马克·库班公司负责这项业务)。最后，它们还有一个[工具栏](https://web.archive.org/web/20210920195732/http://www.icerocket.com/c?p=toolbar)，如果你不像我一样，并且仍然有足够的浏览器空间来实际查看网页，那么这个工具栏是非常棒的。

基本上 IceRocket 很牛逼。他们真正大放异彩的一个领域是搜索结果。结果包括大量有用的链接。除了到源的链接之外，还有到结果标签的链接、博客本身、改进搜索以包括或排除该博客的工具、到博客的链接数量(针对相关性和 ego)，它们甚至链接到博客的 RSS 提要。所有搜索结果页面都有 RSS 提要。

![](img/90e1b08e6edf5c79dc91b862196c5f29.png)

另一个有趣的工具是 [IceSpy](https://web.archive.org/web/20210920195732/http://www.icerocket.com/icespy) ，它显示了一个滚动列表，列出了所有的搜索条目。这让我想起了谷歌办公室里显示搜索结果的平板屏幕(尽管谷歌似乎过滤掉了成人词汇，而 IceRocket 却没有:-)。

![](img/078195ea7c815fd0d8c19c12db12d8cd.png)

在不久的将来，他们会将博客搜索引擎的名字改为 BlogScour。

我们也看到了一个[增长的趋势](https://web.archive.org/web/20210920195732/http://www.technoogle.com/?p=28)，除了 Technorati 标签之外，博客们还用 IceRocket 标签来标记他们的帖子。
