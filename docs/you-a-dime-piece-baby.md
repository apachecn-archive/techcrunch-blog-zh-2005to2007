# 你是一个一毛钱的宝贝

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/19/you-a-dime-piece-baby/>

为了纪念罗技 25 周年和日本娱乐杂志《DIME》20 周年，罗技刚刚发布了特别限量版的 [VX-RD](https://web.archive.org/web/20210305110721/http://www.logitech.com/index.cfm/news/US/EN,contentid=12663,crid=34) ，它采用了非常酷的蓝色。
特别版“一角硬币”售价约为 90 美元(含增值税)。
