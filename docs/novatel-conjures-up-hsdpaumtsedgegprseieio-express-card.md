# Novatel 联想起 HSDPA/UMTS/EDGE/GPRS/EIEIO 快递卡

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/11/novatel-conjures-up-hsdpaumtsedgegprseieio-express-card/>

假设你有一家提供高速数据的手机提供商，比如 Cingular。然后，假设你有一台笔记本电脑，带有这些光滑的新 Express 卡插槽，就像 MacBook Pro 一样。Cingular 出售的传统 PCMCIA 卡不兼容。怎么办？

再等一会儿，就这样。但不会太久了。Novatel 向我们介绍了他们的 Merlin XU870。它是 HSDPA 卡、UMTS 卡、EDGE 卡和 GPRS 卡，所有这些都在一个 Express 卡包中。将它插入 34 或 54 号插槽(ew ),就万事俱备了。

它目前的上限是 3.6Mbps，但通过软件升级(和兼容的网络)，你可以获得 7.2Mbps 的移动速度。现在那是热的。

运送到世界各地，很快就会到达美国。
