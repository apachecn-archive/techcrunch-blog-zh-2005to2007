# Hyperwords 插件

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/12/26/the-hyperwords-plugin/>

我一般不写火狐插件，但是 [Hyperwords](https://web.archive.org/web/20220617201122/http://www.hyperwords.net/) 引起了我的注意。就像 [Om](https://web.archive.org/web/20220617201122/http://gigaom.com/2005/12/21/hyperwords-a-must-have-firefox-plugin/) 说的，它是辅助浏览体验的必备产品。我在这里也看到了一个商业模式。

每当选择一段文本时，Hyperwords 都会给用户提供许多选择。选项只是弹出，右键功能保持不变。

选项包括搜索各种引擎，在维基百科和 dictionary.com 中查找文本，通过电子邮件发送文本，在谷歌地图上搜索，翻译，以及在亚马逊和其他商业网站上搜索。搜索和商业流量将会产生收入(我的猜测是每个活跃用户每年一美元左右)。翻译服务非常有用。

如果你想测试 Hyperwords，请发邮件给 beta@gethyperwords.com。
