# 三星 Q1P UMPC 获得 Vista 认证

![](img/875c8acbc7c7466267d178ba0f8f2b47.png)

三星的 Q1P ultra mobile PC (UMPC)是第一款获得 Windows Vista 认证的设备。现在它已经通过了 Vista 认证，可以坐在学校酷孩子的午餐桌上了。超便携电脑还没有完全点燃世界(看看它的价格——1300 美元？不用了谢谢)，但是很高兴看到越来越多的设备支持微软的 OS X 版本。

Q1P 的统计数据相当简单，60GB 硬盘和 1GB 内存。不过，7 英寸的触摸屏非常宽敞，你可以用它来涂鸦打发时间，然后将手写笔记转换成可搜索的文本。有趣的是，三星希望 Q1P 在企业中大受欢迎，如果有一个群体比企业 it 部门更反对使用 Vista，我还没有见过。
