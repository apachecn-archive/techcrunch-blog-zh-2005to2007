# Podtech 和 SolutionWatch 加入 Web 2.0 工作组

如果你还没有看过[web 2.0 工作组](https://web.archive.org/web/20220124172526/http://www.web20workgroup.com/)，那就去看看吧。我原来的[帖子在这里](https://web.archive.org/web/20220124172526/http://www.beta.techcrunch.com/2005/10/10/web20-workgroup/)。

我们增加了两项新资源—[Podtech.net](https://web.archive.org/web/20220124172526/http://www.podtech.net/?p=183)和[解决方案观察](https://web.archive.org/web/20220124172526/http://www.solutionwatch.com/259/solution-watch-joins-web-20-workgroup/)。

Podtech.net 是一个非常受欢迎的播客网站，由 John Furrier 主持，它正在重新定义企业新闻实践。例如，最近，雅虎给了 Podtech 一个独家预告权，在上周末讨论他们的播客服务。

由 Brian Benzinger 撰写的 SolutionWatch 关注定义 web 2.0 格局的新公司和新产品。就像 TechCrunch 一样，Brian 正在帮助人们思考这些产品如何融入整个网络生态系统。

点击查看 Podtech 关于该小组[的播客讨论。](https://web.archive.org/web/20220124172526/http://www.podtech.net/?p=183)

Podtech.net 和 SolutionWatch 加入了[戴夫·维纳](https://web.archive.org/web/20220124172526/http://www.thetwowayweb.com/)、TechCrunch、[读写网](https://web.archive.org/web/20220124172526/http://www.readwriteweb.com/)和[webreaktuff](https://web.archive.org/web/20220124172526/http://www.webreakstuff.com/)的行列，成为我们已经提供的早期资源。更多的即将到来。
