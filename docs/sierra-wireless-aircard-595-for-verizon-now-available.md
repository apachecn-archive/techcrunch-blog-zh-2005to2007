# 面向威瑞森的 Sierra Wireless AirCard 595 现已上市

![](img/2ccd2bfc35665f45a9512e7d02077e96.png)

如果你正在寻找一些甜蜜的速度在你的笔记本电脑上漫游互联网，并且你是威瑞森的客户，一定要看看 Sierra Wireless 的 AirCard 595。

威瑞森商店售价 99 美元(两年协议售价 50 美元)，这个坏男孩是威瑞森顾客可以买到的第一批 EV-DO Rev. A 卡之一。该卡还可以在威瑞森无线的 CDMA 和标准 EV-DO 网络上使用，所以无论你用这张卡覆盖什么。明年 1 月，你可以在网上或商店买到。
