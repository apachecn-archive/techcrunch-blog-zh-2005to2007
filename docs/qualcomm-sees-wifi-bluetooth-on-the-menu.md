# 高通在菜单上看到了 WiFi 和蓝牙

为我们生产许多手机的高通公司刚刚收购了两家外部公司:RF Micro Devices 和 Airgo。虽然一家芯片公司收购其他公司通常不是新闻，但这一次引起了一些人的关注，因为 RF Micro 以其蓝牙芯片组而闻名，Airgo 以其 WiFi 解决方案而闻名。通过将这两家公司的知识产权和技术结合起来，高通可以为手持设备制造提供 WiFi 和蓝牙功能的组合芯片组，价格比其他任何公司都要便宜。

展望未来，搭载 802.11n 的手机不会比我们现在使用的手机更大，而且价格会低得多。
