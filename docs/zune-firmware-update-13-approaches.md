# Zune 固件更新 1.3 临近| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/01/zune-firmware-update-13-approaches/>

Zune Insider 的 Cesar 昨晚宣布，新的 Zune 固件将于本月晚些时候发布。更新 1.3 将修复一些问题，但不会真正增加任何功能。

无论如何，这是你可以从新固件中期待的:

*   许多用户从 Zune Marketplace 购买内容时遇到的跳过问题将得到解决。*   在设备检测和同步方面，提高了设备和软件的可靠性。*   The FM tuner will no longer drain the battery when in sleep mode.

    新版本的发布日期尚未确定，但据说将在 3 月 20 日发布。
