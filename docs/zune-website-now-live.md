# Zune 网站现已上线

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/11/02/zune-website-now-live/>

 [](https://web.archive.org/web/20210307140217/http://www.zune.net/) 微软新推出的捆绑个人媒体播放器/服务 [Zune](https://web.archive.org/web/20210307140217/http://www.beta.techcrunch.com/2006/07/21/microsoft-zune-to-battle-itunesipod/) 将于 11 月 14 日星期二上市。Zune 官方网站[今天早上](https://web.archive.org/web/20210307140217/http://crunchgear.com/2006/11/02/zunes-a-comin-welcome-to-zunenet-the-most-zune-riffic-site-around/)在[zune.net](https://web.archive.org/web/20210307140217/http://www.zune.net/)上线。

这款设备的零售价为 249 美元。完整规格如下，但 Zune 有一个 320×240 像素的 3 英寸屏幕(比当前 iPod Video 的 2.5 英寸屏幕稍大)，4/14 小时的视频/音乐电池寿命，以及 802.11 b/g 连接。

在未来十天左右的时间里，我们将会在 TechCrunch 以及我们的小工具博客 [CrunchGear](https://web.archive.org/web/20210307140217/http://www.crunchgear.com/) 上对 Zune 设备和服务进行大量报道。我们之前的 Zune 报道是[这里](https://web.archive.org/web/20210307140217/http://ww.beta.techcrunch.com/tag/zune)。
