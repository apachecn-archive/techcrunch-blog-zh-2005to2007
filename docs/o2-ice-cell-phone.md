# O2 Ice 手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/16/o2-ice-cell-phone/>

老实说，我们更喜欢糖果棒形状的手机，而不是蛤壳或滑盖。也许这是 80 年代和 90 年代使用无绳电话的结果。不管怎样，这款来自欧洲运营商 O2 的“冰”手机在我们看来非常性感。

这款 O2 手机可能从 [Netgear](https://web.archive.org/web/20150418204306/http://tools.netgear.com/skype/) 和 [Belkin](https://web.archive.org/web/20150418204306/http://www.belkin.com/skype/howitworks/) Skype 手机中获得设计灵感，拥有 130 万像素摄像头、蓝牙和 MP3 支持。它将于 9 月在英国上映，但就像西方文明一样，美国需要等待轮到它。
