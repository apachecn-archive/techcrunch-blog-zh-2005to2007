# USBCell 只是一个很棒的想法…

时不时会有一个非常棒的想法出现，让你想到“我怎么没想到呢”。这种类型的最后一个创新可能是[发条收音机。](https://web.archive.org/web/20201127145632/http://windupradio.com/)

[](https://web.archive.org/web/20201127145632/http://www.usbcell.com/)[USBCELL](https://web.archive.org/web/20201127145632/http://www.usbcell.com/) 是一种 AA 电池，只需插入任何 USB 端口即可充电。你不再需要扔掉用过的电池，或者拖着充电器或可充电电池的适配器。

![](img/0e662e7e0708ac64f353a9881c609b2e.png)

每年有超过 150 亿个电池被不必要地扔掉，随着现代电子设备对便携式电源的需求不断增加，不起眼的电池根本无法满足现代生活的需求。

USBCell 是西蒙·丹尼尔和克里斯·赖特创立的莫伊克萨能源(英国)有限公司的产品。

> “我们的使命是发明和提供技术，为消费者的移动或家庭用电需求提供更好的解决方案，这些解决方案更实用、更经济、更环保。”

[继 USBCell 在](https://web.archive.org/web/20201127145632/http://mobilecrunch.com/2006/09/26/moixa-energy-dump-your-ac-adaptors-and-save-the-planet-in-the-process/) [DEMO](https://web.archive.org/web/20201127145632/http://demo.com/) 上的成功展示之后，mobile runch对其进行了更长时间的回顾，同样有趣的是看到了来自夏沫的其他[未来产品](https://web.archive.org/web/20201127145632/http://www.usbcell.com/products)。
