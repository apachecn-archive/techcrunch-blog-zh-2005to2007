# 为什么 Xbox Live 要 50 美元

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/26/why-xbox-live-costs-50/>

[![xblwhypay.jpg](img/3a7dc27d915f016c65117c98fee54f40.png)](https://web.archive.org/web/20201024180051/https://beta.techcrunch.com/wp-content/uploads/2007/06/xblwhypay.jpg "xblwhypay.jpg")

我和许多像我一样的人每年支付 50 美元，享受在玩 Xbox Live 时听到 12 岁孩子诅咒和希望我的家人倒霉的特权。但是当我可以在个人电脑上免费做同样的事情时，为什么我要为此支付 50 美元呢？Xbox 社区团队的一名成员说:

> …它又回到了这个统一的系统。我们必须有语音通信，文本通信，现在我们正在做跨平台。有市场，有演示，所有这些东西都不会神秘地盘旋在你周围，你可以把它们从以太中取出来

虽然一位公关代表后来补充说，人们认为每月支付 100 美元的有线电视费用没有什么奇怪的，那么每年支付 50 美元有什么大惊小怪的呢？他说得有道理:每年 50 美元*对我来说差不多就是 0 美元，但真正让我不安的是支付的想法。叫我脾气暴躁的老头，我不在乎。*
