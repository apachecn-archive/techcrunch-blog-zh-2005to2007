# Slingbox 将于今年登陆 Symbian

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/17/slingbox-to-hit-symbian-this-year/>

诺基亚和索尼爱立信的粉丝们一直叫嚣着要支持 Symbian Slingbox，现在他们痛苦的呼喊被听到了。据 Zatz 报道，用于 S60 和 UIQ 手机的 Slingbox 客户端将于今年登陆欧洲和亚洲，明年登陆美国。

> Sling Media 写道:新的 SlingPlayer Mobile for Symbian OS 软件包将使用户能够将他们在 S60 和 UIQ 上支持的 Symbian 智能手机转换为个人的移动数字电视，随时随地提供对他们客厅电视体验的访问。
> 
> 迄今为止，超过 8200 万部 Symbian 智能手机已在全球范围内销售给 250 多家主要网络运营商。根据 Gartner 的数据，2006 年第二季度，Symbian 占全球智能手机出货量的 71%。
