# Vyew 协作套件现已上市

 [](https://web.archive.org/web/20220810115052/http://vyew.com/) [Vyew](https://web.archive.org/web/20220810115052/http://vyew.com/) 是一个新的协作工具套件，让用户共享一个公共的在线可视工作区和他们的桌面。令人印象深刻。这项服务今天结束了测试阶段，现在除了已经提供的免费账户之外，还提供高级账户。

Vyew 有点像 WebEx 和其他高端会议产品。它也有点像[thinknature](https://web.archive.org/web/20220810115052/https://beta.techcrunch.com/tag/thinkature)和其他简单的协作工具。这绝对是一家值得关注的公司，尽管我自己在测试时并没有轻松地使用它。

免费帐户允许创建两个包含 10 个上传项目的工作区。每月支付 10 美元或 20 美元，你就可以大幅提高这些限额。

Vyew 不需要用户下载桌面客户端。这是新用户快速轻松采用的关键。使用 Java 小程序在 Flash 和桌面中共享工作区。有文本聊天，只需点击一下，Vyew 就会为参与者生成一个唯一的登录信息，用于免费的电话会议。

这里有一个制作精良的深度截屏，声音非常认真。

工作区工具是标准的尖端。指向箭头是单向的，这是一件小事，但我希望文本在项目的右边，指向左边的箭头连接两者。不过这并不容易做到，几乎没有人会费心把它添加到这样的服务中。有很多 alpha 插件，比如下面截图中的雅虎地图。

性能如何？管理功能做得非常好——所有可用的选项都是一种享受。在 Safari 中，当桌面共享打开时，我在工作区遇到了一些问题。我发现桌面共享很容易开始，但没有任何实践就有点难以使用。

我建议试用 Vyew，看看你是否能使用它。如果可以，价格和功能都很棒。
