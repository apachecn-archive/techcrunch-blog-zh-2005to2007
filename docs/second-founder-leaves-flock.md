# 第二个创始人离开羊群

 [](https://web.archive.org/web/20211205084037/http://www.flock.com/) 有传言称，总部位于硅谷的 [Flock](https://web.archive.org/web/20211205084037/http://www.flock.com/) 的联合创始人兼首席战略官杰弗里·阿隆将离开该公司，成为 [Bessemer Venture Partners](https://web.archive.org/web/20211205084037/http://www.bvp.com/) (最初的 Flock 投资者之一)的常驻企业家。另一位联合创始人、前首席执行官巴特·德塞姆(Bart Decrem)去年年底离开了 Flock。

弗洛克最近用雅虎元老肖恩·哈丁取代了德雷姆。目前还不知道谁会接替阿伦。也没有消息表明 Arone 将在 Bessemer 从事什么新的创业项目。

尽快在官方 [Flock 博客](https://web.archive.org/web/20211205084037/http://www.flock.com/blog)上寻找一篇博文。我们之前对羊群的报道[在这里](https://web.archive.org/web/20211205084037/http://www.beta.techcrunch.com/tag/flock)。

***更新:**这个现在确认了。Arone 在下面评论说:“我将与 Flock 密切合作，这是一家贝塞麦公司。我很喜欢和贝塞麦一起工作，这个机会出现了。Flock 与合作伙伴紧密合作，正在挑灯夜战以推出 1.0 版本。我的群发电子邮件仍将保持完整，如果你知道任何好的业务拓展/战略伙伴，请发给我。干杯，杰弗里"*
