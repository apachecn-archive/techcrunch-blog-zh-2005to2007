# SplashCast 为脸书带来“性感回归”

[![](img/6d5e26a982fd710c21aeaa99b33e6018.png)](https://web.archive.org/web/20221006195556/http://splashcastmedia.com/) 在线多媒体播放器 SplashCast】宣布了一些新的合作伙伴，并在脸书发布了一个新品牌的多媒体播放器。

推出合作伙伴包括贾斯汀·汀布莱克、平克和克瑞丝·布朗，他们提供官方品牌的球员。

SplashCast 脸书应用程序允许用户在他们的个人资料中添加一个播放器，该播放器包括一个定制的皮肤，并预载了创作者的最新视频和图像的 RSS 源。

[![jtplayer.png](img/cc0724d81c87539debe2fe9f227f81ab.png)](https://web.archive.org/web/20221006195556/http://www.facebook.com/apps/application.php?id=2734735952&b) 使用该播放器，用户可以将内容组织成不同的频道(如专辑、视频、音乐)。

这个新球员是 SplashCast 在脸书的一笔大投资。他们专门为脸书的应用程序制作了一个新的管理面板。通过管理面板，所有者可以皮肤播放器，管理他们的渠道，甚至可以控制新闻提要更新消息。作为回报，他们将收取覆盖前 1000 名用户的基本开发费用，之后费用会逐渐增加。

已经有很多“粉丝应用”可以让你跟踪脸书的某个节目，但这些都是定制工作。SplashCast 播放器使公司可以轻松创建自己的多媒体粉丝应用程序。

我不断听到脸书应用程序开发人员从希望进入脸书的公司获得工作机会的故事。像这样的白标应用程序可能是自动化该过程的一种方式。
