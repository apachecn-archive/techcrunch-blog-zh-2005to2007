# 用 Tabblo 你的照片海报

 [](https://web.archive.org/web/20220730222316/http://tabblo.com/) 图片网站 [Tabblo](https://web.archive.org/web/20220730222316/http://tabblo.com/) 今天对他们的服务进行了大量升级，现在结合了维基精神和基于良好布局一般规则的设计工具。Tabblo 不是传统的照片分享网站；Tabblo 不与你可能熟悉的网站竞争，它让你很容易从 Flickr(以及 iPhoto 或 Picasa)等地方导入照片，然后让你将这些照片与文本排列在一起，制作大型拼贴画般的海报，称为 Tabblos，以供打印。

时至今日，Tabblo 设计者不仅可以允许其他用户查看和评论 Tabblo，还可以对其进行修改。例如，每个家长都可以定制一个运动队标签，以便轻松地将他们孩子的照片放在海报的中心，但保持与整个球队海报相同的设计。它就像一个视觉设计的维基，保留了原始的模板。

Tabblo 中的模板设计功能相当好用。它是在 DHTML 中，而不是 ajax 或 flash，有缩放，平移和裁剪功能以及其他几种可以改变演示的方式。例如，尝试看看实时大小转换是如何发生的，这很有趣。对原始尺寸的忠实度通常是好的，但不是在所有情况下(我的狗被拉长了一点，以适应 Tabblo)。所有这些仍然会产生一些非常丑陋的布局，但该网站提供了另一种服务来帮助。他们想出了一系列好的设计规则，并创建了一个按钮来为你的 Tabblo 推荐替代设计。你可以点击通过尽可能多的建议，现在可以返回并撤销尽可能多的建议。类似于完全自由形式的写作和其他类型的媒体创作正在通过维基百科等地方的外部专家控制形式进行调节——Tabblo 为用户提供全面的设计控制，但如果需要，也提供一些指导。

最终结果是非常好看的图像集。Tabblo 的账户是免费的，该公司通过将 Tabblo 打印成海报，并很快打印成卡片和相册来获得收入。海报确实是网站产品的最佳形式。

比起那些想把自己的照片放到网上分享的人，想要打印出带有文字的海报的人可能要少得多，但是就像[onetrumedia](https://web.archive.org/web/20220730222316/http://www.beta.techcrunch.com/tag/One-True-Media/)正在利用从照片和视频刻录的 DVD 赚钱一样，许多网站正在让自助出版书籍变得容易 Tabblo 成为“互联网打印按钮”的目标很可能是可行的。不管是不是这样，使用起来肯定很有趣。
