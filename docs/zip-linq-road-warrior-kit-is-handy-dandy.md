# Zip-Linq Road Warrior 套件非常方便

毫无疑问，成为一个技术呆子有它的优势。当一个漂亮的女孩大声询问“天桥风云”什么时候开始时，通过在你的 Treo 上查找很容易给她留下深刻印象。但是保持所有的齿轮连接可能会有问题。

进入 Zip-Linq 的 Road Warrior Kit 2，这是你的笔记本电脑版本的剃须套件。该套件包含一个 USB 鼠标，几个 USB 适配器和性别转换器，一个 USB 笔记本灯，以太网电缆和分频器，以及一个麦克风和 Skype 耳机，如果你喜欢的话。它们都装在可伸缩的性感拉链卷轴里。

这些东西都装在一个真正的假皮革拉链袋里，在机场安检处不会引起任何人的注意。大约 66 美元，有点贵，但是方便不是免费的。
