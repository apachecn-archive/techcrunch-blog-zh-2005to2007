# 挪威回应约伯的开放数字版权管理

以公开信的形式。题为“关于音乐的想法”，乔布斯概述了 DRM 的状态，以及他认为这种做法可能会结束的地方，有一句名言，如果主要工作室允许解散 DRM，苹果将接受它的心跳。

即使对 iTunes 巨头苹果来说，这也是一个大胆的举动。不出所料，这一声明开始得到回应。挪威消费者委员会的托吉尔·沃特豪斯回复乔布斯的信说:

> 我们很高兴看到史蒂夫·乔布斯承担起苹果作为数字领域领先公司之一的责任，并对挪威消费者委员会发布的投诉做出评论。
> 
> 我们的担心当然是苹果和 iTunes 音乐商店应该解决唱片公司和 DRM 本身的问题，如果需要解决的话——正如我们之前所说的，iTunes 音乐商店为消费者提供服务，因此有责任提供消费者友好的产品。

所以挪威站在苹果一边。我在想制片厂什么时候会介入。
