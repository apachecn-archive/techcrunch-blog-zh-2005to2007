# 索尼 DVP-FX860DT 便携式 DVD 播放器

[![sonyfx860dt.jpg](img/1c460b0dc3dd982bab61dfc81d724481.png)](https://web.archive.org/web/20130628173452/http://tctechcrunch2011.files.wordpress.com/2007/07/sonyfx860dt.jpg "sonyfx860dt.jpg")

索尼有一个新的便携式 DVD 播放器，对不对，叫做 DVP-FX860DT。嗯，它不是*叫*DVP-FX 860 dt，但这是型号。她是标准配置，分辨率为 800×480，8 英寸显示屏，支持 MP3，还有一个电视调谐器。我要说 500 美元对于便携式 DVD 播放器来说太贵了。
