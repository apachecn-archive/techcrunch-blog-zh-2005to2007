# PS3 将于 2008 年推出 DVR

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/01/ps3-goes-dvr-in-2008/>

![drunk_santa.jpg](img/9308e2b9728e2c28ee1e05960efc7dc8.png)

我:圣诞老人，我想要一台 PS3 作为圣诞礼物。

圣诞老人:我以为你讨厌 PS3。

我:嗯，我说了，但是索尼公司在新西兰的营销主管沃里克·莱特发表了一个声明，让我收回我说的话。你看，Freeview 是新西兰的一家免费数字电视和广播提供商，索尼希望在 2008 年 3 月之前支持他们的地面数字广播服务。

圣诞老人:那是不可能的，你他妈的白痴！

我:有可能，因为索尼将为 PS3 推出一款数字调谐器，它也将允许用户录制电视节目。我认真地收回我对 PS3 说过的每一句难听的话。

圣诞老人:哦。好的。我会考虑的。

我:你会考虑？！WTF！我最好在我的圣诞树下看到一个 PS3，否则我会踢你的屁股，伙计。我拥有你。我知道去年圣诞节你在比格斯家做了什么。

圣诞老人:好，好，好。郑重声明，约翰主动找上我。

我:我不管谁勾搭谁。我只想要一台 PS3。搞定它，胖子。
