# 索尼 Vaio N30 发布

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/15/sony-vaio-n30-announced/>

![](img/7758c1e86882f6fa0550bebef21cc54d.png)

在青蛙腿、香烟和萨科齐的国度，索尼发布了 Vaio N30 系列。虽然它不是 UMPC，但它仍然是一个相当紧凑的小机器，具有相当大的功率。你可以按照你喜欢的方式进行配置，但就标准配置而言，你会发现一个英特尔酷睿 2 双核 CPU，高达 2GB 的内存，120GB 的硬盘和 Windows Vista。此外，部分型号还配有 HSDPA 高速上网。考瓦邦加·布拉赫！

如果需要，您可以将硬盘最大容量增加到 200GB，您还可以找到指纹读取器和 TPM 模块。哦，如果你不小心掉了 N30，没关系。它有一个特殊的硬盘盒，以确保适当的保护。N30 将于本月晚些时候上市。
