# 雅虎团队与活字

 [雅虎](https://web.archive.org/web/20201129133609/http://jeremy.zawodny.com/blog/archives/005869.html)和[六合](https://web.archive.org/web/20201129133609/http://www.sixapart.com/movabletype/)已经合作在雅虎的小企业托管上预装可移动活字。雅虎的主机用户现在可以预装活字。这应该有助于雅虎移动更多的主机单位，并且肯定会扩大 Six Apart 的竞争范围。这对两家公司来说都是一笔好交易，尤其是六家公司。

在真空中，这条新闻相当无聊。

但令人兴奋的是，雅虎现在对消费者市场的攻击是多么咄咄逼人。他们[制造](https://web.archive.org/web/20201129133609/http://www.beta.techcrunch.com/2005/11/29/rss-is-now-integrated-into-yahoo-mail-and-alerts/) [bold](https://web.archive.org/web/20201129133609/http://www.beta.techcrunch.com/2005/11/02/new-yahoo-maps-shows-power-of-flash/) [真正能引起消费者共鸣的](https://web.archive.org/web/20201129133609/http://www.beta.techcrunch.com/2005/11/14/yahoo-shoposphere-launches-tonight/) [新产品](https://web.archive.org/web/20201129133609/http://www.beta.techcrunch.com/2005/12/08/yahoo-answers-launches/)，购买[同类最佳的](https://web.archive.org/web/20201129133609/http://www.beta.techcrunch.com/2005/12/09/yahoo-acquires-delicious/) web 2.0 公司，以及[与他人合作](https://web.archive.org/web/20201129133609/http://www.beta.techcrunch.com/2005/11/16/is-the-gawker-yahoo-deal-important/)。

他们以一种好人的方式做着这一切。高管们开始接纳并倾听博客作者和其他人的意见，90 年代末的傲慢完全消失了。雅虎现在正在各条战线上点击。谷歌的傲慢和他们“去他妈的客户”的心态是他们最大的弱点，而雅虎正在充分利用这一点。成功和谦逊是非常强大的组合。
