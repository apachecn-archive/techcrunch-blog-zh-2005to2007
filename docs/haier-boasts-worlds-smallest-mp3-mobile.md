# 海尔号称世界上最小的 MP3 手机

海尔的最新款手机是一个庞然大物，好东西确实来自小包装。为什么？这很简单，真的，优雅，因为它被称为，是一个小 lil 手机与捆绑的功能，使它非常诱人。这些功能包括 MP3 播放器、调频收音机、蓝牙、游戏和长达 200 小时的待机时间。这款手机的大小与玛氏巧克力棒相比，并不算太寒酸。它包括一个漂亮的彩色有机发光二极管和一个滚轮，可以让你浏览菜单。虽然它没有像[诺基亚 5300](https://web.archive.org/web/20160925071456/http://crunchgear.com/2007/03/03/nokia-5300-xpressmusic-hands-on/) 那样的 MP3 播放专用按钮，但它确实通过将键盘兼作控制器来最小化空间。查看视频，你会大吃一惊。
