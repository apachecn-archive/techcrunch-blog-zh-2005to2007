# 索尼 Cyber-Shot DSC-G1 是公司第一个包含 WiFi | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/08/sony-cyber-shot-dsc-g1-is-companys-first-to-include-wi-fi/>

![](img/82fa58fbfd27a1b56138049b4232a661.png)

索尼终于给它的一款相机添加了 Wi-Fi。600 万像素的 Cyber-shot DSC-G1 能够接入 802.11 b/g 网络，因此用户可以立即将他们的照片上传到互联网，无论是 Flickr、脸书还是他们完全未读的个人博客。更好的是，这款相机可以与其他数字生活网络联盟(一个致力于在不同公司之间建立共同标准的行业组织)设备连接。这可能是另一台相机、一台电脑或一种新的支持 WiFi 的火成岩。

DSC-G1 也有一个宽敞的 3.5 英寸液晶显示器。可惜光学变焦只有 3 倍。索尼甚至扔给我们一块骨头，包括 2GB 的内置存储。可惜他们仍然坚持让我们使用他们专有的记忆棒存储格式。

很高兴看到索尼最终采用了 WiFi 摄像头，使该公司现在只落后于时代两年。哦，他们还对它收取全额索尼溢价，要求 600 美元的相机。我们会忽略一个事实，那就是你可以花这么多钱买到低端数码单反，因为，嘿，WiFi 就是这么棒。
