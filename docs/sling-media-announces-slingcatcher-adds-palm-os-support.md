# Sling Media 宣布推出 SlingCatcher，增加 Palm OS 支持

![](img/27f378a655a9a8c219ffd2804eb6ea7d.png)

Sling Media 今天早上宣布，它不仅将为 Palm OS 生产一个版本的 SlingPlayer 软件，而且他们将发布一个全新的产品:SlingCatcher。什么是 SlingCatcher？听起来像是被顶起的弹弓。从发布:

> “SlingCatcher 包括标准清晰度和高清晰度 AV 输出，用于连接电视，包括 HDMI、分量、S-Video 和复合视频，以及模拟和数字音频。在网络连接方面，SlingCatcher 具有集成无线和标准以太网插孔，可连接到家中。SlingCatcher 将于今年年中上市，该公司的目标价格在 200 美元以下。

你还可以从多台电视机上访问你的 DVR。它与 SlingProjector 捆绑在一起，sling projector 是一款允许您发布任何网站、媒体、音频、视频等内容的应用程序。无线连接到你的电视。看起来 Sling 想扩展到其他设备等等。我们喜欢。
