# iPhone 的 Zork:你被 Grue 吃掉了

![zork-iphone.jpg](img/b6de6192ab8cbfc46c9c63c7913e4de2.png)

我不想把 CG 变成 iPhoneAppCentral.com，但这太好了，不能错过。他们已经将 Frotz 移植到 iPhone 上，所以现在你可以在光滑的触摸屏上舒适地“去东方”和“拿起剑”。
 我唯一的问题是打字。即使在使用 iPhone 几个星期后，我仍然无法获得良好的打字速度。我讨厌坐在那里打猎，啄穿*行星坠落*，然后意外杀死弗雷德。也许有人应该尝试一下拇指操作。
