# 个人资料–卡哈纳(Hotmail 测试版)–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/16/profile-kahuna-hotmail-beta/>

卡哈纳(带有 Ajax 的新 hotmail)还没有推出，但是 Start.com 团队([简介](https://web.archive.org/web/20201130025942/http://www.beta.techcrunch.com/?p=125))从 2005 年 5 月开始就一直在认真研究它，现在它正在[私人 beta 测试](https://web.archive.org/web/20201130025942/http://spaces.msn.com/members/mailcall/Blog/cns!1pFgRKa8Lr6GIMM5UtTma4pQ!151.entry) ( [更新](https://web.archive.org/web/20201130025942/http://spaces.msn.com/members/mailcall/Blog/cns!1pFgRKa8Lr6GIMM5UtTma4pQ!155.entry))。

从 [Kahuna](https://web.archive.org/web/20201130025942/https://crunchbase.com/organization/kahuna) 团队(见下文)和各种 [beta](https://web.archive.org/web/20201130025942/http://belfioreguillaume.typepad.com/main/2005/08/sign_up_for_mic.html) [测试人员](https://web.archive.org/web/20201130025942/http://spaces.msn.com/members/cookiezhou/Blog/cns!1ptBormZg9f3WS2LMHmKuIUw!688.entry) ( [和](https://web.archive.org/web/20201130025942/http://www.ajaxian.com/archives/2005/08/hotmail_kahuna.html) [其他](https://web.archive.org/web/20201130025942/http://www.davidsmalley.com/blog/archives/2005/08/15/hotmail-beta/)观看空间)的帖子来看，它对 Hotmail 的显著增强就像 [Start](https://web.archive.org/web/20201130025942/http://www.beta.techcrunch.com/?p=125) 对旧的 MSN 门户一样。看起来他们将会在 mail.start.com 这个网址下推出这项服务。

关键特性包括自由使用 ajax 来消除屏幕刷新。允许不离开收件箱阅读电子邮件的方法，以及更快更干净的用户界面。

卡哈纳最近的历史细节可以在这里找到。

来自[伊姆兰·库雷希](https://web.archive.org/web/20201130025942/http://spaces.msn.com/members/mailcall/)，卡哈纳团队成员:

> **我喜欢邮件测试版的五大原因——原因 1 和 2**
> 
> 邮件测试版是一个全新的网络邮件体验，致力于比现有的网络邮件服务更快、更简单、更安全。该团队专注于阅读和发送邮件的基础知识。邮件测试版正在开发中，大量的测试版用户正在推动它的发展(我们几乎无法跟上他们所有的想法)
> 
> 以下是我在 mail beta 中阅读邮件时最喜欢的一些事情:
> 
> 1.越来越快，越来越快还是邮件测试版明显更快，我的意思是快了一个数量级:
> a)用户界面对许多动作立即做出响应，对其他动作也很快做出响应
> b)很少的上下文切换(整个页面发生变化，你的眼睛必须重新扫描)
> c)你需要更少的点击来完成日常任务
> d)更整洁的外观(包括更多的“空白”?)这样你的眼睛可以放松，更快地找到东西
> 
> 2.使用阅读窗格阅读邮件而不离开收件箱
> 阅读窗格允许你在不离开收件箱的情况下阅读邮件。如果您喜欢 Outlook 中的阅读窗格，现在您可以在 web mail 中使用它。其他 web 邮件服务会强制您打开每封邮件，然后在阅读下一封邮件之前关闭它。如果你有 10 条信息要读，它们需要 20 次点击，卡哈纳:10 次。(这对你的腕管来说怎么样？)
> 
> 想看宽邮件？只需双击消息，瞧！

我们期待亲自测试这一点。
