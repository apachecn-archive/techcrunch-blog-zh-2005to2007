# MyStrands 获得 2500 万美元的 B 轮融资

[![mystrands.png](img/44f681461d97ce599b3c95e8e9f19657.png)](https://web.archive.org/web/20220703212829/http://www.mystrands.com/) 音乐探索平台 [MyStrands](https://web.archive.org/web/20220703212829/http://www.mystrands.com/) 已经完成了由西班牙第三大媒体集团 Grupo Zeta 首席执行官 Antonio Asensio 领投的 2500 万美元 B 轮融资，现有投资者还有 Debaeque 和 Sequal。

MyStrands 提供的产品虽然与 Last.fm 有相似之处，但也扩展到了桌面，在其推荐等式中包含了本地存储的音乐。

首席执行官弗朗西斯科·马丁(Francisco Martin)表示，公司致力于保持独立，这笔资金将帮助公司继续开发其产品。

MyStrands 一直在与 Last.fm、iLike 和 Pandora 等竞争对手进行斗争。如果没有 MusicStrands 的更名，这个市场很难打开。话虽如此，MyStrands 是一个体面的产品；该平台很好地连接到 iTunes，并在你无缝播放歌曲时推荐音乐。2500 万美元不是一个小数目，我相信公司会好好利用它的。
