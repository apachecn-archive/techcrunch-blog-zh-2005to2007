# Linux 桌面:一个 Linux Mac Mini

即将推出:一款 Mac 迷你版 Linux 电脑，专为网上冲浪、浏览和快速上网而设计。Linutop 几乎是图书馆、酒店、医院和网吧中不能做太多事情的较旧、较大机器的替代品，它配有 AMD Geode 处理器、512MB ROM、256MB RAM、VGA 显示器和 100 Mbps 以太网。

它带有一种 Linux 的味道(他们没有说是哪一种)，并且预装了火狐、GAIM、Evince、AbiWord 和 Totem。你最有可能安装和定制你自己的 Linux 映像，但是它必须适合 512MB 的存储空间。没有价格方面的消息，但它的尺寸是 9.3 x 2.7 x 15 厘米，非常小。当你开车的时候，呆在你的车里进行一些 SNES 模拟并不坏，你更有可能在你的图书馆里看到它们作为互联网机器。
