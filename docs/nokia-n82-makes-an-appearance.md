# 诺基亚 N82 亮相

![](img/67840a69b6be06712a0c58a76de3418e.png)

诺基亚已经变得越来越疯狂，每一个 N 系列手机，他们宣布。每一款新手机都倾向于拥有比前一款更时尚的设计以及更高的价格。N82 也不例外。凭借 3G，500 万像素摄像头，320×240 TFT 显示屏，内置 WiFi 和蓝牙，这款手机看起来绝对是杀手级的。关于 N82 将于何时发布或售价多少，目前还没有太多的细节公布。Q4 是目前的传闻，但谁真正知道在这一点上。它可能会在某个时候来到美国，但同样，所有这些细节在这一点上都是粗略的。只是坐下来，盯着一个有 500 万像素摄像头的手机可以有多小。
