# Psile 媒体电脑:时尚而安静

华盛顿西南部的一家邮购电脑公司 CoolTechPC 向我们介绍了一款时尚、令人垂涎的新电脑 Psile 系列。这些媒体电脑配备了令人印象深刻的规格，但这并不是它们酷的原因。

设计耐人寻味:CoolTechPC 消除了嘈杂的风扇，因此即使在机器启动时，也几乎不会发出任何噪音。这里的想法是，你会忘记它在那里，这是一个很好的媒体中心电脑在你的客厅。但是他们在制造一个非常艺术的机器方面更进了一步，有你自己设计的可定制的皮肤，所以如果你碰巧注意到它，你真的会注意到它。

Psiles 具有 Wi-Fi、DVR 功能、无线键盘和鼠标、1.83GHz 的英特尔酷睿双核处理器、320GB 高清硬盘和高达 2GB 的内存。当然包括电视进出。加上运费，它将花费你大约 1700 美元，但如果你喜欢高端娱乐，这是一个不错的选择。
