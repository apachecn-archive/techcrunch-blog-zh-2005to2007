# TiVo 远程调度在威瑞森推出

![](img/4d651f7d344945690ad8e03b21193c99.png)

终于！天知道 TiVo 的主人长久以来一直想远程录制节目，现在，多亏了威瑞森无线，这完全成为可能。拥有 Series 2 或 Series 3 TiVo box 并希望在旅途中录制节目的威瑞森客户可以下载应用程序。所有“现在就买”的手机都与这项服务兼容，包括三星 A950 和 LG 巧克力等热门产品。

但是要注意，便利不是免费的。你将不得不每月支付 1.99 美元来录制节目和远程编程你的 TiVo。
