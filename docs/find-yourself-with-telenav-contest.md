# 在电视导航竞赛中发现自己

![](img/68dddcf437560bc915cd5e796bd3e9e9.png)
为了庆祝我们对 [TeleNav](https://web.archive.org/web/20210416202842/http://telenav.com/) 服务的高度赞赏，并认可其持续的全球扩张(美国、加拿大、巴西、中国、意大利以及更多国家)，我们将发起一场竞赛，TeleNav 服务范围内的任何人都可以参加。要参赛，请发送电子邮件至 contest at crunchgear dot com，解释你关于定向障碍的最佳故事(使用主题:“迷路”)。

获胜者将获得一个 TeleNav 蓝牙 GPS 接收器和三个月的服务。比赛从今天开始一直持续到美国东部时间周五中午。我们会发布最好的故事，然后大家投票选出获胜者。

所以让我们开始吧。
