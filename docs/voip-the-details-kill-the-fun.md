# VOIP——细节扼杀乐趣——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/09/23/voip-the-details-kill-the-fun>

一些网络电话服务已经推出，帮助人们用普通电话打更便宜的电话。没有一个对大众市场有吸引力。

当您从 VOIP 电话或软件(Skype、Vonage 等)发起呼叫时，VOIP 非常有用。).这些 VOIP 网络可以呼叫其他 VOIP 电话，或者接入普通电话网络进行相对便宜的呼叫。Vonage 很久以前就取代了我的普通电话服务，越来越多的人正在使用 VOIP 解决方案，而不是普通电话。

但是一批新的公司已经开始尝试让人们从一部普通的 POTS(普通老式电话服务)电话(通常是一部手机)到另一部 POTS 电话进行免费或廉价的 VOIP 通话。如果有人做对了，就会有一个巨大的市场等着去摧毁。问题是没有人做对。除非这些服务简单易用，否则大众市场不会采用它们。

这些服务通常采用两种方法之一来允许人们拨打 VOIP 电话。一种方法是告诉服务人员您正在拨打哪个号码以及您想拨打哪个号码。然后，该服务调用双方并连接它们。第二种方法是分配特殊的电话号码来代替普通的电话号码。这些特殊号码由 VOIP 服务控制，绕过 POTS 系统进行最昂贵的通话。

这两种方法都不允许人们在飞行中给某人打电话。这两种方式都需要多个步骤才能打电话，通常还需要使用网站(这意味着你必须坐在电脑前，或者尝试通过手机浏览器访问服务)。

以下是我们一直在追踪的几个例子:

[Jajah](https://web.archive.org/web/20211022033352/http://www.jajah.com/) :进入网站，告诉它你的电话号码和你要呼叫的号码，两部电话就开始通话。打电话[费率](https://web.archive.org/web/20211022033352/http://www.jajah.com/info/rates/)很便宜，有时免费。但是你必须在你的电脑前使用它，并且如果你打非免费电话的话，你必须和 jajah 有付费关系。然而，本周他们有一些重大消息传出，值得注意。

[瑞博特尔](https://web.archive.org/web/20211022033352/http://www.rebtel.com/):我们先在这里报道瑞博特尔[。他们刚刚宣布了高达 2000 万美元的风险投资](https://web.archive.org/web/20211022033352/http://www.beta.techcrunch.com/2006/07/26/rebtel-makes-international-calling-easy-and-cheap/)[。Rebtel 打电话的方法极其混乱。基本费用是每周 1 美元。然后他们会给你的每个朋友分配当地的电话号码。你打那个号码而不是那个朋友的正常号码。你的朋友拿起电话，挂断并拨打刚才呼叫他们的号码，以便与你联系。然后通话就免费了。如果这还不清楚，你可以在这里看到说明](https://web.archive.org/web/20211022033352/http://business2.blogs.com/business2blog/2006/09/with_a_rebtel_y.html)。您也可以在对方没有挂断电话的情况下使用 Rebtel，但您将为通话付费(费率低于正常电话费率)。

[Hullo](https://web.archive.org/web/20211022033352/http://www.hullo.com/) :我们在这里介绍了 Hullo [。和 Jajah 很像，功能稍微好一点。](https://web.archive.org/web/20211022033352/http://www.beta.techcrunch.com/2006/08/22/hullo-to-enter-voip-war-with-free-product/)

[ConnectMeAnywhere](https://web.archive.org/web/20211022033352/http://www.connectmeanywhere.com/):Sam Sethi[在英国 TechCrunch 上写了关于](https://web.archive.org/web/20211022033352/http://uk.beta.techcrunch.com/2006/09/14/is-moip-the-next-buzzword/) ConnectMeAnywhere 的文章。像 Rebtel 一样，ConnectMeAnywhere 为您的联系人分配本地号码，您使用这些本地号码而不是他们的正常电话号码。与 Rebtel 不同的是，这里没有免费的选项，用户可以挂断电话，然后回电。相反，CMA 收取的费用比你的电话公司要低。他们的价格在这里是。

这些服务都不足以改变大众市场的用户行为。对大多数人来说，坐在电脑前或拨打特殊的电话号码太麻烦了。当然，强迫接听电话的人挂断电话并回电不是很有吸引力。传统的 POTS 费率继续快速下降，这意味着与难以使用的 VOIP 提供商合作的动机降低了。

当然，我们会在新服务推出时对其进行监控。或许有人会提出更好的解决方案。在那之前，我不会押注任何一家当前的公司。
