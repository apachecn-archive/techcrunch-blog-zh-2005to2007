# Wii 光盘即将被…备份用于合法用途

![](img/45453730c92dc585dd4006ba2f50add2.png)

今天早上在万能的 Digg 上冲浪时，我无意中发现了一个 Wii 光盘转储器的链接。不，这不是一个恶心的马里奥腹泻的 scat 视频，这是一个小程序，当它与 PC 中适当的 DVD 驱动器结合时，可以将 Wii 游戏光盘的原始内容转储到你的硬盘上。

似乎你需要一个 LG-8164b 或 LG-8163b DVD-ROM 驱动器来完成这项工作，这个过程需要 50 多个小时才能完成。当然这需要一段时间，但想想所有的~~盗版场景发布副本~~备份你将能够做到！
