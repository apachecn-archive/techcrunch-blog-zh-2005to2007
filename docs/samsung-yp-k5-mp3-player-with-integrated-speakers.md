# 带有集成扬声器的三星 YP-K5 MP3 播放器

可能只是 iPod 疲劳，但三星的这款 YP-K5 看起来比 iPod Nano 更好看，更经典，而且我们敢说，更性感。该播放器有一个像素类型的图标显示，并支持 MP3/WMA/ASF，加上甚至开源呆子支持 OGG。它有 1GB、2GB 和 4GB 三种尺寸。

即使有集成扬声器带来的额外体积，我们可能大多数时候都不需要，YP-K5 看起来仍然很薄。它现在只在韩国提供，但是如果它在美国找到出路，我们会让你知道。
