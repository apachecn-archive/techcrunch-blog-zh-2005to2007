# Zune Insider 详细介绍 Zune 盒子内的配件

除了具有 WiFi 功能的媒体播放器中的下一件大事，如果你决定在 11 月 14 日以略高于传言的 249.99 美元的价格购买一台 Zune，自然也会有一大堆配件。这都是标准票价，但我们认为你会想知道。就个人而言，我们觉得如果它被包含在内，它就不是一个真正的配件，但我们将默认张敬利在这一点上。

你会得到你的 Zune 播放器，你的磁性耳塞，USB 同步电缆，和一个微型麂皮拉绳袋。这里没有什么能真正让粉丝们激动的，尽管看起来你的袋子会和你的 Zune 相配，这很了不起。
