# 三星:你有厌食症

三星又一次推出了整个[世界上最薄的手机](https://web.archive.org/web/20210303051627/http://crunchgear.com/2006/12/27/samsung-does-it-again/)。有人有体重问题。
 超极本 10.9 也就那么 10.9 毫米厚。明白了吗？10.9?UE 10.9 slider 一开始就配备了 320 万像素的摄像头、MP3/视频播放、立体声蓝牙、microSD 卡插槽、数字功率放大器(wha？！？)、电视输出端口和 80MB 内存。
