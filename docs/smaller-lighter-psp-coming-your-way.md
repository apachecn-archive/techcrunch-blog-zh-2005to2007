# 更小、更轻的 PSP 向你走来

![](img/8484cabf53a49407c086ac23020b72ad.png)

你还记得索尼重新设计了外形更小、设计更薄的 PS2 吗？还记得它是如何像野火一样到处销售，直到今天还在销售吗？索尼记住了这一点，现在正在对它的 PSP 做同样的事情。SCEUK 负责人 Ray Maguire 证实了这一传闻，并表示尽管控制台将变得更薄(正如 99%的科技作家所说)“更时尚”，但屏幕尺寸将保持不变。

至于发布日期或制作时间框架，马奎尔暂时保持沉默。所以你要像个真正的男人一样耐心等待一次。
