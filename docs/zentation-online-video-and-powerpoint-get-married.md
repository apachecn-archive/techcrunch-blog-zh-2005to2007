# Zentation:在线视频和 Powerpoint 结婚

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/06/28/zentation-online-video-and-powerpoint-get-married/>

[![zentation.png](img/20762eeb879df20997118d1d12af0a50.png) ](https://web.archive.org/web/20161130130925/http://www.zentation.com/) [Zentation](https://web.archive.org/web/20161130130925/http://www.zentation.com/) 是一项免费服务，将视频和 PowerPoint 演示结合在一起，提供两者的混合版本供在线观看。

使用 Zentation 很简单。用户将他们的视频上传到谷歌视频，登录 Zentation，复制并粘贴谷歌视频的 URL，然后上传一个 PowerPoint 文件。最后一步是使用“ZenSync”工具为演示文稿中的每个图形提供精确的开始时间。

Zentation 目前只支持谷歌视频；显然，谷歌视频是唯一允许用户跳转到视频中任何一点的主要视频分享服务，即使它还没有完全下载。YouTube 不提供这一功能，Zentation 认为这是这项服务的关键。

可以在 Zentation.com 上直接观看混合演示，或者通过一个像下面这样的嵌入式小工具来观看。

一些人会对将 PowerPoint 演示与在线视频结合的想法感到不满。不得不耐着性子听完任何严重依赖 PowerPoint 的会议或演示已经够糟了，但有时 PowerPoint 幻灯片可以增加这种体验。PowerPoint 上瘾的专业演讲者会喜欢 Zentation，因为它使他们能够将自己的幻灯片添加到录制的视频演示中。

Zentation 并不是第一个试图将 PowerPoint 风格的演示带到网络上的应用。像 [Slideshare](https://web.archive.org/web/20161130130925/http://www.techcrunch.com/2006/10/04/introducing-slideshare-power-point-youtube) 这样的网站提供在线 PowerPoint 托管，WebEx、 [DimDim](https://web.archive.org/web/20161130130925/http://www.techcrunch.com/2006/09/07/dimdim-launches-foss-challenge-to-webex/) 和 [TeamSlide](https://web.archive.org/web/20161130130925/http://www.techcrunch.com/2006/09/21/teamslide-web-presentations-go-hosted/) 也在这个领域竞争。Zentation 与众不同，它提供了一个带有视频的混搭服务，但所有这些都表明了人们对在线演示空间越来越感兴趣。
