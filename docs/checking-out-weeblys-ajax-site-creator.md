# 查看 Weebly 的 Ajax 站点创建者

 [](https://web.archive.org/web/20220627082156/http://www.weebly.com/) 我不得不称赞[weely](https://web.archive.org/web/20220627082156/http://www.weebly.com/)，它正在变成互联网上最好的简单网站创建工具之一。几个月前，当我们第一次听说他们的时候，我看到的只是另一个错误的 Ajax 网站创建者。我们真正想看到的是比现有工具更好的东西——[Google Page Creator](https://web.archive.org/web/20220627082156/http://www.beta.techcrunch.com/2006/02/22/google-pages-released/)、 [Sitekreator](https://web.archive.org/web/20220627082156/http://www.sitekreator.com/) 和 [Synthasite](https://web.archive.org/web/20220627082156/http://www.beta.techcrunch.com/2006/09/04/very-early-look-at-synthasites-ajax-website-builder/) 。

在过去的几天里，我们重新审视了一下，Weebly 在这两个月里取得了长足的进步。

Weebly 允许用户通过拖放 Ajax 界面创建简单的多页面网站，只需点击几下鼠标。然后，用户可以将这些网站保存到 Weebly 服务器上，或者下载到其他地方使用(如果你下载了该网站，Weebly 标题栏将被删除)。可以包括 RSS 元素。除了标准的文本和图像工具以及 RSS 支持，Weebly 还支持基于 Javascript 的元素，如 Google Maps、Flickr Badge 等。

一些最近的评论者在博客中抱怨这个网站太慢了，尽管在我们的测试中它的反应非常好。这可能是 Weebly 站点的流量负载问题。我们将在未来几天的定期测试中看看它的表现如何。

Weebly 还很好地利用视频向人们展示了这一切。这是我们继续向新公司建议的事情，最近的简介帮助创业公司做到了这一点。我们在下面提供了 Weebly 的演示视频(如果有人知道他们从哪部色情电影中偷了音乐，请告诉我)。还有一些视频可以帮助用户了解该服务的某些方面。
