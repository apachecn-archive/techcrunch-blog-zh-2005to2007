# 龟滩耳力 X1:好功能，坏名声| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/03/turtle-beach-ear-force-x1-good-features-bad-name/>

也许是我，但是“耳力”这个名字就是不能让我想跑出去买这些耳机。或者告诉任何人我买了它们，不管它们有多好——这个轻便的耳机看起来不错。正如您在图中看到的，X1 是您游戏生态系统的一部分。他们能够混合语音聊天和游戏音频，因此两者都可以在立体声中听到，而不是只在左右声道之间分离。集成放大器增强音频，并通过麦克风静音按钮控制音量。这款售价 59.95 美元的耳机由 USB 端口供电，可以与任何游戏主机或 PC 配合使用。
