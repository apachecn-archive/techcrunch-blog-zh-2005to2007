# 面向生态极客的太阳能键盘

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/07/solar-keyboard-for-eco-geeks/>

如果你担心你的色情下载习惯对环境有害，那么你的良心观念比我的更混乱。但你拿起这个天才键盘后可能会感觉更好。通过内置的太阳能电池板，无线键盘可以利用环境光充电，而不是使用现成的电池，电池会毒害地面并杀死你的家人。可悲的是，包括老鼠需要这些有毒的铅酸细胞，但至少你有一半的责任。

当然，你妈妈的地下室光线不太好，所以这可能不适合你。但是如果你住在，比如说，一个温室里，或者在山顶上，或者在一座山上，或者在西雅图之外的一个城市里，那么你可能会得到足够的阳光来让这件事情变得有价值。
