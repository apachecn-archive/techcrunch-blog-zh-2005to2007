# 缩略图:截图的连续画面

[![thumbstrips.jpg](img/88115fed000306761d442e9e71b239f8.png) ](https://web.archive.org/web/20210803141914/https://addons.mozilla.org/en-US/firefox/addon/5045) [ThumbStrips](https://web.archive.org/web/20210803141914/https://addons.mozilla.org/en-US/firefox/addon/5045) 是一个 Firefox 插件，可以记录一系列截图。

该插件来自 Intuit 的[创新实验室](https://web.archive.org/web/20210803141914/http://innovation.intuit.com/)，这是一个由 Quicken Roy Rosin 前总经理领导的 Intuit R & D 部门。

ThumbStrips 记录用户浏览会话的连续画面，以便于返回和共享用户访问过的页面。这个插件是在观察了人们在后退按钮和历史列表上的挣扎后开发的。共享组件支持协作搜索或人们做研究项目，试图相互分享他们去过的地方和他们在那里发现的东西。

到目前为止，这个相当新的插件已经被下载了 2 万次。

记住你几小时前访问过的网站的问题是很常见的，至少对我来说是这样。ThumbStrips 是一个聪明的想法，它提供了一个访问浏览器历史的简单方法。
