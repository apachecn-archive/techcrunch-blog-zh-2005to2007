# 十部:一百万个小片段

![](img/c6b5b00ec3c16d32a3eca2c667494a68.png)

我最近撕开了一张 40GB 的 iPod 照片，非常懊恼地发现，我在匆忙中撕破了耳机接口。不幸的是，几天前我还不知道 Octopart，但我确实在 [iFixIt](https://web.archive.org/web/20210301204705/http://www.ifixit.com/) 找到了我需要的东西，这让我松了一口气。

Octopart 是一个网络电子产品聚合器，为我们所有人的极客寻找各种组件。你输入你要找的东西，Octopart 就会搜索出来。 [TechCrunch](https://web.archive.org/web/20210301204705/http://www.beta.techcrunch.com/2007/03/05/octopart-vertical-product-search-electrical-engineers/) 发现它是由两个厌倦了在网上寻找晦涩难懂的作品的物理学毕业生创建的，它似乎是一个适合 DIY 和自制爱好者的伟大系统。
