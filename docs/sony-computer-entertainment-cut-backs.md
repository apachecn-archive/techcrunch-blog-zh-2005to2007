# 索尼电脑娱乐削减| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/07/sony-computer-entertainment-cut-backs/>

那些不会留在任天堂营销和销售团队的员工在向索尼投简历之前最好三思。路透社报道称，视频游戏行业很快会有更多的人失业。索尼打算在美国裁员，甚至考虑在日本进行重组！

根据路透社的报道，索尼尚未透露其在美国的约 1，500 名员工中有多少人将因重组而失业，但根据过去的数据，我们猜测可能高达 10%。今年 4 月，索尼电脑娱乐公司宣布计划裁员约 160 人，占员工总数的 8%。目前，游戏部门在全球雇佣了大约 4500 名员工，但这个数字只会越来越小。看看不买 PS3 会怎么样。人们失业了！
