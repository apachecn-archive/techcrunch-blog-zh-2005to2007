# 恐怖啊！恐怖啊！Vibram 吮吸的五指

有时候，当你看到某样东西时，你会立刻希望你能看不见它。这就是我穿着 Vibram 五指运动凉鞋的地方。这些“鞋”就像加固的脚趾袜，有单独的脚趾。这是没有借口的，老天作证，我看到的第一个穿这个的人会被一拳打在头上。

真的，即使在最下雨的日子里，当你穿着人字拖时，我也不得不看着你的脚趾。我甚至会忍受反复出现的趋势，即 Ugg 靴子。但这让鞋类进化走得太远了。你不应该希望你的脚底像烧伤病人的一样，这就是这里发生的事情。

我找不到词语来描述这些是多么的错误。这比那些胶水粘的人字拖还要糟糕。这比希特勒还糟糕。这比清醒的罗汉还糟糕。这比 Windows Mobile 还糟糕。如果你想买这些东西，我会让你的孩子患雷神的屁眼癌。你听到了吗？
