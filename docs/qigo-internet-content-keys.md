# QiGO 互联网内容密钥

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/30/qigo-internet-content-keys/>

![untitled1.png](img/657d35b545055ea44584584094c7388e.png)

今天，QiGO 宣布了一种新的营销工具，这可能会彻底改变公司营销新产品的方式。好吧，这可能是一派胡言，但也有可能发生。超级独特的 USB 设备将自动启动营销人员希望您看到的任何互联网内容。最重要的是，按键可以变形为任何标志。可能性是无穷无尽的，你只需要发挥你的想象力。

例如，您最喜欢的乐队可以在新 CD 发布之前发布这些专辑，包括特殊的后台图片、mp3 样本，甚至是您最喜欢的热门歌曲的“不插电”版本。你也可以为你所有疯狂的维加斯图片和视频制作一批。就像我说的，可能性是无限的。你有足够的时间来考虑你将在这些键上扔什么，因为它们要到秋天才可用。
