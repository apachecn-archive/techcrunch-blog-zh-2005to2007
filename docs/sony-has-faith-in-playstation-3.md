# 索尼对 PlayStation 3 有信心

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/04/sony-has-faith-in-playstation-3/>

![](img/3c285fc88e7bb0d238002c142d7d5bbd.png)

我们担心索尼可能会因为 PlayStation 3 受到的所有负面新闻而有所退步。我的意思是，在游戏机发布两周后更换管理人员不是现在最好的迹象，不是吗？好吧，索尼坚决支持它的新主机，声称 PlayStation 3 比它最初声称的处理能力更强，并且它正在利用这种能力制作更好的游戏。

> 比他们最初预期的有更多的处理空间，这导致了新游戏功能的开发

–*SCEE 副总裁保罗·霍尔曼*

索尼还计划为 PlayStation 3 发布键盘和鼠标插件，允许用户轻松浏览互联网和使用任何第三方操作系统。霍尔曼表示，随着开发人员找到利用游戏机功能的新方法，PS3 将获得几项固件升级。如果 PS3 是最后一款索尼游戏机的传言引起了你的注意，不要担心，霍尔曼也提到了 PS4:

> 说因为管理层变动就不会有 PS4 有点牵强

看起来索尼的目标不仅是将 PS3 改造成一个不同于以往任何游戏主机的游戏主机，而且还要使 PS3 成为一个强大的家庭影院媒体中心，就像它是一个游戏主机一样。希望在不久的将来，我们将最终开始看到 PlayStation 3 的能力，在此之前，我们所能做的就是等待。
