# 未来学家:亲手操作最整洁的新纳米技术

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/15/the-futurist-hands-on-with-the-neatest-new-nanotech/>

![](img/827702c9f806c819baaa331f82bed3e1.png)

所以我现在和音频文件麦克·T2 在一起。原来，这个高地人 T4 和流氓的家乡不仅因为给世界带来了电视、青霉素和克隆羊而声名大噪，而且现在是新的纳米技术研究的热气腾腾的温床。我们花了一周时间检查出血边缘纳米，包括使用有机发光二极管光消除皮肤癌的方法，对抗重力的微型机器，以及能够在肠道中进行类似旅行的可消化药丸。

正如纳米技术研究所的创始人 Ottilia Saxl 所说，“我称纳米技术为科学派——就像科幻小说一样，但却是真实的。”

本周，我们将关注下一波纳米技术研究，它将改变我们做任何事情的方式。

根据我们对十几个苏格兰纳米技术设施的第一手观察，我挑选了几个未来学家认为最有希望改变世界的项目。人们——这些东西太疯狂了。其中一些东西是直接从漫画书和老科幻电影中撕下来的。我们不只是在谈论食肉纳米机器人(现场的专家似乎都不太有兴趣讨论这个想法)，我们还在谈论喷雾皮肤和癌症治疗。

**喷雾状皮肤:**

这里的纳米技术研究所给了我关于德国马尔堡大学一位名叫安德烈亚斯·格雷纳的研究员所做研究的独家新闻。似乎马尔堡博士开发了一种“静电纺丝”技术，基本上允许你在开放的伤口上喷洒一种无形的网络，就像抗菌喷雾一样。其应用是显而易见的:不再有战场感染，移除创可贴的痛苦撕裂可能成为过去。

**可吞咽式人体测量仪:**

还记得电影《内部空间》中丹尼斯·奎德用微型容器拍摄马丁·肖特的内脏吗？这可能缺乏喜剧经典的明星效应，但格拉斯哥大学分支凯尔文纳米技术公司的科学家们让我们了解了所谓的“药丸实验室”想法是:病人吞下一个无线发射器，当它通过你的系统时，它将你的重要数据无线传输给医生。曾经需要侵入性手术的读数现在就像一大口水抑制你的呕吐反射一样简单。虽然可能性很大，但这里的医生似乎对该设备抗击肠癌的能力最感兴趣。

**对抗重力的微型机械:**

毫无疑问，我们看到的最整洁的东西之一是能让牛顿耸耸肩的纳米机器。爱丁堡大学的有机化学教授大卫·利博士(和他的乌合之众科学家团队)已经开发出一种方法，利用光动力显微机器真正使水跑上斜坡。虽然往山上移动几滴液体除了让那些疯狂的巨型吸管更有效之外不会有太大作用，但未来的可能性是 x 战警式的。“目前，我们可以移动微小的液滴，”利博士说。“但是有一天，移动固体物体是可能的。如果我想要我的那杯茶，我可以发射激光，它就会来找我。这听起来像科幻小说，但没有理由不可能是真的。”校园恶作剧应该变得更加有趣。

**治疗皮肤癌的新方法:**

我不知道，但显然大约 40%的英国人和 75%的澳大利亚人在他们生命中的某个时刻会患上某种形式的皮肤癌。这太疯狂了。目前，大多数治疗是痛苦的，昂贵的，并且需要重复去医院。因此，当我们看到一种只需使用乳液和有机发光二极管灯(就像可以照亮你的数字音频播放器的那种灯)就能消除这种最严重的皮疹的方法时，我被震惊了。

光动力疗法治疗皮肤癌已经有一段时间了。也就是说，患者将光敏洗液涂在患处，然后进行一些治疗，让强光与癌细胞相互作用。光动力疗法的问题在于，它非常痛苦，非常昂贵，并且需要反复去医院。这里最大的突破是发明了一种可穿戴的有机发光二极管绷带(见上图),它可以包裹住癌变区域。像传统的光动力疗法一样，你涂上乳液，只是现在你可以在一天中简单地戴上绷带，电源夹在你的腰带上，靠近你的黑莓手机。这里的优势是，有机发光二极管不仅是一个更加一致的光源(意味着甚至可以全方位治疗)，而且它更便宜，痛苦更少，不需要去医院(它很轻，便于携带)，并且足够灵活，几乎可以在任何地方佩戴。正如该项技术背后的公司 Lumicare Ltd .的安德鲁·麦克尼尔博士所说:“阳光是皮肤癌的主要原因，但正确使用光线可以治疗皮肤癌。”
