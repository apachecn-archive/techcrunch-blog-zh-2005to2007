# 我的雅虎上的新东西——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/05/16/new-stuff-at-my-yahoo/>

巨头们真的开始关注个性化主页，其中雅虎主导了 T4，谷歌强势崛起。谷歌表示，他们的个性化主页是过去几个财政季度中增长最快的产品。Pageflakes 和 [Netvibes](https://web.archive.org/web/20220121103114/http://www.netvibes.com/) 是这一领域两个著名的初创公司。

今晚，雅虎宣布对最近重新推出的“我的雅虎”进行一些改进:一个新的日历模块，一个使用网飞 API 的“我的网飞”模块，以及一个对雅虎书签模块的更新。

日历模块具有读/写功能(正如它应该的那样)。“我的网飞”可以让用户看到最近发行了哪些电影的 DVD、你正在排队的电影以及你家里的电影。书签模块的主要新增功能是能够直接从模块中添加新书签。

新功能将于周四推出。屏幕截图如下。第一个显示了日历和网飞模块。
