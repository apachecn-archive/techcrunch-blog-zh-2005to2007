# 三星 smrgsbord 闻起来像……

![samsung-anycall-5200-gaming-phone.jpg](img/dc7f815efbd02ae92e0cd6df6367f744.png)

我通常会使用三星在市场上推出的手机，但 5200 让我挠头，重新思考我的忠诚度。双滑盖设计很好，但你不能为了游戏控制而把 QWERTY 拆了。WTF！它运行在 WinMo 上，所以我可以不太关心 2.8 英寸的触摸屏，300 万像素的摄像头和 HDSPA。还有人注意到背面类似 PSP 的 logo 吗？
