# RFID Guardian 为手里拿着芯片的家伙提供安全保障

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/07/rfid-guardian-provides-security-from-that-dude-with-a-chip-in-his-hand/>

![](img/f69a413cf3fa3de64a1f0aea9ab353f0.png)

并不是每个人都真正理解 RFID 以及如何真正利用它。我的意思是肯定的，它被商业和物流公司广泛用于跟踪货物等，但个人使用 RFID 是罕见的，并受到重视。RFID Guardian 是一个移动的、电池供电的“个人防火墙”,可以抵御您附近未经授权的 RFID 标签。所以基本上，如果一个人在他的手里植入了 RFID 芯片来看足球比赛，而你不想让他来，你可以使用 RFID 卫士来提醒你，当他进入你的私人空间时。

RFID 卫士的真正用途是丰富的。主要的担忧是，随着技术的进步，隐私将会减少，RFID 阅读器将变得更加普遍。假设你有一张使用新的“快速通行证”系统的信用卡，但是因为害怕你的信用卡信息以不安全的方式被传送而不使用该系统。你可以随身携带 RFID Guardian，当它开始在你当地的 7-11 里完全疯狂时，你会知道他们提供那些讨厌的“快速通行证”系统，并将友好地把你的业务带到其他地方。你也可以设置 RFID Guardian 允许友好的 RFID 芯片进入你的空间，以防你碰巧喜欢那个手里拿着芯片的家伙。在他们的[网站](https://web.archive.org/web/20150427004810/http://www.rfidguardian.org/)上关注 RFID Guardian 的进展。
