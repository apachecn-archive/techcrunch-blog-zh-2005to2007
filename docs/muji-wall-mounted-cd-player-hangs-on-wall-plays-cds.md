# 无印良品壁挂式 CD 播放器挂在墙上，播放 CD

这个 CD 播放器的简单性让我非常喜欢。只需打开光盘，拉动琴弦就可以播放了。扬声器内置在框架中，其他控制装置位于顶部。199 美元，你可以认为它是一件价格过高的老式立体声设备，也可以认为它是一件实用的艺术品(我倾向于后者)。
