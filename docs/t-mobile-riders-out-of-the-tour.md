# T-Mobile 车手出局

来自欧洲体育台的一则与手机相关的新闻。对于 T-Mobile 赞助的职业自行车队来说，这是悲伤的一天。德国人帕特尼克·辛克维兹在与一名观众相撞后退出了环法自行车赛。这场艰苦的长达一个月的比赛于一周前在伦敦开始，然后前往法国，几乎没有障碍阻止粉丝们，但具有讽刺意味的是，这场事故发生在辛克韦茨越过终点线之后。截至今天，辛克维兹在去酒店的路上与一名观众发生高速碰撞后仍在医院。这名伤势严重的观众已经被飞机送往格勒诺布尔的一家医院接受治疗。

事实上，T-Mobile 一天之内就失去了六分之三的乘客；除了辛克维兹之外，英国短跑运动员马克·卡文迪什(他在上周日从伦敦到坎特伯雷的第一阶段就一直受到问题的困扰)，以及澳大利亚队领队麦克尔·罗杰斯也退出了比赛。作为最受瞩目的职业自行车赛中最大的车队之一，T-Mobile 今年的表现令人遗憾地有所下降。

今天休息了一天后，明天从瓦勒迪泽尔出发继续游览。
