# TLA 推出饲料广告

 [](https://web.archive.org/web/20220522112158/http://www.text-link-ads.com/feedvertising/) [文字链接广告](https://web.archive.org/web/20220522112158/http://www.text-link-ads.com/)(一家允许网站在其网站上包含简单文字广告的广告网络)今天早上推出了一款名为[feedbertising](https://web.archive.org/web/20220522112158/http://www.text-link-ads.com/feedvertising/)的新产品。Feedvertising 允许博客作者和其他网站所有者在他们的 RSS 订阅源中包含广告或其他消息。

我们喜欢 Feedvertising(请注意最后的披露),因为它简单、免费，让出版商能够直接销售他们自己的广告，包括 TLA 销售的广告，或者简单地放入出版商希望包含的任何信息和链接。

我们已经通过一个 WordPress 插件(其他博客平台的插件即将推出)将 Feedvertising 集成到了 TechCrunch 的主要 RSS [Feed](https://web.archive.org/web/20220522112158/http://feeds.feedburner.com/Techcrunch) 中。我们选择不包含广告，而是在 feed 内容的末尾添加一个简单的文本项来推广其他 Crunch 网站。

有关该产品如何工作的更多信息，请参见他们制作的视频教程[这里](https://web.archive.org/web/20220522112158/http://www.tubetorial.com/feedvertising/)。相信很多博主都会想用这个。

**披露:** TLA 是 TechCrunch [的长期赞助商](https://web.archive.org/web/20220522112158/http://www.beta.techcrunch.com/advertise/)。
