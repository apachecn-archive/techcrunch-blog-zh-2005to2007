# 推特移动化|技术危机

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/06/twitter-goes-mobile-2/>

我经常玩推特，但它并没有对我的生活产生巨大或深远的影响，所以我的账户一直在等待。推特最终推出了一个移动网站，这可能会让我改变主意。这并不是说，短信/短信影响了我，因为我有无限的，但它仍然是一个讨厌在我的书。无论如何，如果你喜欢叽叽喳喳，并希望它能变得更简单，那么你的一天就来了。享受生活，让每个人都知道你在做什么。
