# RIYL:憎恨索尼的阅读器

昨日在水中死去。哦，等等，也许是的。至少 TeleRead 的 David Rothman 是这样说的，他将它与 1958 年的汽车事故 Edsel 进行了比较。他的关键论点肯定不是离谱的——单色屏幕、DRM 的普遍糟糕、只有大约 10，000 个标题、没有背光，以及非常有限的产品相对较高的成本。然而，除此之外，在昨晚玩了一个之后，我忍不住想要一个。我在一个非常暗的房间里，可以轻松而不费力地阅读显示屏。这是足够简单的操作和打印放大很好，将是近盲人方便。但是我认为它会失败吗？当然啦我真的希望它不会完全失败，因为我真的想看到它的下一代。
