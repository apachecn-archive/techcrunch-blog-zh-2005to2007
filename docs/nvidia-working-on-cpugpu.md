# nVidia 致力于 CPU+GPU | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/19/nvidia-working-on-cpugpu/>

nVidia 已经被在 GPU 上比在 CPU 上做得更好冲昏了头脑。在他们相互击掌并击落了 374 架 Coronas 之后，他们开始实施一项计划，开发一款具有 GPU 功能的 CPU。

根据 nVidia 的说法，OEM 制造商希望 CPU 将 CPU 功能与图形功能相结合，以降低成本，并且在其主板上只有一个芯片，而不是两个。这种产品可能会对 UMPCs 产生相当大的影响，在不增加尺寸或功耗的情况下提供更丰富的图形体验。

凭借当今显卡的强大功能，nVidia 寻求在日常应用中利用这种能力是有道理的——你不能一直玩游戏。
