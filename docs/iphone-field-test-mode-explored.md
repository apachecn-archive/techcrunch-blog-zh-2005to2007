# 探索 iPhone 现场测试模式

想成为卡车里那个可以四处走动测试手机信号接收的酷哥吗？只需在你的 iPhone 中输入“* 3001 # 12345 # *”——你甚至可以在通话过程中这样做。这个组合键打开了 iPhone 的现场测试模式，这基本上是一个小集合或读数，报告当前的蜂窝条件、固件信息以及运营无线公司所需的所有好东西。你能改变什么吗？不，但你可以像我一样，用它来测试你房子各个区域的接收情况，以决定你是否需要一个无线助推器。
