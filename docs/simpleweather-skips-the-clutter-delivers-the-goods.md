# SimpleWeather 跳过杂乱，交付货物| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/19/simpleweather-skips-the-clutter-delivers-the-goods/>

![](img/5688461188293ac01a8abe5e026f7fd9.png)

大多数天气网站通常充斥着大量的广告，无用的信息，比如下一次当地高尔夫锦标赛什么时候举行，并且总是让你不停地输入你的邮政编码。对于我们这些只想知道天气的人来说，看看简单天气。

该网站是极简的，易于使用。只需输入一次您的邮政编码，SimpleWeather 就会永远记住您的位置(或者直到您清空 cookies)。它提供了易于阅读和理解的数据，没有混乱，并能够从华氏温度转换为摄氏温度。目前只有美国，但他们确实计划在今年夏天加入加拿大和英国。

现在我确定你们中的很多人会去“但是我有一个天气预报的小工具！”或者“我的雅虎！主页也给我显示天气！”这太棒了，但并不是每台电脑都能运行 widgets。至于起始页，它们充满了你在寻找外界条件时不想看到的其他信息。简单天气让它保持简单，我们喜欢它。检查一下，看一看，试着用它作为你的主页一个星期。你再也不想使用另一个天气网站了。
