# Splashcast 启动一个玩家来绑定他们所有人

经过漫长的测试期后，位于 T2 的,[splash cast,](https://web.archive.org/web/20220816065202/http://www.beta.techcrunch.com/2006/11/06/splashcast-aims-to-offer-a-frictionless-web-media-player/)于今天早上发布。Marshall Kirkpatrick 在演示了该产品后，于去年年底写了一篇很长的评论(他非常喜欢它，现在在那里工作)。

Splashcast 有点难以描述，但是一旦你明白了，它就有意义了。这是一个 Flash 媒体播放器，用户可以制作各种频道的内容，包括文本、视频、图片和音频文件，然后将包含这些频道的播放器嵌入到网站上。它基本上整合了许多其他可嵌入产品的功能。YouTube 视频、照片幻灯片展示，如由 [Slide](https://web.archive.org/web/20220816065202/http://www.beta.techcrunch.com/2005/09/28/speaking-of-loops-slide/) 提供的，播客馈送等。都可以作为频道添加，并且将占用所有其他服务所需的屏幕空间的一小部分。对于已经很忙的网站来说，Splashcast 播放器比来自不同公司的 3-4 个插件更容易接受。

每个频道的内容都有一个 RSS 提要，因此感兴趣的观众可以订阅它，而不必回到您的网站进行访问。

Splashcast 将直接托管内容，但真正的效用来自于插入来自 Youtube 的视频、来自 Flickr 的照片等。，这样您就可以将媒体保存在您想要的位置，而不必将其复制到 Splashcast。你甚至可以像马歇尔在下面所做的那样，给任何一个节目片段添加画外音。

Splashcast 最初名为 QMind，是一家企业电子学习软件的创建者，已经从天使那里筹集了 130 万美元。他们去年夏天重新推出了 Splashcast，目前正在进行首轮融资。
