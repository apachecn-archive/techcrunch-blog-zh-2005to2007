# Zune 2:更薄，更无线

许多人对 Zune 感到失望，它是微软的无线网络，而不是 iPod。最令人失望的似乎是车载无线设备的低效利用。没有太多其他 Zune 可以分享，即使有，无线也很可能关闭以节省电池电量。即使那样，谁会想要一首只有三天的歌呢？

所有这些问题似乎都将在最早于今年春天发布的 Zune 2 中得到解决。传言称，这将是一款基于闪存的播放器，尺寸更小，WiFi 功能更丰富，包括空中下载。

引发这些谣言的不是别人，正是克里斯·斯蒂芬森，他在戛纳音乐博览会上证实，雷德蒙正在开发一种更薄的基于 Flash 的 Zune，可以从赞助的零售热点下载音乐。这意味着当你在星巴克等拿铁咖啡的时候，你可以下载新的 Jay-Z(他要退休了，朋友)。所说的 mini-Zune 的其他细节是什么？我们不确定，但你知道我们会找到答案的，所以把它放在这里作为你所有的 Zunage 信息。
