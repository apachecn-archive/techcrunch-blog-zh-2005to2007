# 玩具探索为孩子们发布 PMP/PVR

![](img/7317c4fdff9e64a811978b200b7cfa3e.png)

几周前，我在一个老虎电子帐篷前停下来，这个帐篷是用来展示这个假期所有的新玩具的。它们都有一个共同的主题:“成人拥有而孩子想要的蹩脚的小玩意。”产品范围从蹩脚的投影仪到蹩脚的视频 MP3 播放器。然而，ToyQuest 似乎意识到，孩子们应该得到和他们的大人一样的品质。

PMP/PVR 的 RipRoar Video Free 拥有 2.5 英寸的彩色屏幕，5 小时的视频播放，512MB 的内存，一个 SD 端口，一个内置扬声器和一个 USB 端口。它也可以从电视上录制。又一个很酷的功能？你可以把红外发射器连接到它上面，这样它就会改变频道来录制一些东西。价格？$199.对于成年人来说，这是一笔足够好的交易。
