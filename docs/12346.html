扔掉TechCrunch，赢取Web 2.0博览会的免费入场券

原文：https://web.archive.org/web/http://www.techcrunch.com:80/2007/04/04/trash-techcrunch-and-win-a-free-pass-to-the-web-20-expo/

我们拿到了三张4月15日至18日在三藩市举行的 Web 2.0博览会的免费入场券，每张价值1500美元。我们用这些通行证换取了我们网站上多余的广告库存，我们的计划是把它们送给读者。

我想在分发这些东西时找点乐子。去年，我们向最有理由去SXSW 的读者免费赠送了一张门票，但结果证明这是一场争夺最大催人泪下的故事的大规模竞争。令人沮丧。

因此，对于Web 2.0博览会，我们将做一些稍微不同的事情。我要你告诉我我们(偶尔)有多烂。事后来看，有时我们的预测会相差甚远。或者他们一开始就没有逻辑基础。或者我们弄错了一些重要的事实。不管是什么，我希望你找出TechCrunch历史上最差的帖子，并写下它为什么如此糟糕。一个好的起点是我们的公司指数。

规则如下:你可以攻击任何发表在TechCrunch上的帖子(而不是mobile runch、 CrunchGear 或我们的任何其他网站)。但你不能亲自攻击编剧，除非是我(迈克尔·阿灵顿)。原创性、创造性和幽默将被加分，但公然和无根据的刻薄和/或人身攻击将被扣分。在我看来，最好的参赛作品将是那些发现我们对一家初创公司或细分市场的未来发表了片面看法的作品，事实证明这是完全错误的。

要想加入，你需要在你的博客上写下你的文章，或者成功地追踪到这篇文章，或者在评论中添加一个你所写内容的链接。如果你没有博客，我建议你在 WordPress 、 Vox 或 Blogger 上创建一个。设置只需要一分钟，希望这将是你写的许多有趣的博客帖子的第一篇。

我们将选出前十名左右的作品，然后让读者投票选出三名获胜者。参赛作品必须在太平洋标准时间4月7日(星期六)中午之前写好并添加到下面的引用通告或评论中(届时评论和引用通告将被关闭)。投票将于4月9日星期一开始，持续48小时。

虽然这主要是为了好玩，但我会仔细阅读每一篇文章，并尽可能多地提出建设性的批评。这将是成为一名更好的作家的痛苦但有用的方法。
