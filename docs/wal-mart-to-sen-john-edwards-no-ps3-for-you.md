# 沃尔玛对参议员约翰·爱德华兹说:没有 PS3 给你！

这很有趣:在沃尔玛今天的一份声明中，前参议员约翰·爱德华兹让一名员工联系北卡罗来纳州罗利一家商店的电子产品经理，试图得到一台 PlayStation 3。问题是，那天晚上晚些时候，据报道，他在联合食品和商业工人(UFCW)工会发起的电话会议上向人们讲述了一个故事，他的儿子嘲笑了一个同学，因为他的鞋子是在沃尔玛买的。

听到这个消息后，该公司邀请爱德华兹参观当地的商店，查看其电子产品以及男士和男孩的鞋子。然后他们基本上说他可以和其他人保持一致。

> 该公司指出，PlayStation3 是今年圣诞节期间非常受欢迎的产品，当其他美国工薪家庭耐心排队等候时，参议员爱德华兹想插队。虽然我们不能保证爱德华兹会是第一批获得 [PlayStation3、](https://web.archive.org/web/20201020121417/https://crunchbase.com/product/playstation) 的人之一，但我们肯定爱德华兹能够为他圣诞清单上的每个人找到很棒的礼物——许多礼物的价格都是沃尔玛的“回滚价”

我不得不把它交给沃尔玛。他们真好，基本上就这样打击了爱德华兹。
