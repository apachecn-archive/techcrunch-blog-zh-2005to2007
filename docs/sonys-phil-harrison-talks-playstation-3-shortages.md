# 索尼的菲尔·哈里森谈论 PlayStation 3 短缺

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/06/sonys-phil-harrison-talks-playstation-3-shortages/>

![](img/96a139643d4455f3f15fb049290d7f1a.png)

为了安抚欧洲游戏市场，索尼的菲尔·哈里森就 PlayStation 3 在欧洲的延迟发布了一份声明，称索尼实际上试图通过在其新主机中实施蓝光来做得太多。据索尼称，蓝光二极管的制造是 PS3 在全球范围内短缺的原因，PS3 推迟到其他国家。

> 我不能否认，我们在蓝光组件的生产上走得太远了。但这是你采用全新、前沿技术的代价，这些技术将经得起未来考验。我们将解决这些问题——我们已经在迎头赶上。
> 
> 我们将继续追赶产量，如你所知，我们没有改变全年 600 万台的预测[到 2007 年 3 月]，所以我们只是在谈论一个扩大生产的问题。我们不是在谈论产品本身的基本设计，”他补充道。不可否认，我们遇到了一些非常公开的挑战。今天，这看起来是一个非常困难的局面——但在未来的几周、几个月和几年里，这种局面将会消失。

我不确定哈里森所说的“未来证明”是什么意思，因为在科技行业，真的没有这种东西。Harrison 还谈到了在 PS3 的低端 20GB 型号中实施 HDMI，指出这是遵循显示器制造商的市场趋势，他们正在销售更多支持 HDMI 的显示器。索尼是时候对欧洲市场说些什么了，因为它已经得到了所有的负面宣传，它需要停止表现得好像它不在乎它的消费者。
