# Powerset TechCrunch 上的 PowerHype

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/12/powerhype-at-powerset/>

[![](img/9ed202c53f835bfbb78619f29c06a676.png)](https://web.archive.org/web/20230220132743/http://www.powerset.com/) 我对这家新成立的搜索引擎公司 [Powerset](https://web.archive.org/web/20230220132743/http://crunchbase.com/company/powerset) 了解得越多，我就越怀疑它是否只不过是一座纸牌屋。他们是一家有点人格分裂的奇怪公司。例如，在某些方面，他们是非常保密的——每一个看到“演示”的人都必须签署一份保密协议。创业公司很少会有这种要求。然而，另一方面，首席执行官巴尼·佩尔(Barney Pell)大肆宣传该产品(这仍只是演示，根据我们听到的消息，Powerset 可能会在 2007 年上市，也可能不会上市)，他们已经筹集了 1250 万美元的风险投资，并从雅虎聘请了十几名搜索专家(雅虎拥有期权，搜索专家渴望下一件大事)。

所有这些大肆宣传引起了媒体的广泛关注。通常即使是左撇子的马特·马歇尔每次写作时都会滔滔不绝地谈论它们(他有机会看了演示)。《纽约时报》今年已经写了两次关于他们的文章。

这的确是个好故事。人们对搜索的现状感到沮丧，想要更好的东西。谷歌最初推出时也曾被批评试图解决一个太难解决的问题。人们将 Powerset 比作 Powerset，或许认为每十年就会有一家新公司推出并消灭旧的、根深蒂固的竞争对手。

尽管如此，Powerset 很少有机会被评判这类事情的搜索引擎精英所利用。

看过演示的人说这是一个完全受控的环境。该指数仅限于像纽约时报网站这样的高质量网站的非常小的样本，搜索查询也是由 Powerset 员工驱动的。有了这种设置，几乎任何人都可以展示令人惊叹的演示。我采访过的看过这个演示的真正的专家说，它很少或根本没有展示真实的东西。其他人留下深刻印象。

该公司上周末在旧金山举行了一场派对，庆祝他们的风险投资。莎拉·迈耶斯经常带着摄影师参加派对，拍摄派对上发生的事情(包括我们的事情)，她在那里与佩尔和其他员工交谈。视频嵌在下面。佩尔对产品进行了高水平的介绍，并没有说太多我们已经听到的内容。然而，在大约 2 点 47 分的时候，一些可怜的人被摄像头捕捉到，并回答了 Meyer 关于 Powerset 到底是什么的问题。他的回答是:“我们有一个演示，你可以，比如，搜索网页并得到结果，比如，儿童书籍与儿童书籍的对比…这就是我们声称我们可以做到的…我们将在一年内看到。”不完全是自信诱导的东西。

 **雅虎角度** 所有这些都说不通。该公司距离上市还太远，不可能大肆宣传其产品和举办派对。但据大家所说，佩尔是一位杰出的企业家。他到底想干什么？

我想唐·道奇已经搞定了。Powerset 承诺雅虎需要什么，他们正在与前雅虎员工一起建立它。Powerset 现在可能正在大肆宣传该产品，因为他们想尽快卖出去。道奇说:

> Powerset 应该怎么做？没有简单明了的答案……它们只是在 10 年后才显得简单。Powerset 的优势在于自然语言处理，或者理解单词的含义和上下文。许多像在杂志、教科书或报纸文章中发现的词。与其将所有这些 NLP 能力集中在理解典型的 2 或 3 个词的搜索查询上，为什么不帮助广告商更好地将他们的广告瞄准非结构化内容呢？
> 
> 雅虎的问题是什么？无目标的交通。雅虎的主页和雅虎邮箱有大量的流量，但却没有有效的定向广告。AOL、MSN 和其他所有门户网站都有同样的问题。因此，他们最终都销售低成本、低利润的 CPM 广告，而不是像谷歌那样的高利润每次点击付费(PPC)广告。如果 Powerset 技术可以用来“理解”电子邮件信息中的上下文和含义，他们可以有效地定向广告…并使门户网站的收入增加两倍。如果 Powerset 可以扫描门户网站的动态主页，或者每个用户的个性化主页，他们就可以更好地定位广告。

如果道奇是正确的，佩尔是在玩一个非常高赌注的扑克游戏。如果他赢了，他们会很快被收购，所有的雅虎人都会回到雅虎。如果他输了，他的产品最好能在上市后变得非常好。因为很多人会关注这种炒作是否合理。如果不是这样，Powerset 将很难得到任何媒体的关注。
