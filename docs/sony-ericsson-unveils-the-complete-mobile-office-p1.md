# 索尼爱立信推出完整的移动办公室，P1 

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/08/sony-ericsson-unveils-the-complete-mobile-office-p1/>

[![sony_ericsson_p1_1.jpg](img/0f3d6f59d3c39926c8caeabe811b009d.png)](https://web.archive.org/web/20160421182512/http://old.crunchgear.com/wp-content/uploads/sony_ericsson_p1_1.jpg "sony_ericsson_p1_1.jpg")

索尼爱立信今天发布了 P1，希望它能让智能手机先锋重回巅峰。SE 将 P1 吹捧为一个完整的移动办公室，具有推送电子邮件、网络浏览应用、GPRS、UMTS 和 WiFi(802.11b)等功能。P1 用塞班操作系统和 UIQ 3.0 来组合拳。

这些功能大部分都很棒，但是是什么让它成为一个完整的移动办公室呢？首先，有一个双功能字母数字 QWERTY 键盘(boo)，320 万像素的相机和名片扫描仪，以及 Blackberry Connect 和 Exchange ActiveSync 电子邮件支持。它还包括 VoIP 支持、手写识别、视频通话、VPN 和 Opera 8.0。通常的标准功能，如蓝牙，调频收音机，视频录制，日历和录音机都包括在内，所以没有必要为这些小事烦恼。第三季度，部分市场的 P1 下降。
