# 费城终于有了无线网络| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/23/philadelphia-finally-getting-wifi-network/>

![](img/b1328ab547651c97ed7345c7694d6d56.png)

刚刚听到一些关于 NPR 的好消息。建立费城全市 WiFi 项目的公司已经完成了城市北部 15 英里宽的测试，据说该系统已经准备好部署到整个费城。到今年第三季度，该网络将建成，这样费城市区的任何人都可以以低廉的价格获得无线互联网接入。

价格还没有确定，但是去年，10 美元一台的电脑到处都是。我也听说过 15.95 美元，尽管最终宽带质量的无线网络接入会低于 20 美元。这是费城的主场。已经完成这个 WiFi 项目。
