# 索尼回应 PS3 更新传闻

 FCC 备案。在制造商发给 Kotaku 的一份声明中，索尼表示:

> 正如我们在发布 PS3 产品时提到的那样，该系统将有不同的配置(因此，60GB 硬盘带存储卡插槽和 Wi-Fi，20GB 硬盘不带存储卡插槽或 Wi-Fi)。考虑到各种可能性，已经向 FCC 提出了申请，但是，目前还没有发布新产品。

没有人否认 PS3 的更新。我甚至可以说，这封回信甚至比预期的还要早。但可以肯定地说，80GB 型号没有超出存储容量的扩展。哦好吧。
