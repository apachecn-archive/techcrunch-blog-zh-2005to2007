# TailRank 2.0 发布，挑战 TechMeme

 [](https://web.archive.org/web/20201128054008/http://www.tailrank.com/) [TailRank](https://web.archive.org/web/20201128054008/http://www.tailrank.com/) 增加了一堆[新功能](https://web.archive.org/web/20201128054008/http://blog.tailrank.com/2006/10/tailrank_20_is_.html)今天在一个重大的服务中重新推出。如果你不熟悉 TailRank，[看看这篇文章](https://web.archive.org/web/20201128054008/http://www.beta.techcrunch.com/2006/02/04/a-look-at-the-memeorandum-killers/)，它也讨论了它的竞争对手。基本的想法是，它跟踪博客写了什么，谁链接了什么，以确定当天的主要讨论是关于什么的。只要浏览一下网站，读者就会看到一些博客开始写的重大新闻。

这个领域无可争议的领导者是 TechMeme，大多数博客每天都要查看好几次。我喜欢 TailRank 的一点是，它有一个简单的搜索功能，TechMeme 莫名其妙地忽略了这一点。这意味着如果你正在寻找昨天或上周的主要博客新闻，TailRank 是搜索的好地方。我还喜欢 Tailrank 索引了 150，000 个“主要”博客，而 TechMeme 只跟踪最大的几千个博客。这是史蒂夫鲁贝尔和其他人偶尔会讨论的问题，称 TechMeme 是一个拥有专属会员资格的“[博客乡村俱乐部](https://web.archive.org/web/20201128054008/http://www.micropersuasion.com/2006/10/techmeme_is_a_b.html)”。
