**是什么？**

Technorati 是 Web 2.0 的“老派”:最初的(也是最好的)实时搜索引擎之一。它要求客户反馈，并使用它来启动一个广泛的重新设计他们的网站作为一个公共测试版。最初的网站仍然在 www.technorati.com[和测试版，现在，在 beta.technorati.com](https://web.archive.org/web/20230315095318/http://www.technorati.com/)。Technorati 声称每天索引 800，000 +的新帖子，这与竞争对手对博客世界规模的估计一致。

Technorati 通过在其引擎中允许标签(或类别)搜索，帮助增加了利用博客站点元数据的流行度。通过标签搜索，用户可以在某些类别下找到出版商特别标记的内容。

Technorati 100 是网上流行博客的权威列表。

新的用户界面有几个关键的升级:

–简化的界面
–标签搜索的 RSS 源
–标签搜索返回索引结果以及 flickr、furl、delicious 和 buzznet
–更多主页个性化—包括观察列表、声明的博客和个人资料信息
–新的观察列表功能
