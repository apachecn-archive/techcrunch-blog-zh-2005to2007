# Xbox 360 精英拆开，不与黑客工具

[![llammaelite.jpg](img/322a75e85cd529b2c31397954b34ea4d.png)](https://web.archive.org/web/20201025203136/http://old.crunchgear.com/wp-content/uploads/llammaelite.jpg "llammaelite.jpg")

llamma.com 的侦探们得到了一台 [Xbox 360 Elite](https://web.archive.org/web/20201025203136/http://crunchgear.com/2007/04/17/special-trade-in-deals-for-the-xbox-360-elite-confirmed/) ，并好心地为我们打开了它。虽然 CPU 和 GPU 似乎与常规的 360 不同，但 llamma 不知道它们是否是传言中的 65 纳米设计。(65 纳米芯片应该有助于降低制造成本，最终导致价格更低的 360。)另外值得注意的是，硬盘与目前可用的黑客工具不兼容，如[硬盘黑客](https://web.archive.org/web/20201025203136/http://xbox360.qj.net/The-Specialist-s-HDDHacker-v0-5B/pg/49/aid/80613)。这意味着精英们暂时不能使用自己的 SATA 硬盘。
