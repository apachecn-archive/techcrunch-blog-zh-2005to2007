# 诺基亚 5300 XpressMusic 评论

![](img/5a620cda1de73e8b3788d158d657bf97.png)

音乐手机市场每月都在增长。对整合和组合的需求给索尼和诺基亚等制造商增加了压力，要求他们将 MP3 播放器的功能整合到手机中。诺基亚的最新作品，T-Mobile 的 5300 XpressMusic 是一项勇敢的努力，它既有缺点也有优点。如果你正在考虑把 iPod 留在家里，把你的手机当作音乐播放器使用，请继续阅读，看看是否值得说再见。
 就设计而言，诺基亚 5300 XpressMusic 注重的是功能，而不是风格。这是市面上最厚的滑盖手机之一，放在我的牛仔裤口袋里感觉真的很大。然而，如果你穿着一件有内口袋的外套或夹克，那么你不会介意带着它到处走。5300 XpressMusic 具有专用的音乐控制按钮、音量控制和一个摄像头按钮，所有这些都位于显示屏的两侧。还有一个游戏手柄，中央按钮，4 个附加按钮，以及一个当你垂直滑动手机时的数字小键盘。

就我个人而言，我并不喜欢塞班 60 操作系统，诺基亚大部分时间都对它着迷。它非常笨重，很难使用。谢天谢地，这手机没用。诺基亚的用户界面非常友好、简洁，对于 VCAST 和摩托罗拉菜单的世界来说，这是一个受欢迎的变化。它的反应非常灵敏，而且在命令之间从来没有滞后过。诺基亚的音乐播放器软件运行良好，尽管没有什么特别之处。只需拖放您的文件，您就可以开始了。
![](img/212ac561025a3fce6687cd3af552ca49.png)
在 5300 XpressMusic 上与音乐相关的功能中，还有其他几项资产使这款手机值得购买。那些经常使用 AIM 或其他即时通讯工具的人会对这款手机的全功能即时通讯客户端感到满意。那个“手机好友”列表？删除它。这次你不需要它了。附带的 130 万像素摄像头比周围的大多数摄像头(RAZR、Sidekick 3)都要好，只要你有良好的照明，就可以拍出适合网络观看的像样的照片。

但是招待会怎么样？诺基亚一向以信号强、接收清晰著称，5300 XpressMusic 也不例外。电话听起来很悦耳，你不会发现自己在说“喂，喂？”反反复复。也不用担心这个手机会没电。待机时间带开/关通话时间让我三天半不充电。
![](img/0ed923953efb32771f74a709d3a3249b.png)
随着诺基亚重回 T-Mobile 阵容，是时候换人了吗？如果不是为了额外的功能，那些需要更好的 Mo 接收效果的人绝对应该考虑 5300。此外，随着目前的回扣 schpiel 目前正在进行，你可以用不到 100 美元的价格买到它。还不错。然而，由于音乐存储仅限于 microSD 卡，你可能想继续前进，并保持 iPod 在手边。
