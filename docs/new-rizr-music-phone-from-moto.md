# Moto 新推出的 RIZR 音乐手机

移开 RAZR，不要让它变得更滑。Z6 MOTORIZR 是一款以音乐为中心的手机，配有 microSD、立体声蓝牙和 220 万像素的拍照器，设计与疲惫的 RAZR 非常相似。真正酷的是，Z6 运行的不是 MOTO 的 RAZR 表兄弟通常运行的标准 MOTO slim phone piece-o-crap 操作系统，而是我们一直听说的尚未看到的基于 Linux 的操作系统。希望我们以后能得到更多的细节。
