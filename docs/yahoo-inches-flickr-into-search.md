# 雅虎！将 Flickr 推入搜索

 [](https://web.archive.org/web/20201028181850/http://yahoo.com/) 显然是第一次，来自 [Flickr](https://web.archive.org/web/20201028181850/http://flickr.com/) 的有限照片现在出现在 [Yahoo！](https://web.archive.org/web/20201028181850/http://yahoo.com/)搜索结果。用户驱动的搜索索引是 Flickr 一直令人兴奋的部分原因，我知道这是我第一次去寻找植物图片(试着用拉丁名称搜索)、地点和事件的地方。

丁玲贤在 [Yahoo！搜索博客](https://web.archive.org/web/20201028181850/http://www.ysearchblog.com/archives/000350.html)今天，这种整合开始于搜索像[有趣的照片](https://web.archive.org/web/20201028181850/http://search.yahoo.com/search?p=funny+photos)、[有趣的照片](https://web.archive.org/web/20201028181850/http://search.yahoo.com/search?p=interesting+photos)、旅行和黑白照片。这是一个非常有限的试验(这些实际上是唯一能够返回 Flickr 结果的搜索查询)，但该公司可能会及时扩展更多搜索结果的集成。

正如我昨晚在初创公司 [Pixsy](https://web.archive.org/web/20201028181850/http://pixsy.com/) 的简介中所写的[，图片搜索是搜索世界中迅速扩张的一部分。让图片搜索保持最新也是一项艰巨的任务。在 Flickr，雅虎！拥有一个由时尚摄影师组成的用户群，他们不断实时上传和标记照片。事情会变得非常有趣，如果雅虎！开始将最近的照片放入新闻搜索，尽管这可能说起来容易做起来难。](https://web.archive.org/web/20201028181850/http://www.beta.techcrunch.com/2006/08/23/pixsy-to-monetize-visual-metasearch/)

值得注意的是，雅虎和苹果都没有。谷歌也没有在他们的图片搜索页面上投放广告(Ask.com 和 Pixsy 都有)，所以任何可以在有广告的页面上利用高质量图片的方式都是有意义的。

雅虎！照片网站比 Flickr 大得多，但是它的用户[刚刚被引入与世界分享他们的照片的想法](https://web.archive.org/web/20201028181850/http://www.beta.techcrunch.com/2006/08/17/yahoo-photos-to-leave-beta-today-api-finally-coming/)。Flickr 用户对此习以为常，但也可能发布更尖锐的照片。据《T4》报道，截至去年，雅虎有 15 亿张图片被编入索引显然是指望 Flickr 的质量而不是数量。传统的图片搜索通常会出现大量与你的搜索词没有太大关系的低质量照片——而 Flickr 却不是这样。
