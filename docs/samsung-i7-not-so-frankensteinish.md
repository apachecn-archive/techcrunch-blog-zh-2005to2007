# 三星 i7 不那么科学

三星的 i7 可能不像它的哥哥 T2 的 i70 那样古怪，但它确实包括一些漂亮的小玩意。三星很好地融合了 MP3 播放器、旋转的 3 英寸 LCD 屏幕和模拟 SRS 3D 声音。规格？720 万像素 3 倍光学变焦三星 NV 镜头，先进的防抖技术，ISO 1600 设置，能够录制 SVGA 质量的 MPEG-4 视频。
