# Windows Longhorn Server Beta 3 可用

![longhorn.jpg](img/1771c07fca9602157f12856f7308fb1c.png)想要一些下一代服务器技术，IT 呆子？为什么不去[这个 32 位](https://web.archive.org/web/20130628152309/http://download.microsoft.com/download/d/d/2/dd2c8e3b-0263-4d2f-ace2-9c5f93a30dd9/wsl_6001.16510.070417-1740_x86fre_server-KB3SFRE_EN_DVD.iso)或者这个 [64 位](https://web.archive.org/web/20130628152309/http://download.microsoft.com/download/d/d/2/dd2c8e3b-0263-4d2f-ace2-9c5f93a30dd9/wsl_6001.16510.070417-1740_x64fre_server-KB3SxFRE_EN_DVD.iso) ISO 下载网站，为自己服务一些 Longhorn。

自从我的合法 IT 时代以来，我就没有关注过微软服务器 warez，所以我不太确定是否有人特别感兴趣，但这里有一些改进:

> *集成 IIS 7.0 Web 服务器的“上线”许可证:IIS 7 上线许可证允许用户托管 Web 应用程序和。Longhorn Beta 3 上的. NET 3.0 Web 服务。
> *将 Windows PowerShell 脚本环境集成到 Longhorn Server 中。
> * Active Directory 联合身份验证服务的改进，使用户能够实施新的策略，并更轻松地在可信任的合作伙伴之间建立关系。
> *支持八个服务器核心角色(而不是之前宣布的四个)。
> *额外的远程管理工具可提供更加集成的管理环境。*默认情况下，Windows 防火墙现在是打开的。
> *网络访问保护(NAP)与 Microsoft Update 和 Windows Update 相集成，允许管理员决定哪些更新是关键的，并相应地设置策略。

需要八个服务器核心角色？你当然知道。我们都是，真的。
