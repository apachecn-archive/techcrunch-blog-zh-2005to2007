# 雅虎发布即时搜索

雅虎[昨晚宣布](https://web.archive.org/web/20221006203257/http://www.ysearchblog.com/archives/000186.html) [即时搜索](https://web.archive.org/web/20221006203257/http://instant.search.yahoo.com/)(更多信息在此[登陆页面](https://web.archive.org/web/20221006203257/http://next.yahoo.com/))。

像 [Google Suggest](https://web.archive.org/web/20221006203257/http://www.google.com/webhp?complete=1&hl=en) 和 [Snap](https://web.archive.org/web/20221006203257/http://www.snap.com/) 一样，雅虎也在搜索中加入了 ajax 功能。

不过，这不一样。Google Suggest 和 Snap 使用 ajax 根据你目前输入的内容来建议搜索查询，而 Yahoo 已经将基本搜索结果集成到界面中。这意味着你可以在不刷新页面的情况下获得简单而流行的搜索(比如天气)的答案。

 [试出来](https://web.archive.org/web/20221006203257/http://instant.search.yahoo.com/)。雅虎建议查询诸如“波士顿天气”和“维基百科”。这种类型的搜索会给出很好的结果。

雅虎对谷歌进行了小小的抨击，说“当你可能是对的时候，为什么感到幸运”，这是指谷歌的[感觉幸运](https://web.archive.org/web/20221006203257/http://www.google.com/help/features.html#lucky)按钮。约翰·巴特利说“哎呀，爪子伸出来了……”:-)
