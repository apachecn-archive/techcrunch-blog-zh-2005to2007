# 谣言:EB Games 员工可能因 Ebay 拍卖被解雇

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/25/rumor-eb-games-may-get-fired-over-ebay-auctions/>

![](img/b10455123d4f1a9d688a3a5b3ae1a0d7.png)

据说 EB Games/Gamestop 正在密切关注所有拥有任天堂 Wii 或 PlayStation 3 游戏机的员工。显然，员工不允许在 Ebay 上出售他们通过公司收到的游戏机。我希望这是真的——如果知道员工在销售游戏机，美国游戏玩家将不得不在百思买前过夜才能买到游戏机，我会非常失望。

现在一切都只是传闻，但是，如果这是真的，EB Games/Gamestop 很可能会将 Ebay 上的序列号与他们给每个员工的游戏机序列号进行匹配。
