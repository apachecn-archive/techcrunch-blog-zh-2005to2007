# 威瑞森将提供 OTA 联系人同步

这个男人为什么在笑？因为威瑞森今天宣布将出售 Plaxo 的一项服务，该服务允许其无线用户通过无线方式自动同步其电脑和移动设备之间的联系人。

Java BREW 应用程序与目前大多数威瑞森手机兼容，每月售价 4.49 美元。虽然我们更愿意将此视为一次性的服务收费，但威瑞森和 Plaxo 似乎认为，其客户会大量更新他们的电话簿，而实际上，除非有新手机，否则用户不会这么做。但是，如果你是一个新客户或获得一部升级的手机，使用这项服务一个月可能不是一个坏主意，尽管它通常可以在任何一家零售店支付少量费用。它仍然不能阻止你酒后打电话给那个人。
