# VidZone 数字媒体获得索尼 BMG 英国公司的移动产品目录

贾斯汀·汀布莱克和碧昂斯进入了移动区……实际上是视频区。英国领先的移动网络音乐提供商 VidZone Digital Media 签署了一项协议，该协议将授权其完全访问索尼 BMG 音乐娱乐英国公司的数字目录，供移动设备和 PC 使用。该提供商将提供音频和视频格式的音乐流和下载，同时还提供铃声音乐。

VidZone Digital Media 向英国境内的 3、O2、T-Mobile、Virgin Mobile 和沃达丰以及国际上的另外 35 个移动网络和移动在线网站分发音乐内容。
