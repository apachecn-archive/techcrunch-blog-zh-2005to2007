# 战略卫星完成首次飞行测试

![](img/ca69148abb63203d6dddc4104b3a2de0.png)

GlobalTel 通信公司的一家子公司已经证实，它已经成功地驾驶了一艘飞艇，他们希望有一天能够将宽带互联网传送到下面的城市。这个想法已经被作为一种新的解决方案抛来抛去，为大众带来宽带，但现在看来这个想法终于起飞了(双关语)。

这次飞行只不过是一次简单的飞行测试，以确保它能够真正离开地面，但在接下来的 90 天里，GlobalTel 计划为飞艇添加不同的系统和功能。每次添加后，工程师将确保它仍然正常工作，最终，在不久的将来，我们应该有一个漂浮在主要城市上空 12-13 英里的飞艇。
