# OLPC 儿童机英寸更近

![](img/d941235f3e50c135a65d7b94a202b426.png)

一个孩子一台笔记本电脑倡议今天宣布，它已将其第一台“100 美元笔记本电脑”命名为儿童机，又名 CM1。每个单元将配备一个 400Mhz AMD Geode 处理器，一个 7.5 英寸 LCD 显示器，并将兼容网状网络。它还将包括 VoIP 和一个 SD 插槽。CM1 实际上用于完全偏远的地区，因为它实际上是由拉绳发生器提供动力的。

现在你知道了，OLPC 已经命名了他们的第一台 140 美元的笔记本电脑，还有一些来自公司的非连续新闻。我真诚地希望看到这个产品开花结果，但有时它似乎很有问题。信息几乎每周都在变化，它是否能投入生产常常令人怀疑——我真的不希望如此。
