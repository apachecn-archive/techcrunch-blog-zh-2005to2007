# TiVo 用户可以直接从盒子里获得亚马逊电影

从今天开始，TiVo 用户将可以通过他们的机顶盒从亚马逊购买和下载电影。

自 3 月起，TiVo 用户可以从亚马逊 Unbox [购买或租赁电影。以前，这需要用户在亚马逊网站上购物。从今天起，TiVo 系列 2 和 3 的用户将可以不用电脑就能浏览和购买电影。](https://web.archive.org/web/20220619053743/http://www.beta.techcrunch.com/2007/03/08/amazon-to-the-tv-before-apple/)

纽约时报[指出【TiVo 已经开发了一种安全措施，通过要求一个五位数的密码来完成交易，这样用户就不会意外地订购电影](https://web.archive.org/web/20220619053743/http://www.nytimes.com/aponline/technology/AP-TiVo-Amazon.html?ex=1341720000&en=24ef875b426fc16b&ei=5088&partner=rssnyt&emc=rss)

这项服务与苹果电视形成对比，苹果电视需要使用 iTunes 来下载、购买内容，然后将内容同步到苹果电视上。苹果电视已经被一些人认为是苹果最近成功故事的一个失败；鉴于竞争产品的持续改进，这可能不是一个没有价值的电话。当然，有人会质疑是否需要一个独立的苹果电视风格的设备，既不能录制电视也不能播放 DVD，当有大量的替代品时，从 TiVo 风格的单元，HD DVD 播放器和网络 DVD 播放器，以类似或更低的成本提供更好的服务。
