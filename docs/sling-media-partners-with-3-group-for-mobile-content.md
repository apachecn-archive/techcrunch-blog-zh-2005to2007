# 与 3 家移动内容集团进行媒体合作

![](img/50b3d22190d29e03cc92342e207a0c97.png)

今天，Sling Media 宣布，它将与 3 Group 合作，作为帮助将电视引入更多移动手机的计划的一部分。目前，只有数量有限的手机和 PDA 可以运行 SlingPlayer 移动软件，该软件用于通过 Slingbox 从家中播放电视节目。现已增加对诺基亚 N73 和索尼爱立信 W950 手机的支持。这种合作将使更多的消费者能够在世界任何地方观看他们的本地新闻和色情频道。
