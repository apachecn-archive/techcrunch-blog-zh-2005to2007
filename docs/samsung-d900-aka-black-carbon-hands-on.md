# 三星 D900 又名黑炭动手

![](img/64cf2fb4003823d68e9a4749cc594571.png)

是另一款 Ultra 机型–D900 是一款超薄滑盖，配有 300 万像素摄像头和与 Adobe 联合设计的独特用户界面。三星的人没有提到时间表，但它应该很快就会登陆我们的海岸，价格约为 350-400 美元。
 ![](img/8e9bdfa66005dee634ec8a6961530601.png)像 Trace 一样，D900 非常薄，设计有高速互联网和一些独特的功能，包括 uGo，它可以根据您漫游的国家将背景图像更改为当地的地标(显然在欧洲效果更好)，以及 uTrace，如果有人窃取了您的手机并放入他们自己的 SIM 卡，它会悄悄地向您发送短信。此型号仅适用于 GSM，并且未锁定。

![](img/1be94a4f6b07c0b3e24e3f47b37d0db2.png)

出于一个小小的原因，UI 非常有趣。菜单有“弹出”部分，所以当你选择某样东西时，会出现一个小窗口，让你不用离开原来的屏幕就可以向下钻取。这节省了几次点击，比标准的“卡片组”用户界面友好得多，标准的“卡片组”用户界面需要你在两个独立的菜单之间来回切换。为了更好地理解其中的区别，想象一下在 Windows 或 OS X 主菜单选项下的子菜单。总体感觉比较优雅，不那么杂乱。
![](img/f8de4705674307a1ddb8808c63facf7f.png)
包装遵循诺基亚的领导，创造一个完整的，独特的包装，而不是一般的盒子。这绝对是一部地位手机，尽管缺乏闪亮的材料可能会对它不利。它还将有多种颜色，包括红色和银色。对于三星来说，这是一个有趣的举动，类似于诺基亚的旗舰店模式，直接向消费者销售手机，几乎完全不通过运营商。如果你在韩国有什么感受的话，那就是我们在手机功能和手机方面严重落后。幸运的是，像这样的举措会让我们更接近 21 世纪。
