# SMC 用 802.11n 很酷，发布了一堆东西

![](img/a8acca49e79d703393d0cb99e93cce9f.png)

如果你想尽早赶上下一代潮流，你可能想去当地的电子商店看看 SMC Networks 的最新产品。SMC 发布了一堆 802.11n 好东西，仍然是 802.11n 的草案版本，包括从路由器到适配器的所有东西。售价 120 美元的 barrier N 无线路由器可以连接高达 300Mbps 的数据，背面有四个以太网端口，还有 SPI 防火墙提供保护。

当然，还有适配器。你可以选择:90 美元的 EZ Connect N PCI-card 或者 80 美元的 CardBus 版本。两者都可以让你实际使用可能来自该路由器的 300Mbps。
