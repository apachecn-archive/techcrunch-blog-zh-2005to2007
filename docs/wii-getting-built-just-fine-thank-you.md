# Wii 做得很好，谢谢

所以看起来 Wii 将会是最大的赢家。显然，任天堂有 200 万台 Wii 等待上市，预计 3 月份将有 600 万台，到 2007 年底将有 1100 万台。这意味着我们所有人，从最大的到最小的，都可以享受 Wii。

我从来没有真正想要一个 game cube——虽然我很喜欢在我的好友菲尔的地方玩它——但 Wii 似乎只是把心放在了正确的地方。任天堂回到了它的根本——以一个不错的价格创造一个引人注目的产品，然后看着钱滚滚而来。你想知道最糟糕的是什么吗？想想 Wii 这个名字。我们都很紧张，现在它只是脱口而出。那是为你营销。
