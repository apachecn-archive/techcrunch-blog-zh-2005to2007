# oDesk 宣布从 Benchmark | TechCrunch 获得 800 万美元资金

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/09/27/odesk-announces-8m-funding-from-benchmark/>

[![oDesk](img/ec68771ab49dbb7a0ab6ba1ff4dd7e9e.png)](https://web.archive.org/web/20150927063506/http://www.odesk.com/)

oDesk 是一家位于门洛帕克的公司，主要在编程服务领域为合同人才提供市场。我们第一次[写 oDesk](https://web.archive.org/web/20150927063506/http://www.techcrunch.com/tag/odesk) 是在几周前，从那以后，他们经历了强劲的增长，项目和供应商的数量都在增加。今天 oDesk 宣布了由基准资本牵头的 800 万美元的 B 轮投资，这是一家受人尊敬的硅谷公司，之前曾投资过易贝、Handspring、RedHat 和 Juniper 等公司。oDesk 之前的所有投资者也参加了这轮投资，Benchmark 的凯文·哈维将加入 oDesk 董事会。

在最近于三藩市举行的[未来网络应用大会](https://web.archive.org/web/20150927063506/http://www.futureofwebapps.com/)上，迈克尔·阿灵顿(Techcrunch 的编辑)将 oDesk 列为“值得关注”的公司之一。我们已经看到 oDesk 在一个以前由 eLance 和 Guru 主导的领域中强劲增长。在一个极度渴望高质量开发和其他 IT 服务的市场中，oDesk 提供了一种对旧模式的扭转。oDesk 允许其客户跟踪他们发布到市场上的项目和工作的各个方面，从寻找合适的人才到管理您与供应商的关系。它也有一个不同的经济模式，因为所有的工作都是按小时收费，而不是其他网站上的固定项目成本。

我们看好 oDesk 的前景，不仅因为对优秀供应商的需求不断上升，还因为他们连接和管理供应商与客户之间关系的方式。正如我们之前帖子中的评论所指出的，根据我们自己的经验，oDesk 似乎受到服务提供商和 it 客户的喜爱，双方似乎都坚持使用这项服务。随着 Benchmark 现在将筹码放在 oDesk 上，这项服务应该会更进一步，成为主导者。oDesk 表示，从 B 轮融资中筹集的资金将用于扩大营销活动和刺激增长，看来他们的平台是正确的，它对所有参与者都很有效，现在是时候打开闸门了。
