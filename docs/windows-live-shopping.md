# Windows Live 购物–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/03/17/windows-live-shopping/>

[](https://web.archive.org/web/20220131205949/http://shopping.live.com/)[Windows Live 购物](https://web.archive.org/web/20220131205949/http://shopping.live.com/)的域名现已上线，[微软的 Iam McAllister](https://web.archive.org/web/20220131205949/http://spaces.msn.com/ianmcallister/blog/cns!AFAD344C48A05587!770.entry) 指出，他是 MSN 购物平台的项目经理。目前还不知道这项服务将于何时推出，但一旦推出，它将成为零售商和买家的一个非常大的平台。

Windows Live 购物、Expo 和 MSN 购物是 MSN Marketplaces 的一部分。关于他们活动的更多信息[在这里](https://web.archive.org/web/20220131205949/http://shopping.live.com/jobs/)。
