# 苹果正在筹备一个完全开发的媒体中心？

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/12/apple-prepping-a-fully-developed-media-center/>

[![applemedia.jpg](img/9603fb649275a8c4bb53bf525a775b10.png)](https://web.archive.org/web/20150428211819/http://old.crunchgear.com/wp-content/uploads/applemedia.jpg "applemedia.jpg")

如果最近的专利申请是可信的，那么苹果正在研究一个实际的媒体中心解决方案，而不是一些半心半意的前排尝试。这项专利最初是在 2005 年底申请的，但今天才被披露，它描绘了一个似乎位于媒体中心心脏的中央“模块控制器”。显然，它是一个位于其他软件和硬件之间的软件，决定着什么东西何时何地去哪里。

你会看到很多关于这个媒体中心将能够做什么的猜测——比如与即将到来的 iPhone 无线接口——但有一点是相当肯定的:苹果不仅对你的[客厅](https://web.archive.org/web/20150428211819/http://crunchgear.com/2007/04/10/new-day-new-apple-tv-ad/)有设计，而且对你生活娱乐的所有方面都有设计。我想知道人们多久会像讨厌微软一样讨厌苹果？
