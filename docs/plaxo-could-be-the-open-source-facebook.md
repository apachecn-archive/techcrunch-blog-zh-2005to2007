# 普拉克索可能是开放的脸书

[![](img/221bef645ea780b69b95e3b4ba99febf.png) ](https://web.archive.org/web/20220816015241/http://www.crunchbase.com/company/plaxo) [Plaxo](https://web.archive.org/web/20220816015241/http://www.crunchbase.com/company/plaxo) 最近[发布了一个新的 3.0 版本](https://web.archive.org/web/20220816015241/http://www.beta.techcrunch.com/2007/06/24/new-version-of-plaxo-launched-more-sync-more-often/)，这不仅仅是一个 AJAX 换脸。Plaxo，[曾经是臭名昭著的垃圾邮件制造者](https://web.archive.org/web/20220816015241/http://www.beta.techcrunch.com/2006/03/22/plaxo-now-with-less-evil/)，现在已经超越了作为网络地址簿的核心服务。新版本不仅同步联系人信息，还同步你的信息订阅源，像 Flickr，日历，每天通过 Plaxo Pulse 添加新的订阅源。今天，Plaxo 加入了其他支持 OpenID 平台的站点的行列，因为他们更大的愿景正在形成，它正在成为一个中心而不是一个目录。

脸书多次谈到他们的“社交图”(计算机科学术语，指社交网络背后的数据结构)。这是他们平台的核心，提供了使他们的应用程序与用户相关的环境。唯一的问题是它仍然相对封闭。应用程序可以通过其 API 存在于脸书内部或外部。不管怎样，脸书仍然拥有用户。

Plaxo 不像脸书那样是一个“社交网络”,但它切中了“社交图”的核心 Plaxo 可以访问他们 1500 万用户的联系信息。这不仅包括电话号码和电子邮件，还包括构成社交图上链接的用户之间的联系。

Plaxo 并没有让这个“社交图”成为一个有围墙的花园。他们正在成为联系信息的瑞士，并一直专注于与新服务(雅虎，MSN，美国在线)的互动。OpenID 的加入降低了上图的摩擦。

虽然没有脸书那么激动人心，但 Plaxo 正在向他们的方向发展。Plaxo Pulse 将网络上不同的服务联系在一起，而不像 news feed 只将脸书的内容联系在一起。虽然 Plaxo 还没有向一群迫不及待的开发人员推出平台，但他们已经悄悄地专注于链接到网络上现有的应用程序。目前，为商务人士关心的社交订阅源、电子邮件、联系人和日历应用程序的同步提供了单一界面。不难看出，这将发展成为与更多 web 应用程序更深层次的集成。
