# 维基以 200 万美元收购 ArmchairGM

据知情人士透露，维基百科将于本周宣布收购 [ArmchairGM](https://web.archive.org/web/20221208031944/http://armchairgm.com/) 。该公司以 200 万美元现金和维基百科股票被收购。

几个月前，我们在一篇关于竞争对手的[帖子](https://web.archive.org/web/20221208031944/http://www.beta.techcrunch.com/2006/08/29/yardbarker-wants-to-be-sports-20/)中简要提到了 ArmchairGM。这项服务是由纽约的四个人创建的，有点像 digg-wiki-sports 的混合体。显然，这项技术很有趣，并且与 Wikia 兼容，我猜测这部分是一种收购后租用类型的交易。
