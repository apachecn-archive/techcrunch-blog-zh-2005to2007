# Netvibes 获得了 1500 万美元的投资

总部位于巴黎/伦敦的公司 NetVibes 将于周一宣布一轮 1500 万美元的融资。[现有投资者](https://web.archive.org/web/20220112011856/http://www.beta.techcrunch.com/2006/03/22/netvibes-to-announce-seed-financing-today/) Index Ventures 联合新投资者 Accel 牵头投资，这是今年欧洲公司最大的投资之一(英国公司 Bebo[筹集了类似金额](https://web.archive.org/web/20220112011856/http://www.beta.techcrunch.com/2006/05/22/bebo-closes-15m-financing/))。估值没有披露。

NetVibes 是一个个人的 ajax 主页，自从发布以来，我们已经介绍过[。Netvibes 取得了巨大的增长，甚至与 Pageflakes 等资金雄厚的竞争对手相比也是如此，Pageflakes 从 Benchmark](https://web.archive.org/web/20220112011856/http://www.beta.techcrunch.com/tag/Netvibes/)获得了大量投资。该网站现在有超过 500 万用户，并声称在营销上几乎没有花费。

他们的产品也一直在很好地发展，整合了他们的生态系统，并增加了 Meebo IM，注入了一些社区味道。

前一轮种子期于 3 月结束，投资者包括 Index Ventures、马克·安德森、Pierre Chappaz(kel koo 和 Wikio 的创始人)和 Martin Varsavsky(jazz tel、March 和 Fon 的创始人)。

我们知道这项投资将主要致力于建立一个业务和技术合作者团队(他们实际上已经开始[做](https://web.archive.org/web/20220112011856/http://freddymini.blogspot.com/2006/07/pourquoi-je-rejoins-netvibes.html)做[做](https://web.archive.org/web/20220112011856/http://hublog.typepad.com/hubert_michaux/netvibes/index.html))，同时也致力于升级他们的技术基础设施，改进他们的产品(最近有点慢，但是[也在升级](https://web.archive.org/web/20220112011856/http://blog.netvibes.com/?2006/08/05/74-maintenance-upgrade))和建立合作伙伴关系。

NetVibes 最艰难的挑战现在开始了。他们在一个年轻的市场中有足够的牵引力开始将流量货币化——他们只需要找到正确的方法，在不疏远用户的情况下做到这一点。NetVibes 也必须成为主流，这意味着专门的营销努力和相关的分销协议，以达到尚未开发的，非早期采用者的受众。

他们当然有足够的资源来实现他们的愿景，并将这些挑战提升到一个新的水平。NetVibes 会加入其他欧洲的世界级互联网公司吗，比如 [Skype](https://web.archive.org/web/20220112011856/http://www.beta.techcrunch.com/tag/skype) (也是一项指数投资) [Bebo](https://web.archive.org/web/20220112011856/http://www.beta.techcrunch.com/tag/bebo) 和 [Habbo](https://web.archive.org/web/20220112011856/http://www.beta.techcrunch.com/2006/07/10/habbo-hotel-microtransacts-itself-to-30m-in-revenue/) ？
