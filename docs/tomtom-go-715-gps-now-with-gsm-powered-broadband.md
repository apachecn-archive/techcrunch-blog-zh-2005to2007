# TomTom Go 715 全球定位系统:现在与 GSM 供电的宽带

法国顶级 GPS 制造商 TomTom 已经制造出 Go 715 导航装置。杀手级应用:SIM 卡插槽可以让你利用被称为互联网的数据传输媒介的力量来发送和接收数据。这可能很酷:地图和兴趣点的轻松更新是最明显的应用，但还有其他应用，如搜索和在路上阅读小工具博客。
