# NewsVine 推出 ElectionVine:基于 Widget 的选举投票

[http://i.newsvine.com/_vine/swf/electionwidget.swf](https://web.archive.org/web/20221001162622/http://i.newsvine.com/_vine/swf/electionwidget.swf)[election vine](https://web.archive.org/web/20221001162622/http://elections.newsvine.com/)是[新闻网](https://web.archive.org/web/20221001162622/http://www.beta.techcrunch.com/tag/newsvine)今天推出的一项新服务，允许用户在他们的网站上嵌入美国总统候选人投票。

NewsVine 将 ElectionVine 视为他们对主流媒体传统投票方法的回应，通过提供一个可以被美国各地的博客/网站所有者使用的分布式投票系统，彻底改变了传统的投票模式。任何人都可以在自己的网站上举行自己的选举，甚至支持某个候选人。特定网站的结果显示了每个网站的读者中有多少百分比是民主党人、共和党人或某个候选人的支持者。

该应用程序的优势来自于在分布式博客圈的眼中，选举是如何形成的聚合视图。距离初选还有几个月的时间，NewsVine 预计，在整个网络上，如果没有几十万个这样的独立投票站，也会有几万个。

有什么更好的方法来测试这个系统，然后把它嵌入在这个职位。通过参与有史以来第一次 TechCrunch 总统投票，让我们(匿名)知道你支持哪位候选人；右侧的轮询小部件。

总的来说，ElectionVine 是一个聪明的小工具实现，它一定会在美国博客中找到一个有欣赏力的观众。
