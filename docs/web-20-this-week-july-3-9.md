# 本周(7 月 3 日至 9 日)web 2.0 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/07/08/web-20-this-week-july-3-9/>

![](img/e1c00f08c486b283327e4e214db88b54.png)

本周，我们看到了一些有趣的 web 2.0 开发。然而，7 月 7 日对伦敦的恐怖袭击远远超过了其他任何事件。博客和互联网在向人们传递信息方面发挥了很大的作用，因为电视远远落后，手机网络堵塞。

**周总结:**
 **1。[伦敦恐怖袭击](https://web.archive.org/web/20230131031621/http://en.wikipedia.org/wiki/2005_London_transport_explosions)**

我正飞往欧洲参加一个会议，享受着我有史以来第一次在飞机上的 wifi 体验，这时来自伦敦朋友的即时消息开始传来。“有一个暴露在管，他们认为这是一个电力转换器”一个朋友说。然后，几分钟后，“天啊，到处都有爆炸的报道——可能是公共汽车炸弹。”很快，我就收到了乘务员和其他的传单，询问最新消息。飞行员派了一名空姐回来听正在发生的事情，因为她无法通过无线电获得新闻。我似乎是飞机上唯一的新闻来源。

英国广播公司的网站流量下降，人们转向博客获取新闻。[维基百科](https://web.archive.org/web/20230131031621/http://en.wikipedia.org/wiki/2005_London_transport_explosions)被引用为最新的信息来源。[其他人](https://web.archive.org/web/20230131031621/http://www.chrisnolan.com/archives/trevino/2005/07/the_bloody_seventh.html)写博客讲述他们的经历。

从大媒体与博客的角度来看，弗雷德·威尔逊完美地总结了这一点，他说*“这就是全部，也是为什么拥有 24×7 新闻频道的美国有线电视新闻网无可救药地过时了。”*(链接至[杰夫·贾维斯](https://web.archive.org/web/20230131031621/http://www.buzzmachine.com/archives/2005_07_07.html#010010))。

我们的心[继续向所有受影响的人伸出](https://web.archive.org/web/20230131031621/http://techcrunch.com/?p=73)。911 事件时我在曼哈顿。它需要几天的时间来适应现实。

**4。[Ian Davis—“Web 2.0 是一种态度而不是技术”](https://web.archive.org/web/20230131031621/http://internetalchemy.org/2005/07/talis-web-20-and-all-that)**

Ian 写了一篇精彩的文章，进一步阐述了 web 2.0 的定义:*“这是我的看法: **Web 2.0 是一种态度，而不是一种技术。**这是关于通过开放应用和服务来支持和鼓励参与。我说的开放是指技术上的开放，包括适当的 API，但更重要的是，社会上的开放，包括在新的激动人心的环境中使用内容的权利。当然，网络一直都是关于参与的，没有参与就什么都不是。这是一个最大的成就，网络超链接，从一开始就鼓励参与。不知何故，在 90 年代后期，网络失去了与它的根源的联系，自私的利益占据了主导地位。这就是为什么我认为 Web 2.0 标签是狡猾的:语义上它将我们链接回最初的网络和它所倡导的理想，但同时它也意味着新版本的再生。技术在不断发展，重要的是网络的社交面要跟上步伐。”*
