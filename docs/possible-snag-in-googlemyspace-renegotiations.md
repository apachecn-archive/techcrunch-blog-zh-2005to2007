# 谷歌/福克斯重新谈判可能遇到障碍

《华尔街日报》报道关于谷歌与福克斯的[9 亿美元广告交易的](https://web.archive.org/web/20210224174948/http://www.beta.techcrunch.com/2007/02/02/google-fox-renegotiating-and-extending-advertising-deal/)重新谈判可能遇到了阻碍。障碍可能是易贝。

> 据知情人士透露，几个月来，MySpace 一直在与易贝讨论他们在 MySpace 所谓的“对等商务”上的合作方式。这些人说，这个想法是让 MySpace 的用户利用易贝的在线商务技术和 PayPal 支付系统互相买卖商品。据一位知情人士透露，MySpace 用户将能够在个人资料上发布待售商品，他们的易贝拍卖将会自动更新……谷歌不太可能支持任何推广与其竞争的易贝服务的交易。在过去的 18 个月里，谷歌越来越多地侵占易贝的领土。谷歌增加了一个名为 Base 的列表服务，卖家可以在这里发布待售产品，还增加了一个名为 Checkout 的在线支付服务，与易贝的 PayPal 竞争。

我不会说 [Google Base](https://web.archive.org/web/20210224174948/http://www.beta.techcrunch.com/2005/11/15/google-base-launched-yuck/) 是易贝的竞争对手，但可以肯定的是 [Google Checkout](https://web.archive.org/web/20210224174948/http://www.beta.techcrunch.com/2007/01/18/googles-roundhouse-punch-to-paypal/) 正在努力挑战 PayPal。

和我们一样,《华尔街日报》迷恋谷歌和福克斯，经常报道他们的活动。不过，他们有时也会犯错。上个月,《华尔街日报》报道称，福克斯向鲁迈拉运营公司投资了 1200 万美元。事实证明，这是一笔纯粹的换股换绩效的交易，没有现金交易。向《华尔街日报》泄露这一信息的人可能有自己的目的，信息可能不完全准确。我们已经给福克斯发了一封电子邮件来征求意见。
