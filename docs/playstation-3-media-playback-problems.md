# PlayStation 3 媒体播放问题

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/11/playstation-3-media-playback-problems/>

![](img/6a9715cd0b83ad9e4f1ce92a85fb1247.png)

当我们其余的人不得不在寒冷中露营几天，以便下周得到一台 PlayStation 3 时，Gizmodo 的嬉皮士们得到了一台游戏机，并测试了它的媒体播放兼容性。照片方面，看起来 PS3 只能处理 JPG 的文件。在音乐方面，PS3 可以播放 ATRAC、MP3、AAC、MPEG-4 和索尼自己的音乐格式。

Gizmodo 在让 PS3 从 SD 或 CF 卡读取媒体时遇到了一些问题，PS3 在识别通过 USB 连接的拇指驱动器时也遇到了问题。问题可能已经解决了，已经有了从 1.0 到 1.1 的版本更新。由于游戏机要到下周才发布，我不认为任何人都可以下载更新。说真的，你们中有多少人会在全新的 PlayStation 3 上随意浏览媒体？
