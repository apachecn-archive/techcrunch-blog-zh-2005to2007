# 马科夫知道什么？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/08/what-does-markoff-know/>

刚刚看了约翰·马克夫的一篇文章，是关于“苹果”将于“明天”在“苹果世界”上发布的一款潜在的“移动电脑”他用如此神秘的方式表达出来，几乎让人觉得他知道些什么——不过话说回来。你说呢？

> 行业高管和竞争对手认为，苹果已经开发出第一代更接近个人电脑的袖珍新一代设备，这意味着它们将轻松处理音乐、娱乐、生产力任务以及蜂窝和其他无线网络上的通信。
