# 威瑞森的 VZ 航海家评论道:只要你不走路，它就能工作

我完全赞成利用你现有的技术，而不是为你想做的每件事购买一个单独的设备。威瑞森新近推出的每月 10 美元的 VZ 导航服务就是一个很好的例子。InfoSync World 对这项服务进行了彻底的测试，总体而言，由于威瑞森的 EV-DO 网络，该网站给出准确方向和快速更新的能力给人留下了深刻的印象，尤其是在郊区。它还表示，兴趣点工具提供了一些有趣的建议，包括当地博物馆和一些历史酒吧。

不利的一面是，该服务的步行模式毫无用处，基本上就像你在汽车里一样给出相同的方向，并告诉用户他们到达了他们的位置，即使他们仍然在一个街区之外。这是一个真正的耻辱，因为我会更有可能签署 VZ 导航，如果它有一个良好的步行模式。看起来我会坚持使用 [ALK 的 CoPilot Live](https://web.archive.org/web/20131013040809/http://www.alk.com/copilot/smartphone.asp) ，它非常适合在开车或步行时用我的手机找路。
