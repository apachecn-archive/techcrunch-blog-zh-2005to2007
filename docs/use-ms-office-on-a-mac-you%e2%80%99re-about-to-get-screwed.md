# 在 Mac 上使用 MS Office？你就要完蛋了

![](img/fd42020df5ad441958e87adff75433c3.png)

你可能知道也可能不知道，微软已经推出了 Office 2007。该套件是对以前版本的巨大改进，现在当你保存一个单词时使用新的文件格式。docx)、Excel(。xlsx)，或者 Powerpoint(。pptx)文档。这些被称为“微软 Office 开放 XML 格式”的新文件类型似乎无法在 Mac[OS X上运行。尝试在 Word 2004 中为 OS X 打开一个文档，会出现一堆随机字符和无法辨认的垃圾，如上图所示。](https://web.archive.org/web/20201123194851/https://crunchbase.com/product/os-x)

那些在 Windows XP 上使用以前版本的 Office 的用户可以下载一个转换器和兼容性包作为解决办法，但 Mac 用户似乎被冷落了。显然，没有人能说 OS X 版本的转换工具什么时候可以使用。是什么给了微软？要么 100%支持你的 OS X 用户，要么就别费心了。我们希望能够在 Office 文档之间进行协作，但是这种新的文件格式将会杀死我们！
