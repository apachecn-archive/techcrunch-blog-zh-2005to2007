# Thinkpad Reserve:伪装的

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/17/thinkpad-reserve-x61-in-disguise/>

![](img/fa64ffef4860969896271af68e960951.png)

最终，我对奶牛的厌恶和对科技的热爱汇聚到了一台皮面 Thinkpad 上。这个牛美人实际上是一个普通的 X61，带有一个皮夹克，可以在笔记本电脑使用时保持连接。

它有一个体面的硬件阵容，配有低压 1.6GHz Core 2 Duo，2GB 内存和 160GB 硬盘。还包括一个 80GB 的外置硬盘用于备份。据联想称，真正的卖点是额外的支持。是的，我们也不为所动。

如果您在主要城市遇到储备问题，联想将在 4 小时内派技术人员飞往您那里。这款只接受邀请的笔记本电脑还提供 24/7 技术支持，不过你必须决定这一切是否值得 5000 美元的价格。
