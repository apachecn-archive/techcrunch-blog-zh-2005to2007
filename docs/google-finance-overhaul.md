# 谷歌金融改革——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/12/12/google-finance-overhaul/>

 [](https://web.archive.org/web/20210925214900/http://finance.google.com/) 正如[预料到的](https://web.archive.org/web/20210925214900/http://gigaom.com/2006/12/10/google-finance/)，谷歌金融今天推出了一个相当广泛的重新设计和功能变化。主页上有更多的信息。旧布局。现在可以为多只股票创建对比图表。热门股票可以按市值、价格、搜索热度等排序。谷歌还增加了 40 年来的股票市场数据，并为那些数百万可能想转换的雅虎财经和其他网站创建了新的导入工具。
