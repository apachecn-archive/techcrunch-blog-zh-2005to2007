# 松下 DMP-BD10A 蓝光播放器:比去年的型号便宜 700 美元

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/15/panasonic-dmp-bd10a-blu-ray-player-700-cheaper-than-last-years-model/>

[![bd10a.jpg](img/cc5391c9812b7ce4eb29f1e49ec5c4c3.png)](https://web.archive.org/web/20160322170652/http://old.crunchgear.com/wp-content/uploads/bd10a.jpg "bd10a.jpg")

松下今天刚刚发布了第二代蓝光播放器 DMP-BD10A。它支持 DTS-HD 和 Dolby TrueHD，这两种新的 7.1 声道音频格式正开始获得关注。(顺便说一下，微软的 HD DVD 播放器[现在支持](https://web.archive.org/web/20160322170652/http://crunchgear.com/2007/05/15/microsoft-updates-hd-dvd-player-more-audio-options-better-disc-compatibility/)这些格式。)播放器还将 H.264 添加到其支持的视频编解码器列表中。所以，没错，它和去年推出的松下蓝光播放器差不多，只是便宜了 700 美元。没错，只需 600 美元，蓝光的力量就属于你了。更棒的是，松下捆绑了《加勒比海盗:黑珍珠的诅咒》、《加勒比海盗:亡灵徽章》、《传送者》、《神奇四侠》和《与玩家一起坠毁》。

照这样的速度，从明年的款式开始，我将能舒适地买得起一辆。
