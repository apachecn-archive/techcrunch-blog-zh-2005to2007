# PriceGrabber:双核 CPU 价格将继续暴跌

我们 CG 热爱双核处理器。事实上，我们的办公室座右铭是“更多核心，更好”因此，我们很高兴地从 Market Reporter 阅读了 PriceGrabber.com 的双核处理器市场趋势数据，Market Reporter 是该公司跟踪该网站上消费者购物行为的统计数据库，它预测 AMD 和 Intel 的双核芯片价格将在 2006 年剩下的几个月里大幅下降。据估计，从 2005 年 7 月到 2006 年，房价已经平均下跌了 28%。目前前 15 名双核 CPU 的平均价格为 379 美元。

> 观察 2006 年下半年各制造商的表现将会很有意思。随着双核处理器的大量出货，第三季度将反映 AMD 和英特尔之间的竞争加剧。AMD 的目标是在 2006 年上半年出货处理器数量的六倍，估计为 980 万个 Socket-AM2 和 Socket-939 Athlon 64 X2 处理器。英特尔的 Core 2 Duo 处理器被称为“Conroe”芯片，旨在将性能提高 40%，同时将功耗降低 40%。对于这两家制造商来说，超越竞争对手的关键将是平衡价格和性能的正确组合，以吸引消费者。

我们自己也说不出比这更好的了。事实是，尽管围绕 Core 2 Duo 处理器的性能提升大肆宣传，AMD 制造了一些非常有竞争力的 CPU，其价格仍然低于英特尔。现在随着[戴尔出售基于 AMD 的系统](https://web.archive.org/web/20201020070735/http://crunchgear.com/2006/09/13/dell-launches-first-amd-powered-desktops/)，AMD 的品牌认知度肯定会提高。哦，是的，让我们记住，这些英特尔性能的提高是与整体性能不佳的奔腾 D 芯片相比较的。
