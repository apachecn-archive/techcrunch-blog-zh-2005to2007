# 沃尔玛正式进入电影下载大战| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/06/walmart-officially-enters-movie-download-war/>

![](img/8b92c61386c6d3a13c50251ac79e63e9.png)在[涉足电影下载测试](https://web.archive.org/web/20230228020104/https://techcrunch.com/2006/11/28/walmart-bundles-digital-downloads-with-physical-dvds/)之后，沃尔玛现在首先进入电影和电视下载业务领域，并将于今天晚些时候[在](https://web.archive.org/web/20230228020104/http://www.techmeme.com/070206/p5#a070206p5)[walmart.com/videodownloads](https://web.archive.org/web/20230228020104/http://www.walmart.com/videodownloads)推出他们的新网站。

这已经是一个非常拥挤的市场，尽管沃尔玛已经与所有主要的电影公司达成了协议——华特·迪士尼、华纳兄弟、派拉蒙、索尼、20 世纪福克斯和环球。关于被其他电影下载服务锁定的工作室交易，请参见本文中的图表[—](https://web.archive.org/web/20230228020104/https://techcrunch.com/2006/10/24/aol-video-adds-paramount-product-still-lags-market/)[CinemaNow](https://web.archive.org/web/20230228020104/https://techcrunch.com/2006/10/15/itunes-movies-v-the-rest/)、 [MovieLink](https://web.archive.org/web/20230228020104/https://techcrunch.com/2006/10/15/itunes-movies-v-the-rest/) 和 [Amazon Unbox](https://web.archive.org/web/20230228020104/https://techcrunch.com/2006/09/07/amazon-unbox-goes-live/) 也都有类似的交易。iTunes 和其他竞争对手没有。

已经同意提供内容的电视网络包括福克斯、CW 和维亚康姆的 MTV、Nickelodeon、喜剧中心、Logo 和 VH1。电视下载市场目前由 iTunes 主导。

电影的价格将是 10-20 美元。电视节目将是 2 美元。下载和观看内容需要一台 Windows PC，上面提到的所有服务都需要 Windows PC，iTunes 除外。

 **这会成功吗？**

沃尔玛在他们的网上活动中取得了喜忧参半的结果。当他们无法与网飞竞争时，他们已经完全退出了电影邮购租赁市场。然而，在这种情况下，没有既定的玩家试图杀死。电影下载市场正处于萌芽阶段，任何人都可能获胜。

CinemaNow 和 MovieLink 应该特别担心。他们的利润率非常低，已经受到了 iTunes 和其他公司的攻击，现在不得不与沃尔玛竞争。他们不太可能有足够的持久力做到这一点，而且看起来没有任何一个主要网站打算在这个领域购买。制作一个基于 Windows 的充满数字版权管理的播放器并直接与电影公司进行交易更便宜。

然而，iTunes 仍然是唯一一个跨平台的服务，可以在 iPod 上播放，现在可以在客厅的电视上播放。他们已经主导了电视下载市场。沃尔玛短期内不会轻易伤害 iTunes。

别忘了网飞已经发布了非常非常好的免费(ish)电影和电视下载服务，而 [Joost](https://web.archive.org/web/20230228020104/https://techcrunch.com/2007/01/15/venice-project-launch-name-joost/) 是另一项即将推出的服务。这些公司都不会在沃尔玛的竞争面前屈服，他们知道沃尔玛会从他们无法主导的市场中脱身。

所有这些对消费者来说都是好事，但价格仍然太高。电影下载仍然比简单地购买 DVD 昂贵。
