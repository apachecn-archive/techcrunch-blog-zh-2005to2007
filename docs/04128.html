顶级社交媒体用户获得报酬；平衡正在改变吗？

杰森·卡拉卡尼斯在他发布的一份AOL备忘录中说，最近Digg的大火和网景页面浏览量的快速增长证明了他为T2网景设计的模型是正确的。他说，最近的事件证明，社交媒体网站的顶级贡献者需要认可和批准，如果不是付费的话，以便继续努力工作，使社交网站充满活力。迈克·阿林顿称卡拉卡尼斯通过向支付网景的巨大红旗来从其他网站雇佣顶级用户，但我不同意迈克的观点，我认为社交新闻等领域的当前发展，尤其是视频分享表明奖励顶级用户可能是一个可靠的策略。(更新:见下面的评论，迈克说我对他的职位描述不准确，他澄清了。)我不认为它像卡拉卡尼斯那样清晰，但我想不出比这更有趣的问题了。

随着社交新闻的发展，广告商寻求进入MySpace和YouTube等网站，业余和专业之间的界限继续模糊——有许多事情正在发生，可能会改变媒体，就像民主党全国代表大会上的博客作为该媒体的一个关键转折点被载入史册一样。当雅虎！他们说这个系统最大的吸引力之一是用户可以免费建立社区。根据卡拉卡尼斯的逻辑，这不是这些天事情发展的方向。有很多证据支持这个观点；这些网站通过奖励顶级用户的工作，结合高质量、非常非正统的企业广告而变得可行。

为了将最近关于Digg ( 我们的覆盖范围)和Netscape ( 我们的覆盖范围)的争论放在一个背景中，这里有一个关于一些正在发生的关键事件的概述，这些事件模糊了业余用户和专业内容制作者之间的界限。

以下是这个迷因的一些要点:

  • 一些顶级Digg用户觉得自己没有得到赏识
  • 据卡拉卡尼斯称，网景公司雇佣的顶级贡献者帮助页面浏览量快速增长
  • 一个顶级YouTube用户被证明是一个专业制作的作品
  • YouTube用户变得专业，专业人士在YouTube上获得成功
  • MySpace不再是一个训练场，而是一个销售平台
  • Revver视频社区有自己的明星，他们正在获得报酬
  • 人们讨厌帕丽斯·希尔顿。

详情如下。

关于这些问题，YouTube 是目前最有趣的网站。上周，YouTube上最受关注的视频制作者之一，孤独女孩15 被披露是强大的贝弗利山人才机构创意艺术家机构的作品——这不是一个刻板印象，一个受保护的16岁女孩背着父母制作并发布视频。争议是巨大的；Lonelygirl15是商业前艺术的合法作品，还是对社区媒体分享真实性的操纵攻击？Danah Boyd 有一些关于这次活动的最好的博客报道，并说纽约杂志有迄今为止最好的主流报道。

社区的反应从 Bravesgirl15 试图成为谴责《孤独女孩》背后的公司的领导者，到长期网站领导者Renneto录制了在我看来是一段虚假的愤慨聪明地跟随《孤独女孩》的领导和风气。还有一些人发布了一个模拟新闻发布会，上面有一个紫色的猴子木偶，看起来像孤独女孩，还有无数其他不那么有趣的回复。这是目前推动YouTube的讨论，对所有这类网站的未来都很重要。

尽管YouTube和帕丽斯·希尔顿的交易毫无疑问是一个失败，并在网站上引发了强烈的反弹，斯米尔诺夫的茶话会音乐视频即使不成功，至少也非常引人注目。在制作了不到30个视频后，网站favorite Brookers 已经与NBC签署了一项协议，其中许多视频是她与商业版权歌曲的对口型同步。

认为从网站外窃取版权视频是支撑YouTube的唯一因素的观点似乎比以往任何时候都不那么站得住脚，但认为网站上的商业活动不可能成功的观点也是如此。

MySpace已经成为YouTube明星的新伙伴主页，通过将于今年晚些时候生效的谷歌交易和音乐人的直接市场途径，MySpace成为一个主要的广告平台。虽然北极猴子在今年早些时候凭借MySpace的成功获得了巨大的离线唱片销量，但这些新的发展表明，社交媒体网站有潜力不仅仅是主流成功的培训场所。

竞争对手 Revver 并不满足于将病毒式视频让给YouTube，它有自己的明星和一个基于每个视频结尾静态图像广告的收入分享计划。基于广告的收入分享对于任何系统的绝大多数用户来说都不太可能是足够的激励，但是对于一个网站最大的明星来说可能就不是这样了。这些明星可能会被激励使用特定的服务，并吸引大量观众，这些观众本身也是长尾内容生产者；他们可以从网站的总体收益中获利。Ze Frank在Revver上表示，新闻视频博客不一定要由乏味的漂亮女孩来表演，才能吸引大量观众。甚至史蒂文·科尔伯特也被指控剽窃了弗兰克的几个笑话。Revver上的另一个项目“询问忍者”正在与维亚康姆的Atom Films合作制作一部商业电影，并在他们的网站上销售商品。

在所有这些例子中，关于“孤独的女孩”的争论可能是最及时和最有趣的，但我认为所有这些都揭示了最近Digg/Netscape的一些争论。正如我的朋友亚历克斯·威廉姆斯所说，病毒式媒体网站是搜索的新星，对顶级用户的认可，无论是通过经济补偿还是地位，都可能是这些网站生存的关键驱动力。相反，商业活动在这些社区是可能的，但它可以采取的形式仍有待讨论:巴黎希尔顿否茶会是，但对于短期活动和孤独女孩可能是——我不认为有共识，或任何迹象表明这种模式可以复制得足够好以至于可持续。作为概念的证明，这很有趣。

在这些领域做广告需要很多技巧，我们会看到失败的人比成功的人多得多，但偶尔的成功可能有助于建立对大量尝试的宽容，而不是全盘拒绝与病毒式媒体社区的商业接触。各公司正在努力寻找有能力在社交媒体空间做广告的人。“第二人生”完全是另一回事。

谁来推动这些网站并使它们具有商业可持续性？顶级用户将成为其中重要的一部分，他们会希望自己的工作得到回报。做这些人正在做的事情是很难的，即使没有报酬，认可也被证明是非常重要的。
