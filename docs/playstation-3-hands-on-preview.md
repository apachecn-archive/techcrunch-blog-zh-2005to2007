# PlayStation 3 动手预览

索尼在纽约举办的 PlayStation 3 活动终于让游戏玩家们拿到了接近完整版本的 PlayStation 3 游戏。除了漂亮的 1080p 高清电视和看起来很酷的椅子，游戏玩家还可以玩大型 PS3 发布游戏*抵抗:人类的堕落*。今天早些时候我也尝试了这个游戏，我不得不说我对这个游戏很失望。虽然 PS3 支持 1080p，但《T3》的阻力看起来与我们已经在 PC 和 Xbox 360 上看到的游戏没有任何不同。这并不是说图形不好，因为它们并不差，但它们不是我们以前没有见过的东西。最重要的是，在我玩这个游戏的两个小时里，没有任何很酷的功能展现给我。这是一个基本的跑轰 FPS，没什么好疯狂的。你会很高兴地知道，SIXAXIS 的运动感应功能将成为*抵抗:人类的堕落*的一部分，因为它被用来在吸血的外星人咬你时让他们离开你的身体。

我印象最深的游戏是索尼的*老巢*。我们没有玩游戏的演示，但索尼确实展示了它的预告片，它被承诺是所有游戏中的镜头，没有那种挑逗性的废话。老巢展示了一些我见过的最好的视觉效果。这款游戏要到 2007 年 3 月才上市，所以至少在游戏上市的时候你会有机会拥有一台 PS3。跳伞后再看一些照片。
