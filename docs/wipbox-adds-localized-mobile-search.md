# WipBox 增加本地化移动搜索

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/17/wipbox-adds-localized-mobile-search/>

如果你在易贝或 Craigslist 上卖东西，很有可能你已经听说过 WipBox。这家初创公司与 DataUnison 合作，为其用户提供信息，现在正在走向移动，将允许用户发送“WipGBox”以及商品名称和邮政编码到 44636。这将反过来提供一个平均和高价格的项目在他们的本地地区。

因此，如果你想卖一些太大而无法运输的东西，或者你更愿意在当地销售，你可以根据易贝和 Craigslist 网站找到你所在地区的现行价格。毫无疑问，这可以帮助你确定卖二手冲浪板是否值得！
