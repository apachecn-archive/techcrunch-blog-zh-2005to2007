# 公关代理惹恼记者的方式，以及一些做得好的公司

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/25/ways-pr-agents-piss-off-journalists-and-a-few-companies-that-do-it-right/>

在一个完美的世界里，最好的产品会得到大量的媒体报道，而劣质产品会幸运地在天空购物中心找到一个位置。不幸的是，植入式广告要比这复杂一些，公司雇佣公关人员是有原因的。

我从事科技写作已经很多年了，在杂志、网络电视、博客和几十个国家电台上预定或投放产品。不用说，我有足够多的公关恐怖故事让一个伦理学家哭泣。我把它们中的大部分留到了我即将出版的《真相大白》一书中，但这里有一个有趣的圣诞节清单，列出了一些我反复看到的致命公关错误，以及一些我认为做得很棒的公司。虽然好的公关很少能为一个毫无价值的产品找到位置，但激怒心软的记者可以确保产品永远不会出现在页面上。

作为一个见多识广的消费者，当你看到某些产品统治着每本杂志，而其他一些产品却从未露面时，请记住这一点。

**几个致命的公关错误:**

**你永远无法确认广告投放**
如果每个让我“确认广告投放”或质疑为什么他们认为“已确认”的产品没有出现在最后一期的公关人员给我一美元，我就能创办自己的公关公司了。事情是这样的:在我编辑、出版和购买我自己杂志上的所有广告之前，没有可能确认任何事情。我可能会认为你的产品很棒，但是有一千个其他的编辑有一千个不同的观点，他们可能不敢苟同。广告页面进进出出，这意味着编辑页面的增加和减少都是一时兴起。东西总是会被剪掉。记住，仅仅因为我是你在杂志社的联系人，并不意味着我能掌控一切。

**永远不要问我为什么没有包括某个产品**
这不是你的错，除非你问了这个问题。见上文。事情发生了。下次再试一次。

**不要把我当成客户一样跟我说话**
有几句话是任何作家都不想从公关人员(或电子邮件)口中听到的。任何听起来像他们在初级公关课程中教你的那种管理语言都应该留在那里。这么说吧，听到认真使用“pitch”这个词是不人性化的(而且有点侮辱人)。

不要把我当成你的朋友，除非我真的是
从另一方面来说，没有什么比拿起电话，让某人对我施展“高效人士的七个习惯”更令人毛骨悚然的了。我知道自助书籍可能会说，假装是某人最好的朋友是“同意”的最佳方式，但当一个陌生人假装他比你母亲更了解你时，这绝对令人不安。如果你介绍自己时直呼其名，却不说你是从哪里打来的，并且表现得好像我应该知道你是谁一样，我们最好至少在会议上握过手。

**永远不要让我越界**
我最近收到一封公关代理的电子邮件，大意是这样的:“我知道你偶尔为[杂志名称编辑]撰稿。我们通常与[姓名编辑]打交道，但由于他们的道德政策，他不被允许为一篇报道做我们希望他做的事情。你愿意为我们做这件事，写下来，不告诉他们你做了什么吗？”不，我不会。

**永远不要让我支付退回产品的运费**
我知道 UPS 不便宜，但我是一名自由职业者，退回 DLP 的账单可能会超过我的评论支付给我的费用。

永远不要在我的办公桌前打电话给我我很忙，有些晚上我会在办公室呆到半夜，我有很多会议和截止日期，我可能不一定有来电显示。永远给我发邮件，永远不要打电话来“跟进”我没有回复的新闻稿邮件。会让我生气，我生气的时候你不想看到我。

**几个好彩蛋:**

以下是我打交道最有经验的几家公司。没有强行推销，没有虚假的友好，没有仅仅因为和他们通电话而感到肮脏。当然，将 Belkin 与微软相比是不公平的(预算，有人吗？)所以，那些充分利用最少资源的公司会得到额外的加分:

一些公关人员把硬推销铺得如此之厚，以至于你想用漂白剂洗澡。我和这些人打了多年的交道，我不记得他们真的要求植入广告，更不用说乞求或试图强迫我植入广告了。他们只是告诉我他们得到了什么，然后让卡片尽可能地落下。令人耳目一新。他们会举办很棒的派对。

**Belkin:** 每次你看到 Belkin 的产品印刷出来，记住他们的整个公关人员由两名年轻女性组成，没有代理机构，也没有广告预算(这意味着如果他们得不到他们想要的，他们不能威胁撤下广告——这确实发生了！).记住这一点。

三星:这些家伙因为一件事而受到称赞:与他们幸运的金星半岛同行不同，他们知道不应该在他们的活动上给他们的韩国高管 30 分钟的发言时间。科技记者最喜欢的游戏是在这样的活动中“发现睡着的记者”。
