# Zooomr 2.0 截图，新功能

17 岁的克里斯托弗·塔特(他现在 18 岁了)在 3 月份创办的公司 Zooomr 将在 6 月底发布新的测试版 2.0。新版本的最大焦点将是增加稳定性和速度——Zooomr 在发布时多次被大规模流量高峰所拖累。

我很喜欢 Zooomr，但他们真正突破极限的地方是照片元数据，特别是通过谷歌地图的混搭进行地理标记。Zooomr 还在创造技术，帮助人们了解事件发生的时间，可能是通过来自彼此不认识的用户的照片。当 Zooomr 看到在一个时间窗口内上传的照片具有相似的地理标记时，它会假设事件正在发生，并相应地对这些照片进行分组。所有这些中唯一困难的部分是，用户需要花费大量时间将位置信息与照片关联起来……这意味着许多用户不会这样做。

期待 2.0 版本中的智能集，让用户能够根据标签、用户和其他信息动态创建动态相册。我真的很喜欢新雅虎照片的这个功能，并希望随着时间的推移，它能被所有照片网站广泛效仿。

Zooomr 还致力于用尽可能多的不同语言创建本地化版本的服务。在 2.0 版本中，他们将在目前支持的 16 个版本中加入罗马尼亚和挪威。对于不讲英语的人来说，Zooomr 可能是最吸引人的照片分享选项。

还有许多其他功能的补充。新版本发布后，我会及时更新。
