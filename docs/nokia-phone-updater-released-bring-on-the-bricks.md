# 诺基亚手机更新发布:带来了砖块！

![](img/f725e26cf5309fba82c8f6842a875759.png)

一年来，诺基亚一直致力于一个项目，让消费者不用去当地授权的诺基亚商店就能更新他们自己的手机。现在终于可以下载了。这个基于 Windows 的程序(beat it，Mac 用户)只在某些诺基亚手机上运行，但是如果你的手机在列表上，这个过程似乎并不痛苦。只需将 USB 线插入手机和 PC，让软件发挥它的魔力。就我个人而言，我不能等到人们开始用砖头砸他们的手机，向诺基亚抱怨他们如何毁了自己的生活。
