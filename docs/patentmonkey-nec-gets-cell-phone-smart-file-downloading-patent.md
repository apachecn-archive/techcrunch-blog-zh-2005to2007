# 专利猴:NEC 获得手机智能文件下载专利

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/24/patentmonkey-nec-gets-cell-phone-smart-file-downloading-patent/>

![necwifi2_cg.jpg](img/f1eda7bd1bdef2ec2a4038a1d1c33b0a.png)

苹果指出，它是唯一一家在手机覆盖不太好的情况下提供扩展覆盖范围的智能手机(T4)的供应商之一(T2)。本周，NEC 获得了一项关于通过选择蜂窝或 WiFi 网络无线下载文件的相关专利[,提醒我们 iPhone 将推出如此多的酷功能，并且将非常高兴地在这样做时踩一些脚趾。](https://web.archive.org/web/20160301034930/http://www.patentmonkey.com/PM/patentid/7233784.aspx)

NEC 一直在与支持 WiFi 的手机技术合作。关于跳跃后的最新细节…

使用手机来降低下载成本，NEC 的技术可以确定网络的可用性，并在了解收费格式(例如 WiFi +宽带的统一费率与大多数手机套餐的可变费率)的情况下，可以自动确定文件下载的更便宜路径。

关于一些细节:

> 1.一种用于将数据文件下载到便携式通信终端的方法，包括:
> 所述便携式通信终端自动从具有统一费率计费的第一无线路径和具有测量费率计费的第二无线路径中进行选择，所述自动选择包括:
> 检测所述第一无线路径是否可用于所述便携式通信终端在当前位置下载所述文件，并且响应于检测到所述第一无线路径在所述当前位置可用， 所述便携式通信终端使用所述第一无线路径下载所述文件，并且响应于检测到所述第一无线路径在所述当前位置不可用，检测所述第二无线路径是否可用，并且响应于检测到所述第二路径可用，所述便携式通信终端从所述第二无线路径下载所述文件。

在蜂窝网络中[仍然相当昂贵]所有你可以得到的数据计划，下载速度可能比成本更重要，但不是每个人都愿意接受每月约 40 美元的完全接入数据计划，这使得这个概念值得扩展。
