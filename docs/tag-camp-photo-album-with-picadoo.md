# 用 Picaboo TechCrunch 标记营地相册

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/10/31/tag-camp-photo-album-with-picadoo/>

几天前，我第一次从罗伯特·斯考伯那里听说了[皮卡布](https://web.archive.org/web/20221006225715/http://www.picaboo.com/)。Picaboo 是一种创建、共享和打印相册的方式。

Picaboo 由霍华德·菲尔德(Howard Field)和凯文·麦柯迪(Kevin McCurdy)于 2002 年 9 月创立，由凯鹏华盈(Kleiner Perkins)和软银(Softbank)出资。他们于 2005 年 5 月启动。

它目前只在 Windows 上运行(尽管 Mac 用户已经为这种东西内置了运行良好的软件)。Picaboo 需要客户端下载，相册编辑在此客户端上进行。虽然我更喜欢 web 应用程序(flash 或 ajax)，但拥有桌面客户端确实会大大加快速度。

Picaboo 可以免费下载，创建和分享相册也是免费的。他们从印刷专辑和 DVD 的销售中获得收入。一个打印的相册是 25 美元加上运费，最多 20 页。每加一页 2 美元。

今天晚上，我花了相当多的时间测试这个软件，制作了一张专辑并与布莱恩·本辛格分享，以测试其功能。

用户界面非常易于使用，几乎不需要帮助或 FAQ 参考。你只需上传图片，选择相册格式，添加文字等。

有各种各样的专辑可供选择。您还可以添加音乐或任何其他声音文件(例如解说)。我不喜欢的一点是，图片必须在上传前保存在本地。例如，你不能给它一个 flickr 用户名，然后使用这些图片。

一旦你创建了一个相册，你可以与“好友”分享，如果他们已经下载了 picaboo 软件，他们可以查看相册并添加到相册中，创建他们自己的版本。您还可以将相册发布到 web 上，供任何人查看。出于某种原因，网络版的专辑不会播放任何你与之关联的音乐或其他声音文件(在我看来这是一个严重的缺陷)。
