# Netvibes 今天宣布种子融资

非常受欢迎的 Ajax 主页产品[](https://web.archive.org/web/20220810213710/http://www.netvibes.com/)，我说过它是一家 [Web 2.0 公司，没有它我无法生存](https://web.archive.org/web/20220810213710/http://www.beta.techcrunch.com/2005/12/30/web-20-companies-i-couldnt-live-without/)，它将宣布由 Index Ventures、马克·安德森、Pierre Chappaz(kel koo 和 Wikio 的创始人)和 Martin Varsavsky(Jazztel、Ya.com 和 Fon 的创始人)提供种子资金。这轮融资的规模没有披露。

Netvibes 上已经创建了 100 多万个主页，而且这项服务有多种语言版本。他们现在正在开发一个 API，允许创建第三方小工具。

Netvibes 的总部设在巴黎和伦敦。我在 Netvibes 上的原帖是[这里](https://web.archive.org/web/20220810213710/http://www.beta.techcrunch.com/2005/09/16/netvibes-personal-homepage/)。
