# TalentSpring 旨在扰乱简历市场

[![talentspring.png](img/cb6362dae841ddb4dd01306010fcaedb.png)](https://web.archive.org/web/20211024222004/http://www.talentspring.com/) 西雅图 [TalentSpring](https://web.archive.org/web/20211024222004/http://www.talentspring.com/) 今天发布了一款独特的产品，有可能颠覆在线简历市场。

由前 Windows Live 开发负责人 Bryan Starbuck 和 Andrew Boardman 领导的 TalentSpring 将 Digg 等社交投票网站背后的理念引入了简历市场。但是将这项服务与 Digg 及其克隆大军相提并论是不公平的；只是很难和其他东西相比。

像传统的简历市场一样，TalentSpring 是求职者或寻找新就业机会者的简历数据库。传统的简历市场通常是搜索驱动的数据库，根据用户提交的位置、经验等进行分类，TalentSpring 使用一个排名系统来对简历进行评级，这样顶部的简历就会浮到 TalentSpring 的顶部，而较少的简历会被丢弃。

简历被其他求职者排名。注册并提交简历后，要想让你的详细信息出现在 TalentSpring 上，你**必须**在你自己的专业领域给 12 组其他候选人打分。

投票系统是一个简历或另一个命题 x 12。用户会看到两份并排的简历，他们必须标出自己认为哪个候选人更适合那个特定领域的职位。

我问布莱恩·斯达巴克为什么只有 12 个而不是更多？TalentSpring 在雇主、招聘公司和求职者中广泛测试了这个系统。他们发现 12 是一个理想的数字。它允许 TalentSpring 获得足够广泛的用户提交数据范围，以便对候选人进行适当的排名，而不会使最终用户的审查任务太长。

作为一名招聘人员，最糟糕的事情莫过于收到 300 份简历，然后又收到一封同样的电子邮件，申请一个招聘职位。几天的时间都浪费在筛选候选人，试图制定一份面试入围名单上。TalentSpring 的目标是通过只为一个职位提供优秀的候选人来绕过招聘人员的大部分程序。排名系统与深度过滤系统相结合，根据招聘人员和求职者的需求来限制候选人。例如，如果一个求职者只寻找薪水在 100，000 美元及以上的工作，那么他们只能通过招聘人员找到该类别的工作机会。

星巴克将 TalentSpring 比作 Zillow.com 和 FareCast.com。所有这些都使用高级数学和数据挖掘，从交易双方读取数据，并通过查看整个市场的整体视图来汇总/计算估值。

这项服务的费用极具竞争力，从 195 美元一个位置开始，还提供各种无限使用包。

TalentSpring 作为一项服务将会吸引招聘岗位上的大量人员。我可能在我的评估中有偏见，因为我以前在涉及招聘的职位上工作过，所以我知道招聘人员的痛苦。如果 TalentSpring 获得了关注，并建立了一个体面的简历范围，这是一个我很乐意使用的服务，我相信其他人也可能会使用它。
