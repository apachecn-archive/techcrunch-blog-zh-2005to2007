# 海龟滩还在，做 Wii 耳机

![](img/b768ba3b930a7f510028df812526d28e.png)

自从 Turtle Beach 在 20 世纪 90 年代早期停止生产声卡后，我就再也没有听到过它的消息，但我想随着 Wii 最近人气的飙升，现在是赶上这股潮流的好时机了。无论哪种方式，这些来自海龟海滩的 Ear Force W3 无线耳机都是“为 Wii 做好准备”的，并配有音频分离器电缆。

由一节 AAA 电池供电，这些手机将花费你 60 美元和你的骄傲。
