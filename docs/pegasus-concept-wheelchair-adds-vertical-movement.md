# 飞马概念轮椅增加了垂直运动

[![pegasus.jpg](img/b2cb9b59c48f87bdb193e6e42a5de285.png)](https://web.archive.org/web/20220926073509/http://old.crunchgear.com/wp-content/uploads/pegasus.jpg "pegasus.jpg")

现在你应该知道我是一个爱发牢骚的孩子，就像“黑道家族”里的那个孩子一样，而且我总是看好那些试图帮助穷人的小玩意。因此，Pegasus 是另一个试图改善轮椅使用者生活的设计概念。除了作为日常轮椅之外，Pegasus 还允许一定程度的垂直移动，希望给用户更大的机动性。简单的，像去市场这样的日常事情可以变得更容易；曾经遥不可及的物品现在已经触手可及，等等。我喜欢这个主意。
