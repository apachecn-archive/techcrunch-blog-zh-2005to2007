# Numark 最终确定 iDJ2 的发布日期

![](img/2287b4f1d85f4a24580a007fa824f255.png)

受延迟困扰，Numark 的最新 iPod DJ 混音器 iDJ2 终于要发布了。到了九月，你就可以花 600 美元买一个 iPod 混音器了，这应该会给 DJing 带来革命性的变化。iDJ2 具有双 iPod 基座、便笺簿、音高控制、节拍匹配、按键锁定以及同时播放同一 iPod 中的两首歌曲的能力，这确实是革命性的。

有大量的输入和输出，包括 USB。还包括一个全彩色屏幕和播放从 iTMS 上购买的歌曲的能力。功能丰富？绝对的。太贵了？下个月我们就知道了。
