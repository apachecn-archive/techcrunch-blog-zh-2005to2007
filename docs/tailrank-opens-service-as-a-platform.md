# TailRank 开放服务作为平台

与 [TechMeme](https://web.archive.org/web/20220112001223/http://www.techmeme.com/) 和 [Megite](https://web.archive.org/web/20220112001223/http://www.megite.com/) 竞争的热门博客新闻组织者 [](https://web.archive.org/web/20220112001223/http://www.spinn3r.com/) [TailRank](https://web.archive.org/web/20220112001223/http://www.beta.techcrunch.com/2006/10/16/tailrank-20-launches-to-take-on-techmeme/) 已经开放了其后端，并将作为一个平台向希望添加博客索引和排名服务的应用程序开发者提供服务。这项新服务名为 [Spinn3r](https://web.archive.org/web/20220112001223/http://spinn3r.com/) ，将于今天中午上线。有传言说它已经为 Gnoos 提供了动力，去年六月我们已经报道过了。

Spinn3r 索引了 100 万个博客。他们对每 1000 个被抓取、索引和发布的博客收取月许可费。
