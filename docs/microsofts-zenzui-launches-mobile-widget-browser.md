# 微软 ZenZui 推出移动小工具浏览器

[![zenzuilogo.png](img/a11c58ae6e5d6f543128fad7d30f0037.png)](https://web.archive.org/web/20201127201529/http://zenzui.com/)TechCrunch 已经[报道了](https://web.archive.org/web/20201127201529/http://www.beta.techcrunch.com/2007/03/27/zenzui-on-mobile-browsing-the-microsoft-way/)新的移动 widget 平台 [ZenZui](https://web.archive.org/web/20201127201529/http://zenzui.com/) 的发布，他们今天早上推出了他们的移动 widget 浏览器。该浏览器允许用户通过触摸屏或键盘在 36 个小部件的网格中平移和缩放来搜索。浏览器中的每一个小部件都可以作为 Zillow 或 Amazon 等网站的移动优化界面。这基本上是一种增加牌组位置的方法。

小工具由开发者创建，并通过“公认的广告原则，如 CPA 和 CPM(我们称之为 CPZ-每变焦成本)”在服务上赚钱。ZenZui 将与 widget 开发者分享收入，并提供一个无广告版本，显示来自非营利组织的信息。

ZenZui 的缩放用户界面最初是由微软的雷德蒙研究实验室开发的，后来被剥离到它自己的公司。该公司与微软 IP Ventures 合作，从 SeaPoint Ventures 和其他投资者那里获得了 1200 万美元的首轮融资，微软保留了该公司的股份。

[mobile runch](https://web.archive.org/web/20201127201529/https://crunchbase.com/product/mobilecrunch)还覆盖了其他 widget 平台，如 [Bluepulse](https://web.archive.org/web/20201127201529/http://mobilecrunch.com/2006/12/03/bluepulse-20-is-bigger-slicker-broader-and-deeper-and-maybe-the-ultimate-mobile-media-platform/) 和诺基亚的 [Widset](https://web.archive.org/web/20201127201529/http://mobilecrunch.com/2006/10/03/nokia-launches-widset-widgets-for-mobiles/) 。
