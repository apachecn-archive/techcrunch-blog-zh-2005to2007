# OWC 售后市场超级驱动| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/15/owc-aftermarket-superdrive/>

Other World Computer 今天发布了一款价格实惠的售后 SuperDrive，可用于最新的 MAC 电脑。这些驱动器旨在替换旧的只读驱动器。他们使用先锋公司的安静 SuperDrive 技术，可以以 16 倍的速度刻录双层 DVD。

这些驱动器现在的价格从 OEM 的 29.99 美元到捆绑版的 99 美元不等。
