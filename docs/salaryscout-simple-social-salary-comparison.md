# SalaryScout:简单的社会工资比较

 [](https://web.archive.org/web/20221002133056/http://www.salaryscout.com/) [SalaryScout](https://web.archive.org/web/20221002133056/http://www.salaryscout.com/) 是一个刚刚推出的薪酬比较网站，它做了很多[以前的简介网站 PayScale](https://web.archive.org/web/20221002133056/https://beta.techcrunch.com/tag/payscale) 做的事情，但融入了更多网络新发展的典型社交功能。它有一个很好的轻量级的感觉。几乎每个人都想知道，他们是应该挣更多的钱，还是应该为自己相对于从事类似工作的其他人挣了多少钱而感到自豪。对这种服务肯定有需求，但大多数网站使用起来太麻烦，除非用户支付订阅费，否则没有什么价值。我喜欢 SalaryScout，因为它很简单。

SalaryScout 上的用户提交相对较少的关于他们工作的匿名细节，包括工作描述、在工作场所等级中的位置、福利和其他信息。作为对贡献你的工作和报酬信息的交换，你可以获得其他人工作的详细信息。

用户可以对其他人的个人资料进行评级，将个人资料标记为伪造，发表评论，并订阅 RSS 源进行搜索。在 SalaryScout 获得大量用户群之前，对 RSS 的支持是唯一可以让它免于一文不值的东西——这是社交软件的一个常见陷阱。即使现在没有很多(或任何)符合我兴趣的工作的搜索结果，我也可以订阅 SalaryScout 中所有符合我标准的未来条目。如果从来没有人这样做，那么没有伤害，但几乎没有任何障碍，很容易在未来监测网站。

其他薪资比较网站试图提供各种形式的附加值，但最终都以糟糕的导航和令人沮丧的用户体验告终。SalaryScout 很简单，而且比一些做薪资比较的网站要好看得多。

这可以通过多种方式实现盈利——广告和综合数据销售是最明显的方式。如果网站创建者 Ben Thomas 将这项服务作为副业，那么他没有理由不能在做正常工作的同时赚点小钱(不管是什么，这里所有的数据都是匿名的)。)我喜欢这个网站的开发方式，虽然没有大量的推广、全面的商业实践和相应的网站膨胀——像这样的小网站不太可能提供大量的信息——但随意使用还是不错的。我已经订阅了一个写作的 RSS 源，我很有兴趣看看其他人提交了什么。谁知道呢？或许 SalaryScout 可以成为薪资比较网站 Craigslist——尽管在 Craigslist 初创时期驱动它的嬉皮士们不太可能对薪资比较感兴趣，你永远也不会知道。

这些被称为 Web 2.0 的简单部分——用户评论、用户评级、标记为不适当、RSS 订阅——它们的发展都是有原因的，正如 SalaryScout 所证明的那样，你可以将它们与良好的视觉设计结合在一起，并拥有一个可靠的网站。
