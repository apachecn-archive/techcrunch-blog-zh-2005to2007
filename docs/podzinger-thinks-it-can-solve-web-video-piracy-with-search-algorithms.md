# PodZinger 认为它可以用搜索算法解决网络视频盗版问题

[![podzinger.jpg](img/08c2c7e9c2915dde040d25a0299f5ff7.png)](https://web.archive.org/web/20201129074146/http://old.crunchgear.com/wp-content/uploads/podzinger.jpg "podzinger.jpg")

没人喜欢盗版。嗯，[我们有](https://web.archive.org/web/20201129074146/http://crunchgear.com/2007/03/14/ids-ceo-says-piracy-killing-pc-gaming-industry/)，[当然有](https://web.archive.org/web/20201129074146/http://crunchgear.com/2007/02/23/riaa-names-top-piracy-schools/)，但是内容提供商并不像我们这样冷静。PodZinger 是一家初创公司，它的主要产品——真正的视频搜索——最终可以拯救 YouTube 等网站，使其免于不断地被全球的维亚康姆以侵犯版权为由起诉。凭借其视频搜索技术，用户能够精确定位视频中的确切对话，目前准确率为 70%。寻找乔治和杰里讨论成长和改善生活的场景？简单搜索一个相关的关键词——这里是“我们不是男人”——视频就弹出来了。Neat-o.
 这就是盗版部分的由来。如果公司可以正确地标记他们的视频——也就是说，让 PodZinger 的算法分析它们——那么他们就可以根据视频内容销售广告。如果在观看宋飞的片段之前，有一个关于让你的生活回到正轨的广告，不要感到惊讶。我不知道，想想一些愚蠢的银行广告，说在他们那里开个账户是负责任生活的第一步。像这样滑稽的事情。

因此，请留意 PodZinger 和越来越多的嵌入式广告。因为海盗们对广告如此着迷。
