# Bittorent 上提供 Windows Vista 测试版

Windows Vista Beta 2 下载量高达 3.2 GB(甚至[产品指南](https://web.archive.org/web/20220701023333/http://www.microsoft.com/downloads/details.aspx?FamilyID=BBC16EBF-4823-4A12-AFE1-5B40B2AD3725&displaylang=en)也超过 70 mb)。您可以下载该文件或订购 DVD。微软正在大力推广 DVD 选项，指出他们的服务器正承受着下载的巨大压力，许多客户根本无法通过这种方式获得软件:

![](img/3599ea11ba32fac0837e7910c9e61bd6.png)

当然，微软已经选择忽略现存最有效的文件分发方法——bittorent。克里斯·皮里略和杰克·卢丁顿在没有得到微软任何认可的情况下，创建了 [VistaTorrent](https://web.archive.org/web/20220701023333/http://www.vistatorrent.com/) ，并给出了通过 Bittorent 下载文件的说明。他们还指出，微软网站将为该软件生成一个密钥。
