# MySpace 养活死水一潭，资助另一家创业公司

[](https://web.archive.org/web/20211206061525/http://www.datinganyone.com/)[MySpace](https://web.archive.org/web/20211206061525/http://www.myspace.com/)[274 亿英镑的大猩猩](https://web.archive.org/web/20211206061525/http://www.beta.techcrunch.com/2006/06/13/myspace-the-27-billion-pound-gorilla/)，今天又踩死了一家创业公司。DatingAnyone.com 已经停止了它的服务。

上周，初创公司 Singlestat.us 关闭了大门，以回应 MySpace [的停止和终止函](https://web.archive.org/web/20211206061525/http://www.beta.techcrunch.com/2006/06/15/myspace-nukes-singlestatus/)。同一封信，不同的名称和地址，是关闭 DatingAnyone.com 的原因。这些服务几乎完全相同，当某人在 MySpace 上的关系状态发生变化时，它们会让你知道。

TechCrunch 的死池增长。
