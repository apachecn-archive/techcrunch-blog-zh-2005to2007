# Purity 原型笔记本电脑:走路像 MacBook，说话像 MacBook

[![puritylap1.jpg](img/7a42846adfa8cbd6b14e68086469847c.png)](https://web.archive.org/web/20130628164628/http://old.crunchgear.com/wp-content/uploads/puritylap1.jpg "puritylap1.jpg")

马上，这款笔记本电脑原型 Purity 笔记本——让你想起了什么？对我来说，这看起来就像任何数量的个人电脑都在努力，真的在努力模仿 MacBook。[并不是说](https://web.archive.org/web/20130628164628/http://youtube.com/watch?v=cTDVEb6Wu-w)有什么问题。作为一台个人电脑，它运行的是 Windows Vista，就原型机运行的*任何东西*而言，我想。

[![puritylap2.jpg](img/36715e250dd22f97b4b641737b55640e.png)](https://web.archive.org/web/20130628164628/http://old.crunchgear.com/wp-content/uploads/puritylap2.jpg "puritylap2.jpg")

称之为极简主义是一种老生常谈:什么笔记本电脑，不管是真的还是假的，不是“极简”的？

再说一次，我是个厌倦了的卑鄙小人。我会说黑色的比白色的好看多了。我想是因为光线反射的方式。

即使只有一个，也是如此吗？
