# Sonace 扬声器几乎是看不见的

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/08/sonace-speakers-are-nearly-invisible-sorta-kinda/>

[![sonancespeaks.jpg](img/42c3f43b6cbbe71f8bbda7488e3e351f.png)](https://web.archive.org/web/20150920140358/http://old.crunchgear.com/wp-content/uploads/sonancespeaks.jpg "sonancespeaks.jpg")

你那个唠叨的妻子不让你在你的起居室里安装大而有男子气概的扬声器？我的第一个建议是申请离婚，但我的第二个建议是检查一下这些“看不见的”Sonance 建筑系列扬声器，特别是筋膜 Z8F。他们退到墙内*和*你会注意到没有格栅，没有可见的螺丝，什么都没有！简而言之，这些扬声器就像你能从同类产品中期待的那样干净。
