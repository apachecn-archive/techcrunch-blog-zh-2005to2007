# iPhone 之旅:我放弃

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/22/the-iphone-tour-i-give-up/>

迈克，请不要恨我们。你看，读者迈克给我发邮件，要求我们停止谈论 iPhone。但是我们不能。我们真的不能。它占据了整个新闻周期。每个人都害怕 iPhone，他们不会推出任何东西来竞争它。想象一下，我不知道，一部相当酷的科幻电影在同一周上映*《星球大战 7:乔治·卢卡斯前列腺的进攻》*。你会做吗？不。也许你会等到假期。你会选择恰当的时间来避开《星球大战》给风景投下的巨大阴影。

所以拜托，去看这个疯狂的视频吧。厂家:去看两遍。从现在开始，每部手机都必须是这个样子。我重复一遍。没有人再想要你的手机了，除非某个穿黑衬衫的家伙谈论它 20 分钟，除非点击电话让我想起更多的*少数派报告*而不是*诺比特*。这是你的 Windows/MS-DOS 时刻——在这款手机之前的一切都是 MS-DOS。当然，你有诺顿指挥官和那个奇怪的图形 DOS 外壳，但是当 Windows——和 MAC OS——上市时，游戏完全变了。MS-DOS 没有削减它，在这之前的一切都是 MS-DOS。咆哮结束。对不起，迈克，但我不得不这样做。再过几天，我们就可以继续我们的生活了。
