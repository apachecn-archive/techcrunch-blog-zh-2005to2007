# 诺基亚推出手机 WidSet 小工具

![](img/182620955528dd9eeeb992db273b0bc6.png)
[WidSets](https://web.archive.org/web/20201127205159/https://www.widsets.com/index) 是一个适用于 Java MIDP 2.0 手机的移动窗口小部件平台，完全独立于制造商和浏览器。就像电脑上的小部件一样，WidSets 提供了对从 RSS 提要到博客文章到照片上传网站的所有信息的即时访问。诺基亚已经为你的手机增加了相当多的选项。模板也可用于创建您自己的模板。只需访问该网站，注册，下载安装客户端，然后加载——它们是免费的。
