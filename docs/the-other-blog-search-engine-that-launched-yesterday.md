# 昨天发布的另一个博客搜索引擎

 [](https://web.archive.org/web/20211130070941/http://www.gnoos.com/) Bloglines 并不是昨天推出[新博客搜索引擎的唯一一家公司](https://web.archive.org/web/20211130070941/http://www.beta.techcrunch.com/2006/05/31/askcombloglines-launch-blog-search/)—[gnos](https://web.archive.org/web/20211130070941/http://www.gnoos.com/)也发布了。尽管他们的推出时机非常糟糕，但这个专注于澳大利亚的博客搜索引擎还是有很多让人喜欢的地方。

Gnoos 专注于索引澳大利亚的博客以及具有“全球声音”的博客。由于关注点不同，很难将特定查询的结果集与其他博客搜索引擎进行比较。相反，我将重点放在特性集上，在这方面，Gnoos 是最好的。

每个结果(一篇博客文章)都可以在 Gnoos 上评论，并进行评级。一篇文章的总投票数会被统计出来，并且很快会被用来决定后续搜索的相关性。点击“更多”按钮会弹出整篇文章。这似乎是一个显而易见的功能，其他一些新的博客搜索引擎也有这个功能，但 Technorati 继续忽略这一点，强迫点击博客来阅读整篇文章。最后，我最喜欢这个功能，标签可以被添加到摘要下面的框中，虽然标签现在还没有用，但 Ben 告诉我他们很快就会围绕标签构建功能。

除了这些特性，Gnoos 网站还有一些让人感觉不错的地方。这可能是因为它整洁的设计，对搜索的快速响应时间或一些无形的东西，但它似乎只是一个我可以花很多时间的网站。本，你应该考虑把这个扩展到澳大利亚以外。
