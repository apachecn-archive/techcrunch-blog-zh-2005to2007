# 东芝的 REGZA WLT68 高清电视

东芝表示，东芝的 REGZA 系列电视是第一批带有三个 HDMI 输入的电视。这些电视采用了 Active Vision M100 技术(提供“最平滑、最清晰、最逼真的画面”)，增强了细节、色彩、对比度和集成的数字调谐器。它们还拥有独立的低音和高音扬声器系统，并提供可选的低音炮。还没有定价，这很重要，因为这些设备的最大卖点是三个独立的 HDMI 输入，而不是只购买一个 HDMI 选择器。
