# MyNetscape 将于今日发布:更多 Ajaxy 垃圾

MyNetscape (太平洋标准时间凌晨 3 点关闭)今天正式作为用户可定制的 Ajax 主页重新发布。网景博客有详细报道。

像 Netvibes、Pageflakes、GoogleIG、MyYahoo、Live.com 和许多许多其他网站(我错过了谁？)，用户将能够从“不到 100 个模块”的定制内容中进行选择，并为喜欢的提要添加 RSS 模块。

Netscape 的用户基础并不完全是尖端的，AOL 显然正在从新的创业公司中吸取好的想法，并观察他们的用户是否会消费它们。Netscape 在 2006 年中期成为了 Digg 的克隆版本，现在 my.netscape 将加入 Ajax 主页的潮流。没什么好批评他们的，但也没什么好激动的。这家曾经伟大的公司长期缓慢的衰落仍在继续。

请看 [ReadWriteWeb](https://web.archive.org/web/20220817205757/http://www.readwriteweb.com/archives/mynetscape_20.php#more) 了解更多信息，它询问“网景的用户群能应对另一次 web 2.0 的大变革吗？”
