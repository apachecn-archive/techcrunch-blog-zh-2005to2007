# 简单火花:网络应用目录

[![simplespark.png](img/3f287fe1aa1f1ec20ff407a5ca9dd8ba.png)](https://web.archive.org/web/20220929222839/http://simplespark.com/) 总部位于纽约的 [Simple Spark](https://web.archive.org/web/20220929222839/http://simplespark.com/) 旨在通过广泛的在线应用网站目录使寻找网络应用变得容易。

超过 3000 个 web 应用程序被跟踪，并通过截图、图标、提要、评论和简明摘要进行分类。应用程序被编入超过 70 个类别的索引，包括电子表格、工作空间和维基、教育、时尚、绿色生活和家庭与孩子

简单的 Spark 搜索工具也是语言友好的，搜索 delicious 或 Flicker 会得到 del.icio.us 和 Flickr 的正确结果；我们以前都拼错过 Web 2.0 应用程序的名称。

每一个竞争空间都可以通过基于单独类别、搜索或目录最新添加的 RSS 源进行跟踪。

即使是业内人士也很难跟上每个领域的众多竞争对手，更不用说外部观察者了。我发现自己对 Simple Spark 印象深刻，顾名思义，这是一个简单的想法，我肯定会在将来研究产品类别时使用它。
