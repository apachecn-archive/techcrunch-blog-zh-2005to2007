# 三星开发 8GB MicroSD 卡

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/17/samsung-develops-8gb-microsd-card/>

![](img/9003b7a5d4af34540d0f774fdf2dd70d.png)

不太清楚你们中有多少人用音乐手机代替 iPod，但是如果你用了，三星有一些令人兴奋的消息。今天，该公司宣布开发出了 8GB 的 microSD 卡。这种新型号的写入速度为 8MB/秒，而以前的老式卡的写入速度为 2MB/秒。这是一个好消息，因为传输 8GB 的视频和音乐并不是我所说的快乐时光。

如果你想知道，8GB 足够两到三张翻录和编辑过的 DVD。你也可以在上面放上 2000 首歌曲，确保无论飞行、坐火车或慢跑多长时间，你都会有不间断的音乐。三星还没有公布芯片的发布日期，但我想明年的某个时候你会在商店里看到这些。

[三星为音乐手机准备 8GB microSD 卡](https://web.archive.org/web/20150312091606/http://www.electronista.com/articles/07/05/17/samsung.8gb.microsd.card/)
