# 微型 3 端口 USB 集线器:小巧、光滑| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/15/tiny-3-port-usb-hub-small-slick/>

我们得到的设备越多，我们需要更多的 USB 端口来支持它们。如今，大多数计算机都只有四个或更少的通用串行总线类型的端口，一个好的 USB 集线器是必不可少的。

我们发现自己喜欢上了这个来自日本 Elecom 的天才们的整洁的小动物，我们在 EverythingUSB 上找到了它。它将你可怜的一个 USB 端口变成了 3 个，顶部和底部端口成角度，这样你的设备就不会纠缠在一起。它可能也不会倾家荡产。
