# 新的“社交”词典——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/26/new-social-dictionary/>

 [](https://web.archive.org/web/20210920200938/http://www.word.sc/) 一个名为 [WordSource](https://web.archive.org/web/20210920200938/http://word.sc/) 的新在线词典今天发布。与 Dictionary.com 等网站相比，它的主要优势在于网站设计非常简洁，不含广告。你也可以通过简单地将一个单词添加到 URL 中来查找它(因此[word.sc/example](https://web.archive.org/web/20210920200938/http://word.sc/example)将调出“示例”的定义)。

该网站还有许多社交功能，允许用户上传与单词相关的照片，添加标签，给单词评分等。将一个社交网络植入在线词典似乎有点矫枉过正，但这就是你要做的。如果他们没有建造它，其他人可能会。

我倾向于使用 Google 来定义单词(只需键入您的查询作为" [define: example](https://web.archive.org/web/20210920200938/http://www.google.com/search?q=define%3A+example&ie=utf-8&oe=utf-8&aq=t&rls=org.mozilla:en-US:official&client=firefox-a) ")，许多人也使用[这个](https://web.archive.org/web/20210920200938/https://addons.mozilla.org/firefox/1171/) Firefox 插件。
