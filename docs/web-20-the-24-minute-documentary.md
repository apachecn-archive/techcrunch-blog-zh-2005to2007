# web 2.0:24 分钟的纪录片

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2006/08/0a8/web-20-the-24-minute-documentary/>

几周前，迈克尔·阿灵顿召集了一些初创公司的首席执行官和高管，就 Web 2.0 进行了视频讨论。参与讨论的有阿龙·科恩([博尔特](https://web.archive.org/web/20100420032659/http://www.bolt.com/))、斯科特·米莱纳和史蒂文·卢里([布劳斯特](https://web.archive.org/web/20100420032659/http://www.browster.com/))、基思·蒂尔([埃奇奥](https://web.archive.org/web/20100420032659/http://www.edgeio.com/))、史蒂文·马德尔([尤里克斯特](https://web.archive.org/web/20100420032659/http://www.eurekster.com/))、乔·克劳斯([乔·斯波特](https://web.archive.org/web/20100420032659/http://www.jot.com/))、杰里米·韦尔巴([皮佐](https://web.archive.org/web/20100420032659/http://www.piczo.com/))、奥伦·霍夫曼([拉普利夫](https://web.archive.org/web/20100420032659/http://www.rapleaf.com/))、克里斯·奥尔登([罗约](https://web.archive.org/web/20100420032659/http://www.rojo.com/))、高塔姆·戈德瓦尼(。 大卫·西弗里([泰诺拉蒂](https://web.archive.org/web/20100420032659/http://www.technorati.com/))、马特·桑切斯([视频蛋](https://web.archive.org/web/20100420032659/http://www.videoegg.com/))和迈克尔·坦纳([眨眼](https://web.archive.org/web/20100420032659/http://www.wink.com/))。

讨论的主题包括:

1.  什么是 Web 2.0？
2.  我们是在泡沫中吗？
3.  今天在网络上行得通的商业模式有哪些？
4.  在用户创造的世界里，出版商的角色是什么？
5.  早期采用人群有多重要，有多大？
