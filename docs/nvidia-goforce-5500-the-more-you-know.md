# NVIDIA GoForce 5500:你知道的越多| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/05/nvidia-goforce-5500-the-more-you-know/>

你知道 NVIDIA GoForce 5500 移动 GPU 能够在手机上完全运行 Quake 3 吗？我没有，这让我有点奇怪。

我记得当地震 3 第一次出现的时候。系统要求非常苛刻，我不得不对我的电脑进行全面检查。现在，它可以在手机上运行，所有功能都已启用。疯狂的谈话。

你看到的是 NVIDIA 开发者单元。它允许他们自由地互换零件。它拥有比大多数手机更快的处理器，但我仍然对 GPU 的能力印象深刻。这不是一个小任务。
