# 奥尼尔 H3 Campack 是最酷的背包

![](img/ba5f7cc315321f73f9c65b120ab20cfc.png)

铁杆滑雪板玩家比你酷多了。克服它。他们想尽一切办法踩你。现在，他们又多了一个工具。奥尼尔 H3 Campack 是你在斜坡上需要插入的一切。这个背包上有控制器，可以插入你的 DV 摄像机，iPod 和蓝牙手机。嗯嗯…一边滑雪一边打电话。然而，这还不是全部。当然，包有防风雨的隔间，这是这种类型的包的惯例；它也有一种操纵杆，你可以用它来控制你的相机镜头或你的 iPod 的播放列表。花 240 英镑(449 美元)买一个，假装你和那些从不下斜坡的人一样酷。
