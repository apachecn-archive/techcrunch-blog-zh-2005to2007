# 光环 3 预演，意见不一

上周五，微软和 Bungie 发布了即将于今年晚些时候发布的《光环 3》的预览版。即将推出的第一人称射击游戏是广受欢迎的光环系列的最新产品，可能是微软 Xbox 360 游戏机急需的“杀手级应用”。在游戏的公开测试之前，游戏的预览版在纽约和旧金山上演，游戏将于本周三上线。虽然《光环 3》提供了新的进步，包括改进和更加真实的物理效果，但一些与会者认为这些图形仍有一段路要走，才能成为真正的“下一代”
