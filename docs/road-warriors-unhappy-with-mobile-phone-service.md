# 移动办公族对手机服务不满

这项调查的标题听起来像是一部以内地为背景的后世界末日电影的剧本。但 In-Stat 最近的研究“2007 年出差族:爱他们或失去他们”实际上给无线运营商带来了一些坏消息。高平均每用户收入(ARPU)北美的客户对他们的服务不满意，但这真的令人惊讶吗？然而，超过 60%的客户感到不受重视，80%的客户认为提供商需要做得更多。

In-Stat 分析师比尔休斯(Bill Hughes)表示:“这些用户希望看到的增值包括忠诚度计划，用户可以通过奖励获得积分，免费或低价的查号服务，以及免费电池和旅行充电器。”“对运营商来说，好消息是这三个奖项可以转化为更多的收入。例如，目录辅助直接鼓励更多的呼叫，提供更多的电池和充电选项可以确保重度用户始终可以拨打电话。”

无线运营商可能会认真考虑比尔的建议。如果出差族太不开心，一些无线运营商可能会被迫在梅尔·吉布森的下一部电影中采取行动。
