# UnZoone 移除有缺陷的 Zune 软件

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/22/unzoone-removes-flawed-zune-software/>

![](img/5fae6e0668be8b99a92868c9f19ef645.png)

我们今天偶然发现了一个可爱的小网站。这个网站上有一个叫 UnZoone 的软件。UnZoone 是一款专为移除 Zune 软件和所有随之而来的 bug 而设计的软件。尽管是一个小工具(128kb 大小)，UnZoone 是强大的。

它会删除所有东西，从注册表项到那些讨厌的 DLL 文件，即使你使用卸载工具(不是 UnZoone)也会留下来。看来作者是真的铁了心要确保 Zune 软件不会留在你的电脑上，因为他计划“不断更新”UnZoone。最优秀的人。在我们找到像样的东西之前，为什么要退而求其次呢？
