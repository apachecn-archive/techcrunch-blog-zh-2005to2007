# SpinVox 自旋到状态

请注意，SpinVox 现已在我国境内推出。

他从去年 12 月开始使用这项服务，似乎非常满意。每当你收到语音邮件，SpinVox 会将信息转换成文本，并通过短信发送到你的手机上。这样，你就能及时、不引人注目地获得信息。

虽然它目前正在与运营商谈判，但你可以通过该公司的[博客](https://web.archive.org/web/20130628184322/http://simoncrowfoot.livejournal.com/)测试这项服务。
