# Tzero Tech 和 ADI 公司宣布推出无线 HDMI

![](img/ce3c638ebe287719240f3d872f72c78c.png)

Tzero Technologies 和 ADI 公司推出了适用于任何兼容 HDMI 的设备的无线 HDMI。如果你问我，我会说时机非常好，对高清和高清配件的需求正在快速攀升。那么界面是如何工作的呢？根据 Gizmodo 的说法，HDMI 接口使用 JPEG 2000 压缩数据，JPEG 2000 是一种图像压缩标准，几乎没有明显的伪像。当然，很难说“几乎不引人注意”的质量实际上等同于什么，但是我们很快就会发现，不是吗？Tzero Technologies 和 ADI 公司表示，该产品将于 11 月上市。尚未定价，敬请关注。
