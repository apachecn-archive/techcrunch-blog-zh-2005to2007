# Evergreen NAS 是 Mac 迷你版

日本的 Evergreen 公司生产了一种具有 Mac Mini 外形的 NAS。白色和银色的 Mac Mini 风格将包含一个 IDE 或 SATA 硬盘驱动器-台式机，而不是笔记本电脑-除了以太网连接外，还将具有 USB 2.0。

这实际上看起来非常适合您想要与其他组件融合在一起的家用 NAS，尤其是苹果的某台小型计算机。
