# TomTom 将购买 Tele Atlas

![](img/8e4288fdd5920ea0a005db6997ce43bd.png)

激动人心的消息？不尽然，但随着 TomTom 宣布收购地图提供商 Tele Atlas，这可能意味着更好的服务和更好的产品。据传 TomTom 以 25 亿美元收购了该公司。当然这是一大笔钱，但是想想你从这笔交易中得到了什么。Tele Atlas 目前是 TomTom 数据的供应商。

这个消息也会震动其他地图公司。Mapquest 很烂，老实说，自从谷歌地图出现后，我就没用过它。幸运的是，这次收购肯定会动摇两家公司的高管，可能会迫使他们推出新功能，以保持新鲜感和竞争力。一旦交易完成，每日地图更新也可能成为现实。听起来像是消费者和 TomTom 的双赢。
