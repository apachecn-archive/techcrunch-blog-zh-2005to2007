# Skype 推出实时语音翻译

 [](https://web.archive.org/web/20230125121528/http://www.skype.com/) 通过与 Voxeo 和 Language Line Services 的合作，Skype 现在[为 Skype 语音通话提供](https://web.archive.org/web/20230125121528/http://share.skype.com/directory/%3Cb%3Elanguage_line%c2%ae_personal_interpreter%3C%10b%3E/view/)实时语言翻译服务。支持 150 种语言。

这就是我喜欢它的地方:它易于使用(将该服务添加到您的 Skype 联系人列表中)，它并不贵得吓人(每分钟 2.99 美元，没有其他费用，从 Skype Out 点数中扣除)，而且您可以随时使用它，无需安排时间(您可以在首次请求后平均 45 秒内获得翻译)。

有一个问题:我已经验证了这项服务可以在 PC 上运行，但在我的 Mac 上根本无法运行。

现在，如果 Skype 增加了通话记录功能，那就太完美了。
