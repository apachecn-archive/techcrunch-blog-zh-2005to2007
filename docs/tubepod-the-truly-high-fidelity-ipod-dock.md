# TubePod——真正高保真的 iPod 基座

![](img/24d78b93a1112e6e834a4d7c54a93f78.png)

大约有无数(实际数字)的 iPod 基座，今天我们带给你更多。然而，这一个声称是不同的。所有这些 200-500 美元的人都可以去踩，因为 Sonic Integrity 已经用他们 1000 美元的 TubePod 加强了游戏。在这一点上，仅仅是名字前面没有“我”这个事实就让它物有所值。但是这一个实际上也比其他的都有进步。它的功率输出为 13Wx2，频率响应为 20Hz–20k Hz(1.5 db)，谐波失真为 0.5%，信噪比为 86Db。它甚至包括一个“清洁用的刷子/手套”只有两个书架扬声器，但我们听说这个东西听起来非常非常好。
