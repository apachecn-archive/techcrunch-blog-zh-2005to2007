# 雅虎收购 MyBlogLog。不，他们没有。等等，是的。

 [](https://web.archive.org/web/20220815032048/http://www.mybloglog.com/) 好的，这是官方消息，并从雅虎得到证实:他们购买了 [MyBlogLog](https://web.archive.org/web/20220815032048/http://www.mybloglog.com/) 。这是第一次[传言发生在 11 月](https://web.archive.org/web/20220815032048/http://www.beta.techcrunch.com/2006/11/17/mybloglog-to-be-acquired-by-yahoo/)，但从未得到证实，我们更新了我们的帖子来反映这一点。今天早上这条新闻又被爆出，但随后立即被撤下。现在又弹出了。由于我不知道这一次会不会成功，我给雅虎产品战略副总裁布拉德利·霍洛维茨打了电话，他确认了这次收购。交易敲定时，奥姆·马利克实际上正在与联合创始人斯科特·拉夫共进晚餐，所以这是一笔确定的交易。

MyBlogLog 将成为雅虎开发者网络的一部分，并向 Chad Dickerson 汇报(后者向 Bradley 汇报)。该交易的规模尚未得到证实，尽管有传言称在 1000 万到 1200 万美元之间。

MyBlogLog 将会出现在雅虎开发者网络中，这是一个很好的信号。从用户关怀的角度来看，雅虎最近的收购处理得相当好，看起来他们也将采取类似的方式。布拉德利确实说过，最终用户将能够用他们的雅虎 id 登录。这将大大扩大潜在用户群。

祝贺 MyBlogLog 团队。这东西像火箭一样起飞，再也没有回头，我们是一个自豪的成员。
