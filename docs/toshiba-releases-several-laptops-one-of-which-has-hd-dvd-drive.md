# 东芝发布了几款笔记本电脑，其中一款配备了 HD DVD 硬盘

[![qosmioav12.jpg](img/76ed8c07ac9a405fe0428c114e5fe5e2.png)](https://web.archive.org/web/20210226111940/https://beta.techcrunch.com/wp-content/uploads/2007/06/qosmioav12.jpg "qosmioav12.jpg")

我喜欢公司同时发布一系列产品。这要么意味着 1)他们想让电视广播充斥他们产品的新闻，要么意味着 2)没有一个产品是那么热门，所以他们同时发布所有产品以减轻痛苦。我的观点是，这里没有其他人。对，所以，东芝刚刚宣布了一个 HD DVD-有 Qosmio 和一些新的卫星。Qosmio F45-AV412，以下简称 AV412，是一款多媒体主力产品，配有前面提到的 HD DVD 驱动器，以及 HDMI 输出和宽敞的 250GB 硬盘，尽管转速仅为 4,200RPM。它也有 802.11n .义！

至于卫星，他们失去了 HD DVD 驱动器，只有 802.11b/g。然而，他们确实在一段时间内第一次使用 AMD 处理器。

嗯…带上 iPhone！
