# 合成音铃声 RIP

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/10/polyphonic-ringtones-rip/>

![ultimatecd1.jpg](img/093b3dc103493c61473288d8ad2ac575.png)音乐媒体观察曾报道，复调铃声已死。虽然我们甚至不知道这是病，我们觉得我们需要说几句话:

复音铃声有一个很好的生活。它让无数的手机用户能够骚扰同事、电影赞助人和周围的任何人，当过度使用的莫扎特曲调、洛基的主题或其他简单的音乐提示突然开始提醒来电时。复音铃声是移动行业的主力，就像 SMS 一样被开发出来，尽管事实上它从来都不是真正需要的。

然而，复调铃声设法取得了很大进展。这是第一次一个大众市场产业被创造出来，几乎独立于主要唱片公司，直接向个人客户销售音乐。更重要的是，它利用了那个时代的技术，包括类似 MIDI 的文件大小，最后，复调铃声让音乐家的工作可能会减少到在婚礼乐队中演奏。

我们不知道什么会完全取代彩铃。它已经有了许多(非婚生)子女，如 MP3 铃声、回铃音和其他烦人的声音提示。复音铃声会被一些人怀念。更重要的是，我们知道，只要有那些廉价的人保留着他们的旧手机，我们就知道在某个地方，有人会接到电话，我们会听到你的铃声！
