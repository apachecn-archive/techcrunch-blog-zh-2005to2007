# 部落被收购了，这次是真的

对于总部位于硅谷的 Tribe.net 来说，去年是动荡的一年。创始 CEO 马克·平卡斯于 2005 年 4 月被[赶下](https://web.archive.org/web/20221129042825/http://www.siliconbeat.com/entries/2005/04/07/silicon_valley_shakeups_continue_tribe_replaces_pincus.html)，然后[于 2006 年 8 月重返](https://web.archive.org/web/20221129042825/http://www.beta.techcrunch.com/2006/08/24/is-mark-pincus-back-at-tribe/)。平卡斯离开后，有传言称该公司已被 NBC 收购，但交易从未完成。

但是现在《纽约时报》报道说，这个八人公司的资产已经被思科收购。两周前，他们收购了社交网络基础设施服务公司的五家公司。他们希望利用这两家公司的技术来帮助思科的企业客户建立自己的社交网络，所以还不清楚部落服务本身是否会继续存在。

部落交易的价格尚未披露，但通常这些事情会在几天内泄露。

需求是显而易见的，正如路透社今天发布的公告所示，他们正在寻找建立自己的 Myspace 克隆版，用于金融类。如今，似乎每个人都想拥有自己的社交网络。

思科现在是社交网络领域的一员。这肯定不在任何人对 2007 年的预测中。随着思科涉足这项新业务，硅谷周围肯定会有一些窃笑声。

宁(刚刚升级了自己动手搭建的社交网络平台)的创始人马克·安德森在《NYT》的文章中就这个话题发表了精彩的观点。他说:“思科将成为社交网络领域的一支力量，这种想法就像宁成为光开关领域的一支力量一样可信。”
