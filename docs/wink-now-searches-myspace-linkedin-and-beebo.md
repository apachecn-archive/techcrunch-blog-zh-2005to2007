# Wink 现在搜索 MySpace、LinkedIn 和 Bebo

 [](https://web.archive.org/web/20220813215109/http://wink.com/) 社交搜索网站 [Wink](https://web.archive.org/web/20220813215109/http://wink.com/) 本周末发布了一项名为人物搜索的新功能，我认为这将是该公司的一大举措。Wink People Search 搜索 [MySpace](https://web.archive.org/web/20220813215109/http://myspace.com/) 、 [LinkedIn](https://web.archive.org/web/20220813215109/http://linkedin.com/) 和 [Bebo](https://web.archive.org/web/20220813215109/http://bebo.com/) 的用户资料。它不是网站自身搜索功能的混搭，而是这三个社交网站上超过 1 亿份个人资料的原始索引。

Wink 表示，它将每两周向人物搜索添加新的社交网络，这些网络将由注册 Wink 用户投票决定。随着小众社交网络的激增，聚合搜索变得非常智能。来自开放 ID 社区的人应该在 Wink 人物搜索的结果页面上买一个大广告。

许多社交网站的匿名性质使得用它们来发现老朋友很困难，尽管不是不可能。Wink 表示，人物搜索的主要用途是在社交网络中寻找兴趣相似的人。结果可以通过网络、性别、年龄和单身/已有状态进行过滤。年轻人会想靠兴趣跨网络搜索吗？我不确定。营销人员和研究人员会吗？我想他们会的。这些网站会如何看待 Wink 的广告与其用户资料的搜索结果相冲突？这可能会引起一些关注，但我相信，带有链接的简短汇总摘录通常被认为是投放广告的公平竞争。

在不知道如何使用它的情况下，人民搜索给我的感觉就是太酷了。它现在集成了基本的 Wink 书签和共享功能；Wink 通过允许您在其他用户的书签收藏中进行搜索来增强谷歌搜索。我们之前对 [Wink 的报道在这里](https://web.archive.org/web/20220813215109/https://beta.techcrunch.com/tag/wink)。该公司去年从寒武纪风险投资公司、Greylock Partners 和天使投资公司筹集了 620 万美元的资金。他们告诉我们，他们有一个良好的，缓慢的燃烧率，这是伟大的-这让他们有时间来提出和实施坚实的功能，如人民搜索。这种价值主张可能会驱使大量的人眨眼，从而增加其基本社交搜索的用户。社交搜索可能还不足以独立存在，所以 Wink 的明智之举是开始构建类似人们在它周围搜索的东西。

更新:一些人显然觉得同时搜索多个社交网络令人不快。那对我来说没有意义。如果需要去多个网站搜索是你和危险之间的唯一障碍，你就不太安全。我认为这将对邪恶的活动产生零影响，并将对研究和评估不同的社交网络产生最大的影响，以了解在你感兴趣的领域哪些是最强的。
