# 用例:公司如何正确使用图片共享

当谈到硬件制造商试图利用社交网络/共享网站的兴奋时，我们 CrunchGear 的 gearheads 相当怀疑——我想到了电话运营商和 MySpace 但这里有一个实际上相当不错的例子。

尼康翻遍了 Flickr，挑了几个很棒的摄影师，发给他们 [D80s](https://web.archive.org/web/20190616220657/http://www.nikonusa.com/template.php?cat=1&grp=2&productNr=25412) ，真是不错的 DSLR。然后，他们在一些广告中使用他们拍摄的照片，并创建了一个华丽的网站，详细介绍每个摄影师，从业余爱好者到专业的婚礼摄影师。

这种事情让每个人都很高兴:Flickr 作为人才孵化器获得了一些认可，尼康获得了一些好感，酷摄影师获得了不错的相机。现在，如果有人会在 YouTube 上搜索下一个 NBC 新闻主播，那么我们会有所进展。
