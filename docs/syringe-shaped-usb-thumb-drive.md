# 注射器形状的 USB 拇指驱动器

USB 闪存驱动器多如牛毛，这意味着公司需要增加他们的产品来获得消费者的认可。一种方法是把它做得很大，像 T2 东芝的 16GB T3 硬盘。另一种是把你的塑造成注射器的形状。是的，我想我们都知道我们想要哪一个。

注射器驱动器符合 USB 2.0 标准，具有高达 1GB 的空间，此外，它还让机场安检部门非常满意。
