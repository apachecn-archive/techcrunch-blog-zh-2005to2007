# 想买搜索引擎？

一个伟大的元搜索引擎 [Jux2](https://web.archive.org/web/20221001145536/http://www.jux2.com/) 的创始人昨天把它放到[易趣](https://web.archive.org/web/20221001145536/http://offer.ebay.com/ws/eBayISAPI.dll?ViewBids&item=7554979251)上出售。约翰·巴特利昨天也写了这篇文章。

创始人之一(也是我的朋友)Aaref Hilaly 发邮件告诉我这件事，他说创始人没有足够的时间投入到这个项目中去，处理增长问题。

起步价？0.01 美元，无保留。现价？还剩 7 天 26100 美元。

Jux2 今年早些时候获得了搜索引擎观察的[最佳元数据搜索引擎奖](https://web.archive.org/web/20221001145536/http://searchenginewatch.com/awards/article.php/3494141#meta)。这是并排查看大型搜索引擎结果的一个很好的方式，并且有一个非常干净和可用的界面。
