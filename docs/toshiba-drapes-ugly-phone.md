# 东芝窗帘丑陋的手机

![](img/26c977c10f883a37c3c8e0dc51591fb7.png)

东芝今天推出了由 KDDI 开发的丑陋手机 DRAPE。虽然她外表看起来不怎么样，但内在才是最重要的，对吧？这款窗帘配有一个 320 万像素的摄像头，包括自动对焦、数字变焦和夜间模式，可在弱光条件下拍摄。像往常一样，这个布帘还可以作为 MP3 播放器。目前还不知道这款产品何时何地上市，但如果让我猜的话，我会说:永远不会在美国上市，很快会在日本上市。
