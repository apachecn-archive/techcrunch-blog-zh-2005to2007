# 老年人现在有了一个闲逛的地方

Monster.com 创始人杰夫·泰勒今天创办了一家名为 [Eons](https://web.archive.org/web/20210306022551/http://www.eons.com/) 的新公司。但是除非你至少 50 岁，否则不要注册——这是一个为老年人群服务的社交网络。杰夫本人只有 45 岁，太年轻了，不能使用自己的服务。

这个想法很好——这是一个拥有大量社交时间的受众群体——但执行起来很糟糕。在许多方面，Eons 是在拥抱新网络(社交网络博客，等等)的宣传，而不是拥抱精神。简而言之，Eons 试图做的太多了，我不知道它到底想做什么。

他们还在兜售一个讣告功能，这个功能看起来既噱头又有点令人反感。该部分包括值得注意的死亡和额外的功能，如你所在位置附近的死亡，都有品位地包装在殡仪馆，医疗保险和网飞(？)广告。

但除了噱头，正如我在上面所说的，我对该网站的主要问题是，他们试图成为一个博客，一个 myspace 类型的主页，一个待办事项列表和一个用户门户，以及其他许多东西。一下子扔给人太多了，这个年龄段完全有能力用雅虎做门户，用 Vox 做博客平台等等。我可能会从 Myspace 的角度出发，随着用户的增加而增加功能，这种模式在许多其他创业公司都很成功。做一件事比其他人都好。

成功的企业家往往在新的创业公司上遇到麻烦——Eons 看起来正在走这条路。更多关于 [CNN Money](https://web.archive.org/web/20210306022551/http://money.cnn.com/blogs/browser/2006/08/myspace-for-baby-boomers-launches.html) 。
