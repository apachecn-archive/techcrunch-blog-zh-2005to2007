# 英伟达:没有超频的 G80 给你

据 Inquirer 报道，由于 GeForce 7900 GT 的回报太多，Nvidia 禁止其合作伙伴超频即将推出的 G80 卡。该网站通过提出 Nvidia 如何在双 GPU 7950 GX2 卡上锁定超频来支持该报告。然而，I4U 称之为扯淡，因为它拥有超频的 XFX 7950 GX2 XXX。

这一点我不得不同意 I4U。有很多厂商靠工厂超频卡赚钱，完全切断市场并不是最明智的举动。让我们打赌这只是 Nvidia 的恐吓战术，这样它就可以销售更贵的更快时钟速度的卡。
