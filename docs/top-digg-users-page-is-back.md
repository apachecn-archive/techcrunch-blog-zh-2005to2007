# Digg 顶级用户页面又回来了

Digg 的凯文·罗斯昨天宣布，他们将从 Digg 网站上移除顶级用户名单。如今，这份名单又回来了，但不是在 Digg 上。

为 Digg 的竞争对手 Netscape 工作的 Christopher Finke 正在从个人资料中剔除用户排名，并将用户链接到 Digg 的首页故事。他在这里整理了自己的清单。

这些信息仍然可用，因为 Digg 仍然在用户的个人资料页面上显示每个用户的排名。简单地浏览所有的用户页面就会显示出 Digg 的顶级用户，而且这些数据每天都会更新。

我想这份名单会比以前更受关注。告诉人们他们得不到的东西，他们会更加想要。
