# 松下用 microSD 卡玩俄罗斯嵌套玩偶

我们有一部手机、一个 GPS 装置和一部掌上电脑，它们都使用不同大小的 SD 卡。手机使用 microSD，GPS 单元，有一个大屏幕用于照片播放，使用 miniSD，Palm 使用标准 SD。即使它们都是 SD 的，你也不能在不购买花哨的适配器的情况下共享它们。

松下已经解决了我们的问题，它在新的 microSD 卡中安装了迷你和标准 SD 适配器。这不仅仅是为了在设备之间共享，而且现在当我们在旅途中时，我们可以在笔记本电脑包中随身携带一个 SD 读卡器。

谢谢你，松下，让我们这些极客的生活变得简单多了。
