# 索尼推出 PSP 摄像头 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/22/sony-unveils-psp-cam/>

![](img/56b30f61f7b98f4b9e2cb372308c8cb3.png)

索尼什么时候会放弃 PSP 配件，开始专注于实际的游戏？也许这样任天堂 DS 就不会用 PSP 游戏擦屁股了。接下来，索尼将于 11 月发布 PSPJ-15003。PSPJ-15003 是一款 130 万像素的 PSP 相机，分辨率为 640 x 480 像素。640 x 480？因为普通手机可以以那样的速度(或者更高的速度)拍摄分辨率，所以真的没有理由购买这个。不过，如果你想让你的 PSP 变得更有趣，你只需花费 44 美元。至少它能让你免于购买 PSP 游戏，但愿不会如此。
