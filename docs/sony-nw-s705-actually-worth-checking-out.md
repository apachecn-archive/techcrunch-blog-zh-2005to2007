# 索尼 NW-S705 实际上值得一试

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/10/sony-nw-s705-actually-worth-checking-out/>

尽管我非常喜欢索尼的设计，但它仍然处于数字音频播放器的最底层，至少在这个国家是这样。索尼可能不值得在这一点上有任何类型的打击，但在阅读了 CNET 亚洲评论后，我对 NW-S705 有点兴奋。2GB 网络随身听当然是闪亮漂亮的，有机发光二极管屏幕，接口不易出错。但最酷的可能是索尼将降噪麦克风内置于播放器本身，而不是依赖集成在耳机内的庞大处理器单元。

评论称音质也很好，它的额定电池寿命为 47 小时，还有一些很酷的可定制选项，如两个用户定义的均衡器预设(大多数 DAP 只有一个)。最大的缺点是，你必须使用索尼的 SonicStage 软件来传输歌曲，这与其他闪存 MP3 播放器相比速度较慢。
