# LG VX9900 enV 来到威瑞森

据 Gizmodo 报道，LG VX9900，显然被称为 enV，将于 11 月 27 日发布。这款智能手机具有 200 万像素的摄像头、EV-DO、VZNavigator、microSD 和双立体声扬声器。

传言零售价为 320 美元，但如果你愿意把你生命中的两年典当给威瑞森，你可以把它降到更现实的 150 美元。
