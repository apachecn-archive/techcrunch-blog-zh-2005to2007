# Nvidia 8800GTX 真的真的很好；ATI 又开始追赶了。

你能通过一个人的 GPU 散热器和风扇组件的尺寸来判断他吗？因为，哇，这东西太大了。闪亮，黑色，漂亮，但是很大。无论如何，我猜它在即将到来的顶级 768MB Nvidia 8800GTX 上是如此之大，因为该卡就是如此强大。DailyTech 得到了它肮脏的基准测试手，并使用《半条命 2:迷失海岸》、《地震 4》、《猎物》和《3DMark06》进行了基准测试。由 2GB 内存、10，000rpm Raptor 驱动器和英特尔酷睿 2 至尊 QX6700 四核处理器支持的卡使 ATI 目前的性能冠军镭龙 X1950 XTX 看起来像一个傻瓜，在所有测试中轻松击败它。

关于该卡的其他一些有趣的事实:它需要两个 PCI Express 电源连接器，占用两个扩展槽，并具有两个 SLI 桥夹连接器，该网站推测，这将允许制造商在一个系统中安装三个这样的卡。
