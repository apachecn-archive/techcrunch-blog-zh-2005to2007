# OLPC 1000 台笔记本电脑接近量产

在绿色桌面的映衬下，你在照片中几乎看不到它，但这是作为“每个孩子一台笔记本电脑(OLPC)”计划的一部分，在上海装配线上下线的 1000 台[广达 XO 2B1 笔记本电脑中的第一台](https://web.archive.org/web/20130627213125/http://crunchgear.com/2006/08/28/olpc-laptop-no-longer-cm1-now-2b1/)。

第一批 1000 台机器将进行严格的环境测试，以模拟它们最终将被使用的恶劣条件。测试将包括 XOs 从不同高度跌落，儿童敲击键盘，并确保外壳防尘。开发人员也可以使用调试工具来测试构成笔记本操作系统的应用程序。

在这一轮测试之后，下一批产品将送到阿根廷、巴西、利比亚、尼日利亚和泰国的学校儿童手中，让他们在真实生活条件下接受锻炼，这是 2007 年夏季开始大规模生产之前的最后一个测试阶段。
