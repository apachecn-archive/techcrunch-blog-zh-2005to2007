# Tellme 推出免费 411 服务| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/16/tellme-launches-free-411-service/>

![tellme.jpg](img/8825cf0af21eefb8652e4662e523524f.png) [Tellme Networks](https://web.archive.org/web/20160407215623/http://www.tellme.com/) 今天在 Web 2.0 博览会上宣布，它将增加三种新的方式来访问其早在一月份推出的[信息服务。该服务的功能类似于 411，但用户可以通过语音、短信或移动网络请求信息。该声明是在谷歌宣布其免费 411 服务](https://web.archive.org/web/20160407215623/http://www.techcrunch.com/2007/01/23/tellme-mobile-launches/) [Goog-411](https://web.archive.org/web/20160407215623/http://labs.google.com/goog411/index.html) 后一周发布的。

要通过语音使用该服务，只需拨打 1.800.555。告诉并说出“企业搜索”即可从任何电话上查找企业名录或搜索特定类别，如“花店”。短信和移动网络同样简单。向 TELLM 发送一条短信，详细说明您正在寻找的业务。该服务将以包含地图链接的详细信息做出响应。如果你不想耗尽你的短信配额，快速访问 www.tellme.com 应该会产生类似的结果。

[TechCrunch 最近报道](https://web.archive.org/web/20160407215623/http://www.techcrunch.com/2007/03/14/microsoft-acquires-tellme/)微软已经收购了 Tellme Networks，Inc. Tellme 声称负责接听美国近 80%的自动 411 电话。这项收购将于 4 月 30 日生效。
