# 威瑞森获得黑莓 8830 世界电话

对于威瑞森的顾客来说，今天是个不错的日子。它刚刚宣布将携带新的黑莓 8830 气旋。这款新设备具有 CDMA 和 GSM 功能，可以在美国和欧洲使用。

这款手机配有 QVGA 屏幕、microSD 和 EV-DO。没有相机，但我不知道有多少人会用手机拍照，所以这没什么大不了的。

还没有关于价格或可用性的消息。
