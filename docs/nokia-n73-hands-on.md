# 诺基亚 N73 动手操作

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/24/nokia-n73-hands-on/>

自从 8260 T2 之后，我就再也没有拥有过诺基亚，那是很久以前的事了。我对此没有意见，但我对 AT & T wireless 被 Cingular 收购有意见。有人记得收到短信是免费的吗？花一些时间在 N73 上是一种回家的感觉，但我觉得自己像一个 85 岁的老人试图学习如何使用电脑。这个 3D 蛇是怎么回事？Lifeblog，这是 wtf？！

一如既往，诺基亚是市场上最友好的手机，我只花了几分钟就浏览了用户界面。N73 包含了如此多的功能和应用程序，有时我会不知所措，这在手机上很少发生。诺基亚喜欢把这个宝石称为“多媒体计算机”,我倾向于同意这种说法，但请记住，它很好地完成了它的主要工作——作为一部电话。通话清晰无误。

我觉得没有必要介绍太多，因为你们大多数人都很清楚这个时代手机自带的简单功能，所以让我们聊聊这款手机自带的很酷的新功能和应用。

在此之前，我将讨论几个老问题，但值得一提。N73 上的蓝牙没什么特别的，但我还没有一部手机能够将媒体传输到我的 Mac 上，所以利用这一功能非常有趣。我通过蓝牙传输所有的照片和视频，从来没有遇到过故障。有些人可能会认为 WiFi 不是一个必要的功能，但出于一些奇怪的原因，我渴望它。N73 缺乏这一点，但 T-Mo 的优势发挥了作用，所以我不能抱怨太多。HTML/XHTML 浏览器做得和任何手机浏览器一样好，但是我被宠坏了，所以我没怎么用它。

N73 还支持电子邮件功能，这似乎是这个时代的必备功能。尽管我很想远离电子邮件，做一个偶尔查看邮件的“正常人”，但我做不到。带有附件的 SMTP、IMAP4 和 POP3 帐户以及 VPN 都在这款 N 系列产品上运行。无聊的老东西说够了；是时候吃好吃的了。

N73 上的 320 万像素摄像头非常好用！卡尔·蔡司光学镜头、自动对焦功能和 20 倍数码变焦只是 N73 的众多选项中的冰山一角。前置摄像头没什么可夸的，画质也不值一提，但拥有两个摄像头总是一个不错的卖点。画质是 VGA，但确实有 2 倍变焦。主摄像头对比度，亮度，曝光，白平衡和色调都可以调整，我认为，这对于一个拍照手机来说已经很多了。您还可以使用自拍器，为最多六张照片调整序列模式，还有一个减少红眼的 LED 闪光灯。我们还在谈论诺基亚 N73，对吗？的确如此。该相机还具有白天和夜晚的正常模式，以及三种图像质量设置。相机有多好？看看这张照片。您还可以捕获 MP4 或 3GP 格式的带声音的视频，如下所示。

用卡尔·蔡司 320 万像素相机拍摄的一张照片

这个坏男孩上的应用程序很可爱。例如，LifeBlog 是一个多媒体日志工具，它可以同步你的照片和所有信息，这样你也可以成为一个博客写手。Quickoffice 可以让你阅读 Word、Powerpoint 和 Excel 文档，还有一个 PDF 阅读器，所以商务人士绝对可以不用一部成熟的智能手机。PIM 有日历、计算器、时钟等基本功能。

板载内存限制为 42MB，但 miniSD 插槽允许扩展。音乐播放器完成工作，它支持 MP3，AAC，WMA，M4A 和 eAAC+文件。内置的均衡器是一个很好的触摸，我承认我用了播放器几天，很喜欢它。只要你拿着耳机，调频收音机也很棒，因为调谐器内置在耳机中。请放心，N73 拥有你所期望的所有功能，但我决定不考虑这些，而是专注于主要功能。如果你想知道更多的细节，请给我发邮件。

蓝色灯光！！！

用负色调拍摄的视频和最后几秒钟在电视上被放大。我有巨大的鼻孔。

总的来说，N73 是一个受欢迎的惊喜，大大超出了我的预期。我从未遇到过诺基亚的问题，如果你想买一部基本上是智能手机的非智能手机，N73 是一个安全的选择。你不会失望的。它现在已经上市，零售价为 429.99 美元。
