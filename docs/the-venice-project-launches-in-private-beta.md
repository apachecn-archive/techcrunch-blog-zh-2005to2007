# 威尼斯项目在私人测试中启动

[![veniceproject_logo.jpg](img/dbc50996193403497a0121b0b7d3fa79.png)](https://web.archive.org/web/20210806044526/http://www.theveniceproject.com/)[威尼斯计划](https://web.archive.org/web/20210806044526/http://www.theveniceproject.com/)的测试版今天推出。这是由 Kazaa 和 Skype 的创始人尼可拉斯·曾斯特罗姆和杰纳斯·弗里斯创办的互联网电视公司。

TechCrunch 第一次[得到这个项目的风声](https://web.archive.org/web/20210806044526/http://www.beta.techcrunch.com/2006/10/14/the-stars-align-for-the-venice-project/)是在十月份。

“我们开始尝试融合电视和互联网的精华，我认为我们刚刚在漫长的旅程中迈出了一大步，”弗里斯今天在他的博客上说。“几个月来，我们一直在一小群人中间悄悄进行测试。现在，我们将扩大这个圈子，邀请越来越多的人。如果你想尝试一下，可以从现有的测试者那里获得邀请。在接下来的几个月里，终端用户和内容提供商将会带来更多的产品特性。更重要的是，越来越多的内容。您现在看到的是您将找到的一些内容类别的预览。”

[GigaOm](https://web.archive.org/web/20210806044526/http://gigaom.com/) 正在[报道](https://web.archive.org/web/20210806044526/http://gigaom.com/2006/12/12/venice-project-letting-in-beta-testers/#more-7546)早期的测试者并没有留下深刻的印象，称该网站有一个“糟糕的界面，没有关于按钮含义的文字描述，而且视频质量起伏很大。”
