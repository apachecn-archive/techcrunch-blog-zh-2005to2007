# 松下的新 4GB 3CCD 摄像机

![s150leica.jpg](img/2bbe2e32e9293010519f91fd7b493c22.png)

松下的 SDR-S150 SD 有许多与竞争对手不同的特点。它具有 4GB 存储的 MPEG2 录制功能，可容纳长达 3 小时 20 分钟的视频。

除了录制功能，摄像机 SD 还采用了徕卡迪考马镜头和先进的纯色引擎，以提供令人满意的图像质量。SDR-S150 SD 将于 9 月上市，建议零售价为 999.95 美元。
