# Treo 680:现在有更多无聊的内容

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/06/treo-680-now-featuring-more-boring/>

在摩托罗拉，我们还得到消息，广受欢迎的 Treo 680 现在可以不带摄像头了。为什么会有人想要一个没有相机的 Treo？因为在一些企业环境下更安全，这就是原因。

在许多企业环境中，比如说，在 RIAA，偷偷抓拍并发送照片的能力是一场噩梦，让有安全意识的人夜不能寐。但是随着现代手机摄像头的普及，不可能有一个 100%安全的环境。不过，在公司计划中发放不包括摄像头的手机，至少会让公司自己承担更少的责任。但实际上，这只会让手机更差劲。抱歉，公司计划类型，你太差劲了。等等，你有免费的 Treo？
