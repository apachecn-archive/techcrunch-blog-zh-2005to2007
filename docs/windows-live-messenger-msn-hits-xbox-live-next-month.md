# Windows Live Messenger (MSN)将于下月推出 Xbox Live

[![msn360.jpg](img/51750ee6f8da9c4aed0f76e86e509c30.png)](https://web.archive.org/web/20230322164148/https://techcrunch.com/wp-content/uploads/2007/04/msn360.jpg "msn360.jpg")

从下个月开始，Xbox Live 用户将能够相互发送即时消息，并与所有 Windows Live Messenger (MSN)好友发送即时消息，无论他们是在 PC、Windows mobile 设备上还是在 360 上。你可以使用屏幕键盘或者你身边的任何一个通用串行总线键盘来打字。Xbox Live 的春季更新不仅为在线服务带来了 Windows Live Messenger，还带来了一个改进的 Xbox Live 街机(你可以看到你的朋友正在玩什么街机游戏)和一个改进的成就通知:它不会只说“成就解锁”，它会显示准确的解锁内容和多少玩家积分。

更新将于 5 月 7 日上线，就在微软发布其混合控制器+键盘产品的几个月前。
