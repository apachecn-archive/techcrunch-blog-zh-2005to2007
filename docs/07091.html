O2令人敬畏的智能手机是山中之王

偶尔，一部手机会影响到隔海相望的网络，这是如此的糟糕，以至于我们的流行文化参考或幼稚的夸张都无法公正地对待它。一个这样的设备是德国O2网络的火焰。我们喜欢21点，我们喜欢Treos，我们喜欢智能手机，但这个傻瓜属于自己的类别。也许是Geniusphone，因为这个怪物什么都不缺。

因此，我们不会描述它有多么坚如磐石、令人垂涎三尺，我们只是给你规格，让你自己解决它。注意，并随时做笔记。

  • 3.6寸VGA(！？)触摸屏
  • 520MHz Xscale CPu
  • 英伟达GoForce 5500 GPU
  • UMTS/HSDPA(欧洲风味，抱歉)
  • 支持MPEG视频的200万像素摄像头
  • 2GB内部存储
  • 802.11磅/克
  • 带SRS环绕声的电视输出(！？)
  • 移动USB
  • Windows Mobile 5:手机版
  • Powered by the blood of virgins

    其中一个是假的，但我们会让你找出是哪一个。然而，其余的都是实话。这东西比某些人的笔记本电脑还强大。最重要的是，它可以打电话。我们可能永远也不会在美国看到这一幕，但我们应该知道，每当他们在欧洲或亚洲突破极限时，我们都会在这里产生甜蜜的涟漪效应。这意味着全息激光驱动的蒸汽发电WiMax蓝牙耳机可能会在我们的未来。
