# Toufee 推出快速 Flash 电影制作

 [](https://web.archive.org/web/20210920044440/http://toufee.com/) [Toufee](https://web.archive.org/web/20210920044440/http://toufee.com/) 是一个新的所见即所得的在线 Flash 电影生成器，旨在让非技术用户轻松快速地创建令人印象深刻的多媒体项目，并嵌入到网页中。一些关键功能包括拖放元素的位置，对演示时间的良好控制，以及具有六种相对人类声音的文本到语音转换引擎。我认为这里有很大的潜力——不缺乏那些想要在没有复杂或昂贵的桌面软件的情况下制作 Flash 演示文稿的人。

如果你读过我们对令人印象深刻的多媒体工具 [Scrapblog](https://web.archive.org/web/20210920044440/https://beta.techcrunch.com/tag/scrapblog) 的评论，Toufee 更加轻量级，面向短片。如果你读了我们对非常令人兴奋的 Flash 电影创作工具 [TagLoops](https://web.archive.org/web/20210920044440/https://beta.techcrunch.com/tag/tagloops) 的评论，Toufee 要简单、易用和成熟得多。一些演示样本非常有趣，据说只需要几分钟就可以完成，可以在 Toufee 网站上查看[。](https://web.archive.org/web/20210920044440/http://toufee.com/samples.html)

该产品是完全托管，在点和点击选择元素和媒体选项丰富，并产生非常小的文件。支持 Flickr 和 YouTube 导入。用起来很好玩。该公司旨在取悦从 MySpacers 到销售专家和易贝用户的每一个人。

该产品由新德里的 Sanchit Bhatnagar 和 Navneet Rai 全职开发。如果 Toufee 能够在性能和稳定性方面站稳脚跟，它可以在目前由 Slide 和 RockYou 这样的公司主导的领域做得很好。Toufee 还没有出现，但它显示了很大的潜力。

工具响应和与浏览器的交互可能需要一些工作。Toufee 也更容易学。该服务也可能使用名称更改。如果 Toufee 这个词有什么特别的含义，我不知道它是什么。我忍不住想象豆腐做的假发，但也许这只是我。

总体来说，我觉得 Toufee 是一个值得看的公司和产品。一旦它站稳脚跟，我认为它将满足一个重要的需求。
