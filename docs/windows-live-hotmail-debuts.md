# Windows Live Hotmail 首次亮相

Hotmail 可能是 Web 1.0 的第一个重大成功，它已经发展了。

微软宣布在全球范围内推出 36 种语言的 Windows Live Hotmail 的[版本](https://web.archive.org/web/20220628230727/http://www.microsoft.com/presspass/press/2007/may07/05-06WLHotmailLaunchPR.mspx),并辅以 AJAX 技术。

这项新服务是对之前 Hotmail 服务的巨大改进，吸收了超过 2000 万测试人员的意见。然而，今天的产品仅仅是从测试版到正式推出的一步。

我们之前报道过 Windows Live Hotmail:早在二月迈克尔·阿灵顿查看了当时的测试服务，结果并不乐观，新的 Hotmail 排名第三，仅次于 Gmail 和 Yahoo！邮件。

Windows Live Hotmail 最初拥有 3000 万用户，在未来几个月内将增加超过 2.5 亿用户。

新的 Hotmail 基于全新的代码，标志着 Windows Live 品牌的持续整合。

未来几周，微软将发布一款新的免费邮件客户端，旨在取代 Outlook Express、Windows Mail 和 Windows Live desktop。新的客户端被称为 Windows Live Mail，将是一个桌面客户端，在用户桌面上本地提供 Hotmail 的功能集。它还将支持非 Hotmail 帐户的管理。

登录新的 Hotmail 还有一些额外的好处；支持是内置的，允许即时通讯的接触，或 VOIP 呼叫接触，与一次点击，这也将配合与微软的桌面即时通讯平台。
