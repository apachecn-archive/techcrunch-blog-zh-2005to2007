# 索尼-爱立信样机泄露

![](img/54b43e84ec81b8afec372b29ce7f6d8e.png)

一款新的索尼-爱立信手机出现在波兰的拍卖网站上，这款手机看起来非常漂亮。卖家将这款手机列为“索尼 X123i”，这在我们听来像是一个原型模型或完全虚构的名字。显然，这款手机将成为 W850 的继任者，我们喜欢我们所看到的。这款橘红色滑块的规格包括 1670 万彩色 LCD、3G、200 万像素主摄像头和用于视频会议的 VGA 前置摄像头，以及半英寸厚的设计。

关于这款手机的任何官方细节还没有消息，但是对它的需求似乎相当高。目前的出价约为 711 美元，预计还会上涨。我们的常驻波兰人约翰看了整版报纸，没有发现任何可疑之处——例如，他们卖的是实物，而不是照片——但你必须去罗兹才能买到。斯马茨内戈！
