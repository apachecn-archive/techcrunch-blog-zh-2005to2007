Novint获得VTech来制造猎鹰游戏控制器

原文：https://web.archive.org/web/http://techcrunch.com/2006/11/14/novint-gets-vtech-to-build-falcon-game-controller/

诺维特获得伟易达建立猎鹰游戏控制器


Novint Technologies Falcon game controller是第一款让消费者计算应用实现高保真交互式三维触摸的控制器。(诚然，有些东西我不知道我需要，但现在觉得我必须要。)猎鹰让用户在玩启用的游戏时感受重量、形状、纹理、尺寸、动态、3D运动和力的效果。Novint已经与手机制造商VTech签订合同，处理生产任务，所以即使它要到2007年才开始发货，它至少会在发货。
