# 奥林巴斯 SP-550UZ 动手——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/28/olympus-sp-550uz-hands-on/>

![](img/6505df08e81f18577749842370c3aeed.png)
我的天，你的变焦镜头真大。

盒子上写着 18x 广角变焦镜头。作为一个变焦类型的人，这是一见钟情。奥林巴斯 SP-550UZ 是第一款宽 18 倍光学变焦(28 毫米至 504 毫米)，范围从真正的广角到长焦，毫无疑问非常棒。事实上这很神奇；28 毫米比市场上的任何紧凑型车都要宽，504 毫米无疑比任何竞争对手都更接近您。
 即使使用单反相机，你也需要两个独立的镜头来实现这种灵活性。周期性地出现一种数码相机，试图既小巧又单反。奥林巴斯 SP-550UZ 就是这样一款相机。当你考虑到买数码相机时只有两种选择时，你会觉得很棒。你要么买一个小巧的可以放在手掌里拍出一般照片的相机，要么买一个需要两只手才能拍出好照片的单反相机。不管怎样，这都是一种妥协，很糟糕。

![](img/84c03af6728cbfb0b801bd064bb66ef4.png)

一架变焦很大的好相机。710 万像素，原始拍摄，准确的色彩和取景器是我们所寻找的，这就是你用 SP-550UZ 得到的。EVF(电动取景器)比主液晶显示器暗，但对比度很好，非常可行。当然，像任何液晶显示器一样，快速移动需要重新绘制时间，但这是非常好的。当你按下快门时，它会消失一会儿，以便在回来之前刷新。主 LCD 很漂亮，亮度为 2.5 英寸(230，000 像素)，可以在光线变化时快速调整。当然，你仍然有标准的曝光模式，程序，光圈优先快门优先和手动，全自动，场景和指南模式。

尽管 550UZ 大部分是塑料，但橡胶握持和金属装饰让它感觉很结实。镜头筒很坚固，但长得惊人，变焦时有一个很好的平滑机制。按钮和刻度盘直观且符合逻辑。内置闪光灯弹出很好看。它需要 4 节 AA 电池，允许大约 450 次拍摄。

550UZ 配备了一系列令人印象深刻的功能，但遗憾的是，有一点炒作的肉。我越来越多地看到这种趋势，相机制造商沉迷于营销的双重谈话和古怪的解释。奥林巴斯不幸也不例外；奥林巴斯声称 550UZ 的自吹自擂的突发速率为 15fps。从表面上看，这非常令人印象深刻，但为了完成这一壮举，550UZ 以 120 万像素拍摄，使大肆宣传的爆炸成为泡影。

如果你想要一个全分辨率的连拍，那么你必须在两秒钟内将连拍减慢到三帧。更夸张的是 ISO 速度高达 3200 或 5000，但在那个 ISO 下，你的分辨率只有 310 万像素。相机的 ISO 速度不可靠，经常导致模糊和颗粒状的拍摄。事实上，只有较低的 ISO 速度是可用的。你在开玩笑吗？！

双重图像稳定也是如此:它工作得很好，但事实上双重部分是一种延伸。传感器移位机械是检测相机运动并移动 CCD 传感器进行补偿以减少模糊的硬件。当使用长变焦长度或在低光下拍摄时，当快门速度较慢时有效且重要。另一半，BS 部分，是软件。这是 ISO 提升，让你更快的快门速度与一些增强的噪声处理。没什么大不了的。

此外，我没有看到高质量(HQ)和超高质量 JPEG 图像设置之间的任何差异，所以我会坚持 HQ，并在 xD 存储卡上获得三倍的拍摄量。xD 卡！只有富士胶片和奥林巴斯使用 xD 卡(尽管富士胶片正在其较新的相机上切换到 SD/xD 插槽),虽然这些卡工作正常，但它肯定是一种模糊的格式。至于内存，550UZ 的板载内存为 20MB。

奥林巴斯的自动对焦一直很慢，虽然 550UZ 比奥林巴斯以前的型号要好，但我不认为它快，只是过得去，白天比晚上快。

“我的模式”功能允许您保存多达四种设置，从测光模式到色彩调节和 ISO 速度，所有设置都非常先进。然后是向导模式，对于那些不想考虑这些模式意味着什么的用户来说。这个常见拍摄情况的列表允许用户选择情况，相机选择模式。想要拍摄背光或模糊背景等等。这是一个很容易理解的功能，但是人们会使用它吗？我怀疑是否有很多人知道。如果他们不能被模式所困扰，他们中有多少人会偶然发现这个特性？

别误会，我喜欢这个相机。这是一个很好的消费相机，变焦很厉害，但它不是制造梦遗的相机。
