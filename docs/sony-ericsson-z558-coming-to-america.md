# 索尼爱立信 Z558 登陆美国

已获 FCC 批准。你可能还记得，Z558 是一款基本的翻盖手机，除了 dope 触摸屏和触控笔。由于这款手机原本是为中国市场设计的，(触摸屏对汉字来说似乎更容易些)，我们从未想过会在美国看到它。有了 T9，我们为什么还需要触摸屏？

显然索尼爱立信认为我们会。这是低端手机中高端功能的又一次融合，[很像诺基亚 6085](https://web.archive.org/web/20201125113450/http://crunchgear.com/2006/09/13/nokia-6085-high-end-features-on-a-consumer-clamshell/) 。这可能只是因为手机制造商在手机中塞进了太多的功能，使得手机过于昂贵。通过让手机各有一两个特殊功能，消费者可以挑选他们想要的。这可能会成为一种趋势，一种让我们的父母不仅仅使用手机打电话的方式。
