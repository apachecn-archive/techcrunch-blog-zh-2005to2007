# 雅虎！Messenger 发布 Mac IM 3.0

[![](img/32f1d19a4677272ed78301816acf468d.png)](https://web.archive.org/web/20221002134034/http://messenger.yahoo.com/mac.php) 雅虎！IM 团队刚刚发布了雅虎的 3.0 版本。我支持麦克。雅虎！Mac 版 Messenger 最后一次更新是在 2003 年 9 月。它的功能集很有趣，但它似乎缺乏 PC 版最引人注目的两个品质: [VOIP](https://web.archive.org/web/20221002134034/http://www.beta.techcrunch.com/2006/03/21/new-voip-yahoo-messenger-launches-tomorrow/) 和基于[最近发布的软件开发工具包](https://web.archive.org/web/20221002134034/http://www.beta.techcrunch.com/2006/06/19/yahoo-opens-im-to-developers/)构建的不断增长的[插件库](https://web.archive.org/web/20221002134034/http://us.gallery.yahoo.com/messenger)。当我可以使用 Adium 同时连接所有主要的即时通讯系统时，我很难想象在我的 Mac 上使用任何一个即时通讯客户端。

![](img/70531c73f33837dfed4c7170edf6dbac.png)以下是它的内容:

*   可可风格的新界面，非常 OS X 化。
*   头像和动画表情，我相信这将吸引广大观众。
*   网络摄像头支持，兼容 PC 和 Macs 尽管我个人在不想亲密交流时会发即时消息。
*   自定义隐形设置，例如，如果我不想和我的前女友说话，除了她以外的所有人都可以看到我有空。
*   使用[咆哮](https://web.archive.org/web/20221002134034/http://growl.info/)系统的视觉通知。
*   雅虎即时消息提醒！日历、邮件和个人信息。对他们来说，这是个好主意，但是任何 RSS 提要的即时消息提醒会更好。

即将推出:与 Windows Live (MSN)用户聊天。如果 IM 兼容性方面哪怕出现一丁点的进步，那都将是美好的一天。

我将非常好奇该公司承诺的未来功能中是否包括可供 PC 用户使用的 VOIP 和插件。
