索尼推出PSP摄像头TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2006/09/22/sony-unveils-psp-cam/

索尼什么时候会放弃PSP配件，开始专注于实际的游戏？也许这样任天堂DS就不会用PSP游戏擦屁股了。接下来，索尼将于11月发布PSPJ-15003。PSPJ-15003是一款130万像素的PSP相机，分辨率为640 x 480像素。640 x 480？因为普通手机可以以那样的速度(或者更高的速度)拍摄分辨率，所以真的没有理由购买这个。不过，如果你想让你的PSP变得更有趣，你只需花费44美元。至少它能让你免于购买PSP游戏，但愿不会如此。
