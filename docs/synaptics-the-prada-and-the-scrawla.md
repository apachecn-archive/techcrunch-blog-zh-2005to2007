# Synaptics:普拉达和 Scrawla

今天，我心血来潮，顺便去了趟 Synaptics，幸运地看到了 LG 普拉达的原貌:躺在一张桌子上，没人喜欢。所缺少的只是一张信用卡和一些打击，我们将有一个手机的真实未来环境的图像。但是我跑题了。
 ![](img/a9d052492f3f1e6cf630ee05a5518d8f.png)
Synaptics 在屏幕上制作了[电容式](https://web.archive.org/web/20201127060220/http://en.wikipedia.org/wiki/Capacitor)传感器，让你可以在普拉达手机的用户界面上移动一条小鱼。
![](img/3ac854508a54493f23cd4cab2782d44b.png)

然而，一个很酷的东西是这个奇怪的小俄罗斯手机。你只需在键盘上“画”出你的信息，而不必键入。这些按钮兼作触摸板，再加上手写识别，你就有了一个相当酷的组合。狡猾，狡猾的俄罗斯人。
