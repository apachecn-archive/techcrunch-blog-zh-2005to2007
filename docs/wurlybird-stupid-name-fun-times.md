# WurlyBird——愚蠢的名字，有趣的时代

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/05/wurlybird-stupid-name-fun-times/>

![](img/7a42cef12bc7cdb3450fe13e1878e7a9.png)
十恶不赦！

虽然这个名字听起来像是你愿意花 10 美元去找妓女的东西，但 WurlyBird 实际上是为孩子们设计的。这基本上是一个室内的公平乘坐，让你旋转，直到你呕吐或下车。3-10 岁的孩子可以加入这项活动，在打碎客厅里的所有瓷器的同时享受一次小小的乐趣。

信不信由你，WurlyBird 不会让你落后太多。基础版只要 195 美元，豪华版 300 美元。还是没印象？[偷看这个视频](https://web.archive.org/web/20160925073631/http://www.wurlybird.co.uk/video.htm)我有一种感觉，你会改变主意。
