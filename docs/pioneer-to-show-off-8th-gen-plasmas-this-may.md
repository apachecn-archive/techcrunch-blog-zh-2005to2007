# 今年五月率先展示第八代等离子

![](img/19cece6c72a727588d147b8f11f1b282.png)

先锋遍布等离子电视，看起来它将于今年五月在罗马的一次活动中推出其第八代等离子电视。先锋在 CES 上一直在说它如何想让等离子成为首要的电视技术，这一声明应该最终会揭示它一直在做什么。

过去我对等离子有点失望，但像先锋和松下这样的公司，昨天在纽约这里展示了他们新的等离子生产线，已经将技术改进到我会买一台的程度。也就是说，如果我有钱的话。(等离子还是贵一点，你看。)
