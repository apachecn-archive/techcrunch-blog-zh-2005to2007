# SNOCAP 宣布与主要唱片公司结盟

[![snocap.jpg](img/e084ab8648e62f1f4024e5c7f7f0c884.png)](https://web.archive.org/web/20221129055624/http://snocap.com/) 早在九月[我们就报道过【MySpace 已经开始允许艺术家使用](https://web.archive.org/web/20221129055624/http://www.beta.techcrunch.com/2006/09/02/myspace-gets-into-music-biz/) [SNOCAP](https://web.archive.org/web/20221129055624/http://snocap.com/) MyStores 直接销售音乐。这项服务使用 MP3，没有任何形式的数字版权管理。由于无数字版权管理的格式，当时还不清楚 MySpace 和 SNOCAP 是否会与任何主要的唱片公司达成协议，因为他们通常更喜欢将他们的曲目放在胸前。

然而，随着 SXSW 的 SNOCAP 发布了一个重要公告[,这一切都改变了。该公司今天透露，它现在将提供 Sub Pop、New Line 和 Dangerbird 等主要独立唱片公司的歌曲。虽然乍一看这似乎无关痛痒，但我应该指出，所有这些标签都在华纳的保护伞下运作，这使得这一声明更加美味。](https://web.archive.org/web/20221129055624/http://snocap.com/press/releases/?id=27)

也许我的评估过于急切了，但我认为这是 DRM 大坝的一个关键突破。大量的听众从 MySpace 上获得他们的音乐，我敢肯定，随着 MySpace 提供流行音乐的无版权管理拷贝，这个数字将会成倍增长。我相信，在不久的将来，我们将开始看到更多来自主要唱片公司的声明。
