# NewsGator Go！:现在针对 Java 和 Blackberry

NewsGator Go！以前只适用于 Windows Mobile，现在适用于黑莓和支持 Java 的手机，包括诺基亚、摩托罗拉和三星的手机。

这个版本与你的在线 [NewsGator](https://web.archive.org/web/20210303052522/http://www.newsgator.com/) 账户同步，并缓存文章供以后离线使用。你也可以通过程序的界面用电子邮件发送故事。
