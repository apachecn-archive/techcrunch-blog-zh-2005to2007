# TechPresident:跟踪候选人对 MySpace、YouTube 和 AdSense 的使用

[](https://web.archive.org/web/20220815081158/http://techpresident.com/)[tech president](https://web.archive.org/web/20220815081158/http://techpresident.com/)是一个新的在线项目，跟踪和汇总所有美国总统候选人使用在线社交媒体的信息。尽管该小组由比其他任何人都多的民主党支持者组成，但该小组的研究已经产生了一些有趣的发现。

候选人对 MySpace，YouTube，Google 和 Yahoo！到目前为止，广告是网站上被观看的主要内容。网站的侧边栏显示了每个候选人在 MySpace 上好友列表的增长情况。巴拉克·奥巴马在民主党中领先，增长最快，得克萨斯人罗恩·保罗在共和党中处于同样的位置(注意到一个统计异常，否则有利于米特·罗姆尼)。昨天一篇题为“YouTube:谁得到了它？”记录候选人对媒体的使用。除了丹尼斯·库钦奇(他也相信政府资助的飞机轨迹是用来控制天气的)之外，所有民主党候选人都在 YouTube 上有官方频道，而共和党中只有米特·罗姆尼有。

在其他有趣的发现中，只有六名候选人购买了搜索引擎关键词广告，而共和党人在这方面做得更好。只有共和党人在雅虎上购买广告！据报道，除了谷歌，只有共和党人会购买搜索对手名字的广告。

TechPresident 是个人民主论坛的一个项目。其他贡献者包括扎克·埃克斯利，在线组织和沟通主任，凯瑞/爱德华兹 04 年，迈克·特克，前共和党全国委员会和布什-切尼 04 年的电子竞选主任，林恩·d·约翰逊，FastCompany.com 高级编辑，视频博客明星史蒂夫·加菲尔德和 NetCentricCampaigns.com 的鲁比·辛里奇等人。
