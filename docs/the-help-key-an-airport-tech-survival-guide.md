# 帮助关键:机场技术生存指南

作为一个非常、非常、非常频繁的旅行者，我得到了一些如何处理现代航空系统弱点的建议。如今，公司可能会尽最大努力让你失去免费 Wi-Fi 和一个体面的座位，但这并不意味着你不能继续领先。点击跳转，查看我多年来收集的一些在现代机场生存的技巧。

**挑选最佳座位:**

选择飞机座位是一件冒险的事——很难知道 14A 是否会倾斜，直到你登机，但为时已晚。补救措施:参观 Seatguru.com。输入你的航空公司和飞机模型，它会准确地告诉你要什么样的座位，要避免什么样的座位。

记住一些在网站上反复出现的常见陷阱。例如，虽然紧急出口排提供了最好的腿部空间，但直接在他们前面的座位往往不能倾斜。此外，除了一些例外，飞机后面的座位通常更窄，厕所旁边的座位也容易闻到气味。

**如何找到免费的机场 Wi-Fi:**

如今，在机场找到免费 Wi-Fi 比找到便宜的食物还难。除了少数例外，大多数机场都试图每天收取你八九美元，以换取你在航班起飞前半小时查看电子邮件的特权。但是也有免费冲浪的方法:

在捷蓝航空的枢纽机场，比如纽约的 JFK，他们通常会在航站楼区域提供免费 Wi-Fi。把你的笔记本电脑放在附近，吸收阳光。否则，你最好的选择是靠近航空公司的豪华会员休息室。这些家伙总是有无线网络，而且很少用密码保护。进入红地毯俱乐部的范围，尽情冲浪。

**保持充电:**

我开始注意到一个令人不安的趋势:机场禁用电源插座，以防止人们喝免费果汁。这是一种失控的廉价在纽约的拉瓜迪亚机场是一个特别糟糕的问题，那里的营业网点少之又少。那么，如何解决这个问题呢？寻找喷泉。这些吸盘需要插上电源，插座通常是成对的，这意味着机场不能在不打开另一个插座的情况下打开喷泉的插座。只需将笔记本电脑插入饮水机旁边的插头，然后开机。
