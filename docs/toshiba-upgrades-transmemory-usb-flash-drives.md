# 东芝升级 Transmemory USB 闪存驱动器

![8gb_usb_readyboostapp1500px.jpg](img/1ea26ab9d372c4fe56493584275268fa.png)
Windows Vista 用户会很高兴地知道，东芝已经宣布了 Windows ReadyBoost Transmemory 盘的 U2K 系列。为了提高 Vista 机器的性能，新的驱动器现在将充当外部存储器。理论上这是一个好主意，但我怀疑任何东西都会帮助 Vista。U2K 系列将于今年 10 月推出 1GB、2GB、4GB 和 8GB(200 美元)容量。限量版 32GB(665 美元)Transmemory 硬盘将于 12 月上市。
