# Plextor PlexEraser 可以销毁光盘，因此您不必担心

我把收到的无数美国在线光盘扔进办公室的费罗斯碎纸机，从中获得了相当大的乐趣。不幸的是，这产生了一大堆塑料碎片，并使覆盖光盘的有机染料逸出，并可能造成环境破坏。Plextor 旨在通过其独立的 PlexEraser PX-OE100E 驱动器解决这一问题。这款售价 249 美元的设备专为金融、政府/军事和研究等行业以及拥有敏感信息的大公司设计，它使用激光束来销毁存储在光盘上的所有数据，并支持 12 厘米和 8 厘米两种尺寸的八种光盘格式。这可能不如把它们扔进碎纸机那么令人满意，也没有那么快，因为销毁一张光盘只需要大约三分钟，但至少它让它们可以回收利用，不会有助于创造三眼鱼。
