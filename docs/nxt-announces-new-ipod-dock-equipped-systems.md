# NXT 发布新的配备 iPod Dock 的系统

NXT 今天宣布推出两款新的超薄高保真系统，都配有 iPod 基座。这两种新产品(CD-X10i 和 MC-DX22i)利用了深厚的 SurfaceSound 技术，将从一对薄扬声器中发出完整的声音。

该系统具有带背光的多功能 LCD 显示屏和带定时器和睡眠模式的时钟。扬声器和主单元可壁挂安装，低音炮结构紧凑，易于隐藏。支持 CD-MP3/CD-R/RW 播放，以及带有预设内存的 AM/FM 调谐器，使这些产品更加完美。CD-X10i 和 MCDX22i 的建议价格分别为 249 美元和 299 美元。
