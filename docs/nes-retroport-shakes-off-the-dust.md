# NES RetroPort 抖落灰尘

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/07/nes-retroport-shakes-off-the-dust/>

面对现实吧，当我们玩马里奥和俄罗斯方块时，我们的手和大脑都停留在过去，Wii-motes 就是不能满足我们。按钮的间距都是关闭的，这只是简单的尴尬。肌肉记忆可能是一个婊子，但如果你有 NES 控制器躺在周围，我知道任天堂粉丝这样做，那么你很幸运。RetroPort 可以让你通过 USB 端口在 Wii 上即插即用你原来的 NES 控制器，直到现在我还没有发现它的用处。仅售 22 美元。
