# 来自微软| TechCrunch 的 XNA Game Studio Express

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/14/xna-game-studio-express-from-microsoft/>

![](img/4b236f3064c421007d92b52cdea29d5e.png)

给所有 C#开发者提个醒。微软的新开发套件 XNA Game Studio Express 现在允许你同时为 Windows 和 Xbox 360 编写应用程序。您最终只是为每个平台使用两个独立的编译器来完成这项工作。自由游戏开发者请注意，这是将你正在开发的基于 Windows 的游戏移植到 360 的一个好方法。
