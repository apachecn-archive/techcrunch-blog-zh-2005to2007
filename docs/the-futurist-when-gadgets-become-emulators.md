# 未来主义者:当小工具变成模拟器时

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/28/the-futurist-when-gadgets-become-emulators/>

整个文化界都知道，苹果公司将在周五大张旗鼓地发布 iPhone。当失业者和富人在工作日排队等着最终拿到他们 600 美元的宣传盒时，他们将第一次发现，“iPod”不再仅仅指他们的口袋大小的 MP3 播放器，还指另一个设备中的软件程序。

这只能被视为技术行业的一种趋势:曾经独立的硬件项目越来越多地被移植到简单的软件仿真器中，这些仿真器有可能被加载到许多设备上。

这样的例子比比皆是。以 ESPN 移动为例。在 MVNO 倒闭之前，他们的十几个客户花了几个本雅明才买到一部界面与最新体育赛事合拍的手机。今天，几乎任何威瑞森的用户(至少那些携带相对较新的手机的用户)都可以加载 V-Cast 并下载“ESPN 移动”:一个小型的 BREW 应用程序，它模仿了过去只有特定硬件才能实现的功能。虽然是“iPod ”,但这个程序目前只能在一个非 iPod 的硬件上找到:iPhone。但是如果苹果喜欢，我想将程序移植到 Windows Mobile 或 Symbian 会非常容易。

虽然小工具融合并不是什么新鲜事，非 iPhone 手机也已经播放了一段时间，但有趣的是著名的品牌硬件设备退化为纯粹的软件程序。由于分发可下载软件的成本和库存风险与把一个坚固的小玩意放在商店货架上的必要投资相比可以忽略不计，这也可以为那些商业上失败的设备延长寿命，但仍然建立了狂热的粉丝基础。这一点在视频游戏模拟器上已经很明显了——即使市场不同意你的 TurboGrafx-16 crush，你仍然可以随时在你的 PC 上玩 Bonk。即使没有其他人喜欢 Gizmondo，重温它的辉煌岁月不是很棒吗？)在智能手机的安全范围内？

这种硬件仿真对我们未来几年将携带的小工具有着巨大的影响。可以想象，在几年内，我们可能会得到很少或没有内置功能的空白设备。这将取决于我们，最终用户，挑选我们想要我们的小工具做什么，并相应地安装硬件模拟器。想让它玩游戏？安装“PSP”想和一些剪影一起跳舞吗？加载“iPod”
