# YouTube 移动上传 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/05/10/youtube-mobile-uploading/>

 [](https://web.archive.org/web/20221007223457/http://www.youtube.com/) [ YouTube 今天宣布了](https://web.archive.org/web/20221007223457/http://www.youtube.com/t/pr_060510_mobile)一项新功能，允许从支持多媒体消息服务(MMS)的移动设备上传视频。用户需要在上传视频前创建一个移动档案。[在这里报名](https://web.archive.org/web/20221007223457/http://www.youtube.com/mobile)。

我一直用拍照手机直接上传图片到 Flickr(比如昨天的香蕉黄可笑[租车](https://web.archive.org/web/20221007223457/http://www.flickr.com/photos/michaelarrington/143529790/))。如果我的手机有制作视频的功能，我也会一直用它来看 YouTube。

顺便说一句，我永远也不会明白为什么 Flickr 会把视频市场拱手让给 YouTube。也许他们(Flickr)知道一些 YouTube 不知道的关于这项业务的真实成本。
