# 潘多拉发布分享功能

[![](img/31441dd5ed3079f13fc4b39c1ace76b5.png) ](https://web.archive.org/web/20220626073556/http://www.crunchbase.com/company/pandora) [潘多拉](https://web.archive.org/web/20220626073556/http://www.crunchbase.com/company/pandora)，最近成为[与竞争对手 Last.fm 激烈比较的主题](https://web.archive.org/web/20220626073556/http://www.beta.techcrunch.com/2006/01/23/655/)，今晚将发布许多旨在分享的新功能。

许多功能被添加到用户的“收藏夹”页面(你听过的你喜欢的歌曲的书签列表)。收藏夹页面的一个例子是这里的。用户现在可以在他们的收藏夹页面(实际上是任何人的收藏夹页面——每个用户都有一个 URL)上试听 30 秒钟的任何歌曲。每首歌曲还链接到乐队和歌曲的摘要。最后，当您访问其他用户的收藏夹页面时，您可以点击并添加他们创建的任何电台。潘多拉上的一个电台是根据用户喜好高度定制的音乐源。共享电台变得简单多了。

Pandora 还增加了许多 RSS 订阅选项。有五类源:

*   针对特定听众的收藏歌曲提要:跟踪你的朋友正在发现的歌曲
*   针对听众的电台提要:保存一份你朋友的潘多拉电台列表
*   Pandora 上播放的顶级歌曲的 Pandora 范围提要
*   在潘多拉上播放的顶级艺术家的潘多拉范围提要
*   在潘多拉上播放的最受欢迎的电台的潘多拉范围的提要

潘多拉还增加了键盘控制和链接网站的方式。我真正想看到的是一个将潘多拉播放器直接添加到另一个网站的 API。
