# 三星 MM-X5:iPod 基座

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/12/samsung-mm-x5-an-ipod-dock/>

![dock_good.jpg](img/b5a297fd8f65c1c874280470fe27362d.png)

需要闹钟吗？收音机调谐器？CD 播放器？一个 iPod 基座？好吧，三星用他们 199 美元的 MM-X5 覆盖了你。它还可以从 USB 驱动器播放音频，看起来像达斯·维德的坐浴盆。它应该在 10 月份上市，但你也可以在等待的时候获得其他 5000 个时钟收音机/iPod 基座中的任何一个。
