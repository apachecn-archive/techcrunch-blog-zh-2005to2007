# 三洋 MAJ-U02 犹太爆破工| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/02/sanyo-maj-u02-ghetto-blaster/>

MAJ-U02 配备了 CD 播放器、USB 端口、卡式录音机(太棒了！)、数字 FM 调谐器和一个 SD/MMC 插槽。扬声器还配备了一个低音反射系统，一个带低音增强的放大器(所有贫民区爆破手都需要)，以及一个光学数字音频输出。但是我真的想花 250 美元买一个能播放我的旧混音带的东西吗？可能没有，但你应该听说过“我们去睡觉吧”是如何演变成“阳光下的水泡”的经典。
