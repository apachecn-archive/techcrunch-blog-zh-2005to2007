# 威瑞森无线有助于安全出行

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/25/verizon-wireless-helps-for-safe-travel/>

今年夏天，威瑞森无线公司为旅行者提供一些帮助。威瑞森正在提供几个附加选项，以方便您上路时的夏季旅行计划，包括 VZ 导航器，这是一个支持 GPS 的 Get It Now 应用程序，可以将手机转换为导航设备。VZ 导航器的无限月租费为 9.99 美元，在未启用 Get It 功能的手机上使用一天的费用为 2.99 美元。还提供路边援助服务，每月 3 美元，全年 365 天，每天 24 小时提供紧急援助。这种帮助包括牵引、锁定服务、燃料输送和轮胎服务。

该公司还要求客户通过在任何威瑞森无线商店捐赠他们的旧无线电话来帮助其他需要安全和保密通信的人。这些电话将会捐给家庭暴力受害者的希望热线项目。
