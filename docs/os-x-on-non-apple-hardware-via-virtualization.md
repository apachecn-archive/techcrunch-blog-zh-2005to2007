# 通过虚拟化在非苹果硬件上实现| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/24/os-x-on-non-apple-hardware-via-virtualization/>

随着苹果电脑从 PowerPC 转向基于英特尔的硬件，兼容 X86 的 OSX 版本在互联网上流传，允许任何人在个人电脑上安装苹果操作系统(你的也包括在内)。这是苹果不想要的东西，但是随着 VMware 和 Parallels 都致力于 Mac 平台的虚拟化，这很有可能成为现实。

除了允许 Windows 在 Mac 电脑的窗口中运行，虚拟化阵营也来自另一个方向:在戴尔或惠普电脑上运行的 OSX。这种前景可能会让苹果公司的高层夜不能寐，但我们认为虚拟化是一个强大的力量，不会因为硬件的变化而停止，所以今年晚些时候会有一些痛苦的诉讼或收购。
