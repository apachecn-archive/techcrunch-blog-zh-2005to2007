# V Moda 为 V-Day  更新 Vibe 耳塞

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/02/v-moda-updates-vibe-earbuds-for-v-day/>

情人节马上就要到了，V Moda 已经注意到了他们的红色和摩卡色耳塞 Vibe 系列的更新。如果你的女性朋友有一辆红色 Nano，那么这将是完美的搭配。如果她有一个便便的棕色 Zune，那么我认为摩卡咖啡将是一个很好的礼物，如果你喜欢谈论她有一个便便的棕色 Zune 的事实。在 V Moda、Virgin Mega Stores、Apple 和 Bloomies 的售价为 101 美元。
