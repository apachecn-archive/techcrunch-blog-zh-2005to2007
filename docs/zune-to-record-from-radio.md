# Zune 从电台录音？

![](img/97dd5f247b75d3f9bb48a56d36e36d49.png)
目前在[艺术家和唱片公司联盟](https://web.archive.org/web/20220204011316/http://www.aarcroyalties.com/)(类似于 RIAA，但不是)有一点大惊小怪，关于 Zune 从 FM 录音的能力。它声称录音应该被征税，因为 Zune 能够在其他 Zune 之间共享音乐，所以消费者可以分发他们不允许交换的音乐。

如果你觉得这不对，那你就对了。目前，Zune *无法从 FM*录音。这引发了一些猜测。

AARC 知道一些我们不知道的事情吗？从硬件的角度来看，Zune 不仅仅能够录制和编码 FM 流，这实际上只是升级固件的问题。

这是怎么回事？推测！
