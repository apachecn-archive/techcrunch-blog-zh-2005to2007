# Vista 的媒体中心远程预览

Vista 爱好者已经知道 Windows Media Center 观看和录制套件将包含在所有更高端的 Vista 版本中。媒体中心不再是一个独立的 SKU，现在已经为大众市场观众做好了准备，它的遥控器也是如此。就像所有大众市场技术一样，它是流线型的，更容易使用。

与旧的媒体中心遥控器相比，新的 Vista 遥控器的大多数按钮都完好无损:数字键盘、停止、播放、音量和频道键都在。底部的四个彩色按键很可能对应于 MCE2005 遥控器上的“录制的电视、指南、直播电视和 DVD 菜单”选项。我们喜欢老式的遥控器，但是把这些菜单键放在遥控器的最底部似乎很笨重。他们把音量、导航和播放控件放在你拇指所在的中间位置，这很好，但这四个重要的键需要移到更高的位置。

尽管如此，这款遥控器的特点是方形的边角，据报道，它和旧款遥控器一样容易握持。我们期待着亲自测试它。
