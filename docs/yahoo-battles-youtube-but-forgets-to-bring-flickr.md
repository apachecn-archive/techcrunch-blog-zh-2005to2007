# 雅虎与 YouTube 竞争，却忘了带 Flickr

 [](https://web.archive.org/web/20220817204351/http://video.yahoo.com/) 新的[雅虎视频](https://web.archive.org/web/20220817204351/http://video.yahoo.com/)产品[将于今晚(周三)晚上 9 点(太平洋标准时间)发布](https://web.archive.org/web/20220817204351/http://www.ysearchblog.com/archives/000307.html)。大变化？雅虎视频通过允许用户在网站上制作视频来瞄准 YouTube。最大的失望？该产品尚未集成到 Flickr 中。

雅虎视频将继续允许主要视频发行商直接访问该网站，并在网上搜索相关视频。但现在，普通互联网用户将有一个简单的方法来上传视频，并在雅虎视频上创建自己的视频博客频道。

上传的唯一限制是文件大小——每个文件不能超过 100 MB——但没有时间、带宽或总存储空间的限制。视频可以设置为公开或私人，观众可以根据流行度、类别(由雅虎预先定义)或标签(用户创建)搜索和排序视频。像 YouTube 一样，视频可以由观众评级、评论、嵌入到其他网站等。

与其他竞争对手相比，使用雅虎的最大好处可能是潜在的曝光率。当新的雅虎主页[发布时，受欢迎的用户制作的视频将定期出现在该主页的多个地方。](https://web.archive.org/web/20220817204351/http://www.beta.techcrunch.com/2006/05/16/yahoo-launches-new-home-page/)

与 YouTube 和其他视频网站不同，雅虎并不完全支持 Flash。他们从头开始构建上传工具，默认播放器是 Windows Media 格式。对于 Mac 用户和其他不能使用这种格式的用户，将自动使用 Flash 播放器。

这个产品很棒，肯定会很受欢迎——部分原因是因为这是雅虎，部分原因是因为他们创造了一个非常强大和完整的视频搜索工具，可以从网上抓取自己制作的视频和内容。但我真正想要的是看到视频上传功能直接包含在 Flickr 中，这是一个完全不同的用户群。人们会在雅虎视频上上传视频，以获得分发和名气。人们将视频上传到 Flickr，主要是与朋友和家人分享，并有一个安全的长期存储它们的地方。我对后者更感兴趣。
