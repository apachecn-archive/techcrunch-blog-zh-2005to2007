# 不包括电池，有印象吗？

![text.jpg](img/dfda151eb49661196cb6578fca6c4d69.png)

当我看到 Boynq Alibi 的那一刻，它立即引发了我童年的美好回忆。在我像往常一样离题之前，我先说明一下不在场证明。这是一个多合一设备，比 [Brando USB 集线器](https://web.archive.org/web/20210119095828/http://crunchgear.com/2007/08/06/brando-chromatic-usb-hub/)在你的桌子上清理出更多的空间。Alibi 将网络摄像头、扬声器和麦克风结合成一个时尚现代的东西。它现在在亚马逊和 Target 上售价 79.99 美元。言归正传。

还记得不包括电池的*吗*？这就像，有史以来最好的基于机器人的电影。不在场证明看起来是不是像一个可爱的机器人宝宝？你知道的。不太记得了？然后点击跳转进行快速刷新。
