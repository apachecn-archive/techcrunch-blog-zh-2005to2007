# Proporta 推出新的横向案例

![](img/1ea06920ea6b4393211480ed4c152c6f.png)

外壳和配件制造商 Proporta 发布了一款新的消费电子产品外壳设计。横向表壳设计首次亮相 Palm Treo 700。该保护套采用优质皮革和航空级铝材，保护屏幕和设备免受震动和刮擦。它还配有一个皮带夹，可以容纳两张 SD/MMC 卡。

这会花掉你大约 65 美元。但这是商业级的案子。所以，如果你真的想保护你的设备，你应该去看看。
