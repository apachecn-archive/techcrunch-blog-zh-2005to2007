# TrendNet ClearSky 蓝牙 Skype 手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/29/trendnet-clearsky-bluetooth-skype-handset/>

这款 TrendNet ClearSky Skype 手机配有一个可插入电脑的蓝牙加密狗。这对你有什么帮助？嗯，附带的软件连接到 Skype，然后通过蓝牙将音频传输到手机。表现如何？嗯，音质是“合理的自由滞后”，但你必须使用包括蓝牙加密狗，因为软件是直接捆绑到 ClearSky 实用程序。这是没有办法的。

这款手机的大小相当于一部大型手机。通话时有音量控制开关和静音控制。它通过迷你 USB 充电，但实际上它没有 AC 适配器。电池寿命为六小时通话时间和六十小时待机时间。如果您能忍受让电脑一直开着来转接您的 Skype 通话，这款手机可能适合您。否则，我们建议[选择使用 WiFi](https://web.archive.org/web/20160317230443/http://crunchgear.com/2006/08/21/belkin-wifi-skype-phone-review/) 的手机。
