# 纽约时报重新设计

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/04/03/new-york-times-redesign>

[](https://web.archive.org/web/20220815063311/http://www.nytimes.com/)[《纽约时报》](https://web.archive.org/web/20220815063311/http://www.nytimes.com/)几分钟前推出了一个重新设计的网站。许多变化是细微的，但它们增加了一些我非常欣赏的新功能。首先，他们现在在网站上有一个[专用区域](https://web.archive.org/web/20220815063311/http://video.on.nytimes.com/ifr_main.jsp?nsid=a-1d0baee9:10a5e4e7eae:33b8&st=1144052791187&mp=FLV&cpf=false&fr=040306_042714_w1d0baee9x10a5e4e7eaex33b9&rdm=929166.9710286823)来展示视频剪辑。更重要的是，他们有一个“[最受欢迎的](https://web.archive.org/web/20220815063311/http://www.nytimes.com/gst/mostpopular.html)”区域，包含了最多的电子邮件、最多的博客和最多的搜索文章。我喜欢《纽约时报》的发展方向——我也注意到他们正在尝试直接从文章链接到博客文章。

> 您将看到的一些变化和新功能是焕然一新的外观、简化的导航、视频和其他多媒体的扩展使用，以及查看其他读者正在查看、搜索和谈论的内容的更好方式。
