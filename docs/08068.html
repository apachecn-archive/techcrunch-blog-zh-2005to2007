kdice 是一款类似于风险的免费多人在线游戏，我刚刚浪费了一整个晚上玩它(而且大部分时间都在输)。它是由Ryan Dewsbury使用Google Web Toolkit 创建的，有一个Flash界面。Ryan还创建了gpoker，这是一个Ajax德州扑克游戏，也是用Web Toolkit构建的首批应用程序之一。早在7月份，我们就在这里介绍过gpoker。

kdice很像风险棋盘游戏，尽管游戏进行得非常快。像gpoker一样，kdice也有玩家之间的实时聊天。没什么可说的——这是一个有趣的应用程序，让我想起了小时候玩冒险游戏的时光。如果有人知道任何其他好的在线冒险游戏(除了大战略)，请告诉我。
