# 适用于 MAC 电脑的诺基亚媒体传输套件

终于，终于。诺基亚终于发布了 MAC 电脑的传输套件，这是大约 5000 年来一直非常缺乏的东西。你只需下载该应用程序，就可以开始从 iPhoto 和 iTunes 中拖动文件和音乐，以及从 Symbian 手机和应用程序安装中导入 iPhoto。
