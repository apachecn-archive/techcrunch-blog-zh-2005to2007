# TechMeme 开放档案——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/12/11/techmeme-opens-archives/>

 [](https://web.archive.org/web/20220128130415/http://www.techmeme.com/) 热门博客新闻网站 [TechMeme](https://web.archive.org/web/20220128130415/http://www.techmeme.com/) 今天上午推出了旗下各种[科技](https://web.archive.org/web/20220128130415/http://www.techmeme.com/river)[政治](https://web.archive.org/web/20220128130415/http://www.memeorandum.com/river)[棒球](https://web.archive.org/web/20220128130415/http://www.ballbug.com/river)和[名人八卦](https://web.archive.org/web/20220128130415/http://www.wesmirch.com/river)网站的[档案](https://web.archive.org/web/20220128130415/http://blog.memeorandum.com/061211/river)。新网站展示了过去五天的新闻标题的“新闻河”。创始人 Gabe Rivera 今天早些时候告诉我，他也计划以这种格式加入旧档案的链接。

这也是对新闻的另类看法。TechMeme 根据受欢迎程度(取决于有多少博客链接到一个头条新闻)对新闻条目进行排名，而新的新闻专题 river 只是按照时间倒序排列标题。这可能是一些用户的首选。

TechMeme 及其姐妹网站仍然没有任何搜索功能，我们已经多次敦促他们实现这一功能，以协助新闻研究。尽管如此，TechMeme 仍然是博客爱好者的必读网站。我们之前对公司的报道是[这里](https://web.archive.org/web/20220128130415/http://www.beta.techcrunch.com/tag/techmeme)。
