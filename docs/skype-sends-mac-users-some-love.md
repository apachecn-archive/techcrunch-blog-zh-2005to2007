# Skype 升级 Mac 版——现在有了视频

 [](https://web.archive.org/web/20211205111521/http://skype.com/) 感觉被 Windows 开发的 [Skype](https://web.archive.org/web/20211205111521/http://skype.com/) 抛弃的 Mac 用户可以振作起来了——该公司今天发布了[Skype for Mac 1.5 的测试版](https://web.archive.org/web/20211205111521/http://www.skype.com/download/skype/macosx/15beta.html)。视频支持终于包括在内，[虽然它是一个单独的下载](https://web.archive.org/web/20211205111521/http://share.skype.com/sites/mac/2006/07/video_has_arrived_if_youre_bra.html)。拖放联系人添加和文件传输都很流畅。您甚至可以将联系人列表中的某个人拖放到群聊中。

最大的变化是用户界面的外观和感觉。很好，Skype 感觉比以前流畅多了。这是一个测试版，所以可能会有更多的变化。虽然你可以在即时消息中留下语音邮件，但从聊天工具中发起呼叫的功能已经消失了。

![skypescreen1.png](img/c8938a58ec8fc1500f0379802b8e7ce7.png)

Skype for Windows 上次更新到 2.5 版是在 5 月。该版本支持 Skypcasts，即最多 100 人的群组通话。以前，非英特尔电脑上的 Skype 用户被限制为 6 名参与者(事实上，这是 AMD 传票的主题)。新的 Mac Skype 仍然不支持 SkypeCast，我的英特尔 Mac 仅限于与另外四个人通话。

这也是 Skype 的另一个版本，没有提供通话录音功能——这是竞争对手 Gizmo 项目的一个很棒的功能。

今天有消息称，至少根据一项研究，随着网络需求的增加，网络电话的质量正在下降，这个新版本的 Skype 让我觉得我想从媒体中得到更多。消费者网络电话已经存在了足够长的时间，很高兴看到易贝将其巨额收购变成真正了不起的东西。我觉得还没到那一步。
