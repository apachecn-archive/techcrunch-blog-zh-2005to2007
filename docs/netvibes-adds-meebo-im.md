# Netvibes 增加了 Meebo IM-

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/26/netvibes-adds-meebo-im/>

![](img/ea7be46609d166c54d87a51d1964431f.png) [Netvibes](https://web.archive.org/web/20220124072257/http://www.netvibes.com/) ，领先的在线 ajax 桌面之一，[今天](https://web.archive.org/web/20220124072257/http://blog.netvibes.com/?2006/07/25/71-netvibes-introduces-meebo-and-alexa-modules#c)宣布他们为 [Meebo](https://web.archive.org/web/20220124072257/http://www.meebo.com/) 增加了一个模块，这是一个跨平台的 web IM 系统。Netvibes 是让 RSS 在视觉上更具吸引力和可用性的一个好方法，但是在平台中包含 Meebo 在交互性方面更上一层楼。

使用多台电脑的人现在可以使用任何终端通过 AIM、MSN、Yahoo 和 GMail/Jabber 阅读他们在 NetVibes 和 IM 中的信息，只需一次登录。早期的集成有点不稳定，但是希望随着更多的使用和修补，事情会变得更好。这可能说起来容易做起来难，因为这里有太多的 ajax 了，但假设他们真的能做到，我认为这是一个伟大的合作伙伴。

Meebo 实现在 NetVibes 的一个单独的标签中打开，但是如果你在 NetVibes 的主标签中，它会发出声音通知你有新的即时消息。如果您使用的是浏览器以外的应用程序，也可以听到音频通知。这是一个很好的系统，很高兴看到这两个流行的工具一起工作。我预计这将是一个双方用户的大热门。

我们上一次写 Meebo 是在为 Firefox 和 Flock 开发扩展的时候。该扩展是由外部开发者创建的，但今天的迁移是内部进行的，正如围绕红杉资助的服务的许多创新一样。该公司的主要竞争对手是电子伙伴，其前身是电子信使。

总部设在巴黎和伦敦的 Netvibes [也得到了资助](https://web.archive.org/web/20220124072257/http://www.beta.techcrunch.com/2006/03/22/netvibes-to-announce-seed-financing-today/)，拥有数百万用户和整个[资源生态系统可以利用](https://web.archive.org/web/20220124072257/http://www.beta.techcrunch.com/2006/05/31/netvibes-gets-an-ecosystem/)。该公司是上个月 TechCrunch 在 SuperNova 主办的活动中的 [12 个互联创新者之一。Netvibes 面临着来自一长串 ajax 主页的竞争。](https://web.archive.org/web/20220124072257/http://www.beta.techcrunch.com/2006/07/03/the-supernova-12/)

这两家公司的合作在 web 服务集成的方向上迈出了非常酷的一步，这有助于这两家公司在各自的类别中脱颖而出。

**更新:**集成应该工作，这样一个单一的登录 NetVibes 也可以让你通过 Meebo 登录所有的 IM 帐户。当这种情况发生时，这些服务将真正很好地融合在一起，创造出比多个浏览器标签更流畅的体验。Meebo 告诉我这项功能即将推出。
