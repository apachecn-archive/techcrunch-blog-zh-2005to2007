# 松下 Toughbooks 获得升级

![](img/ea06bab8e4a12a9c62de78e6d746365a.png)
松下今天宣布推出 Toughbook 52，它将取代去年推出的 Toughbook S1。和以前的 Toughbooks 一样，它的设计可以承受泼溅和跌落——这款笔记本可以承受高达 2.5 英尺的跌落而不会出现裂缝。

然而，这一款比上一代有所改进，它给了你一些 Santa Rosa 芯片功能，内置手柄，更宽的屏幕和可选的内置 3G 无线功能(由威瑞森·EV-DO 提供)。

7 月份购买，价格为 1，699 美元(标准配置)，如果你希望它“针对改进的视频和 Vista 性能进行优化”，价格为 2，499 美元。
