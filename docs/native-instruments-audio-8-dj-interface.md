# 本地乐器音频 8 DJ 接口

![](img/4f37be98d1dcab009de76e30a77ee4c6.png)

专业的 DJ 都知道，在 2007 年，要在一家高级夜总会完成工作，需要两种以上的技术——1200 和一个 RANE 混音器(除非你是卡尔·考克斯)。所有音乐软件 elite 的制造商 Native Instruments 也知道这一点，并在 NAMM 展示了其新的数字 DJ 界面。Audio 8 DJ 接口带有四个 RCA 输入和输出、MIDI 输入/输出、麦克风输入和音频输出插孔、双唱机前置放大器(太棒了！)，以及 96KHz Cirrus 逻辑模数转换器。

这都是什么意思？这意味着无缝集成和原始的音频质量为您的轨道。如果你已经对这个美丽的音乐装置垂涎三尺，你最好拿张纸巾。Audio 8 DJ 接口要到 4 月份才会上市，售价 450 美元。
