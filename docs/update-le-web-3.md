# 更新:Le Web 3

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/16/update-le-web-3/>

[![Conferenceleweb3paris2006_1](img/30348677bf03306cf5de962a33cfce0e.png "Conferenceleweb3paris2006_1")](https://web.archive.org/web/20201202034001/http://www.leweb3.com/) 对于那些将要参加 Le Web 3 (30 名英国与会者，还在增加)的人，Loic 刚刚设立了一个公开的 [Google group LeWeb3](https://web.archive.org/web/20201202034001/http://groups.google.com/group/leweb3) 供任何与会者或对会议感兴趣的人围绕它进行对话。如果你错过了，他们已经把 [Planet LeWeb3](https://web.archive.org/web/20201202034001/http://planet.leweb3.com/) 放在一起，这是在 LeWeb3 注册的所有人的最近帖子的一页。
