# 日本移动电话将增加地震预警

法新社报道，日本最大的两家移动运营商 NTT DoCoMo 和 KDDI 宣布，他们正在开发一种先进的地震预警系统。这个系统会在地震发生前通知客户。该系统预计将在几个月内投入运行，并将传递来自日本气象厅的信息。还没有宣布将使用哪种类型的信息，但有人认为电子邮件可能会淹没移动网络。
