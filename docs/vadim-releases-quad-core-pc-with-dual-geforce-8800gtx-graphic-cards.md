# Vadim 发布配备双 GeForce 8800GTX 显卡的四核电脑| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/03/vadim-releases-quad-core-pc-with-dual-geforce-8800gtx-graphic-cards/>

![](img/3fa24b0a7f267abd288a8f712e782f97.png)

让英特尔酷睿 2 双核至尊四核游戏电脑的战斗开始吧！今天早些时候，我们报道了 VoodooPCs 全新四核[预兆](https://web.archive.org/web/20150701123229/http://crunchgear.com/2006/11/03/voodoopc-announces-quad-core-gaming-rig/)。现在，Vadim Computers 宣布，它也将发布一款配备四核 Core 2 Duo Extreme PC 的 PC 游戏设备。为了不被 VoodooPC 破坏，Vadim 钻机还配备了两个全新的 [GeForce 8800GTX 显卡](https://web.archive.org/web/20150701123229/http://crunchgear.com/2006/11/03/nvidia-8800gtx-is-really-really-good-ati-playing-catch-up-again/)。

除了四核处理器和显卡，Vadim rig 还将配备 2GB DDR2 RAM、两个 150GB Raptor HDs、一个 750GB 希捷 Barracuda HD、双层 DVD 刻录机、CD 刻录组合驱动器和一个创意 7.1 X-Fi Extreme Music 声卡。新的 Vadim 钻机中的所有硬件都是完全可定制的，这让我想到了钻机的另一个重要方面:它的价格标签是 8800 美元。该设备没有配备鼠标、键盘或显示器，但好消息是你可以获得两年的保修。是的，这个新的游戏装备可能比你的车更值钱，但是，嘿，这似乎是目前硬核电脑游戏的价格，不是吗？
