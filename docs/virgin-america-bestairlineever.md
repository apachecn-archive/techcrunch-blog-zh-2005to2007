# 维珍美国:有史以来最好的航空公司

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/08/virgin-america-bestairlineever/>

![virgin](img/169746a882ac6c8c3246abaeba963b1c.png)

Boing Boing 的 Xeni Jardin 参加了维珍航空的首航，并发布了一篇很棒的文章。哦，请，请，请让这成为对西北航空公司的最后一击。我已经等了那些落后贪婪的混蛋很多年了。

维珍美国的价格很低(旧金山到纽约 278 美元)，WiFi 接入，超级舒适的座椅，每个头枕内置的触摸屏信息娱乐系统，Doom(！)、谷歌地图、Linux、电影、电视、柔光。听起来熟悉吗？当然不会。这是所有其他航空公司都没有的。

维珍航空目前(或即将)在旧金山、洛杉矶、纽约、华盛顿特区和拉斯维加斯之间飞行。希望在不久的将来，美国的其他城市也会加入进来。
