# 如果你抄袭 Digg，至少感谢他们

 [](https://web.archive.org/web/20221207181221/http://www.supergu.com/) [go2web2](https://web.archive.org/web/20221207181221/http://go2web2.blogspot.com/2007/01/find-differences.html) ，通常是一个温和的博客，指出新的创业公司 [SuperGu](https://web.archive.org/web/20221207181221/http://www.supergu.com/) 复制了关于 Digg 的一切，除了他们的名字，并对他们有点反感。雪上加霜的是，他们[说](https://web.archive.org/web/20221207181221/https://beta.techcrunch.com/wp-content/uploads/2007/01/supergu565h.jpg)它“受到了 Meneame、Del.icio.us 和 Reddit 的启发”，完全忽略了 Digg。

截屏看起来几乎一模一样——super gu 不仅复制了 Digg 的理念，还复制了外观和感觉。我们将会看到 Digg 是否会激怒律师，还是会袖手旁观。
