# UMD:大 PDA 还是小笔记本？

起初我以为这位女士只是有一双男人的大手，但结果是 VIA NanoBook UMD 只有 1.2×6.7×9 英寸。这款 1.87 磅重的笔记本是与第一国际计算机公司(FIC)合作制造的参考设计，已被 Packard Bell 采用，作为 EasyNote XS 出现在欧洲市场。它目前正在 Computex 2007 上展示。

这款小巧的笔记本电脑采用 1.2GHz VIA C7-M ULV 处理器，由高达 1GB 的 DDR2 内存和 30GB 或 60GB 硬盘支持，可以挤出长达 5 小时的 Microsoft Windows XP 或 Vista Basic excitation 电池续航时间。

 ![005_opennohandsweb.jpg](img/76ccceb14257abc663234eb63e7001eb.png)
它有一个全键盘，配有我见过的最小的触摸板，一个 7 英寸的 WVGA 屏幕，内置 802.11g Wi-Fi，蓝牙和以太网支持，外加一个 DVI 和两个 USB 2.0 端口。显然，在屏幕旁边有一个 USB 插槽，用于各种附加模块，如世界时钟和日历、GPS 接收器、VoIP 和宽带无线。(我假设其中一个模块是彩色屏幕旁边的单色东西。)
