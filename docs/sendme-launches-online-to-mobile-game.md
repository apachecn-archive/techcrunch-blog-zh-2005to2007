# SendMe 推出在线移动游戏

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/19/sendme-launches-online-to-mobile-game/>

今天，直接面向消费者的移动娱乐提供商 SendMe 宣布推出 SoLow，这是一款在线移动游戏，玩家们可以通过竞争来到达底部。用户可以为奖品免费出价，参加抽奖，以独特的方式，最低的独特出价获胜。

SendMe 的联合创始人兼首席执行官拉塞尔·克莱因(Russell Klein)表示:“SoLow 背后的概念已经在世界其他地方爆发，拥有成千上万的玩家。“SoLow 在美国处于领先地位，为消费者带来了新的、简单的和有趣的东西，让他们可以随时随地参与其中。在我们的测试期间，我们发现用户非常喜欢抽奖活动，平均每个玩家都会为我们的赠品出价几十次。”

虽然这听起来好得令人难以置信，但关键是要有最低的独特出价。因此，每个人都会试着为几乎任何东西输入 1 美分，所以你需要找到别人没有选择的那个数字。奖品实际上是你可能真正喜欢的东西，包括任天堂 Wii 和索尼 Bravia 高清电视，甚至是去佛罗里达群岛度假。

拍卖是免费的，以前可以在网上进行，SoLow 现在允许用户通过短信发送他们的猜测。如果你有几分钟可以消磨，为什么不呢？你可能会赢得一次旅行或者一台新电视。
