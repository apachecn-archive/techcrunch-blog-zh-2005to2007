# 退出率下降

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/15/rakout-rates-drop/>

IP 语音(VoIP)已经成为一种以低廉的价格给远方的朋友打电话的可靠方式，但是全球互联网通信和信息公司 Raketu 准备将他们的 RakOut VoIP 服务的拨出电话费率降低高达 66 %!这是将持续到 6 月 30 日的促销活动的一部分。

Ratetu 可以在 Windows PC 上的任何互联网连接上运行，并使用点对点技术来提供高质量的 VoIP 通话——因此当您拨打这些电话时，不会听起来像是在隧道中。问题是，你必须为他们的免费推广不包括的拨出分钟数预付费用。但是考虑到你可以免费打电话到世界上的许多地方，而其他电话只需几分钱而不是几美元，似乎没什么损失。所以现在你可以打那些超级长途电话，而不会觉得比飞机票还贵。
