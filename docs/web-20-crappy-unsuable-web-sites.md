# Web 2.0 =蹩脚的、不可持续的网站

[![web20stinks.jpg](img/13766ae982e3750b691b1f9fa8584ecd.png)](https://web.archive.org/web/20210301202336/https://beta.techcrunch.com/wp-content/uploads/2007/05/web20stinks.jpg "web20stinks.jpg")

Web 2.0 在网页设计方面很烂。这是雅各布·尼尔森(Jakob Nielsen)最近在一次会议上说的话，他在会上痛斥了那些为了时髦而牺牲可用性的花哨、浮华的 Web 2.0 网站。(我希望 CrunchGear 不会被认为是以牺牲可用性为代价的华而不实。)太多的网站，通常是由对网页设计一无所知的人运营和/或资助的，把“把它做好”放在一边，这很快就会反过来，用这么多的话来说，咬他们的屁股。

本质上，尼尔森认为网站应该回归基本，而那些“明白”的网站将是受益者。Web 2.0 的另一个问题是:过于依赖用户输入。事实已经表明，很少有人真正参与到网络娱乐中来，所以那些只是想获得尽可能多的用户而没有给他们一个真实的、对他们有用的注册理由的网站也会被毁掉。

但是不，Web 2.0 才是它现在的位置。一个。
