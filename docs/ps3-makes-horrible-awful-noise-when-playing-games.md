# PS3 玩游戏的时候发出很恐怖很难听的噪音？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/12/ps3-makes-horrible-awful-noise-when-playing-games/>

[![ps3whine.jpg](img/536e4e5f847cc62da14beadbf4474134.png)](https://web.archive.org/web/20150929063957/http://old.crunchgear.com/wp-content/uploads/ps3whine.jpg "ps3whine.jpg")

用户抱怨许多 PS3 在玩游戏时发出令人讨厌的高音调噪音。他们已经将有问题的 PS3 缩小到序列号为 CE133212XXX 或 CE133190XXX 的 PS3。(还记得 PSone 在长时间的游戏过程中总是过热，或者 PS2 一直向你抛出的“脏盘错误”吗？我看到一种粗制滥造的模式。)

所以如果你不想整天和这个打交道，确保你闪亮的新 PS3 没有这两个系列。
