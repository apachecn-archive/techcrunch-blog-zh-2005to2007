Windows Live测试视频搜索

Live.com正在他们的测试版搜索服务中开发一些有趣的新功能，最著名的是视频搜索。它还没有出现在首页，但讽刺的是，在网站的一部分，它的标志上缺少了单词beta:beta.search.live.com。视频搜索现在炙手可热:两周前，谷歌将其视频搜索提升到了首页，美国在线的新视频网站在搜索和初创公司 Pixsy通过明确定位垂直搜索而引起了最近的轰动。Windows Live旨在为用户提供完整的服务起始页体验，如果没有这一点，它将是不完整的。

直播视频搜索指数似乎可以与谷歌和美国在线视频搜索相媲美——尽管每个人都被雅虎毁掉了！从搜索结果的数量来看，视频搜索。到目前为止，结果页面非常简单，一个单独的栏在干净的框中显示缩略图、标题、来源、持续时间和摘要。文件类型和更多信息可在每个结果角落的下拉列表中找到。

视频搜索在测试版页面的“更多”标签中，但在一次视频搜索后，它会移动到页面顶部自己的标签中。在高级搜索页面的典型选项中有一个地理位置字段，虽然网站说这将有助于提高相关性，但显然还没有使用，因为我的结果没有什么不同。

一旦Live在全球范围内传播开来(这是不可避免的),那么在众多服务中加入视频搜索将是一件好事。如果地理相关性被打开，如果视频有一个功能，Live正在试验图像搜索(见下文)，那么这可能会更有吸引力。

Live beta的另一个值得注意的新增功能是图像搜索便笺——它可以让你在多次搜索后将单个图像保存到侧边栏。可以构建和命名多个集合，尽管还不清楚可以用它们做什么。如果你感兴趣的话，请看我们之前对FlickrStorm 的评论，这是一个轻量级服务，为Flickr搜索提供了这个功能和更多功能。

如果现在每个人都有视频搜索，Live.com应该是未来，那么很高兴看到它推动视频搜索等事物向前发展，而不是仅仅成为另一个实用工具。然而，也许这就是创业的目的。由于Live将会出现在大量的人面前，看起来他们很快就能使用一个可行的视频搜索引擎，这很好。感谢奥莱利雷达，在那里第一次发现了这个评论。
