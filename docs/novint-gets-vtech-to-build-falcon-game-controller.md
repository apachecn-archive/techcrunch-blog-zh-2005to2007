# Novint 获得 VTech 来制造猎鹰游戏控制器

# 诺维特获得伟易达建立猎鹰游戏控制器

Novint Technologies Falcon game controller 是第一款让消费者计算应用实现高保真交互式三维触摸的控制器。(诚然，有些东西我不知道我需要，但现在觉得我必须要。)猎鹰让用户在玩启用的游戏时感受重量、形状、纹理、尺寸、动态、3D 运动和力的效果。Novint 已经与手机制造商 VTech 签订合同，处理生产任务，所以即使它要到 2007 年才开始发货，它至少会在*发货。*
