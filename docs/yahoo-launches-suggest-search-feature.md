# 雅虎推出建议搜索功能

[![](img/de7e34d2d8c30d8cffd10f5fd563666d.png)](https://web.archive.org/web/20211205105956/http://www.yahoo.com/) 雅虎[推出了](https://web.archive.org/web/20211205105956/http://www.ysearchblog.com/archives/000469.html)一项“建议搜索”功能，当用户键入时，它会自动提示用户建议的搜索查询。

雅虎表示，这项功能可以限制用户在查询中输入的字数。通过提出类似的查询，用户应该有更短的搜索时间，更快地找到他们要找的东西。

该功能并不是全新的；雅虎火狐工具栏的用户以前可以使用建议搜索，尽管它对 Yahoo 来说是新的。值得注意的是，在现阶段，该功能在特定国家的雅虎网站上不可用。

雅虎之前推出了一个类似的产品，即时搜索，它以类似于 2005 年 9 月建议搜索提示查询的方式提示搜索结果，尽管该产品从未出现在雅虎的首页。谷歌提供与谷歌建议搜索相同的服务。
