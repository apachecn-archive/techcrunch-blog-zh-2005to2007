# CrunchGear HiFi 周

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/23/crunchgear-hifi-week/>

![crunchgear_hifi.jpg](img/814a16dd0c504c594b44c81a4a7ddf15.png)
本周 CrunchGear 将举办音频庆祝活动。我们有很多很棒的评论和见解，所以请做好准备。

我们还推出了一个全新的读者反应系列，与 HiFi 周同时进行。待价而沽的是两台 [SanDisk Sansa Connect](https://web.archive.org/web/20160329181428/http://crunchgear.com/2007/04/12/sansa-connect-by-sandisk/) 机组。我们将在周五宣布两位获奖者，但整个星期的评论都会被考虑。跳跃后的全部细节。

要想加入，你所要做的就是尽可能多地发表令人信服的评论。我们将从整个星期中选出最好的两位评论者，他们每人将获得 SanDisk 的 Sansa Connect。

你必须在评论者信息栏留下你的电子邮件地址。没有有效电子邮件地址的评论将不予考虑。我强调这一点，因为总有我想奖励的伟大评论，但它缺少一个地址。

感谢 [SanDisk](https://web.archive.org/web/20160329181428/http://www.sandisk.com/) 为比赛提供连接。就这样，比赛开始了，开始评论吧！
