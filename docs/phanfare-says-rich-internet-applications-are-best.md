# Phanfare 说富互联网应用程序是最好的

[![](img/5f49d9c6ae7cc09fadd848fb615c913b.png)](https://web.archive.org/web/20201124094638/http://www.phanfare.com/) 照片和视频存储与分享服务 [Phanfare](https://web.archive.org/web/20201124094638/http://phanfare.com/) 从 2005 年初开始提供 Windows 和 Mac 的可下载桌面客户端。这是一个令人印象深刻的工具，但该公司发现人们希望在线访问。今天，Phanfare  在网上发布了一个简单版本的桌面客户端，名为 [My。Phanfare](https://web.archive.org/web/20201124094638/http://my.phanfare.com/) 。这是一个有趣的案例研究，支持所谓的富互联网应用程序 web 和桌面应用程序之间的交叉，一些重要的处理在客户端完成。

Phanfare 提供无限存储档案和网络质量的照片和视频，最长可达 10 分钟(大小为 1.5 GB)，每月，每年和终身计划的费用分别为 7 美元，55 美元和 300 美元。首席执行官安德鲁·埃里克森的上一家公司 Flashbase 于 2000 年被 DoubleClick 收购。他们面临许多其他公司的竞争，包括资金雄厚的 Sharpcast 和与 Maxtor/Seagate 合作的 Fabrik。

Phanfare 桌面客户端专注于提供流畅的工作流程，在这一点上它是成功的。从一次直观地显示你所有的照片，快速添加字幕，到在应用程序的后台进行大量的上传——桌面上的 Phanfare 可以做一些 web 服务通常不能做的事情，在某些情况下也不能。这是一个强大的、反应灵敏的系统，但仅仅这样还不够。

该公司一直提供简单、无广告的网页来展示公开发布的照片、视频和配有音乐的幻灯片。它们提供安全的在线存储，并可以快速重新填充媒体的本地缓存。该公司发现，虽然人们希望能够上传，编辑和删除照片，从道路上，从多个地点和从浏览器。该公司表示，客户现在已经非常习惯于网络访问，以至于他们经常在试图通过网络浏览器登录桌面客户端时感到困惑。

虽然基于网络的照片分享现在非常流行，Phanfare 指出，该公司的大多数客户向不到 15 人展示他们的媒体，几乎没有人停止使用他们的桌面照片整理工具。没有桌面客户端，客户无法离线处理照片，媒体无法从硬件自动导入，工作流程也无法快速运行。但是，如果没有网络备份，那么文件就像任何单独的硬件一样都是短暂的。

这是一个很好的组合，桌面的繁重工作和 web 的敏捷访问。不同的公司从不同的有利位置开始认识到有价值的中间产品，但是 Phanfare 的新网络访问强调了即使是一个非常好的桌面客户端也是不够的。
