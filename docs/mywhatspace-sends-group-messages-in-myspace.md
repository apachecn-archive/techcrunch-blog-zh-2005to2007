# MyWhatSpace 在 MySpace | TechCrunch 中发送群发消息

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/15/mywhatspace-sends-group-messages-in-myspace/>

[![](img/cfe358fbeb271c4cb5d6fe485c86be33.png)](https://web.archive.org/web/20161120065827/http://www.mywhatspace.com/) 我不知道这是否违反了 MySpace 的服务条款，我不会感到惊讶，但这很酷。MyWhatSpace 是一款电脑桌面应用，可以让你将 MySpace 上的朋友分成不同的群组，并同时向任何群组的所有成员发送消息。所以我可以给大学朋友发一条信息，给家人发一条，等等。

这对于 MySpace 来说是一个非常合乎逻辑的特性，但是它没有。与此同时，让新墨西哥州的四个人来组装一个工具并加入一些 AdSense。这是一个小团队卓越敏捷性的绝佳例子。它非常容易使用，非常方便，可能会成为小型企业的一部分。

想象一下，如果所有开发垃圾邮件工具和其他使在线社交网络空间变得更加肮脏的人都把时间花在制作这样的工具上。希望 MySpace 不会让这些人不好过。最近几个月，该公司关闭了与 MySpace 同时运行的另外两项服务 (DatingAnyone 和 SingleStat.us)。当一家公司关闭了与一家大型大众服务公司合作而非竞争的小型创业公司时，这对每个人来说都是一种损失。
