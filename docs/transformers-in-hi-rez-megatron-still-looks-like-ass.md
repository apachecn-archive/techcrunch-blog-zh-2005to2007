我们都为变形金刚做好了准备。是的，我们知道，这不是一个电影博客，但从技术上讲，变形金刚是小玩意，巨型机器人或动作人物。他们是我们许多人的入门玩具，所以我们知道你想知道他们的一切。

这让我想起了迈克尔·贝在他的新闻博客上发布的一些照片:八张我最喜欢的伪装机器人的高清图片，四个汽车人，四个霸天虎。我的工作就是向你指出它们。踢屁股。
