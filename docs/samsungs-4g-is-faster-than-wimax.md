# 三星的 4G 比 WiMAX 快

在美国，我们刚刚开始启动 3G，并与我们的大多数提供商合作，但在世界其他地方，4G 和 WiMAX 之间的大战已经开始升温。潜在的赢家？4G。

三星在野外乘坐了一辆演示巴士，以 60 公里/小时(37 英里/小时)的速度行驶，以展示 4G 的速度和在手机之间切换电话的能力。谢天谢地，他们得到的数据速度比陆地速度快得多。4G 达到了 1Gbps，比 WiMAX 目前的 802.16e 规格快 50 倍

与此相关的是，Sprint Nextel 与三星签订了合同，在未来几年内部署其全国性的 WiMAX，这意味着我们将在美国的 4G 网络之前获得 WiMAX。
