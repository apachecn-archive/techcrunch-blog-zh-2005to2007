zimbra:400万个付费邮箱，还在增加

原文：https://web.archive.org/web/http://www.techcrunch.com/2006/10/17/zimbra-4-million-paid-mailboxes-and-counting/trackback/

Digg、Mozilla、瑞士伯尔尼大学、印度时报有什么共同点？根据今天的公告，他们都是基于web的开源通信和协作套件 Zimbra 的1000多个客户。

该公司于2003年在加利福尼亚州圣马特奥成立，今天宣布它已经通过了400万个付费托管和现场邮箱。与微软交易所销售的1.5亿多个席位相比，这是一个很小但不断增长的非常重要的数字。Zimbra从Benchmark Partners、Redpoint Ventures和Accel Capital获得了1600万美元的资金。

我们已经描述了谷歌、Zoho、微软和无数小公司向基于网络的办公室发展的趋势。 Zimbra包括一长串其他公司刚刚开始提供的功能。微软表示，Exchange Server 2007将于今年年底或明年年初推出；如果它真的在那个时间框架内成为最终版本，看看它是否能做Zimbra能做的事情将会很有趣。

网络邮件客户端的外观和操作与GMail非常相似，但支持Outlook中的电子邮件、日历和联系人。
有RSS提要阅读、Salesforce集成、无需下载单独客户端的移动访问(尽管显然没有Windows mobile支持)、标记、文档和电子表格共享和协作，以及富文档的Ajax嵌入。Zimbra明确支持混搭和微软产品需要第三方工具来执行的语音集成。

这是一个令人印象深刻的产品，显然被越来越多的人采用。如果你正在寻找支持Web 2.0在企业中的可行性的证据，Zimbra今天的客户公告提供了优质的素材。
