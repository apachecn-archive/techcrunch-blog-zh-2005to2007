# 赛格威推出新的热点

![segway_i2.jpg](img/e2e22e1056ef64d6ed09706e371fd473.png)

赛格威正式推出了他们最新的个人运输车，i2 和 x2。这些设备具有一系列新功能和更新的功能。其中最引人注目的是新设计的转向机构 LeanSteer 和显示各种信息的高度直观的启动设备 InfoKey。

本月早些时候宣布，Segway 已经获得了 FCC 对新车型的认可——可拆卸的 InfoKey 可能是他们需要批准的原因。如果有人试图篡改你停放的 Segway，该装置将锁定其车轮，振动其底盘，发出警报并向你的 InfoKey 发送信息，提醒你盗窃。

![x2-1.jpg](img/a3aa5fbc85986a3ab6d7fe8f1c1fd986.png)

i2 型号主要面向通勤者和都市人，价格为 4995 美元起。另一方面，x2 更坚固，具有多节的车轮和高架底盘，用于清除可能位于您道路上的障碍。它的价格也很高，不含配件的起价为 5495 美元。每个单位都能够达到 12.5 英里/小时的速度，所以不要指望在这些事情上取得任何进展。
