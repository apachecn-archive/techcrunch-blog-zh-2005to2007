# Vonage 律师将裁决等同于“切断氧气”

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/06/vonage-lawyer-equates-ruling-to-cutting-off-oxygen/>

一名联邦法官今天裁定，Vonage 在上诉侵犯威瑞森专利期间不能为新客户提供服务。法官还命令该公司缴纳 6600 万美元的保证金。

Vonage 的律师罗杰·沃林(Roger Warin)向法庭表示，禁止 Vonage 寻求新客户对 Vonage 来说相当于“切断氧气或一颗子弹射向头部”。

早在三月初，陪审团发现 Vonage 侵犯了威瑞森的三项专利。它命令 Vonage 为未来的销售支付 5800 万美元和 5.5%的版税。
