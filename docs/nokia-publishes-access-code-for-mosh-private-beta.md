# 诺基亚发布 Mosh 私人测试版的访问代码

如果你打算进行私人测试，最好不要在你的论坛上发布网站的访问代码。但这正是诺基亚[和 T2](https://web.archive.org/web/20220813221009/http://forum.nokia.com/)[的新 Mosh 服务](https://web.archive.org/web/20220813221009/http://www.beta.techcrunch.com/2007/07/09/screen-shot-of-yahoo-mosh-plus-nokia-has-a-mosh-too/)所做的。

如果你想了解诺基亚 Mosh，只需在[登录页面](https://web.archive.org/web/20220813221009/http://mosh.nokia.com/)的框中输入“ALLACCESS”即可。然后，您可以注册该服务。

Mosh 实际上是一个伟大的想法，尽管还不足以对抗即将到来的 iPhone 浪潮。这是一个可以从普通电脑或诺基亚移动设备访问的门户网站，用户可以在这里上传和分享内容——应用程序、游戏、图像等。然后将这些下载到移动设备上。

巧合的是，雅虎也在开发一项新服务，工作名称是“Mosh”
