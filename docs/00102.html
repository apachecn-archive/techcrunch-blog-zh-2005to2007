服务: 雅虎我的网站2.0

发布:2005年6月29日

是什么？

MyWeb2.0是一个社交搜索引擎“它通过让用户除了搜索网络之外，还搜索他们的朋友和社区的知识和专业技能来补充网络搜索。”我们已经使用和滥用它一天了，我们认为它很好——有点像普通的yahoo plus furl ( profile )。

它于今天作为早期测试版“为有限数量的用户”发布。可能会有一个截止期，所以如果你想提前了解情况，最好尽快签约(这是个很好的营销想法)。

一旦你注册了(你可以使用现有的雅虎账户)，你可以做很多事情。如果你想为网页添加书签，我们建议你下载雅虎工具栏，它可以让你为你正在浏览的网页添加书签。否则，您只能为普通雅虎搜索中找到的页面添加书签。我们不太喜欢工具栏，因为几乎有一半的屏幕都被它们占据了，但是如果你想使用我的网站2.0，它必须成为你生活的一部分(嘿，也许你已经使用了雅虎工具栏)。

当您为页面添加书签时，会出现一个弹出窗口，允许您在网站上输入元数据，包括标题、注释、标签、访问控制和“保存页面”选项(同样，所有这些看起来和感觉都很像furl:

您也可以邀请朋友(请随意添加我们-archimedesventures@yahoo.com)(tech crunch摄于:-)，并查看他们的书签页面。整个想法是，与你的朋友相关的东西也很可能与你相关。

这是用户标记在起作用(关于这种危险的讨论，见我们在名人闪烁和 Feedster 上的个人资料)，但在这里你有真正的动机(比如美味和furl)去正确地做这件事——既为了以后找到东西，也为了和你的朋友分享。

是的，这是另一个添加朋友的服务，并且不需要给网站添加书签，但是它有输入选项(包括RSS订阅)(耶！)来减轻负担。我导入了我的个人美味页面RSS提要，它似乎工作得相当好。

有很多关于这方面的文章(见下面的链接)，所以我们的建议是试着阅读评论，亲自尝试一下。感谢雅虎在Web 2.0中启动了这个实验。
