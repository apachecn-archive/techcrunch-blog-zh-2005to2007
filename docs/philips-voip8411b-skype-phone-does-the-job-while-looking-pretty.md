# 飞利浦 VOIP8411B Skype 电话在漂亮的同时完成了工作

[![voip8411b.jpg](img/d536823be32da1a11fa1b338297555f4.png)](https://web.archive.org/web/20210415015020/https://beta.techcrunch.com/wp-content/uploads/2007/04/voip8411b.jpg "voip8411b.jpg")

凭借 VOIP8411B，飞利浦跃入 Skype 手机的喧嚣世界。幸运的是，这家荷兰企业集团的手机是所有产品中最好看的。它没有其他 Skype 手机没有的功能——拨打 Skype 至 Skype 和 SkypeOut 电话——但它也可以作为 POTS 手机使用。了不起！整个包装由三部分组成:手机本身，一个相当大的集线器和一个充电站。集线器是你插入电话和以太网插孔以及充电站的地方，充电站可以给手机充电。是的，飞利浦今天全力以赴。

VOIP8411B 真的不比其竞争对手的[产品](https://web.archive.org/web/20210415015020/http://crunchgear.com/2006/09/12/netgear-ships-wifi-skype-enabled-sph101/)更好或更差，但 CNET 认为它时尚、超酷的风格使它值 150 美元。当然，为什么不呢？
