# Wibree 加入蓝牙大家庭| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/12/wibree-joins-the-bluetooth-family/>

随着诺基亚 Wibree 的加入，蓝牙扩展了它的神奇领域。Wibree 技术现在将成为超低功耗蓝牙规范。这反过来将推动蓝牙的个人区域网络，供更小的设备无线使用。Wibree 于 2001 年成立，这是一个漫长的过程。展示 Wibree 集成到蓝牙规范中的首批产品将于 2008 年上半年首次亮相。
