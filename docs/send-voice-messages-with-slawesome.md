# 使用 Slawesome 发送语音消息

上周四在 TechCrunch [Meetup](https://web.archive.org/web/20210505232528/http://www.beta.techcrunch.com/2005/09/23/successful-meetup-last-night/) 上演示的 [Slawesome](https://web.archive.org/web/20210505232528/http://www.slawesome.com/) ，是一个通过电子邮件录制和发送语音信息的简洁工具。

与 [vemail](https://web.archive.org/web/20210505232528/http://www.beta.techcrunch.com/2005/08/16/profile-vemail/) 不同，slawesome 不需要下载。与 Springdoo 不同，Slawesome 是免费的。

注册后，您可以录制长达 2 分钟的消息，并将其发送到任意数量的电子邮件地址。电子邮件中包含该文件的链接。目前没有下载声音文件的功能。

邮件被存档，并可以设置为私人或公共状态。

这是一项年轻、原始的服务，但值得一试。

免责声明， [Fred Oliveira](https://web.archive.org/web/20210505232528/http://www.webreakstuff.com/) ，TechCrunch 的编辑之一(也是网站设计师)，在 Slawesome 上工作。
