# VideoEgg 发布

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/09/21/videoegg-launches/>

由三名耶鲁毕业生创建的 [VideoEgg](https://web.archive.org/web/20220120015710/http://www.videoegg.com/) ，于周一在加州亨廷顿海滩 [DEMO](https://web.archive.org/web/20220120015710/http://www.demo.com/) 发布。

鉴于他们网站上的清淡内容，我敢打赌，VideoEgg 的人在过去几周一直在努力工作，为产品的发布做准备。

VideoEgg 是一种基于网络的发布服务，允许用户从几乎任何设备和格式捕捉视频内容，并将其发布到网络上。点击[此处](https://web.archive.org/web/20220120015710/http://www.videoegg.com/Player.html?timestamp=1127333563187)查看关于[巴斯·布鲁格曼](https://web.archive.org/web/20220120015710/http://buzzmodo.typepad.com/buzznovation/)的示例。

乍一看，VideoEgg 有一些非常优秀的功能。他们的网站上有一个[现场演示](https://web.archive.org/web/20220120015710/http://www.videoegg.com/Publisher.html)，允许你将视频拖到浏览器中，体验一下它的易用性。 [Flash 8](https://web.archive.org/web/20220120015710/http://www.macromedia.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash) 允许观看质量更高的视频。

从他们的[发布新闻稿](https://web.archive.org/web/20220120015710/http://www.videoegg.com/PressRelease091905.html):

> 今天，VideoEgg 公司推出了 VideoEgg Publisher，这是一种新的基于网络的视频发布技术，使日常互联网用户首次能够轻松捕捉、编码、上传和观看在线视频。VideoEgg Publisher 是一个“通用适配器”,可以直接从数百个设备中捕获并读取数十种格式，允许用户轻松发布任何人都可以观看的视频，而不必担心播放器兼容性、编码设置或额外的软件。VideoEgg 在 DEMOfall 2005 大会上宣布了 Publisher，这是一个只接受邀请的发布创新技术产品的领先行业展示。
> 
> 目前，在网上发布视频对用户来说很复杂，需要对格式、编码器、播放器和服务器有高度的技术理解。VideoEgg Publisher 简化了困难的视频编码和发布过程，允许用户从摄像机、网络摄像头和移动电话直接将视频捕获到网站上。它还通过一个简单的拖放界面接受视频文件。在电影被编码和发布之前，用户可以使用发行商的简单编辑工具进行基本编辑。
> 
> 使用该系统，可以通过基于 Flash 的 VideoEgg 播放器立即在线观看视频，这是一种“无播放器”解决方案，不需要像 Windows Media 和 QuickTime 这样的外部播放器。

上面有很多关于这项服务的有趣信息…最后一段真的很有趣。出版商只需添加一个 html 代码片段，就可以轻松地将内容直接集成到他们的网站上。

这是我非常喜欢 VideoEgg 的一点——看起来他们真的很专注于让他们的产品尽可能的简单易用。

我已经下载了他们的软件(现在似乎已经不在网站上了)，正在为我的摄像机挖一个盒子。我希望得到一些好的内容，并在 [Truveo](https://web.archive.org/web/20220120015710/http://www.beta.techcrunch.com/2005/09/21/truveo-video-search/) 上被索引。
