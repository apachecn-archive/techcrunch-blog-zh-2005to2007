# 专利猴:数字显示魔术

![](img/c7671da9c02950791d4afc0d8cc5efe0.png)

在本周关于数码相机和显示器的知识产权回顾中，我们发现了数码相机的光滑[柯尼卡可滚动液晶显示屏](https://web.archive.org/web/20210227222618/http://www.patentmonkey.com/PM/PatentID/7184086.aspx)和柯达申请专利的[“身临其境的剧院”](https://web.archive.org/web/20210227222618/http://www.patentmonkey.com/PM/PatentID/7180529.aspx)概念。

从电影[少数派报告](https://web.archive.org/web/20210227222618/http://imdb.com/title/tt0181689/)中可以看出，柯尼卡显示技术被描述为使用“电子纸，[其中]红色、绿色和蓝色的液晶层被叠加，两层之间的调色剂被吸引和分离，有机 EL 在薄膜上发光。”高度紧凑，我们真的很想看到柯尼卡推动这一发展成为一个原型。有什么在进行中吗？ ![](img/68bd79856d5fb661b3ced7216c891096.png)

关于数码相机可滚动显示器的潜在实现，聚合体视觉宣布与意大利电信建立[合作关系，将他们的技术引入移动设备。他们的网站还讨论了他们的可滚动显示技术，称其“弯曲半径仅为 0.75 厘米——这就是为什么他们实际上可以‘消失’在移动设备中”。Polymer 是飞利浦的副产品，飞利浦从 2002 年开始在该领域开展工作。屏幕更大，尺寸更小，很好。](https://web.archive.org/web/20210227222618/http://www.polymervision.com/News-Center/Press-Releases/TelecomItaliaandPolymerVisionannouncetheCE.html)
