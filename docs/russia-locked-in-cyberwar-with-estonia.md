# 俄罗斯与爱沙尼亚陷入网络战

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/17/russia-locked-in-cyberwar-with-estonia/>

![](img/714583a9ff7e8da3a1caf6d858a348d5.png)

在第一次类似的战争中，俄罗斯已经攻击小国爱沙尼亚几个星期了。爱沙尼亚人将一座苏联士兵的铜像从纪念碑上移走后，事件开始了。俄罗斯被激怒了，发动了黑客攻击、DDoS 攻击和许多其他针对该国的伎俩，攻击政治成员、报纸、银行、公司以及政府网站。局势如此危急，以至于北约被召来帮助评估局势并带来一些平静。攻击似乎永无止境，因此专门的网络恐怖主义专家被部署来纠正问题。在我听来像是某个家伙在看[黑客](https://web.archive.org/web/20160914095343/http://en.wikipedia.org/wiki/Hackers_(movie))并想黑掉 Gibson，但是却陷入了一场网络战争。
