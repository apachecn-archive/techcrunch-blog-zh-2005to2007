# NAB 2007:东芝的 56 英寸几乎 4K 显示器

![_mg_0328a.jpg](img/9890ab5c73dd76825513f5bc55b86be2.png)

红色的非常棒，因为它能够在 4k 模式下进行原生拍摄。但是如果你不能回放，那有什么用呢？东芝展示了一款 56 英寸的显示器，它称之为 4k，尽管它在技术上有所欠缺。3840×2160，P56QHD 不是真正 4k 要求的 4200+像素，但它显示了一个漂亮的画面。这比你留着看的 1080p 电视的 3 倍还多，废物。1200:1 的对比度不是最高的，但比一些好。最棒的是显示器为您提供了大量输入，四个 DVI 插孔。很好。
