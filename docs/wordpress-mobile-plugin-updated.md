# WordPress 移动插件已更新

![emu.gif](img/6c7f8df4e7d5c26b60e918f8b1b81d05.png)
这个世界上有两种人:用 WordPress 的人和不用的人。如果你在第一组，你很幸运。Andy Moore 发布了一个更新的 WordPress 移动插件，允许读者在移动电话或浏览器功能减少的设备上访问你的网站。

这个插件构造得相当好，它能立即感知到一个进来的手机浏览器并做出相应的反应。在安装到 CrunchGear 上之后，我能够在一个模拟的 Symbian 设备上[浏览网站](https://web.archive.org/web/20210118050740/http://emulator.mtld.mobi/emulator.php?emulator=nokiaN70&webaddress=crunchgear.com&Submit=Update)。最重要的是，你可以自动将 AdMob 广告添加到网站中。安迪甚至为自己拿了一份提成，这样就省去了我们为如此伟大的编程工作给他寄支票的麻烦。

为移动浏览重构网页是困难的。这个插件让它变得异常简单。
