原文：https://web.archive.org/web/http://www.techcrunch.com/2005/08/16/profile-kahuna-hotmail-beta/trackback/

卡哈纳(带有Ajax的新hotmail)还没有推出，但是Start.com团队(简介)从2005年5月开始就一直在认真研究它，现在它正在私人beta测试 ( 更新)。

从卡哈纳团队(见下文)和各种 beta 测试人员 ( 和 其他观看空间)的帖子来看，它看起来像是对旧MSN门户网站的 Start 一样对Hotmail的重大增强。看起来他们将会在mail.start.com这个网址下推出这项服务。

关键特性包括自由使用ajax来消除屏幕刷新。允许不离开收件箱阅读电子邮件的方法，以及更快更干净的用户界面。

卡哈纳最近的历史细节可以在这里找到。

来自伊姆兰·库雷希，卡哈纳团队成员:

我喜欢邮件测试版的五大理由——理由1和2

邮件测试版是一个全新的网络邮件体验，致力于比现有的网络邮件服务更快、更简单、更安全。该团队专注于阅读和发送邮件的基础知识。邮件测试版正在开发中，大量的测试版用户正在推动它的发展(我们几乎无法跟上他们所有的想法)

以下是我在mail beta中阅读邮件时最喜欢的一些事情:

1.越来越快，越来越快更快邮件测试版明显更快，我的意思是快了一个数量级:
a)用户界面即时响应许多动作，并快速响应其他动作
b)很少的上下文切换(整个页面发生变化，你的眼睛必须重新扫描)
c)你需要更少的点击来完成日常任务
d)更整洁的外观(包括更多的“空白”?)这样你的眼睛可以放松，更快地找到东西

2.使用阅读窗格阅读邮件而不离开收件箱阅读窗格允许您在不离开收件箱的情况下阅读邮件。如果您喜欢Outlook中的阅读窗格，现在您可以在web mail中使用它。其他web邮件服务会强制您打开每封邮件，然后在阅读下一封邮件之前关闭它。如果你有10条信息要读，它们需要20次点击，卡哈纳:10次。(这对你的腕管来说怎么样？)

想看宽邮件？只需双击消息，瞧！

我们期待亲自测试这一点。
