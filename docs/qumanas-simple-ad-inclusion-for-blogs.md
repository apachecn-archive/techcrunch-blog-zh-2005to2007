# Qumana 简单的博客广告收录

Qumana 是一款优秀的所见即所得的在线/离线博客编辑工具。[我们已经在](https://web.archive.org/web/20220811105843/http://www.beta.techcrunch.com/?p=18)之前写了关于 [Qumana 的文章并描述了他们的服务，完全免费。](https://web.archive.org/web/20220811105843/http://www.beta.techcrunch.com/2005/08/11/update-qumana-new-feature/)

昨天 Qumana [发布了他们产品的 2.0 版](https://web.archive.org/web/20220811105843/http://blog.qumana.com/blog/_archives/2005/10/12/1294618.html)。新增的主要功能是能够非常容易地将点击付费广告直接添加到博客帖子中。

广告可以放在任何地方，有几个格式选项可供选择(见下面屏幕截图中的箭头)。收入在 Qumana 和出版商之间五五分成。

广告是基于你为文章设置的标签的关键字驱动的，并且毫无问题地流入 RSS。每个广告都通过关键词和点击量来追踪。广告也可以在最终发布前预览。发布者可以根据自己的意愿灵活地添加或多或少的广告。

还有其他新功能，包括改进的旧帖子编辑，更好的拖放工具，以及与 wordpress 和 blogger 的更好集成。

这是我见过的小型(和大型)博主在帖子中轻松加入广告的最佳工具。
