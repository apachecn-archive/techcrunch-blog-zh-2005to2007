# 牛顿外设 MoGo 耳机

![](img/0e0d6e09711be15406afe4adc365388a.png)
[MoGo 耳机](https://web.archive.org/web/20130628182124/http://crunchgear.com/2007/01/16/teeny-tiny-bluetooth-adapter/)上个月引起了一点轰动，现在看起来它已经到了重要的出货阶段。这款耳机只有 5 毫米(440 公里或 0 . 333 码)厚，而且有计划——注意，有计划——将它们嵌入手机中，安静地充电。

最棒的是？媒体 CD 上有一张莫戈女士的时髦照片，这是一个迷人的年轻女孩，耳朵上戴着蓝牙耳机。
