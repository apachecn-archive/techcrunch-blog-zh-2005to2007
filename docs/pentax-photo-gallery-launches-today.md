# 宾得照片库今天发布

![pentaxphotogalleryhome.jpg](img/218866eb6d9b4e2c17bfda91c89779f5.png)

Pentaxians 现在有了 Flickr、Photobucket 或 Picasa 之外的选择来展示他们的照片。宾得照片库是一个分享你的作品和从一群极具天赋的摄影师那里获得资源的专属空间。您将立即获得来自欧洲、亚洲、美洲和非洲 40 多个国家的 300 多位艺术家的 2500 多张独特照片。该图库提供英语、西班牙语、德语、法语、意大利语和日语版本。

PPG 与其他网站的不同之处在于，你可以通过机身和镜头搜索照片。你可以检查任何给定镜头的精确设置。这对崭露头角的摄影师来说是非常有用的工具。不久前我亲眼看到了这个网站，我可以说它的用户界面和照片选择给我留下了深刻的印象。首页展示了首映式系列的十几张照片，并不断轮换展示最新最棒的照片。

不是每个 Joe Shmoe 都能上传昨晚啤酒聚会的照片，考虑到外面有那么多的 Pentaxians，这已经很不错了。图像是根据艺术和技术价值选择的。该网站还提供可选的艺术家投票，以帮助管理和指导成为画廊一部分的图像的选择。画廊中接受的每幅图像都由宾得评选委员会最终批准。如果你觉得自己的工作符合标准，或者有其他问题，请随时给 photogallery@pentax.com 发电子邮件。
