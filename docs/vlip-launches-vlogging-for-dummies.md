# VLIP 推出虚拟博客

[![vliplogo.png](img/9278c6833a2ebeb00696bdecb3471ce1.png)](https://web.archive.org/web/20220516080809/http://www.vlip.com/) 视频会议公司 [Sightspeed](https://web.archive.org/web/20220516080809/http://www.beta.techcrunch.com/2006/07/26/sightspeed-50-launches/) 随着新的视频博客网站 [Vlip](https://web.archive.org/web/20220516080809/http://www.vlip.com/) 的推出，正在跳上 web 2.0 的列车。Vlip 允许用户在 Vlip 网站上发布网络摄像头视频评论，以及一系列视频回复。它采用了 YouTube 上的帖子和回复视频，并对其进行了一些改进，将其放在了前面和中心。像 YouTube 视频一样，Vlips 可以嵌入到网站中，也可以让用户回复和查看帖子。一个很好的举措是，Vlip 使得回复 Vlip 变得非常容易，即使你不是会员，甚至可以实时连接到也是 SightSpeed 会员的 Vlippers。在主网站上，Vlips 可以通过评级、日期和视图进行搜索。查看下面的小工具。

我们以前见过一些其他非 YouTube 视频评论线程。NYU 有一个 WordPress 的插件。Flikzor 也有一个可嵌入任何页面的视频评论部件。Vlip 希望成为所有这些对话的目的地，这让我想起了很多 [Chinswing](https://web.archive.org/web/20220516080809/http://www.beta.techcrunch.com/2006/12/08/chinswing-brings-audio-to-discussion-boards/) 。Chinswing 也是一个目的地网站，但是是针对音频的。像 Chingswing 一样，Vlip 不幸地将只使用键盘的人群排除在评论之外，这在我心中是一个很大的“不不”。Vlip 要想成功对抗 YouTube 和 Flikzor，他们必须搞定界面。首页缺乏在 YouTube 上运行良好的快速热门视频列表或导演频道。如果他们做对了，Vlip 的被动和主动视频聊天的结合会真正流行起来。
