# 防松鼠滚筒给料机

如果你是一个观鸟的人，当松鼠弄脏了你珍贵的喂鸟器时，你会变得焦躁不安，那么考虑一下 Solutions 公司的滚轴喂鸟器。它实际上就是它听起来的样子。进料器由两根杆悬挂着。当重量放在喂食器的一端时，它会滚动，将攻击你的松鼠扔进深渊(也就是草地)。

来自路易斯安那州，我逐渐意识到:没有什么是防松鼠的。当然，这可能会让这些小混蛋们受挫几天，但最终他们会明白的。记住我的话，79.95 美元的滚筒饲料可能会保护你宝贵的鸟食一两周，但之后，它又回到了绘图板。他们是不可阻挡的，真的。
