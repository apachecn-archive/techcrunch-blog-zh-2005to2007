# 三星蓝光+ HD DVD 播放器，支持两种格式的特殊功能

[![samsungbrhdvd1.jpg](img/1f5caf5a5fbfbb348aac2c3b83adb99c.png)](https://web.archive.org/web/20220121173021/https://beta.techcrunch.com/wp-content/uploads/2007/07/samsungbrhdvd1.jpg "samsungbrhdvd1.jpg")

随着三星 [BDP-UP5000](https://web.archive.org/web/20220121173021/http://crunchgear.com/2007/04/13/samsungs-blu-ray-and-hd-dvd-compatible-duo-hd-player-here-by-the-holidays/) 组合蓝光/HD DVD 播放器的发布，越来越多的细节正在互联网上出现，或者用我们最近用来描述互联网的任何有趣的词。来自德国的报道(还有哪里？)暗示组合播放器将支持蓝光和 HD DVD 阵营的所有功能。我知道[东芝上周刚刚宣布了](https://web.archive.org/web/20220121173021/http://crunchgear.com/2007/07/01/hd-dvd-makes-big-moves/)其 HD DVD 播放器的固件更新，这些播放器完全依赖于活跃的网络连接，所以看看“所有”是否真的意味着“所有”会很有趣该播放器还可以以每秒 24 帧的速度输出电影(对于“[雅各布的梯子](https://web.archive.org/web/20220121173021/http://www.youtube.com/watch?v=lCZN60mBeaQ)”[01:46 in]或“The Ring”快动作效果，甚至可以达到每秒 60 帧)。

BDP-UP5000 仍有望在 2007 年末发布。格式战，见鬼去吧！[又来了！](https://web.archive.org/web/20220121173021/http://crunchgear.com/2007/01/17/lg-super-multi-blue-hd-player-already-on-shelves/)
