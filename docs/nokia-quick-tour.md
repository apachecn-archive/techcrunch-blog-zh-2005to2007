# 诺基亚快速游览

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/12/nokia-quick-tour/>

![](img/fc4fda89eb4be8579a4b30b54fbfde85.png)

去了趟诺基亚展台。一个有趣的小花絮是来自诺基亚的新免提车载套件，CK-15W。它包括一个安装在仪表板上的迷你屏幕和一个用于浏览联系人和通话的滚轮。屏幕非常紧凑，非常明亮。

获得了一些 E61i 和 N77 在跳跃后的照片。
