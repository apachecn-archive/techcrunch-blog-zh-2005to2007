# 索尼说:你会得到复合材料并喜欢它

如果你打算买一台 PS3，跑回家把它连接到你的高清电视上，享受一些高清蓝光和/或游戏的美好时光，请确保在离开商店之前买一根 HDMI 或分量线。没错，索尼在 PS3 的包装盒中只包括一根复合电缆。根据 IGN 的一份报告，“60g 或 20g 型号的购买者将获得一根电源线、USB 电缆、以太网电缆和标准复合 AV 电缆。”至少，在日本版本中是这样的。然而，如果你有来自 PS2 的电缆，PS3 的多输出接口将接受它们。
