# Zing 播放器有 WiFi，比 Sirius Stiletto 还小

如果你期待着 Sirius Stiletto 来听 Sirius 电台，你现在有了一个选择。Zing 播放器还使用 WiFi 来抓取互联网和 Sirius 频道的访问权限。它也很小，尤其是和天狼星相比。

还有一个 dock 和一个 click-wheel，它不是基于触摸的，但仍然很酷。目前还没有定价和供货信息。
