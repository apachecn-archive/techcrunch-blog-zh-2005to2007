# 狼王战士 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/18/wolf-king-warrior/>

![wolfkingwarrior.jpg](img/09953f7e977ce5dd82bf7d989490cfa4.png)

狼王的新[战士控制器](https://web.archive.org/web/20220121194222/http://www.wolfkingusa.com/prdts_wolfking.html)基本上是对他们先前存在的[木材狼](https://web.archive.org/web/20220121194222/http://www.wolfkingusa.com/prdts_timberwolf.html)设计的重新评估。通过移除键盘组件，狼王创造了一个高度便携的 55 功能设备，适用于局域网聚会环境。它可以作为一个独立的单元工作，也可以与键盘配合使用。

虽然它被吹捧为与几种游戏类型同质，但它看起来最适合 FPS 风格的游戏。狼王战士 if [现已发售，售价 34.99 美元](https://web.archive.org/web/20220121194222/http://shop.wolfkingusa.com.cn/displayProductDocument.hg?productId=1&categoryId=2)。
