# 滚动闹钟为了得到它想要的东西而大发脾气

仿佛这个假期里，所有在每家有某种零售产品的商店里尖叫的孩子还不够，还在摇着闹钟(那不是 Beefheart 上尉专辑的名字吗？)给你一次机会…一次机会…起床。失败了，它真的会穿过房间跑到某个切尼式的秘密地点，然后尖叫，直到你起来找到它。我们的建议是:大多数时候使用你的手机闹钟，把这个留到重要工作面试的早上。这东西会引发严重的动脉瘤。
