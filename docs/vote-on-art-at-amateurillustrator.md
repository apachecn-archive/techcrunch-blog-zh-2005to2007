# 业余插画师钉钉小众社交网络 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/08/20/vote-on-art-at-amateurillustrator/>

# 业余插画指甲小众社交网络

如果你想看一个很棒的小众社交网站，去看看 AmateurIllustrator.com 的。这个位于英国林肯郡的网站今天发布，我认为它很有可能会蓬勃发展。任何有针对性的、热门的社交网站都会很好地检验其外观和功能集。有相当多的社交网站以小众兴趣为目标，但却丑陋、空洞、无趣。这个不是这样的。

这是一个业余插画爱好者分享他们的作品、互相投票和交流的地方。这个网站设计得非常好——考虑到开发者作为艺术家的背景，这也许并不奇怪，它看起来很漂亮。尼克·富兰克林和一位不知名的合作者在林肯大学学习插图时创建了这个网站。

 [](https://web.archive.org/web/20221204194441/http://www.amateurillustrator.com/articles/?p=23) 我最喜欢业余插画师的一点就是突出质量，原创文字内容。在发布会上，该网站对四位技术娴熟的插画师进行了采访——这篇文章中的插画应该归功于他们，点击图片进入艺术家的采访。管理员表示，他们将在未来推出更多对顶级插画师的采访、交易技巧和小奖品竞赛。引人注目的、开发良好的内容很难获得，但在社交网络空间中提供这些内容是一个很好的组合。它只是提高了整个网站的质量；我希望这个网站将能够拉动他们在这个部门的目标。

该网站的真正关键当然是上传图片集的插画师的社交网络。任何人都可以评价个人艺术作品并发表评论。注册用户可以互相发送消息，上传艺术和参与论坛。会员是免费的，如果这个网站能够吸引更多的观众，那么我敢肯定，已经存在的广告将会得到巩固。考虑到网站的主题和关注的受众，广告可能会比许多大众感兴趣的广告驱动型网站有更高的价值和转化率。我敢肯定，创始人希望让网站更上一层楼，让它成为一家企业，这将带来全新的挑战——但目前看来还不错。

有一些小的接触也很好。当你注册一个账户时，系统会询问你是否需要你的电子邮件、联系方式或者在你的个人资料页面上没有联系方式。

这个网站刚刚上线，显然是非常有才华的人的作品……嗯，业余爱好者——所以举例来说，一旦你登录了一个帐户，就很难访问服务条款，很高兴看到服务条款明确规定艺术家保留他们艺术作品的权利。你可以看出这是他们的意图，我想管理员们很快就会完善这些问题。

我能确定的竞争对手包括 [M.A.D.E.Planet](https://web.archive.org/web/20221204194441/http://madeplanet.com/) ，它对我来说似乎很刺耳，但却在后泡沫上获得了[好评，还有](https://web.archive.org/web/20221204194441/http://www.postbubble.com/2006/07/07/madeplanet-social-networking-for-artists/) [eSnips](https://web.archive.org/web/20221204194441/http://esnips.com/) ( [我们的报道](https://web.archive.org/web/20221204194441/https://beta.techcrunch.com/tag/esnips))。 [DeviantART](https://web.archive.org/web/20221204194441/http://www.deviantart.com/) 显然是这个空间的遗产地；它有免费和付费账户以及大多数标准功能，但缺少一些让业余插画师与众不同的东西。业余插画师没有为艺术家提供直接销售其作品的途径，但人们的联系信息是可用的，整体用户体验可能是最重要的。如果后来通过该网站进行销售，我不会感到惊讶，但如果不是这样，我也不会感到惊讶。

总的来说，这可能是我一段时间以来看到的执行良好的利基社交网站的最佳例子。看一看吧，我想你也会印象深刻的。
