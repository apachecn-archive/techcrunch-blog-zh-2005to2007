# 视频搜索

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/09/21/truveo-video-search/>

 [Truveo](https://web.archive.org/web/20230306024818/http://www.truveo.com/) 是几天前测试版推出的视频搜索引擎。投资人杰夫·克拉维尔[在这里](https://web.archive.org/web/20230306024818/http://blog.softtechvc.com/2005/09/truveo_the_litt.html)写了关于 Truveo 的文章。

如果您正在寻找视频内容，这是一个好去处。

Truveo 两年前由一个由搜索专家组成的非常聪明的团队成立，正在解决从视频中创建元数据的非常困难的问题——与基于文本的内容不同，在没有人实际查看的情况下，很难确定视频和音频内容的上下文(这很容易出错，并且无法扩展)。

Truveo 采用创新的两步方法来索引新内容。第一步是一个直接的网页抓取来找到视频。第二步，也是更重要的一步，是创建关于内容的元数据，而不仅仅是文字记录和提要(大多数视频内容都没有)。他们已经创建了一个“[视觉爬虫](https://web.archive.org/web/20230306024818/http://www.truveo.com/about.php)”，它查看周围的内容以确定上下文。

> 在网上找到所有的视频文件只是挑战的一部分。为了使视频可被搜索，还需要收集有意义的文本元数据来与每个视频文件相关联。当然，我们依赖于标准的技术，比如挖掘隐藏字幕和导入 RSS 提要。然而，网络上的绝大多数视频没有任何隐藏字幕或 RSS 元数据。幸运的是，我们的视觉爬虫拯救了我们。每当我们的视觉爬虫在网络上发现一个新的视频时，它们也可以“视觉地”检查周围 web 应用程序的上下文。在大多数情况下，这种检查揭示了与每个视频相关的丰富而详细的元数据。
> 
> 通过我们独特的爬行方法，我们不仅可以找到其他爬行器遗漏的视频，还可以为每个视频收集丰富的相关元数据。因此，当您使用 Truveo 搜索视频时，您总能找到高质量的相关搜索结果。下次你想看网络视频时，我们邀请你试试我们的搜索引擎，亲自看看。

最棒的是，这东西真的有用。尝试搜索飓风丽塔和每日秀。

有[高级搜索](https://web.archive.org/web/20230306024818/http://www.truveo.com/advancedSearch.php)选项和[偏好](https://web.archive.org/web/20230306024818/http://www.truveo.com/prefs.php?refpage=http%3A%2F%2Fwww.truveo.com%2Findex.php%3FfromPrefs%3D1)将帮助用户找到相关内容。可悲的是，默认情况下“成人过滤器”是打开的。:-)也可以过滤掉付费内容。

我唯一想看到添加的是搜索的 RSS 提要。
