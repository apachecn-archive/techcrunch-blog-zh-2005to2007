Web 2.0本周(10月9日至15日)

Web 2.0 This WeekOctober 9 – 15Location:

硅谷

本周大量新公司成立。我猜他们的目标是上周的Web 2.0会议，但不得不推迟。这可能是最好的结果，因为在上周的媒体混乱之后，他们得到的报道可能要好得多。

大多数博客作者可能已经注意到戴夫·温纳回到了湾区。他的大脑又回到这里真好。我被邀请参加他和史蒂夫·吉尔默在T2的几次每周早餐会。我试着闭上我的嘴，尽可能地保持我的听/说比率。这些人并不总是对的，但是他们已经看了很多，做了很多，即使他们错了，你也能从他们身边学到很多。

我们向理查德·麦克马努斯道别，他在硅谷旅行期间和我们呆在一起。根据他受到的热情接待，我预计理查德会很快回来。他雄辩的告别帖非常精彩。

上周，我们启动了一个叫做web 2.0工作组的项目。这是一个实验性的沙盒，一群专注于写新网络的博客作者正在研究这个沙盒。现在它是一个参与博客的目录。很快就会多得多。

这是本周的总结:

1.TechCrunch本周简介

卡哈纳(更新)、加达.贝、谷歌阅读器(更新)、雅虎博客搜索、谷歌书签、眨眼、 MeasureMap (更新)、 Memeorandum (更新)、 Yelp 、 Qumana (更新)、 RememberTheMilk

2.聪明的公关和愚蠢的公关

聪明:给博客圈一个独家预览权和发布权，让的荣誉滚滚而来。

哑巴:推出一款产品攻破火狐，无视博主意见，给《纽约时报》一个独家。博主回应。

3.1亿个博客

博客先驱报报道称，博客数量超过1亿。按国家分列。太棒了。英国广播公司《T2》报道称，今年网络比以往任何时候都增长。

4.戴夫·怀纳在书呆子电视上

戴夫·怀纳在书呆子电视节目中接受罗伯特·克林格利的采访。这是一个大下载，但值得。我很高兴像这样的东西被记录下来用于历史目的。

5.罗伯特·斯考伯写实时搜索

这是一篇咆哮和一篇经典文章的混合体，很重要。他继续他在现有博客搜索引擎上比较搜索结果的主题。

让我们谈谈基于时间的搜索的现状。

用一句话来说:太糟糕了。

没有人做得很好。

我能听到每个人都说“嗯”?我认为Feedster、Technorati、IceRocket、Bloglines和Pubsub等公司都在进行基于时间的搜索？â€

是的，但是它们都不令人满意。为什么？好吧，首先，他们永远不会有MSN Search、Yahoo或Google的流量。我周围的大多数“正常”人除了这三个之外，从来不会使用搜索引擎。见鬼，世界上大多数人甚至从未点击过“高级搜索”,你会试图让他们访问类似于http://blogsearch.google.com的东西？没错。

在谷歌之前，每个人都说搜索“足够好”。博客搜索甚至还不够“好”。

6.大文件共享应用程序列表

Massimo crecerla花时间列出大文件共享应用程序。把它做成书签。

7.Oodle失去Craigslist Feed

当他们采用分散的内容方法时，Oodle做对了。Craigslist感到受到威胁，于是关闭了它们。Oodle 外交回应。

这对Oodle来说不太好，但我也认为这对Craigslist来说不太好。数据必须是开放的。试图聚集它的网站最终会失败。

8.标记文章

Fred Oliveira 写了关于标记的文章，重点是为什么人们会标记。我希望10月28日他仍会在湾区参加标签营，与该团体分享他的想法。

9.克里斯·皮里略在博客上开炮

Chris Pirillo的一篇帖子引发了一场关于垃圾博客的大讨论，事实上博客似乎是垃圾博客的主要来源。谷歌拥有BlogSpot，并有明显的动机允许这种情况发生——他们在这些博客上提供广告。这不是一个新问题。谷歌需要采取行动。

10.和解吧

戴夫迈出了第一步。昨天他又伸手。这两个人需要一起努力，让博客世界和网络成为一个更好的地方。我希望他们能找到方法。
