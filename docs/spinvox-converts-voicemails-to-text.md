# Spinvox 将语音邮件转换成文本

[](https://web.archive.org/web/20211202045805/http://www.spinvox.com/)[mobile runch](https://web.archive.org/web/20211202045805/http://mobilecrunch.com/2006/05/25/spinvox-makes-mobile-messaging-hyper-efficient)的 Oliver Starr 写了一篇关于 [Spinvox](https://web.archive.org/web/20211202045805/http://www.spinvox.com/) 的文章，这是一家总部位于英国的公司，将手机语音邮件转换成短信或电子邮件(或两者兼有)。这是我会用的东西。

这在英国已经存在了一段时间，但在美国刚刚推出(它需要与你的手机运营商达成协议才能工作)。目前还没有宣布任何交易，但显然在英国人们正在疯狂地使用它——而且它并不便宜。详见[mobile runch](https://web.archive.org/web/20211202045805/http://mobilecrunch.com/2006/05/25/spinvox-makes-mobile-messaging-hyper-efficient)。
