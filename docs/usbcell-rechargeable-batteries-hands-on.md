# USBCell 充电电池动手操作

USBCell 非常友好地送来一些充电 AA 电池给 CrunchGear 测试，它们不可能更快到达。我对 Xbox 360(和它的无线控制器)的不健康的沉迷已经耗尽了我几个星期的钱包。

**更新:**上周，USBCell 入围了 2006 年设计周最佳消费产品奖的决赛。Apple Shuffle 是该类别的获胜者，不算太寒酸。
 一包两节镍氢 AA 电池要 19.50 美元。电池的正极向后翻转，露出一个 USB 连接器。将电池插入电脑上的任何 USB 端口或电源插座，瞧，五个小时后你就有了新充电的电池。

充电时，电池边缘会持续发出绿光。当电量达到 90%时，它开始闪烁，并最终在充满电时熄灭。很可爱。

该公司计划在不久的将来推出 AAA、C、D 和 9 伏电池。它还在研发手机电池。我们等不及了。
