# 对 400 美元游戏的追求

![](img/8bb475cced3029d41fb6bacd5b0556b6.png)

随着*吉他英雄 2* 的成功，一些出版商和开发商说“嘿…如果我们用**整个乐队**做同样的概念会怎么样？这只是时间问题?”？!"摇滚乐队是下一步，将于今年 11 月推出。该游戏将包括一个完整的乐队设置，使用独特的吉他、贝斯和鼓控制器，以及麦克风来发出这些声音。但是随着早期定价在 [Gamestop 网站](https://web.archive.org/web/20130628183832/http://www.gamestop.com/search.asp?cookie%5Ftest=1&Ntk=TitleKeyword&Ntx=mode%2Bmatchallpartial&Ntt=rock+band&N=0&find.x=0&find.y=0&find=Search)上设定，这款游戏可能是有史以来最昂贵的游戏之一。

如果你打算全力以赴，你需要鼓(80 美元)，两把吉他(120 美元)，一把贝斯(60 美元)，比如说两个麦克风(80 美元)，用于主唱和伴唱，当然，还有该死的游戏(60 美元)。把它们加起来，你已经在你的虚拟乐队上花了 400 美元，而这个虚拟乐队永远不会离开客厅。当然，这比组建一个真正的摇滚乐队要便宜 2000 美元，所以你可以说你在省钱。这款游戏将同时在 PS3 和 Xbox 360 上发布，所以无论你站在哪一边，都最好开始存钱。孩子呢？去买一个漂亮的吉他、一些鼓和一个麦克风，然后组建自己的乐队。相信我们:无论你在“最后的倒计时”中有多出色，粉丝们都不会蜂拥到你的虚拟风格中。
