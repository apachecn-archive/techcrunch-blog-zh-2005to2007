# 飞利浦生产第 100 万台流光溢彩平板电视

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/29/philips-makes-millionth-ambilight-flattv/>

飞利浦在比利时布尔吉斯的制造工厂生产了第 100 万台 42 英寸流光溢彩平板电视。这算不算里程碑？希望它是佛兰德欢呼和斯特拉回合。坦率地说，我很惊讶他们生产了这么多这种动物。

据说流光溢彩系统通过从布景后面投射环境光来改善观看体验。我对这些功能的实用性有点犹豫，但我还没有实际看到它的运行，所以我不太了解第一手资料。
