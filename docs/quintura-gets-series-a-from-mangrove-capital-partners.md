# Quintura 从红树林资本合伙人获得 A 轮投资

[![](img/0849bf8e3299645fb281dde87fad9684.png)](https://web.archive.org/web/20221201172920/http://www.quintura.com/) 基于视觉的搜索引擎 [Quintura](https://web.archive.org/web/20221201172920/http://www.quintura.com/) 已经从该公司的原始种子投资者红树林资本合伙人那里获得了“数百万美元”的首轮融资。

这笔资金将用于扩大 Quintura 网站搜索联盟模型的规模，并使用 Quintura 的“神经网络技术”建立语义网络索引。

Quintura 显示了一个由相关关键词和短语组成的交互式搜索云，以优化和缩小搜索范围。

Quintura 附属计划将涉及网站所有者在他们的网站上放置 Quintura 搜索云，并基于付费搜索位置采用收入分成模式。

迈克尔·阿灵顿[之前曾用肯定的措辞描述过这个网站，他写道，他发现 Quintura“对于基于搜索的研究或浏览是有用的，在那里我试图找到关于给定主题的更多信息”；在众多新的 Web 2.0 搜索引擎中，它无疑是较好的产品之一。](https://web.archive.org/web/20221201172920/http://www.beta.techcrunch.com/2007/02/26/quintura-search-engine-relaunches/)
