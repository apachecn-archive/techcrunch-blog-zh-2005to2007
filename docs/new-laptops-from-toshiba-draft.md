# 东芝新推出的笔记本电脑

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/02/new-laptops-from-toshiba-draft/>

![](img/6812ffd3dc0f75ead4ec22bf3a13a348.png)

东芝似乎真的已经在市场两端的笔记本电脑上迈出了步伐:超便携和游戏。

Portege R500 是东芝在移动领域的最新产品。令人惊讶的是，这款两磅重的笔记本配备了一个光盘驱动器，它号称是世界上最薄的，厚度为令人印象深刻的 7 毫米。

点击查看更多图片和规格…

R500 包含一些很酷的功能和选项，如可用的 64GB 固态硬盘存储，当结合没有光驱时，笔记本电脑的重量下降到令人难以置信的 1.72 磅**。固态硬盘和 LED 背光显示屏的结合显然为 R500 提供了世界上最好的电池续航时间，估计为 10 小时。**

除了上面提到的硬件，这款超便携设备还配备了蓝牙 2.0 和 draft n wireless。1.2 GHz Core 2 Duo 和标准 1GB RAM 确保它是一款出色的启动性能。

起价 1999 美元

[产品页面](https://web.archive.org/web/20201022181732/http://www.toshibadirect.com/td/b2c/cmod.to?coid=-33781)

东芝的怪物 17 英寸卫星 X205 配备了所有常见的桌面替换装饰品。虽然与外星人的同类产品相比略显逊色，但 1999 美元的起价也相当便宜。

基本配置包括:

–Nvidia GeForce Go 8700m GT，配备 256 MB DDR 3 图形内存

–2GB DDR2 内存，配有 1gb 英特尔睿频内存

–17 英寸 WXGA 显示屏

–蓝牙 2.o EDR(增强型数据速率)

–802.11 a/g/n 无线

–高清 DVD 光盘

–4 个带低音炮的扬声器

–240 GB 存储 2×120(抱歉，没有 raid！)

多花 500 美元，CPU 就能提升 200MHz，硬盘空间提升到 320(仍然没有 raid ),显存提升到 512MB。
