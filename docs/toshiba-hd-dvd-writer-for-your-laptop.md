# 适用于您笔记本电脑的东芝 HD DVD 刻录机

![toshiba.jpg](img/d04f41a225f5cccd964d2f229b2a199f.png)

你对拥有一个超薄的 HD DVD RW 硬盘的前景感到兴奋吗？很确定 HD DVD 会赢得这场格式战吗？那么，你很幸运，因为东芝已经宣布了 SD-L912A，这是世界上第一款满足你对可擦写 HD DVD 媒体渴望的产品。

除了 HD DVD 不可避免的衰落之外，唯一的问题是 1 倍的读/写速度。Qosmio 还可以写入 HD DVD-R DL、DVD RW 和 CD RW 格式。寻找七月的驱动器。
