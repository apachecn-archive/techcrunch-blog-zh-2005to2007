# 突破:XM 和天狼星合并

ABC 新闻报道 XM 和 Sirius 正在合并，并且即将完成。实际的合并预计将在今天下午晚些时候举行新闻发布会。早些时候，有一些传言说 XM 和 Sirius 想要合并，但由于联邦通信委员会的原因而无法实现。尽管联邦通信委员会将会给两家公司带来麻烦，但他们应该能够以某种方式解决这个问题。
