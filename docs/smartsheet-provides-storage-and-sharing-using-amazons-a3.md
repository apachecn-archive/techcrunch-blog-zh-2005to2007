# Smartsheet 使用亚马逊的 S3  提供存储和共享

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/12/06/smartsheet-provides-storage-and-sharing-using-amazons-a3/>

[![](img/8eb5b16bdbc8106ef27726c3165a5440.png) ](https://web.archive.org/web/20131110050544/http://www.smartsheet.com/home) [Smartsheet](https://web.archive.org/web/20131110050544/http://www.smartsheet.com/home) ，一个基于网络的电子表格软件，将于明天在亚马逊的[简单存储服务(S3)](https://web.archive.org/web/20131110050544/http://www.amazon.com/gp/browse.html?node=16427261) 上发布新的存储和共享功能。

之前，该公司的服务器位于德克萨斯州的达拉斯，但随着共享文件和发送附件的新功能的推出，Smartsheet 需要亚马逊的强大动力。S3 允许开发者使用存储来运行他们自己的全球网络，这是 Smartsheet 在 12 月发布时所需要的。新版本允许用户从硬盘或服务器上附加文档，并通过电子邮件发送对文档所做的任何更改的更新。

Smartsheet 还宣布了一个新的[定价结构](https://web.archive.org/web/20131110050544/http://www.smartsheet.com/products/pricing)，它将包括一个没有存储和电子邮件功能的免费软件版本。

Smartsheet 总裁马克·马德尔(Mark Mader)在周三的电话简报中表示:“我们正在从企业团队定价结构转向扩大受众范围的结构。”“我们正在进入高容量类型模式，在社区环境中，你最不想做的事情就是引入使用它的障碍。”

Smartsheets 可以与 Excel 互操作，但 Mader 说它不是程序的复制。当被问及为什么 Smartsheet 与 Google Spreadsheets 或 Zoho Sheet 有所不同时，他指出了三个与众不同的功能:(1)它为任何文档变更提供了简化的更新；(2)它允许“相关警报”，这意味着您可以在您指定的时间间隔内仅订阅您想要的关于文档的警报；(3)它给出了所有领域的变化历史。

Smartsheets 还允许用户通过电子邮件请求对文档进行更改。因此，如果我想让其他人更改某个单元格的值，我可以通过电子邮件发送请求，这样他们就可以在电子邮件中填写答案，这样就可以在这个人没有登录到他们的 Smartsheet 帐户的情况下进行更改。Mader 说，该公司将在下一个版本中研究 Outlook 集成。

Smartsheet 是一个可靠的产品，但一个只有电子表格的 Web 应用程序真的能生存吗？Zoho 展示了它的演示应用程序，作为区别于谷歌文档&电子表格的因素，现在它有了办公插件，使它更加普遍有用。另外，它是免费的。我不确定谁会为 Smartsheets 付费，只是因为它更方便一些，但该公司表示，自 8 月份推出以来，已有约 2000 家公司在使用它的程序。我完全支持市场竞争，但我真的认为，如果你想成为一个 SaaS 文档程序，你需要一套全面的办公类应用程序。
