# 个人资料-小浏览器 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/16/profile-bitty-browser/>

小小浏览器是一个可以嵌入网页的迷你浏览器。

这是斯科特·马修斯的创造。今天早些时候我和斯科特谈过了。他正在收集客户反馈，并将很快发布新功能。

斯科特称他的项目为网络“画中画”。他还指出，这是弥合移动内容和普通内容之间差距的完美方式——Bitty Browser 是将移动内容引入网站的好方法。

Bitty Browser 是基于 javascript 构建的，几乎可以在所有的浏览器上运行，并且有一些很好的快捷方式来查看 feeds，delicious tags 等等。更多信息，请参见他们的[帮助页面](https://web.archive.org/web/20221007212224/http://www.bitty.com/info/)。

我们在几分钟内创建了一个显示 TechCrunch RSS 提要的浏览器(我没有将其嵌入到帖子中，因为提要阅读器似乎经常会破坏 html)。

欲了解更多信息，请参见[弗雷德·威尔逊](https://web.archive.org/web/20221007212224/http://avc.blogs.com/a_vc/2005/08/bitty_browser.html)和[埃里克·肖恩菲尔德](https://web.archive.org/web/20221007212224/http://business2.blogs.com/business2blog/2005/08/a_bitty_browser.html)。
