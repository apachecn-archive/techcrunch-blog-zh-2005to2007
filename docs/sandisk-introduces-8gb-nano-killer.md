# 闪迪推出 8GB“纳米黑仔”

![e280.jpg](img/f772db84dc11f7c72b48ee6c5c22508a.png)

闪存制造商 SanDisk 公司今天宣布对其 DAP 播放器系列进行大规模调整，其中最引人注目的是即将推出的 e280。这款新播放器将配备 8GB 闪存和一个 microSD 扩展槽。除了新型号，现有的 2g、4g 和 6g 的 Sansa 播放器的价格将降低 30%，使这些播放器的价格范围达到 140 美元至 240 美元。
