# Trulia 测试版推出新功能和部件

[![](img/931a7bda6a4a8ec92bad06a59c11245c.png)](https://web.archive.org/web/20220816010559/http://trulia.com/) 房地产列表和搜索引擎 [Trulia](https://web.archive.org/web/20220816010559/http://www.beta.techcrunch.com/2005/10/30/real-estate-vertical-search-with-trulia/) 现已退出测试，其特色是一些可用性升级、新的答案服务和网站插件。

Trulia 的首页和搜索功能进行了一些可用性调整。新的首页更加清晰地突出了搜索 Trulia 数据的四种不同方式，并显示实时数据馈送。Trulia 的搜索页面有一个更简单的侧边栏界面，用于优化您的搜索。该引擎还会根据其他用户的搜索方式，建议搜索相关的社区。他们的热图和搜索订阅功能(RSS 和电子邮件)仍然存在，但电子邮件订阅现在将跟踪单个物业的生命周期，并推荐与所选物业或搜索类似的房源(类似的社区、价格、类型等)。).

随着改造的主要释放是他们自己的问答服务，Trulia“声音”。这项服务让会员发布和回答关于社区质量方面的问题。问题可以通过地理位置和标签进行搜索。其他成员可以对每个问题或答案进行评分，这会影响您对网站内容的整体 karma 评分。这将有助于一定程度的自我推销，因为提问和回答的评分很高，可以让你在问答排名中获得更好的位置。我可以想象，让发帖者**获得**列表搜索排名，而不是通过将 Q & A 排名与列表排名挂钩来付费，将会让这项服务火起来。Zillow 有一项问答服务，但它主要是针对特定商品的提问。

最后，Trulia 公开宣布 widgets.trulia.com 和 housingwidgets.com。widgets 网站目前有四个 widgets，它们是基于几个月前发布的 API 开发的。这四个小部件是房屋地图(预先存在)、Trulia Stats(跟踪平均房价)、Home roll(新房源，已过滤)和 Trulia 搜索框。Housingwidgets 是一个新的网站，它聚集了与房屋租赁清单相关的热门 widgets 的链接，如 MeeboMe 或 MyBlogLog。遗憾的是，您不能使用 Trulia 帖子中的任何小工具。

Trulia 声称在 4 月份有 150 万独立访客。在同一时期，Zillow 索赔 400 万英镑。这两个服务功能相近，涵盖搜索、热图、趋势和指南。然而，Trulia 跟踪超过 200 万个实时列表的关注数据和历史价格趋势。Zillow 为雅虎房地产的历史价格图表提供了动力，但也为超过 7000 万套房屋提供了其标志性的基于公式的 Zestimates。
