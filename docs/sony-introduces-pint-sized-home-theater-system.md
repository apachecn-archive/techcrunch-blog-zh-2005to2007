# 索尼推出小型家庭影院系统

![dav_is10_angled_view_med.jpg](img/138018a878e84d5a949fafefc4aeca60.png)

索尼今晚在纽约发布了一款小型 5.1 声道家庭影院系统。这些卫星大约有一个苹果那么大，但是不要被它的大小所迷惑。这个小包装总共能发出 450 瓦的能量。五颗卫星各输出 50W，而双向低音炮每通道输出 100W。对于这样一个小系统来说还不错吧？

DAV-IS10 以 DVD/CD 播放器为中心，只要你有一台高清电视，它就可以通过 HDMI 输出将视频升级到 720p 或 1080i。如果这还不够，该系统还利用了 Bravia 剧院同步技术。啊？如果您有一台 Bravia 电视，那么您可以利用上述 HDMI 接口，通过触摸按钮轻松地从一个输入导航到另一个输入。很可爱，是吧？

数字媒体端口输入还允许您通过蓝牙和 Wi-Fi 连接配件。不够精明，无法设置您的多声道环绕声系统？没关系，因为不是每个人都这样，这就是为什么索尼的数字电影自动校准使设置变得轻而易举。

该系统将于 7 月上市，售价高达 800 美元。你可能会奇怪，当我知道这件事的时候，为什么我要用新闻照片，对吗？这很容易回答。在过去的十年里，索尼一直令人失望，我只是不忍心看到他们的另一个产品可能会失败。觉得我错了？如果我是，我们现在都会带着随身听到处走，而不是 iPods。但是他们真的忽略了这一点，所以到处都是 iPods。我还需要提到 PS3 的[或者](https://web.archive.org/web/20201130042704/http://crunchgear.com/2007/05/14/sony-losing-the-game/)[他们燃烧的电池](https://web.archive.org/web/20201130042704/http://crunchgear.com/2007/05/16/oh-boy-another-sony-battery-catches-fire-in-a-laptop/)吗？我试着爱你，索尼，但你让这变得如此艰难。也许我的醉酒状态让我感到痛苦，或者我只是在说实话。不管怎样，我都会停下来睡觉。花 800 美元买这东西，玩得开心。
