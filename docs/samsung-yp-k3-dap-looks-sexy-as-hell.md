# 三星 YP K3 DAP 看起来性感极了

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/30/samsung-yp-k3-dap-looks-sexy-as-hell/>

![](img/c7d212bb13e7ddba7578157a5dc8020c.png)

这就是我们所说的性感 MP3 播放器。上图是 YP-K3，三星 YP-K5 音频播放器的继任者。这个坏男孩有一个伟大的设计，据说是三星迄今为止最薄的球员。预定在 2007 年国际消费电子展上首次亮相的 YP-K3 不会像它的哥哥姐姐那样有内置扬声器。此外，显示器现在是 1.8 英寸，没有人真正知道它看起来有多好。不过，从这些美味的照片来看，这是一个我实际上会买的 MP3 播放器，而不是 iPod(这说明了很多)。当 CES 2007 在一月份到来的时候，我们一定会给你一个最新的消息。
