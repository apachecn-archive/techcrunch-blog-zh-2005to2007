# Plugaid 将把引用通告带到 MySpace，Xanga

[![](img/2517f7688b1ae909203bb66f8d8c758c.png) ](https://web.archive.org/web/20220812125147/http://www.plugaid.com/) [Plugaid](https://web.archive.org/web/20220812125147/http://plugaid.com/) 是一种创新而简单的方式，让博客作者使用不支持自动引用通告发现的系统来显示他们的帖子所属的对话。该工具针对 MySpace、Xanga 和 Blogger.com 等系统。我认为在那里会很受欢迎。

Plugaid 是由韩国高等科学技术学院的学生 Trent Kang 共同创立的。它目前处于 Alpha 模式，为感兴趣的未来用户收集电子邮件。创作者已经开始了一个可以参与测试的对话，正如我在这篇文章的底部所做的那样。

它使用嵌入式 flash 来显示参与讨论特定帖子的博客帖子，下拉摘录和完整帖子的链接。如果可能的话，还会显示每个链接博主的照片。用户只需提供一个密码，如果他们希望以后从显示中删除该链接，就可以使用该密码。

在下面的例子中，我点击了“参与”，输入了一个密码，并得到了几行代码来包含在我的帖子中。这意味着我正在参与关于 Plugaid 演示的对话，其他参与的人应该很快会在他们的 Plugaid 框中看到这篇文章的链接。当你点击“更多”按钮时，出现的链接可以显示完整的摘录，在这里你还会看到链接 URL 和一个访问该博客文章的按钮。现在摘录都很短，因为只有少数人写非常短的测试帖子。

一个不利的方面可能是，参与 plugaid 讨论似乎不能提供 HTML 格式的到对话中其他帖子的直接链接，因此对 link love 没有帮助。不过，我想这对于许多潜在用户来说并不是什么大问题。

Plugaid 使用的语言也有点不清楚，需要澄清一下。

这是一个简单的工具，还没有明确的商业模式，但我认为它真的会流行起来。
