# 三星 Q1/Q30 设备使用固态闪存

![](img/4c6ebff6173a7dc0d0928ec8034ca60b.png)三星没有给超便携设备增加使用更多电能、产生更多热量、移动太频繁容易损坏的硬盘，而是在 Q1 和 Q30 移动设备中安装了固态硬盘(SSD)。

Q30 是一款 12.1 英寸的笔记本电脑，Q1 是一款配有 32GB 固态硬盘的 UMPC。与旧的硬盘驱动器 Q1 相比，新的 Q1 启动速度快 25%到 50%，读取速度为 53MB/s，写入速度为 28MB/s。这种新的驱动器并不便宜，因为它将设备的价格提高到了 1399 英镑，或 2644 美元。真的会有人花近 3000 美元买这些东西吗？我们不这么认为。
