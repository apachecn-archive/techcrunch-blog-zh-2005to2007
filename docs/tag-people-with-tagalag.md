# 用 Tagalag 标记的人

 [](https://web.archive.org/web/20220527125422/http://www.tagalag.com/) [Tagalag](https://web.archive.org/web/20220527125422/http://www.tagalag.com/) 是一项让你通过电子邮件地址标记*人*的服务。它不是像 [43 人](https://web.archive.org/web/20220527125422/http://www.43people.com/)那样的“致敬”网站，因为只有知道一个人邮箱地址的人才能为那个人添加标签。

如果您创建了个人资料，您可以添加自己的个人和地理信息。

我不知道 Tagalag 是否有可行的商业模式，但我喜欢给人贴标签的想法。随着它的发展，这可能会变得有趣。
