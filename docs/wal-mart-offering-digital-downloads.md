# 沃尔玛提供数字下载

![](img/27ddf8853225af74fd7b770e24daaaa6.png)

沃尔玛开始涉足数字下载业务，与所有六大电影公司达成协议:派拉蒙、华纳兄弟、迪士尼、索尼、福克斯和环球。不仅如此，超市还能让电视网络站在自己这边。来自 Nickelodeon、FX、Comedy Central 和其他维亚康姆电视网的节目每集收费 1.95 美元。

很显然，沃尔玛也委托惠普设计了这个网站，看起来有点朴素，但也没关系。电影的价格从 7.50 美元到 19.88 美元不等，这取决于电影的名称——后者对于数字下载来说有点太贵了。我想在抨击或赞扬沃尔玛之前，我们还得等着看它的服务表现如何。
