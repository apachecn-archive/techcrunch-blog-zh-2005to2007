# 蜘蛛侠获得 BSOD-

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/21/spiderman-gets-bsod/>

![](img/19b7ba5a3740b251ff420603c73fdeaf.png)

可怜的蜘蛛侠。他想做的就是在这个兆字节的环境中使用他的计算机。但不幸的是，当他去打开 Word 98 时，他得到了臭名昭著的蓝屏死亡。也许如果蜘蛛侠为他的超级计算机选择了一个更可靠的操作系统，如 Linux 或 OS X，他就能让他的计算机工作，这样他就可以去踢毒液的屁股了。但是，唉，他坚持试图拉下一个注册表编辑器，并安装诺顿防病毒。
