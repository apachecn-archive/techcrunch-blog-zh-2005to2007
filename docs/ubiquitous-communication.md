# 无处不在的通信

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/14/ubiquitous-communication/>

如今的大学生常常有无数的设备和太多的信息需要管理。但是 clearTXT 最近推出了 Ubiquity，这是一个面向学院和大学的分布式信息传递系统。它目前被 70 多个教育机构使用，包括辛辛那提大学、西顿霍尔大学和孟菲斯大学。

这个程序不需要下载或安装软件应用程序，但可以在大多数手机和服务运营商上运行，提供一种传播信息的方式，如紧急警报、校园消息、班级作业、日历事件和学生政府新闻。因为这个系统可以由学校管理，所以它不发送商业信息或其他广告。
