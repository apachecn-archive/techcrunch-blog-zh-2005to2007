# 儿童机器来了

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/25/the-childrens-machine-arrives/>

我最喜欢的技术项目之一刚刚宣布了它的新名字。由尼古拉斯·尼葛洛庞帝带头的百元笔记本电脑以前被称为 OLPC，现在被称为 CM1，或儿童机。

这款 7.5 英寸、1,200 x 900 像素可配置、可联网的笔记本电脑运行由 400MHz AMD Geode 处理器支持的 Fedora Linux 发行版，旨在为发展中国家的孩子提供一种超廉价的方式来访问互联网，并弥合所谓的技术鸿沟。![](img/fcde240b2c516d83dfd0ba2a541783d9.png)

和以往一样，微软当然不会对这种潜在的威胁视而不见。最近，研究和战略官员 Craig Mundie 展示了一款名为 **FonePlus** 的原型手机，它可以连接键盘和电视，为人们提供基本的计算功能。

微软已经提供了 Windows XP 的完全缩小版 [Starter Edition，并一直在较贫穷的国家推广 FlexGo 即付即用计划，在这些国家，人们支付较低的前期成本购买一个系统，然后使用预付费卡访问它。](https://web.archive.org/web/20140121145238/http://www.winsupersite.com/showcase/winvista_editions.asp)

两周之后，我希望能在英国电信的 21 世纪全球峰会上看到儿童机器的实物，尼古拉斯·尼葛洛庞帝将是主要发言人。我将有机会对 Nicholas 进行简短的采访，所以如果你有任何问题想让我问他，请告诉我。
