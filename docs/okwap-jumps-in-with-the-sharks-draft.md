# OKWAP(谁？？)推出手机

台湾的 OKWAP 宣布他们正跳入鲨鱼横行的智能手机市场。S868 运行 Windows Mobile 5.0，并集成了一个 2.4 英寸触摸屏，200 万像素的自动对焦摄像头，GPRS，蓝牙，红外，手写笔和字母数字键盘，但它仍然保持了其精致的外形，重量仅为 4.3 盎司。

为了让自己与众不同，OKWAP 增加了自己的快捷菜单以及键盘上的虚拟滚轮。例如，在浏览图片时，你只需绕着“5”键打圈，而不是按下按钮或用滚轮滚动。S868 采用英特尔 PXA270 416Mhz CPU，拥有 128MB ROM 和 64MB RAM，可通过 SD 或 MMC 卡进行扩展，适合大男孩使用。S868 将于 2 月底在台湾上市，售价 470 美元。
