# WeShow:人力视频目录

[![weshow.jpg](img/5858bcc94aefafa936a9f220b3424ef6.png)](https://web.archive.org/web/20221004035011/http://www.weshow.com/) 纽约人力在线视频指南 [WeShow](https://web.archive.org/web/20221004035011/http://www.weshow.com/) 今天推出。

WeShow 不是这种类型的第一个网站，但它确实有强大的支持。投资者包括 Pilot Group(由 MTV 之父、美国在线时代华纳前首席运营官鲍勃·皮特曼拥有和管理)和比尔·萨尔曼(哈佛商学院副院长兼创业学教授)。资金数额此前没有透露，但据信约为 500 万至 700 万美元。

WeShow 从 YouTube、DailyMotion、Metacafe、Google Video 和其他网站中选择和组织最佳在线视频，并提供了一种编辑方法。该网站还提供每月一次的“WeShow Awards”，号称是“世界上最大的在线视频竞赛”，用户可以对 WeShow 有本地版本的每个国家(美国、英国和巴西)的内容进行投票。WeShow TV 通过提供一个以一些“世界上最好的视频”为特色的评论式节目来完善这一产品。

这项服务的吸引力实际上取决于在线视频消费者是喜欢原始视频还是精致视频。我一直认为找到有趣的视频是一半的乐趣，但其他人可能更喜欢被填鸭式喂养。有创意的是，WeShow 开始发布网站的调查结果表明，60%的美国人被大量的在线视频所淹没，其中 46%的人不再看更多的在线视频，因为他们害怕从太多的搜索结果中筛选。我总是对这种数字有点怀疑，但这是应该的:公司在提供解决方案之前就证明了一个问题，这是一些创业公司很好的教训。

总的来说，它很擅长自己的工作。导航简单，视频选择丰富多样。对于袖手旁观来说，这是否是一个足够强大的想法，这只是一个只有时间才能回答的问题。
