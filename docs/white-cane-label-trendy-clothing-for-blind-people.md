# 白手杖标签:盲人的时尚服装

罗彻斯特理工学院的两名学生 Jaimen Brill 和 Asmah Abushagur 正在为盲人和视障人士创建一个在线服装网站。这个名为 [White Cane Label](https://web.archive.org/web/20210511192426/http://whitecanelabel.com/) 的网站是一个非营利组织，旨在帮助盲人在网上购物，并在没有视力正常的朋友帮助的情况下轻松跟踪他们的衣柜。

该网站的界面将由声音和文本驱动，而不是图像。顾客将收到基于他们对服装问卷的回答的建议，该问卷评估他们的个人风格。这些建议将包括每件商品的详细描述，包括款式和价格范围。订购一件商品后，他们会收到标有盲文标签的服装，标签上标明了设计师、服装类型、尺寸和颜色，以便于参考。第二个标签将包括洗涤说明。

他们完全依赖设计师的捐赠来供应库存，但将收取全价，以努力阻止有远见的买家利用任何交易。

该网站最初是一个学校项目，但随着他们的深入研究，很快就吸引住了这两个人。在罗马夏季时装周期间，White Cane Label 在罗马举行的时尚圆桌会议上宣布[将于今年秋天推出。](https://web.archive.org/web/20210511192426/http://www.reuters.com/article/internetNews/idUSPAR36781320070713?feedType=RSS)
