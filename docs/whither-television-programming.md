# 电视节目向何处去？–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/09/whither-television-programming/>

作为我们上一篇关于电视网络在互联网和 iTunes 上的活动的文章的后续，我们进一步探索了电视节目以及它在主流网络之外的网络上是如何播放的。

我会避开雅虎、谷歌、YouTube……所有在这个网站上被[广泛覆盖的](https://web.archive.org/web/20211206235434/http://www.beta.techcrunch.com/2005/11/06/the-flickrs-of-video/)网站。相反，我想强调几个较少覆盖的服务。

很难在我刚刚提到的网站和我将在下面讨论的网站之间划清界限，因为它们共享许多功能(并且许多使用 Flash 作为底层技术)。一个区别是上面的网站不为用户做编程决定。下面的网站确实提供了某种形式的节目——有些人决定在这些网站上展示什么以及何时展示。这两种方式在网上都有空间，尽管现在大多数目光和风险资金都流向了视频门户网站，但我相信我们最终会看到初创企业社区对编程产生类似的兴趣。

**我太老了，不适合这个！！！**

毫不奇怪，许多在线节目，如 [MTV Overdrive](https://web.archive.org/web/20211206235434/http://www.mtv.com/overdrive/player.jhtml?launchedFrom=/overdrive/) ，迎合了 25 岁以下的人群，但对我们其他人来说也有一些好东西: [BloombergTV](https://web.archive.org/web/20211206235434/http://www.bloomberg.com/) 和 [MLB.tv](https://web.archive.org/web/20211206235434/http://mlb.mlb.com/NASApp/mlb/mlb/subscriptions/mlbtv.jsp) ，例如，提供相关内容和类似频道的在线体验。

在年轻人市场，MTV 的影响力是毋庸置疑的。ManiaTV 、 [Fuse.tv](https://web.archive.org/web/20211206235434/http://www.fuse.tv/) 、 [MusicplusTV](https://web.archive.org/web/20211206235434/http://www.musicplustv.com/) 和 [CurrentTV](https://web.archive.org/web/20211206235434/http://www.currenttv.com/) 虽然各自的权利肯定不同，但他们相信音乐、现实、前卫和/或政治和社会意识的某种结合是成功的关键。

疯狂电视

 [](https://web.archive.org/web/20211206235434/http://www.maniatv.com/) 那么汤姆·格林就是这样了？也许我是*而不是*太老了，因为我喜欢疯狂电视。快看畸形秀！这里有足够多的好东西可以看，事实上我很惊讶它没有更多的观众，特别是考虑到过去 6 个月视频网站的爆炸式增长。我认为这种编程是有空间的，虽然 Alexa 的数据并不意味着失败，但它肯定不会蓬勃发展。他们的受众是否足以盈利，我不知道。至少他们不用担心巨大的生产成本。

**Current TV**
 [](https://web.archive.org/web/20211206235434/http://www.currenttv.com/) 最著名的是戈尔汽车，Current 是为那些更有政治头脑和社会活动的人设计的。有很好的原创编程阵容，似乎有很强的社区参与，我发现这是一个很好的可用网站，但它还没有起飞。也许阿尔·戈尔的电影会给它打一针强心剂，但迄今为止，它还没有表现出放弃其微不足道的流量排名的迹象。下届总统选举时会升温吗？谁知道呢？我敢打赌，它会在美国政治中的许多重大两极分化问题上度过它的一天。也许它最大的问题是它更倾向于知识分子。让你的朋友们看一个好色的毛驴视频并不难，但是试着让同一群人看一个关于在学校教禁欲的视频。

**iBlueTV**

 [](https://web.archive.org/web/20211206235434/http://www.ibluetv.com/) 另一个有点不同的有趣编程概念是 [iBlue](https://web.archive.org/web/20211206235434/http://www.ibluetv.com/) 。IBlue 从他们的网站流几个不同的频道。好像范围从摩托越野到科幻再到独立电影制作。这里有一些不错的消磨时间的内容。至少，我可以看到它找到了同样喜欢激光节目的大学生观众。

**我还没老到这个地步！！！**

经典电视节目和老电影(多年来一直是非黄金时段的电视素材)也在网络上找到了自己的家。最近的一些交易，比如 Guba 与华纳的交易，将会给复古运动增添活力。今天，AOL 以[在电视](https://web.archive.org/web/20211206235434/http://television.aol.com/in2tv)中领先群雄。可以看 42 个“经典”节目从蝙蝠侠动画片到八个就够了(那时候就够了)。节目可以以相对较低的质量进行流式传输，也可以以更高分辨率的格式(Hi-Q)向您推送内容，这允许全屏观看。由于一些老节目灯光不好，高 Q 格式是必要的。我通过普通频道看了一集《斯潘塞出租》，当夜幕降临时，我分不清斯潘塞和坏人。

同样，如果你对游击队情有独钟，你可能会想看看 [Retrovision.tv](https://web.archive.org/web/20211206235434/http://www.retrovision.tv/) 和 [AmericafreeTV](https://web.archive.org/web/20211206235434/http://www.americafreetv.com/) (还有其他的)，它们提供老的电视节目和电影。记住，你在这里得到你所得到的——依靠广告收入的业余爱好者网站——所以不要期望完美。

**聚合器**

也有网站聚合内容，并开始类似于传统的有线电视频道。虽然有许多像 [MeeVee](https://web.archive.org/web/20211206235434/http://www.meevee.com/) 这样的免费网站将你引向免费的流媒体，但像 [MobiTV](https://web.archive.org/web/20211206235434/http://www.mobitv.com/) 、 [VDC](https://web.archive.org/web/20211206235434/http://www.vdc.com/) (仅限 Windows)或 [JumpTv](https://web.archive.org/web/20211206235434/http://www.jump.tv/index.html?&language=english) 这样的网站提供了一个通过订阅观看许多频道的枢纽:

VDC

 [](https://web.archive.org/web/20211206235434/http://www.vdc.com/) [VDC](https://web.archive.org/web/20211206235434/http://www.vdc.com/) 基本上就是把有线模型搬到网上(并和有史以来最烂的 logo 结合)。声音和图像质量都很好，而且很容易使用。但是 11.95/月，频道阵容还没有。它的 20 个左右的频道主要是新闻和购物的混合，尽管他们最近增加了 TLC、Discover 和 Animal Planet(目前都只针对手机)。他们在网站上记录了他们对改进阵容的兴趣，所以我确信更多的东西正在工作中。虽然这将增加他们成功的机会，但我认为 VDC 更大的问题在于将电缆移植到网络的模式。我要问你:当有像 Slingbox 和 [Orb](https://web.archive.org/web/20211206235434/http://www.beta.techcrunch.com/2006/03/23/orb-stream-pc-media-to-mobile-device/) 这样的选项可以在你的电脑或手机上提供你的家庭有线电视阵容时，第二个有线电视订阅有什么真正的优势吗？我喜欢更便宜的有线电视，我不介意通过网络获得，但如果没有与我的电视相当的阵容，我不会添加或切换。

**jump TV**
 [](https://web.archive.org/web/20211206235434/http://www.jumptv.com/) [jump TV](https://web.archive.org/web/20211206235434/http://www.jump.tv/index.html?&language=english)聚合了全球数百个频道。每个频道每月 9.95 英镑，可能会变得很贵，所以 Jump 提供了一些套餐，比如每月 22.95 英镑的 20 个频道的阿拉伯语套餐。我怀疑，从长远来看，这个特殊的利基市场将证明对 Jump 有利可图。尽管这不是网络上最性感的电视节目，但它显然满足了消费者的需求，因为如今有这么多的人生活在自己的出生国之外。虽然目前在 Alexa 的流量排名中排名约为 13，000，但其稳定的流量上升趋势表明存在一些需求。世界杯可能会进一步扩大它的影响范围。

**移动电视**

 [](https://web.archive.org/web/20211206235434/http://www.mobitv.com/) [MobiTV](https://web.archive.org/web/20211206235434/http://www.mobitv.com/) 可能最终会拥有这个领域最引人注目的产品。他们为移动蜂窝设备提供 50 个频道的有线电视订阅，最近还为任何接入 Wi-Fi 热点的设备提供订阅。如果他们的服务可以通过网络从任何 ISP 获得，继续传统的有线电视或卫星电视订阅的需要就成了严重的问题。

**总结**

互联网所带来的巨大的内容革命在我之前说过的那种网站上可以看到，比如 Youtube。现在，任何人都可以在几分钟内廉价地上传、分享和修改内容。因此，我们这些业余爱好者制作和观看视频内容的规模是我们几年前无法想象的。每天至少有 100，000 个视频被上传到像 YouTube 这样的网站上，而且这个数字还在快速增长。

但这并不意味着我们所知道的电视节目即将消失。事实上，我认为现在编程的整体质量比几年前更高，也许是因为来自网络的思想分享的竞争。一些电视制作人在播出之间从网络中收获了足够多的好处(如《迷失》)，我认为下一阶段将意味着改变节目内容，因为电视继续从被动向某种程度的互动迁移。在电视广告中已经司空见惯，那么在节目中还要多久才会司空见惯呢？看看我们信任的男人中的[。如果它像广告宣传的那样有效，那么我们会一边看节目一边看里面的节目，我们会决定故事的走向。这是适应其媒体的编程内容。交互性和社交网络——网络擅长的两件事——将会推动它的发展。](https://web.archive.org/web/20211206235434/http://www.personalscreen.com/projects.html)
