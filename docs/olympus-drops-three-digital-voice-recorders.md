# 奥林巴斯推出三款数字录音机

奥林巴斯拥有数码录音机市场，因此它很容易放松，不用担心更新或改进其产品系列。相反，该公司发布了三款新机型，DS 系列，正好赶上了~~假期~~ CES。

奥林巴斯 DS-30 (256MB，150 美元)、DS-40 (512MB，200 美元)和 DS-50 (1GB，250 美元)拥有高音质和录音质量，以及一系列功能，使它们在娱乐和工作中同样出色。每个型号都有一个可拆卸的立体声麦克风和内部单声道麦克风，以及三个灵敏度级别的设置(演讲、会议和听写)。也有三种类型的噪声消除(低切滤波器、语音滤波器和噪声消除)，它们能够以超高质量的立体声 SXQ 模式录制，可以录制和播放高达 44.1kHz、128Kbps 的音频内容。此外，只需两节 AAA 电池，您就可以连续工作 32 小时。

对于那些不太擅长学习新设备的人，或者即使你只是想在弱光环境下工作，所有型号都具有声音指令和菜单选项和选择确认功能。

奥林巴斯想提供的不仅仅是一个录音机，所以每个单元都支持播放 MP3 和 Audible.com 文件格式的音乐，以便听你最喜欢的有声读物。加载音频是通过捆绑的 DSS 播放器软件完成的。您也可以设置它来听播客。只需通过随附的 USB 电缆将其中一台录像机连接到您的 PC，然后将 RSS 源拖放到播客文件夹中。每次你把录像机连接到电脑上，它就会更新信号。
