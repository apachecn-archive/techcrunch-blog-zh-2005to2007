## 概述:

展望很酷。Skype 很酷。让几个澳大利亚人一起做爱，你会得到什么？ [Skylook](https://web.archive.org/web/20201124163137/http://www.skylook.biz/) 。它 skypefies 的前景，这是真棒。

[Skype](https://web.archive.org/web/20201124163137/https://crunchbase.com/organization/skype) 确实已经有了一款 Outlook 产品，我们[上个月](https://web.archive.org/web/20201124163137/http://www.beta.techcrunch.com/?p=70)对其进行了简介。Skylook 要高级得多，也很容易使用(尽管与 Skype 的 Outlook 工具栏不同，它不是免费的)。

下载大约 2.2 mb。你可以免费试用 14 天。29.95 美元后购买。

安装后，有三个我喜欢的主要功能区域。

我喜欢的第一个功能是自动记录所有 skype 通话和 IM 聊天。录制后，Skylook 将文件放在一个特殊的文件夹中，可以通过电子邮件转发，搜索等。

我喜欢的第二个功能是，您可以直接在 Outlook 中查看 skype 联系人的状态信息，并且可以直接从 Outlook 中启动即时消息和语音通话。

我喜欢的第三个功能是通过 skype 或 skype out 直接从 vcard“一键”呼叫联系人的能力。

Skylook 的首席执行官杰里米·黑格(Jeremy Hague)写信给我，提供了以下信息:

> 【Skylook 与 Skype Outlook 工具栏有何不同？
> 
> Skylook 不仅仅是 Skype 的 Outlook 工具栏。Skylook 对语音和即时消息的作用相当于 Outlook 对电子邮件的作用。Skylook 允许您存储并在以后检索您所有的 Skype 通讯，就像您现在存储和检索电子邮件一样。
> 
> Outlook 是管理电子邮件的好工具。它可以让你以多种不同的方式存储、组织、分类和搜索。如果您不仅可以在电子邮件中使用所有这些出色的 Outlook 功能，还可以在所有通信中使用这些功能，岂不是很棒？Skylook 让你做到这一点。
> 
> Skylook 将所有语音通话作为项目记录在一个特殊的 Outlook 文件夹中。每一项都包括诸如你给谁打电话、通话时间等细节。，但最棒的是，实际对话被录制为高质量的 MP3 文件，并作为附件包含在项目中。因为这是作为标准的 Outlook 项目保存的，就像电子邮件一样，所以您可以像处理电子邮件一样将其存档，以供以后参考。
> 
> Skylook 还将所有文本聊天记录为 Outlook 中的项目。每个项目都包括细节，如谁参加了聊天等。因为这是作为标准 Outlook 项目保存的，所以您可以将其存档以供以后参考。以后，你可以对它进行搜索、分类、存档，或者使用通常只适用于电子邮件的 Outlook 功能。
> 
> 为了让您不必经常在 Outlook 和 Skype 之间切换，Skylook 在 Outlook 工具栏中为每个当前在线的 Skype 联系人提供了一个在线状态按钮。此按钮显示联系人的状态(如在线、离开等。)带有一个方便的图标，还可以方便地访问联系人详细信息、最近的电子邮件和 Skype 对话录音。
> 
> 除上述功能外，Skylook 还提供了您所期望的 Skype Outlook 工具栏的所有功能。您可以使用 Skype 给 Outlook 联系人打电话，与 Outlook 联系人开始文字聊天，选择一封电子邮件并立即与其中一位收件人开始语音或文字聊天，选择一个约会并立即与其中一位收件人开始语音或文字聊天，等等。

Skylook 是我们的守护者。不仅仅是因为杰里米给了我一把免费的激活钥匙。:-)我会付钱的。
