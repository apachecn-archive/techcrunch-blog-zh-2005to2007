# CES: Cobra 将红光摄像头检测添加到队列中

首先我们有雷达探测器，这是警察不喜欢的。所以警察开始用激光追踪我们过分热心的超速行为。因此，雷达检测器中增加了激光检测器，为我们提供了雷达/激光检测器。所以警察开始用这些可怕的相机给我们的车牌拍照，因为我们快速通过黄色，所以他们可以用引文弄脏我们的邮箱。但我们终于能反击了！
