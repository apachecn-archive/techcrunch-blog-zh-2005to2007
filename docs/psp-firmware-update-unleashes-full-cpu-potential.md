# PSP 固件更新释放全部 CPU 潜力

![psp_3pt50_275.jpg](img/a4a0e45119307b5fc1b663ded21d2cbf.png)

我们很多人都认为 PSP 的固件升级到 3.5 版只允许远程访问 PS3，但也有一些猜测认为它也释放了 CPU 的全部潜力。谣言是有趣的，但铁的事实让每个人都感到温暖和模糊。

索尼现已正式确认固件升级，确实释放了处理器的全部潜力，达到 333 MHz。但你不会知道，因为目前的游戏只有 266 兆赫。升级确实允许开发者开始为更快的处理器速度编码，所以未来的游戏肯定会比现在更好看，更流畅。这种升级的负面影响可能是电池寿命的下降，但如果游戏不以最高速度运行，很难说这是真的。
