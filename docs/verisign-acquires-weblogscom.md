# 威瑞信收购 Weblogs.com

 Verisign 已经[证实](https://web.archive.org/web/20221015161258/http://www.verisign.com/press_releases/pr/page_035944.html)他们收购了[Weblogs.com](https://web.archive.org/web/20221015161258/http://www.weblogs.com/)ping 服务器网络及相关资产。也可以在这里查看 Verisign [的博客文章](https://web.archive.org/web/20221015161258/http://infrablog.verisignlabs.com/2005/10/weblogs_20_1.html)。

我们不会在 TechCrunch 上写太多，因为 Keith Teare 和我是 Weblogs.com 的交易顾问。我会说，我相信这是一个改变博客圈游戏规则的事件。

Dave Winer，Weblogs.com 的创始人和博客 ping 服务器的发明者[在这里讲述了这个故事](https://web.archive.org/web/20221015161258/http://archive.scripting.com/2005/10/07#whatADay)(并友好地提到了我们的参与)。

如果你想了解更多关于 ping 服务器的信息，我在今年七月写了一篇关于 Weblogs.com 的简介。

更多的报道，你还应该读读[杰森·科特基](https://web.archive.org/web/20221015161258/http://www.kottke.org/05/10/weblogscom-sold-to-verisign)、[斯泰西·克莱默](https://web.archive.org/web/20221015161258/http://www.paidcontent.org/pc/arch/2005_10_06.shtml#051712)、[罗伯特·斯考伯](https://web.archive.org/web/20221015161258/http://radio.weblogs.com/0001011/2005/10/06.html#a11389)和[迈克尔·巴兹利](https://web.archive.org/web/20221015161258/http://www.siliconbeat.com/entries/2005/10/06/confirmed_verisign_buys_weblogscom.html)。
