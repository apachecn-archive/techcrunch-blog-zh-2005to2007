# SlingPlayer 即将登陆 Symbian 手机，目前处于测试阶段

Sling Media 正在对基于 Symbian 的手机进行为期 4 周的测试。如果您有诺基亚 N75、N95 或 E65，您就有资格。

你将被要求对你收到的任何敏感信息保持沉默，你将不得不填写至少两份调查，并且你将被期望在此过程中通知 Sling 任何故障或错误。

显然，完成测试申请需要大约 10 分钟，仍然不能保证你会被杰出项目接受，但如果你觉得这一切都很好，请随时前往[beta.slingmedia.com](https://web.archive.org/web/20210126013520/https://beta.slingmedia.com/callout/default.html?callid=CEEB5F14208F48FD8FAE02450C6798D0)开始这个过程。
