# Panthius.com 演示销售力量模型

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/13/panthiuscom-demos-salesforce-model/>

爱尔兰软件初创公司[Panthius.com](https://web.archive.org/web/20210421165552/http://panthius.com/)推出了一个演示网站。托管电子商务套件提供了相当广泛的集成工具，以帮助人们经营他们的业务，这与 Salesforce.com ASP 模式提供的在线会计软件包没有什么不同。目前，该系统正在运行，并向用户展示演示公司，以便在日后全面推出之前对网站进行测试和试用。

在接受 TCUKI 的电子邮件采访时，Panthius 的首席建筑师 Robert O'Leary 说:“该模式与 Salesforce.com 相似，系统作为一种服务提供给订户，每月支付相关费用。然而，入门成本非常低，每月仅 49 欧元，我们提供全套工具，而不仅仅是 CRM。”

Panthius Plus 版本提供了诸如网络商店管理、CRM 工具、仓库管理、供应链管理、商业智能工具、完整的会计包等工具。

奥利里说，所有的工具都集成在该网站上，该网站还将在新的一年为第三方开发者提供定制服务和公共 API。(太好了，我们喜欢 TechCrunch 上的公共 API)。

该公司成立于 2004 年初，总部设在都柏林的贝莱德，除承包商外，有 6 名全职员工。这是私人支持的，尽管他们说风险投资正在“排队”

从技术角度来看，该系统是一个 J2EE 企业应用程序。整个平台基于开源工具，并且——根据 O'Leary 的说法——运行在带有 Postgres 后端的 JBoss 应用服务器上，所有这些都托管在 Linux 服务器上。
