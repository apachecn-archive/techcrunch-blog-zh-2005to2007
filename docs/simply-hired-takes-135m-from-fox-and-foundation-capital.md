# 《简单雇佣》从福克斯和基金会资本获得 1350 万美元

 [](https://web.archive.org/web/20210925215830/http://www.simplyhired.com/) [简单雇佣](https://web.archive.org/web/20210925215830/http://www.simplyhired.com/) ( [ TechCrunch 简介](https://web.archive.org/web/20210925215830/http://www.beta.techcrunch.com/2005/08/05/profile-simply-hired/))已经[从新公司的福克斯互动集团和基金会资本筹集](https://web.archive.org/web/20210925215830/http://today.reuters.com/news/articlenews.aspx?type=industryNews&storyid=2006-04-19T052037Z_01_N18401953_RTRIDST_0_INDUSTRY-MEDIA-NEWSCORP-SIMPLYHIRED-DC.XML)1350 万美元。两位投资者都将在该公司获得董事会席位。

这对刚被雇佣的人来说是个大新闻。福克斯的投资几乎可以肯定地表明，福克斯的其他资产，包括 MySpace，在不久的将来可能会与 Simply Hired 有非常密切的关系。而且——简单雇佣公司的竞争对手[的确](https://web.archive.org/web/20210925215830/http://www.indeed.com/)从《纽约时报》那里拿钱[。这些创业公司现在都有一些重量级人物支持他们，看看事情如何发展会很有趣。](https://web.archive.org/web/20210925215830/http://www.beta.techcrunch.com/2005/08/08/update-indeed-raised-money/)

关于这笔交易的另一个角度是没有发生的事情—**这不是福克斯**的直接收购，福克斯最近一直在进行收购。我的猜测是，收购谈判已经进行了一段时间，但对单纯雇佣公司的估值可能太高，不适合直接收购。

当然，所有这些都是推测。但是工作空间变得更加有趣了。
