# NeoNode:小、黑、不同

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/12/neonode-little-black-different/>

有机会看了 NeoNode，一个奇怪的小设备，本质上是一个专为媒体设计的 GSM 手机。它有一个光学触摸屏，在一个大规模改造的外壳下运行 Windows CE。它可以播放视频和音乐，并且非常小，大约 2 英寸乘 2 英寸。美国没有，但看起来它将在 5 月份登陆欧洲。

我和公司的 CEO 聊过，印象颇深。这只是一个音乐手机，看起来它做了它应该做的事情。

前面唯一的东西就是那个小操纵杆。该设备的其余部分都是基于触摸屏和手势的。不知道是怎么回事。
