# PayPerPost 记录了他们的非道德性

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/12/03/payperpost-chronicles-their-amorality>

总部位于佛罗里达州的 PayPerPost 刚刚推出了一个名为 [RockStartup](https://web.archive.org/web/20150911191618/http://www.rockstartup.com/) 的新网站，该网站记录了他们受到道德挑战的初创公司[的诞生(我希望是最终的失败)](https://web.archive.org/web/20150911191618/http://www.techcrunch.com/tag/payperpost)。

RockStartup 似乎受到了电影[Startup.com](https://web.archive.org/web/20150911191618/http://imdb.com/title/tt0256408/)的严重影响，这部电影通过一家 web 1.0 初创公司追踪了卡莱勒·图兹曼和他的团队(以及多个女朋友)。《摇滚创业》前两集中的一些场景几乎是《Startup.com》中场景的翻版。当然，RockStartup 是实时消费的，而 Startup.com 直到公司倒闭后才发布。

我等不及 Ted(创始人)坐牢的那集了。:-)

我不会再讨论整个付费后的争论了。在 YouTube 上看到一个裸男在床上诋毁我之后。如果你想知道我对 PPP 的看法，可以看看我们的[往期帖子](https://web.archive.org/web/20150911191618/http://www.techcrunch.com/tag/payperpost)，听听我们在[他们融资](https://web.archive.org/web/20150911191618/http://www.talkcrunch.com/2006/10/01/episode-13-payperpost-raises-3-million/)后对创始人的采访。
