# 雅虎！Go 2.0 演练

[![](img/e4a086fe9fe22667f147fffb11a2a220.png)](https://web.archive.org/web/20210228230428/https://beta.techcrunch.com/wp-content/uploads/2007/05/img_7010.jpg)

我有机会玩雅虎 Go 的 Symbian 版本，这是雅虎的独立网络应用程序，提供新闻、天气、交通和雅虎！从手机搜索。它的功能相当全面，甚至支持雅虎邮件。

我被吹走了吗？没有，但对于一个 1MB 的手机应用程序来说，它肯定有很多优势。在某种程度上，它非常类似于 Helio 的 HOT service，并且指出了手机应用程序的发展方向。制造商并没有提供一个空白屏幕——可以说是拨号音——而是试图抓住开机时弹出的初始手机屏幕。虽然这一领域曾经神圣不可侵犯，但现在被认为是成熟的货币化死角。

[![](img/acded6ebfa0eba201419f618ff8e3188.png)](https://web.archive.org/web/20210228230428/https://beta.techcrunch.com/wp-content/uploads/2007/05/img_7011.jpg) 
我们才刚刚开始……

这里真正吸引人的是搜索和地图。尽管大多数地图网站都在手机中被屠杀，但像这样的独立应用程序允许对渲染进行一些相对控制。我带来的纽约地图非常有用，而且相当快。尽管如此，其余的功能都是肉汁。没有太多的实质内容，但有一小会儿的乐趣。
