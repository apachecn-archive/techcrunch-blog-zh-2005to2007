# 射杀信使(如果消息是“醒醒吧，混蛋”)；或者，格洛克时钟| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/22/shooting-the-messenger-if-the-message-is-wake-up-jerk-or-the-glock-clock/>

几个月来，我们已经在 CrunchGear 的页面上看到了太多有趣的[闹钟](https://web.archive.org/web/20150531011006/http://crunchgear.com/?s=clock)，事实上还有很多我们没有涵盖的。太多了。但有时我们会遇到用完全独特的方式让我们醒来的时钟，我们会注意到它。如果它们够酷，我们会在这里与你分享。

一种这样的钟是枪控闹钟。它有双重用途:通过强迫你用 Ninendo Zapper 风格的枪瞄准球体使其沉默，它强迫你醒来并清理蜘蛛网。它还通过让你打破烦人的唧唧喳喳的时钟给你一点点满足感。作为一个额外的奖励，杀死某些东西的兴奋感会帮助你振作起来。

读者们，你们可以以后再感谢我们。我们想要一个。
