# 表演陷入混乱——股东公开争吵

 [](https://web.archive.org/web/20221003212750/http://performancing.com/) 昨天晚上我们写了关于三个服务中的第二个[的关闭，以及首席执行官 Nick Wilson 最近的](https://web.archive.org/web/20221003212750/http://www.beta.techcrunch.com/2007/01/22/performancing-heading-to-deadpool/)[离职](https://web.archive.org/web/20221003212750/http://performancing.com/node/5609)。

显然，威尔逊，谁说他仍然拥有 35%的业务，是不高兴的关闭表演的广告网络。事实上，他似乎并不知道发生了什么。在他的个人博客上，他写道:

> 我不知道绩效管理之间的沟通问题是什么，但似乎有一些决策没有掌握所有的事实。
> 
> 当我从公司辞职并将领导权移交给 Chris 时，情况需要做很多工作，但还不错——我们有几个前进的选择，包括与一位著名的博客玩家就收购 PFF 或现在众所周知的 ScribeFire 进行会谈。
> 
> 我已经给 Chris 和 Patrick 发了电子邮件，虽然当时我不知道 Patrick 是否还会扮演积极的角色，因为他和 Bill 在收购 MediaWhizz 后需要放弃他们今年在表演方面的股份。
> 
> 不要指望表演了。

听起来表演公司需要召开股东大会来解决这个问题。很快。
