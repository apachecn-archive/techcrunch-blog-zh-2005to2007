# 点播赶上 DVD 发行

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/18/on-demand-catches-up-to-dvd-releases/>

康卡斯特已经开始了一项计划，在电影发行 DVD 的同一天提供点播服务。这是一个增加付费服务销售的尝试。这项实验目前正在匹兹堡和丹佛进行。它允许观众以 4 美元一部的价格观看电影。

这种服务有点容易，但我很高兴看到康卡斯特开始这样做。考虑到有多少人已经有了这样的设备来看电影，这是非常合理的。

如果更多的供应商加入进来，它可能会切实改变我们观看电影的方式。然而，更有可能的是，他们会懈怠，像微软这样的公司会通过迫使人们购买额外的设备(见: [Xbox 360](https://web.archive.org/web/20141101154932/http://crunchgear.com/2006/11/22/xbox-marketplace-video-rentals-go-live/) )来获得他们本可以按需获得的相同服务，从而在该领域获得主导地位。
