# Sound ID SM100 蓝牙耳机评论| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/28/sound-id-sm100-bluetooth-headset-review/>

周围有这么多蓝牙耳机，我可以看到哪里很容易忽略声音 ID SM100。这家公司很可能对你来说并不熟悉——不像随处可见的诺基亚、缤特力、摩托罗拉和索尼爱立信——你的家人和朋友可能也没有听说过它们。见鬼，我几个月前才知道他们。此外，多年来测试了许多耳机，我已经学会不要对任何制造商声称的回声降低和噪音消除技术过于乐观。

所以，当 129 美元的 SM100 实际上是一个很棒的小耳机，不仅名副其实，而且有一些我在其他设备上没有见过的功能时，这是一个惊喜。

![sid_controls.JPG](img/ea1bf86a15a37d86409c96c45677d076.png)
**特点**
这是一款令人难以置信的轻巧、超紧凑的蓝牙耳机。它的重量不到 0.4 盎司，感觉里面有点空(因为它很可能是)，有些人可能会发现塑料看起来比市场上的其他耳机稍差。但是，它通过了“你会穿这个吗？”对我问过的每个人进行测试——男人或女人。

SM100 有一些独特的功能，这无疑增加了它的吸引力。它有一个环境模式，允许音频从外面通过并放大，所以你可以与人交谈，而不需要拿出耳机。当你接听电话的时候，扩音系统就关闭了，你所听到的就是你的电话。这也可以使它成为一个临时的助听器。而且如果你不想用这种模式(确实有点影响续航)，可以关机。

集成的噪音导航系统可以确定传入声音的响度、频率、音高和节奏，因此 SM100 可以抵消任何风或背景噪音，同时改善语音和声音质量。此外还有 PersonalSound 技术，只需按下一个按钮，你就可以调整耳机以提高清晰度，而无需提高音量。

最后，有一个 One2One 模式，让您将两个 SM100 配对在一起，以便您可以在它们之间通话。这是我唯一质疑其有用性的功能，因为你说的是离其他用户最多 30 英尺远。但是，在嘈杂的区域，大喊大叫并不理想，或者作为一个临时的助听器，这样你就不必对着重听人大喊大叫，这可能会很方便。此外，这只是一个额外的，所以没有伤害没有犯规，如果你从来没有使用它。

![sid_inside.JPG](img/f6f37ad059d7b6959125460e255673ab.png)sm 100 没有耳挂，这使得从左耳切换到右耳非常简单，只需将耳机转到 R 或 l 即可。

![sid_inear.JPG](img/f1c249e6057945a73349e0e3193e159e.png)如你所见，它很小。不过，没必要为了让别人听见而大喊大叫；双，全方位硅麦克风做了一个神奇的工作，拿起你的声音在正常的讲话水平。如果你打开了环境模式，你可以听到你的声音有多好。

![sid_eartips.JPG](img/d24d1c333fc068ffb9624b2aac05f0ef.png)与我测试过的大多数无夹子耳机不同，SM100 不依赖不同尺寸的耳塞来将设备固定在耳朵里。相反，耳塞的尺寸都是一样的，但“RealComfort 耳挂”的尺寸会有所不同。这个环通过在你耳朵后面的软骨下滑动来锁定耳机。很舒服，也很管用。

**性能**
![pairing.gif](img/d705347e7ac08cd4a6eb77372592e518.png)配对过程与其他蓝牙耳机相同。按住主按钮(设备外部的标志)，等待背面的活动灯开始闪烁，然后等待您的手机或其他设备识别它。主按钮还可以接听电话，打开和关闭耳机。所有其他功能由其余三个按钮控制。

至于性能，虽然，我并没有期待太多。我测试过的大多数耳机要么让你听起来像是在用扬声器说话，要么把你的声音数字化到让人分心的程度。当我走在纽约繁忙的街道上时，SM100 让我这边的声音听起来响亮而清晰。我打过电话的每个人都说他们不知道我在用耳机说话，我收到的其他评论是“清晰”、“声音自然”、“正常”和“好”

电池寿命也很好，可以持续几天的通话和待机状态。(该公司规定手机模式下的超时时间为 8 小时，待机模式下的超时时间为 72 小时。)另外，它在大约 3 小时内充满电。总的来说，SM100 打破了我的预期，当然值得被列为当今最好的蓝牙耳机之一。
