# 威瑞森的新三星弯曲，旋转，摇摆

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/14/verizons-new-samsung-bends-twirls-rocks/>

三星 SCH-u740 现已在威瑞森无线公司上市。对于那些无法在小翻盖外形和 QWERTY-txting 便利性之间做出决定的人来说，这是一款非常棒的手机，因为它可以翻转和转换。此外，它支持 V-Cast，带有外部音乐控制。

真正棒的是，这款功能齐全的手机仅售 149 美元(捆绑两年)，考虑到它的所有功能，包括前面提到的短信、130 万像素的闪光灯和 microSD，这是一个非常棒的价格。这款手机还支持 VZ 导航服务，并配备了索姆松新的无线网络浏览器。

我们甚至喜欢外部显示屏上的复古模拟时钟。
