# OS X 豹| TechCrunch 需要改进的十大事项

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/06/top-ten-things-to-improve-in-os-x-leopard/>

虽然这些大多相当深奥，图奥还是有一些好主意，让 OS X 比一般的熊更好。例如:

> 移除应用程序及其所有包袱:在 Mac OS X 上删除(或“卸载”)一个应用程序很简单:你只需将它移到垃圾桶。但是，当你使用应用程序时，它们产生的所有额外数据(数据库、媒体库等)怎么办呢？)?

因为苹果付钱让我们爱他们——实际上他们不爱，但那不是很好吗——我们认为这个列表是一组“想要”而不是需要，除了空闲磁盘移除和卸载。卸载是一个一键操作的过程，但是弹出的小窗口提醒你已经销毁了所有数据，可能会使你不育，这有点过分。只要给我们一个优雅的卸载程序，不要激动。
