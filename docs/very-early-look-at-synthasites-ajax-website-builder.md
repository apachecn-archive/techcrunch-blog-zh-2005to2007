# 很早就看了 Synthasite 的 Ajax 网站生成器

总部位于南非的 T4 将很快推出一个名为 Synthasite 的网站，帮助人们完全在线建立网页。它将包括许多桌面应用程序的功能，如 Frontpage 和 Dreamweaver(想想 Writely 或 Zoho Writer v. Word)。

像 [Sitekreator](https://web.archive.org/web/20211127062251/http://www.sitekreator.com/) 和 [Google Page Creator](https://web.archive.org/web/20211127062251/http://www.beta.techcrunch.com/2006/02/22/google-pages-released/) 一样，Synthasite 将是一个完全在线的应用程序。这项服务还处于私人测试阶段，要到今年秋天才会推出。然而，我们有机会测试它。

合成石将会非常受欢迎。像 Sitekreator 一样，它有各种各样的模板可供选择，该团队也有一些关于如何帮助网站创建者与第三方 web 服务集成的想法。它比 Google Page Creator 有更多的功能。然而，测试版产品目前只能在 Internet Explorer 中运行(Firefox 将很快发布)，我们遇到了网站反复崩溃和关闭的很多麻烦。当我们坚持的时候，结果是好的，但是，我们假设在网站完全或者至少大部分稳定之前不会公开发布。

一旦它在 Firefox(以及 Mac)上可用，我们将进行详细的审查，并与 Sitekreator 进行比较。
