# Nowsy:新 Ajax 主页

总部设在荷兰的诺西有些独特之处。这是一个 Ajax 主页，拥有许多获得资助的初创公司的功能，如 [Netvibes](https://web.archive.org/web/20210618112610/http://www.beta.techcrunch.com/tag/netvibes) 和 [Pageflakes](https://web.archive.org/web/20210618112610/http://www.beta.techcrunch.com/tag/pageflakes) (以及 [Google IG](https://web.archive.org/web/20210618112610/http://www.beta.techcrunch.com/2005/12/14/googles-ajax-desktop-is-now-extensible/) 和大约[十个其他](https://web.archive.org/web/20210618112610/http://www.beta.techcrunch.com/2005/12/18/yep-one-more-ajax-desktop-pageflakes/)未获得资助的网站)。

Nowsy 没有 Netvibes 或 Pageflakes 的所有功能，但它拥有大多数基本功能——基于 RSS 提要的简单模块添加、拖放和删除功能以根据您的喜好定制页面，以及简单的新页面创建。

Nowsy 有一点与其他服务不同，它很聪明。他们在网站顶部添加了一个搜索框，当您输入搜索词时，它会查询页面上所有模块中的所有服务(我假设他们是通过在自己的服务器上索引提要以供用户稍后搜索来做到这一点的)。如果你已经在 Nowsy 中添加了你喜欢的网站，搜索功能会非常有用。
