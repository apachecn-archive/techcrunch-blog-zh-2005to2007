# 雅虎阿尔法搜索推出混乱的测试版

[![alphalogo.png](img/0b690e990d274b58f01ec4b82b47ae58.png)](https://web.archive.org/web/20210622153301/http://au.alpha.yahoo.com/) 雅虎澳洲今日推出名为[阿尔法](https://web.archive.org/web/20210622153301/http://au.alpha.yahoo.com/)的全新个性化搜索引擎。雅虎还有另一个个性化搜索产品叫做[雅虎搜索构建器](https://web.archive.org/web/20210622153301/http://www.beta.techcrunch.com/2006/08/07/yahoo-launches-personalized-search/)，意在为你的网站定制一个雅虎搜索引擎。很多人都称 Alpha 是 Google Coop 的竞争对手，但事实并非如此。Yahoo Alpha 是一个元搜索引擎，具有 Netvibes 的外观和感觉，让你在一个框中输入搜索，然后在几个搜索引擎中查看结果。 [Google Coop](https://web.archive.org/web/20210622153301/http://www.beta.techcrunch.com/2006/10/23/google-custom-search-tomorrow/) 和 [Rollyo](https://web.archive.org/web/20210622153301/http://www.beta.techcrunch.com/2006/08/03/rollyo-adds-a-ton-of-features/) 交错排列来自多个按相关性排序的来源的结果。

搜索结果页面是搜索结果小部件的两列布局，左列包含主要搜索结果，右列包含较小结果提要的列表。默认设置是雅虎网站搜索在左边，较小的 Flickr、YouTube、雅虎新闻、维基百科和雅虎搜索营销小工具在右边。当您在主框中输入搜索时，每个源的结果都会显示在小部件中。

您可以通过对页面上的小部件重新排序和添加新的搜索提要小部件来自定义结果页面。添加 feed 的过程很混乱，需要输入一个深奥的 Opensearch url(例如 http://blogsearch.google.com/blogsearch_feeds?hl=en q = { search terms }&ie = utf-8&num = 10&output = RSS)和验证码答案。Opensearch 是 Amazon A9 的一项技术，允许第三方通过 RSS 访问您的搜索结果。每个部件都可以是公开的或私有的，尽管我们还没有看到这些部件的市场。

看到雅虎推出这样的产品令人惊讶——[Rollyo](https://web.archive.org/web/20210622153301/http://www.beta.techcrunch.com/2005/09/28/rollyo-search-launches-today/)用雅虎自己的 API 做得更好。
