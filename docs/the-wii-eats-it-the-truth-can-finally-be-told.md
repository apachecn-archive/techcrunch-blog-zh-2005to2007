# Wii 吃了它:真相终于可以大白了

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/23/the-wii-eats-it-the-truth-can-finally-be-told/>

![spm.jpg](img/8637fa3a0d74d4ca03d7ebe45918e3cd.png)
我在尝试[喜欢 Wii](https://web.archive.org/web/20140111025602/http://crunchgear.com/2007/01/26/one-mans-wii-pinion/) 。我用尽全力去爱*超级纸马里奥*。但是这张图片基本上总结了我对游戏、平台和其他可用游戏的看法，包括*塞尔达*。拜托，任天堂。求你了。别说了。把我当成有脑子的人。我已经厌倦了“异想天开”被等同于“神话”和“经典角色”，成为拙劣游戏和愚蠢故事的借口。

任天堂专注于婴儿。直截了当。如果他们能让控制器穿过子宫颈，他们会的。他们的游戏又长又复杂，而且毫无必要地令人费解，几乎没有增加故事情节或满足感。以塞尔达原版*传说为例。你的任务是杀死加农。很公平。但是你要怎么做才能到达那里呢？烧掉一棵树又一棵树。推倒一个又一个墓碑。看起来不像会被炸的炸弹墙。这一切都“在那里”，显而易见，但游戏被不必要的和没有挑战性的复杂性弄得一团糟。这对 NES 来说没问题，但近 20 年后就不好了。*

快进到*超级纸马里奥*。这个故事像《半衰期》一样线性，但却不那么引人注目。去这里，做这个。去那里，做那件事，回到这里，看一些巫师谈论四十分钟，去那里。再做一次。除了最不可救药的玩家——我认为自己是个不可救药的玩家——这些谜题对所有人来说都是显而易见的，而包括收集卡片在内的支线任务就像看着油漆变干一样令人兴奋。但是约翰！我喜欢那些东西！你会尖叫。好吧，但是我很生气。

上周末我和一个朋友玩了几十个小时的《光晕 3》，玩得很开心。这不是火箭科学，这个光环专营权。一些很酷的武器，一些汽车，一些下一代图形。我没有时间收集暮光之城公主世界里所有的 500 只蚂蚁。相信我。我有抵押贷款。给我一个让我忘记自己在玩游戏的游戏。给我一些有生命的东西，而不是一些糟糕的石头怪物和烦人的孩子指定的罐头。我想通过自己的行动推进剧情，而不是通过非玩家角色的行动。给我一些好的图形和有趣的谜题。当然，这并不总是一个伟大的游戏——*猎物*，有人吗？—但至少你已经有了基本的部分。想做聚会游戏，就做聚会游戏。别在我对马里奥和塞尔达的美好回忆上撒尿了，做个不烂的游戏吧。
