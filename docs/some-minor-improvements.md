# 一些小的改进

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/15/some-minor-improvements/>

只是简单提一下我添加到页面上的一些东西。首先，我决定显示标签。这些标签本质上是微小的提要，所以如果你喜欢 iPod 的东西【http://crunchgear.com/category/gear/ipod/feed】订阅(注意 URL 末尾的“提要”)。你也可以和[的作者](https://web.archive.org/web/20141017044244/http://crunchgear.com/author/johnbiggs/feed)一起这么做。

接下来，在边上有一个小小的最近评论框。这将让我们所有人都了解信息，但也可能会受到垃圾邮件的攻击。我们正在尽最大努力清除垃圾邮件，Akismet 运行得非常好，但是 feh 和 fie 和 foe——它仍然存在。无论如何，它可以让你了解一些你最喜欢的帖子的最新评论。够好了。

最后，我们的增长加上这些特性，以及我们的服务器显然是 64MB 内存的 286 这一事实，就像《Punch-Out》中的[河马王一样击倒了我们。我们伟大的读者 Andrew 指出了这一点](https://web.archive.org/web/20141017044244/http://en.wikipedia.org/wiki/List_of_Punch-Out!!_boxers)[所以我们不会再有 IE 爆炸的问题，但是现在请容忍我们的废话。](https://web.archive.org/web/20141017044244/http://support.microsoft.com/default.aspx/kb/927917)
