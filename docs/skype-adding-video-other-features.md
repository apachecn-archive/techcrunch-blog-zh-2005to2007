# Skype 添加视频和其他功能

上周，Skype 在波士顿的 VON 发布了部分产品路线图，截屏在 [SkypeJournal](https://web.archive.org/web/20201128105952/http://www.skypejournal.com/blog/archives/2005/09/skypes_product.php) 发布。

海，马克蓝中国布拉！

即将到来 1.5 版本计划在 10 月份发布，将增加视频和客户端网络呈现功能。11 月推出的 1.6 版本简化了客户端，增加了社交网络等功能。

后续版本包括博客/网站管理员工具、PTT、用户奖励计划、离线即时消息、共享群组和视频邮件。
