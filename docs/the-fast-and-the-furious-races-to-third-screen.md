# 速度与激情比赛进入第三幕

*速度与激情*中的第四个游戏刚刚登陆移动平台。自 2004 年推出以来，这个系列相当成功，获得了 700 万次付费下载。通过与环球移动娱乐和 I-Play 的协议，最新的游戏将于本周推出。

*速度与激情:逃犯*基于两部电影之间的时间，并着眼于两部电影之间的全新篇章。这款游戏是为 BREW 平台开发的，于本周在 BREW 2007 大会上发布。

“BREW 的顾客在商店里可以享受到*速度与激情:逃犯 3D* 。I-play 首席执行官 David Gosen 表示:“这款游戏由我们的内部团队开发，他们不断提高创新标准，创造了视觉上令人惊叹的 3D 游戏。“一些增加游戏真实感的视觉效果包括后窗的云和建筑物的反射，以及直接从电影中提取的赛前摄影。”

这个游戏现在可以从 [I-Play](https://web.archive.org/web/20130628174846/http://www.iplay.com/) 开始玩了。
