# Zune 与 iPod:磁带的故事

当有争吵时，我们会变得疯狂。布雷克在 Crunch Gear 已经在他的地下室经营斗鸡联盟三年了，就是为了这场冲突。这当然能赚很多钱，但是没有什么比让两个主要的竞争对手面对面，一对一，让他们一决雌雄更好的了。

这就是我的技术列表中的迈克尔对 Zune 和 iPod 所做的。他把它们的相似之处和不同之处放在戒指上，让他们去城里，比较 iPod 没有调频调谐器和 Zune 较短的电池寿命，iPod 的 Clickwheel 和 Zune 的大小。这是相当全面的，但鉴于 Zune 甚至不是一个你可以购买的产品，我们很可能会在圣诞节前看到一场重赛。

最终的赢家是 iPod，尽管 Zune 确实有着相当不错的表现，并且可以昂首挺胸。没有消息说 iPod 接下来会和桑德曼或迈克·泰森竞争，但我们打赌 iPod 会保持冠军地位，至少到明年初。
