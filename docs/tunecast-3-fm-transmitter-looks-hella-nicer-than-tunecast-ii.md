# TuneCast 3 FM 发射机看起来比 TuneCast 2 更好的海拉

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/09/tunecast-3-fm-transmitter-looks-hella-nicer-than-tunecast-ii/>

[![tunecast3.jpg](img/d6d183e9cdcb2a053dc3c5ecea745a37.png)](https://web.archive.org/web/20180204184614/http://old.crunchgear.com/wp-content/uploads/tunecast3.jpg "tunecast3.jpg")

Belkin 继续推进其用于 DAPs 的调频发射机 TuneCast 系列。TuneCast 3 与 [TuneCast II](https://web.archive.org/web/20180204184614/http://catalog.belkin.com/IWCatProductPage.process?Product_Id=166893) 有许多相同的功能:向任何未使用的 FM 电台广播、自动省电、兼容所有 iPods 和其他 DAP 等。显然它们看起来不同，我不得不说 TuneCast 3 看起来更好，尤其是显示器。可读性更强，在光线较暗的情况下更是如此。

根据我使用这些调频发射机的经验，如果你想让它像宣传的那样工作，你真的需要在一个偏远的地方。试着在像纽约和洛杉矶这样无线电密集的地方使用这些设备，你将会忍受比你想的更多的静电干扰。
