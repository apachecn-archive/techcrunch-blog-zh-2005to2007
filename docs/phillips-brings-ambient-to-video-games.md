# 菲利普斯将环境带入视频游戏

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/13/phillips-brings-ambient-to-video-games/>

没有什么比玩电子游戏和没有周边刺激更糟糕的了。当然，你有大电视，你在玩光晕，你有幽灵，但是你的 42 英寸平板电视周围的一切都让你心烦意乱。没有补救办法。

但是飞利浦有一个想法。它的 amBX 系统将“环境背光平板电视”提升了一个档次。这些分布在你的“游戏环境”中的灯光，是为了配合你的视频游戏的环境照明，所以你是“在”游戏中。此外，他们有风扇，模拟当你的士兵跑过田野时，空气从他的头上掠过。

真的。

我们不确定这将于何时首次亮相，也不知道它的价格，但它是关于本·阿弗莱克家的一个毫无用处的电影剧本，如果它像他们的电视一样受欢迎，那就找我们冲出去买这个吧。
