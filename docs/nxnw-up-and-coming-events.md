# NxNW 上升和未来的事件。

几周来，我一直在哀叹大理石拱门以北缺乏创业活动。当然，如果我仔细看的话，我会发现优秀的伯明翰的万用包和曼彻斯特的 T2 极客开发团队。这两个小组都已经举办了一段时间的活动，并且都将很快举办活动——请查看下面的详细信息。

我很高兴地宣布，事情肯定开始在“北方”发生，因为这个月有两个新的事件启动。第一个是明晚的“**第一个星期五**”，由 FreeServe fame 的 Ajaz Ahmed 组织。

另一项活动名为“**西北创业 2.0** ”，由 Manoj Ranaweera([ebdex](https://web.archive.org/web/20130627213325/http://www.ebdex.co.uk/)的首席执行官)于 11 月 23 日在曼彻斯特举办。我必须声明，Manoj、Ivan Pope(Snipperoo[的首席执行官)和我已经就组织一次企业家活动讨论了几个星期，我很高兴地宣布现在已经就绪。](https://web.archive.org/web/20130627213325/http://www.snipperoo.com/)

该活动将免费参加，但我们要求您提前注册活动[在这里](https://web.archive.org/web/20130627213325/http://upcoming.org/event/120380/)在 Upcoming.org，其他发言人包括[Imran Ali](https://web.archive.org/web/20130627213325/http://imran.typepad.com/)Orange 技术研究团队(总部位于利兹)主管的副手。晚上的计划是做一个非常简短的主题演讲，然后是与小组成员的自由讨论。然后当这一切结束后，我们将退休到酒吧社交和网络。
