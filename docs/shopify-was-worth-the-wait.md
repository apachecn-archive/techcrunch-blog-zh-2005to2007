# Shopify 值得等待

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/04/15/shopify-was-worth-the-wait/>

总部位于渥太华的 Shopify(T4)比最初承诺的(去年秋天的)T6(T5)晚了很多，但是等待是值得的。他们推出了一个私人测试版，并计划在五月永久开放。

Shopify 是一个集成支付的托管电子商务解决方案。在网上卖东西或者想卖东西的人会喜欢它。您可以立即建立一个商店，添加要销售的商品，上传图片，添加标签和分组商品，并集成 paypal 或信用卡处理器进行支付。报告非常出色，他们甚至想到要包含一个最近订单的 RSS 提要来跟踪销售情况。

参见[此处](https://web.archive.org/web/20230405042734/https://checkout.shopify.com/services/signup)了解更多功能详情。一般来说，卖家只需几分钟就可以创建一个非常专业的多页面网站，只需要很少或不需要编程技巧。

商店有许多预制的主题，主题可以通过直接操作 CSS 来修改。Shopify 还发布了一款名为 [Vision](https://web.archive.org/web/20230405042734/http://vision.shopify.com/) 的产品，这是一款可供下载的客户端应用程序，供设计师构建和修改自己的 shopify 商店主题。

创建商店是免费的。Shopify 从所有销售额中抽取 3.75%的佣金。

所以 Shopify 会帮你以一个合理的价格建立一个非常好的商店。但是他们应该尽快开发工具来帮助人们推广这些商店(例如，Vendio 有一些基本的推广工具，他们也帮助 ebay 列表)。如果 shopify 也能帮助店主进行营销，它将成为一项更有用的服务。

想了解更多关于 shopify 的信息，请看他们的博客和开发主管鸢·卢特克的采访。如果有人知道好的样品店，请告诉我。
