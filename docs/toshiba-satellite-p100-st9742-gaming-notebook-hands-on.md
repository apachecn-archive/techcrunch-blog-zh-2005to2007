# 东芝卫星 P100-ST9742 游戏笔记本动手操作 TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/25/toshiba-satellite-p100-st9742-gaming-notebook-hands-on/>

![](img/7dd4d4f6b56aa1385f16ee6c61504ae7.png)

上次我去看东芝笔记本电脑时，我看到了 P105-S9722。它有一些很棒的功能，但也有一些真正令人讨厌的缺陷，妨碍了日常使用。这一次，我希望 P100-ST9242 (P100)卫星能解决这个问题。是吗？性能提高了吗？酷睿 2 双核处理器怎么样？别冲动，冠军。你必须继续阅读，自己去寻找答案。
 打开笔记本电脑的包装，我立刻被这台机器的一个突出特点所吸引:它的喷漆工作。显示器的背面具有令人惊叹的金属青铜色。它很突出，给它所在的环境增添了一些“活力”。然而，P100 不仅仅只有好看的外表。它还配备了高端功能，以证明其 2900 美元的价格是合理的。

在这个 7.1 磅重的庞然大物中，您会发现一个 T7200 英特尔酷睿 2 双核 CPU、一个带 TruBrite 的 17 英寸宽屏显示器、NVIDIA GeForce Go 7900 GTX 512 MB gddr 3 SDRAM 显卡、2GB PC 5300 DDR2 内存、一个 100 GB 7200 RPM SATA 硬盘、集成无线、5 合 1 读卡器、Harman/Kardon 扬声器、大量 USB 端口、一个支持 11 种格式的 DVD 刻录机、Windows XP Professional(抱歉，没有等等，去掉最后一个特征。不用说，这东西是为高性能计算打造的。

在东芝的 P105 上，我们看到了某些区域的问题。键盘有一个设计糟糕的换档键，黯淡无光的扬声器，大量预装的垃圾软件，以及故障无线。不幸的是，我必须报告，P100 仍然存在许多这样的问题。扬声器实际上听起来好多了，但垃圾仍然存在，仍然令人难以置信地讨厌。

![](img/31bbd9528e12716e9c5926c6a1b13d1e.png)

最让我恼火的是，东芝必须在它的大多数笔记本电脑中使用相同的无线网卡/驱动程序。使用 P105，每次我连接到 NETGEAR RangeMax 240 路由器时，我的互联网连接都会完全中断。同样的事情发生在 P100 身上。没有其他笔记本电脑可以做到这一点，我的 iMac G5 工作正常，我朋友的戴尔和索尼 Vaio 工作正常，为什么你的东芝不行？我建议对此做进一步的调查，因为我感觉我不是唯一一个有这个问题的人。

但是抛开所有的烦恼不谈，P100 仍然是一台不错的机器。您可以多任务处理多个应用程序，如入无人之境。让《魔兽世界》,一个播放着 YouTube 视频的浏览器，iTunes 和 MS Word 同时打开。它不会慢下来，不会有任何东西崩溃。这款笔记本电脑对于基于 Windows 的机器来说非常稳定，我觉得东芝承诺使用高质量的部件(无线部分除外)是导致这种情况发生的原因。这不像是黑色星期五你在沃尔玛买的一台蹩脚的笔记本电脑，这像是一台推土机，可以处理任何扔向它的东西。

那些欢迎预装软件的人可能会对现有的选择非常满意。大量的杀毒软件、一些游戏、流行软件的试用版以及其他一些软件对那些想要这些东西的人来说是很好的。拥有这款笔记本电脑的普通消费者不会介意 P100 附带这些应用程序，但像我这样的超级用户会把这个软件视为一堆我们不想要的垃圾。也许东芝可以开始提供一个没有安装操作系统的干净笔记本电脑的选择？这肯定会大大有助于解决这个问题。

如果你喜欢电影或观看大量视频，P100 一定会让你欣喜若狂。这款 17 英寸宽屏显示器绝对令人惊艳——是我见过的笔记本电脑中最好的之一。也就是说，TruBrite 技术确实让它在动作或恐怖电影中看起来很棒。此外，还有大量的房地产可供选择。我不介意带着这个东西在沙发上看 3 小时的电影，尽管它 7.1 磅有点重。

总的来说，如果你在家里不使用 NETGEAR 产品，并且想在上面安装一个新的 Windows Vista 版本，我认为这款笔记本电脑是值得购买的。P100 拥有的强大功能弥补了它的缺陷，为您的应用程序和游戏提供了无缝的工作流程。还应该提到的是，这是一个在线交易。你不会在商店里找到这个坏男孩，使它比其他笔记本电脑更个性化。如果你有多余的钱买一台昂贵的笔记本电脑，你会喜欢它的喷漆和 17 英寸的显示屏，以及强大的媒体功能。对 P105 的改进？当然，但是东芝真的需要尽快解决他们的无线问题。
