# 特别提款权很棒。如果你不相信我，去问 MobileCrunch

我们的姐妹网站 MobileCrunch 独家报道了来自 Sandbridge Technologies 的一款新型灵活基带无线电。软件定义无线电(SDR)将是移动手机中无线电的可行替代品，允许您通过更改无线电软件在不同标准之间切换。或者至少我的小脑袋是这样处理信息的。虽然真正的生产就绪设备被认为还很遥远，所以这可能是一个突破，最终使我们所有人受益。奥利弗在 MobileCrunch 上对它的分解促使我学习更多，你也应该这样做——那就给我解释一下。
