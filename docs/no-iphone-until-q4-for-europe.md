# iPhone 要到第四季度才会在欧洲上市

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/26/no-iphone-until-q4-for-europe/>

如果你在欧洲，你可以不排队买 iPhone。取而代之的是排队等候卢浮宫(并不是说那里的队伍有时移动得很快)。据我们的姐妹网站 CrunchGear 报道，[iPhone 要到今年第四季度才会发布](https://web.archive.org/web/20160305014553/http://crunchgear.com/2007/07/26/iphone-heading-to-europe-in-q4-not-q3/)。不用担心，2007 年第三季度(应该是现在)正在飞逝。

我们听说史蒂夫·乔布斯认为苹果可以在第四季度末卖出一百万部 iPhones。当然，公司需要实际发布它们来销售。我们将继续追踪这个故事。
