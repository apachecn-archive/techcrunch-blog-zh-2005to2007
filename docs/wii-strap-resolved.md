# Wii 表带已解决

本周早些时候，如果你觉得没有必要使用 [Wii 表带修复](https://web.archive.org/web/20130627212241/http://crunchgear.com/2006/12/05/wii-strap-fix/)，你可能会没事。来自澳大利亚的报道称，有问题的皮带已经被更新为新的、更厚的皮带。这大概可以解决问题，尽管为了安全起见，使用一些鱼线也无妨。只是一个想法。
