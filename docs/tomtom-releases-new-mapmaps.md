# TomTom 发布新地图

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/21/tomtom-releases-new-mapmaps/>

万一你打算在这个假期带着你的 TomTom 上路，帮自己一个忙，为导航公司今天放弃的美国和加拿大客户获取更新的地图。

新地图包括 410，000 英里的新道路，超过 600 万个新的家庭和商业地址，500，000 个新的兴趣点，以及改善的道路细节以获得更好的视觉表现。

TomTom 为 TomTom GO x10 和 TomTom ONE 用户以及其他通过[验证](https://web.archive.org/web/20150518051513/http://www.tomtom.com/products/maps)的合格用户提供免费地图。另外，在 2007 年 3 月 31 日之前，TomTom GO 300、GO 700、TomTom RIDER 和 TomTom NAVIGATOR 5 客户的更新地图价格为 49.95 美元，此后所有产品的更新地图价格为 149.95 美元。
