# 三星 Vista LCD | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/19/samsung-vista-lcd/>

虽然我不是 Windows 的粉丝，但还是要感谢三星推出了两款获得 Windows Vista 认证的新宽屏显示器。是的，没错，我说的是 Windows Vista 认证。这些漂亮的产品有 20 和 22 英寸的型号(CX206BW 和 CX226BW ),带有时髦的黑色高光钢琴饰面，以加速新的 Syncmaster LCD 系列。细节不多，但我们确实知道这么多，对比度是 3000:1，响应时间为 2 毫秒。它们将分别花掉你大约 400 美元和 475 美元。
