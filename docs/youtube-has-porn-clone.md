# YouTube 有色情克隆 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/25/youtube-has-porn-clone/>

色情行业一直有一种“早期采用者”的模式。我们已经报道了一些在过去采用了新的网络功能的色情相关网站(特别是 [Heatseek](https://web.archive.org/web/20221209001247/http://www.beta.techcrunch.com/2006/07/11/porn-browser-heatseek-launches-yeah-porn-browser/) 和[社交色情](https://web.archive.org/web/20221209001247/http://www.beta.techcrunch.com/2006/01/07/social-porn-it-had-to-happen-eventually/))。集成了类似 YouTube 功能的 PornoTube (注:非工作安全)是这一趋势的最新例证。YouTube 有禁止色情的明确政策，而 PornoTube 的政策正好相反。

PornoTube 提供用户上传色情内容的点播 Flash 视频(我相信我们的赞助商 Adobe 会为此感到自豪)，包括视频、音频和照片。该网站的设计令人惊讶地好，并有像评级，评论等社会功能。所有视频都有标签，可以通过标签云或直人或同性恋内容浏览。有各种类别的 RSS 提要。视频也可以嵌入到其他网站。

该网站对用户免费。对于更多技术上的 saavy 用户来说，bittorent 长期以来一直是免费色情内容的来源。但是任何有电脑的人都可以使用的 PornoTube，可能会破坏对类似内容收费的网站。不过，我想 PornoTube 将会面临与 YouTube 类似的版权问题，而且随着时间的推移，它将不得不撤掉这些内容。

他们甚至有一个“测试版”在他们的标志。看看吧，如果你喜欢的话。
