# 三星 SGH-a707 又名 Cingular SYNC 发布

尽快，三星同步，或 SGH-a707，今天全部上市。Cingular 独有的超薄翻盖手机针对高速下行分组接入(HSDPA)进行了优化，可通过一键 Cingular 音乐键“即时访问”新的音乐服务。它还有外部触摸感应多媒体按钮和一个 microSD 插槽，可满足您的存储需求。哦，它支持蓝牙 2.0 上的立体声音乐，并有一个内置的 200 万像素的闪光灯，可以在你听果酱的时候拍照。所有这些只需 49.99 美元，带两年服务合同，并有 100 美元的邮寄折扣。
