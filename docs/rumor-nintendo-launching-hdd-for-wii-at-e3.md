# 谣言:任天堂在 E3 发布 Wii 硬盘

![38759-500-375.jpg](img/bab6bdda7dec55ebf00cbcbf92b2cebd.png)

今天是刚垩满的[谣言](https://web.archive.org/web/20130628150739/http://crunchgear.com/2007/06/04/new-mystery-app-spotted-on-iphone/)是不是？每个人都围着水冷却器谈论着 [iPhone](https://web.archive.org/web/20130628150739/http://crunchgear.com/2007/06/04/rumor-apple-to-announce-way-to-convert-mac-os-x-apps-for-use-on-iphone-at-wwdc/) ，所以是时候去别处寻找一些有趣的新闻/谣言了。

这一个应该让 Wii 的拥有者高兴，但不是约翰，他讨厌 Wii。据传任天堂将在 E3 展示一些新硬件，所有迹象都指向硬盘。粉丝们尖叫着要更多的存储空间，因为 Wii 只有微不足道的 512MB 内部存储空间。那么，是什么引发了这个谣言？宣布近地天体游戏加入 VC 是罪魁祸首。当游戏容量达到 330MB 左右时，你最好相信任天堂会打破桂冠，拿出让 Wii 粉丝满意的东西。
