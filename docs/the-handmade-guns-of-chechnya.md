# 车臣的手工枪

![scaled11.jpg](img/b495a9d645205741b795970d0eec2dea.png)

有没有玩过电子游戏，看着古怪的、未来派的或者蒸汽朋克武器，然后说“哇，如果那是真的，那该多好啊？”好吧，就去饱受战争蹂躏的车臣吧，在那里你可以享受发射一个盗版的，手工制作的肮脏的哈利风格的大酒瓶的刺激，以及让它在你手中爆炸的所有兴奋。

这些枪支被俄罗斯军队没收，显示出对可互换零件价值的公然漠视，让人感觉爷爷可能不仅仅是在车库里修理[特拉班](https://web.archive.org/web/20201129144951/http://en.wikipedia.org/wiki/Trabant)——他可能正在为该死的傻瓜游击队手工成型枪支零件。
