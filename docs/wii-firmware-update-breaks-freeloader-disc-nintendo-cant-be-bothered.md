# Wii 固件更新打破白吃光盘，任天堂不能被打扰

[![freeloaderbrokeloader.jpg](img/5b6d9f8ade4711a9df6eaab29a2bf978.png)](https://web.archive.org/web/20210120221830/http://old.crunchgear.com/wp-content/uploads/freeloaderbrokeloader.jpg "freeloaderbrokeloader.jpg")

昨天，任天堂 Wii 的固件更新使得 Datel 的免费光盘完全无用。免费下载让用户可以在 GameCube 上玩进口游戏，直到昨天，还可以在 Wii 上玩。更新之后，那个吃白食的…她死了。是的，现在想在 Wii 上玩进口 GameCube 游戏的玩家将不得不购买一种保修失效、肯定会被认定为非法的 modchips。

不用说，任天堂没有兴趣“修复”它的错误。由于免费下载是未经许可的软件，这种软件破坏了任天堂的整个地区特定的发行球拍，该公司不能费心去补救这种情况。

游戏魔方太难盗版了。
