# NPR 说 Wii 对儿童有害

![](img/2835a2e2272126516fd6df0da1fda930.png)

NPR 讨厌你，也讨厌你给孩子买的 Wii。Kelly McBride，一位家长，前几天在电视上攻击其他家长给他们的孩子买任天堂 Wii 游戏机。麦克布赖德说，虽然 Wii 让你的孩子从沙发上起来，手臂四处活动，但它给了孩子“一种在世界上竞争的错误感觉。”哎哟！她还能说什么更糟的话吗？你打赌。评论继续解释 Wii 运动如何糟糕，孩子们如何需要学习真正的运动，以及其他一些趣闻。

我听了很多 NPR 的音乐，所以我知道这篇评论可能呈现的风格，并意识到它可能有点刺耳。但是我同意凯莉和她的反 Wii 运动吗？绝对的。孩子们需要把屁股从沙发上挪开，一直走到门口，这样他们就可以出去玩了。下雪了？管他呢，拿起那个廉价的橄榄球扔来扔去。也许你们这些孩子真的会变得有趣些。我认为玩 Wii 运动和模拟身体活动的游戏是可以的，但你不应该用它来完全代替锻炼。只是我的两分钱。
