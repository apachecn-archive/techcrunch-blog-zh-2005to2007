# 专利猴子:斯巴鲁的骑士专利| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/20/patent-monkey-night-rider-patent-by-subaru/>

。

上周，富士工业/斯巴鲁获得了自动控制、GPS 导航汽车的最新[专利。斯巴鲁在这一领域最引人注目的是大众汽车，我们在](https://web.archive.org/web/20160526210100/http://www.patentmonkey.com/PM/PatentID/7177760.aspx)[斯坦福的正式声明](https://web.archive.org/web/20160526210100/http://www.eetimes.com/news/design/showArticle.jhtml?articleID=197006955)中快速浏览一下自动驾驶汽车，他们将参加今年的 DARPA 大挑战。

DARPA 早在 2004 年就举办了名为[大挑战赛](https://web.archive.org/web/20160526210100/http://www.darpa.mil/grandchallenge/index.asp)的自动驾驶汽车比赛，包括[斯坦福车队(由大众赞助)](https://web.archive.org/web/20160526210100/http://cs.stanford.edu/group/roadrunner/)在内的参赛者在该领域做出了一些[惊人之举](https://web.archive.org/web/20160526210100/http://www.darpa.mil/grandchallenge/video/StanleyCD001.wmv)。虽然在不久的将来很难想象，但 DARPA 参赛者推动这一领域向前发展的努力将渗透到我们的日常通勤中，并改善那些加入这些技术的汽车制造商和供应商的命运。疯狂？iRobot 的 Roomba 在使用机器人技术清洁和拖地方面取得了成功。

。

至于斯巴鲁的专利:坦率地说，用于车辆导航的 GPS 是不够的，因为 GPS 是一种视线技术，在 GPS 无法使用的城市中，导航真的很难。
