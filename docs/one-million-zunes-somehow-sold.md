# (不知何故)卖出了 100 万枚金币

![](img/51dfa85b1a96c4797c4553381566a232.png)

谈谈周末的震惊。微软设法卖出了一百万台 Zunes。微软娱乐和设备总裁 Robbie Bach 表示，微软已经实现了目标，今年晚些时候将会推出更多 Zune 产品。我可以理解也许一个小公司会因为卖出 100 万台设备而兴奋，但是苹果仅在去年第四季度就卖出了 1400 万台 iPods。是的，1400 万人买了 iPods。相比之下这些 Zunes 就显得微不足道了。

微软甚至有时间做一个新的 Zune 并推出它吗？它成功的可能性微乎其微吗？在这一点上我非常怀疑。
