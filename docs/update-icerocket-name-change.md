# 更新–ice rocket(名称变更)

**公司:** [IceRocket](https://web.archive.org/web/20200919182155/http://www.icerocket.com/)

![](img/9cb42b5cfa14799806304cd6a1cabb72.png)

**以前简介:**[2005 年 7 月 29 日](https://web.archive.org/web/20200919182155/http://www.beta.techcrunch.com/?p=111)

**有什么新消息？**

我们在 IceRocket 上的前一篇简介[中写道，他们将更名为 BlogScour(基于马克·库班在 AlwaysOn 说过的话)。](https://web.archive.org/web/20200919182155/http://www.beta.techcrunch.com/?p=111)

IceRocket 的首席执行官布莱克·罗德斯(Blake Rhodes)打电话告诉我，我们的事实并不完全正确(他还感谢了我们的帖子)。他们不会把名字改成 BlogScour，但是他们会推出一个名为 BlogScour 的网站，包含他们所有的博客搜索功能。

几分钟前我在 [Blogherald](https://web.archive.org/web/20200919182155/http://www.blogherald.com/2005/07/31/icerocket-dumps-blogscour-name/) 看到了这个(我找不到提到的 SEW 文章)，并给 Blake 发了邮件确认事实。他证实了他周五告诉我的事情

*“麦克——*

我们将推出一个名为 Blogscour.com 的网站。我没有约会对象。基本上，这将是我们目前拥有的博客搜索，减去我们目前在 IceRocket 上拥有的所有网络和图像搜索功能。这将是博客而已。祝你晚上愉快。

布莱克"

所以你有它。

就我个人而言，我一点也不在乎，我只是喜欢他们的搜索引擎。他们可以称之为 searchcrap.com，我们仍然每天用它来研究公司 20 次。
