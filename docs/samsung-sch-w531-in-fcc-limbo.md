# 三星 SCH-W531 在 FCC 的冷宫

![](img/822b088c0fb43b174fb8b20cb93d2662.png)

请关注三星 SCH-W531 的未来。除了看起来可以与几台零下冰箱融为一体，这款手机还有一些你可能在其他三星机型上看不到的不错的功能。它具有双模 GSM 和 CDMA，摄像头和视频捕捉，蓝牙，作为大容量 USB 存储设备的能力，以及支持 HTML 的网络浏览器。

它甚至可以处理 MS Office 和 PDF 文档，因此轻型企业用户肯定可以从 SCH-W531 中受益。还没有关于价格或可用性的消息。
