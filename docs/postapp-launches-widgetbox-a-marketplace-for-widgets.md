# PostApp 推出 WidgetBox，一个小工具市场

 [](https://web.archive.org/web/20221006080043/http://www.postapp.com/) 隐形初创公司 [PostApp](https://web.archive.org/web/20221006080043/http://postapp.com/) 在周四的 SuperNova Connected Innovators 会议上宣布，明天将推出 [WidgetBox](https://web.archive.org/web/20221006080043/http://www.widgetbox.com/) ，这是其新的基于网络的托管小工具的测试版市场，Hummer Winblad 提供了 150 万美元的资金。

如果你喜欢小部件，你的博客或个人资料页面中会有更多的小部件可供使用。如果你想开发小部件并让其他人来处理细节，这可能适合你。如果你对小工具不熟悉，可以看看 TechCrunch 右边的“社区搜索”框——那是一个小工具。可以放入网页的视频、幻灯片、音乐播放器和新闻收报机都是常见的小工具。

PostApp 负责，B2B 资深人士 Ed Anuff，Giles Goodwin 和 Dean Moses。该公司在 3 月底与 Typepad 合作推出了其首款产品易贝小工具。

PostApp 将管理将网络服务转化为小部件的过程，博客、社交网络用户和其他人可以将小部件插入他们的页面。外部开发人员将创建 web 服务，将它们提交给 PostApp 以转换为小部件，内容发布者如博客、拍卖卖家和社交网络用户将从 WidgetBox 市场选择他们想要的小部件。该服务还将管理涉及附属链接或订阅等金融交易的 widgets 的资金，尽管开发者将在决定其项目的商业规则方面拥有优先权。PostApp 将视情况充当主要附属机构或订阅中心。

随着网上照片、视频、电子商务和日历小工具的数量激增，我不禁认为，一个集中存放大量小工具的地方听起来是个好主意。我还希望这将使开发者更容易创建其他类型的小部件，因为现有的选择可能会让人觉得很陈旧。

第一批突出显示的小部件之一将使用 Yahoo images API 将上下文相关的图像插入任何网站。RSS 将是许多小部件的基础，但不是全部。如果您和我一样喜欢 RSS，您可能会想到几乎所有信息都是通过 feed 传递的，并显示在一个小部件中。如果一个有趣的各种各样的饲料被 widget 化，然后与断断续续的上下文广告混合，那么每个人都赢了。我将热切期待未来的小部件。
