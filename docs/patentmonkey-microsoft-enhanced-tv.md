# PatentMonkey:微软增强型电视

![MSEnhancedTV](img/54cae80dedfb51c025fdc7b8052bb120.png)微软最近获得了一项专利，涵盖了[通过将互动层与广播内容相结合来增强电视](https://web.archive.org/web/20201020164727/http://www.patentmonkey.com/PM/patentid/7228560.aspx)。事实上，这是他们拥有的众多[视频相关专利](https://web.archive.org/web/20201020164727/http://www.patentmonkey.com/PM/FolderID/1bf0f9e9b19e44f79eff06c2612963be.aspx)之一。

虽然任天堂 Wii 让我们离点击式电视更近了一步，却没有对其潜力发表太多言论，Joost [也获得了创新型新电视平台的荣耀](https://web.archive.org/web/20201020164727/http://crunchgear.com/category/joost/)，但在过去十年的幕后，微软和其他公司一直在构建系统，为你的电视提供丰富的互动媒体体验。

微软的发明目标是使用低端机顶盒作为中继，以 MPEG 流媒体的形式加入交互式内容，并将其加入到标记的现有内容中。

具体来说，微软的新专利描述了:

> 根据本发明的增强节目编排技术识别与要传送到娱乐模块(例如，电视)的节目编排相关联的增强节目编排内容，远离娱乐模块将这种增强节目编排内容准备成娱乐模块可以呈现给观众的形式，并且在接收到来自观众的输入时，将增强节目编排内容传送到娱乐模块。

微软最近采取的所有这些措施都是为了让[将](https://web.archive.org/web/20201020164727/http://www.zune.net/) [融入](https://web.archive.org/web/20201020164727/http://www.xbox.com/en-US/) [硬件](https://web.archive.org/web/20201020164727/http://www.microsoft.com/surface)以及整合线上和线下广告的竞赛，为微软继续保持谷歌作为其主要竞争对手的努力带来了一系列有趣的潜在事件。

一个简单的举措是利用 X-Box 作为一个增强的有线电视观看门户，将有线电视和互联网内容结合在一起，在同一屏幕上显示。把 Adobe Interactive Flash 想象成两层，第一层是典型的电视节目，第二层通过互联网传输，提供增强的内容。好吧，微软在专利中没有描述很多用例，但我们发现将两个流结合到一个具有交互式组件的查看器中的能力非常有趣(例如，单击从 IMDB 获取信息，单击查看产品广告，单击为节目中的内容添加标签，等等)。

![](img/ef834a8abc64b11bce7379386eb6eaf1.png)这一领域值得关注的设备开发商是 [Hillcrest Labs](https://web.archive.org/web/20201020164727/http://www.hillcrestlabs.com/products/freespace.php) ，它正在开发一种交互式遥控器和框架，使用设备中的运动传感器来操纵你的电视或 PC(参见[演示](https://web.archive.org/web/20201020164727/http://news.com.com/1606-2_3-6181659.html?tag=vid))

这个领域令人兴奋，许多设备变得更具成本效益。互动电视有可能把我们的电视变成互联网，或者可能是一个更被动和可控的版本。
