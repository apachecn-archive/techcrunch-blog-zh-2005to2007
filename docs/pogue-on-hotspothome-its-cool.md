# HotSpot@Home 上的 Pogue:很酷

![367367.jpg](img/280ed65a986332f485317cdf38962eb8.png)

本周，我一直在使用 T-Mobile 的 [HotSpot@Home](https://web.archive.org/web/20210123020500/http://crunchgear.com/2007/06/27/t-mobile-hotspothome-review/) ,老实说，如果你经常跳池塘，这项服务是你的救命稻草。David Pogue 倾向于同意，他说 T-Mobile 这样做是疯狂的，但也有一些好的理由。

> 因为 T-Mobile 也受益。让我们面对现实吧:T-Mobile 的移动网络无法与威瑞森相提并论。但是改善它的网络意味着花费数百万美元在新的信号塔上。仅仅分发免费家用路由器要便宜得多。

有人说，这是一个策略，让我们处理必要的蜂窝回程，以建立一个无线网络。不过，如果我在家能获得更好的语音服务，在旅行时能获得一些不错的额外功能，我愿意多付几块钱。
