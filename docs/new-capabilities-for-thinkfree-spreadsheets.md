# Thinkfree 电子表格的新功能

[![thinkfreelogo.png](img/975a06cbd944bffb079321419a183196.png)](https://web.archive.org/web/20201026010444/http://thinkfree.com/) 在线办公套件 [ThinkFree](https://web.archive.org/web/20201026010444/http://www.thinkfree.com/) 将于周二宣布与[Team and Concepts Limited](https://web.archive.org/web/20201026010444/http://www.beta.techcrunch.com/wp-admin/Team%20and%20Concepts%20Limited)(TnC)建立新的合作伙伴关系，将 TnC 的 EditGrid 电子表格应用程序与 ThinkFree 进行整合。该协议允许 ThinkFree 用更高级的 EditGrid 替换其当前的快速编辑 Calc。这笔交易包括两家公司之间的收入分成协议。

据说这笔交易将为 ThinkFree 提供“市场上最先进的实时协作电子表格”。

使用 EditGrid，ThinkFree 用户将能够同时在电子表格上进行协作，并实时看到发生的变化。远程数据功能将在电子表格中提供对外部数据的访问，允许这些源实时更新。

Power Edit Calc 允许更复杂的函数集，并提高了与 Microsoft Excel 的兼容性。

Thinkfree 是第一家推出在线办公产品的公司，尽管面临来自谷歌的激烈竞争，该公司仍在不断创新和发展。之前的 TechCrunch 报道[点击这里](https://web.archive.org/web/20201026010444/http://www.beta.techcrunch.com/tag/thinkfree)。
