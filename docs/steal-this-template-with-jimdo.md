# 用 Jimdo 窃取我们的网站模板

对技术人员来说，制作 HTML 可能很简单，但对其他人来说就不够简单了。如今，不仅仅是设计和编码一次页面，而是心血来潮地重新设计。MySpace 和它反复无常的设计爱好者社区就是一个很好的例子，他们经常改变他们的布局，就像他们最喜欢的乐队一样。

一些服务通过提供简单的网站创建工具来满足这种需求。它们就像 Geocities v 2.0。Jimdo 是今年早些时候推出的另一个来自德国汉堡的易于使用的 AJAX 站点编辑器。您可以使用 Jimdo 轻松快速地创建个人网站，包括照片、文本、留言簿、rss 源和 YouTube 视频。他们是我们之前介绍过的另一个 AJAX 编辑器 Weebly(我最喜欢的编辑器之一)的竞争对手。

我对这些网站的一个问题是，设计仍然会受到相当大的限制。如今，Jimdo 已经解决了这个问题，让你只需几个步骤就能从任何网站轻松获得设计。

1.  将 HTML 源文件从目标站点复制并粘贴到 Jimdo 中。
2.  点击“XHTML”——系统会自动剪切出相关代码(内容、侧边栏、导航、页脚)——并验证。
3.  将你的 CSS 复制到 Jimdo 中。
4.  在“文件”标签下上传你的布局所需的图片。

Jimdo 基本上使用他们的站点理解的特殊标签将他们的小部件放入模板的关键部分。结果是模板显示了您的 Jimdo 导航栏和内容。你可以在这里看到 TechCrunch 网站[的一个例子。不过，我仍然不得不摆弄 HTML 代码来使它工作，手动将特殊的标签放在内容需要去的地方。](https://web.archive.org/web/20210805232804/http://templateapidemo-nick.jimdo.com/)

一个由三个朋友组成的团队最初以 CMS 解决方案的形式为企业销售简单的站点创建工具，但是今年决定增加一个消费者版本。他们目前拥有超过 25，000 名用户和三种不同语言(德语、英语和中文)的网站。他们的基本帐户从 500 MB 的存储空间开始，专业帐户每月 6 美元，可以在自己的域名上托管它们。

还有其他一些针对不同人群的网站设计工具。RealEditor 专门为 MySpace 制作编辑器。 [Synthasite](https://web.archive.org/web/20210805232804/http://www.beta.techcrunch.com/2006/09/04/very-early-look-at-synthasites-ajax-website-builder/) 是一个很好的基于 AJAX 的编辑器，可以详细控制页面布局。 [Sampa](https://web.archive.org/web/20210805232804/http://www.beta.techcrunch.com/2007/06/08/sampa-brings-personalized-pages-to-facebook/) ，虽然有一个较弱的布局编辑器，但针对的是希望创建一个团体网站的家庭。Webjam 是一个由编辑创建的网站社区。
