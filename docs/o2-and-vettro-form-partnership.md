# O2 和 Vettro 结成合作伙伴关系| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/15/o2-and-vettro-form-partnership/>

![vettro.jpg](img/2d170a326fa96b0eb0bbbd544fce9d9e.png)英国移动运营商 O2 正与全球移动点播业务应用的领导者 Vettro 合作。这将允许 O2 销售人员直接销售 Vettro 360 移动应用套件。这一合作伙伴关系将加强 O2 的产品组合，其目标是不断增长的移动工作人员。这将为 Vettro 提供一个在国际市场获得更大立足点的机会，同时也让 O2 增加其在业务领域的份额。
