# 用户对触摸屏界面很敏感

Strategy Analytics 的一份报告“触摸屏设备的积极学习曲线”发现，视觉反馈和耐用性是手机触摸屏界面成功的关键。虽然为手机操作配备手写笔是可以的，但根据定义和研究，触摸屏必须对手指手势做出反应。“屏幕的耐用性仍然是触摸屏设备最关心的问题。然而，准确性增强技术和视觉反馈——而不是触觉反馈——也是成功的关键，”Strategy Analytics 的高级用户体验分析师保罗·布朗说。
