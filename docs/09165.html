Zoho，Omnidrive办公文档存储合作伙伴| TechCrunch

原文：https://web.archive.org/web/http://www.techcrunch.com/2007/01/19/zoho-omnidrive-partner-for-office-document-storage/trackback/

在线办公套件 Zoho 和在线存储公司 Omnidrive (见下文披露)今天宣布合作，扩展两家公司的功能。集成利用了Zoho最新更新的 API 。

这种合作关系允许用户在Omnidrive中存储他们所有的Zoho office文档，给他们一个集中的在线文件存储区域。此外，点击Omnidrive中的任何office文档都会自动打开该文档，以便在Zoho中进行编辑。它还允许用户通过使用Omnidrive的共享特性在文档上进行协作。

这种新的合作关系也意味着Omnidrive现在可以说提供了在你的桌面和在线制作和编辑文件之间的完整链接。Omnidrive的文件链接使得在你的桌面或Zoho Writer中进行的更新也能更新/同步其他位置的文件。当我们之前分析在线存储服务时，这种链接是吸引我们眼球的主要特性。

欲了解更多信息，请参见 Omnidrive 和 Zoho 博客。关于这些公司的背景，请参见我们之前对 Zoho 和 Omnidrive 的报道，并在此处查看Webware最近对Omnidrive 的报道。

披露:2006年12月，TechCrunch创始人迈克尔·阿灵顿投资了Omnidrive。更多信息参见TechCrunch 关于第页。
