# Sphere 会解决博客相关性问题吗？

 [奥姆·马利克](https://web.archive.org/web/20230216142409/http://gigaom.com/2005/10/08/sphere-the-relevant-blog-search/#more-5190)写了一篇关于 [Sphere](https://web.archive.org/web/20230216142409/http://www.sphere.com/) 的文章，这是一种新的博客搜索服务，似乎正准备推出。

Sphere 正在尝试建立一个更相关的博客搜索引擎。传统的链接分析对博客文章不起作用，因为新文章没有时间收集链接。相反，Sphere 似乎在试图确定某个给定主题领域的博客权威，并根据这些博客指向的人来确定新的权威博客。

从 Sphere 的[了解更多信息](https://web.archive.org/web/20230216142409/http://www.sphere.com/about.html)页面:

> 是什么让我们比现有的博客搜索网站更好？
> 
> 它始于相关的结果和快速的性能。我们新的基于相关性的算法在新的博客帖子创建时发现它们，在发布后几分钟内对它们进行索引，应用丰富的语义分析，并使它们可以按相关性或时间进行搜索。此外，我们还有一些有趣、有用的功能，我们认为这些功能可以带来更丰富的用户体验。

当然，在他们发布之前，没有办法知道 Sphere 是否已经解决了博客搜索相关性的问题。我很期待找到答案。
