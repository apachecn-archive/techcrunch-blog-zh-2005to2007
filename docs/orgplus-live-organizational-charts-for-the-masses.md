# OrgPlus Live:面向大众的组织结构图

[![grabberraster-0001-14-24-32.jpg](img/70acd1750a34f41cdc00764aa38740fd.png)](https://web.archive.org/web/20210225233839/http://www.orgpluslive.com/ "grabberraster-0001.jpg") 我知道这既不是“设备”，也不是“小工具”，但我喜欢告诉你，读者，当我遇到好的软件时。软件供应商 Human Concepts 开发了 OrgPlus，这是创建组织结构图的首选应用程序。OrgPlus Live 是该软件的新网络版本。不认为你需要组织结构图？好吧，也许只是你不知道更好。

当然，创建组织结构图的一些目标是获得公司结构的视觉概念，建立项目团队，并随着组织的变化跟踪每个人的角色和联系信息。但是我用组织结构图来安排故事，建立我的赌博公司一个 NCAA 绅士的办公室，甚至为一个棒球联盟设计团队——基本上是任何需要结构和秩序的事情。

OrgPlus Live 带来了基于 Windows 的软件(起价 189.95 美元)的组织结构图优势，并将其应用于基于网络的版本，由于 Flash 的奇迹，该版本可以在任何平台上使用。拖放式创建、简单的文本输入和字段编辑、简单直观的界面、可靠的安全性、从 Excel 导入现有数据的能力等等，现在每月费用低至 9.99 美元。如果你仍然看不出它有多大用处，我敢说你可以自己尝试一下——有 14 天的免费试用期，不需要安装或下载任何东西。
