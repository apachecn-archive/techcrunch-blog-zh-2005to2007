# Oracle Linux:明天终于推出了？

严格来说，这不算是小玩意新闻，但却是大新闻。拉里·埃里森(Larry Ellison)是硅谷唯一一个比苹果公司的乔布斯还自负的人，他将于明天在旧金山举行的甲骨文全球大会(Oracle OpenWorld)上向地球投掷甲骨文的 Linux 炸弹。Oracle 是数据库世界中 800 磅重的庞然大物，在 145 个国家设有办事处。埃里森本人一度是《福布斯》杂志的世界首富。这将标志着该公司在操作系统领域的首次努力。

如果是真的，这将证实关于该项目的超过一百万互联网年的谣言，并使许多 Linux 粉丝陷入狂热。O'Reilly 似乎认为 Ubuntu——当前 Linux 界的美女——将成为这一新计划的基础，这让 Red Hat 很是懊恼。

明天一整天回来看看埃里森给我们带来了什么，如果有的话。虽然这个声明已经被期待了一段时间，但它最终还是要发布了。
