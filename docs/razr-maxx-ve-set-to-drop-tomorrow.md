# RAZR MAXX Ve 设置为明天下降？

威瑞森明天将发布 RAZR 系列中唯一一款我们有点喜欢的手机，即 [MAXX Ve](https://web.archive.org/web/20201202043602/http://crunchgear.com/?s=maxx) 。MAXX 的外形与其他 RAZRs 相似，但增加了一个巨大的 2MP 自动对焦摄像头，带真正的闪光灯和一个超大的外部 LCD。凭借内置的 EV-DO 连接，这个小蛤壳注定要 flickr。

两年合约的价格应该在 200 美元左右，我们预计几周后会降到 100 美元，如果你能等的话，我们知道你能等。
