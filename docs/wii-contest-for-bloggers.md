# 博主 Wii 竞赛

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/14/wii-contest-for-bloggers/>

![](img/56507edcb17834a41ad7211b37da1f3b.png)

懒得(或穷到)在百思买外面等上几个小时想要一台 Wii？别担心，[wHooiz.com](https://web.archive.org/web/20201124123000/http://www.whooiz.com/)的官方博客正在免费赠送一台任天堂 Wii(押韵让我自我感觉良好)。你所要做的就是去 wHooiz 博客做你自己的简介。

wHooiz 博客基本上是一个列表网站，允许你查找其他人的简介，阅读他们的传记。这个网站刚刚起步，所以你还不会在上面找到太多东西，但如果它起飞了，它可能是一个可靠的参考资料，我们可以用它来找到更多关于某个人的信息，这个人还没有突出到像维基百科这样的东西(编辑:或 Myspace 或脸书-布莱克)。

当你有了 wHooiz 个人资料后，你必须把 wHooiz 插件放到你的博客上，并给客服发邮件，让他们知道你已经注册了。现在还不算太糟，是吗？谁知道呢，也许你会遇到一些喜欢你资料的辣妹。呃，也许不是。
