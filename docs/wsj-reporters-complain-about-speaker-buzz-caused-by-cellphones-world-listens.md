# 《华尔街日报》记者抱怨手机引起的扬声器嗡嗡声，全世界都在听

[![bzzz.jpg](img/4bae892a83d4db25947bea42775c70e3.png)](https://web.archive.org/web/20210227140010/https://beta.techcrunch.com/wp-content/uploads/2007/05/bzzz.jpg "bzzz.jpg")

BZZZZZZ。每当手机或智能手机将要响起，接收短信，或者对于你[黑莓](https://web.archive.org/web/20210227140010/http://crunchgear.com/2007/05/03/att-gets-first-crack-at-blackberry-curve-surprised/)用户来说，接收电子邮件时，扬声器就会发出这种声音。而且噪音很烦人，让我告诉你。即将被默多克拥有的《华尔街日报》深入研究了这个问题，听取了每个人的意见。许多人表示，听到扬声器嗡嗡作响会让他们感到尴尬，尤其是在董事会议进行中或演讲时。不知何故，很多人都没有关掉这个愚蠢的东西，这是一个遗憾，因为这是目前唯一真正解决问题的方法。
 也许我只是个混蛋，但这不就是几年前一个[牛逼的男生](https://web.archive.org/web/20210227140010/http://en.wikipedia.org/wiki/Jerky_Boys)咬的吗？“我的扬声器发出可怕的嗡嗡声。我的扬声器，我说，哦，我的上帝。”这真的是一个大问题，或者至少是一个烦恼吗？当我输入东西的时候，我自己的扬声器发出嗡嗡声。没有疯狂在这里报告，敲木头。
