# UBiQUiO 501 掌上电脑手机| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/23/ubiquio-501-pocket-pc-phone/>

![](img/5130150ebddbdbc40fda8a43c1f99fc6.png)

还记得 Tech Faith Wireless 的 UBiQUiO 501 智能手机吗？是的，智能手机看起来和现在无处不在的[摩托罗拉 Q](https://web.archive.org/web/20140530140124/http://www.motorola.com/motoinfo/product/details.jsp?globalObjectId=113) 非常相似。《移动新闻》写道，UBiQUiO 已经出现在 [Expansys](https://web.archive.org/web/20140530140124/http://www.mobileplanet.com/product.asp?code=140098&tab=0) 上，售价仅为区区 514.95 美元。所以，如果你有兴趣多花 300 美元买一个 Q，拿起你的信用卡，去 Expansys 吧。

公平地说，UBiQUiO(我感觉我在 AOL 上打字)确实提供 802.11 b/g 连接，所以它不是完全相同的手机。然而，考虑到无线宽带的可用性和 WiFi 消耗的巨大能量，将其包括在内似乎基本无关紧要。
