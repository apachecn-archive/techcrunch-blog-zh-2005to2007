# 专利猴:微软入股 DRM 专利索赔

他们想知道他们在数字版权管理上投入了多少努力。

2001 年初，苹果公司采取了一项重大举措，与主要唱片公司签约，提供合法的音乐下载，尽管有很多限制~~[公平竞争](https://web.archive.org/web/20201204055447/http://en.wikipedia.org/wiki/FairPlay)。公平地说，当时许多人认为 DRM 是合法启动数字派对的好方法。虽然 iPods 已经售出，微软也在开发 playsforsure 和 Zune，但我们在 DRM 上收集了更多的图片，这让我们更加深入地了解了史蒂夫对音乐的看法。~~

最近，微软发布了第 7185363 号美国专利，涵盖了第一设备向第二设备转移 DRM 权利的方法。当阅读这篇文章时，Zune 的喷射很容易在文章中看到，虽然没有具体说明。从战略的角度来看，微软似乎已经准备好进入 DRM 市场。不幸的是，市场在变化。

回顾微软的 DRM 专利[(微软-DRM 专利列表)](https://web.archive.org/web/20201204055447/http://www.patentmonkey.com/PM/FolderID/c9091e658d9145578bc8fdcab85e485b.aspx)，初步统计总计 32 项。[DRM](https://web.archive.org/web/20201204055447/http://www.patentmonkey.com/PM/PatentID/7185363.aspx)的一个奇怪的例子不言而喻:*“如果[数字许可证]不可用，DRM 系统会尝试在没有用户干预的情况下从许可证服务器上悄悄地获取许可证。”*听你一清二楚 Windows 正版优势。

尽管苹果公司利用 DRM 取得了巨大的成功，但他们似乎并没有发布任何涵盖音乐 DRM 的专利。在我们看来，苹果[不需要 DRM](https://web.archive.org/web/20201204055447/http://crunchgear.com/2007/02/08/apple-vs-the-world-patentmonkey-ip-review/) ,因为这个游戏(re: money)更多的是关于精巧的设备，而不是媒体销售。所有这些都给史蒂夫·乔布斯关于 DRM 应该消亡的声明增加了一些洞察力。声明 iPods 上 97%的音乐不是从 iTunes 上购买的，这是苹果公司的一大承认。再加上 iTunes v .苹果其他产品的销售额，这一点就非常清楚了。DRM 不会对苹果的产品机器造成影响。

随着 Zune 注入如此多的营销推动，如果 DRM 最终解除，微软将有一些品牌、产品和专利重组。
