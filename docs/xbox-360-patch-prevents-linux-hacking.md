# XBox 360 补丁防止 Linux 黑客攻击

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/06/xbox-360-patch-prevents-linux-hacking/>

昨晚这可能会影响到你，特别是如果你在 XBox 360 上运行 Linux 以便更快地处理 SETI@Home。否则，你很好。

该补丁填补了一个允许未签名代码注入的漏洞，允许用户在他们昂贵的游戏控制台上运行 [Linux](https://web.archive.org/web/20150910174335/http://news.com.com/Group+claims+Linux+advance+on+Xbox/2100-1043_3-1021692.html) 或其他自制代码。说真的伙计们。这是一个 XBox 360。黑你的正规 XBox，玩几年游戏，然后装 Linux。在把你所拥有的撕成碎片之前，先享受一会儿。
