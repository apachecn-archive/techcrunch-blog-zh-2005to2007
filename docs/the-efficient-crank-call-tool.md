# 高效的骚扰电话工具

![](img/fdbec14cbb98b4682d9ec75386f0e4c4.png)

我们昨天听说了一家新的“创业公司”。你输入一个电话号码，告诉它你希望服务呼叫它多少次。然后，该服务将呼叫电话该次数。

我们不打算说它的名字，或链接到它，因为它没有实际用途，可以用来不停地骚扰电话。这位企业家认为，这对试图寻找丢失手机的人很有用。我们说，用另一部电话打给丢失的那部。相反，他创造了一个 15 岁孩子的完美骚扰电话工具——设置为最多 9 次重复呼叫，让它在凌晨 3 点响起。

我们测试过了。它工作了。而且很蠢。

但是这里有一点。人们抱怨我们在 TechCrunch 上没有覆盖足够多的初创公司，更多(或所有)提交个人资料的公司应该被覆盖。大多数都不像这个服务那样愚蠢，但是应用过滤器是有目的的。我们每天都看到这样的东西，就在这一次，我们想和你分享其中的一个。
