# 诺基亚 7900 是一个大骗子

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/18/nokia-7900-is-a-big-faker/>

![nokia-7900-lamour-collection.jpg](img/3be0dd8dc1a38864166c92ab4f81d613.png)

我喜欢在醒来时听到最热门的新诺基亚手机上市的消息，甚至无聊的手机偶尔也会让我笑一笑，但今天早上的[太可怕了](https://web.archive.org/web/20160531103726/http://crunchgear.com/2007/06/18/nokia-releases-low-end-phones-snooze-gasm/)，对吧？幸运的是，一些德国粉丝找到了最新款爱慕手机 7900 的图片和规格。

它有一个时髦的对称图案，如果你在面前快速挥舞，可能会让你眩晕。有人猜测这可能是一个大骗子，因为它看起来完全像 7500，但 3G UMTS 规范似乎奠定了这种猜测休息。在进一步查看“规格表”后，你会注意到一些明显的事实，让你想知道这可能有多合法。任何以爱情系列发布的手机都必定至少有一个 200 万像素的摄像头，然而，这似乎有一个 130 万像素的摄像头。WTF？！一两个月后就知道了。
