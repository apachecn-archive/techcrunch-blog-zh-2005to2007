# 雨中的仪式笔记本

![](img/a165f0ef4949eae3505fc7d40d73e8ed.png)

嘿嘿嘿！在计算机出现之前，人们使用叫做铅笔、T2 和笔记本的东西。问题是笔记本，就像他们的电脑一样，当你把它们放在雨中时，它们会被淋湿并毁坏。不好玩。

幸运的是，我们在雨中的笔记本里有仪式！它们很便宜，根据大小只需 2-4 美元，并且被一种防水物质 Polydura 覆盖。所以，去吧，在野营旅行、自然远足或音速驾车探险时，留一辆在外面淋雨。你会没事的。
