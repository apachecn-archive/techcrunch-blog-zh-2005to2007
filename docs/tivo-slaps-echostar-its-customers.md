# TiVo 打了它的客户 EchoStar 一巴掌

![dish.jpg](img/2f7d540bca9a71fcb437d421cd4901af.png)

在最近的一次裁决中，德克萨斯州的一名法官发现 EchoStar (Dish Network)故意侵犯了许多 TiVo DVR 版权。EchoStar 现在必须向 TiVo 支付近 9000 万美元的赔偿金。除了现金结算之外，数百万元的 DVRS 菜网将被有效作废。

现在，我可以理解一家公司想要维护其知识产权和所有这些噪音，但对如此多无辜的消费者产生负面影响是一件令人难过的事情。我认为对消费者更公平的做法是评估正在使用的 DVR 的价值，然后相应地调整结算价格。损害已经造成了，所以将它们从市场上移除除了做出一个荒谬和浪费的声明之外，什么也做不了。
