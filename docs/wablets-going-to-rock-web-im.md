# Wablet 将震撼网络即时通讯

 [](https://web.archive.org/web/20220212225736/http://wablet.com/) 昨天我得到了即将发布的网络即时通讯系统 [Wablet](https://web.archive.org/web/20220212225736/http://wablet.com/) 的预览。三周前，我发布了关于 Wablet 的第一个[谣言。这是一个 Flash 服务，可以访问你在主要 IM 服务上的所有帐户，或者用户可以获得一个 Wablet 用户名。即时消息会话可以嵌入到博客和社交网络中，也可以出现在任何页面的弹出框中。这并不是什么新鲜事，但 Wablet 可以让你同时管理与看到不同公开个人资料描述你的人的多个聊天。](https://web.archive.org/web/20220212225736/http://www.beta.techcrunch.com/2006/08/04/meebo-to-face-more-competition-from-stealth-startup-wablet/)

这种服务有潜力成为重要的业务，看看最近美国在线对用户平面的收购和红杉资本对 Meebo 的投资。

Wablet 利用了身份、声誉、权限、队列和一个非常好的 flash 界面——它很有野心。商业模式将在很小程度上是上下文广告，但在很大程度上是基于小企业对服务的使用，作为其 CRM 战略的一部分。我可以肯定地说，对核心功能丰富的网络即时消息感兴趣的消费者将会爱上 Wablet。

该公司在旧金山和马尼拉设有办事处。这是由大卫·富特领导的，他是菲律宾社交网站 Groovenet 的主席和创始人。Foote 说，在接下来的几周内，将会有多达 500 个测试账户开通，你现在可以在网站上提交你的电子邮件地址来申请一个。

 **下面是它的工作原理。**一个 Wablet 聊天框可以作为弹出窗口嵌入或链接。寻求与您聊天的访问者必须首先请求许可，您可以在接受即时消息之前查看此人的个人资料和通过与系统其他用户聊天建立的声誉。到目前为止，只有 Wablet 即时通讯服务的用户才是这种情况，但其他主要即时通讯供应商的用户名可能会在未来获得声誉(我希望如此)。用户配置文件被设置为包含一长串其他系统的配置文件页面的徽章，因此当某人试图与您聊天时，会有指向他们在 Flickr 或 MySpace 上的页面的链接。当你有很多人想和你聊天时，他们的头像会出现在你当前所在的盒子上方的一个格子里。等待聊天显示该人等待批准的时间，正在进行的聊天显示在另一个窗口中与您聊天的人头像上方的思想气泡中输入的线条。

Wablet 的主要创新围绕着身份。该服务在基本即时消息的基础上增加了第二层，用户可以创建多个具有不同个人资料的角色。如果你有一个商务页面，那么有兴趣通过该页面与你聊天的人将会看到你为商务建立的个人资料，其中的徽章可能会链接到你的 Linkedin 页面和其他与商务相关的内容。但是，在不同环境下访问您页面的访问者将会看到不同的个人资料，其中包含关于您的不同信息。当您将代码插入到您的配置文件页面时，外部配置文件将被验证。Wablet 的管理界面可以很容易地选择你的所有信息中的哪些信息将填充每个角色的复选框，以及你希望在任何特定的聊天情况下使用哪个角色。创始人大卫·富特说，当 OpenID 标准成熟时，Wablet 将作为一个简单的接口来强大地使用 OpenID。

您可以通过您管理的聊天访问点为每个站点设定不同的在线状态。聊天档案和联系人可以添加标签，并且可以在您的私人帐户中添加注释。声誉很容易建立，拇指向上和向下，评论与你的角色联系在一起。可以将消息发送给脱机的用户，以便以后发送。该公司将为 Flash 开发者发布一个 API，这些开发者认为他们可以添加到已经详尽的功能集，或者进一步将 Wablet 与其他服务集成。
