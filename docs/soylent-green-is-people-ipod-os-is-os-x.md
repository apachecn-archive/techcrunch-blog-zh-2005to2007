# Soylent Green 是人，iPod OS 是 OS X！

![converge.jpeg](img/714335a625fa6e7c5b0c990b1f195e92.png)
更新——[这个故事也完全是假的。](https://web.archive.org/web/20130628160457/http://daringfireball.net/2007/07/os_x_ipod_nonsense)

如果现在对你来说是清晨，你还没有喝咖啡，请随意跳过这个故事。基本上，它说 OS X，以 Mach 内核的形式，一直存在于 iPod 中——除了它是 [Pixo OS](https://web.archive.org/web/20130628160457/http://sfgate.com/cgi-bin/article.cgi?f=/c/a/2004/08/16/BUGTG878AR1.DTL) 的短暂时期——还有……ZZZZZZ……嗯？
 哦，耶。嗯。无论如何，他们认为由于 iPod 的处理器如此之弱，苹果基本上只能在其标准的、交叉兼容的内核上实现第三方解决方案。然而，随着 iPhone 的崛起，他们终于能够将 OS X 与石英表示层(Quartz presentation layer)一起放入一个小包装中，该软件使 OS X、苹果电视和 iPhone 如此纯净。你在乎吗？好吧，当然。它表明，通过使用 BSD，苹果基本上已经完成了对 Sun 和微软的“一次编写，随处运行”的竞争。与 WinCE 及其所有相关操作系统不同，OS X 核心出现在每台设备中，这意味着，从总体上看，每台设备都是一样的。嘿，我警告过你这会很无聊的。

那些 OS X 的 iPods？他们已经来了！Pixo、ARM 和 Mac OS 【粗略起草】
