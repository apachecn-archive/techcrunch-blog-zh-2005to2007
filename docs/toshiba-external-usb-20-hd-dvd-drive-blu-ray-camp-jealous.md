# 东芝外置 USB 2.0 HD DVD 硬盘，蓝光阵营眼红

![](img/ee77796400289e1e03a660215e8a72d7.png)

东芝似乎突然开始出货 PA3530U-1HD1，这是一款外置 USB 2.0 HD DVD 驱动器。它甚至没有那么贵，在新蛋网上商店售价 370 美元。(是的， [360 的](https://web.archive.org/web/20210410223344/http://crunchgear.com/2006/11/10/xbox-360-hd-dvd-drive-unveiling/)外置 HD DVD 硬盘仅售 200 美元，但这让你不得不在 360 上观看高清电影。你有更多的选择在哪里看这个。)它也很纤薄，所以不难想象，在长途飞行中，为了看高清电影，把它放在笔记本电脑旁边。

除了读取 HD DVD 光盘，该驱动器还可以刻录 CD、单层和双层 DVD。这个发现不错。太糟糕了，蓝光的[胜利](https://web.archive.org/web/20210410223344/http://crunchgear.com/2007/02/01/the-war-is-over-blu-ray-outselling-hd-dvd/)是不可避免的，至少索尼是这样认为的。
