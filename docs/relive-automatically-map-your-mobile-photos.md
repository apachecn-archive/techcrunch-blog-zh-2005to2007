# 重温:自动映射您的手机照片

[![relivelogo.png](img/abc4e7e3a321ce315904156563fbc26c.png)](https://web.archive.org/web/20201125084006/http://relive.atlasct.com/) 今日测绘老兵图册(est。1987)正在为 Java ME 推出一个新的移动照片地理标记应用程序，名为[revive！](https://web.archive.org/web/20201125084006/http://relive.atlasct.com/)。好像和雅虎差不多！Flickr 的地理标记程序。然而，重温！并不意味着只对单个照片进行地理标记，还可以在旅行的背景下将它们连接到地图上。当搭配 GPS 系统，重温！允许您自动映射您在旅途中制作的带标签的照片、视频或虚拟明信片。因为每张照片都有时间戳，所以照片、视频和明信片也可以在旅行途中订购。整个旅程也可以通过他们网站上的账户查看和分享。他们目前默认的例子是在巴塞罗那漫步。

该程序在巴塞罗纳 3GSM 举行的 NAVTEQ 全球 LBS 挑战赛上赢得了欧洲社交网络奖。该应用程序旨在展示 AtlasCT 基于矢量的地图程序，该程序需要较小的数据传输，并可随移动屏幕大小的变化而变化。AtlasCT 没有直接针对消费者。在与诺基亚达成的一项协议中，所有诺基亚智能手机都预装了地图软件。他们目前正在美国寻求更多的 OEM 交易，并相信至少有一家相机制造商准备将 GPS 集成到他们的设备中。

在 GPS 变得更加普及之前，OEM 路线似乎是一个明智之举。只有一小部分美国手机支持 GPS，而没有某种蓝牙功能(Helio 和 Boost 支持整个网络的 GPS)。如果你真的对地理标记感兴趣，看看这款天生支持地理元数据的[三洋手机](https://web.archive.org/web/20201125084006/http://www.oreillynet.com/etel/blog/2006/03/first_us_phone_to_add_gps_data.html)。

Shozu 也有一个地理标记应用程序，可以支持多种服务。
