# 诺基亚 E62 现已从 Cingular 发货

我们想知道 Cingular 什么时候会进入中端的非黑莓 QWERTY battle royale，那一天就是今天。诺基亚 E62 现已在 Cingular 网站上出售，两年合约价格为 149.99 美元。

这款轻薄的智能手机实际上是几天前在 T2 推出的，但现在正在发货，这对那些真正想要一部实体手机的人来说是个好消息。黑莓风格的 E62 有一个 320 x 240 的屏幕，运行 Symbian 的 S60 智能手机操作系统，这是 Windows Mobile 和 PalmOS 的替代品，我们在美国不像在欧洲那样常见。它还配备了 EDGE 高速无线、扩展卡插槽和蓝牙。

E62 直接面向摩托罗拉 Q 市场，Cingular 正通过其黑莓连接服务销售它，这意味着你可以拥有你的(黑莓)蛋糕并吃掉它。
