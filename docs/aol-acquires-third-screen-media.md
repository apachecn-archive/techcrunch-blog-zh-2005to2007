# AOL 收购第三屏幕媒体

mocoNews 报道，AOL 已经完成了收购移动广告网络和移动广告服务平台提供商 Third Screen Media 的交易。两家公司自 2 月份以来一直在谈判，但交易的财务条款没有披露。mocoNews 还报道说，第三屏幕媒体将继续作为 AOL 的全资子公司，总部设在波士顿。
