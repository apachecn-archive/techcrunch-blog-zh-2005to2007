虽然我通常反对让便携式 AV 播放器过于复杂的想法，但我确实意识到内容将很快以无线方式交付——购买、下载或同步。MusicGremlin 可以让你通过 Wi-Fi 下载和分享，Archos 604 Wi-Fi 运行 Opera 浏览器的一个版本，让你像在 PC 上一样上网冲浪。在本周的消费电子展上，我还在 MP3 播放器上看到了大量很酷的 Wi-Fi 应用；例如，iRiver 的 W10 可以从你的电脑上播放音乐到打网络电话。所有这些都不会使音乐播放器变得太复杂而无法使用。但是由于更快和更低功率的处理器，玩家现在可以做更多的事情。

看，iPhone 将运行 Mac OS X，尽管还不清楚它会是一个多么成熟的操作系统版本。这与运行 Windows Mobile 的东芝 gigabeat 播放器不同；它更像是没有体积的 UMPC。多么强大的概念！忘掉空中下载吧——你很快就能在这些东西上运行 BitTorrent 了。但是有一件事一直在帮助 iPod 超越巅峰，那就是软件集成——也就是 iTunes。目前这一切都很好，但这种情况不会持续太久。DAP 进化的下一个阶段显然是无线同步，这将从根本上消除 PC 上的音乐文件和 MP3 播放器之间的中间人，并可能使游戏环境更加公平。

人们想要与他们的机器配合得最好的设备，这就是为什么 iPods 直到与 Windows 盒子配合后才真正爆发。但是如果微软能设法制造一个触摸屏播放器，运行一个足够完整的 Windows 版本~~并使其正常工作~~，该公司可能真的会获得巨大成功。无线透明同步与 Windows 一样无缝，仍然覆盖世界上大多数电脑，是一个杀手级应用程序。如果他们能为同步媒体制作一个 Mac 客户端，如果为 Mac 设置双重引导变得比现在更容易，那就更好了，MS 可能会有和 iPod 一样通用的东西。

正如我上周所说的那样，我相信史蒂夫·乔布斯会保留 iPhone 所有的无线优势，并让 iPod 或多或少保持原样——一个简单且相对便宜的音乐播放器。毕竟，价格才是真正推动非 Wi-Fi 播放器市场的因素，而不是技术的进步。但至于支持 Wi-Fi 的 DAP(不具备手机功能)，微软最有前途，因为它仍然拥有操作系统业务，随着 MP3 播放器获得更强大的处理器，这将成为新的最重要的因素。关键要素是让播放器和你的电脑真正无缝连接——当它们本质上运行相同的软件时，这就容易多了。

我敢打赌，最初的 iPhone 发布不会看到 OS X 是一个很大的因素，但第二代肯定会。Zune 品牌的下一款设备可能会运行几乎完全成熟的 Windows 版本，这可能会使它成为像 MP3 播放器一样的个人电脑，而没有提高价格和复杂性的手机功能。一旦人们意识到拥有一个运行真正操作系统的价格合理的 DAP 的力量，数字对微软非常有利。

> 3.苹果还能对 iPod 做些什么？
> 尽管人们都在谈论真正的视频 iPod，但我开始怀疑这款产品是否会令人失望。如果苹果推出一款和普通 iPod 很像但屏幕更大的东西，你会满意吗？我想我不会。iPod 最大的创新在于它的物理和软件界面。我不确定为这款设备设计的触摸面板或触摸滚轮是否足够。为了捍卫自己的领先地位，苹果必须重拾自己的优势。例如，我认为 iPod 有足够的空间变得更轻更薄，就像 MacBook Pro 为笔记本电脑所做的那样。它还需要变得更便于握持，并成为所有家庭娱乐的核心。与电视公司合作，整合 iPod 基座，轻松观看节目。从这个角度出发，尽一切努力建立在人们知道的、需要的和创新的基础上。从顶峰跌落是很容易的——问问索尼就知道了。

虽然我同意上面的一些观点(更薄更轻是好的，更大的屏幕可能不令人满意)，但我认为这个问题本身有些错位。苹果不需要在 iPod 上进行更多的创新——它需要推出下一个改变生活的小工具，让 iPod 保持原样:一个极其简单有效的音乐播放器。iPhone 和仍然模糊的 iTV 都很有可能成为它，特别是如果苹果利用它与谷歌的关系(谷歌的首席执行官最近加入了苹果的董事会)。这意味着可以访问 2006 年最大的科技现象之一:YouTube。

想想看:你已经可以在 LG 巧克力等支持威瑞森 V-Cast 的手机上观看 YouTube 剪辑，但这仍处于起步阶段；如果乔布斯能够绕过 YouTube 与威瑞森签订的手机服务独家合同，苹果可以通过提供与 YouTube 更紧密的手机集成来轻松进入并利用这一点。iTV 可以与 YouTube 建立真正美好的关系，而不仅仅是从 iTunes 上下载内容。

iPod 如何融入这个计划？一个更大屏幕的视频 iPod 可能就是你所需要的，但是每个人都在说的话(感谢像 Archos，MusicGremlin 和微软这样的公司)是无线的。如果 iPod 的下一代内置 WiFi，并运行 Mac OS 的缩小版——就像据报道将在 iPhone 上运行的版本一样——那么天空几乎是无限的。通过 iTunes 的嵌入式版本进行直接无线数字下载(还有 iTMS 订阅服务，拜托，哦，拜托)将是一个非常棒的开始。如果你也能像 YouTube 一样订阅一个完全没有数字版权管理的服务会怎么样？首先，会有更多的人在观看一个人被一个骑着三轮车的 3 岁小孩撞到屁股的最新片段时撞到电线杆上。

现在把我在上一段中说的都拿出来，揉成一团，扔进垃圾桶，因为这是不可能的(也许除了 iTMS 订阅服务)。乔布斯很聪明。他将把所有的 WiFi 好处留给 iPhone，让 iPod 少了无线功能，因为不管 iPhone 在播放音乐、获取移动内容和嗯……通话方面有多棒？…数百万人仍然愿意花大价钱购买一个简单且没有太多花里胡哨的专用音频设备。

当然，新功能对我们这些精通技术的人来说是很棒的，他们可以为 iPod 设想各种可能性，但事实是 iPod 首先是一种主流产品，这就是苹果公司如何谋生的。多年来，我一直恳求苹果公司加入像 A-B 重复这样的简单细节，他们坚持认为保持简单可以让玩家从货架上飞起来。我不能不同意。

我不是说 iPod 可能需要变薄和更多的屏幕空间，但坦白地说，它在大约四分之三的市场上没有被打破，所以它不需要修复。如果你想让它成为最简单、设计最好的音乐播放器，那么未来的迭代确实会令人失望。换句话说，iPod 不会因为缺少 WiFi、蓝牙等通信设备领域的功能而突然开始走下坡路。也就是说……直到每个手机运营商都提供 iphone，并且闪存足够便宜，可以以现实的价格购买 20GB 范围内的大容量机型。但是到那时，我们都会吵着要第一个皮下 MP3 播放器。
