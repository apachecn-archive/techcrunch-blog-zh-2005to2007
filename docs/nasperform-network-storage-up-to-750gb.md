# NASPerform 网络存储:高达 750GB 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/25/nasperform-network-storage-up-to-750gb/>

[![ownnasperform.jpg](img/e0caff1323cdfc60265daa61637f1677.png)](https://web.archive.org/web/20150117072904/http://old.crunchgear.com/wp-content/uploads/ownnasperform.jpg "ownnasperform.jpg")

在我的圆滑设置中，我缺少一个网络连接存储设备。另一个世界的计算机今天推出了 NASPerform，它可以与任何网络(尽管它必须与路由器物理连接)以及任何品牌和型号的计算机一起工作。它是一个 NAS(但你也可以把它插入 USB 2.0 端口，用作普通的外置硬盘)，所以没什么大不了的:网络上任何一台计算机都可以访问的巨型硬盘。

你可以花 60 美元买到不带硬盘的型号，或者花 300 美元买到 750GB 的。希望 NASPerform 和 OWC 的 RAM 一样好，我现在已经把它安装在我的 iMac *上了！*
