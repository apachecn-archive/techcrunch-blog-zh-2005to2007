# 地毯闹钟:来吧，踩吧

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/22/the-carpet-alarm-clock-go-ahead-step-on-it/>

[![carpetalarm.jpg](img/30dfa82f93af733ca38a51592b175c95.png)](https://web.archive.org/web/20150531011815/http://old.crunchgear.com/wp-content/uploads/carpetalarm.jpg "carpetalarm.jpg")

对于那些可怕的工作日早晨，这是垫闹钟。它采用了与[时钟](https://web.archive.org/web/20150531011815/http://crunchgear.com/2007/03/23/clocky-a-brief-video-review/)相似的策略:为了关掉它，你需要起床。一旦你踩下它，闹钟就会自动关闭，你就可以开始新的一天了。这似乎很容易生产，所以希望它真的会。
