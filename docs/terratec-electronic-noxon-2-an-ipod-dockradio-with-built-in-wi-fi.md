# TerraTec Electronic Noxon 2:一个内置 Wi-Fi 的 iPod 基座+收音机

[![noxon2.jpg](img/f08e906b0c522238b90ccd3bc8ee0029.png)](https://web.archive.org/web/20130628163418/http://old.crunchgear.com/wp-content/uploads/noxon2.jpg "noxon2.jpg")

我已经有一段时间没有写过任何种类的 iPod 扬声器了，但 TerraTec Electronic 的 Noxon 2 试图满足所有人的所有需求，这一点值得一提。这个小家伙有一个内置的 iPod 基座(咄！)、Wi-Fi 和 FM 收音机。它还有 USB 端口，所以你可以连接外部硬盘等。Wi-Fi 用于连接互联网广播流，尽管如果你很穷，负担不起 Wi-Fi，也有有线连接。

现在，抓紧你的袜子，因为这个坏男孩的价格可能会让你失去它们。我不骗你:465 美元。一个带内置基座和 Wi-Fi 的扬声器将近 500 美元。所以我想，如果你刚刚赢得了政府国防合同，需要一个新的 iPod 基座，这将是一个不错的选择。否则，我们中的一些人有账单要付。
