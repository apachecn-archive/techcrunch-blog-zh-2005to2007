# 在 Paltalk 9.0 上看到、听到和键入 Amanda Congdon

视频聊天信使 [![](img/3bdce64f0585aaea40362a7b760d368c.png)](https://web.archive.org/web/20221001155648/http://paltalk.com/) [PalTalk](https://web.archive.org/web/20221001155648/http://paltalk.com/) 在几位[名人](https://web.archive.org/web/20221001155648/http://www.paltalk.com/launch/)的帮助下，将在未来两天推出其新的 9.0 客户端，其中包括因 Rocketboom 而出名的博主 Ariana Huffington 和 Amanda Congdon。Paltalk 的 messenger 客户端拥有您所期望的所有基本 IM 聊天功能，包括与 AIM、Yahoo！，和 ICQ，但通过将视频和聊天室融入其中而脱颖而出。

![](img/9c9970d7cad440fa77db84acd3f1f85d.png)

新的 9.0 客户端具有他们所谓的“SuperIM”功能，在这里你可以与多达九个人进行实时视频和音频聊天。你在窗口顶部的一个长条上观看你朋友的网络摄像头的视频，如上图所示。从 strip 中，您可以打开并展开一个视频源，以便更仔细地查看。如此丰富的聊天体验可能会让大多数习惯进行多次对话的用户感到兴奋，但在窗口之间切换焦点和音频的能力有助于控制混乱。

如果你每月额外支付 4.99 美元(59.95 美元/年)购买最好的版本，Paltalk x-treme 的视频质量非常好。然而，基本版本只显示用户快照，而第二层版本显示视频的帧率约为 x-treme 的一半，但每年的费用为 39.95 美元。这里有一个很好的对比预览。

Paltalk 以从其 400 万用户(过去八年积累的)中创建社区而闻名。他们有数百个按主题分类的聊天室，如宗教、政治和技术，子类别进一步缩小了范围。像 Opie 和 Anthony 这样的广播节目在播出时会使用 Paltalk 向几千名粉丝广播，我希望他们的嘉宾阵容也会如此。

对于不习惯的用户来说，PakTalk 上的常规聊天室可能会过于刺激。经常同时有几个即时消息，管理员的房间里可以播放音乐。你可以通过静音、忽略用户、发送个人即时消息或举手讲话(类似于 Skype Live)来控制其中的一部分。如果事情真的失控，用户可以通过一个由 250 名志愿者组成的团队报告虐待行为，他们负责监管房间并对投诉做出回应。
