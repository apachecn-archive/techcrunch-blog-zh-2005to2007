# T-Mobile Dash 动手

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/11/t-mobile-dash-hands-on/>

随着智能手机逐渐成为主流，我们想起了为什么它们长期以来被放在出差族和不可救药的电子邮件迷的臀部皮套里。除了极少数例外，它们的操作系统对普通消费者来说过于强大，它们的设置屏幕令人费解，与普通翻盖手机相比，它们的价格也高得离谱。

T-Mobile 知道一些其他运营商显然不知道的事情。他们知道，电子邮件和即时消息形式的数据可能是向高中学生、足球妈妈和其他曾经认为他们需要一个 QWERTY 键盘的人销售 3G 服务的最佳方式之一。看看他们目前的产品 SideKick 3、Pearl 和现在的 Dash。这些手机直接面向那些对黑莓感兴趣，但不想让他们的办公室 IT 部门仅仅为了阅读几封电子邮件而参与进来的人。

![](img/3dd8d944feb47675dd9d77befff01d6b.png)

HTC 为 T-Mobile 生产的 Dash 是一款 Windows Mobile 5.0 智能手机，大约与摩托罗拉 Q 一样薄，包括许多用户友好的功能。完全披露:我不喜欢 Windows Mobile 5.0，但这种偏见不足以阻止我重新审视一款出色、纤薄的智能手机。

然而，走近一点看，你会惊讶地发现这款迷你手机内置了 Wi-Fi 和 130 万像素的摄像头。当你没有或不想使用 EDGE access 时，Wi-Fi 是一个救命稻草，相机非常好，可以拍摄出大多数手机只能梦想的清晰照片。再加上包含全面 T-Mobile 热点接入的数据计划，我们可能就有赢家了。

一个很棒的功能是 MyFaves。这是对电话的一个惊人的补充，使给妈妈、爸爸、妻子和我的赌注登记经纪人打电话变得非常简单。这款手机通过无线方式进行自我更新，并在主屏幕上显示我的音乐。因为所有与 MyFaves 的通话都是免费的，所以我们鼓励你多给奶奶打电话，前提是她还活着。

也就是说，如果你在家里、学校、工作中使用 Windows，请不要让我的抱怨阻止你考虑这款手机，比如说，一款没有智能功能的标准手机。即时消息和邮件功能非常容易使用，设置起来也很容易。PIM 功能总体上是可以接受的，对于 Outlook 来说是理想的。最后，这款手机纤薄、稳重，而且非常便于携带。
