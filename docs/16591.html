网络治安官DMCA对华纳音乐上传的YouTube内容的搜索

在线音乐推荐和分享网站 Scouta 收到了来自知识产权执法组织网络治安官的DMCA通知，要求YouTube上托管并在Scouta上显示的一段白色条纹视频因侵犯版权而从Scouta上删除。尽管视频是由华纳音乐上传到YouTube的，根据维基百科，白条乐队目前的唱片公司。

这个问题不同于TechDirt 报道的类似争议。由于YouTube上的内容的合法性受到质疑，在Scouta的情况下，视频是由权利持有人合法上传到YouTube 的，并授予视频嵌入第三方网站的全部权利。

Web Sheriff等公司的愚蠢显然与DMCA通知Scouta ( pdf )未能遵守该法案没有关系。

Web Sheriff未能按照DMCA通知中的要求演示在哪里可以查看原始内容；简单地说，他们无法证明原始内容的来源是侵权的，因为Scouta 显示原始内容是由权利持有人合法上传的。该通知要求Scouta“从侵权文件位置移除被侵权的标题”，尽管Scouta没有托管该文件(YouTube托管)。该法案明确规定，此类通知只应发送给那些托管侵权文件的人；如果内容确实违反了该法案(显然没有)，那么YouTube应该已经收到了通知。

像Web Sheriff这样的公司滥用DMCA凸显了法律的可悲失败。像Scouta和其他公司这样的初创公司不应该受到像这样的错误指控。美国立法者迫切需要将DMCA扔进它理应属于的历史领域。

对于那些喜欢白色条纹的人来说:有问题的视频，由华纳音乐和谷歌/ YouTube提供。
