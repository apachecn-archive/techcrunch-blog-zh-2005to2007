# 孢子开发商抨击任天堂

![](img/7af69b57d8dc93eb41f85bf4825a9b97.png)

既然布雷克显然正在 GDC 小睡片刻，看着[柯尔特](https://web.archive.org/web/20201202130921/http://www.edbaran.com/brickhouse/#Press%20Releases)的电影，我想让你注意一些事情。他们在 GDC 进行了一轮名为“燃烧的疯狂——一个游戏发行商的咆哮”的会谈基本上，一定数量的出版商和开发商可以大声抱怨他们讨厌的东西。当*孢子*团队的克里斯·赫克轮到他时，他就任天堂如何糟糕发表了一通疯狂的长篇大论。

克里斯宣称“Wii 是一坨屎！”还有那个任天堂应该做一个不“吸屁股”的主机。克里斯为什么这么沮丧？Wii 的处理能力不足以开发下一代游戏，任天堂只想制作“有趣”的游戏。你知道任天堂并不认为视频游戏是艺术吗？可怕！
