# 用 ClickTale | TechCrunch 观察互联网用户的行为

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/07/11/watch-internet-users%E2%80%99-behaviour-with-clicktale/?year=2006&monthnum=07&day=11&like=1>

总部位于以色列的 [ClickTale](https://web.archive.org/web/20120511005756/http://www.clicktale.com/) 提供了一项新服务，填补了在线统计服务领域的一个重要空白。这项服务目前处于私人测试阶段，但应该很快就会发布。在那之前，你可以在[ClickTale.com](https://web.archive.org/web/20120511005756/http://www.clicktale.com/)登记邀请，我们有机会获得独家私人预览。

如果你是一个网站所有者，了解你的用户是优化你的在线服务可用性的首要考虑。大多数统计服务，无论是以用户为中心(如 [NielsenNetRatings](https://web.archive.org/web/20120511005756/http://www.nielsen-netratings.com/) )还是以网络为中心(如 [HitWise](https://web.archive.org/web/20120511005756/http://www.hitwise.com/products-services/how-we-do-it.php) )或以站点为中心(如 [Omniture](https://web.archive.org/web/20120511005756/http://www.omniture.com/) 用于网站和 SiteMeter，Performancing，Mint 或 BlogBeat 用于博客)，都会生成关于页面浏览量、入站和出站链接、花费时间、访问者概况和来源以及最常点击的链接和其他参数的非常全面的报告。在某些情况下([眼球追踪](https://web.archive.org/web/20120511005756/http://www.eyetracking.com/)，谷歌分析和 MeasureMap)，它们甚至会生成最受关注或点击区域的“热区”。但是据我所知，没有人能够精确地解释互联网用户在给定网页上的动态行为(想想鼠标手势、动态浏览……)。

声称“每个用户都有一个故事”的 ClickTale 将通过向任何网站或博客所有者提供观看电影个人浏览会话的可能性来解决这一问题。

> 与只产生纯统计数据的传统网络分析不同，ClickTale 让网站管理员能够观看用户个人浏览会话的电影。鼠标的每一次移动，每一次点击，每一次击键都被记录下来，方便回放。有了 ClickTale，网站管理员可以提高网站的可用性，增强导航，并增加网站的有效性。

就像商店经理直观地监控他的顾客的购物习惯一样，ClickTale 让网站所有者能够观察他们的访问者的浏览习惯。

但是如果原始数据不能被分析和消化，那就没有那么多帮助了。尤其是高访问量的网站。我肯定会对前 50 个视频感到兴奋，但过一会儿就会很快厌倦，更糟糕的是，甚至无法从成千上万的视频会话中学习。因此，该公司还设计了统计工具，从所有这些丰富的数据中得出结论。

> 除了电影之外，ClickTale 还提供了一组独特的统计数据来解决重要的可用性问题。例如，“页面查看百分比”统计可以回答“用户查看了网页的多少内容，以及他们滚动到网页底部的频率如何？”“活跃浏览时间”统计可以回答“相对于只打开不活跃的浏览器，用户活跃地浏览网页多长时间？”还有很多很多…

在不久的将来，ClickTale 甚至将包括新的分析功能，这些功能将聚合数百万客户的记录，为他们提供独特的全球视角，并帮助您更好地了解用户的行为，以改进您的网站。

有趣的一点是，这项服务将面向所有人，而不仅仅是那些贪婪于用户数据的大公司。ClickTale 是一项托管服务，因此无需在服务器端或客户端进行安装，安装只需几分钟。网站管理员在他们的网页上添加一小段 javascript 代码。javascript 收集浏览数据并将其传输到 ClickTale 服务器进行处理。ClickTale 几乎可以即时创建浏览会话的电影，网站管理员可以随时安全地登录观看这些电影。这一点很重要:事实上，该技术是以网站为中心的，这意味着不是基于一组用户，而是基于网站的总活跃用户群。

隐私是 ClickTale 高度关注的问题。只有授权的网站人员才能观看他们网站的录像。网页之外的任何活动都不会被记录:没有个人文件，没有互联网历史，没有与本地安装的软件的交互，用户在网站之间不会被跟踪。在网页内部，密码永远不会被记录。

我认为 ClickTale 可以很好地补充现有的统计服务，因为它们将有助于理解动态行为，特别是如果它们能够以更容易理解的方式汇总原始数据。商业模式还不确定，但我们可以假设这项服务的一部分将会变得更好。关于他们的[博客](https://web.archive.org/web/20120511005756/http://blog.clicktale.com/)的更多信息。
