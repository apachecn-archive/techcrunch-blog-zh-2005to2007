# PlayStation 3 在 Ebay 上预订…再次

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/18/playsation-3-preorders-up-on-ebayagain/>

对于那些渴望海狸游戏的玩家和想要取悦他们孩子的父母来说，PlayStation 3 的预购又在 Ebay 上出现了。至少这一次，他们是合法的 T2。实际上，我有点惊讶地看到 PS3 预购最高售价为 1500 美元；我以为他们会出更高的价。不过，这可能只是因为离圣诞节还有两个多月。知道这一点也很好，即使你得到了预购，你可能不会在发售当天得到一台游戏机，因为供应是如此有限。我现在可以看到:父母对抗父母，游戏玩家对抗游戏玩家，易趣玩家对抗…嗯，其他易趣玩家，所有人都在为比别人先拥有一个 PlayStation 3 而高兴。我知道，有专门的球迷在那里已经等待了很长时间，这个控制台出来，但来吧…
