# 诺基亚 6086 通过 FCC  认证

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/05/nokia-6086-approved-by-fcc/>

![](img/f6a7120a4c554658f5960dada74e2544.png)

大约一周前，我们告诉过你关于诺基亚 6086 手机的事。最初，它计划于 2007 年初在欧洲发行。现在 6086 已经通过了联邦通信委员会的批准，将很快在美国上市。显然，T-Mobile 将首先获得这款手机，并将在 2007 年的某个时候以 200 美元左右的价格出售。

此外，尽管这款手机有 UMA 和使用 WiFi 作为保持通话连接的备用方法的能力，但其他功能，如仅仅是 VGA 摄像头，并不令人兴奋。乌玛？很性感。VGA？不是。
