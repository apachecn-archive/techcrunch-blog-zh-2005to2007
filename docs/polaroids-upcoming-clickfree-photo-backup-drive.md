# 宝丽来即将推出的免点击照片备份硬盘

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/28/polaroids-upcoming-clickfree-photo-backup-drive/>

这个月早些时候，我错过了它，但它是摄影爱好者特别感兴趣的一个非常酷的小驱动。这是一款即将推出的存储设备，由[宝丽来](https://web.archive.org/web/20160526205331/http://www.polaroid.com/us/index.jsp?bmLocale=en_US?bmUID=1167338691935)和[存储设备公司](https://web.archive.org/web/20160526205331/http://www.storage-corp.com/)合作开发。它被称为宝丽来媒体备份照片版，本质上是一个便携式 40GB 硬盘驱动器，旨在备份你的照片——而且只备份你的照片。我还没有测试过它，但是如果 129 美元的驱动器像承诺的那样工作，它绝对是我见过的对无知的人来说最好的技术。

Storage Appliance 具有内置于驱动器中的免点击技术，一旦通过 USB 连接到您的 Windows 2000 或 XP 计算机(Vista 支持将于 2007 年在 Q1 推出)，它就会搜索任何图像文件(支持 60 多种不同的文件类型)并自动备份它们。无需安装软件，无需完成设置过程。没什么。它只是找到它们并保存到硬盘上。即使它们在压缩文件夹中。

这个设备连电源按钮都没有。你唯一会发现的是顶部边缘和旁边的一个迷你 USB 端口，这是内置电源适配器的输入。而且如果你的电脑供电充足，就不用用适配器供电了；USB 电缆携带足够的电力来运行驱动器。

与任何好的备份设备一样，它不会在每次连接时进行完全覆盖。它使用单向增量备份，只保存新文件和那些已经在您的计算机上修改。如果您愿意，该硬盘的恢复功能可用于在另一台电脑上共享您的文件，同样无需安装任何软件。
