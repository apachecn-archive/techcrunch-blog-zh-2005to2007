# Vigor Gaming 的 AMD 四核 FX4 处理器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/20/vigor-gamings-amd-quad-fx4-powerhouse/>

没听说过活力博彩？也许你听说过它的[季风热电台式制冷系统](https://web.archive.org/web/20150711135315/http://www.vigorgaming.com/main/monsoon.html)？好吧，那么你只能相信我的话，然后活力建立一些顶级性能的台式机和笔记本电脑，如活力部队侦察 QX4。

Vigor Force Recon QX4 是为该公司所谓的“大任务”(在多线程游戏中并行运行多个应用程序)而制造的，由一对 AMD Athlon 64 FX-74 系列双核处理器提供支持，主板采用新的双插槽直接连接(DSDC)架构。基本系统的售价为 2899 美元，这很高，但对于这个庞然大物中的所有组件来说，这是一个不错的价格。

如果你要去一月份的 CES，一定要去看看 Vigor 的展位。
