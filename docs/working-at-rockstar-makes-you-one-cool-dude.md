# 在摇滚之星工作让你变得很酷

![](img/2a58306d44d660ba453cb415cc0a1c0a.png)

不是！但是看到所有你喜欢的公司的内部运作总是很有趣的。杰夫，曾经为 Rockstar Games 工作，最近发布了一份关于他在该公司工作时间的巨大宣言。这是一本充满有趣的珍闻的好书，尽管有些东西是可以预见的。以下是一段尖刻的摘录:

> 在那里工作也确实在某些圈子里买到了摇滚明星的待遇。游戏玩家都是极客，Rockstar 绝对是所有游戏公司中最嬉皮最“街”的，所以任何知道你在那里工作的游戏玩家都会把你当神一样看待。我们穿着摇滚明星夹克在游戏店里走来走去，就好像我们是某种直接来自勇士的穿制服的帮派。人们实际上认出了我们，并清出了一条路。

该死的兄弟！你是男人！16 岁的孩子和从未有过性行为的男人实际上为你扫清了道路？请允许我把你的名字写进我每日的礼拜册。我不会一直破坏你的故事。读一读，享受一下。
