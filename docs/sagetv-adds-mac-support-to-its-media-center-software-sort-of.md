# SageTV 为其媒体中心软件增加了 Mac 支持

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/18/sagetv-adds-mac-support-to-its-media-center-software-sort-of/>

SageTV 为 Windows 和 Linux PCs 开发了一个方便的媒体中心应用程序，它做了各种各样的好事，但主要是把你的计算机变成一个强大的 DVR。该公司制造并销售了一款名为 SageTV Placeshifter 的配套应用，用于接入媒体中心，从任何连接到互联网的计算机上控制和观看直播或录制的电视节目，很像一个[弹弓](https://web.archive.org/web/20140930210004/http://crunchgear.com/2006/09/27/slingmedia-drops-three-slingboxes-for-the-holidays/)。今天它宣布了一个 Mac OS X 的 Placeshifter 版本。

因此，如果你是那些拥有基于 Windows 或 Linux 的个人电脑来录制电视节目的人之一，你现在可以用你的 Mac OS X 电脑来访问它。假设你在用 SageTV 录制东西。你完全应该这样，现在你可以在三个不同的操作系统平台上访问这些东西。

Mac Placeshifter 将于本月上市，作为 SageTV Media Center 99.95 美元套装的一部分。额外的并发易地播放连接每个 29.99 美元。
