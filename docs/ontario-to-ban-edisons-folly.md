# 安大略省禁止爱迪生的愚蠢行为

安大略湖，一个生活和成长的地方，在 2012 年宣布白炽灯泡为非法，以鼓励全国范围的节能。该开关每年将节省 600 万兆瓦时。

该禁令不会影响特殊的使用情况，如医药，并将鼓励使用紧凑型荧光灯，比旧的白炽灯泡节省 75%的能源。
