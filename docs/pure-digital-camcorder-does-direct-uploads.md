# 纯数码摄像机可以直接上传

Pure Digital 升级了其廉价的街头便携式摄像机，让你可以通过三个步骤拍摄视频并上传到网上。在你兴奋之前，上传体验并不像你想象的那么直接。纯数字点&拍摄摄录机(30 分钟，129 美元；60 分钟，169 美元)现在有软件可以让你通过 USB 连接将视频从相机下载到 Windows 或 Mac 电脑上，并直接上传到共享网站 Google Video 或 Grouper Networks。捆绑的软件还可以一步完成电子邮件、视频贺卡和定制电影混音。这些相机将于 11 月上市。

我真正想知道的是，我离能够突然进入一个目标，CVS 或 RiteAid，以不到 200 美元的价格抓住一台相机，拍摄我的朋友做非法活动的视频，并在一个热点无线上传到视频共享网站还有多远？
