# YouTube 已经终结了嘻哈大战

[![youtuberakim.jpg](img/cfbb36446c74dd972a8d4a3fb3b88308.png)](https://web.archive.org/web/20220807164041/https://beta.techcrunch.com/wp-content/uploads/2007/07/youtuberakim.jpg "youtuberakim.jpg") 
你去了哪里？

像埃里克 B 和拉基姆这样的家伙，前 G-单位 Mobb 深和 Nas 在他决定调用它之前使嘻哈伟大。hip hop 文化的部分魅力在于 MCs 之间不断的争吵——牛肉。一个说唱歌手会“开始一些事情”，而另一个会以同样的方式回应，通常是在一些带有“dis track”的混合磁带上。不幸的是，YouTube，加上说唱歌手完全缺乏创造力，已经摧毁了备受庆祝的嘻哈大战。

聪明的俏皮话已经一去不复返了，除了延续宿怨之外，没有任何理由的互相攻击。现在剩下的只是 YouTube 上粗制滥造的咆哮，其中主持人吹嘘自己“很性感”或一些废话。你可能会认为他们会花时间写一两首诗，但事实并非如此。现在他们需要的只是廉价的网络摄像头、失业的亲信和宜家照明。这就像射频视频拍摄视频，只是夹杂着俏皮，MTV 批准的嘻哈方言。

埃里克·B 竞选总统。
