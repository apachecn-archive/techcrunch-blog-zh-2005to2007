# RebTel 上了索引| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/24/rebtel-gets-on-the-index/>

几周前，我在写了关于两家总部位于伦敦的新公司 [Truphone](https://web.archive.org/web/20140711095356/http://www.truphone.com/) 和 [ConnectMeAnywhere](https://web.archive.org/web/20140711095356/http://www.connectmeanywhere.com/) 的文章，这两家公司通过 VoIP 重定向本地电话，在你的手机上提供廉价的国际电话服务。我还简单提到了提供类似服务的瑞典公司 RebTel。

据商业 2.0 博客报道，“RebTel 刚刚从 Index Ventures 和 Benchmark Capital 的 [Danny Rimer](https://web.archive.org/web/20140711095356/http://www.indexventures.com/cgi-local/ivw_team_detail_xy.cgi?spgid=4) 那里筹集了 2000 万美元。”

虽然我不清楚这笔投资为何如此重要，但这笔投资确实表明，尽管 VoIP 市场开始充斥着许多模仿服务，但直到现在，风投们才开始押注于他们认为最终将成为 21 世纪下一家大型“数字”电话公司的公司。

当然，20 世纪的模拟(POTS)电话公司，如英国电信，希望他们能够通过(缓慢地)将 1800 万用户转移到基于 VoIP 的服务，如英国电信软电话和英国电信融合，来保持我们现有的通信提供商地位。

就我个人而言，我更喜欢在我的 T-Mobile MDA Pro 上使用 Skype 和 Skype mobile。例如，最近在法国时，我不仅收到了预期的免费 Skype 至 Skype 通话，还收到了来自英国的 Skype in-calls，也就是说，即使我在法国，人们也拨打我当地的 Skype 英国号码，并且只支付当地通话费用。同样，使用 Skype 也意味着我只需在电脑和移动设备上管理一个 Skype 联系人列表。

目前，我尽可能多地使用我的 Skype 手机，在任何可能的地方和时间拨打免费的 VoIP 电话，但由于家里或 hotposts 上的 wifi 接入有限，这种情况并不常见。但是每次我这样做，我或者节省了我的国际通话费，或者节省了我的本地通话费。

因此，我可以想象，一旦现在正在安装的大都市 wifi 网络变得无处不在，我将开始更频繁地使用我的 VoIP 电话来取代一些 GSM/GPRS 电话。

RebTel 当然不是 Danny Rimer 唯一的投资。他的另一项投资(1800 万美元)是 FON T1。FON 是一个网状网络，用户为了其他 [FON](https://web.archive.org/web/20140711095356/http://www.fon.com/) 成员的利益，免费分享他们的家庭/热点 wifi。如果(我对此表示怀疑)FON 能够设法创建一个全国性和/或国际性的免费网状网络，那么像 RebTel 这样的服务将变得非常有价值，这就是为什么我猜想 Danny 投资了 RebTel？

就我个人而言，我不确定这些公司中的任何一家，RebTel、CMA、TruPhone 或 FON，最终会成为下一个 21 世纪(纯数字)电话公司。但据我所知，丹尼是 Skype 的早期投资者之一，所以也许他能看到一些我在 RebTel 和 FON 中看不到的东西，但谢天谢地，我不是唯一一个[表达我的担忧和疑虑的人。](https://web.archive.org/web/20140711095356/http://www.techcrunch.com/2006/09/23/voip-the-details-kill-the-fun/)
