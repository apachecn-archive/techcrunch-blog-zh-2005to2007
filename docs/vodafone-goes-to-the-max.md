# 沃达丰走到最大

![vodafone2.jpg](img/450ae3389bbe895a1f9e8218458b4cd2.png)今天，沃达丰宣布成为 WiMAX 论坛的主要成员，这是一个致力于推广和认证互操作 WiMAX 产品的非营利组织。这项技术允许移动设备在比以前的技术(如 Wi-Fi)更远的距离上进行高速互联网接入。

沃达丰全球首席技术官 Steve Pusey 表示:“我们的 WiMAX 论坛成员身份将补充我们现有的其他主要行业团体成员身份，如 GSMA、3GPP 和下一代移动网络计划。“就我们未来的网络选择而言，沃达丰保持技术中立，加入 WiMAX 论坛是我们寻求评估该技术的全部能力和潜力的合理步骤。”

WiMAX 论坛目前由 470 多家公司组成，包括 141 家运营商。
