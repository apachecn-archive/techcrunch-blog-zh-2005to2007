# 为天生寒冷的人准备的微型桌上加热器

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/20/tiny-desk-heater-for-the-inherently-frigid/>

这个小加热器不是一个新的产品，但我想提一下，因为它太棒了。我不太喜欢 Sharper Image 的产品，因为它们看起来总是有点劣质。然而，我的手变得很冷，由于手指僵硬，打字成了一件苦差事。

Sharper Image 称其为[世界上最小的大型加热器](https://web.archive.org/web/20151205143227/http://www.sharperimage.com/us/en/catalog/productdetails/sku__VX351)，虽然我还没有对小型加热器市场进行全面调查，但我不得不说它至少在名单上名列前茅。这款 150 瓦的加热器尺寸为 4.75×3.75×3.5 英寸，重约 1 磅，可以插入标准的美国插座。它有高低两种设置，也可以作为一个小风扇使用。

当暖气打开时，垂直可调通风口周围的蓝色 LED 指示灯会亮起，如果你打翻它，它会自动关闭。40 美元还不错。
