# 来自海盗湾未经审查的免费图片托管

众所周知且经常引起争议的 BitTorrent 追踪网站[海盗湾](https://web.archive.org/web/20221205091528/http://www.beta.techcrunch.com/tag/thepiratebay)推出了 [BayImg](https://web.archive.org/web/20221205091528/http://bayimg.com/) ，这是一项未经审查的免费图片托管服务。

海盗湾此前一直在对一个视频托管网站进行私人测试，该网站尚未推出。BayImg 现在可以提供高达 100mb 的托管空间，并承诺只要图像合法，他们就不会审查它，尽管有趣的是“合法”并没有得到解释。“我不同意你说的话，但我誓死捍卫你说话的权利”也被错误地归因于伏尔泰(这个短语是由后来的一位作者发明的，作为伏尔泰对言论自由态度的缩影)，然而，尽管法律上有问号，但无论如何，任何东西被海盗湾团队审查的可能性都接近于零。

希望上传文件的用户不需要注册，据说 BayImg 支持 140 种不同的文件格式。

BayImg 直接与 CNet 的 AllYouCanUpload 竞争，但有一个明显的优势:DMCA 对 BayImg 的请求将没有地位。
