# 溢出 2.0 是一个干净，简单的启动器

[](https://web.archive.org/web/20221129043629/http://www.stuntsoftware.com/)Mac launcher[Overflow](https://web.archive.org/web/20221129043629/http://www.stuntsoftware.com/Overflow/)2.0 版本今天发布了，真的很喜欢。启动器应该有助于平衡一方面让文件和应用程序易于访问的需求，另一方面让您的计算机保持有序的需求。我发现最流行的 Mac 启动程序 [Quicksilver](https://web.archive.org/web/20221129043629/http://quicksilver.blacktree.com/) 比我的需求更麻烦。一方面，溢出不太依赖击键。

这个应用程序的新版本本质上是第一个真正可用的版本。你可以拖放文件到溢出，创建多个类别或页面，并使用热键启动托盘。你可以把它放在屏幕上任何你想放的地方，每次你打开它的时候，它都会在相同的位置打开。

溢出需要 15 美元，但开始时可以免费使用。有些人抱怨钱太多了，但是温尼伯的特技软件应该为他们的工作得到奖励。像这样购买软件的好处在于它非常简单——我可以试用它，如果我喜欢它，那么 100%的钱都会给开发者。任何人都可以使用分销渠道(我通过 [Mac 更新](https://web.archive.org/web/20221129043629/http://www.macupdate.com/info.php/id/14815#changesContainer_link)发现了这一点)，这些渠道运行广告。这与在网上购买音乐有很大不同。如果他们的工作被证明是有用的，我会觉得不好意思不付钱给这样的公司；当谈到主要标签的音乐时，情况肯定不是这样。

如果其他 Mac 发射器对您不起作用，请尝试溢出。你可能真的会欣赏它的简单和有用。
