# 抱歉，加拿大，没有 iPhone 给你(可能):怪罗杰斯吧

[![iphonecanada1.jpg](img/af5b7def8da255a78afc77744032f1e3.png)](https://web.archive.org/web/20210304202436/https://beta.techcrunch.com/wp-content/uploads/2007/04/iphonecanada1.jpg "iphonecanada1.jpg")

忘记你的手在 iPhone，加拿大，因为罗杰斯，你唯一的 GSM 供应商，刚刚说，任何和所有的信息，iPhone 的可用性在其网络上是纯粹的猜测。(请记住，iPhone 是一款仅支持 GSM 的设备。)不仅如此，分析师还表示，如果不能证明 iPhone 具有成本效益，罗杰斯甚至可能根本不会携带 iPhone。这就是当你有一个全民医疗保健系统时会发生的事情。
 但是，对罗杰斯来说，拒绝发布(可能是)有史以来最重要的手机有多现实？我们已经知道，很多美国人一直在为知道他们什么时候能买一台而烦恼。天知道如果人们拿不到他们珍贵的 iPhone 会发生什么。加拿大，祝你好运，一定要感谢罗杰斯坚持“顾客永远是对的”这句话
