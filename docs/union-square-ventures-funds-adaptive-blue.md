# Union Square Ventures 资助 Adaptive Blue——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/02/20/union-square-ventures-funds-adaptive-blue/>

# 联合广场风险投资基金适应性蓝

语义网 Firefox 插件提供商 Adaptive Blue 今天宣布，该公司已经从 Union Square Ventures 获得了一笔未披露金额的资金。该公司的产品名为[蓝色组织者](https://web.archive.org/web/20220119222911/http://www.adaptiveblue.com/)，是一款标签和搜索工具，具有令人难以置信的一系列功能，专注于解析与之交互的网页的语义。Union Square Ventures 最出名的是在收购雅虎之前对 Del.icio.us 的投资。收购。该公司还投资了 Feedburner。

Blue Organizer 本周也推出了一系列实质性的功能升级；11 月我在这里回顾了产品的前一个版本。

Adaptive Blue 是一家四人公司，成立于一年前，由创始人亚历克斯·伊斯科尔德自筹资金。当蓝色组织者被选为 Firefox 2.0 附加页面的官方推荐时，该公司迎来了重大突破。用户下载量从上市前的 2 万增长到 11 月的 13 万，再到今天的 34 万。有趣的是，自 Firefox 最新版本发布以来，增长率一直保持不变。

为了更好地了解 Blue Organizer 的强大功能，我建议要么尝试一下浏览器插件，要么阅读我之前对它的评论。下面是一个截图，让你体验一下当我在一个包含音乐评论的网页上使用 Blue Organizer 时的样子。该产品自动确定我正在阅读关于音乐的内容，以及哪些搜索选项最相关。这非常令人印象深刻，但我个人对网络的使用足够集中在一个主题上，它的主题辨别能力和垂直搜索对我来说不太相关，因为它们可能对那些在网上阅读葡萄酒和书籍的更随意的网络用户来说更重要。
