# 威瑞森将关闭 Amp-d 手机？

![](img/12e0389e8617d310a1a1e44b29f0f5ab.png)

随着[Amp d Mobile 申请第 11 章保护](https://web.archive.org/web/20201028054217/http://crunchgear.com/2007/06/03/ampd-mobile-goes-bankrupt-claims-theyre-getting-too-big/)并欠威瑞森大量现金，对垂死的 MVNO 来说，未来的事情不会变得更糟。但我认为威瑞森威胁关闭你的网络，有效地减缓你的业务到磨停止，是有点糟糕。威瑞森最近要求 Amp'd 在 10 天内支付 450 万美元，否则将面临关闭。当你的公司没有 450 万美元来偿还债务时，你知道离倒闭还有几个月。

威瑞森似乎厌倦了扮演讨债人的角色，所以在接下来的几天里，Amp 的客户可能会看到一个可能的关闭。最初是什么卷入了这场混乱？许多麻烦都归咎于签订了 18 个月合同的不付款的拖欠客户。真可惜。也许 Amp 'd 只是不适合 MVNO 世界。
