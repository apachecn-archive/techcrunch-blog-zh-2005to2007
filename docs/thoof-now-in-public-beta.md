# Thoof 现已公开测试| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/06/27/thoof-now-in-public-beta/>

[![](img/711731b0fd7f1545df5dd7ddaa5d0a35.png)](https://web.archive.org/web/20160422000407/http://www.thoof.com/) 社交新闻网站 [Thoof](https://web.archive.org/web/20160422000407/http://www.crunchbase.com/company/thoof) ，大约两周前由私人发起，只接受邀请，现在对任何人开放。[我们的发射台在这里](https://web.archive.org/web/20160422000407/http://www.techcrunch.com/2007/06/15/revver-founder-launches-thoof-personalized-news-service/)。

Thoof 是关于个性化新闻的，并融入了维基、Digg/del.icio.us 投票功能的元素。用户将故事作为新闻条目的链接，连同标题、描述和标签一起提交给服务。如果 Thoof 确定其他用户会喜欢这条新闻，他们就会开始看到这条新闻。然而，提交的内容可以很容易地被认为有所欠缺的其他用户编辑。
