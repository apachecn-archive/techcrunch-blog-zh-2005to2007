# Sandio 3D 游戏 O' Mouse 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/31/sandio-3d-game-o-mouse/>

3D 输入技术公司 Sandio 已经开始发售其 [3D 游戏 O' mouse](https://web.archive.org/web/20230322164155/http://www.sandiotech.com/index.php?id=41) 。该鼠标拥有独特的 6 自由度 3D 输入技术，使其适用于 3D 应用。

激光鼠标具有高达 2000 dpi 的分辨率，当 dpi 变化时，它会发出声音。不过，这个设备的关键是它独立的 3D 6DOF 命令生成。

3D 6DOF 技术允许用户在鼠标上进行额外的 3D 控制。举个例子，在谷歌地球中，人们可以操纵相机。然而，在第一人称射击游戏中，pad 可以用于移动控制，如前进、后退、侧身、跳跃、下蹲、俯仰、偏转和滚动，所有这些都不需要使用键盘。

玩了多年的雷神之锤竞技游戏，我可以比一些花哨的新控制器更容易地支持这样的想法。我知道一些玩家在他们的鼠标上做跳跃和行走的动作。有一个专门用于运动的垫子会很有效。

3D 游戏 O '现在售价 79.99 美元
