# 红色 iPod Nano 明天上市？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/12/red-ipod-nano-coming-tomorrow/>

![](img/ceaba923a5f20a817fb17d504f405f9b.png)

iLounge 报道称，博诺本人将于明天在奥普拉·温弗瑞秀上宣布一款新的红色 iPod nano。红色 iPod 将成为“红色产品”( T2)项目的一部分，该项目目前正在将城市涂成红色以抗击艾滋病。这一集显然是本周早些时候录制的，主要是波诺和 4GB 的“iPod nano RED”它的价格应该是同样的 199 美元，其中 10 美元将捐给全球基金。我不怀疑这个谣言的真实性，因为为什么*不会发生在*呢？
