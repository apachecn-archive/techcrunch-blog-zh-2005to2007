# 环球拥抱互联网，DRM

这是注定要发生的，大唱片公司开始看到光明了。环球公司今天宣布计划通过一项名为 [SpiralFrog](https://web.archive.org/web/20201129165545/http://www.spiralfrog.com/) 的创业服务免费提供其全部音乐目录。该项目完全由 ads 支持。

[SpiralFrog](https://web.archive.org/web/20201129165545/https://crunchbase.com/organization/spiralfrog) 的用户将能够加载其专有软件，并在 Windows Media Player 和两个便携式设备上播放通用歌曲，iPod 不支持。然而，问题是用户必须每月至少登录一次网站，否则就会被广告淹没。如果用户每月不登录一次，受 DRM 保护的文件将停止工作。我不相信广告足以维持这个项目，但很高兴至少看到一些前进的势头。
