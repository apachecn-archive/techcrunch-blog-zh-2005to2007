# 两份新的购物清单，目标不同

总部位于英国的 [Gifttagging](https://web.archive.org/web/20220817233220/http://www.gifttagging.com/) 刚刚推出，总部位于佛罗里达州的 [MyPickList](https://web.archive.org/web/20220817233220/http://www.mypicklist.com/) 将在两周后推出私人测试版。这些是类似的产品(产品书签)，但功能不同。Gifttagging 关注的是一个愿望清单(想想亚马逊愿望清单)，而 MyPickList 关注的是根据产品推荐赚取现金。

杰里米·贝恩斯(Jeremy Baines)和他的团队创建的[](https://web.archive.org/web/20220817233220/http://www.gifttagging.com/)Gifttagging.com 是亚马逊愿望清单(Amazon Wish List)和 del.icio.us 的结合。使用他们的 bookmarklet 或 firefox 扩展将产品标记为书签，并将其放在你的“礼物清单”或“我有清单”中。额外功能:你可以导入你的亚马逊愿望清单，并以此开始。许多功能与 [Kaboodle](https://web.archive.org/web/20220817233220/http://www.beta.techcrunch.com/2005/10/25/kaboodle-launch-bookmarking-wiki/) 相似，尽管这是一个高度专业化的网站，只专注于礼物。我喜欢。

Jeff Eichel 的 MyPickList 更容易理解为雅虎购物圈，这是一项通过向他人推荐他们喜欢的产品来帮助人们赚钱的服务。我已经在这里发布了该服务的截图[。](https://web.archive.org/web/20220817233220/http://www.flickr.com/photos/michaelarrington/tags/mypicklist/)

通过书签功能，收集物品将非常有用。您的名单将通过 RSS 发布在 mypicklist.com 网站上。但更有趣的是，MyPickList 将向用户提供工具，在他们自己的博客和网站上推广这些列表。用户可以推广包含他们正在推广的商品的小部件，或者用这些商品创建自定义横幅广告。

任何对这些物品的购买都会为用户产生金钱。MyPickList 正在与在线商家建立联盟关系。推广商品的用户将获得“大约”40%的代销商收入，剩下的归 MyPickList 所有。一个很好的特性是用户不需要直接注册会员关系。MyPickList 将负责记账并向发布者签发支票。

对于许多博主来说，MyPickList 将是一个通过推荐各种产品赚钱的好方法。例如，我可以想象，数百万热情的 MySpace 用户把他们最喜欢的音乐和电影放在我的选择列表上。皮特·凯什摩尔称之为“一个非常聪明的想法”。我同意。只不过他们需要将用户的收入分成提高到 85%左右。
