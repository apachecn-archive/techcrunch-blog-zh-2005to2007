# TagWorld 推出音乐。战争开始了。TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/12/12/tagworld-launches-music-the-war-begins/>

上周末 [TagWorld](https://web.archive.org/web/20221208220853/http://www.tagworld.com/) ，一家在圣莫尼卡成立一个月的年轻的自筹资金创业公司[推出了](https://web.archive.org/web/20221208220853/http://www.tagworld.com/-/Main/articles.aspx)一款吸引眼球的音乐产品。

TagWorld 的目标是 MySpace 的用户群——通常是 25 岁以下的年轻人，他们都希望在网上有自己的博客/主页。像 MySpace、 [FaceBook](https://web.archive.org/web/20221208220853/http://www.beta.techcrunch.com/2005/11/09/facebook-users-are-passionate/) 和 Xanga 这样的网站正在产生大量的页面浏览量——MySpace 和 FaceBook 在页面浏览量上都与谷歌不相上下(尽管它们还没有达到这个水平)。原因？这些用户每天大部分时间都在这些网站上，更新他们的网站，点击朋友。

TagWorld [已经有了令人印象深刻的特性集](https://web.archive.org/web/20221208220853/http://www.beta.techcrunch.com/2005/11/11/a-profile-of-tagworld/)，包括 1gb 的存储空间和用于创建网站的优秀 ajax 工具。顾名思义，标记一切。

新的音乐功能比 MySpace 和其他公司领先整整一代。乐队可以上传 1gb 的音乐，并有许多 DRM 选项来控制发行。用户可以将这些音乐混合到自己的媒体播放器中，也可以上传自己的音乐(和其他文件，如照片和电影)。他们可以创建定制的音乐播放列表，在 TagWorld 上存储音乐，并在任何可以上网的地方访问音乐。用户还可以设置播放列表，以便其他人在访问用户网站时可以收听。所有这一切只需要点击音乐并将音乐播放器拖到他们的网站上。不需要 HTML 或其他技术技能。

TagWorld 能否挑战 MySpace，成为[社交网络 3.0](https://web.archive.org/web/20221208220853/http://www.ventureblog.com/articles/indiv/2005/001226.html) 之王？

是的，我想他们会的。原因有很多。

首先，创始人弗雷德·克鲁格(Fred Krueger)和埃文·里夫金(Evan Rifkin)在让顶级乐队开始使用该平台方面做了大量工作。查看标签世界页面[死亡出租车](https://web.archive.org/web/20221208220853/http://www.tagworld.com/deathcabforcutie)、[小腿](https://web.archive.org/web/20221208220853/http://www.tagworld.com/theshins)和[邮政服务](https://web.archive.org/web/20221208220853/http://www.tagworld.com/thepostalservice)。

第二，TagWorld 上线仅一个月就有 160，000+用户。“时尚”人群开始注意到了。

第三，我认为青少年/学生社交网络不像 ebay 那样受到限制。TagWorld 的目标用户变化无常，不想被认为是主流。他们会尝试新事物，如果功能已经存在，他们会留下来。随着儿童成为青少年(新客户)和年轻人转向更专业的工具(客户离开)，这个目标市场不断更新自己。

所以，简而言之，我看好 TagWorld。坦率地说，其中一些功能吸引了比青少年和年轻人更多的观众。带有便携式播放器的在线音乐储物柜市场非常开放。TagWorld 可以占领这个市场。
