# Retrevo.com:小工具、电子产品的搜索引擎

[![retrevo.jpg](img/f26e2b7905ea47e444b232431c433123.png)](https://web.archive.org/web/20210228075611/http://crunchgear.com/?attachment_id=5737 "retrevo.jpg") 
一个专门用于查找小工具和消费电子产品信息的搜索引擎？我恋爱了。

Retrevo.com 的搜索集中在关于你要找的产品的评论和文章上，而忽略所有其他无关的信息(店面、广告、易贝网页等)。在页面的左侧，你会看到一组标签，可以点击进入，比如评论、购物网站或产品文档。(曾经需要从一本遗失已久的说明书中挖掘信息吗？)

它现在在“伽玛”中，所以有些东西仍然有点小故障。但总的来说，这绝对是一个值得你下次去搜索新技术信息的书签网站。
