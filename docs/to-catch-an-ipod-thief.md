# 去抓一个偷 iPod 的贼

![](img/8df059b5aa2ba01f39ca211d6bbeb870.png)

虽然[网站](https://web.archive.org/web/20130628175315/http://protect.gadgettheft.com/)已经被盗，但一项名为“小工具盗窃”的免费在线服务最近推出，帮助用户找回被盗的小工具。

该网站为用户提供了一个小型的“间谍软件”应用程序，可以加载到 USB 拇指驱动器、MP3 播放器或数码相机的内存中。如果检测到设备被盗，软件会记录并向用户发送 IP 数据。该应用程序使用了一个聪明的社交工程，当连接到电脑时，会显示一个弹出窗口，要求安装 USB 驱动程序。

小工具盗窃不会帮助找到设备，要靠用户说服执法部门使用数据来找到物品，但至少这是一个开始。

遗憾的是，这款“间谍软件”只能在基于 Windows 的电脑上运行，所以苹果电脑窃贼仍然会逃脱。
