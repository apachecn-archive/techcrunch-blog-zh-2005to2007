# 快照图片搜索

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/12/19/snap-image-search/>

搜索引擎 Snap 今天推出了[图片搜索](https://web.archive.org/web/20150923155910/http://images.snap.com/search.php?query=flower)功能。结果并不比大公司提供的好，但 Snap 独特的两列界面让一些用户觉得很酷。缺乏可用的照片元数据来帮助查询的问题继续困扰着所有的搜索引擎，当然，导致不太好的结果。看看一些新推出的服务，如[极地玫瑰](https://web.archive.org/web/20150923155910/http://www.techcrunch.com/2006/12/19/polar-rose-europes-entrant-into-facial-recognition/)，是否可以通过与他们合作并利用用户生成的描述性数据来提供帮助，这将是一件有趣的事情。更多关于 Snap [这里](https://web.archive.org/web/20150923155910/http://www.techcrunch.com/tag/snap)。
