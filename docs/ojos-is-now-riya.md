# Ojos 现在是 Riya 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/10/20/ojos-is-now-riya/>

从未拥有过 ojos.com(他们使用 ojos-inc.com)的奥霍斯找到了一个永久的名字——[日亚](https://web.archive.org/web/20230218074923/http://www.riya.com/)。更多信息请访问蒙加尔·沙阿的博客和塔拉·亨特的博客。

Riya 拥有面部和文本识别技术，可以自动标记照片中的人。我们[在早些时候](https://web.archive.org/web/20230218074923/https://techcrunch.com/2005/08/31/ojos-auto-name-tag-your-photos/)对他们进行了剖析。

Riya 正准备开始他们的私人阿尔法，并希望在几个月内全面推出。如果你有兴趣参加阿尔法，电子邮件 alpha@riya.com。在阿尔法期间，他们将只支持 IE6。
