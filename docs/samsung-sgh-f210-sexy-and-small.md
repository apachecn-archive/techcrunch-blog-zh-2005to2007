# 三星 SGH-F210:性感小巧

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/25/samsung-sgh-f210-sexy-and-small/>

![](img/496edf616a012a9e4452f43eecb63280.png)

虽然我们可能要过几百万年才能看到这款三星手机登陆美国，但它仍然是一款设计激进的酷炫手机。SGH-F210 本周在亚洲亮相，反响非凡。它看起来有点像市场上新的索尼随身听 MP3 播放器，除了更坏。这个小封装中的功能包括三频 GSM，一个 200 万像素的摄像头(漂亮！)、262k 色 1.46 英寸液晶显示屏、支持 MP3/ACC/ACC+/eACC+/WMA 的媒体播放器，以及 1GB 带 microSD-slot 的内部内存用于扩展。

哦，对了，在他们敲定交易之前，蓝牙不知怎么就被塞进了包装里。我不明白蓝牙和 200 万像素的摄像头是如何被塞进这么小的手机里的，但这肯定是三星的功劳。在跳跃后查看一张图片，看看显示屏如何滑动，以显示一个用于拨号的小数字小键盘。
