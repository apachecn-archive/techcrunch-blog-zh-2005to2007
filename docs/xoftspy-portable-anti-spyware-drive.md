# XOFTspy 便携式反间谍驱动器

![sub_u3.jpg](img/7841e0dae2b18faca27eac57d38e1799.png)

任何一个称职的世界旅行者都知道要带一些重要的东西——毛巾、旅游指南和防护用品。这就是为什么 ParetoLogic 创造了 XOFTspy，这是一个保护你在路上免受间谍软件攻击的 USB 密钥。只需将它放入一个容器中，你就可以在瞬间完成即时抗疱疹清洗。

该计划可以检测和杀死大多数恶意和间谍软件应用程序，甚至有一个企业使用模型:

> IT 管理员现在可以向员工发放 U3u 盘，确信他们不会将间谍软件带入公司网络

相当巧妙，但如果要成功，最好有某种 USB 调制方式。想象一下，告诉你的员工只使用这个花哨的 XOFTspy 密钥，而不是他们在 Circuit City 购买的那个很酷的 2GB 密钥。
