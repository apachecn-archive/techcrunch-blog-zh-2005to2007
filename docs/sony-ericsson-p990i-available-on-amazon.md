> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/20/sony-ericsson-p990i-available-on-amazon/>

当然，这需要 649.99 美元，但这难道不值得吗？没有吗？是的，可能不是，但它仍然是一个非常性感的野兽。

P990i 具有三频 GSM/GPRS、802.11b WiFi、蓝牙 2.0 和 FM 调谐器。它也有 UMTS，但它支持的频率仅在欧洲可用。你还会得到一个 240 x 320 260K 的彩色 TFT 触摸屏和两个摄像头。正如你所看到的，一个是用于视频会议的，而背面的一个有 2MP，自动对焦，2.5 倍变焦和闪光灯。

它运行的是漂亮稳定的 Symbian OS 9.1，所以你不必担心 Windows Mobile 在通话中崩溃。
