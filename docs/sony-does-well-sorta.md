# 索尼表现不错，算是吧

![](img/489583259c55518bbf9f96b18a401198.png)

当我第一次购买索尼 PSP 时，我只想在上面玩经典的 PSOne 游戏。不用说，我很失望，因为像《最终幻想 7》《T2》和《异星齿轮》《T4》这样的游戏从未被移植到 PSP 上。幸运的是，索尼已经采取措施改变这种状况，允许用户从在线 PlayStation 商店下载 PSOne 游戏。现在你可以下载*铁拳 2* 、*虹吸过滤器*、*速成橡皮筋*、*酷玩玩家*和*热拍高尔夫 2* 。

索尼表示，本月晚些时候将会有更多的游戏上市，并会继续扩大 PSOne 图书馆的规模。这些游戏也相当便宜，每款 5.99 美元。坏消息是，你需要一个 PlayStation 3 来访问在线 PlayStation 商店，所以如果你没有，你就不走运了。
