# 欧盟降低漫游费

本周，欧盟国家之间的通话可能会变得便宜一点。根据新的欧盟规则，电信公司必须在 7 月底之前(和今天一样)向客户提供新的定价结构，这意味着更便宜的“漫游”费。用户有两个月的时间来表明他们是想转向新的计划还是坚持现有的合同。

欧洲委员会计算出移动电话运营商每年从漫游费中获利约 85 亿欧元！让你想永远呆在网络里，不是吗！
