# 出售 Napster 但这不是一家健康的公司

 [](https://web.archive.org/web/20220818112721/http://www.napster.com/) 音乐下载和订阅服务公司 Napster [宣布](https://web.archive.org/web/20220818112721/http://biz.yahoo.com/prnews/060918/nym264.html?.v=15)他们已经聘请了一家投资银行来帮助他们在今天早些时候出售公司。这一举措是*“为了回应最近第三方对建立战略合作伙伴关系或潜在收购该公司的兴趣。”*

乍一看，该公司看起来非常健康，年收入超过 1 亿美元，另有 1 亿美元现金。然而，问题是他们的业务利润极低。上个财政季度，该公司运营亏损近 1000 万美元(折合年率 4000 万美元)。在这个非常拥挤的音乐市场，让 Napster 盈利是不可能的。关于 Napster 竞争对手的更多信息，请看我们最近对音乐[下载服务](https://web.archive.org/web/20220818112721/http://www.beta.techcrunch.com/2006/04/07/lets-buy-some-music-part-1/)和音乐[订阅服务](https://web.archive.org/web/20220818112721/http://www.beta.techcrunch.com/2006/04/11/subscription-music-services-compared-part-2/)的分析。Napster 在这两个类别中都不提供最好或最便宜的产品。

不过，这并不意味着 Napster 不会卖很多钱。一家好的投资银行基于恐惧和贪婪进行销售。许多玩家害怕被排除在正在发生的音乐革命之外。Napster 可能正是他们正在寻找的。举一个最近的例子，看看蒙哥马利证券公司最近以 6500 万美元的现金将 Grouper 出售给索尼，这是没有人预料到的事情。
