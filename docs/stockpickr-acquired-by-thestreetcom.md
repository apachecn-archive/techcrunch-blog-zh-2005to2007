# TheStreet.com 收购 Stockpickr

一月份 [](https://web.archive.org/web/20221002235059/http://www.stockpickr.com/) 斯托克宣布与 TheStreet.com 达成投资股权和合伙协议。看起来双方都喜欢这笔交易；今天晚上，TheStreet.com 宣布他们收购了这家公司。交易条款没有披露。

Stockpickr 是我们一直在跟踪的金融市场领域为数不多的新创公司之一。参见我们对 [SeekingAlpha](https://web.archive.org/web/20221002235059/http://www.beta.techcrunch.com/tag/seekingalpha) 、 [Zecco](https://web.archive.org/web/20221002235059/http://www.beta.techcrunch.com/2006/09/22/3014/) 、 [Motley Fool](https://web.archive.org/web/20221002235059/http://www.beta.techcrunch.com/2006/10/05/caps-takes-wisdom-of-the-few-to-stock-picking/) 的报道。
