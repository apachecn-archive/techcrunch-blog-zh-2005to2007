# Skype 将提供游戏连接| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/17/skype-to-offer-gaming-connectivity/>

![](img/f25f9039981c8fb3eebfb6b26238f1da.png)

Skype 宣布将为其新的 GameXN 框架提供软件开发工具包(SDK)。这项计划是为了让 Skype 用户能够邀请其他人加入在线游戏。目前还没有消息表明 Skype 是否计划允许用户在比赛期间通过网络电话进行交流，但我可以想象它正在进行中。Windows Live Messenger 已经有了一些类似的功能，但是很多人都使用 Skype，而且它是跨平台的，所以它可能在这里有所发现。
