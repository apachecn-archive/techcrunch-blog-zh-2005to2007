# 更多实时硬盘在线存储谣言的时间到了

微软[即将推出的在线存储产品](https://web.archive.org/web/20210125150003/http://www.beta.techcrunch.com/2006/08/16/windows-live-drive-info-quickly-deleted-post-from-team-member/)，将被称为 Windows Live Folders(之前称为 Live Drive)，向发布又迈进了一步。我们已经[跟踪](https://web.archive.org/web/20210125150003/http://www.beta.techcrunch.com/2006/04/20/microsoft-live-drive-may-launch-before-google-drive/)项目大约一年了。当网站启动时，在 folders.live.com 寻找它。

更多的[细节](https://web.archive.org/web/20210125150003/http://liveside.net/blogs/main/archive/2007/05/12/windows-live-folders-beta-review.aspx)，包括截图，都在 LiveSide，它以某种方式(短暂地)进入了网站。目前没有太多需要回顾的内容，该服务显然还处于测试阶段。此时，该服务允许用户上传文件，并将其保存在公共、私人或共享文件夹中。总存储限制为 500 MB，并且没有桌面上传程序来协助文件管理。

LiveSide 将其描述为“非常基本的服务”，我同意。它目前的功能比我们跟踪的一些初创公司要少，比如 Box.net。
