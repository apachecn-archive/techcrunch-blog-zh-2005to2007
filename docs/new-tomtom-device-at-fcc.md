# FCC | TechCrunch 上的新 TomTom 设备

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/28/new-tomtom-device-at-fcc/>

除了几张图片外，没有关于这个的真实信息。我们知道这将是一个全球定位系统(duh 这是 TomTom)，标清，蓝牙拨号和语音短信和界面的变化。它还将有一个新的对接系统，允许你把你的 TomTom 放在底座上，用它来控制 iPod 或 USB 存储设备。
