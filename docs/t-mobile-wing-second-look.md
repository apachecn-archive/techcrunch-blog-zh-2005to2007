# T-Mobile Wing 第二外观| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/24/t-mobile-wing-second-look/>

前几天，当 T-Mobile 开始发售 Wing(由 HTC 的好伙计们制造)时，他们因为成为第一家为我们提供 Windows Mobile-6.0 手机的供应商而赢得了吹嘘的权利。

那么这个小玩意是如何支撑的呢？我花了很多时间玩全新的 T-Mobile Wing，现在准备对它进行评判。点击跳转查看完整的评论和大量近距离照片。

-橡胶表面令人难以置信，绝对让这款商务专用设备更具“生活方式”感。这样做的好处是，如果你想把它用作媒体播放器(使用附带的 Windows Media Player 应用程序),它在你的手掌中实际上感觉不错。

-它出奇的薄。比 MDA 薄得多，仅比 RAZR 厚几毫米。假设这大约和一块 LG 巧克力一样薄。因此，这款手机实际上可以很好地放在裤子的后口袋里(我选择的手机皮套)，并且对于那些绝对不想带皮带夹的人来说也很方便。)

![](img/44155571b57776fe0189d3ee0698ef18.png)

-只要光线充足，200 万像素的摄像头就能拍出不错的照片(抱歉，没有闪光灯！)我知道其他手机也有这种功能，但它很好地利用了触摸屏，在预览图像时可以快速调整。这绝对是独立消费类相机应该使用的那种东西，而不是强迫你在无尽的菜单中挖掘。

-他们称之为快速菜单，这是一个很好的 WM6 小功能，可以让你监视正在运行的程序。这一点尤其重要，因为 Windows Mobile 有一种倾向，让你以为你已经关闭了一个程序，而实际上你只是关闭了窗口。令人欣慰的是，这个问题在这次迭代中大大减少了，因为快捷菜单还提供了对复选框的轻松访问，允许您设置右上角的“X”来终止整个程序。

![](img/938785fa63bbc7f8d7a624df5d5bb0e5.png)

-当您滑出键盘时，显示器需要一两秒钟的时间来自动切换方向(当键盘对接时为垂直方向，当您准备键入时为水平方向。)虽然这不是一个巨大的麻烦，但它肯定不是一个无缝的过渡，屏幕令人恼火地打嗝。

-键盘看起来和感觉都很棒，照亮的按键在黑暗中看起来很整洁。太棒了。

-正如 John 在他的[评论](https://web.archive.org/web/20130628200930/http://crunchgear.com/2007/05/22/t-mobile-wing-review/)中指出的，当你点击 Enter 时，IM 客户端不会自动发送消息。对于快节奏的交谈者来说，这可能很烦人。

![](img/eda7b440e3b3c06b94d4065cb36b4210.png)

-即使只有一个程序在后台运行，摄像机也可能无法获得足够的系统资源来加载。我试图在 Internet Explorer 运行时加载它，但没有成功。如果当你正疯狂地试图关闭你打开的任何程序时，一生只有一次的拍照机会与你擦肩而过，这可能是一个悲剧。

Windows Mobile 电子邮件客户端仍未完成。虽然黑莓用户已经习惯于他们的设备从以太网中即时提取电子邮件，但 Windows Mobile 仍然需要“发送/接收”才能获得你的最新消息。除非您每分钟手动点击“发送/接收”，否则您可以将设备设置为每五分钟自动获取一次信息。蹩脚。如果我四分钟前就需要这条信息呢？更糟糕的是，整个发送和接收过程很慢，只需要几秒钟。啊。加油微软！

![](img/43232fa358c4063bc455db6cb50cb546.png)

-当您设置电子邮件时，Wing 会自动合并您设置的所有电子邮件帐户的地址簿。这对于有很多账户的人来说非常有用，因为这让查找他们的联系人变得轻而易举。

-扬声器工作正常，但在高音量下听起来有些低沉。

-Wi-Fi 清道夫工作得非常好，拉起我们的本地网络，让我们输入加密代码。

-机翼上的电池寿命，像所有 Windows Mobile 设备一样，糟透了。

-媒体播放器工作良好，这是该设备比 MDA 更像“生活方式”产品的主要原因。不幸的是，唯一的耳机插孔是通过该死的 USB！它甚至没有大多数手机都有的小插孔(这让我们很恼火)，更不用说全尺寸的了。新的黑莓 Curve 在这方面做得很好——如果有人在一副耳机上投资一大笔钱，他们希望能够在所有设备上使用它们。

-默认的“今天”屏幕可轻松定制，突出显示您的 T-Mobile my aves(整洁！)，而且看起来很棒。你可以很容易地选择你最喜欢的程序或功能，并立即访问它们，这是非常有用的。然而，深入菜单系统可能会很麻烦，菜单组织在直观性方面类似于 Windows 95 计算机。基本上，有时很难找到你的文件。

-你仍然不能与 MAC 进行企业同步。

-你可以设置 Gmail，但这需要几分钟时间，即使这样第一次也可能无法工作(对我来说不是这样)。Hotmail 和 Yahoo 都内置在菜单中，因此是无缝和即时的。

-我用过 Windows Mobile 5.0。我有过 Windows Mobile 5.0 手机在 Windows 5.0 下死机，失速，甚至砖头的情况。我已经想干掉 Windows Mobile 5.0 了。虽然有时仍然有点慢，但 Windows Mobile 6.0 要稳定得多，实际上可以让你完成工作，而不用担心会因为时髦的操作系统而失去一切。它仍然可以使用一些改进(特别是在电池部门)，但这是对其前身的巨大改进，感觉像是在准备好之前就推向市场的测试版。

**总体:**

Wing 是一款非常棒的手机，尤其是如果你是 T-Mobile 的忠实用户(或者仅仅是签约用户),而 Sidekick 有限的软件支持无法满足你对程序的渴望。它也是一个非常复杂的设备，如果你不是从另一个 Windows Mobile 设备迁移过来的话，你需要花很多时间去适应它。然而，回报是巨大的:这真的是一部手机，我可以看到自己在路上用它来完成工作。从各方面来说，它也是非常可定制的。当然，最令人惊讶的是，他们是如何设法掩饰这款手机的商业头脑的。橡胶涂层(它只是乞求被触摸)、媒体播放器和轻薄的外形确实为这个小玩意增添了急需的生命力。

**最终拍摄:**

如果你是一个 T-Mobile 用户，并且没有绑定到黑莓，这简直是最好的商务 PDA 手机，在 Sidekick 的幼稚内涵之间达到了一个很好的中间地带(你不能想象自己把那个东西带到会议室，你能吗？)，以及 MDA 的无趣感。
