# ReviewMe 发布:一个更好的 payper post-

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/11/09/reviewme-launches-a-better-payperpost/>

 [](https://web.archive.org/web/20220816080246/http://www.reviewme.com/) [ReviewMe](https://web.archive.org/web/20220816080246/http://www.reviewme.com/) ，这是一项[类似 PayPerPost 的服务](https://web.archive.org/web/20220816080246/http://www.beta.techcrunch.com/tag/payperpost)，向博客写手支付撰写广告客户产品的费用，刚刚推出。该公司得到了 TechCrunch 赞助商 Text-Link-Ads 的支持，后者最近被收购。

ReviewMe 有一个与 PayPerPost 有些不同的模型。PayPerPost 上的广告商设定了一个单一的费用，支付给所有博客，无论其规模大小，ReviewMe 使用一种基于 Alexa、Technorati 和其他统计数据的算法来确定博客的重要性，并根据计算结果对每个博客收取不同的费用。博主的报酬从每篇文章 30 美元到 1000 美元不等。

还有，博主必须公开评论是付费广告。他们可以用他们选择的任何方式做到这一点，例如“以下是付费评论:”付费广告:“等等。这是对 PayPerPost 的另一个改进，payper post 受到了严厉的批评，因为它不要求披露。

最后，广告商可以购买帖子，但他们不能要求帖子是正面的。博主可以选择写出他们真实的观点，而不用担心得不到报酬。唯一的要求是评论必须至少 200 字。

在一封电子邮件交流中，一位公司发言人表示，“我们正计划埋葬 PayPerPost。”虽然我们不认可这种商业模式，但我们确实注意到，ReviewMe 删除了 PayPerPost 商业模式中最令人震惊的方面:没有披露要求，要求写一篇正面的帖子。

ReviewMe 正在吃他们自己的狗粮，今天[送出 25000 美元](https://web.archive.org/web/20220816080246/http://www.reviewme.com/blog/2006/11/02/launch-set-for-nov-9-details-on-the-25000-giveaway/)给博客写手，让他们写关于这项服务的文章。
