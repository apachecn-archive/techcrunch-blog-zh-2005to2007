# Proporta 展示新款 iPod 保护套| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/15/proporta-shows-off-new-ipod-cases/>

一层铝不足以满足您的纳米需求？双袋你的 8GB 坏男孩与新的铝从 Proporta 水晶案件。这些铝制外壳有一个小的聚碳酸酯窗口，当你使用时，它可以紧紧地拥抱你的 iPod。

但这不是普罗波塔镇上唯一的游戏。哦，不。如果有弹性的更合你的口味，他们还提供了一个更软的箱子，显然是以我前女友的名字命名的。双层硅胶很薄，类似于上面的铝，但是，你知道，硅胶。它还带有一个集成的腰带/系索孔、挂绳环，并有多种颜色可供选择。

为了不被超越，新的 Shuffle 有它自己的铝和硅胶版本，为你的 iPodish 按钮点击留下了一个小洞。

现在这些美味的新款 Shuffle 和 Nanos 已经上市，你不希望你现在的手机也有一个这样的吗？祝你在 Craiglist 上赚到最高价，Scratchyboy。
