# 高通公司推出采用 65 纳米芯片组的新型 3G 手机

![qualcomm-logo1.gif](img/0ee45103cd308e1557a1fb3508a4b4eb.png)三款采用最新 65 纳米(nm)芯片组的新手机现已上市，高通公司今年还将发布 40 多款采用这一新技术的手机。该公司本周宣布了半导体工艺中的这一重要里程碑，这些新型号提供了更高的能效、更薄的外形和对高速数据能力的更大支持。

高通 CDMA 技术公司产品管理高级副总裁史蒂夫·莫伦科夫表示:“高通公司致力于不断提高无线用户的标准，提供更丰富、更实惠的功能和更好的体验。“我们很高兴与我们的客户一起达到这一尖端工艺技术的里程碑，我们期待为全球市场提供更多纤薄、智能和节能的 3G 设备。”

新发布的 65 纳米手机型号包括:

*   华为 WCDMA (UMTS) U120 手机*   LG 电子的 WCDMA (UMTS) KU250 手机*   三星 HSDPA U700 手机
