# 比特洛德采访了来自 TPB 的 Brokep

刚刚看到比特大陆对海盗湾的 Brokep 进行的精彩采访。在采访中，世界上最受欢迎的比特洪流追踪器的负责人讨论了传播盗版，利润和盗版，以及更多。Brokep 讨论了如何超过 50%的种子来自海盗湾，以及如何免费使用文件和版权将导致全球免费使用盗版。
