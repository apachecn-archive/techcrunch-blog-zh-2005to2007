# VeohTV 承诺提供不同的在线电视体验

![](img/0a2987837c21ac002b6d3f276b39accb.png)

最近，Joost 可能受到了很多关注，但它肯定不是网络电视游戏中唯一的新玩家。Veoh 声称，虽然 Joost 拥抱有线电视的老派频道风格，但他们的 [VeohTV 服务](https://web.archive.org/web/20201125083323/http://www.beta.techcrunch.com/2007/06/20/veoh-announces-veohtv-a-sort-of-distributed-joost/)在类似 DVR 的功能上更进了一步。 [VeohTV](https://web.archive.org/web/20201125083323/https://crunchbase.com/organization/veoh) 让您观看网络上的任何视频，而不仅仅是预选内容的频道，并允许您录制您稍后想要观看的内容。甚至是流媒体视频。VeohTV 将于本周晚些时候推出有限的测试版，并将于今年晚些时候向所有人开放。
