# 索尼爱立信发布新的 Cybershot 手机

![](img/48d0d9d6c06ee70adb736a8fdedd1609.png)

正当你认为 W610 是索尼-埃里克森推出的唯一产品时，K810(左)和 K550(右)Cybershot 手机出现了。

这两款手机都有漂亮的设计，并有很大的电量。K810 只有 17 毫米厚，有一个 320 万像素的自动对焦和氙气闪光灯摄像头，一个用于视频通话的摄像头，记忆棒微型支持和三频 GSM。K550 配备了一个 200 万像素的摄像头和 EDGE 网络支持，以及一个记忆棒微型插槽。

不幸的是，美国在 K810 上得到了轴，这将在欧洲和亚洲只在 Q2 推出。然而，K550 同时在北美销量下降。也没有关于价格的消息。真扫兴。
