# v 演员姓名调整| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/22/v-cast-names-that-tune/>

当你听到一首你喜欢的歌，但不知道是谁唱的，你会怎么做？嗯，老的学校方式是在唱片店唱这首歌，但这就像参加美国偶像的选拔一样遗憾。在 21 世纪，你有办法给那首曲子命名，而不必带进歌里。威瑞森 Wireless V CAST Song ID 可以让您听到一首歌曲，将手机对着音乐，然后捕捉音乐样本。从这个 V Cast 歌曲 ID 将识别音乐，甚至允许您购买匹配的完整曲目歌曲，铃声或回铃音。所以下次你在俱乐部或者只是听收音机的时候，你可以用这个功能给歌曲命名，这肯定比问别人“你知道这是谁吗？”
