# Ifrogz 周二

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/08/tuesdays-with-ifrogz/>

![](img/5bbbd1069b1096fb47d46477cc753336.png)

制造定制硅 iPod 保护套的 Ifrogz 每周二都会有一个独特的[赠品](https://web.archive.org/web/20210228073426/http://ifrogz.com/snatchit.php)。MST 时间下午 2 点到 4 点之间访问该网站的人(没错，该公司位于犹他州)会注意到一个闪烁的绿色灯泡，并“抓住它！”选项卡在网站上每个页面的中上部。客户将能够免费向他们的购物车添加特色产品。

可以预见的是，Ifrogz 对于这个神奇的灯泡什么时候会熄灭含糊其辞，但是可以把它看作是为明年的 Amazon.com 圣诞节进行的训练。
