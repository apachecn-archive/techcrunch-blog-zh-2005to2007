# 史努比 iPod Nano 套件:来自花生画廊

刚刚在日本发布的是限量版史努比 iPod Nano 套装。它的特点是在狗屋顶上用激光雕刻了一张可爱的史努比和伍德斯托克的照片，还有一个同样蚀刻的漂亮皮套和一些其他的小玩意来增加交易的甜头。动作要快，因为他们只在日本发行 1000 台。

为这些特别的限量版 Nano's 支付大约 285 颗花生。
