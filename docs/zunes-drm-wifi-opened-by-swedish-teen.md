# Zune 的 DRM，瑞典少年开的 WiFi？

虽然我们坚信 Zune 黑客攻击应该首先局限于让设备在所有操作系统(OS X、Linux、Windows)上运行，但我们仍然希望看到这一领域取得任何进展。

Nutzipper，一个来自瑞典的聪明的黑客，声称不仅成功破解了 Zune 的数字版权管理系统，还成功破解了基于 WiFi 的音乐共享系统。这种被称为“Zukebox”的二合一设备允许 Zune 用户从给定区域内任何其他支持 WiFi 的 Zune 上浏览和下载任何歌曲。然后，它剥去被盗 MP3 的 DRM 枷锁。这不像比尔·盖茨那么喜欢 DRM(T2)，但是你必须知道，如果是真的，这正是 RIAA 在得知 Zune 计划的音乐共享功能时所担心的。

这也许解决了许多用户抱怨的 DRM 限制，但它没有解决 Zune 用户面临的最大问题:找到其他 Zune 用户。

或许是吧？

如果能够从任何其他用户那里找到并使用任何音乐的能力确实存在，这可能会增加许多购买者心目中 Zune 的价值。毕竟，Zune 的全部意义就在于能够从其他人那里即时抓取歌曲，但这种方式并没有(对许多人来说)实现微软做出的宏伟承诺:共享音乐是未来。
