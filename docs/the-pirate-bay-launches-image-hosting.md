# 海盗湾推出图像托管

![](img/1d4b666c7a472e21e1adcb3de516609c.png)

美国海盗知道海盗湾团队一直在进行几个新项目，其中一个项目成为了 YouTube 的竞争对手。不过现在，我们有幸拥有来自 TPB 的独一无二的图片托管服务 BayImg。不像 Photobucket 和 Imageshack 这样的网站，如果认为你的照片“不合适”就会删除，BayImg 允许你上传任何未经审查的图片，超过 100 种文件格式。

所以，拿出你拍摄 JFK 的照片，毫无顾虑地上传吧。BayImg 不会收集你的任何个人信息，并会给你一个网址，以便你删除图片时使用。标签也可用于方便浏览。似乎是个很棒的服务。那么，它能与大玩家一较高下吗？很有可能。
