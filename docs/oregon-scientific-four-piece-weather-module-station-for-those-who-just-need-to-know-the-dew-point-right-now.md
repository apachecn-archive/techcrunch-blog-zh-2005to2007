# 俄勒冈科学四件气象模块站:对于那些只需要知道露点的人来说

[![weathermodule.jpg](img/dc71bc2e9913304c2e970101688539d8.png)](https://web.archive.org/web/20210301233521/https://beta.techcrunch.com/wp-content/uploads/2007/05/weathermodule.jpg "weathermodule.jpg")

俄勒冈科学公司的 BHT668A 是一个模块化、无电缆的迷你气象站。它由四种不同的仪器组成:一个气压计，一个雨量计，一个显示和记录降雨数据的室内模块和一个温湿度计。那个测量湿度。因为它是模块化的，所以你可以把四个部分分开，放在不同的房间里。你可以有一个“湿度室”或“雨水室”你也可能会失去所有的朋友！

你可以在这个宝贝上花 200 美元，或者像我一样只需要装上再装上[简单天气](https://web.archive.org/web/20210301233521/http://crunchgear.com/2007/04/19/simpleweather-skips-the-clutter-delivers-the-goods/)。酷小孩都是这样的。
