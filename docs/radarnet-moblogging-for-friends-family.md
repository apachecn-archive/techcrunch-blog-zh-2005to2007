# Radar.net:朋友、家人的移动博客

这是一个测试版手机照片系统，旨在与朋友和家人分享照片和评论，而不是与威斯康星州最深处的无聊暴露狂夫妇分享。

![](https://beta.techcrunch.com/wp-content/uploads/2007/01/the inside of a pocket. Phone cams were useful little toys, fun for a moment or two and then forgotten. Radar, however, seems to be changing that equation.</root>)

雷达允许每个用户拥有自己的照片流。您会收到一个个性化的电子邮件地址，然后通过彩信或电子邮件将您的照片发送到流媒体。它们会立即显示出来，你的朋友可以将你的视频流放在他们的主页面上，并对你的图片发表评论。该系统可以在你朋友的信息流更新时发送电子邮件——这是完全可配置的——图像几乎会在传输后立即显示出来。

![](img/bbc7207bc5f17edcd5b130f9e248ae4c.png)

然而，最好的部分是社交方面。如果你有朋友和/或家人，你们都可以分享你们的照片，并对每张照片发表评论。例如，我们显然要给[宝宝](https://web.archive.org/web/20210227232732/https://beta.techcrunch.com/wp-content/uploads/2006/09/img_2888.jpg)拍各种可爱状态的照片，所以我们会与姻亲、姐妹、兄弟等分享这些照片。每个人都可以在这个封闭的小组中发表评论。你也可以阻止人们看到某些照片——也许我和我的妻子想要一个 mobog 时刻——并将用户分组以便于管理。

雷达也有一套适用于各种手机的客户端，包括一个 Java 客户端，它可以控制你的相机并传输图像，而不必手动选择它们。这些客户端相当精巧，网站上的所有功能都被压缩在一个小小的屏幕上。

该网站显然仍处于测试阶段，但它非常有用。计划包括添加广告，以及创建一个优质服务。从 Radar 的服务器上获取你的图像*应该也是一个即将到来的附加功能。*

总的来说，这是一个有趣而有用的工具，除了给宜家家具拍照以备日后查阅之外，你还可以将手机用于其他用途。
