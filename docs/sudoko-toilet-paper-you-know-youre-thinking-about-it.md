# Sudoko 卫生纸:你知道你正在考虑它

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/21/sudoko-toilet-paper-you-know-youre-thinking-about-it/>

数独卫生纸就像它听起来的那样:上面印有疯狂流行的数独游戏的卫生纸。所以，如果你在马桶后面放一支笔，你可以做数学智力题，而不是一边做你不敬的事情一边读《T2》格言。

如果你确实在那里放了一支笔，我们想知道为什么，因为你在今天之前从未听说过这个产品。是在*格言*中泰拉·瑞德的照片上画胡子吗？提示:你不需要把它画进去，只要在三天的狂欢后，当她忘记去她的沙龙时找到她。小心，汤姆·塞立克。

然而，5 美元一卷似乎有点贵。没有关于双层绗缝的消息。
