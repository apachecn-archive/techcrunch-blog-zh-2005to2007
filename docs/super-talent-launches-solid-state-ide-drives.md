# 超级天才推出固态 IDE 驱动器

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/10/24/super-talent-launches-solid-state-ide-drives/>

单个驱动器的标准消费硬盘容量很快接近 1TB，内存制造商 Super Talent 的这款 16GB 闪存驱动器(市场价约 500 美元)显得微不足道。然而，该公司的 IDE 闪存驱动器(有 4GB、8GB 和 16GB 容量可供选择)非常适合 UMPCs 和笔记本电脑，因为它们重量轻、功耗低且高度可靠。此外，由于没有移动，它们不太容易受到跌落、振动和冲击造成的数据损坏。

不，这不是新技术，但在不到两年的时间里，IDE 驱动器将与硬盘并列出现在您最喜爱的笔记本电脑供应商的配置程序中。相信我，一旦你用这些启动了你的笔记本电脑，你就再也不用担心转速了。
