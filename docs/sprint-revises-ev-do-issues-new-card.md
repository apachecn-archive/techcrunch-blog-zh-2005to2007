# Sprint 修订 EV-DO，发行新卡| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/08/25/sprint-revises-ev-do-issues-new-card/>

![](img/7cbda62c528eb6bfa57179f68b9981f9.png)

在与威瑞森争夺 EV-DO 市场主导地位的竞赛中，Sprint 遥遥领先。该公司已宣布，它正在升级其 EV-DO 网络的版本 A，并将很快发布卡，利用新的实施。修订版 A 卡将支持平均 450–800 kbps 的下载速度和 300–400 kbps 的上载速度。

新的版本 A 卡是向后兼容的，将出售 99.99 美元的两年合同和 179.99 美元的一年合同。我想，我们可以预计威瑞森在未来几周也会采取类似措施。
