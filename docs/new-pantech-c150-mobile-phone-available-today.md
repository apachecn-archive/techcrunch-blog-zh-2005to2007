# 新的泛泰 C150 手机今天上市

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/25/new-pantech-c150-mobile-phone-available-today/>

![pantech c150](img/0e9a0054ddf22c21986cf29111bba92e.png)从今天开始，在& T，客户将能够购买泛泰 C150 照相手机。值得一提的是，C150 获得了“2006 年国际卓越品牌设计 iF 设计奖”

显然，这是一个相当不错的奖项，因为它已经举办了 50 多年，来自世界各地的参赛作品数以千计。我想这就是我没有跟上德国设计竞赛的原因。

这款直板手机——哇，当我写这篇文章的时候，我刚刚注意到手机屏幕上的照片，哈哈哈！—附带蓝牙，通过 Yahoo！、AOL 和 Windows Live Mail、MP3 铃声和一个也能拍摄视频的 VGA 摄像头。

这款手机不到半英寸厚，两年合约的价格是 199 美元。
