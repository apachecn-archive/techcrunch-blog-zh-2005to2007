# 罗杰的幻听

把这篇文章归入“我说过了”一类。或者更准确地说是“罗杰说的”。终于有人谈到了下一代手机的发展方向，我完全同意。我们真的需要像普拉达或 iPhone 这样的触摸屏手机吗？“以语音为中心的方面发生了什么？”罗杰问。说得好。一部带语音功能的手机…这才是真正的智能！

和大小。iPhone 将会很大。便携式设备不是越小越好吗？当然，对于移动网络和游戏来说，尺寸很重要。好吧，我想我们不能拥有一切。
