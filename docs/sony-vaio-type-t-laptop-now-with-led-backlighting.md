# 索尼 Type T 型笔记本电脑现在与 LED

![img_002.jpg](img/384f19f2ed8e770f9b107bb808b140d2.png)

有什么比在黎明时分醒来发现有一台 LED 背光笔记本电脑更好的呢？实际上，很多事情，但是如果日本的科技巨头能给我们送来最新的 VAIO 就好了。我们都在谈论盛传的 LED Macbook(T2)。索尼可能会大赚一笔，但他们多年前就已经把头埋得太深了，所以这永远不会发生。
 总之，索尼 VAIO Type T 十周年笔记本电脑拥有我们想要/需要的一切，包括前面提到的 LED 屏幕、可选的 32GB 固态硬盘和 8 小时电池续航时间，所有这些都装在闪亮的黑色外壳中，重量为 2.68 磅。其他值得注意的规格包括 80GB 硬盘、1GB RAM、11.1 英寸全景屏幕和双层录制 DVD 驱动器。如果你觉得辣，那么你可以选择 32GB 固态硬盘和 160GB 硬盘的硬盘和 DVD-R。还有一个可选的电池，你可以升级到 18 小时的任何你选择沉迷的乐趣，但等等，还有更多。

Type T 有两个 USB 2.0 插槽，一个 Firewire，Expresscard/34，SD/MMC/Memory Stick Pro/Duo，英特尔 945GMS 高速芯片组，802.11a/b/g，网络摄像头，指纹扫描仪和蓝牙 2.0+EDR。真的，谁想要这些？
