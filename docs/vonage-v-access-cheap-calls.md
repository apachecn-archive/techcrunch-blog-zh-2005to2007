# Vonage V-Access:廉价通话

![](img/64073427ba8988b7b9df1534c22015d9.png)

Vonage 正在提供一种叫做 V-Access 的新“服务”。旨在为您和您的家人/朋友节省长途费用，您可以通过本地接入号码拨打 Vonage 号码。它是这样工作的:

*   1.拨打本地接入号码，无论您身在何处都是免费的。*   2.拨打您要拨打的电话号码。*   3.你可以打市话(通常是免费的)。

你还可以使用 V-Access 来避免国际电话费——这对交换生来说似乎很好。如果你有 Vonage，让我们知道这有多好。
