# scanR 将为脸书发布版本

可以拍照的时候谁还需要做笔记？scanR 的人做了笔记，记录了过去的事情。有了这个漂亮的小服务，你可以用手机拍一张照片，然后发送到 scanR，然后通过电子邮件发送给你。现在，该公司给我们的话，他们将发布一个版本的脸书了。

有了这项服务，学生可以在课堂上使用他们的照相手机来复制笔记、论文、书籍甚至白板。抓拍的照片将被转换成 PDF 格式，并可以上传到脸书。不去上课从来没有这么容易过！
