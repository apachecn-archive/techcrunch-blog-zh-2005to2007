# TrustedPlaces 是一种有益的体验。

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/22/trustedplaces-is-a-rewarding-experience/>

![TrustedPlaces](img/3b3c01109be932d1a2318d395305acbe.png "TrustedPlaces") [TrustedPlaces](https://web.archive.org/web/20151003012546/http://www.trustedplaces.com/) 是一个“口口相传”的社交网络，人们可以在这里评论、评价和为朋友推荐他们知道的地方，或者只是为了自己的利益使用该网站(重新)搜索地方。

TrustedPlaces 是 Walid Al Saqqaf 和 Sokratis Papafloratos 在 Peter Nixey 的咨询帮助下，自今年年初以来一直在积极开展的一个项目。本周，TrustedPlaces 宣布他们已经成为公开测试版，在这个版本中，他们还增加了一些新功能。昨晚在 Mashup 上，我遇到了 TrustedPlaces 团队，他们对达到这一里程碑感到非常兴奋。他们还告诉我，他们已经在着手下一个候选版本，并且会有更多的新特性来扩展网站的价值/功能。例如对 hcard、hreview 和 geo 微格式的微格式支持。

> “在这个版本中，你会看到一个新的主页，一些非常有用的评论选项，收藏夹和朋友的收藏夹，相当多的设计调整，但你也可以搜索你附近的地方。”

我认为他们做得非常好，已经实现了你现在在社交网络上期望的大多数有用的 web 2.0 特性和功能——标签云、个人资料、朋友网络、评级，但 TrustedPlaces 最有趣的方面之一是他们为评论者实现的[奖励系统](https://web.archive.org/web/20151003012546/http://trustedplaces.com/docs/faq#rewards)。

> “您在网站上进行的某些操作会获得不同的积分，例如添加评论或向其他人的评论添加图片。这些分数中的每一分都被视为我们将定期举行的抽奖的一张票。您可以通过在网站上更加活跃和收集更多积分来增加获奖的机会。系统通过将您分为三个不同的类别来通知您获胜的可能性:

> *   冷:胜率> 20%
> *   温暖:21–60%的胜算
> *   热门:胜算超过 61%
> 
> 每个月底我们会随机抽取一张彩票。如果你的彩票被抽到，你就赢得了月度奖。每年年底还会随机抽取一张年度大奖的门票。如果你的彩票被抽到，你就赢得了年度大奖。"

和 TrustedPlaces 一样，我坚信那些依靠“用户生成内容”来增加盈利能力或整体价值的网站——亚马逊、YouTube、雅虎！以 Answers 和 TechReview 为例——应该以某种方式奖励他们的贡献用户，无论是奖品、优惠券、积分还是小额现金支付(如果可行),以确保有价值内容的*可持续*流。

例如，亚马逊应该很容易使用他们的 [mTurk 服务](https://web.archive.org/web/20151003012546/http://www.mturk.com/mturk/welcome)来奖励在他们网站上发表评论的人。我知道微软正在开发一个积分奖励系统，类似于他们对 [Xbox Live 积分](https://web.archive.org/web/20151003012546/http://www.xbox.com/en-US/live/features/microsoftpoints.htm)的奖励系统，他们将在他们的 Live 网站上使用，例如 [Expo](https://web.archive.org/web/20151003012546/http://expo.live.com/) 。

雅虎！点已经可用，我可以想象这最终链接到他们最近推出的[雅虎！答题积分](https://web.archive.org/web/20151003012546/http://uk.answers.yahoo.com/info/scoring_system;_ylt=AlPw74fZcIzWQHPLDUqcPYgdBgx.)奖励系统。为了不被淘汰，谷歌当然计划拥有自己的[积分](https://web.archive.org/web/20151003012546/http://www.revenews.com/bethkirsch/archives/002032.html)系统。

我预见到的用积分奖励人们的唯一问题是，我们曾经和 [Beenz](https://web.archive.org/web/20151003012546/http://news.bbc.co.uk/1/hi/sci/tech/423369.stm) 一起来过这里，他试图创建一个全球积分系统。也许这一次，像“联合 ID”一样，也许我们会得到一个联合积分交换？唯一美中不足的是一个积分值多少钱，当涉及到积分交换时，1 个谷歌值 2 个雅虎吗？s？？？
