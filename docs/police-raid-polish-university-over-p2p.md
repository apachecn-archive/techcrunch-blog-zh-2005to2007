# 警方因 P2P  突袭波兰大学

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/20/police-raid-polish-university-over-p2p/>

![](img/00fd41747f15cd56954e292a74a8af08.png)

世界各地的大学和学院的盗版行为都在增加，但波兰的一所学校肯定走得太远了。周三，由于大量盗版活动正在进行，警方突袭了科萨林科技大学。他们发现的远远超出了一个孩子自制的服务器。联邦政府没收了一台运行 DC++ hub 软件的 PC、10 台笔记本电脑和 60 个硬盘，其中包含超过 35，000GB(您没有看错，先生)的媒体供学生下载。

三名拥有服务器管理权限的学生被拘留，很可能会受到多项指控。报告称，学生们使用 DirectConnect P2P 软件来分享大量的电影、音乐和软件。我知道很多人因为文件共享而在校园里被抓，所以如果你在学校经常盗版，并且你在使用 DirectConnect，我们可以建议你暂时退出吗？这也许能救你一命。
