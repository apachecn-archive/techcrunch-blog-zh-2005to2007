# Zune 终究不会因为分享而还你钱

你想知道我们讨厌的事吗？我们讨厌有一个非常热门的网络谣言，那种引起我们的注意，让你，我们的读者，站起来注意，然后它被证明是完全胡说八道。完全是垃圾。差不多完成了。
 [我们昨天提到，我们已经听到了关于微软对以病毒方式分享音乐的 Zune 用户进行补偿的传言](https://web.archive.org/web/20130627211735/http://crunchgear.com/2006/10/23/zune-to-pay-you-back-for-sharing-songs/)，因为这是 Zune 存在的主要原因之一。简而言之，这个理论是，如果其他 Zune 用户购买他们从你的 Zune 下载的音乐，你就可以获得积分。积累足够的信用点数，它们可以用来购买歌曲。

这对 Zune 来说是一个绝妙的想法，因为它鼓励人们使用它的独特功能，如果它计划完全取代 iPod，这是它成功的关键。唉，都是废话。昨晚我们和几个 Zune 团队成员谈了谈，喝了几杯鸡尾酒后，我们让他们确认这不是真的。

他们确实承认这是一个好主意，而且“以后可能会有类似的产品”，但是，至少在发布时，它是垃圾。抱歉，Zune 的粉丝们，我们不是故意让你们这么激动的。好吧，我们确实想让你兴奋，但我们不想给你留下 Zune 蓝球。
