# Wii 想要我们的 Wii-

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/06/wii-want-our-wii/>

![](img/6d3732ed131a54af08423b80cc90101c.png)
Wii 系列大约在 2006 年夏天。现在更糟糕了。

不管你喜欢它还是讨厌它，Wii 是目前最受欢迎的游戏机，任天堂有一些事情要做。Wii 于去年推出，但人们仍然需要在百思买外面排队几个小时，并且[出售他们的肾脏](https://web.archive.org/web/20201028054024/http://www.outtastock.com/wii/)才能获得这一大堆乐趣。Kotaku 威胁使用暴力，而任天堂美国公司的 Reggie Fils-Aime 说:

> 我们与零售商合作，定期持续向他们供货。一些零售商决定保留产品，以支持他们通常在周日投放的广告。但我现在正在看一份报告，报告称 93%的零售店中，可用的硬件设备不到三个。这是一个巨大的问题，我们正在努力解决。

雷吉，游戏机在哪里？【筝乐】
