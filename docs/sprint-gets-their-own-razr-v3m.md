# Sprint 获得自己的 RAZR V3m

“迟到总比不到好”是我母亲过去常说的话，因为我又把田径比赛的最后一名带回了家。显然，Sprint 的高管们也是这么想的，他们最终将摩托罗拉 RAZR V3m 纳入阵容就是明证。

这款手机和威瑞森之前的所有产品一样，有蓝牙、130 万像素的摄像头、EV-DO 和 290 美元的巨大价格标签。真的吗？幸运的是，有 150 美元的“即时储蓄”和 90 美元的邮寄回扣，可以降到 50 美元。如果我们是你，我们会留下来看一些 Moto KRZR 或 Moto RIZR 的行动，而不是把我们辛苦赚来的 50 美元花在这个爷爷身上。
