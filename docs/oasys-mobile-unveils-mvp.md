# Oasys Mobile 发布 MVP 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/27/oasys-mobile-unveils-mvp/>

![MVP.jpg](img/baa37cbda62f06561788e2753e4aa90e.png)本周，Oasys Mobile 宣布为威瑞森无线公司推出 *AROD MVP 棒球挑战赛*，这是一项以扬基队 A·罗德里格兹为主角的移动棒球赛事。街机风格的游戏侧重于击中一系列目标，包括各种体育场人物。这种快速的游戏风格基于击球和投球，并以音乐、人群噪音、甚至裁判判罚和击球声为特色。
