# 魔方 MP3 播放器概念

![](img/700e5dec392c7ddf85bfca977e4be78c.png)

那些不太擅长字谜/音频的人不需要在这里申请。Yanko Design 公司的这个概念 MP3 播放器是基于我们小时候都失败过的非常流行的魔方。

但它不仅仅是一个可爱的 MP3 播放器，你实际上需要解决这个难题才能执行某些功能。例如，向右旋转顶部可以向前跳过一首歌曲，而向后旋转可以转到上一首歌曲。踢球者？要关闭设备，你需要解决这个难题。我猜新的*旅程* CD 肯定是第八次真正的摇滚。
