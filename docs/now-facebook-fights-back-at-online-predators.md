# 现在脸书反击在线掠夺者

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/07/30/now-facebook-fights-back-at-online-predators/>

[![shadowyfb.jpg](img/67b864253159679943effb01abd61131.png)](https://web.archive.org/web/20160422045703/http://old.crunchgear.com/wp-content/uploads/shadowyfb.jpg "shadowyfb.jpg")

[脸书](https://web.archive.org/web/20160422045703/http://crunchgear.com/category/facebook/)是我的白鲸:我似乎每天都在写它，尽管我[声称](https://web.archive.org/web/20160422045703/http://crunchgear.com/2007/06/15/facebook-how-complicated-has-this-site-become/)不想与它有任何关系。尽管如此，很高兴看到该网站处理其业务，使儿童捕食者远离不知情的未成年人。就像之前的 MySpace 一样，这个社交网站已经吸引了《捉迷藏》的玩家，因为所有的孩子都在上面。现在当局正在打击脸书。“镇压”意味着脸书不合作，这与事实相去甚远。

脸书在成立的三年里，对媒体和技术了如指掌，通常会在几个小时内删除令人不快的简介。所有投诉都很快得到纠正。

《泰晤士报》指出的是，有人如何建立了一个虚假账户，以此来看看与年轻人联系有多容易。此人加入了露骨的性团体，并不断受到网络怪人的骚扰。我不想重复一些留言的内容。

但是脸书是积极主动的:它正在汇编一个性犯罪者姓名和电子邮件地址的数据库，以便从一开始就防止他们注册。

脸书不希望有任何负面的宣传——一个性犯罪者通过该网站结识某人可能是你能想到的最糟糕的事情——所以它将继续做它必须做的事情，以保持其清白的形象。而且，你知道，不被起诉遗忘。
