# 视频游戏玩家是真正的英雄

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/29/video-gamer-a-true-hero/>

![](img/42f87d68266eacca84e2520f7e1db8ac.png)

不管在什么场合，任何英雄都应该得到荣誉。因此，应该给住在大洋彼岸的 24 岁 it 顾问亚当·梅普森(Adam Mapleson)一份这样的礼物是有道理的。当梅普森看到一名女保安被两名男子持枪抢劫时，他立即采取行动，并抓住了那两名劫匪。在此过程中，亚当胸部中枪。

他在一个火车站附近被发现，奄奄一息地被送往医院。但是作为亚当这样的英雄，他为生活而战，现在他的情况正在好转。显然，英国的一些小报称亚当“危险”，因为他的 MySpace 页面显示他喜欢游戏(主要是 FPS)和黑色金属。这家伙玩射击游戏并不意味着他是个威胁。我是说那家伙因为大声哭喊而被枪杀。他是个英雄，毫无疑问。我认为亚当应该因为他的勇敢而获得一枚奖章和一台免费的 Xbox 360。
