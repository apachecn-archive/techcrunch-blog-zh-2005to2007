# PS3 上的索尼:游戏上路！不要慌！

索尼 PS3 的用户们，振作起来。日本巨人已经宣布，你将获得 34 个即将到来的游戏形式的系统内容，包括侠盗猎车手 4，时间转移，和一些关于世界尽头的海盗。我觉得是 a 片。
