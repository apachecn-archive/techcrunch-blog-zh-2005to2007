# 索尼 Vaio 型，带 32GB 闪光灯

索尼的 Vaio Type U，类似 UMPC 的手持设备，存储空间从 16GB 增加到 32GB，价格翻了一倍，几乎肯定超出了任何人的价格范围。为什么？因为它是 1800 美元。

现在还有 1GB 的内存，相同的滑出式键盘，相同的网络摄像头，以及与旧的 16GB 版本相同的所有其他东西。
