# 更新:Antec 九百很安静

所以昨天我提到了 XYZ 计算公司对 Antec 的九百台游戏台式机的评论。大黑盒子里装着三个 120 毫米的风扇和一个 Antec 的大男孩 200 毫米风扇，网站报道说，我也报道了，因为“不太安静”。好吧，现在我开始怀疑 XYZ 是否让粉丝们兴奋了。

另外三篇评论强调了这一点，即此案是彻头彻尾的秘密，这与你没有亲身经历的人可能会想到的相反。

[来自汤姆的硬件](https://web.archive.org/web/20130627213606/http://www.tomshardware.com/2006/10/27/the_gaming_case_showdown/index.html):

> 由于开箱后安装了四个风扇，并且还能容纳三个，Antec 设法避免了通常伴随这种充满风扇的箱子的所有噪音。即使是顶部的 200 毫米大排气扇也几乎安静地运转，同时从机箱中抽出足够量的空气。超频者可能会有一些乐趣，看看他们能从这样的装备中得到什么。

 [来自硬件回顾](https://web.archive.org/web/20130627213606/http://www.hardwareinreview.com/cms/content/view/54/):

> 说到风扇，你可能会认为 900 听起来像吸尘器，但它实际上是我这里噪音最低的 PC 机箱。即使当它们都被调到最高设置时，这 900 个仍然发出咕噜咕噜的声音。很有可能你的显卡和 CPU 风扇会比这个机箱发出更大的噪音。

[来自超频俱乐部](https://web.archive.org/web/20130627213606/http://www.cluboverclocker.com/reviews/cases/antec/NineHundred/index.htm):

> 从内部观察那个巨大的 TriCool 200 mm 风扇，会发现它实际上比 120 mm 风扇大得多。就像 120 毫米 TriCool 风扇一样，200 毫米风扇也可以通过树位置开关进行调节。我把它设置为中等，风扇无声地推动一吨空气。在高处，风扇是可以听到的，但对于超频系统来说，这个风扇真的可以将热空气从顶部吸出来。

我亲眼见过这种情况，但它没有打开，所以我真的不能说它是不是很响。但是现在三比一的人支持九百安静，尽管它有大量的粉丝。
