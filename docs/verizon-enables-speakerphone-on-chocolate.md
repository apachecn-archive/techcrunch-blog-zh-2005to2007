# 威瑞森启用巧克力扬声器| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/27/verizon-enables-speakerphone-on-chocolate/>

威瑞森今天发布了巧克力爱好者期待已久的固件更新。性感的触摸感应滑块的固件版本 V04 已经发布，它启用了扬声器功能，由于某种原因，该功能在首次发布时被禁用。

缺点是你不能更新自己。要刷新固件，你必须把手机拿到离你最近的 VZN 授权经销商那里，通过号码 V04 请求升级。他们把它插到他们的魔法盒子里，一些奥柏伦柏人会唱一首关于你的歌，然后，瞧，你得到了会说话的巧克力。
