# 工作中的诺基亚 5700 XpressMusic

![](img/1bc70cc9194a1dcca082c6d16166ccfe.png)

一则广告出现在网络上，上面有诺基亚 5700 XpressMusic 的图片和“5700 XpressMusic”的字样。该设备本身看起来很像诺基亚 3250，但具有 5300 的一些更好的功能。除了它将运行 Symbian S60 第三版并安装了 Feature Pack 1 之外，我们对这款手机知之甚少。

然而，就航母而言，我们一无所知。这款手机可能会跟随 5300 XpressMusic 甚至欧洲运营商的脚步，为 T-Moibile 推出。随着更多细节的出现，我想我们只能拭目以待了。
