# Skype 即将进入 PSP？闭上你的嘴

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/09/skype-coming-to-psp-shut-yo-mouth/>

有没有人钻研过 PSP 的固件更新，四处窥探隐藏的宝藏？我们没有，但是快跳有。根据 QJ 的说法，如果你查看固件 3.10 的 vshmain.prx 的解密版本，你可能会相信某种形式的 VoIP 功能将在未来出现。所有迹象都表明 Skype 目前是供应商。

> 检查转储，没有实际的 PRX 的名字，这意味着实际的功能仍在这个阶段工作。我也不认为有人在虚张声势。早在功能开放之前，固件中就已经有相机、GPS 和 POPS 程序的例子了。在相机和 GPS 的情况下，参考资料可以追溯到固件 2.70。在固件 2.80 转储中发现了 POPS 引用。

这是否意味着 Mylo 的灭亡？
