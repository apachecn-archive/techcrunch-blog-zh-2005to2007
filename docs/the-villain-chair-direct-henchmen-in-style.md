# 恶棍之椅:时尚的直接追随者

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/14/the-villain-chair-direct-henchmen-in-style/>

布洛费，达斯·维德，邪恶博士，莱克斯·卢瑟，爪博士。这些是我们最喜欢的几部电影中的一些绝对致命和邪恶的坏人。共享邪恶的天才，全面征服的计划，和光滑的 knavish 风格。这种风格的一个常见元素是标志性的恶棍椅。

由深色皮革制成的威胁性框架，恶棍椅足以让任何人觉得自己是一个超级坏蛋。我们喜欢想象我们的编辑们在他们的年代里慢慢地轮换着，把次要的博主喂给贪婪的鳄鱼，或者强迫他们看着他们的家园被摧毁。棘手的部分是背景。当然，在黑白分明的产品网站上，它看起来非常令人印象深刻，但我们有一种感觉，如果我们把它搬到，比如说，我的垃圾，拥挤的公寓，它会看起来非常不合适。

这让我们认为，有人(宜家的邪恶双胞胎兄弟，他看起来就像宜家，但留着山羊胡子)应该推出一系列看起来很邪恶的家具。我们肯定会买的，如果我们的亲戚允许的话。你知道，他们不会的。
