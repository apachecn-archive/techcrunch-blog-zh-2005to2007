# NBC 油墨与 Alltel | TechCrunch 达成协议

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/18/nbc-inks-deal-with-alltel/>

你可以在 Alltel Wireless 的手机上观看 NBC Universal 最喜欢的内容。mocoNews 报道称，NBC Unversal (NBCU)已与 Alltel 签署了一项协议，该协议将允许运营商通过 Alltel 的接入数据服务分发 NBCU 11 个视频点播频道的内容，以及铃声、回铃和壁纸。

各种短格式的视频内容将包括来自 Access Hollywood、Telemundo、NBC News、*深夜与柯南·奥布莱恩*、Brave、科幻频道和美国网络的内容，以及来自 dotcomedy.com 的原创内容，甚至还有一点来自*星期五之夜*的长篇内容。在后一种情况下，似乎没有人会收看电视节目，所以为什么不把它送给 Alltel 的客户呢！
