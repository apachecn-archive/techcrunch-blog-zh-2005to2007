# MyBlogLog 将被雅虎收购

更新:雅虎的公关公司之一 OutkastPR 发邮件给我们说这个故事是不准确的，雅虎没有收购 MyBlogLog。我们深入调查了一些内部人士，似乎雅虎和 MyBlogLog 正处于早期的收购讨论中，仅此而已。当被问及今天的谣言时，创始人斯科特·拉夫完全沉默了——暗示他不介意得到所有媒体的关注。

 [](https://web.archive.org/web/20220816171841/http://www.mybloglog.com/) 第二次被雅虎社交媒体收购故事发生在过去的 12 个小时里，有消息称雅虎！已经同意收购 [MyBlogLog](https://web.archive.org/web/20220816171841/http://mybloglog.com/) 。MyBlogLog 是一个博客社区和分析工具，用于 TechCrunch(见侧栏)和网络上的许多其他博客。该网站刚刚退出测试，很快就获得了关注。我们第一次[对 MyBlogLog](https://web.archive.org/web/20220816171841/http://www.beta.techcrunch.com/2006/10/19/mybloglog-readers-network-around-their-favorite-blogs/) 进行侧写是在十月份。据传该公司出价约 1，000 万美元。置评请求尚未得到回复。

昨晚我们还了解到雅虎！[是否同意收购](https://web.archive.org/web/20220816171841/http://www.beta.techcrunch.com/2006/11/16/yahoo-acquires-contest-site-bix/)在线竞赛网站 [Bix](https://web.archive.org/web/20220816171841/http://bix.com/) ，这是一家由 [Epinions](https://web.archive.org/web/20220816171841/http://epinions.com/) 创始人 Mike Speiser 领导的公司。凯蒂·费伦巴赫报道雅虎！今天也证实他们已经收购了瑞典移动公司 Kenet Works，这笔交易几个月前就已经完成。我们被告知雅虎！也将在明年 1 月的 CES 上发布一款怪兽产品。
