# 专利申请显示固定电话显示基于互联网的广告:万岁，更多的广告

[![adsphone.jpg](img/7d7ad082055fb87364accf303935ed02.png)](https://web.archive.org/web/20130628202046/http://old.crunchgear.com/wp-content/uploads/adsphone.jpg "adsphone.jpg")

ZDNet 几天前披露的一项专利申请显示，一部普通的老式电话能够在屏幕上显示基于互联网的广告。这包括谷歌广告，这也是谷歌目前价值无限的部分原因。这款手机使用互联网协议电话系统(就像麻烦中的 Vonage 一样)，所以它也可以接入互联网广播流，前提是它们没有被立法禁止存在。

我想知道的是，如果我们在手机上接受这些广告，我们会看到任何形式的服务价格折扣吗？不然你为什么想在手机上看到广告？这只是另一种尝试，在你醒着的时候，向你做广告，哦，忠实的消费者。
