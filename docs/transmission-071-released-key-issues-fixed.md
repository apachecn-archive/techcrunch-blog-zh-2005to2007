# 发布传输 0.71–修复关键问题| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/24/transmission-071-released-key-issues-fixed/>

![](img/b2b462f62e3f181c70e251a79e7fa4a5.png)

Mac OS X 用户总是对体面的 Bit Torrent 客户端有点反感。总有 Azureus，但是它很混乱，很慢，而且很耗资源。直到最近发布的 Bit Rocket，除了传输之外，就没有什么别的了。过去，网络上几乎所有私人比特种子追踪器的传输都被禁止。为什么会这样？传输过去一直不停地向追踪者宣布，导致整个 torrent tracker 变慢，有点像 DoS 攻击。

但是随着 0.71 版本的发布，传送又回到了游戏中。一个更受欢迎的私人音乐追踪者今天宣布，Transmission 已经从 torrent 客户端的“禁止”列表中删除，现在可以在网站上使用。如果你不是 Azureus 破坏你的 CPU 的粉丝，或者讨厌 Bit Rocket 仍然存在的一些错误，请查看 Transmission 并享受愉快的 UI 和低 CPU 使用率。
