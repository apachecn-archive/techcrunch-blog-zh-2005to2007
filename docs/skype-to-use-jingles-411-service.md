# Skype 将使用叮当的 411 服务

 [Skype](https://web.archive.org/web/20210928043814/http://www.crunchbase.com/company/skype) 和[叮当网络](https://web.archive.org/web/20210928043814/http://www.crunchbase.com/company/jingle-networks)，一家免费 411 服务的运营商，正在合作——美国 Skype 用户将可以使用叮当打 411 电话。用户还可以将“Free411USA”添加到他们的 Skype 好友列表中。用户必须安装 Skype 3.5 for Window。

在叮当[宣布了广告支持的 411 电话专利](https://web.archive.org/web/20210928043814/http://www.beta.techcrunch.com/2007/06/15/jingle-awarded-patent-for-free-411-calls/)之后，这条新闻很快就出现了。创始人斯科特·克里格表示，该公司将在未来几个月内利用这项专利来打击竞争对手——这些竞争对手包括谷歌和谷歌。

就我个人而言，我觉得这笔交易有点愚蠢。Skype 用户通常使用连接互联网的设备，而网络搜索几乎总是查找商务信息和 411 电话信息的一种更简单的方式。

我还认为叮当的专利近乎荒谬，凸显了我们美国专利制度的许多问题。如果接受付费服务，将其与广告配对，然后免费分发是可以申请专利的，那么我们就真的有麻烦了。当他们试图让 411 电话对消费者来说更便宜时，叮当是个好人。现在，每当另一家公司尝试同样的事情时，他们都想从中分一杯羹。那给我留下了不好的印象。
