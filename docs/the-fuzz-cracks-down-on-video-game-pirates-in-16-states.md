# 美国警察在 16 个州打击视频游戏盗版

[![piracypiracy.jpg](img/4a4d36e488c34b7cfafb4d86628b6360.png)](https://web.archive.org/web/20130628211101/http://old.crunchgear.com/wp-content/uploads/piracypiracy.jpg "piracypiracy.jpg")

联邦调查局昨天突击搜查了一些家庭和企业，寻找盗版视频游戏和允许视频游戏盗版的设备。你知道，mod 芯片之类的。(庆幸他们没把我抓回 2002 年，lol。)联邦政府没有公布任何名字或关联，但在 16 个州执行了 32 份逮捕令。显然，这是长达一年的视频游戏盗版阴险犯罪调查的一部分。

嗯，这是[今天](https://web.archive.org/web/20130628211101/http://crunchgear.com/2007/08/02/dont-copy-that-floppy-pirate-gets-sentenced/)关于逮捕海盗的第二个故事。很高兴看到我们的政府为我们工作。
