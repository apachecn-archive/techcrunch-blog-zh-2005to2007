# Diggville 的麻烦 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/09/06/troubles-in-diggville/>

[![](img/efce28be54deb9bfefe5524c61df0842.png)](https://web.archive.org/web/20230218164235/http://www.digg.com/)[令人难以置信的成功](https://web.archive.org/web/20230218164235/https://techcrunch.com/2006/06/22/digg-30-to-launch-monday-exclusive-screenshots-and-stats/)新闻网站 [Digg](https://web.archive.org/web/20230218164235/http://www.digg.com/) 最近遇到了一些困难。Digg 是一个新闻网站，根据其他 Digg 用户的投票，将用户提交的新闻故事推广到其主页。如果一个故事被足够多的用户“挖掘”，它就会出现在主页上，大量的流量就会被导向新闻故事中的链接。

除了最近 AOL 针对 digg 的业务，将 netscape.com 的大量资产变成了 Digg 的克隆版本之外，最近许多人大声抱怨 Digg 上的用户群通过集体行动，能够将一个故事放到主页上，或者从主页上删除。

政治博客作者 Michelle Malkin 是第一批抱怨 Digg 的保守派或自由派用户团体删除另一方权威人士的帖子的人之一。最近，[另一个博客作者](https://web.archive.org/web/20230218164235/http://jesusphreak.infogami.com/blog/is_digg_rigged)分析了 Digg 主页的故事，并得出结论，一小部分强大的 Digg 用户一起行动，控制了主页故事的很大一部分。

对一些人来说，这是令人不安的，因为它去除了 Digg 的民主本质。与《纽约时报》等报纸不同，在《纽约时报》等报纸上，由一小群编辑决定什么是“新闻”，并将其纳入报纸，Digg 是一个更加精英化和民主的过程，读者实际上决定什么是有新闻价值的。如果 Digg 被一个相对较小的用户群体所破坏，那么 Digg 和 NYT 之间的区别就不那么明显了。

其他人回应说，这些群体只是 Digg 的铁杆用户，他们每天花很多时间在网上寻找好的故事在 Digg 上推广。Digg 根据用户提交的故事的成功程度对用户进行排名，少数用户对他们的 Digg 排名竞争激烈。理由是这些用户只是更擅长寻找故事。

今天，Digg 的联合创始人凯文·罗斯回应了这些抱怨。他支持争论的双方。凯文说，一群人互相推荐故事实际上是一件好事。但他也表示，Digg 将很快实施一种新的算法，这种算法对多样化的挖掘者群体比对集体行动的群体更为重要:

> 这一算法更新将着眼于挖掘故事的个人的独特挖掘多样性。遵循游戏模式的用户将具有较少的促销权重。这并不意味着这个故事不会被推广，这只是意味着一个更加多样化的个人群体需要认为这个故事值得放在主页上。

我认为这样做是对的。Digg 需要继续鼓励人们向他们的朋友推荐故事，但也要想办法在没有 Digg 用户组赞助的情况下，将真正独特和有趣的故事放到主页上。希望算法的改变会有所帮助。我最近向 digg 首席执行官 Jay Adelson 提出的另一个改进建议是:如果一个故事来自于阅读“即将到来的故事”区，那么 Digg 更重视这个故事。由于朋友们经常通过直接链接发送电子邮件或即时消息，这更有可能是一个群体的投票。一个来自即将到来的故事页面的挖掘者更有可能只是一个浏览故事并挑选他或她认为重要的故事的用户。
