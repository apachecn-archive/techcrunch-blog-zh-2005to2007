# MySpace 为美国西班牙语用户推出网站

 [](https://web.archive.org/web/20221003200906/http://latino.myspace.com/) 超过 2800 万居住在美国的人把西班牙语作为他们的第一语言(美国人口普查，2000)。他们现在在 latino.myspace.com T4 有了自己的 MySpace 版本。

在今天之前，美国、英国、日本、法国、德国、澳大利亚、爱尔兰、意大利、西班牙、加拿大、新西兰和墨西哥的互联网用户可以访问本地化版本的 MySpace。用户根据其 IP 地址的位置被重定向。有了 Latino.Myspace.com，用户可以选择直接去那个网站，而不是美国版。

MySpace 还推出了 la.myspace.com，这是一个面向拉丁美洲西班牙语使用者的泛区域网站。

这些本地化版本的服务并不禁止来自不同 MySpace 网站的人成为朋友。任何人都可以与 MySpace 中的任何人成为朋友，即使他们使用不同的 MySpace 网站，说完全不同的语言。
