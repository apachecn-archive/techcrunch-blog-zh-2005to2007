# 雅虎收购 SearchFox 资产

据可靠消息称，[宣布将于 2006 年 1 月 25 日关闭](https://web.archive.org/web/20221209121439/http://www.beta.techcrunch.com/2006/01/10/searchfox-to-shut-down/)的 [SearchFox](https://web.archive.org/web/20221209121439/http://rss.searchfox.com/) 已经将其资产出售给雅虎。至少一些员工也转投了雅虎。

没有标明价格。目前还不清楚这是一次直接的收购导致了宣布的关闭，还是雅虎在宣布后介入了一次机会主义交易。不管怎样，现在 SearchFox 产品将(或可能)继续存在下去，这很好。
