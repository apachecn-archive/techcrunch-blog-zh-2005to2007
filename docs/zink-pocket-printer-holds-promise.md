# Zink 袖珍打印机有希望

![](img/5ecc9af279e0063c39adce26c37daf35.png)

Zink 正在使用一种新的印刷技术，利用嵌入纸中的结晶墨水。结果是一个热激活打印解决方案，将创建快速，好看的照片。Zink 的第一款产品是一台小型袖珍打印机，大约一副扑克牌大小，售价不到 100 美元。

这台打印机可以很容易地连接到手机或电脑上，打印出有趣的小图片，你和你的朋友可以把它们放在冰箱上。发布日期的细节还不清楚，但这真的会让[宝丽来](https://web.archive.org/web/20201020064622/https://crunchbase.com/organization/polaroid) (Zink 的母公司)重新成为人们关注的焦点。
