# 《今日美国》走向移动

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/21/usa-today-goes-mobile/>

很难抗拒一个关于*《今日美国》*的小新闻变得字节大小的笑话，但我还是会。美国最畅销的报纸《今日美国》,将印刷新闻、短信、移动广告和移动搜索结合在一起。

用户注册*今日美国*的新闻、体育、八卦等短信提醒。将收到实时更新和赞助互动广告。简短的代码每天都出现在报纸的印刷版上。Gannett Digital (Gannett 拥有 *USA Today* )的移动战略和运营总监马特·琼斯说，这项服务“允许我们采用一个非交互式平台，一份报纸，并使印刷产品具有交互性。”
