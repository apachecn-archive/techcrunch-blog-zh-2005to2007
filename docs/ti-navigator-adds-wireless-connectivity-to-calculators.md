# TI Navigator 为计算器增加了无线连接

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/16/ti-navigator-adds-wireless-connectivity-to-calculators/>

![](img/75af6fc46b423ee51cc464ffdd3234c3.png)

我承认，当我第一次看到 TI Navigator 时，我不禁想为什么有人需要在计算器上使用无线连接。根据德州仪器的说法，TI Navigator 更多的是一种课堂设备，而不是学生个人的设备。它允许教师或教授实时查看学生的计算器，这样他们就可以看到人们在数学问题上遇到的困难。教师也可以使用它让学生提交问题，并通过 TI 计算器显示他们的工作。

只要确保你不开始玩一个疯狂粗糙版本的*超级马里奥兄弟*与 TI 导航仪连接，否则你的老师会看到你不能通过世界 1-3。最好在那些热门的 TI-84+技能上下功夫。
