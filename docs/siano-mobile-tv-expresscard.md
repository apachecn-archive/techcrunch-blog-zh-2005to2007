# 夏诺移动电视快捷卡

![](img/abe830747d66f78fa66fafc67ae9dc8a.png)

自从 ExpressCard 34 出现在苹果的 Macbook Pro 中，这个新插槽的一些项目就到处出现——没有什么真正的价值。夏诺移动和三星推出了 SuwonExpress 卡。将卡插入笔记本电脑的侧面，随时随地享受移动电视。该卡将与 CyberLink 的一些软件捆绑在一起，因此它应该不会是一个太糟糕的包。您将能够观看直播电视、录制节目和浏览屏幕上的节目指南。

发布日期和价格尚未确定，但请关注下个月 CES 上的一些公告。
