# 维亚康姆获得 Joost-TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/20/viacom-gets-joost-up/>

![](img/085d466e4fcf55731b9a374717df1551.png)

你可能还记得，上周，维亚康姆要求 YouTube 撤下 10 万部属于该媒体巨头的版权视频。事实证明，这不仅仅是因为有人上传了一个旧的霍尔奥茨视频，而是维亚康姆已经决定与 Joost 合作。

没错:由于 Joost 和 Viacom 之间的一项新协议，Joost 用户将能够享受来自 MTV、Comedy Central 和 Nickelodeon 等网络的节目。现在，如果每个人都正确出牌，Joost 很可能最终成为 YouTube 杀手。
