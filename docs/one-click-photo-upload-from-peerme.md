# 从 PeerMe 一键上传照片

今天我们将了解 PeerMe 的一键式照片上传工具。这是一个新推出的 Java 应用程序，允许用户将手机上拍摄的照片直接传输到他们的脸书页面，或与 Twitter 集成的一键博客。任何其他照片或博客网站也可以集成 PeerMe 的 API，并为用户提供相同类型的易用功能。

“我们正在开放我们的 API，以便任何其他公司都可以连接到这些 PeerMe 功能，并向其用户群提供一键照片上传、博客和其他受欢迎的功能。这是将 PeerMe 转变为领先的互联应用移动分发平台的最新举措，”公司首席执行官汤姆·拉塞特说。“PeerMe 产品的 Java 版本允许用户在移动时继续享受 PeerMe。你只需要一个无限制的数据计划，你就可以用 PeerMe 和你的朋友进行即时通讯。我们相信，一键式照片上传和一键式博客功能将会在全球用户中大受欢迎。”
