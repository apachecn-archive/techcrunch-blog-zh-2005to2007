# 基于富国银行浏览器的移动银行

![wellsfargo.jpg](img/65b5c5fa723ec5d04a1f814e55303b09.png)本周，富国银行&公司推出了富国移动，一种基于浏览器的移动银行解决方案，全国所有客户都可以使用。这项新服务允许客户通过支持移动的网络浏览器访问他们的 Well Fargo 账户。

富国银行执行副总裁兼互联网服务集团董事总经理吉姆·史密斯表示:“有了富国银行移动服务，客户无论身在何处，无论是想确保账户中有足够的资金还是想在账户间转账，都可以立即采取行动。”。“我们的移动银行解决方案受到了客户的启发。他们通过参与我们的试点项目并提供有价值的反馈，帮助我们设计了这种体验。客户的财务需求具有时效性和复杂性，我们在构建移动设备解决方案时就考虑到了这一点。"

通过该网站，客户可以检查他们账户的余额，查看交易历史，甚至在符合条件的账户之间转账，为您节省去银行的费用。
