# Profile Builder 今晚在 TechCrunch Meetup 上发布

[![null](img/0b4eccf183801ef56afdc68a234e19bf.png)](https://web.archive.org/web/20221130202722/http://www.crunchbase.com/company/profilebuilder) 在今晚的第九届 TechCrunch 见面会上，身份管理服务 [Profile Builder](https://web.archive.org/web/20221130202722/http://profil.es/) 在 Profile 上发布并演示了他们的产品。Profile Builder 的解决方案解决了由于越来越多的服务争夺我们的注意力而导致的在线身份碎片化问题。

使用配置文件生成器，您可以创建一个配置文件，将您所有的在线身份链接到一个可自定义的配置文件中。用户可以控制可见性和访问，同时跟踪其个人资料的性能和活动的统计数据。一个独特的 Profil.es 图标可以很容易地添加到任何可以放置 HTML 代码的地方，例如电子邮件、博客和个人网站。

我们目前的评论是稀疏的，因为我们将在今晚发布时第一次看到该网站。然而，有几个其他网站也试图将我们的在线身份联系在一起。[profile link](https://web.archive.org/web/20221130202722/http://www.beta.techcrunch.com/2006/12/28/profilelinker-takes-meebo-approach-to-social-networking/)、[profile actic](https://web.archive.org/web/20221130202722/http://forums.beta.techcrunch.com/forums/thread.jspa;jsessionid=D7011C30CD811FEBE50838EB5D2C99CA?messageID=4496&#4496)和 [Loopster](https://web.archive.org/web/20221130202722/http://www.beta.techcrunch.com/2007/04/07/loopster-makes-friends-of-social-networks/) 是我们见过的一些身份聚合器。像 [WikiYou](https://web.archive.org/web/20221130202722/http://www.crunchbase.com/company/wikiyou) 、 [Spock](https://web.archive.org/web/20221130202722/http://www.crunchbase.com/company/spock) 和 [Wink](https://web.archive.org/web/20221130202722/http://www.crunchbase.com/company/wink) 这样的身份搜索引擎也在帮助解决身份问题。
