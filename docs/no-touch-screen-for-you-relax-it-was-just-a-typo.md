# 没有触摸屏:放松，这只是一个打印错误

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/02/no-touch-screen-for-you-relax-it-was-just-a-typo/>

老天，我真的等不及苹果发布一款[触屏](https://web.archive.org/web/20150506124256/http://crunchgear.com/2006/10/18/touchscreen-ipod-let-the-speculation-continue/) / [触控式](https://web.archive.org/web/20150506124256/http://crunchgear.com/2006/10/26/apples-new-ipod-touch-bezel-not-touchscreen/)的东西了，因为我实在受不了这种谣言废话太多了。这一次，有人在苹果电脑 iPod 开发者文档中发现了一个所谓的秘密失误。原始文档只有一条注释，说明“从触摸屏机型开始，带显示屏的 iPod 机型支持本文档中描述的所有其他 Notes 功能。”但正如你在上面的图片中看到的，苹果已经介入并澄清了这个笔记，将其改为“触摸滚轮”当然，你可以继续猜测。
