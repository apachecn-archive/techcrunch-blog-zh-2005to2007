# RoombaCtrl 用手机控制你的 Roomba

哦，是的。现在你可以用你自己的来覆盖自动导航的 Roomba 控件。使用你布满火腿的手指来引导你的机器人女仆穿过你的房子，除了一部支持蓝牙和 Java 的手机之外什么都不用。哦，你还需要一个 RooTooth 设备来连接 Roomba，以便在 Roomba 上实现蓝牙连接。
