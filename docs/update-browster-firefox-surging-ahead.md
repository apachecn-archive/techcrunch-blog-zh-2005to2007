# 更新-浏览器(火狐向前冲)

Browster 的首席执行官 Scott Milener 刚刚在我们上一次发布的 Browster 上留下了一个非常有趣的评论。在支持 Firefox 仅仅几个星期后(实际上，大约一个星期)，它就占据了总下载量的 1/3:

> 谢谢你的发帖和对布朗斯特的赞美。我们非常高兴能够支持 Firefox。我们现在收到了在所有平台和所有浏览器上使用 Browstr 的请求——有很多事情要做！
> 
> 有趣的是，我们的 Firefox 安装数量已经是我们安装数量的 1/3(每天数千)。正是这种接受推动了 Firefox 社区和像我们这样的公司支持这个平台。请写信给我，告诉我任何关于改进的想法或任何其他想法。
