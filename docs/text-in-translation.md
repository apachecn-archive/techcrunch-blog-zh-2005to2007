# 翻译文本

![fen_network_logo.gif](img/b8f73a37036fb61379c84f098fb44e7b.png)数字世界的发展导致了 LOL、K 甚至 BFF 等首字母缩写词的使用，但是当你的孩子开始键入 A/S/L、PRW 或 MorF 时，你需要参考[词汇表](https://web.archive.org/web/20130628200040/http://life.familyeducation.com/internet-safety/instant-messaging/51253.html?page=3&detoured=1)进行翻译。FamilyEducation.com 提供了一个家长指南来理解网上缩略语，其中就有这样一个词汇表。它适用于即时消息和 FaceBook 等网站，以及孩子手机上发送和接收的短信。即使你不是发短信的青少年的父母，你可能会发现词汇表有助于缩短你的信息以适应 160 个字符的限制，或者翻译朋友发来的信息。
