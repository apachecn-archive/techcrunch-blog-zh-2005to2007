# 硬盘今天

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/13/the-hard-drive-is-50-today/>

![](img/e80debbfb5871ea3d539abfff7b772c7.png)

50 年前的今天，IBM 推出了 RAMAC 305 和 RAMAC 350。RAMAC 系统是 IBM 首次尝试在磁盘上存储信息。原来的硬盘驱动器甚至在周二晚上为它举办了一个派对，目前存放它的博物馆表示，他们希望将 RAMAC 恢复到工作状态。我不知道他们会用那惊人的 5MB 存储空间做什么，但是，你知道，去吧。

所以亲亲你的电脑吧，今天是他的曾曾曾祖父的生日。
