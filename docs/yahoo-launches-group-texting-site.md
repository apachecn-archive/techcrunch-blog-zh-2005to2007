# 雅虎！推出群发短信网站

 [](https://web.archive.org/web/20220810115843/http://mixd.yahoo.com/) 雅虎！随着他们的新网站 [Mixd](https://web.archive.org/web/20220810115843/http://mixd.yahoo.com/) 的推出，他们开始涉足群发短信业务。TechCrunch 今天被告知该网站的存在，但我们无法确定它已经存在了多长时间。

Mixd 允许用户建立群组，并在群组内发短信或分享照片。文本和照片可以在线管理，也可以在手机上管理。

![mixdscreen.jpg](img/a30497d3b798e9d255ccddc6844295e7.png)短信收件人可以回复所有人，通过只回复一个人来“耳语”，或者通过关闭信息链来“静音”。但是从一些初始测试来看，所有这些功能都不是最容易弄清楚的。注册这项服务很简单，但管理联系人、群组以及分配给群组和命令的号码却很难。雅虎！试图通过让用户输入一个数字而不是一个单词来简化事情，我不确定这是否会让事情变得更简单。

它看起来像雅虎！会从“酷”的角度出发。[“关于”页面](https://web.archive.org/web/20220810115843/http://mixd.yahoo.com/about)说，“Mixd 是关于外出的。协调最后一分钟的聚会，分享手机中的图片和视频，并在我们为您创建的网站上自动记住昨晚。”

我们最近已经盯上了几个[群组短信服务](https://web.archive.org/web/20220810115843/http://www.beta.techcrunch.com/2006/09/27/a-look-at-eight-multi-person-sms-services/)，比如 [Zemble](https://web.archive.org/web/20220810115843/http://www.beta.techcrunch.com/tag/zemble/) 、[躲避球](https://web.archive.org/web/20220810115843/http://www.beta.techcrunch.com/tag/Dodgeball/)和 [Twitter](https://web.archive.org/web/20220810115843/http://www.beta.techcrunch.com/tag/Twitter/) 。有点令人惊讶的是，雅虎！想参与这个游戏，但是为什么不呢？

到目前为止，这项服务只适用于美国的电话号码。收取普通短信费。
