# 沃达丰将收购威瑞森？

大 V 们会合并吗？这是网上流传的谣言。正如我们的姐妹网站 CrunchGear 所发布的，[沃达丰已经表达了对威瑞森通信](https://web.archive.org/web/20130628164834/http://crunchgear.com/2007/07/16/vodafone-denies-verizon-takeover-rumors/)的兴趣。两家公司已经是合作伙伴，但合并可能会成为真正的大型手机巨头。

由于沃达丰的 3G 网络在英国占主导地位，而威瑞森在美国也有自己的“最可靠的网络”，这可能会使国际移动电话更便宜、更可靠、更酷！

然而，沃达丰迄今否认有任何交易。但是我们都知道否认只是任何交易的第一步！
