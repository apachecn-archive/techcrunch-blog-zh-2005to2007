# Oodle 融资 1100 万美元——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/03/21/oodle-adds-11-million/>

[![oodle](img/dd332c344a58e499d64468e1f4fee375.png)](https://web.archive.org/web/20210614234251/http://oodle.com/) 分类搜索引擎[不减当年](https://web.archive.org/web/20210614234251/http://oodle.com/)获得[一堆现金](https://web.archive.org/web/20210614234251/http://venturebeat.com/2007/03/20/oodle-giving-you-classifieds-with-a-touch-of-help/)今天，来自新投资者 JAFCO Ventures 的 1100 万美元，以及他们之前[轮 500 万美元](https://web.archive.org/web/20210614234251/http://www.thealarmclock.com/mt/archives/2007/03/classifieds_pub.html)的领导人，Greylock Partners 和 Redpoint Ventures。像 LiveDeal、 [Edgeio](https://web.archive.org/web/20210614234251/http://www.beta.techcrunch.com/2006/10/24/edgeio-closes-5-million-series-a-financing/) 和 [Vast](https://web.archive.org/web/20210614234251/http://www.beta.techcrunch.com/2006/03/14/vast-%e2%80%93-aggregating-listings-from-the-whole-web/) 一样，Oodle 重新打包并聚合了来自网络的分类列表，使它们可以很容易地按类别、属性和位置进行搜索。Google Base 也涉足了这一领域。Oodle 的模式通过托管汽车、房地产、租赁、工作、个人、商品、门票、宠物和服务的列表，几乎覆盖了所有已建立的垂直搜索。

Oodle 通过直接发布和一系列本地列表合作伙伴扩大了其列表，它在 Yell.com、local 等网站上联合了这些合作伙伴。包括 Engage.com、Local.com、售票员、房地产经纪人、大学以及报纸。然而，去年他们失去了 Craigslist 的订阅。Edgeio 和 Vast 的发展方式略有不同。Edgeio 一直在与国际网站的 feed 合作，通过分享上市费来赚取收入(Edgeio marketplaces 就是一个很好的例子)。通过搜索登录和搜索表单背后的深层网络，Vast 一直在发展壮大。

Oodle 报告了来自美国和英国 75，000 多个来源的 2，000 万个活跃列表，相比之下，Edgeio 的国际列表超过 1 亿个，Vast 的分类列表超过 70，000 个。然而，Oodle 有一些比这些更先进的搜索功能，在有意义的地方增加了价格跟踪和地图。他们的价格跟踪是 Mpire 在垂直购物领域提供的更基本的版本。

***披露:**迈克尔·阿灵顿是 Edgeio 的创始人和董事。*
