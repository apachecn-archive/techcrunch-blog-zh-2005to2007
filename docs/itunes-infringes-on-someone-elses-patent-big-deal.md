# iTunes 侵犯了别人的专利——大事| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/01/itunes-infringes-on-someone-elses-patent-big-deal/>

哦，苹果，你什么时候才能学会？硅谷巨头的最新诉讼是针对他们的面包和黄油，iTunes。个人网络有限责任公司(Individual Network LLC)提交的诉状称，iTunes store 侵犯了一项迎合定向媒体交付的专利。对我来说听起来像很多废话，但是我不是专利律师。

有争议的专利提供了一个基于个人信息的定制数字内容列表，这听起来很像 iTunes 的“只为你”和其他一些功能。如果个人网络赢了这场对苹果的诉讼，它可能会让整个房子倒下来——某种程度上。他们不仅会从 iTunes 上销售的所有内容中获得版税，还会从 iPod、Apple TV 和即将销售的 iPhone 中获得分成。还能更糟吗？是的，它可以。个人网络也在争取永久禁令，禁止所有未经许可的 iPod 和 iTunes 商品的销售。苹果害怕了吗？大概不会。他们会用 iPods 支付所有人，并在 iTunes 上支付免费积分。哈！
