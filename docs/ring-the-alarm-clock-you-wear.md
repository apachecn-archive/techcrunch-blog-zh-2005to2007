# 戒指:你戴的闹钟

![](img/8170c140907f9ef859818eafe85e52da.png)

又是一天，另一个由 Yanko 设计的产品出现了。这一次，我们考察了一个名为 Ring 的闹钟的新设计理念。Ring 由两个弹性振动环组成，在闹钟底座上充电。你给每个铃声设定你想醒来的时间，然后让它们充电。一旦充好电，在你昏过去之前戴上它，你会以一种全新的方式醒来。

每个戒指都会轻轻震动，慢慢将用户从睡眠中唤醒。不再有嘈杂的闹钟或[时钟](https://web.archive.org/web/20220119233523/http://www.crunchgear.com/?s=clocky)跑来跑去要求你离开舒适的床。遗憾的是，这有可能永远不会发展成成熟的工作产品。似乎是早上醒来的最好方式之一。
