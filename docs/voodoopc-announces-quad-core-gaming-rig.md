# VoodooPC 发布四核游戏平台

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/03/voodoopc-announces-quad-core-gaming-rig/>

![](img/28fc5831f0684c5b4203af412ec60534.png)

如果你想买一台新的游戏电脑，那么看看 VoodooPC 的新四核 Omen 吧。Omen 采用英特尔新的[四核酷睿 2 双核至尊处理器](https://web.archive.org/web/20160302005018/http://crunchgear.com/2006/11/02/intel-core-2-quad-damn-thats-fast/)，2GB DDR2 内存，80GB 硬盘和一个 Creative Labs X-Fi Xtreme 音乐声卡。Omen 是完全可升级的，但由于其 5500 美元的价格，您可能需要自己添加额外的硬件。

VoodooPC 使用中间冷却器和“先进的室内气流布局”来保持电脑凉爽和安静。虽然 VoodooPC 说预兆是一个近乎无声的系统，但在我相信之前，我必须亲自尝试一下。即使高端游戏个人电脑有液体冷却，他们仍然有正常的风扇你可以听到。至于显卡，你可以在 ATI Radeons 和 Nvidia GeForces 之间挑选。如果你想获得金牌，你可能需要安装两个 GeForce 7950s，以实现四核 CPU 和四 SLI 设置。
