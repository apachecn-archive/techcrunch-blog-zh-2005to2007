# iPhone 没有预售

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/14/no-pre-sales-for-iphone/>

![iphone425.jpg](img/d53a0ce47362b545200c8ddf4a0cb34f.png)

好，好，好。BGR 又一次得到了独家新闻，这次是关于 iPhone 的。根据一份内部文件，iPhone 将以先到先得的方式出售。如果你认为你的名字出现在名单上会确保你在发布日会有一个，那么再想想吧。我现在就能看到…成千上万疯狂的人在美国电话电报公司商店前露营。这些台词将超过等待第一集的星战迷们的台词。还有一些其他的趣闻，但没有什么是我们不知道的，比如 iPhone 拥有“领先任何手机 5 年的高级软件”，或者定价和发布日期不能与客户讨论的事实。整个非预售的事情将会激怒一些人。
