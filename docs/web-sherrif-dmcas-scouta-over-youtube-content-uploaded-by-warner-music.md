# 网络治安官 DMCA 对华纳音乐上传的 YouTube 内容的搜索

在线音乐推荐和分享网站 [Scouta](https://web.archive.org/web/20221204183248/http://www.scouta.com/) 收到了来自知识产权执法组织[网络治安官](https://web.archive.org/web/20221204183248/http://www.websheriff.com/)的 DMCA 通知，要求 YouTube 上托管并在 Scouta 上显示的一段白色条纹视频因侵犯版权而从 Scouta 上删除。尽管视频是由华纳音乐上传到 YouTube 的，根据维基百科，白条乐队目前的唱片公司[。](https://web.archive.org/web/20221204183248/http://en.wikipedia.org/wiki/The_White_Stripes)

这个问题不同于 TechDirt 报道的[类似争议。由于 YouTube 上的内容的合法性受到质疑，在 Scouta 的情况下，视频是由权利持有人合法上传到 YouTube **的，并授予视频嵌入第三方网站的全部权利。**](https://web.archive.org/web/20221204183248/http://techdirt.com/articles/20070703/144358.shtml)

![scouta.png](img/e77055b3508a30042d300d6e757b7ef2.png)

Web Sheriff 等公司的愚蠢显然与 DMCA 通知 Scouta ( [pdf](https://web.archive.org/web/20221204183248/https://beta.techcrunch.com/wp-content/uploads/2007/07/scoutadmca.pdf) )未能遵守该法案没有关系。

Web Sheriff 未能按照 DMCA 通知中的要求演示在哪里可以查看原始内容；简单地说，他们无法证明原始内容的来源是侵权的，因为 Scouta **显示原始内容是由权利持有人合法上传的。该通知要求 Scouta“从侵权文件位置移除被侵权的标题”，尽管 Scouta 没有托管该文件(YouTube 托管)。该法案明确规定，此类通知只应发送给那些托管侵权文件的人；如果内容确实违反了该法案(显然没有)，那么 YouTube 应该已经收到了通知。**

像 Web Sheriff 这样的公司滥用 DMCA 凸显了法律的可悲失败。像 Scouta 和其他公司这样的初创公司不应该受到像这样的错误指控。美国立法者迫切需要将 DMCA 扔进它理应属于的历史领域。

对于那些喜欢白色条纹的人来说:有问题的视频，由华纳音乐和谷歌/ YouTube 提供。
