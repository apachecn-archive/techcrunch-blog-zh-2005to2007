# Wii 的 Opera“试用版”将于周五登陆 Wii 商店

![](img/6790fa7defc7deeb960da4301bb1a55a.png)

从周五开始，你将可以在 WiiSports 的间隙赶上 CrunchGear 的帖子。

Opera 刚刚宣布，Wii 的“试用”版 Opera 将于本周五登陆 Wii 商店。最棒的部分:它是免费的，至少现在是这样(“试用”一词往往意味着价格飙升随时可能到来。)所以趁勺子免费的时候把它舀出来。
 [戏曲](https://web.archive.org/web/20210412010555/http://www.opera.com/pressreleases/en/2006/12/19/)

编辑:Opera for Wii 的最终版本将于 3 月至 6 月底对所有 Wii 用户免费开放。之后，它将在 Wii 商店渠道以 500 Wii 积分出售。布雷克
