# Wellsphere 推出健康 2.0

[![](img/0ae6688d7bd5a5d277d91955d7472f55.png)](https://web.archive.org/web/20230215122319/http://wellsphere.com/)[wellshepere](https://web.archive.org/web/20230215122319/http://wellsphere.com/)明天将推出他们健康社区网站的 alpha 版本。它加入了我们已经报道过的许多其他健康相关的搜索网站。Wellsphere 关注的是构成“健康”的日常健康选择，这种选择你不会问你的医生，而是咨询你的同行。Wellsphere 旨在通过为用户提供与相似的人和他们所在地区的健康相关资源的访问，来帮助人们进行这些选择和激励。这些资源被分散在他们收集的个人资料和健康相关位置的数据库中。

他们的个人资料很像任何社交网站，只是增加了一个类似博客的功能，叫做“井圈”。健康圈是附在你的个人资料上的评论线索，涉及你想要实现的目标或你对健康的一些具体知识。任何注册用户都可以订阅这个领域的更新，或者通过发表自己的意见来加入对话。你可以想象会有很多重复的讨论，Wellsphere 通过允许成员为其他用户投票来处理这些问题。

与社交网站上的纯粹交友不同，Wellsphere 有“活动伙伴”。希望人们更有可能坚持他们的健身目标，如果他们作为一群依靠他们出现的人的一部分安排活动，我同意。然而，虽然搜索功能可以让你找到你所在地区有相似兴趣的人，但活动计划则由会员决定。这似乎是一个类似于 MatchActivity 的解决方案，将人们匹配起来进行马拉松训练，或者每周一次的重量训练将是一个理想的补充。埃米尔似乎是这个职位的完美候选人，考虑到她已经在寻找竞选伙伴。

网站的最后一部分是健康相关场所的数据库，如健身房和治疗师，以及健康餐馆的建议。你可以按地点搜索数据库，寻找你所在地区的健身房或者脊椎指压治疗师。看起来你甚至可以通过健身房播放的音乐来过滤你的搜索。数据库与网站的其他部分连接起来，包括哪些成员去哪里，从而更容易找到你所在地区的健身伙伴。
