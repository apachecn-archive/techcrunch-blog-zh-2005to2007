# TomTom GO 910 被 Virii 困扰

![](img/c27b4c5361e7413ca8cd01e3a13293d9.png)

如果你买了一个 TomTom GO 910，你可能要找的不仅仅是方向。结果发现，一小批设备包含两种不同的非致命木马，当你将 910 连接到 PC 时，你的反病毒程序可以检测到它们。TomTom 迟迟不承认其设备确实存在问题，但表示这没什么大不了的，如果你有受感染的设备，请联系客服。

想象一下，如果病毒影响了用户的旅行目的地，结果会有多滑稽…
