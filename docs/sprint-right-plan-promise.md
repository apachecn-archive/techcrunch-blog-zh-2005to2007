# 冲刺正确的计划承诺

![](img/5fedc8d879058fd8eb89d203e4c020fa.png)

我敢肯定 Sprint 非常爱你，但是留住新客户是无线行业赚大钱的方法。现在，斯普林特公司的新客户将能够使该公司信守其新的正确计划承诺。它的工作原理是允许新客户在激活的前六个月内修改他们的计划。这意味着，如果你需要更多的通话时间，需要一部新手机，或者不需要额外的数据计划，你可以轻松地做出必要的调整，而不必续签两年的合同。

听起来像是朝着 Sprint 的正确方向前进。虽然它可能有一个强大的数据网络，但 Sprint 的语音通话要么是成功的，要么是失败的，我肯定希望能够根据我的服务做出改变。还增加了一项新的 30 天保证，如果您不满意，可以完全取消您的计划。
