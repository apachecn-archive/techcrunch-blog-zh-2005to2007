Windows Live Messenger速度超出测试版

原文：https://web.archive.org/web/http://www.techcrunch.com/2006/06/19/windows-live-messenger-speeds-out-of-beta/trackback/


第一款 Windows Live 产品全球脱离beta状态将于今晚由微软发布。Windows Live Messenger公测版仅用了六周时间。当测试版发布时，我们回顾了现在可用的大多数功能。

最值得注意的是通过界面和专用硬件包括语音通话，离线信息和一个简单的文件共享系统。

在等待Vista发布的人们多年的嘲笑和微软痛苦缓慢的开发的普遍声誉之后，Live Messenger测试版的发布可能是这个新的后盖茨时代运营严重转变的第一个迹象。当周四宣布盖茨将放弃他作为首席技术架构师的职位给前首席技术官雷·奥茨时，Live被认为是公司未来的主要发展方向。

该公司表示，明年将在全球推出20多项新的Windows Live服务。
