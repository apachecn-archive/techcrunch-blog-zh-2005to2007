# 索尼凭借 XDR-S3HD 等产品进入高清收音机市场

如果你在高清收音机上坚持到索尼加入游戏(那些没有跑出去买[剑桥的收音机](https://web.archive.org/web/20210305022427/http://crunchgear.com/2007/05/25/cambridge-soundworks-radio-820hd-review/)的人)，你的等待正式结束了。索尼宣布，它将长期完全放弃高清收音机(请注意，是在假期)，并将在未来几年开发和销售支持高清收音机的产品。这一切都始于 XDR-S3HD 台式收音机。

> 具有模拟环绕声功能的运动立体声扬声器、内置 AM/FM/HD 数字调谐器和独立的低音和高音控制，具有高清收音机功能的台式收音机在一个小包装中提供高质量的立体声。

这个小盒子还有一个辅助输入，这样你就可以把一个 ~~iPod~~ S2 运动随身听 MP3 播放器连接到它上面。哦，这也是一个闹钟。预计 7 月份售价约为 200 美元。

索尼还宣布了一款带高清收音机(XT-100HD)的移动调谐器，但我没有贴上照片，因为它看起来很无聊。这是一个有输入和输出的小黑盒。你明白了。
