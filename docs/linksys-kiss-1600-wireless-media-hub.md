# Linksys KiSS 1600 无线媒体集线器

今天在 CeBIT 上推出的是 Linksys KiSS 1600 无线媒体集线器。这是一个必备的媒体设备，可以通过互联网或网络上设置的任何其他设备将包括高清视频在内的各种数字内容传输到您的电视。亲吻取代你的 DVD 播放器，因为它会通过 HDMI 将你的整个 DVD 收藏升级到 720p。大多数规格仍然未知，但它将花费你大约 500 美元。
