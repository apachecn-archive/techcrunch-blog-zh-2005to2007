# 吴:苹果对所有 iPods 和笔记本电脑都采用固态技术？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/07/wu-apple-going-solid-state-for-all-ipods-portables/>

据 Apple Insider 报道，基于硬盘的 iPod 将很快成为过去。在分析了苹果基于 NAND 闪存的存储供应链后，美国技术研究公司的 Shaw Wu 得出结论，苹果 iPod(标准版，不仅仅是 Shuffle 和 Nano)的未来版本将基于闪存，这意味着更高的可靠性和更长的电池寿命。NAND 比标准 HDD 更贵，但价格在下降，同时容量在增加，这是唯一能让变化奏效的组合。

此外，吴认为批量购买高容量 NAND 驱动器是对基于闪存驱动器的“MacBook Thin”笔记本电脑的确认，这种笔记本电脑已经传闻了几个月。虽然基于 NAND 的笔记本现在已经上市，但苹果很可能不会生产这么多。

这两条产品线的综合购买力将直接有助于获得笔记本和 iPod 设备所需的 32 和 64GB 驱动器的合理价格，因为众所周知，苹果不仅批量购买 NAND，而且还决定三星和其他制造商的价格。这是否会降低这些新型号的价格？不太可能，预计一开始会支付溢价，随着产量的增加，价格会下降。

总的来说，我们认为 iPod switch 是一个显而易见的选择，但是笔记本电脑却是个问题。苹果可能会在某个时候将 MacBook 系列切换到 NAND，至少是 Pro，以利用速度和电池寿命的节省。然而，全新产品线的想法仍然令人怀疑。
