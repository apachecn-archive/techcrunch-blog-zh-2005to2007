# Velocity Micro 宣布推出四核工作站

与[其他消息](https://web.archive.org/web/20130627215438/http://crunchgear.com/2006/09/26/intel-challenges-pc-designers-to-think-small-sexy/)今天，在英特尔信息技术峰会上，Velocity Micro 发布了其最新的 ProMagix W160 工作站，该工作站采用了英特尔即将推出的四核处理器(代号为“Kentsfield”)。这对普通消费者意味着什么？不是很多。但是，如果您是一名拥有大量电子表格的日间交易员，并且喜欢同时听音乐、下载盗版电影和玩游戏，那么单个芯片上的四个处理核心将会非常方便。
