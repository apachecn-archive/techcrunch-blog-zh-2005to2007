# 适用于 iPod 的 Mustek Paragon TW450

就动力而言，这可能不算过分，但我认为这款更符合美学。Mustek 的 Paragon TW450 配备了 45 瓦的放大器和 5.25 英寸的双线圈低音炮。它有 38 英寸高，有光滑的黑色或白色表面。

这里的主要卖点是外观。它不是最强大或声音最好的 iPod 基座，但我不认为它是针对音响发烧友的(他们可能不会用 iPod 来做家庭音频)。它将很快在 Mustek 上市，售价 250 美元。
