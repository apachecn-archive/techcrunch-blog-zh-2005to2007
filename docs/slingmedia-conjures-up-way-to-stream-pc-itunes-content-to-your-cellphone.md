# SlingMedia 想出了一种将电脑 iTunes 内容传输到手机的方法

[![atvsling.jpg](img/625f25970d6b7e2a5a04507fd78cf7ad.png)](https://web.archive.org/web/20210227030624/https://beta.techcrunch.com/wp-content/uploads/2007/04/atvsling.jpg "atvsling.jpg")

在不久的将来，你将能够把你所有的 iTunes 内容传输到你的手机上。怎么了？SlingPlayer 的制造商 SlingBox 表示，它正在开发能够与苹果电视(T4)配合使用的软件，让用户能够将内容从个人电脑传输到苹果电视，最后传输到手机。(前提是手机能与 SlingPlayer 兼容——目前仅限于运行 Windows Mobile、Palm 和 Symbian 的手机。)这不是最精简的方法，但这是一个开始。也许如果苹果公司设计的 iPhone 是开箱即用的，我们就不必依赖第三方来收拾残局了。
