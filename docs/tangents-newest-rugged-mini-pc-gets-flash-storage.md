# Tangent 最新的加固型迷你电脑配有闪存

如果我有一辆车，车上有一台电脑，我希望那台电脑是 Tangent 的坚固的迷你无风扇电脑。这种小型计算机设计用于“工厂车间、野外、安装在手推车或车辆上，或其他恶劣环境中”，能够承受冲击、灰尘、振动、极端温度和电磁干扰。在这个最新版本中，该公司甚至放弃了传统的硬盘驱动器，代之以紧凑型闪存驱动器。少了一个需要担心的活动部件。

这个盒子只有 4 磅重，待机时仅用 6 瓦，开机和闲置时用 24 瓦，完全使用时用 44 瓦。它运行在 Windows XP Pro 上，您可以选择英特尔赛扬 M、酷睿单核或酷睿双核处理器。它还使用了兼容 Cisco LEAP 的 Wi-Fi 接收器，因此您可以打开无线网络。

这款坚固的 Mini 现在可以直接从 Tangent.com 的[买到，起价 1195 美元。有点贵，但这是一个专业的电脑，所以你要期待这一点。](https://web.archive.org/web/20201020051057/http://www.tangent.com/)
