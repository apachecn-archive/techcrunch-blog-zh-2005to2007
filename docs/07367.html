三星二十一点:问谁？–TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2006/11/30/samsung-blackjack-q-who/



见见我的新宠手机。Cingular的三星21点是相对普通的竞争对手中名副其实的神童。不仅每次你拿出来它都会引人注目，而且它的性能会超过大楼、街区、邻居等任何一部智能手机。

Blackjack最引人注目的方面是它的尺寸，尺寸仅为4.45 x 2.32 x 0.46英寸，仅略大于Razr (2.08 x 3.85 x 0.5英寸)，小于其最接近的竞争对手Moto Q (4.6 x 2.5 x 0.5英寸)。与我用过的其他智能手机不同，它非常适合放在我的口袋里，这真是一个巨大的福利。

但不要以为只是一张漂亮的脸。21点的出拳无与伦比。在它的外表之下是220Mhz TI OMAP双核处理器和64MB RAM/128MB ROM。这足够完成任务了。

在使用Cingular 8125大约一年后，我开始厌恶Windows Mobile 5。它又慢又笨重，使用起来几乎总是很麻烦。我已经不再认为WinMo5应该为8125的失败负责。我错了。

21点使用相同的操作系统，精确度更高。屏幕切换迅速，没有明显延迟，应用程序几乎没有任何等待时间。例如，敏捷信使可以在几秒钟内启动。

开箱即用，它支持几乎所有可以想象的电子邮件平台，包括push和POP3/IMAP4。我想这有点苦乐参半。能够保持联系是很好的，但是一直保持联系会有点烦人。不过，这不能怪21点，这只是它的工作。

到目前为止，Blackjack最大的缺点是电池寿命，三星似乎已经意识到这个问题，因为它包含了两块电池和一个独立的充电器。手机文献说它可以达到5.5小时的通话时间，但我非常怀疑这种说法。它还声称手机可以待机11天，这几乎是不可能的。它在我的桌子上闲坐一天就用完了半块电池。我不确定实现这些要求需要什么条件，但我猜这两个条件都不会是真实的。

21点的另一个缺点实际上源于它的小尺寸。钥匙很小，尤其是对于像我这样手大的人来说。这需要一些时间来适应，但这是可行的。

另一个巨大的装饰性吸引力是可以投影1280 x 960的SXGA屏幕(好的，这是一个可以显示320× 240的QVGA屏幕。虽然不是SXGA，但看起来也不错。感谢理查德抓住了疏忽)。这可能不是我见过的最好的手机屏幕，但它绝对接近顶部。

有一堆扣带的具体功能，特别是一些音乐相关的。除了XM广播流，还有一个有争议的功能，音乐ID服务。基本上，当你听到一首你喜欢但不知道名字的歌时，拿出你的21点，让音乐为你播放这首歌。我试了大概五首歌。就流行歌曲和主流歌曲而言，这是相当准确的(你可以很容易地说出这些歌曲的名字)，但在更难的东西上，它完全失败了；你真正需要的东西。

电话接收非常好。还没有掉线或未接电话。拨号系统工作正常，你可以通过一个专用的主屏幕按钮快速访问联系人列表。

另一个不错的触摸是允许快速浏览窗口的滚轮。如果你像我一样有一个庞大的联系人列表，那么这将是一个受欢迎的功能。

哦，还有一件小事，我猜你们中的一些人可能会感兴趣:21点支持UMTS/HSDPA无线。很快，很快，很快。在我的8125上使用EDGE一段时间后，我开始认为“无线宽带”只是一个公关术语。公关人员抛出的一些东西，让移动产品听起来很花哨。我真的没有考虑到真正的无线宽带的含义，但我承认错误。

我在纽约、芝加哥和新奥尔良都玩过21点。纽约是三个城市中3G接入最差的。芝加哥和新奥尔良的信号覆盖同样出色，我的3G小图标总是在那里。

鉴于21点的纤薄外观，奇特的功能和一般的酷因素，我认为三星和Cingular在这里有一个真正的赢家。就智能手机而言，我真的认为目前没有更好的了。我给了21点最好的字节，我敢说这是我用过的最好的全能手机。

Cingular现在可以以199.99美元的价格购买两年合约，如果你不想放弃自己的灵魂，价格为449.99美元。
