# Update - Del.icio.us(搜索和推荐)

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/15/update-delicious-search-and-recommend-features/>

## 概述:

社交书签服务 Del.icio.us (见上面之前的简介)最近推出了两项新的重要功能——[推荐](https://web.archive.org/web/20230219030328/http://blog.del.icio.us/blog/2005/08/people_who_like.html)和[搜索](https://web.archive.org/web/20230219030328/http://blog.del.icio.us/blog/2005/08/search_me.html)。

这两个功能都是非常需要的工具，帮助用户找到标记的 URIs。这些工具将极大地帮助用户研究相关标签，并找到他们感兴趣的好的相关内容。

## 建议:

Josh [前几天发布了一个推荐工具](https://web.archive.org/web/20230219030328/http://blog.del.icio.us/blog/2005/08/people_who_like.html):

> **喜欢推荐的人也喜欢…**
> 
> 我们已经发布了新的推荐引擎。如果你在一个标签中保存了十个以上的网址，你将会得到几个网址以及系统认为你会感兴趣的其他人的标签。URL 页面现在也提供了查看其他相关 URL 的机会。

如果你在某个特定的 del.icio.us 用户页面(del.icio.us/username/tag)，有一个链接可以看到其他人用过的“推荐”标签。这些信息对于研究特定标签以及其他人如何标记相似的 URIs 非常有用。书签的 URL 页面中也推荐相关的 URL。

[布莱恩·德尔·维奇奥](https://web.archive.org/web/20230219030328/http://hybernaut.com/delicious-recommendation)(很好的截屏)[库剪辑](https://web.archive.org/web/20230219030328/http://libraryclips.blogsome.com/2005/08/15/delicious-recommendations/)和[绍里尔·杜瓦尔](https://web.archive.org/web/20230219030328/http://www.socio-kybernetics.net/saurierduval/2005/08/delicious-recommends.html)也写到了这一点。

## 搜索

[约什几小时前发布了搜索](https://web.archive.org/web/20230219030328/http://blog.del.icio.us/blog/2005/08/search_me.html):

> **搜我**
> 
> 我发布了一个简单的面向所有用户的搜索实现。如果你登录了，点击“搜索”，然后确保你勾选了“整个网站”

这是一项急需的服务，其他人试图填补乔希建立这项服务时的空白。现在在你的个人 Del.icio.us 页面的顶部有一个搜索栏(你必须登录)，你可以搜索你自己的标签或者点击“搜索整个网站”，得到所有用户的标签结果。我已经测试过了，效果很好。

我不确定新的搜索功能是否允许搜索多个标签-一次尝试给了我一个内部服务器错误。这是一个重要的特性，可以过滤多个标签的结果。

我希望他能在不久的将来为结果添加一个 feed 链接。
