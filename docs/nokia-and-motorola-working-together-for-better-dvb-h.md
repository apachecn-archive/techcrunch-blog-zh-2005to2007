# 诺基亚和摩托罗拉携手建设更美好的 DVB-H

![](img/5267531db6db2374c5d5a99cf5e01c98.png)

看起来摩托罗拉和诺基亚都需要给我们带来像样的移动电视。今天，诺基亚发布了一份[新闻稿](https://web.archive.org/web/20130627212057/http://press.nokia.com/PR/200609/1074515_5.html)声明他们将开始与摩托罗拉交换意见，只要他们在 DVB-H 领域有新的发现。显然，两家公司都认为“移动电视市场将呈指数级增长——到 2010 年，预计全球销售的支持 DVB-H 的移动设备将超过 5000 万台。”所以理论是，如果他们一起工作，他们可以确保一旦技术为现实世界做好准备，他们将准备好帮助我们购买它。
