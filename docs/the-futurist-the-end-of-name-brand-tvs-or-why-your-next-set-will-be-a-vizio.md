# 未来主义者:名牌电视的终结。为什么你的下一套会是 Vizio 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/14/the-futurist-the-end-of-name-brand-tvs-or-why-your-next-set-will-be-a-vizio/>

当我满足每天五个小时坐地铁的习惯时，我发现了一个让我完全震惊的广告。这则广告并不特别血腥或淫秽——让我措手不及的是广告客户。Vizio 是那些本土位于山姆会员店和好市多过道的非品牌平板电视制造商之一，它实际上是在电视上做广告。

从我所看到的一切来看，Vizio 的画面质量虽然可能不如松下清晰，但也足够好，一般人无法区分。而且它的价格肯定是降价的——这显然是消费者的反应，因为这家新贵公司现在是北美十大电视销售商之一。对于一家五年前还不存在的公司来说，这已经不错了。

那么 LG 和夏普是不是吓坏了？像 Vizio、Maxent 和 Sceptre 这样的公司会让大公司害怕吗？几乎没有。事实上，这些新贵可能会成为继有线电视之后发生在大型制造商身上的最好的事情。

生产一台电视的所有零件需要高超的制造技术。事实上，只有少数公司拥有制造等离子面板等关键部件所需的资源和专业技术。因此，大多数电视公司——尤其是较小的电视公司——将大部分制造和零件生产外包给拥有大型工厂的人。

出于显而易见的原因(没有人喜欢被告知他们与另一家公司没有什么区别)，大公司和小公司都不愿透露谁为谁制造零件的细节，然而，一位 Vizio 代表告诉我，“五家不同的公司”制造其面板，包括我听说过的公司。考虑到你一只手就能数出主要的面板制造商，他们的来源对任何使用谷歌的人来说都应该是显而易见的。

当然，索尼和 Sceptre 之间的质量差异多年来一直在消退。如果目前的趋势保持下去，名人和无名小卒之间的唯一区别将是名字本身。尽管如此，由此导致的对他们业务的蚕食很可能会成为巨头们的一件好事。

和消费电子行业的其他部门一样，电视业务也很艰难。保持竞争力需要巨大的人力和财力。市场营销，广告，法律部门，公共关系不要自己掏钱。通过向较小的公司打开工厂大门，大公司有一个重要的机会从根本上转变他们的主要电视相关收入流，从向消费者出售电视机到向其他公司出售零件——这种收入流可能远没有那么变化无常，需要浪费在前台系统工资支票上的资源也少得多。

那么谁是赢家呢？当然是带着植物的人:松下，LG。我想到了飞利浦和夏普。

但是小公司也有独特的机会。打入任何一个行业，尤其是像消费电子产品这样大规模、高风险的行业，都是极其困难的。但是因为搬运植物的巨人实际上从小人物的成功中获得了一些东西，他们不仅没有像蚊子一样压碎他们，他们实际上给了他们生存所需的资源。结果是更多的竞争，更低的价格，大概更好的产品质量，这意味着最终的赢家是你，讨价还价的消费者。
