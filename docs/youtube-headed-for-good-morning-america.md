# YouTube 走向早安美国

 [](https://web.archive.org/web/20210305195500/http://youtube.com/) YouTube [今晨](https://web.archive.org/web/20210305195500/http://rds.yahoo.com/S=53720272/K=YouTube/v=2/SID=w/l=NSR/R=3/;_ylt=A9iIgNQHbhFFloABdBTQtDMD;_ylu=X3oDMTBjcXBoZjEwBHBvcwMzBHNlYwNzcg--/SIG=12g4rh7is/EXP=1158856583/*-http%3A//news.yahoo.com/s/nm/20060920/tc_nm/media_youtube_dc_5)宣布，该网站将举办一场在线“乐队之战”，由用户投票选出的获胜者将出现在《早安美国》节目中，作为 Cingular 铃声。天哪，你能有多蠢？

紧随更有趣的华纳交易之后，今天的公告似乎进一步证实了这样一个信念，即从他们庞大的用户群中赚钱对该公司来说只是时间问题。

之前与 NBC 的一项协议使块茎在本周末登陆杰伊·雷诺，尽管财务安排没有在片段中披露，而且“我们在 YouTube 上发现的视频”的基层真实性值得怀疑。接下来，你知道山姆大叔会把“这是你吸毒的大脑”的视频放到 YouTube 上——等等，[这已经发生了](https://web.archive.org/web/20210305195500/http://youtube.com/profile?user=ONDCPstaff)。

一旦 YouTube 成为与主流电视和手机铃声捆绑在一起的过时音乐促销的新家，它还能保持优势吗？YouTube 用户社区似乎对帕丽斯·希尔顿反应不佳。随着 YouTube 的贵族化，下一个流行的、低租金的病毒视频社区会出现在哪里？似乎没有任何特别引人注目的竞争对手，尽管昨晚的[雅虎！与 Current.tv](https://web.archive.org/web/20210305195500/http://www.beta.techcrunch.com/2006/09/19/yahoo-launches-better-video-content/) 的合作可能会很有趣，Revver 可能是目前为止在创业公司中尝试最努力的。
