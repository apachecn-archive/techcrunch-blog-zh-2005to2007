# 挫败互联网审查的 Psiphon 项目

众所周知，住在加拿大的人是一群冷血的左翼分子。如果你需要证据，你只需要求助于奇妙的灵能工程。Psiphon 由草根互联网活动家拼凑而成(我们拒绝使用“黑客活动家”这个词)，是一个公共工具，用于帮助其他人访问他们的政府拒绝他们访问的部分万维网。

限制性政权，如中国或朝鲜，经常对其公民屏蔽大量网络，从持不同政见者网站和色情网站到西方新闻网站和博客。这导致了这些地区的巨大痛苦，并且出现了某种程度的互联网军备竞赛，因为那些想要信息自由的人和那些采取行动压制信息自由的人都在相互竞争。

这就是 Psiphon 背后的酷想法的来源。

许多绕过互联网路障的方法都采用了这样或那样的代理形式。这里的问题是，代理人可以被识别，而托管代理人的一方会受到惩罚。灵能将代理人带出受影响的国家。在一个不受审查的国家，任何读者都可以下载该软件，并为那些不那么幸运的人托管。审查国家的网络用户然后使用你的连接来浏览和你一样不受限制的互联网。

这个想法是基于数字的:如果只有少数用户安装了这个软件，那么它很容易被屏蔽。但是如果有几千人使用它，那就很难了。越多的用户分享他们的连接，互联网警察就越难阻止他们。

点对点流量经过加密，不会在客户端计算机上留下任何使用痕迹。这对那些躲在数字幕后的人来说是一个很好的窥视方式，对普通网络用户来说也是一个很好的帮助他人的方式。毫无疑问，这是一个值得期待的项目。
