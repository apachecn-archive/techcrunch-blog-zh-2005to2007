# OnAir GT 移动高清电视接收机

OnAir GT 移动高清电视接收器可能不是你第一个接触的笔记本配件，但它肯定是最酷的之一，让你可以在笔记本电脑上观看高清和模拟无线和数字(QAM)和模拟有线电视广播。由美国 [Autumnwave](https://web.archive.org/web/20210116220308/http://www.autumnwave.com/) 发行，这个小黑盒通过 USB 电缆获取电力，并通过附带的 A/V 适配器电缆传输视频，迅速将您的系统变成一台具有完整 DVR 功能的电视，配有 EPG。你也可以在设备上连接一个游戏机游戏系统，让你直接在笔记本电脑屏幕上玩游戏。OnAir GT 的零售价为 179 美元，但如果你已经有了一台能够显示高清分辨率的笔记本电脑，这是一种在任何地方观看高清节目的廉价方式。
