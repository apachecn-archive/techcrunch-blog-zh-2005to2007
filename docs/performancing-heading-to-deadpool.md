# 向死亡池前进

 [](https://web.archive.org/web/20220521070800/http://performancing.com/) 演出[关闭](https://web.archive.org/web/20220521070800/http://performancing.com/node/5651)他们的广告网络，[演出合作伙伴](https://web.archive.org/web/20220521070800/http://www.beta.techcrunch.com/2006/10/18/performancing-launches-ad-network-for-bloggers/)，今天。他们的度量服务，一个独立的产品，在本月早些时候被 PayPerPost 收购失败后被关闭。

那笔 PayPerPost 交易也让 Performancing 将他们的第三个产品，一个流行的博客编辑器，更名为 ScribeFire。我们的理解是，该公司目前并不打算保留 ScribeFire 的名称，但该网站仍然是活跃的。

Performancing 公司的首席执行官最近也离职了。我们正在联系该公司，看看我们是否可以期待其最后剩余产品的未来更新，或者该公司是否基本上已经死亡，应该被添加到[死亡池](https://web.archive.org/web/20220521070800/http://www.beta.techcrunch.com/tag/deadpool)。
