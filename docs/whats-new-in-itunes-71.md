# iTunes 7.1 有什么新功能？

![](img/0cef588e2445da52aefaad5ba5eeb5e2.png)
iLounge 对 iTunes 7.1 及其所有改进/变化进行了深入的分析。最有趣的是全屏覆盖流视图，提供了一点前排的好处，没有太多的开销。锁定键盘，你就有了一个伟大的小点唱机应用程序。

他们还增加了 Apple TV 支持——根据苹果商店的消息，似乎是 3 月 20 日发货——以及一些高级排序字段，如艺术家专辑。对于那些对苹果电视上的 iPod 游戏感到兴奋的人，对不起。那个谣言已经被揭穿了。
