# 昨晚的聚会很成功

感谢[昨晚在阿泽顿参加](https://web.archive.org/web/20220816064748/http://wiki.beta.techcrunch.com/second_meetup_bbq#attendees)我们聚会/烧烤的每一个人。我们肯定会至少每个月做一次，昨晚我们收到了一些企业赞助的提议，包括食物、饮料等。在未来。
