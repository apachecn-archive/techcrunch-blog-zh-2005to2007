# 纽约时报:PS3 让这个地方变得臭气熏天

Seth Schiesel，一个喜欢 Xbox 360 的人，基本上是这样对索尼说的:

> 你公司的新视频游戏系统并没有那么好。

什么？他怎么敢！他实际上是在说 PS3 感觉还没完成，用户界面很糟糕。由此产生的迷你咆哮是合理的和聪明的，所以你不能在这方面指责他。显然，索尼已经失去了他们的触觉和/或有一些惊人的未来更新他们的袖子。我个人认为是前者。
