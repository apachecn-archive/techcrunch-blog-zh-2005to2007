# 黄色潜水艇 iPod 即将推出，包括甲壳虫乐队的目录

![beatles-ipod.jpg](img/2072496c3e1fc1fc709751be0917a779.png)

我喜欢甲壳虫乐队。真的。当我还在上小学的时候，我父亲的黑胶唱片收藏是有史以来最酷的东西，他有《滚石》和一些休·马塞克勒的唱片，还有不少披头士的专辑。美好时光。美好时光。

所以现在我们相当确定一个黄色潜水艇版本的 iPod 将会和整个甲壳虫乐队的备份目录一起发布，我不得不问“我为什么要关心这个？”苹果和苹果之间的诉讼早已不了了之，就像这个乐队与年轻观众的关系一样。他们做出了很棒的音乐，但如果是这样的话，为什么不做一张辛纳特拉特别版呢？对我来说，这有点像婴儿潮后嬉皮士时代的爱情，但我只是一个习惯了肤浅的营销策略的愤青。

那里。我说了。我真的不知道为什么人们会在 iTunes 上关注披头士。保罗爵士和小伙子们的粉丝们已经拥有了他们想要的所有专辑和盗版，这只是史蒂夫·乔布斯说的“我终于得到了这些家伙。去你的，因为我在小学时一直唱‘在你里面，没有你’而打我的孩子们！”
