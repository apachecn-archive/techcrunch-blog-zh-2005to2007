# 三星宣布推出全球首款带 DisplayPort 的液晶显示器

![104632.jpg](img/ca4cf9697e59eac6ebe65ac6c8a18199.png)

三星宣布了另一款世界首款采用单显示端口的 30 英寸液晶面板，将在即将推出的三星液晶显示器中取代 DVI、LVDS 和 VGA。下一代接口以 10.8Gbps 的速度输出所有漂亮的图片，这允许 2560×1600 的分辨率或 10.7 亿种颜色，没有任何可见的延迟。DisplayPort 面板使用一个四通道、2.7 Gbps/通道接口芯片，而通常至少需要三个 DVI 或四个 LVDS 接口芯片才能达到相同的效果。三星专有的超级图案垂直排列液晶技术允许 180 度的视角和 300 尼特的亮度。这部电影将于 2008 年在 Q2 上映。
