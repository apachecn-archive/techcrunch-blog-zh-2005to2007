# 虚拟世界很有趣，天空是蓝色的

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/18/virtual-worlds-are-fun-sky-is-blue/>

![](img/40ae7152cb93c304b91ffa11dffe13c7.png)

“第二人生”运营商林登实验室的主席米切尔·卡普尔周五在 T2 的虚拟世界会议上发表了主题演讲，谈到了虚拟教育和培训的好处，其中包括提供一个学员可以安全失败的环境。尽管人们没有从通过互联网学习工作中获得任何真实世界的经验是一个明显的挫折，但 Kapor 说互动和交流变得更容易了。他继续将虚拟世界比作毒品旅行，因为它们将人们置于新的环境中，这可以带来灵感。
