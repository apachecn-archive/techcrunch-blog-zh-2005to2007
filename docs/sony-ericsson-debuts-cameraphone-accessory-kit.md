# 索尼-爱立信推出照相手机配件包

我们喜欢索尼爱立信手机的一点是，该公司将它们巧妙地分为两个不同的类别:音乐手机和照相手机。与其他手机制造商不同，SE 手机的音乐欣赏和拍照功能并不是半吊子的尝试，它对音乐手机使用了真正的 Walkman 技术，对照相手机使用了真正的 Cybershot 技术。

不过，相机手机长期以来缺少的东西是像样的摄像头配件。索尼-爱立信已经开始用 IPK-100 套件进行补救，这是一组用于 Cybershot 手机的配件，包括外壳、皮带夹、三脚架和其他基本配件。我们喜欢这种融合，但我们不欣赏它通常包含的功能的损失，因此我们发现这是 SE 相机手机用户的必备功能。去吧，你知道你想要一个。
