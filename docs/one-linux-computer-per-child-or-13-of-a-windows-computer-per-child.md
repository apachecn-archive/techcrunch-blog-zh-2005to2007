# 每个孩子一台 Linux 电脑或每个孩子 1/3 的 Windows 电脑

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/20/one-linux-computer-per-child-or-13-of-a-windows-computer-per-child/>

微软正在准备一款售价 3 美元的操作系统，它可以在发展中国家销售的低成本硬件上运行。该软件包包括一个精简版的 XP、MS Office 家庭和学生版，以及一些其他的儿童应用程序。这应该在硬件上运行，比如 T2“一童一机”项目的 XO 笔记本电脑。该软件将只出售给政府。

但是等等？OLPC 不会运行 Linux 吗？而且 Linux 不是免费的吗？微软是不是想通过增值税来培养一大批微软用户，同时还能在无形的业务上获得可观的利润？是的，是的，还有双倍的是的。
