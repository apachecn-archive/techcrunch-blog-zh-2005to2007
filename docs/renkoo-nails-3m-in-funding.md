# Renkoo 获得 300 万美元融资

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/04/13/renkoo-nails-3m-in-funding>

 [](https://web.archive.org/web/20160216045604/http://www.renkoo.com/) 融资列车本周继续行驶——总部位于帕洛阿尔托的初创公司 [Renkoo](https://web.archive.org/web/20160216045604/http://www.renkoo.com/) ，仍处于私下测试阶段，已获得来自 [Matrix Partners](https://web.archive.org/web/20160216045604/http://www.matrixpartners.com/) 的 300 万美元风险投资。

Renkoo 是我有机会预览的许多新的 Web 2.0 活动创建公司之一(参见 [Skobee](https://web.archive.org/web/20160216045604/http://www.techcrunch.com/2006/03/09/skobee-just-launched-so-start-using-it/) )。Renkoo 由首席执行官亚当·莱福金(KnowNow 的创始人)和首席技术官 Joyce Park(Friendster 的前首席工程师)创建。发射日期:“很快”。
