# 手机广告流行了

还记得互联网上弹出广告的早期吗？他们一开始很有趣，然后变得非常讨厌……哦，五秒钟后。好吧，准备好第二轮。弹出广告会出现在你附近的第三个屏幕上。一些初创公司已经开始推动将弹出窗口发送到手机的技术。

更糟糕的是，当使用浏览器时，广告不一定会弹出，但当手机屏幕空闲时，广告会弹出！这些广告背后的一些公司表示，他们将致力于用户不会觉得讨厌的广告。但是我来问这个问题:你上一次不觉得弹窗广告很烦是什么时候？
