# 未发布的游戏机本来可以，但没有

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/20/unreleased-consoles-that-could-have-been-but-werent/>

当我们在地平线上寻找黑色 Xbox 或打折 PS3 或任何有 Wii 库存的商店时，我们想分享 Techeblog 列出的从未上架的五大游戏系统，更不用说游戏玩家的大脑了。

我们最喜欢的是世嘉海王星，主要是因为它实际上看起来很酷(当时)。当然，还有其他的。我个人认为爱立信的红玉应该榜上有名。爱立信(索尼之前的合作伙伴)投资了数百万在这个设备上，却被来自诺基亚的 N-Gage 抢了风头。在看到人们不想要手机/lameboy 混合手机后，爱立信明智地终止了这个项目。

但有史以来最好的未发布游戏系统一定是[共和游戏机](https://web.archive.org/web/20160305074733/http://www.republic-console.00page.com/index.html)。我们希望有一天它会回来。
