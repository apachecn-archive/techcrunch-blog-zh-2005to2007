# 沃达丰会保留威瑞森无线吗？

本周的问题是移动电话巨头沃达丰集团是否会出售其在威瑞森无线的股份。路透社报道称，沃达丰似乎不太可能出售其股份，并援引一位消息人士的话说，“沃达丰已决定暂时保留威瑞森无线。”

关键词是“目前”，世界第二大移动电话公司有可能在不久的将来抛售其在威瑞森无线的 45%股份。分析师对沃达丰股份的估值约为 450 亿美元，但尽管如此，这一价值并没有反映在沃达丰自己的股价中。

Verizon Communications 控制着威瑞森无线的大部分股份。但到目前为止，沃达丰的投资者拒绝了维权股东高效资本结构公司(Efficient Capital Structures)提出的将股份分拆成一家独立上市公司的要求。
