分享文档的网站Scribd 今天早上结束了私人测试，在他们最初的12000美元的投资基础上，又获得了30万美元的天使投资。Scribd最容易被描述为YouTube的文本版本。它是一个社交网络，可以让你标记、分享和评论上传的文档。doc，。pdf，。txt，。ppt，。xls，。ps，。lit)。

Scribd不仅仅是YouTube的翻版。他们借鉴了许多基本的设计原则，但也利用了书面格式，包括灵活的下载和上传文件格式以及一些有趣的分析跟踪。文档可以显示和嵌入为html或未充分利用的、比pdf更快的Flash纸格式。它们可以下载为。pdf格式，。文档。txt，甚至. mp3文件。mp3版本是由Scribd的文本到语音转换包(由 Nuance 提供支持)创建的，它可以让你用颤抖的英国口音听你文档的文本(这里有可下载的例子)。人们已经上传了各种私人测试版的文档，比如这个约会和诱惑傻瓜指南，或者这个不到合法版本的Visual c++ 21天。Scribd还可以让你“极客”地分析你发布的文件所产生的所有数据，比如你的文章获得了多少投票和浏览，以及把读者带到那里的地理位置和http referrer。

我们已经看到许多不同的社交网络围绕着不同的媒介出现，照片、视频甚至音频，但是主导一个媒介并不能保证一个简单的商业模式，正如专注于音频的Odeo上的“出售”标志提醒我们的那样。到目前为止，围绕书面文字的社交网站处理的是书籍，而不是用户生成的，或者至少是用户上传的内容。Scribd让人们做一些新的事情，我们只需要等待，看看人们会用它走多远。
