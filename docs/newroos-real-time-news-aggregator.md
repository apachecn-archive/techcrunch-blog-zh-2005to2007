# Newroo 的实时新闻聚合器

由布朗大学毕业生 Brian Norgard 和 Dan Gould 于 2005 年创立的总部位于旧金山的 new roo(T4)离正式成立只有两周时间了。

与 [Megite](https://web.archive.org/web/20230123213713/http://techcrunch.com/2006/02/03/is-megite-a-contender/) 一样，Newroo 最终也将面向与 [Memeorandum](https://web.archive.org/web/20230123213713/http://tech.memeorandum.com/) 相同的市场。然而，它们有许多类似 squidoo 的特性(这是一种恭维)，允许任何人创建自己的基于主题的主服务版本。

基本引擎就像 Memeorandum——它们根据主题对博客帖子进行索引和分组，以便近乎实时地显示新出现的新闻，并在一个地方显示分布式博客讨论。

Newroo 推出了许多垂直业务，包括 Brin Rank(谷歌新闻)、Series C(风险投资新闻)、Jockzilla(前沿体育新闻)和 Celeb Trash(名人八卦)。图中是 JockZilla，尽管目前所有的垂直市场都有相似的外观和感觉。

上线后不久，Newroo 将允许用户创建和定制自己的网站，从现有网站的克隆开始。这种收入模式是基于展示与新闻相关的商品的亚马逊联盟链接，最终他们计划与网站的创建者分享这一收入。

我认识创始人，但与他们没有任何关系。尽管如此，我还是很欣赏他们当前登陆页面上的链接。
