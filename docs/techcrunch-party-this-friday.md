# 本周五的 TechCrunch 派对

本周五(10 月 21 日)下午和晚上，我们将在阿泽顿举办下一场 TechCrunch 派对。[在这里报名](https://web.archive.org/web/20230216134410/http://wiki.beta.techcrunch.com/third_meetup)。我们的[上一次聚会](https://web.archive.org/web/20230216134410/https://techcrunch.com/2005/09/23/successful-meetup-last-night/)吸引了大约 100 名企业家、风险投资家、博客作者和啤酒爱好者，我希望这次聚会至少能吸引这个数字。

我们正在转向 socialtext wiki，它将在今天晚些时候与确认帖子一起发布。如果您想展示您的产品，请[现在就给我们发电子邮件](https://web.archive.org/web/20230216134410/mailto:editor@beta.techcrunch.com)。我们有 8 个插槽。

我们还接受了两个额外的企业赞助商——这使得我们可以免费举办这项活动。如果你感兴趣，请给我们发电子邮件。套餐包括张贴横幅的权利、长达 2 分钟的祝酒词/演讲以及在活动 t 恤上印有您的标志。加上知道人们会爱你和你的公司的快乐。:-)
