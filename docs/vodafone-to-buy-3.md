# 沃达丰收购 3？–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/16/vodafone-to-buy-3/>

约翰·豪克斯韦尔说:“今天伦敦流传着这样的报道:香港集团[和记黄埔](https://web.archive.org/web/20201129141028/http://www.hutchison-whampoa.com/)准备出售其在英国的手机业务 [3 英国](https://web.archive.org/web/20201129141028/http://www.three.co.uk/)。高盛已被任命为顾问，官方声明预计将于周四发布。艾伦·帕特利克也开始关注这个故事。

同样,《泰晤士报》报道称，最近对新收购采取谨慎态度的首席执行官 Arun Sarin 表示，如果 3 的欧洲资产被出售,“那么我们会考虑”。我相信沃达丰有他们的理由。

我仍然认为，在接下来的几年里，我们会看到更多的合并活动，这将导致更少的 quad 玩家为我们提供折扣服务和内容。最近我们见证了 Tiscali [收购](https://web.archive.org/web/20201129141028/http://www.computeractive.co.uk/computeractive/news/2162301/tiscali-buys-homechoice) HomeChoice 和 CarphoneWarehouse 收购 AOL 英国的宽带接入业务。

上周，我们看到 NTL 宣布有意收购独立电视台。英国电信首席执行官 Ben Verwaayen 还宣布，英国电信肯定会在圣诞节前推出其 BT (IPTV)视频服务，未来几周将会宣布更多类似迪士尼的[内容交易。](https://web.archive.org/web/20201129141028/http://business.timesonline.co.uk/article/0,,9076-2445926,00.html)

我仍然认为，英国的整合才刚刚开始；我想知道下一个是谁？
