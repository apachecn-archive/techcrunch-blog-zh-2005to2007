# Netvibes 旧金山派对:TechCrunch 读者收到最后 50 份邀请

[![](img/e69677ae92745613f2e647fbe918db69.png)](https://web.archive.org/web/20220516091055/http://eco.netvibes.com/sfparty/)

[Netvibes](https://web.archive.org/web/20220516091055/http://www.netvibes.com/) 将为他们 600 名最亲密的朋友举办一场派对，庆祝《Netvibes Universe》即将发布(期待我们的帖子)。这个聚会可以免费参加，但你必须事先答应。Netvibes 创始人塔里克·克里姆(Tariq Krim)同意将最后 50 份邀请函送给 TechCrunch 的读者。如果你想参加，现在就在[eco.netvibes.com/sfparty/](https://web.archive.org/web/20220516091055/http://eco.netvibes.com/sfparty/)回复。50 个位置消失后，该网站将开始创建一个等候名单。

如果你真的打算去，请回复我。否则你会把其他人拒之门外。

**时间:**晚上 9 点
**地点:**加利福尼亚州旧金山市明娜街 111 号，邮编:94105
