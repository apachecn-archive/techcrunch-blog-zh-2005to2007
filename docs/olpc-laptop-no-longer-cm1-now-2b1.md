# OLPC 笔记本不再是 CM1，现在是 2B1

就在上周，每个人都报道了“每个孩子一台笔记本电脑”项目的第一台笔记本电脑更名为“CM1 ”,代表“儿童机器 1”。但现在他们的官方维基已经删除了所有提及 CM1 的内容，取而代之的是“2 B1——孩子们的机器”。“2B”代表“将来”吗？

无论如何，这款笔记本电脑仍然拥有 7.5 英寸、1200×900 屏幕、视频/静态摄像头、三个 USB 端口、SD 插槽、VoIP 功能、400MHz AMD Geode 处理器和 128 MB RAM。价格从 100 美元涨到了 140 美元，但对于孩子们将要得到的所有东西来说，还是很便宜的。
