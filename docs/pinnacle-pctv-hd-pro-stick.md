# 顶峰 PCTV 高清专业棒| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/20/pinnacle-pctv-hd-pro-stick/>

![](img/7a5ad9ce165372515058a67c105b512d.png)

Pinnacle 今天宣布发布 PCTV 高清专业棒，这是一种超紧凑的 USB 2.0 电视调谐器，具有吸引力的棒形。它支持 ATSC 数字标清、EDTV 和高清电视。Pro Stick 还可以播放模拟 NTSC 信号，例如来自有线电视和 DVD 播放器的信号。当与 PCTV MCE 伴侣，它是完全兼容的 Windows 媒体中心版。该伴侣使您的 Media Center PC 能够用作 DVR，并具有时移功能，可以直接保存到 DVD、DivX、iPod 和索尼 PSP。

总而言之，这似乎是一个坚实的高清电视调谐器。它包括一个遥控器和所有有趣的东西。PCTV 高清专业棒售价 129.99 美元。而 PCTV MCE 伴侣是一个额外的 99 美元。
