# USB 组织认证手机微型 USB

如果你的手机有一个 USB 端口，这样你就可以抓取你的照片和铃声，并备份你的联系人，这不是很好吗？当然，我们中的一些人足够幸运，在我们的手机上有这样的东西。我们中的一些人甚至有蓝牙功能来完成这些任务。但是对于美国的绝大多数手机来说，获取信息的唯一途径是通过其运营商的网络(这是一个昂贵的提议)，或者通过专有电缆，也是昂贵的(如果你能找到一个的话，前提是你知道要找什么)。

USB 实现者论坛已经提出了一个新的标准，称为微型 USB，它希望制造商采用的手机标准。为所有手机制定一个标准解决了许多显而易见的问题，但不能保证会被大规模采用。不过，如果配件制造商开始使用它，它可能会推动手机制造商朝着那个方向发展，就像 90 年代末第一个 USB 规范所发生的那样。

提议的端口实际上比大多数支持 USB 的手机的 4 针迷你 USB 端口更小，这意味着更小的手机具有更大的能力，这是我们喜欢的组合。
