# 葡萄酒爱好者现在有了一个 Web 2.0 网站

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/05/24/wine-lovers-now-have-a-web-20-site/>

 [](https://web.archive.org/web/20220112013243/http://www.winelog.net/) [葡萄酒日志](https://web.archive.org/web/20220112013243/http://www.winelog.net/)，一个新的葡萄酒爱好者网站，刚刚推出。这是一个用户生成葡萄酒信息的网站，包括标签、评级和评论。

内容还不深入，鼓励用户向数据库中添加更多的葡萄酒(一个新的葡萄酒条目在出现在网站上之前经过审核)。但那里有足够的内容表明，这将是一个葡萄酒爱好者会充满热情的网站。我也希望它能开发一个新的市场，吸引那些可能还不熟悉“web 2.0 网站”的人，让他们接触标签、RSS、评论等等。

如果找到喜欢的葡萄酒，可以添加到自己的葡萄酒日志中。这些页面有一个永久的公开的共享网址(隐私是[承诺很快](https://web.archive.org/web/20220112013243/http://www.winelog.net/blog/?p=27)，以及 RSS，博客小工具和其他功能)。你也可以通过短信将任何葡萄酒的信息直接发送到你的手机上。

服务仍然很粗糙，我有一些意见/建议。葡萄酒可按类型(红、白等)浏览。)，品种和地域。我也希望能够按酒厂浏览。点击那里的标签会显示我可能喜欢的其他葡萄酒。另一个 nit:当你对一款葡萄酒发表评论时，它不会出现在你的葡萄酒日志中。你必须在那里留下单独的评论。还有一些导航错误。

但总的来说，我喜欢这个网站和服务。至少，它比 Winespectator 收费提供的个性化选项有用得多。
