# 虚拟诺基亚 E61i 在野外

![](img/f9380d56c1b3da39041f9fc677d58b9e.png)

我喜欢 E61，我想我会喜欢 E61i。这款诺基亚塞班智能手机的升级版拥有 E61 的所有纤薄外观，但希望没有令人讨厌的减速问题。

你在这里看到的实际上是一个手机商店的“虚拟”版本，对于那些有此倾向的人来说，价格仅为 22.99 美元。对诺基亚恋物癖者来说，这是一个有趣的机会，但对我们其他人来说，这没什么实际意义。
