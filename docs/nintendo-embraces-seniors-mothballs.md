# 任天堂拥抱老年人，樟脑丸

![](img/8c745d8576c16433187135b7a25396a8.png)

任天堂本周末出现在一个你可能不会想到会在视频游戏中遇到的大会上:在加利福尼亚州阿纳海姆举行的 AARP Life@50+大会上。该公司展示了其 Wii in force，向 2 万名与会者展示了 Wii 不仅仅是给孩子看的。它还展示了 DS 的“大脑年龄”，任天堂操作人员戴着写有“问我的大脑年龄”的徽章在场地上游荡。

我相信这将成为任何游戏制造商最辉煌的举动之一。这是一座尚未开发的消费者金矿。有大量现金的退休人员正在寻找更有趣的方式来打发时间。但真正的美妙之处在于，这些退休人员中的许多人都有孙子，他们将进一步拥抱这项技术——真的很棒。
