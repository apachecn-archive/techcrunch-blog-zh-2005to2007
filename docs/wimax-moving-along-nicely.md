# WiMax 进展顺利

![](img/ad0f3261ef0ceabeb8121189816ddfbc.png) DSL 报告正在撰写一篇文章，发现 Sprint 的 WiMax 系统进展顺利。据报道，4G 网络将于 2008 年初在芝加哥和 DC 铺开，用户网速将达到 2-4 Mbps。更重要的是，据称它每月只需 55 美元，并支持开放访问系统。这意味着你可以把它连接到你家里的路由器上，并允许多个系统共享链接。

我以前说过，但我真的看到移动宽带在未来接管。我们看到人们使用手机作为他们的主要电话，并完全放弃座机。再加上移动宽带连接带来的便利，这真的是显而易见的。
