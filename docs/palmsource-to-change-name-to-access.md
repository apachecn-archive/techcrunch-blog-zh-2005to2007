# 更改名称以访问| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/12/palmsource-to-change-name-to-access/>

Palm OS 的制造商 PalmSource 正在经历又一次更名。日本移动软件制造商和 PalmSource 的母公司 Access 显然正在将该公司并入其 Access 品牌。虽然 Palm OS 仍将被这样称呼，但 PalmSource 现在将被称为 Access。

对于 Palm 粉丝来说，这种以 Palm 为主题的绰号并不是一个好兆头，因为许多人认为 Palm OS，尤其是即将推出的基于 Linux 的版本，是 Windows 世界的救星。操作系统的下一次迭代最好是 kickass，否则可能是最后一次。

Palm 软件制造商的最新身份:Access
