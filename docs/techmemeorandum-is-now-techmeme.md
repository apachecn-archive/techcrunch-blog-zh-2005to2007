# 技术。Memeorandum 现在是 TechMeme

 [](https://web.archive.org/web/20220127055502/http://www.techmeme.com/) 这是我最后一次写 Memeorandum，因为，感谢上帝，它已经被创始人 Gabe Rivera 改名了。博客聚合器 Memeorandum 的技术部分，以前在 tech.memeorandum.com，现在是 [TechMeme](https://web.archive.org/web/20220127055502/http://www.techmeme.com/) 。

谢谢你加布，终于把那个名字拿出来拍了。我不能告诉你我打错了多少次。我以前在 Memeorandum 上的帖子，现在是 TechMeme，[在这里](https://web.archive.org/web/20220127055502/http://www.beta.techcrunch.com/tag/Memeorandum)。如果你不知道什么是 TechMeme，读读我关于它的前两篇文章，这里的[这里的](https://web.archive.org/web/20220127055502/http://www.beta.techcrunch.com/2005/09/18/memeorandum-is-exceptional/)和这里的[这里的](https://web.archive.org/web/20220127055502/http://www.beta.techcrunch.com/2005/10/12/memeorandum-is-changing-the-web/)，来理解它为什么重要。

政治博客聚合器仍然可以在 memeorandum.com 找到。
