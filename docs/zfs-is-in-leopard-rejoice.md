ZFS 是里的[，ZFS 是](https://web.archive.org/web/20200806014153/http://crunchgear.com/2007/06/06/leopard-will-use-zfs-file-system-sun/)[外的](https://web.archive.org/web/20200806014153/http://crunchgear.com/2007/06/12/sorry-no-zfs-in-mac-os-x-leopard/)……好了，现在我们知道 ZFS 文件系统支持*是 Mac OS X 豹里的*。根据网上的报道，读/写支持只是开发者的种子，除非苹果将它设为默认文件系统，否则除了完全的书呆子没有人会注意到。Leopard 的 WWDC 版本仍然将 HFS+作为默认文件系统。
