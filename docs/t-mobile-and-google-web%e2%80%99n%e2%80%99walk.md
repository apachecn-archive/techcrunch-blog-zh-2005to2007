# T-Mobile 和谷歌网络漫步

匈牙利最大的移动运营商 T-Mobile 现在将依靠谷歌的搜索引擎技术来支持他们的 web'n'walk 门户网站。以前国内的匈牙利 OK.hu 搜索引擎被用于移动搜索，但现在 T-Online 和 T-Mobile 都改用了谷歌。
