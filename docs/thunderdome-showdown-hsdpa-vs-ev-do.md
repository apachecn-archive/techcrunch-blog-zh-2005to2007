# Thunderdome 摊牌:HSDPA vs EV-DO

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/21/thunderdome-showdown-hsdpa-vs-ev-do/>

![](img/e1c407faf9149f9f79ffa7d6bd825af2.png)年度战役！Cingular 的新人 HSDPA 与 Sprint 的大众最爱 EV-DO 展开较量。关于哪种技术会更好的猜测从互联网的一端散落到另一端，我在这里澄清事实。

通过使用 Novatel Merlin Xu 870 HSDPA express card 和 Merlin EX720 EV-DO，我在全国各地旅行时能够充分了解这些激烈竞争对手的利弊。EV-DO 能否凭借其稳定的表现保持其在 WWAN 舞台上的统治地位，或者 HSDPA 能否凭借其快速而花哨的步法赢得冠军？

正如我上面提到的，HSDPA 是这个街区的新人。它出现在过去的一年中，最近在网络可用性方面有了一些广泛的发展。我和 HSDPA 在 21 点上有过相当多的经历，大部分都是正面的。可用性很好，性能令人满意。鉴于这些因素，我很高兴能在我的 MacBook Pro 上尝试专用的 HSDPA ExpressCard。

Cingular HSDPA 的速度为 7.2Mbps，是 Sprint 提供的微不足道的 3.1Mbps EV-DO 修订版的两倍。我发现的问题是，你不可能在任何地方达到这样的速度。在我的旅行中，我无法在 HSDPA 冲刺达到最大带宽。但是不要让这动摇了你，每个灵媒都有自己的权利。

![](img/5fe69e58680e4d05ef5736b94f956fdb.png)虽然 Cingular 目前没有直接提供 Merlin XU870 ExpressCard，但它可以很容易地在 MacBook Pro 上顺利运行。只要更新 OS X 软件，你就可以了。安装简单是一个伟大的设备功能，尤其是与 Mac 上的 EX720 设置的灾难相比。

HSDPA 变得相当苦乐参半。我发现很少有地方达到了它声称的 7.2Mbps 的能力，不管怎样，我确实找到了不少地方比它的对手产生了明显更快的速度。我注意到的是，三个实心条可以让你快速浏览网页。然而，我失望地发现下载速度并没有明显提高。我怀疑丢包是罪魁祸首，但不可否认这是显而易见的。

在我看来，HSDPA 有大约一年的时间可以支配。大约在那个时候，WiMAX 将从 Sprint 的胎盘中喷出它美丽的头。当这种情况发生时，HSDPA 提供的 7.2 兆比特看起来就不那么美好了，原因和 EV-DO 的 3.1 兆比特看起来不再那么美好一样。为什么？因为 WiMAX 将成为 9376481641^17Mbps(结果可能会有所不同)，当您可以获得 WiMAX 时，谁还会想要 HSDPA。

因此，Cingular 的提示是，让您的游戏在一起。你现在有更好的技术，但是你的速度比保龄球之夜的奶奶还慢。那些球瓶不会自己倒下。你得放些太空时代的聚合物在里面，奶奶。召集桥牌俱乐部，打电话给你的老朋友，我不在乎。只要比你现在做的更快就行了，因为即使有更好的技术，你也是在扔臭球。

正如我之前解释的那样，EV-DO 之所以受到青睐是有原因的:一致性。当然，当 HSDPA 兴奋时，EV-DO 无法竞争，但 HSDPA 很少发挥其全部潜力。我发现 Sprint 没有在最广泛的可用性上撒谎。它的 EV-DO 随处可见，我试过了。在建筑物里，在挑檐下，在壁橱里，在厕所里，德克萨斯。似乎没有什么能阻止它。我还实现了相当快的浏览速度，下载大约 150KB，上传大约 40KBps。这和我的第一个家庭宽带连接一样快，但它在任何地方都可用——猜猜哪个更理想。

与此同时，我的 HSDPA 卡片孤零零地放在我的背包里，我知道如果教练信任他，他会做得更好。一个可怕的命运，但考虑一下事实。梅林 XU870 经常陷入边缘模式，这对任何事情都没有好处。说真的 EDGE 就是不好。就像使用拨号一样。这比什么都让我沮丧，到最后我准备把我的系统砸到地上。EV-DO 从未经历过这种不安。总是 EV-DO。它可能达不到它的最高速度，但它的表现足够好，你仍然可以完成你的大部分工作。

![](img/8a008944f299777a61113156f23598cf.png)
Sprint 目前提供的 Merlin EX720 ExpressCard 非常棒，一旦你让它工作起来。你看，EX720 在 Mac 上无法激活。要使其工作，您必须在具有 ExpressCard 插槽的 PC 上激活它。如果你有材料，这是很容易的，但是要做好准备。如果你不这样做，该设备将无法定位运营商，将完全无用。

现在重要的是要考虑到这里的竞争环境是公平的。鉴于两张卡都来自 Novatel，它们至少是从相同的性能口径切入的。这些变化完全取决于介质。也就是说，我认为 EV-DO 是目前最好的选择。如果你有一张 WWAN 卡，我猜你的目标是在各种场合使用它。除非你能确认 HSDSPA 在大多数领域都可用，否则你会使用它(我希望你会在很多意想不到的领域使用它)。

如果 HSDPA 获得了足够的牵引力，那就要小心了。它会把 EV-DO 炸成碎片。可悲的是，在 WiMAX 出现之前，它似乎不会传播得足够广泛。
