# 索尼展示 Energy Link 便携式 USB 小配件充电器

USB 确实是新的充电标准。索尼并没有试图给每个人强加一种新的充电方式(这不会太不符合其性格)，而是表现得很好，展示了一种有用的新型便携式 USB 充电器。只需插入电池，这个东西就能给你选择的小工具充电——我要说，这比给你的笔记本电脑充电容易多了。它将于今年夏天上市，售价 35 美元，包括一对充电电池。
