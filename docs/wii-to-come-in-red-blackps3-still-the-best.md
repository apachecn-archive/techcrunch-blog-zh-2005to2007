# Wii 将推出红色和黑色——PS3 仍然是最好的

也许你听说过这个[任天堂 Wii](https://web.archive.org/web/20220624004501/http://crunchgear.com/2007/01/26/one-mans-wii-pinion/) 每个人和他们的叔叔都在谈论。GameDrift 上的消息是，他们已经确认任天堂将推出多种颜色的 Wii。目前，红色和黑色已经确定，但不要惊讶地看到
