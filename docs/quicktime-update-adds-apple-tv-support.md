# Quicktime Update 增加了 Apple TV 支持

![](img/68309c02958a74acd359c2e15f051b8f.png)

随着苹果电视[](https://web.archive.org/web/20201020055610/https://crunchbase.com/product/apple-tv)*终于*开始发货，未来的所有者将高兴地欢迎 Quicktime 的最新更新。隐藏在 Quicktime 丰富功能中的是一个名为“导出到 Apple TV”的导出选项它是做什么的？它可以为你的苹果电视导出高清视频，这样你就可以真正使用 42 英寸的等离子电视了。该功能将使用 H.264 编解码器将视频导出为 720×404 或 1280×720 (720p)分辨率的电影文件，这相当于 DVD 质量的电影。

当然，high-rez 不是没有代价的。每部 90 分钟的电影将会为你节省大约 3GB 的硬盘空间。还应该提到的是，你不能将结果文件传输到你的 iPod 上，但你可以在 iTunes 中查看它们。没关系，反正我也不想在我的 iPod 屏幕上看高清电影。
