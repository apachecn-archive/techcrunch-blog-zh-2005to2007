# Nero Linux 3 获得蓝光和 HD DVD 支持

![linux3_header2.gif](img/989647804771815d355a35e9597f02c1.png) Nero 今天宣布 Nero Linux 3 现已上市。该软件的两个有趣之处在于它可以刻录蓝光和 HD DVD 光盘。Nero 声称 Linux 3 是最强大和最通用的 Linux 刻录应用程序——我倾向于同意。

除此之外，它仍然只是刻录软件。伟大的燃烧软件，但它是什么。
