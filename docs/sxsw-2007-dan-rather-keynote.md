# SXSW 2007:丹·拉瑟主题演讲

我们刚刚参加了丹·拉瑟的 SXSW 主题演讲。我不知道具体会有什么与技术相关的内容，但我会在博客上直播。

2:16–丹·拉瑟已经走上舞台。他引用了一句可爱的林肯名言。

2:20–主持人提出了一个问题，询问指责尼克松说谎是什么感觉。声学问题。她不得不重复三遍，但现在她已经明白了这个问题。

他解释说，记者有责任成为信息经纪人。他们是向公众传递消息的渠道。那些问普通人没有能力问的问题的人。这个答案带有对当前记者的批评，他们只问一些轻微的问题，很少直接询问总统为什么他的答案与事实不一致。

他开了一个玩笑——他说美国新闻业需要脊椎移植。他将此定义为，新闻业、政府和美国企业界变得过于亲密。这是一种你帮我，我也帮你的情况，记者被当作木偶。

2:35–现在没有太多与技术相关的事情发生…

2:39——相反，这表明美国新闻业的理想应该是一种监督模式。他说，尽管新闻业已经变成了一个哈巴狗的行业，寻找容易的独家新闻，从来不想节外生枝，并有可能失去消息来源。

2:54–讨论新闻业保密等级的差异。相反，解释“在背景中”意味着不透露来源，“深层背景”意味着甚至不透露来源可能位于何处，“不公开记录”意味着什么都不说。
