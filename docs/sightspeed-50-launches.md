# SightSpeed 5.0 发布–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/26/sightspeed-50-launches/>

 [](https://web.archive.org/web/20211025043412/http://sightspeed.com/) 总部位于加州伯克利的 [SightSpeed](https://web.archive.org/web/20211025043412/http://www.sightspeed.com/) ，一家 IP 视频和语音服务公司，将于今晚发布其 5.0 产品。

虽然许多变化都是对其视频产品套件(包括新的视频编解码器)和用户界面(已经很好)的升级，但 SightSpeed 在此次发布中推出了几款产品。其中，它增加了新的 PSTN 出局和入局功能，将 SightSpeed 功能扩展到移动电话和普通电话。它还利用其视频专业知识增加了一项新的“异地”电视服务，在没有硬件的情况下给用户带来“类似弹弓”的体验。一个小的遥控图标可以让你在家里的电视上切换频道。

凭借一款以 30 fps 和超低延迟兜售视频的备受推崇的视频产品，增加电视是其技术的自然延伸。此外，这可能被证明是一个精明的竞争举措，Novac 基于 Skype 的电视产品将成为易地空间的众多参与者之一。

在一个竞争激烈的领域，有大量资金雄厚的竞争对手，如 Skype，产品质量至关重要。视觉速度真的可以传递。他们负担得起的视频会议服务是惊人的。我们并不孤单。例如，阅读 Davis D. Janowski 关于 SightSpeed 在 [PCMag](https://web.archive.org/web/20211025043412/http://www.pcmag.com/article2/0,1895,1979189,00.asp) 的新功能集的令人屏息的描述。
