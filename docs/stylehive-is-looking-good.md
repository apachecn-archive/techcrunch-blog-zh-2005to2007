# Stylehive 看起来不错

 [](https://web.archive.org/web/20220810104632/http://www.stylehive.com/) [Stylehive](https://web.archive.org/web/20220810104632/http://www.stylehive.com/) 是一个专注于购物的社交书签网站。我相信这个领域是有希望的，包括刚刚获得资助的 [Kaboodle](https://web.archive.org/web/20220810104632/http://www.kaboodle.com/) ，以及 [Wists](https://web.archive.org/web/20220810104632/http://www.wists.com/) 和即将推出的 [Plum](https://web.archive.org/web/20220810104632/http://www.plum.com/) 。为什么？因为网购 80%是“研究”，真正“购买”的只有 20%。Stylehive 占据了 80%的市场份额。

Stylehive 由 Emily Chang 和 Max Kiesler 在 [Ideacodes](https://web.archive.org/web/20220810104632/http://www.ideacodes.com/) 设计，也是一个视觉上令人惊叹的网站——只要看看[主页](https://web.archive.org/web/20220810104632/http://www.stylehive.com/)就知道了。他们特别关注热门设计和趋势。

style hive 和普通书签引擎的主要区别在于关注不同类型的元数据(描述性标签、价格等)。)，以及对图像的关注。当您使用 Stylehive bookmarklet 为页面添加书签时，图像也会自动包含在内。

新功能定期推出，包括昨天所有书签的评论功能。该公司由首席执行官迈克尔·凯利创建，总部设在旧金山。
