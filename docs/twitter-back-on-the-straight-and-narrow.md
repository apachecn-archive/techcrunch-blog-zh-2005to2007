# Twitter 重回正轨:创始人比兹·斯通访谈

[![twitter.png](img/cfc4c916db1c317ad3e80613b4f8666a.png)](https://web.archive.org/web/20230327204052/http://www.twitter.com/techcrunch)5 月 17 日我在群组即时通讯服务 [Twitter 上分享了我对持续停机的沮丧。这些回答很有趣。Twitter 用户对这项服务充满热情，许多人将其视为一项关键任务应用。](https://web.archive.org/web/20230327204052/http://www.twitter.com/)

两个星期的时间可以改变很多事情。自从写了那篇文章后，Twitter 的正常运行时间有了显著的改善。仍然存在停机问题；据 Pingdom 报道，Twitter 周一离线超过 5 小时[；然而，这些数字是欺骗性的，因为当 Twitter 上线时，质量会有明显的提高。Twitter 没有受到间歇性功能故障的困扰，如 API 支持问题，这些问题阻止了第三方工具如](https://web.archive.org/web/20230327204052/http://uptime.pingdom.com/site/month_summary/site_name/twitter.com) [Twitbin](https://web.archive.org/web/20230327204052/https://techcrunch.com/2007/05/05/twitbin-inline-fifefox-twittering/) 的连接，就像两周前的情况一样。

我问推特的比兹·斯通推特的幕后发生了什么。

**继本月早些时候的问题之后，Twitter 变得更加稳定了吗？Twitter 团队从这次经历中学到了什么？**

比兹·斯通:“我们努力隔离我们系统的各个部分，以防止一次中断导致整个网络瘫痪。对前几次不稳定原因的调查让我们对我们的使用模式有了更多的了解，并为我们展示了一条为未来的增长和使用浪潮做准备的更清晰的道路”。
 **Twitter 的可靠性从哪些方面提高了？**

比兹·斯通:“我们最近在可扩展性方面的努力使我们能够保持当前的增长曲线，甚至在我们添加新的发送和接收更新的方式时提高速率，如我们最近的脸书应用程序”。

**下一步的可扩展性是什么？**

比兹·斯通:“我们将继续隔离系统的不同方面，并对高使用率区域进行微优化。我们目前正在实施额外的内部工具，这将使我们能够对使用模式进行更有意义的研究。最后，我们正在增加更多的机器”。

让我们希望 Twitter 保持正直。
