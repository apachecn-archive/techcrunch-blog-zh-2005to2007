# 松下 Lumix DMC-TZ3:10 倍变焦，小型封装

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/09/panasonic-dmc-tz3-10x-zoom-small-package/>

想找一个可以一起环游世界的好旅伴吗？松下 Lumix DMC-TZ3 看起来是首选。720 万像素的摄像头采用广角(28 毫米)徕卡镜头，能够实现高达 10 倍的平滑光学变焦。

虽然它不是最纤薄的相机，但它实际上很紧凑，可以轻松放入夹克口袋。ISO 高达 3200，你可以捕捉从明亮的白天照片到黑暗的晚餐室内照片的任何东西。

松下的变焦控制在快门按钮周围的弹簧环上。与翘板开关相比，它可以更流畅地控制放大和缩小，但如果你习惯了翘板开关，也需要一些时间来适应。模式盘的位置非常适合拇指操作，当然，除非你的拇指很短。

TZ3 上没有取景器。只有这个巨大的 3 英寸屏幕。
