# 东芝召回 10，000 块索尼制造的电池

![laptopfire](img/dd469a4afff436a80dc84f13cb03e940.png)

大约 5 年前，我正在信息高速公路上漫游，想着自己的事情，突然我的显示器发出一声尖锐的爆裂声，从顶部射出一个小到中等大小的火焰。我很害怕，但没有受伤。

阅读这些关于笔记本电脑电池爆炸的帖子和文章让我想知道如果多年前我的显示器在我的腿上，或者更糟的是，不是显示器而是笔记本电脑(显示器不属于膝盖)，我会有什么感觉。

显示器爆炸只是让我感到害怕，但如果是笔记本电脑爆炸，我会感到害怕，痛苦，愤怒，最重要的是，诉讼。为了避免类似事件的发生，东芝已经召回了 10，000 个索尼制造的笔记本电脑电池，此前其三个笔记本电脑(使用索尼电池)起火。幸运的是，没有人受伤，而且预计索尼不会花费太多来更换危险的电池。索尼公司可能要花更多的钱来替换某人的胯部。
