# 诺基亚斯普林特公司将 WiMax 引入 N800 

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/09/sprint-nokia-bring-wimax-to-n800/>

从表面上看，[诺基亚 N800](https://web.archive.org/web/20151230155453/http://crunchgear.com/?s=N800) 是一款相当简单的设备。这是一种互联网设备，或者诺基亚喜欢称之为“互联网平板电脑”。凭借其内置的 Wi-Fi 和蓝牙，如果你有手机或热点，它可以让你在任何你想去的地方浏览网页和收发电子邮件。如果没有，你就不走运了。

不过，在不久的将来，你会变得更有联系。诺基亚已经与 Sprint 合作，将 WiMax 纳入新的 N800s，以配合 Sprint 即将推出的 4G 网络。Sprint 正在积极推进其 4G 部署，因为该标准允许语音和数据通过相同的通道传输，这意味着更快的带宽和更便宜的通话。

此外，由于 N800 具有全面的 Skype 支持功能，所以听起来我们*就要*实现无限制通话和数据计划了，这正是我们非常非常想要的。
