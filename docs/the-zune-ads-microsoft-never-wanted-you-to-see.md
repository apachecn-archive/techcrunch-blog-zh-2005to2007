# 微软不想让你看到的 Zune 广告

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/17/the-zune-ads-microsoft-never-wanted-you-to-see/>

![](img/11ca89fdf50c24e507d723f4fbbe944e.png)

啊是的。Zune 发布几天后，每个人都在模仿一切，从安装屏幕到愚蠢的“欢迎加入社交”口号。我们最喜欢的无疑是我们在 [deepsignal 的 flickr 账户](https://web.archive.org/web/20160417041654/http://www.flickr.com/photos/deepsignal/298765089/)上发现的那个巨大的狂欢派对。但是等等，这还不是全部。我们还发现了一些原本面向大学生和青少年的新的竞选广告，微软在这个家伙的 flickr 页面上完全放弃了这些广告。祖恩，在成为福尔摩斯兄弟中的一员之前，你还有很长的路要走。尽管微软做了一些正确的事情——crunch gear 编辑布莱克·罗宾逊在纽约找不到 Zune！但他来自新奥尔良，当曼哈顿的 R-train 停止运行时，他会哭，所以他可能还不够努力。继续找伙计。

[更新，来自 Matt Hickey]为了完全公开，我们应该注意到你在这张照片中看到的人实际上是我的街头帮派 Yahtzee Posse 的成员。文斯发现了这张照片，它越来越受欢迎，再加上我为这个博客写作，这一切都只是一个很大的巧合。其实直到有人指出来我才认出他们。他们也是我保龄球队的成员。真的。你好，来自西雅图。他们有时都住在这里。
