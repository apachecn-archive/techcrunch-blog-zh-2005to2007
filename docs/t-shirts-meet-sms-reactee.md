# t 恤遇到短信:Reactee

[![reactee2.png](img/855362f76293a0784e9f853486722a2a.png) ](https://web.archive.org/web/20220701020524/http://reactee.com/) [Reactee](https://web.archive.org/web/20220701020524/http://reactee.com/) 宣布推出一系列互动 t 恤，通过“回短信的衬衫”将时尚、短信和行动主义结合在一起。

Reactee 允许用户创建包含个性化信息的 t 恤，如“停止全球捕鲸”或“安德鲁·基恩是一个卢德分子”，然后在衬衫上加上一个独特的关键词，如寿司或低能儿。看到这件衬衫的人可以通过短信发送关键字到 41411 来回复它。反过来，发件人会收到 t 恤制作者制作的定制短信回复。

给出的 Reactee 客户示例包括个人，如希望分享其播放列表的 DJ、宣传候选人的政治活动家、只想一吐为快的人，或者整个组织，这些组织可以制作许多带有相同关键字的衬衫，并使用它们来宣传其独特的事业。

用户可以创建文本警报列表，与对其事业感兴趣的人进行交流。此外，用户可以公开他们的设计，并将其纳入收到最多短信的衬衫的 Reactee gallery。

现有用户包括[美国爱乐乐团](https://web.archive.org/web/20220701020524/http://www.poamericas.org/)和[青年噪音](https://web.archive.org/web/20220701020524/http://www.youthnoise.com/)。
