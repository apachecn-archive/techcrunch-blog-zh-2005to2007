# 回顾:鱼鹰子午线和光环包

![54_5501_lg.jpg](img/5eda40d45bc2688c04bbdc963c4e9491.png)

虽然在过去的几个月里，我没有像其他一些 CrunchGear 作家那样经常旅行，但我已经做了足够多的事情来让一套鱼鹰背包——[子午线 28 英寸](https://web.archive.org/web/20211017123154/http://www.ospreypacks.com/Packs/MeridianSeries/Meridian2875L/)和[光环 65](https://web.archive.org/web/20211017123154/http://www.ospreypacks.com/Packs/AuraSeriesWomens/Aura65/)——通过铃声。他们去过意大利、韩国、波多黎各和纽约周围的许多地方。我很高兴地说他们成功地通过了。我将从经络开始，因为它是我过去几个月的主要工作。我最喜欢的功能是拉链式背包，有足够的空间放我所有的旅行装备。daypack 内部有一个笔记本电脑专用的填充隔层，顶部带扣，以确保您的宝贝不会四处碰撞。顶部还有一个小拉链，可以打开一个小网袋，可以存放数码相机、PMP 或钱包。侧面的网状瓶袋确保你不必脱下包装就能得到你喜欢的美味饮料。总的来说，这个包超级方便，尤其是当你不像我一样喜欢把东西放在口袋里的时候。这个小家伙拉上拉链，扣在主行李箱上，这使得通过机场变得轻而易举。

这个手提箱有足够的空间进行短途旅行，它不仅可以滚动，而且如果你不想拖着包到处走，它还有可以收起的带子。这是令人惊讶的舒适，虽然我最喜欢把它推来推去。外面的一个大隔间足够放书、杂志和一双人字拖。当我把包放在酒店房间时，我喜欢做的一件事就是把一些贵重物品放在可以系带子的地方。

在技术方面，子午线有鱼鹰的豪华悬挂系统，让你的下背部免受压力。这也有助于消除你背着包时下背部出汗的情况。包带的唯一缺点是，它消除了主隔间的许多包装空间，因为系统弓入。总的来说，我发现子午线是一个完美的融合，以适应我的一日游和航空旅行的需要。

Aura 65 是一个完美的徒步旅行包，它有很多隔间，有足够的空间去意大利旅行一周。我的旅行伙伴仍然遭受着一场车祸的不良影响，这场车祸使她的下背部容易紧张和疲劳，但空速悬挂系统工作得如此之好，以至于她在整个旅程中几乎没有抱怨过下背部疼痛。在我看来这是一个很大的优势。Aura 内部还有一个方便的水隔间，有一个槽可以穿过你的软管，但我事先不知道这一点，所以我无法安装一个类似骆驼的袋子。

唯一的缺点是一个弯曲的框架，在从纽约到罗马的途中受到了影响，但我们会把这归咎于行李搬运工。我花了几分钟把框架弯回来，我们准备好了，虽然，我不能让它回到原来的形式。

这两种包装现在分别售价 299 美元和 229 美元。
