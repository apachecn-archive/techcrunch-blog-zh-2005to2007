# 证实:MySpace 将以 2.5 亿美元收购 Photobucket

 [](https://web.archive.org/web/20220702220059/http://www.photobucket.com/) 显然，一名过于热心的 Photobucket 员工是这一[谣言](https://web.archive.org/web/20220702220059/http://www.techmeme.com/070507/p46#a070507p46)的源头，但我们已经向更资深的人士证实了这一点:MySpace 将以 2.5 亿美元的现金收购 Photobucket。我们听说还有高达 5000 万美元的额外收入。

自从 3 月以来，Photobucket 一直在寻找买家，当时他们聘请了雷曼兄弟来帮助出售公司。他们希望获得 3 亿美元或更多，但除了 MySpace 之外，可能没有其他竞标者。

过去几周，两家公司一直在进行认真的收购讨论——一场涉及 Photobucket 视频在 MySpace 上被屏蔽的[纠纷](https://web.archive.org/web/20220702220059/http://www.beta.techcrunch.com/2007/04/10/photobucket-videos-blocked-on-myspace/)导致了收购讨论，该屏蔽[被移除](https://web.archive.org/web/20220702220059/http://www.beta.techcrunch.com/2007/04/24/photobucket-back-on-myspace-i-want-to-know-the-backstory/)。

Photobucket 去年创造了 630 万美元的收入，今年计划达到 2500 万美元或更多。他们有 4000 万注册用户，并且每天增加 85000 人。

我们第一次报道 Photobucket 是在一年前。他们在两轮融资中筹集了 1500 万美元。
