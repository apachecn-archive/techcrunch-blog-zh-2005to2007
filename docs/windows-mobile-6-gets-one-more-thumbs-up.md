# Windows Mobile 6 又多了一个大拇指

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/08/windows-mobile-6-gets-one-more-thumbs-up/>

如果在不久的将来你会有一台运行 Windows Mobile 的设备，看起来你肯定会想要一台运行最新版本的。笔记本电脑杂志的主编马克·斯普瑙尔对新推出的版本 6 进行了彻底的测试，不仅通过了测试，而且实际上很喜欢这个操作系统的更新。

Spoonauer 称赞它比它的前辈更快更稳定，改进了电子邮件和网络搜索功能，还有一个实际上可用的日历视图。另外，微软在界面上增加了一些类似 Vista 的外观。

总体而言，缺点似乎很少，比如 Live Messenger 不支持其他 IM 客户端，以及浏览器对多媒体的支持较差。但最大的阻力是，许多更酷的功能是为 Exchange 2007 用户保留的，例如在公司服务器上搜索不在您设备上的电子邮件。

就我个人而言，我非常不喜欢使用 WM 5，任何东西都将是一种改进。但看起来微软确实解决了我在使用它时遇到的许多问题，主要集中在它缓慢的性能和在我最需要的时候死机或崩溃的不可思议的能力上。
