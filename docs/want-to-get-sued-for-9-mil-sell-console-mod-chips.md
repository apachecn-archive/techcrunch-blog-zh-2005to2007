# 想被起诉索赔 900 万美元吗？销售控制台 Mod 芯片

根据 Ars Technica 的一份报告，总部位于法国的公司 Divineo 和另外两名被告将不得不为销售索尼 PlayStation 游戏机的 mod 芯片和 HDLoader 支付超过 900 万美元的赔偿金，HDLoader 是一款可以让你将游戏抓取并存储到连接到系统的硬盘上的应用程序。出售这些芯片违反了数字千年版权法，因为它们破坏了游戏机的版权保护措施。

> “Mod 芯片和 HDLoaders 是助长视频游戏盗版的关键因素，因为它们允许人们在非法改装的视频游戏控制台上玩非法复制的游戏，”ESA 知识产权执法高级副总裁 Ric Hirsch 说。

HDLoader 仍然很容易获得，就像控制台 mod 芯片一样，所以我不知道这对 ESA 来说是多大的胜利。我所知道的是，仅仅是阅读这个故事就让我想对我的游戏机做肮脏、非法的事情。
