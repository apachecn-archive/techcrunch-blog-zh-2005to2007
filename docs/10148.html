Attensa发布其Feed服务器| TechCrunch的新版本

原文：https://web.archive.org/web/http://www.techcrunch.com/2007/02/12/attensa-releases-new-version-of-its-feed-server/trackback/

企业RSS供应商 Attensa 今天早上发布了其新一轮关注数据的RSS服务。看哪家企业RSS供应商能够想出如何让商业世界使用一种似乎无可否认对其有利的技术(RSS)的竞赛仍在继续。

Attensa的enterprise feed server 1.1版本新增了以下功能，所有这些功能都是对RSS套件的合理、智能的补充。该产品以前是作为已经安装了软件的服务器交付的，而该公司现在提供了一个完整的安装包，可以安装在以其他方式购买的硬件上，以及一个完全托管的解决方案。“虚拟服务器”旨在减轻安全顾虑，托管解决方案面向小型企业。

部门或项目负责人能够决定他们自己的团队对订阅提要的选择，这也是一项新功能。这以前是在一个层次上管理的，自上而下。提要服务器产品中还添加了一组持久搜索选项。对于消费级RSS用户来说，这两个步骤可能看起来很奇怪，但是在一些高度控制的企业中，自由地订阅他们找到的任何和每一个提要并不是许多公司为他们的员工所寻求的。Attensa告诉我，一家大型银行的客户让银行柜员使用他们的产品来接收促销信息，而另一家制药公司的客户希望他们的研究馆员能够订阅任何内容。

该服务的其他变化包括报告的复杂性增加，可以报告和搜索个人的阅读习惯，以及允许或阻止特定类型的RSS附件的新管理功能。

Attensa在企业RSS市场面临来自Newsgator和KnowNow的竞争。这三者都是非常不同的服务；Attensa专注于阅读列表的自动定制和报告注意力数据或用户行为。

这三家公司的任何一款产品都有能力彻底改变组织与信息的关系，但似乎它们都没有卖出大量产品。老牌公司要么开始使用RSS，要么很快就会被新贵吃掉，对他们来说，feed阅读是工作流程的重要组成部分。
