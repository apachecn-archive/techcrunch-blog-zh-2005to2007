# Helio 漂移实践

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/06/helio-drift-hands-on/>

![](img/5f0d8c4a00c1937e6f2ef376e7ccaeaf.png)

在过去的几年里，手机领域发生了一些巨大的变化。Nextel 与 Sprint 合并，摩托罗拉重新成为主要竞争对手，Cingular 继续增长，等等。也许这些发展中最特别的是移动虚拟网络运营商(MVNO)的出现和发展。

MVNOs 本质上是通过从更大的提供商那里租赁容量来运行的。Helio 就是这样一家 MVNO，由韩国 SK 电讯和 Earthlink 合资成立，两家公司持有同等股份。它从 Sprint 租赁网络容量。

![](img/92606be99770d859c5034ed3d91160e2.png)

Helio 将自己定位为我们 MVNOs 中的佼佼者。自去年 5 月推出以来，它一直以其引人注目的独特口号“不要称我们为电话公司”而大放异彩。不要叫它电话。”所以如果它不是一家电话公司，也不提供电话，你可能会想这到底是怎么回事？

![](img/3505c178dd0abbb3d9c3bd318f5ddd28.png)

事实是，Helio 是一家电话公司，但它是一家非常不同的电话公司。它主要寻求创造一种不同的用户体验，一种主要针对年轻人和时髦人士的体验。证明这一点的是 Helio 与 Myspace 的合作伙伴关系，它是唯一有任何合理方法访问 Myspace 的移动提供商(MVNO 或其他)。Myspace 被集成到 Helio 设备的用户界面中。但不仅仅是这种伙伴关系。

![](img/5f0d8c4a00c1937e6f2ef376e7ccaeaf.png)

事实上，Helio 已经通过收购一系列非常受欢迎的~~手机~~设备的专属领域而脱颖而出。在推出了 Hero 和 Kickflip(以前的 YT)之后，Helio 又推出了 Hydrid WLAN 卡，现在又推出了三星的漂移。

Helio Drift 或许可以被视为 MVNO 的第一款设备，它可能会引起大型提供商用户的嫉妒。目前，它可以被视为乔恩·赫斯(曾出演《大人物拿破仑》的导演)导演的一个大型广告活动的一部分:

漂移有一个性感的滑块体，有黑色或限量版白色。它的背面印有 HELIO 火焰，在其 2.12 英寸 QVGA 显示屏下写有 Helio 字样。在功能方面，Drift 拥有蓝牙、microSD、EV-DO 和媒体播放器等标准功能，200 万像素摄像头的非标准功能和集成 GPS 的尖端功能。

重要的事情先来。虽然它可能不能被称为“电话”，但它确实能打很好的电话。在经历了我的高度漫游习惯后，我印象深刻地说，通话质量通常是一流的，除了我用这款设备打的第一个电话，我没有掉线。Helio 得一分。

界面很好，很直观。当你滑开手机时，它会亮起，你可以用手指在方向键上滑动。选项屏幕以太阳为中心，选项围绕着你的联系人和群组图标，这似乎是 Helio 的说法，即服务围绕着用户。

我很高兴看到拍照手机的质量在攀升。漂移是迈向这一现实的一步。它的 200 万像素相机包括一个闪光灯，可以拍摄合理的照片。

EV-DO 的网络在获取电话数据时速度很快。我完全不确定这款手机是否支持拨号接入——我从未能让它与我的 Powerbook 配对，这很不幸，但并不罕见。

迄今为止，漂移最令人期待的功能是它的 GPS 功能。简单来说，它的 GPS 挺酷的。这一增加为订户打开了不少大门。用户可以使用谷歌地图来精确定位。此外，通过添加好友信标，您可以跟踪您的好友，并找到他们的具体位置。这为专横的男朋友提供了一个简单的手段来跟踪他们的女朋友，反之亦然(没有理由女士们不能跟踪)。Helio 还预计全球定位系统的其他几项功能将在不久的将来开发出来。

我在漂移中遇到的最大问题是按钮。正如我之前提到的，我的手相对较大，所以这可能是更大的人的专属，但我在漂移中使用方向垫有些困难。这不是可怕的残疾或任何东西，但它可能有点乏味，当试图快速移动时，你最终会因为距离太近而打错东西。至少通过练习会变得更容易。

所以结论是，尽管按钮有点小问题，我真的挖掘了漂移。Helio 花了很大力气在这里创造独特的用户体验。从你拿到盒子的那一刻起，这就是显而易见的事实。它坐落在这样一个像样的时尚，你不能不觉得拥有它有点特别。这款设备本身轻薄时尚，你不会后悔把它装进口袋。有了 Helio 全包会员资格，你可以使用漂移的许多功能，直到你的手麻木。
