# 松下滴科学在兴奋剂复古耳机|科技博客

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/16/panasonic-drops-science-in-dope-retro-headphones/>

如果你是那种喜欢闪回风格的人，松下是不二之选。它的 RP-HTX7 耳机看起来像你可能从你父亲的衣柜里偷出来的东西，在踢出老派节拍时看起来很好，但这种外观掩盖了现代动态响应驱动器对清脆、现代声音和紧凑、智能低音的需求。

然而，质量不仅仅在于技术。这款复古头盔还配有真皮和五种颜色的塑料，所以至少有一种可以搭配你的鞋子。59 美元，你知道你可以买得起这款高保真音响。

好吧，我不想再听街头音乐了。
