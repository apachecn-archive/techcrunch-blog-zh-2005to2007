# 索尼背投:优步薄

![](img/135d65c28d7dc6b92f2f8609739177a5.png)

我和一些认识在索尼工作的人聊天，他们很生气。从本质上说，营销预算被这样的东西吃掉了——一台 1080p 液晶电视，采用了超薄设计——公司的其余部分得到了轴。这款名为 KDF-46E3000 的产品是一款薄型背投产品，有 37 英寸、46 英寸和 50 英寸三种型号。

随便你怎么说索尼电视，但这场战争已经结束了，伙计们。普通消费者想要好市多(Costco)售价 1500 美元的 42 英寸等离子电视，而不是售价 1300 美元、正面印有 WEGA 的 37 英寸平板电视。如果你给类似规格的等离子索尼定价，你会大吃一惊。

抱歉，我对此大发雷霆，但索尼的这些东西只是提醒我，他们需要做得更好。PS3，[不管我们怎么说](https://web.archive.org/web/20210307133503/http://crunchgear.com/2007/01/26/the-playstation-3-is-the-best-console-ever/)相反，挥霍了 PS2 的爱，正在输给 [Wii](https://web.archive.org/web/20210307133503/http://crunchgear.com/2007/01/26/one-mans-wii-pinion/) 。他们最近采取了一些笨拙的行动，我担心这是一种趋势。
