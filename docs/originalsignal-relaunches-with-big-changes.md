# 原始信号重新推出，有很大变化| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2006/11/23/originalsignal-relaunches-with-big-changes/>

 [](https://web.archive.org/web/20161022225450/http://originalsignal.com/) 流行的单页面聚合器 [OriginalSignal](https://web.archive.org/web/20161022225450/http://originalsignal.com/) 已经[重新推出](https://web.archive.org/web/20161022225450/http://blog.originalsignal.com/?p=12#respond)，它现在不仅仅是一个为懒人准备的不灵活的起始页——它增加了一系列功能，使它比以往任何时候都更有用。

该网站展示了各种主题的最受欢迎博客的最新标题。当你将鼠标悬停在标题上时，会显示每篇文章的前几行。自上次查看该页面以来的新项目在“已读”中突出显示。这使得它很容易在一天中弹出到 OriginalSignal，并查看所选博客是否已经更新，甚至不需要阅读标题。

需要说明的是，几个月来，我一直是使用 OriginalSignal 的懒人之一，而不是花时间编写自己的起始页来补充我的 feed 阅读器。不过，随着今天的重新推出，OriginalSignal 带来了比以往更多的东西。

第一个页面是在 Web 2.0 上，然后在过去的几个月里增加了其他几个页面，今天该网站增加了关于科学、游戏、15 个博客、苹果、世界新闻、政治、体育、金融、营销、电影和视频的页面。每个页面都有一个“最受欢迎”的部分，按日、周、月划分。

除了新的热门页面，该网站现在还包括雅虎提供的搜索服务。，能够调整显示中的字体大小，改变显示的提要的顺序，抓取一个小部件来显示您博客上某页的提要，以及*也许最重要的*，在手机上查看每一页的大量新闻版本。除此之外，还有几个其他的功能更新，网站现在看起来也很不一样。

视频页面显示 YouTube、MySpace Video、Yahoo Video、Google Video、iFilm、Metacafe、Break.com、Dailymotion 和 Grouper 上最近添加的视频的屏幕截图。如果色情冒犯了你，你很可能会在任何时候在视频页面上看到你不喜欢的东西。

像 [Netvibes](https://web.archive.org/web/20161022225450/http://netvibes.com/) 、 [Pageflakes](https://web.archive.org/web/20161022225450/http://pageflakes.com/) 或 [Webwag](https://web.archive.org/web/20161022225450/http://webwag.com/) 这样的起始页仍然提供更多的灵活性，成熟的 RSS 阅读器是必不可少的——但 OriginalSignal 是一个方便、快速的聚合工具，它为几乎所有人提供一些内容和一些功能。这是让你熟悉一些流行博客的简单方法。我预计网站的新版本会比第一版更受欢迎。

如果你对单页聚合感兴趣，一定要查看一下 [Popurls](https://web.archive.org/web/20161022225450/http://popurls.com/) 和 [Brian Benzinger 的帖子](https://web.archive.org/web/20161022225450/http://www.solutionwatch.com/501/tracking-the-web-with-single-page-aggregators/)，评论许多做类似事情的服务。
