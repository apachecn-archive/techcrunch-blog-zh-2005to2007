# x-Arcade tank stick–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/18/x-arcade-tankstick/>

![](img/bbb7647007294ab0ded028abf280f4f5.png)

我喜欢这些东西。我的朋友在 Dreamcast 时代就有一个，我们会玩“漫威对卡普空”好几个小时。啊回忆。

不管怎样，X-Arcade 发布了一款重新设计的广受欢迎的街机控制器。它有两个操纵杆，每个操纵杆有八个按钮，还有一个轨迹球，这使得它可以兼容大多数街机游戏。Tankstick 宽 30 英寸，重 20 磅，由重型工业级材料制成，X-Arcade 声称这种材料坚不可摧。它是否真的坚不可摧并不重要，因为 X-Arcade 为它提供了终身保修。它带有一个用于 PC 和 Mac 的 USB 连接器，但适配器可用于 XBOX、PS2、Dreamcast 和 game cube——没有关于下一代兼容性的消息。
