# 新闻藤进入社会新闻行列

 [](https://web.archive.org/web/20221129072104/http://www.newsvine.com/) [Newsvine](https://web.archive.org/web/20221129072104/http://www.newsvine.com/) 尚未推出，但创始人兼首席执行官 Mike Davidson 已经[发布了一旦推出将会是什么样的](https://web.archive.org/web/20221129072104/http://www.mikeindustries.com/blog/archive/2005/11/get-ready-for-newsvine)。

Newsvine 将是一个合作性的社交新闻网站。像其他网站一样，Newsvine 将展示主要的新闻出版物。然而，读者可以对新闻条目进行评论，从而进行讨论。

他们还增加了标签。任何用户都可以用“体育”或“伊拉克”等标签为一条新闻(或任何网页)添加书签，它将在 Newsvine 的 URL newsvine.com/[TAG]下提供。如果您对某个标签上的新闻感兴趣，您当然也可以直接访问该 URL。

Newsvine 将使用一个投票系统，如 Digg，来确定一个给定新闻条目的相关性。我认为这些由用户决定的排名系统，像 Digg 和 Memeorandum，是将好内容推上顶端的好方法。我很期待看到现场直播。(经由[史蒂文·科恩](https://web.archive.org/web/20221129072104/http://www.librarystuff.net/)和[奥姆·马利克](https://web.archive.org/web/20221129072104/http://gigaom.com/2005/11/09/introducing-newsvine/))。
