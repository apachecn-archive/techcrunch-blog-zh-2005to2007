# TomTom ONE GPS 在北美发布

在推迟了很久之后，这个带有 3.5 英寸大屏幕的微型 GPS 导航系统本月登陆美国。它只有 5.6 盎司重，1 英寸厚，对于 GPS 系统来说是很小的。当你的朋友开车时，把这个放在你的口袋或包里，以防万一。

它有 32MB 的 SD 内存，可以访问 TomTom PLUS 服务，如交通和天气。作为一款入门级的设备，还是挺体面的。但 499 美元，对于入门级用户来说可能有点太贵了。
