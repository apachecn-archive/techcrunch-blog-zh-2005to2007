# Dave Winer 加入 Web 2.0 工作组

我非常高兴地宣布[戴夫·怀纳](https://web.archive.org/web/20221202095844/http://www.thetwowayweb.com/2005/10/12#a2075)撰写的网络已经加入我们的[网络 2.0 工作组](https://web.archive.org/web/20221202095844/http://www.web20workgroup.com/)。在这里查看我们对工作组[的介绍。](https://web.archive.org/web/20221202095844/http://www.beta.techcrunch.com/2005/10/10/web20-workgroup/)

TheTwoWayWeb 是 Dave 的博客之一，他在这里写关于 web 2.0 主题的文章。与脚本新闻不同，戴夫在这个网站上允许评论和引用。它是 web 2.0 思维和新闻的绝佳资源。

从[关于页面](https://web.archive.org/web/20221202095844/http://www.thetwowayweb.com/about):

> 双向网络是网络作为一个简单的写作和出版环境的愿景。这并不是一个新的想法，事实上这是它的发明者蒂姆·伯纳斯·李在 1997 年 12 月定义的网络的最初愿景:“使创作成为日常生活自然部分的直观编辑界面仍在成熟。”
> 
> 大多数对这些技术感兴趣的其他公司想要建立一个金钱、商品和服务的网络。我为此鼓掌，但这不是我的目标。我想把网络变成一个强大且易于使用的写作环境。

欢迎你戴夫。
