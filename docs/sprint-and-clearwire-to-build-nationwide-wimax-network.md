# Sprint 和 Clearwire 将建立全国性的 WiMax 网络

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/19/sprint-and-clearwire-to-build-nationwide-wimax-network/>

![wimax.jpg](img/5f48caf88750f9cf69ca3302ed98567d.png)今天，Sprint Nextel 和 Clearwire 宣布了联合构建全国首个使用 WiMax 技术的移动宽带网络的计划，并进一步推动基于 WiMax 的服务在全球的发展。该网络旨在在全国范围内提供移动宽带服务，包括城市、郊区和农村地区，Sprint Nextel 和 Clearwire 希望建立自己的部分，并实现各自地区之间的漫游。

Sprint Nextel 董事长兼首席执行官 Gary Forsee 表示:“这一安排将为快速增长的宽带服务市场带来更激烈的竞争，并将为消费者、国家企业和其他企业、教育工作者和公共安全机构提供更多选择和更快地获得革命性的移动宽带技术。“这将推动我们在全国范围内提供移动 WiMAX 服务的目标，并帮助实现我们在所有网络中提供移动产品和服务的愿景。”

这份协议不是板上钉钉的事；因此，不要出去寻找 WiMax 笔记本电脑卡或 WiMax 手机。该交易取决于最终协议的执行，但这些协议预计将在未来 60 天内完成。当然，政府需要对此进行审查，包括 FCC 的频谱许可分配和转让。

根据网络扩建计划，Sprint Nextel 将建设覆盖最大的 50 个市场的网络，到 2008 年底，覆盖范围将包括大约 1 亿人，在部署区域内实现无缝漫游。移动 WiMax 服务将在一个共同的服务品牌下销售。
