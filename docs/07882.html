Skype for Windows Mobile 2.2 beta | TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com:80/2006/12/12/skype-for-windows-mobile-22-beta/


Skype刚刚发布了其Windows Mobile 2.2软件的测试版，目前支持智能手机版。我从未能让以前版本的Skype for Windows Mobile正常工作，但也许这个版本会有所不同。

除了智能手机支持之外，新版本还具有增强的今日/主屏幕和代理支持。它还具有所有常规的Skype通话功能。还有移动设备功能，比如可以用手机摄像头拍摄个人资料照片，然后直接发布到你的个人资料中。是的，很明显，但还是很酷。我热爱科技。

正如我以前提到的，我从来没有能够让移动Skype工作，但这个概念对我很有吸引力。我打算今晚在我的21点上试试这个新版本，看看它是否像我怀疑的那样有用。我会在不久的将来给你们一个更新。
