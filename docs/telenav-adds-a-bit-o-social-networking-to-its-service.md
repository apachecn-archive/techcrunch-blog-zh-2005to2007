# TeleNav 为其服务增加了一点社交网络

随着所有苹果发生的事情，或者缺少这些事情，这可能不是发布新版本移动设备应用程序的最佳时机，但不是每个人的世界都围绕着苹果。TeleNav 今天发布了其最新版本的 TeleNav GPS Navigator (v. 5.2)，该版本增加了一些很酷的社交网络工具，可以利用平台(你的手机)和 TeleNav 的服务(GPS)。立即可用，用户现在可以将他们的当前位置发送到任何其他可以接收文本消息的移动电话，从而在您外出时创建一种快速简便的方式来与朋友、家人或同事见面。

您还可以分享最喜欢的商店的位置，包括餐馆、酒店或电影院。给自己找了个便宜的加油站？通过燃油查找功能让每个人都知道。其他新工具包括对附近商家的评论和评级，以及通过手机直接对餐馆进行评级的功能。

新版本的 TeleNav GPS Navigator (v. 5.2)可立即在所有主要运营商的精选手机上使用。[查看该公司的网站，了解你可以获得的所有功能以及你可以在哪些手机上获得这些功能的全部细节](https://web.archive.org/web/20201129021943/http://www.telenav.com/)。**然后，下周回到 CG，因为我们将向几个幸运的读者赠送 3 个月的免费服务以及 TeleNav 的蓝牙 GPS 单元。**
