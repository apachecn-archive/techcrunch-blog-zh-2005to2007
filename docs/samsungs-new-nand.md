# 三星的新 NAND

![](img/0e0e3cdcd985a27253d2cbca3771ce11.png)

三星开发了一种新形式的 NAND 内存，容量和性能大大提高。它被称为相变随机存取存储器(PRAM ),目前只以原型形式存在，但三星打算在 2008 年将其推向市场。

由于其电荷陷阱闪存(CTF)架构，该存储器实现了显著更高的速度，从而降低了单元间噪声水平并以更快的速度驱动。PRAM 的速度提高了 30 倍，因为它可以重写数据，而不必擦除先前存储的数据。三星表示，它可以实现高达 64GB 的密度，但迄今为止，它只展示了 512MB 的版本。
