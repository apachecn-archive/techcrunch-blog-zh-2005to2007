# 沃达丰坚持 3G iPhone

许多分析师认为，沃达丰可能处于一个有利的位置，获得新的苹果 iPhone 的独家、泛欧洲锁定，但有一个条件。iPhone 不能在快速的 3G 网络上运行，预计最早要到明年 3 月才会推出 3G 版本。iPhone 在美国运行在较慢的 2.5 代网络上，由于沃达丰在 3G 网络和服务上投入了大量资金，该公司可能会坚持 3G 或 HSDPA(升级的 3G)版本。
