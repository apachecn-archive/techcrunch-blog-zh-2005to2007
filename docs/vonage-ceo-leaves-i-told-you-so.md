# Vonage 首席执行官离职——我告诉过你！

![](img/6b9cc5fd1bc3b55e02fb42edf9d3cf59.png)

我这么叫它！Vonage 完全是吱吱作响。今天，Vonage 的首席执行官 Michael Snyder 辞去了他的职位，这样他就不会成为一个失败的公司的一员。有传言称，Vonage 也裁减了大量其他职位，当你的股票高达每股 3.11 美元时，这可能是真的。

尽管最近出现了各种问题，Vonage 仍然公布了一份不错的 Q1 收益报告，收入为 1.95 亿美元。不算太寒酸，但是公司真的开始削减开支了。Vonage 表示，计划通过削减开支和解雇员工节省近 1.5 亿美元。老鼠？沉船？
