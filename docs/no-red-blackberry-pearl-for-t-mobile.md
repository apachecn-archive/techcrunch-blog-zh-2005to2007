# T-Mobile  没有红色黑莓珍珠

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/12/no-red-blackberry-pearl-for-t-mobile/>

![](img/01f23566ad5ab1714a58b2b0796e9afb.png)

周末，[一些网站](https://web.archive.org/web/20150926032839/http://www.rimarkable.com/archives/1176) [发布谣言](https://web.archive.org/web/20150926032839/http://www.boygeniusreport.com/2007/02/09/red-blackberry-pearl-coming-to-t-mobile/)关于[红色黑莓珍珠即将上市](https://web.archive.org/web/20150926032839/http://www.blackberrycool.com/2007/02/09/003549/)。我给为 T-Mobile 做公关的同事发了电子邮件，他们说不会很快推出红色或其他颜色的黑莓珍珠。

抱歉，上班族、公司女职员们——看来你们要在情人节选择黑珍珠还是白珍珠了。
