# 索尼爱立信 Cybershot K770 发布

![](img/e1282879ddb0473e9f2c4fac0ad6f7f3.png)

索尼爱立信今天正式公布了备受期待的 Cybershot K770。这款功能强大的照相手机将于第四季度上市，厚度仅为 14 毫米！与其他一些手机不同，它配备了 320 万像素的摄像头[，内置 256MB M2 卡用于存储所有高分辨率照片，一个 LED 闪光灯和一个漂亮的 1.9 英寸 QVGA 显示屏。](https://web.archive.org/web/20201205225213/http://crunchgear.com/2007/08/10/t-mobile-samsung-blast-reviewed/)

再加上 lil' tri-band GSM，你就拥有了一部超棒的手机。好吃！
