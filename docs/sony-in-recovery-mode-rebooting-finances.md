# 索尼处于“恢复模式”，重启财务

奥耶兹奥耶兹。尊敬的霍华德·斯金格爵士先生。第一三类死者现在把索尼公司纳入财务恢复模式，努力扼杀所有这些 PS3 的污秽在萌芽状态。索尼曾依赖这款新游戏机来提高其市场份额和现金储备，但众所周知，它两者都没有做到。

但是“恢复模式”是什么意思呢？不多。他真正想说的是，股东不应该担心，一切都会好的。这也意味着他戴着一朵可爱的花。

> “所有的生产问题都已经解决了。我们已经卷土重来了，”斯特林格说，他承诺增加游戏产品，支持机器的网络平台。
> 
> “我们最初总是在硬件上赔钱，然后我们逐渐收回这些钱，”他说。“我们相信 PS3 的发展对我们的未来至关重要，并会取得成功。”

至少没有裁员或其他削减成本的措施，这个过程在大多数情况下相当于放血。我想我们只能等等看，希望斯特林格知道他在做什么。
