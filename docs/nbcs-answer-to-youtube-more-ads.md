# NBC 对 YouTube 的第二个回答:更多广告(但至少它有个名字)

随着综艺 [![](img/26074130d279cc27b25eb39d5002b476.png)](https://web.archive.org/web/20221007053732/http://flickr.com/photos/pierrefoucart/431326690/) 被[报道](https://web.archive.org/web/20221007053732/http://www.variety.com/article/VR1117969864.html?categoryid=14&cs=1&nid=2562)，NBC 正摩拳擦掌准备推出它的第一部(嗯，[第二部](https://web.archive.org/web/20221007053732/http://www.beta.techcrunch.com/2007/03/23/what-we-know-so-far-about-newtube-isnt-good/))《大突击》对抗 [YouTube](https://web.archive.org/web/20221007053732/http://www.crunchbase.com/company/youtube) 、Didja.com(目前是一个停着的页面)。

该网站将“把广告当作娱乐来庆祝”，并以“当前和经典电视广告、电影预告片和其他品牌相关内容的免费在线档案”为特色。这个名字，Didja，是“你看到了吗”的缩写，这是他们希望从观众那里得到的明显反应。该网站将在 NBC 的电视频道上交叉推广，最终将在 NBC 环球的所有部门推广。它将于明年初推出，在与新闻集团(News Corp .)仍未命名的合资企业之后的某个时间推出，该合资企业被谷歌高管称为“[小丑公司](https://web.archive.org/web/20221007053732/http://www.beta.techcrunch.com/2007/03/23/what-we-know-so-far-about-newtube-isnt-good/)”。

该网站是所有关于广告和 NBC 获得报酬。NBC 希望广告商为网站的显著位置付费，或者创建他们自己的品牌页面(例如，全麦当劳频道)。可定制的品牌页面将允许广告商向网站上传商业广告，包括产品报价、商店定位器和其他工具。

对于观众来说，该网站将具有广泛的社交网络组件，以及混合他们自己的“广告”的混搭工具。

这片土地以前被严重践踏过。TBS 有他们自己的 [Veryfunnyads](https://web.archive.org/web/20221007053732/http://veryfunnyads.com/) 网站，特色是*gasp*“非常有趣的广告”。Clipland 是一个较小的网站，也在做同样的事情。尤其是 YouTube 本身，它吸引了人们真正想看的广告和预告片的最大份额。
