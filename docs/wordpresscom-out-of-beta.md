# WordPress.com 脱离了贝塔

Matthew Mullenweg [今天在](https://web.archive.org/web/20220528190723/http://photomatt.net/2005/11/21/wordpresscom-open/)宣布[WordPress.com](https://web.archive.org/web/20220528190723/http://www.wordpress.com/)，托管(免费)(可扩展)版 [WordPress](https://web.archive.org/web/20220528190723/http://www.wordpress.org/) 现在无需邀请即可使用。我已经创建了一个测试博客，除了你不能设置额外的 ping 服务器(一个只对某些人重要的高级功能)，这是一个很棒的产品。如果你正在寻找一个免费的托管博客，去看看吧。
