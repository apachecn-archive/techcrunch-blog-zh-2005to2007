# 雅虎枪支 CNET 评论

 [](https://web.archive.org/web/20221203090030/http://tech.yahoo.com/) [雅虎科技](https://web.archive.org/web/20221203090030/http://tech.yahoo.com/)今晚推出，明确瞄准大受欢迎的 [CNET 评论](https://web.archive.org/web/20221203090030/http://reviews.cnet.com/)属性。

快速了解这个网站的最好方法是参加[之旅](https://web.archive.org/web/20221203090030/http://tech.yahoo.com/tg/site-tour)，也是从主页链接的。Yahoo Tech 从 Yahoo Shopping、Yahoo Answers、内部专家和第三方来源(如消费者报告)提供有关技术产品的内容。

可以通过标签云或搜索栏访问产品评论。雅虎技术包括产品评论、操作指南和其他内容。雅虎科技也有四个常驻博主为不同的客户群提供产品建议。

他们正在尽最大努力让非技术人员看不懂，并把自己与 CNET 和其他评论网站区分开来。它比 CNET 好吗？不，还差得远呢。评论的深度和质量就是不存在。网站是不是慢的要命？是的，可能是因为交通拥挤和大量使用闪光灯。随着事态的发展，我们将继续关注此事。
