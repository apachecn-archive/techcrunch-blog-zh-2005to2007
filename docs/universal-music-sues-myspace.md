# 环球音乐起诉 MySpace

据[华尔街日报](https://web.archive.org/web/20221124225123/http://online.wsj.com/article/SB116379906035126666.html?mod=home_whats_news_us)(需要订阅)报道，在其首席执行官[于 9 月](https://web.archive.org/web/20221124225123/http://www.wired.com/news/wireservice/0,71791-0.html?tw=rss.index)表示 YouTube 和 MySpace 欠环球音乐集团“数千万美元”后，这家媒体巨头兑现了其威胁，并起诉 MySpace 侵犯版权。**该诉讼最有趣的部分是，环球公司[指控【MySpace 参与了侵犯版权的行为，对有版权的视频进行代码转换，以便可以重播并发送给其他用户。](https://web.archive.org/web/20221124225123/http://today.reuters.com/news/articleinvesting.aspx?view=CN&storyID=2006-11-17T205228Z_01_WEN0071_RTRIDST_0_MEDIA-UNIVERSAL-MYSPACE-URGENT.XML&rpc=66&type=qcna)**

《华尔街日报》指出，今天也是 MySpace 推出新工具的日子，该工具将允许版权所有者标记网站上未经授权的内容。这个工具([我们的报道](https://web.archive.org/web/20221124225123/http://www.beta.techcrunch.com/2006/10/29/myspace-moves-to-protect-copyright-holders/))使用了 Gracenote 的技术。MySpace 在一份新闻稿中表示，该诉讼“没有必要，也没有价值。”对我来说，代码转换的争论听起来很严肃。

在今年秋天威胁 YouTube 和 MySpace 几周后，环球影业[在谷歌宣布收购的前一天与 YouTube](https://web.archive.org/web/20221124225123/http://www.beta.techcrunch.com/2006/10/09/google-youtube-sign-more-separate-deals/) 签署了许可协议。它还起诉索尼的 Grouper 和 Bolt 侵犯版权。

新一轮重量级版权大战是否正在爆发？这两家公司有可能庭外和解吗？本周 Craigslist 法院的判决立刻浮现在脑海中。周三，在线目录被裁定不对其网站上的歧视性住房帖子负责。法院发现 Craigslist 是一个渠道，而不是出版商。这一裁决很复杂，属于《联邦通信法》的范畴。环球公司对 Grouper、Bolt 以及现在的 MySpace 的诉讼似乎是对 DMCA 安全港条款的直接考验，该条款被认为是保护版权侵权方，只要他们应要求删除受版权保护的内容。关于视频转码的争论可能是本案的杀手锏。
