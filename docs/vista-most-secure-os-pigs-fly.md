# Vista:最安全的操作系统，猪会飞

![](img/14f21359cc9ae6d718aa4875e7230e67.png)

基于 eWeek 的一篇令人难以置信的长文章，微软自己的安全策略主管 Jeff Jones 得出结论，Vista 是最不容易受到攻击的操作系统。琼斯说，与 Red Hat Enterprise 中的 129 个、Ubuntu 中的 29 个和 OS X 中的 16 个相比，Vista 只有 12 个公开已知的漏洞。分析师认为，这些结果是基于 Vista 的有限部署以及其他操作系统已经有 10 年寿命而 Vista 刚刚发布的事实。微软似乎很自豪，它这次确实做了正确的事情。然而，在 Vista 上市几年之前，没有人会知道这些威胁的真实性质。

当然，琼斯在微软的职位对他的成绩没有任何影响…
