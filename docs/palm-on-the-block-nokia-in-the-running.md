# 手掌放在木块上，诺基亚在奔跑

像已故的伟大的康柏公司一样，Palm 看起来将在未来几天内被一家更大的公司吞并。关于诺基亚可能收购的传闻比比皆是，尽管摩托罗拉也在竞争之列。

这对我们意味着什么？可能是基于 Symbian 的 Treo-esque 设备，或者，老天保佑，来自芬兰巨人的 Windows Mobile 或 Palm 设备。事实上，唯一真正的先例是惠普收购康柏，推出 iPaq 掌上电脑，然后是手机。不幸的是，莫托和诺基亚的文化和美学都非常封闭，我怀疑他们会像蜜蜂之狼一样，采用这种技术，而把品牌的无流体外壳留在花的底部。再见，特里奥？
