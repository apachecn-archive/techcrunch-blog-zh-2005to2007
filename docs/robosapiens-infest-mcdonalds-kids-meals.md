# 机器人寄生在麦当劳的儿童餐中

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/23/robosapiens-infest-mcdonalds-kids-meals/>

现在，我们看到它们大量出现在孩子们的食物中。

从今天开始，如果你的孩子在麦当劳吃儿童餐，他将得到七个机器人玩具中的一个。每个玩具都有不同的功能，从跛脚的屁股发条动作，到跑步，说话和咀嚼。

全家人的乐趣。收集整套，然后弯腰被你的新主人铰。
