总结:

维基百科已经存在5年多了，是最大的维基百科(到目前为止)，每月有超过4亿的页面浏览量(相比之下，《今日美国》大约有3亿)。

维基百科是一个免费的、在线的、用户编写的维基格式的百科全书，由拉里·桑格和吉米·多纳尔·威尔士于2001年创建。它以惊人的速度增长，现在包括160万篇文章。其中超过60万是英语，超过25万是德语，超过10万是日语和法语(维基百科有205种语言的版本，其中92种是活跃的)。

很多人没有意识到任何人在维基百科上添加内容是多么容易(我已经做过几次了)。每个月，都有成千上万的人向这个项目添加内容。

我们今天写它是因为它成为了最全面和最可靠的知识中心，因为关于7月7日伦敦爆炸案的信息变得可用。我们在这里发布了关于攻击的，并在上周 Web 2.0综述中提到，当BBC和其他新闻网站跟不上时，维基百科却一路顺风，让世界了解最新消息。克莱·舍基谈到了这一点，吉姆·菲尔普斯在这里又谈到了这一点:

每当发生真正重大的灾难却没有即时新闻时，有线电视上的人经常在瞎编，因为他们必须不断重复基本的故事，甚至在没有任何新信息的情况下，也可能有人刚刚收听了。维基百科解决了这个问题，同时与这些新闻媒体建立了共生关系，因为它将人们引向故事的书面形式

参见大卫·温伯格关于这个的帖子。

对我们中的许多人来说，我们在2005年7月7日开始充分理解维基百科作为文化/知识聚合者的巨大力量。

维基是什么？

Wiki是一种网络应用程序，允许用户添加内容，就像在互联网论坛上一样，但也允许任何人编辑内容。第一个wiki是沃德·坎宁安在1995年3月25日创建的波特兰模式知识库。

wiki技术的一个显著特征是页面创建和更新的便捷性。一般来说，在修改被接受之前没有审查。维基人通常实践使错误易于改正的哲学，而不是使犯错误变得困难。

这个名字是基于夏威夷术语wiki wiki，意思是“快速”或“非正式”。它通常在夏威夷使用，是夏威夷洋泾浜语的一部分。

百科全书是知识的书面概要。一个很好的总结可以在维基百科的这里找到:

在一个屋檐下触手可及的地方收集世界上所有知识的想法可以追溯到古代的亚历山大和佩加蒙图书馆。古代的许多作家(如亚里士多德)试图全面地写出人类所有的知识

"传统的百科全书是由许多受雇的文字作者编写的，通常是有学位的人。"

今天，大多数百科全书都提供在线内容，并对内容收取高达每年100美元的订阅费。

维基百科有什么不同(和更好)？

维基百科结合了维基和百科全书的概念，维基是用户创建的内容的集合。进一步总结将是愚蠢的——我们无法与维基百科成千上万的编辑竞争。查看他们在维基百科的条目这里。

其核心是，维基百科提供了大量的主题内容(如 web 2.0 ，并提供了额外的功能，如每个条目的论坛，添加内容的编辑工具，以及显示文本历史修订的页面。

Samuel Klein写了一篇漂亮的文章，总结了我们对维基百科现在、过去和将来的想法。

维基百科肯定有问题，但这些问题是维基百科的结构所固有的(当每个人都可以写时，有些人写的是废话)。但它具有极强的弹性和自我修复能力，是互联网文化极其重要的资产。
