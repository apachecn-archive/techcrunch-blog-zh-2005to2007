# 耐克地板曲棍球清洁概念

![](img/af027eee1978c292794aa858ae64b64b.png)

耐克似乎得到了今天的年轻人。除了制造酷踢，该公司花了一些时间来搅拌这个概念清洁系统。仿照地板曲棍球设置，有一个真空，兼作球门。你必须用类似曲棍球棒的扫帚把灰尘和垃圾扫进去。你得了几分，妈妈开心了，客厅也不再乱了。听起来像是一个成功的设计，耐克。但愿这不是一个愚蠢的概念。
