# 好吧，也许麦凯恩应该回到旧的模式

 [](https://web.archive.org/web/20220926211948/http://mike.newsvine.com/_news/2007/03/28/635861-proof-that-john-mccain-needs-new-interns) 今天早上我们提到总统候选人[约翰·麦凯恩的 MySpace 页面被“增强”](https://web.archive.org/web/20220926211948/http://www.beta.techcrunch.com/2007/03/27/john-mccains-myspace-page-hacked/)了一条新的支持同性恋婚姻的政治信息。该网站很快被撤下，大约 13 个小时后修复。不过，这个新网站有一个[错误拼写](https://web.archive.org/web/20220926211948/http://www.myspace.com/johnmccain)—“Gorup”应该是“Group”

这第一次可能会被当作一个简单的错误而一笑置之，尽管它凸显了麦凯恩在同性恋婚姻问题上摇摆不定的立场。但现在有些人会质疑这个家伙能否雇到有能力的人来围绕他。原故事将在今天早上登上主流媒体。很多人会去麦凯恩的 MySpace 页面。他们会看到一个邋遢的网站。关于 Newsvine 的更多信息。
