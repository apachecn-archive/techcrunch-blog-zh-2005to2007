> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/22/windows-media-player-11-drm-out-the-arse/>

我们喜欢对数字版权管理的下意识反应——这让我们觉得我们在经营一个政治博客——但不幸的是《问询者报》几乎完全正确。简而言之，WMP11 可能是我们迄今为止遇到的最严厉的 DRM 之一。

让我们从无法备份音乐许可证开始。许可证绑定在一台机器上，所以如果你把购买的文件拖到奶奶的笔记本电脑上，而你的电脑死机了，你就只能处理无用的文件了。也许你可以用它们制作分形？或者给随机数发生器供电？显然，他们提供的工具可以让你移动你的垃圾，但是如果你的电脑被撞了，下载“微软密钥管理器 2007 测试版”是你最不需要的。当你尝试使用你购买的音乐时，你需要微软检查你吗？Best Buy 的店员会过来看看你几年前买的 Fiddy CD 是怎么回事吗？以确保你没有滥用它？

然后，当你试图复制的时候，WMP11 会感染被翻录的 CD 和手机。多好啊！你必须明白，这保护了唱片业的利益。显然，以数字形式跟踪一个垂死的媒体是至关重要的，因为有人，上帝保佑，可能正在分享贾斯汀·汀布莱克的最新作品*“我像新生婴儿*一样没有头发”。

最后，这个程序帮了你一个大忙，三天后删除了 TiVoToGo 的内容。这确保你真的看了你拖过来的那 50 集流言终结者，而不只是带着它们到处走。再说一次，DRM 文章多如牛毛，他们写的文章通常有点过火。但是询问者知道发生了什么，这是个坏消息，伙计们。我会让他们关闭它。

> WiMP11 所代表的是我所能想到的对你权利的最大盗窃之一。MS 策划了这个，慢慢推各个棋子，这是第一个掉下来的大锤。你的权利，他们所做的承诺，以及其他任何阻碍内容黑手党赚钱的东西都会被抛弃。为什么？贪婪。你的权利？历史。你傻到让它发生，不要说我没有警告你。
