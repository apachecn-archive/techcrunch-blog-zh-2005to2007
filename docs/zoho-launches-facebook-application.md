# Zoho 推出脸书应用

[![](img/ace02644f2d00f9cce03d695792cb903.png)](https://web.archive.org/web/20220626075005/http://www.zoho.com/) 在线办公套件提供商 Zoho [已经在](https://web.archive.org/web/20220626075005/http://blogs.zoho.com/general/zoho-apps-on-facebook/)推出了一款 Zoho 脸书应用。

Zoho for 脸书允许用户在脸书创建文档、电子表格和演示文稿。支持包括从 Zoho Writer、Sheet 和 Show 查看和编辑所有现有文档、电子表格和演示文稿(个人和共享的)。

虽然脸书上的 Zoho 将 Zoho 展示给更广泛的观众，新的应用程序有一个大问题:与脸书用户/登录系统的集成完全失败；脸书的 Zoho 仍然需要一个 Zoho 账户。这当然引出了一个问题:如果不支持脸书登录，为什么用户会从直接使用 Zoho 转向在脸书上使用 Zoho？或许 Zoho 更懂，申请会受到好评；为消费者提供选择是积极的，即使提供这种选择的逻辑并不清晰。
