# 使用 Suunto X9i 追踪您的冒险

![](img/8230201a7ecb13aa935f3febd3a291e5.png)

Suunto X9i 是一款功能强大的腕表，适合喜欢旅行或参与极限运动的人。X9i 具有 GPS 系统、气压计、指南针、防水功能、高度计和基于菜单的界面，让您轻松浏览各种功能。X9i 还通过 USB 连接到 PC，并配有 Suunto 的跟踪软件，因此您可以绘制出您的路线，并通过电子邮件发送给朋友。

你不仅可以从高空视角查看你的路线，还可以在谷歌地球上查看。当你下次去露营，或者在冬天去滑雪时，这款手表可以证明是一个强大的设备。指南针和全球定位系统总是好东西。如果你做一些研究，你可以在网上找到价格略低于 400 美元的手表，这一点也不算太差。
