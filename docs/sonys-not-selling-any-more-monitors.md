# 索尼不再销售任何显示器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/09/sonys-not-selling-any-more-monitors/>

![](img/7bf1d0e89df29933c616f380c2656d14.png)

让“索尼正在衰落”的传言重新开始。它向 [heise online](https://web.archive.org/web/20160226044237/http://www.heise.de/english) 宣布，他们将不再是营销监督员。这一举动甚至让索尼的欧洲办公室感到惊讶，他们宣布他们将继续生产的唯一显示器是他们的“专业”显示器，如用于广播、医疗技术和视频监控系统的显示器。平板电视业务似乎没有足够的钱可赚，索尼去年在该市场排名第十。这就是生活。
