裸奔搜索让社交网络变得无所不包

Vivek，over at Startup Squad，最近发现了一个新的社交网络和社交网络元搜索引擎裸奔。主URL仍然显示网站即将推出。新的引擎可以让你在主流网络(MySpace、Hi5、Bebo和脸书)以及它自己的社交网络上搜索个人资料。这似乎是一个吸引人们进入他们主要服务的诱饵，就像Wink推出自己的个人资料搜索和Rapleaf推出UpScoop一样。档案管理工具profile link也有一个搜索引擎。

裸奔者的社交网络对酷孩子来说就像美味，对Trig来说不那么花哨。它包括一个个人资料，一个工具栏，和一个偶然发现的功能，可以让你浏览给定类别中的链接。这里是视频的一个。

这些简介看起来很像MySpace，由常见的细节、关于我、照片和诱发癫痫的布局组成。森雅是裸奔者的汤姆。然而，在MySpace放博客和评论的地方，裸奔会放上你最喜欢的链接和你对每个链接的“赞”评级。您可以手动将链接输入到您的个人资料中，或者使用裸奔工具栏将链接添加到您的个人资料中并对其进行投票。工具栏还提供了一个到主站点上所有其他功能的接口，目前仅用于IE，需要。NET framework，下载和安装需要一天的时间。

还有几个其他网站提供社交网络元搜索。以下是一些内幕:

眨眼
眨眼快速简单。它搜索Friendster、MySpace、Bebo、LinkedIn和Live Spaces。它还具有高级搜索功能，如位置、性别、身份、年龄和兴趣。它还允许您在第一次搜索后通过这些字段缩小搜索范围。

profile link
profile link是最全面的搜索引擎，拥有84个社交网站，包括综合、博客、文化、交友、专业、学生和特殊兴趣网络。不幸的是，你必须登录才能使用它。

up scoop
up scoop领先于易用性。不像其他的，UpScoop根据你地址簿中的所有联系人通过电子邮件进行搜索。它搜索Bebo、同学会、Ecademy、Flickr、Friendster、Hi5、Livejournal、Multiply、MySpace、Ringo、Tickle、Tribe、Yelp、Mog和LinkedIn。虽然它可以一下子找到你的大多数朋友，但它的一些缺点是，搜索最后几个朋友可能需要几个小时，并且需要交出你的电子邮件凭据。
