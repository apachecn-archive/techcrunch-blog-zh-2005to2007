# 雅虎！messenger 3 OS X 测试版现已推出

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/09/15/yahoo-messenger-3-beta-for-os-x-out-now/>

对于美国 Mac 用户来说，雅虎！信使是福也是祸。多年来，雅虎！在向 Windows 用户推出之前，在 Mac 版本中添加了一些功能。然后，在向 OS X 过渡的某个时候，雅虎！换挡。

Mac 用户已经有几年没有更新了。微软和美国在线保持他们的消息系统是最新的，而 Mac 用户还停留在 OS 9.2.2 时代的计算时代。伙计们，为雅虎欢呼吧！凭借他们无穷的智慧，发布了 Mac 3 版 Messenger 的测试版。我已经用了一整天了，我可以说它真的很棒。

它比以前更好地整理了信息，并具有我们喜欢的现代照片与您的对话相邻的功能。此外，文件传输工作，网络摄像头的工作，它的码头意识，有更好的整合与手机信使，与 Windows Live！信使，并有多聊天会议，就像 Windows 版本。

现在可以下载了，我推荐它，即使它不是严格意义上的小工具。
