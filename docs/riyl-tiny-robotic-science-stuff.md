# RIYL:微型机器人科学的东西

你现在看到的是世界上最小的机器人手。够小，够细腻，可以抓住一个鱼卵而不会把它挤成小块。它是由加州大学洛杉矶分校机械和航空航天工程系的延·鲁文和常·金真开发的。

这只手是由聚合物气球接头连接的硅块制成的。气球的充气和放气控制着四指手。所用的材料和气动技术使其适用于液体或空气中的生物样品。该死，太酷了。
