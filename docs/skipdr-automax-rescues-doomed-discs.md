# SkipDR AutoMax 拯救注定失败的光盘

像这样的设备已经出现了一段时间，但这是全自动的。SkipDR AutoMax 的尺寸大约相当于一个电脑扬声器的大小，它可以自动将有划痕的光盘进行抛光处理。

只需用表面修复液喷洒光盘，并将其装入插槽中。然后，该设备旋转一个电动径向轮一分钟，从游戏侧去除一层微小的塑料，这应该可以去除表面划痕。每个轮子适用于大约 50 次光盘修复，可以以 12.99 美元的价格更换。

它可以运行在六节 AA 电池或一个交流适配器，在 sheisty 注意到，这是不包括在内。SkipDR AutoMax 现已上市，售价 39.99 美元。
