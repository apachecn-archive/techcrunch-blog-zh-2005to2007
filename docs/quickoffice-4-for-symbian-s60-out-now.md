# 面向 Symbian S60 的 Quickoffice 4 现已上市

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/01/22/quickoffice-4-for-symbian-s60-out-now/>

虽然小屏幕可能不适合写伟大的美国小说，但它非常适合在最后一刻修改或审阅文件。Quickoffice 4 是流行移动软件的最新版本，可以做到这一点，现在可用于 Symbian S60 手机，为用户提供动态读写功能。

对于手机软件来说，50 美元的价格似乎有点高，但如果你是一个真正的出差族，那么能够在旅途中编辑 PowerPoint 幻灯片的 moola 可能是值得的。

不过，如果你只需要阅读文档，Quickoffice 提供了一个免费的文件浏览器，这是一种不用预先付费就能体验移动文档的方式。这两款软件现在都可以在运行 S60 第三版的各种不同智能手机上下载。
