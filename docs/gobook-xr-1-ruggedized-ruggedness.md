# GoBook XR-1:加固的耐用性

Itronix 为重型任务制造军用级硬件，GoBook XR-1 也不例外。这个怪物有多达三个无线系统，包括 WWAN、内置 GPS，并且“符合 MIL STD 810F(军用标准)的跌落/冲击和振动等级以及 IP54(入口保护)”，我怀疑这意味着它很难被攻破，除非军方对他们的笔记本电脑做了一些我不知道的事情。

这款 6.8 磅重的庞然大物的基本配置售价为 4330 美元，将于 9 月上市。
