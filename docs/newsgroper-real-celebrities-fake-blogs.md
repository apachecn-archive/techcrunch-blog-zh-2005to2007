# 真正的名人，虚假的博客

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/07/16/newsgroper-real-celebrities-fake-blogs/>

 [](https://web.archive.org/web/20150924201641/http://www.newsgroper.com/) [NewsGroper](https://web.archive.org/web/20150924201641/http://www.newsgroper.com/) 今天发布了 50 多个恶搞博客，每个博客都是匿名作者写的，就好像他们是真正的名人一样。巴拉克·奥巴马、布兰妮·斯皮尔斯甚至达赖喇嘛都是一些不好笑的评论的目标。

这个新网络是由格雷戈·加兰特、亚当·沃尔高、贾里德·纽马克和科什兰·迈尔-布莱克威尔创建的。很明显，它的目的是利用山寨史蒂夫·乔布斯和其他模仿真人的网站的成功。他们雇佣了 50 名匿名作者来写博客。

如果他们写出聪明机智的东西，这显然会受欢迎。如果很无聊，即使背后有再巧妙的想法也不会阻止它陷入默默无闻。不过，到目前为止还不错——最初的一些内容相当有趣。其中一些…不是。
