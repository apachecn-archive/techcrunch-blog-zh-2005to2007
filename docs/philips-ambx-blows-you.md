# 飞利浦 amBX 吹你

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/09/philips-ambx-blows-you/>

![](img/64641e07670f355393ce2c84594582df.png)

奇怪的是，这是我第一次有机会看到飞利浦 amBX 系统，这是一个为 PC 游戏创造周围环境的附加设备。它由三个灯组成，两个在侧面，一个在笔记本电脑/PC 的背面。然后键盘上有两个风扇和一个隆隆声条。当你在太空中飞行时，结果是清新的新鲜空气，或者根据屏幕上的动作发出怪异的光。我们看到的设置有点过时，只是因为环境不太合适，但如果你想在游戏中被炸，这绝对是值得考虑的事情。
