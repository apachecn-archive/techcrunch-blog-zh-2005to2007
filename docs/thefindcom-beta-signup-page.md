# TheFind.com 测试版注册页面

 [](https://web.archive.org/web/20220808124819/http://www.thefind.com/) 位于硅谷的[find](https://web.archive.org/web/20220808124819/http://www.thefind.com/)有一个测试注册页面，将在未来 45 天内推出。这项服务将是一个技术含量很高的比较购物引擎。

去年，Find 一直在围绕二级门票销售运行一个名为 [Fatlens](https://web.archive.org/web/20220808124819/http://www.fatlens.com/) 的购物搜索引擎。这项新服务将利用与 Fatlens 相同的技术，但扩展到更多的购物类别。“我们将拥有一切，除了旅行”联合创始人西瓦·库马尔今天在电话里告诉我。

TheFind 成立于 2003 年，已经从红点风险投资、光速风险投资伙伴&寒武纪风险投资获得了 800 万美元的风险投资。
