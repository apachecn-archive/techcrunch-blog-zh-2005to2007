# WiFi 解放者将大众从付费 WiFi 中解放出来

我们在克朗彻奇不容忍盗版或偷窃。这并不意味着我们不会向你展示方便这种背叛的酷装备，比如 WiFi 解放者，一个罗宾汉式的天才作品。

一个美化的 USB WiFI 加密狗，解放者作为一个继电器，使一个私人网络公开。举例来说，你在当地的星巴克，你有一个 T-Mobile 热点数据计划([，你可以免费破解](https://web.archive.org/web/20201028231302/http://crunchgear.com/2007/01/30/free-steamy-t-mobile-hotspot-access/))，你可以使用解放者分享你的互联网连接到该地区的其他用户。你登录了，在你的合法账户上冲浪，你的邻居也是。一种非常实用的利他方式。

此类设备的合法性仍悬而未决，但话又说回来，WiFi 也是如此，其理念是，从那里它就应该被共享。
