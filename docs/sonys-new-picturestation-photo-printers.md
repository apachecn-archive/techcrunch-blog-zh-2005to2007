# 索尼的新型 PictureStation 照片打印机

![](img/f62e2a2a6ff90061eb4a2cbb74d0fb52.png)

索尼今天推出了一批新的照片打印机——PictureStation DPP-FP90(如上图)和 PictureStation DPP-FP70(跳转后的图)。两者都能在 45 秒内打印出“实验室质量的 4×6 英寸照片”(还记得你在 CVS 等了一个小时吗？)，而且该公司似乎特别自豪的是内置的编辑功能，可以让你在液晶显示器上调整曝光、对焦和红眼(FP90 为 3.6 英寸，FP70 为 2.5 英寸)。)

额外收获:如果你有一台索尼相机，他们声称可以通过读取嵌入数据来“优化”照片。我们会看到的。

FP90 的价格大约为 200 美元，而 FP70 的价格大约为 150 美元。两者都兼容 Vista。点击跳转查看《全盛时期》中更便宜的……
