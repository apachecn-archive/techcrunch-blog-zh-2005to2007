# DirecTV 将通过电力线提供互联网

![](img/1e492e7c37215e7dbc836f71e82e69fd.png)

这里没有太多的信息，但华尔街日报报道，DirecTV 将最终开始通过电力线分配提供宽带互联网接入。通过与 Current Group 的合作，这项服务成为可能，Current Group 通过电力线提供宽带通信。

你会花多少钱购买直播电视的宽频网络？有了 20 美元的卫星电视，这可以构成一个邪恶的一揽子交易的一部分。
