# 现在的智能手机:T-Mobile Dash

![](img/fa90e6b771b4211f3781db45b38bfd70.png) * [智能手机现在](https://web.archive.org/web/20130627212210/http://www.crunchgear.com/smartphones-now)是由 CrunchGear 作者在美国已经上市或即将上市的最新智能手机上开发的一组特殊功能。在这一期中，我们回顾一下 T-Mobile Dash。*
