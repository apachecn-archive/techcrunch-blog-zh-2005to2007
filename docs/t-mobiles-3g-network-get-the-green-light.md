# T-Mobile 的 3G 网络获得批准

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/06/t-mobiles-3g-network-get-the-green-light/>

T-Mobile 今天宣布了它的漂亮的新 3G 网络计划，这是我们期望的的[。该网络将基于 UMTS，其计划在该项目上花费超过 20 亿美元，该项目将于 2009 年完成。不幸的是，它必须工作在与欧洲 UMTS 标准不兼容的频率上。](https://web.archive.org/web/20151103012149/http://crunchgear.com/2006/10/04/t-mobile-to-launch-true-3g-data-network-friday/)

这并不理想，因为现有的手机将无法利用新的带宽。相反，T-Mobile 希望在明年年中推出自己的设备系列。这可能包括电话、PDA、智能手机和笔记本电脑卡。由于 T-Mobile 最近[一直是 HTC](https://web.archive.org/web/20151103012149/http://crunchgear.com/2006/09/27/t-mobile-dash-coming-october-16th/)的粉丝，很有可能 [HTC TyTN](https://web.archive.org/web/20151103012149/http://crunchgear.com/2006/08/31/htc-tytn-xda-trion-3g-windows-mobile-pda-phone-available-now/) 会出现，而 Catherine Zeta Jones 不会出现。

但这并不意味着它懒洋洋地躺在集体 T-Mobehinds 上。事实上，它立即开始推出，因为它已经开始了。这意味着如果你在一个中等的大都市地区，你可以在不到一年的时间里开始使用无线宽带。
