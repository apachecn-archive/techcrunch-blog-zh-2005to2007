# 黑莓 8300 代托纳泄露

最新的黑莓设备已经泄露，这是 RIM 粉丝一直在等待的。Daytona/8300 拥有一系列我们期望从 8800 中获得的功能，这些功能并没有出现，但我们确实听到了今年晚些时候将发布其他新 RIM 设备的传言。Daytona 配备了 Wi-Fi，一个带 LED 闪光灯和 GPS 的 200 万像素摄像头。其他值得注意的事实是扬声器和软件的改进，但细节有限。
