# Zooomr 推出可缩放的画中画功能

[![](img/d9e099b443f688cfa1a45f2200e96c46.png)](https://web.archive.org/web/20220119221844/http://www.zooomr.com/) 图片分享新贵 [Zooomr](https://web.archive.org/web/20220119221844/http://zooomr.com/) 今天推出了许多新功能，其中最引人注目的一个叫做 Portals。这是一种将可缩放图片相互链接的方式。挺惊人的，从下面的视频可以看出来。

Zooomr 今天还发布了注释功能，包括在图片上留下注释的能力，这些注释由朋友状态和人物注释进行颜色编码，或者显示被悬停的人物的 Zooomr 用户名的注释。

门户的工作方式是，就像文本注释可以添加到照片中一样，门户允许图像发布者将照片与当图像悬停时出现的框联系在一起。查看者可以在嵌入的图片内平移或单击查看图片本身。

出版商可以利用门户网站赋予他们的图片深度和细节。单个图像中可以放置的门户数量没有限制，一个门户图像可以指向另一个。因此，从远处拍摄可以有多个特写镜头；对于建筑、自然和集体照片来说，这可能是很棒的东西。
