# 索尼蓝牙耳机:惊人的吸引力

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/27/sony-bluetooth-headphones-surpisingly-attractive/>

我见过很多蓝牙耳机，包括 Koss 和 Plantronics 的产品，但这是一副我可能会把头埋进去的耳机…呃…你知道我的意思吗？缠绕在我的头上？

索尼 DR-BT21G 使用所有最新的蓝牙协议，并与两部手机和您喜欢使用的任何蓝牙加密狗配对。没有价格，它看起来像亚洲-只有现在。它有 20 到 20，000 赫兹的响应范围，看起来听起来很棒。索尼在这方面做得很好，可能需要重新考虑它更强的东西，停止[在电子书阅读器和 Mylos](https://web.archive.org/web/20160322120606/http://crunchgear.com/2007/02/25/the-best-anti-ps3-song-of-the-weekend/) 上的徒劳。

![](img/b6a446ab2d1a0a15f57d06f445f6f2d2.png)

他们还销售 DR-BT21G，这是一种较小的环绕式手机。两者都没有价格，也没有供货。希望他们能在这里成功，因为我已经厌倦了那些看起来像是被《星际迷航》拒绝的人，包括我自己。
