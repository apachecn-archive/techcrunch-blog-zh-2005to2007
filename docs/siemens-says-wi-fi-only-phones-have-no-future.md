# 西门子称只有 Wi-Fi 的手机没有未来

[![wifiphones.jpg](img/c68af26ee304149ba9706e5cff07e00e.png)](https://web.archive.org/web/20210123140712/http://old.crunchgear.com/wp-content/uploads/wifiphones.jpg "wifiphones.jpg")

西门子不再是皇家马德里足球俱乐部的球衣赞助商，它昨天承认了自己的过失，并对只支持 Wi-Fi 的手机已经彻底失败的事实表示了歉意。该公司一直是这种手机的大力支持者，这种手机旨在连接 Skype 和其他 VoIP 供应商，但现在准备弃船而逃。前两句话的关键词是“只支持 Wi-Fi”，因为西门子仍然看到了 Wi-Fi 兼容手机的市场，只是不是那些只支持 Wi-Fi 的手机。这项技术已经发展到了公司现在可以把 Wi-Fi 集成到手机上的程度，这几乎是事后的想法。

手机的圣杯？它结合了蜂窝电话、POTS 无绳电话和 Wi-Fi 服务，这与我们在一月份的 CES 上听到的非常相似。
