# 东芝推出 16GB 闪存驱动器

![](img/b457c6bc1cd72360c6bcd30df072a3cf.png)这些 16GB 的闪存驱动器不是用于固态笔记本电脑的。这些是你放在口袋里的小拇指棒，里面有你所有的期中考试和书面报告。东芝将发布这些 TransMemory U3 系列闪存驱动器的 512MB、1GB、2GB、4GB 和 16GB 尺寸。

你可以通过东芝的在线商店在 12 月拿起你的 16GB 怪物。其他尺寸可在 11 月下旬购买。它们还可以支持直接从东芝公司获得的基于密码的保护软件，这样你就不会把你所有的宝贵数据都丢失给网吧里的一些手脚不干净的小混混。有了 16GB 的空间，将会有大量的数据。
