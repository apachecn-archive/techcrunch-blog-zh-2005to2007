# 为您的所有小工具提供 USB cell 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/13/usbcell-for-all-your-gadgets/>

广受欢迎的 [USBCell 电池](https://web.archive.org/web/20150912161012/http://crunchgear.com/2006/09/20/usbcell-aa-batteries-are-usb-rechargeable/)的制造商已经采取了下一步措施，渗透到移动设备市场。我非常喜欢随身携带额外的电池，因为像大多数极客一样，我的手机(Sidekick3)有助于我在旅途中保持联系。我可以处理我的 SK3 crapping out，因为我通常有我的黑莓手机，但甚至不要让我开始使用数码相机。因为电池没电了，我已经记不清错过了多少精彩的照片。

所以你可以明白为什么我会对用于移动设备的新型 USBCell 电池感到非常兴奋。只要你在 USB 端口旁边，你的手机、相机和手持游戏设备就会一直为你提供动力。
