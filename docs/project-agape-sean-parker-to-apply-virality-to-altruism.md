# Agape 项目:肖恩·帕克将病毒传播应用于利他主义

 [](https://web.archive.org/web/20230329034607/http://www.project-agape.com/) 昨天，我和肖恩·帕克在旧金山[创始人基金](https://web.archive.org/web/20230329034607/http://www.thefoundersfund.com/team_bios.html)的办公室坐在一起，观看他新的尚未命名的创业公司的演示(这个项目的工作名称是 [Project Agape](https://web.archive.org/web/20230329034607/http://www.project-agape.com/) )。

27 岁的帕克是一位传奇人物，他是 Napster 和 Plaxo 的联合创始人，也是脸书的首任总统。在过去的八个月里，他一直全职从事 Agape 项目，同时还在 Founders Fund 担任管理合伙人。

帕克知道如何将病毒原理应用到想法中。在我们 1.5 小时的会议中，有一半时间是在讨论这些原则，以及如何将想法微调到可以成倍增长的程度。他说，唯一能阻止一个好的病毒式创意的是当它耗尽人口时。如果 Napster、Plaxo 和脸书是例子的话，他可能是对的。

尽管我看到了一个演示和一些额外的概念性工作，但 Project Agape 仍然处于重重的保密之下。他说，帕克的目标是将在他之前的项目中非常有效的病毒式传播理念应用到利他主义和行动主义的理念中。

慈善机构、政党和类似团体都依赖于拥有相同信仰和理想的人的参与。但是招募和筹款在很大程度上停留在前互联网时代:社会压力和负罪感被用来让其他人为白血病协会的马拉松捐款，或者贡献时间与无家可归者一起工作。帕克想利用这些行之有效的激励结构，用他的新公司来提高效率。

像[Change.org](https://web.archive.org/web/20230329034607/https://techcrunch.com/tag/change.org)和 [dotherightthing](https://web.archive.org/web/20230329034607/https://techcrunch.com/2007/01/27/do-the-smug-thing/) 和 [Six Degrees](https://web.archive.org/web/20230329034607/https://techcrunch.com/2007/01/18/six-degrees-will-help-people-and-we-can-make-fun-of-it/) 这样的新网站帮助人们在网上讨论问题，但是它们在利用病毒式传播获得新用户并让他们实际做事方面做得还不够。帕克想要脸书看到的围绕这些组织的那种活动——成千上万的每日新用户和数小时的社交互动。他说，结果将是一个更有效的引擎，让组织招募志愿者和筹集资金。

该公司总部位于伯克利，将在未来几周内发布一些公告，测试产品将在几个月后推出。敬请关注更多内容。
