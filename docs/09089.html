MobileCrunch独家配件:诺基亚N800

原文：https://web.archive.org/web/http://www.techcrunch.com/2007/01/18/mobilecrunch-exclusive-nokia-n800/trackback/

在过去的一个月里，在NDA的领导下，Oliver Starr对新的诺基亚N800平板手机进行了全面的测试。奥利弗将800与770进行了对比，称其外形更好，立体声扬声器更大。N800具有800 x 480彩色显示屏，两个存储卡插槽(Micro SD、MMC、SD和Mini SD ),每个插槽可容纳高达2 GB的卡(用户体验为4GB)，比770更快的处理器，电池续航时间长，无需充电即可使用一天。

查看Oliver的帖子，了解他的详细评论。
