# 诺基亚 N80 拥抱互联网

![](img/ab05eb9d21521a5e75923be037e05252.png)

当我走在这条壮丽的大道上，寻找打发时间的好方法时(这似乎越来越频繁)，我通常会在[诺基亚旗舰店](https://web.archive.org/web/20130627214840/http://www.nokiausa.com/about/store/chicago/1,8737,,00.html)停下来，看看他们有什么样的对我来说太贵的新手机可以玩。通常只有大约 30 辆一般的 [8801](https://web.archive.org/web/20130627214840/http://www.nokiausa.com/phones/8801/0,7747,,00.html) ，以及几辆为女士准备的 [7370](https://web.archive.org/web/20130627214840/http://www.nokiausa.com/phones/7370/0,7747,,00.html) 和 [7380](https://web.archive.org/web/20130627214840/http://www.nokiausa.com/phones/7380/0,7747,,00.html) 。

然而最近，他们开始展示 N80 智能手机。我用它玩了一小会儿，就足以让我相信，它已经可以与所有其他智能手机竞争了。但看起来诺基亚的人已经决定将其升级到 2.1 英寸屏幕，352×416 像素分辨率的屏幕，GSM/EDGE 850/900/1800/1900 MHz 网络运行手机，以利用 Wi-Fi，支持第三方 VoIP 应用程序，亚马逊购物，Yahoo Go！，Mobipocket 电子书阅读器和更多互联网就绪的好东西。

新版本将于 9 月推出。
