# Omnidrive 的在线存储实际上是有效的

自从我在 11 月份发布了关于对一个好的在线存储服务的需求的帖子后，我一直在和总部位于澳大利亚悉尼的 Omnidrive(T4)的创始人尼克·库布里洛维奇(T2)交谈。

在过去的几天里，我有机会测试它。这是预测试版，但将很快推出。他们已经解决了许多与远离网络的存储相关的问题，并且拥有在线和桌面界面。

Omnidrive 将有一个 1g 左右的免费版本，之后会有付费计划。该功能集非常棒——它拥有你所要求的一切，包括在后台处理大量文件上传。完整评论即将推出——在网站上注册 beta 版公告[。](https://web.archive.org/web/20220119234742/http://omnidrive.com.au/)
