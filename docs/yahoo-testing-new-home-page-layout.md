# 雅虎测试新主页布局

 [](https://web.archive.org/web/20220817211626/http://www.yahoo.com/) [雅虎](https://web.archive.org/web/20220817211626/http://www.yahoo.com/)正在和它的一些用户测试一个新的主页设计，而[史蒂文·科恩](https://web.archive.org/web/20220817211626/http://www.librarystuff.net/)抓取了一些屏幕截图。

有一些不同，但据我所知，这是一个纯粹的化妆改头换面。新网站更加简洁，将关键资源放在左边，比如电子邮件、我的雅虎和其他雅虎服务和内容。
