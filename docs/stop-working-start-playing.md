被你的工作烦透了？隔间生活让你沮丧？只是今天不想工作？查看有线电视台提供的免费在线游戏列表。

有各种各样的选择，包括围绕“南方公园”人物的简单益智游戏，“每日秀”琐事战斗和一种名为*欺骗我的卡车*的东西，让你“在时钟用完之前通过运送你的有效载荷赚取资金来欺骗你的卡车”,从 CMT。事实上，CMT 收集了大量基于 Flash 的小游戏，可以在你等待工作结束的时候消磨时间。
