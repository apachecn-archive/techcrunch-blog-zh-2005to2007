# 索尼爱立信 W970:阿德里亚娜爱它一些视频

[![seadriana.jpg](img/40b94fdd0cc816946ee7ccfa6c3b5c1c.png)](https://web.archive.org/web/20210228090636/https://beta.techcrunch.com/wp-content/uploads/2007/04/seadriana.jpg "seadriana.jpg")

索尼爱立信是一艘[漏船](https://web.archive.org/web/20210228090636/http://crunchgear.com/2007/04/19/sony-ericsson-w999i-revealed/)，一个西班牙论坛不知何故获得了即将到来的 W970 的图片和信息，绰号“阿德里亚娜”据说这是一款真正专注于提供可靠多媒体体验的手机。为此，Adriana 将专注于视频播放，但微不足道的 512MB 内置内存似乎否定了这一想法。

正如在这张高度可靠的图片中看到的，手机使用了一些奇怪的旋转机制来隐藏看起来像键盘的东西。没有日期，见鬼，没有真正证实它的存在，但今天是星期五，每个人都喜欢未经证实的新手机谣言。
