# Steelcase 最差工作区挑战有一个 Winnah

![](img/e2e333fc285d1ff2ae1f08e5582d916f.png)
咻。自从我们发起 Steelcase 最差工作场所挑战以来，已经过去了大约两周。今天我们终于可以宣布获胜者了。

在我们的旅行中，我们看到了一些真正灾难性的工作场所。我觉得很有理由。这些年来，我的父母一直说我的桌子是世界上最乱的，而我坚持说我没有。你证明了我是对的，为此我感谢你。

总之，我猜你想知道谁赢了。

在我们的五名决赛选手中，V.M .脱颖而出，成为明显的赢家，获得了近一半的选票。所以恭喜 V.M .你是一把[钢壳](https://web.archive.org/web/20201130121421/http://www.steelcase.com/)椅子的骄傲的新主人。

我们的其他决赛选手不会得不到补偿！他们的努力将得到 iFrogz 礼品卡的奖励。iFrogz 的员工制作了一些最好的 iPod 保护套。
