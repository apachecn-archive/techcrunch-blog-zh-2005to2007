# Ogo 继续存在！

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/26/the-ogo-lives-on/>

![](img/bfcec1b3ae99cf5d45288e7e8541692d.png)

如果你问我 7-11 和失败的通讯设备有什么共同点，我会说是过去的 Ogo。虽然 Ogo 从未在这里流行起来，但它仍然存在于我们大西洋彼岸的兄弟们的手中。IXI 移动发布了 Ogo 2.0，这是一款老款手机，有着更好的显示效果和重新设计的界面。出于某种原因，它还有立体声扬声器。

不过我必须承认，新的设计看起来并不太破旧，而且很有欧洲的感觉。也许被排斥的奥格人会和那些臭烘烘的瑞士人幸福地生活在一起。
