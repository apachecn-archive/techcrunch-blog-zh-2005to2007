# Podango 将创建播客频道

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/08/29/podango-to-create-podcast-channels/>

总部位于盐湖城的 [](https://web.archive.org/web/20220702145249/http://www.podango.com/) 尚未推出的 [Podango](https://web.archive.org/web/20220702145249/http://www.podango.com/) 正试图通过将流行的专题播客聚合到专家驱动的频道来垄断播客市场。目标是将可能每周发布的多个播客合并成一个每天 3-5 个节目的频道，并提供一个 RSS 源。就像老式的电视和电台一样。

播客目录，像雅虎和 iTunes 创建的目录，是寻找新的好的播客内容的非常流行的方式。Podango 希望更进一步，将人才整合到特定主题的频道中，减少听众寻找好内容的工作。如果他们成功了，就找那些已经建立起来的玩家迅速效仿。

Podango 将招募电台总监，他们将为他们的频道招募播客，使用 Ajax 拖放管理界面管理频道，并帮助招募广告商。希望听众能从让一个人选择给定类别的最佳播客中受益，并避免因节目将一个接一个排队而造成的广播时间停滞。播客会赢得观众。电视台台长将有机会一年赚 2.5 万到 25 万美元，甚至更多。在总收入中，Podango 将保留 30%，剩下 70%给电视台台长和个人播客。
