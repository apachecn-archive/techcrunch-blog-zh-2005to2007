# Vista 在全球销售强劲

微软今天宣布其新的 Vista 操作系统首次亮相就取得了强劲的销售数字。它声称备受宣传的操作系统已经超过 2000 万个许可证，是 Windows XP 最初销售数字的两倍多。这些数字代表了发送给 PC 制造商的副本，以及销售给零售商的升级副本和完整包装的产品，以及通过 Windows Vista Express 升级计划订购的升级。

相对于 XP 的增长并不令人惊讶，因为与 2002 年的这个时候相比，现在有如此多的 PC 用户。
