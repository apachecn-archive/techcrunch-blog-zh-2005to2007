# T-Mobile:现金换旧手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/08/t-mobile-cash-for-old-cell-phones/>

上周我们报道了[手机可能成为明天](https://web.archive.org/web/20150303052512/http://mobilecrunch.com/2007/07/31/today%e2%80%99s-cool-phone-is-tomorrow%e2%80%99s-toxic-waste/)的有毒废物，但现在有消息称一些移动公司正寻求对此采取行动。T-Mobile 在英国推出了一项新举措，让人们回收他们的旧手机。

在这一努力下，T-Mobile 要求英国人发送他们自己的手机，不管它们是否能用，甚至可能因此获得一些现金。我们必须关注这个故事，目前还不清楚这是否包括旧的 T-Mobile 手机。但据报道，英国有超过 5000 万部手机没有被使用，这可能是让它们远离垃圾填埋场的一项重大努力。
