# 雅虎我的网站今晚重新启动

 [](https://web.archive.org/web/20220818165750/http://myweb2.search.yahoo.com/) 大约一年前，雅虎[在](https://web.archive.org/web/20220818165750/http://www.beta.techcrunch.com/2005/06/29/profile-yahoo-my-web-20/)推出了名为 [Yahoo My Web 2.0](https://web.archive.org/web/20220818165750/http://myweb2.search.yahoo.com/) 的新社交书签服务。从那以后，这项服务基本上没有变化，当然，雅虎在 2005 年 12 月收购了 del.ico.us。

但是，正如雅虎继续将雅虎照片作为独立于子公司 Flickr 的财产运营一样，雅虎我的网站甚至在收购 del.icio.us 之后也在继续发展。今晚太平洋标准时间 7 点，该网站将[重新推出](https://web.archive.org/web/20220818165750/http://www.ysearchblog.com/archives/000310.html)，有更好的用户界面和许多功能增强。正如 Yahoo Photos 吸收了 Flickr 的最佳功能一样，Yahoo My Web 也吸收了 del.icio.us 以前没有的一些更好的功能。

下面是新用户界面的截图。当前 UI 的截图，晚上 7 点后会消失，[在这里](https://web.archive.org/web/20220818165750/http://www.flickr.com/photo_zoom.gne?id=161315337&size=o)。

新的雅虎我的网站功能:

*   添加了“热门标签”和“今日趣闻”以供探索
*   通过页面顶部突出显示的标签进行搜索
*   浏览书签不需要登录
*   用户可以在未经许可的情况下监控其他标记者的书签
*   书签可以导出到其他应用程序
*   更多 Ajax(例如:将鼠标放在名字上查看用户的个人资料)

雅虎还从应用程序的名称中去掉了“2.0”。他们的理由？为了避免与“web 2.0”混淆。虽然我怀疑这与最近围绕这个词的商标问题有什么关系，但我确实觉得更名的时机很有趣。

雅虎对 My Web 用户的数量不予置评，但指出自该服务推出以来，已有 3000 万网页被用户收藏。
