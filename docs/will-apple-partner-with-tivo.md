# 苹果会和 TiVo 合作吗？

在博客圈有传言说史蒂夫·乔布斯将在下一届 [MacWorld](https://web.archive.org/web/20130828124010/http://www.macworldexpo.com/live/20/) 上宣布与 [TiVo](https://web.archive.org/web/20130828124010/http://www.techcrunch.com/tag/Tivo/) 的某种合作关系。

在 MacWorld 时间前后，谣言通常会变得相当疯狂。在去年的英特尔发布会上，我听说所有出席的媒体都将获得新的笔记本电脑，但这并没有发生。这可能只是一个谣言，但如果我们按照 iTV 的思路去想，这并不是一个坏主意。PVRWire 认为合作的可能性极小，但是许可协议是有可能的。TiVo 拥有品牌知名度，但市场份额在下降。苹果公司拥有不可否认的实力，尤其是考虑到他们已经在 iTunes 上为迪士尼卖出的电影数量，但他们在电视市场上完全没有立足之地。他们需要彼此。问题是，苹果并不经常承认需要任何东西或任何人。
