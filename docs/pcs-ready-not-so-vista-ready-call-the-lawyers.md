# 电脑准备好了，但 Vista 还没准备好:打电话给律师！

Dianne Kelley 厌倦了虚假的 Vista 贴纸。她走进一家商店，买了一台电脑，发现它不能运行微软 Vista 的漂亮的 Aero 界面。一个可怜的女人能做什么？苏！在一项拟议的集体诉讼中，Kelley 起诉抗议将 Vista-Ready 放在任何东西和所有带有键盘和鼠标的东西上的做法，而我们都非常清楚，一台 256MB 和绿屏显示器的戴尔电脑不会显示 Vista 的标志性任务切换功能。

因为我们 CG 公司的人是[微软的骗子](https://web.archive.org/web/20130628160408/http://crunchgear.com/2007/04/03/sonys-phil-harrison-cant-give-away-ps3/)，我们很难说母公司的坏话，但这是我们从一开始就抱怨的。所有这些荒谬的版本——基础版、终极版、倒数第二版、超级骗子版——只会让无聊和爱打官司的人感到沮丧。把一切都做到极致，让所有人都买新 PC，把低端销售的损失吸干。
