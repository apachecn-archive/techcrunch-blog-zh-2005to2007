# SanDisk Sansa Shaker 评论:你知道，对孩子们来说

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/13/sandisk-sansa-shaker-review-you-know-for-kids/>

不想让你热爱音乐的 8 到 11 岁的孩子带着 iPod 到处跑吗？SanDisk 的新 Sansa Shake 面向“青少年”,这是一款适合儿童的播放器，可以将音乐存储在安全的数字存储卡上，使用一节 3a 电池。最重要的是，它非常便宜，仅售 39.99 美元，包括一个 512MB 的 SD 卡。大一点的孩子可能不会对摇壶的大小和功能留下深刻印象，但我费了很大劲才从两个 8 岁的孩子那里把它借回来。

这个摇动器有粉色或蓝色，有两个可旋转的白色环，用于调节音量和跳过曲目。在一端，有一个集成的扬声器，另一端弹出，露出一个迷你 USB 端口，SD 卡插槽和电池盒。有两个耳机插孔——一个方便又有趣的 plus——侧面有一个电源/播放/暂停按钮。振动筛没有滤网，但这对孩子们来说可能是一个好处，因为这样就少了一个坏掉的东西。

直径约为 1.6 英寸，长约 2.8 英寸，比苹果的第二代 iPod Shuffle 和创造性的 Zen Stone 要大得多，但这可能会使它更难丢失。Shuffle 和 Stone 有更多的颜色，尽管 Shaker 有一堆可爱的贴纸。另一个考虑是 Shuffle 的价格是 Shaker 的两倍，并且没有扬声器，尽管它的存储容量是 Shaker 的两倍。

配件包括一对可能对年轻人来说较大的耳塞、一节 3a 电池和一根 USB 电缆，以及一张 512MB 的 SD 卡，可以存储大约 125 首歌曲(128Kbps MP3s)。Shaker 还预装了一些有趣的儿童歌曲，比如《巴士上的轮子》。

对于 Windows 用户来说，在 Shaker 上加载音乐很简单:只需将它插入 PC 的 USB 端口，就可以在 Windows 中传输音乐文件——不需要音乐管理软件。如果你正在传输大量文件，请准备好等待一会儿，因为 Shaker 以 USB 1.1 的速度运行。

Shaker 不支持 WMA 文件或订阅音乐，只支持 MP3 和 WAV 文件。它也缺乏对 Mac 的支持，尽管你可以为你的 Mac 买一个便宜的读卡器，直接把音乐加载到 SD 卡上。

对于小孩的小手来说，这种控制比我自己的成人手套更容易使用，而且没有什么太复杂的事情需要解决。在他们的父母说“全球开发者大会”之前，大多数孩子将能够绕过它

若要打开和关闭播放器，请按住播放/暂停按钮几秒钟，直到您听到奇怪的声音效果。您可以使用其中一个白色环来跳过曲目，或者您可以按住播放/暂停，同时摇晃播放器，让它跳到随机歌曲(伴有另一种声音效果)。

当您在播放器暂停时摇动播放器、电池电量低或在播放过程中取出 SD 卡以及 SD 卡上没有音乐时，会播放其他声音效果。不幸的是，没有电池指示灯，尽管根据电池的品牌和类型，你可以获得大约 15 个小时的播放时间——对于这种类型的播放器来说还不错。我喜欢你可以使用商店购买的碱性电池或可充电电池，这样当播放器需要充电时，孩子们就不会被电脑束缚。

我的评测单元的音质有问题，这似乎是 SanDisk 早期运行的标准。失真到你可以听到原始音乐中没有的幽灵般的音符。例如，当你听到一个喇叭时，听起来就像在平行维度中有一个或两个其他发出刺耳声音的喇叭在微弱地演奏。SanDisk 的一名代表表示，由于目标年龄，该公司故意在音质上偷工减料，但这在我看来有些过分。

总的来说，这个摇动器足够便宜(39.99 美元)，可以作为一个有趣的圣诞礼物，这肯定是它的主要吸引力。Shaker 以其古怪的功能和声音效果让成年人兴奋吗？没有。但是从我的两个 8 岁的邻居为它打架的方式来看，这可能会对孩子们产生影响。
