# 益智闹钟叫醒你，让你生气

![](img/d12eab98154167234594435228f4cc62.png)

也许[亲自起床去找你的闹钟](https://web.archive.org/web/20210225032218/http://crunchgear.com/2007/01/30/clocky-is-now-shipping/)不适合你。没关系。如果你更喜欢思考，那么仍然有一个独特的、烦人的闹钟适合你。这个拼图闹钟的顶部有三块拼图，当闹钟响起时，它们会弹出来。

要关闭警报，你需要用你的脑袋来解谜。这有点像醒来后玩那个蹩脚的老游戏[完美](https://web.archive.org/web/20210225032218/http://www.playthingspast.com/hb717.html)的混合。如果你想要一个，准备好拿出 60 美元和你的解谜技巧。
