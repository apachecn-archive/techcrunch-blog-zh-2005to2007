# Skype+Outlook = sky look | TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2006/03/19/skype-outlook-skylook/>

 [](https://web.archive.org/web/20150924121848/http://www.skylook.biz/) 我第一次[写](https://web.archive.org/web/20150924121848/http://www.techcrunch.com/2005/08/20/profile-skylook-for-skype/)关于基于澳大利亚的 [Skylook](https://web.archive.org/web/20150924121848/http://www.skylook.biz/) 早在八月份他们发布 v. 1.0 的时候。这是 Outlook 的一个插件，可以很好地将 Skype 与电子邮件结合在一起。

他们不断创新，以保持 Skype 自己的 [Outlook 工具栏](https://web.archive.org/web/20150924121848/http://www.techcrunch.com/2005/07/06/profile-skype-outlook-toolbar/)产品的领先地位。虽然 Skylook 不是免费的，但它有许多出色的功能，包括在线状态(从 Outlook 中查看谁在线)、与通讯录的集成、通话录音和转换为 MP3 格式，以及在 Outlook 中存档。相比之下，Skype toolbar 基本上是一种仅从 Outlook 通过 Skype 发起即时消息和语音通话的方式。

您不必在工具栏中启动呼叫来记录它们。只要 Outlook 在后台运行，Skylook 就可以自动记录通话和即时消息对话，并为您存档。请注意，Skylook 不支持视频通话录音。

还有许多其他出色的功能，包括同步 Outlook 和 Skype 联系人，以及在 Outlook 收件箱中接收 Skype voicemals。有关升级和新功能的完整列表，请参见此处的[新功能页面。](https://web.archive.org/web/20150924121848/http://www.skylook.biz/whatsnew/)

正如我提到的，Skylook 不是免费的，尽管他们提供了不包括录音的免费基本选项。许可费高达 50 美元。
