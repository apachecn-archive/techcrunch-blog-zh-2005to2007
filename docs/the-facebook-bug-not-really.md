# 脸书的“虫子”不完全是。

只有重度[脸书](https://web.archive.org/web/20200808212438/http://www.crunchbase.com/company/facebook)用户才会对这个帖子感兴趣。

许多博主都在[抱怨](https://web.archive.org/web/20200808212438/http://www.techmeme.com/070723/p7#a070723p7)脸书在询问你如何认识一个人并接受他们为新朋友时取消了“跳过这一步”选项。“bug”出现在周末。

首先，它可能是一个 bug，将被修复，但有一个简单的方法来回避它。只需点击“取消”和点击“跳过这一步”的效果是一样的。你不必想出正确的方式来说你是如何从[非常有限的选项](https://web.archive.org/web/20200808212438/http://www.beta.techcrunch.com/2007/06/01/request-to-facebook-broaden-the-how-i-know-you-options/)中认识这个人的。点击取消。相信我，当你进入你的个人资料时，这个人已经被添加为好友了。

然而，每个人似乎都忽略了一个事实，上周晚些时候[脸书](https://web.archive.org/web/20200808212438/https://crunchbase.com/organization/facebook) 移除了每次添加好友时的整个页面重定向，并代之以弹出框。对大多数人来说，这没什么大不了的，但是我倾向于添加很多朋友(我有超过 1200 个朋友)，并且需要几秒钟的时间通过多次页面刷新来确认每个朋友，这是一个巨大的痛苦。现在整个过程缩短到一秒钟。巨大的生产力收益，脸书牺牲了页面浏览量给了我们这个功能。高尔夫为脸书鼓掌。

**更新:** bug 没了。跳过这一步又回来了。每个人都冷静。
