# Vista 落在自己的剑下，被砍成碎片

![](img/ed11b67afce452dc0686598fcc792c76.png)

Haxors 发现了如何欺骗 Vista，让它认为自己在 OEM 机器上运行，并完全绕过激活和有效性检查。这种黑客行为包括添加一个特定于某个 OEM 厂商的 BIOS 证书，然后 Vista 将其视为合法。

在暴力破解之后——这似乎是[骗局](https://web.archive.org/web/20210227045402/http://www.engadget.com/2007/03/03/windows-vista-brute-force-keygen-a-hoax/)——以及类似[黑客](https://web.archive.org/web/20210227045402/http://thepiratebay.org/tor/3631593/Windows_Vista_Activation_Crack_(vstaldr)_NO_TIME_STOPPER)的事情，很明显 Vista 激活只是一个减速带。不过，我可以建议你出去买 Vista。这是一个[好产品](https://web.archive.org/web/20210227045402/http://crunchgear.com/2007/02/07/apple-vs-the-world-os-x-or-vista/)，一个强大的升级，也是一个名为黑客的猫捉老鼠游戏中值得一提的对手。
