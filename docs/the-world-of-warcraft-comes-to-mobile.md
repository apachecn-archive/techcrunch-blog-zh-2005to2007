# 《魔兽世界》来到了移动领域

从今天开始，玩家将有机会随时随地追踪他们的魔兽世界角色。GoWare 为其 DoMo 主页个性化移动门户发布了一个插件，它将使大型多人在线角色扮演游戏的玩家能够从移动手机上访问和共享他们的 WoW 角色数据。

有了 DoMo 主页快照功能，玩家甚至可以发送角色内容短信。世界(魔兽世界)可能永远不会一样了。
