# 三星 SCH-i760 获得 FCC 批准| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/12/samsung-sch-i760-gets-fcc-approval/>

![](img/cf2a3d44c26b7bb3d40172df4a3f0123.png)

那些威瑞森的人应该停下来，花点时间，看看你可能会得到的下一部手机:三星 SCH-i760。HTC 手机的设计看起来像是与 21 点做爱，并具有蓝牙、130 万像素摄像头、WiFi 和触摸屏等功能，大多数人都会很高兴有一部像样的手机出现在你面前。

i760 还将运行 Windows Mobile 6.0，可以录制视频和播放 MP3。预计将在未来六个月内上市。
