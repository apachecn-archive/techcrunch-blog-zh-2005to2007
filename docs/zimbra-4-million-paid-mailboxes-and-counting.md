# zimbra:400 万个付费邮箱，而且还在增加

[](https://web.archive.org/web/20230401063248/http://www.zimbra.com/)Digg、Mozilla、瑞士伯尔尼大学、印度时报有什么共同点？根据今天的公告，他们都是基于 web 的开源通信和协作套件 [Zimbra](https://web.archive.org/web/20230401063248/http://zimbra.com/) 的 1000 多个客户。

该公司于 2003 年在加利福尼亚州圣马特奥成立，今天宣布它已经通过了 400 万个付费托管和现场邮箱。与微软交易所销售的 1.5 亿多个席位相比，这是一个很小但不断增长的非常重要的数字。Zimbra 从 Benchmark Partners、Redpoint Ventures 和 Accel Capital 获得了 1600 万美元的资金。

我们已经描述了谷歌、Zoho、微软和无数小公司向基于网络的办公室发展的趋势。 **Zimbra 包括一长串其他公司刚刚开始提供的功能。微软表示，Exchange Server 2007 将于今年年底或明年年初推出；如果它真的在那个时间框架内成为最终版本，看看它是否能做 Zimbra 能做的事情将会很有趣。**

网络邮件客户端的外观和操作与 GMail 非常相似，但支持 Outlook 中的电子邮件、日历和联系人。
有 RSS 提要阅读、Salesforce 集成、无需下载单独客户端的移动访问(尽管显然没有 Windows mobile 支持)、标记、文档和电子表格共享和协作，以及富文档的 Ajax 嵌入。Zimbra 明确支持混搭和微软产品需要第三方工具来执行的语音集成。

这是一个令人印象深刻的产品，显然被越来越多的人采用。如果你正在寻找支持 Web 2.0 在企业中的可行性的证据，Zimbra 今天的客户公告提供了优质的素材。
