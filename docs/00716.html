windows Live–不仅仅是一个AJAX桌面

今天，我在CrunchNotes 上实时发布了我在微软新闻发布会上未经编辑的原始笔记。两个主要公告之一是 Windows Live (当然最好在IE上观看)。还宣布了Office Live，我将在今晚晚些时候写一写。

Windows Live是一个免费的、支持广告的AJAX虚拟桌面。大多数功能可以在名为 Start 的微软沙盒项目中看到，我们几个月前对其进行了分析。

然而，微软增加了许多新功能，为产品增加了很多价值。其中包括电子邮件集成，一个新的即时消息客户端，类似plaxo的联系人管理和类似skype的功能，允许向普通POTS电话拨打电话。Windows Live还可以通过“小工具”进行扩展。在今天的经历之后，我对许多硅谷初创公司感到绝望。

桌面

核心特性包括一个(现在的)标准AJAX桌面，一个拖放界面，改变列数的选项，保存搜索的能力，等等。左栏有一个预填充的feed阅读器，添加新的feed只需一键操作。

然而，真正特别的东西是小工具。这些是第三方应用程序，可以通过左上角的“添加内容”功能添加到Windows Live。对于当前的小工具，检查出MicrosoftGadgets.com。

我添加的一个小工具是Flickr浏览器。添加它需要几个步骤，你可以通过标签查看Flickr照片。既然任何人都可以创建一个小工具，我希望在不久的将来能看到数以千计的so选项。

电子邮件

您可以将邮件帐户直接添加到Windows Live页面，并直接在浏览器上获得类似Outlook Express的功能。这项技术看起来直接来自卡哈纳项目，我们已经写了好几次了。

即时消息和VOIP

这是我今天看到演示的最酷的东西。Windows Live将很快在网站上嵌入新的即时消息客户端。即时消息功能将包括像Skype一样拨打POTS外拨电话的功能。在这方面，他们已经超越了谷歌的Gtalk。

还有一个高级联系人管理功能，类似plaxo的功能(你可以控制自己的联系信息，人们可以随时更新)。

布莱克·埃文带领我们进行了一次演示，其中包括搜索旧金山泛美大厦附近的餐馆。当他放大并缩小选择范围时，他选择了一家餐馆，并通过点击拨打餐馆的普通电话进行预订。我拍了一张照片，在flickr 上了。左边是VOIP呼叫者的照片。

即时消息功能将于12月发布。
