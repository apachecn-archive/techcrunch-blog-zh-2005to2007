# 特工侠| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/19/secret-agent-man/>

来自祖国的更多好消息，孙杨 DNT 刚刚发布了沃尔沃(你不知道我现在不笑出声来有多难)。这个聪明的小 USB 记忆棒在母国的市场上支持 VoIP 摄像头功能。有了 USB 线，你可以调整视角，显然你甚至可以通过一些相机调整来制作电影。
支持 30 万像素摄像头，30fps，640×480 分辨率。使用阳光 DNT 的摄像头调谐技术，用户可以使用 Skype 或 MSN 等网络信使服务来即时使用网络摄像头，同时还可以将其作为记忆棒随身携带。你现在有能力成为你梦寐以求的间谍，发送所有的秘密文件。
