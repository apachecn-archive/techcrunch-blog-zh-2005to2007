# 十四行诗科技给你四个输入，没有等待

本周早些时候，我们在[贴出了一组 Mac 配件](https://web.archive.org/web/20210305111041/http://crunchgear.com/2007/02/07/apple-vs-the-world-playing-dress-up-with-your-apple/)，其中包括一款西部数据 My Book Pro，具有三个输入:FireWire 800、FireWire 400 和 USB 2.0。然而，Sonnet Technologies 的新 Fusion R400Q 四托架机架安装驱动器更上一层楼，有四种接口选择——FireWire 400、FireWire 800、USB 2.0 和 SATA II。集成端口倍增器通过任何连接支持多达四个硬盘驱动器。

尽管 699 美元的独立于平台的 R400Q 具有可热插拔的 3.5 英寸驱动器托盘，并且能够与另外三个 R400Q 一起设置在 RAID 阵列中，但它与 My Book Pro 不是同一市场。但是，如果您需要更昂贵的存储解决方案的最佳连接和可扩展性，它绝对值得一试。好吧，至少它会在三月份发行。
