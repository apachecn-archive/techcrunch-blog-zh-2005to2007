# 未来主义者:手机、苹果和 RIM 的下一步是什么

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/26/the-futurist-whats-next-in-cellphones-apple-and-rim-edition/>

![](img/32dbc892fdfeff7651ad03c43a9c41d9.png)

手机——每个人都有。甚至你 12 岁的侄子也带着 RAZR，他的父母通过内置 GPS 监视他。那么，这个无处不在的设备的下一步是什么呢？

在本周的专栏中，我们来看看 iPhone 的未来，以及我们可以通过它了解苹果即将推出的 Macbooks，以及为什么我们可以期待在不久的将来看到黑莓操作系统无处不在。

**我们会得到哪个版本的 iPHONE？**

甚至在 iPhone 正式存在之前，博客圈就已经有了这样的猜测，即首次发售将只是苹果制造的多层次音乐手机系列中的第一款。推出三种尺寸的 iPods 对苹果公司来说效果很好，那么他们为什么不在手机上做同样的事情呢？

大多数猜测都是基于这样一种假设，即 500 美元的机器只能是顶级的(就像 80g 的 iPod)，随后的版本只会更便宜。然而，尽管所有 iphone 的价格最终都会下降，但最初的硬件几乎肯定不会是最高端的手机，而是可能会演变成“纳米”产品。

在短时间内，苹果公司以“纳米”品牌打出了全垒打。随着这第一部手机的出现，我们对“纳米”产品的所有联想都已经具备——它很薄，使用闪存。

现在，让我们更进一步。如果苹果将他们的“纳米”品牌扩展到 iPod 和 iPhone 之外，进入他们的笔记本电脑，会怎么样？如果大家已经讨论了大约一年的神秘超薄 Macbook 或苹果平板电脑实际上被冠以“Macbook Nano”的品牌会怎么样？如果它运行在固态存储器上呢？

 **黑莓操作系统将无处不在**

PDA 的手机粉丝们有一个小问题——大多数最热门的新手机都与特定的操作系统相关联，几乎所有的操作系统都有严重的缺陷。Windows Mobile 比冰柜还冷，会耗尽你的电池，Palm OS 很少见，而且越来越少，Symbian 还不错。但是黑莓 OS 呢？它近乎完美，这也是黑莓手机成为各地商人默认同款手机的原因。它永远不会死机，处理电子邮件时工作得非常好，从你的电池中吸取的电量少得惊人(不像 Windows Mobile 设备，需要在中午前插上电源，你可以把你的黑莓手机带到国外一周，不需要打包你的插座转换器就能应付过去)。这款操作系统的一个传统缺点是缺乏与第三方软件的兼容性，现在这几乎已经不是问题了，部分原因是谷歌一直在向它移植所有的移动应用程序。

如今，黑莓操作系统的一个缺点是它只能在 RIM 自己的设备上使用。我预测我们会看到这种情况的结束——而且很快。只要 RIM 能够避免重蹈 Palm 试图在软件和硬件方面瓜分其王国的覆辙(这一决定以他们的一些硬件向微软挥舞白旗而告终)，RIM 就可以通过将他们的软件授权给其他手机制造商来赚取大量现金。

当然，他们会犹豫这样做——他们可能担心这样做会部分蚕食他们自己硬件的销售(正如他们所说，当他们可以免费许可牛奶时，为什么有人会买奶牛呢？)然而，这种担心被误导了，原因有二。

首先:已经开发的授权软件的利润率可能是天文数字。他们可能不需要投入太多资金进行研发，就能让他们的操作系统在其他手机上运行，而且发布这些操作系统也是轻而易举的事情。这意味着从软件中获得的几乎每一分钱都是纯利润。

第二:让他们的软件更广泛地被使用，可以为他们自己的手机做广告。如果有人是一个 Treo 上瘾者，一周内从未使用过黑莓手机，而他们购买了一部黑莓驱动的 Treo，并逐渐爱上了它，那么他们的下一部手机很有可能是 RIM 的。
