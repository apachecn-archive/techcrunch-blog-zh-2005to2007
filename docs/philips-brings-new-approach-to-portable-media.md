# 飞利浦为便携式媒体带来新方法

飞利浦正在做一件通常不为人知的事情——在数字媒体领域进行创新。他们的新 PMP PMC 7230 在 3.5 英寸的屏幕上播放 MPEGs、MP3 和安全的 Windows Media 文件。虽然没有什么特别的，但它有一些其他独特的东西，使它从拥挤的人群中脱颖而出。

以其画框式的外形为例。PMC7230 的背部有一个旋转臂，不适合拿在手中，因此您可以将其支撑起来，像相框一样放在桌子上或飞机上的座椅靠背上。此外，它内置的 A/V 输入允许您录制任何标准的 NTSC 或 PAL 信号源，因此您可以加载您最喜爱的节目，以便稍后观看，而无需使用 PC。干净利落。

PMC7230 还配有 30GB 硬盘和电池，能够播放 18 小时的音频或 5 小时的视频。它的零售价约为 350 美元，应该很快就会上架。凭借其深思熟虑的附加功能，这将成为今年流行的 iPod 替代品。
