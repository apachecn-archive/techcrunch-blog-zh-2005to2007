# Wii Mod-Chip:成功了

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/05/wii-mod-chip-it-works/>

我已经确认 CycloWiz 是真实的，它是一款 Wii mod-chip，允许备份、导入、DVD-/+R 兼容性以及增强的 GameCube 兼容性。

CG 团队的一位密友在这个周末安装了他的 CycloWiz，目前正在享受他焊接工作的成果。他说升级非常容易，他只有打开机箱的问题，因为它有任天堂最喜欢的[“三角螺丝”](https://web.archive.org/web/20191223211529/http://en.wikipedia.org/wiki/Tri-Wing)把东西固定在一起。
