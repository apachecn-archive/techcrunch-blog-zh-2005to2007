# 研究人员将下载速度提高了 500%

[![fastp2p.jpg](img/a86aca21e7323ee66d7d6a4278255296.png)](https://web.archive.org/web/20230322164155/https://techcrunch.com/wp-content/uploads/2007/04/fastp2p.jpg "fastp2p.jpg")

美国研究人员想出了一种将文件共享速度提高 500%的方法，让你的“快速”BitTorrent 速度大打折扣。这种新方法不是像 BitTorrent 那样将单个文件分成几个部分，而是扫描一大堆文件，然后从那里重新组合数据。例如，查找你最喜欢的丽兹·费儿歌曲——当然是“离婚歌”——会从所有的歌曲中抓取相关元数据，从而加快下载速度。

当从某些 BitTorrent 追踪器下载时，我通常最大限度地使用我的连接(大约 1MB/s ),所以五倍的速度是官方没有必要的，尽管完全赞赏。让我们看看罗杰斯公司的[油门](https://web.archive.org/web/20230322164155/http://crunchgear.com/2007/04/12/use-rogers-your-torrents-might-be-useless/)那个。
