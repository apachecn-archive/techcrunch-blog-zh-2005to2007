# TXTDrop.com:免费短信

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/20/txtdropcom-free-sms/>

本着 [AllCallsForFree](https://web.archive.org/web/20230322181414/http://crunchgear.com/2007/02/05/free-international-calling-yanked-not-quite/) 的精神，或者不管爱荷华州最新的免费国际号码是什么，TXTDrop 正在提供免费的短信通话。没有陷阱，只是在广告服务的消息末尾添加了一个标签。甚至还有一个 OS X 插件和强制性的 Web 2.0 标志。

试一试——趁它还是免费的。
