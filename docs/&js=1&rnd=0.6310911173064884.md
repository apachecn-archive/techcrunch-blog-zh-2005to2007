[![](img/e3655cd4fd3b1666e90e43642e24ae0a.png)](https://web.archive.org/web/20070727035945/http://www.hotswap.com/) 二手车销售是一项大生意——在美国每年至少有 3700 亿美元，其中 810 亿美元左右发生在网上。新创公司 [HotSwap](https://web.archive.org/web/20070727035945/http://www.hotswap.com/) 希望将网络上的一些最新趋势应用到那个市场，并把更多的买家和卖家聚集在一起。虽然二手车销售并不十分吸引人，但 HotSwap 是一个非常酷的网站。想想二手车销售的 [YouTube](https://web.archive.org/web/20070727035945/http://www.crunchbase.com/company/youtube) 。

这家位于加州伯克利的公司允许任何人挂牌出售二手车。除了基本信息之外，人们还被要求在列表中加入车辆的视频。他们说，视频是展示汽车的更好方式。随着简单摄像机的普及(大多数数码相机也拍摄视频)，几乎每个人都可以轻松地创建和上传视频。详情见帖子最后的视频。

个人列表是完全免费的。交易商为每笔交易支付少量费用，还会购买更先进的软件来跟踪交易情况。

该网站有相当有用的 Ajax 工具来优化搜索，允许访问者寻找特定的颜色、汽车类型、位置、里程和价格。可以通过列表直接提供报价，或者观众可以提问等等。列表也将能够嵌入到其他网站。

[![](img/ae19bb1c45ce9701a7879d971d5261a6.png)](https://web.archive.org/web/20070727035945/http://www.techcrunch.com/wp-content/hotwapb.png) 房源清淡，因为网站是预推出的，没人听说过。但是他们已经和 Red McCombs Automotive 签订了协议，该公司每年销售额约为 8 亿美元，将在 HotSwap 上列出每辆二手车。该公司表示，他们也在与年收入近 200 亿美元的 AutoNation 进行谈判。

该网站尚未正式上线。没有凭证阻止人们访问，但许多功能还没有完成，还有一些错误。该公司表示，他们将在未来几天正式推出。

HotSwap 上个月从金赛希尔集团(Kinsey Hills Group)筹集了“大约”200 万美元的资金(该集团也资助了 Scribd。当然，他们与易贝正面竞争，还有像 [Vast](https://web.archive.org/web/20070727035945/http://www.crunchbase.com/company/vast) 这样的网站，以及在较小程度上的 [Edgeio](https://web.archive.org/web/20070727035945/http://www.crunchbase.com/company/edgeio) (我共同创立的)。
