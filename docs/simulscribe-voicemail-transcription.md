# 同步语音邮件转录

![](img/ea66e3c8b9bc06aed583b43bc6e9db76.png)
Simulscribe 是一项手机服务，可以监控和转录您的语音邮件。当你收到一条信息，它会把它转换成文本，并通过电子邮件把信息的副本发给你。它还包括原始邮件的音频文件，以便您可以检查错误。

当我谈到转录这个话题时，有人知道一个可以准确转录的程序吗？wav 还是. mp3？我一直在寻找一种将讲座录音转换成文本的方法，但至今没有成功。
