# MyLifeBrand 聚合社交网站

![mylifebrand.jpg](img/58077cbd6d54096fa5263e6799699cf7.png)MyLifeBrand 是一项服务，它允许用户聚合社交网络会员，并从一个地方在它们之间导航。

[MyLifeBrand](https://web.archive.org/web/20201020151059/https://crunchbase.com/organization/mylifebrand) 支持 [Friendster](https://web.archive.org/web/20201020151059/http://crunchbase.com/company/friendster) 、 [LinkedIn](https://web.archive.org/web/20201020151059/http://crunchbase.com/company/linkedin) 、 [Bebo](https://web.archive.org/web/20201020151059/http://crunchbase.com/company/bebo) 、[脸书](https://web.archive.org/web/20201020151059/http://crunchbase.com/company/facebook)、H15、Orkut、 [MySpace](https://web.archive.org/web/20201020151059/http://crunchbase.com/company/myspace) 和 TagWorld 等，并致力于支持许多小众社交网络，包括钓鱼大师、海豹突击队和醉鸭。

用户还可以将外部网络中的联系人添加到他们的 MyLifeBrand 好友列表中，从而创建一个主好友列表。

管理多个社交网络的困难是真实的，任何重度 Web 2.0 网站用户都会理解这个问题。我们在 2006 年 11 月报道了发言人，此后还报道了许多类似的服务。

MyLifeBrand 给了社交网络聚合一个不错的机会，但是在一个框架中呈现外部网站对我来说不起作用(见截图)。

[脸书的](https://web.archive.org/web/20201020151059/https://crunchbase.com/organization/facebook) [F8 平台](https://web.archive.org/web/20201020151059/http://www.beta.techcrunch.com/2007/05/24/facebook-launches-facebook-platform-they-are-the-anti-myspace/)的引入显示了这个空间的真正方向。[脸书](https://web.archive.org/web/20201020151059/http://www.beta.techcrunch.com/tag/facebook)允许外部网站和服务直接集成到脸书，而不是像 MyLifeBrand 那样通过框架集成，提供卓越得多的用户体验。这并不是说 MyLifeBrand 将无法找到用户群，只是他们可能在发布方面晚了 12 个月，在交付和集成方面落后 6-12 个月，无法成为主要参与者。长尾是长的，总是有新来者的空间，所以我祝他们好运；与脸书竞争将是一个挑战。
