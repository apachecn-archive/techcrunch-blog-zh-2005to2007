# WWDC 横幅揭示秘密

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/06/07/wwdc-banners-reveal-secrets/>

一些狡猾的摄影师在 WWDC 会议开始前就设法拍下了一些关键镜头。在里面，有一些横幅写着“欢迎来到 WWDC07 拓展你的宇宙。”和“Mac OS X 豹火箭燃料为您的应用程序。”显然这意味着 OS X 豹将成为今年 WWDC 的明星。除了 Leopard，新的 IMAC、Mac Pros、iPods、扬声器、USB 电缆和 Mac Minis 都将在大会上宣布。为什么？因为这是我说的，这些横幅就是我的证据。

看到那个写着“拓展你的宇宙”的奇怪横幅有多长了吗？嗯……这意味着任何大的产品都会更新。这不包括 iPod 和 Mac Mini，但肯定包括 Mac Pro 和 USB 电缆。另外，仔细看看印在横幅上的行星和恒星。一种是蓝色，这是 OS X 默认壁纸的颜色。这是否意味着我们可以在 OS X 看到新的壁纸？绝对的。星星表明，史蒂夫·乔布斯在介绍一些新的 Leopard 功能时，将会跳艾拉·费兹杰拉的混音版霹雳舞。孩子们，WWDC 快到了。不要担心谣言，让事件展开。
