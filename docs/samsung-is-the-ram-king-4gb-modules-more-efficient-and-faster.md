# 三星是内存之王:4GB 模块，效率更高，速度更快

[![samsungdram.jpg](img/64d03fbabb8f085b83bc9f88305dd841.png)](https://web.archive.org/web/20210301114647/https://beta.techcrunch.com/wp-content/uploads/2007/04/samsungdram.jpg "samsungdram.jpg")

三星已经找到了一种将 DRAM 模块堆叠在一起的方法，这最终会带来——你猜对了——更快、更高效的内存。另外，它还允许三星制造 4GM 内存模块，这是第一次。这项新技术被称为晶圆级加工堆叠封装，或 WSP，它基本上允许三星将内存模块相互咬合，而不会在它们之间留下空间。以这种方式生产的 RAM 可以达到每秒 1.6 千兆位的速度。

没有什么比“更好、更快”的公羊故事更能让这些周一轻松起来了。
