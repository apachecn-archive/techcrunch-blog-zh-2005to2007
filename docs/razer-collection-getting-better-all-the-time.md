# 雷蛇系列:越来越好

![](img/579d724fc31cf59d55fdf70d2a0c6031.png)
[在大张旗鼓之后，雷蛇](https://web.archive.org/web/20210225063845/http://crunchgear.com/?s=razer)已经开始稳步发布其高端游戏外设系列的硬件。最新的是某种被称为樱井真子的 2.1 声音系统。THX 认证的系统使用两个卫星扬声器，一个低音炮，和一个像游戏棒棒糖一样挂在外面的摇摇晃晃的小遥控器。

它会改善你的游戏音频吗？CES 上展示的只是一个模型，但如果 Razer 的其他设备是任何迹象表明你可能会很幸运。还可以找找 Razer 的 AC-1 声卡、Taranula 键盘、某种蓝色 LED 战斗灯，以及长着黑加法器胡子的死亡加法器游戏鼠标。
