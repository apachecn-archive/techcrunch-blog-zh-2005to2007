# 摇滚乐队芬德尔·斯特拉特亮相

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/19/rock-band-fender-strat-unveiled/>

作为一名吉他手，*吉他英雄*既很好玩又很讨厌。不管 GH 系列有多有趣，玩电子游戏是很难忘记吉他的。下一部， *[摇滚乐队](https://web.archive.org/web/20140113155435/http://www.crunchgear.com/?s=rock+band)* 将推出一款令人惊叹的授权 Fender Stratocaster 吉他控制器。我要说的是他们这次真的做到了。吉布森 SG 控制器是平庸的，但这是一件艺术品！

像其他*吉他英雄*控制器一样，它的颈部有彩色按钮，并像真正的 Strat 一样有一个拾音器选择器和三个音量旋钮。我敢打赌，这种感觉甚至就像 Strat 一样——光滑流畅。我等不及要开始舔这个东西了，我相信你也一样兴奋。不过现在，你要么买一个真的，要么等到它发布。我敢打赌，通过为这些吉他控制器发布不同颜色的撬棍等配件，可以赚很多钱。你先在这里听到的！
