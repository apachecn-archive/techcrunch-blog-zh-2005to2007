# Treo 750 实践

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/02/treo-750-hands-on/>

作为 Palm 的最高产品，Treo 750 可谓任重道远。作为移动通信领域的先驱之一，它的产品已经产生了如此凶猛的粉丝，以至于它们可能只会被苹果粉丝的“想要史蒂夫·乔布斯的孩子”所取代。可以肯定的是，Palm 过去已经制造了一些精致的设备，但是它的最新产品能像火炬一样明亮吗？

我将是第一个承认，这是一个漫长的未来。这个设备我已经用了几个星期了，但是我花了一段时间才适应它。在对 750 进行测试时，我经历了一段学习曲线(阅读:[21 点](https://web.archive.org/web/20210228211555/http://crunchgear.com/2006/11/30/samsung-blackjack-q-who/)退出)。这并不是对 Palm 或任何东西的抨击，它只是与我完全适应的手机不同。也就是说，我对 750 的体验基本上是积极的。

与早期 Palms 相比，最显著的改进是取消了突出的天线。这使得该设备更适合口袋交通。它仍然保留了相当大的体积，这将导致口袋膨胀，但 750 弥补了其性能的周长。毫无疑问，Palm 手里有一个强击手。

750 配备了 300MHz 处理器，可以轻松完成任务。我使用该设备时没有延迟，这可能是我整个体验中最令人耳目一新的方面。我开始期待，甚至接受手机设备的延迟反应。我不喜欢这样，但这是游戏的本质。然而，很明显，这种情况正在开始改变。

Palm 加大了赌注，为有趣的一年设定了赌注。其他制造商将被迫加快性能，以便涉足新的竞争领域。这对乔消费者来说显然是一个好的发展。

除了运行流畅之外，750 还有 63MB 的内存，这对于我安装的两三个程序来说绰绰有余。不过，如果你想用它的 130 万像素摄像头拍照，你可能需要购买一张 miniSD 卡。它可以接受高达 2GB 的卡。

它有效地利用了 Windows Mobile Pocket PC Edition 操作系统。Pocket PC edition 一开始可能会有点不知所措，因为它太全面了。Palm 做了一些很棒的功能调整，在保持大量功能的同时抑制了膨胀。我最喜欢的功能仍然是它的信息中心。短信线程像即时消息一样运行，并保持对话，这是一个非常好的补充。

750 的另一个伟大之处是它非常小，基本上是一个不用动脑筋就能想到的东西。该设备顶部有一个调节铃声的小拨动开关。向一个方向转动可以打开铃声，向另一个方向转动可以使它振动。我知道其他手机有这个功能，但我的手机似乎都没有，而且非常有用。

该设备采用分辨率相当低的 240 x 240 显示屏。这可能是 750 最大的败笔。当然，如果 Palm 有一个标准的 320 x 240 显示屏，750 将会是一个带有喇叭接收器的老式旋转式电话的大小。但是缩小的屏幕对整个包装有负面影响。网站单调乏味，程序显示令人恼火，房地产变得比在 OC 更有价值——我很难接受这一点。不管怎样，这个屏幕确实投影得很好很清晰。

另一个问题是它缺乏 HSDPA。是的，它有 UMTS，这并不糟糕。它会把 EDGE 当早餐吃，过一会儿再回来吃，但它不是 HSDPA。对于那些有 HSDPA 功能手机的人来说，你知道我在说什么。一旦你尝到了移动宽带之神甜美的甘露，就很难再忍受其他任何东西了。这就像试图使用拨号调制解调器(那些东西还存在吗？);它超越了愤怒。Palm 表示，HSDPA 补丁将在未来几个月推出，但现在还没有。与此同时，我试图加载的那个网站也没有。

它确实在很多方面弥补了这些抱怨。我有一个首席运营官朋友，他对 750 深信不疑，因为它能够高效地(用他的话说是“完美无缺地”)与他的服务器、大型机以及首席运营官们摆弄的任何东西进行交互。正是在这里，你不得不欣赏 Palm 的创作之美。它可能不像某些设备那样漂亮或花哨，但它是一款备受推崇的主力产品。行业专业人士已经抓住了它的执行能力，一次又一次。消除怀疑是一种隔离。一家公司在设备上可能会有一些缺点。它会把一些事情搞砸。但如果总的来说，它让自己的产品发挥了作用，那么赦免将会以极快的速度写出来，因为没有比可靠性更大的卖点了——这就是 Palm 在这里的卖点。

当然，就像他们在南方老家说的那样，“可靠性可不便宜”。他们还说你得到了你所付出的。我认为真相介于这两种陈词滥调之间。Treo 750 的合同售价为 399 美元，649 美元免费，是一个小引擎。

值得吗？我认为，重要的是要考虑到一个产品是其各个部分的总和，在这里，这些部分加起来就相当有价值。它不是一个适合所有人的设备，但是很少有产品能够得到普遍推广——而且很少有产品是小玩意。所以回答我的问题，我会说是的。来自 Palm 的 Treo 750 是一个值得的投资，但是当然，我不会很快放弃我的 21 点。
