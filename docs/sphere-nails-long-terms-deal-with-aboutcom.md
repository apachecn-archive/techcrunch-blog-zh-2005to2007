# 球形钉与 About.com 长期合作

总部设在三藩市的环球公司的 T2 成立仅三个月，但是它已经锁定了两项重要的交易。这两笔交易都利用了其“Sphere It”技术，该技术对被搜索页面内的文本进行语义分析，并返回与文章相关的博客结果。

五月，我们写了《环球》与 Time.com 的交易。下周，Sphere 将宣布与《纽约时报》的子公司 About.com 达成协议，在《About.com》的文章中嵌入核心的 Sphere It 技术。

与 Time.com 的交易不同，About.com 将只使用这项技术来整理自己的内容。结果可以在“相关文章”标题下的文章正文中看到见[此处](https://web.archive.org/web/20201125150227/http://golf.about.com/cs/beginnersguide/a/golftipping.htm)举例。

这两家公司已经对这项技术进行了长达一年的付费测试(在核心的 Sphere 搜索引擎推出之前很久就开始使用)。据 Sphere 首席执行官 Tony Conrad 称，测试的表现远远超出了预期，About.com 已经同意签署一项长期协议来授权该技术。

竞争对手 Technorati 与华尔街日报在线、华盛顿邮报和新闻周刊等主要合作伙伴有自己的交易。然而，这些交易指向博客上发表的特定文章，而 Sphere It 专注于内容本身，找到类似的文章，不管它们是否链接到原始来源。见[这篇文章](https://web.archive.org/web/20201125150227/http://sphere.wordpress.com/2006/05/12/week-one-in-the-rearview-mirror/)更详细地描述了两家公司提供的服务之间的区别(坦白地说，两者都有用，尽管 Sphere It 更适合链接较少的新内容)。
