# 洛卡让你的家安全

![](img/672ac5d80f2c08974902bc79e787e91d.png)

它的名字让我们想起了一位意大利绅士会说的话，你真的不会错的。Locca 是一种数字无线开锁器，可在 50 米外工作。你只要按一下按钮，viola，你的门就开了——非常适合你手忙脚乱的时候。

如果 Locca 失败了，你可以用你的普通钥匙作为替代，所以不用担心被锁在雪地里。整套系统售价 290 美元，现已上市。
