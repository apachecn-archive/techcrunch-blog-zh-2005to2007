# 苹果因 iPhone 推迟 OS X 豹

苹果公司宣布 OS X 豹现已正式延迟发布。原因？iPhone。

没错，Cupertino 不得不从 Leopard development“借用一些关键的软件工程和 QA 资源”，并应用于 iPhone 生产。因此，Leopard 将于 10 月份首次亮相，而不是在 6 月份的 WWDC 活动上。

我仍然认为 iPhone 将会是一场灾难。
