雅虎推出Messenger浏览器版本——TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com:80/2007/05/02/yahoo-launches-browser-version-of-messenger/amp/

雅虎推出信使浏览器版本

eBuddy 和 Meebo 明确表示，通过浏览器进行即时通讯的需求很大——许多用户在无法下载软件的电脑上工作，因此基于网络的即时通讯是他们使用该服务的唯一方式。

微软有自己的网络信使，谷歌的Gtalk被整合到Gmail和其他资产中。雅虎之前也尝试过网络聊天，将聊天整合到雅虎邮箱网站中。

今天晚上，雅虎将在webmessenger.yahoo.com推出一个专门的雅虎聊天网站。它应该在太平洋标准时间下午6点左右直播。稍后他们还会在这里发布一段服务的视频。

与大多数其他基于网络的聊天服务不同，Yahoo Web Messenger是建立在Flash 9之上的。在发布时，它将只包括基本的功能，并与MSN Messenger集成(您可以将Live/MSN Messenger联系人添加到您的雅虎朋友列表中)。用户也可以添加头像和表情符号的支持。雅虎表示，以后可能会增加VOIP功能。
