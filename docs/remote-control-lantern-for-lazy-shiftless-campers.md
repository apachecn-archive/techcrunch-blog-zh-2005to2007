# 懒懒散散的露营者的遥控灯笼

我们记得在我们祖父母家发现的一本 60 年代后期的科普读物，内容是关于“未来的露营”。他们的大多数预测都没有实现:当然，我们有自己组装的帐篷，但我们不吃“食物药丸”而不是 trail mix，我的超声波熊击退火箭瑞士军腰带肯定在邮件中丢失了。

在 60 年代，他们没有预料到的一件事是，我们现在非常懒。拿着这个遥控个人大小的灯笼。它很像你带去露营的任何灯笼，因为它由电池供电，并配有一个荧光灯泡，但它还配有一个遥控器，所以你甚至不必拉开你的太空时代睡袋来睡觉。

确实很艰苦。科尔曼出 26 美元。
