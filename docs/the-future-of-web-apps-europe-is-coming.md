# 欧洲网络应用的未来正在到来。

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/06/the-future-of-web-apps-europe-is-coming/>

今天，我很荣幸地与赖安·卡森谈论了“2007 年网络应用的未来”，这个项目目前正在筹划中！任何有幸去过最近在伦敦[和 T4](https://web.archive.org/web/20160202195943/http://www.carsonworkshops.com/summit/2006/london/index.html)[举办的电影节或者最近在三藩市](https://web.archive.org/web/20160202195943/http://www.carsonworkshops.com/summit/)举办的电影节的人都会告诉你，这些电影节绝对是顶级的，不容错过。Ryan 和我讨论了一些将在今年的活动中发言的演讲者，也讨论了我们可能希望看到发言的其他人。我提名本·哈默斯利作为我想听到的人选。如果有一个好的演讲者，你可能想向 Ryan 求婚，你能在这篇博客上留下你的评论吗？
