# 三星抛弃 WinMo，Symbian，Linux，转而支持家酿智能手机操作系统！？

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/26/samsung-to-dump-winmo-symbian-linux-in-favor-of-home-brew-smartphone-os/>

好吧，我们对此有所保留，但只是一点点。我们的小鸟，谁是智能手机操作系统的知情者，给我们发来了一个关于三星的想法的提示，这家公司负责全国许多功能丰富但纤薄的手机，正在开发一些全新的东西，非常棒。这位消息人士如是说，他接着断言三星将在即将火热的智能手机市场独辟蹊径。

真正有趣的是，消息人士声称三星将为其未来的智能手机抛弃 Windows Mobile、Symbian *和* Linux。那么它要用什么呢？自己的操作系统。和一个定制的双核处理器，带有片上 Java。更重要的是，它计划开发基于 MPEG 的引擎、内置 3D 加速和原生 Mp4 编码和解码的移动操作系统。雄心勃勃的任务！

当然，任何网络谣言都需要被视为谣言。但话虽如此，我们的线人从未让我们失望，所以我们有充分的理由相信这是萨米的新策略。
