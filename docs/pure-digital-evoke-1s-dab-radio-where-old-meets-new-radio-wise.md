# 纯数字唤起 1S DAB 电台:旧与新的相遇(电台明智)

[![puredigitalevoke1s.jpg](img/91123875b00e867b7d059a1f5fb3beb0.png)](https://web.archive.org/web/20210301132000/http://old.crunchgear.com/wp-content/uploads/puredigitalevoke1s.jpg "puredigitalevoke1s.jpg")

早上在这里写博客，我要么听豪斯医生和/或 trance 混音，要么听电台访谈，无论是 NPR 还是诺顿和朋友秀。如果我住在一个 DAB 友好的地区，我可能会使用类似纯数字的唤起-1。不是因为它是“革命性的”或任何东西，而是因为它看起来像是直接从 1954 年出来的。不过，这并不是说收音机只是养眼的东西。唤起有必要的输入连接一个 DAP，和纯数字投入调频收音机的良好措施。

唤起的两行，有机发光二极管显示器显示什么是目前正在播放(无论是 DAB 或 RDFified 调频收音机)和它的木材外观的情况？对，实木！我都不记得我拥有的最后一台收音机了，更别说是木制的了。这台收音机还带有 EcoPlus 荣誉徽章，让你知道你每买一台收音机，你就帮助补贴了戈尔私人飞机的燃料费。

这款老收音机将于 9 月上市，售价约为 200 美元。
