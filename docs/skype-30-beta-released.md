# Skype 3.0 测试版发布

Skype 进行了彻底的改造，现在可以在 Windows beta 中为那些有此倾向的人使用。新功能包括直接访问 SkypeCasts，这很酷，如果，嗯，在这个节骨眼上没有用，以及一些用户界面的调整和文件共享的额外功能。

![](img/cb39b2596d2e6b85c8526bb894798f00.png)

你也可以创建公共聊天室，邀请全世界的人和你谈论任何话题。它附带禁止为巨魔提供服务。Mac 版本还没有消息，但请放心，他们正在努力。
