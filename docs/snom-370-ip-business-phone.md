# snom 370 IP 商务电话

VoIP 现在炙手可热是有原因的。它通过允许高带宽语音通信在互联网上进行，而不是通过具有额外费用的电信路由，从而大大降低了电话成本。这项技术对每个人都有意义。

总部设在柏林的 snom technology AG 公司开发了一种 VoIP 电话，允许企业优化其 IP 通信。snom 370 具有大屏幕、扩展内存和 VPN 接入，因此企业用户可以简单安全地管理呼叫。

用户可以定制显示器，以显示高分辨率的公司图形和图片。370 还同时支持几种音频设备，如听筒、耳机和扬声器——我不明白为什么有人会同时使用这三种设备。

无论如何，370 将于 4 月上市，售价 280 美元。它与基于 SIP 的电话系统和组件兼容。
