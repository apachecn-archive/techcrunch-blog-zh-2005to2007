# 沃达丰，谷歌地图合作伙伴

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/12/vodafone-google-partner-for-maps/>

![](img/cf42aa24b08e8e9cac06b1ab48089372.png)

沃达丰的客户可能会从谷歌获得非常受欢迎的升级。今天，沃达丰宣布与谷歌合作，为这家电信提供商开发地图服务。根据协议，沃达丰的谷歌地图将拥有易于使用的地图、本地列表、导航和搜索功能。

沃达丰的 Frank Rovekamp 说:

> 沃达丰将谷歌地图视为两家公司正在发展的关系中的一个新的里程碑。这是沃达丰如何通过其定位服务和分销规模来补充谷歌产品并打开令人兴奋的服务的大众市场的一个主要例子。
