# 东芝 Gigabeat U101 和 U201 DAPs 直接编码 MP3

![](img/6dfd0efcb14322a48c6dee39f2ea981f.png)

是的，我们都知道 iPod 是无可争议的数码产品之王，但这并不意味着公司不应该被允许向 R&D 投入大量资金试图取代它。U101 和 U201 这些微型 Gigabeats 与其他“me，too”的区别在于它能够从任何音频源直接编码为 MP3 格式。方便，是的，但是很难称之为实用。除此之外，Gigabeats 只提供最基本的功能:1 或 2GB 的闪存、FM 调谐器和发射器以及 1.1 英寸的屏幕。我认为苹果不应该太担心这些小玩家。
