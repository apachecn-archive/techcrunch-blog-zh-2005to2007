联邦通信委员会需要倾听谷歌TechCrunch

原文：https://web.archive.org/web/https://techcrunch.com/2007/07/22/the-fcc-needs-to-listen-to-google/amp/

我很高兴看到谷歌公开介入明年年初即将到来的700MHz频谱的FCC拍卖。如果一切顺利，新的频谱可以用来创建一个新的开放接入无线宽带“管道”，进入人们的家庭和设备。如果事情进展不顺利，现有的无线巨头将购买频谱，并对当今美国的蜂窝网络实施类似的使用限制，使我们进一步落后于欧洲和亚洲。

首席执行官Eric Schmidt向FCC主席凯文·帕丁发送了一封信，声明如果四个关键平台规则被采纳，他们将承诺在拍卖中出价至少46亿美元。这些规则将定义获胜者可以提供的服务类型，并要求第三方访问带宽:

  1. 开放应用程序:消费者应该能够下载和利用他们想要的任何软件应用程序、内容或服务；
  2. 开放式设备:消费者应该能够利用手持通信设备和他们喜欢的任何无线网络；
  3. 开放服务:第三方(经销商)应该能够根据合理的非歧视性商业条款，以批发方式从700 MHz许可证持有者处获得无线服务；和
  4. 开放网络:第三方(如互联网服务提供商)应能够在700 MHz许可证持有者的无线网络中的任何技术可行点进行互连。

鉴于美国移动领域的现状，我希望联邦通信委员会能够毫不犹豫地采纳这些保护消费者的规则。但是包括AT T和威瑞森在内的现有玩家表示，他们反对开放存取，如果这些规则被采纳，他们可能不会参与。谷歌的公开举动是为了让联邦通信委员会(和公众)知道，有公司非常乐意在一个开放接入的世界里竞标。

美国电话电报公司对谷歌信件的回应是惊人的大胆:

董事长马丁提出的妥协方案满足了谷歌的大部分条件，但谷歌对这个方案不满意，因此它向美国政府发出了“要么全有，要么全无”的最后通牒，坚持要求美国政府“必须”满足所有条件，否则谷歌将不会参与频谱拍卖。谷歌要求政府为其提供更多有利条件，限制竞争性投标，并切实迫使无线运营商按照谷歌的喜好调整业务模式。我们要重申的是，谷歌要么闭嘴，要么闭嘴——他们可以通过自己喜欢的任何商业模式竞标进入无线市场，然后让消费者决定自己最喜欢哪种模式。

对于那些不太关注这个问题的人来说，美国电话电报公司的回答似乎非常合理:不要让政府监管插手，让市场来决定哪些服务会赢。但实际情况根本不会是这样。如果政府对带宽的限制减少，中标者就能以牺牲竞争对手和消费者的利益为代价来获取更多利润。因此，他们自然不希望看到类似谷歌(Google)推荐的开放存取规则。现有巨头们也不希望看到谷歌钻他们的空子，公开叫板——所以他们还有另一个理由反对自己的提议。

公平竞争委员会的竞争目标是从拍卖中获得最大收益(建议减少监管)和保护公众(建议制定更多规则来迫使竞争)。谷歌提出的开放访问要求将刺激这一领域的竞争，推动经济增长。它还将对移动运营商和宽带公司施加商业压力，要求他们减少对现有宽带和移动服务的限制。

谷歌并不总是那么邪恶，但在这种情况下，面对一些在提供消费产品方面历史相当糟糕的公司，他们会为我们所有人提供打击。在这件事上我支持他们。对联邦通信委员会(FCC)说:请从过去的错误中吸取教训，这次不要理会那些游说者，做些符合公众最佳利益的事。
