# 三星 L74 Wide 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/30/samsung-l74-wide/>

![](img/4fa0bba9200dd16c1416f4b305d98bd6.png)
三星的相机部门最近比较忙。昨天我们看到了 NV11 和 i7，今天我们看到了 L74 Wide。这款 700 万像素的相机在其 28 毫米广角镜头上配备了 3.6 倍光学变焦。它配有一个 3 英寸的触摸屏 LCD，通过它可以控制大多数设备功能。

与三星其他新相机一样，它具有面部识别自动对焦和自动曝光功能。它还具有三星花哨的 ASR 图像稳定功能，这应该有助于提高 iso。

![](img/ee2f68a29047a07a1ba2ca18e689af00.png)
有趣的是，L74 Wide 有导游功能，提供 30 个国家 4500 个地区的旅游信息。它配备了 450 兆内存，用户可以存储所有必要的旅行信息。

L74 Wide 将于今年春季晚些时候上市，售价 350 美元。
