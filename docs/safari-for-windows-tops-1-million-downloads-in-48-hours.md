# Safari For Windows 在 48 小时内下载量突破 100 万次

[![safwinmil.jpg](img/a8e9010e774d49cb280142337e4d372d.png)](https://web.archive.org/web/20201130055021/https://beta.techcrunch.com/wp-content/uploads/2007/06/safwinmil.jpg "safwinmil.jpg")

尽管有[问题](https://web.archive.org/web/20201130055021/http://crunchgear.com/2007/06/14/safari-for-windows-updated-to-301-fixes-exploits/)，在 Safari 浏览器推出的前 48 小时内，超过 100 万人下载了它。苹果声称这款浏览器是世界上最快的，但我在网上看过一些报告显示并非如此。(此外，自从它最初发布以来，在 Mac OS X 上使用过它，你肯定会觉得它有些“不对劲”，比如它缺少*某些东西*。而且在我的系统上运行起来好像比 Firefox 还慢；我只关心这个。)

因此，祝贺苹果公司的这一里程碑。在与 Firefox 和 Internet Explorer 的竞争中，仍有一些地方需要弥补，一些 Windows 用户似乎在积极反对苹果(为什么这么讨厌？这是一个愚蠢的公司)，但这仍然是一个良好的开端。
