# 治疗情绪照明

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/22/therapie-mood-lighting/>

我们都知道光线会影响人的情绪。让你的房间始终像一个潮湿的地窖，你很可能会变成一些邪恶的迪克·切尼式的人物。大量的阳光和诸如此类的东西，很可能你会经历一个更加积极的蜕变。

对于那些被困在阴湿的地窖里，但不想以邪恶告终的人来说，也许一些情绪照明是有序的。除了让你的环境适合 70 年代的摇摆派对，这款由设计师 Andre Keilani 设计的帆布壁灯还遵循光疗法理论，让路人沐浴在健康的光线中，据说可以逆转坏脾气。

Therapie 有一堆双色调的醋酸盐。1100 美元就能搞定。我的建议？弄个 [Mathmos](https://web.archive.org/web/20210115185008/http://www.mathmos.com/) 就行了。
