# Widgetbox 远程图库:一个开放的微件平台

[![](img/9e4200cc0a3347d785100fb42370d3c2.png) ](www.widgetbox.com) [Widgetbox](https://web.archive.org/web/20180713114313/http://www.widgetbox.com/) 宣布广泛采用其新的 Widgetbox 远程画廊功能，该功能可以简化向社交网络开放平台的过渡。

Widgetbox 远程图库功能使社交网络能够嵌入来自第三方开发人员的选定数量的微件，并提供对 Widgetbox 主图库中超过 10，000 个微件的访问，使其用户能够轻松找到并使用微件来定制他们的个人资料、博客和网页。

合作伙伴可以创建自己的 Widgetbox 远程图库，并控制 Widgetbox 远程图库中的小部件选择。画廊可大可小，小部件的选择可以根据需要旋转。

发布合作伙伴包括 Freewebs、imbee.com、Xanga 和 [Six Apart 的](https://web.archive.org/web/20180713114313/https://www.crunchbase.com/organization/six-apart/) TypePad、LiveJournal 和 Vox。
