# 你应该垂涎的 iPhone 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/29/the-iphone-you-should-lust-over/>

我将再次承认，当 iPhone 第一次出现时，我被它迷住了，但后来我知道了硅谷的小苹果掌握的所有[肮脏的小秘密](https://web.archive.org/web/20160305042140/http://crunchgear.com/2007/01/15/multi-touch-technology-and-where-its-going-nextoh-and-apple-didnt-invent-it-either/)。我开始抬起鼻子，对任何提到它的人发出难闻的气味。当然，它有全屏触摸板和所有这些功能，但它缺乏 3G 功能，而且在 orange guys (AT & T/Cingular)网络上发布应该是显而易见的。但是我从这个庞大的共产主义国家的地平线上看到了什么？

魅族 M8 与 iPhone 有着惊人的相似之处，据称配备了 3.3 英寸 720×480 显示屏，白色外壳尺寸为 57×105×11.5 毫米，包装有 GSM 和中国 TD-SCDMA 3G 收音机、蓝牙、300 万像素摄像头和能够以 30fps 的全分辨率录制视频的 ARM11 CPU。这是真的还是假的还有待观察，但你可以打赌，即使苹果的法律团队一直在努力工作，中国的克隆产品也将在 iPhone 在亚洲首次亮相之前很久发布。
