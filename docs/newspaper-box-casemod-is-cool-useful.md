# 报纸盒 Casemod 很酷，很有用

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/14/newspaper-box-casemod-is-cool-useful/>

凯斯莫德人来了又走，我通常不在乎他们。当然，它们很整洁，你的电脑真的是用乐高积木做的。太棒了。但有时我会遇到一个简洁的*和*功能，这引起了我的注意。

看看用户 stwalker 设置的 flickr，他用一个废弃的报纸盒、一个 17 英寸的 LCD、一个 Mac Mini 和一个小 AppleScript，把它们放在一起，得到了上面的结果。它获取标题(大概是通过 RSS)并将它们格式化成报纸的头版。我想这么做。太棒了。
