# Newsvine 是完美的 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com/2006/03/03/newsvine-is-perfect/>

 [](https://web.archive.org/web/20221130193939/http://www.newsvine.com/) [Newsvine](https://web.archive.org/web/20221130193939/http://www.newsvine.com/) 昨天解除了其网站的密码保护，[正式上线](https://web.archive.org/web/20221130193939/http://www.mikeindustries.com/blog/archive/2006/03/newsvine-launches)(我在 Newsvine 上的首发帖子是[这里是](https://web.archive.org/web/20221130193939/http://www.beta.techcrunch.com/2005/11/09/newvine-to-enter-social-news-ranks/))。

Newsvine 结合了 Digg 和 Google News 等许多公司和产品的最佳功能，以及聊天和博客式评论等强大功能。结果是一个完美的新闻网站(注意，像 spotback(这里提到的)这样的新竞争对手已经瞄准了推翻 Newsvine)。

结果是一次真正精彩的社会新闻体验。他们还计划与新闻撰稿人进行收入分成(我们将拭目以待)。

祝贺 Mike Davidson 和 Newsvine 的团队。现在，我至少每天都会访问这个网站。

仅供参考，在今天的吉尔摩帮中，Newsvine 被提到了很多次。
