# Earthlink 什么时候变得这么酷了？

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/13/when-did-earthlink-get-so-cool/>

Earthlink 不会让人联想到尖端的新网络应用。事实上，当我想到 Earthlink 时，首先想到的是我的旧拨号 ISP 帐户，我花了很长时间才取消它。我不止一次地问过史蒂夫·吉尔摩，他自诩 Earthlink 是他有影响力的技术秀[吉尔摩帮](https://web.archive.org/web/20220516091252/http://gillmorgang.podshow.com/)的赞助商，他的听众(早期采用者类型)中是否有人真的会使用 Earthlink 的产品。

在过去的几个月里，有一些值得注意的 Earthlink 新闻——他们[将 Pandora](https://web.archive.org/web/20220516091252/http://blogs.earthlink.net/2006/07/pandora_comes_to_myearthlink_1.php) (一种[新的网络音乐服务](https://web.archive.org/web/20220516091252/http://www.beta.techcrunch.com/tag/pandora))整合到他们古老的门户网站中，他们[正在西雅图和旧金山测试零售店](https://web.archive.org/web/20220516091252/http://blogs.earthlink.net/2006/06/rosemary_from_the_earthlink_to_1.php)。他们还通过戴夫·库斯坦写的博客与世界交流。所有的好迹象。

今天我们收获更多。Earthlink 突然推出了 Bloglines 风格的 RSS 阅读器和 del.icio.us 风格的社交书签网站。史蒂夫鲁贝尔注意到了他推荐人日志中的网站，并查看了一下。

两者都不是最前沿的，但他们在抓住大量观众可能想要的重要功能方面做得很好。更重要的是，服务很快。

[RSS 阅读器](https://web.archive.org/web/20220516091252/http://reader.earthlink.net/)允许 OPML 进口，并有很好的推荐内容选择。它没有罗约和其他人即将推出的高级功能，如标签、基于阅读习惯的内容优先排序等。([查看这篇概述](https://web.archive.org/web/20220516091252/http://www.beta.techcrunch.com/2006/03/30/the-state-of-online-feed-readers/)了解其他在线 RSS 阅读器)，但它确实很好地涵盖了基础知识。它还能近乎完美地呈现 HTML 和 CSS，这是我定期向抱怨的事情。

像 del.icio.us 和其他网站一样，[书签网站](https://web.archive.org/web/20220516091252/http://myfavorites.earthlink.net/)有一个方便书签的浏览器按钮，并允许标记。界面是干净的。

这些发布最让我兴奋的是，大量新观众将接触到新的 Web 2.0 理念。每个月大约有 1200 万人访问 Earthlink，产生近 10 亿的页面浏览量(仅 MyEarthlink 就有大约 200 万个唯一的和 1.15 亿的页面浏览量)(来源:康姆斯克)。
