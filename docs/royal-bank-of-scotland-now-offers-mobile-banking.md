# 苏格兰皇家银行现在提供手机银行服务

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/08/14/royal-bank-of-scotland-now-offers-mobile-banking/>

本周，苏格兰皇家银行集团宣布推出了一项新的手机银行服务，苏格兰皇家银行和 NatWest 的客户将可以在任何有信号的地方使用手机追踪他们的账户。这项服务由 MONILINK 提供，moni link 是 VocaLINK 和 Monitise plc 的合资企业。客户可以注册这项服务，这样他们就可以通过 WAP 浏览器查看账户余额和获取迷你报表。

苏格兰皇家银行计划增加包括移动支付和账户转账在内的其他服务。目前，用户还可以给任何移动预付费电话充值，并接收短信余额提醒。这项服务是安全的，因为数据实际上并不在手机上，而只是信息的显示。使用服务和软件是免费的，没有月费或年费，尽管一些运营商可能会定期收取数据费用。
