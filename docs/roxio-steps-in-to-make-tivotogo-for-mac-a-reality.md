# Roxio 的介入让 Mac 版 TiVoToGo 成为现实

你可能已经读到过这些，但是现在有一种合法的方法可以在运行 Mac OS X 的电脑上获得 TiVoToGo。Mac 用户现在可以体验到将录制的视频长时间传输到笔记本电脑和台式机的痛苦，只有 Windows 用户才能做到这一点。虽然这都不是萝西的错。

Roxio 通过 MacWorld 发布的 Toast 8 插件，从头到尾出色地完成了整个流程。传输速度是最快的(通过无线网络传输一个 1GB 的文件大约需要 30 分钟)，幸运的是，Roxio 很快就完成了文件转码，使其与 iPod 或 PSP 兼容。

这个故事的寓意是，如果你要传输它，请以较低的质量录制。我将对 Toast 8 的到来进行全面的回顾，但现在每个有 Mac 的人都应该把新版本视为必备应用。

当然，TiVo 用户将获得额外的好处，能够将内容从 TiVo Series 2 单元转移到他们的计算机上(对不起，Series 3 的所有者，看起来你仍然被冷落)，但还有一些其他漂亮的功能，包括界面(比 7 大，漂亮的改进)和一些不错的功能，包括我个人最喜欢的，可以在 Mac 或 Windows 系统上使用的光盘跨越(跨多张光盘刻录内容)。对于双操作系统家庭或企业来说，这是一个巨大的优势。售价 99 美元，价格也很合理。
