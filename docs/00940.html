雅虎团队与活字

雅虎和六合已经合作在雅虎的小企业托管上预装可移动活字。雅虎的主机用户现在可以预装活字。这应该有助于雅虎移动更多的主机单位，并且肯定会扩大Six Apart的竞争范围。这对两家公司来说都是一笔好交易，尤其是六家公司。

在真空中，这条新闻相当无聊。

但令人兴奋的是，雅虎现在对消费者市场的攻击是多么咄咄逼人。他们制造 bold 真正能引起消费者共鸣的 新产品，购买同类最佳的 web 2.0公司，以及与他人合作。

他们以一种好人的方式做着这一切。高管们开始接纳并倾听博客作者和其他人的意见，90年代末的傲慢完全消失了。雅虎现在正在各条战线上点击。谷歌的傲慢和他们“去他妈的客户”的心态是他们最大的弱点，而雅虎正在充分利用这一点。成功和谦逊是非常强大的组合。
