# 在线日历大战——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/01/04/online-calendar-wiars/>

最近在[谷歌日历](https://web.archive.org/web/20220629012705/http://calendar.google.com/)上的 Hitwise 流量[数据](https://web.archive.org/web/20220629012705/http://weblogs.hitwise.com/leeann-prescott/2007/01/google_calendar_up_threefold_s_1.html)出来了，它显示了相对于来自雅虎和微软的竞争产品的稳定增长。现在这三个都差不多大了，虽然只有谷歌日历指向了正确的方向。

在经历了太多的同步问题后，我最近放弃了使用 Mac 日历。我仍然从 Mac 桌面应用程序查看我的日历，但所有数据都存储在 Google 上，读写权限与团队共享。看到它受到雅虎和微软的攻击，我并不感到惊讶——这些家伙最近没有把太多注意力放在自己的产品上。

正如我们几周前在[用谷歌博客搜索](https://web.archive.org/web/20220629012705/http://www.beta.techcrunch.com/2006/12/28/google-v-technorati-and-hitwise-v-comscore/)所做的那样，我们也看了一下这三种产品的最新康姆斯克数据，看看他们有什么要说的。我很惊讶地说，康姆斯克显示出与 Hitwise 相同的趋势和整体排名，这似乎很少发生。康姆斯克仍然显示雅虎稳固领先，但迅速下滑。
