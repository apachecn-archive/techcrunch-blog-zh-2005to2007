# 星巴克的床 iTunes，出生邪恶的三头野兽

![](img/0503ca4bf5834b8b17c520de570793f6.png)

所以我现在坐在芝加哥。加文的网络连接毫无价值，所以我一整天都呆在星巴克，听着有史以来最可怕的音乐(除了现在他们碰巧在播放十二月党，这是他们六个小时以来播放的第一首好歌)。

无论如何，我相信很多人走进星巴克，心里会想，“我怎么会听到我在喝低脂奶油、无糖和婴儿呕吐物的冰摩卡摩卡时听到的那首可怕的歌？”如果你是其中之一，你很幸运。

星巴克宣布与苹果达成协议，将在 iTunes 上发布其 [Hear Music](https://web.archive.org/web/20210412223619/http://phobos.apple.com/WebObjects/MZStore.woa/wa/viewGrouping?id=41) 目录。如果我没有消除噪音的耳塞，我可能会用冰锥敲我的耳膜。
