# SMSJock:你的音乐节目主持人关系

那些音乐节目主持人现在有麻烦了！一项新技术已经发布，这将使听众能够直接向音乐节目主持人发送信息，当然只收取象征性的费用。现在听众可以纠缠不休地让骑师演奏他们想要的任何曲子。利用任何具有短信功能的手机，听众可以使用这种基于网络的服务，不幸的是，这种服务叫做[短信](https://web.archive.org/web/20210225063249/http://www.smsjock.com/index.php)，直接给音乐节目主持人发短信。同样，这也是一个通信枢纽，因为音乐节目主持人可以通过这项服务向听众的移动设备进行广播。
