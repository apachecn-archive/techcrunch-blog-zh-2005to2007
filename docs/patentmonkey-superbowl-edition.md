# PatentMonkey:超级碗版

![](img/867fe8500762170dfcfa0a56908a4671.png)我们在超级碗周末有一些创新的想法:

**火箭激光武器检测器**

网络显示棒球投掷速度。我建议足球比赛中也开始展示 T4 的投掷速度。在如此强调臂力的情况下，让我们让技术满足所有关于谁有臂力，谁没有的主观评论。

他们为什么不直接用激光呢？

是的，在球场上带上锁链增加了比赛的戏剧性，但是真的，必须有一个更有效的方法来判断这是不是第一次摔倒。Normal Harty [发明了](https://web.archive.org/web/20230322164148/http://www.patentmonkey.com/PM/PatentID/6851198.aspx)一种小巧的基于激光的第一羽绒检测器。也许他们会把“把锁链拿出来”改成“他们在球场上发射激光”。

**美泰经典足球，“旧”是新的“新”**

虽然我会在 YouTube 上跟踪“最受欢迎的商业广告”的[投票](https://web.archive.org/web/20230322164148/http://www.youtube.com/superbowl)，并在周日的自动游戏中多次运行我的任天堂 Wii [Madden Football](https://web.archive.org/web/20230322164148/http://www.easports.com/madden07/) (看看熊队是否有任何获胜的机会)，但我也会用一个旧的最受欢迎的游戏进行反击，我的[美泰经典足球游戏](https://web.archive.org/web/20230322164148/http://www.handheldmuseum.com/Mattel/ClassicFB.htm)——这项技术作为[专利](https://web.archive.org/web/20230322164148/http://www.patentmonkey.com/PM/PatentID/4162792.aspx)免费供任何人使用，现在已经过期。当然，它缺少细节，但是你不记得躲避那些小光点有多棒了吗？我当然知道。
