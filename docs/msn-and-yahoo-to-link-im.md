# MSN 和雅虎联手 IM TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/10/11/msn-and-yahoo-to-link-im/>

有传言说，微软和雅虎明天将宣布他们的即时消息客户端将实现互操作。催化剂似乎是 Skype 和 Google Talk 使用量的激增。MSN 和雅虎共占如今即时通讯总使用量的 44%(不包括 Skype 和 Google)，每月独立用户达到约 3350 万。

更新:这个传闻今天早上通过[雅虎新闻发布会](https://web.archive.org/web/20221129064627/http://docs.yahoo.com/docs/pr/release1265.html)得到了证实。这两家公司宣布，其 2.75 亿联合客户的互操作性将在 2006 年 Q2 会议前后实现。嘿，现在怎么样？:-)
