# T-Mobile 提前加入黑莓珍珠

我们喜欢 T-Mobile。他们很友好，最近他们在与 Cingular 在酷派手机领域达成平等方面取得了很大进展。今天，他们又迈出了一步，将黑莓珍珠添加到他们的网站上作为“现已上市”。

他们已经接受预订有一段时间了，并开始在店内销售，但这是第一次你可以从他们的网站上订购现货，并在一夜之间发货。

我们已经在这里咀嚼珍珠[有一段时间了](https://web.archive.org/web/20130627204830/http://crunchgear.com/?s=pearl)，我们喜欢它。这是一款小巧纤薄的智能手机，配有 130 万像素的摄像头、蓝牙和 RIM 的推送电子邮件解决方案。它使用熟悉的 SureType 混合键盘，因此当前 CrackBerry 上瘾者的迁移应该是无痛的。很高兴看到 T-Mobile 提前放弃了一些东西。我们觉得今天是我们的生日什么的。
