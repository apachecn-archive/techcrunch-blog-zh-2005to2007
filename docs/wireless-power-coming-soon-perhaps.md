# 无线电源即将推出？或许

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/15/wireless-power-coming-soon-perhaps/>

![](img/05facd922c2ea96f98cf16e56a1acb2b.png)

我想在这一点上，每个人都会喜欢无线电力。被称为我的“工作区”的火灾隐患非常渴望无线上网，腾出一些电源插座。幸运的是，这可能比我想象的要快。美国研究人员已经开始开发和研究一种无线供电的方法。他们借鉴了尼古拉·特斯拉等人的新旧技术，而答案其实就在物理和共振之中。共鸣通常可以在乐器中听到，是无线传输能量的关键。

科学家说，“如果你让另一个具有相同频率的共振物体足够靠近这些尾巴，那么事实证明能量可以从一个物体隧穿到另一个物体。”

不过，现在还不要扔掉你的电源板。发展处于初级阶段，还有很长的路要走。
