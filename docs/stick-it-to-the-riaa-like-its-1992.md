# 像 1992 年一样贴在 RIAA 上

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/18/stick-it-to-the-riaa-like-its-1992/>

早在 90 年代，我就加入了几个“受欢迎”的本地独立摇滚乐队。那是在数字世界模拟录音的时代，只有高端同行才有 CD-R 功能，而且这些功能大多用于数据目的。因此，我最喜欢的吉他音乐和关于女孩的悲伤歌曲都在模拟磁带上。

谢天谢地，我碰到了 Plus Deck 2c，这是一款适用于您电脑的模拟盒式磁带驱动器。它可以安装在你计算机的任何 5.25 英寸驱动器上，并且有合适的电缆连接到你的声卡上。外面的控制都是硬件，这里没有什么真正的高科技，但这是为什么它很棒的一部分。

简而言之，你可以通过电脑的音频子系统播放磁带，这意味着你可以将磁带直接数字化成数字副本，然后你可以对其进行数字版权管理并出售，每盘赚几分钱。不过，我们希望它包括数据服务。你知道，把你的硬盘备份到磁带上会很有意思。

从 CD 中翻录是本世纪的*所以*。在 2007 年，我们要脱离模拟，宝贝。模拟。
