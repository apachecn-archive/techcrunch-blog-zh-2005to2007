# 这就是 Vista 支持 Halo 2 的原因

![](img/2ae5af5b0750c0da0140fc482c0d9118.png)

想要 Vista 版的 Halo 2？[ESRB](https://web.archive.org/web/20210307083302/http://crunchgear.com/2007/05/24/halo-2-for-vista-delayed/)不想让你拥有它，跳跃后的图像就是为什么。准备哭吧。

 ![untitleddf7uv9.png](img/c5185e0fbf89c348a9b8bf32b0eb2012.png)

> “在 Vista 版《光环 2》发布 M(成熟 17 岁以上)分级分配后，就在该游戏即将上市之前，微软通知 ESRB，在游戏附带的地图编辑器工具中发现了相关内容。尽管所讨论的内容可能对于绝大多数用户来说是不可访问的，但是当特定错误发生时，该内容向用户显示个人的裸露背部的照片，并且因此保证“部分裸露”内容描述符，以便警告消费者其存在于产品中。因此，微软在将运往美国和加拿大零售的几乎所有副本的包装上都贴上了正确的 ESRB 分级信息标签。微软已经通知我们，未来的游戏运行将没有问题的内容，因此不需要在这些版本上显示描述符。
> 
> “ESRB 的工作是确保游戏得到适当的标签，这正是我们在这个案例中所做的。我们非常感谢微软在这件事上的合作。”–娱乐软件评级委员会
