# Sprint 推出首个社交地图服务 loopt

你的朋友在躲着你吗？他们真的没有你就在一起了吗？当 Sprint 宣布 loopt 时，他们可能没有想到跟踪，但这项在美国主要运营商中首次推出的社交地图服务将帮助你知道你是否被拒绝。更重要的是，它将有助于确保你不会错过你的朋友，因为该服务使用特定 Sprint 手机上的 GPS 功能来促进朋友之间的真实世界互动。您现在可以共享您的位置，发送基于邻近度的消息并查看地理标记的照片。好吧，所以这不是一种跟踪你的朋友的方法，但确保你们不会错过彼此是一个很好的功能。

“世界上最常见的手机问题是‘你在哪里？’我们很高兴 loopt 能够为选择参与的 Sprint 客户回答这个问题。我们在手机上的交流方式将永远改变，因为 Sprint 上的 loopt 结束了错过的连接，并促进了现实世界的互动。"

这项服务可以自动更新用户私人朋友网络中每个人的位置，并将信息直接显示在手机地图上。loopt 还可以提醒你何时有朋友在附近。这项服务是 100%基于许可的，并且这些数据只能在私有网络中获得。所以你不会报名让老大哥看你，即使一开始听起来是这样。它将在超过 25 款 Sprint 和 Nextel 手机上提供，每月 2.99 美元，外加标准数据费用。
