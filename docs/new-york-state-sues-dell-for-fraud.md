# 纽约州起诉戴尔欺诈

[![dellsued.jpg](img/e91c360716197f064fbe52c8eea8058d.png)](https://web.archive.org/web/20210411014049/https://beta.techcrunch.com/wp-content/uploads/2007/05/dellsued.jpg "dellsued.jpg")

这是周三的坏消息，最近遭受打击的公司是戴尔。纽约爱管闲事的司法部长安德鲁·科莫，你我的朋友，正以欺诈罪起诉电脑制造商。该诉讼将在今天晚些时候的新闻发布会上解决，指控戴尔误导其客户，让他们认为他们获得了低融资，而事实上它在购买电脑时应用了高信用。所以，说一套做一套。该州还将证明戴尔的客户服务、保修和折扣并不像宣传的那样好。

我从未买过或拥有过戴尔电脑，但这些年来我肯定听过所有客户服务的恐怖故事。是朝着正确方向迈出的一步，还是一个只想扬名立万的人？
