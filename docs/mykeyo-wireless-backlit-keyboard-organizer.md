# myKeyO 无线背光键盘管理器

myKeyO 键盘管理器获得了 Web 2.0 更新，带有掌托、背光和无线连接。它仍然在零售那些像中学课桌一样的小储物格，用来放你的铅笔、钥匙和剪刀；但不像旧的，不像是 1995 年的。

好消息是，它只比普通键盘厚 1/10 英寸，但价格差不多(有线版 34 美元，带鼠标的无线版 49 美元)。他们现在接受秋季送货的订单。
