# 试用 Pinger

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/08/14/trying-out-pinger/>

上周，我会见了苹果公司(T4)的创始人 Greg Woock 和 Joe Sipher，观看了他们今年晚些时候发布的新移动产品的演示。

这是一家严肃的公司，在移动语音信息领域有着非常简单的病毒式产品。Pinger 当时被称为 Project Edgar，2005 年 11 月，它从凯鹏华盈(Kleiner Perkins)筹集了 300 万美元，凯鹏华盈合伙人兰迪·科米萨尔(Randy Komisar)加入了董事会。

两位创始人都来自 Handspring (Greg 是销售副总裁，Joe 是营销副总裁),他们还在维珍一起合作了一个从未启动的项目。

关于这款产品，我们还不能说太多，但我可以说这是我见过的为数不多的我会每天使用的移动应用之一。私有测试版比我们测试过的大多数非测试版服务更加保守。一旦公司批准，我们会尽快发布更多消息。在 [Pinger](https://web.archive.org/web/20211024201524/http://www.pinger.com/) 主页上注册以获得发布通知。
