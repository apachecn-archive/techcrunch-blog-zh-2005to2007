# 平流层宽带

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/03/stratospheric-broadband/>

由约克大学牵头，由欧盟 T2 框架计划资助的,[*【CAPANINA】*](https://web.archive.org/web/20150911145934/http://www.capanina.org/)*项目旨在推动无线宽带高智能平台(HAPs)的发展。对于外行人来说，hap 本质上是系留气球和飞艇，提供覆盖大城市和农村地区的宽带；它是 WiMAX 和 3G 的空中竞争对手，也有可能弥合卫星和地面无线宽带之间的差距。*

 *该大学最近在约克举行的为期一周的会议上分享了该项目的首批成果。CAPANINA 的目标是开发高达 120Mbit/s 的网络速度，理论上可以解决英国数字基础设施的许多限制因素，如通过 DSL 和有线网络分配高清电视。然而，就像 WiMAX 和 3G 一样，很可能只有运营商和现有公司可以进行必要的投资来利用这种发明…除了英国电信，他们不太可能在 3G 内爆后进行大规模的基础设施投资。

尽管 HAP 技术仍处于早期发展阶段，但令人欣慰的是，伦敦是互联网创新和商业的近期中心，而英国其他地区也在对新兴技术和学术合作关系进行长期投资。
