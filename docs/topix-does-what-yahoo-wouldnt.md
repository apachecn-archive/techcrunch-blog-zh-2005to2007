# Topix 做了雅虎不会做的事情

今天，Topix.net 新闻引擎增加了 15000 个热门博客。

雅虎采取了将博客搜索结果整合到新闻区的步骤，但[将它们推到了](https://web.archive.org/web/20221129071730/http://www.beta.techcrunch.com/2005/10/10/yahoo-blog-search/)的一边，与此不同，Topix 已经将博客帖子整合到结果中，其方式与其他 12，000 个新闻源相同。

Topix 博客帖子[宣布了](https://web.archive.org/web/20221129071730/http://blog.topix.net/archives/000082.html)的增加，提供了大量信息，说明他们如何选择 15，000 个订阅源，以及博客报道与传统新闻报道有何不同。好东西。看到这个我真的很开心。
