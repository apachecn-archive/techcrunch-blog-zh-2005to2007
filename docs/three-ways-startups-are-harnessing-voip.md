# 初创公司提供 VOIP 的三种方式 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/08/15/three-ways-startups-are-harnessing-voip/>

当消费者“替代固定电话”的 VOIP 之战继续进行时(有线电视公司现在控制了该市场 70%的份额，Vonage 还在[与](https://web.archive.org/web/20221130215543/http://www.beta.techcrunch.com/2007/08/09/hey-maybe-vonage-isnt-dead-yet/)作战)，许多灵活的纯软件创业公司正在试验他们自己的服务。

所有这些都允许用户以大大降低的成本拨打普通的非 VOIP 电话。无论电话交谈的各方是否使用支持 VOIP 的电话，都可以获得这些节省，因为在某些连接处，传输可以从 PSTN 跳到 VOIP，反之亦然。例如，打给全国各地朋友的移动电话可能从 PSTN 开始，快速跳转到 VOIP 进行长途旅行，然后跳转回目的地附近的 PSTN。

关键是使用 VOIP 来剥离部分或大部分通话成本，让这些初创公司向消费者提供成本非常低的通话。不过，这不是免费电话——任何时候，只要一条普通电话线被用于至少部分通话，特别是终止通话，电信公司就要付费。

理解所有新的 VOIP 创业公司是令人生畏的，所以我们按照用例对它们进行分类。要比较功能、价格和更多公司，请查看这张图表。

**我很贱而且我有一台电脑**

到目前为止，打电话最便宜的方法是买一个桌面客户端。桌面上的 VoIP 客户端允许用户通过 VoIP 网络从一台计算机向另一台计算机拨打电话。如果需要额外付费，您可以连接到 PSTN 电话网络上的标准电话，与您的计算机进行通话。你们大多数人都知道这是 Skype-in 和 Skype-out。

最著名的桌面客户端是 Skype，拥有超过 1 亿用户。这些大公司——[雅虎](https://web.archive.org/web/20221130215543/http://voice.yahoo.com/)、[微软](https://web.archive.org/web/20221130215543/http://get.live.com/messenger/overview)和[谷歌](https://web.archive.org/web/20221130215543/http://www.google.com/talk/)——也有他们自己的 VoIP 桌面客户端。由于 VoIP 产品已经内置到他们的即时通讯客户端中，它们共同构成了一个拥有超过 3 . 4 亿用户的潜在市场。

一个更年轻的初创公司，去年 7 月推出的 [Gizmo 项目](https://web.archive.org/web/20221130215543/http://www.crunchbase.com/company/gizmoproject)。据报道，他们的应用程序有 200 万次下载。该应用程序的功能类似 Skype，支持 IM 和 VoIP 通话。Gizmo 项目的独特之处在于，它不仅提供 IM 和 VoIP 通话，而且如果你向朋友推销他们的产品并成为一名活跃用户，还可以免费拨打标准电话网的电话。

Hullo 和 Nimbuzz 是我们讨论过的其他桌面 VOIP 应用。

**喜欢 WiFi 又省钱**

如果把笔记本电脑放在耳边和朋友通话的想法听起来不吸引人，诺基亚的 WiFi 电话可能适合你。诺基亚 N800 就是一个很好的例子，它利用了桌面应用程序的免费网络通话功能。 [Fring](https://web.archive.org/web/20221130215543/http://www.crunchbase.com/company/fring) ，通过 3G/GPRS 和 WiFi 提供类似 Skype 的功能，对诺基亚非常友好，刚刚转移到 Windows Mobile。然而，你仍然需要支付拨打标准电话的费用，并购买一个真实的电话号码，这样你那些过时的 PSTN 电话上的朋友就可以给你回电。他们最近筹集了另一轮[1200 万美元](https://web.archive.org/web/20221130215543/http://www.beta.techcrunch.com/2007/08/10/voip-startup-fring-raises-12-million-series-b/)，过去也从我们这里获得了很多[好评](https://web.archive.org/web/20221130215543/http://uk.beta.techcrunch.com/2006/09/26/fring-brings-mobile-voip-to-the-next-level/)。

**我有社交生活 WiFi 不能包含**

如果你不在 WiFi 天堂(山景)或经常在 WiFi 热点闲逛，有一些其他的半 VoIP 解决方案仍然可以为你节省一些钱，至少在长途电话上。移动 VoIP 提供商并没有放弃 PSTN 线路，而是通过用更便宜的 VoIP 线路将主叫和被叫打给本地号码的两个电话连接起来，从而为客户省钱。然而，这些解决方案最适合长途电话，在这种情况下，桥接本地电话是有意义的，并且仍然需要花费您的移动计划。最大的好处是，它可以在你阅读了一篇关于 CrunchGear 的评论后买到的热门新手机上运行。

这一类的玩家不少，包括桌面 VOIP 客户端 Skype 自带的播放器， [iSkoot](https://web.archive.org/web/20221130215543/http://iskoot.com/) 。iSkoot 是 Skype 的移动版本，它允许您通过呼叫 Skype 服务器转接来电，向您的 Skype 联系人拨打电话。Shape Services 最近拼凑了一个 Skype 的 iPhone 版本，但有报道称该版本受到 AT T 的低传输速率的影响。另一家初创公司， [EQO](https://web.archive.org/web/20221130215543/http://www.beta.techcrunch.com/2007/05/23/eqo-launches-mobile-voipchatmessaging-client/) 也在这一领域竞争，直到他们推出了我们之前提到的 VOIP、IM 和即时通讯移动应用。

最大的两家公司分别获得了 2800 万美元和 2450 万美元的融资，分别是 Jajah 和 T2 Truphone。这两家初创公司让你可以轻松地用手机打电话。然而，Jajah 使用 VOIP 来桥接两条标准电话线，而 Truphone 可以进行真正免费的通话，如果你的电话有足够快的数据连接的话。随着 Jajah 的投资者 T-Mobile[将 Truphone 踢出他们的网络](https://web.archive.org/web/20221130215543/http://gigaom.com/2007/06/15/tmobile-truphone/)，他们的关系变得更加复杂。T-Mobile 通过“[家庭热点](https://web.archive.org/web/20221130215543/http://www.google.com/url?sa=t&ct=res&cd=1&url=http%3A%2F%2Fwww.theonlyphoneyouneed.com%2F&ei=xMWNRrWnBZC-gAOX5rzQCA&usg=AFQjCNHMMaa0IWVntQxOqZLzRZl9WnuE_g&sig2=Lt1ho8VI69s2I9TB7DVaaw)”进行了自己的 WiFi 通话尝试，让你每月额外支付 9.95 美元，就可以从你的 WiFi 网络无限制地拨打电话。

**谁赢了**

虽然 Skype 显然在为易贝赚钱，但据我们所知，其他初创公司都没有盈利。但是通信行业本身也受到了伤害。随着语音收入从 2000 年的每月 51 美元下降到去年的每月 43 美元，尤其是在移动行业，一种转变正在发生。数据计划被广泛认为是增加电信公司每用户年收入的未来(ARPU)。

然而，这并不意味着 VOIP 的道路一帆风顺。手机上的 VOIP 正面临着相当多的挑战。最基本的问题是在过多的移动平台上分发你的应用程序。移动运营商没有帮助，因为他们仍然不愿意加速他们的语音和内容服务的消亡。威瑞森和他们的各种订阅服务(VCast，maps)是后者的完美例子。

我们也对这些应用程序的可用性表达了很多不满。安装到你的手机上后，VOIP 服务可以增加另一个不同通话费率的老巢，有时只能为你节省长途电话费用，但仍然要花几分钟。由于美国的许多手机套餐中都包含了全国长途，所以对许多用户来说可能没有意义。尽管如此，仍有数十家 VOIP 运营商(看看我们的[图表](https://web.archive.org/web/20221130215543/http://www.beta.techcrunch.com/wp-content/voipcomparison.html))竞相压低通话费率。

然后是带宽要求。移动数据网络通常不够快，无法确保足够高的服务质量。无论谷歌如何努力，通过 WiFi 提供 VOIP 的最佳方式仍未普及。3G 提供了更好的覆盖和足够的带宽，但仍然由运营商控制，他们可以限制上行带宽来影响 VOIP 的质量。威瑞森[据报道](https://web.archive.org/web/20221130215543/http://www.crn.com/it-channel/160401324)计划提供 3G 网络电话，但自 2005 年以来一直没有兑现承诺。

因此，VOIP 仍然分散在固定电话、桌面和移动平台上。

问题的关键在于，这一类别的赢家必须与运营商友好相处。即使是纯粹依靠数据计划或你的桌面工作的创业公司也需要运营商来提供移动网络基础设施。Jajah 最适合与运营商合作，它提供更便宜的长途电话，同时仍然使用运营商赖以生存的通话时间。Fring 和 Gizmo Project 等通过数据网络运营的服务可能会绕过运营商在越来越快的数据网络上的语音计划，从而为消费者提供更好的交易，但与运营商直接竞争。削减这些现有公司的利润将最终导致他们正面交锋，就像 TruPhone 对 T-Mobile 所做的那样，或者一些运营商已经在 N95s 上禁用了 VOIP。除非这些初创公司改变他们的革命性言论，否则这看起来是一场艰苦的战斗。
