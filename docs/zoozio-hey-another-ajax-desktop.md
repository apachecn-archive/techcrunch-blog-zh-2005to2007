# 嘿，又一个 AJAX 桌面！

看来总部位于雷德伍德城的 Zoozio 将成为 AJAX 桌面竞赛的下一匹马。

我们已经有了 [Protopage](https://web.archive.org/web/20220928114010/http://www.beta.techcrunch.com/2005/08/18/profile-protopage/) 、 [Netvibes](https://web.archive.org/web/20220928114010/http://www.beta.techcrunch.com/2005/09/16/netvibes-personal-homepage/) (我们的最爱)和(基于 flash 的) [Goowy](https://web.archive.org/web/20220928114010/http://www.beta.techcrunch.com/2005/10/17/goowy-charges-ahead/) 。哦，对了，还有谷歌。等等，[微软](https://web.archive.org/web/20220928114010/http://www.beta.techcrunch.com/2005/11/01/windows-live-more-than-an-ajax-desktop/)也是。

网站上还没有太多信息。如果这些人想进入这个已经很拥挤的市场，我希望他们有一些有趣的特点。谢谢史蒂文的提示。皮特·凯什摩尔昨天也看到了这一点。
