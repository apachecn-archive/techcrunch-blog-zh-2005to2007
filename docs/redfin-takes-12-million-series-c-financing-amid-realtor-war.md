# 在房地产经纪人大战中，Redfin 获得 1200 万美元的 C 轮融资

总部位于西雅图的房地产网站 Redfin 已经完成了 1200 万美元的 C 轮融资，该网站允许用户绕过与使用房地产经纪人相关的大部分费用。除此之外，他们在前两轮融资中筹集了大约 800 万美元。这轮由德雷珀·费希尔·朱韦森领衔。之前的投资者 Madrona Venture Group、Vulcan Capital、BEV Capital 和 Hillman Company 也参与其中(参见所有融资历史[此处](https://web.archive.org/web/20220615052227/http://www.crunchbase.com/company/redfin))。

Redfin 正在尽最大努力将房地产代理商和经纪人从至少一半的房屋销售中彻底清除出去。该公司将 MLS 列表信息(待售房屋)与历史销售数据(已售出房屋)结合到一张地图中。如果你找到一个你喜欢的房子，并想提出报价，Redfin 将在购买过程中代表你(他们有一个呼叫中心，有持牌的房地产专业人士指导你)。然后，他们会在交易结束时将买方房地产费用的 2/3 退还给你。普通购房者在一笔交易中可以节省大约 1 万美元。该公司还将在国内销售交易中代表卖方。

该公司已经惹恼了房地产行业，引起了主流的关注。五月，他们被制作成 60 分钟节目。从那时起，该公司已经完成了超过 3.5 亿美元的房地产交易，并为客户节省了约 600 万美元的佣金。

房地产经纪人的反击主要是试图限制该公司可以在其网站上显示的 MLS 数据，并试图阻止该公司在这些数据旁边显示客户评论。他们还看到越来越多的房地产经纪人拒绝接受 Redfin 客户的报价，一些“待售”的标志已经从院子里砍掉。

考虑到他们业务的破坏性，我对此并不感到惊讶。在我们之前的帖子中，房地产经纪人对该公司的一些评论是恶毒的。不过，我感觉房地产经纪人和 Redfin 之间真正的大战还在后头。
