# 蝎子手表公司暗礁潜水员

![](img/1eb0d007964c59f7dc2b07a73cd2daff.png)

刚刚得到了新的 Scoripion 礁潜水员审查。这是由丹佛的几个人定制的自动手表，看起来很漂亮。它有一个自动 ETA 机芯，带有黑客秒针-这意味着当你拔出表冠时秒针会停止-并且防水 30ATM 或 1000 英尺。他们只生产了 100 名红礁潜水员，但他们可能会在一段时间内到工厂生产更多。完全署名——表冠、表壳、表扣都说蝎子——相当醒目。

初步印象:它很轻，但没关系，799 美元的价格标签可能会让一些人失去兴趣。然而，如果你想买一款别人没有的潜水表，这款手表在[排名前三的](https://web.archive.org/web/20130628171544/http://bathyswatch.com/main.php)微酿、[美国](https://web.archive.org/web/20130628171544/http://dfreemontwatches.com/)型号中名列前茅。
