# 博客有用——这些家伙创造了很酷的东西

 [Weblogs Work](https://web.archive.org/web/20221006205136/http://www.weblogswork.com/) ，总部设在得克萨斯州的达拉斯，是一家 web 2.0 咨询公司、孵化器和总体很酷的一群人。我发现我花了越来越多的时间与这些人谈论新的想法，我们彼此联系得如此频繁，以至于人们开始谈论。:-)

博客的工作是设计、构建和推广新的应用程序，如果合适的话，还会帮助第三方构建应用程序。现有的和未来的应用程序包括像 [eflURL](https://web.archive.org/web/20221006205136/http://www.elfurl.com/) (我们的[配置文件](https://web.archive.org/web/20221006205136/http://www.beta.techcrunch.com/?p=96))、 [Gahbunga](https://web.archive.org/web/20221006205136/http://www.gahbunga.com/) 、Frankenfeed、EgorRSS 和 EgorOPML。在不久的将来，许多或所有这些都将在 TechCrunch 上进行描述。

Brian Oberkirch(他有一个名为 [Like It Matters](https://web.archive.org/web/20221006205136/http://www.lightbox5.com/likeitmatters/) 的非常出色的个人博客)是首席执行官。他以前是文学教授、大学棒球运动员和营销主管，后来成为企业家。该公司有 50%的营销/公关人员像 Brian，50%的技术/编码人员像 Scott Ryan(总共 26 名员工/承包商)。

[M Ventures](https://web.archive.org/web/20221006205136/http://www.m-ven.com/) (我喜欢去这个网站只是为了听听很酷的背景音乐)资助和指导博客工作。亚历山大·缪斯是 M Ventures 的主要负责人，他在这里保持着他的[博客](https://web.archive.org/web/20221006205136/http://texasvc.weblogswork.com/)(另一个必读)。
