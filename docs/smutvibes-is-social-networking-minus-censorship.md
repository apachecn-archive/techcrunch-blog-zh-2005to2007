# SmutVibes 是没有审查的社交网络

[![](img/ec771090f6e2e900d1d53fcb6e459005.png)](https://web.archive.org/web/20210924033055/http://www.smutvibes.com/) 牙买加金斯敦 [Vibesnetwork](https://web.archive.org/web/20210924033055/http://vibesnetwork.com/) 今天正式推出其最新的在线社交网络服务——[smut vibes](https://web.archive.org/web/20210924033055/http://smutvibes.com/)。(警告:这个网站到处都是非常裸体的人。VibesNetwork 首席执行官兼总裁德尔罗伊·弗雷泽表示，Smutvibes 将为许多成年用户提供他们想从社交网络服务中获得的东西，一个进行未经审查的性对话的地方，并将在其他地方的青少年社交网站之外为这些对话创造一个单独的空间。

这个系统拥有你所期望的社交网络服务的所有功能，或许不包括 VOIP。照片上传、私人聊天、博客、评论等等。还有一个事件板，看起来它得到了很好的利用。我已经用化名下的照片测试了所有的功能，也遇到了很多非常棒的人。开玩笑的。

真正的吸引力可能是露骨的照片不会被删除，而是被鼓励。该系统的熟悉度和免审查政策可能真的会吸引 MySpace 现在 7500 万用户中的一部分人。

如何让青少年远离淫秽物品？也许可笑的是，那些提供出生日期表明他们不到 18 岁的人是不允许登记的。嗯……我认为这还不够。

会员是免费的，我可以想象广告收入是一个真正的可能性。另见该公司的合作伙伴网站[Christianvibes.com](https://web.archive.org/web/20210924033055/http://christianvibes.com/)。
