# Scouta 为 iTunes 带来了推荐

 [](https://web.archive.org/web/20210115141017/http://www.scouta.com/) 社交音乐和视频推荐初创公司 [Scouta](https://web.archive.org/web/20210115141017/http://www.scouta.com/) 今天推出了免费的 Mac 软件，可以自动为 iTunes、iPods、iPhones 和 Apple TV 提供社交推荐。

Scouta 在二月份推出了基于个人内容偏好向会员推荐音频和视频内容的服务。

新的代理允许 Scouta 基于会员的 iTunes 库进行推荐，包括播客和视频播客订阅。然后，iTunes 会自动提供推荐，供本地播放或上传到连接的设备。

Scouta 还推出了网站重新设计和新的网站功能，以配合推出。

Scouta 计划在未来的版本中包括电影、音乐视频和电视节目推荐。
