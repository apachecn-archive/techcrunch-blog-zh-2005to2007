# 三星、阿迪达斯在新手机上的混搭

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/30/samsung-adidas-mashup-on-new-mobile-phone/>

![adidas-samsung-sgh-f110.jpg](img/39d8fac0f5a6c2b7d9bdc027a182d9d3.png)

就在几天前，我去慢跑，想知道是否有人会填补移动电话行业的空白，推出集计步器、心率监测器和 MP3 播放器于一体的产品。安静我的心。关于三星和阿迪达斯合作发布 SGH-F110 的谣言正在流传。

如果传言是真的，那么我们很快就会看到一款手机，它在一个漂亮的小包装中完成我之前提到的所有功能，包括 2 英寸屏幕、200 万像素摄像头、FM 收音机、microSD、USB 2.0 和蓝牙 2.0。音频播放器还会根据你的心跳节奏调整你的播放列表，一个声音会对你大喊大叫，让你快点。F110 应该可以在四频 GSM 上运行，但是 EDGE 也在其中，这可能会让你们有些人望而却步。
