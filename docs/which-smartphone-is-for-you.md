# 哪款智能手机适合你？TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/11/05/which-smartphone-is-for-you/>

[](https://web.archive.org/web/20160319020018/http://crunchgear.com/2006/11/03/smartphones-now-first-time-buyers-guide/)CrunchGear 团队发布了一份非常详细的[主要智能手机选择对比](https://web.archive.org/web/20160319020018/http://crunchgear.com/2006/11/03/smartphones-now-first-time-buyers-guide/)，以帮助首次购买者做出正确的选择。这是该团队正在进行的许多产品比较研究帖子中的第一篇。

他们比较了目前排名前七的智能手机。我使用的电话，摩托罗拉 Q，包括在内。坦率地说，这是一个糟糕的选择，在这一点上，我希望我只是买了一个黑莓或 Treo。Q 上的按键太小了，用户界面的设计也很糟糕。我一直在摔这个该死的东西，希望它会坏掉(给我一个买新手机的借口)，但它是一个坚固的小手机。我们将看看它在“哎呀，我把它掉在厕所里了”测试中表现如何。

CrunchGear 还发布了“[我讨厌智能手机的十件事](https://web.archive.org/web/20160319020018/http://crunchgear.com/2006/11/03/smartphones-now-10-things-i-hate-about-smartphones/)”，这是买家的另一个好资源。
