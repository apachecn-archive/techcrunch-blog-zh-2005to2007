# 煎蛋卷 1.0——第一款苹果电视游戏

![](img/28c366aeaa02463aeca029b31ad8a446.png)

现在苹果电视已经上市一段时间了，人们已经习惯了，啊，游戏现在开始流行起来。煎蛋卷 1.0 是第一批登陆网络的游戏之一，看起来相当不错。游戏玩法看起来类似于一个*栏*，你四处移动鸡蛋，并匹配三个相同颜色的鸡蛋。

这个游戏是一个简单的下载，但要玩它，你需要修改/破解你的苹果电视盒子。一个简单的谷歌搜索就能解决这个问题，所以如果你喜欢冒险，而且你是个电视迷，那么试试煎蛋卷吧。最糟糕的情况是你把你的苹果电视弄坏，扔向墙壁，然后去餐馆吃花椰菜、切达干酪和培根煎蛋卷。Mmmmmm，好吃。
