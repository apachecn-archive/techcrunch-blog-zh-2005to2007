# GoogleTube 会在本周公布吗？

过去几天，谷歌/YouTube 被收购的传言迅速传开。在我们最初的[帖子](https://web.archive.org/web/20230219141530/https://techcrunch.com/2006/10/06/completely-unsubstantiated-googleyoutube-rumor/)讨论了这个谣言之后，[华尔街日报](https://web.archive.org/web/20230219141530/http://online.wsj.com/article/SB116014813857884917.html?mod=technology_main_whats_news)(在付费墙后面)和[纽约时报](https://web.archive.org/web/20230219141530/http://www.nytimes.com/2006/10/07/technology/07google.html?_r=1&oref=slogin)很快跟进了他们自己的文章。两家出版社都没有联系我，我也不知道他们的消息来源是谁。

猜测从那里开始。例如，Hitwise 的 LeeAnn Prescott 公布的数据显示，YouTube 比谷歌视频大多少(LeeAnn 还指出，谷歌可以很容易地复制 YouTube)。马克·库班继续声称这笔交易毫无意义。

其他媒体现在宣称交易即将达成。不管它是否发生，这似乎是硅谷现在的话题。

除了马歇尔·柯克帕特里克正在追踪的一个发生在周五和周六的离奇故事之外，我们没有任何其他经过核实的信息。我们得到消息，代表谷歌的律师可能已经告诉至少一个人“交易已经正式通过。”据说被告知此事的人随后给许多人发了一封总结谈话内容的电子邮件，我们听到了风声。我和律师谈过，他否认“对交易有任何了解”，也没有发表过这样的声明。我也和写邮件的人谈过，他收回了原话，说他夸大了律师的陈述，简而言之，他写的邮件不是真的。那时，马歇尔和我决定放弃它。但我要说的是，律师反复告诉我他不知道交易，而不是“无可奉告”，这意味着他要么在撒谎(不太可能)，交易正在发生，他不知道(不太可能)，要么根本没有交易。我们走着瞧。
