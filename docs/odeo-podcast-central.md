# Odeo 简介–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/06/13/odeo-podcast-central/>

**公司:** [奥迪奥](https://web.archive.org/web/20220704040930/http://www.odeo.com/)

![](img/df531a5b0342f72bee9f42cc5d7a3ca9.png)

**地点**:旧金山

**状态:**私测
 **什么事？**

Odeo 将是一个播客服务，让用户搜索、查找和订阅播客，也是一个用最少的硬件创建你自己的播客的工具。Odeo 建立在[轨道](https://web.archive.org/web/20220704040930/http://www.rubyonrails.org/)上。

用他们自己的话说，*“嗨。我们是 Odeo(发音像 rodeo)。在旧金山的某个公寓里，我们正在让你更容易地为你的 iPod(或任何你喜欢的 MP3 播放器类型的交易)发现、创建和订阅新鲜、独立的音频内容。”*

还没什么可看的，因为他们的产品只能通过邀请测试版获得。然而，有几个人已经测试了该产品，并在博客上发表了文章。

来自[webbreakstuff](https://web.archive.org/web/20220704040930/http://webreakstuff.com/blog/2005/06/the-rodeo-thoughts-on-odeo/):

*“通过浏览类别，你可以轻松订阅自己喜欢的音频内容，并查看其他人在听什么。此外，他们允许某人添加任何系统中还没有的节目，如果您实际上是当前频道的发布者，Odeo 可以让您轻松控制和定制它。*

不幸的是，Odeo Studio 功能还没有上线。Odeo Studio 是，我引用一下，一个基于浏览器的工具，它使录制和发布变得容易。有了录音棚和一个便宜的麦克风(甚至是你笔记本电脑内置的麦克风)，你就拥有了你需要的一切。这很有趣，如果他们做得对，这可能是我在之前关于 Odeo 和 iTunes 的帖子中提到的给他们带来优势的特性。我们必须等等看。
我喜欢概览页面(Odeo 的入口页面)列出特色内容、时代精神(人们现在在听什么)和 Odeo 新闻的方式。"

看起来 Odeo 也将支持内容标记，这将是找到好东西的一个重要途径——与文本不同，音频很难预览，因此元数据更加必要。
