# 现在的智能手机:Linux 

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/03/smartphones-now-linux/>

如果你认为基于 Linux 的操作系统是为密室、铁杆极客和计算机科学研究生准备的，那你就对了。然而，最近，它在变得更加“用户友好”方面取得了长足的进步，甚至发展到可以作为手机使用的地步。但是是什么让你想选择一部 *Linux* 智能手机呢？

这个问题有几个答案:如果你是 Linux 的忠实粉丝，你会想要一部运行 Linux 的智能手机，而不是 PalmOS、Symbian 或 Windows Mobile。您使用 Linux 已经有一段时间了，您渴望在更便携的产品中获得相同的构建模块功能。第二个原因是，如果你是一个喜欢自己动手修补新事物的人。用于手机的 Linux 是一个尚处于起步阶段的媒介，但是由于它的“开源”特性，如果你对编程有点熟悉的话，你可以轻松快速地编写自己的程序。Linux 手机通常还不适合普通用户或商业用户，但在几年内，它们可能会获得一些可观的市场份额。

这些都很棒，但是你能用 Linux 手机做什么呢？嗯，Linux 手机上一些更有趣的功能来自于它运行 Linux 的事实。当然，你有所有普通的智能手机功能，如办公文档编辑、网页浏览、电子邮件、即时消息和媒体播放。但你也有多年的优势，和书呆子，用他们的 Linux 经验支持你。你可以自由下载和编译不同的软件包来适应你的手机。

想在路上远程控制您的电脑吗？当然，抓住 VNC，让它发生。想打免费电话吗？买一部基于 SIP 的软电话，你就可以用 802.11b/g 和你的妻子通话了。想做点什么吗？自己写，或者上 sourceforge 看看是否有人已经写好了。就这么开着。

这一切对你来说听起来很完美，不是吗？只有一个障碍:Linux 智能手机在许多方面仍然是一个白日梦。当然，它们确实存在，但数量远不及其他智能手机。我们的快速调查显示，目前没有美国供应商提供运行 Linux 的智能手机。这并不意味着它不会发生。事实上，Palm OS 的下一个版本是基于 Linux 的。除了支持开源 Linux 项目之外，据说它还向后兼容流行和成熟的 Palm OS 应用程序，这对两个平台都是一个真正的福音。此外，有传言称，摩托罗拉将在不久的将来推出基于 Linux 的智能手机平台，并且已经有一款手机通过了 FCC 认证。

如果这对你来说还是有点不确定，你可以自己酿造。狂热分子在让 Linux 在为其他智能手机操作系统设计的硬件上运行方面取得了一定的成功。例如，众所周知，Palm 的 Treo 可以很好地运行 Linux，其他硬件应该不会太难破解。谁知道呢，将来我们可能会像一些人购买没有操作系统的计算机硬件一样购买智能手机，并计划安装自己的操作系统。

Linux 在智能手机市场面临的障碍之一是与其他智能手机的互兼容性。如果我的手机有 WiFi，我想给你发送一个联系人，谁能保证你的联系人列表会识别我的数据呢？这就是嘴唇的用处。LIPS 是 Linux 电话标准论坛。Palm OS 的制造商 Access(以前的 Palm Source)是 LIPS 的创始成员，由于其操作系统的下一个版本是基于 Linux 的，它将为不断发展的标准论坛提供真正起飞所需的腿，并为其他制造商使用其标准。

这并不意味着现在没有对 Linux 智能手机的大力支持。摩托罗拉、NEC、松下和三星都有运行 Linux 的成功手机，事实上，新的三星 SCH-i819，一款面向中国商人的智能手机，运行的是支持触摸屏的 Linux 的特殊版本，非常适合输入汉字。此外，它还具有 BREW 功能，支持大多数移动下载。

如果你是第一次购买智能手机，并且从未拥有过任何种类的 PDA，那么 Linux 设备的学习曲线与 Windows Mobile 或 Palm OS 或 Symbian 设备没有太大不同:你指着你想要的东西，告诉它做什么，它应该会做得很好。不幸的是，没有美国制造商在美国销售基于 Linux 的智能手机，尽管是由于 Linux 的性质，但这种情况将会改变。不过，对于高端用户来说，这是一个将尽可能多的功能集成到一个非常小的设备中的理想平台。
