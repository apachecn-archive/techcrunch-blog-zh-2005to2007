# 公爵抵达威瑞森无线电台

虽然仍在开发中的《永远的杜克·核弹》(Duke Nukem Forever for the PC)在这一点上仍然是一款蒸汽软件(以至于许多人给这款游戏贴上了“杜克·核弹”的标签)，但杜克已经在第三个屏幕上首次亮相了。SkyZone Entertainment 和 MachineWorks Northwest 宣布，杜克 Nukem Arena 3D 将在精选的威瑞森 Wireless Get It Now enabled 手机上直播。

这款游戏使用 RIPP3D 图形引擎，保留了传统的第一人称射击游戏(FPS)体验，玩家可以扮演公爵、猪头或伞兵的角色，并与最多三名其他玩家进行战斗。

“SkyZone 对字幕内容的获取非常认真，我们与 MachineWorks NorthWest 的关系使我们有能力向订户提供更高端的游戏体验，”SkyZone Entertainment，Inc .的首席出版官 Bob Shireman 说，“除了补充我们的热门游戏产品组合之外，Duke Nukem Arena 3D 对广泛的游戏玩家群体具有吸引力，代表着运营商的出色价值驱动力。”

杜克 Nukem 竞技场 3D 现在可以通过 Get It Now 虚拟商店以每月 3.49 美元的价格或 9.99 美元的价格无限制地使用精选手机。
