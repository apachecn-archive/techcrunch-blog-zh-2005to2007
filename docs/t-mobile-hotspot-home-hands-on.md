# T-Mobile 热点@ Home Hands On

T-Mobile HotSpot @ Home 是一项刚刚在西雅图推出的服务，它可以让你无缝地将电话从蜂窝切换到 WiFi。该服务配有 T-mobile Linksys 路由器和 Samsumg UMA(免授权移动接入)电话，允许您在家中使用 WiFi 进行通话。

该计划每月花费 19.99 美元，不包括你为常规计划支付的费用，那么为什么它值得你花时间呢？因为任何通过 WiFi 的通话都是免费的，不包括在您的正常通话时间内。此外，如果你的手机信号在家很差，使用 WiFi 会比根本不打电话好得多。第三个好处是可以在星巴克、机场和联邦快递/Kinkos 等 T-Mobile 热点地区免费拨打电话。我们不确定你是否已经注册了 T-Mobile Hot Spot access，或者这是否包含在每月 19.99 美元的计划中。你还必须为双频手机支付 49.99 美元。

随着这项服务在全国推广，请继续关注更多新闻。
