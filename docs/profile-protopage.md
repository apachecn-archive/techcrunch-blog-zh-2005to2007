# 个人资料–原型页面–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/08/18/profile-protopage/>

## 概述:

虚拟桌面本周很流行。不像 Goowy ( [昨天的简介](https://web.archive.org/web/20210731004047/http://www.beta.techcrunch.com/?p=160))是基于 flash 的， [Protopage](https://web.archive.org/web/20210731004047/http://www.protopage.com/) 全是 Ajax，宝贝。

视觉上既实用又美观。拖放、添加和删除。非常酷。

目前应用程序有些有限:便笺、待办事项、收藏夹链接和快速搜索。此外，最初的起始页有点拥挤，尽管只花了几分钟来定制它。

我想他们很快就会添加 RSS、电子邮件和其他很酷的小工具——肯定需要更多。

商业模式是什么？不知道。它是完全免费和无广告的，至少现在是这样。使用它根本不需要注册，尽管你需要通过电子邮件验证来命名页面。看看这个。
