# Truveo 每月增长 50%，表示视频搜索变得越来越重要

 [](https://web.archive.org/web/20230306024818/http://www.truveo.com/) [Truveo](https://web.archive.org/web/20230306024818/http://www.truveo.com/) ，2006 年初被 AOL 收购的[，被认为是互联网上最好的视频搜索引擎之一。它超越了附加到视频文件本身的元数据，探索了视频周围的网站内容，从而产生了更多的数据来索引和更好的搜索结果。](https://web.archive.org/web/20230306024818/https://techcrunch.com/2006/01/10/aol-acquires-truveo/)

交通数据证明了这一点。据康姆斯克报道，该网站每月有 4000 万独立访问者，并且以每月 50%的速度增长。

Truveo 说，这是他们的搜索方法的结合，也是版权所有者越来越倾向于删除 YouTube 和其他网站上未经授权的上传的事实。其结果是，YouTube 不再是唯一能找到好视频剪辑的地方。例如，人们开始在喜剧中心网站上搜索日常节目片段，而不仅仅是在 Youtube 上寻找。

该公司还与 AOL 以外的公司合作，以获得该引擎更广泛的分销渠道。Brightcove、Clevver、CSTV、Flock、Pageflakes、PureVideo、Qwest、Search.com、Sportingo、Netvibes、Widgetbox 和 YourMinis 现在都使用 Truveo 进行视频搜索，加入了现有的合作伙伴 AOL Video、AOL Search、Excite、Infospace。

然而，有一个新的孩子，叫做 [CastTV](https://web.archive.org/web/20230306024818/https://techcrunch.com/2006/10/03/casttv-will-revolutionize-video-search/) ，它最近宣布了来自 Draper Fisher Jurvetson 的[310 万美元](https://web.archive.org/web/20230306024818/https://techcrunch.com/2007/04/24/casttv-raises-31-million-battles-google-and-aol-in-video-search/)A 轮融资。他们还没有推出，但演示结果显示它和 Truveo 一样好，甚至更好。鉴于 AOL 已经拥有 Truveo，如果技术如演示所显示的那样好，可以考虑迅速收购 CastTV，可能是由谷歌收购。
