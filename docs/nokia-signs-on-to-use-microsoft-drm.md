# 诺基亚注册使用微软数字版权管理

![](img/4fd8a8e9ff71bf26cd8ab16fda845b4b.png)

你知道吗，我很高兴你有一部 iPhone。虽然它被过度炒作和谈论了几个月，但至少它没有使用微软的低劣 PlayReady DRM。诺基亚做出了愚蠢的举动，与微软签约，在使用 S40 和 S60 接口的手机上使用 DRM。

诺基亚手机往往功能丰富，尤其是在媒体部门。引入数字版权管理，你的手机就会变成垃圾。这个举动最终肯定会伤害诺基亚，等着瞧吧。当你可以让消费者自由的时候，为什么要限制用户在*该死的手机*上做什么？哦，也许摩托罗拉现在会走出低谷。
