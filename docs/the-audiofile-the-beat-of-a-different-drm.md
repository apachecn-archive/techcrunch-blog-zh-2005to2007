# 音频文件:不同 DRM 的节拍

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/18/the-audiofile-the-beat-of-a-different-drm/>

![](img/f9659311fe315791fad81e63f7051f40.png)
唱片业是否正式失去对数字音乐的控制？更重要的是，它将采用哪些新的限制来打击必然会出现的猖獗的盗版行为？还有:数字音乐仍然有一个主要缺点:没有转售市场。但是想象一下，有一种方法可以利用二手 MP3 的潜力…我必须每隔一周写一篇关于数字音乐的文章，但是我能说什么呢？最近发生了很多事情，百代在 iTunes 和 Amazon.com 等在线商店发布了没有版权保护的整个音乐目录*，以及爱可视的下一代产品威胁要将盗版移动化。*

 *鉴于亚马逊目前计划在今年晚些时候推出仅销售*未受保护的*曲目，它在让其他三大唱片公司与 EMI 一起放弃 DRM 方面的作用可能非常大。结合 iTunes 音乐商店和史蒂夫·乔布斯致各大公司的公开信，亚马逊可以利用其庞大的用户群成为版权憎恨者的音乐爱好者的梦遗。

据[ArchosLounge.net](https://web.archive.org/web/20140309015011/http://www.archoslounge.net/Cinquieme-generation-Flash.html)报道，爱可视即将于 6 月份推出的第五代便携式媒体播放器将支持 Bit Torrent 和 CinemaNow 然后你将可以选择合法或非法地下载电影和音乐，只要你在热点附近。瞧，随时随地的盗版！

如果 RIAA 想要控制盗版，它应该与互联网服务提供商更紧密地合作，以应对数字音乐共享领域不断变化的局面。当然，这可能意味着某种使用税。呸。但是这比试图控制消费者端的事情要有效得多；他们不能阻止人们分享，但他们至少可以让网络盗版不那么方便。

倒卖你的灵魂我本质上是个捡便宜货的人。我喜欢以低于全价的价格买东西，作为一个音乐迷，我天生就喜欢二手 CD(和 LP)。我已经买到了一些非常棒的东西，5 或 6 美元一张的二手 CD 或 3 美元一张的唱片，但我真的不能用数字音乐做到这一点——二手数字产品没有很大的折扣。一个寻找数字便宜货的人该怎么做？

问题是，当你复制一份数字文件时，你得到的是与原件完全相同的东西，而且你可以无限复制。但是，如果数字文件质量下降，比如说，像黑胶唱片一样的播放次数下降，或者更好的是，文件被复制的次数下降，那该怎么办？这将确保原件得以保存，但拷贝越多，拷贝的质量就越低。

然后你可以合法地转售歌曲的拷贝，但是根据质量降低价格。因此，在你卖出一个文件后，拷贝的质量会下降到每秒特定的千比特数，以此类推，直到拷贝几乎不可收听。换句话说，你可能很难卖出一个 64Kbps 的 MP3 文件……但至少你有最原始的版本。

不幸的是，无论你如何分割，这都只是另一种形式的 DRM——但限制较少。遗憾的是，对于大型音乐公司来说，这太少了，也太晚了，因为 DRM 显然是一个晚期病人。
