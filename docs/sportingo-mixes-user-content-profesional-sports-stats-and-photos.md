# Sportingo 混合了用户内容、专业体育统计数据和照片

 [](https://web.archive.org/web/20220929205551/http://www.sportingo.com/) [Sportingo](https://web.archive.org/web/20220929205551/http://sportingo.com/) 是一个体育新闻和分析网站，将球迷撰写的原创内容和来自场外的博客内容与获得许可的体育统计数据、时间表和专业摄影相结合。它是在成功的韩国网站 [OhMyNews](https://web.archive.org/web/20220929205551/http://ohmynews.com/) ( [英文版](https://web.archive.org/web/20220929205551/http://english.ohmynews.com/))和以色列 [Scoop](https://web.archive.org/web/20220929205551/http://scoop.co.il/scoop/index.html) 的模仿和帮助下开发的。

Sportingo 最初专注于欧洲足球和网球，为用户提供了一个全面的内容管理系统，上传他们的原创作品，然后由 sportigo 编辑团队进行编辑。该公司的编辑撰写标题，修改语法，批准标签的一致性，将搜索引擎优化元数据应用于每篇文章，附上相关的统计数据和从外部供应商那里获得许可的照片，并确定故事在 Sportingo 网站上的位置。编辑可以使用简单的表单在每个故事的模板中添加、删除和放置元素。读者与每个故事的互动也会影响故事的位置。显示比赛和体育新闻的赔率是因为人们觉得它们有趣，而不是通过 Sportingo 进行投注。收入模式完全基于广告。

这家由九个人组成的公司是由塔尔·巴纳奇和泽夫·罗佐夫创建的，他们曾领导一家体育多媒体百科 DVD 公司，该公司于 1996 年在伦敦上市。

Sportingo 与韩国的 OhMyNews 对世界新闻的做法非常相似。该网站非常成功，在该网站的报道被认为是上次韩国总统选举期间最重要的媒体活动之一后，这位新当选的总统接受了 OhMyNews 的选举后首次采访。Sportingo 团队与 OhMyNews 和 Israel's Scoop 合作，为他们的体育网站开发编辑流程。Sportingo 在注册时要求比许多人习惯的更多的个人信息，但这是一种将新闻与真实的人的身份联系起来的努力。OhMyNews 走得更远，要求投稿人扫描并发送个人 ID。

除了原创文章之外，Sportingo 还从非官方网站收集精选的博客文章，重点介绍顶级体育论坛的经理提交的论坛讨论以及路透社的突发体育新闻。该公司计划授权直播体育赛事，推出西班牙语和其他语言的本地化版本，并希望发展专注于报道突发新闻的用户贡献者团队，以便专业内容可以不受重视。

用户贡献者通过他们的故事的读者评级来发展声誉，状态级别从新秀开始，上升到 Sportingo 名人堂。高地位的贡献者将获得特权，如与运动员的私人会议和对网站上他们自己部分的编辑控制权。与许多类似的网站不同，贡献者没有收入分成计划。说到类似的网站，一些人在评论中建议也去看看。

我觉得这太棒了。从体育到个人理财和投资，现在有许多初创公司试图为博客世界的热门和多产部分带来结构，但 Sportingo 在编辑控制和需要专业发展的外部资源方面的投资确实可以为用户提供一个参与非常引人注目的体育新闻社区的机会。我们都知道主流媒体的许多缺点，但纯用户生成内容的缺陷也变得越来越明显。这种混血儿挺耐人寻味的。
