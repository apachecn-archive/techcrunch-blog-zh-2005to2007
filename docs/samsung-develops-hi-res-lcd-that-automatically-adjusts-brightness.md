# 三星开发自动调节亮度的高分辨率液晶显示器

三星(Samsung)和 LG 几乎每天都在推出“世界第一”，难怪 SK 在移动电话和尖端技术方面拥有世界市场，让其他人垂涎三尺。例如，三星刚刚发布了一款 2.1 英寸的液晶面板，配备了自动调节亮度的传感器，确保你不必眯着眼看屏幕上的内容。

由于三星的自适应亮度控制技术，这种新面板提供了 QVGA 分辨率，同时比相同尺寸和分辨率的其他显示器功耗低 20-30%。ABC 是一种在不到一毫米的显示结构中嵌入超薄黑色矩阵的设计。高端面板还有一个温度计，可以在任何环境下保持液晶显示器的恒温。这种电池板预计将在今年下半年开始生产。
