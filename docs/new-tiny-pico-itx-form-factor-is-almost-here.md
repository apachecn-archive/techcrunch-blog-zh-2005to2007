# 新的微型微微 ITX 外形(几乎)在这里

![Pico](img/16ebd6171b9b3e1a47f4845e383de2a3.png)

准备好迎接一个非常非常小的主板吧。如果你正在寻找建立，比方说，一个系统，适合在一个盒子里的流行馅饼，皮可 ITX 的形式因素是正确的。

虽然早在 2004 年就宣布了，但现在才开始少量供应，应该很快就会大量供应。

它的尺寸为 3.9 英寸×2.8 英寸，包含一个 1.0GHZ 的嵌入式 VIA C7 处理器，支持高达 1gb 的 RAM，一个 SATA 硬盘连接，四个 USB 插槽，以及对 MPEG-2/4 和 WMV9 视频的集成支持。
