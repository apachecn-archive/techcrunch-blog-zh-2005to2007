# WhosOff:轻松的工作场所休假管理

WhosOff 是一个在线的工作场所休假管理服务，它使组织休假变得容易。

该服务面向直接处理休假管理的行政人员、公司所有者和经理，从记录休假到平衡未来休假请求的需求。

WhosOff 不仅仅是一个由管理员控制的收集信息的数据库。WhosOff 支持员工全面访问；员工可以提交休假申请，甚至可以记录病假。被任命的管理员可以接受或拒绝请求，无论他们在哪里；作为一个基于网络的平台，分散了用户的访问。

世卫组织通过鼓励员工提前预订，鼓励他们在预订和计划假期时更加周到。通过共享跨业务或部门的休假信息，可以最大限度地减少最后一刻的请求，这样所有员工都可以看到任何假期请求的影响。

将休假管理包作为一个独立的产品来提供可能有点奇怪。许多大型企业已经在专业人力资源软件中内置了类似的功能，不需要这样的东西，但是对于中小企业来说，Off 是一种管理员工休假的智能、免费的方式。
