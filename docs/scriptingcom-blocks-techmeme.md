# Scripting.com 阻止了 TechMeme

 [](https://web.archive.org/web/20210702095300/http://www.techmeme.com/) [TechMeme](https://web.archive.org/web/20210702095300/http://www.techmeme.com/) 的博客新闻搜索引擎越来越受欢迎，变得[越来越有影响力](https://web.archive.org/web/20210702095300/http://searchengineland.com/070117-083259.php)。或许太有影响力了。

戴夫·维纳是广受欢迎和尊敬的博客 Scripting.com 的作者，他已经要求 TechMeme 停止索引他的网站。TechMeme 通过分析有影响力的博客之间的链接模式，自动确定其网站上的新闻标题(“有影响力”是由 TechMeme 的创始人 Gabe Rivera 定义的)。Winer 不喜欢这样的事实，当他链接到一个故事时，它会在 TechMeme 上给这个故事增加额外的能量。

博主很少会采取行动降低她/他自己的影响力，但这正是 Winer 所做的。他说，目前这只是一个实验。直觉告诉我，我们不会看到成群结队的博客作者争相采取类似的行动。
