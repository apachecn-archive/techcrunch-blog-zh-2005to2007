Technorati发布主要新功能

原文：https://web.archive.org/web/http://www.techcrunch.com/2005/09/01/technorati-releases-major-new-feature/trackback/

戴夫·西弗里今晚宣布推出 Technorati博客发现者。Technorati的Derek Powazek也在Blog Finder 上发表了文章。

这是什么？这是博客的排名列表。按标签。大卫·波瓦泽克称之为“你能想到的任何话题的迷你前100名”。

这是卡特里娜飓风。下面是 Web 2.0 。这是 Skype 。

有多好？我觉得它有些问题。标签是出版商创建的，所以出版商会添加大量的标签，并且会有动力为他们的博客设计一个合适的标签。举例来说，我相信只要在TechCrunch博客上添加“烹饪”标签，就能让TechCrunch成为一个非常受欢迎的烹饪网站。

添加标签有三种方式。根据Sifry的数据，Technorati已经标记了“超过200万个博客”，这似乎是准确的。添加标签的另外两种方式是通过publisher工具(注意下面的文本是为添加“web2.0”标签而定制的):

快进来。要将您的博客添加到此页面，声明您的博客并将“web2.0”添加到您的博客标签中，或者将此代码放在您的网站上。

然而，Technorati在选择描述Techcrunch的标签方面做得相当不错。我肯定会做出一些改变，但这是一个好的开始。

博客搜索器是Technorati 上的第三个标签。

乍一看，我实际上认为在这种情况下我更喜欢罗约的方法。罗约的列表使用读者用来描述博客发布的单个帖子的标签。请点击此处查看我们的个人资料更新，了解详情。然而，我确实相信，如果列表有用，它必须是基于主题的。这是解决这个问题的一个有趣的实验。
