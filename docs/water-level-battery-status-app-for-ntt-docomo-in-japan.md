# 日本 NTT DoCoMo 的水位电池状态应用程序

虽然这在 NTT DoCoMo 上只适用于日本的 N702is 3G 手机，但它太酷了，不能不给你们看。这款手机带有一个屏幕保护程序，当它被激活时，看起来就像一杯水的里面。702 有一个内置的运动传感器，所以当你倾斜它时，水位会改变以匹配你的手机的方向。不仅如此，它的水量还反映了你还剩多少电池电量！这是不是太棒了，还是太棒了？
