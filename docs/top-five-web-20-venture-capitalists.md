# 五大 Web 2.0 风险投资家 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/10/19/top-five-web-20-venture-capitalists/>

大多数风险资本家不理解标签、博客、rss、ajax 或双向网络。他们对社交书签网站嗤之以鼻，认为播客是一种时尚(也许是)。他们非常想“加入”这场网络复兴，但却懒得打开最简单的 RSS 阅读器，也不想了解分散内容的力量。简而言之，他们不明白。

大多数传统风险投资基金只有一个专注于网络的合作伙伴，他们可能对 web 2.0 消费者公司感兴趣，也可能不感兴趣。在很大程度上，这些风投并不是最有趣的交易的一部分。如果他们真的得到了你推销的东西，他们会向公司投入更多的钱。

以下是我试图指出那些真正明白这一点的风险投资家们。其中一些有新的基金，最近才成为风险资本家。有些已经存在很久了。只有两种共同的性格特征。首先，他们明白网络发展已经发生了根本性的转变。其次，他们渴望找到交易，主动寻找下一个机会的时间比他们坐在那里看自然流入任何风险基金的商业计划流的时间要多。

这个列表纯粹是主观的。这是明显的偏见。一些风投不属于这里，而许多属于这里的风投被拒之门外。尽管如此，这是我在过去几周采访企业家、风险投资家和记者时列出的清单。

## 大卫·考恩

 [大卫·考恩](https://web.archive.org/web/20230216134402/http://www.bvp.com/about/bio.asp?id=7)是[贝塞麦风险投资公司](https://web.archive.org/web/20230216134402/http://www.bvp.com/)的合伙人，他写了一个名为[的博客，谁有时间写这个](https://web.archive.org/web/20230216134402/http://whohastimeforthis.blogspot.com/)。他在这个名单上，部分原因是因为他孵化了目前网络上最热门、最受期待的公司 Flock。

这家伙挑选赢家。在 43 项早期投资中，19 项已经上市，16 项被收购。这可能是因为他的教育——他拥有哈佛大学的数学/计算机科学学位和工商管理硕士学位。

## 蒂姆·德雷珀

蒂姆·德雷珀投资了 Skype。完成了。

他还是 SocialText 的董事会成员，他的基金在百度。

为了合适的交易，蒂姆会去世界上任何地方。大多数净资产只有他 1/10 的风投不会离开硅谷去开董事会。

## 大卫·霍尼克

David Hornik 是 August Capital 的普通合伙人，他写了一个拥有超过 10，000 名 RSS 读者的博客。

大卫最大的优点是他认识每个人。他是那种你想边看足球赛边喝啤酒的人。如果你是他的朋友，他会很乐意把你介绍给任何人，并且经常把两个人放在一起，只是因为他认为他们应该互相认识。

他投资了定义当今网络的公司:SixApart 和 Technorati 就是其中之一。即使他不投资，他也会比大多数投资过的风投做得更多来帮助你。这里有一个例子——看看[对潘多拉](https://web.archive.org/web/20230216134402/http://www.ventureblog.com/articles/indiv/2005/001216.html)的评价，潘多拉不是一家投资组合公司。

大卫在许多场合告诉我“网络是一个生态系统”，他想为它的健康发展尽自己的一份力量。他当然知道。

## 乔希·科佩尔曼

乔希·科佩尔曼通过 [FirstRoundCapital](https://web.archive.org/web/20230216134402/http://www.firstroundcapital.com/) 悄悄过滤了几乎所有年轻的 web 2.0 公司，并投资了其中的许多公司。

乔希拥有沃顿商学院的 MBA 学位，是 Half.com 的创始人，该公司于 1998 年被卖给了 ebay。他的投资包括 Feedster、LinkedIn、Del.icio.us，据说还有 Browster(以及其他公司)。

Josh 比大多数企业家更了解 web 2.0。他作为企业家和白手起家发展庞大企业的经验是一笔不可思议的财富。

## 弗雷德·威尔逊

弗雷德·威尔逊是联合广场风险投资公司的创始合伙人，并撰写了非常受欢迎的风险投资。如果你是 web 2.0 的新手，从他的博客 1.0 的文章开始。然后读他博客的其余部分(甚至是关于音乐的东西)。

弗雷德是 del.icio.us 的最初主要投资者，那时候人们还对它摸不着头脑。这足够让他上名单了。但是联合广场的其他投资也可能是主要的全垒打。

从他的博客和人们对他的评价来看，很明显他花了很多时间在他的投资组合公司上。他是那种当你投入战斗时你希望站在你一边的人。

## 名人

杰夫·克拉维尔–[杰夫](https://web.archive.org/web/20230216134402/http://softtechvc.blogs.com/about.html)是一名前风险投资人，仍在进行零星的天使投资(Feedster、Truveo 和其他一些公司)。他的[新公司](https://web.archive.org/web/20230216134402/http://www.softtechvc.com/)允许他与预投资公司合作，让他们为黄金时间做好准备。去年，他与 Userplane、Buzznet、UltraDNS、Glenbrook Networks、Loomia 和其他两家未披露的公司合作。如果你有一个热门的想法，需要建议、资金、介绍和其他任何东西，杰夫可能会把你作为客户，尽一切努力帮助你取得成功——但像风投一样，他必须相信你的潜力。布拉德·菲尔德(Brad Feld)——布拉德是莫比乌斯风险投资公司(Mobius Venture Capital)的董事总经理，他写了一篇必读的 web 2.0 博客，名为 Feld Thoughts。如果你正在筹集资金，请阅读他在条款清单上的文章。我一直想见到他，但是还没有。布拉德的投资包括投资 NewsGator、Feedburner 和 Technorati。

O ' Reilly alpha tech Ventures——这是这里唯一的非人类。OATV 刚刚关闭了一只投资年轻公司的 5000 万美元的基金。鉴于蒂姆·奥莱利对这些公司的难以置信的影响力，OATV 可能很快成为 web 2.0 领域的一支重要基金。

**皮埃尔·奥米迪亚**–[皮埃尔](https://web.archive.org/web/20230216134402/http://pierre.typepad.com/)创立了 ebay，并且是 [Omidyar Network](https://web.archive.org/web/20230216134402/http://www.omidyar.net/index.html) 的联合创始人，在那里他投资了许多有趣的公司，包括 EVDB、SocialText 和 Feedster，以及一些未披露的公司。有个亿万富翁站在你这边总是好的。

彼得·瑞普是一个 1 亿美元基金的创始合伙人。彼得还写了另一个必读博客[早期风投](https://web.archive.org/web/20230216134402/http://earlystagevc.typepad.com/)。他的投资包括 ojos，这是一个令人难以置信的新照片元数据服务，将极具颠覆性(也非常有用)。就像这篇文章中的其他人一样，他是我想加入的人。作为一名企业家，他经历了许多战斗(他在 80 年代初创建了自己的第一家公司，并在 90 年代末在 Infoseek 工作)。

 ****彼得·泰尔**——贝宝前 CEO 彼得投资了 LinkedIn、Friendster、LinkedIn 和其他 web 2.0 公司。他刚刚创建了[创始人基金](https://web.archive.org/web/20230216134402/http://www.thefoundersfund.com/index.html)。迫不及待地想看看它能带来什么投资。

## 年轻的枪

还有三家风投公司尚未进行重大投资。但他们非常关注这个领域，渴望投资 web 2.0 公司。你会发现他们出现在每一个活动中，而且似乎决心要为自己扬名。我现在把它们放在这里是因为我很有可能在一年后更新这个列表时，它们会在最上面。

**托马斯·包尔**–[汤姆](https://web.archive.org/web/20230216134402/http://www.austinventures.com/team/teammember.asp?id=81)是 Austin Ventures 的风险合伙人，该基金管理着 30 亿美元。他是他们的消费者和 web 2.0 专家，似乎花了很多时间在硅谷和 web 2.0 活动上。在加入 AV 之前，Tom 是一名企业家，创办了几家成功的互联网公司。这为他的 web 2.0 投资哲学增加了一个独特的视角。他与包括 Pluck 在内的许多 AV 投资组合公司积极合作，我怀疑他将在不久的将来对 web 2.0 进行重要投资。

Dan Grossman**–[Dan](https://web.archive.org/web/20230216134402/http://www.venrock.com/bio_deg.html)是 Venrock Associates 的一名负责人，他最近开设了一个名为[A Venture fouth](https://web.archive.org/web/20230216134402/http://www.aventureforth.com/)的博客(他在这里写了一篇关于 [Ajax](https://web.archive.org/web/20230216134402/http://www.aventureforth.com/2005/09/06/top-10-ajax-applications/) 的帖子，被收藏了很多)。在文洛克之前，丹在 Garage.com。你还会在大多数 web 2.0 会议上发现 Dan，他为接触社区和网络做出了巨大的努力。**

Jason Pressman–[Jason](https://web.archive.org/web/20230216134402/http://www.shastaventures.com/pressman.htm)是 Shasta Ventures 的负责人，Shasta Ventures 是一家年轻的 2 亿美元基金，在以消费者为中心的业务方面有着深厚的承诺和专业知识。Jason 在来 Shasta 之前在 Walmart.com，是管理这个地方的一个小型高管小组的成员，他是战略、运营和业务发展副总裁。他对 web 2.0 有着敏锐的洞察力，并担任过该领域许多公司的顾问。Jason 知道如何将初创公司发展到大规模，他总是在思考他可以帮助哪些新公司达到这样的规模。**
