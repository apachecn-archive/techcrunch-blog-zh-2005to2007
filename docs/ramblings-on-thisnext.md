# 漫谈这个 next

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/09/11/ramblings-on-thisnext/>

 [](https://web.archive.org/web/20220818164529/http://www.thisnext.com/) 一些初创公司正在“社交购物”领域摸索，他们谈论的是如何从花费在研究产品而不是购买产品上的 80%的在线购物时间中赚钱。

这个领域的王者是 [CNET 评论](https://web.archive.org/web/20220818164529/http://reviews.cnet.com/)，它将编辑评论与大量图片和结构化数据以及直接用户评论结合在一起。五月，雅虎推出了雅虎科技(Yahoo Tech)产品(T6)，也直接针对 CNET 评论。当我研究产品时，我通常不会去比这两个网站更远的地方(尽管我在小工具博客上花了很多时间试图找出我打算研究的新东西)。当然，CNET 和雅虎科技只专注于科技产品。

其他服务也在这一领域推出，通常专注于愿望清单、推荐清单或两者兼有，并经常为用户提供工具，将推荐或想要的产品放在他们的博客或网站上。参见我们在 [Kaboodle](https://web.archive.org/web/20220818164529/http://www.beta.techcrunch.com/tag/kaboodle) 、 [Stylehive](https://web.archive.org/web/20220818164529/http://www.beta.techcrunch.com/2006/04/20/stylehive-is-looking-good/) 、[雅虎购物圈](https://web.archive.org/web/20220818164529/http://www.beta.techcrunch.com/2005/11/14/yahoo-shoposphere-launches-tonight/)和 [MyPickList](https://web.archive.org/web/20220818164529/http://www.beta.techcrunch.com/tag/mypicklist) 上的帖子。Wists 是这一类别中的另一个站点，尽管我们还没有写它。这些网站在其产品评论中远远超出了技术范畴。

这些网站通过会员费(通过电子商务网站的链接)和/或通过网站上的上下文广告(通常是谷歌)产生收入。

现在是 [ThisNext](https://web.archive.org/web/20220818164529/http://www.thisnext.com/) ，这是一家总部位于洛杉矶的公司，获得了 [NYT](https://web.archive.org/web/20220818164529/http://www.nytimes.com/2006/09/11/technology/11ecom.html?_r=1&ref=technology&oref=login) 的好评以及[杰森·卡拉卡尼斯](https://web.archive.org/web/20220818164529/http://www.calacanis.com/2006/09/10/shopcasting-with-thisnext/)的赞许(杰森曾与首席执行官共事，这可能解释了他为何提到该产品)。

这个网站是我迄今为止见过的最好的，有着清晰的项目结构和非常简单的方式让用户通过评论、标记、评论评级和添加东西到愿望清单来添加他们自己的观点。如果你想购买该商品，很容易找到“购买”链接。用户还可以创建一个网站小工具来展示他们真正喜欢的东西。好东西，执行得好。

但是我从来没有对像这样的网站和上面提到的其他网站感到兴奋。我不相信他们会成功地从 CNET 和雅虎这样的互联网大品牌那里吸引顾客，这些大品牌已经在整合用户评论、编辑评论和内容方面做得很好了。

NYT 和贾森都没有提到 CNET 已经在那里，并主导着这个技术产品领域。NYT 应该……作家鲍勃·泰德斯基显然打算给 ThisNext 一个积极的评价(配有一张首席执行官拿着冲浪板散步的照片)，但他似乎永远无法指出它为什么如此伟大。

然而，杰生钉它。他认为 Next 会成功，因为它将展示消费者喜欢的新趋势。

> 这有点像把“临界点”的概念变成一个你可以实际使用的产品。例如，随着社区的发展，你将能够首先看到谁推荐了一个产品。一段时间后，你可能会找到擅长发现——或创造——趋势的人。

CNET 不这样做，他们只是简单地评论产品，并把他们的评论放在网站上。它们没有显示出某些产品是如何随着时间的推移而变得炙手可热的，也没有显示出人们擅长挑选哪些赢家。

这种类型的东西可能会找到一个合适的位置——它不会取代致力于谈论特定领域最新最棒产品的博客，也不会取代像 CNET 这样的大型评论网站，让人们可以找到他们已经知道的产品的评论。但是也许他们可以为那些想成为决定什么产品是热门，什么不是热门的社区的一员的人服务。

我兴奋吗？不。但我有足够的知识在凌晨 3:30 写这个。或者也许我不兴奋*因为*现在是凌晨 3:30。我不知道。我们看看会发生什么。
