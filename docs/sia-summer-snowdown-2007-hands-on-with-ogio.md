# SIA 夏日雪景 2007:与奥吉欧牵手

![img_4769.JPG](img/89b6a0be69bd296de1aa83622b458505.png)

SIA 展是我期待了几个月的东西，虽然它不是冲浪博览会，但夏天的雪景将满足我户外运动的需求。有一吨很酷的东西，我会尽可能快地扔下去，所以请继续关注，因为每个人都有一些东西，从装备包到液体装甲服。

我们将以几个 OGIO 装备包开始党卫军的报道，任何相机/视频家伙都会喜欢，以及一个添加了一些耀斑的经典。首先出场的是 [Ty 视频包](https://web.archive.org/web/20201125111624/http://www.ogio.com/product.php?product=57&nav=0)。它的空间比你用小玩意填满它的空间还要大。主隔间包括大量的空间和填充物，可根据您的装备需求进行拆卸。甚至有一个外部三脚架携带带，所以你所有的装备都在一个包里。

![img_4770.JPG](img/27eb324da7df1458b5df956adae808a9.png)
正如您在这里看到的，所有拉链都经过涂层和保护，以确保您的相机/摄像机和所有配件保持干燥，以便您可以继续拍摄。

![img_4771.JPG](img/dcfe5da7bd52165c98a625f7ff68508c.png)

一个很好的笔记本电脑软垫隔层，这样你就可以在旅途中编辑和上传你的封面照片。

![img_4772.JPG](img/f278d0da74e8fe4f5f3220d74ad8d9af.png)

在我看来，这是包里最酷的部分，因为 OGIO 包里有一个 LED 手电筒，以防你在黑暗中需要找到超宽变焦镜头。

![urboe_black.png](img/691185d693fe843ca49914c1b6283189.png)

我挑选展示的下一个包是 [Urboe](https://web.archive.org/web/20201125111624/http://www.ogio.com/product.php?product=302&nav=0) 。它可能看起来像一个普通的背包，但它是装备包的缩影。所有隔间都有衬里和衬垫，因此它们随时准备发出隆隆声。笔记本电脑隔间紧靠您的背部，外壳看起来非常舒适。进出都那么容易。底部有一个甜蜜的保险杠，防止液体进入，为您的小狗提供额外的保护，以防您的包掉在地上。侧板可存放电源线和鼠标。Urboe 是一款非常棒的笔记本电脑包，适合经常外出的人使用，并且有足够的空间存放您随身携带的所有物品。它们现在以 59 美元的价格提供黑色、军红色和石榴石色。
