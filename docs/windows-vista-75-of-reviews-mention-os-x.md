# Windows Vista: 75%的评论提到 OS X

您不需要我们告诉您，视窗 Vista 的大部分是苹果公司的 [OS X](https://web.archive.org/web/20201031160730/https://crunchbase.com/product/os-x) 的翻版。你甚至不需要[CNN 来告诉你](https://web.archive.org/web/20201031160730/http://crunchgear.com/2007/01/31/cnn-calls-gates-out-on-vistas-design/)。不，你已经知道了，其他人也是。

在最近一项针对 Windows Vista 评论的调查中，无限循环发现 75%的评论都引用了 OS X。如果这不能说明问题，我们也不确定到底是什么。
