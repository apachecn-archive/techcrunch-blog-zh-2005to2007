# CrunchGear 与 Zunes 亲密接触

塞萨尔·梅嫩德斯，我们在 T2 Zune Insider 的兄弟博主，很友好地邀请我们去加强咖啡馆参加 Zune 的预发布会，他在那里为西雅图地区的所有 Zune 粉丝举办了一个小聚会。此外，他很友好地让我们拍了几张他带来的硬件的照片(界面的照片是不允许的，它仍然是一项正在进行的工作)。

我们期待一个单一的 Zune，这样我们就可以给你们一些齿轮色情(不要假装你不喜欢它)，但我们的人从雷德蒙带来了三个 Zune 的，每一个发射颜色。因此，下面我们有一些 Zunes 的快照(我们认为应该有比“Zunes”更好的多元化，但我们将留待以后)，以及一个让你们垂涎三尺的核心视频幻灯片，直到 11 月 14 日，当然，你们可以花 249.99 美元获得你们的。
