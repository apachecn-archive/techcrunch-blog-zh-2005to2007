# 苹果电视的 USB 端口现在被黑了，而且(几乎)功能齐全

![](img/f80ebac640fd78d1d4bc87b042b32784.png)

到目前为止， [Apple TV 的](https://web.archive.org/web/20200806011854/https://crunchbase.com/product/apple-tv) USB 端口就像你的附录:在那里，但并没有真正做什么。不可救药的黑客现在已经找到了一种解锁端口的方法，让苹果电视使用任意数量的不同 USB 设备(比如鼠标、键盘、硬盘等)。)另一面是来自*的最后一位苏格兰国王*Nicholas 摆弄一个新破解的 USB 端口的视频。

视频

自从上周发布以来，苹果电视就像最初的 Xbox 一样被黑客攻击。新的视频格式支持，[升级了](https://web.archive.org/web/20200806011854/http://crunchgear.com/2007/03/27/iresq-will-upgrade-your-apple-tvs-hard-drive-for-you/)内置硬盘，现在还有一个功能齐全的 USB 端口。USB 端口欺骗背后的黑客将在未来几天发布一个完整的步骤。去西部吧，年轻人，把苹果电视拆成碎片。
