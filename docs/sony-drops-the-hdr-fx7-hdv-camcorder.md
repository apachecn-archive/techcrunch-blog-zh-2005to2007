# 索尼放弃 HDR FX7 高清摄像机

![](img/21eec1488aee25ab4a2fc583501b43fb.png)

索尼在其高端机型的低端增加了一款新的 HDV 摄像机。HDR-FX7 使用三个 CMOS 成像审查，而不是典型的三个 CCD，它能够改善慢动作和弱光记录。它具有 HDMI 和 RCA 连接功能。FX7 的重量也只有 3.63 磅，比 FX1 轻 40%。

它没有达到目前可用的 FX1 的标准，但它应该与估计的价格差异大致平衡。我们可以假设索尼也将很快更新其高端机型。HDR-FX7 将于 10 月上市，建议零售价为 3500 美元。
