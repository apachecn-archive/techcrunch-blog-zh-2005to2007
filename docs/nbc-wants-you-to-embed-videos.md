# NBC 希望你嵌入视频| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/06/nbc-wants-you-to-embed-videos/>

[http://www.nbc.com/Video/export/swf/nbcsyndpl.swf?id=snlbackstage](https://web.archive.org/web/20160526214646/http://www.nbc.com/Video/export/swf/nbcsyndpl.swf?id=snlbackstage)

随着 YouTube 每天都在增长，在线视频的需求量非常大。不提供视频？继续前进。这是一个 Web 2.0 的世界，人们想要内容。NBC 抓住了这个概念，现在将提供视频，用户可以在他们的网站或博客上发布。像《会见新闻界》和《新闻联播》这样的节目片段会让你的读者无聊透顶。

所以，如果你想在你的博客上放些视频，去 NBC.com 找一个你喜欢的剪辑吧。不过，要找到一个可以立即嵌入的对象有点困难。我花了 10 分钟在网站上搜索，仍然没有找到一个。15 分钟后，我想到了上面的 SNL 片段。对于那些没有屈服于 YouTube 的人来说，这是一种支持。
