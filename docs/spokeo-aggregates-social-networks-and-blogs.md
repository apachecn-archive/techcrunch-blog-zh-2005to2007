# Spokeo 聚合了社交网络和博客

[![spokeologo.jpg](img/4b1d2c2844f0177a9a22a4aa3f8a8a1a.png) ](https://web.archive.org/web/20230105074028/http://www.spokeo.com/) [代言人](https://web.archive.org/web/20230105074028/http://www.spokeo.com/)想成为你的主页。他们想把你最喜欢的博客内容、MySpace 更新、新的 YouTube 视频和朋友的相册一目了然地呈现给你。

Spokeo 通过多媒体 RSS 阅读器将前 20 个社交网络目的地和你喜欢的任何 RSS 源结合在一起。但是你实际上不能在网络中导航。它只会让你疲惫不堪。有点像我的雅虎！对于 Web 2.0 网站。

“像‘我的雅虎’这样的起始页的问题在于 Spokeo 的创始人哈里森·唐(Harrison Tang)在周三的一次电话交谈中表示。“人们并不真正知道如何添加内容。这需要十个步骤，很多人不知道怎么做。因此，我们假设大多数人都对阅读同一页面上的在线信息感兴趣，这些信息会告诉你你的朋友在哪里，他们在拍什么照片，以及你最喜欢的博客(如 TechCrunch)在说什么。”【我发誓他说的是 TechCrunch！]

这是一个伟大的想法，但它需要一些工作。首先，(这是我内心的女人说的)，它不漂亮。它可以学习我的雅虎！至少告诉我当地的天气。此外，将信息导入网站并不容易。我试图用它来填充我的 MySpace 信息，但不知何故，结果却变成了 MySpace 好友和一个叫 Gnatalie 的人的信息。我不应该需要密码吗？因此，我花了很长时间才弄明白如何将 Gnatalie 从我的页面上移除，并在 MySpace 上获取我自己的信息。

您的 Spokeo 主页在边栏上分为三类:我、我的朋友和特色用户。然后，用户可以根据他们的标签或活动导航他们的朋友或受欢迎的代言人在做什么。

Spokeo 是由四个斯坦福的朋友创建的，他们热爱 MySpace 和 Flickr，显然也热爱 TechCrunch，他们厌倦了不能把所有信息都放在一个地方。到目前为止，这个网站是通过“天使投资者”资助的…意思是他们的父母。
