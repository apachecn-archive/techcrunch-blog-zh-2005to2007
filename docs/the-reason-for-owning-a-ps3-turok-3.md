# 拥有 PS3 的理由:Turok 3

![](img/fe5919877187a85e1429f7ccac68efd6.png)
地狱是的

很久以前，任天堂 64 曾经有一个简短的系列叫做*图罗克:恐龙猎人*。许多人不明白的是，Turok 是游戏史上最好的第一人称射击游戏之一。在大头作弊代码、广阔的关卡和出色的图形之间，你还得到了**杀死恐龙**。我觉得这已经是最好的了。

不过那是过去，未来就在现在。Turok 正准备进行下一代登陆，任何有 Wii 的人都不走运。今年秋天来到 PS3，Turok 就是 PS3 的初衷。Strategy Informer 上的这 27 张新截图展示了 Turok 将在 PS3 上拥有的令人难以置信的图形和郁郁葱葱的风景。如果我说我现在没有兴奋得头晕目眩，那我是在撒谎。九月不会来得更快了。
