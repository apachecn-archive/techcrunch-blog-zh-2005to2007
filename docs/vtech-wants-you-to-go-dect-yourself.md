# 伟易达希望你自己去 DECT

如果你还在寻找一款外观不丑的新无绳电话(说真的，为什么大多数无绳电话都这么丑？)并且不会受到周围其他无线信号的干扰(我的 802.11g 客户端接收了不少于 22 个网络，更不用说其他无绳产品的信号了)，看看这五款最新上市的伟易达 DECT 手机。DECT(数字增强型无绳电信)技术最近获准在美国使用，其通话质量和覆盖范围优于 2.4GHz 和 5.8GHz，且不受无线网络干扰。

6031 是基础型号，建议零售价为 49.95 美元，具有手机扬声器和音量控制、存储最近 50 次呼叫的来电显示以及蓝色背光键盘和显示屏等功能。6032 增加了一个应答机，6041 失去了应答机并获得了一部手机，6042(如上图)得到了应答机和两部手机，顶级的$99.95 6053 得到了三部手机和应答机。此外，它们看起来都很棒，键盘上还添加了一些 RAZR 的外观。

6053 可以在全国范围内的好市多和山姆俱乐部找到，而其他的可以在沃尔玛、塔吉特、Radio Shack 和在[伟易达的网站](https://web.archive.org/web/20210302021149/http://www.vtechphones.com/)上找到。
