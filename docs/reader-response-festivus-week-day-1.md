# 读者反应:Festivus 周第一天

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/18/reader-response-festivus-week-day-1/>

想想看，我们在五个多月前刚刚成立了 CrunchGear。它的发展很快，浏览帖子，几乎每个条目都有评论。当我想到我们的小网站成长得如此之快时，我有点伤感。很快它就会开始在不方便的地方长出毛发，并以新的方式思考其他博客。

我正为那些坐在椅子上焦虑不安的漫漫长夜做准备。手边有一杯尊尼黑，可以镇定神经，防止恐慌和胁迫。会发生的。我准备好了。但现在，我很享受那甜蜜纯真的阶段。神奇的世界。虽然我知道这不会长久，但这很好。

我想除了开始送人之外，没有其他庆祝方式了。所以我想宣布第一个读者反应的获胜者:Festivus 周。*鼓声*
