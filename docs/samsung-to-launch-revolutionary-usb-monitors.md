# 三星将推出革命性的 USB 显示器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/21/samsung-to-launch-revolutionary-usb-monitors/>

你是否梦想过有五台或更多的显示器连接到你的电脑上？唉，我们的显卡无法完成这项任务，但三星即将让这些香槟梦想成为现实。下个月在 CeBIT 上首次亮相的是通过 USB 连接的 19 英寸 Syncmaster 940UX。你问这怎么可能？用户将首次可以选择通过 USB 以及传统的 DVI 和 VGA 连接。

只要您的 CPU 能够处理负载，您可以选择连接尽可能多的显示器。显示器将依赖于 CPU 的图形泵魔术，而不是一个图形卡。你个人电脑的 USB 端口数量是唯一阻碍你获得宇宙先生般的 T2 地位的因素。

没有定价的消息，但预计它将于 5 月上市。唯一的缺点是，它是为有 Vista 驱动程序的 XP 用户设计的。更糟糕的是，OS X 和 Linux 用户可能根本看不到驱动程序。伤心难过。
