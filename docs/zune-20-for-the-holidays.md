# Zune 2.0 假期版| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/23/zune-20-for-the-holidays/>

![zune-jesse_m.jpg](img/d8f1a536f497cb4cfeb43a23d73020ec.png)

在 Whiz Byte 的极客们有关于 G2 Zune 的独家新闻，以及我们什么时候能看到它。据微软的一位销售代表说，我们可以期待在假日购物高峰时看到它。有点奇怪的是，微软想把 Zune 推向“与 iPod 平起平坐”，但话说回来，微软一直充当苹果的副手已经有一段时间了。真是浪费时间。热勺，嗖字节，热勺！

如果你非常想知道 MS 代表说了什么，那就来吧:

> 我们已经看到了传言，但我们还没有宣布下一代 Zune 设备或服务的具体日期或细节。也就是说，Zune 遵循消费电子产品周期的周期，所以你可以期待在今年晚些时候假期之前的更新。
> 
> 我们将沿着三条可预测的路径扩展 Zune:
> 设备家族——增加新的尺寸、款式和价位
> 功能——我们将在支持视频、播客等方面与 iPod building 并驾齐驱，同时建立无线功能、FM 调谐器、 等
> 地理市场——随着时间的推移，我们将开始扩大 Zune 的国际足迹，但只有当我们能够确保提供适合该市场的产品时
> , Zune 对消费者真正有利的一点是，该设备是经得起未来考验的。 消费者可以购买，而不用担心他们的设备会过时。Zune 的软件(固件)可以更新，所以当新软件推出时，他们的设备会自动更新新功能。
