# OCZ 发布水冷 RAM

散热器和散热器已经成为过去；OCZ 发布了水冷内存模块。OCZ 的 Flex XLC 版内存模块的核心设计元素是水冷选项。水冷 DDR2 PC2-6400 CL 3 flex xlc Edition 2GB 内存套件旨在实现低速时的最高效率。如果您的主板支持 EPP，这些 800MHz 模块会自动运行在 CAS 3-4-4。这允许真正的即插即用超频功能，使手动配置过时。其他特性包括高达 2.4V 的 EVP 保护、6.4 GB/s 带宽和 12.8 GB/s 内存带宽。零售价在 265 美元左右。
