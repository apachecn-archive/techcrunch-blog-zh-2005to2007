# 史蒂夫和比尔:打！–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/31/steve-and-bill-fight/>

实际上，他们有点喜欢对方。听听史蒂夫轻柔轻快的声音，观察一下比尔脸上羞涩的微笑。想象自己是绿屋子里墙上的一只苍蝇*波浪形的电视画面和音乐*

“嗯，就这样，比尔？”史蒂夫弄平了他的高领毛衣，用熨斗擦掉了一些棉绒。
“圆周率！”比尔说，放下他的 iPaq。然后他对 OS X 进行逆向工程，然后将 WinFS 纳入核心。
 “那是什么账单？”史蒂夫问道。他走向比尔，好像要拥抱他，但又弯下腰去用一些 iPolish 擦鞋。
“霹！我能背到第一百名。”比尔说。他开始查看电灯开关，并在上面安装了 Vista。
“好的，比尔。让我们出去，让他们大吃一惊。梅林达给我做了冰淇淋。是的，她说了，比尔。是的，她做到了。"
