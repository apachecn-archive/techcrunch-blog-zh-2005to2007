# 使用 Webaroo 离线存储网页内容

 [](https://web.archive.org/web/20220516085719/http://www.webaroo.com/) 对于我们这些有时仍然离线，想要访问至少一些网络内容(我)并运行 Windows 机器(不是我)的人来说，位于圣克拉拉的 [Webaroo](https://web.archive.org/web/20220516085719/http://www.webaroo.com/) 将是一项有用的服务。今天发布。

Webaroo 将“最高质量”的网站内容编入索引，并创建基于主题的网页包供下载。这些网络包中的内容离线存储在您的计算机上，并定期更新。为了测试它，我启动了我的旧电脑，安装了 5 MB 的应用程序，并下载了《世界新闻》和《旧金山网络包》。你也可以让 Webaroo 为特定网站编制索引，以便离线查看。我加了 TechCrunch。

然后我拔掉了网，试了一下。网络包很棒，让我可以搜索或浏览内容。在飞机上我会喜欢这个的。具体的网站索引并没有很好地工作——所有的格式和 CSS 都被从页面中去除了，网站看起来很糟糕。尽管如此，内容还是在那里。

Webaroo 还允许我选择索引链接到网站的内容，因此 TechCrunch 的链接也是可见的。很棒的功能。

Webaroo 也适用于运行 Windows Pocket PC 操作系统的移动设备。他们宣布了与 PC 制造商宏碁的第一笔交易，在新的宏碁笔记本电脑上预装 Webaroo。更多关于 [Memeorandum](https://web.archive.org/web/20220516085719/http://tech.memeorandum.com/060410/p60#a060410p60) 的信息。
