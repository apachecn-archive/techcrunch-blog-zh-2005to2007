# 抱歉，麦克 OS X 豹没有 ZFS

[![nozfs.jpg](img/627dd264c0847faff978dc2718b25592.png)](https://web.archive.org/web/20210304133353/http://old.crunchgear.com/wp-content/uploads/nozfs.jpg "nozfs.jpg")

我有一些毁灭性的消息要告诉你:ZFS 不会成为 Mac OS X Leopard 的默认文件系统。至少从今天，也就是 2007 年 6 月 12 日开始。Leopard 的一位高级营销人员明确表示，“‘ZFS’是不可能发生的”，这与孙上周所说的话完全相反。Leopard 即将使用 HFS+文件系统，你知道吗？

真正的考验将是当一些行善者将[豹 WWDC](https://web.archive.org/web/20210304133353/http://crunchgear.com/2007/06/11/help-key-what-youll-use-in-leopard/) 测试版泄露到新闻组或者我最近听说[的那些种子网站之一。在那之前，我们仍然一无所知。](https://web.archive.org/web/20210304133353/http://crunchgear.com/2007/05/14/help-key-the-essential-guide-to-piracy/)
