# 未来主义者:MacWorld ' s-A-coming 或……完美的音乐手机——TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/28/the-futurist-macworlds-a-comin-or-the-perfect-musicphone/>

MacWorld 即将来临，随之而来的是对 2007 年的无休止的猜测，这一年不仅是苹果公司将帽子扔进手机圈的一年，也是音乐手机真正取代你选择的数字音乐播放器的承诺最终实现的一年。

当然，这一切对我们来说都似曾相识。我很确定 2006 年应该是音乐手机年，甚至可能在那之前的 2005 年。消费者还没有真正接受音乐/手机融合有很多很好的原因，这不仅仅是缺乏苹果制造的音乐/手机融合。

那么，怎样才能制造出完美的(苹果或非苹果)音乐手机呢？点击跳转查看在我们永远离开我们的 iPods 之前需要做的事情……

**问题:耳机插孔过小**
也许音乐手机广泛采用的最大障碍是制造商愚蠢地坚持在他们的产品上填充微小的 2.5 毫米耳机插孔，而不是普通 MP3 播放器上的 3.5 毫米插孔。大多数人都不喜欢使用任何尺寸可变的适配器，而且意识到你 300 美元的一副眼镜放不进你的手机，这通常足以让人们在很长很长一段时间内对你失去兴趣。

当然，你在 90%的手机上找到 2.5 毫米是有一个熟悉的原因，这不是尺寸限制(如果超薄 Nano 可以适合 3.5 毫米插孔，你的 RAZR 可以)。确切地说，是我们的好朋友老美国佬。正如一位手机高管最近私下告诉我的那样，如果你需要一副单独的耳机，你必须花更多的钱购买更多的配件。我甚至不能告诉你这种态度对创新有多糟糕，我也不应该告诉你。

**问题:内存太小**
RAZR、SLVR 和(上帝禁止的)ROKR 被宣传为音乐手机，但一旦你加载了 Bejeweled、你的联系人和 130 万像素的聚会照片，就几乎没有足够的空间来播放足够的音乐让你度过早晨的通勤。除非手机的容量比 Shuffle 更大，否则我不会把我的 DAP 放在家里。

问题:笨重的手机
当然，也有专门销售的音乐手机，其容量足以容纳你新买的汤姆·威兹盒装手机。以诺基亚 N92 为例，它是芬兰的 4-gigger 手机。不幸的是，这是一种荒谬的手机，像大多数手机一样，在一个或两个以上的包，应该比 RAZR 和 Nano 粘在一起笨重。与亚洲人或欧洲人不同，美国人重视微小胜过任何前沿特征。由于像 Nano 和新 Shuffle 这样的 DAP 向我们展示了它们可以压缩几个内存的能力有多小，现在官方没有理由继续这种扩大口袋的废话了。

**问题:低劣的流媒体音乐**
最终，音乐手机的未来几乎肯定会是流媒体内容——它们永远在线的无线连接是一个杀手级应用，应该会让你的 Zune 的 Wi-Fi 哭出来。任何想要一窥这项技术潜力的人，只要看看像 Sprint A900(又名..刀片)。如果你有一个高速网络来传输质量不错的音乐(通过与 Sirius 的许可协议提供)，手机内存不足应该不是问题。)不过，这项技术还太年轻。目前 EV-DO 的网络限制甚至阻止你在播放音乐的时候收到来电通知，这让你的电话毫无用处。此外，可用的内容是有限的(Sirius 只授权他们的几个音乐站，没有他们的谈话——所以没有霍华德),而且价格有点荒谬。尽管如此，一旦你的手机能够让你在摇滚的时候保持连接，并且可以以合理的价格获得合理的内容，这几乎肯定会是音乐手机的未来。不幸的是，苹果的历史表明，这一功能几乎肯定不会是他们首次涉足手机的一部分。
