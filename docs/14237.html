从Twitter转向Jaiku | TechCrunch

原文：https://web.archive.org/web/http://www.techcrunch.com/2007/05/17/making-the-switch-from-twitter-to-jaiku/trackback/

“我想我看到了一个油灰纹身。我看到了，我看到了一个油灰纹身！”

这些著名的话从来不是Tweety Bird的最后一句，但对于越来越多的Twitter用户来说，它们将是最后一句。他们在放弃Twitter之前看到的最后一个东西将是Twitter服务器猫，这是服务完全关闭时的默认屏幕。

不仅仅是Twitter最近的宕机让这项服务处于令人沮丧和无用之间。即使当Twitter启动时，更新/刷新失败，页面无法加载，第三方工具无法连接。已经有很多停工。

Twitter是一项你想要热爱的服务。就像Evan William的早期创业公司Blogger一样，它不仅成为了市场领导者，而且在创建一个专注于即时通讯的新的在线服务市场方面发挥了至关重要的作用。

我们在2006年9月报道了竞争服务，但随着Twitter上的大众开始关注其他地方，一项服务正在迅速被接受。服务是 Jaiku 。

它没有Twitter那样的人流量，但至少你可以在Jaiku上可靠地与人交流。像Leo Laporte这样的高调皈依者为越来越多寻求改变的Twitter用户引领了道路。

新平台需要新工具。下面是一些尝试在Jaiku上重建(积极的)Twitter体验的开始。

视窗

Jaikaroo :类似于Twitteroo的桌面应用，灵活的可定制界面

Jay-q :又一个桌面app，界面丑但是好用。

Mac

聚狐:为Jaiku提供Twitterific风格的服务。为什么所有漂亮的应用最后都在Mac上？

Jaiku Dashboard Widget :从Mac OS X桌面发布到Jaiku，无需使用浏览器。

十字平台

NitWit :版本适用于Linux/ Mac/ Windows，支持Jaiku和Twitter

斯蒂夫·克里福德有一个更广泛的指南，包括移动应用和对WordPress的支持。我唯一找不到的是一个Twitbin风格的Firefox插件，可以在浏览器中使用，但肯定会有人很快开发出来。如果你有任何喜欢的Jaiku应用程序想要分享，请在评论中告诉我们。
