# 垃圾邮件猎人成为掘客垃圾邮件最好的朋友

新服务[集体行动](https://web.archive.org/web/20220928183306/http://www.collactive.com/)来到这个世界是失败的以色列机构蓝色安全背后的团队的好意，蓝色安全因运行反垃圾邮件服务蓝蛙而闻名。起初，BlueFrog 的拒绝服务攻击非常流行(这是一个道德制高点上的拒绝服务攻击),让垃圾邮件发送者感到不快，去年 5 月，所有人都一败涂地，SixApart 和其他公司都被搞垮了。他们随后让步，关闭了这项服务。

我们当时不知道的是，在与垃圾邮件的战争中失败后，他们实际上为对方制造武器。

Collactive 为他们所谓的所有要点公告(APB)提供分发服务。你在 Digg 或 APB 系统的任何地方添加一个社会化书签样式的页面，然后其他人会得到这个页面的通知。在这一点上，服务有所不同。用户被引导去收集它自己，而不是直接收集被标记的页面。该页面在屏幕左侧显示一个集体框架，其中包括所需操作的注释，例如“为此投票”。你提交页面的流量来自其他集体用户，此外还支持给朋友发电子邮件，一个浏览器扩展和一个显示你的 APB 的博客小部件。

该网站有无争议的用途，任何种类的页面都可以列出。然而，在使用方面的真正意图是通过网站清楚地推广。当我访问该网站时，排名第一的 APB 是请求对 Digg 的一个故事进行投票，其他社交网络也在 APB 之列，社交网站的徽标被用于宣传材料中。

Collactive 由红杉投资。是的，*那棵*红杉。

也许这项服务有比滥发 Digg 更多的内容？当然，除了一个听起来像药物的名字之外，还有一种药物；它也适用于垃圾邮件 Reddit 和 Netscape！
