# 8 月推出新的人脸识别相机

三星刚刚宣布了两款即将上市的数码相机 L730 和 L830。摄像头将分别为 700 万像素和 800 万像素，并将采用三星的智能人脸识别技术，这是一种新型技术，*自动识别并聚焦于人的面部。*

视频捕捉模式将在四种不同的质量设置下运行，从 320 x 240 到 800 x 592，这是一种奇怪的分辨率，但这一信息来自英国的新闻稿，所以当这些相机在美国上市时，这个数字可能会变成 800 x 600。这两款设备都配有 3 倍光学变焦、2.5 英寸液晶显示屏和 16Mb 内部存储，并可选择插入高达 4GB 的 SD 内存。

在 8 月底寻找它们，价格在 240 美元(L730)和 300 美元(L830)左右。
