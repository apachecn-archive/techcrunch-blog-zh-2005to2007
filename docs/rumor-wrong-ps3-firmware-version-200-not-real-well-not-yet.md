# 谣言错误:PS3 固件版本 2.00 不真实(嗯，还没有)| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/15/rumor-wrong-ps3-firmware-version-200-not-real-well-not-yet/>

[![ps3v200.jpg](img/6ed48d853b067beb1c24742c46bb477e.png)](https://web.archive.org/web/20160305142900/http://old.crunchgear.com/wp-content/uploads/ps3v200.jpg "ps3v200.jpg")

不，假警报。网络上流传的谣言详细描述了 PS3 固件版本 2.0 的[假定功能](https://web.archive.org/web/20160305142900/http://www.ps3scene.com/news/static/PlayStation3Firmwarev2.0FeaturesRevealed-1179117253.php)，包括游戏内 XMB 访问和内置 RSS 频道。但是，是的，这不是命中注定的。泄露功能列表的人“Leah”现在声称她的留言板帐户(发布列表的地方)被黑客攻击，列表是假的。现在，这并不是说我们永远不会看到这些功能的一天，但在它目前的化身谣言是假的。
