# 诺基亚 N77 将在 3GSM 发布

注意到市场上缺乏负担得起的电视手机，诺基亚将于下周在 3GSM 发布 N77。这将是中等价位，所以诺基亚可以渗透到大众。“当手机价格达到 200 欧元(250 美元)-300 欧元(400 美元)的价位时，移动电视就会进入大众市场，”该消息人士说。

N77 使用 DVB-H 技术接收电视信号。没有其他细节或发射日期已经宣布，诺基亚是保持沉默，直到 3GSM。
