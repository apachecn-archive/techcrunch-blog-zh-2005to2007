# T-Mobile Wing 发布

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/05/22/t-mobile-wing-released/>

![](img/724e7f96a09c65b6caecf511d0d57a63.png)

不是说你不知道机翼，但你们中的 T-Mobile 现在可以购买机翼。这款支持 MyFaves 的手机并不便宜，两年合约的价格为 300 美元，但功能丰富吗？除了滑动式 QWERTY-pad，您还可以找到蓝牙 2.0、WiFi、EDGE、全 HTML 浏览器、推送电子邮件、microSD、媒体播放器和具有 8 倍数码变焦的 200 万像素摄像头。

Wing 最突出的一点可能是它搭载了 Windows Mobile 6.0。尽管 WM6 比 5.0 有了很大的改进，但它仍然是 Windows Mobile。你要么喜欢，要么讨厌。如果你想像鸟儿一样用翅膀飞走，就去当地的 T-Mobile 商店吧。今天晚些时候期待我们的全面审查。
