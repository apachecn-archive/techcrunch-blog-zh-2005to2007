# Shifd:《纽约时报》的一款智能手机应用

来自《纽约时报》的移动应用软件[![shifd.jpg](img/e4aac3df654b7ea15f36314f0dac726b.png)](https://web.archive.org/web/20221203084224/http://nytlabs.com/shifd/)[Shifd](https://web.archive.org/web/20221203084224/http://nytlabs.com/shifd/)允许用户在台式电脑和移动电话之间轻松共享任何内容，从网页提要、列表和地图到个人笔记和数据。

原型通过放置在手机内部的 RFID 芯片记录存在；当用户移除手机时，数据会通过网络界面同步。

下面的视频对此有更好的解释；这不是一个很容易解释或与现有产品相比较的应用程序。
