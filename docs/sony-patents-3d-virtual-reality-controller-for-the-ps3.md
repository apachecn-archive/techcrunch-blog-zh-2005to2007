# 索尼为 PS3 的 3D 虚拟现实控制器申请专利

[![3dsony.jpg](img/b140ed4b7921aba97ebbc2f837ceb76e.png)](https://web.archive.org/web/20210225192117/http://old.crunchgear.com/wp-content/uploads/3dsony.jpg "3dsony.jpg")

索尼为 PS3 和 Vaio 电脑申请了 3D 控制器的专利，它不仅能对你疯狂的[挥舞动作](https://web.archive.org/web/20210225192117/http://crunchgear.com/2007/02/02/conan-whips-serena-in-wii-tennis/)做出反应，还能对你的手握和施加的压力做出反应。正如图中精彩展示的那样，控制器握在你的手掌中。也有触觉反应，所以可以说你能够[感受游戏](https://web.archive.org/web/20210225192117/http://youtube.com/watch?v=O3oGylkA2S4)。我敢打赌，索尼的真正快乐 it 终于[在几周前用力反馈“发明家”沉浸解决了](https://web.archive.org/web/20210225192117/http://crunchgear.com/2007/03/19/sony-and-immersion-finish-legal-battle/)。

最重要的信息——这项专利是否会成为真正的产品——目前还不得而知。
