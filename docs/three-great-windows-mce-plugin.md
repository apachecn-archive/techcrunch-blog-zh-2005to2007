# 三个伟大的 Windows MCE 插件

随着 Windows Vista 发布日期的临近，美国微软 Windows Media Center 的早期采用者需要停止沾沾自喜，并开始为即将到来的事情做好准备——例如，通过 CableCARD 的高清内容。但在此之前，这里有三个很棒的媒体中心插件，可以让你在媒体中心听 iTunes 购买的歌曲。

通过扩展，我们认为 iTunes 插件将允许人们在他们的 Xbox 360s 上听他们的 iTunes 音乐，因为 360 只是 MCE 的一个扩展。还有什么比一边听着从 iTunes 上买的《机器人先生》Domo Arigato 的歌，一边在东京的街道上飙车更有趣的呢？没什么，就是这样。
