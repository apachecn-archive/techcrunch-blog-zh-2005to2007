# YouTube 上的 Beeb:BBC+YouTube 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/02/the-beeb-on-the-yout-bbc-youtube/>

和 BBC 新闻，而且没有广告。BBC 全球内容将播放广告和热门节目的剪辑，最后，BBC 新闻将只在英国以外播放，也将播放广告。它将包括来自常规新闻报道的剪辑。

在世卫组织的互联网概念上，英国广播公司(BBC)在英国被称为“烟雾”( Old Farlettrousers)一直处于领先地位，他们只能为在 YouTube 上提供高质量的酷而鼓掌。现在，我们喝点茶，在芭比娃娃上再放一份牛排和腰子馅饼，好吗？
