# SimpleTech 放弃了两个巨大的 3.5 英寸固态硬盘

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/04/19/simpletech-drops-two-ginormously-tiny-35-inch-ssd-drives/>

SimpleTech 宣布了几个 3.5 英寸的巨型 NAND 存储驱动器，我想我的心脏漏跳了一拍。你真的需要 3.5 英寸 512GB 或 256GB 的固态硬盘吗？可能不会，但你可以为它辩护。SimpleTech 声称宙斯-IOPS 硬盘的随机事务处理性能比 15,000rpm 的企业硬盘快 200 倍。像哇哦！256GB 驱动器目前已经上市，512GB 驱动器将在第三季度上市。价格肯定会高得离谱，但该死的是，如果它不适合 DVR/媒体中心使用。
