# Pownce 在易贝邀请出售

为了证明一切都有市场，凯文·罗斯的 P2P 服务 [Pownce](https://web.archive.org/web/20220927021728/http://www.crunchbase.com/company/pownce) 正在[易贝](https://web.archive.org/web/20220927021728/http://search.ebay.com/pownce_W0QQfromZR40)上出售。

Pownce 邀请的起拍价为 1c，现在购买价格为 5 美元。

就在不久前，Gmail 邀请还可以在易贝上出售，尽管我认为(当时)这个极具创新性的电子邮件平台和 Pownce 这样的 P2P 平台有着天壤之别。对于那些渴望得到邀请的人来说，问题变成了:是或不是，谢谢；Pownce Desktop Client 用户每天至少会看到十几次错误消息上的确切选项。
