# 周末 wee war | TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/06/02/weewar-for-the-weekend/>

 [](https://web.archive.org/web/20160421213321/http://www.weewar.com/) 我刚碰到 [Weewar](https://web.archive.org/web/20160421213321/http://www.weewar.com/) ，一款融合了冒险元素和其他策略游戏的网络战争游戏。这是相当简单的像素艺术，但他们在测试期间获得了极其忠实的追随者。

像 [Kdice](https://web.archive.org/web/20160421213321/http://www.techcrunch.com/2006/12/17/i-spent-sunday-evening-playing-kdice/) (我写这篇文章的时候正在玩这个游戏)[桌面塔防](https://web.archive.org/web/20160421213321/http://www.techcrunch.com/2007/04/16/desktop-tower-defense-should-be-banned/)和其他我们报道过的游戏一样，Weewar 是一个非常容易上瘾的游戏。你会很快被吸引去和其他人类对手玩上几个小时。

这是一个回合制游戏。任何玩家都可以开始游戏，首先从[地形](https://web.archive.org/web/20160421213321/http://weewar.com/map/narrowpath)中选择一个[号](https://web.archive.org/web/20160421213321/http://weewar.com/map/fortress)，然后邀请其他玩家。你接管工厂，让他们生产更多的军队，坦克等等。然后继续接管更多的工厂或与其他竞争者竞争。不是每个人都必须在同一时间玩游戏。轮到你了，然后下一个玩家会收到电子邮件通知，说该轮到他们了。

这款游戏仍处于私人测试阶段——他们的[博客](https://web.archive.org/web/20160421213321/http://blog.weewar.com/)指出，他们已经有多个服务器崩溃，同时有太多玩家运行缓慢。在此注册[进入测试版。](https://web.archive.org/web/20160421213321/http://weewar.com/requestInvite)
