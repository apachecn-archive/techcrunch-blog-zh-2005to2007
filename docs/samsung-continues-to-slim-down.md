# 三星继续精简液晶显示器

[三星又做到了。](https://web.archive.org/web/20200806021253/http://crunchgear.com/2007/01/22/samsung-you-have-a-problem/)三星 SDI，三星显示接口公司，已经超越三星电子，制造出世界上最薄的手机液晶显示器。最新的原型只有 0.74 毫米厚，而三星电子的 LCD 只有 0.82 毫米厚。SDI 还拥有世界上最薄的有机发光二极管，厚度仅为 0.6 毫米。目前还没有大规模生产的计划。

SDI 也将在下一季度出货 1.9 毫米液晶显示器，它们将被用于 [Ultra Edition II。我甚至不确定我是否想要一部那么薄的手机，我可能会在 5 分钟内把它弄坏。](https://web.archive.org/web/20200806021253/http://crunchgear.com/2007/02/13/hands-on-with-all-the-samsung-ultras-can-you-say-ultra-purdy/)
