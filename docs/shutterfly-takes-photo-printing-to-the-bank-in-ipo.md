# Shutterfly 将照片打印带到 IPO 的银行

 [](https://web.archive.org/web/20211202093004/http://shutterfly.com/) 在线照片分享和打印服务 [Shutterfly](https://web.archive.org/web/20211202093004/http://shutterfly.com/) [今天在 IPO 中筹集了 8700 万美元](https://web.archive.org/web/20211202093004/http://online.wsj.com/article/SB115954277498678039.html?mod=hpp_us_at_glance_technology)，收盘上涨 3.7%，每股 15.55 美元。VentureBeat 指出，价格曾一度攀升至 16.73 美元。

出生于 1999 年的 Shutterfly 是上一次泡沫和崩溃的幸存者。现在，这是一个考虑其他可能估值的有趣案例。这也让我们有机会看看一家成功上市的公司是如何上市的。相比之下，据传 Flickr 已经被雅虎收购。3000 万到 3500 万美元。

在摩根大通、Piper Jaffray 和杰富瑞出售了 Shutterfly 2360 万股中的 580 万股后，该公司以超过 3.5 亿美元的市值收盘。该公司约有 200 名员工，去年的收入约为 8400 万美元，净利润率不到 34%。该公司迄今已获得 6700 万美元的风险投资。网景公司的联合创始人吉姆·克拉克和 Shutterfly 董事会主席购买了今天售出股票的 30%。

在这个时候，IPO 是 Web 2.0 初创公司最不常见的流动性事件之一，相对于我们在这里介绍的所有初创公司，一个图片共享网站保持独立并上市是有趣的。

Shutterfly 是一家有大量开销的公司，面临竞争对手的降价，并不缺乏寻求进入其市场的小型创业公司。它的大部分收入来自航运，并面临着可扩展性方面的问题。在 MrWaveTheory 有一些有趣的分析。

虽然 Shutterfly 提供 150 多种印刷和 DVD 产品，但照片印刷初创公司，如[onetrumedia](https://web.archive.org/web/20211202093004/https://beta.techcrunch.com/tag/OneTrueMedia)、 [Tabblo](https://web.archive.org/web/20211202093004/http://www.beta.techcrunch.com/tag/tabblo) 和 [Scrapblog](https://web.archive.org/web/20211202093004/https://beta.techcrunch.com/tag/scrapblog) 提供更复杂的设计能力、更多的病毒式在线分发，在某些情况下还拥有知名的媒体合作伙伴。大多数在线照片分享网站现在允许用户打印或刻录他们的档案到 DVD 上，尽管很少有网站像 Shutterfly 那样将整个商业模式建立在打印上。

Shutterfly 在用户体验上提供了什么？明显吸引主流用户的产品。

*   无限免费存储。
*   适用于 Windows 和 Mac 的优秀批量上传器(尽管不是英特尔 Mac)。
*   公共和私人相册。
*   共享收藏的小组协作。
*   干净的网址，对照片的评论，基本的作物，边界，红眼消除和颜色变化的影响。
*   他们还为经销商提供商业解决方案。

它的主要竞争对手，惠普的 SnapFish 提供了许多相同的功能，外加通过电子邮件进行移动上传。

在 Shutterfly 中，除了网站导航之外，我觉得一切都令人愉快。作为用户，我的一个抱怨是它不提供标签。相册导航意味着一张照片无法在多个群组中找到。标签最近流行起来是有原因的——它真的很有用，让一组页面更容易移动。

公司上市后会成功吗？很难说，考虑到它自己对管理费用、价格竞争和日益多样化的市场的担忧。不过，你不能说 Shutterfly 是静止不动的。这种新的现金注入，它提供的当代功能及其已经庞大的用户群看起来肯定不错。
