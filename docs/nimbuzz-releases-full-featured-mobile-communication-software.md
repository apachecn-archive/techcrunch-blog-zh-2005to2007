# Nimbuzz 发布全功能移动通信软件

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/08/22/nimbuzz-releases-full-featured-mobile-communication-software/>

[](https://web.archive.org/web/20160915170751/http://nimbuzz.com/)mobile runch 的 Oliver 今天早上[在](https://web.archive.org/web/20160915170751/http://mobilecrunch.com/2006/08/22/nimbuzz-could-change-the-face-of-mobile-communications-a-mobilecrunch-exclusive/)发布了一项名为 [Nimbuzz](https://web.archive.org/web/20160915170751/http://nimbuzz.com/) 的新免费服务的独家报道，他相信这项服务将会真正震撼移动世界。用户可以在个人和小组聊天中与 MSN 和 GoogleTalk 用户进行即时通讯，以 10 美分的价格在世界各地发送短信，留下和检索信息，并以本地电话的价格在世界任何地方打电话。用户还可以互相发送快速的蜂鸣声或单次铃声，以通知重要的事情。移动设备和 PC 之间的 Nimbuzz 通信也是可能的。

这项服务是荷兰 Rijswijk 的 Buzzaa 媒体集团的产品。该公司最近获得了红树林资本合作伙伴的资助，这是一个为 Skype 提供初始资金的组织。查看[奥利弗的帖子](https://web.archive.org/web/20160915170751/http://mobilecrunch.com/2006/08/22/nimbuzz-could-change-the-face-of-mobile-communications-a-mobilecrunch-exclusive/)了解更多细节和分析。
