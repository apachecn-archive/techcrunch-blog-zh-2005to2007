# FCC 的 Sidekick Shuriken 幻灯片

![shuriken-fcc.jpg](img/f3bc161878b47c95e09b81fdaf5ef5b6.png)

危险手里剑又名 Sidekick LX 又名 Sidekick 4 已经出现在 FCC 网站上，我和其他人一样对细节感到失望。根据型号 PV250 (SK3=PV200，SKiD=PV150)，我们知道它是该家族的下一代产品，但它缺乏 3G 支持，这有点道理，因为 T-Mobile 还没有拿出足够长的时间来做点什么。我希望它能被藏在某个地方，以后再被释放出来。没有这样的运气。自 5 月下旬以来，一个工作原型一直在浮动，这意味着没有进一步的硬件变化。电池已经从 Sidekick 3 上提升了，但这可能与新的 3 英寸屏幕有直接关系。这基本上意味着电池寿命将和 SK3 一样糟糕。保密协议在万圣节结束。期待它在 11 月下降。
