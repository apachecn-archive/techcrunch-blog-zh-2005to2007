# Scouta 媒体推荐:即将在 iTunes 中发布

[![scouta.jpg](img/78a7d5fb85bb5a2570ed3d20272cc0c1.png)](https://web.archive.org/web/20211016235746/http://www.scouta.com/)

Scouta 是最近推出的一项新服务，它根据网站成员感兴趣的内容向他们推荐音频和视频内容。Scouta 已经对各种网站的内容进行了索引，使用户能够从这些网站中的任何一个找到并接收内容，而不是在每个单独的网站上进行搜索。这意味着它有一个庞大的目录，是大型库和目录的超集。

他们今天宣布，在几周时间内，他们将有一个 Mac iTunes 客户端，允许用户直接查看或收听他们的推荐或你自己的书签内容。该插件还会跟踪你在 iTunes 中听和看的内容，并将信息发送回 Scouta。这与 [iLike](https://web.archive.org/web/20211016235746/http://www.ilike.com/) 类似，只是它适用于 iTunes 和 Scouta 本身的所有内容(你可以在大量视频和内容网站上跟踪内容)。他们还将开放一个 API，以便其他人可以开发工具或集成 Scouta 及其建议。

YouTube 等内容网站上的大多数“起点”显示所有用户最受欢迎的视频，或评分最高的视频。虽然你有时可以按地区对这些数据进行分类(在 YouTube 上你还不能——这导致大量外语(好吧，对说英语的人来说是外语)现在占据了主要内容页面)，但更多的时候，你看到的内容通常并不符合每个人可能感兴趣的内容。

Scouta 会跟踪你加了书签的媒体，你的朋友加了书签的媒体，以及你所在的任何团体，并使用这些信息向你推荐内容。从我们最初的测试来看，它似乎工作得很好，我们很快就找到了有趣的链接。Scouta 提供了一个书签小程序，这样您就可以将内容添加到 Scouta 中，而无需返回到您的页面。他们还为网站上的几乎所有内容提供 RSS 提要(让你的推荐在你的新闻聚合器中传递给你是很有趣的)
