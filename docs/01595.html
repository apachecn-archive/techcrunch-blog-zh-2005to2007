Netvibes今天宣布种子融资

原文：https://web.archive.org/web/http://techcrunch.com:80/2006/03/22/netvibes-to-announce-seed-financing-today/comment-page-2/

非常受欢迎的Ajax主页产品，我说过它是一家 Web 2.0公司，没有它我无法生存，它将宣布由Index Ventures、马克·安德森、Pierre Chappaz(kel koo和Wikio的创始人)和Martin Varsavsky(Jazztel、Ya.com和Fon的创始人)提供种子资金。这轮融资的规模没有披露。

Netvibes上已经创建了100多万个主页，而且这项服务有多种语言版本。他们现在正在开发一个API，允许创建第三方小工具。

Netvibes的总部设在巴黎和伦敦。我在Netvibes上的原帖是这里。
