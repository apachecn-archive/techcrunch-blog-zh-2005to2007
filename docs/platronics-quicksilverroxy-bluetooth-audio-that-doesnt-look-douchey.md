# Plantronics + Quiksilver/Roxy =看起来不傻的蓝牙音响

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/22/platronics-quicksilverroxy-bluetooth-audio-that-doesnt-look-douchey/>

(或者，事实上，无线)衣服是我们以前见过的东西，袖子里的 iPod 遥控器之类的，但 Plantronics 与 Roxy 和 Quiksilver 合作，他们似乎做对了，发布了一系列内置立体声蓝牙免提套件的滑雪板设备。

派克大衣和头盔系列不仅可以通过蓝牙手机与斜坡上的同伴聊天，还可以通过头盔和兜帽中的集成扬声器听音乐。一个额外的好处是，在山上戴上蓝牙耳机，你看起来不会像一个工具。

服装并不便宜(点击跳转查看更多细节和价格)，但大多数滑雪板运动员习惯于为好的装备支付溢价，价格与其他同等质量的无线设备相当。他们会卖吗？我们不知道，但如果这条线成为热门，我们不会感到惊讶。

——Quiksilver Double Daffy Snow 夹克，采用 Plantronics 立体声和蓝牙语音技术:该夹克是 Quiksilver 限量版外套系列的一部分，可用于 iPod 和手机。双 Daffy 将在 2007/2008 外套季节上市，在全国范围内的 Quiksilver 商店和更好的零售商处的零售价为 370 美元。

— Quiksilver Pulse 头盔，采用 Plantronics 立体声和蓝牙语音技术:这款轻质头盔由注塑 ABS 制成，具有可调节的侧通风口，并采用黑色迷彩印花。Pulse 将在 2007/2008 外套季节上市，在全国范围内的 Quiksilver 商店和更好的零售商处的零售价为 249.95 美元。

— Roxy Teen Angel Snow Jacket 采用了 Plantronics 立体声和带蓝牙的语音技术:iPod 和手机就绪，这款夹克将在 2007/2008 外套季节上市，在全国范围内的 Quiksilver 商店和 Roxy 零售商处的零售价为 340.00 美元。

— Roxy 颤栗头盔，采用 Plantronics 立体声和蓝牙语音技术:颤栗 Roxy 女式滑雪/雪地安全头盔将于 2007/2008 外套季节上市，在全国范围内的 Quiksilver 商店和 Roxy 零售商处的零售价为 249.95 美元。
