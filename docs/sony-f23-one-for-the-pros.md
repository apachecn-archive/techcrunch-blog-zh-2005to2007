# 索尼 F23:专业人士的不二选择

![](img/a2ab731dd5040c2a095569fda147e02d.png)

对于业余和专业电影制作人来说，索尼并不陌生。它的 Betacam 是那些要求高质量镜头和易用性的人的热门产品。今天，索尼提高了标准，宣布推出 F23，这是一款为严肃电影制作设计的专业摄像机。该相机可以在 HD1080 中以 24fps 拍摄，以给画面带来“电影”般的极致清晰度。想要古怪的规格吗？您可以在 HD1080/60P 和 HD RGB 4:4:4 模式下录制，该相机支持 1，920 x 1，080 分辨率、1080/23.98PsF、24PsF、25PsF、29.97PsF、50P、59.94P、1080/50i 和 59.94i。我们真的不确定这些模式是否完全必要，但话说回来，我们自制的电影 F23 将于 2007 年中期上市，其价格肯定会让你的银行账户血本无归。
