# 我们参加 Sidekick ID 发布会的邀请函呢？

我想我们的邀请函在邮寄过程中丢失了。我们都恨之入骨的搭档 ID 正从凯莉·莫纳科和塔恩·曼宁那里得到一些急需的公关。没什么说的，“买我吧”，就像一个~~小明星~~可互换的皮肤。光是图片就让我想买一个。发布会不在 Chuck-E-Cheese，但是有一些辣妹。

就连詹娜·杰姆逊也从百忙之中抽出时间来参加派对，但她似乎更喜欢德韦恩·韦德版 SK3。根据这些照片，我认为 ID 会受到女士们的欢迎。
