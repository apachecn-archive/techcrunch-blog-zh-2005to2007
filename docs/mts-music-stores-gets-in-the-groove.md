# MTS 音乐商店进入最佳状态

上周五，移动音乐服务提供商 Groove Mobile 宣布与通信公司 MTS Allstream Inc .结盟，通过数字网络将移动音乐带给 30 多万用户。这是 Groove Mobile 的第六个运营商合作伙伴。

Groove Mobile 首席营销官 Adam Sexton 表示:“作为我们发展全球移动音乐市场承诺的一部分，我们很高兴通过此次联盟扩大我们在加拿大的业务。“MTS Allstream 的先进数字网络和 Groove Mobile 的创新音乐发现功能非常适合音乐爱好者，他们需要直接通过手机获得优质的音乐体验。”

MTS 移动音乐商店将由 Groove 支持，提供直接到手机的完整曲目音乐下载，以及对包括音乐排行榜和专辑封面在内的其他内容的访问，并提供购物和播放、浏览和预览音乐的能力。音乐服务将支持双重下载，这将允许客户通过空中下载购买音乐，并将歌曲的副本下载到他们的手机上，同时还可以将副本直接发送到 PC 上。客户还可以在电脑上播放歌曲，或者将歌曲转移到 MP3 播放器或刻录到 CD 上。
