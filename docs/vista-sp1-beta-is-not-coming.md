> 将会有一个 Windows Vista 服务包，我们目前的预期是测试版将会在今年的某个时候推出。服务包是传统软件生命周期的一部分——它们是我们为所有微软产品所做的事情，作为我们持续改进承诺的一部分，提供早期测试版本是一种标准做法，有助于我们整合客户反馈并提高产品的整体质量。

> 服务包只是我们为不断改善 Windows 体验所做工作的一个例子。我们还通过 Windows Update 提供对 Windows 的改进，这是一个为我们的客户提供最新更新的绝佳渠道。此外，自 Windows Vista 发布以来，我们一直在与合作伙伴合作，以提高整体设备覆盖率和应用程序兼容性。现在，Windows Vista 支持超过 210 万台设备和 2，000 多个已登录的应用程序。我们认为客户现在使用 Windows Vista 会有很好的体验。
