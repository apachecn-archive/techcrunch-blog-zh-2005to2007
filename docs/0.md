# Revision3 从 Greylock 筹集了 800 万美元

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/06/24/revision3-raises-8-million-from-greylock/0/>

### 迈克尔·阿灵顿

[![](img/d31e7c00ba0c5967d783fe2f869b8c30.png)](https://web.archive.org/web/20080526060659/http://www.revision3.com/) 在线视频网站 [Revision3](https://web.archive.org/web/20080526060659/http://www.revision3.com/) 已经筹集了 800 万美元的第二轮融资，据一位接近该交易的消息人士透露，这使得筹集的资金总额达到 900 万美元。我们的理解是格雷洛克领先一轮。格雷洛克也是 Revision3 第一轮 100 万美元融资的主要投资者。新闻初创公司 Digg 的创始人凯文·罗斯和杰伊·阿德尔森都持有 revision 3[的股份，该公司也两次从 Greylock](https://web.archive.org/web/20080526060659/http://www.techcrunch.com/2006/12/28/no-acquisition-for-digg-raise-series-b-round-instead/) 募集资金。

Revision3 现在拥有 10 个节目的独家版权，并随着时间的推移继续增加更多人才。然而，Diggnation 是目前最大和最受欢迎的。收入是通过整合广告产生的，节目主持人直接讨论广告产品。

该公司尚未证实这一传闻，也没有回应我们的置评请求。我们的消息来源还称，首席执行官杰伊·阿德尔森可能很快就会离职，专注于 Digg，Digg 也由他运营。该公司可能已经开始寻找新的首席执行官了。
