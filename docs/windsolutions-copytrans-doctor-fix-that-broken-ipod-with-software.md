# 医生:用软件修好那个坏掉的 iPod！

[![fixitunesdeal.jpg](img/3aae6905cc5d1a155c97da26f9c3324d.png)](https://web.archive.org/web/20201127145951/http://old.crunchgear.com/wp-content/uploads/fixitunesdeal.jpg "fixitunesdeal.jpg")

我们都有 iPods，Hickey [已经告诉我们](https://web.archive.org/web/20201127145951/http://crunchgear.com/2007/03/05/crunchgear-how-to-fix-an-ipod-that-wont-boot/)如果它坏了该怎么办。但是，说，你讨厌 Hickey，不相信他说的一个字。很公平，但是你的 iPod 仍然有可能过早死亡。很遗憾，是的，但是 WindSolutions 已经发布了 CopyTrans Doctor iPod，这个软件仍然处于测试阶段，可能会修复你的 iPod。听起来像 DiskWarrior 等人，因为它试图修复磁盘错误。它还会尝试修复你的 iPod-iTunes 资料库中的漏洞。(换句话说，歌曲*在 iPod 中列出了*，但实际上并不存在。)

只有 Windows，但 Mac 是完美的，所以拥有 iPod 的 Mac 用户甚至不必费心。
