Windows Mobile设备中心:吸这个，ActiveSync

嗯，这是一壶好鱼。Windows刚刚推出了32位和64位同步软件Windows Mobile Device Center或NAMBLA的测试版。该软件显然比ActiveSync漂亮，它包括所有常规的同步垃圾以及媒体同步，设备上的文件浏览，以及改进的设置和识别。

我从没想过我会这么说，但Windows看起来越来越好了。
