# 面向摩托车手的 Scala Rider 蓝牙头盔耳机

普通的蓝牙耳机在摩托车上不起作用——我的几个朋友试过。这款 Scala Rider 耳机是专为骑手打造的，配有吊杆麦克风，机身更薄，不会妨碍头盔。耳机夹在头盔里，扬声器放在你的耳朵里，吊杆麦克风放在你的脸前。

《科技文摘》对此怎么看？嗯，为了在高速公路上打电话，你必须先在上面设置语音识别。但是一旦你把它安装好并运行起来，音质就会非常好。他们说，在他们打电话的时候，几乎不可能知道骑车人的时速是 70 英里。不幸的是，这只适用于封闭式头盔，因为露天头盔仍然会受到麦克风周围风的影响。

这个装置上的按钮有点小，戴上手套就能按下，而且没有 MP3 或 GPS 集成。另外，这对摩托车手来说是一个非常棒的耳机，价格为 75 英镑，或者 141 美元
