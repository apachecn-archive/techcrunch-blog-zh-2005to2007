# 创业公司推出帮助 Web 1.0 网站

两家初创公司最近推出了帮助 web 1.0 内容和商业网站增加急需的 web 2.0 功能的服务，即[聚合知识](https://web.archive.org/web/20220930073059/http://www.aggregateknowledge.com/)和[巴扎尔语音](https://web.archive.org/web/20220930073059/http://bazaarvoice.com/)。这两家公司都得到了乔希·科佩尔曼(Josh Kopelman)的第一轮投资的支持，帮助公司增加客户反馈功能，其成本只是内部开发的一小部分。

 [](https://web.archive.org/web/20220930073059/http://www.aggregateknowledge.com/) 总部位于门洛帕克的 **Aggregate Knowledge** ，由首席执行官 Paul Martino 和产品副总裁 Chris Law 领导，于今天在 eTech 大会上正式发布。该公司为网站创建了自动化的“行为推荐”——比如“看这个的人也看了”和“买这个的人也买了”等功能。请参见此处的屏幕截图[获取视觉效果。集成极其简单。](https://web.archive.org/web/20220930073059/http://www.aggregateknowledge.com/products/recengine.html)

Aggregate Knowledge 还为客户创建了白标“ajax 主页”,帮助推广这些信息。参见[这张截图](https://web.archive.org/web/20220930073059/http://www.aggregateknowledge.com/products/dashboard.html)中的示例。

 [](https://web.archive.org/web/20220930073059/http://www.bazaarvoice.com/) 总部位于德克萨斯州的 **BazaarVoice** ，比聚合知识更进一步，为产品提供类似亚马逊的编辑建议——如顾客评论等。这是一笔大生意——BazaarVoice 能够以内部开发成本的一小部分提供服务，甚至像 CompUSA 这样的大型网站也开始采用他们的解决方案。对于较小的客户，起价约为每月 2000 美元。除了乔什·科佩尔曼，BazaarVoice 还获得了奥斯汀风险投资公司汤姆·鲍尔的资助。
