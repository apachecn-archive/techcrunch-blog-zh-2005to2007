# PlayStation 3 为 PSP 带来用途

![](img/e77205a2c081b564a86afcdb980420ee.png)

如果你拥有一台 PSP，你终于可以用上这个失灵的便携设备了。[正如我之前提到的](https://web.archive.org/web/20210227080501/http://crunchgear.com/2006/10/24/playstation-3-23-things-you-want-to-know/)，PlayStation 3 将通过 WiFi 支持 PSP，让你用 PSP 访问 PS3 上下载的内容。旧金山的玩家日展示了我们可以期待的一点预览:PS3 主机已经将詹姆斯·邦德新电影《皇家赌场》的预告片下载到了硬盘上。PSP 能够无线连接到控制台，将预告片下载到自己的存储器中。

这就是在 PSP 发布期间你所能做的一切。索尼表示，未来我们将能够从其市场下载 PSOne 游戏(这是另一件我们不太了解的事情)，并在你的 PSP 上玩这些游戏。抱歉，对于那些期待无定位电视的人来说，索尼没有在这一领域取得进展的计划。

目前，只有在你的客厅里，你才能通过 WiFi 访问 PS3 高清版的所有内容。将来我们可能可以从任何地方访问 PS3，但是现在，整个过程有些受限。
