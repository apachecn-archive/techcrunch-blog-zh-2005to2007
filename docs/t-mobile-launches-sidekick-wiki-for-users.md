# T-Mobile 为用户推出 Sidekick Wiki

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/15/t-mobile-launches-sidekick-wiki-for-users/>

![](img/1041dc02e6fd899214994705718026bb.png)

我们总是喜欢大公司能和我们这样的极客建立联系。所以当我收到一封电子邮件，说 T-Mobile 正在参与维基百科时，我欣喜若狂(真的)！T-Mobile 在即将推出的 wiki 网站 Wetpaint 上为他们的伙伴用户创建了一个 wiki。Sidekick 用户可以浏览和贡献事实，浏览论坛，聊天，等等。我认为看到一大堆 Sidekick 粉丝/用户聚集在同一个地方是很棒的。如果你像现在一半的年轻人一样使用 Sidekick 3，一定要检查一下。
