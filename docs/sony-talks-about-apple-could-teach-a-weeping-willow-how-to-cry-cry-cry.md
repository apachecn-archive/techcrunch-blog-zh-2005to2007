# 索尼谈论苹果，可以教一棵垂柳如何哭，哭，哭

![](img/ee677a9e4f87573a16d04f48b1638536.png)

上周，霍华德·斯金格爵士三世与首席执行官 Exchange 坐下来，讨论了索尼如何在便携式播放器市场陷入困境，以及他希望 PS3 能够战胜 Wii 的冲击，摆脱恶名。

关于数字播放器，他说:

> 1997 年，我们与 IBM 合作电子音乐分销，本可以比 iPod 早五年推出。但是我们无法让我们的员工理解软件。我们是一家音乐公司。他们看到数字媒体，惊慌失措，不喜欢它。

对于索尼的领导者来说，这是一个大胆的举动，尤其是考虑到他们沉默寡言的天性和无失败的文化。这里最重要的一点是，索尼已经被他们最近一连串的坏运气弄得谦卑了。他们能打破自己陷入的随身听/WEGA 的窠臼吗？我希望如此。他们正在被各种各样的小玩家所取代，除了 PS3，他们的知名度仅限于笔记本电脑和电视等跟风设备。所有很酷的东西，圆形电脑，免费的摇摆舞，都藏在一蒲式耳下面。
