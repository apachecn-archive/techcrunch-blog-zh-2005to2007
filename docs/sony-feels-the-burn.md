# 索尼感受到了燃烧| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/19/sony-feels-the-burn/>

![](img/68c03a401769741a4b789ab1e3328755.png)

看来，所有这些电池召回和劣质产品发布最终都影响到了索尼公司。索尼公司周四宣布，将净利润预期下调 38%，至 800 亿日元(约合 6.76 亿美元)。

随着 PS3 的销售加速，用户忠诚度开始再次上升，索尼预计明年春天将会反弹。然而，我对此表示怀疑。它将亏本销售 PS3，该系统与任天堂 Wii 相比已经表现不佳。似乎索尼正在把所有的鸡蛋放在一个即将被践踏的篮子里。
