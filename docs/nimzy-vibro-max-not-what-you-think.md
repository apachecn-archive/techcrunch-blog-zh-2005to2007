# 尼玛兹·振动麦克斯，不是你想的那样

![vibro-max.jpg](img/639eb90c3cdde89bd2d86d91a2797b59.png)

不，Nimzy Vibro Max 不是另一个[性玩具](https://web.archive.org/web/20230322164148/http://crunchgear.com/2007/05/16/vortex-sex-toy-for-the-love-of-all-that-is-holy-please-no/)，它比那个更酷。乍一看，它似乎是一个无害的盒子，但一旦你把它放在一个平面上，它就会把这个物体变成一个扬声器。像哇哦！Nimzy 使用“先进的电声专利技术”，将音频振动发送到木材、塑料或玻璃等表面，产生清晰的声音。*咳，BS，咳*这个可爱的小立方体有一个集成的数字放大器，音量控制和开/关开关。你也可以把你的 CD 播放器、笔记本电脑、MP3 播放器插上电源，把一些音乐发送出去，让你办公室里的其他人也能听到。附带的遥控器可调节低音、高音和音量。只需 160 美元，所有这些都是你的了。
