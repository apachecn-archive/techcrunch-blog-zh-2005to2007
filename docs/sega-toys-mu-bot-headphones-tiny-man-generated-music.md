# 世嘉玩具的 mu-bot 耳机:微小的人造音乐？

![](img/5e7ef711876e0d42e800750832fe3ffa.png)

信不信由你，这个装置是一副耳机。他们的名字叫 mu-bot(日语是“什么鬼东西？”)并计划于下月下旬登陆日本。我不会说一句日语，但我会试着解读产品页面上的象形文字到底是什么意思。

例如，我很确定就在“mu-bot”下面写着“快乐进入耳朵，获得巨大的终极声音乐趣。”当你滚动第一个方块“小人好 iPod 脆娱乐。”

诸如此类。小巧的人形耳机，听起来一定很棒，售价 25 美元。
