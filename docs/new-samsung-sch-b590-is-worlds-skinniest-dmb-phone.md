# 新款三星 SCH-B590 是世界上最薄的 DMB 手机

![](img/3031e065280e62411f5b08505d6e4611.png)

三星今天发布了他们的新型号 SCH-B590，它看起来很棒——你知道，如果你对“更苗条更性感”感兴趣的话。这款手机拥有 DMB、130 万像素摄像头和 MP3 播放器，因此韩国人可以继续玩我们知道存在但只能梦想有一天得到我们的东西。

这款手机号称是“市场上最薄的 DMB 手机”，只有 16.5 毫米，除非你认识一个认识一个人的人，否则你不可能拥有一部手机。即使那样，你在美国也会有一部 DMB 手机，但实际上没有任何东西可以看。所以，玩得开心点。
