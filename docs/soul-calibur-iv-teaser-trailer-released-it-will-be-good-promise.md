# 灵魂 Calibur 四预告发布:它会很好。保证。

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/15/soul-calibur-iv-teaser-trailer-released-it-will-be-good-promise/>

作为一个男孩，战斗游戏一直是我最喜欢的，所以即使是阅读关于*灵魂之剑 4*的一点点信息也会给我带来极大的快乐。Namco 今天发布了这款游戏的预告片，预计将于 2008 年在 [Xbox 360](https://web.archive.org/web/20141016160923/http://crunchgear.com/category/xbox-360/) 和 [PS3](https://web.archive.org/web/20141016160923/http://crunchgear.com/category/ps3/) 上发布。我在前两部电影上花了大量的时间(问问我的大学室友)，目前正在努力完成第三部。

预告中没有透露太多信息。常青藤的半身像仍然很大，而三月树看起来仍然像是历史的火焰在燃烧。我现在更开心了。
