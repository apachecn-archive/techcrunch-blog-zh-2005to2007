# 本周四的 TechCrunch Meetup

继上周四我们在阿泽顿[的第一次聚会和烧烤](https://web.archive.org/web/20211127180611/http://www.beta.techcrunch.com/2005/09/16/techcrunch-conference-tonight/)取得成功后，本周[我们将再次在](https://web.archive.org/web/20211127180611/http://wiki.beta.techcrunch.com/doku.php?id=second_meetup_bbq)聚会和烧烤。上周大约有 25 人到场，我们预计本周会有更多的人到场。

如果你在湾区，请加入我们。该活动将在我们位于阿泽顿的家中/办公室举行。[这里是此次活动的 Wiki，包括具体细节和方向](https://web.archive.org/web/20211127180611/http://wiki.beta.techcrunch.com/doku.php?id=second_meetup_bbq)。如果你要来，请在维基底部加上你的名字。如果你不知道如何编辑维基，你就不能来。:-)

我们将在 5-6:30 烧烤，整个晚上将有大量的葡萄酒、啤酒和软饮料。客厅和后勤办公室都有展示产品的电脑。火坑在外面，让事情持续到晚上。

如果你想结识其他企业家和风险投资家，或者想为你的初创公司造势，这里是你的不二之选(如果你不在演示现场的话)。
