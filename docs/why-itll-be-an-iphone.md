# 为什么会是 iPhone(？？)

大卫·索博塔了解苹果。他在那里工作了 20 年，参加了无数次与史蒂夫的会议。因此，让我们给他听一听。他基本上是说，苹果平板电脑几乎是不可能的，而苹果手机几乎是 99%肯定的。
 现在，我们都知道，即使是最好的预言家也经常会犯可怕的错误。然而，让我们来谈谈群众的智慧。暗示在预测市场中提供了 75%的机会。CrowdIQ，一个类似的系统，说 [1%](https://web.archive.org/web/20210417051025/http://www.crowdiq.com/opex/contractinfodisplay.do?contractid=432) ，这也是我要说的。然而，Sobotta 让我犹豫。数以百万计的眼睛盯着苹果，他们不得不从帽子里拿出一些花哨的东西。
