# 糟糕的未来是现在:车内无线网络

从明年开始，Avis 将为其所有租赁汽车添加 WiFi 服务，作为一项附加服务，这进一步加剧了令人头疼的租车体验。服务提供商 Autonet Mobile 准备在 CES 上宣布这项交易，但我们通过《纽约时报》捕捉到了早期的风声。

一天 11 美元，你就可以以 60 英里/小时的速度下载色情内容，这真的是信息时代最初承诺的实现。我们希望在儿童色情主题重拍的*中看到其中一个，提高音量*，但遗憾的是这不会发生。我们真的不知道这项服务有什么意义，除非你的车上有一整个团队，在这种情况下，他们实在是太不耐烦了。说真的，在美国，星巴克每隔 50 英尺左右就有一个 T-Mobile 热点，我们还需要车内 WiFi 吗？

嗯，可能吧。
