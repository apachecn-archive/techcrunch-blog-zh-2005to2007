# 三星、意法半导体和 Metalink 推出无线高清 IPTV 机顶盒

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/03/19/samsung-stmicroelectronics-and-metalink-unveil-wireless-hd-iptv-set-top-box/>

今天早上，三星、意法半导体和 Metalink 在 CeBIT 上发布了一些令人兴奋的消息。他们合作开发了一款机顶盒，希望能彻底改变你看电视的方式。SMT-H6155 将支持基于 802.11n 无线 IPTV 标准的高清电视。STMicroelectronics STi7109 芯片组负责高清视频解码，Metalink 的 802.11n WLANPLUS 芯片组可通过无线局域网传输多个高清视频流。

SMT-H6155 是一款混合机顶盒，可处理 IP 和 DVB-T 接收。它还将支持视频点播和 IP 多播。革命性的 SMT-H6155 将 MPEG-4 AVC/H264/C1 功能结合在使用意法半导体 SoC 平台生产的单个芯片组上，并支持使用 802.11n 的无线数字视频家庭网络。没有关于价格或上市日期的消息。
