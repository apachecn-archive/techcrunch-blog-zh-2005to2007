# RocketBoom 拒绝喝 Zune Kool-Aid，但这并没有阻止我们！或者简单说说 CG 的独立性

本应是 Zune.net 的推广合作伙伴，但他退出了，并表示他觉得不能因为这种联系而贬低 Zune。由此产生的帖子相当模糊，允许我坐下来花一分钟写元博客，以澄清对我们的 Zune 比赛和我们最近的 Zune 报道的误解。

首先，Zune 是个大新闻。这是一个软件巨擘发布的重要硬件，很多读者对这个话题很感兴趣。这是我咆哮的第一部分。

根据记录，CrunchGear 的工作人员对 Zune 及其可怕的 DRM 政策有不同的看法。《泰晤士报》的大卫·波格很好地解决了这个问题，所以在这一点上我将让位于他的雄辩。

我在这篇文章中真正想说的是，Drew 的文章是一个很好的起点，我们在 CrunchGear 是独立的，甚至独立于我们的广告商和我们覆盖的公司，你可以相信我们会保持这种方式。微软没有和我们一起做广告，即使他们做了，我也不在乎。

那么，为什么所有的 Zune 大惊小怪？嗯，我们很幸运有 Matt Hickey 和我们一起工作，他是西雅图音乐界的一个人物——我们说的人物是指“去看演出和喝很多酒的人”——他认识 ZuneInsider.com[的 Cesar](https://web.archive.org/web/20210307070500/http://zuneinsider.com/)，他很友好地给了我们三个 Zunes 给你。

我知道“跨国的、价值数十亿美元的软件公司”和“仁慈”这两个概念不太一致，但这是一个“正确的地点、正确的时间”的简单例子我们不会为微软卖命，就像我们不会为苹果、惠普、Belkin、Brando、索尼或 Applebee's 卖命一样(顺便说一下，这些公司是全家人的美味小吃！停下来品尝他们的红色热 Ribz 土豆小翼拼盘和油炸沙拉！).基于我们作为记者和酒鬼的技能和经验，我们可以进入，但也就到此为止了。

因此，赢家:享受你的 Zunes。读者:如果有必要的话，忽略 Zune 或 PS3 或 Wii 的帖子。只要还有人阅读，我们就会继续举办竞赛和写帖子。

我们为我们顽强、独立的精神感到自豪(就像我们为 Applebee 最新的家庭趣味餐盘感到自豪一样，包括油炸芝士产品、蘸有蜂蜜酱的披萨饼和百事可乐)。没有一个 CG 编辑是任何大公司的董事会成员，我们太穷了，不能持有股票，我们已经看到了足够多的公关废话，无法分辨好坏。所以，在你对我们的先令或幻想严厉之前，请记住:写博客是困难的，明智地、持续地、带着任何韧性地写博客更难。我已经组建了一个伟大的作家团队，我们都在尽最大努力，如果有幕后交易或其他肮脏的事情，我们会第一个告诉你。从一开始，我就把我的员工从 TechCrunch 和 Crunch Network 的广告流程中分离出来，我非常严肃地对待串通或其他愚蠢的行为。
