# 死党 3，英雄女孩拯救世界

![](img/18d5e1f61bb17ea307dbc3122dfd0b6a.png)

当你面临闯入，只有几秒钟的反应时间时，抓起你的手机藏在壁橱里是一个极好的选择。这正是皇后区 12 岁的凯蒂·鲁伊斯在两名持枪男子闯入她家时所做的事情。凯蒂脑子转得很快，抓起她最近刚收到的助手，跑到壁橱里寻求安全。当她的家人被恐吓和抢劫时，她在里面拨打了 911。

她起初拨打 911 有困难，因为助手的界面对她来说是新的和处女，但在几次尝试后设法得到它。幸运的是，这些人在“像在《黑客帝国》中一样”跳窗试图逃跑时被抓住了。如果罪名成立，他们将被判 25 年监禁。
