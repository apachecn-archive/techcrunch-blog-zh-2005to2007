# systemax Pursuit 4155:999 美元 Vista 旗舰笔记本电脑

微软 Windows Vista 推出时，T2 抱怨最多的话题之一是有效运行操作系统的系统要求。 [Systemax Pursuit 4155 笔记本](https://web.archive.org/web/20130628133212/http://www.productshippingcenter.com/applications/SearchTools/item-details.asp?EdpNo=2937403&Tab=2&NoMapp=0)证明所有的大惊小怪都是没有根据的。

我最近对四款价格不到 1000 美元的笔记本电脑进行了一次全面的总结，它们都运行 Vista Home Premium，这是 Pursuit 运行的终极版的降级版本。这些笔记本电脑在运行 Premium 和 Pursuit 时都没有问题，其 1.73GHz Core 2 Duo T2250 处理器、1GB 内存和集成的英特尔显卡可能也不会在 Ultimate 上遇到问题。4155 的其他规格很普通:60GB 硬盘、DVD/CD-RW 组合驱动器、15.4 英寸宽屏液晶显示器和内置的 130 万像素网络摄像头。

现在，我并不是说你不必更换使用了三年的奔腾 4/Athlon 64 台式机/笔记本电脑来运行该操作系统，但硬件要求并没有高到让你无法购买一个预算系统并仍然享受花哨的 Vista 功能。

跳跃后更多追击 4155 的照片。
