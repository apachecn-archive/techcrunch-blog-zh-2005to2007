# 维基百科筹集了 400 万美元

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/03/27/wikia-raises-4-million/>

 [](https://web.archive.org/web/20220626091001/http://www.wikia.com/wiki/Wikia) [Wikia](https://web.archive.org/web/20220626091001/http://www.wikia.com/wiki/Wikia) ，[原名 Wikicities，](https://web.archive.org/web/20220626091001/http://www.wikia.com/wiki/Press_releases/March_2006)今日宣布 400 万美元 A 轮融资。此次融资由 Bessemer Venture Partners 和 Omidyar Network 牵头，天使投资人丹·吉尔默、雷德·霍夫曼、伊藤穰一和米切尔·卡普尔参与了融资。Bessemer 的 David Cowan 也在他的[个人博客](https://web.archive.org/web/20220626091001/http://whohastimeforthis.blogspot.com/2006/03/wikia-jimmy-wales-startup.html)上写了这个。

维基百科是由维基百科创始人之一吉米·多纳尔·威尔士创办的一家盈利性企业。它托管基于主题的维基，主题数量，任何人都可以添加新的主题。

虽然 Jimmy Wale 的维基百科是网络上访问量最大的网站之一，而且我确信投资者认为他能够带动类似水平的用户忠诚度，但是 Wikia 目前的访问量几乎为零。与之对抗的将是不那么引人注目的主题，以及这是一家盈利性公司的事实。

然而，我很想看到维基百科和维基百科写作界面的彻底改革。他们有很多很好的例子可以借鉴。
