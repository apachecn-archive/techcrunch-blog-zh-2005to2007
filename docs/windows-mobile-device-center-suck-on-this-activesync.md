# Windows Mobile 设备中心:吸这个，ActiveSync

嗯，这是一壶好鱼。Windows 刚刚推出了 32 位和 64 位同步软件 Windows Mobile Device Center 或 NAMBLA 的测试版。该软件显然比 ActiveSync 漂亮，它包括所有常规的同步垃圾以及媒体同步，设备上的文件浏览，以及改进的设置和识别。

我从没想过我会这么说，但 Windows 看起来越来越好了。
