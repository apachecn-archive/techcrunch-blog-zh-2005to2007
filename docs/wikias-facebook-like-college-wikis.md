# 维基的脸书式大学维基

 [](https://web.archive.org/web/20221203093128/http://www.wikia.com/wiki/Wikia) [Wikia](https://web.archive.org/web/20221203093128/http://www.wikia.com/) 已经悄然推出半私立学院/大学 wiki。他们还借鉴了脸书的游戏计划，要求用户拥有一个经过批准的大学电子邮件地址才能编辑维基。然而，维基本身对每个人都是可见的。

目标是通过排除与大学没有直接关系的人来获得更好的数据。当然，不利的一面是，人们必须找到他们大学的电子邮件地址才能添加内容。很明显，脸书为用户提供了足够的激励去这么做。目前还不清楚 Wikia 是否会取得同样的成功。

斯坦福的一个例子是这里的。关于创建半私立大学维基的信息在这里。

这仍然是一个实验，已经存在的关于学院和大学的开放维基可以在 students.wikia.com 找到。维基百科没有透露他们将如何处理每所大学有两个版本的维基百科(一个开放，一个半私有)。

Wikia 在三月份筹集了 400 万美元并雇佣了前易贝高管 Gil Penchina 来领导公司后，继续前进并做得很好。吉尔说，维基百科现在每天有 100 万独立访问者，产生 500 万的页面浏览量。他们最近推出的旅游指南在推出仅八周后，就有超过 1200 篇十种语言的文章，覆盖 160 个城市。
