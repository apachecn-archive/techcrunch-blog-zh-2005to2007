# 威瑞森悄悄放弃第一部美国有机发光二极管手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/05/verizon-quietly-drops-first-american-oled-handset/>

。由于诺基亚(通过泛泰)，有机发光二极管终于带着新的诺基亚 6215i 低调来到了北美。这款轻巧的翻盖手机有两个有机发光二极管显示屏，一个内置 128×128 像素，一个外置 96×64 像素。这款手机还配有带闪光灯和自动定时器的 VGA 摄像头、免提电话和声控拨号功能。

这是值得注意的，因为作为一项技术，[有机发光二极管](https://web.archive.org/web/20131104142323/http://en.wikipedia.org/wiki/OLED)将在不久的将来取代小型设备中的液晶显示器。我们中的一些人认为这已经发生了，因为有机发光二极管比 LCD 更便宜，更轻，更容易制造，而且耗电更少，但这绝对是未来发展的先锋。

它可以从威瑞森以 50 美元的价格获得两年的合同，这是一种非常便宜的进入前沿的方式。
