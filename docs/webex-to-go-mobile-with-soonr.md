# WebEX 将很快实现移动化

 [](https://web.archive.org/web/20220629003608/http://www.soonr.com/) 网络会议巨头 [WebEX](https://web.archive.org/web/20220629003608/http://webex.com/) 和移动/PC 连接初创公司 [SoonR](https://web.archive.org/web/20220629003608/http://soonr.com/) 今晚宣布，他们已经达成合作伙伴关系，将允许用户通过移动设备访问 WebEX 会议。WebEX 推出了一个名为 WebEX connect 的新平台，该平台将精选的第三方服务整合到公司的网络会议中。WebEX 的移动访问预计将在 3 到 6 个月后推出。

网络会议领域的竞争越来越激烈，高端 WebEX 必须继续创新，以与便宜得多的 [GoToMeeting](https://web.archive.org/web/20220629003608/http://gotomeeting.com/) 、Adobe 的 [Breeze](https://web.archive.org/web/20220629003608/http://www.adobe.com/products/breeze/solutions/webconferencing/) 以及即将上市的[DimDim dim](https://web.archive.org/web/20220629003608/https://beta.techcrunch.com/tag/DimDim)等一系列免费网络会议解决方案竞争。

与 Salesforce 的 AppExchange 不同，它向大量开发人员提供 API 访问，有时会导致收购(参见[Kieden 交易](https://web.archive.org/web/20220629003608/http://www.beta.techcrunch.com/2006/08/21/salesforce-dives-deep-into-google-adwords/)), WebEX 似乎正在开放一个更加有限的平台，并雇佣精选的外部合作伙伴来集成他们的服务。

SoonR 是一家我们在之前已经介绍过的公司，它允许用户从 450 多种手机上搜索和访问他们桌面电脑上的文件。这不是该公司通过移动设备提供的第一个应用程序，他们通过电话提供 Skype。SoonR 最近发布了许多重大公告；上周，他们与 Opera Mobile 联合发布了他们所谓的第一个移动 AJAX 服务，并完成了由 Clearstone Venture Partners 和 Intel Capital 牵头的 600 万美元的首轮融资。
