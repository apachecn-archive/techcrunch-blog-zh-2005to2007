# Windows Live 邮件服务 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/11/25/windows-live-email-service/>

Domains.Live.com 是微软提供的免费新电子邮件服务。这是一个简单但非常有用的工具。而且，像 [Office Live](https://web.archive.org/web/20221207202604/http://www.beta.techcrunch.com/2005/11/02/office-live-will-be-huge-productivity-tool/) 一样，它颠覆了现有的域名注册市场(他们甚至没有注意到吗？).

所以基本服务是这样的:微软将在你拥有的域名中为你托管电子邮件和即时消息。

*   在您的域内创建多达 20 个电子邮件帐户
*   为每个帐户准备一个 250 MB 的收件箱
*   从任何支持网络的电脑上查看电子邮件
*   使用 Microsoft SmartScreen 技术的垃圾邮件过滤器保护
*   电子邮件的病毒扫描和清除
*   无缝访问 MSN Messenger、MSN Spaces 等。

关于[自定义域名博客](https://web.archive.org/web/20221207202604/http://spaces.msn.com/members/customdomains/)的更多细节。

以下是我的想法:

首先，这很棒。到目前为止，通常只有域名注册商提供这种服务，它既昂贵又有非常 web 1.0 的界面。

第二，当我看到像这样的服务和 Office Live，在我看来，最需要关注的行业是域名注册商。微软免费提供的许多服务是 reigstrars 今天收费的类型。

微软开始为小型企业创造令人印象深刻的生产工具。这些工具也不收费。竞争者需要小心了。
