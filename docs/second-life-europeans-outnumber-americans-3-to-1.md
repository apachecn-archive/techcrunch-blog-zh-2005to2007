# 第二人生:欧洲人比美国人多 3 倍

如果 Web 2.0 是炒作的泡沫，那么林登实验室的第二人生无疑是其中最闪亮的泡沫。IBM、CNet、路透社、American Apparel、Coldwell Banker 等公司已经在元宇宙建立了业务，但是一项关于“第二人生”的新研究发现，这些支出可能是浪费。

comScore 报道不仅德国人比美国人更积极地使用第二人生，欧洲人作为一个整体比美国人多 3 倍。这些数字本身看起来比比率还要糟糕:三月份，美国只有 20.7 万人至少登录过一次第二人生。统计数据跟踪了独特的用户，没有考虑频率，所以这些游客中的许多人很容易成为一次性用户，注册了一个免费账户，目的是看一眼，一旦遇到性俱乐部、赌场和飞行男性附属物，就永远不会回来。

对于林登实验室来说，这不全是坏消息，欧洲的使用量正在上升，3 月份有 777，000 名活跃用户，比 1 月份增加了 32%，在德国增加了 70%，在法国增加了 53%。

当你将 MySapce 上百万用户的数十亿页面浏览量与同一个月内 Second Life 的 207，000 人的潜在完整独特用户群进行比较时，然后你会认为，由于笨重的世界内搜索和竞争选择的大杂烩，他们访问你的 Second Life 目的地的机会很小，这种价值主张对美国业务来说根本不成立。
