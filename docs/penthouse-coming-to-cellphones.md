# 手机时代的顶层公寓

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/12/penthouse-coming-to-cellphones/>

![](img/bb799f0502b115453aea49973344b3f1.png)

色情出版巨头 Penthouse 今天宣布，他们将开始在整个美洲和欧洲提供移动内容。这些内容将包括游戏，社论，音频和视频剪辑，以及(咄)图片。这项服务甚至会有一个新的移动电视频道。他们说它将在未来几个月内上市。我迫不及待地想和坐在我旁边的某个家伙一起坐在地铁上，用他的手机读一篇顶层公寓的“社论”。
