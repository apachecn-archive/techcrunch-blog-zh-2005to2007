# 索尼 VAIO WA1 无线音乐流媒体工具

[![sonywa1.jpg](img/a20655dfab2f79884b89b6206740bd09.png)](https://web.archive.org/web/20130628175913/http://old.crunchgear.com/wp-content/uploads/sonywa1.jpg "sonywa1.jpg")

我猜这是无线音箱周四在 CG 这里，什么与鹦鹉的[党](https://web.archive.org/web/20130628175913/http://crunchgear.com/2007/06/28/parrot-party-portable-boom-box-cue-aint-no-party-jokes/)和现在的索尼 VAIO WA1。这个小家伙与你的电脑和/或 Wi-Fi 网络协同工作，只传播室内音乐最令人难忘的节拍。索尼可能认为，人们并不真的想用电脑的蹩脚扬声器听音乐，也不想在地板上拉几英里长的扬声器电缆。就像我说的，WA1 可以搭载你的 Wi-Fi 网络，结合播放音乐，或者你可以使用内置的 USB 无线转换器。两者都完成了同样的任务:让无线音乐传遍整个房子。

WA1 支持所有标准音乐格式，并有 128MB 的板载 RAM 用于本地音乐存储。我是一个更喜欢大型音响系统的人(嗯，理论上来说)，但是在音乐云中漫步总是，嗯，很棒。350 美元整？嗯…
