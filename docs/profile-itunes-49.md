# 简介:iTunes 4.9-TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/06/28/profile-itunes-49/>

**服务:** [带播客的 iTunes 4.9](https://web.archive.org/web/20220818164457/http://www.apple.com/itunes/)

![](img/faa8d40699e8cb23a7b8d3bbb5190919.png)

**发射:**2005 年 6 月 28 日
 **什么事？**

正如人们普遍预期的那样，iTunes 4.9 今天发布(22 mb 下载)，适用于 windows 和 mac 平台。它包括重要的新功能和增强功能，其中最显著的是对播客的支持。

![](img/4be603f0c80e38dd72c0db00b2b7d14d.png)

迈克尔·加滕伯格[在 iTunes 4.9 上写了一篇精彩的文章](https://web.archive.org/web/20220818164457/http://weblogs.jupiterresearch.com/analysts/gartenberg/archives/008993.html):*“我可以下载一个，订阅一个 feed，在我的设备上保留一个固定的号码，并在我听完之后自动删除它们。简而言之，苹果为播客做的和他们在 Tiger 中为 RSS 做的一样。他们使它可以被大众市场使用，同时，他们拥有在 Macintosh 和 Windows 上最广泛使用的播客客户端，这将使他们的目录成为列表中的一个。但是还有更多。苹果还调整了所有 iPod 的固件，所以没有单独的播客类别，这意味着播客不会与我的音乐混在一起，并将支持书签，所以我可以听播客，并从我离开的地方继续。因此，它不仅仅是首要的播客个人电脑客户端，iPod 本身现在也是第一个集成了播客支持的设备。结合这一消息、新的定价以及 iPod photo 与核心白色 iPod 系列的整合，你就会明白为什么苹果仍然是这一领域的竞争对手。”*

该软件非常优秀，包括**显著特征:**

–所有播客目前都是免费的
–下载的播客显示在一个名为“播客”的 iTunes 文件夹中
–轻松搜索/查找
–一键订阅新播客
–自动下载所有未来播客的选项
–在 iTunes 上提交 publisher 播客的工具
–在 iPod 中保持独立，因此不会与音乐混在一起
–听完后自动删除(真棒！)
