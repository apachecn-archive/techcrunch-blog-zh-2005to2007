# mPoria 整合了 PayPal | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/22/mporia-integrates-paypal-2/>

mPoria 是一家为零售商和服务提供商提供移动商务解决方案的公司，该公司与 PayPal 合作，将后者的自动化移动支付系统与他们的 GoMobile 移动商务平台相集成。

mPoria 的首席执行官兼联合创始人丹·赖特(Dan Wright)认为，这一整合将“为移动网络上最大、最活跃的购物社区之一提供简单、安全的结账功能。”随着 PayPal 成为第三方支付系统提供商，客户将能够使用信用卡、银行账户或 PayPal 账户中的现有余额进行购物。
