# Treo 750:模糊的照片，功能列表

![Treo 750 leak](img/d0dbbab3a2d12207517015210c3ebdbb.png)

今天早些时候，你[读到了](https://web.archive.org/web/20210726005226/http://crunchgear.com/2006/08/28/palm-treo-750-caught-in-the-wild/)一些据称是 Treo 750 在野外被抓的新照片。我们可以确认这确实是 Treo 750 的一个版本，尽管还不清楚它是否与 10 月份将在 Cingular 推出的“Lennon”是同一款 750，或者它是否是传言中的 750v，即仅在欧洲推出的版本。我们知道的“Lennon”(很可能被命名为 750w)将包括 WiFi，但尚不清楚这款设备是否将包括 UMTS 和 class 10 EDGE。“Nitro”是 750 的补充，这是一款基于 PalmOS 的 750(可能是 750p)，具有类似的统计数据，更好的分辨率(320×320)，没有 WiFi。

我们了解到的是，在功能方面，这些智能手机附带的软件的价值非常可观。通常的 Windows Mobile 应用程序套件将被包括在内，包括 Office 文档读写。此外，寻找 pTunes、出色的 Mp3/WMA/网络电台应用程序和 MobiTV。对于后者，可能会有单独的订阅，但运营商可能会找到补贴的方法。我们可能会在两周内找到更多信息，届时 Palm 已经宣布他们将发布一项声明。你没看错。它几乎使公关炒作的虫洞。

我们还知道，第一版的摄像头将与当前的 Treos 相同，为 130 万像素，分辨率可能会保持在 240×240。
