Saitek Pro玩家指挥台

Saitek的这款新控制器采用直观的20按钮设计，让用户在FPS和RTS游戏中占据优势。Pro Gamer Command Pad具有背光人体工程学键盘，可调节拇指区域，以获得最大舒适度。它有4路帽子开关和多达120个命令可以在设备上编程。

以前我经常去参加许多 CPL 赛事，我不记得曾经见过锦标赛选手使用这样的东西。话说回来，当我退出舞台时，本质上模仿键盘的设备才刚刚开始出现。有谁知道现在有没有重要的玩家使用鼠标和键盘以外的东西？
