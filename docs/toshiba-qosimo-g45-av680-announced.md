# 东芝 Qosimo G45-AV680 发布

![](img/07fc235869da96a6b83130ba4720e13f.png)

过去，东芝笔记本电脑并没有得到我的好评。他们经常被一些问题所困扰，这些问题掩盖了写在盒子侧面的漂亮的规格。然而，新的 Qosimo G45-AV680(以下简称为“Q-Daddy”)似乎要废除东芝所有动荡的过去。

首先，Q-Daddy 采用了 HD DVD-R 硬盘。是的，那是一个 HD DVD **燃烧器**，你可以随身携带。30GB 的色情、盗版或被盗的办公文档，可以放在光盘上。显然，HD DVD 回放也是一个选项，在 1080p 回放和 17 英寸 WUXGA 宽屏显示器之间，你可以很好地去。包括外部高清电视调谐器，802.11n WiFi，512MB DDR2 GeForce 显卡，杜比家庭影院声音，以及使用 SATA 驱动器实现最高速度的 320GB 存储空间。

要有礼貌。在开始全面测试之前，先清理一下你的口水。
