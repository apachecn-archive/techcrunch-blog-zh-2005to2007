# 三星数码相机/相框混搭

![samsung2.jpg](img/9a7da9a65a51f6303fd62291f8335c94.png)

设计师振宇·韩的数码相机/相框混搭概念简直太壮观了。我经常发现自己想向朋友炫耀我的最新镜头，但背面的小液晶显示器就是不行。凭借三星 S700 的概念，你可以拍摄照片，并立即向任何人展示，这要归功于内置的支架。当然，液晶显示器没有那么大，但一个简单的幻灯片应用程序会解决这个问题。我当然希望三星在这个设计理念上有所作为。
