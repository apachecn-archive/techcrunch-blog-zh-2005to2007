# 与 NHL 建立媒体合作伙伴关系

![](img/afd73ce04d2ff208817c21e6945ccd9b.png)

那些渴望鲜血和弹弓的人将会非常高兴地得知 [NHL](https://web.archive.org/web/20200307102547/http://crunchgear.com/2007/04/09/akimbo-courts-pc-users-nhl-fans/) 已经与弹弓媒体签署了一项内容协议。用户现在将能够剪辑+悬挂曲棍球比赛的一部分，并将其发送给他们的朋友和家人。

想知道什么是夹子+吊带？这是 Sling Media 即将向想要分享视频剪辑的客户提供的服务。你只需点击几下鼠标，就能录制和回放你在电视上看到的片段。Sling 也将很快推出一个网站，人们可以张贴他们的剪辑+Sling 捕获和剪辑。听起来 Sling Media 现在正朝着几个不同的方向发展。看看六个月后他们在哪里会很有趣。
