# 互联网地址簿身份搜索引擎。TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/05/the-internet-address-book-the-identity-search-engine/>

大约一个月前，Roberto Bonanzinga 向我提供了互联网地址簿(IAB)T2 的详细信息，这是一个身份搜索引擎，扫描了许多流行的社交网络——MySpace、LinkedIn、Delicious、Skype 等。–查找一个人的各种身份和/或资料。尽管这个特性本身可能很有用，但它相当单一。当时，它似乎只是另一个网站，我不得不投入大量时间注册我的社交网络身份、即时消息 id 和网址，但回报却很少。

从那以后，IAB 发布了许多新的特性和功能，这无疑使这个网站更加有用，值得多花一点时间。[新工具](https://web.archive.org/web/20150901064435/http://www.internetaddressbook.com/tools.php)页面提供了类似于**搜索** [片段](https://web.archive.org/web/20150901064435/http://www.internetaddressbook.com/my_iab_check_snippets.php "http://www.internetaddressbook.com/my_iab_check_snippets.php")的工具，你可以将这些工具放在你的个人博客、个人资料、电子邮件或网站上。

他们还为 Internet Explorer 7 和 Firefox 用户(mozilla 浏览器)开发了一个搜索插件**以及一个为苹果电脑用户开发的小工具**。他们还在互联网地址簿中添加了更多的互联网地址。例如 Profileheaven.co.uk、Ryze.com、Technorati、Picturetrail、Buzznet 等等。要查看完整列表，请访问[互联网通讯录](https://web.archive.org/web/20150901064435/http://www.internetaddressbook.com/ "http://www.internetaddressbook.com/")。

另一个有趣的新工具是**招聘片段**，你可以把它放在你的博客或个人资料上，列出你要找的人。他们还可以让您轻松地从 Live Messenger 或 Outlook 中一次性邀请所有联系人[！](https://web.archive.org/web/20150901064435/http://www.internetaddressbook.com/my_iab_invite.php?firstName=your%20friends "http://www.internetaddressbook.com/my_iab_invite.php?firstName=your%20friends")

它们还被添加到基于 flash 的 PIDY(个人身份)小工具中，您可以配置该小工具，然后将其添加到您的网站、个人资料或博客中。下面是我的 PIDY 的一个例子。

随着我们都拥有的社交网络、即时消息客户端和网站账户的数量不断增加，将它们整合在一起无疑是一种增长趋势。其他做类似事情的网站包括 [PeopleAggregator](https://web.archive.org/web/20150901064435/http://www.peopleaggregator.com/) ，仅仅是因为传统的白页、黄页或主流搜索引擎网站似乎没有很好地找到我们的网络身份。

我可以想象这项服务要么被雅虎收购并整合到他们自己的垂直搜索服务中，要么被谷歌和微软复制。
