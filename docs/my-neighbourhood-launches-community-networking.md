# 我的邻居发起了社区网络。

“我的邻居”是一个新的英国“社区网络”网站，刚刚推出，目的是“改善全英国数千个社区的交流和邻里行为”。正如首席执行官丹尼·布尔所说

仔细观察，该网站是当地新闻和评论加上当地分类广告与社交网络相结合的混搭。也就是说，想象一下[UpMyStreet.com](https://web.archive.org/web/20201124054515/http://www.upmystreet.com/)+[鱼 4](https://web.archive.org/web/20201124054515/http://www.fish4.co.uk/iad/homes) + [朋友重聚](https://web.archive.org/web/20201124054515/http://www.friendsreunited.co.uk/)的组合，除了在这种情况下，到目前为止，大部分初始本地信息都是通过与[内斯特里亚](https://web.archive.org/web/20201124054515/http://www.nestoria.co.uk/)和[酷航](https://web.archive.org/web/20201124054515/http://www.scoot.co.uk/) 的合作关系提供的。

这可能是一个本地化社交网络和列表的好主意，但我有一些保留意见，我希望能在未来的测试版中解决。首先，虽然注册是免费的，但是为了与我的邻居联系和/或通过分类市场销售商品，我必须升级我的会员资格并支付 6.5 英镑的年费——为什么！？

当网站是一个试图发展社区的阿尔法时，这种进入和参与的障碍对我来说似乎很奇怪，特别是当英国已经有更知名的免费列表网站时，如 [CraigsList](https://web.archive.org/web/20201124054515/http://london.craigslist.org/) 、[易贝](https://web.archive.org/web/20201124054515/http://www.ebay.co.uk/)和其他已经建立社区(买家/卖家)的网站。甚至我当地报纸的在线分类列表也是免费的。

就我个人而言，我更希望看到 MN 从当地广告商那里获得收入(但仅限于那些在当地社区获得好评的企业投放广告)，但我猜这可能是“当前”企业列表合作伙伴 Scoot 施加的限制。
