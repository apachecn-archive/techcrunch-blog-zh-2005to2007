# 潘多拉 WiFi 播放器来了

Pandora 昨晚开始移动，并在另一份声明中发布了一款新的 WiFi 设备，该设备是由给我们带来雅虎的 WiFi 音乐播放器 SanDisk 的同一批人生产的。

据 TechCrunch [报道](https://web.archive.org/web/20201101002037/http://www.beta.techcrunch.com/2007/05/23/prototype-of-pandora-wifi-device-shown-tonight-in-san-francisco/)，CTO Tom Conrad 演示了一个设备原型，如下图所示。关于这个球员的细节还有很多悬而未决的事情。它会有硬盘吗？要花多少钱？什么时候发布？然而，我们知道这款播放器将由 Zing 提供动力，并运行你已经知道并喜爱的潘多拉站。
