# 谈论工作的新论坛网站

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/07/25/new-forum-site-to-gossip-at-work/>

我将简单介绍一下[沃克帕尔](https://web.archive.org/web/20150909075553/http://www.wurkpal.com/)。这是一个非常简单的论坛网站，专门为人们八卦/抱怨等而设计。匿名谈论他们的工作环境。该网站根据反向主机名查找猜测您的登录位置，并尝试将您引导至正确的公司论坛。你也可以查看[目录](https://web.archive.org/web/20150909075553/http://www.wurkpal.com/groups/1)，然后从那里开始。该网站急需一个搜索功能和增加新公司的能力。

我们已经介绍了大致处于这一领域的几家公司——参见 [BossBitching](https://web.archive.org/web/20150909075553/http://www.techcrunch.com/2006/08/08/bossbitching-whos-been-your-worst-boss/) 、 [Ovehear.us](https://web.archive.org/web/20150909075553/http://www.techcrunch.com/2007/05/04/overhearus-corporate-gossip-2/) 和 [JobVent](https://web.archive.org/web/20150909075553/http://www.techcrunch.com/2007/06/06/jobvent-takes-dirt-dishing-to-a-broader-audience/) 。对于那些想要打败拒绝他们的风险投资家的企业家来说，资金总是很有趣的。这些都不能与当年的 FuckedCompany 相提并论，但这种类型的网站要经历一次大的衰退才能再次看到辉煌的日子。

Wurkpal 位于圣何塞，并表示他们已经获得了天使投资。
