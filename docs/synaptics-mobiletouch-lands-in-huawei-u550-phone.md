# Synaptics MobileTouch 登陆华为 U550 手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/23/synaptics-mobiletouch-lands-in-huawei-u550-phone/>

![u550.jpg](img/2e7616e7be62d0bfd5003817bfab6a7d.png)如果您从未见过 Synaptics MobileTouch 技术的实际应用，请访问他们的网站并[观看演示视频](https://web.archive.org/web/20131113121316/http://www.synaptics.com/products/mobto.cfm)。但基本上它采用了你在使用该公司产品的笔记本电脑上找到的所有有趣的触摸板技术，并将它们应用到更小的移动设备上——在这种情况下，就是华为 U550 翻盖手机。

背光的三按钮 MobileTouch 接口充当手机外部的 MP3 播放器控件，允许您播放、暂停、快进和快退音乐。是的，这项技术的使用非常简单，但正如我所说，查看演示视频，你可以更好地了解 Synaptics 创建的触敏表面的所有可能。
