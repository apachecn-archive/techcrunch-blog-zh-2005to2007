# Gada.be 的新外观、新名称

 Tag 元搜索引擎 [Gada.be](https://web.archive.org/web/20211027004919/http://gada.be/) (TechCrunch 上描述该服务的帖子是[这里是](https://web.archive.org/web/20211027004919/http://www.beta.techcrunch.com/tag/gada.be))将在本周晚些时候举行的 Gnomedex 大会上进行更名和重新设计(Chris Pirillo 创建了该公司，也负责该会议)。

这项新服务将被称为 TagJag(并将登陆 TagJag.com)。像 Gada.be 一样，TagJag 可以很容易地从移动设备上查询。

Gada.be [于 10 月份推出](https://web.archive.org/web/20211027004919/http://www.beta.techcrunch.com/2005/10/10/gada-tag-meta-search-done-right/)，据该公司称，每天提供约 30，000 次查询。在网站的[关于](https://web.archive.org/web/20211027004919/http://gada.be/about)部分可以看到更多相关信息。
