# T-Mobile 将于周五推出真正的 3G 数据网络？

在美国，这意味着它终于可以迎头赶上了。

那一天似乎很快就要到来。事实上，eWeek 的消息来源透露，10 月 6 日星期五，T-Mobile 将在纽约市举行新闻发布会，宣布推出其新的 3G 数据网络。不仅如此，他们详细说明了该网络将可悲地与欧洲 UMTS 标准不兼容，因为使用的频率是不同的。不管怎样，我们都在使用移动数据。
