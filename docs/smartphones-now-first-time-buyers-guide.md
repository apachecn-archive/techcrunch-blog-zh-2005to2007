# 智能手机现状:首次购买指南

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/11/03/smartphones-now-first-time-buyers-guide/>

所以你决定购买你的第一部智能手机。这太棒了，但在你发疯般地放弃汽车付款，转而选择 PDA 遇上手机的美好时光之前，你需要停下来想想哪一款智能手机适合你，但亲爱的读者，这就是 CrunchGear 出现的原因。

因为市场上有这么多不同的设备，我们应该从确定智能手机到底是什么开始，这比听起来要难，因为没有官方定义。一般来说，智能手机是手机和 PDA 的结合，但许多传统手机正在通过添加电子邮件同步、网页浏览和即时消息等功能来侵占智能手机的领地。

然而，一部真正的智能手机通常还包括办公文档阅读(通常是写作)、替代文本输入(QWERTY、触摸屏、两者或不同的东西)，以及加载和启动第三方应用的能力。

在本指南中，我们不认为黑莓和 SideKicks 这样的设备是真正的智能手机，尽管它们经常被归为智能手机。这些是通讯器，一个奇怪的智能手机同父异母的兄弟，只有当他需要保释金时才会出现。


智能手机类似于 1997 年左右的个人电脑。那时，电脑仍然被认为是商人或极客的领地，不是每个美国足球妈妈都会拥有的东西。不到十年后，很难想象你家里没有至少一台电视机的生活。随着厂商首次将目标瞄准非商业客户，智能手机目前正在跨越同样的门槛，这一细分市场我们称之为“临时智能手机用户”。

这并不意味着它们对每个人都是正确的，还不是，但是它们已经走得足够远，如果你有任何跳跃的倾向，现在是这样做的好时机。但首先我们应该看看为什么你会有一个。面对现实吧，不是为了看起来酷。除了把手机别在腰带上，唯一让手机看起来更像工具的方法就是把智能手机别在腰带上，或者更糟，别在腰带上。你想要一部智能手机，因为你是那种不仅想要有条理，还想保持联系的人。如果你只需要让你的时间表保持直线，那么 PDA 就很好地完成了这项工作。然而，如果你的约会比“24 小时”的忠诚度变化更快，那么你就是理想的智能手机候选人。

好了，我们已经确定了它是什么以及你为什么想要它，这是最简单的部分。现在哪个智能手机是你的智能手机？这可能是有史以来人类被问到的最难的问题，就我们的目的而言，我们认为是在 2002 年左右。

硬件方面，有两条路可走:触摸屏和非触摸屏。虽然触摸屏似乎是显而易见的，但值得考虑的是，这些设备比它们的非触摸表亲要贵得多。它们也往往更大，因为用于校准和记录触摸动作的附加设备占据了自己的空间。例如，T-Mobile MDA 或 Sprint Treo 700p 比它们的标准屏幕兄弟更大。

这并不是说非触摸屏智能手机不好。事实上，T-Mobile Dash 和威瑞森的摩托罗拉 Q 都提供完整的 QWERTY 文本输入和单手导航，非常直观和易于使用。Cingular 3125 避开了标准电话 T9 键盘的文本输入键盘，但具有纤薄的外形。

你应该问问自己，你打算用你的智能手机做什么。如果你正在考虑将现有的手机扩展到下一个级别，非触摸屏设备可能是正确的，因为它将基本的智能手机功能与熟悉的外形和界面相结合。如果你更喜欢阅读电子邮件，而不是写作，那么 QWERTY 键盘可能就太多了。然而，如果你是一个来自另一个方向的超级用户，一个希望将你的设备结合起来的 PDA 用户，那么你会更喜欢触摸屏和 QWERTY 设备，因为它会更接近地模仿你现在使用的设备。

在考虑外形之后，你应该考虑你的操作系统。全球最受欢迎的智能手机操作系统 Symbian 刚刚在美国取得进展，但在 Cingular 的诺基亚 E62 中表现令人印象深刻。Palm OS 始终是一个受欢迎的起点，因为它的易用性和捆绑应用程序，以及在屡获殊荣的 Treo 硬件上运行，使它什么都不缺。如果与您现有企业的通信是一个交易杀手，Windows Mobile 5 设备提供了与 Exchange 和 Active Directory 的内置集成。此外，这款操作系统拥有最多的“口味”，其中之一你很可能会觉得很有吸引力。

另一个真正的问题是数据传输。这些设备在魔法仙尘上不起作用(Nod 3354i Nvrlnd 除外)。如果你的设备不能上网，那么双向电子邮件同步、集中联系人管理器和手机色情都是没有用的。这就是智能手机携带的各种次级无线电发挥作用的地方。在美国，我们有两种服务。EV-DO/RTT，受到 Sprint 和威瑞森等 CDMA 手机网络的青睐，以及 T-Mobile 和 Cingular use 等标准 GSM 提供商的 GPRS/EDGE/UMTS/HSDPA。当检查可用的智能手机时，查看它们的“数据包”格式，以确保它与您的提供商相匹配。

有些版本比其他版本快。所谓的 3G 技术，如 EV-DO 和 UMTS/HSDPA，远远快于 2 或 2.5G 技术，如 GPRS/EDGE 和 RTT。不过，无论哪种方式，你都必须在注册语音套餐的同时注册一个次级数据套餐，这是很多人都没有意识到的，也是一个可能会扼杀交易的东西。根据提供商和速度的不同，这些套餐每月收费 20-40 美元，但如果你选择无限量套餐，你会感谢我们的。

我们已经讨论了很多，但是不要被吓倒。如果你一步一步地遵循这个指南，并仔细研究你希望智能手机为你做什么，你就能利用这些信息找到你的手机。不要被任何其他用户的意见所左右；对他们来说是对的对你来说可能是错的，反之亦然。例如，我们离不开触摸屏 Treo，尽管我们的一位非常出色的同事非常信赖他的 Cingular 3125 翻盖手机。但他也是个有六颗牙齿的瘾君子。

总的来说，最重要的是你的需求，如今市场上有很多智能手机，一定有一款能很好地满足它们。
