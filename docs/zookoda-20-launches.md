# Zookoda 2.0 发布 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/05/07/zookoda-20-launches/>

 [](https://web.archive.org/web/20221007030217/http://www.zookoda.com/) 澳大利亚的 [Zookoda](https://web.archive.org/web/20221007030217/http://www.zookoda.com/) 博客电子邮件工具 2.0 版本今天发布。Zookoda 允许博客定期向读者发送电子邮件。它与 Feedburner 的新电子邮件产品竞争，虽然它没有 Feedburner 庞大的客户群，但它确实提供了一个非常强大的功能集，对许多博客作者来说很有吸引力。

几周前，创始人 Yorke Hinds 和 Nick McNaughton 向我介绍了新产品。像 Feedburner 一样，该产品目前是免费的。虽然 Feedburner 只提供单一的电子邮件选项(每天一次，无法调整电子邮件的外观和感觉)，但 Zookoda 为博客提供了定制产品几乎每个方面的能力。

例如，Zookada 提供模板定制和预定的每日、每周和/或每月定期电子邮件。Zookoda 还提供了非常深入的报告功能，以查看什么被阅读，点击等。

如果你是一个正在寻找解决方案的博客作者，Zookoda 或 Feedburner 都可以。我喜欢 Feedburner，因为坦率地说，我不想花很多时间格式化模板和浏览详细的报告:Feedburner 的一刀切方法非常适合 TechCrunch，我喜欢我所有的 RSS 和电子邮件订阅统计数据都在一个仪表板中管理的事实。对于寻找更强大的、可定制的解决方案的博客作者，或者想要更详细的报告的人来说，Zookoda 是一个非常用户友好的产品，将会工作得非常好。

博客为读者提供电子邮件选项很重要——有些人喜欢通过这种方式接收内容。无论是 Zookoda、Feedburner 还是其他解决方案，都可以考虑提供这种解决方案。
