# 三星发布北京奥运会官方手机

随着奥运会即将来临，我指的是又一年，三星宣布了三款专为即将到来的奥运会设计的高规格手机。配备 200 万像素摄像头和 10.6 毫米高清晰度的 E848 滑盖加入了 BOCOG 网络，并希望在中国提升三星品牌。改善中国移动的努力将来自 P318 和 U308 手机。U308 真的是[超二版翻盖](https://web.archive.org/web/20130628201833/http://crunchgear.com/2007/02/13/hands-on-with-all-the-samsung-ultras-can-you-say-ultra-purdy/)。各位，你们已经收到通知了。如果你明年要去北京，那么你就知道最热门的手机会是什么。
