新闻集团收购Milkround

路透社报道称，新闻集团最终收购了 Milkround ，这是一家面向毕业生和学生的英国求职网站，由汤姆·休斯于1996年创立。收购价号称2000万左右。“milkround”这个名字来源于一个招聘会的表达，或者是那些打算尽早招聘大学毕业生的公司在校园里举行的面试。

在这次英国收购之前，福克斯互动媒体(FIM)在4月份投资购买了美国求职引擎的公司的股份，该公司的特色是现在的 MySpace Jobs 。

更新 : Simon Cast 刚刚写信给我，谈到他关于英国年龄歧视法(2006年就业平等(年龄)条例)的帖子，该法于10月1日生效。

“如果你把招聘限制在大学‘牛奶轮’，你会发现这是间接的年龄歧视，因为这种做法会严重限制25岁以上的人申请你的职位空缺的机会。”
