# Rapleaf 现已上线| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/05/07/rapleaf-is-now-live/>

 [](https://web.archive.org/web/20230301140033/http://www.rapleaf.com/) [ Rapleaf 的](https://web.archive.org/web/20230301140033/http://www.rapleaf.com/)开放反馈系统([这里有更多信息](https://web.archive.org/web/20230301140033/https://techcrunch.com/2006/04/23/rapleaf-to-challenge-ebay-feedback/))于周六深夜上线，[已经有数百名用户在其他人身上留下了数千条反馈帖子。](https://web.archive.org/web/20230301140033/http://rapleaf.blogspot.com/2006/05/were-live-and-stats.html)

首席执行官 Auren Hoffman 承诺在本周的某个时候为核心 Rapleaf 函数提供 API，所以期待很快看到 Rapleaf mashups。用户可以通过代码片段拉出他们的“信誉箱”,并将其发布在网站上。我的在下面。

我在[Rapleaf 上的 TechCrunch 声誉评分在这里 T10。](https://web.archive.org/web/20230301140033/http://www.rapleaf.com/profile/sRyxaC7M)[正如我之前所说的](https://web.archive.org/web/20230301140033/https://techcrunch.com/2006/04/23/rapleaf-to-challenge-ebay-feedback/)，我相信如果 Rapleaf 能够完成正确的业务开发交易，他们将成为赢家。
