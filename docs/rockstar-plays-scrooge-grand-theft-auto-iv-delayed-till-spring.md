# 摇滚明星扮演守财奴:侠盗猎车手四推迟到春天

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/03/rockstar-plays-scrooge-grand-theft-auto-iv-delayed-till-spring/>

[![gta4delay.jpg](img/f3d2e96f3466b89c4b6f748ae80db53d.png)](https://web.archive.org/web/20151211064725/http://old.crunchgear.com/wp-content/uploads/gta4delay.jpg "gta4delay.jpg")

不要怪索尼和微软游戏部门的头头们今天想睡懒觉；他们刚刚得知 [*侠盗猎车手 IV*](https://web.archive.org/web/20151211064725/http://crunchgear.com/category/grand-theft-auto-iv/) 已经推迟到明年春天，扔下各自的假期阵容。以 Two 为例，这款游戏的发行商在与 Rockstar 的问题上拥有最终决定权，它推迟了这款游戏，因为“需要额外的开发时间来完成游戏。”

又一款备受瞩目的视频游戏延期了。我多么希望更多的开发者能够简单地采用“当它完成时”的策略。这个发布日期一两步就像一个漫长的，拖长了戏弄。

游戏的延迟——你真的对它的延迟感到惊讶吗？—对索尼或微软在这个假期的地位没有什么帮助。 *GTA IV* 本应是今年的系统销售商，尤其是在 Konami 推迟 *Metal Gear* 的消息传出后对索尼而言。毫无疑问，任天堂对事态的转变感到高兴。

还剩下什么杀手级 app，卖系统的游戏？我知道*刺客信条*和*天剑*，但就这些吗？我们都知道任天堂的策略:它将推出一些钓鱼模拟器，最终将售出大约 1100 万份。
