# Tascam DAP 是音乐人的天赐之物

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/31/tascam-dap-is-a-godsend-for-musicians/>

[![tascammp3.jpg](img/06750bf8eaf6f732a0caa2227f0bf442.png)](https://web.archive.org/web/20160914070929/http://old.crunchgear.com/wp-content/uploads/tascammp3.jpg "tascammp3.jpg")

显然 Tascam 这个品牌在音乐界很“大”,因为 Vince 刚刚在我们的聊天室说它有多棒。Tascam 设计了这个 DAP，MP-VT1，考虑到音乐家，这是当你会发现一条线和一个麦克风插孔。它还有其他只有音乐家才能欣赏的功能，比如变速试听，可以在不改变音高的情况下减慢歌曲的速度；歌曲可以配置成循环播放，这对 DJ 来说很好；你可以改变演奏音乐的音调。

所有这些推销谈话都让我感到口渴。

哦，那也不是全部。有一个内置的节拍器和乐器调音器。遗憾的是，这款设备的内存只够存储 240 首歌曲，但在我未经训练的眼里，它更像是音乐爱好者的瑞士军刀般的伴侣，其众多功能中只有一个是 MP3 播放。

任何音乐人——这是一个技术术语，是的——有使用 Tascam 的经验吗？还是文斯又在说摇摆舞了？
