# 使用 tag 2 find 为您的桌面材料添加标签

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/12/29/tag-your-desktop-stuff-with-tag2find/>

 [](https://web.archive.org/web/20220819182811/http://www.tag2find.com/) 标记应该是操作系统的一个特性。这通常是一种更好的组织信息的方式，因为一个文件只能放在一个文件夹中，但是多个标签可以与该文件相关联。这是我喜欢 Gmail 的原因之一，因为电子邮件可以用多个描述性术语来标记，而不仅仅是扔进一个文件夹。举例来说，我发现在 flickr、del.icio.us 和 gmail 上找到被标记的项目要比在 Mac mail 上容易得多，Mac mail 是我的桌面电子邮件客户端，不支持标记功能。正确标记的物品更容易在路上找到。

Vista 确实支持有限形式的标记，尽管很难实现。如果您右键单击一个文件，然后单击属性，然后单击详细信息，您可以为该文件输入一组标记。使用 Vista 的搜索功能，您可以通过搜索标签来找到这些文件。Windows XP 不支持标记，Mac 也不支持。

总部位于奥地利的 Tag2Find 创建了一个漂亮的应用程序，可以为任何类型的桌面文件添加标签。它只适用于运行 XP 或 Vista 的 Windows 机器，并且依赖于。NET 框架。在我们对 XP 机器的测试中，它表现得非常好。

要了解它是如何工作的，请看这里的演示。该应用程序位于屏幕底部的系统托盘中。点击它，会出现一个搜索栏来标记文件，或查找标记的文件(见右图)。要标记文件，请在标记中键入内容，然后将文件拖到图标上。Tag2Find 还会根据您输入的内容建议其他标签。

还有一个更大的视图选项，可以打开一个显示标记文件、标记云或标记列表的窗口，以及一个内置的媒体播放器来播放您已标记的媒体文件(见下图)。
