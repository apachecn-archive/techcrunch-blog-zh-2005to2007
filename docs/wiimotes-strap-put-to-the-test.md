# Wiimote 的表带经受考验

![](img/317f17030e53e3b97a5a04bf05043fc4.png)

在过去的几周里，有很多人抱怨 Wiimote 的表带太坚固，而且有断裂的趋势。自从 Wii 发布以来，我已经举办了三次以上的聚会，游戏玩家和非游戏玩家都聚在一起，喝很多啤酒，玩 Wii。我的朋友们也不是那种冷静的人，如果你在这些事件中走进我的公寓，你会看到纯粹的迷你游戏混乱。然而，我的 Wiimote 没有一根带子断过，也没有控制器被扔到房间的另一边。Wiimote 偶尔会撞到离它很近的东西(或人)，但这完全是用户的错。

任天堂决定自己动手，测试 Wii 的表带。Wiimote 表带第一次测试是在玩的时候，“不小心”让 Wiimote 走了。即使当用户试图通过放开 Wiimote 故意导致表带断裂时，表带仍然完好无损。第二个要测量的方面是表带的可拆卸强度(在不折断 Wiimote 的情况下，表带可以承受多少)，请看图片:

即使上面挂着 25 磅重的书和 100 液体盎司的 Mountain Fresh 洗衣液，这种带子也不会松动。这太难了。任天堂的测试人员唯一能做的就是用双手故意把它从 Wiimote 上扯下来。现在，这不是有助于清除整个皮带是“太弱”废话吗？我希望如此。只要记住玩的时候一直把带子绕在手腕上，如果你放开 Wiimote 而没有带上带子，它就会决定在你的墙上打一个新洞，这是你的错。
