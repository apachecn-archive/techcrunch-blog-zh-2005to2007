# 我的个人体重问题，见见我的个人电脑

![](img/61cefbd8190730e8cfe8be197ada4baf.png)

我从来没有说过我特别瘦，但是有些人(布莱克)一直在批评我的体重，最近(加文是个胖杂种——布莱克)。幸运的是，Tanita 开发了 PC InnerScan 量表。乍一看，这只是一个带有体重和体脂数字读数的高端秤。你在图片中看不到的是 USB 存储驱动器，你可以将其连接到你的电脑上，并在一些方便的软件的帮助下，记录你的体重。软件目前是 Windows 专用的，内存驱动只能监控四个人 30 天，但是说真的，什么家庭把自己称在一起？现在在亚马逊日本有售，价格约为 130 美元
