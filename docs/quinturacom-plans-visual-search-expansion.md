# Quintura.com 计划扩大视觉搜索

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/12/05/quinturacom-plans-visual-search-expansion/>

在最近发布之后，视觉搜索引擎[Quintura.com](https://web.archive.org/web/20160925222605/http://quintura.com/)计划发布一个为特定市场领域特别设计的界面。

[![Yakov Sadchikov](img/4b26b6ae7964527a87999810d5b20d79.png "Yakov Sadchikov")雅科夫·萨奇科夫](https://web.archive.org/web/20160925222605/http://company.quintura.com/company/management.asp)，Quintura 的联合创始人兼首席执行官([图片](https://web.archive.org/web/20160925222605/http://www.flickr.com/photos/darrenstraight/311876518/)由[达伦·斯特朗](https://web.archive.org/web/20160925222605/http://www.darrenstraight.com/)拍摄)，今天告诉 TCUKI:“下一步，我们将发布 Quintura 面向妇女和儿童的视觉导航/搜索网络服务。”Sadchikov 最近在 MediaTech 的初创公司展示会和 Firefox party 上展示了该网站。

上个月，这家以俄罗斯人为主的初创公司(其合法总部位于弗吉尼亚州亚历山大，但在莫斯科有软件开发业务)从卢森堡的[红树林资本](https://web.archive.org/web/20160925222605/http://www.mangrove-vc.com/)(Skype 的早期投资者) [ABRT 风险基金](https://web.archive.org/web/20160925222605/http://www.abrtfund.ru/eng/)和波士顿的[OpenView Venture Partners](https://web.archive.org/web/20160925222605/http://www.openviewpartners.com/)的合伙人那里获得了“大量”风险投资。尽管 Quintura 声称是第一家获得西方风险投资的俄罗斯互联网公司，但这似乎不太可能。这些资金将用于更积极地推广网站。

还有更多的计划。

Quintura 服务(通用和图片搜索)目前由 Yahoo XML feeds 提供支持。但事实上，Quintura 可以在任何搜索引擎之上工作，作为一个可视化的表示层。为了这个目的，Sadchikov 说他们也“正在测试一个视频搜索引擎 blinkx”视觉视频搜索将给 Quintura 带来大多数其他利基搜索服务所没有的优势。

那么，为什么昆图拉是一个非常酷的网站呢？视觉标签云非常直观，可以帮助用户完善他们的搜索词。这使得搜索变得不那么令人沮丧，而且速度更快。当你从一个视觉云漫游到另一个视觉云的时候，它也引入了更多的发现。一幅画胜过千言万语。Quintura 在屏幕的一半显示相关关键词和短语的交互式视觉云，而在下方显示传统的搜索结果。当您将鼠标悬停在标签上时，它会显示相关的关键字，并在您单击时动态更新结果。

![quintera.gif](img/d9e3ec1f582b37c7105d696ccdf304fa.png)

用户还可以保存和分享他们的搜索结果:一个联盟计划将允许网站使他们的标签云互动。保存搜索意味着用户可以建立一个预设的搜索云，让访问者进一步深入。

没有提到 Quintura 的投资者将如何退出，但我们认为视觉搜索几乎肯定会对主要搜索引擎有用，他们将要么开发自己的公司，要么收购在该领域拥有良好知识产权的公司。众所周知，Quintura 的系统背后有一套复杂的算法。

Quintura 最近还登上了[商业周刊](https://web.archive.org/web/20160925222605/http://www.businessweek.com/magazine/content/06_48/b4011067.htm)的封面，并在[商业 2.0 杂志](https://web.archive.org/web/20160925222605/http://money.cnn.com/2006/08/09/technology/webaroundtheworld.biz2/)八月刊中被评为世界上最热门的创业公司之一。
