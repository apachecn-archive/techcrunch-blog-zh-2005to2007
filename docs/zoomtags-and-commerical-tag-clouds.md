# ZoomTags 和商业标签云

 [](https://web.archive.org/web/20220519070536/http://www.zoomtags.com/) [ZoomTags](https://web.archive.org/web/20220519070536/http://www.zoomtags.com/) ，总部位于加利福尼亚州森尼维尔，是由 [1000Tags](https://web.archive.org/web/20220519070536/http://www.beta.techcrunch.com/2006/01/10/1000tags-another-simple-brilliant-idea/) 引入的商业 tagcloud 理念的专业实现，我在本月早些时候写过。

ZoomTags 是一个基于 API 的解决方案，允许博客和其他网站在其网站上放置商业标记云。像 1000 个标签一样，Zoomtags 允许广告商对标签云中的标签或关键词进行竞价。当有人点击一个标签时，他们会看到一组广告结果。这是标签云中的 Adsense 类型的广告。

网站所有者将有[多种多样的标签云](https://web.archive.org/web/20220519070536/http://www.zoomtags.com/misc/wtagFeat.html)可供选择。

ZoomTags 是 ZoomGroups 的一部分，自 1999 年以来已经建立了一个相当成熟的广告客户网络。

当然，如果这被证明是一些网站提供广告的更好方式，没有什么能阻止雅虎和谷歌实现类似的界面。
