# Basecamp 的新东西——TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/12/13/new-stuff-at-basecamp/>

昨天和今天,“37 Signals”宣布了他们广受欢迎的(在主页上有 100，000 名用户)和有用的 [Basecamp](https://web.archive.org/web/20210920213600/http://basecamphq.com/) 项目管理产品的几个新功能。

首先，Basecamp 现在[直接在他们的服务器](https://web.archive.org/web/20210920213600/http://basecamphq.com/forum/viewtopic.php?id=1235)上托管文件，不需要建立你自己的 FTP 服务器。可以上传 20 mb 的文件。

第二，Basecamp 创建了一个[基本会员计划](https://web.archive.org/web/20210920213600/http://basecamphq.com/forum/viewtopic.php?id=1250)。如果一家代销商向 Basecamp 推送新的付费会员，用户可以从他们的 Basecamp 费用中获得积分。
