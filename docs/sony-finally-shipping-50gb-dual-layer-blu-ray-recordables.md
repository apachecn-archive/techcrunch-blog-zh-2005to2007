# 索尼终于推出 50GB 双层蓝光刻录光盘

蓝光之路崎岖不平，但索尼最终设法将他们的双层可写蓝光光盘推向市场。每张光盘大约 48 美元，以美元对千兆字节的比率，现在输给了 DVD 刻录。这应该在几年内改变，但与此同时，DVD 仍然是节俭型刻录机的冠军。

那些考虑将这些蓝光光盘用于备份目的的人可能应该等到价格下降，或者至少等到 50GB 双层可重写光盘在今年晚些时候上市。
