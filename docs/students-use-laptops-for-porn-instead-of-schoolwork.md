# 学生用笔记本电脑看黄片，而不是做作业

[![kidslaptops.jpg](img/34924acfe1eeaaca757e239b0339a864.png)](https://web.archive.org/web/20210118175336/http://old.crunchgear.com/wp-content/uploads/kidslaptops.jpg "kidslaptops.jpg")

真正令人震惊的消息:高中生没有把学校发放的笔记本电脑用于教育目的，而是用它们来发送即时消息、浏览网页和沉浸在色情中(T4)。说不是这样的。所以全国各地的学区，相信给孩子们笔记本电脑会，我不知道，让他们变得更聪明，正在取消校内笔记本电脑计划，让孩子们用传统的方式学习，用书和铅笔。

最好的一点是，在对有笔记本电脑的学校和没有笔记本电脑的学校进行的大量比较中，有笔记本电脑的学校没有看到考试成绩的提高，因为考试成绩是当今衡量聪明程度的标准。但我想知道的是，学区希望这些孩子用笔记本电脑做什么。*学？*哈！住在隔间里的普通工蜂每天花多少小时在电脑上打发时间？我记得在高中的时候，我们更关心的是如何绕过 Websense 过滤系统，而不是研究交给我们的任何繁忙的工作任务。

底线是，笔记本电脑不会仅仅因为它们的存在而改善教育。但是让我们继续向学校砸钱，那会解决问题的。
