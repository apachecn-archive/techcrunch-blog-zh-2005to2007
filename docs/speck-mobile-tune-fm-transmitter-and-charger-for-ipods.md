# 斯派克移动调谐调频发射机和 iPods 充电器

这里有一个 iPod 配件，它被设计成可以融入你的汽车装饰中。移动调频发射机和充电器插入点烟器，插入汽车内部，看起来像顶部镀铬的换挡手柄。它是用皮革装饰的，有一个背光的蓝色表盘指示器，如果你的车是用皮革装饰的，有背光的蓝色表盘指示器和镀铬装饰。

但你必须找到一些东西来处理那根悬空的电线，因为 Mobile Tune 的通用基座连接到你的 iPod，它必须沿着通向它的电线占据一定的空间。虽然使用充电器/FM 发射器将 iPod 信号传输到车载收音机很方便，但根据我们的经验，这些设备工作不太好，带来的噪音超过了你的承受能力，除非你以恒定的 80 英里/小时的速度行驶。也许这是不同的，但我们对此表示怀疑。
