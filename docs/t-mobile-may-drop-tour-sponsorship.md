# T-Mobile 可能会放弃巡演赞助

环法自行车赛的最大赞助商可能会退出，这可能会在本周末的比赛结束后发生。T-Mobile 团队将在周日得知手机运营商是否会撤回赞助。

T-Mobile 的通信主管克里斯蒂安·弗罗默特说:“我们会考虑一些事情，并在今年年底做出决定。”弗罗默特的公司与车队的赞助合同一直延续到 2010 年。

环法自行车赛是职业自行车赛中最大、最引人注目的比赛，T-Mobile 是最大的支持者之一。上周，T-Mobile 车手帕特里克·辛克维兹(Patrik Sinkewitz)被发现血液中的睾丸激素水平异常，并被车队停赛，这使得德国自行车运动的声誉受到损害，T-Mobile 的声誉也受到了一定程度的影响。金融时报德国版*最近的一项民意调查显示，只有 15%的德国人仍然对今年的巡回赛感兴趣。*

 *今年甚至有传言称，这场比赛将通过第三块屏幕吸引观众，并成为 T-Mobile 的天然发射台。但在上周另外两名车手撞车以及这些药物指控之后，这更可能是 T-Mobile 很快就会忘记的一次旅行。
