# 三星 Blast 现适用于 T-Mobile

![](img/3ccbcda2f689974c0018e4f7940788ec.png)

T-Mobile 可能没有 iPhone，但它肯定有其他不错的手机可供选择。今天推出的是三星 Blast，这款滑盖具有一些令人印象深刻的功能，不会让你倾家荡产。你会发现一个 130 万像素的摄像头，高达 2GB 的存储 microSD，集成的 AOL 和 Yahoo！邮件、媒体播放器、蓝牙和四个即时消息客户端(ICQ、AIM、Windows Live 和 Yahoo！信使)。

还有一个 262k 的彩色屏幕，EDGE 和 myFaves，这样你就可以免费给你的敌人/敌人打电话了。旅游？疾风有四频 GSM，所以覆盖范围应该相当不错。如果你想要一个，你必须支付 100 美元，签订两年的合同。嘘呀！
