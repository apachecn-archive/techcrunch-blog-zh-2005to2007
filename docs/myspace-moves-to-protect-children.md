# MySpace 采取行动保护儿童

据《纽约时报》今天报道， [](https://web.archive.org/web/20220516081022/http://www.myspace.com/) 下周将会看到 [MySpace](https://web.archive.org/web/20220516081022/http://myspace.com/) 试图保护年轻用户的一些举措。最大的变化将是，注册为 18 岁以上的用户将被要求输入 16 岁以下用户的全名或电子邮件地址，以便与他们联系。现在，用户名足以向 MySpace 中的任何人发送好友请求。该计划的明显缺陷是恶意的成年人可以很容易地注册为任何年龄，从而绕过成人和儿童的限制。

要做的其他改变包括任何人都可以将自己的个人资料页面标记为私人的新功能，以及删除青年用户页面上的成人服务广告，如约会网站。自动创建 16 岁以下注册用户的所有资料可能会更有效，但可能会使这项服务在喜欢在离线世界结识其他不认识的孩子的孩子中不受欢迎。

可能没有明确的方法来保护儿童在网上免受试图伤害他们的成人的伤害。同样值得注意的是，根据美国司法部(pdf)[2000 年的一项研究(或在](https://web.archive.org/web/20220516081022/http://www.ojp.usdoj.gov/bjs/pub/pdf/saycrle.pdf) [HTML](https://web.archive.org/web/20220516081022/http://66.102.7.104/search?q=cache:UDf8R5o0l4AJ:www.ojp.usdoj.gov/bjs/pub/pdf/saycrle.pdf+Children+sexually+assaulted+by+family+members&hl=en&gl=us&ct=clnk&cd=6&client=firefox-a) 中)，34%的美国青少年性侵犯罪犯是受害者的家庭成员。教育、父母指导和比网络世界更深刻的文化变革可能是解决儿童危险的唯一可行办法。
