# GadgetLocker 宣布 iPod 回收计划

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/06/thegadgetlocker-announces-ipod-recycling-program/>

![](img/7318bfb578a68eae8fb110d6e11e8ba0.png)

猜猜孩子们。TheGadgetLocker.com 现在提供一个 iPod 回收项目，你可以处理你的坏的或者老化的 iPod。TheGadgetLocker 提供所有的运输标签，所以你不用支付一分钱。此外，发送您的 iPod，您将获得 20 美元的奖励积分。该计划的目标是让消费者了解 GadgetLocker.com 的商品，同时保护环境。

另外，我想宣布我的 iPod 回收计划。你可以把坏掉的、用过的、旧的和新的 iPods 送到我家门口，作为回报，我会微笑着向你竖起大拇指。然后我转身在 Craigslist 上卖一两块钱的 iPods，买一瓶龙舌兰酒来抚慰我的悲伤。等等，什么？
