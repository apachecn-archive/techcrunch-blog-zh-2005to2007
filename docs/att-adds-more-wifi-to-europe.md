# 美国电话电报公司为欧洲增加更多 WiFi

![](img/b385d45d28dc8eb2b39f091195992452.png)

这里没什么特别的，只是对那些将来住在或去欧洲的人来说是个好消息。美国电话电报公司已经与云合作，在欧洲提供超过 9000 个新的 WiFi 热点，使 WiFi 点的总数达到约 37000 个。非常好。
