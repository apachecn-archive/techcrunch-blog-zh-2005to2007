# 在线音乐许可:YouLicense

[![youlicense.png](img/0523eb159dfa4fb8bd4a01c04e579c97.png) ](https://web.archive.org/web/20220929201403/http://youlicense.com/) [YouLicense](https://web.archive.org/web/20220929201403/http://youlicense.com/) 的目标是在不断增长的直接音乐授权交易市场中获得牵引力。

作为一个在线音乐许可市场，YouLicense 使艺术家和那些寻求音乐内容的人能够直接开展业务，而无需与音乐公司打交道。

内容有索引，易于搜索。YouLicense 提供了标准化的合同，因此买卖双方都能立即了解交易的合法性，从而实现快速便捷的交易。

这个网站的明显竞争对手是各种各样的播客，主要是免费和付费音乐市场。虽然获得 YouLicense 许可的音乐可以用于播客和其他形式的网络创作内容，但该网站旨在实现音乐的全方位使用:电影和电视、广告活动、音乐保留、手机内容和音频项目。提供的项目也不受格式的限制。铃声、乐谱和节拍可以与传统的预录音乐一起列出。

YouLicense 的目标是使音乐许可和版权贸易成为一个更简单、更直接的过程。虽然这项服务仍处于私人测试阶段，但我进入了市场，看起来它肯定会实现自己的目标。![youlicense1.png](img/33aac3375a41763751c99b718704a122.png)
