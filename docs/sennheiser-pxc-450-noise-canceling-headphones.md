# 森海塞尔 PXC 450 降噪耳机

我和森海塞尔有点暧昧。在我玩游戏的时候，我用过一对 HD-570。他们完全改变了我的游戏。据我所知，森海塞尔仍然是游戏玩家使用的主要耳机。

森海塞尔在 CES 上展示了一副新的降噪耳机，这对游戏玩家和通勤者都很有用。在我的产品演示中，我对它过滤环境噪音的效率印象深刻。通过通话功能，音频可以通过。

耳机采用 Duofol 振膜，确保声音无失真。它包括飞行适配器和 1/4”耳机插孔。你可以花 499 美元在三月份挑选一套。
