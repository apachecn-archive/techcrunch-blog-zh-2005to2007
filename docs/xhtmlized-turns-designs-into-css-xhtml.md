# XHTML 化把你的设计变成代码

 [](https://web.archive.org/web/20220810204050/http://www.xhtmlized.com/) 如果你有一个网站设计，但没有时间、资源或兴趣将其转化为 CSS 或 XHTML，请查看三个月前的初创公司[XHTML 化](https://web.archive.org/web/20220810204050/http://xhtmlized.com/)。

他们把你的设计外包给全世界的编码员，五天之内还给你。他们做的不仅仅是快速转换代码:XHTML 化声称他们已经想出了一个“提供搜索引擎爱、可访问性知识并利用微格式和 ALA 技术等智能实践的配方”。项目起价非常合理，150 美元，如果你不喜欢这个产品，你就不用付钱。

XHTML 化是来自 [Futuretrack5](https://web.archive.org/web/20220810204050/http://futuretrack5.com/) 的 Dave Rosen，Stan Dzavoronok，Ilya Sabanin，Dmitry Sabanin 的创作。

根据他们提交的材料:

> 近 3 个月前推出的 XHTMLized 正在不断壮大，给了许多满意的客户。每个项目都根据设计进行报价，但平均价格是 149 美元。大多数项目在几天内完成，但是 XHTML 化可以满足任何期限。分布在 7 个国家的 12 位 XHTML/CSS 专家通过高级使用 Basecamp API 进行协作，共同完成 XHTML 的标准化工作。他们共同形成了一个 XHTML/CSS 食谱，提供搜索引擎之爱、可访问性知识，并利用微格式和 ALA 技术等智能实践。XHTMLized 还可以将设计转换成博客或 CMS 主题，以及更新旧的基于表格的网站。
