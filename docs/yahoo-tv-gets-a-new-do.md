# 雅虎！电视有了新玩法

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/11/30/yahoo-tv-gets-a-new-do/>

[![yahootvlogo.jpg](img/04979016fbf5b62d6e7cc26b42b4e0dd.png)](https://web.archive.org/web/20220819152106/http://tv.yahoo.com/) 雅虎！本周重新设计了他们的[电视列表
网站](https://web.archive.org/web/20220819152106/http://tv.yahoo.com/)。某些[博主](//web.archive.org/web/20220819152106/https://paul.kedrosky.com/archives/2006/11/28/yahoo_tv_goes_2.html)表达了他们对改头换面的不满。我认为它看起来不错，但肯定可以更实用。

最雅虎！页面变得越来越闪亮，所以是时候对电视列表页面动刀了。有人抱怨 Ajax 界面会降低速度，但这不是我的经历。

问题不在于“酷”的新配色方案。问题是设计位置。最相关的信息离高层不够近。在进入“我的电视”之前，我不得不从《实习医生格蕾》、《丑女贝蒂》和《实习医生格蕾》的宣传片向下滚动太多，这也是我首先会去这个网站的原因。他们还把“电视新闻”、“有趣的八卦”和“最新摘要”放在实际上市之前。如果我想要这些废话，我会去 PerezHilton 的博客。

我不认为这是雅虎的另一个例子。涂上它的花生酱。我认为这是雅虎！给自己改头换面，但可能试图努力变得很酷。功能先于时尚，雅虎！向 [Meevee](https://web.archive.org/web/20220819152106/http://www.beta.techcrunch.com/2006/09/21/meevee-overhauls-site-i-want-more/) 学习。

更新:显然雅虎已经在他们自己的博客上看到了关于电视列表页面的反弹。不太愉快。希望他们会记下来。
