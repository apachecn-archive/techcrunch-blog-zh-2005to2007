# 三星即将推出的第三代高清产品

![samsung](img/de9e63f39c273cc8fa5fcd7714ddad4b.png)

三星宣布了三款新的第三代高清播放器([，其中两款我们之前在](https://web.archive.org/web/20210118235246/http://crunchgear.com/2007/07/12/samsung-launching-3rd-gen-blu-ray-players/)报道过，包括一款可以播放蓝光和 HD-DVD 光盘的 BD-UP5000 Duo HD。

这三款产品都将支持高达 1080p 和 24Fs/60Fs 模式，杜比数字加音频，并将包括 HDMI 输入和输出以及一系列其他连接。

我们还可以期待一种新的家庭影院系统，称为 HT-BD2，它将有一个 7.1 声道的接收器和扬声器系统，由 1100 瓦的果汁供电。

BD-UP5000 和 HT-BD2 将于第四季度上市，售价分别为 1049 美元和 1499 美元，BD-P1400 和 BD-P2400 蓝光播放器将于 9 月上市，售价分别为 549 美元和 649 美元。
