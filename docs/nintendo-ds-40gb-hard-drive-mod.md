# 任天堂 DS 40GB 硬盘模块

![](img/a70ecfae8ae2b47d187cbc88a4e27421.png)

所以你买了一个任天堂 DS，现在你希望你有一个 PSP。DS 没有足够的存储容量去看那些非法下载的小型色情电影，或者听大家都在谈论的新辛亚飞唱片。嗯，我们的一个加拿大邻居提出了一个可能会吸引你的解决方案。把这个东西拆开，从 CompactFlash 连接中取出所有 IDE 连接，并把它们焊接到 IDE 硬盘触点上。给它一点 11.1 伏锂电池的额外能量，你就可以在它上面放任何你想要的硬盘了。现在，它不是我见过的最漂亮的车，也不是难以置信的移动性，但它是一个很好的例子，说明一点点努力、一些决心和没有性生活能为你做什么。另外，谁不想同时在两个屏幕上看色情片呢？
