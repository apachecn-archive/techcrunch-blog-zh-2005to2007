# Zetabyte 亚马逊存储设备

![](img/6185e217fc65280b60ba361379fd3df3.png)

Zetabytes？磅，宝贝，磅！Zetabyte 存储设备实际上是一个网络存储系统，备份到亚马逊的 S3 网络文件系统。它可以存储高达 690GB 的，这是最有意思的，每月 299 美元。最便宜的套餐是 30GB 每月 49 美元。真正的关键是存储在 zBox 上的所有数据，正如它被称为的那样，直接进入亚马逊的 S3 T2 服务。基本设备最多支持 5 个同时用户，专业版支持 100 个，每月 499 美元，690GB。
