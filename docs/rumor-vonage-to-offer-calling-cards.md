# 谣言:发誓要提供电话卡

一个非常可靠的消息来源称，Vonage 将开始提供品牌电话卡，该卡将提供大约 200 分钟的免费通话时间，并直接按用户电话号码计费。他们的目标是希望在旅途中使用“家庭”电话号码的商务旅行者，并将包括一个独特的安全密钥生成系统。

Vonage 是最引人注目的 VoIP 运营商，如果这一举措是真的，将为已经强大的服务提供一些惊人的自由。我已经广泛使用了 Vonage，能够从任何电话打电话——不仅仅是从任何网吧使用[V-Phone](www.vonage.com/device.php?type=VPHONE)——听起来真的很不错。

像往常一样，对此持保留态度，但我们的消息来源相当坚定。
