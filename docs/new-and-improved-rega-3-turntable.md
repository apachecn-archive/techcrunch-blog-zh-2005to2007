# 新的和改进的 Rega 3 转盘

![](img/c218a319b7804ccda1eb02f2985a7597.png)

还记得你以前放黑胶唱片时的那种感觉吗？唱针在唱片上的第一次接触，那温暖模糊的声音传入你的耳朵，以及背景中那平滑的嘶嘶声？你并不孤单。

Rega 3 转盘以其出色的音色和演示而闻名。你不仅有一堆新的颜色可供选择，而且 Rega 3 还装有一个更高质量的同步电机和一个浮动平板玻璃拼盘。想想看，你会因为在这些性感的唱盘上播出*而得到所有的关注，而这个世界不会听*的。Rega 3 会花掉你大约 940 美元，所以准备好为你的怀旧买单吧。
