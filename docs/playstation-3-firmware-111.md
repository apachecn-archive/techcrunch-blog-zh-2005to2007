# PlayStation 3 固件 1.11 | TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/28/playstation-3-firmware-111/>

索尼昨晚更新了 PlayStation 3 的固件。新的固件版本 1.11，在账户管理器中增加了一个媒体交叉栏，以及其他各种系统调整。

Kotaku 还报告说，当用户试图调整声音设置时，新固件会导致系统锁定。没有具体的原因，但我想我们很快就会看到新的固件。
