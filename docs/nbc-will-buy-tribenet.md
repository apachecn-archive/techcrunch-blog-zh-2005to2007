# NBC 可能收购 Tribe.net 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/07/07/nbc-will-buy-tribenet/>

(由[拉法特·阿里](https://web.archive.org/web/20230218163736/http://www.paidcontent.org/tribenet-being-bought-by-nbc)和[苏珊·默尼特](https://web.archive.org/web/20230218163736/http://susanmernit.blogspot.com/2006/07/nbc-buying-tribe-deal-is-real.html)散布的)谣言称，NBC 正在收购社交网站[Tribe.net](https://web.archive.org/web/20230218163736/http://tribe.net/)。阿里估计这笔交易的价值不超过 5000 万美元，并表示这家媒体公司将利用 Tribe 来支持今年早些时候以 6 亿美元收购的低技术女性社交网站 iVillage。

**更新: [ClickZ story](https://web.archive.org/web/20230218163736/http://www.clickz.com/news/article.php/3618661) 现在不证实上述消息，报道称交易事实上尚未完成，可能的价格远低于阿里的猜测。更多信息请看下面的评论。**

今年早些时候，Tribe 获得了《华盛顿邮报》和 Knight Ridder 的资助。批评人士声称，其用户基础未能在旧金山核心之外大幅增长。

对我来说，这似乎是一个奇怪的举动，但我想当你是 NBC 的人时，对一项成熟的技术投入一些零钱并专注于集成它，而不是花时间建立自己的技术，这不会有什么坏处。天知道现在一个媒体公司不能没有一两个社交网站！有利于部落在这种技术达到绝对的商品地位之前得到一些钱。上周 Valleywag 对此进行了精彩的独家报道。
