# 卡利的路由器现在必须有安全警告

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/01/routers-in-cali-must-now-have-security-warnings/>

![](img/1e86a853d5eb5b65f1d37cbf18a9d94e.png)

根据加利福尼亚州的一项新法律[，任何“包括集成和启用的无线接入点……【该接入点】制造于 2007 年 10 月 1 日或之后，用于小型办公室、家庭办公室或住宅环境，并在联邦政府未授权的频谱中使用”的设备都必须包含一些关于不保护您的无线网络的危险的警告。警告可以以一张纸、贴纸或软件的形式出现，解释如果您的无线网络易受攻击会发生什么，以及如何保护自己免受攻击。](https://web.archive.org/web/20160325042724/http://info.sen.ca.gov/pub/bill/asm/ab_2401-2450/ab_2415_bill_20060821_amended_sen.html)

这对我来说有点道理。我真的不知道这些公司这么长时间不这么做是如何逃脱的。当然，我的网络今天坏了，所以我很感激我的楼里至少有一个人对此一无所知。

编辑:我对这个有点犹豫。对我来说，家庭网络不安全的危险是显而易见的。我认为这只会成为 RIAA 的素材。随着人们通过简单地声明他们有一个开放的网络来摆脱诉讼，我可以看到 RIAA 跳上这一点，并说，“嗯，没有借口，因为你有一个警告。”你们都认为这会影响那些案子吗？布雷克
