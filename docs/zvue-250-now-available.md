# ZVUE 250 现已上市–TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/19/zvue-250-now-available/>

本月早些时候，我报道了一款名为 ZVUE 250 的低端笔记本电脑将在沃尔玛上架(就像我爸爸说的那样)。今天，它在 T2 的亚马逊和全美的沃尔玛商店都有售

它采用 2.5 英寸显示屏，支持高达 30fps 的视频。该播放器配有一个 SD 插槽和一张 128MB 的卡。它可以播放 MP3，WMV，WMA(带 DRM)，OGG，AVI，ASF，Xvid 和 MPEG4。

ZVUE 250 现已上市，建议零售价为 99.95 美元。
