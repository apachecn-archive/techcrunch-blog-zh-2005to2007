# Zonbu 网络 Linux PC:黑盒子？

一家名为 Zonbu 的公司推出了一款售价 99 美元的平装薄台式电脑，运行 Gentoo Linux，内置 4GB 闪存。你所有的文件都存放在哪里？在 Zonbu 每月 12.95 美元的服务器上，这就是。由建造 [Vudu](https://web.archive.org/web/20210124232213/http://crunchgear.com/2007/04/30/vudu-set-top-box-for-movies/) 的同一批人制造，我 100%肯定这将在上市的头两个月成为一个很好的补贴黑客的玩具。一旦服务失败，期待看到一个非常活跃的 Zonbu mod 集体添加硬盘驱动器，网络服务器和遥控器到这个鸡巴。

显然，它内置了许多方便的软件，包括 Firefox 和 Openoffice，但我不知道为什么你不能在易贝上花 50 美元买一台旧奔腾，然后安装 Ubuntu 来达到同样的效果。

傻逼？每一分钟？
