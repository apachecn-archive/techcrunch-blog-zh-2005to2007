# SmugMug :(反)Web 2.0 公司

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/01/22/smugmug-the-anti-web-20-company/>

 [](https://web.archive.org/web/20230301105645/http://www.smugmug.com/) [SmugMug](https://web.archive.org/web/20230301105645/http://www.smugmug.com/) 首席执行官唐·麦卡斯基尔(Don MacAskill)过去曾说过“[也许我只是不明白这个‘Web 2.0’术语](https://web.archive.org/web/20230301105645/https://techcrunch.com/2006/04/06/the-photo-gunners/#comment-25698)”，现在他正着手教导我们这些声称确切知道它意味着什么的人。该公司今晚推出了一套令人惊叹的新功能。

SmugMug 于 2002 年末迎来了第一批客户，是严肃摄影爱好者的首选服务。10 万付费客户上传了超过 1 亿张照片。该公司有 19 名员工，是盈利的，从未接受过外部资助。麦卡斯基尔告诉我，年收入在 1000 万美元左右。

这项服务没有免费版本。人们每年至少支付 40 美元来上传照片到网站。150 美元/年的专业账户为摄影师提供了许多添加水印的工具，并出售他们作品的下载和打印。更高级别的账户也允许客户使用[模板](https://web.archive.org/web/20230301105645/http://purified.smugmug.com/)，完全定制他们相册的外观和感觉(或者 SmugMug 称之为“图库”)，甚至使用他们自己的[域名。](https://web.archive.org/web/20230301105645/http://photos.photoscapedesign.com/)

该网站拥有典型的 Web 2.0 公司 Flickr 的大部分共享和社区功能。RSS 提要、标签、评论、公共 API 等。Flickr 的母公司雅虎定期与 SmugMug 团队会面，雅虎用户界面团队最近在雅虎博客上对 SmugMug 大加赞赏。对于许多(但不是所有)严肃的摄影师来说，SmugMug 只是展示他们照片的一种更强大的方式。

SmugMug 也已经成为亚马逊 S3 存储服务的典型代表，著名的宣称通过转换每年至少节省 50 万美元。

**新 SmugMug 功能**

SmugMug 今晚发布了许多新功能和架构变化，几乎构成了一个完整的重新发布——SmugMug 95%的页面浏览量受到影响。

关键的变化是照片查看界面从 HTML(附加了一些 Ajax 组件)完全重写为动态 javascript。在图片中点击不再需要刷新页面，大大加快了网站导航的速度，也不会在点击一张新照片后将用户带回页面顶部。

和[雅虎地图](https://web.archive.org/web/20230301105645/https://techcrunch.com/2005/11/02/new-yahoo-maps-shows-power-of-flash/)一样，SmugMug 也经历了当人们浏览网站时更新网址的艰难过程。这是一个困扰富互联网应用的问题。在浏览 Flash 或 Ajax 应用程序时，如果没有页面刷新，URL 就不会更新。SmugMug 已经解决了那个问题，甚至对于 Safari 也是如此(雅虎仍然没有解决这个问题)。

SmugMug 还将用户评论移到了与照片相同的页面(节省了一次点击)，并将大部分元数据和照片选项从显示区移到了一个飞出的抽屉中。这样做的效果是突出照片内容，而不是用大量的数据和链接使页面变得杂乱。
 **自视甚高的未来**

麦卡斯基尔说，风险投资家和潜在收购者“不断”与他接触。但他的计划是在没有外界干预的情况下继续运营这家盈利的公司。他说，他想继续为专业摄影师提供最好的产品，而且没有出售的计划。当然，这只会让那些风投和追求者更加想要 SmugMug。

SmugMug 是“Web 2.0”公司吗？是的，几乎在每一个方面…但你不会听到麦卡斯基尔承认这一点。无论如何，短期内不会。
