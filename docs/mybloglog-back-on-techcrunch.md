# 我的博客回到 TechCrunch

我们在 TechCrunch 上重新嵌入了[MyBlogLog](https://web.archive.org/web/20220808130354/http://www.mybloglog.com/)——几个月前，由于[垃圾邮件](https://web.archive.org/web/20220808130354/http://www.beta.techcrunch.com/2007/01/10/mybloglog-got-spammed-and-so-did-we/)和可靠性问题，我们移除了它。创始人 Eric Marcoullier 和雅虎(已被[收购](https://web.archive.org/web/20220808130354/http://www.beta.techcrunch.com/2007/01/08/yahoo-buys-mybloglog-no-they-didnt-wait-yes/)该公司)反复承诺，这项服务是稳定的，垃圾邮件得到了控制，在此之后，我们决定再试一次。

我喜欢这个小工具，它可以显示网站的最近访客，并根据谁访问了什么博客来构建一个社交网络。我们很高兴它回来了，并期待即将到来的[重新设计](https://web.archive.org/web/20220808130354/http://www.beta.techcrunch.com/2007/05/13/yahoo-to-finally-upgrade-mybloglog/)。

小部件在右边栏，也嵌入在下面。很高兴再次在网站上看到每个人的笑脸，以及偶尔出现的垃圾邮件标志。
