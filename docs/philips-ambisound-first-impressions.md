# 飞利浦 Ambisound 第一印象

![hts81001.jpg](img/a5a75824895e4bd5a0c795596b7ffa8d.png)

我很高兴今天在时代广场检查了一些飞利浦即将推出的产品，我没有说谎，我告诉你，我对一些产品非常兴奋。例如，Ambisound 家庭影院系统让我大吃一惊。音质与其他顶级系统不相上下，其流线型设计为任何媒体中心增添了一丝档次。HTS8100 条形音箱在一个系统中输出完整的 5.1 环绕声，低音炮增加了我们一直在寻找的活力。

我看了几分钟《x 战警 3》,在进行演示的酒店房间里听起来相当不错。我还被告知，下一代 Ambisound 系统将是蓝光，而不是旧的被遗忘的标准 DVD 技术，我们应该期待在' 08 年。这个系统的一个很大的特点是它只有三条线:一条用于果汁，HDMI 输出可以升级到 1080p，还有一条连接到低音炮。SoundBar 现已发货，零售价为 799.99 美元。请继续查看我们的 feed，了解更多飞利浦产品，包括与流光溢彩平板电视和三星液晶电视的面对面比较。
