# 三星和 LG 发布超薄 AMOLEDs

当公司不断告诉 AMOLEDs 他们太胖时，我们在教世界上的孩子们什么？为了让世界摆脱肥胖、肥胖、肥胖的 AMOLEDs，三星和 LG 今天都宣布了超薄显示屏。
