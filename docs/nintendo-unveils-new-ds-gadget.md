# 任天堂推出新的 DS 小工具

![](img/b42b06d8da364a366d46cebda6272c0c.png)

任天堂 DS 的拥有者今天欢欣鼓舞，因为任天堂为其广受欢迎的便携式系统推出了又一个创新的小发明。DS 的新滑动传感器夹在 DS 的底部，基本上允许用户将他或她的 DS 变成光学鼠标。其中一个用途是滑动 DS 来探索游戏中的地图。很酷，但是很有用？也许吧。

这种滑动传感器售价约为 48 美元，目前仅在日本有售。对我们来说幸运的是，一个名为*滑行冒险:Mag Kid* 的游戏被包含在内(哇！)和 12 款使用该传感器的迷你游戏。目前还不知道世界其他地方是否会接受这个新玩意，所以先别急着动手。
