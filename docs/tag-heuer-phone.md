# 豪雅手机(？？)

手表制造商 ModeLabs 将发布一款由法国公司 ModeLabs 设计的手机，该公司为 MTV 和 Virgin Mobile 设计手机。这款手机将由不锈钢制成，并从标签系列的其他手表中汲取设计灵感。

这似乎是对诺基亚 Vertu 系列的直接抨击，这意味着豪雅的母公司 [LVMH](https://web.archive.org/web/20130628181230/http://www.lvmh.com/) 的某些人肯定已经从长时间的小睡中醒来，并注意到时代正在发生变化。

这款手机可能看起来有点像那边的[卡雷拉](https://web.archive.org/web/20130628181230/http://www.tagheuer.com/the-news/new-products/index.lbl?uh=9D5FBBC8-15CF-463C-9BDF-119141DADEF0)，豪雅的旗舰手表。我个人从来没有喜欢过他们的作品——太多廉价的石英表和太少酷酷的怪物——但是我们会看到的，不是吗？
