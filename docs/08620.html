SanDisk固态硬盘Ultra ATA 5000已准备好迎接黄金时代

SanDisk认为，它已经准备好推出首款固态硬盘，进军移动个人电脑市场。SSD Ultra ATA 5000 1.8英寸硬盘的大小为32GB，可以在35秒内启动Windows Vista Enterprise。这种驱动器因其速度而受到吹捧，这要归功于固态存储器，而不是像传统硬盘驱动器那样的移动部件。

新闻稿中的怪胎:

"
由于没有移动部件，SanDisk SSD不需要像传统硬盘驱动器那样旋转或寻找文件，从而使SanDisk SSD的工作速度更快。对于512字节的传输，SanDisk SSD Ultra ATA 5000实现了62兆字节(MB)*/秒的持续读取速率和每秒7300次输入/输出(IOPS)的随机读取速率，比任何硬盘驱动器都要好100多倍。
这些性能数据提升了系统性能。例如，闪迪固态硬盘UATA 5000可以在35秒内启动笔记本电脑上的Microsoft Windows Vista Enterprise。SanDisk SSD实现了0.12毫秒的平均文件访问速率。**在Windows Experience Index for Microsoft Vista中，SanDisk SSD在总分5.9分中得了5.4分，而在同一台笔记本电脑中进行的相同测试中，硬盘驱动器得了3.7分。***
”

哦，如果你把这些固态硬盘中的一个放进去，你的笔记本电脑将获得更长的电池寿命。该驱动器在活动时仅使用0.4W，而标准硬盘上使用1.0W。消耗更少的能量意味着更多的能量。每个人都赢了。目前还不知道这要花多少钱，也不知道什么时候会有更大的尺寸。
