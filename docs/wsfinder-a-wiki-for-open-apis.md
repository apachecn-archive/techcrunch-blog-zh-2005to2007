# ws finder——开放 API 的 Wiki

WSFinder 是一个 wiki，提供关于现有开放 API 和 web 服务的信息。它今天在发布了[，急需一个标志。:-)](https://web.archive.org/web/20201205124802/http://wsfinder.typepad.com/web_service_finder/2005/09/we_are_up.html)

克里斯·劳是[部落](https://web.archive.org/web/20201205124802/http:/www.tribe.net)的早期员工之一，也是 ws finder[的创始人，他写道](https://web.archive.org/web/20201205124802/http://wsfinder.typepad.com/web_service_finder/2005/09/we_are_up.html):

> **我们上**
> 
> 我在网上搜索了一段时间，试图编辑一个 API 的初始列表，人们可以用它来创建应用程序，现在我们已经有超过 80 个了。我觉得这很酷。
> 
> 我们希望你会发现这个资源是有用的，并且它会启发你使用这些 Web 服务创建一些很酷的应用程序。一旦您创建了它们，请让我们知道，我们会将它们添加到示例页面中。或者干脆自己加到维基上！
> 
> 我们也希望这一努力将鼓励更多的公司和开发者开放他们的 API，让其他人也能使用它们。我们已经看到了许多使用开放 API 创造的伟大新事物的例子，这在以前是没有人预料到的。一批很酷的新服务正在涌现，我们希望帮助加速这一进程。
> 
> 这个博客将被用来谈论 WSFinder Wiki 的进展以及我们对 Web 2.0 和 Web 服务的有趣想法。
> 
> 给我们写信，让我们知道你的想法！我在 chrislaw(at)gmail(dot)com
> 
> 你可以回到 http://www.wsfinder.com 的 WSFinder Wiki
> 
> 克莉丝

我喜欢维基和人们用它们做的事情。例子无处不在，新的利基维基网站不断涌现。

对于我们这些正在建立新公司，并且想要一个 API 资源来辅助 mashups 的人来说，WSFinder 将会特别有趣。
