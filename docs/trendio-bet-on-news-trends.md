# trend io–押注新闻趋势

 [](https://web.archive.org/web/20221002115932/http://www.trendio.com/) [Trendio](https://web.archive.org/web/20221002115932/http://www.trendio.com/) 是一个新闻趋势的股票市场——注册并获得 10，000 美元的假币进行文字交易。Trendio 从各种新闻来源监控它们的使用情况，并根据该词在这些来源中出现的频率来提高或降低价格。还有一个法语版的网站，[这里](https://web.archive.org/web/20221002115932/http://www.trendio.com/fr/)。

这让我想起了 Alexadex，一个允许人们在任何被 Alexa 追踪的域名上进行交易的网站。几个月前，我用 Alexadex 玩得很开心，它仍然受到一些博客作者的关注。
