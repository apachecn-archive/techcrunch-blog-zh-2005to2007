# Sprint 将于 11 月发布 MOTOKRZR、MOTORAZR 和 MOTOSLVR

对于 MOTO 粉丝来说，这是个好消息，Sprint 宣布 KRZR、RAZR 和 SLVR 手机将于 11 月上市。根据发布的消息，这三款手机都将兼容 EV-DO，并可以访问 NFL Mobile、Sprint TVSM 和 Sprint Movies。

由于离 11 月还有一个月的时间，你有足够的时间来决定是要糖果棒形状的 SLVR，还是要旧的宽而薄的 RAZR，还是新的 lady about town，KRZR。
