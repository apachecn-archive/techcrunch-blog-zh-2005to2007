# 威瑞森 G'zOne 得到审查(裁决:烤！)

![](img/218f76c93a92d57680f92c366a957216.png)

如果你没听说过威瑞森奇怪命名的 G'zOne 手机，那么你们这些粗犷的户外运动者就错过了大好时机。G'zOne 被设计成符合军事标准和一般的惩罚，它应该能够承受很多惩罚而不会让用户失望。一个叫安迪的家伙得到了一个，并把它付诸实践。手机被放在烤箱里煮，剧烈按摩，扔进淋浴间，沉入水下，等等。

电话打得怎么样？它成功地通过了测试，每次测试后都运行良好。如果你正在寻找一部你可以放下，洒上啤酒，生气时踢一脚的手机，那么 G'zOne 和你可能会有一个成功的未来。它的售价为 300 美元，带有来自威瑞森的两年合约。
