# Skype 找到了进入沃尔玛坏死心脏的方法

[![skypewalmart.jpg](img/d6e30e6d982ab29fad596f6de7107f03.png)](https://web.archive.org/web/20210414173128/http://old.crunchgear.com/wp-content/uploads/skypewalmart.jpg "skypewalmart.jpg")

美国又是早晨了。今天，比以往任何时候都有更多的男人和女人将使用像 Skype 这样的 IP 语音服务。了解到这一点，Skype 和沃尔玛联手，这家大型零售商将在其 17 家数不清的破坏小镇生活方式的商店中出售 Skype 硬件和 20 美元的预付费卡。

那些对网络电话感兴趣的人会发现 Skype 耳机、网络摄像头之类的东西在他们当地的商店里已经准备好进行实际测试了。对 Skype 来说，这是一个很好的举措，因为除了拥有一个舒适的沃尔玛商店之外，没有什么更好的方式向大众推销你的产品了。考虑到该公司刚刚经历了一个相当令人失望的月份，沃尔玛可能需要尽可能多的新的高科技产品来进货。不可能永远不停地开店来弥补这个底线。
