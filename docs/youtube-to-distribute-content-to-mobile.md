# YouTube 将向移动设备分发内容

![youtube.jpg](img/3f958a1ccaef6259e6fdeed805ebf81f.png)你第三屏？那是你管的未来吗？据美联社报道，这正是 YouTube 的联合创始人陈士骏的想法。明年将视频分享网站引入移动平台可能是该公司的下一个前沿领域，但这对威瑞森和沃达丰等移动运营商已经提供的 YouTube 服务意味着什么还不太清楚。或许陈看到了我们昨天报道的不断增长的年轻人市场的好处。让我们不要忘记广阔的国际市场，在那里拥有电脑并不普遍，但每个人都有一部手机。最终，这些人将能够看到 20 世纪 70 年代自制的视频和情景喜剧！
