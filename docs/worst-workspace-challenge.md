# 最差工作空间挑战

众所周知，极客住在办公桌前。他们的工作空间会变得有些个性化和杂乱。有时非常严重。如果你认为你的工作环境是最好的(意思是最差的)，那么[去 T2 的 CrunchGear 看看他们的工作环境挑战。谁发来一张工作空间最恐怖的照片，谁就能赢得一把新的钢制椅子。](https://web.archive.org/web/20210928154111/http://crunchgear.com/2007/02/19/the-steelcase-worst-workspace-challenge/)
