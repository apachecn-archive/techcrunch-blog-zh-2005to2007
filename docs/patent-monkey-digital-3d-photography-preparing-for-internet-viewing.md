# 专利猴:数码 3D 摄影为互联网观看做准备

![](img/e24526c4e37052ba02b53b3d94186f7f.png)

本周，宾得公司获得了一项关于带距离控制的[3d 成像系统](https://web.archive.org/web/20210304043700/http://www.patentmonkey.com/PM/PatentID/7190389.aspx)的专利，同时两名个人发明家获得了一项在互联网上观看[3d 数字图像](https://web.archive.org/web/20210304043700/http://www.patentmonkey.com/PM/PatentID/7190371.aspx)的专利。立体视觉艺术已经存在了一个世纪，但令人着迷的是随着数字领域的爆炸，这一空间将走向何方。

宾得的专利涵盖了使用两个取景镜头、一个距离传感器和将镜头调整到合适的取景距离并以普通摄影覆盖范围捕捉两幅图像的技术。立体图像依靠图像的轻微重叠来创造深度，但观看它们的方式已经从观看盒中的两张照片发展到更近的 3d 眼镜。随着数字和互联网的扩散，现在可以开发一个新的景观。

发明家 Palm 和 Lynn 已经申请了一项用于互联网的 3d 立体图像浏览器的专利。本质上，图像的 3d 观看不仅需要图像的 X 和 Y，还需要需要需要文件类型的第三个观看平面(Z ),以及能够处理显示的观看器。所有这些都需要摄像机和互联网的新标准。Web3d 正在尝试一套使用 XML 进行实时 3d 通信的开放标准。开放标准有一定的意义，因为立体摄像机技术有着悠久的历史[(立体视觉专利列表)](https://web.archive.org/web/20210304043700/http://www.patentmonkey.com/PM/FolderID/ff6a187ca08e45babee78eb4c247eee3.aspx)。

有趣的技术融合可能最终会在虚拟世界引发新的视觉体验。再加上 PureDepth 的[多层显示器](https://web.archive.org/web/20210304043700/http://crunchgear.com/2007/02/28/patent-monkey-digital-display-magic/)的发展，随着大量采用数字图像来构建虚拟世界，如[谷歌地球](https://web.archive.org/web/20210304043700/http://earth.google.com/)和微软的 Photosynth ，也许是立体摄影(或甚至每次点击更多图像)最终摘下眼镜并进入黄金时间的时候了。
