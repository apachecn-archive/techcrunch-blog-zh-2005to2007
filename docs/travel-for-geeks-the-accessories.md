# 极客旅行:附件

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/20/travel-for-geeks-the-accessories/>

CG 人员因工作原因经常出差，我们自己的迈克·科布林为[准备了他喜欢出差时带的美味装备](https://web.archive.org/web/20150821104701/http://crunchgear.com/2007/03/19/the-ultimate-geek-go-bag/)，但我们不想就此打住。毕竟，旅行给了我们这些极客一个借口去购买我们已经拥有的东西的更多移动版本，寻找新的解决旅行问题的方法，当我们使用我们旅行用的除臭剂时，感觉自己像个巨人。(是的，我们中的一些人确实使用除臭剂。)

为此，我们列出了一些我们最喜欢带在身边的产品——用于商务和休闲——以及一些品牌和网站，当你去购买与旅行相关的配件时，你会想去看看，包括科技和其他方面。

网上有一堆买旅行饰品的地方也就不足为奇了。我们最喜欢的是 LuggagePoint.com。该网站组织良好，价格合理，几乎拥有世界上所有的旅行配件，包括电子产品。此外，您还可以找到 TSA 行李指南和限制的最新信息、货币转换器以及打包和旅行的一般提示。

另一个值得注意的配件网站是 Franzus 公司。Franzus 出售各种各样的配件——从锁到马桶盖，再到我们见过的一些最好的电源转换器。

说到电源，如果你带着很多需要充电的装备旅行，我们建议你在 APC 停下来，带上 90 美元的内置 USB 充电端口的通用电源适配器。你可以用它从汽车、飞机或任何交流墙上插座上获得的电力来运行你的笔记本电脑，它可以同时为最多两个附加设备充电，从而减少你需要携带的充电器的数量。APC 还生产大量其他手机配件。

当我们之前提到“我们不知道自己拥有的旅行问题解决方案”时，我们想到的是像 [Brookstone](https://web.archive.org/web/20150821104701/http://www.brookstone.com/) 和[Sharper Image](https://web.archive.org/web/20150821104701/http://www.sharperimage.com/us/en/index.jhtml)这样的商店。作为旅行小工具爱好者，很难抗拒像 M1·特拉克里蒂这样的东西。我们有很多手电筒，但没有一个像这个 20 美元的 LED 钥匙扣一样，有一个磁性旋转底座。当然，他们有各种各样其他方便的旅行装备，比如可以在其他地方买到的[行话旅行者 12 种语言对话翻译](https://web.archive.org/web/20150821104701/http://www.sharperimage.com/us/en/catalog/product/sku__LG554)，但是这些商店通常是我们发现很多我们不知道自己需要的东西的地方。

我们最终会在外出时比在家时更多地使用手机，所以我们喜欢随身携带 CellBoost 或类似的手机充电器，以备不时之需。当然，你可以在电池没电后跑进商店买一个，但是当你把租来的车开进沟里，然后发现你的电池没电了，这对你有好处。另一个选择是拿起第二块电池来保持充电，并随身携带。为此，我们建议去 BargainCell.com。他们为各种各样的手机和其他移动设备储备配件，包括电池、充电器和外壳，一切都很容易在网站上找到。

当我们旅行时，我们拍了很多照片。为了避免携带读卡器，我们更喜欢使用 SanDisk 的 Ultra II SD Plus 卡，这种卡对折后露出 USB 适配器，因此您可以将卡直接插入 USB 端口。

除此之外，还有像[金士顿 DataTraveler Reader](https://web.archive.org/web/20150821104701/http://www.kingston.com/flash/dt_reader.asp) 这样的产品。售价 34 美元的 2GB 盘内置 SD/MMC 读卡器。放入一张卡，将驱动器插入笔记本电脑，它就会显示为两个独立的驱动器。您可以将照片传输到硬盘或闪存盘。

此外，如果你通常不携带笔记本电脑旅行，但仍然希望能够将闪存卡中的照片转储到另一个存储设备中，那么你应该随处携带 Belkin 的 USB。这款轻薄小巧的设备允许您在两个 USB 存储设备之间传输文件，而无需使用计算机。例如，如果你在 SanDisk Ultra SD Plus 卡中装满了照片，你只需将其插入 USB Everywhere 的一个端口，并将另一个存储设备连接到另一个开放的 USB 端口，如[Memorex 8GB Mega travel drive](https://web.archive.org/web/20150821104701/http://www.buy.com/prod/memorex-8gb-mega-traveldrive-usb-2-0-micro-drive/q/loc/101/202066914.html)，然后按下一个按钮，这个小单元就会将图像从你的卡中取出并放入驱动器中。

度假时，随身携带一个袖珍三脚架也是一件不错的东西。有很多这样的产品，但 Vanguard 生产了各种各样的迷你吊舱，包括 0.12 磅的 Flexi，带有可弯曲的腿，适用于不太平坦的表面。

，让众多手机配件值得一试。

我们没有花时间在任何类型的便携式媒体设备上，因为，嗯，那不是真正的配件。我们还跳过了耳机，因为我们希望你们所有人现在都知道，在汽车后座或飞机上放松的最佳方式莫过于一副良好的隔音耳机或耳挂式降噪耳机。然而，我们确实想添加一个你可能想检查的东西: [Pzizz](https://web.archive.org/web/20150821104701/http://www.pzizz.com/) 。这是一个软件和伴随的声音场景，可以帮助你入睡或小睡片刻。是的，这是令人难以置信的新时代，我们通常不从事这种事情，但如果你对它开放，它会帮助你放松和幸福。

最后一点，我们的印象是，飞机上 3 盎司液体的限制仍在继续，因为它促进了小样瓶剃须凝胶、牙膏和洗发水的销售。撇开阴谋论不谈，如果你想储备旅行用的药品，去药店网的旅行商店逛逛。如果在网上购买小瓶漱口水不能让你成为一个大极客，我们不知道什么可以。
