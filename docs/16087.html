TechCrunch派对又多了100张门票

原文：https://web.archive.org/web/http://www.techcrunch.com/2007/06/25/100-more-tickets-now-available-to-techcrunch-party/trackback/

为了庆祝最近的 TechCrunch新闻发布会，我们刚刚制作了100多张7月27日在八月之都举行的TechCrunch派对的门票。价格为10美元，所有收入都将捐赠给旧金山东南部的公立小学 Kipp Bayview Academy ，用于购买计算机相关设备。与会者的身份将在门口接受检查。

我们会在聚会前一周再准备100张票。我们还将为TechCrunch20的与会者预留一些位置。如果你正在参加 TechCrunch20大会，你将能够参加八月的资本聚会。

详情
日期:2007年7月27日
时间:下午5:30 - 10点
地点:八月首都(地图:谷歌 / 微软 / 雅虎 )
