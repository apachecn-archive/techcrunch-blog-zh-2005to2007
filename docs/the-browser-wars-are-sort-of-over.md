# 浏览器大战(某种程度上)已经结束了

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/18/the-browser-wars-are-sort-of-over/>

除了[游戏控制台](https://web.archive.org/web/20150901145212/http://crunchgear.com/?s=console+wars)和 [DVD 格式](https://web.archive.org/web/20150901145212/http://crunchgear.com/2007/04/18/hd-dvd-sprints-ahead/)的战争，我们似乎已经忘记了浏览器战争，但它似乎已经结束，或者至少是一个友好的停火。在本周的 Web 2.0 博览会上，微软、Mozilla、Opera 和谷歌占据了中心舞台，讨论到底发生了什么。讨论的主题是安全性，我认为这是一个非常好的开始。我可以不在乎市场份额，但我会说我厌恶 IE。

Mozilla 的首席技术官布伦丹·艾希(Brendan Eich)说，安全是困难的，并将永远如此。

> 我认为我们不应该轻视安全；他说，这是一个端到端的问题，我们必须跳出目前的模式，才能在这方面取得胜利。谷歌软件工程师克里斯·韦瑟雷尔(Chris Wetherell)表示，让他夜不能寐的场景之一是离线访问浏览器，以及从安全角度看这意味着什么，特别是在用户对用户方面。
