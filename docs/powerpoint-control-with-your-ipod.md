# 用 iPod  控制 PowerPoint

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2006/08/29/powerpoint-control-with-your-ipod/>

![](img/7ed8d5c8a406be386ead0f049207f048.png)

有很多方法可以通过 PowerPoint 展示你最近的销售损失——一个名副其实的家庭手工业已经围绕着一些小设备建立起来，这些设备可以提高或降低你的负面销售数字。他们使用射频，蓝牙，无线网络，巫术和电缆与你的笔记本电脑接口。嗯，iPod 早就应该加入这个列表了。

使用 [iPresent It](https://web.archive.org/web/20151231155503/http://www.zapptek.com/ipresent-it/) ，一个花费不到 20 美元的应用程序，你可以把你的演示上传给你的多媒体小朋友。然后，您可以在屏幕上回放它(作为一系列照片)进行练习，或者您可以使用 [A/V 连接套件](https://web.archive.org/web/20151231155503/http://store.apple.com/1-800-MY-APPLE/WebObjects/AppleStore.woa/wo/1.RSLID?mco=543CBB30&nplm=M9765G%2FA)直接输出到投影仪。

虽然你仍然需要一台电脑来进行编辑，但你可以使用适合你夹克口袋的设备来进行你的演示，而且效果很好。这非常适合在客户的办公室或家里进行推销，应该会给那些关注你的人留下深刻印象，这些人很可能就是你正在推销的任何人。
