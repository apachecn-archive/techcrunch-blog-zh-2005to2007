# 胡桃夹子:少年笑话说够了

[![nutcracker.jpg](img/a929537fa071dc032787f34d63ea084b.png)](https://web.archive.org/web/20191130015641/http://old.crunchgear.com/wp-content/uploads/nutcracker.jpg "nutcracker.jpg")

今天的概念让实际的店内产品相形见绌，是由[克里斯托弗·库](https://web.archive.org/web/20191130015641/http://www.christopherkuh.com/)设计的坚果钳。是的，让我们现在就解决大象:它看起来像一个假阳具。大不了。它能高效系统地敲碎坚果。理论上，这只是一种渲染。

您将坚果(渲染中的核桃)放在两个管之间，并有目的地扭曲它们。片刻之后，你就有一个裂开的坚果可以食用了。太棒了。
