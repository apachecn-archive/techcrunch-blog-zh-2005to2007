# XP 将延续到 2010 年

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/01/24/xp-to-live-until-2010/>

XP 已经上路了，但这并不意味着它会不战而退。微软将支持 XP Home 直到 2010 年左右，这意味着在微软停止接听你的电话之前，你可以间谍软件宝贝了整整三年。

> 这将是显着延长“[达伦休斯顿，微软日本部门总裁]在接受共同社采访时说。我说的显著，是超过 1 年。"

更多具体信息将于周四公布。
