# 德国 Geni 克隆体

[![verwandtlogo.png](img/0df924f62387600694ceaeef3ae74c55.png)](https://web.archive.org/web/20220128123849/http://verwandt.de/) 德国正开始打造一个以自己的名字命名的世界创业克隆之都。流行的美国服务的德国复制品不断出现。推特( [Frazr](https://web.archive.org/web/20220128123849/http://www.beta.techcrunch.com/2007/05/10/frazr-internationalize-or-someone-else-will/) ，[嘟嘟嘟](https://web.archive.org/web/20220128123849/http://www.beta.techcrunch.com/2007/06/21/german-twitter-clone-dukudu-for-sale-on-ebay))。和脸书( [Studi.vz](https://web.archive.org/web/20220128123849/http://www.beta.techcrunch.com/2007/01/03/german-facebook-clone-sells-for-e100-million/) )就是最近的两个例子。TechCrunch 撰稿人 Gregor Hochmuth 将这些德国克隆称为复制/粘贴创新。

最新的德语克隆， [Verwandt](https://web.archive.org/web/20220128123849/http://verwandt.de/) ，在英语中是“相关”的意思。它的设计和功能与美国的同类产品 [Geni](https://web.archive.org/web/20220128123849/http://www.crunchbase.com/company/geni) 非常相似。事实上，它远不止是相似——Verwandt 是一个 Geni 克隆体，穿着一些更可爱的图形。它使用与美国网站相同的布局和基于 flash 的快速注册注册表。他们还复制了家谱导航和个人资料页面。看看下面的内容，自己判断相似之处。

动机和业务流程很清楚:1)在美国或其他地方找到一个成熟的概念，2)克隆服务，3)盈利。他们从这些克隆产品中获利颇丰。山姆威尔兄弟投资了 Alando.de(易贝克隆出售给易贝)和 Studi.vz(脸书克隆以 1 亿美元出售)。他们也是 Frazr 的投资者。见 Gregor 的帖子获得更长的克隆列表。当然，包括美国在内的其他国家也参与了他们的克隆事业。然而，来自德国的克隆潮表明了一种不受欢迎的趋势。

Verwandt 似乎也在复制 Geni 的一些成功。在不到两个月的运营时间里，他们拥有超过 150 万份个人资料，而 Geni 只有 500 万份。他们还获得了一笔未披露金额的融资，其中肯定有 Geni 公司 1 亿美元的估值。

美国之外有很多伟大的创新，但随着真正的麦科伊继续创新和国际化，这种抄袭和翻转的心态可能被证明是短视的，像[邢](https://web.archive.org/web/20220128123849/http://www.xing.com/)这样稳固的德国初创公司在争议中被遗忘。
