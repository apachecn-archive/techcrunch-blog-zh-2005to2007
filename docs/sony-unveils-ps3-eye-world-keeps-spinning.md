# 索尼推出 PS3 Eye，世界继续旋转

索尼推出了下一代 Playstation Eye 网络摄像头，它看起来确实非常…非常蹩脚。好吧，这不是真的蹩脚，但它有一个奇怪的设计。我知道麦克风据说比它的前身有了很大的改进，但如果它在摄像头下面不是更好看吗？我认为最重要的部分是专门拾取语音的适应性，以过滤背景噪音，实现更高质量的聊天和语音命令。我还是觉得很丑，不过跑题了。

一种新的 futuristicexpialidocious 传感器使这些低光情况变得可以忍受，并提高了帧速率，以更好地跟踪运动敏感游戏的运动。VGA 分辨率图像以 60 FPS 的速度摆动，而 QVGA 图像的速度高达 120 FPS。它还包括 EyeCreate，允许 PS3 用户将音频和视频记录到硬盘上，你甚至可以应用麻木效果！该软件还可以让你拍摄慢镜头视频和大幅改善你偷懒的延时快照。

今年夏天寻找基于 USB 的眼睛。没有价格方面的消息。
