# 雅虎音乐——从桌面

Windows Vista 和 IE7 的一个关键新功能是侧边栏，这是一个小工具平台。这将允许 Live.com 小工具直接在 Windows 桌面上使用。在一月份参加了微软的搜索冠军活动后，我[写了一点关于这个的](https://web.archive.org/web/20221006200713/http://www.beta.techcrunch.com/2006/01/25/new-livecom-services-shown-at-searchchamps/)。

Yahoo Music 已经悄悄地发布了一个 Ajax 小工具，可以让用户在 Windows 桌面上播放 Yahoo Music，它给了我很多关于我们很快就会看到的东西的想法。

在此之前，要使用雅虎的[无限音乐产品](https://web.archive.org/web/20221006200713/http://music.yahoo.com/unlimited/)(你可以下载并听音乐，每月 5 美元)，你必须下载雅虎音乐引擎，这是他们的 iTunes 版本(一种基于客户端的媒体播放器)。但是根据伊恩·罗杰斯在雅虎音乐博客上写的文章，情况将不再是这样。你可以直接在 Windows 桌面上听音乐，而不需要使用雅虎音乐引擎。

我已经包含了一些屏幕截图，包括播放器的特写和在 Windows 桌面上显示它的大图。伊恩的帖子上有更多的截图，链接如上。你也可以在[这个链接](https://web.archive.org/web/20221006200713/http://metahost.savvislive.com/microsoft/20060321/mix_belfiore_20060321_300.asx)看到一个演示，直接跳到 28 分 10 秒。

因此，真正有趣的事情是，我们将能够通过 Windows 桌面上的小工具使用其他哪些网络和桌面应用程序？我的猜测是，几乎所有的事情。
