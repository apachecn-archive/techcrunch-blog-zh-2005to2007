# Wikia 收购分布式网络爬虫 Grub

[![grub.jpg](img/77edea14cb0dec49afb48aeb05c1e7ac.png) ](https://web.archive.org/web/20221010063549/http://www.grub.org/) [Wikia](https://web.archive.org/web/20221010063549/http://www.crunchbase.com/company/wikia) 已经从 Looksmart 收购了分布式网络爬行引擎 [Grub](https://web.archive.org/web/20221010063549/http://www.grub.org/) 。

Grub 利用用户计算机上空闲的计算周期来索引网络，这与 SETI@home 在搜索外星生命时使用的概念类似。

Grub 最初开始于 2000 年，2003 年被 LookSmart 收购。Grub 在 2005 年关闭，原因是围绕封闭源代码和未能遵守 Robots.txt 阻塞请求的一些小争议。

Wikia 将“尽快”开源 Grub 代码，并使用来自 Grub 的数据来支持[搜索 Wikia 项目](https://web.archive.org/web/20221010063549/http://search.wikia.com/wiki/Search_Wikia)。

收购价格没有披露。
