# 三星发布奥运镀金手机

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/07/19/samsung-unveils-gold-plated-phone-for-olympic-games/>

![](img/dd29d27e1b86481b881e82ce70dde112.png)

随着奥运会明年在中国北京举行，电子产品制造商知道是时候挺身而出，提供一些荒谬而昂贵的东西来纪念这一事件了。三星今天宣布推出 P318 Anycall 手机的特别版本，采用令人惊叹的 18k 金色喷漆。结合黑色的表面，它看起来像一个不错的手机，尤其是旁边的摩托罗拉 D&G RAZR。

不过有一件事是肯定的。中国将会有这款手机，并且只在特定市场销售。没有价格或发布日期，谁知道三星将如何发挥这一点。他们可以要价 500 美元，5000 美元。我们等着瞧会发生什么。
