# Sennheiser MX75 运动型耳机，适合多汗人士

自从我玩游戏的时候(很久以前)，我就一直使用一副森海塞尔耳机，所以我可以证明它们的神奇。作为一个经常锻炼的人，我总是想买一副好的运动耳机。嗯，这些 MX75 可能正是你想要的。

MX75 采用方便的 twist to fit 技术，为运动员提供贴身穿着。它还有各种耳机附件，适合不同的设置。该系统包括一个夹子，用于将绳子扣在你的衬衫上，防止它四处乱飞。

MX75 的明显缺点是颜色难看。我理解人们希望体育用品是明亮的，这样他们就能被看见，汽车不会从你身上碾过或什么的，但是为什么一定要用*那种*的颜色呢？事实上，似乎森海塞尔在所有的运动耳机上都使用了绿色。一些颜色选择怎么样？
