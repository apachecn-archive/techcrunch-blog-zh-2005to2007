Windows Longhorn Server Beta 3可用

想要一些下一代服务器技术，IT呆子？为什么不去这个32位或者这个 64位 ISO下载网站，为自己服务一些Longhorn。

自从我的合法IT时代以来，我就没有关注过微软服务器warez，所以我不太确定是否有人特别感兴趣，但这里有一些改进:

*集成IIS 7.0 Web服务器的“上线”许可证:IIS 7上线许可证允许用户托管Web应用程序和。Longhorn Beta 3上的. NET 3.0 Web服务。
*将Windows PowerShell脚本环境集成到Longhorn Server中。
* Active Directory联合身份验证服务的改进，使用户能够实施新的策略，并更轻松地在可信任的合作伙伴之间建立关系。
*支持八个服务器核心角色(而不是之前宣布的四个)。
*额外的远程管理工具可提供更加集成的管理环境。*默认情况下，Windows防火墙现在是打开的。
*网络访问保护(NAP)与Microsoft Update和Windows Update相集成，允许管理员决定哪些更新是关键的，并相应地设置策略。
