# 音频文件:耳边的 Windows Vista

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/02/the-audiofile-windows-vista-in-your-ear/>

![](img/44612e48da663e3329b863149bd72bec.png)
Windows Vista 亲密版千疮百孔。

![](img/63f83903aabab5e6608b3aa2aac7ea39.png)本周在 Windows Vista 的狂欢中，我想知道它是否会对我的耳朵产生影响——我的意思是，除了每次启动我的 Windows box 都要听到罗伯特·弗里普的声音之外。微软最新操作系统中包含的新音频功能凸显了 MS 进一步进入你的客厅，从我在网上看到的情况来看，人们似乎对此非常兴奋。
 大问题是:去 Vista 还是不去 Vista。我将在未来几周内升级，以确定这些东西是否正常工作，并获得大量关于这些东西有多有用的实际经验。但是一点常识表明，许多增强功能都非常小——大多数用户可能会禁用它们，或者因为冗余而遇到问题。

**声音够大吗？MS 在 Vista 中选择关注的事情之一是音量控制。首先，您现在可以独立控制各种应用程序和设备的音量。这意味着你可以一边看视频，比如说 WMP 11，一边在线听 NPR，而且你可以调节音量，这样你就可以清楚地听到两者。(最初我认为这很愚蠢，但后来我发现自己前几天也是这么做的。)当然，你可以在不运行 Vista 的情况下调节这些应用程序的音量，因为视频几乎总是有音量控制，互联网广播应用程序也有自己的控制。看起来这实际上给许多人调节音量增加了一层额外的困惑。**

就独立设备卷而言，如果您同时运行多个 USB 声卡，事情会变得有点棘手，但这只会影响极少数人。例如，我有时想在我的家庭工作室中同时运行扬声器和耳机，我使用 M-Audio Transit USB 输出到扬声器，并使用 HeadRoom Total BitHead 输出到耳机。但是我更喜欢用我的硬件上的旋钮(没有委婉的意思)来调节音量。

另一个大问题是 Vista 增加了对麦克风阵列的支持。曾经用你笔记本电脑的内置麦克风录下你自己~~放屁~~或者进行在线语音聊天吗？嗯，听起来有点小，因为你的笔记本电脑只有一个内置麦克风。但最新的笔记本电脑将有多个微型麦克风，为您提供更好的音质，满足您基本的音频捕捉需求。换句话说，当你在没有耳机的情况下进行 Skype 通话或在线游戏聊天时，你会听到更少的环境噪音，但你仍然需要一个专用的外部声卡来进行像样的录音或使用[在线音乐协作](https://web.archive.org/web/20210301112132/http://www.wired.com/news/technology/software/0,72612-0.html?tw=wn_index_4)服务，如[ejaming](https://web.archive.org/web/20210301112132/http://www.ejamming.com/)。

**推出**
~~革命将~~呈现在电视上——高清。与之配套的音频(以及 HDDVD/BluRay)也是高清的，这基本上意味着无损压缩和大量的通道，这要感谢杜比和 DTS 之类的技术。Vista 支持[英特尔高清音频](https://web.archive.org/web/20210301112132/http://www.intel.com/design/chipsets/hdaudio.htm)规范，这将产生比你弱小的人脑所能处理的更高分辨率的音频——立体声 32 位，192kHz，多达 8 个通道 32 位，96kHz(如典型的 7.1 系统)。

但是 Vista 也有一些巧妙的技巧，比如扩展你的弱不禁风的 2 声道音频来填充你拥有的扬声器，或者它会创建“幻影”声道来补偿你没有的扬声器。此外，操作系统将根据扬声器大小和低音炮的存在，确定如何在您的系统中最好地分配低音。Vista 也有自己的专门针对耳机的环绕声功能，但许多笔记本电脑都会内置这种杜比版本，所以…嗯。我不是扬声器或耳机虚拟化的忠实粉丝，但游戏玩家和电影观众可能会很喜欢这些。

MS 甚至声称，流媒体音频中的打嗝或延迟等延迟问题在强大的 Vista 中得到了缓解，这在一定程度上要归功于新的设备驱动程序和重新组织的 API，这样你的系统就不会对音频流的去向感到困惑。最奇怪的是包括一个房间校准系统，它可以帮助你均衡声音输出，使你的扬声器像手套一样适合你的房间。很有可能，要进行这种房间校准的人不会相信 Windows 会为他们做这件事——他们会得到合适的设备，并且做得正确。

总的来说，我喜欢 MS 解决了延迟的问题，因为这是导致大多数用户沮丧的原因；当你在家里或互联网上播放音频时，没有什么比突然掉线更烦人的了。但是这些功能的其余部分很可能会干扰您已经拥有的软件和硬件。举个例子，只要试着让两个耳机虚拟化系统同时开启，你就知道“过度处理”是什么意思了。幸运的是，你可以进入并禁用这些功能，但并不是每个人都是一个大的设置微调者。

说真的，我真希望 MS 在 Vista 启动声音中使用了《贾尔斯的快乐疯狂》、《贾尔斯和弗里普》、《T2》、《T3》中音轨末尾的那些搞笑对白。我喜欢打开我的机器，听到一个干巴巴的英国声音说，“罗德尼是个悲伤的年轻人……”
