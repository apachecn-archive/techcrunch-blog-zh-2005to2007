# Pebble MP3 播放器:穷人终于为大众设计了 MP3 播放器

[![pebble1.jpg](img/d5c84014caee0ba679dd2062abb0c9bc.png)](https://web.archive.org/web/20131011014527/http://old.crunchgear.com/wp-content/uploads/pebble1.jpg "pebble1.jpg")

比如说，如果一系列 MP3 播放器可以被设计成类似于散布在海边的一排鹅卵石，那会怎么样我不知道，戴维，但显然有人想知道。

设计师对 Pebble 的其余描述读起来就像一些 19 世纪的政治宣言:“现在的情况是:千篇一律坐在装配线上；相同存在于架子上；相同性掌握在消费者手中。有必要这样吗——颜色是唯一的区别特征？”坦率地说，我同意他的观点。为什么颜色(我们为之疯狂的颜色)必须成为当今产品中唯一的区别。
