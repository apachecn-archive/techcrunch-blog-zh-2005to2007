# 专利猴:带可拆卸键盘的诺基亚双屏平板电脑

![nokiafold.jpg](img/936123b18bcaf4c1bdabbdd4aad17b91.png)

诺基亚获得了一项新发布的专利，涵盖了一种笔记本电脑式设备，它有两个触摸屏和一个可移动键盘。诺基亚在这项专利中展示了其向笔记本电脑和手机融合的发展广度，并创造了这一概念，以智能地实现更小的设备尺寸和改进的功能。

诺基亚的设计允许用户将显示器固定为一个平坦的单触摸屏表面，同时移除键盘以最大化观看表面。随着手机上越来越多地使用多个小 LCD，以及 iPhone 应该会带来触摸屏的即将爆发，将屏幕集成为更大的观看屏幕的能力与更紧凑和更安全的存储设计相结合，是这种设备设计的明显优势。

来自诺基亚的 US 7221330 描述:*“因为触摸模式功能使得平板设备配置是期望的，所以本发明公开了一种平板配置，以及用于将该设备固定在该平板配置中的滑动件。因此，两个屏幕在平面配置中结合在一起。当滑动件滑开以允许设备关闭时，没有屏幕在未受保护的折叠位置面向外。”*

![N770](img/1aa5f52bfb28e4d1603611a95b35e58a.png)诺基亚在涉足笔记本电脑设计方面有一些经验，如 [US 6903927](https://web.archive.org/web/20210122011131/http://www.patentmonkey.com/PM/patentid/6903927.aspx) 所示。随着 [770 互联网平板电脑](https://web.archive.org/web/20210122011131/http://www.nokiausa.com/770/1,7841,main:1,00.html)和 [N 系列](https://web.archive.org/web/20210122011131/http://crunchgear.com/2007/01/24/nokia-n800-quick-look/)扩大了诺基亚的覆盖范围，这样的设计开始将更多的工业设计引入一种设备，这种设备可能能够与 [OQO](https://web.archive.org/web/20210122011131/http://www.oqo.com/) 竞争，成为另一种有竞争力的超移动 PC 选择。
