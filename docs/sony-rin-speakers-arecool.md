# 索尼 Rin 音箱是…酷？

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/05/15/sony-rin-speakers-arecool/>

![sound_column_350×496.jpg](img/6af742583cce406879f1efbb1ca6242e.png)

这些意大利设计的扬声器让人想起奥斯汀·鲍尔斯的瑞典阴茎泵。设计方面很有趣，但意大利设计和日本电路的混合对于喜欢预算时尚的消费者来说肯定不是好兆头。扬声器的底座由伦巴第大理石制成，透明的杆子向整个区域发出声音，就像鹅卵石在水中激起涟漪一样。很酷，对吧？
