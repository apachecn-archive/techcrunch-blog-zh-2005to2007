# UGO 被赫斯特公司收购，今天应该会宣布

今晚早些时候，我们得到消息，赫斯特公司收购了总部位于纽约的 got 公司，并将于明天宣布这笔交易。听起来福布斯也得到一个提示，而且是一个更好的提示:他们说价格应该在 1 亿美元左右。UGO 是一个受欢迎的新媒体网站，成立于 1997 年，据《福布斯》报道，每年产生约 3000 万美元的收入。

它们经常出现在谣言中，作为一家公司四处推销自己，许多公司都以这个价格错过了它们。该公司已经筹集了 8200 万美元的资金。

最近增长不大——根据最近的康姆斯克统计，该网站的全球独一无二用户总数从一年前的 850 万上升到每月 1100 万。
