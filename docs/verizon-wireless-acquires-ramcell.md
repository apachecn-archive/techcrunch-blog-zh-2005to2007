# 威瑞森无线收购 Ramcell 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/10/verizon-wireless-acquires-ramcell/>

威瑞森无线公司昨天宣布，已经收购了农村移动运营商 Ramcell，该公司在肯塔基州和俄勒冈州使用 TDMA 标准运营。用户和网络将由威瑞森转换为 CDMA，而网络将升级到 EV-DO 版本 a。此次收购包括 Ramcell 频谱许可证，覆盖肯塔基州 Clay 和 Madison 市场的约 300，000 人，以及俄勒冈州 Coos 的另外 274，000 人。

财务细节尚未披露。此次收购将大大增加威瑞森无线在这两个地区的覆盖范围。
