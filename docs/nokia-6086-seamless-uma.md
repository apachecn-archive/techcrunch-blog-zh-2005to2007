# 诺基亚 6086，无缝 UMA 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/03/nokia-6086-seamless-uma/>

除了看起来丑陋之外，诺基亚 6086 实际上提供了一个实用工具。除了能够通过四频 GSM 电路进行通信，它还可以无缝跳转到 UMA WLAN 通信。

因此，如果你走来走去，失去了信号，但有 802.11 b/g WiFi 可用，6086 可以锁定它，而你不会错过任何一个字。这对我来说听起来可疑，但我愿意抑制我的怀疑，直到我看到它在行动。

当然，它目前只在欧洲有售，会花掉你 200 欧元。我敢肯定，在未来十年的大部分时间里，我们不会在这里看到这样的事情。所以现在，让我们嘲笑它看起来有多丑，尽管它可能有多有用。
