# 大学生的掌上电脑:你需要 Treos 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/09/palm-to-college-kids-you-need-treos/>

为了与大学生中的不相关现象作斗争(我的话)Palm 向参加微软学生合作伙伴计划的大学生赠送 [Treo 700wx 智能手机](https://web.archive.org/web/20160226045855/http://crunchgear.com/2006/09/03/sprint-launches-treo-700wx-surprise/)。

> 学生将使用 Treo 700wx，这是 Sprint Power Vision Network 上的第一款基于 Windows Mobile 的 Treo 智能手机，来宣传移动性的优势，并提高对 Palm Connected Campus 的认识，Palm Connected Campus 是一项支持移动计算环境的计划，在移动计算环境中，移动计算将学生、教师、IT、管理员和员工彼此联系起来，并将校园网络和个人信息联系起来。

目前只有 114 所大学参与了这项计划，这很好，因为 Palm 只分发了 228 台设备。所以我想你们应该学会分享。

别误会，我在大学的时候完全可以用 Treo。见鬼，我可以用任何 PDA 或手机。然而，在不到 300 台的情况下，Palm 是否试图让那些没有免费单元的学生出去买一台 Treo？因为标价 499 美元，我在大学时肯定买不起(编辑:我现在买不起！布雷克)。如果 Palm 和 Sprint 加快步伐，降低面向学生的设备和服务的价格，那就太好了。但我到底知道什么？
