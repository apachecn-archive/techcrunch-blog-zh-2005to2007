# Trekstor Vibez，8 和 15GB 高清 MP3 播放器

这个球员的一切都让人觉得“奇怪”。让我们从它名字中的“z”开始，这个名字来自德国公司 Trekstor，很奇怪。向德国人民道歉，但嘻哈街头时尚并不完全来自柏林。然后是大小:8gb 和 15GB。我们能看到的 8g，但是 15g 呢？那是一个奇怪的数字。

其余的规格稍微正常一点:MP3/WMA/OGG/WAV 播放，调频调谐器，20 小时电池寿命。还有一个遥控器和一个即将到来的扩展坞。如果你喜欢独特的椭圆形设计，看起来是一个整洁的球员。
