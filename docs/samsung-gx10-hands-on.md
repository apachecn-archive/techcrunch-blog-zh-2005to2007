# 三星 GX10 动手

> 原文：<https://web.archive.org/web/http://techcrunch.com:80/2007/02/26/samsung-gx10-hands-on/>

![](img/8c9bc0b79e7c12e9fbc5b9c277599095.png)

三星 GX10 是一款结实的 DSLR，来自一家以轻薄性感著称的公司。虽然它以前做过消费级数码单反，但这款 1020 万像素的相机比去年售价约为 550 美元的 600 万像素版本 GX1S 要高得多。

作为一个新兴的相机迷，我很高兴看到 GX10 进入了一个已经拥挤的市场。它有机会对抗全世界的 EOS 和 Alphas 吗？请继续阅读，寻找答案。

快速免责声明。我绝对不是一个照片狂热者，这绝对不是一个照片网站。我寻找工业和功能设计的好例子，并根据它们对普通消费者的价值进行评级，而不是像 Snappypants 先生那样每个月换一台新相机。GX10 似乎没有得到广泛的评论，我们很幸运地得到了一个测试，所以我正在考虑为那些可能准备从标准点升级并拍摄的人提供一个更昂贵的相机。

关于 999.99 美元的 GX10，你首先注意到的是它的尺寸和重量。这绝对是一个主动射手的相机。它重约 10 盎司，所有重要的东西都用橡胶密封，不受天气影响。电池盒感觉像潜水艇一样防水，SD 卡舱需要两种不同的动作才能打开——翻转和扭动小锁。再加上一个 2.5 英寸的 TFT LCD，您会发现三星是认真的。

小心狮子。

在我测试的配置中，这款相机具有光学图像稳定和 CCD 清洁功能，并配有 D-XENON 18-55 mm 镜头。我对这个镜头有一个很大的疑问，那就是从镜头的尾端有一个很小的突起。一个标签控制光圈，在快速更换镜头时，它正乞求被折断。我没有机会看其他可用的 D-XENON 镜头，但如果我是一名专业摄影师，这个小细节可能会成为一个亮点。

我几乎想让把它拍下来。

![](img/276c07f26605a492a5a23d71dda82557.png)

除了这个问题，相机拍出了漂亮、明亮的照片，帧与帧之间几乎没有延迟。一张普通的自动图像对焦和抓拍只需 2 秒钟，没什么问题。在我的大多数测试中——无论是在舞台场景还是现场拍摄——当试图在恶劣的光线条件下捕捉图像时，甚至是在完全静止的情况下，都会出现一些问题。这显然可以通过一点点 f-stop futzing 来改善，但一般消费者会在必要时使用闪光灯，并在大多数拍摄中保持自动。

![](img/eb746eb5275d057eeee8a50c9faad926.png)

GX10 在连续拍摄模式下每秒可以拍摄 3 帧，尽管我在标准条件下每秒只能拍摄 1 帧。该相机支持 JPEG 和 RAW 格式，并具有多种用户模式以及全闪光灯、光圈和 ISO 设置。简而言之，这是一个强大的 DSLR。

![](img/ffd6b97a64759202f78b72e9aaac5c85.png)

![](img/51afafe38ad10bf24e952fd019c03e7a.png)
自动快速拍摄，无闪光灯，直射光下防抖。我可以让它保持静止，但我想模拟一个没有舞台的快速镜头。

这款相机刚刚在美国推出，可以在一些高端经销商处预订。作为一个最近转换到 DSLR 折叠，我认为 GX10 是一个强有力的竞争者，但它肯定是半专业到业余的。

![](img/8c2d36f8792f51223e8b5eada8d1e044.png)

与该领域的其他相机相比，GX10 属于[佳能 EOS 40D](https://web.archive.org/web/20130628154725/http://crunchgear.com/2006/09/05/canon-eos-40d-info-leaked/) 系列，高于[数码反叛 XT](https://web.archive.org/web/20130628154725/http://crunchgear.com/2006/12/15/fujifilm-finepix-s6000fd-hands-on/) 或[尼康 D40](https://web.archive.org/web/20130628154725/http://crunchgear.com/2006/11/27/nikon-d40-hands-on/) ，这是我目前最喜欢的两款相机。这款相机构造精良，开箱即用，易于理解和使用，并包括足够的高端功能，足以让普通旋钮旋转者开心几周。图像明亮而清晰。图像中的噪点很少，颜色也相当真实，如果与类似的数码单反相机相比有点褪色的话。

一个主要问题是缺乏兼容的镜头，至少在这个时候。佳能和尼康有数百个可用的镜头，这个型号，现在，有很少。这当然是一个小问题，应该在今年晚些时候得到解决。

![](img/2229ebcb445c8725a5cc613ce90f2a8b.png)

对于价格，GX10 提供了相当多的钱相机。美国的套装有一个镜头——显然海外版本有两个——三星今年确保有更多镜头，包括鱼眼和变焦镜头。与任何 DSLR 一样，我建议购物者和/或买家在商店里试几个，挑选一个在价格和图像质量方面与他们“对话”的。一旦你摆脱了那些花里胡哨的东西，你想要的是一台在不同光线下都能拍出好照片的相机，而且不会浪费太多时间。在这方面，我可以放心地推荐 GX10 作为较好的相机之一。
