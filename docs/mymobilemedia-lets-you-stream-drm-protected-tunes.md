# myMobileMedia 允许您播放受 DRM 保护的音乐| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/03/27/mymobilemedia-lets-you-stream-drm-protected-tunes/>

DRM 吹了，我们都知道，但至少有人在做一些事情，而不是抱怨。(虽然不是我们；比起行动，我们还是更喜欢抱怨。)CMWare 推出了一项服务，可以让你将受 DRM 保护的音乐从个人电脑传输到支持 MP3 的手机上，但 myMobileMedia 还做了一些其他很酷的事情。

这项技术将流式传输你所有的音乐文件，包括 iTunes 购买的文件，而不会违反任何 DRM 规则。传输到手机的歌曲由文件扩展名标识。那么它是如何处理 iTunes 购买的音乐的呢？当播放 iTunes 购买的歌曲时，myMobileMedia 会在您的电脑上远程启动 iTunes，歌曲会被转码为 MP3 或 AAC+,而不会复制到您的手机上。

让这项服务更好的是，你可以将视频和照片流式传输到使用 Java 或 Symbian 操作系统的手机上。但是好时光并没有就此停止，因为 myMobileMedia 会不断地分析你的网络连接并调整数据流，使你的内容流畅。

这项急需的服务将于今年夏天在美国推出免费和收费服务，但目前还没有价格方面的消息。
