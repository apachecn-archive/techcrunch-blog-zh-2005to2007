# 在 Clearspring 上追踪你的小工具的全球统治地位

[![clearspringlogo.png](img/cb1dd4fb4a1cffa99e5b6eff313f7347.png)](https://web.archive.org/web/20230106022331/http://clearspring.com/) 自去年 11 月以来， [Clearspring 的](https://web.archive.org/web/20230106022331/http://clearspring.com/) widget syndication 平台已经为 Time、NBC、Universal 和 Maxim 等客户提供了超过 42 亿次 widget 浏览量的分析。今晚，Clearspring 将向任何开发者开放他们的平台，让你在众多网站和平台上编写、跟踪和分发 web 部件。他们还将提供一个论坛来支持他们的开发者。这一声明让他们与 WidgetBox 的分析和分发平台和 Widgipedia 的知识库展开竞争。

Clearspring 的平台允许开发者编写一次小部件代码，然后动态地将它嵌入到任何网站、Google Gadgets、Netvibes、Pageflakes 和 Live.com 的包装器中。开发人员需要做的就是将 Clearspring 指向他们的小部件的源代码。

Clearspring 的包装器为你的部件跟踪分析，并为你的部件动态设置参数。它还包括一个可定制的“抓取”按钮，让你获得嵌入代码或将其导入各种社交网站。所有的分析数据都可以通过仪表板获得。仪表板按类型(访问/唯一)、源域和访问者的地理位置对数据进行细分。在仪表板中，您还可以分析您的小工具是如何传播的，并确定帮助您的小工具起飞的“病毒中心”。

由于 Clearspring 可以设置小部件的参数，这不仅意味着用户可以编辑小部件的设置，还意味着您可以通过他们的 API 动态创建小部件。动态小部件的一个例子是下面的 NBA 球员卡，它可以根据输入到 Clearspring 的参数为任何 NBA 球员生成一张卡。
