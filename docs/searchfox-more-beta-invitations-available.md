# 更多 SearchFox 测试版邀请可用

 [ SearchFox ](https://web.archive.org/web/20220701220847/http://www.searchfox.com/) ( [个人资料](https://web.archive.org/web/20220701220847/http://www.beta.techcrunch.com/2005/09/12/searchfox-personalized-rss-reader/))刚刚[宣布](https://web.archive.org/web/20220701220847/http://3gsearch.blogspot.com/2005/09/expanding-beta-program.html)他们测试版的额外邀请。

如果你想收到试用邀请，请发送电子邮件至“support at searchfox dot com ”,附上你的首选用户名和参考代码 xb17。

正如你可能已经在我们的 SearchFox 简介中读到的，我们相信他们已经在 RSS 聚合器/阅读器领域创造了一个潜在的颠覆性产品。如果你对注意力想法如何帮助向你推送相关内容感兴趣，这是值得一试的。
