# 具有 Mac 专属功能的 Skype For Mac 2.6

 [Skype for Mac 2.6](https://web.archive.org/web/20221209132628/http://www.skype.com/download/skype/macosx/) 今天[在](https://web.archive.org/web/20221209132628/http://share.skype.com/sites/mac/2007/04/skype_26_for_mac_beta.html)发布，包含了许多以前只有 Windows Skype 用户才能使用的“追赶”功能。其中包括一个聊天类型指示器(知道你的联系人何时在写消息)、 [Skype Prime 集成](https://web.archive.org/web/20221209132628/http://www.beta.techcrunch.com/2007/03/07/skype-rolls-out-skype-prime-charge-for-calls/)和自动更新。

但最酷的功能是 Mac-只有在这一点上，呼叫转移。当您正在进行 Skype 通话时，您可以将其转移给联系人名单上的另一位 Skype 用户。在通话过程中，可通过“更多”按钮查看。

似乎接收用户也必须使用 Mac 版本 2.6，所以这个特性还没那么有用。不过，很高兴看到 Skype 首先为 Mac 用户发布新功能。
