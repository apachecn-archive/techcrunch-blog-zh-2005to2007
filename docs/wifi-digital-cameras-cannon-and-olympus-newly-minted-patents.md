# Wifi 数码相机:佳能和奥林巴斯新发明的专利

![](img/cd3d73374d22e2e44f0fb801df9b1710.png)

佳能和奥林巴斯在一月份获得的两项新发明的数码相机 wifi 图像传输专利仅仅是个开始…

佳能最近获得了一项[发布的](https://web.archive.org/web/20130628144810/http://www.patentmonkey.com/PM/PatentID/7167258.aspx)专利，涵盖了一个外部设备，如个人电脑，它可以触发家庭或办公室网络中一个或多个数码相机的图像无线传输。

同样在一月份，奥林巴斯获得了一项关于将数码图像从数码相机无线传输到个人电脑等存储设备的专利。更聪明的是，它包括在成功 Wifi 下载到一个人选择的设备(如 PC)后自动删除相机图像的概念，并知道如果下载到不是选择的设备，就不要删除。

这对像柯达 EasyShare v610 和 TechCrunch 的 Eye-Fi 这样的相机意味着什么？如果还没有建立直接的合作关系，解决办法可能包括无线传输，而不需要智能自动删除图像的步骤，以及不使用外部设备来启动该过程。

重要的是，这两项发明凸显了谁拥有无线数码相机图像传输各个步骤的开始——这是一个不断增长的热门领域。显然，后来者苹果的 iPhone 和微软的 Zune 手机将需要考虑无线图像传输的合作伙伴关系。短期来看，苹果似乎很乐意通过手动对接来“启动和更新”程序。Zune 更明确地强调无线是一个主要功能，应该采取行动来抢购一些智能概念。
