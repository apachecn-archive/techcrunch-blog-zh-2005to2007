# 微软将推出“弗里蒙特”

在接下来的几周内，微软将在 Live.com 品牌下发布一项新的 web 2.0 服务。最终的名字是 TDB，但目前的项目名称是“弗里蒙特”和网址[fremont.live.com](https://web.archive.org/web/20220926103843/http://fremont.live.com/)将重定向一旦它完全活跃。我有机会看到了该产品的演示——它非常酷，肯定会引起轰动。
