# 威瑞森将提供移动支付

![](img/72e4aa248a75198f9e4887618747a983.png)

慢慢地，移动支付开始悄悄进入美国。虽然外国倾向于更多地使用这项服务，但当我们都意识到外出时支付账单可以很容易时，美国最终会明白。威瑞森现在宣布，它将使用 Obopay 的服务，允许顾客通过手机向彼此和商店发送移动支付。

这项服务将允许顾客走进参与的商店，在收银台前挥动手机付款。随着万事达卡加入预付费交易，预付费和计费选项都将可用。虽然很方便，但不要指望很快就能在 7-11 买到士力架。在威瑞森和 Obopay 的计划生效之前，需要有更多的商店参与这个项目。目前，威瑞森还没有宣布这项服务的价格。
