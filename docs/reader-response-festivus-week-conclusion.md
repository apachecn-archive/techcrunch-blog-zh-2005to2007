# 读者反应:节日周结束了

![](img/3cf0871fc19083a99ed658dba7b9c3a9.png)

结束了！七天的顽强评论；七天来，读者们争相赢取尼康 D40 相机套装。作为我们读者反应节的一部分，我们已经分发了一批 SanDisk Cruzer 闪存驱动器，但我想今天是你们一直在等待的一天。

那么谁得到 D40 呢？

你看，事情发生了有趣的变化。虽然每个人都发表了重要的评论，但有一个人显然是本周的胜利者。

然而，有一个问题:他在本周早些时候获得了一些奖励。我们该怎么办？

![](img/555574842a122cf88c89bf29b649c0fa.png)
经过深思熟虑，我们决定接近他，看看他是否愿意放弃他的闪存盘，这样他就可以赢得 D40。毫不奇怪，他抓住了这个机会。

当然，我说的是乔恩，我很高兴地宣布他是我们 Festivus 周读者反应的大奖得主。不过，这引发了另一个问题。初始 u 盘应该交给谁？

这是一个艰难的决定，但最终我们决定乔恩李，乔恩的二重身应该接受闪迪克鲁泽微。

恭喜所有人。今天晚些时候我会给你发邮件询问送货地址。下周，除了四张 ifrogz 礼品卡之外，我们还将赠送两套 Ultimate Ears super.fi 3 录音室耳机。稍后会有更多信息。

非常非常感谢尼康和闪迪为本周的读者反应奖做出的贡献。最重要的是，节日快乐！
