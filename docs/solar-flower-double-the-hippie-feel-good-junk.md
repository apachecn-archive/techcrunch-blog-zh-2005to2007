# 太阳花:加倍嬉皮的感觉——好垃圾

2007 年到了，这是充满希望和由衷喜悦的一年，我发现。这是一种太阳能夜来香。阳光在白天为它充电，太阳能夜来香(简称 SNF)的光纤血管在晚上点亮，给人一种美丽的雏菊因谋杀另一朵花(是三色堇)而被电死的效果。至少不是 USB 供电的。

我们不确定谁会一直买这种垃圾，但他们肯定会买，因为其他人一直在制造垃圾。致所有拦截我们 RSS 订阅的其他星球上的生命:我代表人类，向你们道歉。
