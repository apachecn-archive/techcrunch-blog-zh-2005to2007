# 传言:基于圣罗莎的新款 iMacs 将于 8 月发布

![santarosa_imac.jpg](img/5f642bd10cabaff2e1444cf8f20dc695.png)

我们在 6 月份告诉过你的那些定于下个月上市的 IMAC看起来已经步入正轨。Think Secret 的消息来源将它们放在 8 月份发布，这与我们自己的消息来源所说的非常吻合，所以除非出现一些不可预见的运输或制造问题，否则请在一个多月的时间里寻找它们。

新的 iMacs 将与当前的系列有类似的总体形状，但将失去标志性的白色塑料拉丝金属外壳。此外，底部备受争议的“下巴”将从杰伊·雷诺的尺寸大幅缩小到更令人愉悦的布鲁斯·坎伯贝尔的比例。您的 iSight 可能是较新的 130 万像素型号。外壳本身将更薄、更方，厚度仅为 2 英寸。

但在里面，你会发现英特尔最新的 Santa Rosa 芯片组，这是新款 MAC 电脑的真正不同之处。目前的 MacBook Pros 系列使用快速平台，所以在这里看到升级是显而易见的。更酷的是，17 英寸的入门级机型将被淘汰，这意味着 iMac 的最小屏幕将是 20 英寸，价格可能略高于目前的 999 美元。我们打赌它会下降到 1199 美元，当你考虑到你得到多少时，这仍然是一个完美的价格。

高端产品最高将达到 24 英寸和 2.4GHz，只是比当前批次的时钟快一点，但具有更强大的芯片组。并寻找免费升级到豹优惠券。
