# 索尼可能在新 AIBO 工作

![](img/ae9190d7321418d6e7995c8693e6c4ed.png)

是啊！我曾经有一台旧的索尼 AIBO，虽然很多人会说这是浪费 800 美元，但我认为这是自齐柏林飞船以来最酷的东西。它可以做一堆疯狂的事情，在陶醉的时候非常享受(它会因为大声哭出来而做倒立)。

现在有消息称，索尼正在开发之前被取消的 robodog 的新版本。索尼的 Masaya Matsuura 说:

> “艾博背后的工程师正在做 PS3。我们正在讨论打造像新艾博那样的东西。”

新设备可能会连接到 PS3 来增强游戏性。目前，除非你在索尼日本公司工作，否则我们将不得不等待更多的信息泄露出来。
