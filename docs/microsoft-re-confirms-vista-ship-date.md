# 微软再次确认 Vista 发货日期

就像一个丈夫对他的妻子说“亲爱的，我当然为我们的周年纪念给你买了一份礼物”，当他跑去沃尔玛买一些好东西时，微软再次表示他们没有计划改变他们先前宣布的 Vista 的发货日期。截至目前，他们仍在试图在 11 月前将 OEM 版本发送给制造商，并在 1 月前上架。

你可以在 9 月的某个时候寻找候选版本 1，它实际上不会是一个真正的 RC，因为微软直到 10 月才会最终确定 Windows Vista。
