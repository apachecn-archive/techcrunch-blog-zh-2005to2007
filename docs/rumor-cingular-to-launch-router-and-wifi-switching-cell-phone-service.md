# 传言:Cingular 将推出路由器和 WiFi 交换手机服务

![](img/58fd6369815b5015d082114d786ac6e0.png)我发现 [wifi +手机](https://web.archive.org/web/20201202041816/http://crunchgear.com/2007/02/14/cell-phones-smart-switching-phones/)的势头挺有意思，早早就盖了两个[专利](https://web.archive.org/web/20201202041816/http://www.patentmonkey.com/PM/PatentID/7174187.aspx)引领潮流。

我在飞机上，我身后的一个人，坐在一个值得被打动的人旁边，谈论着任何人都想聊的话题，他的公司 Cingular 正在推出他的新手机..是这样的…

“是的，这个产品还不可用，但它可以让你在 WiFi 网络和手机网络之间切换。由于手机信号塔需要穿透建筑物，并且不能有效地做到这一点，这个系统将让你在进入建筑物时接入 WiFi 网络，在不进入建筑物时接入手机网络。我们也将销售路由器，虽然这将适用于你的 Linksys，但我们的工程师说我们的路由器更适用于电话。它将在大约 30 天后推出，作为一项附加服务，每月额外支付 20 美元。”

不，没有 iPhone joy 的尖叫声从那排传来，没有好好观察它，齿轮仍然是一个谜。
