# Orb 压力测试在 36000 英尺

[](https://web.archive.org/web/20221208225637/http://www.orb.com/)mobile runch 的 Oliver 有一篇关于 [Orb](https://web.archive.org/web/20221208225637/http://www.orb.com/) 的[精彩评论](https://web.archive.org/web/20221208225637/http://mobilecrunch.com/2006/03/23/why-buy-sling-mobile-when-orb-is-free/)，这是 Sling Media 的 250 美元弹弓的免费替代品。Orb 允许您将直播电视、视频、音频和图片从基本计算机传输到任何联网设备。

奥利弗让 Orb 经受了一次非常高的压力测试——在他最近从韩国旅行回来的路上，他在 36000 英尺的高空用波音公司的 Conexion 从他的家用电脑上播放电影《黑社会革命》。一切都很顺利，他说。

就我个人而言，我只是想在我的手机上有 100 GB 的存储空间，但鉴于目前这是一个不存在的产品，通过 Orb 流媒体可能是一个很好的免费方式。
