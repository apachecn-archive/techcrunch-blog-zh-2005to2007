# 松下将 BD-Rs 的刻录速度提高了 4 倍

![](img/941098e876507bd720de3feea4e48b30.png)

在这场次世代格式战争中，似乎谁拥有最大的光盘，谁就成为了赢家。虽然这种兄弟会式的态度是幼稚的，但它肯定有利于价格低廉的 BD-R 光盘的消费者。但是现在[松下不再追求规模，而是追求速度。松下现在声称他们已经生产了一种 4x 蓝光光盘，能够比默认的 1x 速度快 4 倍。存储容量相同，但燃烧速度更快。](https://web.archive.org/web/20201020054439/https://crunchbase.com/organization/panasonic)

现在听起来可能有很大的不同，但回想一下 1994 年，4x CD-ROM 驱动器是仅次于 8MB 显卡的最令人惊叹的东西。此外，对于您将要刻录的所有数据，越快越好。目前还没有发布日期或价格的消息。
