# 笔记本电脑冷却器的 T-1000

![](img/61c8db91065a53a271904ab9c65cc4f7.png)

[Thermaltake](https://web.archive.org/web/20130628194951/http://www.thermaltakeusa.com/) 推出了一款名为 iXoft 的无风扇笔记本电脑散热垫，可以折叠和卷起，便于运输。

据 Thermaltake 公司称，这种垫子由十水硫酸钠或化学书呆子用的 Na2SO4.10H2O 制成(没错，我不知道我在说什么)，使用 Thermaltake 的热转移技术，这种技术由内部材料组成，在不使用时表现出寒冷和固体的特性，但在散热时会转化为液体。当笔记本电脑接触到 pad 时，热量会散发出去，让您的笔记本电脑保持凉爽。

还没有关于价格或可用性的消息，但预计很快就会有。
