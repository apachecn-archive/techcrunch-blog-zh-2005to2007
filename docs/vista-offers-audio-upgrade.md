# Vista 提供音频升级

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/08/vista-offers-audio-upgrade/>

终于，Vista-world 传来了一些好消息。似乎 Vista 将拥有相当多的高端音频功能，包括音量均衡和扬声器调谐。

例如，Vista 将支持麦克风阵列以实现更好的语音识别，并将为笔记本电脑用户添加低音增强和环绕声模拟。这些功能中的大多数曾经被卸载到硬件上或由第三方驱动程序运行。有趣的是，现在它们将被嵌入到 Vista 中。
