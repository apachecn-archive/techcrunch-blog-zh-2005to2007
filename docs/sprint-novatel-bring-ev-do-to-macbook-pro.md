# Sprint、Novatel 将 EV-DO 带入 MacBook Pro

当谈到 3G 无线时，我们 Mac 用户在很大程度上一直被冷落，但 Sprint 和 Novatel 在这里解决这个问题。EX720 ExpressCard 是 EV-DO 版本 A 卡，通过其 ExpressCard 插槽，您的 MacBook Pro 能够以令人印象深刻的 1.4Mbps 速度随时随地访问互联网。它将配备 OS X 驱动程序。

这意味着没有疯狂的谷歌搜索第三方或原始设备制造商的驱动程序，一个邪恶的 OS X 用户目前必须的。除了司机之外，该卡还包括用于 Mac 的 GPS 软件，并将“很快”从 Sprint 零售商和网上购买，合同价格为 179.99 美元。
