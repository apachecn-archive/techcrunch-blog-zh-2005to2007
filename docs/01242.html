梅吉特是竞争者吗？

原文：https://web.archive.org/web/http://www.techcrunch.com/2006/02/03/is-megite-a-contender/comment-page-4/

Megite 已经在 Web 2.0工作组电子邮件列表中获得了一些关注，有可能是第一家能够将博客对话分组的公司，还有memorandum(个人资料)。该网站在设计上非常粗糙，但在智能分组，似乎很快就能把流行的东西放到顶部。

Memeorandum重新定义了早期采用者获取和消化新闻的方式。虽然这些网站还很年轻，但它们将很快对大公司产生破坏性影响。tail rank(profiles)是该领域的另一项服务，也在做一些有趣的事情，比如分析用户上传的OPML文件以获取更多数据，并包括正在讨论的帖子/项目的截图。

有关Memeorandum的更多信息，请参见对创始人Gabe Rivera的采访。
