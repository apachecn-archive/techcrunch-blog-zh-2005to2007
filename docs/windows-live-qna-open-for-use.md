# Windows Live QnA 开放使用

 [](https://web.archive.org/web/20220930021449/http://qna.live.com/) 经过四个月的[私测](https://web.archive.org/web/20220930021449/http://www.beta.techcrunch.com/2006/05/05/microsoft-qna-enters-crowded-market/)， [Windows Live QnA](https://web.archive.org/web/20220930021449/http://qna.live.com/) (问答)今日向公众开放。网上并不缺少问答服务，这几乎成了门户网站的一项要求。不过，如果用户数量在这种服务中很重要，那么随着用户数量的增加，直播服务可能会特别有吸引力。

这项服务很像广受欢迎的雅虎。答案，也就是[每月增长 35%](https://web.archive.org/web/20220930021449/http://money.cnn.com/magazines/business2/business2_archive/2006/09/01/8384345/index.htm)。主要区别在于，Live QnA 使用标签而不是类别，并通过 MSN Alerts 提供更复杂的警报。通过点击其他人的分类术语，标签将使网站更容易浏览，而不是在黑暗的全文查询中寻找与自己相似的术语。该服务目前只有英语版本，但直播团队表示要有耐心——更多语言版本即将推出。

雅虎！Answers 在您的仪表板页面上提供了一个观察列表，这很方便。两周前的今天，它还发布了一个 API，所以开发者可能会想出一些有趣的数据用途。雅虎！Answers 也已经拥有了非常忠实的用户群。另请注意我们对这一领域的以色列初创公司 Yedda 的报道，该公司于本月早些时候成立。

现场 QnA 还没有列在 Live.com 网站的首页，所以如果你想看，你必须直接进入它的页面，[qna.live.com](https://web.archive.org/web/20220930021449/http://qna.live.com/)。在这个领域有很多创业公司，有些有创新的模式，比如对答案的补偿。在其中一种模式更好地证明自己之前，Live.com 提供直接的 QnA 服务并添加标签是有意义的。
