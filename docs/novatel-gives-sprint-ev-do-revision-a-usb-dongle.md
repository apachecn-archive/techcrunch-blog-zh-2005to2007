# Novatel 给 Sprint EV-DO 改版一个 USB 加密狗

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/25/novatel-gives-sprint-ev-do-revision-a-usb-dongle/>

现在，Sprint 正在推出它的 EV-DO 修订版 A 网络，以宽带速度接入移动互联网，许多人会想为他们的笔记本电脑买一个那些花哨的新版本 A 兼容 PCMCIA 或 Express 卡。

但是如果他们是数百万没有上述卡槽的笔记本电脑用户中的一员呢？别担心，Novatel 已经搞定他们了。它在 Ovation 720 中提供 USB 解决方案。他们可以将这个加密狗添加到他们的笔记本电脑或台式机上，并设置为以 400-800kbps 的速度巡航。还不错的无线，不管怎么切。
