# Vonage Beta 测试新的 Ambit VOIP 网关–TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/08/03/vonage-beta-testing-new-ambit-voip-gateway/>

消费者网络电话服务 Vonage 正在测试一种新的，自称“酷而时尚，但简单和平易近人”的网关设备，由 T2 的 Ambit 公司生产。这款名为 Ambit VDV21 的设备包括一个 LCD 屏幕，可以显示来电显示和语音邮件通知。Vonage 在一封电子邮件中表示，希望这将“激励消费者自豪地展示他们的 Vonage 路由器，而不是把它藏在桌子下面。”它还没有列在他们的[产品页面](https://web.archive.org/web/20211027163028/http://www.vonage.com/products.php)上。

这听起来像是他们在借鉴 [Ooma 的](https://web.archive.org/web/20211027163028/http://www.crunchbase.com/company/ooma)策略。Ooma 为用户提供了一个更吸引人的 VOIP 设备，作为他们服务的一部分；它包括扬声器、语音邮件控制和其他功能。一旦购买了硬件，Ooma 也是免费的。

Vonage gateway 还包括其他功能——参见下面发送给我们的电子邮件。如果你想测试它，他们会在测试期间提供一个月的免费服务，外加在你的 Vonage 账户上的 30 美元的信用。但是，你必须在月底归还设备。报名[这里](https://web.archive.org/web/20211027163028/https://forums.vonage.com/showthread.php?t=94)。
