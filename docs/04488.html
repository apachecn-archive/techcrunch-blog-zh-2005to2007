NewsGator走向移动| TechCrunch

原文：https://web.archive.org/web/http://techcrunch.com/2006/09/19/newsgator-goes-mobile/

NewsGator明天将宣布增加一个新产品，可能会引起轰动。NewsGator Go！是一个RSS聚合器，设计用于Windows Mobile设备。它将使用户能够在旅途中方便地阅读内容和管理RSS订阅。

新版本能够将文章和订阅与现有的NewsGator程序同步，如NewsGator Online、Inbox、NetNewsWire和FeedDemon。使用NewsGator Online，用户可以向特定的设备发送特定的订阅源，例如，可以设置移动设备只接收最重要的订阅源。它还可以格式化提要，使其非常适合移动显示。

NewsGator Go！for Windows Mobile将与任何运行Windows Mobile 2003或Windows Mobile 5的设备兼容。它将于明天在NewGator上市，售价29.95美元。
