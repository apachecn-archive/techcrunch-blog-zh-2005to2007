你们中有多少人在 [Mac](https://web.archive.org/web/20150924111844/http://crunchgear.com/2007/05/15/apple-updates-macbooks/) 上使用 Firefox，但讨厌它看起来像 Windows 3.1 应用程序？Mozilla 的人知道这一点，听到了你的抱怨，现在正在做一些事情。如果一切按计划进行，Firefox 3.0 将会有一个更像 Mac 的界面。与此同时，新 Mac 版本的主要开发者之一在他的博客上发布了一个预览版本…对我们来说太糟糕了，Mozilla 服务器现在似乎已经死了。让我们燃起希望的方法。
