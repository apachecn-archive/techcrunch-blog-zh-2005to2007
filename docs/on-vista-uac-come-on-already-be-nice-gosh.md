谎言！谎言！谎言！

ZDNet 的乔治欧病了——病了！——在那些广告中，约翰·霍吉曼在玩装有 Vista 的电脑时，不得不“取消或允许”一切。他很反感。事实上，Vista 的用户帐户控制是伟大的，任何说任何不同的人都不明白 Vista 是伟大的。他说得有道理。他基本上是说，Vista 现在有权限提升控制，这是 OS X 几年前就有的，Unix 从 Steve 和 Bill 还在襁褓中时就有了。但是，在 Windows 3.11 期间实现 UAC，让事情变得更安全，这难道不是一个好主意吗？

> Mac OS X 实际上需要你做更多的工作，让你输入管理员密码，而 Vista(主要用户作为有限管理员运行)只提示你点击“允许”。因此，如果我们真的想让苹果的广告准确无误，就应该有第二个保安让“Mac”背出一连串的字母，然后他才同意继续。我们看到的是另一个欺骗性广告的例子，Vista UAC 真的没有那么糟糕。

另一个案子，朋友们。还记得第一个案例吗？回到史蒂夫·乔布斯在 UAC 的照片上拉屎，在 MacWorld 上举起它，说“UAC 是便便？”我们多快就忘记了苹果的抹黑战术。微软绝不会屈尊于这样的 FUD。

苹果在取笑 Vista，这不公平。结案了。Vista 很精致。它就像一朵花，只有轻轻一碰才会开放。我们应该尊重这一点。
