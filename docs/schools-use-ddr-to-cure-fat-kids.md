# 学校使用 DDR 治疗肥胖儿童

[![fatkidsddr.jpg](img/f11a0287d3ce7354751c9d011974ed4f.png)](https://web.archive.org/web/20210417045838/http://old.crunchgear.com/wp-content/uploads/fatkidsddr.jpg "fatkidsddr.jpg")

现在的孩子都很胖。比起像篮球或足球这样的常规运动，孩子们更喜欢玩电子游戏。灯泡！通过让孩子们在全国各地的学校里玩*跳舞革命*，体育老师正在帮助年轻人减肥。超过 1500 所学校准备在 2020 年前安装这款游戏，再加上学校食堂不再供应垃圾食品，应该会有很多孩子减掉他们的大肚子。

或许更好的是，教育者和父母都看到 DDR 成为了通向更传统体育活动的桥梁。但是对于那些从小就害怕直接竞争的孩子来说，也就是说，“每个人都是赢家！，“这个游戏是他们学校生活中一个受欢迎的补充。拜拜胖营，你好跳舞毯？
