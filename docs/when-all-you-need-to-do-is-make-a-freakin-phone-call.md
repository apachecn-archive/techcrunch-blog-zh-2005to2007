# 当你需要做的只是打个电话

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/04/when-all-you-need-to-do-is-make-a-freakin-phone-call/>

有些人只是想用手机打一个清晰的电话，偶尔发一条短信，这并不奇怪。考虑到这一点，InfoSync World 收集了 10 款手机，从昂贵而漂亮的诺基亚 8801 到几乎一次性的[三星 Trace](https://web.archive.org/web/20150925231705/http://crunchgear.com/2006/09/20/samsung-trace-quick-look/) ，这些手机在其他部门可能不够用，但在说话方面做出了坚实的选择。包括 LG AX490，这是第一款使用 [Fastap](https://web.archive.org/web/20150925231705/http://crunchgear.com/2006/09/14/fastap-keypad-revolution/) 键盘的美国手机，我们希望，不，要求很快在更多手机上看到。

编辑的选择:通话和短信电话
