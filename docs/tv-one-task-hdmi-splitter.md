# 电视单任务 HDMI 分离器

如果你发现自己处于需要将一个 HDMI 信号分割成两个或更多信号源的情况，TV One-task 希望能让你的任务变得简单一点。可提供 2 路、4 路和 8 路输出，分配器经过 ATC 测试，并被认证为符合 HDMI 和 HDCP 标准。据说分离器可以保持信号强度，并可以实现 480p、720p、1080i 和 1080p 清晰度。

1T-HDMI-612 的建议零售价为 299 美元，1T-HDMI-614 为 349 美元，1T-HDMI-618 为 599 美元。
