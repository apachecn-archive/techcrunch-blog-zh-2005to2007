# 泛泰再次乘坐:PN-3200

在宣布 [mini-mini-C300](https://web.archive.org/web/20130627203248/http://reviews.cnet.com/Pantech_C300/4660-10619_7-6556019.html) 几周后，泛泰带着 PN-3200 访问加拿大。这款翻盖手机相当简单——CDMA、GPS 和蓝牙——但它是与加拿大第二大运营商 TELUS 合作推出的。泛泰一直在为亚洲和欧洲生产令人垂涎的手机，所以看到它们航行到这些海岸应该很棒。这款手机并不特别漂亮，但它的心意才是最重要的。
