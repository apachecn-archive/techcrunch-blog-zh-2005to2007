# 任天堂更新 Wii 安全说明

![](img/6b5f55bd886128843dabe7f285ec7fea.png)

由于任天堂 Wii 用户正在失去对 Wiimotes 的控制，任天堂官方网站决定在其网站上更新 Wii 的安全说明。如果你有一台 Wii，你会发现大部分安全预防措施和你在游戏机上玩游戏时一样，对于那些还没有设法得到 Wii 的人，看看你必须小心的一些事情。我个人最喜欢的是:

> 在 Wii Sports bowling 中，打保龄球时放开球的正确方式是松开 Wii 遥控器上的“B”按钮——不要放开 Wii 遥控器本身。

其他安全预防措施包括牢牢抓住 Wii 遥控器，玩游戏时总是戴着腕带(显然用户的腕带已经断了)，给自己留出足够的空间，以免不小心撞到什么东西或什么人。我迫不及待地想看看当《超级粉碎兄弟之争》发布时，任天堂会在它的网站上放些什么。可能是类似“不要把 Wiimote 当作武器”的话。我怀疑没人会听。
