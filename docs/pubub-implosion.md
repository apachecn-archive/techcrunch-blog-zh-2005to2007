# 公开内爆

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/06/06/pubub-implosion>

 [](https://web.archive.org/web/20220810114725/http://www.pubsub.com/) 博客搜索引擎 [PubSub](https://web.archive.org/web/20220810114725/http://www.pubsub.com/) 在与 [knownow](https://web.archive.org/web/20220810114725/http://www.knownow.com/) 的合并谈判破裂后，今天进行了大规模裁员。看起来关闭迫在眉睫。

在与联合创始人鲍勃·怀曼(Bob Wyman)的权力斗争中失败后，Pubsub [于 3 月驱逐了创始首席执行官萨利姆·伊斯梅尔](https://web.archive.org/web/20220810114725/http://www.beta.techcrunch.com/2006/03/07/pubsub-ceo-steps-down-acquisition-rumors/)。在萨利姆宣布离职之前，康斯坦丁·格斯·斯派思接任首席执行官。

**更新:**在 Bob Wyman [通过下面的评论](https://web.archive.org/web/20220810114725/http://www.beta.techcrunch.com/2006/06/06/pubub-implosion/#comment-65485)做出回应后，我对上面的帖子做了一些更正。
