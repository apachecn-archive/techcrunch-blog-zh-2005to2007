Windows Vista SP1即将推出(er)

显然Windows无法让它的大嘴巴闭上。经过一番挖掘，一些网站已经发现在微软下载中提到了 Windows Vista SP1 ，这是微软为更新和其他大文件下载提供的服务。

这是什么意思？本质上看起来Service Pack 1存在于微软自己的封闭工作空间中，一些技术作者没有意识到它还没有普及。我怀疑这将很快消失，但它确实意味着有某种SP1候选人在野外的某个地方，这意味着它可能在今年夏天准备好。好侦探，关心这东西的人！
