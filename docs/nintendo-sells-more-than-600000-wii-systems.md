# 任天堂销售了超过 60 万台 Wii 系统

![](img/08dca2b2eef06d785fc4a819d0005b9a.png)

早在 PlayStation 3 发布之前，索尼就宣布在发布当天我们将会看到 40 万台游戏机。然而，任天堂并没有给出一个明确的估计，我们将会看到发售多少台，我们所知道的是，到今年年底，我们将会在全球看到 400 万台游戏机。任天堂现在给我们提供了数字，称 Wii 系统在发布的第一周内售出了 60 万台。

任天堂还在第一方游戏和配件上赚了 1.9 亿美元，售出了 45.4 万份*塞尔达传说:暮光公主*。看来今年圣诞节我们很多人都要享受林克的冒险了。如果你还没有机会去看看新的*塞尔达传说，*一定要去看看，这是一个很棒的游戏。
