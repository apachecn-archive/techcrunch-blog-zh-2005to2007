# Wii 可用性再探

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/25/wii-availability-revisited/>

![](img/018d61f099689fcd701c9be59ee1d438.png)

自从 Wii 发布以来，几乎不可能在当地或网上零售商那里买到。现在，在游戏机发布的头一个月或三个月，这是可以理解的。但是现在已经是五月了，在商店里仍然很难找到任天堂 Wii 游戏机。这是为什么呢？任天堂不应该提高产量吗？《魔鬼经济学》有一篇很好的文章，研究了为什么买不到 Wii 很糟糕。

从任天堂可怕的制造失误到零售商获得额外的销售和增加的流量，这篇文章绝对是必读的。读完之后马上去看看。这个。句子。
