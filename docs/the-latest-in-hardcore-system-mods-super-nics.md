# 最新的硬核系统模块:超级网卡

现在它只是一个插线的地方。就像 90 年代的 WinModems 一样，你的网卡只是一个接口，而你的操作系统和 CPU 完成了下载色情内容的所有实际工作。黑仔从你的电脑上取走那些讨厌的网络相关任务并自己处理，提高了系统性能。

然而，主要卖点是专为游戏玩家设计的内置功能。对于玩大量在线游戏的人来说，他们游戏体验的瓶颈(转化为他们的高分)是他们与其他玩家的联系。他们将为他们的 DSL 每月多付 20 美元。但是如果你的系统由于自身的资源而滞后，那么快速连接就是浪费带宽。黑仔有“LLR”(减少延迟和滞后)来改善那些非常重要的 ping 时间。

此外，它运行一个微型版本的 Linux，所以它实际上是一台 PC 中的 PC。理论上，它可以被~~黑客~~修改，做各种各样的事情，目前运行“后台”而不影响你的系统性能。BitTorrent 之类的出现在脑海中。
