# PMP:这都是屏幕

[![venzeroslickr.jpg](img/e00bf3089d4e738712697aca266d400a.png)](https://web.archive.org/web/20210118180250/http://old.crunchgear.com/wp-content/uploads/venzeroslickr.jpg "venzeroslickr.jpg")

从 Venzero，这个你从未听说过的公司，来了 slick r[slick r](https://web.archive.org/web/20210118180250/http://crunchgear.com/2007/04/03/slickrframe-digital-photo-frame-automatically-displays-flickr-photos/)？)，这是一款基于 flash 的便携式媒体播放器，主要吸引力在于其小巧的 2.83 英寸宽屏显示器。它能够显示 160 万种颜色，对于你可以载入的 AVI 视频来说绰绰有余。其余的都是相当基本的:MP3/WMA/WAV 兼容，JPEG 图片浏览，录音机等。没有什么太疯狂的，但它仍然是像 iPod nano 这样的播放器的一个很好的选择。它配有 2GB 的内置存储空间，可以使用 microSD 卡进行扩展。
