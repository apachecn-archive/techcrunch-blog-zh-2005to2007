# Otterbox 发布两款新的黑莓手机套

![](img/2e4ff66a13d2fbcb4c31ceb122b0ccda.png)

黑莓用户知道这些元素会让你和你珍贵的通讯设备遭殃。因此，你会很高兴地知道，Otterbox 已经宣布了两个新的防水，坚固的黑莓手机外壳。Otterbox 1930 和 1931 的设计是为了让你的黑莓手机保持完整。它们可能不是市场上最漂亮的箱子，但它们最终会很好地完成工作。他们甚至还配有皮带夹，这样你可以在去吃午饭的时候踢它。

1930 适合所有 8700 型号，而 1931 是为仍在使用 7200 系列的老用户设计的。这两个箱子的售价都是 129.99 美元，1930 年的已经上市，1931 年的即将上市。
