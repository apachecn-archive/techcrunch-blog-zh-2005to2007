# 任天堂 DS 新屏幕来了？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/18/new-nintendo-ds-screen-coming/>

![](img/89c1ff8851f33d7b999fef6fbb0e07cd.png)
如果[多彩的任天堂 Wiis](https://web.archive.org/web/20141101152500/http://crunchgear.com/2006/12/15/wii-color-could-be-coming-soon-for-200/) 没有引起你的注意，也许对任天堂 DS 的猜测引起了你的注意。一些受欢迎的日本博客和论坛发布了围绕新任天堂 DS 掌机的信息，这款掌机的顶部和底部面板都将采用更大的屏幕。屏幕尺寸为 3.5 到 3.8 英寸(目前 DS Lite 的屏幕尺寸都是 3 英寸)。)

我有信心明年会有新的任天堂 DS(任天堂也每年都用新的 GBA 来打击我们)，我只是希望新版本不会在今年太早发布。今年圣诞节，许多消费者将外出为某人购买任天堂 DS Lite，如果在他们购买 DS Lite 后立即推出新版本，这将令人失望。
