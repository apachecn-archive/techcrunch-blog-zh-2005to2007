# 雅虎收购 WebJay 

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2006/01/09/yahoo-acquires-webjay/>

据凯文·伯顿消息[并在](https://web.archive.org/web/20221205084510/http://www.feedblog.org/2006/01/exclusive_yahoo.html)[雅虎音乐博客](https://web.archive.org/web/20221205084510/http://ymusicblog.com/blog/?p=20)上证实，卢卡斯·冈兹的 [Webjay](https://web.archive.org/web/20221205084510/http://webjay.org/) 已被雅虎收购，金额未披露。

Webjay 是一个有趣的应用程序，允许用户在网络上发布音乐播放列表。它没有筹集任何外部资金，总部设在夏威夷的檀香山。我敢打赌，雅虎公司开发部的人找到了很多理由对这家公司进行现场尽职调查。:-)

恭喜你，卢卡斯。
