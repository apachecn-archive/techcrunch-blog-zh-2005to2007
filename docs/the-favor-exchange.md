# 优惠交换 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2005/12/30/the-favor-exchange/>

今天是今年的最后一个工作日，这显然让我想到了最后一刻的税务注销。当然，最好的税收减免之一是慈善捐赠。然后，我提醒自己，向慈善机构捐款不仅仅是为了减税——它还有帮助他人的附带好处。:-)

这也是我喜欢多伦多的法沃维尔(T2)的原因。这是一家 web 2.0 公司，有帮助他人的附带好处。

基本系统是:需要帮忙或者有需要帮忙的，注册后发帖。我现在在网站上看到的东西包括一个[免费床垫](https://web.archive.org/web/20221208222028/http://www.favorville.com/toronto/free%20stuff/postview.aspx?id=136)的帖子，以及一个[请求](https://web.archive.org/web/20221208222028/http://www.favorville.com/favorville/writing%20and%20%20editing/postview.aspx?id=140)帮助写一篇维基百科文章。帮忙完成后，成员们被要求对另一个人进行评价。如果你帮助了某个人或者被帮助了，这个人会自动加入你的网络。

Favorville 很年轻，车流量很少。通常我会等事情发展得更好一点再发这样的帖子，但是考虑到公司感觉良好的角度，我现在要指出它们。也许我们可以帮他们一个忙，启动网络。
