# 未开发的 windows Mobile 6 

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/02/12/windows-mobile-6-in-the-wild/>

![](img/e04f0cb73b4713b9da98f0cdcc58c6ee.png)

今天下午，在正式发布会之前，我去了微软展台，想看看 Windows Mobile 6。看起来主要的升级是在信息方面，这还不足以让我今天早上兴奋。然而，它现在有两个版本，标准版和专业版，这两个版本都将运行 MS Office Mobile，并包括一个迷你 MSSQL 客户端和一个 Exchange 客户端。
