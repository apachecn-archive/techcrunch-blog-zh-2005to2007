威瑞森将在年底获得Treo 700wx

威瑞森无线公司将在其Treo 700w中增加一个“x”。700wx目前仅由Sprint提供。他们差别非常非常小，对拨号网络的支持是关键因素，但每个人都想在同一个游戏上。

它将保留EV-DO，配有Windows Mobile 5、全球定位系统、130万像素摄像头、240×240屏幕和128MB内置内存，与Sprint版本相同，但没有提到任何软件。期待天线运动的Windows Mobile 5为基础的Treo在圣诞节前一触即发。
