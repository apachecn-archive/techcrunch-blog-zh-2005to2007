# Palm 750 亮相| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/09/12/palm-750-unveiled/>

![](img/c59497efcdb610306fb014e90f233c88.png)

![](img/378d22846fadec169de457c36a91b3e2.png) Palm 刚刚在欧洲发布了沃达丰的新款智能手机 750。一小时后我们会有独家视频和图像。

> 今天，Palm 发布了第一款这样的设备。750v 看起来像其前身的更光滑、无天线版本，将运行在微软的 Windows Mobile 平台上，并将只出售给欧洲的沃达丰无线用户。更重要的将是 Treo Palm 的第二款产品，一款名为“Lowrider”的电子邮件和网络手机，价格约为 200 美元，约为其前代产品价格的一半。
