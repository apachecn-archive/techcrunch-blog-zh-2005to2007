# 威瑞森无线重组移动 Web 2.0

美国最大的无线提供商刚刚使移动 Web 2.0 更易于使用。一个改进的界面使客户更容易访问新闻和电子邮件，让人们随时了解最新信息。历史是为布满灰尘的图书馆书架准备的，而不是今天的摇摆器和搬运工。

内容提供商 ESPN、美洲银行、*《今日美国》*和 MapQuest 仍然可用，但是在你想要的时候找到你想要的东西要容易得多。威瑞森无线还增加了新的内容频道，本地，连接，工具，通勤和金钱。

“移动 Web 2.0 的重新设计是我们不断努力进一步增强和简化客户无线体验的又一例证，”无线消费产品副总裁李·丹尼尔斯表示。“客户选择、简单导航和可用性是帮助移动数据服务在市场中取得成功的关键因素。”

当前的移动 Web 2.0 客户可以在导航到移动 Web 2.0 时查看新门户。用户只需进入“立即获取”菜单下的“新闻和信息”类别，即可注册更新的服务。我们这些没有移动 Web 2.0 的人必须注册。我要呆在家里打扫卫生。
