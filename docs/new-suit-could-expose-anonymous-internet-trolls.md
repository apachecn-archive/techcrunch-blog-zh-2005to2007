# 新诉讼可能揭露匿名网络钓鱼者

两名来自耶鲁大学的女性针对大学讨论板 [AutoAdmit](https://web.archive.org/web/20220629005314/http://www.autoadmit.com/) 的法律行动可能会对匿名钓鱼构成威胁。

[路透社报道](https://web.archive.org/web/20220629005314/http://today.reuters.co.uk/news/articlenews.aspx?type=internetNews&storyid=2007-06-17T010410Z_01_N15292674_RTRIDST_0_OUKIN-UK-USA-INTERNET-LAWSUIT.XML)6 月 8 日在康涅狄格州纽黑文的美国地方法院提起的诉讼包括对 AutoAdmit 的 28 名匿名用户的传票，所有这些人都诽谤了这些女性，并随后被指控花费了实习费用。

尽管这一案件很可能演变成一场关于网络言论自由的争论，但很难对这些巨魔产生同情。言论自由确实需要捍卫，但也必须得到尊重；任何权力都伴随着责任。匿名诽谤他人，尤其是在这种诽谤有直接后果的情况下，这种做法太过分了。令人惊讶的是，有多少人似乎在网络世界和现实世界之间存在脱节，在现实世界中，完全正常的人在网下不会对任何人说三道四，但在网上却成了截然相反的人。每个人越快意识到在线攻击的另一端有真实的人，整个在线社区就越好。
