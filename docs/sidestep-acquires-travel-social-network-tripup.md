# SideStep 收购旅游社交网络 TripUp

[![tripup.jpg](img/5cbacfc61b543cb507c4907dd462913d.png)](https://web.archive.org/web/20221001074004/http://www.tripup.com/) 旅游搜索公司 [SideStep](https://web.archive.org/web/20221001074004/http://www.sidestep.com/) 收购社交网站 [TripUp](https://web.archive.org/web/20221001074004/http://www.tripup.com/) 未披露金额。

TripUp 是一个专注于旅游的社交网络，包括用户生成的旅游博客、照片、视频、地图和旅游评论。

SideStep 之前在 2006 年 10 月收购了用户生成酒店评级网站[TravelPost.com](https://web.archive.org/web/20221001074004/http://travelpost.com/)。

在宣布之前，我与 SideStep 首席执行官罗布·所罗门进行了交谈。Solomon 认为收购 TripUp 是向数据密集型网络迈进的一步；SideStep 希望成为在线旅游相关信息的领导者。

所罗门说，SideStep 有一个社交网络战略，除了利用 TripUp，还包括脸书。

SideStep 已经有两个脸书应用程序。 [Trips 脸书应用程序](https://web.archive.org/web/20221001074004/http://www.facebook.com/apps/application.php?api_key=f5ee4f391fd3e07c066851f8b6ce2c6a)提供旅行规划服务，目前拥有超过 150，000 名用户。[扩展信息](https://web.archive.org/web/20221001074004/http://www.facebook.com/apps/application.php?id=2374336051&b)允许用户将自己选择的字段和类别添加到脸书个人资料中。SideStep 已经在为 TripUp 开发一款脸书应用，预计将于今年晚些时候推出。

鉴于许多脸书应用程序似乎缺乏任何连贯的商业模式，我向 Solomon 提出了脸书应该采取什么措施的问题。SideStep 将脸书视为获取客户的工具。Solomon 指出，尽管他们在脸书的工作仍处于早期阶段，但该公司已经从他们的两个脸书应用程序中获得了超过 1000 个可靠的线索，并预计随着他们改善交叉推广策略，这一数字将迅速上升。
