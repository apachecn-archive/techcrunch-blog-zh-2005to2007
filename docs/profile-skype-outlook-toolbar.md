**发布日期:**2005 年 7 月 5 日(Outlook 工具栏)

**什么事？**

如今，基本的 Skype 服务允许任何有电脑的人注册，并与任何其他接受他们为联系人的 Skype 用户通话/发送即时消息。把它想象成带声音的即时通讯。很多人都在使用 Skype——他们的客户端下载量超过 1 亿次，付费用户超过 100 万。

网络效应给 Skype 带来了巨大的影响。

![](img/54605fd6f34347e2967829ccfd41134c.png)

**Skype 拥有强大的核心功能:**

–通过“软电话”(您电脑上的软件，而不是电话)免费进行 p2p 通话，一次最多可与 4 人通话
–基本文本即时消息功能
–文件上传
–Skype-Out(拨打任何电话号码，每分钟收取少量费用)
–Skype-In(为您的 Skype 帐户分配一个电话号码，允许任何电话通过您的电脑拨打您的电话)
–语音邮件

至少有两个功能会让它变得更有吸引力——应用程序共享(意味着你可以与通话中的人共享 powerpoint 演示、网站或任何东西)(见我们在 [vskype](https://web.archive.org/web/20221224073013/http://techcrunch.com/?p=17) 上的简介，它有这个功能和视频)，以及通话记录(见我们在 [Gizmo](https://web.archive.org/web/20221224073013/http://techcrunch.com/?p=59) 上的简介，它有这个功能和其他功能，如地图)。

但总的来说，Skype 有趣、便宜，音质也不错。
 **Outlook 工具栏有什么新功能？**

新的 Outlook 工具栏将 Skype 与 Outlook 紧密集成。Peter Henning 在 Skype journal[上对该产品做了非常详细的介绍，如果您打算使用该产品，我建议您阅读一下。](https://web.archive.org/web/20221224073013/http://www.skypejournal.com/blog/archives/2005/07/skype_toolbar_f_1.php)

这是一个非常小的下载(767k 的 PC)和安装速度很快。简而言之，它允许您:

–直接在 Outlook 中使用 Skype(Skype 客户端必须同时运行)
–将您的 Outlook 联系人与其 Skype 用户名相关联。如果他们在线，您可以直接从 Outlook 用 Skype 与他们通话。如果您收到他们的电子邮件，您会在他们的名字旁边看到一个 skype 状态图标，您可以点击与他们聊天。总的来说，这是大多数 PC 用户桌面上两个最重要的应用程序之间的一个很好的集成。

![](img/110207a253fdd28108a9d22e1f747f4b.png)

你可以在上图中看到的另一个很棒的功能是，它可以通过 skype-out 一键拨打联系人的电话号码。这个“点击调用的功能是一个杀手级的功能。这些电话号码不需要单独添加到您的 Skype-Out 通话清单中。

总的来说，对于大量使用 Skype 和 Outlook 的用户来说，这是一个非常好的生产力工具。
