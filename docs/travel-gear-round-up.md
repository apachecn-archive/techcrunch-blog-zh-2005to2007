# 旅行装备综述

![geek.jpg](img/8e1eacc3d17f4250f264b76442f5e9f1.png)

如果我没有随身携带的装备，我就不会是一个极客，对吗？我喜欢接触最新的装备，不管是衣服、鞋子还是太阳镜，这正是我和[鱼鹰背包](https://web.archive.org/web/20130628153936/http://crunchgear.com/2007/06/04/osprey-meridian-and-aura-review/)一起测试和回顾的。你想过竹子可以织成衬衫吗？你认为太阳镜镜片能适应不同的光线强度，而你甚至不知道它会很棒吗？想要一双合脚的凉鞋，并且在任何情况下都有最好的抓地力吗？你想要所有这些东西，相信我。

![d4533m-tecta-shirt.jpg](img/6f3134df7ec635f589da9bff38f6f046.png)

我在不同的气候和大洲穿过的白色 Sierra Tecta 衬衫非常酷(真的)。我从来不知道你可以用聚酯纤维编织竹子，但是这种混合纤维感觉很棒，而且可以自然地吸走湿气。Tecta 不仅能让我保持干爽，还能保护我娇嫩的皮肤免受日晒。它甚至有一些抗菌特性，可以让你的装备包远离恐惧。这件由不含杀虫剂的竹子制成的环保衬衫绝对不会出错。Tecta 目前售价 50 美元。

![5012648_v1_m56577569830620168.jpg](img/b34377acd2ee242ac035c105237611be.png)

我是一个铁杆珊瑚礁迷，所以即使穿上一双不同的凉鞋对我来说也很难，但我很高兴我这样做了。我徒步穿越波多黎各的丛林和爬上南韩的一座山时，美音洋流凉鞋是最完美的选择。当你能微调你的鞋子以适合你的脚时，它总是有帮助的。这些两栖凉鞋真的抓住你的脚与胸腔一样的花边结构，让你在任何情况下都感到舒适。

当走在山边极其光滑潮湿的岩石上时，我害怕任何鞋子或凉鞋。电流通过了测试，让我感到轻松，一旦我意识到他们实际上为我提供了多少抓地力。当然，他们有点傻，但他们肯定打败了 Aquasocks。目前售价为 105 美元。

![julbo_race-2.jpg](img/eca9ec910b100c5923b91b68d157f666.png)

Julbo Speed Race 太阳镜采用了斑马镜片，这是一种光致变色奇观，可以调节保护你的眼球免受猫的伤害。2 至卡特彼勒。4 它们是防雾的。下面的图片比我解释的更好。

![6b5f45d388.jpg](img/3e9ae424718dccf773b4234bb9a7715d.png)

从黑暗的空间调整到明亮的空间需要大约 30 秒的时间，这对我来说没问题。与大多数这种性质的装备一样，这完全取决于你的面部结构和偏好，但当透光率从 40.7%上升到 6.6%时，你可以尽你所能。我不会在苏荷区的街道上穿这些，但它们非常适合白天远足和任何户外活动。现在花 120 美元买一双。
