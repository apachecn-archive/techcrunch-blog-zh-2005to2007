# 社交——位置感知应用

昨天发布的社交之光是一款移动位置感知应用，允许你记下你在街上走过的地方，这样当你的朋友在同一个地方走动时，他们的手机就会收到关于你的笔记的通知。他们的“关于”页面上写着:

> Socialight 是一个基于手机和网络的平台，允许用户创建和共享基于位置的消息，称为 StickyShadows。Socialight 的移动和网络工具让您可以在手机和网络上访问基于位置的媒体。
> 
> StickyShadows 是一种虚拟的多媒体便利贴，您可以使用手机或本网站制作。粘性阴影由媒体(如文本和图片)以及关于谁可以看到它以及何时何地可以使用它的信息组成。

我特别感兴趣的是，通过标记和标注真实地点，在真实世界的基础上叠加信息，同时仍然与社交组织保持联系(获取信息的是你的朋友)。它是否会回升是我们将看到的，但很高兴看到有用的信息突然出现在你熟悉的地方。
