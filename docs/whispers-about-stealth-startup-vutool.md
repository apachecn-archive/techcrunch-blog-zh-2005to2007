# 关于秘密创业公司 vu tool TechCrunch 的传闻

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/03/28/whispers-about-stealth-startup-vutool/>

我们得到了一个叫 Vutool 的秘密创业公司的信息。创始人是斯坦福人工智能实验室主任巴斯蒂安·特龙。我们听说他有一个十人左右的团队，其中大多数是斯坦福的博士生。

Vutool 是一个来自地面的[谷歌地球](https://web.archive.org/web/20221207150723/http://www.beta.techcrunch.com/2006/12/09/google-adds-the-geographic-web-to-earth/)。它是以艰难的方式创造的——一大队汽车在美国主要城市行驶，并在地面上拍照，旋转 6 度。该公司称这种体验为“街道级沉浸式影像”图像是从每个可能的角度拍摄的——默认视图就像你坐在驾驶座上一样。

原则上，这听起来有点像[微软的街头服务](https://web.archive.org/web/20221207150723/http://www.beta.techcrunch.com/2006/02/28/killer-new-livecom-service-street-side/)，但是规模更大。在 Street Side 每个街区只有几张照片的地方，Vutool 是一个更真实的虚拟世界。一位消息人士将其描述为“像侠盗猎车手一样，但却是真实的世界。”据报道，该项目开发了大量的知识产权，尤其是围绕图像处理、过滤、存储和渲染。

但这项服务可能永远不会推出。他们正在与一些硅谷风险投资家洽谈，希望在首轮融资中筹集 1800 万美元或更多。但在收到一些条款清单后，Thrun 退出了，称公司已被收购。

有强烈的迹象表明，谷歌是收购者，并最终将 Vutool 与谷歌地球配对。用户可以随意从街景放大到卫星图像。

谷歌元宇宙可能会比我们想象的更早到来。
