# 新的 Mac 广告:笨重的电脑，薄薄荷 Mac

![stuffed.gif](img/e8a64b1ebd69720432c8c9e3ffe1871b.png)
又一天，又一个 Mac 广告。在这里，约翰·霍吉曼，个人电脑的角色，是病态肥胖的[试用软件](https://web.archive.org/web/20230405204014/http://crunchgear.com/2007/04/06/even-mossberg-cant-figure-out-his-pc/)的化身，像戴尔和惠普这样的 OEM 制造商随他们的个人电脑发货。然而，贾斯汀·朗，苹果的角色，仍然很瘦，而且没有垃圾。

苹果公司对这则广告相当厚颜无耻，暗示市场上的每台 Windows 机器都预装了试用软件，这是又一个贬低个人电脑行业的笼统说法。FUD！
