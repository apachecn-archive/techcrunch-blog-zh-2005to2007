# 2007 年的 YouTube 移动？TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/02/youtube-mobile-in-2007/>

虽然用户已经可以使用手机将视频上传到 YouTube，但目前还没有办法通过手机观看 YouTube 上的视频。然而，明年这一切都可能改变。在昨天的 [OgilvyOne Verge 数字峰会](https://web.archive.org/web/20160407215527/http://verge.ogilvy.com/index_master.html)上，YouTube 创始人查德·赫利表示，到 2007 年底，YouTube 可能会“在移动设备上有所作为”

随着最近谷歌收购 YouTube 和[谷歌变得越来越移动](https://web.archive.org/web/20160407215527/http://mobilecrunch.com/2006/11/02/google-updates-mobile-services-new-downloadable-application-speeds-up-gmail/)，我敢说，我们将在 2007 年年底之前开始在移动设备上看到 YouTube。无论如何，随着更多信息的发展，我们会随时通知你。
