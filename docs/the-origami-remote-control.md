# 折纸遥控器

![](img/5eb38b585f5a0403f9fa2b0de47e3fd9.png)

当我还是个孩子的时候，小鸡们总是做那些小纸“算命师”,你挑几个数字，然后被告知你有疱疹。现在，同样的原理正被应用到电视遥控器上，或者至少是这个。你用纸上算命师来控制诸如换频道或调节音量之类的事情。

根据创造者 Hayeon Yoo 的说法，该遥控器旨在让儿童通过纸张交互“只学习基本功能”。我不知道你的孩子怎么样，但我的孩子最好知道如何每周四录制《实习医生格蕾》,否则我就要被皮带抽了。

想知道它是如何工作的？它使用无线传感器板和 Max/MSP 进行控制。折纸电视遥控器可能看起来很酷，但它实际上只是一个在研讨会期间开发的原型。所以现在还不要急着去找目标，牛仔。
