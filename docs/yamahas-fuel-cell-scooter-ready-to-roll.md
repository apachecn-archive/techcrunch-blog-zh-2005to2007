# 雅马哈的燃料电池滑板车即将推出

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/26/yamahas-fuel-cell-scooter-ready-to-roll/>

雅马哈将于本周在横滨举行的国际电池、混合动力和燃料电池电动汽车研讨会暨博览会上首次推出一款新的伪 125cc 燃料电池驱动的踏板车。踏板车有一个从主氢燃料电池充电的二次锂离子电池。

如果雅马哈将这款小自行车投入生产，它将是地球上第一款大规模生产的燃料电池汽车。目前，燃料电池技术对全尺寸汽车的效率还不够高，但通过证明该技术可以在稍小的规模上工作，这种情况可能会改变。

现在我们所需要的是一些氢动力摩托车来满足疯狂的精神，配乐由世界卫生组织提供。
