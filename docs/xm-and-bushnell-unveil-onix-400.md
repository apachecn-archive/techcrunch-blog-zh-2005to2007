# XM 和 Bushnell 推出 Onix 400

布什内尔并不以令人敬畏的消费电子产品而闻名，但 Onix 400 似乎是一款不错的设备。这是一个防水的 GPS 设备，内置 XM 卫星广播和天气跟踪。它配备了一个 3.5 英寸 320×240 液晶显示屏，一个用于数据和地图的嵌入式 64MB microSD 卡，以及一个可充电电池。还没有价格，但它将在今年夏天与 *[订书机](https://web.archive.org/web/20210225025710/http://www.youtube.com/watch?v=oKBnTUg3RdQ)* 一起亮相。
