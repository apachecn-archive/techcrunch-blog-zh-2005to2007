# 任天堂 Wii 预购明天开始

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/10/12/nintendo-wii-preorders-starts-tomorrow/>

Gamestop 今天早些时候发出了电子邮件，让每个注册了任天堂 Wii 预订通知的人知道预订将于明天，13 号星期五开始。就像我们昨天说的，你可能不需要预订任天堂 Wii，因为在发布日会有这么多游戏机。尽管如此，还是会有一些任天堂粉丝会去预订 Wii 的(我知道我是)。

CrunchGear 的加文(也是任天堂粉丝)通知我，他明天不能为自己预订 Wii 了。他正在向芝加哥的人寻求帮助。如果你，或者你认识的人，可以做一个善良温和的人，为他下订单(他会付钱的)——给他发电子邮件到 gavinrobinson@gmail.com。他感谢你。
