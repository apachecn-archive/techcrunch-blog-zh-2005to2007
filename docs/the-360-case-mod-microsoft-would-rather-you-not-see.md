# 微软不想让你看到的 360 案例

![](img/5547da679a69739e0cc3a07478004e5d.png)

本·黑肯登不是一般的办案人。这个人是“头号人物”，在我长大的拉丁小村庄里，他们这样说。某个很有钱的人告诉本，他们想要一台 Xbox 360，可以随身携带，在任何他喜欢的地方使用。所以本把这个东西放进了一个 2.8 英寸、14 磅重的盒子里，盒子后面只有一根电源线。他不满足于仅仅满足移动性要求，他还改进了原始设计，建立了一个比 360 的股票冷却系统更有效的水冷系统。这是一件美好的事情，真的。我告诉你，移动高清游戏是未来的发展方向。
