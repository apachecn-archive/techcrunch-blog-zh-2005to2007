# 维基百科禁止卡塔尔 TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/01/01/wikipedia-bans-qatar/>

拥有近百万人口的卡塔尔被[禁止](https://web.archive.org/web/20221006085253/http://en.wikipedia.org/wiki/User_talk:82.148.97.69)编辑维基百科上的任何条目，“因为大量的垃圾邮件和恶意破坏”

显然，卡塔尔只有一个 ISP， [Qtel](https://web.archive.org/web/20221006085253/http://en.wikipedia.org/wiki/Qtel) ，整个国家共用一个 IP 地址。维基百科已经阻止了该 IP 地址的匿名编辑，但允许拥有实际维基百科账户的 IP 地址用户继续编辑文章。不过，有一个问题。如果您从该 IP 地址进入维基百科，则无法创建帐户。这有点像第 22 条军规，用户将被迫要么使用代理进入网站(其中许多也被禁止)，要么干脆完全停止编辑。

这个问题的一个答案是维基百科终止匿名编辑，并强制任何想投稿的人使用用户账户。

Digg 也根据 IP 地址屏蔽账户。我想知道他们是否也禁止了卡塔尔。
