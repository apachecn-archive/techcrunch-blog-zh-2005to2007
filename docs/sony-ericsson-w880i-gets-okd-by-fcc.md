# 索尼爱立信 W880i 获得 FCC 批准

![](img/325165581d3f7d77b84f2737daf35320.png)

W800 系列随身听手机对索尼-爱立信来说是一个相当大的成功。我仍然有我的殴打 W800i 在其所有的橙色荣耀，我喜欢它。现在，S-E 刚刚获得了 FCC 对其新 W880i 手机的批准——这是其弟弟们的超薄、深色、金属版本。不幸的是，没有太多的信息得到 FCC 的批准。

目前，我们知道这款手机有 1.8 英寸 QVGA 液晶显示屏，记忆棒微型插槽，音乐播放，GSM 和 USB 2.0。希望随着这款手机投入生产，更多的细节会浮出水面。
