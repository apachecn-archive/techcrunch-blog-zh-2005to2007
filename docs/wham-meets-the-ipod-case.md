# 砰！符合 iPod 外壳

![](img/d5b6f8537a1943f81dd4c87aecca1093.png)

承认吧。你知道你听了一吨*威猛！*回到 80 年代。如果你能把那张唱片做成 iPod 外壳会怎么样？还在听吗？很好。Contexture Design 发布了一系列由真正的回收乙烯基唱片制成的 iPod 外壳。每个盒子都是由乙烯基唱片、毛毡、软木塞特制而成，并配有屏幕保护装置。应该注意的是，价格

这些保护套被设计成适合 5G iPods，有汤姆·琼斯、是的、雪儿和许多其他艺术家的风格，他们实际上在 45s 上发布他们的专辑。哦，是的，这是我通常放一个低级笑话的地方，比如“你可以花 45 美元买一个 45 美元的！”因为这些箱子每箱 45 美元。我想知道他们是否有旧的乔治·卡林唱片盒…
