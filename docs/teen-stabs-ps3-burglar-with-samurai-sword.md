# 青少年用武士刀刺伤 PS3 窃贼

![](img/e7d7cdc8699c95f96baf6652f8373a3d.png)

毫无疑问，这是我很久以来听到的最酷的 PS3 故事。两名窃贼闯入佛罗里达州的一户人家，开始洗劫这个地方。他们不知道的是达米安·费尔南德斯和他 15 岁的妹妹迪安当时都在家。Deanne 躲在她的壁橱里，当她听到强盗从她的前门闯入，看着整个磨难在她的房间里展开。达米安当时正在睡觉，但醒来后发现小偷拿走了他的空 PS3 盒子。他知道他们很快就会回来拿游戏机，于是抓起一把武士刀，等待着。当窃贼进入房间时，达米安向他冲了过去，将剑直插那人的胸膛。吓坏了，受了伤，两个劫匪逃跑了，一个徒步逃跑，达米安拿着一把该死的武士刀追赶另一个。

小偷被抓住并逮捕，但目前已被保释——这是法院不应该允许的。现在达米安是一个英雄，任何想再次抢劫他家的人都应该知道，他们有可能被剑刺中。说说终极 PS3 防盗器！
