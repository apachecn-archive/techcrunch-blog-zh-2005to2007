# zillow–API 和移动特性

刚刚筹集了另一笔[巨款](https://web.archive.org/web/20221204190056/http://www.beta.techcrunch.com/2006/07/24/zillow-raises-another-25-million/)的 [![](img/ae260e78256cef8ebcb64adcf9044edc.png)](https://web.archive.org/web/20221204190056/http://www.zillow.com/) [Zillow](https://web.archive.org/web/20221204190056/http://www.zillow.com/) ，最近宣布了几项新功能。其中一个，Zillow API，将会被众多渴望获得网站流量的房地产经纪人所接受。另一个是 Zillow 的移动功能，可以让你通过电子邮件/短信获得 Zestimate，这是 Zillow 对房屋价值的估计。

Zillow API 仍在开发中，尽管他们已经在 Zillow 博客[上给出了一些细节。雅虎已经在使用 API，以及为代理商建立网站的公司的测试网站。](https://web.archive.org/web/20221204190056/http://www.zillowblog.com/zillow_blog/2006/07/zillows_open_ap.html)

[移动功能](https://web.archive.org/web/20221204190056/http://www.zillow.com/labs/ZillowMobile.htm)可让您通过电子邮件/短信获取房屋价值。要使用它，发短信或电子邮件到 z@labs.zillow.com 的家庭地址。我同意[socket site](https://web.archive.org/web/20221204190056/http://www.socketsite.com/archives/2006/07/zestimates_by_email_sms_coming_soon.html)–Zillow Mobile 你再也不用等到参加完晚宴回到家才知道主人的房子值多少钱了。当然，另一个用途是当你参观开放日时，检查附近的房屋价值。
