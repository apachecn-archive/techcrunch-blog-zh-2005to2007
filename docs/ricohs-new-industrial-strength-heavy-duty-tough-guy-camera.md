# 理光的新型工业级重型硬汉相机

当然，这个丑陋的相机看起来像是 2001 年的，但是它的功能一点也不老。

这个 1200 美元的难看的怪物不是为了给你妹妹的婚礼拍照。除非你姐姐是北美野人，因为这是个户外运动相机。这是那种你会在旅行中带上的东西，因为它很坚固而且防震。或者进行激流泛舟旅行，因为它是防水的。这是海明威在乞力马扎罗山的斜坡上带的相机，如果他没死也没被吸的话，因为它是耐温的，并且设计成戴着厚手套使用。

这款 800 万像素、3 倍光学变焦的相机不仅比性别不确定的竞技表演中的范·迪塞尔更强悍，而且它还具有连接功能，能够通过内置 Wi-Fi 或蓝牙上传。您还可以使用上述无线协议之一与您的 GPS 设备配合工作，为简化地理标记添加时间和位置标记。

所以来吧，你这个坏家伙，带着这个理光去北极，你将能够证明你把你的裤子掉在了你说的地方，以防不幸的冻伤没有成功。
