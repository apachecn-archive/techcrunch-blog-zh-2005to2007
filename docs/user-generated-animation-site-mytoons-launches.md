# 用户生成动画网站 MyToons 发布| TechCrunch

> 原文：<https://web.archive.org/web/http://www.techcrunch.com:80/2007/03/14/user-generated-animation-site-mytoons-launches/>

[http://www.mytoons.com/flash/MyToons_RemotePlayer.swf](https://web.archive.org/web/20160305101711/http://www.mytoons.com/flash/MyToons_RemotePlayer.swf)

我们很幸运，现在有两个优秀的用户生成动画视频网站上线了——以色列的 [Aniboom](https://web.archive.org/web/20160305101711/http://www.techcrunch.com/2006/11/23/animated-short-films-on-aniboom/) 和从今天开始的 MyToons，后者刚刚结束私人测试。

MyToons 在设计和整体可用性上肯定比 Aniboom 有优势，尽管我今天早上发现网站导航很慢。在这两个网站上，用户上传他们创作的动画短片，或者获得版权许可使用的短片。视频按标签、用户、最受关注、最高评价等进行组织。

我喜欢这个东西，在选择上面嵌入的视频之前，我花了相当多的时间浏览视频(尽管我不太确定特定的视频是否符合 MyToon 的版权政策)。

MyToons 也试图对家庭友好，并表示他们将[审查掉在美国构成 NC-17 分级的](https://web.archive.org/web/20160305101711/http://www.mytoons.com/documentation/comicSwear/)内容。如果我们谈论色情内容，这将是一个足够明确的界限。但是如果他们开始审查脏话和其他模糊不清的不当内容，网站将会受到影响。
