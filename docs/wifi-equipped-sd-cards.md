# 配备 WiFi 的 SD 卡| TechCrunch

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/12/wifi-equipped-sd-cards/>

![](img/344205330b88429f5b1fecbed04506ee.png)

一家名为 Eye-Fi 的公司正在寻求将嵌入 WiFi 的 SD 卡引入你的数码相机。你可以购买一个 SD 卡，几乎可以放在任何数码相机中，并允许多个设备进行 WiFi 连接，而不是花费大量金钱购买一个支持 WiFi 的数码相机。

凭借高达 2GB 的容量，Eye-Fi 无疑有机会创造出一种市场将会趋之若鹜的新设备。我个人非常希望我的卡西欧 Exilim 相机可以无线上网。预计为 2GB 型号支付 100 美元，这对于 SD 卡来说是相当昂贵的。但是记住，你得到的是 WiFi 的快乐！
