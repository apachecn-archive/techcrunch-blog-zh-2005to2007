# 尼康 D40 实践

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/11/26/nikon-d40-hands-on/>

![](img/6bfd82d22e00d367c1eab617c5efbee0.png)

[数码单反相机拍摄更清晰的照片，但它们的价格、尺寸和丰富的功能往往会吓退普通人，让他们重新选择紧凑型相机。600 万像素的 D40，尼康的训练轮 DSLR，尽最大努力减少恐惧因素，有望缓解人们一生中在昂贵的镜头上扔本杰明的情况。它将于 12 月上市，但我们已经玩了几天了。看看我们的想法和一些漂亮的相机色情照片，它们是在跳伞后脱下衣服的……
 】600 美元的机身/镜头套件价格是合理的，考虑到最近大多数像样的紧凑型相机都在 400 美元左右，而且巧合的是，当你出现在当地的百思买超市并发现 PS3 货架被清空时，你可能会发现有多少现金。附带的镜头是一个 3 倍变焦的 18-55 毫米镜头，即使目标观众可能从未购买过镜头，它也可以与尼康现有的所有 AF-S 和 AF-I 镜头配合使用。](https://web.archive.org/web/20210414032334/http://www.crunchgear.com/holidays-2006)

![](img/5347547961865901b01907055cdf88df.png)

![](img/f0ecb098cc78cf65c8740912ddbcf956.png)

至于大小，这东西很小，至少与大多数单反相机相比是这样。它不会适合任何人的口袋，除非他们碰巧是袋鼠或小丑，但它的尺寸为 5.3 英寸×2.5 英寸×3.7 英寸，比它的哥哥 D50 的 5.2 英寸×4 英寸×3 英寸大幅缩水。液晶显示器是一个相当明亮的 2.5 英寸。

![](img/958aa8ad40a00314e0cfacfc21457838.png)

至于让新手适应 DSLR 的功能，这是 D40 真正试图与众不同的地方。为了向您介绍他们手指上的大量控件，微小的视觉表示显示了每个设置选项何时为顶部的视觉示例。例如，当你翻阅 ISO 设置时，一个黑暗房间的小图片位于高 ISO 选择的旁边，而一个阳光明媚的风景位于低 ISO 选择的旁边。你可能甚至不知道 ISO 是什么意思，但是你能弄清楚什么设置是最好的。为了测试这个功能，我在度假的时候把相机给了我的勒德派母亲。几分钟之内，她就选好了场景，给她的狗狗们拍了些像样的照片(就像妈妈们习惯做的那样)。就像他们说的:你可以带一个人去西尔斯，他会带着一张漂亮的照片出去，或者你可以教一个人摄影，他会为生活创造漂亮的照片。

![](img/afe137f9710fe3ae595cc3a7517c0861.png)

![](img/d662b3bfa833d740b7365dc7f2da63e3.png)
每个 ISO 设置都有相应的图像来描述场景。

底线:D40 是一个很好的第一 DSLR。它可能不会满足专业人士，但如果你一直使用的是 PowerShot 或 CoolPix，这是一个简单的方法来升级到一些更有活力的东西。
