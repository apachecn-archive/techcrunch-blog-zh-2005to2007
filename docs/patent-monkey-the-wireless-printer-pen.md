# 专利猴:无线打印机笔

![](img/3c0f679044d846daa310eed9363264c3.png)

本周，爱立信获得了一项[无线打印笔专利](https://web.archive.org/web/20210301065013/http://www.patentmonkey.com/pm/patentid/7196825.aspx)，它可以将手写转换为电子文本来发送电子邮件和接收内容，并可以根据笔在纸上的位置将其打印到特殊纸张上。

都这么打字了，书写工具创新还有空间吗？经过几秒钟的思考，我们的答案是肯定的。

打印机笔
爱立信的新专利突出了将电子邮件写在纸上，然后转换成电子文本发送出去的能力。此外，本发明还设想具有依赖于笔的位置和要输出的图像的打印能力。这个概念建立在大量先前发明的基础上[(相关技术列表)](https://web.archive.org/web/20210301065013/http://www.patentmonkey.com/PM/relpatentid/7196825.aspx)。

![](img/4ef9e717a35de141e7f7ecdb9eebcbf8.png) 数码录音笔
罗技 io2 是 Mi-Corporation 为其设计数据采集软件的笔和数据采集设备。搜索手写内容的能力很酷。人们写东西，甚至是用黑莓手机的人，用笔和把笔记数字化是一个很好的补充。丢了一支 150 美元的笔，不太好。

数字控制墨水笔
Silverbrook Research 已经开发了多项墨水技术，包括最近发布的一项允许电子控制钢笔书写能力的技术，例如在不使用时打开或关闭墨水，或者对路径进行不同的设计。Silverbrook 在多墨水、打印机风格的笔领域做了相当多的工作，值得一读，因为在像打印机一样的笔中融合墨水的想法是最终的 [4 墨水点击笔](https://web.archive.org/web/20210301065013/http://www.amazon.com/BIC-4-Color-Ball-Medium-Point/dp/B000F2PFPS)的基础。[(silver brook 钢笔专利列表)](https://web.archive.org/web/20210301065013/http://www.patentmonkey.com/PM/FolderID/93df45f856634aa98a74e0f4b29611db.aspx)

数字白板

![](img/e644b6eccc7015137a7dec9f091413e9.png)
