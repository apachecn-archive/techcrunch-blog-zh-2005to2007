# 助手用户将获得 MySpace 应用程序

![](img/da76080f82c9b738e11faa5da5b30b04.png)
谈两年为时已晚，但我想迟到总比不到好。T-Mobile Sidekick 3 和 Sidekick ID 用户将最终可以通过一个专用的应用程序访问他们的垃圾 MySpace 帐户。用户将能够发送消息，添加朋友，查看个人资料(不知道是否所有自定义 CSS 都会显示)，等等。自定义提醒和通知将可用，以便您可以发现您来自德克萨斯州的 46 岁好友何时向您发送消息。

看起来是一个不错的应用程序，尽管 Sidekick 用户早就应该使用它了。显然，它也将被包括在即将到来的[搭档赞特](https://web.archive.org/web/20201031165244/http://crunchgear.com/2007/06/16/dangermotorola-zante-aka-slide-aka-sidekick-35-details-emerge/)中。很棒的男人。令人遗憾的是，MySpace 垃圾邮件泛滥，使用率下降。然而，一个移动脸书应用程序…
