# 第三屏幕影院寻找暴走族

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/01/third-screen-cinema-looks-for-mobstars/>

你当然可以用手机拍摄视频，但你用它做什么呢？如果你在南非，你可以展示你作为终极独立电影制作人的作品。本月，一项名为 Mobstars 的新竞赛正在进行，它的目标是移动视频领域的创新者。提交可以在网上进行，领域将缩小到 10 个最好的移动电影制作人，他们的内容将在移动内容门户网站上。价格将包括 MacBook Pros，但真正的回报是你作为第三屏幕视频制作的创新者所能获得的尊重。
