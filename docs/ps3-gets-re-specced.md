# PS3 被重新指定

![](img/0255c95ee7132a4a342c67cc2069e598.png)

《问询者报》报道说 PS3 的规格又一次改变了，这次是在视频部门。它的英伟达 RSX GPU 已经从 550/700 核心/内存降级到 500/650。没有说明为什么会发生这种变化，但它标志着 PS3 即将发布的又一次最后时刻的修订。我想知道下一步会发生什么。
