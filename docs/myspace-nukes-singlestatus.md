# MySpace Nukes SingleStat.us

 [](https://web.archive.org/web/20211128174957/http://www.singlestat.us/) 好了， [SingleStat.us 实验](https://web.archive.org/web/20211128174957/http://www.beta.techcrunch.com/2006/06/05/find-out-when-your-friends-become-single/)让人们发现 MySpace 上有人改变了他们的关系状态。MySpace 友好的律师向创建 SingleStat.us 的工程师 David Weekly 发送了一封[停止通知函](https://web.archive.org/web/20211128174957/http://singlestat.us/)，要求他关闭该网站，并声称他的*“活动正在并将继续给 MySpace 造成重大且不可挽回的损害。”*

> 我们注意到您正在运营一个域名为 www.singlestat.us 的网站，您允许您网站的访问者调查任何 MySpace 用户的当前关系状态，并在该用户的状态发生变化时通过电子邮件接收更新。虽然这似乎是一个无伤大雅的想法，但您允许网站访问者调查 MySpace 用户关系状态的方式构成了一个自动脚本程序，MySpace 与其用户之间的使用条款协议第 8 节明确禁止使用该程序。此类程序给 MySpace 服务器带来了过度的负担，从而对网站的运行能力产生了不利影响。(着重部分由作者标明)

直觉告诉我，当 MySpace 说 SingleStat.us 给他们造成了“过度的服务器负担”时，他们可能有点夸大其词。如果说我从采访公司中学到了什么的话，那就是给一家小的、一个人创业的公司发停止函通常不会像你计划的那样奏效。恶业。然而，SingleStat.us 现在已经进入了 TechCrunch 的死池。
