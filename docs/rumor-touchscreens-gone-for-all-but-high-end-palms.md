# 谣言:除了高端掌上电脑，触摸屏已经消失

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/11/rumor-touchscreens-gone-for-all-but-high-end-palms/>

我们从 Palm 谣言箱收到了另一条小道消息。除了最高端的 Treos 之外，该公司似乎正在逐步淘汰所有产品的触摸屏。这意味着今年和明年推出的 Gandolf 和一整套新 Treos 将没有触摸屏。这将如何工作谁也说不准，但我们怀疑这是一个削减成本的措施，也表明 Palm OS 将变得相当不流行。为什么甘多夫会有键盘？
