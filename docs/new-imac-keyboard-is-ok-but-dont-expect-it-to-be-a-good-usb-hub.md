# 新的 iMac 键盘不错，但不要指望它是一个好的 USB 集线器

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/08/10/new-imac-keyboard-is-ok-but-dont-expect-it-to-be-a-good-usb-hub/>

[![newimackeyboardtm.jpg](img/4adaf24812aa1f20d995ec8f4ee99a4d.png)](https://web.archive.org/web/20150912092226/http://old.crunchgear.com/wp-content/uploads/newimackeyboardtm.jpg "newimackeyboardtm.jpg")

ThinkMac 是 NewsLife RSS 阅读器的幕后推手(我以前用过它的前身 NewsMac)，它刚刚拿到了苹果新的 iMac 键盘，并向我们提交了一份简短的、有点褒贬不一的评论。这里没有愚蠢的拆箱，没有“男孩很性感”，只是一些辛勤工作的 Mac 开发者的意见。

键盘本身很好:比旧款薄得多，但与 MacBook one 相似，如果不是更坚固一点的话。然而，有一个相当大的警告:键盘非常耗电。因此，这些内置的 USB 端口只能用于最基本的设备，如廉价的拇指驱动器。不要指望插上你的 500GB 硬盘或笨重的存储卡读卡器。嗯，听起来就像我现在的旧 iMac 键盘。

键盘的软件也有问题。为什么需要一个 30MB 的驱动程序来告诉 F4 键打开 iTunes？可以肯定的是，这是一个小问题，但苹果正在将这款键盘作为其迄今为止最好的一款来销售。

那么它是一个好的键盘吗？嗯，是的，它是。如果你能习惯类似 MacBook 的按键，并且不介意必须使用额外的外部 USB 集线器，那么你应该会觉得花 50 美元买它没什么问题。
