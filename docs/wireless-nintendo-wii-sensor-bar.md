# 无线任天堂 Wii 感应条

> 原文：<https://web.archive.org/web/http://techcrunch.com/2006/12/12/wireless-nintendo-wii-sensor-bar/>

![](img/ee133f9ee3baac386e7b0aab909afd70.png)

如果你喜欢把你的任天堂 Wii 挂在不同的位置，或者想要在你的电视后面少一根电缆，无线 Wii 感应条绝对是值得一试的。无线感应条的功能与普通 Wii 感应条相同，只是它使用 9V 电池。这并不是一件太坏的事情，你只需要确保当你不在的时候你的 Wii 不会开着。无线传感条网站并没有确切说明传感条可以离 Wii 多远，但有可能你不会把传感条放在离你的主机太远的地方。唯一的问题是，它有黑色的吗？
