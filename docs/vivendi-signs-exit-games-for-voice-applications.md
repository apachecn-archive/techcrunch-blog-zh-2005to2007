# Vivendi 签署退出语音应用游戏

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/06/21/vivendi-signs-exit-games-for-voice-applications/>

# 威望迪签署退出语音应用游戏

Exit Games 已经与 Vivendi 签约合作开发手机游戏，尤其是在提供 VoIP 和其他语音及集成功能方面。该协议的第一个版本将是新的海战。如果好友列表中有人邀请他们玩游戏，用户的手机也会被“唤醒”。

Vivendi Games Mobile 的总裁 Paul Maglione 表示:“Neutron 技术的引入将带来复杂的多人游戏体验，这将对手机游戏玩家产生广泛的吸引力。

Exit Games 的 Neutron 4.0 支持 50 多个国家的游戏直播。
