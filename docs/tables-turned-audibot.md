# 桌子变成了听觉机器人

> 原文：<https://web.archive.org/web/http://techcrunch.com/2007/04/26/tables-turned-audibot/>

![audibot-horz.jpg](img/6a7089a5deaa6a6c6d2c2d3c75d06f91.png)

不，这不是一个新的改造运动跑车。这是一个基于 Linux 的广播盒，它连接到互联网，流式传输任何音频源，然后通过 iTunes 或任何 RSS 兼容的 MP3 服务进行下载。

这款设备的专业版售价为 1200 美元，由废旧电脑制成的“重生”版售价为 600 美元。该公司还为您的设备提供托管计划，或者您可以通过自己的网站进行流式传输。
